На этой странице перечислены стандартные сочетания клавиш для навигации и редактирования текста в Obsidian. Эти сочетания предоставляются вашей операционной системой или фреймворком, на котором построен Obsidian, и не могут быть настроены внутри Obsidian.

Информацию о настраиваемых сочетаниях клавиш для команд Obsidian см. в разделе [[Горячие клавиши]].

## Сочетания клавиш для Windows и Linux

### Общие действия

| Действие | Сочетание клавиш |
|-|-|
| Копировать | `Ctrl+C` |
| Вырезать | `Ctrl+X` |
| Вставить | `Ctrl+V` |
| Вставить без форматирования | `Ctrl+Shift+V` |
| Отменить | `Ctrl+Z` |
| Повторить | `Ctrl+Shift+Z` или `Ctrl+Y` |
| Копировать абзац | `Ctrl+C` (без выделенного текста) |
| Вырезать абзац | `Ctrl+X` (без выделенного текста) |

### Редактирование текста

| Действие | Сочетание клавиш |
|-|-|
| Вставить новую строку | `Enter` |
| Удалить предыдущий символ | `Backspace` |
| Удалить следующий символ | `Delete` |
| Удалить предыдущее слово | `Ctrl+Backspace` |
| Удалить следующее слово | `Ctrl+Delete` |
| Удалить текущую строку | `Ctrl+Shift+K` (без выделенного текста) |

### Навигация по тексту

| Действие | Сочетание клавиш |
|-|-|
| Переместить курсор на один символ | `Left/→` |
| Переместить курсор к началу предыдущего слова | `Ctrl+←` |
| Переместить курсор к концу следующего слова | `Ctrl+→` |
| Переместить курсор к началу текущей строки | `Home` |
| Переместить курсор к концу текущей строки | `End` |
| Переместить курсор на предыдущую строку | `↑` |
| Переместить курсор на следующую строку | `↓` |
| Переместить курсор к началу заметки | `Ctrl+Home` |
| Переместить курсор к концу заметки | `Ctrl+End` |
| Переместить курсор на одну страницу вверх | `Page up` |
| Переместить курсор на одну страницу вниз | `Page down` |

### Выделение текста

| Действие | Сочетание клавиш |
|-|-|
| Упростить выделение | `Escape` |
| Выбрать все объекты | `Ctrl+A` |
| Расширить выделение на один символ | `Shift+Left/→` |
| Расширить выделение до начала предыдущего слова | `Ctrl+Shift+←` |
| Расширить выделение до конца следующего слова | `Ctrl+Shift+→` |
| Расширить выделение до начала текущей строки | `Shift+Home` |
| Расширить выделение до конца текущей строки | `Shift+End` |
| Расширить выделение до начала заметки | `Ctrl+Shift+Home` |
| Расширить выделение до конца заметки | `Ctrl+Shift+End` |
| Расширить выделение на одну страницу вверх | `Shift+Page up` |
| Расширить выделение на одну страницу вниз | `Shift+Page down` |

| Действие | Сочетание клавиш |
