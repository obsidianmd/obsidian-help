Мобильное приложение Obsidian для iOS и iPadOS обеспечивает мощные возможности ведения заметок на вашем iPhone и iPad. Вы можете загрузить его из [Apple App Store](https://apps.apple.com/us/app/obsidian-connected-notes/id1557175442).

На этой странице описаны функции, специфичные для iOS, включая виджеты, интеграцию с Siri и быстрые команды.

## Синхронизация

Информацию о синхронизации заметок между устройствами см. в разделе [[Синхронизация заметок между устройствами]].

## Виджеты

Obsidian для iOS предлагает несколько виджетов для быстрых действий с вашим хранилищем.

> [!note] Примечание
> Виджеты доступны на iOS и iPadOS 18 и выше.
> Виджеты недоступны при использовании функции «Требовать Face ID» для разблокировки приложения.


### Виджеты экрана блокировки и Пункта управления

Виджеты экрана блокировки и Пункта управления позволяют:
- Создать новую заметку
- Открыть определённую заметку
- Открыть ежедневную заметку
- Открыть поиск
- Открыть Obsidian

### Виджеты домашнего экрана

Виджеты домашнего экрана позволяют:
- Создать заметку
- Просмотреть заметку
- Открыть ежедневную заметку

### Настройка виджетов

Вы можете настроить виджеты под свой рабочий процесс, например, выбрать хранилище или указать конкретную заметку для открытия.

- **Виджеты домашнего экрана:** нажмите и удерживайте виджет, затем выберите **Редактировать виджет**.
- **Виджеты экрана блокировки:** нажмите и удерживайте экран блокировки, нажмите **Настроить**, выберите экран блокировки, затем нажмите на виджет, который хотите настроить.
- **Виджеты Пункта управления:** откройте Пункт управления, нажмите кнопку **+** в левом верхнем углу для начала редактирования, затем нажмите на виджет, который хотите настроить.

## Быстрые команды

Obsidian интегрируется с приложением Apple «Быстрые команды», позволяя создавать мощные автоматизации. Доступные команды включают:

- **Открыть заметку** — открыть определённую заметку в хранилище
- **Создать новую заметку** — создать новую заметку в хранилище
- **Открыть ежедневную заметку** — перейти непосредственно к сегодняшней ежедневной заметке
- **Добавить в ежедневную заметку** — добавить текст в начало или конец ежедневной заметки без открытия приложения Obsidian
- **Добавить в закладку** — добавить текст в начало или конец заметки из закладок без открытия приложения Obsidian

Команды для быстрого сохранения особенно полезны для оперативного создания заметок, так как они позволяют добавлять содержимое в заметку в фоновом режиме.

## Меню «Поделиться»

Меню «Поделиться» в Obsidian позволяет захватывать содержимое с веб-страниц. Оно также работает с такими приложениями, как YouTube и другие социальные сети.

> [!note]
> - Встроенное меню «Поделиться» доступно на iOS и iPadOS 18 и выше.
> - Функции меню «Поделиться», описанные в этом разделе, требуют Obsidian 1.13.0 или новее.

Используйте меню «Поделиться», чтобы быстро отправить содержимое из другого приложения в Obsidian:
1. В другом приложении нажмите кнопку **Поделиться**.
2. Выберите **Obsidian**.
3. Выберите расположение.
4. Просмотрите или отредактируйте захваченное содержимое.
5. Нажмите **Сохранить**.

### Расположения

Расположения позволяют решить, куда должно попасть содержимое, перед тем как вы его сохраните.

Расположения могут сохранять в:
- **Новая заметка** — создать новую заметку в хранилище или папке.
- **Ежедневная заметка** — добавить содержимое в начало или конец сегодняшней ежедневной заметки.
- **Заметка из закладок** — добавить содержимое в начало или конец заметки из закладок.
- **Заметка** — выбрать существующую заметку в хранилище.
- **Новая закладка** — сохранить переданный URL в закладки Obsidian.

![[ios-share-sheet-locations.png|400]]

### Настройка расположений

Вы можете создавать расположения для типичных рабочих процессов, например, сохранение статей во входящие, добавление цитат в ежедневную заметку или добавление ссылок в закладки.

Чтобы настроить расположения:

1. Откройте Obsidian из меню «Поделиться» в iOS.
2. Нажмите на текущее расположение на панели инструментов.
3. Нажмите кнопку **+**, чтобы создать новое расположение, или выберите существующее для редактирования.
4. Выберите хранилище, поведение и дополнительные настройки.

В зависимости от типа `Поведения` вы можете настроить такие параметры, как:
- Папка
- Шаблон
- Группа закладок
- Позиция добавления (в начало или в конец)
- Захватывать ли для переданных ссылок **Полный текст** или только **URL**

![[ios-share-sheet-add-location.png|400]]

### Использование шаблона при отправке

Вы можете использовать шаблон при отправке содержимого из меню «Поделиться». Шаблоны позволяют форматировать захваченное веб-содержимое с такими деталями, как заголовок страницы, автор, исходный сайт и дата публикации.

Чтобы настроить расположение с шаблоном:

1. Откройте Obsidian из меню «Поделиться» в iOS.
2. Нажмите на текущее расположение на панели инструментов.
3. Нажмите кнопку **+**, чтобы создать новое расположение.
4. Введите имя для расположения.
5. Выберите хранилище.
6. Установите **Поведение** на **Новая заметка**.
7. В разделе **Дополнительно** нажмите **Шаблон**.
8. Выберите заметку из хранилища для использования в качестве шаблона.
9. Нажмите **Сохранить**, чтобы сохранить расположение.

![[ios-share-sheet-set-template.png|400]]

Когда вы делитесь ссылкой с использованием этого расположения, Obsidian сначала применяет шаблон, а затем добавляет переданное содержимое.

Поддерживаемые переменные шаблона:

| Переменная | Описание |
| --- | --- |
| `{{author}}` | Автор статьи |
| `{{description}}` | Описание или краткое содержание статьи |
| `{{domain}}` | Доменное имя сайта |
| `{{favicon}}` | URL фавикона сайта |
| `{{image}}` | URL основного изображения статьи |
| `{{published}}` | Дата публикации статьи в формате даты по умолчанию |
| `{{published: YYYY-MM-DD}}` | Дата публикации в пользовательском формате даты |
| `{{site}}` | Название сайта |
| `{{title}}` | Заголовок статьи |
| `{{url}}` | URL статьи |
| `{{wordCount}}` | Общее количество слов в извлечённом содержимом |

Вы также можете использовать стандартные переменные шаблона для даты и времени:

| Переменная | Описание |
| --- | --- |
| `{{date}}` | Текущая дата |
| `{{date: YYYY-MM-DD}}` | Текущая дата в пользовательском формате |
| `{{time}}` | Текущее время |
| `{{time: HH:mm}}` | Текущее время в пользовательском формате |

## Интеграция с Siri

Вы можете использовать голосовые команды Siri для взаимодействия с Obsidian:

- «Capture using Obsidian»
- «Capture to Obsidian»
- «Open my daily note in Obsidian»
- «Search in Obsidian»

## Интеграция со Spotlight

При поиске «Obsidian» в iOS Spotlight вы увидите быстрые действия:
- Новая заметка
- Поиск
- Ежедневная заметка
