[[Введение в Obsidian Web Clipper|Web Clipper]] включает встроенный режим чтения под названием **«Читатель»**, который убирает лишние элементы с веб-страниц, представляя только основное содержимое статьи в чистом, удобном для чтения формате.

## Активация режима чтения

Вы можете активировать Читатель несколькими способами:

- ![[lucide-book-icon.svg#icon]] **Значок книги** в панели расширения.
- **Сочетание клавиш**, настраиваемое в настройках расширения.
- **Контекстное меню** — щёлкните правой кнопкой мыши по веб-странице, которую вы просматриваете.

Вы также можете настроить кнопку расширения ![[obsidian.svg#icon|20]] для открытия Читателя одним кликом. В настройках Web Clipper перейдите в **Основные** → **Поведение** → **Поведение при открытии** и выберите **Читатель**.

Все перечисленные выше варианты также позволяют выйти из Читателя и восстановить исходную страницу.

## Как это работает

Читатель извлекает основное содержимое со страницы. Исходная страница заменяется чистым представлением для чтения, которое включает:

- Заголовок статьи, автора, дату публикации и исходный домен.
- Основное содержимое статьи с сохранением изображений и форматирования.
- Структуру в боковой панели для статей с несколькими заголовками.
- Подсветку синтаксиса для блоков кода.
- Всплывающие окна для сносок, которые отображаются при нажатии прямо в тексте.

## Настройка представления для чтения

Настройте свой опыт чтения в настройках расширения. Некоторые параметры также доступны в режиме Читателя при нажатии на значок ![[lucide-case-sensitive.svg#icon]].

| Настройка                    | Описание                                        |
| ---------------------------- | ----------------------------------------------- |
| Шрифт                        | Добавление шрифтов, установленных в вашей системе. |
| Размер шрифта                | Настройка размера текста.                       |
| Высота&nbsp;строки           | Расстояние между строками.                      |
| Ширина&nbsp;строки           | Максимальная ширина области содержимого.         |
| Внешний вид                  | Выбор светлого или тёмного оформления.          |
| Тема                         | Выбор цветовой схемы.                           |
| Пользовательский CSS         | Дополнительный CSS, применяемый к странице.      |
