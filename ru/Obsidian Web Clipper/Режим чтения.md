[[Введение в Obsidian Web Clipper|Web Clipper]] включает встроенный режим чтения под названием «Читатель», который убирает лишние элементы с веб-страниц, представляя только основное содержимое статьи в чистом, удобном для чтения формате.

## Активация режима чтения

Вы можете активировать Читатель несколькими способами:

- **Значок книги** в заголовке панели расширения.
- Сочетание клавиш `Alt+Shift+R`, настраиваемое в настройках расширения.
- Повторное переключение режима чтения восстановит исходную страницу.

## Как это работает

Читатель извлекает основное содержимое со страницы. Исходная страница заменяется чистым представлением для чтения, которое включает:

- Заголовок статьи, автора, дату публикации и исходный домен.
- Основное содержимое статьи с сохранением изображений и форматирования.
- Структуру в боковой панели для статей с несколькими заголовками.
- Подсветку синтаксиса для блоков кода.
- Всплывающие окна для сносок, которые отображаются при нажатии прямо в тексте.

## Настройка представления для чтения

Режим Читателя включает скрытую панель настроек, которую можно отобразить, наведя курсор на правый верхний угол страницы. Панель содержит элементы управления для настройки удобства чтения.

| Настройка                    | Описание                                                                                                          |
| ---------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| Размер шрифта                | Настройка размера текста.                                                                                         |
| Ширина строки                | Максимальная ширина области содержимого.                                                                          |
| Высота строки                | Расстояние между строками.                                                                                        |
| Цветовая&nbsp;схема          |                                                                                                                   |
| Тема                         | Автоматически — следует системным настройкам. Или выберите постоянное использование светлой или тёмной темы.       |
