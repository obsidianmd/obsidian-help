После установки расширения [[Введение в Obsidian Web Clipper|Web Clipper]] для браузера вы можете получить к нему доступ несколькими способами, в зависимости от вашего браузера:

1. Значок Obsidian на панели инструментов браузера.
2. Сочетания клавиш для активации расширения с клавиатуры.
3. Контекстное меню, вызываемое правым щелчком мыши на посещаемой веб-странице.

Чтобы сохранить страницу в Obsidian, нажмите кнопку **Добавить в Obsidian**.

## Сохранение страницы

При открытии расширения Web Clipper извлекает данные с текущей веб-страницы в соответствии с настройками вашего [[Obsidian Web Clipper/Шаблоны|шаблона]]. Вы можете создавать собственные шаблоны и настраивать результат с помощью [[Переменные|переменных]] и [[Фильтры|фильтров]].

По умолчанию Web Clipper пытается интеллектуально извлечь только основное содержимое статьи, исключая другие элементы на странице. Однако вы можете переопределить это поведение следующими способами:

- Если задан пользовательский шаблон, используется ваш шаблон.
- Если на странице есть выделение, используется выделенный текст. Вы можете использовать `Ctrl/Cmd+A`, чтобы выделить всю страницу.
- Если на странице есть [[Выделение на веб-страницах|выделенные маркером фрагменты]], используются они.

## Загрузка изображений

Изображения не загружаются автоматически при использовании Web Clipper. Вместо этого изображения ссылаются на свои URL-адреса в интернете. Это экономит место в хранилище, но означает, что изображения не будут доступны в автономном режиме или если URL-адрес перестанет работать.

Вы можете загрузить изображения для любого файла в Obsidian с помощью [[Палитра команд|команды]] **Загрузить вложения к текущему файлу**. Эту команду также можно назначить на сочетание клавиш в Obsidian.

## Сочетания клавиш

Web Clipper включает сочетания клавиш, которые помогут ускорить рабочий процесс. Чтобы изменить назначения клавиш, перейдите в **Настройки Web Clipper** → **Общие** и следуйте инструкциям для вашего браузера. Назначения можно изменить во всех браузерах, кроме Safari, который не поддерживает редактирование сочетаний клавиш.

| Действие                | macOS         | Windows/Linux  |
| ----------------------- | ------------- | -------------- |
| Открыть расширение      | `Cmd+Shift+O` | `Ctrl+Shift+O` |
| Быстрое сохранение      | `Opt+Shift+O` | `Alt+Shift+O`  |
| Режим маркера           | `Opt+Shift+H` | `Alt+Shift+H`  |

## Функции интерфейса

Интерфейс Web Clipper разделён на четыре секции:

1. **Заголовок** — здесь можно переключать шаблоны, включать [[Выделение на веб-страницах|маркер]], [[Режим чтения|режим чтения]] и открывать настройки.
2. **Свойства** — показывает [[Свойства|метаданные]], извлечённые со страницы, которые будут сохранены как [[Свойства]] в Obsidian.
3. **Содержимое заметки** — контент, который будет сохранён в Obsidian.
4. **Нижняя панель** — позволяет выбрать хранилище и папку, а также добавить в Obsidian.

Функции заголовка:

- Выпадающий список **Шаблон** для переключения между сохранёнными [[Obsidian Web Clipper/Шаблоны|шаблонами]], добавленными в настройках Web Clipper.
- Кнопка **Далее (...)** для отображения переменных страницы, которые можно использовать в шаблонах.
- Кнопка **Маркер** для включения [[Выделение на веб-страницах|выделения]].
- Кнопка **Шестерёнка** для открытия настроек Web Clipper.

Функции нижней панели:

- Кнопка **Добавить в Obsidian** для сохранения данных в Obsidian.
- Выпадающий список **Хранилище** для переключения между сохранёнными хранилищами, добавленными в настройках Web Clipper.
- Поле **Папка** для указания папки сохранения.
- **Интерпретатор** для выполнения [[Интерпретация веб-страниц|запросов на естественном языке]] к странице.
