[[Введение в Obsidian Web Clipper|Web Clipper]] позволяет выделять текст на веб-страницах и выбирать элементы, которые вы хотите сохранить в Obsidian. Ваши выделения сохраняются, поэтому вы можете вернуться к ним при повторном посещении страницы.

Выделения можно [[Сохранение веб-страниц|захватить]] и сохранить в Obsidian при открытии расширения.

## Включение маркера

Вы можете включить выделение несколькими способами в зависимости от вашего браузера:

- Значок маркера на панели расширения.
- Сочетания клавиш для активации расширения с клавиатуры.
- Контекстное меню, вызываемое правым кликом по посещаемой веб-странице.

После включения выделения вы можете выбирать текст, изображения и элементы, которые хотите выделить.

## Настройки маркера

Вы можете изменить режим работы маркера в настройках Web Clipper. Здесь же можно экспортировать ваши выделения в файл `.json`.

Существует три варианта вставки выделений в сохранённую заметку через [[Переменные|переменную]] `{{content}}`:

- **Выделить содержимое страницы** — добавляет выделения непосредственно в текст с использованием [[Obsidian Flavored Markdown|синтаксиса]] `==выделение==`.
- **Заменить содержимое страницы** — возвращает список выделений без содержимого страницы.
- **Ничего не делать** — возвращает исходное содержимое без выделений.

Вы можете добавить выделения непосредственно в шаблон с помощью переменной `{{highlights}}`, например:

```
{{highlights|map: item => item.text|join:"\n\n"}}
```
