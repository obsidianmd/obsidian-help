Узнайте, как расширить возможности Obsidian с помощью плагинов, созданных сообществом. Используйте плагины, чтобы адаптировать Obsidian под свои конкретные потребности, например для поддержки дополнительных форматов файлов или интеграции со сторонними сервисами.

> [!warning] Предупреждение
> Плагины сообщества выполняют сторонний код от вашего имени, который потенциально может нанести вред. Чтобы узнать больше о том, что команда Obsidian делает для предотвращения вредоносных плагинов, обратитесь к разделу [[Безопасность плагинов]].

## Обзор плагинов сообщества

1. Откройте **[[Настройки]]**.
2. Выберите **Включите плагины сообщества**. Для получения дополнительной информации обратитесь к разделу [[Безопасность плагинов#Ограниченный режим|Ограниченный режим]].
3. Выберите **Просмотр**, чтобы отобразить список всех доступных плагинов сообщества.

Используйте текстовое поле для фильтрации плагинов по названию, автору и описанию.

Вы также можете просмотреть доступные плагины в браузере, перейдя на [obsidian.md/plugins](https://obsidian.md/ru/plugins).

## Установка плагина сообщества

Чтобы установить плагин сообщества, сначала необходимо отключить [[Безопасность плагинов#Ограниченный режим|Ограниченный режим]].

1. Откройте **[[Настройки]]**.
2. В боковом меню выберите **Плагины сообщества**.
3. Выберите **Просмотр**, чтобы просмотреть доступные плагины сообщества.
4. Выберите плагин, который хотите установить.
5. Выберите **Установить**.

Чтобы использовать установленный плагин, его необходимо включить.

## Включение плагина сообщества

Либо выберите **Включить** сразу после установки, либо включите его из списка плагинов сообщества в разделе **[[Настройки]] → Плагины сообщества → Управление плагинами**.

## Обновление плагинов

В целях безопасности плагины сообщества не обновляются автоматически. Вы можете обновить все плагины сразу или обновить отдельные плагины.

Чтобы обновить все плагины:

1. Откройте **[[Настройки]]**.
2. В разделе **Плагины сообщества → Установленные плагины** выберите **Проверить обновления**.
3. Если обновления доступны, выберите **Обновить все**.

Чтобы обновить отдельный плагин:

1. Откройте **[[Настройки]]**.
2. В разделе **Плагины сообщества → Установленные плагины** выберите **Проверить обновления**.
3. В разделе **Управление плагинами** выберите **Обновить** рядом с плагином, который хотите обновить.

## Удаление плагина сообщества

1. Откройте **[[Настройки]]**.
2. В боковом меню выберите **Плагины сообщества**.
3. В разделе **Управление плагинами** нажмите значок корзины ![[lucide-trash-2.svg#icon]] рядом с плагином, который хотите удалить.
4. Выберите **Удалить** для подтверждения.

## Управление установленными плагинами

В разделе **[[Настройки]] → Плагины сообщества → Управление плагинами** у каждого плагина есть несколько значков для управления:

- **[[Настройки]]** ![[lucide-settings.svg#icon]] — открыть страницу настроек плагина, если у него есть настраиваемые параметры.
- **[[Горячие клавиши]]** ![[lucide-plus-circle.svg#icon]] — назначить сочетания клавиш для команд плагина.
- **Поддержать** ![[lucide-heart.svg#icon]] — просмотреть ссылку для поддержки автора плагина.
- **Удалить** ![[lucide-trash-2.svg#icon]] — удалить плагин из вашего хранилища.
- **Переключатель** — включить или отключить плагин без его удаления.

Вы также можете:

- Нажать значок **обновления**, чтобы перезагрузить все плагины.
- Нажать значок **папки** ![[lucide-folder-open.svg#icon]], чтобы открыть папку плагинов в вашей [[Папка конфигурации|папке конфигурации]].
- Использовать **строку поиска** для фильтрации плагинов по названию.
