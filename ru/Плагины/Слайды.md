Слайды — это [[Основные плагины|основной плагин]], который позволяет создавать презентации из ваших заметок. Чтобы начать презентацию:

**Меню файла:**

- Щёлкните правой кнопкой мыши по вкладке заметки и выберите **Начать презентацию**.

**Палитра команд:**

1. Нажмите `Ctrl+P` (или `Cmd+P` на macOS), чтобы открыть [[Палитра команд|палитру команд]].
2. Найдите команду **Начать презентацию**.
3. Нажмите `Enter`, когда команда выделена, чтобы начать презентацию с использованием текущей заметки.

Для перемещения между слайдами нажимайте на стрелки влево и вправо в правом нижнем углу или используйте клавиши со стрелками влево и вправо на клавиатуре.

Также можно перейти к следующему слайду, нажав клавишу «Пробел».

Чтобы остановить презентацию, нажмите `Escape` или щёлкните по крестику в правом верхнем углу презентации.

Вы можете использовать любой допустимый файл Markdown в качестве презентации. Чтобы разделить слайды, введите `---` в начале строки, окружённой пустыми строками.

```md
# Презентации с помощью Слайдов

Демонстрация создания презентаций с помощью плагина Слайды.

---

## Форматирование

Вы можете использовать обычное форматирование Markdown, например *курсив* и **жирный** текст.

---

## Слайды

Используйте `---` для разделения слайдов.
```
