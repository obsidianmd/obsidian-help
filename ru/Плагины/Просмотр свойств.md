Просмотр свойств — это [[Основные плагины|основной плагин]], который добавляет два представления на боковую панель, которые можно использовать для управления [[Свойства|свойствами]]:

- **Свойства файла** отображает свойства текущей заметки.
- **Все свойства** показывает список всех свойств в вашем хранилище и их тип.

## Представление «Все свойства»

С помощью представления «Все свойства» вы можете:

- Сортировать свойства по имени или по частоте использования (общему количеству использований в хранилище).
- Нажать на свойство, чтобы открыть представление [[Поиск]] с предзаполненным синтаксисом [[Поиск#Поиск по свойствам|поиска по свойствам]].
- Щёлкнуть правой кнопкой мыши по свойству, чтобы переименовать его глобально.
