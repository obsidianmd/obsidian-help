[[Создатель уникальных заметок]] — это [[Основные плагины|основной плагин]], который позволяет создавать заметки с именами на основе времени, также известные как заметки Zettelkasten.

Например, если вы создадите новую заметку в 09:45 1 января 2024 года, Создатель уникальных заметок создаст заметку с именем `202401010945`. Если заметка с таким же именем уже существует, новая заметка получит следующую доступную временную метку.

## Создание уникальной заметки

**Лента:**

- На вертикальной панели нажмите **Создать заметку с уникальным идентификатором** ![[obsidian-icon-sheets-in-box.svg#icon]].

**Палитра команд:**

1. Нажмите `Ctrl+P` (или `Cmd+P` на macOS), чтобы открыть [[Палитра команд|палитру команд]].
2. Найдите команду **Создать заметку с уникальным идентификатором**.
3. Нажмите `Enter`, когда команда будет выделена.

## Создание уникальных заметок из шаблона

По умолчанию новые уникальные заметки пусты. Чтобы использовать другой файл в качестве шаблона:

- Откройте **[[Настройки]]**.
- На боковой панели нажмите **Встроенные плагины → Уникальный идентификатор**.
- В поле **Расположение шаблона** укажите файл, который вы хотите использовать в качестве шаблона.

Для получения дополнительной информации о написании шаблонов обратитесь к разделу [[Шаблоны]].
