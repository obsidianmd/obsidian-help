«Рабочие пространства» — это [[Основные плагины|основной плагин]], который позволяет управлять различными макетами приложения и переключаться между ними в зависимости от вашей задачи, например ведение дневника, чтение или написание текстов.

Рабочее пространство содержит информацию об открытых файлах и вкладках, а также о ширине и видимости каждой боковой панели.

## Сохранение рабочего пространства

**Вертикальная панель:**

1. На вертикальной панели нажмите **Пространства** ( ![[lucide-panels-top-left.svg#icon]] ).
2. Введите имя для рабочего пространства.
3. Нажмите **Сохранить**.

**Палитра команд:**

1. Нажмите `Ctrl+P` (или `Cmd+P` на macOS), чтобы открыть [[Палитра команд|палитру команд]].
2. Найдите команду **Пространства**.
3. Нажмите `Enter`, когда команда будет выбрана.
4. Введите имя для рабочего пространства.
5. Нажмите **Сохранить**.

Чтобы обновить существующее рабочее пространство, сохраните его с тем же именем, что и у пространства, которое вы хотите обновить.

## Загрузка рабочего пространства

**Вертикальная панель:**

1. На вертикальной панели нажмите **Пространства** ( ![[lucide-panels-top-left.svg#icon]] ).
2. Рядом с нужным рабочим пространством нажмите **Загрузить**.

**Палитра команд:**

1. Нажмите `Ctrl+P` (или `Cmd+P` на macOS), чтобы открыть [[Палитра команд|палитру команд]].
2. Найдите команду **Пространства**.
3. Нажмите `Enter`, когда команда будет выбрана.
4. Выберите рабочее пространство для загрузки.

## Удаление рабочего пространства

**Вертикальная панель:**

1. На вертикальной панели нажмите **Пространства** ( ![[lucide-panels-top-left.svg#icon]] ).
2. Рядом с нужным рабочим пространством нажмите **Удалить пространство** ( ![[lucide-x.svg#icon]] ).

**Палитра команд:**

1. Нажмите `Ctrl+P` (или `Cmd+P` на macOS), чтобы открыть [[Палитра команд|палитру команд]].
2. Найдите команду **Пространства**.
3. Нажмите `Enter`, когда команда будет выбрана.
4. Рядом с нужным рабочим пространством нажмите **Удалить пространство** ( ![[lucide-x.svg#icon]] ).
