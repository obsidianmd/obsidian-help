[[Введение в Obsidian Sync|Obsidian Sync]] регулярно проверяет обновления ваших [[Настройки синхронизации и выборочная синхронизация|синхронизируемых файлов]], ведя учёт всех изменений. Они сохраняются как новые записи в [[#История версий]]. Эту информацию можно получить несколькими способами:

- [[#История синхронизации]]
- [[#История версий]]

Хотя это не является частью Obsidian Sync, локальная история версий также доступна на каждом устройстве, если включён плагин [[Восстановление файлов]].

## История синхронизации

Функция истории синхронизации (или боковой панели синхронизации), представленная в Obsidian версии 1.7, позволяет быстро просматривать недавно созданные или изменённые заметки и вложения, которые были синхронизированы. Её также можно рассматривать как историю _редактирования_.

На боковой панели вы можете выбрать элемент, чтобы открыть его файл в активном окне. У элементов также есть контекстное меню, позволяющее выполнять такие действия, как перемещение файла или просмотр его [[#Заметки и вложения|истории версий]].

> [!compatibility] Функция только для десктопной версии
> При наведении курсора на синхронизированный файл в боковой панели вы можете увидеть, кто последним редактировал файл. Это особенно полезно, если вы [[Совместная работа в общем хранилище|работаете совместно]] в общем хранилище.

> [!tip] 
> Настройки и удалённые элементы не отображаются в боковой панели истории синхронизации. Их можно найти в [[#Заметки и вложения|истории версий]].

### Показать историю синхронизации

При включении основного плагина [[Введение в Obsidian Sync|Sync]] история синхронизации автоматически активируется, но по умолчанию не отображается на боковой панели. Вам нужно добавить её вручную с помощью команды или сочетания клавиш.

#### Включение через палитру команд

> [!info] Для этого варианта необходимо, чтобы был включён основной плагин [[Палитра команд]].

**Десктоп/планшет** ![[lucide-monitor-check.svg#icon]]

1. Откройте **палитру команд**. ( ![[lucide-terminal.svg#icon]] )
2. Введите «Sync».
3. Выберите пункт «Sync: Show Sync history».

История синхронизации появится в [[Боковая панель|правой боковой панели]].

**Мобильный** ![[obsidian-icon-smartphone.svg#icon]]

1. Откройте [[Лента|ленту]] ( ![[lucide-menu.svg#icon]] ).
2. Откройте **палитру команд**. ( ![[lucide-terminal.svg#icon]] )
3. Введите «Sync».
4. Выберите пункт «Sync: Show Sync history».

История синхронизации появится как выпадающий пункт в [[Боковая панель|правой боковой панели]].

#### Включение через сочетание клавиш

1. Откройте **[[Настройки]]**.
2. В категории **Настройки** выберите **Сочетания клавиш**.
3. В строке поиска сочетаний клавиш введите «Sync».
4. Рядом с «Sync: Show Sync history» назначьте предпочтительное сочетание клавиш.

## История версий

Помимо [[#История синхронизации|истории синхронизации]], Obsidian также ведёт историю версий для восстановления заметок и вложений. Если вы случайно удалили заметку или хотите вернуться к предыдущей версии, вы можете восстановить её из истории версий.

Срок хранения истории версий зависит от вашего [[Планы и лимиты хранилища|плана Obsidian Sync]]. На плане Standard заметки хранятся 1 месяц, а на плане Plus — 12 месяцев. По истечении этого периода старые версии заметок удаляются.

Для [[Вложения|вложений]] старые версии хранятся <u>две недели</u>.

![[Совместная работа в общем хранилище#^version-history-image]]

### Заметки и вложения

Процесс восстановления заметок и вложений одинаков.

#### Просмотр истории версий файла

**Десктоп/планшет** ![[lucide-monitor-check.svg#icon]]
1. В **файловом менеджере** выберите заметку, которую хотите восстановить.
2. Выберите **Открыть историю версий**.
3. Выберите нужную версию заметки слева. Содержимое будет отображено справа.

**Мобильный** ![[obsidian-icon-smartphone.svg#icon]]
1. В **файловом менеджере** выберите заметку, которую хотите восстановить.
2. Удерживайте для вызова контекстного меню.
3. Выберите **Открыть историю версий**.
4. Во всплывающем меню выберите нужную версию заметки.
5. После выбора содержимое заметки будет доступно для просмотра.

#### Просмотр истории версий удалённого или переименованного файла

1. Откройте **[[Настройки]]**.
2. На боковой панели в разделе **Встроенные плагины** выберите **Sync**.
3. Рядом с **Удалённые файлы** выберите **Просмотр**.
4. Выберите заметку, историю версий которой хотите просмотреть.
5. Во всплывающем окне истории версий выберите нужную версию заметки слева.

#### Восстановление предыдущей версии файла

**Десктоп/планшет** ![[lucide-monitor-check.svg#icon]]
1. В **файловом менеджере** выберите заметку, которую хотите восстановить.
2. Выберите **Открыть историю версий**.
3. Выберите нужную версию заметки слева. Содержимое будет отображено справа.
4. Нажмите кнопку **Восстановить**.
5. Содержимое заметки будет заменено восстановленной версией.

**Мобильный** ![[obsidian-icon-smartphone.svg#icon]]
1. В **файловом менеджере** выберите заметку, которую хотите восстановить.
2. Удерживайте для вызова контекстного меню.
3. Выберите **Открыть историю версий**.
4. Во всплывающем меню выберите нужную версию заметки.
5. После выбора содержимое заметки будет доступно для просмотра.
6. Нажмите кнопку **Восстановить**.
7. Содержимое заметки будет заменено восстановленной версией.

#### Восстановление удалённого файла

1. Откройте **[[Настройки]]**.
2. На боковой панели в разделе **Встроенные плагины** выберите **Sync**.
3. Рядом с **Удалённые файлы** выберите **Просмотр**.
4. Выберите заметку, которую хотите восстановить.
5. В списке версий слева выберите версию, которую хотите восстановить.
6. Нажмите кнопку **Восстановить** справа.
7. Заметка будет восстановлена в исходное расположение в файловой системе.

> [!tip] Вы можете выбрать несколько заметок с помощью **массового восстановления**, отмечая флажки или используя `shift+click`. В этом меню предварительный просмотр файлов будет недоступен.

### История настроек

Obsidian Sync также отслеживает изменения, внесённые в настройки конфигурации хранилища.

#### Просмотр истории версий настройки

1. Откройте **[[Настройки]]**.
2. На боковой панели в разделе **Встроенные плагины** выберите **Sync**.
3. Прокрутите вниз до раздела **Настройка синхронизации хранилища**.
4. Нажмите кнопку **Просмотр** рядом с **Историю версий [[Настройки|настроек]]**.
5. Во всплывающем окне **Файл [[Настройки|настроек]]** выберите файл настроек, который хотите просмотреть.

#### Восстановление предыдущей версии настройки

1. Откройте **[[Настройки]]**.
2. На боковой панели в разделе **Встроенные плагины** выберите **Sync**.
3. Прокрутите вниз до раздела **Настройка синхронизации хранилища**.
4. Нажмите кнопку **Просмотр** рядом с **Историю версий [[Настройки|настроек]]**.
5. Во всплывающем окне **Файл [[Настройки|настроек]]** выберите файл настроек, который хотите просмотреть.
6. В окне изменений нажмите кнопку **Восстановить**.
7. Перезагрузите или перезапустите Obsidian, чтобы настройки вступили в силу. Подробнее см. в инструкциях в разделе [[Настройка Obsidian Sync#Настройка параметров Obsidian Sync|Настройка параметров Obsidian Sync]].
