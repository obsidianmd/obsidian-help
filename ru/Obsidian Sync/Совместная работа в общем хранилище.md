С помощью [[Введение в Obsidian Sync|Obsidian Sync]] вы можете совместно работать в общем хранилище с вашей командой.

Все участники должны иметь активную подписку на Sync для доступа к общему хранилищу. Присоединение к общему хранилищу не учитывается в вашем [[Часто задаваемые вопросы#Сколько удалённых хранилищ у меня может быть?|лимите хранилищ]].

Если удалённое хранилище [[Obsidian Sync/Безопасность и конфиденциальность|зашифровано сквозным шифрованием]], участники должны ввести пароль шифрования при настройке хранилища.

## Управление пользователями

### Добавление пользователей

Чтобы пригласить пользователя для совместного использования удалённого хранилища:

1. Откройте **[[Настройки]]**.
2. В боковом меню выберите **Синхронизация**.
3. Рядом с **Удалённое хранилище** выберите **Настроить**.
4. Рядом с удалённым хранилищем, которым вы хотите поделиться, выберите **Настроить общий доступ** ( ![[lucide-users.svg#icon]] ).
5. В поле **Пригласить пользователя** введите электронную почту пользователя, которого вы хотите пригласить.
6. Нажмите **Добавить**.

### Удаление пользователей

1. Откройте **[[Настройки]]**.
2. В боковом меню выберите **Синхронизация**.
3. Рядом с **Удалённое хранилище** выберите **Настроить**.
4. Рядом с пользователем, у которого вы хотите отозвать доступ, выберите **Удалить пользователя** ( ![[lucide-x.svg#icon]] ).

## Совместная работа с командой

### Разрешения

Детальное управление разрешениями пока не поддерживается. Все участники получают те же разрешения, что и владелец хранилища, за одним исключением: только владелец хранилища может приглашать участников.

### Редактирование в реальном времени

Общие хранилища позволяют командам работать вместе с набором файлов, однако Obsidian пока не поддерживает совместное редактирование одного файла в реальном времени. Вы не увидите курсор другого пользователя, а его изменения появятся только после синхронизации.

Если несколько пользователей одновременно редактируют один и тот же файл, [[Устранение неполадок Obsidian Sync#Разрешение конфликтов|изменения будут объединены]] в процессе синхронизации. Изменения можно просмотреть и восстановить с помощью [[История версий|Истории версий]].

## Ограничения

Обратите внимание, что Obsidian Sync имеет [[Часто задаваемые вопросы|ограничения]], которые могут затронуть вашу команду:

- Максимальное количество участников в общем хранилище — 20 пользователей.
- Максимальный размер файла для вложений зависит от [[Планы и лимиты хранилища|плана]] хоста вашего удалённого хранилища: 5 МБ для стандартного плана и 200 МБ для плана Plus.
