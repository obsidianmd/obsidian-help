**Хранилище** — это папка в вашей файловой системе, которая содержит ваши заметки, [[Вложения|вложения]] и [[Папка конфигурации|папку конфигурации]] с настройками Obsidian. Подробнее о хранилищах смотрите в разделе [[Хранение данных в Obsidian]].

Вы можете управлять хранилищами с помощью **Переключателя хранилищ**. *Профиль хранилища* открывается при первом запуске Obsidian.

Чтобы открыть переключатель хранилищ из существующего хранилища, нажмите **Профиль хранилища** ( ![[lucide-chevrons-up-down.svg#icon]]) в нижней части [[Боковая панель|левой боковой панели]]. Или выберите **Открыть хранилище** в [[Палитра команд|палитре команд]].

## Создание нового хранилища

1. Откройте Obsidian на компьютере.
2. В левом нижнем углу нажмите **Профиль хранилища** ( ![[lucide-chevrons-up-down.svg#icon]]).
3. Появится контекстное меню. Выберите **Управление хранилищами...**.
3. Справа от **Новое хранилище** нажмите **Создать**.
4. В поле **Название** введите имя хранилища.
5. Нажмите **Просмотр**, чтобы выбрать место для нового хранилища.
6. Нажмите **Создать**.

## Создание хранилища из существующей папки

1. Откройте Obsidian на компьютере.
2. В левом нижнем углу нажмите **Профиль хранилища** ( ![[lucide-chevrons-up-down.svg#icon]]).
3. Появится контекстное меню. Выберите **Управление хранилищами...**.
3. Справа от **Открыть папку как хранилище** нажмите **Открыть**.
4. В файловом менеджере выберите папку, которую хотите использовать как хранилище.
5. Нажмите **Открыть**.

> [!tip] Открытие хранилища из Obsidian Sync
> Если вы хотите открыть удалённое хранилище с помощью Obsidian Sync, обратитесь к разделу [[Настройка Obsidian Sync]].

## Переименование хранилища

Поскольку имя хранилища совпадает с именем папки, переименование хранилища также переименовывает папку.

1. Откройте Obsidian на компьютере.
2. В левом нижнем углу нажмите **Профиль хранилища** ( ![[lucide-chevrons-up-down.svg#icon]]).
3. Появится контекстное меню. Выберите **Управление хранилищами...**.
4. В списке хранилищ нажмите **Ещё** ( ![[lucide-more-horizontal.svg#icon]] ) рядом с хранилищем, которое хотите переименовать.
5. Выберите **Переименовать хранилище**.
6. Введите новое имя хранилища и нажмите `Enter`.

## Перемещение хранилища в другую папку

1. Откройте Obsidian на компьютере.
2. В левом нижнем углу нажмите **Профиль хранилища** ( ![[lucide-chevrons-up-down.svg#icon]]).
3. Появится контекстное меню. Выберите **Управление хранилищами...**.
4. Закройте окно текущего хранилища, оставив окно **Управление хранилищами** открытым.
5. В списке хранилищ нажмите **Ещё** ( ![[lucide-more-horizontal.svg#icon]] ) рядом с хранилищем, которое хотите переместить.
6. Выберите **Переместить хранилище**, а затем укажите новое расположение.

Некоторые операционные системы не позволяют перемещать хранилище через переключатель хранилищ. В таких случаях вам потребуется переместить хранилище вручную:

1. Закройте Obsidian.
2. Переместите папку хранилища в новое расположение, избегая папок, синхронизируемых другими сервисами.
3. Снова откройте Obsidian.
4. Нажмите значок **Профиль хранилища** в левом нижнем углу ( ![[lucide-chevrons-up-down.svg#icon]]).
5. Во всплывающем меню выберите **Управление хранилищами...**.
6. Рядом с **Открыть папку как хранилище** нажмите **Открыть**.
7. Перейдите к новой папке хранилища и выберите её.
8. Нажмите **Открыть**.
9. Убедитесь, что содержимое хранилища не изменилось. При необходимости повторно включите плагины сообщества, перейдя в **[[Настройки]] → Плагины сообщества → Отключить ограниченный режим**.

## Удаление хранилища

Удаление хранилища лишь убирает его из списка хранилищ.

1. Откройте Obsidian на компьютере.
2. В левом нижнем углу нажмите **Профиль хранилища** ( ![[lucide-chevrons-up-down.svg#icon]]).
3. Появится контекстное меню. Выберите **Управление хранилищами...**.
4. В списке хранилищ нажмите **Ещё** ( ![[lucide-more-horizontal.svg#icon]]) рядом с хранилищем, которое хотите удалить.
5. Выберите **Убрать из списка**.

## Перенос настроек в другое хранилище

Чтобы использовать те же настройки в другом хранилище, с помощью файлового менеджера (или терминала) скопируйте папку `.obsidian` из корня исходного хранилища в корень целевого хранилища.

Возможно, потребуется перезапустить Obsidian, чтобы изменения вступили в силу.

> [!note] Где найти папку `.obsidian`?
> По умолчанию большинство операционных систем скрывают папки, начинающиеся с точки (`.`). Подробнее о папке `.obsidian` и способах доступа к ней смотрите в разделах [[Хранение данных в Obsidian#Настройки хранилища|настройки хранилища]] и [[Папка конфигурации|папка конфигурации]].
