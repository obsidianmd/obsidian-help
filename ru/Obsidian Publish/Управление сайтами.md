Эта страница описывает, как управлять сайтами [[Введение в Obsidian Publish|Obsidian Publish]].

Сайт — это коллекция заметок, размещённая на Obsidian Publish и доступная онлайн либо по адресу Obsidian Publish, либо через [[Пользовательские домены|пользовательский домен]].

## Создание нового сайта

> [!note] Количество сайтов, которые вы можете иметь одновременно, определяется вашей подпиской на Obsidian Publish. Перед созданием нового сайта убедитесь, что ваша подписка это позволяет.

1. На вертикальной панели слева от окна приложения нажмите **Опубликовать изменения** ( ![[lucide-send.svg#icon]] ).
2. Если у вас уже есть существующий сайт, нажмите **Выбрать другой сайт** ( ![[lucide-repeat.svg#icon]] ).
3. В поле **ID сайта** введите путь, который вы хотите использовать для своего сайта. Например, сайт с идентификатором `my-amazing-site` будет доступен по адресу publish.obsidian.md/my-amazing-site.
4. Нажмите **Создать**.

## Удаление существующего сайта

> [!note] Заметки остаются в вашем хранилище даже после удаления сайта.

1. На вертикальной панели слева от окна приложения нажмите **Опубликовать изменения** ( ![[lucide-send.svg#icon]] ).
2. Нажмите **Выбрать другой сайт** ( ![[lucide-repeat.svg#icon]] ).
3. Нажмите **Удалить сайт** ( ![[lucide-x.svg#icon]] ) справа от сайта, который вы хотите удалить.
4. Нажмите **Удалить** для подтверждения.

## Переключение между сайтами

1. На вертикальной панели слева от окна приложения нажмите **Опубликовать изменения** ( ![[lucide-send.svg#icon]] ).
2. Нажмите **Выбрать другой сайт** ( ![[lucide-repeat.svg#icon]] ).
3. Нажмите **Выбрать** справа от сайта, на который вы хотите переключиться.

## Изменение ID сайта

1. На вертикальной панели слева от окна приложения нажмите **Опубликовать изменения** ( ![[lucide-send.svg#icon]] ).
2. Нажмите **Выбрать другой сайт** ( ![[lucide-repeat.svg#icon]] ).
3. Нажмите **Изменить ID сайта** ( ![[lucide-edit-3.svg#icon]] ) справа от сайта, который вы хотите отредактировать.
4. В поле **ID сайта** введите новый идентификатор для вашего сайта.
5. Нажмите **Изменить**.

## Просмотр настроек сайта

1. На вертикальной панели слева от окна приложения нажмите **Опубликовать изменения** ( ![[lucide-send.svg#icon]] ).
2. В диалоговом окне **Опубликовать изменения** нажмите **Настройки сайта** ( ![[lucide-cog.svg#icon]] ).

## Настройки сайта

### Общие

| Настройка                             | Тип    | Описание                                                                                                                             |
| ------------------------------------- | ------ | ------------------------------------------------------------------------------------------------------------------------------------ |
| Название                              | Ввод   | Публичное название и заголовок вашего сайта Obsidian Publish.                                                                        |
| Домашняя страница                     | Ввод   | Расположение файла Markdown, который вы хотите использовать в качестве главной страницы.                                             |
| Логотип                               | Ввод   | Изображение, которое будет использоваться как баннер сайта. Изображение должно [[Публикация контента#Publish notes\|быть опубликовано]]. |
| Совместная работа                     | Кнопка | Укажите других пользователей, которым вы хотите предоставить доступ к редактированию опубликованных заметок. Пользователи должны иметь аккаунт Obsidian. |
| Личный домен                          | Кнопка | [[Пользовательские домены]]                                                                                                          |
| Запретить индексацию поисковикам      | Тумблер | Запретите уважающим правила поисковым системам сканировать ваш сайт с помощью файла `robots.txt`.                                    |

### Внешний вид

| Настройка                            | Тип              | Описание                                                                                              |
|--------------------------------------|------------------|-------------------------------------------------------------------------------------------------------|
| Тема                                 | Выпадающий список | Выберите, как ваша тема отображается на сайте: **Светлая**, **Тёмная** или **Адаптация к системной теме**. |
| Переключатель Светлой/Тёмной тем     | Тумблер          | Позволяет пользователям переключать режим **Светлая**/**Тёмная** с помощью кнопки на вашем сайте.     |

| Ограничить максимальную длину строки   | Тумблер | Включить или отключить ограничение длины строки на вашем сайте.                                             |
