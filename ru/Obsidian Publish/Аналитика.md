## Предварительные требования

Для использования аналитики на вашем сайте [[Введение в Obsidian Publish|Obsidian Publish]] необходим [[Пользовательские домены|пользовательский домен]].

> [!important] Важно
> Прежде чем включить аналитику, убедитесь, что местное законодательство и нормативные акты позволяют отслеживать посетителей. В зависимости от используемых инструментов вам может потребоваться добавить на сайт баннер согласия на сбор данных.

## Google Analytics

Чтобы включить Google Analytics для вашего сайта Obsidian Publish:

1. На вертикальной панели слева от окна приложения нажмите **Опубликовать изменения** ( ![[lucide-send.svg#icon]] ).
2. В диалоговом окне **Опубликовать изменения** нажмите **Настройки сайта** ( ![[lucide-cog.svg#icon]] ).
3. В поле **Код слежения Google Analytics** введите ваш код отслеживания.

Чтобы использовать Google Tag Manager вместо Google Analytics, используйте пользовательский JavaScript для добавления собственных скриптов.

## Plausible, Fathom и другие сервисы аналитики

Вы можете подключить большинство сервисов аналитики к вашему сайту через файл [[Настройка сайта|publish.js]]. Обязательно замените `yourdomain.com` на ваш домен, а атрибут `src` скрипта — на скрипт от вашего провайдера аналитики.

Вот пример использования [Plausible Analytics](https://plausible.io/):

```js
var analyticsScript = document.createElement('script');
analyticsScript.defer = true;
analyticsScript.setAttribute('data-domain', 'yourdomain.com');
analyticsScript.src = 'https://plausible.io/js/plausible.js';
document.head.appendChild(analyticsScript);
```

Тот же подход можно использовать для [Fathom Analytics](https://usefathom.com/). Обратите внимание на замену `data-domain` на `data-site` — разные провайдеры аналитики могут использовать различный формат для вставки скрипта.

```javascript
var fathom = analyticsScript.createElement('script');
analyticsScript.defer = true;
analyticsScript.setAttribute('data-site', 'yourdomain.com');
analyticsScript.src = 'https://cdn.usefathom.com/script.js';
document.head.appendChild(analyticsScript);
```

## Устранение неполадок

Чтобы проверить, использует ли ваш сайт сервис аналитики, отключите любые расширения браузера для блокировки рекламы, такие как uBlock Origin, которые могут препятствовать выполнению скрипта отслеживания.
