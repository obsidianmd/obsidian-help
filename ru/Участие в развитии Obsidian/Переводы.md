Если вы знаете другой язык, желательно на уровне носителя, вы можете помочь перевести пользовательский интерфейс Obsidian или документацию.

Чтобы помочь с переводом пользовательского интерфейса, отправьте pull request в [репозиторий переводов](https://github.com/obsidianmd/obsidian-translations).

Чтобы помочь улучшить нашу документацию, отправьте pull request в репозиторий [obsidian-help](https://github.com/obsidianmd/obsidian-help). При внесении изменений обязательно следуйте [[Руководство по стилю#Переводы|руководству по стилю для переводов]].
