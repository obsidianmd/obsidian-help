Если вы хотите ссылаться на файл, используя различные имена, рассмотрите возможность добавления _псевдонимов_ к заметке. Псевдоним — это альтернативное имя для заметки.

Используйте псевдонимы для аббревиатур, прозвищ или для обращения к заметке на другом языке.

Если вы хотите изменить внешний вид ссылки только в одном месте, посмотрите, как [[Внутренние ссылки#Изменение отображаемого текста ссылки|изменить отображаемый текст ссылки]].

![[Внутренние ссылки#^callout-internal-links-link-text]]

## Добавление псевдонима к заметке

Чтобы добавить псевдоним для заметки, добавьте свойство `aliases` в [[Свойства|свойствах]] заметки. Псевдонимы всегда должны быть оформлены в виде списка в YAML.

```md
---
aliases:
  - Doggo
  - Woofer
  - Yapper
---

# Dog
```

## Ссылка на заметку с помощью псевдонима

Чтобы создать ссылку на заметку с помощью псевдонима:

1. Начните вводить псевдоним во [[Внутренние ссылки|внутренней ссылке]]. Все псевдонимы отображаются в списке предложений с значком изогнутой стрелки рядом.
2. Нажмите `Enter`, чтобы выбрать псевдоним.

Obsidian создаст ссылку с псевдонимом в качестве пользовательского отображаемого текста, например `[[Artificial Intelligence|AI]]`.

> [!note] Примечание
> Вместо того чтобы просто использовать псевдоним в качестве назначения ссылки (`[[AI]]`), Obsidian использует формат ссылки `[[Artificial Intelligence|AI]]` для обеспечения совместимости с другими приложениями, использующими формат wiki-ссылок.

## Поиск упоминаний без ссылки для псевдонима

С помощью [[Обратные ссылки|обратных ссылок]] вы можете находить упоминания без ссылки для псевдонимов.

Например, после установки «AI» в качестве псевдонима для «Artificial intelligence» вы сможете увидеть упоминания «AI» в других заметках.

Если вы свяжете упоминание без ссылки с псевдонимом, Obsidian превратит это упоминание во [[Внутренние ссылки|внутреннюю ссылку]] с псевдонимом в качестве отображаемого текста.
