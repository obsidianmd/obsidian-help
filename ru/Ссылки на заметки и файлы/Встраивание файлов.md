DESCRIPTION: Узнайте, как встраивать другие заметки и медиафайлы в свои заметки. Встраивая файлы в заметки, вы можете повторно использовать контент по всему хранилищу.

Узнайте, как встраивать другие заметки и медиафайлы в свои заметки. Встраивая файлы в заметки, вы можете повторно использовать контент по всему хранилищу.

Чтобы встроить файл в хранилище, добавьте восклицательный знак (`!`) перед [[Внутренние ссылки|внутренней ссылкой]]. Вы можете встраивать файлы любого из [[Поддерживаемые форматы файлов|поддерживаемых форматов]].

> [!tip] Встраивание перетаскиванием
> На десктопе вы также можете перетаскивать поддерживаемые файлы прямо в заметку, чтобы автоматически встроить их.

## Встраивание заметки в другую заметку

Чтобы встроить заметку:

```md
![[Внутренние ссылки]]
```

Вы также можете встраивать ссылки на [[Внутренние ссылки#Ссылка на заголовок в заметке|заголовки]] и [[Внутренние ссылки#Ссылка на блок в заметке|блоки]].

```md
![[Внутренние ссылки#^b15695]]
```

Текст ниже — пример встроенного блока:

![[Внутренние ссылки#^b15695]]

## Встраивание изображения в заметку

Чтобы встроить изображение:

```md
![[Engelbart.jpg]]
```

![[Engelbart.jpg#outline]]

Вы можете изменить размеры изображения, добавив `|640x480` к адресу ссылки, где 640 — ширина, а 480 — высота.

```md
![[Engelbart.jpg|100x145]]
```

Если указать только ширину, изображение масштабируется с сохранением исходных пропорций. Например, `![[Engelbart.jpg|100]]`.

![[Engelbart.jpg#outline|100]]

Вы также можете встроить изображение, размещённое на внешнем ресурсе, используя Markdown-ссылку. Управление шириной и высотой работает так же, как и для wiki-ссылок.

```md
![250](https://publish-01.obsidian.md/access/f786db9fac45774fa4f0d8112e232d67/Attachments/Engelbart.jpg)
```

![250](https://publish-01.obsidian.md/access/f786db9fac45774fa4f0d8112e232d67/Attachments/Engelbart.jpg)

## Встраивание PDF в заметку

Чтобы встроить PDF:

```md
![[Document.pdf]]
```

Вы также можете открыть определённую страницу PDF, добавив `#page=N` к адресу ссылки, где `N` — номер страницы:

```md
![[Document.pdf#page=3]]
```

Вы также можете указать высоту в пикселях для встроенного просмотрщика PDF, добавив `#height=[number]` к ссылке. Например:

```md
![[Document.pdf#height=400]]
```

## Встраивание списка в заметку

Чтобы встроить список из другой заметки, сначала добавьте [[Внутренние ссылки#Ссылка на блок в заметке|идентификатор блока]] к вашему списку:

```md

- элемент списка 1
- элемент списка 2

^my-list-id
```

Затем создайте ссылку на список, используя идентификатор блока:

```md
![[Моя заметка#^my-list-id]]
```
