Таблица — это тип [[Виды|представления]], который можно использовать в [[Введение в Базы|Базах]].

Выберите ![[lucide-table.svg#icon]]  **Таблица** в меню представлений, чтобы отображать файлы в виде таблицы, где каждая строка соответствует файлу, а столбцы — [[Свойства|свойствам]] этого файла.

![Пример базы с табличным представлением и списком книг](bases-noshadow.png#interface)

## Настройки

Настройки табличного представления можно изменить в [[Виды#Настройки представления|настройках представления]].

### Высота строки

Высота строки позволяет отображать больше информации. Выберите между **низкая**, **средняя**, **высокая** и **очень высокая**.

## Сводки

Вы можете добавить сводки к столбцу таблицы для быстрого вычисления значений, таких как итоги, средние значения или количество строк, видимых в текущем представлении.

Сводки привязаны к представлению, а не к базе. Каждое представление может отображать разные сводки для одного и того же столбца.

### Добавление сводки

1. Щёлкните правой кнопкой мыши по заголовку столбца в табличном представлении.
2. Выберите ![[lucide-calculator.svg#icon]] **Добавить сводку…**.
3. Выберите одну из встроенных функций сводки или нажмите ![[lucide-square-function.svg#icon]] **Добавить сводку**, чтобы создать собственную.

Сводка отображается внизу столбца. Когда результаты [[Виды#Сортировка и группировка результатов|сгруппированы]], сводка для каждой группы отображается в верхней части группы.

После добавления строки сводки вы можете добавить сводки для других столбцов, нажав на ячейку сводки. Строка сводки скрывается, если все сводки удалены.

### Встроенные сводки

Следующие сводки доступны по умолчанию. Доступные варианты могут различаться в зависимости от типа свойства.

#### Все типы свойств

- **Нет значения**: количество строк без значения.
- **Заполнено**: количество строк со значением.
- **Уникальная**: количество различных значений.

#### Числа

- **Среднее**: среднее всех числовых значений.
- **Максимум**: наибольшее значение.
- **Медиана**: медианное значение.
- **Минимум**: наименьшее значение.
- **Диапазон**: разница между максимумом и минимумом.
- **Стандартное отклонение**: стандартное отклонение.
- **Сумма**: сумма всех значений.

#### Даты

- **Самая ранняя**: наименьшая/самая старая дата.
- **Самая поздняя**: наибольшая/самая недавняя дата.
- **Диапазон**: разница между самой ранней и самой поздней датой.

#### Флажок

- **Отмечено**: количество строк, где флажок включён.
- **Не отмечено**: количество строк, где флажок выключен.

### Пользовательские сводки

Вы можете создать собственную сводку с помощью формулы:

1. В меню ![[lucide-calculator.svg#icon]] **Добавить сводку…** выберите ![[lucide-square-function.svg#icon]] **Добавить сводку**.
2. Задайте имя сводки.
3. Введите формулу. Формула выполняется над списком значений в данном столбце (например, с использованием [[Функции|функции]] вроде `values.reduce(...)`).
4. Сохраните сводку.

Пользовательские сводки полезны, когда нужны вычисления, не предусмотренные встроенными вариантами.

## Сочетания клавиш

Вы можете быстро перемещаться по табличному представлению с помощью мыши и [[Сочетания клавиш при редактировании|сочетаний клавиш]].

- Shift+щелчок создаёт выделение ячеек.
- Щёлкните правой кнопкой мыши по выделенным ячейкам, чтобы получить доступ к дополнительным действиям для этих файлов.

| Отменить изменения свойств                                                                                                        | `Ctrl+Z`             | `Cmd+Z`             |
| Повторить изменения свойств                                                                                                       | `Ctrl+Shift+Z`       | `Cmd+Shift+Z`       |
| Выбрать все ячейки в текущей группе                                                                                               | `Ctrl+A`             | `Cmd+A`             |
