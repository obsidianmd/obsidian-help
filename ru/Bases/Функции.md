Функции используются в [[Введение в Базы|Базах]] для обработки данных из [[Свойства|свойств]] в [[Виды#Фильтры|фильтрах]] и [[Формулы|формулах]]. Обратитесь к справочнику по [[Синтаксис Баз|синтаксису Баз]], чтобы узнать больше о том, как использовать функции.

Функции Баз следуют поведению JavaScript. Для полной справочной документации обратитесь к [MDN Web Docs](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference).

Помимо [[Функции#Глобальные|глобальных]] функций, большинство функций зависят от типа значения, которое вы хотите изменить:

- [[Функции#Тип Any|Any]]
- [[Функции#Тип Date|Date]]
- [[Функции#Тип String|String]]
- [[Функции#Тип Number|Number]]
- [[Функции#Тип List|List]]
- [[Функции#Тип Link|Link]]
- [[Функции#Тип File|File]]
- [[Функции#Тип Object|Object]]
- [[Функции#Тип регулярных выражений|Регулярные выражения]]

## Глобальные

Глобальные функции используются без указания типа.

### `escapeHTML()`

`escapeHTML(html: string): string`

- Экранирует специальные символы в строке, чтобы её можно было безопасно использовать в HTML.

### `date()`

`date(date: string): date`

- `date(string): date` разбирает переданную строку и возвращает объект даты.
- Строка `date` должна быть в формате `YYYY-MM-DD HH:mm:ss`.

### `duration()`

`duration(value: string): duration`

- Разбирает строку как длительность. Формат строки `value` описан в разделе [[Синтаксис Баз#Арифметика дат|арифметика дат]].
- Явный разбор не требуется для арифметики с датами (например, `now() + '1d'`), но необходим при арифметике с самими длительностями (например, `now() + (duration('1d') * 2)`).
- При арифметике длительностей со скалярами длительность должна стоять слева. Например, `duration('5h') * 2`, а не `2 * duration('5h')`.

### `file()`

`file(path: string | file | url): file`

- Возвращает объект файла для указанного файла или пути.
- Пример: `file(link("[[filename]]"))` или `file("path to file")`.

### `html()`

`html(html: string): html`

- Преобразует строку во фрагмент кода, который отображается как HTML.

### `if()`

`if(condition: any, trueResult: any, falseResult?: any): any`

- `condition` — выражение для вычисления.
- `trueResult` — результат, если `condition` истинно.
- `falseResult` — необязательный результат, если `condition` ложно. Если не указан, по умолчанию равен `null`.
- Возвращает `trueResult`, если `condition` истинно или является истинноподобным значением, иначе возвращает `falseResult`.
- Пример: `if(isModified, "Modified", "Unmodified")`

### `image()`

`image(path: string | file | url): image`

- Возвращает объект изображения, которое отображается в представлении.
- Пример: `image(image-property)` или `image("https://obsidian.md/images/obsidian-logo-gradient.svg")`.

### `icon()`

`icon(name: string): icon`

- Возвращает значение, которое отображается как значок в представлении. Имя значка должно соответствовать поддерживаемому значку Lucide.
- Пример: `icon("arrow-right")`.

### `link()`

`link(path: string | file, display?: value): Link`

- Разбирает строку `path` и возвращает объект Link, который отображается как ссылка на указанный путь.
- Необязательный параметр `display` позволяет задать отображаемый текст ссылки.

### `list()`

`list(element: any): List`

- Если переданный элемент является списком, возвращает его без изменений.
- В противном случае оборачивает переданный `element` в список, создавая список с одним элементом.
- Используйте эту функцию, когда свойство содержит как строки, так и списки в разных заметках хранилища.
- Пример: `list("value")` возвращает `["value"]`.

### `max()`

`max(value1: number, value2: number...): number`

- Возвращает наибольшее из всех переданных чисел.

### `min()`

`min(value1: number, value2: number...): number`

- Возвращает наименьшее из всех переданных чисел.

### `now()`

`now(): date`

- Возвращает объект даты для текущего момента времени.

### `number()`

`number(input: any): number`

- Пытается вернуть переданное значение как число.
- Возвращает объекты даты как миллисекунды с начала эпохи Unix.
- Возвращает логические значения как `1` или `0`.
- Разбирает строки в число и возвращает ошибку, если строка не является допустимым числом.
- Пример: `number("3.4")` возвращает `3.4`.

### `today()`

`today(): date`

- Возвращает объект даты для текущей даты. Часть времени устанавливается в полночь.

### `random()`

`random(): number`

- Возвращает случайное число от 0 до 1.
- Генерация числа обновляется при каждой загрузке представления. Переключение между представлениями изменяет случайное число.

## Тип Any

Функции, которые можно использовать с любым значением. Это включает строки (например, `"hello"`), числа (например, `42`), списки (например, `[1,2,3]`), объекты и другие типы.

### `isTruthy()`

`any.isTruthy(): boolean`

- Возвращает значение, приведённое к логическому типу.
- Пример: `1.isTruthy()` возвращает `true`.

### `isType()`

`any.isType(type: string): boolean`

- Возвращает true, если значение принадлежит указанному типу.
- Пример: `"example".isType("string")` и `true.isType("boolean")` оба возвращают true.

### `toString()`

`any.toString(): string`

- Возвращает строковое представление любого значения.
- Пример: `123.toString()` возвращает `"123"`.

## Тип Date

Функции, которые можно использовать с датой и временем, например `date("2025-05-27")`. Сравнение дат можно выполнять с помощью [[Синтаксис Баз#Арифметика дат|арифметики дат]].

### Поля

Для дат доступны следующие поля:

| Поле               | Тип      | Описание                    |
| ------------------ | -------- | --------------------------- |
| `date.year`        | `number` | Год даты                    |
| `date.month`       | `number` | Месяц даты (1–12)           |
| `date.day`         | `number` | День месяца                 |
| `date.hour`        | `number` | Час (0–23)                  |
| `date.minute`      | `number` | Минута (0–59)               |
| `date.second`      | `number` | Секунда (0–59)              |
| `date.millisecond` | `number` | Миллисекунда (0–999)        |

### `date()`

`date.date(): date`

- Возвращает объект даты с удалённой частью времени.
- Пример: `now().date().format("YYYY-MM-DD HH:mm:ss")` возвращает строку вида "2025-12-31 00:00:00".

### `format()`

`date.format(format: string): string`

- `format` — строка формата (например, `"YYYY-MM-DD"`).
- Возвращает дату, отформатированную в соответствии с указанной строкой формата Moment.js.
- Пример: `date.format("YYYY-MM-DD")` возвращает `"2025-05-27"`.

### `time()`

`date.time(): string`

- Возвращает часть времени в виде строки.
- Пример: `now().time()` возвращает строку вида "23:59:59".

### `relative()`

`date.relative(): string`

- Возвращает читаемое сравнение даты с текущим моментом времени.
- Пример: `file.mtime.relative()` возвращает значение вида `3 days ago`.

### `isEmpty()`

`date.isEmpty(): boolean`

- Возвращает false.

## Тип String

Функции, которые можно использовать с последовательностью символов, например `"hello"`.

### Поля

| Поле            | Тип      | Описание                         |
| --------------- | -------- | -------------------------------- |
| `string.length` | `number` | Количество символов в строке     |

### `contains()`

`string.contains(value: string): boolean`

- `value` — подстрока для поиска.
- Возвращает true, если строка содержит `value`.
- Пример: `"hello".contains("ell")` возвращает `true`.

### `containsAll()`

`string.containsAll(...values: string): boolean`

- `values` — одна или несколько подстрок для поиска.
- Возвращает true, если строка содержит все указанные `values`.
- Пример: `"hello".containsAll("h", "e")` возвращает `true`.

### `containsAny()`

`string.containsAny(...values: string): boolean`

- `values` — одна или несколько подстрок для поиска.
- Возвращает true, если строка содержит хотя бы одно из значений `values`.
- Пример: `"hello".containsAny("x", "y", "e")` возвращает `true`.

### `endsWith()`

`string.endsWith(query: string): boolean`

- `query` — строка для проверки в конце.
- Возвращает true, если данная строка заканчивается на `query`.
- Пример: `"hello".endsWith("lo")` возвращает `true`.

### `isEmpty()`

`string.isEmpty(): boolean`

- Возвращает true, если строка не содержит символов или отсутствует.
- Пример: `"Hello world".isEmpty()` возвращает `false`.
- Пример: `"".isEmpty()` возвращает `true`.

### `lower()`

`string.lower(): string`

- Возвращает строку, преобразованную в нижний регистр.

### `replace()`

`string.replace(pattern: string | Regexp, replacement: string): string`

- `pattern` — значение для поиска в целевой строке.
- `replacement` — значение для замены найденных совпадений. Когда `pattern` является Regexp, в `replacement` можно ссылаться на группы захвата с помощью `$1`, `$2` и так далее.
- Если `pattern` — строка, все вхождения шаблона будут заменены.
- Если `pattern` — Regexp, флаг `g` определяет, заменяется ли только первое или все вхождения.
- Пример: `"a:b:c:d".replace(/:/, "-")` возвращает `"a-b:c:d"`, тогда как `"a:b:c:d".replace(/:/g, "-")` возвращает `"a-b-c-d"`.
- Пример с группами захвата: `"John Smith".replace(/(\w+) (\w+)/, "$2, $1")` возвращает `"Smith, John"`.

### `repeat()`

`string.repeat(count: number): string`

- `count` — количество повторений строки.
- Пример: `"123".repeat(2)` возвращает `"123123"`.

### `reverse()`

`string.reverse(): string`

- Переворачивает строку.
- Пример: `"hello".reverse()` возвращает `"olleh"`.

### `slice()`

`string.slice(start: number, end?: number): string`

- `start` — начальный индекс (включительно).
- `end` — необязательный конечный индекс (не включительно).
- Возвращает подстроку от `start` (включительно) до `end` (не включительно).
- Пример: `"hello".slice(1, 4)` возвращает `"ell"`.
- Если `end` опущен, извлекает до конца строки.

### `split()`

`string.split(separator: string | Regexp, n?: number): list`

- `separator` — разделитель для разбиения строки.
- `n` — необязательное число. Если указано, результат будет содержать первые `n` элементов.
- Возвращает список подстрок.
- Пример: `"a,b,c,d".split(",", 3)` или `"a,b,c,d".split(/,/, 3)` возвращает `["a", "b", "c"]`.

### `startsWith()`

`string.startsWith(query: string): boolean`

- `query` — строка для проверки в начале.
- Возвращает true, если данная строка начинается с `query`.
- Пример: `"hello".startsWith("he")` возвращает `true`.

### `title()`

`string.title(): string`

- Преобразует строку в формат заголовка (первая буква каждого слова заглавная).
- Пример: `"hello world".title()` возвращает `"Hello World"`.

### `trim()`

`string.trim(): string`

- Удаляет пробелы с обоих концов строки.
- Пример: `"  hi  ".trim()` возвращает `"hi"`.

## Тип Number

Функции, которые можно использовать с числовыми значениями, например `42`, `3.14`.

### `abs()`

`number.abs(): number`

- Возвращает абсолютное значение числа.
- Пример: `(-5).abs()` возвращает `5`.

### `ceil()`

`number.ceil(): number`

- Округляет число вверх до ближайшего целого.
- Пример: `(2.1).ceil()` возвращает `3`.

### `floor()`

`number.floor(): number`

- Округляет число вниз до ближайшего целого.
- Пример: `(2.9).floor()` возвращает `2`.

### `isEmpty()`

`number.isEmpty(): boolean`

- Возвращает true, если число отсутствует.
- Пример: `5.isEmpty()` возвращает `false`.

### `round()`

`number.round(digits: number): number`

- Округляет число до ближайшего целого.
- Необязательный параметр `digits` позволяет округлить до указанного количества десятичных знаков.
- Пример: `(2.5).round()` возвращает `3`, а `(2.3333).round(2)` возвращает `2.33`.

### `toFixed()`

`number.toFixed(precision: number): string`

- `precision` — количество десятичных знаков.
- Возвращает строку с числом в формате с фиксированной точкой.
- Пример: `(3.14159).toFixed(2)` возвращает `"3.14"`.

## Тип List

Функции, которые можно использовать с упорядоченным списком элементов, например `[1, 2, 3]`.

### Поля

| Поле          | Тип      | Описание                           |
| ------------- | -------- | ---------------------------------- |
| `list.length` | `number` | Количество элементов в списке      |

### `contains()`

`list.contains(value: any): boolean`

- `value` — элемент для поиска.
- Возвращает true, если список содержит `value`.
- Пример: `[1,2,3].contains(2)` возвращает `true`.

### `containsAll()`

`list.containsAll(...values: any): boolean`

- `values` — один или несколько элементов для поиска.
- Возвращает true, если список содержит все указанные `values`.
- Пример: `[1,2,3].containsAll(2,3)` возвращает `true`.

### `containsAny()`

`list.containsAny(...values: any): boolean`

- `values` — один или несколько элементов для поиска.
- Возвращает true, если список содержит хотя бы один из `values`.
- Пример: `[1,2,3].containsAny(3,4)` возвращает `true`.

### `filter()`

`list.filter(value: Boolean): list`

- Фильтрует список, оставляя только элементы, для которых выражение возвращает true.
- `value` — значение элемента в списке.
- `index` — индекс текущего значения.
- Пример: `[1,2,3,4].filter(value > 2)` возвращает `[3,4]`.

### `flat()`

`list.flat(): list`

- Разворачивает вложенный список в один плоский список.
- Пример: `[1,[2,3]].flat()` возвращает `[1,2,3]`.

### `isEmpty()`

`list.isEmpty(): boolean`

- Возвращает true, если список не содержит элементов.
- Пример: `[1,2,3].isEmpty()` возвращает `false`.

### `join()`

`list.join(separator: string): string`

- `separator` — строка, вставляемая между элементами.
- Объединяет все элементы списка в одну строку.
- Пример: `[1,2,3].join(",")` возвращает `"1,2,3"`.

### `map()`

`list.map(value: Any): list`

- Преобразует каждый элемент списка с помощью выражения.
- `value` — значение элемента в списке.
- `index` — индекс текущего значения.
- Пример: `[1,2,3,4].map(value + 1)` возвращает `[2,3,4,5]`.

### `reduce()`

`list.reduce(expression: Any, acc: Any): Any`

- Сворачивает список в одно значение, выполняя выражение для каждого элемента. Выражение должно возвращать следующее значение `acc`. Используйте `value` для текущего элемента, `index` для его позиции и `acc` для накопленного результата.
- `expression` вычисляется для каждого элемента списка.
- `value` — значение текущего элемента в списке.
- `index` — индекс текущего элемента.
- `acc` — накопленное значение.
- Пример (сумма): `[1,2,3].reduce(acc + value, 0)` возвращает `6`.
- Пример (максимум): `values.filter(value.isType("number")).reduce(if(acc == null || value > acc, value, acc), null)` возвращает наибольшее число или `null`, если таковых нет.

### `reverse()`

`list.reverse(): list`

- Переворачивает список.
- Пример: `[1,2,3].reverse()` возвращает `[3,2,1]`.

### `slice()`

`list.slice(start: number, end?: number): list`

- `start` — начальный индекс (включительно).
- `end` — необязательный конечный индекс (не включительно).
- Возвращает поверхностную копию части списка от `start` (включительно) до `end` (не включительно).
- Пример: `[1,2,3,4].slice(1,3)` возвращает `[2,3]`.
- Если `end` опущен, извлекает до конца списка.

### `sort()`

`list.sort(): list`

- Сортирует элементы списка от наименьшего к наибольшему.
- Пример: `[3, 1, 2].sort()` возвращает `[1, 2, 3]`.
- Пример: `["c", "a", "b"].sort()` возвращает `["a", "b", "c"]`.

### `unique()`

`list.unique(): list`

- Удаляет дублирующиеся элементы.
- Пример: `[1,2,2,3].unique()` возвращает `[1,2,3]`.

## Тип Link

Функции, которые можно использовать со ссылкой. Ссылки можно создать из файла (`file.asLink()`) или пути (`link("path")`).

### `asFile()`

`link.asFile(): file`

- Возвращает объект файла, если ссылка указывает на существующий локальный файл.
- Пример: `link("[[filename]]").asFile()`.

### `linksTo()`

`link.linksTo(file): boolean`

- Возвращает `true`, если файл, представленный объектом `link`, содержит ссылку на `file`.

## Тип File

Функции, которые можно использовать с файлом в хранилище.

### Поля

Для файлов доступны следующие поля:

| Поле              | Тип      | Описание                                                          |
| ----------------- | -------- | ----------------------------------------------------------------- |
| `file.name`       | `string` | Имя файла.                                                        |
| `file.basename`   | `string` | Имя файла без расширения.                                         |
| `file.path`       | `string` | Полный путь к файлу относительно корня хранилища.                  |
| `file.folder`     | `string` | Полный путь к родительской папке.                                  |
| `file.ext`        | `string` | Расширение файла.                                                 |
| `file.size`       | `number` | Размер файла в байтах.                                            |
| `file.properties` | `object` | Свойства заметки для данного файла.                                |
| `file.tags`       | `list`   | Теги файла. Включает встроенные теги.                              |
| `file.links`      | `list`   | Внутренние ссылки в данном файле.                                  |
| `file.ctime`      | `date`   | Временная метка создания файла.                                    |
| `file.mtime`      | `date`   | Временная метка последнего изменения файла.                        |

### `asLink()`

`file.asLink(display?: string): Link`

- `display` — необязательный отображаемый текст для ссылки.
- Возвращает объект Link, который отображается как рабочая ссылка.
- Пример: `file.asLink()`

### `hasLink()`

`file.hasLink(otherFile: file | string): boolean`

- `otherFile` — другой объект файла или строковый путь для проверки.
- Возвращает true, если `file` ссылается на `otherFile`.
- Пример: `file.hasLink(otherFile)` возвращает `true`, если существует ссылка из `file` на `otherFile`.

### `hasProperty()`

`file.hasProperty(name: string): boolean`

- Возвращает `true`, если файл имеет указанное свойство.

### `hasTag()`

`file.hasTag(...values: string): boolean`

- `values` — одно или несколько имён тегов.
- Возвращает true, если файл имеет любой из тегов, указанных в `values`.
- Пример: `file.hasTag("tag1", "tag2")` возвращает `true`, если файл имеет тег `#tag1` или `#tag2`. Также включает любые [[Теги#Вложенные теги|вложенные теги]], такие как `#tag1/a` или `#tag2/b`.

### `inFolder()`

`file.inFolder(folder: string): boolean`

- `folder` — имя папки для проверки.
- Возвращает true, если файл находится в указанной папке или одной из её подпапок.
- Пример: `file.inFolder("notes")` возвращает `true`.

## Тип Object

Функции, которые можно использовать с коллекцией пар ключ-значение, например `{"a": 1, "b": 2}`.

### `isEmpty()`

`object.isEmpty(): boolean`

- Возвращает true, если объект не имеет собственных свойств.
- Пример: `{}.isEmpty()` возвращает `true`.

### `keys()`

`object.keys(): list`

- Возвращает список, содержащий ключи объекта.

### `values()`

`object.values(): list`

- Возвращает список, содержащий значения объекта.

## Тип регулярных выражений

Функции, которые можно использовать с шаблоном регулярного выражения. Пример: `/abc/`.

### `matches()`

`regexp.matches(value: string): boolean`

- `value` — строка для проверки.
- Возвращает true, если регулярное выражение совпадает с `value`.
- Пример: `/abc/.matches("abcde")` возвращает `true`.
