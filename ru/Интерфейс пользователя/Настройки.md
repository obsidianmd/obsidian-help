Настройки позволяют вам персонализировать работу с Obsidian. Настраивайте общие параметры, предпочтения редактора, внешний вид, сочетания клавиш, а также управляйте основными и сторонними плагинами.

## Открытие настроек

### На компьютере

В [[Боковая панель#Открытие скрытых боковых панелей|левой боковой панели]] выберите **[[Настройки]]** ( ![[lucide-cog.svg#icon]] ). Вы также можете открыть настройки через [[Палитра команд|палитру команд]].

### На мобильном устройстве

В левой боковой панели выберите **[[Настройки]]** ( ![[lucide-cog.svg#icon]] ).

## Организация настроек

Настройки организованы по следующим категориям:

- **Настройки** — Основные параметры приложения, включая [[#Общие]], [[#Редактор]], [[#Файлы и ссылки]], [[#Внешний вид]] и [[#Сочетания клавиш]].
- **[[Основные плагины]]** — Встроенные плагины, поставляемые с Obsidian.
- **[[Сторонние плагины]]** — Сторонние плагины, установленные из сообщества.

## Общие

Страница «Общие» содержит информацию о версии, параметры обновлений, языковые настройки и управление аккаунтом.

### Версия и обновления

В верхней части страницы отображается текущая версия Obsidian и версия установщика. Нажмите **Проверить обновления**, чтобы вручную проверить наличие новых версий, или нажмите **[Открыть журнал изменений](<https://obsidian.md/changelog/>)**, чтобы просмотреть примечания к выпуску.

**Автоматические обновления**

Когда включено, Obsidian автоматически проверяет наличие новых версий и уведомляет вас о доступных обновлениях.

**Получать ранние сборки**

Автоматически обновляться до последних [[Ранние версии|ранних версий]]. Эти версии включают новые функции, но могут быть менее стабильными.

> [!info]+ Требуется лицензия Catalyst
> Этот параметр доступен только для пользователей с [[Лицензия Catalyst|лицензией Catalyst]].

### Язык

Выберите предпочтительный язык интерфейса Obsidian. Нажмите **[[Переводы|Узнайте, как добавить новый язык в Obsidian]]**, чтобы внести свой вклад в перевод.

### Помощь

Нажмите **[[Главная|Открыть]]**, чтобы получить доступ к справочным ресурсам, включая документацию, форумы сообщества и руководства по устранению неполадок.

### Аккаунт

Управление аккаунтом Obsidian и лицензиями.

**Ваш аккаунт**

Просмотр информации об аккаунте, включая имя пользователя и электронную почту. Нажмите **[Настроить](https://obsidian.md/account)**, чтобы перейти к настройкам аккаунта, или нажмите **Выйти из системы**, чтобы выйти из приложения.

**Лицензия Catalyst**

Просмотр статуса лицензии Catalyst. Если у вас активная [[Лицензия Catalyst|лицензия Catalyst]], здесь отображается её уровень.

**Коммерческая лицензия**

Если вы поддерживаете [Obsidian для работы](<https://obsidian.md/blog/free-for-work/>), нажмите **Активировать**, чтобы ввести коммерческий лицензионный ключ, или нажмите **[Приобрести](https://obsidian.md/account/commercial)**, чтобы купить лицензию. Нажмите **[[Коммерческая лицензия|Подробнее]]** для получения информации о коммерческих лицензиях.

### Расширенные настройки

#### Уведомлять, если запуск займёт больше времени, чем ожидалось

Когда включено, Obsidian отображает уведомление, если запуск необычно медленный. Уведомление содержит диагностическую информацию о причине задержки. Нажмите значок таймера ( ![[lucide-timer.svg#icon]] ) рядом с этим параметром, чтобы [[Помощь и поддержка#Проверка времени запуска|проверить время запуска]].

## Редактор

Страница «Редактор» содержит настройки отображения и редактирования заметок.

### Фокусировать новые вкладки

При открытии ссылки в новой вкладке сразу переключаться на неё.

### Режим открытия новых вкладок

Вид по умолчанию, в котором открывается новая вкладка с Markdown-файлом. Выберите между [[Режимы просмотра и редактирования#Режим чтения|Режимом чтения]] или [[Режимы просмотра и редактирования#Режим редактирования|Режимом редактирования]].

### Режим редактора по умолчанию

Режим редактирования по умолчанию для новой вкладки. Выберите между [[Режимы просмотра и редактирования#Динамический просмотр|Динамическим просмотром]] или [[Режимы просмотра и редактирования#Режим исходного кода|Режимом исходного кода]].

### Статус редактора

Показывать переключатель режима редактирования в [[Строка состояния|строке состояния]].

### Отображение

#### Ограничить максимальную длину строки

Повышает удобство чтения длинных абзацев, но уменьшает количество помещающейся на экране информации.

#### Двойной разделитель строк

Спецификация Markdown игнорирует одинарные [[Основной синтаксис форматирования#Переносы строк|переносы строк]] в режиме чтения. Отключите этот параметр, чтобы одинарные переносы строк были видимы.

#### Свойства в документе

Выберите, как [[Свойства|свойства]] отображаются в верхней части заметок. Выберите **Исходный**, чтобы отображать свойства как необработанный YAML.

#### Сворачивать заголовки

[[Сворачивание|Сворачивание]] всего содержимого под заголовком при нажатии на значок сворачивания рядом с ним.

### Сворачивать отступы

[[Сворачивание|Сворачивание]] части отступа, например списков, при нажатии на значок сворачивания.

#### Номера строк

Добавляет нумерацию строк на полях.

#### Направляющие отступов

Добавляет вертикальные линии между пунктами одного уровня в списках.

#### Направления письма (RTL)

Установить отображение направления письма заметок справа налево.

#### Закрывать скобки

Автоматически закрывает скобки и кавычки.

#### Закрывать теги Markdown

Автоматически закрывает теги для жирного и курсивного начертаний.

#### Отступ для списков

Автоматически добавляет отступы и теги для списков.

#### Использовать табуляцию

Использовать табуляцию для отступов при нажатии клавиши Tab. Отключите, чтобы использовать 4 пробела для отступов.

#### Конвертировать HTML

Автоматическая конвертация HTML в Markdown при вставке и перетаскивании с веб-страниц.

Используйте `Ctrl/Cmd+Shift+V`, чтобы вставить HTML без конвертации.

### Режим работы

#### Проверять орфографию

Включение проверки орфографии. Нажмите значок шестерёнки, чтобы удалить сохранённые элементы из пользовательского словаря.

#### Выбрать языки

**Windows и Linux:**

Выберите языки для проверки орфографии. Нажмите значок плюса (+), чтобы добавить языки. Чтобы удалить язык, нажмите X рядом с ним.

**macOS:**

Встроенная проверка орфографии автоматически определяет язык, используемый операционной системой.

#### Ширина табуляции

Количество пробелов, которому соответствует табуляция.

### Расширенные настройки

#### Использовать сочетания клавиш Vim

Позволяет использовать сочетания клавиш Vim в режиме редактирования.

## Файлы и ссылки

Страница «Файлы и ссылки» содержит настройки управления файлами, ссылками и поведения хранилища.

### Место для новых заметок по умолчанию

Где создавать новые заметки. Доступные варианты:

- **В корне хранилища** — Заметки создаются в корневой папке хранилища.
- **В той же папке, где расположен файл** — Заметки создаются в той же папке, что и текущая активная заметка.
- **В папке, указанной ниже** — Заметки создаются в указанной вами папке.

### Место для вложенных файлов по умолчанию

Где размещать новые [[Вложения|вложения]]. Доступные варианты:

- **В корне хранилища** — Вложения размещаются в корневой папке хранилища.
- **В папке, указанной ниже** — Вложения размещаются в указанной вами папке.
- **В той же папке, где расположен файл** — Вложения размещаются в той же папке, что и текущая активная заметка.
- **В подпапке в текущей папке** — Вложения размещаются в подпапке внутри папки текущей заметки.

### Ссылки

#### Формат новой ссылки

Какие ссылки вставлять при автоматическом создании [[Внутренние ссылки|внутренних ссылок]]. Доступные варианты:

- **Самый короткий путь к файлу из возможных** — Используется кратчайший уникальный путь к связанному файлу.
- **Относительный путь к файлу** — Используется путь относительно текущего файла.
- **Абсолютный путь в хранилище** — Используется полный путь от корня хранилища.

#### Всегда обновлять внутренние ссылки

Когда включено, Obsidian автоматически обновляет внутренние ссылки при переименовании файла. Отключите, чтобы получать запрос на обновление ссылок после переименования.

#### Использовать wiki-ссылки

Автоматически создавать wiki-ссылки для `[[ссылок]]` и `![[изображений]]` вместо Markdown-ссылок и изображений. Отключите этот параметр для создания Markdown-ссылок.

#### Отображать все типы файлов

Показывать файлы с любым расширением, даже если Obsidian не может открыть их самостоятельно. Вы сможете их видеть и ссылаться на них в файловом менеджере и быстром переходе.

### Корзина

#### Подтверждение удаления файла

Запрашивать подтверждение перед удалением файла.

#### Удалённые файлы

Что делать с файлами после удаления. Доступные варианты:

- **Перемещать в системную корзину** — Файлы перемещаются в корзину операционной системы.
- **Перемещать в корзину Obsidian** — Файлы перемещаются в папку `.trash` в вашем хранилище.
- **Удалять безвозвратно** — Файлы удаляются безвозвратно и не могут быть восстановлены.

### Расширенные настройки

#### Исключённые файлы и папки

Исключённые файлы будут скрыты в [[Поиск|поиске]], [[Граф|виде графа]] и упоминаниях без ссылки ([[Обратные ссылки]] и [[Исходящие ссылки]]), а также менее заметны в [[Быстрый переключатель|быстром переходе]] и подсказках ссылок. Нажмите **Настроить**, чтобы настроить исключённые файлы.

#### Место хранения конфигурации хранилища

Использовать другую [[Папка конфигурации|папку конфигурации]] вместо стандартной. Должна начинаться с точки.

#### Разрешить обратные вызовы URI

Включить использование x-callback-url через x-success или x-error при обработке [[Obsidian URI|URI Obsidian]].

#### Восстановить кэш хранилища

Перестраивает [[Хранение данных в Obsidian#Кэш метаданных|кэш метаданных]] хранилища. Восстановление кэша может занять от нескольких секунд до нескольких минут в зависимости от размера вашего хранилища. Нажмите **Восстановить**, чтобы начать процесс.

## Внешний вид

Страница «Внешний вид» содержит настройки для персонализации отображения Obsidian.

### Основная тема

Выберите основную цветовую схему Obsidian. Доступные варианты:

- **Адаптация к системной теме** — Соответствует светлому или тёмному режиму вашей операционной системы.
- **Светлая** — Использует светлый режим.
- **Тёмная** — Использует тёмный режим.

### Акцентный цвет

Выберите акцентный цвет, используемый во всём приложении. Нажмите на палитру цветов или значок сброса для настройки.

### Темы

Управление установленными [[Темы|темами]] и просмотр тем сообщества. Нажмите **Настроить**, чтобы просмотреть и установить темы. В раскрывающемся списке отображается текущая активная тема. Значок папки ( ![[lucide-folder-open.svg#icon]] ) открывает папку тем в вашей [[Папка конфигурации|папке конфигурации]].

### Текущие темы сообщества

Отображает количество установленных тем сообщества.

### Шрифт

#### Шрифт интерфейса

Выбор основного шрифта интерфейса Obsidian. Нажмите **Настроить**, чтобы выбрать пользовательский шрифт или ввести имя шрифта.

#### Шрифт текста

Выбор шрифта для областей редактирования и просмотра. Нажмите **Настроить**, чтобы выбрать пользовательский шрифт или ввести имя шрифта.

#### Моноширинный шрифт

Выбор шрифта для блоков с кодом и начальных метаданных. Нажмите **Настроить**, чтобы выбрать пользовательский шрифт или ввести имя шрифта.

#### Размер шрифта

Размер шрифта в пикселях в области редактирования и предварительного просмотра. Настраивается с помощью ползунка.

#### Быстрое изменение размера шрифта

Изменение размера шрифта с помощью `Ctrl+Прокрутка` (Windows/Linux) или `Cmd+Прокрутка` (macOS), либо с помощью жеста масштабирования на трекпаде.

### Интерфейс

#### Встроенный заголовок

Отображает имя файла как редактируемый заголовок в тексте заметки. Встроенный заголовок работает как заголовок внутри Obsidian, но не добавляет заголовок в вашу заметку.

#### Строка заголовка

Отображать строку заголовка в верхней части каждой вкладки.

#### Вертикальная панель

Отображать вертикальную панель инструментов в боковой части окна. Подробнее о [[Лента|вертикальной панели]].

#### Настройки вертикальной панели

Выбор команд для отображения в вертикальной панели. Нажмите **Настроить** для настройки.

> [!info]+ Настройка вертикальной панели на мобильном
> На мобильных устройствах настройка вертикальной панели работает иначе. Подробнее см. в разделе [[Лента#Мобильное устройство|Настройка вертикальной панели на мобильном]].

### Расширенные настройки

#### Уровень масштабирования

Управление общим уровнем масштабирования в приложении. Настраивается с помощью ползунка.

#### Системное контекстное меню

Отображение контекстных меню приложения в соответствии с операционной системой, независимо от выбранной темы оформления.

#### Заголовок окна

Выбор оформления заголовка окна Obsidian. Изменение данного параметра потребует перезапуска приложения. Доступные варианты:

- **Obsidian** — Использует пользовательское оформление окна Obsidian.
- **Системный** — Использует оформление окна по умолчанию вашей операционной системы.
- **Скрытый** — Полностью скрывает заголовок окна.

#### Пользовательская иконка приложения

Установить собственную иконку для приложения. Нажмите **Выбрать**, чтобы выбрать файл иконки (`.icns`, `.ico`, `.png` или `.svg`). Подробнее о [[Внешний вид#Пользовательская иконка приложения|пользовательских иконках приложения]].

#### Полупрозрачное окно

Добавляет интерфейсу ощущение глубины, особенно в тёмном режиме. Не работает под Linux.

> [!info]+ Только для macOS
> Эта функция предназначена в первую очередь для macOS. Подробнее о [[Внешний вид#Полупрозрачность|полупрозрачности]].

#### Аппаратное ускорение

Включает аппаратное ускорение с помощью ГП для более плавной работы Obsidian. При отключении производительность приложения может значительно снизиться, но это может решить некоторые редкие проблемы.

### CSS-сниппеты

Управление и просмотр [[CSS-сниппеты|CSS-сниппетов]], хранящихся в вашем хранилище. CSS-сниппеты хранятся в `/vault/.obsidian/snippets/`. Нажмите значок обновления, чтобы перезагрузить сниппеты, или значок папки ( ![[lucide-folder-open.svg#icon]] ), чтобы открыть папку сниппетов.

## Сочетания клавиш

Страница [[Горячие клавиши|Сочетания клавиш]] позволяет просматривать, устанавливать и настраивать [[Сочетания клавиш при редактировании|сочетания клавиш]] для команд в Obsidian.

Используйте строку поиска для фильтрации команд и нажмите значок фильтра, чтобы показать только команды с назначенными сочетаниями клавиш. Чтобы добавить сочетание клавиш, нажмите значок плюса (+) рядом с командой. Чтобы удалить сочетание клавиш, нажмите значок X рядом с комбинацией клавиш.

## Основные плагины

[[Основные плагины]] — это встроенные плагины, поставляемые с Obsidian. Каждый плагин добавляет определённую функциональность в приложение.

Чтобы включить или отключить основной плагин, нажмите переключатель рядом с его названием. Нажмите значок плюса ( ![[lucide-plus-circle.svg#icon]] ), чтобы просмотреть [[Горячие клавиши|сочетания клавиш]] для плагина. Нажмите значок настроек ( ![[lucide-settings.svg#icon]] ), чтобы открыть страницу настроек плагина.

Используйте строку поиска для фильтрации плагинов по имени.

## Сторонние плагины

[[Сторонние плагины]] — это плагины, созданные сообществом Obsidian. Просматривайте и устанавливайте плагины, нажав **Просмотр**.

### Ограниченный режим

[[Безопасность плагинов#Ограниченный режим|Ограниченный режим]] отключает все сторонние плагины для безопасности. Нажмите **Включить**, чтобы активировать ограниченный режим.

### Установленные плагины

Отображает количество установленных сторонних плагинов. Нажмите **Проверить обновления**, чтобы проверить наличие обновлений плагинов.

### Управление плагинами

Просмотр и управление установленными сторонними плагинами. Используйте строку поиска для фильтрации плагинов по имени. Каждый плагин имеет значки для настроек, сочетаний клавиш, финансовой поддержки, удаления и переключатель для включения или отключения.
