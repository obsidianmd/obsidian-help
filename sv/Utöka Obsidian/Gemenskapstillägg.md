Lär dig hur du utökar Obsidian med tillägg byggda av gemenskapen. Använd tillägg för att anpassa Obsidian efter dina specifika behov, som stöd för ytterligare filformat eller integration med en tredjepartstjänst.

> [!warning] Varning
> Gemenskapstillägg kör tredjepartskod å dina vägnar som potentiellt kan orsaka skada. För att läsa mer om vad Obsidian-teamet gör för att förhindra skadliga tillägg, se [[Tilläggssäkerhet]].

## Bläddra bland gemenskapstillägg

1. Öppna **[[Inställningar]]**.
2. Välj **Slå på gemenskapstillägg**. För mer information, se [[Tilläggssäkerhet#Begränsat läge|Begränsat läge]].
3. Välj **Bläddra** för att lista alla tillgängliga gemenskapstillägg.

Använd textfältet för att filtrera tillägg baserat på namn, författare och beskrivning.

Du kan också bläddra bland tillgängliga tillägg i din webbläsare genom att gå till [obsidian.md/plugins](https://obsidian.md/plugins).

## Installera ett gemenskapstillägg

För att installera ett gemenskapstillägg måste du först stänga av [[Tilläggssäkerhet#Begränsat läge|Begränsat läge]].

1. Öppna **[[Inställningar]]**.
2. I sidomenyn, välj **Gemenskapstillägg**.
3. Välj **Bläddra** för att utforska tillgängliga gemenskapstillägg.
4. Välj det tillägg du vill installera.
5. Välj **Installera**.

För att använda ett installerat tillägg behöver du aktivera det.

## Aktivera ett gemenskapstillägg

Välj antingen **Aktivera** direkt efter installation, eller aktivera det från listan över gemenskapstillägg under **[[Inställningar]] → Gemenskapstillägg → Installerade tillägg**.

## Uppdatera tillägg

Av säkerhetsskäl uppdateras inte gemenskapstillägg automatiskt. Du kan antingen uppdatera alla tillägg på en gång eller uppdatera enskilda tillägg.

För att uppdatera alla dina tillägg:

1. Öppna **[[Inställningar]]**.
2. Under **Gemenskapstillägg → Nuvarande tillägg**, välj **Sök efter uppdateringar**.
3. Om uppdateringar finns tillgängliga, välj **Uppdatera alla**.

För att uppdatera ett enskilt tillägg:

1. Öppna **[[Inställningar]]**.
2. Under **Gemenskapstillägg → Nuvarande tillägg**, välj **Sök efter uppdateringar**.
3. Under **Installerade tillägg**, välj **Uppdatera** bredvid det tillägg du vill uppdatera.

## Avinstallera ett gemenskapstillägg

1. Öppna **[[Inställningar]]**.
2. I sidomenyn, välj **Gemenskapstillägg**.
3. Under **Installerade tillägg**, välj papperskorgsikonen ![[lucide-trash-2.svg#icon]] bredvid det tillägg du vill avinstallera.
4. Välj **Avinstallera** för att bekräfta.

## Hantera installerade tillägg

Under **[[Inställningar]] → Gemenskapstillägg → Installerade tillägg** har varje tillägg flera ikoner för hantering:

- **[[Inställningar]]** ![[lucide-settings.svg#icon]] — Öppna tilläggets inställningssida om det har konfigurerbara alternativ.
- **[[Snabbkommandon]]** ![[lucide-plus-circle.svg#icon]] — Ange tangentbordsgenvägar för tilläggets kommandon.
- **Finansiering** ![[lucide-heart.svg#icon]] — Visa tilläggsförfattarens finansierings-URL för att stödja deras arbete.
- **Avinstallera** ![[lucide-trash-2.svg#icon]] — Ta bort tillägget från ditt valv.
- **Växla** — Aktivera eller inaktivera tillägget utan att avinstallera det.

Du kan också:

- Välja **uppdateringsikonen** för att ladda om alla tillägg.
- Välja **mappikonen** ![[lucide-folder-open.svg#icon]] för att öppna tilläggsmappen i din [[Konfigurationsmapp|konfigurationsmapp]].
- Använda **sökfältet** för att filtrera tillägg efter namn.
