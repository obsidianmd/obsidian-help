Obsidian låter dig migrera dina anteckningar från Craft. Detta konverterar dina Craft-data till hållbara Markdown-filer som du kan använda med Obsidian och många andra appar.

## Exportera dina data från Craft

Öppna Craft-appen på din stationära dator:

1. Gå till vyn "Organize" eller "All Documents"
2. Klicka på trepunktsmenyn
3. Välj alternativet Exportera.
4. Välj Markdown.

Beroende på hur många anteckningar du har i Craft kan denna process ta en stund.

## Importera dina Craft-data till Obsidian

Härifrån kan du följa instruktionerna i [[Importera Markdown-filer]].

## Felsökning

Craft Docs använder icke-standardiserad syntax för vissa länkar. Du kan hitta resurser för att migrera dessa på [Importer GitHub-repo](https://github.com/obsidianmd/obsidian-importer/issues/27).
