Obsidian använder Markdown `.md`-filer som primärt format för anteckningar. Detta gör det enkelt att importera Markdown-filer till Obsidian. Många appar kan redigera Markdown-filer eller exportera till detta format.

Det finns flera sätt att lägga till Markdown-filer i ditt Obsidian-valv:

1. Dra och släpp filer eller mappar till [[Filutforskare]]-fönstret
2. Använd din systemfilhanterare (t.ex. Utforskaren i Windows eller Finder på macOS) och flytta dina filer direkt till Obsidian-valvmappen.

### Konvertera från andra varianter av Markdown

Obsidian använder [[Obsidian Flavored Markdown]]. Vissa appar använder något annorlunda Markdown-syntax. Du kan använda [[Markdown-formatimportör]] för att konvertera dina filer till Obsidians Markdown-syntax.
