Webbplatser med [[Introduktion till Obsidian Publish|Obsidian Publish]] optimeras automatiskt för sökmotorer och [[Förhandsvisning av länkar på sociala medier|delningskort för sociala medier]]. Du kan också anpassa sidmetadata med beskrivningar, permalänkar och bilder.

Observera att du kan tillåta inte sökmotorindexering i dina [[Hantera webbplatser#Webbplatsalternativ|webbplatsalternativ]].

### Google Search Console

För att spåra SEO med Google, registrera din webbplats på [Google Search Console](https://search.google.com/search-console). Du kan också följa våra rekommenderade steg för att [[Analys|konfigurera Google Analytics]].

### Webbplatskarta

Webbplatskartan för din Publish-webbplats finns tillgänglig på `/sitemap.xml`, till exempel:
https://help.obsidian.md/sitemap.xml

Webbplatskartan finns också tillgänglig som ett RSS-flöde på `/rss.xml`:
https://help.obsidian.md/rss.xml

### Permalänkar

Överväg att använda [[Permalänkar]] för att definiera permanenta URL:er på din webbplats.

### Metadata

Sidmetadata kan anpassas med hjälp av [[Egenskaper#Egenskaper för Obsidian Publish|Egenskaper]], vilket kan vara användbart för [[Förhandsvisning av länkar på sociala medier|delningskort för sociala medier]].

| Egenskap      | Beskrivning                                                                                                       |
| ------------- | ----------------------------------------------------------------------------------------------------------------- |
| `publish`     | Se [[Publicera ditt innehåll#Välj automatiskt data att publicera\|Välj automatiskt data att publicera]]. |
| `permalink`   | Se [[Permalänkar\|Permalänkar]].                                                                                   |
| `description` | Se [[Förhandsvisning av länkar på sociala medier#Beskrivning\|Beskrivning]].                                                      |
| `image`       | Se [[Förhandsvisning av länkar på sociala medier#Bild\|Bild]].                                                                  |
| `cover`       | Se [[Förhandsvisning av länkar på sociala medier#Bild\|Bild]].                                                                  |
