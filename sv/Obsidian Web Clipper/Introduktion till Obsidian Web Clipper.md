Obsidian Web Clipper är ett gratis webbläsartillägg som låter dig markera sidor och spara webbinnehåll till ditt valv.

## Installera Web Clipper

Hämta tillägget Obsidian Web Clipper från den officiella katalogen för din webbläsare:

- [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
	- För Chrome, Brave, Arc, Orion och andra Chromium-baserade webbläsare.
- [Firefox Add-Ons](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
	- För Firefox och Firefox Mobile.
- [Safari Extensions](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
	- För macOS, iOS och iPadOS.
- [Edge Add-Ons](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
	- För Microsoft Edge.

## Hur du använder Web Clipper

- [[Klipp webbsidor]]
	- Kom igång och lär dig hur du sparar innehåll från sidor.
- [[Highlighter|Markering]]
	- Markera viktiga passager och välj element du vill spara.
- [[Reader|Läsare]]
	- Läs webbsidor i en ren, distraktionsfri vy.
- [[Interpreter|Tolk]]
	- Använd naturliga språkpromptar för att fånga och modifiera data på en sida.
- [[Felsök Web Clipper|Felsökning]]
	- Lös vanliga problem med Web Clipper.
- [[Obsidian Web Clipper/Mallar|Mallar]]
	- Skapa anpassade mallar för specifika webbplatser.
- [[Variabler]]
	- Fyll i siddata automatiskt i din mall.
- [[Filter]]
	- Modifiera utdata från mallvariabler.
- [[Logik]]
	- Använd villkor, loopar och sätt variabler i mallar.

## Integritet

Obsidian Web Clipper sparar innehåll lokalt till ditt Obsidian-valv och följer vår [integritetspolicy](https://obsidian.md/privacy). Din data samlas inte in, och vi samlar inte in några användningsmetrik. Koden är [öppen källkod](https://github.com/obsidianmd/obsidian-clipper) och granskningsbar.
