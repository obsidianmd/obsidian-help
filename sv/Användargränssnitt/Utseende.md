### Färgschema

Obsidian levereras med två färgscheman, ljust och mörkt. Som standard är färgschemat inställt på att anpassa sig efter ditt operativsystem. Du kan ändra detta genom att gå till **[[Inställningar]] → Utseende**.

### Anpassade teman

Du kan ändra utseendet på Obsidian genom att använda [[Teman|anpassade teman]]. Bläddra bland hundratals gemenskapsskapade teman genom att gå till **[[Inställningar]] → Utseende** och sedan klicka på **Hantera** för att visa och installera teman.

Du kan också justera teman med [[CSS-instick]] och [bygga ditt eget tema](https://docs.obsidian.md/Themes/App+themes/Build+a+theme).

### Anpassad appikon

Du kan ändra Obsidians skrivbordsappikon. Prova ikonredigeraren som finns på [vår blogg](https://obsidian.md/blog/new-obsidian-icon/) för att anpassa ikonfärgerna efter dina önskemål.

För att ändra Obsidians appikon, följ dessa steg:

1. Öppna **[[Inställningar]]**.
2. Välj **Utseende** till vänster.
3. Rulla ner till avsnittet **Avancerat**.
4. Under **Anpassad appikon** klicka på **Välj**.
5. Välj önskad ikon från ditt filsystem. Accepterade filformat är `.icns`, `.ico`, `.png` och `.svg`.
6. Klicka på **Starta om**.
7. Efter att Obsidian startat om bör din nya ikon visas i aktivitetsfältet på skrivbordet och på din skärm på mobil.

> [!tip] Anpassad appikon i Windows aktivitetsfält
> För vissa Windows-operativsystem kan ett extra steg behövas för att justera den anpassade appikonen i aktivitetsfältet.
> 
> Följ dessa steg:
> - Högerklicka på Obsidian-ikonen i aktivitetsfältet eller på skrivbordsgenvägen.
> - Högerklicka på namnet Obsidian (hoppa över detta steg för skrivbordsikonen).
> - Välj **Egenskaper**.
> - Klicka på **Byt ikon**.
> - Hitta ikonen i ditt filsystem.
> - Klicka på **OK**.
> 
> Efteråt har du möjlighet att antingen starta om `explorer.exe` i aktivitetshanteraren eller starta om datorn för att tillämpa ändringarna du gjort.

### Genomskinlighet

På MacOS kan du göra Obsidian-fönstret genomskinligt genom att aktivera detta alternativ i inställningsfliken **Utseende**.

> [!info]- Windows och genomskinlighet
> Obsidian-teamet tog bort genomskinlighet i Windows för version 1.15.11 på grund av att Electron tog bort funktionen.
