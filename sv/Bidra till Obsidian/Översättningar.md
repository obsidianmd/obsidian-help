Om du kan ett annat språk, helst som modersmål, kan du hjälpa till att översätta Obsidians användargränssnitt eller dokumentationen.

För att hjälpa till att översätta användargränssnittet, skicka in en pull request till [översättningsrepot](https://github.com/obsidianmd/obsidian-translations).

För att hjälpa till att förbättra vår dokumentation, skicka in en pull request till [obsidian-help](https://github.com/obsidianmd/obsidian-help)-repot. Se till att följa [[Stilguide#Översättningar|stilguiden för översättningar]] när du gör ändringar.
