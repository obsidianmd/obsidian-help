Om du är utvecklare kan du bidra till Obsidian på olika sätt.

### Tillägg och teman

Du kan utveckla [[Gemenskapstillägg]] och [[Teman]] med hjälp av Typescript och CSS. Läs mer genom att besöka [Obsidians utvecklardokumentation](https://docs.obsidian.md).

### Öppen källkod-projekt

Obsidian utvecklar och underhåller flera öppen källkod-projekt. Du kan skicka pull requests med buggfixar och förbättringar till följande GitHub-arkiv:

- [JSON Canvas](https://github.com/obsidianmd/jsoncanvas) (MIT), ett öppet filformat för oändlig canvas-data.
- [Obsidian Importer](https://github.com/obsidianmd/obsidian-importer) (MIT), ett tillägg som konverterar data från en mängd olika format till Markdown-filer i oformaterad text.
- [Obsidian Maps](https://github.com/obsidianmd/obsidian-maps) (MIT), ett tillägg som visar kartor i Obsidian-baser.
- [Obsidian Web Clipper](https://github.com/obsidianmd/obsidian-clipper) (MIT), ett webbläsartillägg för att markera och spara webbsidor som Markdown-filer i oformaterad text.
- [Defuddle](https://github.com/kepano/defuddle) (MIT), HTML-till-Markdown-tolken som används av Obsidian Web Clipper.

### Dokumentation

För att hjälpa till att förbättra vår [utvecklardokumentation](https://docs.obsidian.md/Home), skicka en pull request till arkivet [obsidian-developer-docs](https://github.com/obsidianmd/obsidian-developer-docs).

För att hjälpa till att förbättra denna dokumentation, skicka en pull request till arkivet [obsidian-help](https://github.com/obsidianmd/obsidian-help).

### Gemenskap

Kanalen [plugin-dev](https://discord.com/channels/686053708261228577/840286264964022302) på vår officiella Discord-server är platsen där du kan diskutera tilläggsutveckling och hjälpa andra utvecklare med deras tillägg.
