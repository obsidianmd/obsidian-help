I vissa jurisdiktioner är Obsidian skyldigt att ta ut moms å kunders vägnar vid köp av programvarulicenser och tjänster. Moms kan tillkomma för [[Kommersiell licens|Kommersiella licenser]], [[Introduktion till Obsidian Sync|Obsidian Sync]] och [[Introduktion till Obsidian Publish|Obsidian Publish]]. Moms gäller i allmänhet inte för [[Obsidian-kredit]] och [[Catalyst-licens|Catalyst]], även om undantag kan förekomma.

För att bestämma vilka skatter som tillämpas på ditt köp använder vi det land och postnummer du anger vid kassan eller i faktureringssektionen på din instrumentpanel. Denna information lagras på ditt konto så att vi korrekt kan debitera moms om det sker en ändring i skattelagstiftningen eller andra krav som förpliktar oss att göra det.

## Skattebefrielse

Beroende på ditt land och din status kan du vara befriad från skatter genom att ange ditt skatteregistreringsnummer (TIN), momsnummer, moms-ID eller skattebefrielsecertifikat.

Du kan ansöka om skattebefrielse via faktureringssektionen på din [kontoinstrumentpanel](https://obsidian.md/account/billing). Om din ansökan godkänns kommer framtida köp inte att belastas med skatt.

### Ansöka om befrielse

1. Logga in på ditt [Obsidian-konto](https://obsidian.md/account/billing).
2. Navigera till sektionen [Fakturering](https://obsidian.md/account/billing).
3. Till höger om _Skattebefrielse_, välj knappen **Hantera**.
4. I popup-fönstret:
    1. Ange ditt land.
    2. Ange ditt postnummer, om tillämpligt.

Beroende på ditt land kan du behöva ange ett momsnummer eller skattebefrielsecertifikat.

- Momsnummer
	- Ange ditt momsnummer och välj **Spara** för att skicka in din ansökan.
- Skattebefrielsecertifikat
	1. Under _Skattebefrielsecertifikat_, välj knappen **Bläddra** för att välja ditt skattebefrielsecertifikat från din enhet. Se till att filen är i `.pdf`-format och är mindre än 3 MB.
	2. Efter att du valt filen, välj **Ladda upp** för att skicka in din ansökan.

Verifieringsprocessen kan ske omedelbart beroende på din plats. När ansökan godkänts kommer framtida köp inte att belastas med skatt.

> [!note] Observera: att ansöka om skattebefrielse kommer också att uppdatera din faktureringsadress till det postnummer du valde.
