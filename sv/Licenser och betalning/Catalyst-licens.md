Catalyst-licensen är ett engångsköp som ger dig tidig åtkomst till betaversioner och hjälper till att stödja den fortsatta utvecklingen av Obsidian. Du kan stödja Obsidian genom att [köpa en catalyst-licens](https://obsidian.md/buy/catalyst).

Fördelarna inkluderar:

- Tidig åtkomst till [[#Aktivera tidig åtkomst-versioner|nya versioner]] av Obsidian
- Åtkomst till exklusiva [[#Gå med i våra Insider-kanaler|Insider-kanaler]].
- Exklusiva [[#Hämta ditt forummärke|forummärken]] och [[#Hämta ditt Discord-märke|Discord-märken]].

## Catalyst-nivåer

Det finns tre nivåer av Catalyst baserat på hur mycket du bidrar med: *Insider*, *Supporter* och *VIP*.

| Fördelar                                 | Insider | Supporter |   VIP    |
| ---------------------------------------- | :-----: | :-------: | :------: |
| Tidig åtkomst till nya versioner         |    ✓    |     ✓     |    ✓     |
| Åtkomst till exklusiva utvecklingskanaler |    ✓    |     ✓     |    ✓     |
| Åtkomst till exklusiv loungekanal        |         |     ✓     |    ✓     |
| Insider-märke                            |    ✓    |           |          |
| Supporter-märke                          |         |     ✓     |          |
| VIP-märke                                |         |           |    ✓     |
| **Bidrag (USD)**                         | **$25** |  **$50**  | **$100** |


> [!info] Uppgradera din Catalyst-nivå
> Om du redan äger en catalyst-licens kan du uppgradera till en högre Catalyst-nivå genom att betala prisskillnaden mellan din nuvarande och den nya nivån.

## Få åtkomst till Catalyst-fördelar

När du har köpt en catalyst-licens kan du få åtkomst till en rad fördelar. Lär dig hur du aktiverar dem här.

### Hämta ditt forummärke

**Observera:** För att koppla ditt Obsidian-konto och ditt forumkonto måste du använda samma e-postadress för båda kontona.

1. Om du inte redan gjort det, gå med i vårt [forum](https://forum.obsidian.md/).
2. Gå till din [Obsidian-kontopanel](https://obsidian.md/account/catalyst).
3. Under **Catalyst**, välj **Get forum badge**.

### Hämta ditt Discord-märke

1. Om du inte redan gjort det, gå med i vår [Discord-server](https://discord.gg/veuWUTm).
2. Gå till din [Obsidian-kontopanel](https://obsidian.md/account/catalyst).
3. Under **Catalyst**, välj **Get Discord badge**.

### Gå med i våra Insider-kanaler

Insiders kan chatta med Obsidian-teamet och andra Insiders i våra exklusiva Discord-kanaler. Du går automatiskt med i Insider-kanalerna efter att du [[#Hämta ditt Discord-märke]].

- [vip-lounge](https://discord.com/channels/686053708261228577/714234422379151400) för diskussioner med Catalyst-medlemmar.
- [insider-desktop-release](https://discord.com/channels/686053708261228577/716028884885307432) för de senaste tidig åtkomst-versionerna på desktop.
- [insider-mobile-release](https://discord.com/channels/686053708261228577/817515900349448202) för de senaste tidig åtkomst-versionerna på mobil.
