Den här sidan listar standardtangentbordsgenvägar för navigering och redigering av text i Obsidian. Dessa genvägar tillhandahålls av ditt operativsystem eller det ramverk som Obsidian är byggt på, och kan inte anpassas inom Obsidian.

För anpassningsbara tangentbordsgenvägar för Obsidian-kommandon, se [[Snabbkommandon]].

## Genvägar för Windows och Linux

### Vanliga åtgärder

| Åtgärd | Genväg |
|-|-|
| Kopiera | `Ctrl+C` |
| Klipp ut | `Ctrl+X` |
| Klistra in | `Ctrl+V` |
| Klistra in utan formatering | `Ctrl+Shift+V` |
| Ångra | `Ctrl+Z` |
| Gör om | `Ctrl+Shift+Z` eller `Ctrl+Y` |
| Kopiera stycke | `Ctrl+C` (utan markerad text) |
| Klipp ut stycke | `Ctrl+X` (utan markerad text) |

### Textredigering

| Åtgärd | Genväg |
|-|-|
| Infoga ny rad | `Enter` |
| Radera föregående tecken | `Backspace` |
| Radera nästa tecken | `Delete` |
| Radera föregående ord | `Ctrl+Backspace` |
| Radera nästa ord | `Ctrl+Delete` |
| Radera aktuell rad | `Ctrl+Shift+K` (utan markerad text) |

### Textnavigering

| Åtgärd | Genväg |
|-|-|
| Flytta markören ett tecken | `Left/→` |
| Flytta markören till början av föregående ord | `Ctrl+←` |
| Flytta markören till slutet av nästa ord | `Ctrl+→` |
| Flytta markören till början av aktuell rad | `Home` |
| Flytta markören till slutet av aktuell rad | `End` |
| Flytta markören till föregående rad | `↑` |
| Flytta markören till nästa rad | `↓` |
| Flytta markören till början av anteckningen | `Ctrl+Home` |
| Flytta markören till slutet av anteckningen | `Ctrl+End` |
| Flytta markören upp en sida | `Page up` |
| Flytta markören ned en sida | `Page down` |

### Textmarkering

| Åtgärd | Genväg |
|-|-|
| Förenkla markering | `Escape` |
| Markera allt | `Ctrl+A` |
| Utöka markering ett tecken | `Shift+Left/→` |
| Utöka markering till början av föregående ord | `Ctrl+Shift+←` |
| Utöka markering till slutet av nästa ord | `Ctrl+Shift+→` |
| Utöka markering till början av aktuell rad | `Shift+Home` |
| Utöka markering till slutet av aktuell rad | `Shift+End` |
| Utöka markering till början av anteckningen | `Ctrl+Shift+Home` |
| Utöka markering till slutet av anteckningen | `Ctrl+Shift+End` |
| Utöka markering en sida upp | `Shift+Page up` |
| Utöka markering en sida ned | `Shift+Page down` |

## macOS-genvägar

### Vanliga åtgärder

| Åtgärd | Genväg |
|-|-|
| Kopiera | `Cmd+C` |
| Klipp ut | `Cmd+X` |
| Klistra in | `Cmd+V` |
| Klistra in utan formatering | `Cmd+Shift+V` |
| Ångra | `Cmd+Z` |
| Gör om | `Cmd+Shift+Z` |
| Kopiera stycke | `Cmd+C` (utan markerad text) |
| Klipp ut stycke | `Cmd+X` (utan markerad text) |

### Textformatering

| Åtgärd | Genväg |
| ---- | ---- |
| Fet text | `Cmd+B` |
| Kursiv text | `Cmd+I` |

### Textredigering

| Åtgärd | Genväg |
|-|-|
| Infoga ny rad | `Enter` |
| Radera föregående tecken | `Backspace` |

| Radera till början av aktuell rad | `Cmd+Backspace` |
| Radera till slutet av aktuell rad | `Cmd+Delete` |
| Radera aktuell rad | `Cmd+Shift+K` (utan markerad text) |

### Textnavigering

| Åtgärd | Genväg |
|-|-|

| Flytta markören till början av föregående ord | `Option+←` |
| Flytta markören till slutet av nästa ord | `Option+→` |
| Flytta markören till början av aktuell rad | `Cmd+←` |
| Flytta markören till slutet av aktuell rad | `Cmd+→` |
| Flytta markören till föregående rad | `↑` |
| Flytta markören till nästa rad | `↓` |
| Flytta markören till början av anteckningen | `Cmd+↑` |
| Flytta markören till slutet av anteckningen | `Cmd+↓` |
