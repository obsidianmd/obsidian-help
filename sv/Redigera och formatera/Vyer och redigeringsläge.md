Obsidian låter dig styra hur du läser och redigerar anteckningar skrivna i Markdown med hjälp av _vyer_ och _lägen_.

- **Vyer** växlar mellan att läsa och redigera dina anteckningar.
- **Lägen** styr hur Markdown visas medan du redigerar.

> [!note] Observera
> Som standard öppnar Obsidian nya flikar i redigeringsläge. Du kan ändra detta under **[[Inställningar]] → Redigerare → Standardvy för nya tabbar**.

## Läsvy

*Läsvy* visar din anteckning utan Markdown-syntax och erbjuder ett rent, lättläst format för fokuserad granskning.

För att växla till *Läsvy*:

- Klicka på vyväxlaren ![[lucide-book-icon.svg#icon]] i det övre högra hörnet av redigeraren.
- Eller klicka på den interaktiva statusikonen ![[lucide-edit-3.svg#icon]] eller ![[lucide-code-xml.svg#icon]] i statusfältet och välj **Läsvy**.
- Eller tryck `Ctrl+E` (`Cmd+E` på macOS).

> [!note] Observera
> Aktivera **[[Inställningar]] → Utseende → Visa tabbars titel-bar** för att se vyväxlarikonen.
> 
> Om det är inaktiverat, aktivera **[[Inställningar]] → Redigerare → Redigerarstatus** för att växla vyer från statusfältet.

> [!tip] Visa sida vid sida
> För att öppna en anteckning i både *Redigerare* och *Läsvy* samtidigt, håll `Ctrl` (eller `Cmd` på macOS) och klicka på vyväxlaren.

## Redigerare

*Redigerare* låter dig göra ändringar i din anteckning.

I *Redigerare* bestämmer *Redigeringsläget* hur Markdown visas. Du kan välja ett av två *Redigeringslägen*: *Live-förhandsvisning* eller *Källkodsläge*.

### Live-förhandsvisning

*Live-förhandsvisning* visar formaterad text inline medan den döljer de flesta Markdown-syntaxer. När din markör hamnar i formaterat innehåll blir den underliggande syntaxen synlig för redigering.

För att växla till *Live-förhandsvisning*:

- Klicka på vyväxlaren ![[lucide-edit-3.svg#icon]] i det övre högra hörnet av redigeraren.
- Eller klicka på den interaktiva statusikonen ![[lucide-book-icon.svg#icon]] eller ![[lucide-code-xml.svg#icon]] i statusfältet och välj **Live-förhandsvisning**.
- Eller tryck `Ctrl+E` (`Cmd+E` på macOS).
- Eller använd [[Kommandopalett|kommandot]] **Växla redigerings-/förhandsvy**.

> [!note] Observera
> Som standard är redigeringsvyn inställd på *Live-förhandsvisning*. Du kan ändra detta under **[[Inställningar]] → Redigerare → Normala skrivläget**.

> [!tip] I många fall kan Live-förhandsvisning eliminera behovet av att växla till [[#Läsvy]].

### Källkodsläge

*Källkodsläge* visar all Markdown-syntax exakt som den är skriven. Använd det om du föredrar oformaterad text eller behöver exakt formateringskontroll.

För att växla till *Källkodsläge*:
- Klicka på den interaktiva statusikonen ![[lucide-book-icon.svg#icon]] eller ![[lucide-edit-3.svg#icon]] i statusfältet och välj **Källkodsläge**.

> [!note] Observera
> Som standard är redigeringsvyn inställd på *Live-förhandsvisning*. Ändra detta till *Källkodsläge* under **[[Inställningar]] → Redigerare → Normala skrivläget**.
> 
> För att växla till *Källkodsläge* kan du nu dessutom:
> 
> - Klicka på vyväxlaren ![[lucide-edit-3.svg#icon]] i det övre högra hörnet av din anteckning.
> - Eller trycka `Ctrl+E` (`Cmd+E` på macOS).
> - Eller använda [[Kommandopalett|kommandot]] **Växla redigerings-/förhandsvy**.

> [!tip] Växla *Redigeringsläge*
> För att snabbt växla mellan *Live-förhandsvisning* och *Källkodsläge* kan du ställa in en [[Snabbkommandon|tangentbordsgenväg]] för [[Kommandopalett|kommandot]] **Växla förhandsvy/källkodsläge**.
