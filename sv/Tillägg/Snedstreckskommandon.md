Snedstreckskommandon är ett [[Kärntillägg|kärntillägg]] som låter dig utföra kommandon i redigeraren genom att skriva ett snedstreck (`/`) i början av en rad eller efter ett mellanslag.

För att köra ett kommando med Snedstreckskommandon:

1. I redigeraren, skriv ett snedstreck (`/`).
2. Ange namnet på kommandot du vill köra för att söka i listan över tillgängliga kommandon.
3. Navigera till kommandot med piltangenterna.
4. Tryck på Enter.

För att avsluta sökningen i Snedstreckskommandon utan att köra ett kommando, tryck på `Esc` eller `Mellanslag`.

> [!TIP] Suddig matchning
> Precis som [[Kommandopalett|kommandopaletten]] stöder Snedstreckskommandon suddig matchning som låter dig söka efter kommandon även om du inte vet deras exakta namn. Skriv till exempel "saf" för att hitta kommandot **Spara aktuell fil**.
