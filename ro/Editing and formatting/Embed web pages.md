Aflați cum să folosiți elementul HTML [iframe](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/iframe) pentru a încorpora pagini web în notele dvs.

Pentru a încorpora o pagină web, adăugați următorul cod în nota dvs. și înlocuiți textul substituent cu URL-ul paginii web pe care doriți să o încorporați:

```html
<iframe src="INSERT YOUR URL HERE"></iframe>
```

> [!note] Notă
> Unele site-uri web nu permit încorporarea lor. În schimb, este posibil să ofere URL-uri destinate special încorporării. Dacă site-ul web nu acceptă încorporarea, încercați să căutați numele site-ului urmat de „embed iframe”. De exemplu, „youtube embed iframe”.

> [!tip] Sfat
> Dacă folosiți [[Canvas|Pânza]], puteți încorpora o pagină web într-un card. Pentru mai multe informații, consultați [[Canvas#Add cards from web pages]].

## Încorporați un videoclip YouTube

Pentru a încorpora un videoclip YouTube, folosiți aceeași sintaxă Markdown ca pentru [[Basic formatting syntax#External images|imaginile externe]]:

```md
![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)
```

![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)

## Încorporați un tweet

Pentru a încorpora un tweet, folosiți aceeași sintaxă Markdown ca pentru [[Basic formatting syntax#External images|imaginile externe]]:

```md
![](https://twitter.com/obsdmd/status/1580548874246443010)
```

![](https://twitter.com/obsdmd/status/1580548874246443010)
