Notlarınıza web sayfaları gömmek için [iframe](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/iframe) HTML öğesini nasıl kullanacağınızı öğrenin.

Bir web sayfası gömmek için notunuza aşağıdaki kodu ekleyin ve yer tutucu metni gömmek istediğiniz web sayfasının URL'si ile değiştirin:

```html
<iframe src="URL'NİZİ BURAYA EKLEYIN"></iframe>
```

> [!note] Not
> Bazı web siteleri gömülmelerine izin vermez. Bunun yerine, gömülmek için tasarlanmış URL'ler sağlayabilirler. Web sitesi gömmeyi desteklemiyorsa, web sitesinin adını "embed iframe" ifadesiyle birlikte aramayı deneyin. Örneğin, "youtube embed iframe".

> [!tip] İpucu
> [[Tuval]] kullanıyorsanız, bir karta web sayfası gömebilirsiniz. Daha fazla bilgi için [[Tuval#Web sayfalarından kart ekleme]] bölümüne bakın.

## Bir YouTube videosu gömme

Bir YouTube videosu gömmek için [[Temel biçimlendirme söz dizimi#Harici görseller|harici görseller]] ile aynı Markdown söz dizimini kullanın:

```md
![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)
```

![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)

Bir tweet gömmek için [[Temel biçimlendirme söz dizimi#Harici görseller|harici görseller]] ile aynı Markdown söz dizimini kullanın:
