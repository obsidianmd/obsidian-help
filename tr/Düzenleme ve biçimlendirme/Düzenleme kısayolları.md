Bu sayfa, Obsidian'da metin gezinme ve düzenleme için varsayılan klavye kısayollarını listeler. Bu kısayollar işletim sisteminiz veya Obsidian'ın üzerine inşa edildiği çatı tarafından sağlanır ve Obsidian içinde özelleştirilemez.

Obsidian komutları için özelleştirilebilir klavye kısayolları hakkında bilgi almak için [[Kısayol tuşları]] sayfasına bakın.

## Windows ve Linux kısayolları

### Genel eylemler

| Eylem | Kısayol |
|-|-|
| Kopyala | `Ctrl+C` |
| Kes | `Ctrl+X` |
| Yapıştır | `Ctrl+V` |
| Biçimlendirmeden yapıştır | `Ctrl+Shift+V` |
| Geri al | `Ctrl+Z` |
| Yinele | `Ctrl+Shift+Z` veya `Ctrl+Y` |
| Paragrafı kopyala | `Ctrl+C` (metin seçili değilken) |
| Paragrafı kes | `Ctrl+X` (metin seçili değilken) |

### Metin düzenleme

| Eylem | Kısayol |
|-|-|
| Yeni satır ekle | `Enter` |
| Önceki karakteri sil | `Backspace` |
| Sonraki karakteri sil | `Delete` |
| Önceki kelimeyi sil | `Ctrl+Backspace` |
| Sonraki kelimeyi sil | `Ctrl+Delete` |
| Geçerli satırı sil | `Ctrl+Shift+K` (metin seçili değilken) |

### Metin gezinme

| Eylem | Kısayol |
|-|-|
| İmleci bir karakter taşı | `Left/→` |
| İmleci önceki kelimenin başına taşı | `Ctrl+←` |
| İmleci sonraki kelimenin sonuna taşı | `Ctrl+→` |
| İmleci geçerli satırın başına taşı | `Home` |
| İmleci geçerli satırın sonuna taşı | `End` |
| İmleci önceki satıra taşı | `↑` |
| İmleci sonraki satıra taşı | `↓` |
| İmleci notun başına taşı | `Ctrl+Home` |
| İmleci notun sonuna taşı | `Ctrl+End` |
| İmleci bir sayfa yukarı taşı | `Page up` |
| İmleci bir sayfa aşağı taşı | `Page down` |

### Metin seçme

| Eylem | Kısayol |
|-|-|
| Seçimi basitleştir | `Escape` |
| Tümünü seçin | `Ctrl+A` |
| Seçimi bir karakter genişlet | `Shift+Left/→` |
| Seçimi önceki kelimenin başına genişlet | `Ctrl+Shift+←` |
| Seçimi sonraki kelimenin sonuna genişlet | `Ctrl+Shift+→` |
| Seçimi geçerli satırın başına genişlet | `Shift+Home` |
| Seçimi geçerli satırın sonuna genişlet | `Shift+End` |
| Seçimi notun başına genişlet | `Ctrl+Shift+Home` |
| Seçimi notun sonuna genişlet | `Ctrl+Shift+End` |
| Seçimi bir sayfa yukarı genişlet | `Shift+Page up` |
| Seçimi bir sayfa aşağı genişlet | `Shift+Page down` |

## macOS kısayolları

### Genel eylemler

| Eylem | Kısayol |
|-|-|
| Kopyala | `Cmd+C` |
| Kes | `Cmd+X` |
| Yapıştır | `Cmd+V` |
| Biçimlendirmeden yapıştır | `Cmd+Shift+V` |
| Geri al | `Cmd+Z` |
| Yinele | `Cmd+Shift+Z` |
| Paragrafı kopyala | `Cmd+C` (metin seçili değilken) |
| Paragrafı kes | `Cmd+X` (metin seçili değilken) |

### Metin biçimlendirme

| Eylem | Kısayol |
| ---- | ---- |
| Kalın metin | `Cmd+B` |
| İtalik metin | `Cmd+I` |

### Metin düzenleme

| Eylem | Kısayol |
|-|-|
| Yeni satır ekle | `Enter` |
| Önceki karakteri sil | `Backspace` |
| Sonraki karakteri sil | `Delete` |
| Önceki kelimeyi sil | `Option+Backspace` |
| Sonraki kelimeyi sil | `Option+Delete` |
| Geçerli satırın başına kadar sil | `Cmd+Backspace` |
| Geçerli satırın sonuna kadar sil | `Cmd+Delete` |
| Geçerli satırı sil | `Cmd+Shift+K` (metin seçili değilken) |

### Metin gezinme

| Eylem | Kısayol |
|-|-|
| İmleci bir karakter taşı | `Left/→` |
| İmleci önceki kelimenin başına taşı | `Option+←` |
| İmleci sonraki kelimenin sonuna taşı | `Option+→` |
| İmleci geçerli satırın başına taşı | `Cmd+←` |
| İmleci geçerli satırın sonuna taşı | `Cmd+→` |
| İmleci önceki satıra taşı | `↑` |
| İmleci sonraki satıra taşı | `↓` |
| İmleci notun başına taşı | `Cmd+↑` |
| İmleci notun sonuna taşı | `Cmd+↓` |
| İmleci bir sayfa yukarı taşı | `Fn+↑` |
| İmleci bir sayfa aşağı taşı | `Fn+↓` |

### Metin seçme

| Eylem                                                  | Kısayol                    |
| ------------------------------------------------------ | -------------------------- |
| Seçimi basitleştir                                     | `Escape`                   |
| Tümünü seçin                                           | `Cmd+A`                    |
| Seçimi bir karakter genişlet                           | `Shift+Left/→`   |
| Seçimi önceki kelimenin başına genişlet                | `Option+Shift+←`  |
| Seçimi sonraki kelimenin sonuna genişlet               | `Option+Shift+→` |
| Seçimi geçerli satırın başına genişlet                 | `Cmd+Shift+←`              |
| Seçimi geçerli satırın sonuna genişlet                 | `Cmd+Shift+→`    |
| Seçimi notun başına genişlet                           | `Cmd+Shift+↑`       |
| Seçimi notun sonuna genişlet                           | `Cmd+Shift+↓`     |
| Seçimi bir sayfa yukarı genişlet                       | `Ctrl+Shift+↑`      |
| Seçimi bir sayfa aşağı genişlet                        | `Ctrl+Shift+↓`    |
