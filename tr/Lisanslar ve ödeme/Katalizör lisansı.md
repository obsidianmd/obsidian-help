Catalyst lisansı, beta sürümlerine erken erişim sağlayan ve Obsidian'ın sürekli geliştirilmesini destekleyen tek seferlik bir satın alımdır. [Catalyst lisansı satın alarak](https://obsidian.md/buy/catalyst) Obsidian'ı destekleyebilirsiniz.

Avantajlar şunlardır:

- Obsidian'ın [[#Erken erişim sürümlerini etkinleştirin|yeni sürümlerine]] erken erişim
- Özel [[#Insider kanallarımıza katılın|Insider kanallarına]] erişim.
- Özel [[#Forum rozetinizi alın|forum rozetleri]] ve [[#Discord rozetinizi alın|Discord rozetleri]].

## Catalyst kademeleri

Katkıda bulunduğunuz miktara göre üç Catalyst kademesi vardır: *Insider*, *Supporter* ve *VIP*.

| ---------------------------------------- | :-----: | :-------: | :------: |

| Özel geliştirme kanallarına erişim       |    ✓    |     ✓     |    ✓     |
| Özel salon kanalına erişim               |         |     ✓     |    ✓     |
| Insider rozeti                           |    ✓    |           |          |
| Supporter rozeti                         |         |     ✓     |          |
| VIP rozeti                               |         |           |    ✓     |
| **Katkı (USD)**                          | **$25** |  **$50**  | **$100** |


> [!info] Catalyst kademenizi yükseltin
> Zaten bir Catalyst lisansınız varsa, mevcut kademeniz ile yeni kademe arasındaki fiyat farkını ödeyerek daha yüksek bir Catalyst kademesine yükseltebilirsiniz.

## Catalyst avantajlarına erişin

Catalyst lisansı satın aldıktan sonra çeşitli avantajlara erişebilirsiniz. Bunları nasıl etkinleştireceğinizi buradan öğrenin.

### Forum rozetinizi alın

**Not:** Obsidian hesabınızı ve forum hesabınızı bağlamak için her iki hesapta da aynı e-posta adresini kullanmanız gerekir.

1. Henüz yapmadıysanız, [forumumuza](https://forum.obsidian.md/) katılın.
2. [Obsidian hesap panonuza](https://obsidian.md/account/catalyst) gidin.
3. **Catalyst** altında **Get forum badge** seçeneğini seçin.

### Discord rozetinizi alın

1. Henüz yapmadıysanız, [Discord sunucumuza](https://discord.gg/veuWUTm) katılın.
2. [Obsidian hesap panonuza](https://obsidian.md/account/catalyst) gidin.
3. **Catalyst** altında **Get Discord badge** seçeneğini seçin.

### Insider kanallarımıza katılın

Insider'lar, özel Discord kanallarımızda Obsidian ekibi ve diğer Insider'larla sohbet edebilir. [[#Discord rozetinizi alın|Discord rozetinizi aldıktan]] sonra Insider kanallarına otomatik olarak katılırsınız.

- [vip-lounge](https://discord.com/channels/686053708261228577/714234422379151400) Catalyst üyeleriyle tartışmalar için.
- [insider-desktop-release](https://discord.com/channels/686053708261228577/716028884885307432) masaüstünde en son erken erişim sürümleri için.
- [insider-mobile-release](https://discord.com/channels/686053708261228577/817515900349448202) mobilde en son erken erişim sürümleri için.
