[[Obsidian Web Kırpıcı'ya giriş|Web Kırpıcı]], web sayfalarındaki gereksiz öğeleri kaldırarak yalnızca ana makale içeriğini temiz ve okunabilir bir biçimde sunan, **Okuyucu** adlı yerleşik bir okuma moduna sahiptir.

## Okuma modunu etkinleştirme

Okuyucu'yu birkaç yolla etkinleştirebilirsiniz:

- ![[lucide-book-icon.svg#icon]] Eklenti panelindeki **Kitap simgesi**.
- Eklenti ayarlarından özelleştirilebilir **klavye kısayolu**.
- Ziyaret ettiğiniz web sayfasına sağ tıklayarak **bağlam menüsü**.

Ayrıca ![[obsidian.svg#icon]] eklenti düğmesini tek tıklamayla Okuyucu'yu açacak şekilde ayarlayabilirsiniz. Web Kırpıcı ayarlarında **Genel** → **Davranış** → **Açma davranışı** bölümüne gidin ve **Okuyucu** seçeneğini belirleyin.

Yukarıdaki seçeneklerin tümü ayrıca Okuyucu'dan çıkmanıza ve orijinal sayfayı geri yüklemenize olanak tanır.

## Nasıl çalışır

Okuyucu, sayfadaki ana içeriği çıkarır. Orijinal sayfa, aşağıdakileri içeren temiz bir okuma görünümüyle değiştirilir:

- Makale başlığı, yazar, yayın tarihi ve kaynak alan adı.
- Görseller ve biçimlendirme korunmuş şekilde ana makale içeriği.
- Birden fazla başlığı olan makaleler için kenar çubuğunda bir anahat.
- Kod blokları için söz dizimi vurgulama.
- Tıklandığında satır içi görüntülenen dipnot açılır pencereleri.

## Okuma görünümünüzü özelleştirme

Eklenti ayarlarından Okuyucu deneyiminizi özelleştirin. Bazı seçenekler ayrıca Okuyucu modunda ![[lucide-case-sensitive.svg#icon]] simgesine tıklayarak da kullanılabilir.

| Ayar                    | Açıklama                                        |
| ----------------------- | ----------------------------------------------- |
| Yazı tipi               | Sisteminizde yüklü yazı tiplerini ekleyin.      |
| Yazı tipi boyutu        | Metin boyutunu ayarlayın.                        |
| Satır&nbsp;yüksekliği   | Satırlar arasındaki boşluk.                      |
| Satır&nbsp;genişliği    | İçerik alanının maksimum genişliği.              |
| Görünüm                 | Açık veya koyu görünümü seçin.                   |
| Tema                    | Renkleri seçin.                                  |
| Özel CSS                | Sayfaya uygulanan ek CSS.                        |
