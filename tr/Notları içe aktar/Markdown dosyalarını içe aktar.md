Obsidian, notlar için birincil biçim olarak Markdown `.md` dosyalarını kullanır. Bu, Markdown dosyalarını Obsidian'a aktarmayı kolaylaştırır. Birçok uygulama Markdown dosyalarını düzenleyebilir veya bu biçime dışa aktarabilir.

Obsidian kasanıza Markdown dosyaları eklemenin birkaç yolu vardır:

1. Dosyaları veya klasörleri [[Dosya Gezgini]] penceresine sürükleyip bırakın
2. Sistem dosya tarayıcınızı (örneğin Windows Gezgini veya macOS'ta Finder) kullanarak dosyalarınızı doğrudan Obsidian kasa klasörüne taşıyın.

### Diğer Markdown çeşitlerinden dönüştürme

Obsidian, [[Obsidian Flavored Markdown]] kullanır. Bazı uygulamalar biraz farklı Markdown söz dizimi kullanır. Dosyalarınızı Obsidian'ın Markdown söz dizimine dönüştürmek için [[Markdown formatı içe aktarıcı]] kullanabilirsiniz.
