Tällä sivulla luetellaan oletusnäppäinyhdistelmät tekstin navigointiin ja muokkaamiseen Obsidianissa. Nämä yhdistelmät ovat käyttöjärjestelmäsi tai Obsidianin pohjakehyksen tarjoamia, eikä niitä voi mukauttaa Obsidianissa.

Obsidianin komentojen mukautettavat pikanäppäimet löytyvät kohdasta [[Pikanäppäimet]].

## Windows- ja Linux-pikanäppäimet

### Yleiset toiminnot

| Toiminto | Pikanäppäin |
|-|-|
| Kopioi | `Ctrl+C` |
| Leikkaa | `Ctrl+X` |
| Liitä | `Ctrl+V` |
| Liitä ilman muotoilua | `Ctrl+Shift+V` |
| Kumoa | `Ctrl+Z` |
| Tee uudelleen | `Ctrl+Shift+Z` tai `Ctrl+Y` |
| Kopioi kappale | `Ctrl+C` (kun tekstiä ei ole valittuna) |
| Leikkaa kappale | `Ctrl+X` (kun tekstiä ei ole valittuna) |

### Tekstin muokkaus

| Toiminto | Pikanäppäin |
|-|-|
| Lisää uusi rivi | `Enter` |
| Poista edellinen merkki | `Backspace` |
| Poista seuraava merkki | `Delete` |
| Poista edellinen sana | `Ctrl+Backspace` |
| Poista seuraava sana | `Ctrl+Delete` |
| Poista nykyinen rivi | `Ctrl+Shift+K` (kun tekstiä ei ole valittuna) |

### Tekstissä liikkuminen

| Toiminto | Pikanäppäin |
|-|-|
| Siirrä kohdistinta yhdellä merkillä | `Left/→` |
| Siirrä kohdistin edellisen sanan alkuun | `Ctrl+←` |
| Siirrä kohdistin seuraavan sanan loppuun | `Ctrl+→` |
| Siirrä kohdistin nykyisen rivin alkuun | `Home` |
| Siirrä kohdistin nykyisen rivin loppuun | `End` |
| Siirrä kohdistin edelliselle riville | `↑` |
| Siirrä kohdistin seuraavalle riville | `↓` |
| Siirrä kohdistin muistiinpanon alkuun | `Ctrl+Home` |
| Siirrä kohdistin muistiinpanon loppuun | `Ctrl+End` |
| Siirrä kohdistinta sivu ylöspäin | `Page up` |
| Siirrä kohdistinta sivu alaspäin | `Page down` |

### Tekstin valitseminen

| Toiminto | Pikanäppäin |
|-|-|
| Yksinkertaista valintaa | `Escape` |
| Valitse kaikki | `Ctrl+A` |
| Laajenna valintaa yhdellä merkillä | `Shift+Left/→` |
| Laajenna valinta edellisen sanan alkuun | `Ctrl+Shift+←` |
| Laajenna valinta seuraavan sanan loppuun | `Ctrl+Shift+→` |
| Laajenna valinta nykyisen rivin alkuun | `Shift+Home` |
| Laajenna valinta nykyisen rivin loppuun | `Shift+End` |
| Laajenna valinta muistiinpanon alkuun | `Ctrl+Shift+Home` |
| Laajenna valinta muistiinpanon loppuun | `Ctrl+Shift+End` |
| Laajenna valintaa sivu ylöspäin | `Shift+Page up` |
| Laajenna valintaa sivu alaspäin | `Shift+Page down` |

## macOS-pikanäppäimet

### Yleiset toiminnot

| Toiminto | Pikanäppäin |
|-|-|
| Kopioi | `Cmd+C` |
| Leikkaa | `Cmd+X` |
| Liitä | `Cmd+V` |
| Liitä ilman muotoilua | `Cmd+Shift+V` |
| Kumoa | `Cmd+Z` |
| Tee uudelleen | `Cmd+Shift+Z` |
| Kopioi kappale | `Cmd+C` (kun tekstiä ei ole valittuna) |
| Leikkaa kappale | `Cmd+X` (kun tekstiä ei ole valittuna) |

### Tekstin muotoilu

| Toiminto | Pikanäppäin |
| ---- | ---- |
| Lihavointi | `Cmd+B` |
| Kursivointi | `Cmd+I` |

### Tekstin muokkaus

| Toiminto | Pikanäppäin |
|-|-|
| Lisää uusi rivi | `Enter` |
| Poista edellinen merkki | `Backspace` |
| Poista seuraava merkki | `Delete` |
| Poista edellinen sana | `Option+Backspace` |
| Poista seuraava sana | `Option+Delete` |
| Poista nykyisen rivin alkuun | `Cmd+Backspace` |
| Poista nykyisen rivin loppuun | `Cmd+Delete` |
| Poista nykyinen rivi | `Cmd+Shift+K` (kun tekstiä ei ole valittuna) |

### Tekstissä liikkuminen

| Toiminto | Pikanäppäin |
|-|-|
| Siirrä kohdistinta yhdellä merkillä | `Left/→` |
| Siirrä kohdistin edellisen sanan alkuun | `Option+←` |
| Siirrä kohdistin seuraavan sanan loppuun | `Option+→` |
| Siirrä kohdistin nykyisen rivin alkuun | `Cmd+←` |
| Siirrä kohdistin nykyisen rivin loppuun | `Cmd+→` |
| Siirrä kohdistin edelliselle riville | `↑` |
| Siirrä kohdistin seuraavalle riville | `↓` |
| Siirrä kohdistin muistiinpanon alkuun | `Cmd+↑` |
| Siirrä kohdistin muistiinpanon loppuun | `Cmd+↓` |
| Siirrä kohdistinta sivu ylöspäin | `Fn+↑` |
| Siirrä kohdistinta sivu alaspäin | `Fn+↓` |

### Tekstin valitseminen

| Toiminto                                               | Pikanäppäin                |
| ------------------------------------------------------ | -------------------------- |
| Yksinkertaista valintaa                                | `Escape`                   |
| Valitse kaikki                                         | `Cmd+A`                    |
| Laajenna valintaa yhdellä merkillä                     | `Shift+Left/→`   |
| Laajenna valinta edellisen sanan alkuun                | `Option+Shift+←`  |
| Laajenna valinta seuraavan sanan loppuun               | `Option+Shift+→` |
| Laajenna valinta nykyisen rivin alkuun                 | `Cmd+Shift+←`              |
| Laajenna valinta nykyisen rivin loppuun                | `Cmd+Shift+→`    |
| Laajenna valinta muistiinpanon alkuun                  | `Cmd+Shift+↑`       |
| Laajenna valinta muistiinpanon loppuun                 | `Cmd+Shift+↓`     |
| Laajenna valintaa sivu ylöspäin                        | `Ctrl+Shift+↑`      |
| Laajenna valintaa sivu alaspäin                        | `Ctrl+Shift+↓`    |
