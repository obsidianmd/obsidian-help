Catalyst-lisenssi on kertaosto, joka antaa sinulle ennakkopääsyn beta-versioihin ja tukee Obsidianin jatkokehitystä. Voit tukea Obsidiania [ostamalla Catalyst-lisenssin](https://obsidian.md/buy/catalyst).

Etuja ovat:

- Ennakkopääsy [[#Ota käyttöön ennakkojulkaisuversiot|uusiin versioihin]] Obsidianista
- Pääsy suljettuihin [[#Liity Insider-kanavillemme|Insider-kanaviin]].
- Yksinoikeudelliset [[#Hanki foorumimerkki|foorumimerkit]] ja [[#Hanki Discord-merkki|Discord-merkit]].

## Catalyst-tasot

Catalyst-tasoissa on kolme tasoa lahjoituksen suuruuden mukaan: *Insider*, *Supporter* ja *VIP*.

| Edut                                                                                 | Insider | Supporter |   VIP    |
| ------------------------------------------------------------------------------------ | :-----: | :-------: | :------: |
| Ennakkopääsy uusiin versioihin                                                       |    ✓    |     ✓     |    ✓     |
| Pääsy suljettuihin kehityskanaviin, `#insider-desktop` ja `#insider-mobile`          |    ✓    |     ✓     |    ✓     |
| Pääsy suljetulle lounge-kanavalle, `#vip-lounge`                                     |         |     ✓     |    ✓     |
| Insider-merkki                                                                       |    ✓    |           |          |
| Supporter-merkki                                                                     |         |     ✓     |          |
| VIP-merkki                                                                           |         |           |    ✓     |
| **Lahjoitus (USD)**                                                                  | **$25** |  **$50**  | **$100** |


> [!info] Päivitä Catalyst-tasosi
> Jos sinulla on jo Catalyst-lisenssi, voit päivittää korkeammalle Catalyst-tasolle maksamalla nykyisen ja uuden tason välisen hintaeron.

## Catalyst-etujen käyttöönotto

Kun olet ostanut Catalyst-lisenssin, saat käyttöösi useita etuja. Lue täältä, miten ne otetaan käyttöön.

### Hanki foorumimerkki

**Huom:** Jotta voit yhdistää Obsidian-tilisi ja foorumitilisi, sinun on käytettävä samaa sähköpostiosoitetta molemmissa tileissä.

1. Jos et ole vielä liittynyt, liity [foorumillemme](https://forum.obsidian.md/).
2. Siirry [Obsidian-tilisi hallintapaneeliin](https://obsidian.md/account/catalyst).
3. Valitse **Catalyst**-kohdasta **Get forum badge**.

### Hanki Discord-merkki

1. Jos et ole vielä liittynyt, liity [Discord-palvelimellemme](https://discord.gg/veuWUTm).
2. Siirry [Obsidian-tilisi hallintapaneeliin](https://obsidian.md/account/catalyst).
3. Valitse **Catalyst**-kohdasta **Get Discord badge**.

### Liity Insider-kanavillemme

Insider-jäsenet voivat keskustella Obsidian-tiimin ja muiden Insider-jäsenten kanssa suljetuilla Discord-kanavillamme. Pääset Insider-kanaville automaattisesti sen jälkeen, kun olet [[#Hanki Discord-merkki|hankkinut Discord-merkkisi]].
