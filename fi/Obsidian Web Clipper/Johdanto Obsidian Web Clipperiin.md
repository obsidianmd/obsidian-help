Obsidian Web Clipper on ilmainen selainlaajennus, jolla voit korostaa sivuja ja tallentaa verkkosisältöä holviisi.

## Asenna Web Clipper

Hanki Obsidian Web Clipper -laajennus selaimesi virallisesta hakemistosta:

- [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
	- Chromelle, Bravelle, Arcille, Orionille ja muille Chromium-pohjaisille selaimille.
- [Firefox Add-Ons](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
	- Firefoxille ja Firefox Mobilelle.
- [Safari Extensions](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
	- macOS:lle, iOS:lle ja iPadOS:lle.
- [Edge Add-Ons](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
	- Microsoft Edgelle.

## Web Clipperin käyttö

- [[Verkkosivujen leikkaaminen|Verkkosivujen leikkaaminen]]
	- Aloita käyttö ja opi tallentamaan sisältöä verkkosivuilta.
- [[Verkkosivujen korostaminen|Korostustyökalu]]
	- Korosta tärkeitä kohtia ja valitse elementit, jotka haluat tallentaa.
- [[Lukija|Lukija]]
	- Lue verkkosivuja selkeässä, häiriöttömässä näkymässä.
- [[Verkkosivujen tulkinta|Tulkki]]
	- Käytä luonnollisen kielen kehotteita sivun tietojen kaappaamiseen ja muokkaamiseen.
- [[Web Clipperin vianmääritys|Vianmääritys]]
	- Ratkaise Web Clipperin yleisiä ongelmia.
- [[Obsidian Web Clipper/Mallineet|Mallineet]]
	- Luo mukautettuja pohjia tietyille verkkosivustoille.
- [[Muuttujat]]
	- Täytä sivun tiedot automaattisesti pohjaasi.
- [[Suodattimet]]
	- Muokkaa pohjamuuttujien tuotetta.
- [[Logiikka]]
	- Käytä ehtolauseita, silmukoita ja aseta muuttujia pohjissa.

## Yksityisyys

Obsidian Web Clipper tallentaa sisällön paikallisesti Obsidian-holviisi ja noudattaa [tietosuojakäytäntöämme](https://obsidian.md/privacy). Tietojasi ei kerätä, emmekä kerää mitään käyttötilastoja. Koodi on [avointa lähdekoodia](https://github.com/obsidianmd/obsidian-clipper) ja tarkastettavissa.
