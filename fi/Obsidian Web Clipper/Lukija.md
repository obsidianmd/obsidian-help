[[Johdanto Obsidian Web Clipperiin|Web Clipper]] sisältää sisäänrakennetun lukutilan nimeltä **Lukija**, joka poistaa verkkosivuilta häiriötekijät ja esittää vain artikkelin pääsisällön selkeässä, helposti luettavassa muodossa.

## Lukutilan aktivointi

Voit aktivoida Lukijan useilla tavoilla:

- ![[lucide-book-icon.svg#icon]] **Kirjakuvake** laajennuspaneelissa.
- **Pikanäppäin**, muokattavissa laajennuksen asetuksissa.
- **Kontekstivalikko** napsauttamalla hiiren oikealla painikkeella vierailemaasi verkkosivua.

Voit myös asettaa ![[obsidian.svg#icon]] laajennuspainikkeen avaamaan Lukijan yhdellä napsautuksella. Siirry Web Clipperin asetuksissa kohtaan **General** → **Behavior** → **Open behavior** ja valitse **Reader**.

Kaikilla yllä mainituilla tavoilla voit myös poistua Lukijasta ja palauttaa alkuperäisen sivun.

## Miten se toimii

Lukija poimii sivulta pääsisällön. Alkuperäinen sivu korvataan selkeällä lukunäkymällä, joka sisältää:

- Artikkelin otsikon, tekijän, julkaisupäivämäärän ja lähdetunnuksen.
- Artikkelin pääsisällön, kuvat ja muotoilut säilytettynä.
- Sisällysluettelon sivupalkissa artikkeleille, joissa on useita otsikoita.
- Syntaksikorostuksen koodilohkoille.
- Alaviiteponnahdusikkunat, jotka näytetään tekstin seassa napsautettaessa.

## Lukunäkymän mukauttaminen

Mukauta Lukija-kokemustasi laajennuksen asetuksissa. Jotkin vaihtoehdot ovat saatavilla myös Lukijatilassa napsauttamalla ![[lucide-case-sensitive.svg#icon]] -kuvaketta.

| Asetus            | Kuvaus                                        |
| ----------------- | --------------------------------------------- |
| Kirjasin          | Lisää järjestelmääsi asennettuja kirjasimia.  |
| Kirjasinkoko      | Säädä tekstin kokoa.                          |
| Rivin&nbsp;korkeus | Rivien välinen väli.                         |
| Rivin&nbsp;leveys | Sisältöalueen enimmäisleveys.                 |
| Ulkoasu           | Valitse vaalea tai tumma ulkoasu.             |
| Teema             | Valitse värit.                                |
| Mukautettu CSS    | Sivulle sovellettava lisä-CSS.                |
