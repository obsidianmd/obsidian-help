Jos osaat toista kieltä, mieluiten äidinkielenäsi, voit auttaa kääntämään Obsidianin käyttöliittymää tai dokumentaatiota.

Auttaaksesi käyttöliittymän kääntämisessä, lähetä pull request [käännösrepositoriin](https://github.com/obsidianmd/obsidian-translations).

Auttaaksesi dokumentaation parantamisessa, lähetä pull request [obsidian-help](https://github.com/obsidianmd/obsidian-help)-repositoriin. Varmista, että noudatat [[Tyyliopas#Käännökset|käännösten tyyliopasta]] tehdessäsi muutoksia.
