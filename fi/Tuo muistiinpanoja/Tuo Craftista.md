Obsidian mahdollistaa muistiinpanojesi siirtämisen Craftista. Tämä muuntaa Craft-tietosi kestäviksi Markdown-tiedostoiksi, joita voit käyttää Obsidianin ja monien muiden sovellusten kanssa.

## Vie tietosi Craftista

Avaa Craft-sovellus tietokoneellasi:

1. Siirry "Organize"- tai "All Documents" -näkymään
2. Napsauta kolmen pisteen valikkoa
3. Valitse Export-vaihtoehto
4. Valitse Markdown.

Riippuen muistiinpanojesi määrästä Craftissa, tämä prosessi voi kestää jonkin aikaa.

## Tuo Craft-tietosi Obsidianiin

Tästä eteenpäin voit seurata [[Tuo Markdown-tiedostoja]] -ohjeita.

## Vianmääritys

Craft Docs käyttää epästandardia syntaksia tietyille linkeille. Löydät resursseja niiden siirtämiseen [Importer GitHub -arkistosta](https://github.com/obsidianmd/obsidian-importer/issues/27).
