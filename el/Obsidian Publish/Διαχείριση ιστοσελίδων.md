Αυτή η σελίδα εξηγεί πώς να διαχειρίζεστε ιστοσελίδες [[Εισαγωγή στο Obsidian Publish|Obsidian Publish]].

Μια ιστοσελίδα είναι μια συλλογή σημειώσεων που φιλοξενείται από το Obsidian Publish και είναι διαθέσιμη στο διαδίκτυο, είτε μέσω μιας διεύθυνσης Obsidian Publish είτε μέσω ενός [[Προσαρμοσμένα domains|προσαρμοσμένου domain]].

## Δημιουργία νέας ιστοσελίδας

> [!note] Ο αριθμός των ιστοσελίδων που μπορείτε να έχετε ταυτόχρονα καθορίζεται από τη συνδρομή σας στο Obsidian Publish. Πριν δημιουργήσετε μια νέα ιστοσελίδα, βεβαιωθείτε ότι η συνδρομή σας το επιτρέπει.

1. Στην κορδέλα, στα αριστερά του παραθύρου της εφαρμογής, κάντε κλικ στο **Δημοσίευση αλλαγών** ![[lucide-send.svg#icon]].
2. Αν έχετε ήδη μια υπάρχουσα ιστοσελίδα, κάντε κλικ στο **Αλλαγή ιστοσελίδας** ![[lucide-repeat.svg#icon]].
3. Στο **ID ιστοσελίδας**, εισαγάγετε τη διαδρομή που θέλετε για την ιστοσελίδα σας. Για παράδειγμα, μια ιστοσελίδα με ID `my-amazing-site` είναι διαθέσιμη στο publish.obsidian.md/my-amazing-site.
4. Κάντε κλικ στο **Δημιουργία**.

## Διαγραφή υπάρχουσας ιστοσελίδας

> [!note] Οι σημειώσεις παραμένουν στο θησαυροφυλάκιό σας ακόμα και μετά τη διαγραφή μιας ιστοσελίδας.

1. Στην κορδέλα, στα αριστερά του παραθύρου της εφαρμογής, κάντε κλικ στο **Δημοσίευση αλλαγών** ![[lucide-send.svg#icon]].
2. Κάντε κλικ στο **Αλλαγή ιστοσελίδας** ![[lucide-repeat.svg#icon]].
3. Κάντε κλικ στο **Διαγραφή ιστοσελίδας** ![[lucide-x.svg#icon]] στα δεξιά της ιστοσελίδας που θέλετε να διαγράψετε.
4. Κάντε κλικ στο **Διαγραφή** για επιβεβαίωση.

## Εναλλαγή μεταξύ ιστοσελίδων

1. Στην κορδέλα, στα αριστερά του παραθύρου της εφαρμογής, κάντε κλικ στο **Δημοσίευση αλλαγών** ![[lucide-send.svg#icon]].
2. Κάντε κλικ στο **Αλλαγή ιστοσελίδας** ![[lucide-repeat.svg#icon]].
3. Κάντε κλικ στο **Επιλογή** στα δεξιά της ιστοσελίδας στην οποία θέλετε να μεταβείτε.

## Αλλαγή του ID ιστοσελίδας

1. Στην κορδέλα, στα αριστερά του παραθύρου της εφαρμογής, κάντε κλικ στο **Δημοσίευση αλλαγών** ![[lucide-send.svg#icon]].
2. Κάντε κλικ στο **Αλλαγή ιστοσελίδας** ![[lucide-repeat.svg#icon]].
3. Κάντε κλικ στο **Επεξεργασία ID ιστοσελίδας** ![[lucide-edit-3.svg#icon]] στα δεξιά της ιστοσελίδας που θέλετε να επεξεργαστείτε.
4. Στο **ID ιστοσελίδας**, εισαγάγετε το νέο ID για την ιστοσελίδα σας.
5. Κάντε κλικ στο **Αλλαγή**.

## Προβολή ρυθμίσεων ιστοσελίδας

1. Στην κορδέλα, στα αριστερά του παραθύρου της εφαρμογής, κάντε κλικ στο **Δημοσίευση αλλαγών** ![[lucide-send.svg#icon]].
2. Στο παράθυρο διαλόγου **Δημοσίευση αλλαγών**, κάντε κλικ στο **Αλλάξτε ρυθμίσεις ιστοσελίδας** ![[lucide-cog.svg#icon]].

## Ρυθμίσεις ιστοσελίδας

### Γενικά

| Επιλογή                                          | Τύπος    | Περιγραφή                                                                                                                           |
| ------------------------------------------------- | -------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| Όνομα ιστοσελίδας                                 | Εισαγωγή | Το δημόσιο όνομα και τίτλος της ιστοσελίδας σας στο Obsidian Publish.                                                               |
| Αρχείο αρχικής σελίδας                            | Εισαγωγή | Η τοποθεσία του αρχείου Markdown που θέλετε να λειτουργεί ως σελίδα προορισμού.                                                     |
| Λογότυπο                                          | Εισαγωγή | Η εικόνα που θέλετε να λειτουργεί ως banner της ιστοσελίδας σας. Η εικόνα πρέπει να [[Δημοσίευση του περιεχομένου σας#Δημοσίευση σημειώσεων\|είναι δημοσιευμένη]]. |
| Συνεργασία ιστοσελίδας                            | Κουμπί   | Δηλώστε άλλους χρήστες στους οποίους θέλετε να δώσετε πρόσβαση για επεξεργασία των δημοσιευμένων σημειώσεών σας. Οι χρήστες πρέπει να έχουν λογαριασμό Obsidian. |
| Προσαρμοσμένο domain                              | Κουμπί   | [[Προσαρμοσμένα domains]]                                                                                                           |
| Εμποδισμός ευρετηρίασης από μηχανές αναζήτησης    | Εναλλαγή | Αποτρέψτε τις μηχανές αναζήτησης από τη σάρωση της ιστοσελίδας σας προσθέτοντας ένα αρχείο `robots.txt`.                           |

### Εμφάνιση

| Επιλογή                      | Τύπος         | Περιγραφή                                                                                               |
|------------------------------|---------------|---------------------------------------------------------------------------------------------------------|
| Θέμα                         | Αναπτυσσόμενο | Επιλέξτε πώς εμφανίζεται το θέμα σας στην ιστοσελίδα: **Φωτεινό**, **Σκοτεινό** ή **Προσαρμογή στο σύστημα**. |
| Αλλαγή φωτειού/σκοτεινού    | Εναλλαγή      | Επιτρέψτε στους χρήστες να εναλλάσσουν τη λειτουργία **Φωτεινό**/**Σκοτεινό** με ένα κουμπί εναλλαγής στην ιστοσελίδα σας. |

## Εμπειρία ανάγνωσης

| Επιλογή                          | Τύπος    | Περιγραφή                                                                                                          |
|----------------------------------|----------|---------------------------------------------------------------------------------------------------------------------|
| Προεπισκόπιση κάτω από κέρσορα  | Εναλλαγή | Ενεργοποίηση ή απενεργοποίηση της προεπισκόπισης σελίδας κατά την αιώρηση πάνω σε ενεργό εσωτερικό σύνδεσμο.       |
| Απόκρυψη τίτλου σελίδας         | Εναλλαγή | Ενεργοποίηση ή απενεργοποίηση της εμφάνισης του τίτλου στο έγγραφο μιας δημοσιευμένης σημείωσης.                   |
| Μήκος ευανάγνωστης γραμμής      | Εναλλαγή | Ενεργοποίηση ή απενεργοποίηση του μήκους ευανάγνωστης γραμμής στην ιστοσελίδα σας.                                 |
| Αυστηρή αλλαγή γραμμής          | Εναλλαγή | Ενεργοποίηση ή απενεργοποίηση της εμφάνισης μεμονωμένων αλλαγών γραμμής στην ιστοσελίδα σας.                       |
| Χρήση συρόμενων παραθύρων        | Εναλλαγή | Ενεργοποίηση ή απενεργοποίηση [[Καρτέλες#Στοίβαξη ομάδων καρτελών\|Στοιβαγμένων καρτελών]] στην ιστοσελίδα σας.   |
