## Προαπαιτούμενα

Για να χρησιμοποιήσετε αναλυτικά στοιχεία στην ιστοσελίδα σας στο [[Εισαγωγή στο Obsidian Publish|Obsidian Publish]], χρειάζεστε ένα [[Προσαρμοσμένα domains|Προσαρμοσμένο domain]].

> [!important] Σημαντικό
> Πριν ενεργοποιήσετε τα αναλυτικά στοιχεία, βεβαιωθείτε ότι οι τοπικοί νόμοι και κανονισμοί σας επιτρέπουν να παρακολουθείτε τους επισκέπτες σας. Ανάλογα με τα εργαλεία που χρησιμοποιείτε, μπορεί να χρειαστεί να προσθέσετε ένα banner συγκατάθεσης στην ιστοσελίδα σας.

## Google Analytics

Για να ενεργοποιήσετε το Google Analytics για την ιστοσελίδα σας στο Obsidian Publish:

1. Στην κορδέλα, στα αριστερά του παραθύρου της εφαρμογής, κάντε κλικ στο **Δημοσίευση αλλαγών** ![[lucide-send.svg#icon]].
2. Στο παράθυρο διαλόγου **Δημοσίευση αλλαγών**, κάντε κλικ στο **Αλλάξτε ρυθμίσεις ιστοσελίδας** ![[lucide-cog.svg#icon]].
3. Στο **Google Analytics κωδικός εντοπισμού**, εισαγάγετε τον κωδικό εντοπισμού σας.

Για να χρησιμοποιήσετε το Google Tag Manager αντί του Google Analytics, χρησιμοποιήστε προσαρμοσμένο JavaScript για να προσθέσετε τα δικά σας scripts.

## Plausible, Fathom και άλλες υπηρεσίες αναλυτικών στοιχείων

Μπορείτε να προσθέσετε τις περισσότερες υπηρεσίες αναλυτικών στοιχείων στην ιστοσελίδα σας μέσω του αρχείου [[Προσαρμογή της ιστοσελίδας σας|publish.js]]. Βεβαιωθείτε ότι έχετε αντικαταστήσει το `yourdomain.com` με το domain σας και το `src` του script με το script από τον πάροχο αναλυτικών στοιχείων σας.

Ακολουθεί ένα παράδειγμα χρησιμοποιώντας το [Plausible Analytics](https://plausible.io/)

```js
var analyticsScript = document.createElement('script');
analyticsScript.defer = true;
analyticsScript.setAttribute('data-domain', 'yourdomain.com');
analyticsScript.src = 'https://plausible.io/js/plausible.js';
document.head.appendChild(analyticsScript);
```

 Η ίδια προσέγγιση μπορεί να χρησιμοποιηθεί για το [Fathom Analytics](https://usefathom.com/). Σημειώστε την αλλαγή από `data-domain` σε `data-site` — διαφορετικοί πάροχοι αναλυτικών στοιχείων μπορεί να έχουν διαφορετική μορφή για τον τρόπο εισαγωγής του script.

```javascript
var fathom = analyticsScript.createElement('script');
analyticsScript.defer = true;
analyticsScript.setAttribute('data-site', 'yourdomain.com');
analyticsScript.src = 'https://cdn.usefathom.com/script.js';
document.head.appendChild(analyticsScript);
```

## Αντιμετώπιση προβλημάτων

Για να επαληθεύσετε ότι η ιστοσελίδα σας χρησιμοποιεί την υπηρεσία αναλυτικών στοιχείων σας, απενεργοποιήστε τυχόν επεκτάσεις φυλλομετρητή που μπλοκάρουν διαφημίσεις, όπως το uBlock Origin, οι οποίες μπορεί να εμποδίζουν την εκτέλεση του script παρακολούθησης.
