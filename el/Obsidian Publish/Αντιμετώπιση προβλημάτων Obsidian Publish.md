Αυτή η σελίδα παραθέτει συνηθισμένα ζητήματα που μπορεί να αντιμετωπίσετε με το [[Εισαγωγή στο Obsidian Publish|Obsidian Publish]] και πώς να τα αντιμετωπίσετε.

Βεβαιωθείτε ότι έχετε ελέγξει πρώτα τα [[Αρχεία πολυμέσων]] και τους [[Περιορισμοί Publish|Περιορισμούς Publish]].

## Γενικά

### Δημοσίευση σημειώσεων

**Λαμβάνω σφάλμα hash κατά την προσπάθεια δημοσίευσης μιας σημείωσης.**

Χρησιμοποιείτε κάποιο [[Πρόσθετα κοινότητας|πρόσθετο κοινότητας]] που τροποποιεί τον χρόνο τροποποίησης του αρχείου κατά την ενημέρωση; Αν ναι, αυτό το πρόσθετο μπορεί να δημιουργεί σύγκρουση με το Publish. Παρακαλούμε υποβάλετε αναφορά σφάλματος στον προγραμματιστή του πρόσθετου για την επίλυση του ζητήματος.

**Λαμβάνω ένα ασυνήθιστο σφάλμα δικτύου και έχω μια πολύ μεγάλη ιστοσελίδα Publish.**

Είναι πιθανό να χρειαστεί να εξετάσουμε τη βάση δεδομένων σας. Παρακαλούμε [[Βοήθεια και υποστήριξη#Επικοινωνία με την υποστήριξη Obsidian|επικοινωνήστε με την υποστήριξη Obsidian]] για βοήθεια.

## CSS και θέματα

**Το CSS στον [[Φάκελος ρυθμίσεων|φάκελο ρυθμίσεών]] μου δεν λειτουργεί στο Publish. Γιατί;**

Το Publish δεν διαβάζει από τον φάκελο ρυθμίσεων. Αντί αυτού, θα πρέπει να δημιουργήσετε ένα αρχείο `publish.css` στον κατάλογο ανώτατου επιπέδου του θησαυροφυλακίου δημοσίευσής σας. Μπορείτε να μάθετε περισσότερα σχετικά στο [[Προσαρμογή της ιστοσελίδας σας]].

**Το CSS μου δεν εμφανίζεται το ίδιο στο Publish όπως στην εφαρμογή. Γιατί;**

Το CSS του Obsidian Publish δεν είναι ακριβώς το ίδιο με αυτό στην εφαρμογή. Συνιστούμε να [αναπτύξετε οποιοδήποτε CSS και θέματα](https://docs.obsidian.md/Themes/Obsidian+Publish+themes/About+Obsidian+Publish+themes) ειδικά για το Publish από την αρχή.

Σε _γενικές γραμμές_, ό,τι λειτουργεί για την [[Προβολές και λειτουργία επεξεργασίας#Προβολή ανάγνωσης|Προβολή ανάγνωσης]] έχει μεγάλη πιθανότητα να λειτουργεί στο Publish.
