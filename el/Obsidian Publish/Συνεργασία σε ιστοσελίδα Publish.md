Μάθετε πώς να συνεργάζεστε στην ιστοσελίδα σας [[Εισαγωγή στο Obsidian Publish|Obsidian Publish]] με άλλους χρήστες του Obsidian. Προσθέτοντας φίλους και συναδέλφους ως συνεργάτες, μπορούν να δημοσιεύουν αλλαγές στην ιστοσελίδα σας.

Μόνο ο ιδιοκτήτης της ιστοσελίδας χρειάζεται ενεργή συνδρομή στο Obsidian Publish. Οι συνεργάτες χρειάζονται μόνο έναν [λογαριασμό Obsidian](https://obsidian.md/account).

> [!warning] Πριν δημοσιεύσετε αλλαγές σε μια κοινόχρηστη ιστοσελίδα, βεβαιωθείτε ότι έχετε [[#Συγχρονισμός αλλαγών μεταξύ συνεργατών|συγχρονίσει τις αλλαγές μεταξύ συνεργατών]]. Διαφορετικά, κινδυνεύετε να αντικαταστήσετε αλλαγές άλλων συνεργατών.

## Προσθήκη συνεργάτη σε ιστοσελίδα

1. Στην [[Κορδέλα|Κορδέλα]], επιλέξτε **Δημοσίευση αλλαγών** ![[lucide-send.svg#icon]] ή ανοίξτε την [[Παλέτα εντολών]] και πληκτρολογήστε **Publish: Publish changes...**
2. Στο παράθυρο διαλόγου **Δημοσίευση αλλαγών**, κάντε κλικ στο **Αλλάξτε ρυθμίσεις ιστοσελίδας** ![[lucide-cog.svg#icon]].
3. Δίπλα στο **Συνεργασία ιστοσελίδας**, επιλέξτε **Διαχείριση**.
4. Στο **Προσκαλέστε χρήστη**, εισάγετε το ηλεκτρονικό ταχυδρομείο του συνεργάτη.
5. Επιλέξτε **Προσθέστε**.

## Αφαίρεση συνεργάτη από ιστοσελίδα

1. Στην [[Κορδέλα|Κορδέλα]], επιλέξτε **Δημοσίευση αλλαγών** ![[lucide-send.svg#icon]] ή ανοίξτε την [[Παλέτα εντολών]] και πληκτρολογήστε **Publish: Publish changes...**
2. Στο παράθυρο διαλόγου **Δημοσίευση αλλαγών**, κάντε κλικ στο **Αλλάξτε ρυθμίσεις ιστοσελίδας** ![[lucide-cog.svg#icon]].
3. Δίπλα στο **Συνεργασία ιστοσελίδας**, επιλέξτε **Διαχείριση**.
4. Δίπλα στον συνεργάτη που θέλετε να αφαιρέσετε, επιλέξτε **Αφαίρεση χρήστη** ![[lucide-x.svg#icon]].

## Συγχρονισμός αλλαγών μεταξύ συνεργατών

Το Obsidian Publish δεν συγχρονίζει αυτόματα τις δημοσιευμένες αλλαγές μεταξύ τοπικών θησαυροφυλακίων. Αντιθέτως, οι συνεργάτες πρέπει να συγχρονίζουν χειροκίνητα τις αλλαγές από άλλους συνεργάτες.

Για να ενημερώσετε μια τοπική σημείωση με αλλαγές από τη ζωντανή ιστοσελίδα:

1. Στην [[Κορδέλα|Κορδέλα]], επιλέξτε **Δημοσίευση αλλαγών** ![[lucide-send.svg#icon]] ή ανοίξτε την [[Παλέτα εντολών]] και πληκτρολογήστε **Publish: Publish changes...**
2. Κάντε δεξί κλικ ή πατήστε παρατεταμένα στην αλλαγή που θέλετε να συγχρονίσετε και μετά επιλέξτε **Χρήση ζωντανής έκδοσης**. **Αυτό θα αντικαταστήσει τη σημείωση στο τοπικό σας θησαυροφυλάκιο.**

> [!tip] Συνιστούμε να χρησιμοποιείτε ένα άλλο εργαλείο για τον συγχρονισμό αλλαγών μεταξύ θησαυροφυλακίων, όπως το [[Εισαγωγή στο Obsidian Sync|Obsidian Sync]] ή το [git](https://git-scm.com/).

## Δικαιώματα

Ο παρακάτω πίνακας παραθέτει τα διαθέσιμα δικαιώματα ιστοσελίδας για ιδιοκτήτες και συνεργάτες:

| Ενέργεια                                     | Συνεργάτης | Ιδιοκτήτης |
|----------------------------------------------|:----------:|:-----------:|
| Δημοσίευση νέων σελίδων                      | ✓          | ✓           |
| Δημοσίευση αλλαγών σε δημοσιευμένες σελίδες | ✓          | ✓           |
| Κατάργηση δημοσίευσης σελίδων                | ✓          | ✓           |
| Τροποποίηση ρυθμίσεων ιστοσελίδας            |            | ✓           |
| Διαχείριση δικαιωμάτων                        |            | ✓           |
