Μπορείτε να μετονομάσετε τη διεύθυνση URL των σημειώσεών σας, χρησιμοποιώντας _μόνιμους συνδέσμους_.

Για παράδειγμα, μπορείτε να μετατρέψετε αυτό:

```
https://publish.obsidian.md/username/Company/About+us
```

Σε αυτό:

```
https://publish.obsidian.md/username/about
```

Για να δημιουργήσετε έναν μόνιμο σύνδεσμο για μια σημείωση, προσθέστε την ιδιότητα `permalink` στις [[Ιδιότητες]] σας.

```yaml
---
permalink: about
---
```

Αν κάποιος επισκεφτεί μια σημείωση χρησιμοποιώντας την αρχική διεύθυνση URL, θα ανακατευθυνθεί αυτόματα στον μόνιμο σύνδεσμο.

## Ανακατεύθυνση παλιών σημειώσεων

Η μετονομασία και η αφαίρεση σημειώσεων είναι ένα φυσικό μέρος της συντήρησης ενός ζωντανού θησαυροφυλακίου. Ενώ το Obsidian ενημερώνει αυτόματα τους συνδέσμους όταν μετακινείτε μια σημείωση μέσα στο τοπικό σας θησαυροφυλάκιο, άλλοι ιστότοποι μπορεί να συνδέουν ακόμα στις παλιές σημειώσεις σας στη δημοσιευμένη ιστοσελίδα [[Εισαγωγή στο Obsidian Publish|Obsidian Publish]]. Μπορείτε να ανακατευθύνετε τους αναγνώστες από μια σημείωση σε μια άλλη.

Φανταστείτε ότι θέλετε να μετακινήσετε μια σημείωση από έναν φάκελο σε έναν άλλο:

- **Οδηγοί**
  - ~~Making friends.md~~ (αφαιρέθηκε)
- **Εκπαιδευτικά**
  - *How to make friends.md* (προστέθηκε)

Αφού μετακινήσετε τη σημείωση, το Obsidian ενημερώνει αυτόματα όλους τους συνδέσμους μέσα στο θησαυροφυλάκιο. Ωστόσο, αν δημοσιεύσετε την αλλαγή στην ιστοσελίδα Publish σας, οποιοιδήποτε σύνδεσμοι προς `/Guides/Making+friends` θα καταλήξουν σε σφάλμα 404.

Για να ανακατευθύνετε τους αναγνώστες από `/Guides/Making+friends` στο `/Tutorials/How+to+make+friends`, πρέπει να προσθέσετε ένα [[Ψευδώνυμα|ψευδώνυμο]] στο `How to make friends.md`, τη σημείωση στην οποία θέλετε να γίνει η ανακατεύθυνση.

```md
---
alias: Guides/Making friends
---

# How to make friends
```

> [!important] Σημαντικό
> Βεβαιωθείτε ότι συμπεριλαμβάνετε την πλήρη διαδρομή προς την παλιά σημείωση στο ψευδώνυμο. Ενώ η χρήση μόνο του ονόματος της σημείωσης ως ψευδώνυμο λειτουργεί στο τοπικό σας θησαυροφυλάκιο, το Publish χρειάζεται την πλήρη διαδρομή προς τη σημείωση για να μπορέσει να ανακατευθύνει σε αυτήν.

Μπορείτε να ανακατευθύνετε πολλαπλές σημειώσεις προσθέτοντας ένα ψευδώνυμο για καθεμία.

```md
---
aliases: 
  - Guides/Making friends
  - Developing friendships
---

# How to make friends
```
