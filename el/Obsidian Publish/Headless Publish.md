Το [[Εισαγωγή στο Obsidian Publish|Obsidian Publish]] προσφέρει έναν headless client για δημοσίευση θησαυροφυλακίων χωρίς τη χρήση της εφαρμογής επιφάνειας εργασίας. Χρήσιμο για CI pipelines και αυτοματοποιημένες ροές εργασίας. Δημοσιεύστε τις τελευταίες αλλαγές σας σε προγραμματισμένο χρονοδιάγραμμα ή ως μέρος ενός build pipeline.

Εγκαταστήστε το [[Obsidian Headless]] **(ανοιχτή beta)** για να αλληλεπιδράσετε με το [[Εισαγωγή στο Obsidian Publish|Obsidian Publish]] από τη γραμμή εντολών χωρίς την εφαρμογή επιφάνειας εργασίας του Obsidian.

## Γρήγορη εκκίνηση

Εγκαταστήστε το [[Obsidian Headless|Obsidian Headless]] **(ανοιχτή beta)**:

```shell
npm install -g obsidian-headless
```

Πρέπει να έχετε μια ενεργή [[Εισαγωγή στο Obsidian Publish|συνδρομή Obsidian Publish]].

```shell
# Σύνδεση
ob login

# Εμφάνιση των ιστοσελίδων publish σας
ob publish-list-sites

# Σύνδεση τοπικού θησαυροφυλακίου με ιστοσελίδα publish
cd ~/vaults/my-vault
ob publish-setup --site "my-site"

# Προεπισκόπηση αλλαγών χωρίς δημοσίευση
ob publish --dry-run

# Δημοσίευση αλλαγών
ob publish
```

## Εντολές

### `ob publish-list-sites`

Εμφάνιση όλων των ιστοσελίδων Publish που είναι διαθέσιμες στον λογαριασμό σας.

### `ob publish-create-site`

Δημιουργία νέας ιστοσελίδας Publish.

```
ob publish-create-site --slug <slug>
```

| Επιλογή | Περιγραφή |
| --- | --- |
| `--slug` | Αναγνωριστικό URL για την ιστοσελίδα σας (π.χ. `my-notes` δημιουργεί `publish.obsidian.md/my-notes`) |

### `ob publish-setup`

Σύνδεση τοπικού θησαυροφυλακίου με ιστοσελίδα Publish.

```
ob publish-setup [--site <id-or-slug>] [--path <local-path>]
```

| Επιλογή | Περιγραφή |
| --- | --- |
| `--site` | Αναγνωριστικό ή slug ιστοσελίδας |
| `--path` | Διαδρομή τοπικού θησαυροφυλακίου (προεπιλογή: τρέχων κατάλογος) |

### `ob publish`

Δημοσίευση αλλαγών θησαυροφυλακίου στην ιστοσελίδα σας. Από προεπιλογή, συμπεριλαμβάνονται μόνο αρχεία με `publish: true` στα μεταδεδομένα αρχής τους.

```
ob publish [--path <local-path>] [--all] [--dry-run] [--yes]
```

| Επιλογή | Περιγραφή |
| --- | --- |
| `--path` | Διαδρομή τοπικού θησαυροφυλακίου (προεπιλογή: τρέχων κατάλογος) |
| `--all` | Συμπερίληψη όλων των αρχείων, όχι μόνο αυτών με σημαία δημοσίευσης |
| `--dry-run` | Εμφάνιση αλλαγών χωρίς δημοσίευση |
| `--yes` | Δημοσίευση χωρίς ερώτηση επιβεβαίωσης |

### `ob publish-config`

Προβολή ή αλλαγή ρυθμίσεων συμπερίληψης/εξαίρεσης φακέλων για ένα θησαυροφυλάκιο. Εκτελέστε χωρίς επιλογές για να εμφανιστεί η τρέχουσα ρύθμιση.

```
ob publish-config [--path <local-path>] [options]
```

| Επιλογή | Περιγραφή |
| --- | --- |
| `--path` | Διαδρομή τοπικού θησαυροφυλακίου (προεπιλογή: τρέχων κατάλογος) |
| `--includes` | Φάκελοι προς συμπερίληψη, χωρισμένοι με κόμμα (κενό κείμενο για εκκαθάριση) |
| `--excludes` | Φάκελοι προς εξαίρεση, χωρισμένοι με κόμμα (κενό κείμενο για εκκαθάριση) |

### `ob publish-site-options`

Προβολή ή ενημέρωση ρυθμίσεων εμφάνισης και πλοήγησης σε επίπεδο ιστοσελίδας. Εκτελέστε χωρίς επιλογές για να εμφανιστούν οι τρέχουσες ρυθμίσεις.

```
ob publish-site-options [--path <local-path>] [options]
```

| Επιλογή | Περιγραφή |
| --- | --- |
| `--path` | Διαδρομή τοπικού θησαυροφυλακίου (προεπιλογή: τρέχων κατάλογος) |
| `--site-name` | Όνομα ιστοσελίδας |
| `--index-file` | Διαδρομή αρχείου αρχικής σελίδας |
| `--logo` | Διαδρομή αρχείου λογοτύπου (κενό κείμενο για εκκαθάριση) |
| `--show-navigation` | Προβολή πλαϊνής μπάρας πλοήγησης |
| `--show-graph` | Προβολή γράφου |
| `--show-outline` | Προβολή πίνακα περιεχομένων |
| `--show-search` | Προβολή αναζήτησης |
| `--show-backlinks` | Προβολή backlinks |
| `--show-hover-preview` | Προβολή προεπισκόπησης αιώρησης |
| `--show-theme-toggle` | Προβολή εναλλαγής θέματος |
| `--default-theme` | Προεπιλεγμένο θέμα: `light` ή `dark` |
| `--readable-line-length` | Μήκος ευανάγνωστης γραμμής |
| `--strict-line-breaks` | Αυστηρή αλλαγή γραμμής |
| `--hide-title` | Απόκρυψη τίτλου στο έγγραφο |
| `--sliding-window` | Λειτουργία συρόμενου παραθύρου |
| `--nav-order` | Σειρά πλοήγησης, διαδρομές χωρισμένες με κόμμα (κενό κείμενο για εκκαθάριση) |
| `--nav-hidden` | Κρυφά στοιχεία πλοήγησης, διαδρομές χωρισμένες με κόμμα (κενό κείμενο για εκκαθάριση) |

### `ob publish-unlink`

Αποσύνδεση θησαυροφυλακίου από ιστοσελίδα Publish.

```
ob publish-unlink [--path <local-path>]
```
