Οι ιστοσελίδες [[Εισαγωγή στο Obsidian Publish|Obsidian Publish]] βελτιστοποιούνται αυτόματα για μηχανές αναζήτησης και [[Προεπισκοπήσεις συνδέσμων κοινωνικών δικτύων|κάρτες κοινωνικής κοινοποίησης]]. Μπορείτε επίσης να προσαρμόσετε τα μεταδεδομένα σελίδας με περιγραφές, μόνιμους συνδέσμους και εικόνες.

Σημειώστε ότι μπορείτε να αποτρέψετε την ευρετηρίαση από μηχανές αναζήτησης, στις [[Διαχείριση ιστοσελίδων#Ρυθμίσεις ιστοσελίδας|ρυθμίσεις ιστοσελίδας]] σας.

### Google Search Console

Για να παρακολουθείτε το SEO με την Google, εγγράψτε την ιστοσελίδα σας στο [Google Search Console](https://search.google.com/search-console). Μπορείτε επίσης να ακολουθήσετε τα προτεινόμενα βήματά μας για να [[Αναλυτικά στοιχεία|ρυθμίσετε το Google Analytics]].

### Χάρτης ιστοτόπου

Ο χάρτης ιστοτόπου για την ιστοσελίδα Publish σας είναι διαθέσιμος στη διεύθυνση `/sitemap.xml`, για παράδειγμα:
https://help.obsidian.md/sitemap.xml

Ο χάρτης ιστοτόπου είναι επίσης διαθέσιμος ως τροφοδοσία RSS στη διεύθυνση `/rss.xml`:
https://help.obsidian.md/rss.xml

### Μόνιμοι σύνδεσμοι

Σκεφτείτε να χρησιμοποιήσετε [[Μόνιμοι σύνδεσμοι|Μόνιμους συνδέσμους]] για να ορίσετε μόνιμα URL στην ιστοσελίδα σας.

### Μεταδεδομένα

Τα μεταδεδομένα σελίδας μπορούν να προσαρμοστούν χρησιμοποιώντας [[Ιδιότητες#Ιδιότητες για το Obsidian Publish|Ιδιότητες]], κάτι που μπορεί να είναι χρήσιμο για τις [[Προεπισκοπήσεις συνδέσμων κοινωνικών δικτύων|κάρτες κοινωνικής κοινοποίησης]].

| Ιδιότητα      | Περιγραφή                                                                                                       |
| ------------- | ----------------------------------------------------------------------------------------------------------------- |
| `publish`     | Δείτε [[Δημοσίευση του περιεχομένου σας#Αυτόματη επιλογή δεδομένων για δημοσίευση\|Αυτόματη επιλογή δεδομένων για δημοσίευση]]. |
| `permalink`   | Δείτε [[Μόνιμοι σύνδεσμοι\|Μόνιμοι σύνδεσμοι]].                                                                                   |
| `description` | Δείτε [[Προεπισκοπήσεις συνδέσμων κοινωνικών δικτύων#Περιγραφή\|Περιγραφή]].                                                      |
| `image`       | Δείτε [[Προεπισκοπήσεις συνδέσμων κοινωνικών δικτύων#Εικόνα\|Εικόνα]].                                                                  |
| `cover`       | Δείτε [[Προεπισκοπήσεις συνδέσμων κοινωνικών δικτύων#Εικόνα\|Εικόνα]].                                                                  |
