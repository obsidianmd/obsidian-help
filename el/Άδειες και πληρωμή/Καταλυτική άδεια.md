Η άδεια Catalyst είναι μια εφάπαξ αγορά που σας παρέχει πρώιμη πρόσβαση σε εκδόσεις beta και βοηθά στη συνεχιζόμενη ανάπτυξη του Obsidian. Μπορείτε να υποστηρίξετε το Obsidian [αγοράζοντας μια άδεια Catalyst](https://obsidian.md/buy/catalyst).

Τα οφέλη περιλαμβάνουν:

- Πρώιμη πρόσβαση σε [[#Ενεργοποίηση εκδόσεων πρώιμης πρόσβασης|νέες εκδόσεις]] του Obsidian
- Πρόσβαση σε αποκλειστικά [[#Συμμετοχή στα κανάλια Insider|κανάλια Insider]].
- Αποκλειστικά [[#Λήψη σήματος φόρουμ|σήματα φόρουμ]] και [[#Λήψη σήματος Discord|σήματα Discord]].

## Βαθμίδες Catalyst

Υπάρχουν τρεις βαθμίδες Catalyst βάσει του ποσού που συνεισφέρετε: *Insider*, *Supporter* και *VIP*.

| Οφέλη                                                                              | Insider | Supporter |   VIP    |
| ---------------------------------------------------------------------------------- | :-----: | :-------: | :------: |
| Πρώιμη πρόσβαση σε νέες εκδόσεις                                                  |    ✓    |     ✓     |    ✓     |
| Πρόσβαση σε αποκλειστικά κανάλια ανάπτυξης, `#insider-desktop` και `#insider-mobile` |    ✓    |     ✓     |    ✓     |
| Πρόσβαση σε αποκλειστικό κανάλι σαλονιού, `#vip-lounge`                           |         |     ✓     |    ✓     |
| Σήμα Insider                                                                       |    ✓    |           |          |
| Σήμα Supporter                                                                     |         |     ✓     |          |
| Σήμα VIP                                                                           |         |           |    ✓     |
| **Συνεισφορά (USD)**                                                               | **$25** |  **$50**  | **$100** |


> [!info] Αναβάθμιση της βαθμίδας Catalyst
> Εάν διαθέτετε ήδη άδεια Catalyst, μπορείτε να αναβαθμίσετε σε υψηλότερη βαθμίδα Catalyst πληρώνοντας τη διαφορά τιμής μεταξύ της τρέχουσας και της νέας βαθμίδας.

## Πρόσβαση στα οφέλη Catalyst

Μόλις αγοράσετε μια άδεια Catalyst, μπορείτε να αποκτήσετε πρόσβαση σε μια σειρά από οφέλη. Μάθετε πώς να τα ενεργοποιήσετε εδώ.

### Λήψη σήματος φόρουμ

**Σημείωση:** Για να συνδέσετε τον λογαριασμό σας Obsidian με τον λογαριασμό σας στο φόρουμ, πρέπει να χρησιμοποιείτε την ίδια διεύθυνση ηλεκτρονικού ταχυδρομείου και στους δύο λογαριασμούς.

1. Εάν δεν το έχετε κάνει ήδη, εγγραφείτε στο [φόρουμ](https://forum.obsidian.md/) μας.
2. Μεταβείτε στον [πίνακα ελέγχου του λογαριασμού Obsidian](https://obsidian.md/account/catalyst).
3. Στην ενότητα **Catalyst**, επιλέξτε **Get forum badge**.

### Λήψη σήματος Discord

1. Εάν δεν το έχετε κάνει ήδη, εγγραφείτε στον [διακομιστή Discord](https://discord.gg/veuWUTm) μας.
2. Μεταβείτε στον [πίνακα ελέγχου του λογαριασμού Obsidian](https://obsidian.md/account/catalyst).
3. Στην ενότητα **Catalyst**, επιλέξτε **Get Discord badge**.

### Συμμετοχή στα κανάλια Insider

Τα μέλη Insider μπορούν να συνομιλούν με την ομάδα του Obsidian και άλλα μέλη Insider στα αποκλειστικά κανάλια Discord μας. Συμμετέχετε αυτόματα στα κανάλια Insider αφού [[#Λήψη σήματος Discord|λάβετε το σήμα Discord]].
