Το Obsidian προσφέρει αυτή τη στιγμή δύο τύπους εκπτώσεων: εκπαιδευτική και μη κερδοσκοπική. Και οι δύο σας παρέχουν 40% έκπτωση στις πρόσθετες υπηρεσίες μας [[Εισαγωγή στο Obsidian Sync|Obsidian Sync]] και [[Εισαγωγή στο Obsidian Publish|Obsidian Publish]].

## Πώς να υποβάλετε αίτηση

1. Μεταβείτε στον [πίνακα ελέγχου του λογαριασμού σας](https://obsidian.md/account/billing).
2. Στην ενότητα **Χρέωση**, βρείτε την ενότητα **Έκπτωση** και κάντε κλικ στον σύνδεσμο **Εφαρμογή**.
3. Επιλέξτε τον τύπο έκπτωσης ανάλογα με την περίπτωσή σας, συμπληρώστε το **εκπαιδευτικό ή οργανωτικό ηλεκτρονικό ταχυδρομείο σας** (όχι το ηλεκτρονικό ταχυδρομείο του λογαριασμού σας).
4. Κάντε κλικ στο **Αποστολή αίτησης**.
5. Στα εισερχόμενα του εκπαιδευτικού ή οργανωτικού ηλεκτρονικού ταχυδρομείου σας, θα λάβετε ένα μήνυμα από εμάς για την επαλήθευση της διεύθυνσής σας. Κάντε κλικ στον σύνδεσμο για επαλήθευση.
6. Μπορεί να εγκριθείτε άμεσα για την έκπτωση. Αν όχι, η ομάδα μας θα προσπαθήσει να εξετάσει την αίτησή σας εντός 3 εργάσιμων ημερών.
7. Θα λάβετε ένα μήνυμα ηλεκτρονικού ταχυδρομείου όταν εγκριθεί η έκπτωσή σας.

Συγχαρητήρια! Τώρα θα βλέπετε την έκπτωση όταν προσπαθείτε να αγοράσετε Obsidian Sync ή Publish.

## Πώς να ανανεώσετε

Μόλις έχετε μια ενεργή έκπτωση, μπορείτε να δείτε πότε λήγει στην ενότητα **Χρέωση → Έκπτωση** του [λογαριασμού σας](https://obsidian.md/account/billing).

Μία εβδομάδα πριν λήξει η έκπτωσή σας, θα λάβετε μια υπενθύμιση μέσω ηλεκτρονικού ταχυδρομείου για ανανέωση. Μπορείτε να ακολουθήσετε τα ίδια βήματα στην ενότητα «[[#Πώς να υποβάλετε αίτηση]]» για να ανανεώσετε την έκπτωση, εφόσον εξακολουθείτε να πληροίτε τις προϋποθέσεις.

## Συχνές ερωτήσεις

**Ποιοι είναι επιλέξιμοι;**
Αν είστε φοιτητής ή μέλος του ακαδημαϊκού προσωπικού ενός εκπαιδευτικού οργανισμού όπως πανεπιστήμιο ή λύκειο, γενικά πληροίτε τις προϋποθέσεις για εκπαιδευτική έκπτωση. Αν διευθύνετε ή εργάζεστε σε μη κερδοσκοπικό οργανισμό, γενικά πληροίτε τις προϋποθέσεις για έκπτωση μη κερδοσκοπικού οργανισμού.

**Πόσο χρόνο διαρκεί η διαδικασία αίτησης;**
Αν η αίτησή σας δεν εγκριθεί άμεσα μετά την επαλήθευση του ηλεκτρονικού ταχυδρομείου σας, η ομάδα μας θα την εξετάσει. Θα χρειαστούν έως 3 εργάσιμες ημέρες. Δεν χρειάζεται να ελέγξετε ξανά, θα σας στείλουμε ένα μήνυμα ηλεκτρονικού ταχυδρομείου όταν εγκριθεί ο λογαριασμός σας. Ευχαριστούμε για την υπομονή σας!

**Αυτές οι εκπτώσεις λειτουργούν με την τιμολόγηση πρώιμης πρόσβασης;**
Ευχαριστούμε που ήσασταν από τους πρώτους! Καμία από αυτές τις εκπτώσεις δεν αθροίζεται με την τιμολόγηση πρώιμης πρόσβασης, καθώς η πρώιμη πρόσβαση είναι πιθανότατα η μεγαλύτερη έκπτωση που θα δώσουμε ποτέ.

Αν είστε ήδη χρήστης πρώιμης πρόσβασης, γενικά δεν έχει νόημα να υποβάλετε αίτηση για εκπτώσεις. Αλλά λάβετε υπόψη ότι η τιμολόγηση πρώιμης πρόσβασης είναι ξεχωριστή για Sync και Publish, οπότε αν χάσατε την τιμολόγηση πρώιμης πρόσβασης για ένα από αυτά, μπορεί να θέλετε να υποβάλετε αίτηση για έκπτωση.

**Τι γίνεται αν ο εκπαιδευτικός ή μη κερδοσκοπικός οργανισμός μου δεν παρέχει διεύθυνση ηλεκτρονικού ταχυδρομείου;**
Σε αυτή την περίπτωση, μπορείτε να στείλετε άλλες μορφές επαλήθευσης, όπως φοιτητική ταυτότητα, αντίγραφο αναλυτικής βαθμολογίας ή ταυτότητα εργαζομένου στην [[Βοήθεια και υποστήριξη#Contact Obsidian support|υποστήριξη Obsidian]]. Βεβαιωθείτε ότι συμπεριλαμβάνετε και τη διεύθυνση ηλεκτρονικού ταχυδρομείου του λογαριασμού σας Obsidian. Η ομάδα μας θα εξετάσει την αίτησή σας και θα επικοινωνήσει μαζί σας. Ευχαριστούμε για την υπομονή σας!
