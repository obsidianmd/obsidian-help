Το Obsidian σας επιτρέπει να μεταφέρετε εύκολα τις σημειώσεις σας από το Notion χρησιμοποιώντας το [[Εισαγωγέας|πρόσθετο Εισαγωγέα]]. Αυτό θα μετατρέψει τα δεδομένα σας από το Notion σε ανθεκτικά αρχεία Markdown που μπορείτε να χρησιμοποιήσετε εκτός σύνδεσης με το Obsidian και πολλές άλλες εφαρμογές.

Το Obsidian προσφέρει δύο τρόπους εισαγωγής των δεδομένων σας από το Notion:

1. **Εισαγωγή μέσω API** διατηρεί ολόκληρο τον χώρο εργασίας σας, συμπεριλαμβανομένων των Βάσεων Δεδομένων και των τύπων, οι οποίοι μετατρέπονται σε [[Εισαγωγή στις Βάσεις|Βάσεις]], αλλά απαιτεί ένα token ενσωμάτωσης Notion και σύνδεση στο διαδίκτυο.
2. **Εισαγωγή αρχείων** δεν διατηρεί τις Βάσεις Δεδομένων αλλά δεν απαιτεί token API ή σύνδεση στο διαδίκτυο.

## Εισαγωγή μέσω API

### Δημιουργία token ενσωμάτωσης Notion API

Για να αποκτήσετε πρόσβαση στα δεδομένα σας στο Notion μέσω του API, χρειάζεστε ένα token ενσωμάτωσης. Αυτό το βήμα χρειάζεται περίπου 2 λεπτά για να ολοκληρωθεί.

Το token είναι ένα μεγάλο αλφαριθμητικό που συνήθως ξεκινά με `ntn_...` και σας επιτρέπει να κατεβάσετε τα δεδομένα σας από το Notion.

1. Συνδεθείτε στον πίνακα ελέγχου [Notion Integrations](https://www.notion.so/profile/integrations/internal).
2. Επιλέξτε **New integration**.

![[notion-integration.png#interface]]

2. Δώστε στην ενσωμάτωσή σας ένα όνομα, π.χ. "Personal". Μπορείτε να χρησιμοποιήσετε οποιοδήποτε όνομα.
3. Επιλέξτε τον χώρο εργασίας που θέλετε να εξαγάγετε.
4. Κάντε κλικ στο **Save** και συνεχίστε στο **Configure integration settings**.
5. Στην καρτέλα **Configuration**, το token API σας είναι διαθέσιμο στο πεδίο **Internal Integration Secret**.
6. Επιλέξτε **Show** και μετά **Copy**.
7. Αποθηκεύστε το token σε ένα ασφαλές μέρος, όπως τον διαχειριστή κωδικών πρόσβασής σας.

![[notion-token.png#interface]]

Στη συνέχεια, δώστε στην ενσωμάτωσή σας πρόσβαση στις σελίδες και τις βάσεις δεδομένων του Notion που θέλετε να εισαγάγετε.

1. Μεταβείτε στην καρτέλα **Access** της ενσωμάτωσης που μόλις δημιουργήσατε.
2. Κάντε κλικ στο **Edit access**.
3. Προσθέστε τις σελίδες και τις βάσεις δεδομένων που θέλετε να εισαγάγετε.

Τώρα μπορείτε να μετατρέψετε τα δεδομένα σας χρησιμοποιώντας τον Εισαγωγέα του Obsidian.

### Εισαγωγή δεδομένων Notion μέσω API

Θα χρειαστείτε το επίσημο πρόσθετο [[Εισαγωγέας]] του Obsidian, το οποίο μπορείτε να [εγκαταστήσετε εδώ](obsidian://show-plugin?id=obsidian-importer).

1. Ανοίξτε τις **[[Ρυθμίσεις]]**.
2. Μεταβείτε στα **Πρόσθετα κοινότητας** και [εγκαταστήστε τον Εισαγωγέα](obsidian://show-plugin?id=obsidian-importer).
3. Ενεργοποιήστε το πρόσθετο Εισαγωγέα.
4. Ανοίξτε το πρόσθετο **Εισαγωγέα** χρησιμοποιώντας την παλέτα εντολών ή το εικονίδιο στην κορδέλα.
5. Στην επιλογή **Μορφοποίηση αρχείου** επιλέξτε **Notion (API)**
6. Στο πεδίο **API token**, επικολλήστε το **Internal Integration Secret** από το Notion.
7. Κάντε κλικ στο **Φόρτωση** για να επιλέξετε τις βάσεις δεδομένων και τις σελίδες που θέλετε να εισαγάγετε.
8. Ελέγξτε και επεξεργαστείτε τις επιλογές εισαγωγής.
9. Επιλέξτε **Εισαγωγή** και περιμένετε μέχρι να ολοκληρωθεί η εισαγωγή
10. Τελειώσατε!

### Περιορισμοί

> [!info] Η εισαγωγή μέσω API είναι νέα
> Ο εισαγωγέας Notion API είναι νέος. Λόγω της πολυπλοκότητας των χώρων εργασίας του Notion, ορισμένες ακραίες περιπτώσεις μπορεί να μην έχουν ληφθεί υπόψη. Εάν αντιμετωπίσετε προβλήματα με τη μετατροπή, [υποβάλετε αναφορά σφάλματος](https://github.com/obsidianmd/obsidian-importer/issues) ώστε να μπορέσουμε να τη βελτιώσουμε.

Λόγω των ορίων ρυθμού του Notion API, η εισαγωγή μεγάλων χώρων εργασίας μπορεί να χρειαστεί αρκετό χρόνο. Παρακαλούμε κάντε υπομονή.

Λόγω περιορισμών στο Notion API, ορισμένα δεδομένα δεν είναι διαθέσιμα ή δεν μπορούν να μετατραπούν:

- Εισάγεται μόνο η κύρια προβολή για κάθε βάση δεδομένων.
- Οι [συνδεδεμένες πηγές δεδομένων](https://developers.notion.com/docs/working-with-databases#additional-types-of-databases) δεν εισάγονται: *"Το API του Notion δεν υποστηρίζει προς το παρόν συνδεδεμένες πηγές δεδομένων. Όταν κοινοποιείτε μια βάση δεδομένων στην ενσωμάτωσή σας, βεβαιωθείτε ότι περιέχει την αρχική πηγή δεδομένων!"*
- Συναρτήσεις `People`: `name()` και `email()`
- Συναρτήσεις `Text`: `style()` και `unstyle()`

Επιπλέον, ο Εισαγωγέας θα κάνει τις ακόλουθες αλλαγές:

- Οι σελίδες χωρίς θυγατρικές σελίδες ή βάσεις δεδομένων θα εισαχθούν ως `[filename].md` αντί για `[filename]/[filename].md`.
- Οι βάσεις δεδομένων αναπαρίστανται πάντα ως φάκελοι με όνομα `[database name]` με ένα αρχείο `[database name].base` μέσα.

## Εισαγωγή αρχείων

Η εισαγωγή αρχείων είναι εναλλακτικός τρόπος εισαγωγής των δεδομένων σας από το Notion. Αυτή η μέθοδος δεν διατηρεί τις Βάσεις Δεδομένων αλλά δεν απαιτεί token API ή σύνδεση στο διαδίκτυο.

### Εξαγωγή δεδομένων από το Notion

Για να προετοιμάσετε τα δεδομένα σας για εισαγωγή, θα πρέπει να εξαγάγετε ολόκληρο τον χώρο εργασίας σας χρησιμοποιώντας τη μορφή εξαγωγής HTML του Notion. Συνιστούμε να μη χρησιμοποιήσετε την εξαγωγή Markdown του Notion, καθώς παραλείπει σημαντικά δεδομένα. Πρέπει να έχετε δικαιώματα διαχειριστή στον χώρο εργασίας του Notion για να εξαγάγετε όλο το περιεχόμενο.

1. Μεταβείτε στις **[[Ρυθμίσεις]]** στο πάνω μέρος της πλαϊνής μπάρας του Notion.
2. Στην ενότητα **Workspace** επιλέξτε **General**.
3. Βρείτε και επιλέξτε **Export all workspace content**.
4. Στο πεδίο **Export format** επιλέξτε **HTML**.
5. Επιλέξτε **Include everything**.
6. Ενεργοποιήστε **Create folders for subpages**.
7. Θα λάβετε ένα αρχείο `.zip` μέσω ηλεκτρονικού ταχυδρομείου ή απευθείας στο πρόγραμμα περιήγησης.

### Εισαγωγή του αρχείου .zip του Notion

Θα χρειαστείτε το επίσημο πρόσθετο [[Εισαγωγέας]] του Obsidian, το οποίο μπορείτε να [εγκαταστήσετε εδώ](obsidian://show-plugin?id=obsidian-importer).

1. Ανοίξτε τις **[[Ρυθμίσεις]]**.
2. Μεταβείτε στα **Πρόσθετα κοινότητας** και [εγκαταστήστε τον Εισαγωγέα](obsidian://show-plugin?id=obsidian-importer).
3. Ενεργοποιήστε το πρόσθετο Εισαγωγέα.
4. Ανοίξτε το πρόσθετο **Εισαγωγέα** χρησιμοποιώντας την παλέτα εντολών ή το εικονίδιο στην κορδέλα.
5. Στην επιλογή **Μορφοποίηση αρχείου** επιλέξτε **Notion (.zip)**
6. Επιλέξτε το αρχείο `.zip` με τα αρχεία Notion που θέλετε να εισαγάγετε. *Συνιστάται να εισαγάγετε όλα τα δεδομένα σας από το Notion ταυτόχρονα, ώστε οι εσωτερικοί σύνδεσμοι να μπορούν να αντιστοιχιστούν σωστά.*
7. _Προαιρετικά_, επιλέξτε έναν φάκελο για την εισαγωγή. Οι σελίδες και οι βάσεις δεδομένων σας από το Notion θα είναι ένθετες μέσα σε αυτόν τον φάκελο.
8. Ενεργοποιήστε **Αποθήκευση γονικών σελίδων σε υποφακέλους** για να διατηρήσετε τη δομή του Notion. *Σημειώστε ότι στο Notion μπορείτε να γράψετε περιεχόμενο σε Φακέλους, κάτι που δεν είναι δυνατό στο Obsidian και αυτές οι σελίδες θα προστεθούν ως υποσελίδα κάτω από τον φάκελο.*
9. Επιλέξτε **Εισαγωγή** και περιμένετε μέχρι να ολοκληρωθεί η εισαγωγή
10. Τελειώσατε!

### Αντιμετώπιση προβλημάτων

Εάν αντιμετωπίσετε προβλήματα κατά την εισαγωγή από το Notion:

- Βεβαιωθείτε ότι χρησιμοποιείτε **HTML** ως μορφή εξαγωγής στο Notion, **όχι Markdown**.
- Εάν το Obsidian φαίνεται να παγώνει κατά την εισαγωγή, απενεργοποιήστε τα πρόσθετα κοινότητας και δοκιμάστε ξανά.

Αντιμετωπίζετε κάτι άλλο; Αναζητήστε στο [αποθετήριο του Εισαγωγέα](https://github.com/obsidianmd/obsidian-importer/issues) για να δείτε αν το έχουν αντιμετωπίσει και άλλοι.

#### Εισαγωγή μεγάλων χώρων εργασίας

Εάν εισάγετε έναν χώρο εργασίας με πολλά gigabyte δεδομένων, η εξαγωγή από το Notion μπορεί να περιέχει ένθετα αρχεία `.zip`. Σε αυτήν την περίπτωση μπορεί να δείτε ένα μήνυμα σφάλματος εισαγωγής που μοιάζει κάπως έτσι:

```
Import failed {id}.zip/{id}-Part-1.zip undefined.
```

Εάν δείτε αυτό το σφάλμα, μπορείτε να αποσυμπιέσετε το αρχείο από το Notion και στη συνέχεια να εισαγάγετε τα ένθετα αρχεία `Export-{id}-Part-1.zip`.
