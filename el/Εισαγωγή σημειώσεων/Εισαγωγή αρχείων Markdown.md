Το Obsidian χρησιμοποιεί αρχεία Markdown `.md` ως την κύρια μορφή για τις σημειώσεις. Αυτό διευκολύνει την εισαγωγή αρχείων Markdown στο Obsidian. Πολλές εφαρμογές μπορούν να επεξεργαστούν αρχεία Markdown ή να εξάγουν σε αυτήν τη μορφή.

Υπάρχουν διάφοροι τρόποι για να προσθέσετε αρχεία Markdown στο θησαυροφυλάκιό σας στο Obsidian:

1. Σύρετε και αποθέστε αρχεία ή φακέλους στο παράθυρο της [[Εξερεύνηση αρχείων|Εξερεύνησης αρχείων]]
2. Χρησιμοποιήστε τον διαχειριστή αρχείων του συστήματός σας (π.χ. Windows Explorer ή Finder στο macOS) και μετακινήστε τα αρχεία σας απευθείας στον φάκελο του θησαυροφυλακίου του Obsidian.

### Μετατροπή από άλλες παραλλαγές Markdown

Το Obsidian χρησιμοποιεί [[Obsidian Flavored Markdown]]. Ορισμένες εφαρμογές χρησιμοποιούν ελαφρώς διαφορετική σύνταξη Markdown. Μπορείτε να χρησιμοποιήσετε τον [[Εισαγωγέας μορφής Markdown|Εισαγωγέα μορφής Markdown]] για να μετατρέψετε τα αρχεία σας στη σύνταξη Markdown του Obsidian.
