Μάθετε πώς μπορείτε να ενσωματώνετε άλλες σημειώσεις και πολυμέσα στις σημειώσεις σας. Ενσωματώνοντας αρχεία στις σημειώσεις σας, μπορείτε να επαναχρησιμοποιήσετε περιεχόμενο σε όλο το θησαυροφυλάκιό σας.

Για να ενσωματώσετε ένα αρχείο στο θησαυροφυλάκιό σας, προσθέστε ένα θαυμαστικό (`!`) μπροστά από έναν [[Εσωτερικοί σύνδεσμοι|Εσωτερικό σύνδεσμο]]. Μπορείτε να ενσωματώσετε αρχεία σε οποιαδήποτε από τις [[Αποδεκτές μορφές αρχείων]].

> [!tip] Ενσωμάτωση με Μεταφορά και Απόθεση
> Στην επιφάνεια εργασίας, μπορείτε επίσης να σύρετε και να αποθέσετε υποστηριζόμενα αρχεία απευθείας στη σημείωσή σας για να τα ενσωματώσετε αυτόματα.

## Ενσωμάτωση σημείωσης σε άλλη σημείωση

Για να ενσωματώσετε μια σημείωση:

```md
![[Εσωτερικοί σύνδεσμοι]]
```

Μπορείτε επίσης να ενσωματώσετε συνδέσμους σε [[Εσωτερικοί σύνδεσμοι#Σύνδεση σε επικεφαλίδα σε σημείωση|επικεφαλίδες]] και [[Εσωτερικοί σύνδεσμοι#Σύνδεση σε μπλοκ σε σημείωση|μπλοκ]].

```md
![[Εσωτερικοί σύνδεσμοι#^b15695]]
```

Το παρακάτω κείμενο είναι ένα παράδειγμα ενσωματωμένου μπλοκ:

![[Εσωτερικοί σύνδεσμοι#^b15695]]

## Ενσωμάτωση εικόνας σε σημείωση

Για να ενσωματώσετε μια εικόνα:

```md
![[Engelbart.jpg]]
```

![[Engelbart.jpg#outline]]

Μπορείτε να αλλάξετε τις διαστάσεις της εικόνας, προσθέτοντας `|640x480` στον προορισμό του συνδέσμου, όπου 640 είναι το πλάτος και 480 είναι το ύψος.

```md
![[Engelbart.jpg|100x145]]
```

Αν καθορίσετε μόνο το πλάτος, η εικόνα κλιμακώνεται σύμφωνα με τις αρχικές αναλογίες της. Για παράδειγμα, `![[Engelbart.jpg|100]]`.

![[Engelbart.jpg#outline|100]]

Μπορείτε επίσης να ενσωματώσετε μια εικόνα που φιλοξενείται εξωτερικά χρησιμοποιώντας σύνδεσμο Markdown. Μπορείτε να ελέγξετε το πλάτος και το ύψος με τον ίδιο τρόπο όπως με έναν σύνδεσμο wiki.

```md
![250](https://publish-01.obsidian.md/access/f786db9fac45774fa4f0d8112e232d67/Attachments/Engelbart.jpg)
```

![250](https://publish-01.obsidian.md/access/f786db9fac45774fa4f0d8112e232d67/Attachments/Engelbart.jpg)

## Ενσωμάτωση PDF σε σημείωση

Για να ενσωματώσετε ένα PDF:

```md
![[Document.pdf]]
```

Μπορείτε επίσης να ανοίξετε μια συγκεκριμένη σελίδα στο PDF, προσθέτοντας `#page=N` στον προορισμό του συνδέσμου, όπου `N` είναι ο αριθμός της σελίδας:

```md
![[Document.pdf#page=3]]
```

Μπορείτε επίσης να καθορίσετε το ύψος σε εικονοστοιχεία για τον ενσωματωμένο προβολέα PDF, προσθέτοντας `#height=[number]` στον προορισμό. Για παράδειγμα:

```md
![[Document.pdf#height=400]]
```

## Ενσωμάτωση λίστας σε σημείωση

Για να ενσωματώσετε μια λίστα από διαφορετική σημείωση, πρώτα προσθέστε ένα [[Εσωτερικοί σύνδεσμοι#Σύνδεση σε μπλοκ σε σημείωση|αναγνωριστικό μπλοκ]] στη λίστα σας:

```md

- στοιχείο λίστας 1
- στοιχείο λίστας 2

^my-list-id
```

Στη συνέχεια συνδέστε τη λίστα χρησιμοποιώντας το αναγνωριστικό μπλοκ:

```md
![[My note#^my-list-id]]
```
