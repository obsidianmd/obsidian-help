Αν γνωρίζετε μια άλλη γλώσσα, κατά προτίμηση ως μητρική, μπορείτε να βοηθήσετε στη μετάφραση της διεπαφής χρήστη του Obsidian ή της τεκμηρίωσης.

Για να βοηθήσετε στη μετάφραση της διεπαφής χρήστη, υποβάλετε ένα pull request στο [αποθετήριο μεταφράσεων](https://github.com/obsidianmd/obsidian-translations).

Για να βοηθήσετε στη βελτίωση της τεκμηρίωσής μας, υποβάλετε ένα pull request στο αποθετήριο [obsidian-help](https://github.com/obsidianmd/obsidian-help). Βεβαιωθείτε ότι τηρείτε τον [[Οδηγός στυλ#Μεταφράσεις|οδηγό στυλ μεταφράσεων]] κατά την πραγματοποίηση αλλαγών.
