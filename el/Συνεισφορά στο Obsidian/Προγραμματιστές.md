Αν είστε προγραμματιστής, μπορείτε να συνεισφέρετε στο Obsidian με διάφορους τρόπους.

### Πρόσθετα και θέματα

Μπορείτε να αναπτύξετε [[Πρόσθετα κοινότητας]] και [[Θέματα]] χρησιμοποιώντας Typescript και CSS. Μάθετε περισσότερα μεταβαίνοντας στην [Τεκμηρίωση Προγραμματιστών Obsidian](https://docs.obsidian.md).

### Έργα ανοιχτού κώδικα

Το Obsidian αναπτύσσει και συντηρεί αρκετά έργα ανοιχτού κώδικα. Μπορείτε να υποβάλετε pull requests με διορθώσεις σφαλμάτων και βελτιώσεις στα ακόλουθα αποθετήρια GitHub:

- [JSON Canvas](https://github.com/obsidianmd/jsoncanvas) (MIT), μια ανοιχτή μορφή αρχείου για δεδομένα άπειρου καμβά.
- [Obsidian Importer](https://github.com/obsidianmd/obsidian-importer) (MIT), ένα πρόσθετο που μετατρέπει δεδομένα από μεγάλη ποικιλία μορφών σε αρχεία απλού κειμένου Markdown.
- [Obsidian Maps](https://github.com/obsidianmd/obsidian-maps) (MIT), ένα πρόσθετο που εμφανίζει χάρτες στις Βάσεις του Obsidian.
- [Obsidian Web Clipper](https://github.com/obsidianmd/obsidian-clipper) (MIT), μια επέκταση προγράμματος περιήγησης για επισήμανση και αποθήκευση ιστοσελίδων σε αρχεία απλού κειμένου Markdown.
- [Defuddle](https://github.com/kepano/defuddle) (MIT), ο αναλυτής HTML-σε-Markdown που χρησιμοποιείται από το Obsidian Web Clipper.

### Τεκμηρίωση

Για να βοηθήσετε στη βελτίωση της [τεκμηρίωσης προγραμματιστών](https://docs.obsidian.md/Home), υποβάλετε ένα pull request στο αποθετήριο [obsidian-developer-docs](https://github.com/obsidianmd/obsidian-developer-docs).

Για να βοηθήσετε στη βελτίωση αυτής της τεκμηρίωσης, υποβάλετε ένα pull request στο αποθετήριο [obsidian-help](https://github.com/obsidianmd/obsidian-help).

### Κοινότητα

Το κανάλι [plugin-dev](https://discord.com/channels/686053708261228577/840286264964022302) στον επίσημο διακομιστή Discord μας είναι ο χώρος όπου μπορείτε να συζητήσετε για την ανάπτυξη πρόσθετων και να βοηθήσετε άλλους προγραμματιστές με τα πρόσθετά τους.
