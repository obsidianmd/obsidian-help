Ο Διερμηνέας είναι μια λειτουργία του [[Εισαγωγή στο Obsidian Web Clipper|Web Clipper]] που σας επιτρέπει να αλληλεπιδράτε με ιστοσελίδες χρησιμοποιώντας φυσική γλώσσα. Ο Διερμηνέας σας βοηθά να καταγράφετε και να τροποποιείτε δεδομένα που θέλετε να αποθηκεύσετε στο Obsidian. Για παράδειγμα:

- Εξαγωγή συγκεκριμένων τμημάτων κειμένου.
- Σύνοψη ή επεξήγηση πληροφοριών.
- Μετατροπή κειμένου από μια μορφή σε άλλη.
- Μετάφραση κειμένου σε διαφορετική γλώσσα.

Ο Διερμηνέας αξιοποιεί γλωσσικά μοντέλα για την επεξεργασία πληροφοριών σε μια ιστοσελίδα και επιστρέφει αποτελέσματα χρησιμοποιώντας [[Μεταβλητές]] που μπορείτε να προσθέσετε στα [[Obsidian Web Clipper/Πρότυπα|Πρότυπα Web Clipper]] σας.

## Παραδείγματα εντολών

Οι εντολές χρησιμοποιούν τη σύνταξη [[Μεταβλητές|μεταβλητών]] `{{"η εντολή σας"}}`. Μπορείτε να χρησιμοποιήσετε αυτή τη σύνταξη με οποιοδήποτε ερώτημα σε φυσική γλώσσα, π.χ.

- `{{"a summary of the page"}}` για εξαγωγή σύνοψης της σελίδας.
- `{{"a three bullet point summary, translated to French"}}` για εξαγωγή κουκκίδων σχετικά με τη σελίδα και μετάφρασή τους στα Γαλλικά.
- `{{"un resumé de la page en trois points"}}` για εξαγωγή τριών κουκκίδων χρησιμοποιώντας εντολή στα Γαλλικά.

Η έξοδος των εντολών σας μπορεί να τροποποιηθεί περαιτέρω χρησιμοποιώντας [[Φίλτρα]]. Τα φίλτρα εφαρμόζονται μετά τη λήψη της απάντησης από το μοντέλο. Για παράδειγμα: `{{"a summary of the page"|blockquote}}` θα μετατρέψει την απάντηση σε παράθεση σε μπλοκ.

## Ξεκινήστε

Ο Διερμηνέας λειτουργεί με σχεδόν οποιονδήποτε πάροχο γλωσσικών μοντέλων, συμπεριλαμβανομένων επιλογών που εκτελούνται ιδιωτικά στη συσκευή σας. Για να ρυθμίσετε τον Διερμηνέα:

1. Μεταβείτε στην ενότητα **Διερμηνέας** στις ρυθμίσεις του Web Clipper.
2. Ενεργοποιήστε το **Ενεργοποίηση διερμηνέα**.
3. Ρυθμίστε τον πάροχο και το μοντέλο σας, δείτε την ενότητα [[Ερμηνεία ιστοσελίδων#Μοντέλα|μοντέλα]] παρακάτω.
4. Προσθέστε [[Μεταβλητές|μεταβλητές εντολών]] στα [[Obsidian Web Clipper/Πρότυπα|πρότυπά]] σας.
5. Εάν το πρότυπό σας περιλαμβάνει μεταβλητές εντολών, η ενότητα Διερμηνέα θα είναι ορατή όταν [[Αποκοπή ιστοσελίδων|αποκόπτετε μια σελίδα]]. Κάντε κλικ στο **ερμηνεία** για να επεξεργαστείτε τις μεταβλητές εντολών.

## Πώς λειτουργεί

Όταν ο Διερμηνέας είναι ενεργοποιημένος *και* το πρότυπό σας περιέχει [[Μεταβλητές#Μεταβλητές εντολών|μεταβλητές εντολών]], μια νέα ενότητα Διερμηνέα εμφανίζεται στο παράθυρο της επέκτασης, πάνω από το κουμπί **Προσθήκη στο Obsidian**. Αυτή η ενότητα σας επιτρέπει να επιλέξετε ένα μοντέλο και να εκτελέσετε τον Διερμηνέα για την τρέχουσα σελίδα.

Όταν κάνετε κλικ στο **ερμηνεία**, ο Διερμηνέας στέλνει το πλαίσιο της σελίδας στο επιλεγμένο μοντέλο σας, μαζί με *όλες* τις εντολές στο πρότυπό σας σε ένα αίτημα. Ανάλογα με τον πάροχο μοντέλου που επιλέγετε, αυτό μπορεί να είναι μια εξωτερική κλήση ή τοπική στη συσκευή σας. Το μοντέλο αξιολογεί τις εντολές σας σε σχέση με το πλαίσιο της σελίδας και επιστρέφει τις απαντήσεις του. Ο Διερμηνέας στη συνέχεια αντικαθιστά τις μεταβλητές εντολών με τα δεδομένα απάντησης.

Η όλη διαδικασία μπορεί να διαρκέσει χιλιοστά του δευτερολέπτου ή περισσότερα από 30 δευτερόλεπτα ανάλογα με το μοντέλο που χρησιμοποιείτε και τον όγκο δεδομένων που επεξεργάζεστε.

## Πλαίσιο

Ο όρος *πλαίσιο* αναφέρεται στα δεδομένα σελίδας που χρησιμοποιεί ο Διερμηνέας για την επεξεργασία εντολών. Όσο μικρότερο είναι το πλαίσιο, τόσο ταχύτερα εκτελείται ο Διερμηνέας.

Από προεπιλογή, ο Διερμηνέας χρησιμοποιεί ολόκληρο το HTML της σελίδας ως πλαίσιο, ωστόσο αυτό μπορεί να κάνει τις εντολές πιο αργές και πιο δαπανηρές από ό,τι είναι απαραίτητο.

Μπορείτε να αντικαταστήσετε το προεπιλεγμένο πλαίσιο στις **Σύνθετες ρυθμίσεις** του Διερμηνέα και να ορίσετε πλαίσιο ανά [[Obsidian Web Clipper/Πρότυπα|πρότυπο]].

Για να ορίσετε ένα πιο στοχευμένο πλαίσιο χρησιμοποιήστε [[Μεταβλητές#Μεταβλητές επιλογέα|μεταβλητές επιλογέα]] (ή άλλους τύπους μεταβλητών) για να ερμηνεύσετε ένα τμήμα της σελίδας. Για παράδειγμα, μπορείτε να χρησιμοποιήσετε την ακόλουθη μεταβλητή επιλογέα στο πλαίσιο Διερμηνέα του προτύπου σας:

```
{{selectorHtml:#main}}
```

Αυτό θα εκτελέσει τον Διερμηνέα μόνο στο στοιχείο `#main` μιας ιστοσελίδας, εάν υπάρχει. Τα [[Φίλτρα#Επεξεργασία HTML|φίλτρα επεξεργασίας HTML]] όπως `remove_html`, `strip_tags` και `strip_attr` μπορούν να είναι χρήσιμα για την περαιτέρω μείωση του μεγέθους πλαισίου και την επιτάχυνση της επεξεργασίας.

### Επιλογή μοντέλου

Γενικά συνιστούμε τη χρήση μικρών μοντέλων με το Web Clipper επειδή είναι ταχύτερα και αποδίδουν αρκετά ακριβή αποτελέσματα για αυτή την εργασία. Παραδείγματα μικρότερων μοντέλων περιλαμβάνουν το **Claude Haiku της Anthropic**, το **Google Gemini Flash**, το **Llama** με 3B ή 8B παραμέτρους, ή τη σειρά μοντέλων **Mini της OpenAI**.

### Προσαρμοσμένοι πάροχοι και μοντέλα

Για να προσθέσετε έναν προσαρμοσμένο πάροχο ή/και μοντέλο μεταβείτε στις **[[Ρυθμίσεις]]** του Web Clipper → **Διερμηνέας**:

- **Προσθήκη παρόχου** για τη ρύθμιση προεπιλεγμένων και προσαρμοσμένων παρόχων.
- **Προσθήκη μοντέλου** για τη ρύθμιση προεπιλεγμένων και προσαρμοσμένων μοντέλων.

Κατά την προσθήκη ενός προσαρμοσμένου παρόχου, συνιστούμε να χρησιμοποιήσετε το endpoint ολοκλήρωσης συνομιλίας τους για τη **Βασική διεύθυνση URL** — τυπικά τελειώνει με `/chat/completions`.

### Τοπικά μοντέλα

Ο Διερμηνέας μπορεί να χρησιμοποιήσει τοπικά μοντέλα που προσφέρουν μεγαλύτερο απόρρητο και συμβατότητα εκτός σύνδεσης. Υπάρχουν αρκετές επιλογές για την εκτέλεση τοπικών μοντέλων. Μία από τις πιο εύκολες στη ρύθμιση είναι το Ollama.

#### Ollama

Το [Ollama](https://ollama.com/) σας επιτρέπει να εκτελείτε γλωσσικά μοντέλα τοπικά και ιδιωτικά στη συσκευή σας.

Αφού κατεβάσετε και εγκαταστήσετε το Ollama, προσθέστε το Ollama χρησιμοποιώντας **Προσθήκη παρόχου** στις ρυθμίσεις του Διερμηνέα. Το Ollama δεν απαιτεί κλειδί API. Στη συνέχεια επιλέξτε ένα μοντέλο από τη [λίστα μοντέλων](https://ollama.com/search). Για παράδειγμα, εάν θέλετε να χρησιμοποιήσετε το [Llama 3.2](https://ollama.com/library/llama3.2), κάντε κλικ στο **Προσθήκη μοντέλου**, στη συνέχεια:

- **Πάροχος:** Ollama
- **Εμφανιζόμενο όνομα:** Llama 3.2, αυτή η τιμή είναι προσαρμόσιμη.
- **Αναγνωριστικό μοντέλου:** `llama3.2`, αυτό πρέπει να ταιριάζει ακριβώς με το αναγνωριστικό μοντέλου από το Ollama.

**Εκκίνηση του διακομιστή Ollama**

Για να επιτρέψετε σε μια επέκταση προγράμματος περιήγησης να αλληλεπιδρά με το Ollama πρέπει να [δώσετε ρητή εντολή](https://github.com/ollama/ollama/issues/2308) κατά την εκτέλεση του διακομιστή, αλλιώς θα δείτε σφάλμα `403`.

Κλείστε την εφαρμογή Ollama και εκτελέστε την ακόλουθη εντολή στο τερματικό σας. Το πρωτόκολλο θα πρέπει να αλλάξει στο πρωτόκολλο επέκτασης του προγράμματος περιήγησής σας εάν δεν χρησιμοποιείτε Chrome ή Firefox.

```
OLLAMA_ORIGINS=moz-extension://*,chrome-extension://*,safari-web-extension://* ollama serve
```

Στη συνέχεια εκτελέστε το μοντέλο σας με το Ollama με τον κανονικό τρόπο, π.χ.

```
ollama run llama3.2
```

**Μήκος πλαισίου**

Το παράθυρο πλαισίου του Ollama είναι προεπιλεγμένο στα 2048 tokens. Αυτός είναι ο μέγιστος αριθμός tokens για το μήνυμα και την απάντηση. Κατά την αποκοπή μιας μεγάλης ιστοσελίδας μπορείτε εύκολα να υπερβείτε αυτό το όριο. Το Ollama θα αποτύχει σιωπηλά και θα επιστρέψει άσχετα αποτελέσματα. Μερικές επιλογές:

- Αυξήστε την παράμετρο `num_ctx` του Ollama. Λάβετε υπόψη ότι μεγαλύτερο πλαίσιο απαιτεί περισσότερη μνήμη.
- Χρησιμοποιήστε το πεδίο [[#Πλαίσιο]] στο πρότυπό σας για να παρέχετε ένα πιο στοχευμένο τμήμα της σελίδας, ή περικόψτε το πλαίσιο χρησιμοποιώντας ένα [[Φίλτρα|φίλτρο]] π.χ. `{{content|slice:0,1000}}`.
