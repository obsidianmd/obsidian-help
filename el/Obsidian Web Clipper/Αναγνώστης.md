Το [[Εισαγωγή στο Obsidian Web Clipper|Web Clipper]] περιλαμβάνει μια ενσωματωμένη λειτουργία ανάγνωσης που ονομάζεται **Αναγνώστης**, η οποία αφαιρεί την ακαταστασία από τις ιστοσελίδες, παρουσιάζοντας μόνο το κύριο περιεχόμενο του άρθρου σε μια καθαρή, ευανάγνωστη μορφή.

## Ενεργοποίηση λειτουργίας ανάγνωσης

Μπορείτε να ενεργοποιήσετε τον Αναγνώστη με διάφορους τρόπους:

- ![[lucide-book-icon.svg#icon]] **Εικονίδιο βιβλίου** στο πάνελ της επέκτασης.
- **Συντόμευση πληκτρολογίου** με δυνατότητα προσαρμογής στις ρυθμίσεις της επέκτασης.
- **Μενού περιβάλλοντος** κάνοντας δεξί κλικ στην ιστοσελίδα που επισκέπτεστε.

Μπορείτε επίσης να ρυθμίσετε το κουμπί επέκτασης ![[obsidian.svg#icon|20]] ώστε να ανοίγει τον Αναγνώστη με ένα κλικ. Στις ρυθμίσεις του Web Clipper μεταβείτε στο **General** → **Behavior** → **Open behavior** και επιλέξτε **Reader**.

Όλες οι παραπάνω επιλογές σας επιτρέπουν επίσης να βγείτε από τον Αναγνώστη και να επαναφέρετε την αρχική σελίδα.

## Πώς λειτουργεί

Ο Αναγνώστης εξάγει το κύριο περιεχόμενο από τη σελίδα. Η αρχική σελίδα αντικαθίσταται με μια καθαρή προβολή ανάγνωσης που περιλαμβάνει:

- Τον τίτλο του άρθρου, τον συγγραφέα, την ημερομηνία δημοσίευσης και τον τομέα πηγής.
- Το κύριο περιεχόμενο του άρθρου, με εικόνες και μορφοποίηση διατηρημένες.
- Μια διάρθρωση στην πλαϊνή μπάρα για άρθρα με πολλαπλές επικεφαλίδες.
- Επισήμανση σύνταξης για μπλοκ κώδικα.
- Αναδυόμενα παράθυρα υποσημειώσεων που εμφανίζονται ενσωματωμένα όταν κάνετε κλικ.

## Προσαρμογή της προβολής ανάγνωσης

Προσαρμόστε την εμπειρία Αναγνώστη στις ρυθμίσεις της επέκτασης. Ορισμένες επιλογές είναι επίσης διαθέσιμες μέσα στη λειτουργία Αναγνώστη κάνοντας κλικ στο εικονίδιο ![[lucide-case-sensitive.svg#icon]].

| Ρύθμιση                        | Περιγραφή                                          |
| ------------------------------ | -------------------------------------------------- |
| Γραμματοσειρά                  | Προσθέστε γραμματοσειρές εγκατεστημένες στο σύστημά σας. |
| Μέγεθος γραμματοσειράς         | Ρύθμιση του μεγέθους κειμένου.                     |
| Ύψος&nbsp;γραμμής              | Απόσταση μεταξύ γραμμών.                           |
| Πλάτος&nbsp;γραμμής            | Μέγιστο πλάτος της περιοχής περιεχομένου.          |
| Εμφάνιση                       | Επιλέξτε φωτεινή ή σκοτεινή εμφάνιση.            |
| Θέμα                           | Επιλέξτε τα χρώματα.                               |
| Προσαρμοσμένο CSS              | Πρόσθετο CSS που εφαρμόζεται στη σελίδα.          |
