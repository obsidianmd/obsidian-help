Το [[Εισαγωγή στο Obsidian Web Clipper|Web Clipper]] περιλαμβάνει μια ενσωματωμένη λειτουργία ανάγνωσης που ονομάζεται Αναγνώστης, η οποία αφαιρεί την ακαταστασία από τις ιστοσελίδες, παρουσιάζοντας μόνο το κύριο περιεχόμενο του άρθρου σε μια καθαρή, ευανάγνωστη μορφή.

## Ενεργοποίηση λειτουργίας ανάγνωσης

Μπορείτε να ενεργοποιήσετε τον Αναγνώστη με διάφορους τρόπους:

- Το **εικονίδιο βιβλίου** στην κεφαλίδα του πάνελ της επέκτασης.
- Συντόμευση πληκτρολογίου `Alt+Shift+R`, με δυνατότητα προσαρμογής στις ρυθμίσεις της επέκτασης.
- Η εκ νέου εναλλαγή της λειτουργίας ανάγνωσης θα επαναφέρει την αρχική σελίδα.

## Πώς λειτουργεί

Ο Αναγνώστης εξάγει το κύριο περιεχόμενο από τη σελίδα. Η αρχική σελίδα αντικαθίσταται με μια καθαρή προβολή ανάγνωσης που περιλαμβάνει:

- Τον τίτλο του άρθρου, τον συγγραφέα, την ημερομηνία δημοσίευσης και τον τομέα πηγής.
- Το κύριο περιεχόμενο του άρθρου, με εικόνες και μορφοποίηση διατηρημένες.
- Μια διάρθρωση στην πλαϊνή μπάρα για άρθρα με πολλαπλές επικεφαλίδες.
- Επισήμανση σύνταξης για μπλοκ κώδικα.
- Αναδυόμενα παράθυρα υποσημειώσεων που εμφανίζονται ενσωματωμένα όταν κάνετε κλικ.

## Προσαρμογή της προβολής ανάγνωσης

Η λειτουργία Αναγνώστη περιλαμβάνει μια κρυφή γραμμή ρυθμίσεων που μπορείτε να εμφανίσετε με αιώρηση στην επάνω δεξιά γωνία της σελίδας, με στοιχεία ελέγχου για την προσαρμογή της εμπειρίας ανάγνωσης.

| Ρύθμιση                       | Περιγραφή                                                                                                                          |
| ----------------------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| Μέγεθος γραμματοσειράς        | Ρύθμιση του μεγέθους κειμένου.                                                                                                     |
| Πλάτος γραμμής                | Μέγιστο πλάτος της περιοχής περιεχομένου.                                                                                          |
| Ύψος γραμμής                  | Απόσταση μεταξύ γραμμών.                                                                                                           |
| Χρωματικό&nbsp;σχήμα          |                                                                                                                                    |
| Θέμα                          | Αυτόματα ακολουθεί τις προτιμήσεις του συστήματός σας. Ή επιλέξτε να χρησιμοποιείτε πάντα φωτεινή ή σκοτεινή λειτουργία. |
