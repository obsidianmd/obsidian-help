Το [[Εισαγωγή στο Obsidian Web Clipper|Web Clipper]] σας επιτρέπει να δημιουργείτε πρότυπα που αυτόματα καταγράφουν και οργανώνουν μεταδεδομένα από ιστοσελίδες. Παραδείγματα προτύπων είναι διαθέσιμα στο [αποθετήριο clipper-templates](https://github.com/kepano/clipper-templates).

## Δημιουργία ή επεξεργασία προτύπου

Για να **δημιουργήσετε** ένα πρότυπο, μεταβείτε στις ρυθμίσεις του Web Clipper και κάντε κλικ στο κουμπί **Νέο πρότυπο** στην πλαϊνή μπάρα. Μπορείτε επίσης να **διπλασιάσετε** ένα πρότυπο στο μενού **Περισσότερα** ενεργειών στην επάνω δεξιά γωνία.

Για να **επεξεργαστείτε** ένα πρότυπο, επιλέξτε ένα πρότυπο από την πλαϊνή μπάρα. Οι αλλαγές σας θα αποθηκευτούν αυτόματα.

Τα πρότυπα χρησιμοποιούν [[Μεταβλητές]] και [[Φίλτρα]], τα οποία σας επιτρέπουν να προσαρμόσετε τον τρόπο αποθήκευσης του περιεχομένου.

## Εισαγωγή και εξαγωγή προτύπων Web Clipper

Για να εισαγάγετε ένα πρότυπο:

1. Ανοίξτε την επέκταση και κάντε κλικ στο εικονίδιο γραναζιού **[[Ρυθμίσεις]]**.
2. Μεταβείτε σε οποιοδήποτε πρότυπο στη λίστα.
3. Κάντε κλικ στο **Εισαγωγή** στην επάνω δεξιά γωνία ή σύρετε και αποθέστε τα αρχεία προτύπων `.json` σε οποιοδήποτε σημείο στην περιοχή του προτύπου.

Για να εξαγάγετε ένα πρότυπο, κάντε κλικ στο **Εξαγωγή** στην επάνω δεξιά γωνία. Αυτό θα κατεβάσει το αρχείο προτύπου `.json`. Μπορείτε επίσης να αντιγράψετε τα δεδομένα του προτύπου στο πρόχειρό σας μέσω του μενού **Περισσότερα**.

## Ρυθμίσεις προτύπου

### Συμπεριφορά

Καθορίστε πώς το περιεχόμενο από το Web Clipper θα προστίθεται στο Obsidian:

- **Δημιουργία νέας σημείωσης**
- **Προσθήκη σε υπάρχουσα σημείωση**, στην αρχή ή στο τέλος
- **Προσθήκη στην ημερήσια σημείωση**, στην αρχή ή στο τέλος (απαιτεί το πρόσθετο [[Ημερήσιες σημειώσεις|ημερήσιες σημειώσεις]] να είναι ενεργό)

### Αυτόματη ενεργοποίηση προτύπου

Οι ενεργοποιητές προτύπου σας επιτρέπουν να επιλέγετε αυτόματα ένα πρότυπο βάσει της τρέχουσας διεύθυνσης URL ή δεδομένων [schema.org](https://schema.org/). Μπορείτε να ορίσετε πολλαπλούς κανόνες για κάθε πρότυπο, χωρισμένους σε νέα γραμμή.

Η πρώτη αντιστοιχία στη λίστα προτύπων σας καθορίζει ποιο πρότυπο χρησιμοποιείται. Μπορείτε να σύρετε τα πρότυπα πάνω και κάτω στις ρυθμίσεις του Web Clipper για να αλλάξετε τη σειρά αντιστοίχισης.

#### Απλή αντιστοίχιση URL

Η απλή αντιστοίχιση ενεργοποιεί ένα πρότυπο αν η τρέχουσα διεύθυνση URL *ξεκινά με* το δεδομένο μοτίβο. Για παράδειγμα:

- `https://obsidian.md` θα αντιστοιχηθεί σε οποιαδήποτε διεύθυνση URL που ξεκινά με αυτό το κείμενο.

#### Αντιστοίχιση με κανονική έκφραση

Μπορείτε να ενεργοποιείτε πρότυπα βάσει πιο σύνθετων μοτίβων URL χρησιμοποιώντας κανονικές εκφράσεις. Περικλείστε το μοτίβο κανονικής έκφρασης σε πλάγιες κάθετες (`/`). Θυμηθείτε να χρησιμοποιείτε χαρακτήρα διαφυγής για ειδικούς χαρακτήρες στα μοτίβα κανονικών εκφράσεων (όπως `.` και `/`) με ανάστροφη κάθετο (`\`). Για παράδειγμα:

- `/^https:\/\/www\.imdb\.com\/title\/tt\d+\/reference\/?$/` θα αντιστοιχηθεί σε οποιαδήποτε σελίδα αναφοράς IMDB.

#### Αντιστοίχιση Schema.org

Μπορείτε να ενεργοποιείτε πρότυπα βάσει δεδομένων [schema.org](https://schema.org/) που υπάρχουν στη σελίδα. Χρησιμοποιήστε το πρόθεμα `schema:` ακολουθούμενο από το κλειδί schema που θέλετε να αντιστοιχίσετε. Προαιρετικά μπορείτε να καθορίσετε μια αναμενόμενη τιμή. Για παράδειγμα:

- `schema:@Recipe` θα αντιστοιχηθεί σε σελίδες όπου ο τύπος schema είναι "Recipe".
- `schema:@Recipe.name` θα αντιστοιχηθεί σε σελίδες όπου το `@Recipe.name` υπάρχει.
- `schema:@Recipe.name=Cookie` θα αντιστοιχηθεί σε σελίδες όπου το `@Recipe.name` είναι "Cookie".

Οι τιμές schema.org μπορούν επίσης να χρησιμοποιηθούν για [[Μεταβλητές#Μεταβλητές Schema.org|προ-συμπλήρωση δεδομένων σε πρότυπα]].

### Πλαίσιο διερμηνέα

Όταν ο [[Διερμηνέας|Διερμηνέας]] είναι ενεργοποιημένος, μπορείτε να χρησιμοποιείτε [[Μεταβλητές#Μεταβλητές εντολών|μεταβλητές προτροπής]] για εξαγωγή περιεχομένου σελίδας με φυσική γλώσσα. Για κάθε πρότυπο μπορείτε να ορίσετε το [[Διερμηνέας#Πλαίσιο|πλαίσιο]] στο οποίο έχει πρόσβαση ο Διερμηνέας.
