Εάν αντιμετωπίζετε προβλήματα με το [[Εισαγωγή στο Obsidian Web Clipper|Web Clipper]] μπορείτε να λάβετε βοήθεια μέσω του [επίσημου καναλιού Discord](https://discord.com/channels/686053708261228577/1285652864089198672). Μπορείτε επίσης να αναφέρετε σφάλματα στο [αποθετήριο GitHub](https://github.com/obsidianmd/obsidian-clipper).

## Γενικά

### Κάποιο περιεχόμενο λείπει

Από προεπιλογή, το Web Clipper προσπαθεί να καταγράψει ευφυώς το περιεχόμενο από τη σελίδα. Ωστόσο, μπορεί να μην είναι επιτυχές σε όλες τις ιστοσελίδες.

Το Web Clipper χρησιμοποιεί το [Defuddle](https://github.com/kepano/defuddle) για να καταγράψει μόνο το κύριο περιεχόμενο της σελίδας. Αυτό εξαιρεί κεφαλίδα, υποσέλιδο και άλλα στοιχεία, αλλά μερικές φορές μπορεί να είναι υπερβολικά συντηρητικό και να αφαιρέσει περιεχόμενο που θέλετε να διατηρήσετε. Μπορείτε να [αναφέρετε σφάλματα](https://github.com/kepano/defuddle) στο Defuddle.

Για να παρακάμψετε το Defuddle στο Web Clipper χρησιμοποιήστε τις ακόλουθες μεθόδους:

- Επιλέξτε κείμενο ή χρησιμοποιήστε `Cmd/Ctrl+A` για επιλογή όλου του κειμένου.
- [[Highlighter|Επισημάνετε περιεχόμενο]] για να επιλέξετε ακριβώς τι θέλετε να καταγράψετε.
- Χρησιμοποιήστε ένα [[Obsidian Web Clipper/Πρότυπα|προσαρμοσμένο πρότυπο]] για την ιστοσελίδα.

### Δεν εμφανίζεται περιεχόμενο στο Obsidian

Εάν δεν βλέπετε περιεχόμενο στο Obsidian όταν κάνετε κλικ στο **Προσθήκη στο Obsidian**:

- Ελέγξτε για σφάλματα στην [[Βοήθεια και υποστήριξη#Καταγραφή αρχείων καταγραφής κονσόλας|κονσόλα προγραμματιστή]] του Obsidian.
- Βεβαιωθείτε ότι το όνομα του θησαυροφυλακίου σας στις ρυθμίσεις του Web Clipper ταιριάζει ακριβώς με το *όνομα θησαυροφυλακίου* στο Obsidian και *όχι τη διαδρομή του θησαυροφυλακίου*.
- Ελέγξτε ότι το όνομα φακέλου έχει σωστή μορφοποίηση.

## Linux

#### Το Obsidian δεν ανοίγει

- Βεβαιωθείτε ότι το πρωτόκολλο [[Obsidian URI]] είναι [[Obsidian URI#Εγγραφή Obsidian URI|καταχωρημένο]].
- Εάν χρησιμοποιείτε Firefox μπορεί να χρειαστεί να [το καταχωρήσετε στις ρυθμίσεις του προγράμματος περιήγησης](https://kb.mozillazine.org/Register_protocol).

#### Το Obsidian ανοίγει αλλά αποθηκεύεται μόνο το όνομα αρχείου

Είναι πιθανό ότι το Obsidian δεν μπορεί να αποκτήσει πρόσβαση στο πρόχειρό σας. Η πρόσβαση στο πρόχειρο είναι απαραίτητη για τη μεταφορά δεδομένων από το πρόγραμμα περιήγησής σας στο Obsidian. Η ρύθμιση παραμέτρων σας μπορεί να επηρεάσει τον τρόπο με τον οποίο οι εφαρμογές είναι απομονωμένες σε sandbox, καθώς και τα δικαιώματα πρόχειρου.

Εάν χρησιμοποιείτε Wayland, βεβαιωθείτε ότι το Obsidian έχει τα δικαιώματα να διαβάζει το πρόχειρο όταν η εφαρμογή δεν είναι σε εστίαση. Αυτή η προτίμηση μπορεί να βρίσκεται στον διαχειριστή παραθύρων πλακιδίων σας, π.χ. Hyprland ή Sway.

Εάν χρησιμοποιείτε Hyprland:

```ini
# ~/.config/hypr/hyprland.conf
misc {
    focus_on_activate = true
}
```

Εάν χρησιμοποιείτε Sway:

```ini
# ~/.config/sway/config
for_window [class="obsidian"] focus_on_window_activation focus
```

- Εάν χρησιμοποιείτε Flatpak, δοκιμάστε μια [επίσημα υποστηριζόμενη έκδοση του Obsidian](https://obsidian.md/download).
- Εάν χρησιμοποιείτε KDE, μεταβείτε στις **Ρυθμίσεις Συστήματος** → **Διαχείριση Παραθύρων** → **Κανόνες Παραθύρων** και επιτρέψτε στο Obsidian να λαμβάνει εστίαση, [[web-clipper-kde.png|δείτε στιγμιότυπο οθόνης]].
- Ως εναλλακτική λύση, δοκιμάστε να μεταβείτε σε **Λειτουργία παλαιού τύπου** στις **Ρυθμίσεις Web Clipper** → **Γενικά**. Αυτό θα παρακάμψει το πρόχειρο και θα αποθηκεύσει το περιεχόμενο απευθείας μέσω URI. Σημειώστε ότι αυτό θα περιορίσει τον αριθμό χαρακτήρων που μπορούν να αποκοπούν ανάλογα με το πρόγραμμα περιήγησής σας και τη διανομή Linux.

## iOS και iPadOS

Για να ενεργοποιήσετε την επέκταση Web Clipper για το Safari:

1. Μεταβείτε στο Safari, πατήστε το αριστερότερο κουμπί στη γραμμή URL του προγράμματος περιήγησης, μοιάζει με ορθογώνιο με γραμμές από κάτω.
2. Πατήστε **Διαχείριση Επεκτάσεων**.
3. Ενεργοποιήστε το **Obsidian Web Clipper** στη λίστα Επεκτάσεων.
4. Κλείστε το μενού.
5. Για να χρησιμοποιήσετε την επέκταση **πατήστε το εικονίδιο παζλ** στη γραμμή URL.

Για να επιτρέψετε στο Web Clipper να εκτελείται σε όλες τις ιστοσελίδες:

1. Μεταβείτε στις **[[Ρυθμίσεις]]** iOS → **Εφαρμογές** → **Safari** → **Επεκτάσεις**.
2. Στην ενότητα **Δικαιώματα** επιτρέψτε την εκτέλεση σε όλες τις ιστοσελίδες.

Για να επιτρέψετε στο Obsidian να λαμβάνει πάντα περιεχόμενο από το Web Clipper:

1. Μεταβείτε στις **[[Ρυθμίσεις]]** iOS → **Εφαρμογές** → **Obsidian**.
2. Ορίστε το **Επικόλληση από άλλες εφαρμογές** σε **Αποδοχή**.
