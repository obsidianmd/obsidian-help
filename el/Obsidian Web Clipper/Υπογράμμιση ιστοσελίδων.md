Το [[Εισαγωγή στο Obsidian Web Clipper|Web Clipper]] σας επιτρέπει να επισημαίνετε κείμενο σε ιστοσελίδες και να επιλέγετε τα στοιχεία που θέλετε να αποθηκεύσετε στο Obsidian. Οι επισημάνσεις αποθηκεύονται, ώστε να μπορείτε να τις βλέπετε όταν επιστρέψετε σε μια σελίδα.

Οι επισημάνσεις μπορούν να αποθηκευτούν στο Obsidian χρησιμοποιώντας την επέκταση. Μπορείτε να ορίσετε τις επιλογές του επισημαντή στις ρυθμίσεις της επέκτασης.

## Χρήση του επισημαντή

Ο επισημαντής μπορεί να χρησιμοποιηθεί σε ζωντανές ιστοσελίδες ή στο [[Αναγνώστης]]. Στην προβολή Reader, η δημιουργία μιας επιλογής κειμένου σας δίνει τη δυνατότητα να την επισημάνετε.

Μπορείτε επίσης να προσθέσετε αυτόματα τις επιλογές σας στις επισημάνσεις, ενεργοποιώντας τον επισημαντή:

- ![[lucide-highlighter.svg#icon]] **Εικονίδιο επισημαντή** στον πίνακα της επέκτασης ή στη γραμμή εργαλείων του Reader.
- **Συντόμευση πληκτρολογίου** προσαρμόσιμη στις ρυθμίσεις της επέκτασης.
- **Μενού δεξιού κλικ**, κάνοντας δεξί κλικ στην ιστοσελίδα που επισκέπτεστε.

Μόλις ο επισημαντής είναι ενεργοποιημένος, οποιοδήποτε επιλεγμένο κείμενο, εικόνες και στοιχεία θα προστεθούν στις επισημάνσεις σας. Όλες οι παραπάνω μέθοδοι σας επιτρέπουν επίσης να απενεργοποιήσετε τον επισημαντή.

## Προσθήκη επισημάνσεων σε σημειώσεις

Υπάρχουν τρεις επιλογές για τον τρόπο εισαγωγής των επισημάνσεων στις αποκομμένες σημειώσεις σας:

- **Επισήμανση περιεχομένου σελίδας** — προσθέτει τις επισημάνσεις απευθείας στο κείμενο με τη [[Obsidian Flavored Markdown|σύνταξη]] `==highlight==`.
- **Αντικατάσταση περιεχομένου σημείωσης** — επιστρέφει μια λίστα επισημάνσεων, χωρίς κανένα από το περιεχόμενο της σελίδας.
- **Καμία ενέργεια** — επιστρέφει το αρχικό περιεχόμενο χωρίς επισημάνσεις.
 
Αυτές οι επιλογές αλλάζουν τη [[Μεταβλητές|μεταβλητή]] `{{content}}` στο πρότυπό σας. Μπορείτε επίσης να προσθέσετε επισημάνσεις απευθείας στο πρότυπό σας χρησιμοποιώντας τη μεταβλητή `{{highlights}}`, για παράδειγμα:

```
{{highlights|map: item => item.text|join:"\n\n"}}
```

## Προβολή και αναζήτηση επισημάνσεων

Όλες οι επισημάνσεις που δημιουργείτε μπορούν να προβληθούν στη σελίδα Επισημάνσεων. Μπορείτε να ανοίξετε αυτή τη σελίδα μεταβαίνοντας στις **Ρυθμίσεις** → **Επισημαντής**.

## Εξαγωγή επισημάνσεων

Οι επισημάνσεις μπορούν να εξαχθούν σε αρχείο `.json`, είτε στη σελίδα Ρυθμίσεων είτε στη σελίδα Επισημάνσεων.
