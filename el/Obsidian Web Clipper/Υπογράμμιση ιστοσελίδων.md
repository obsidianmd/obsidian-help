Το [[Εισαγωγή στο Obsidian Web Clipper|Web Clipper]] σας επιτρέπει να επισημαίνετε κείμενο σε ιστοσελίδες και να επιλέγετε τα στοιχεία που θέλετε να αποθηκεύσετε στο Obsidian. Οι επισημάνσεις σας αποθηκεύονται, ώστε να μπορείτε να τις επισκεφτείτε ξανά όταν επιστρέψετε σε μια σελίδα.

Οι επισημάνσεις μπορούν να [[Αποκοπή ιστοσελίδων|αποτυπωθούν]] και να αποθηκευτούν στο Obsidian όταν ανοίξετε την επέκταση.

## Ενεργοποίηση επισημαντή

Μπορείτε να ενεργοποιήσετε την επισήμανση με διάφορους τρόπους, ανάλογα με τον περιηγητή σας:

- Το εικονίδιο του επισημαντή στον πίνακα της επέκτασης.
- Συντομεύσεις πληκτρολογίου, για να ενεργοποιήσετε την επέκταση από το πληκτρολόγιό σας.
- Μενού δεξιού κλικ, κάνοντας δεξί κλικ στην ιστοσελίδα που επισκέπτεστε.

Μόλις η επισήμανση είναι ενεργοποιημένη, μπορείτε να επιλέξετε κείμενο, εικόνες και στοιχεία που θέλετε να επισημάνετε.

## Ρυθμίσεις επισημαντή

Μπορείτε να αλλάξετε τη συμπεριφορά του επισημαντή μεταβαίνοντας στις ρυθμίσεις του Web Clipper. Εδώ μπορείτε επίσης να εξάγετε τις επισημάνσεις σας σε αρχείο `.json`.

Υπάρχουν τρεις επιλογές για τον τρόπο εισαγωγής των επισημάνσεων στην αποκομμένη σημείωσή σας μέσω της [[Μεταβλητές|μεταβλητής]] `{{content}}`:

- **Επισήμανση περιεχομένου σελίδας** — προσθέτει τις επισημάνσεις απευθείας στο κείμενο με τη [[Obsidian Flavored Markdown|σύνταξη]] `==highlight==`.
- **Αντικατάσταση περιεχομένου σημείωσης** — επιστρέφει μια λίστα επισημάνσεων, χωρίς κανένα από το περιεχόμενο της σελίδας.
- **Καμία ενέργεια** — επιστρέφει το αρχικό περιεχόμενο χωρίς επισημάνσεις.

Μπορείτε να προσθέσετε επισημάνσεις απευθείας στο πρότυπό σας χρησιμοποιώντας τη μεταβλητή `{{highlights}}`, για παράδειγμα:

```
{{highlights|map: item => item.text|join:"\n\n"}}
```
