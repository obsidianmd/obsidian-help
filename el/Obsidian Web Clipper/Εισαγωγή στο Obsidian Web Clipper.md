Το Obsidian Web Clipper είναι μια δωρεάν επέκταση προγράμματος περιήγησης που σας επιτρέπει να επισημαίνετε σελίδες και να αποθηκεύετε διαδικτυακό περιεχόμενο στο θησαυροφυλάκιό σας.

## Εγκατάσταση του Web Clipper

Αποκτήστε την επέκταση Obsidian Web Clipper από τον επίσημο κατάλογο για το πρόγραμμα περιήγησής σας:

- [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
	- Για Chrome, Brave, Arc, Orion και άλλα προγράμματα περιήγησης βασισμένα στο Chromium.
- [Firefox Add-Ons](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
	- Για Firefox και Firefox Mobile.
- [Safari Extensions](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
	- Για macOS, iOS και iPadOS.
- [Edge Add-Ons](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
	- Για Microsoft Edge.

## Πώς να χρησιμοποιήσετε το Web Clipper

- [[Αποκοπή ιστοσελίδων]]
	- Ξεκινήστε και μάθετε πώς να αποθηκεύετε περιεχόμενο από σελίδες.
- [[Υπογράμμιση ιστοσελίδων|Επισημαντής]]
	- Επισημάνετε σημαντικά αποσπάσματα και επιλέξτε στοιχεία που θέλετε να αποθηκεύσετε.
- [[Αναγνώστης|Αναγνώστης]]
	- Διαβάστε ιστοσελίδες σε μια καθαρή προβολή χωρίς περισπασμούς.
- [[Ερμηνεία ιστοσελίδων|Διερμηνέας]]
	- Χρησιμοποιήστε προτροπές φυσικής γλώσσας για να καταγράψετε και να τροποποιήσετε δεδομένα σε μια σελίδα.
- [[Αντιμετώπιση προβλημάτων Web Clipper|Αντιμετώπιση προβλημάτων]]
	- Επιλύστε κοινά προβλήματα με το Web Clipper.
- [[Obsidian Web Clipper/Πρότυπα|Πρότυπα]]
	- Δημιουργήστε προσαρμοσμένα πρότυπα για συγκεκριμένες ιστοσελίδες.
- [[Μεταβλητές]]
	- Προσυμπληρώστε δεδομένα σελίδας στο πρότυπό σας.
- [[Φίλτρα]]

- [[Λογική]]
	- Χρησιμοποιήστε συνθήκες, βρόχους και ορίστε μεταβλητές σε πρότυπα.

## Απόρρητο

Το Obsidian Web Clipper αποθηκεύει περιεχόμενο τοπικά στο θησαυροφυλάκιο Obsidian σας και ακολουθεί την [πολιτική απορρήτου](https://obsidian.md/privacy) μας. Τα δεδομένα σας δεν συλλέγονται και δεν συγκεντρώνουμε κανένα μετρικό χρήσης. Ο κώδικας είναι [ανοιχτού κώδικα](https://github.com/obsidianmd/obsidian-clipper) και ελέγξιμος.
