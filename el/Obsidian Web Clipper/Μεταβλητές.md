Τα [[Obsidian Web Clipper/Πρότυπα|πρότυπα Web Clipper]] μπορούν να χρησιμοποιούν μεταβλητές για την αυτόματη προσυμπλήρωση δεδομένων από τη σελίδα σε ένα πρότυπο. Οι μεταβλητές μπορούν να χρησιμοποιηθούν στο **όνομα σημείωσης**, στην **τοποθεσία σημείωσης**, στις **ιδιότητες** και στο **περιεχόμενο σημείωσης**. Οι μεταβλητές μπορούν επίσης να τροποποιηθούν χρησιμοποιώντας [[Φίλτρα|φίλτρα]].

Χρησιμοποιήστε το εικονίδιο `...` στην επέκταση [[Εισαγωγή στο Obsidian Web Clipper|Web Clipper]] για πρόσβαση στις μεταβλητές της τρέχουσας σελίδας προς χρήση σε πρότυπα. Υπάρχουν πέντε τύποι μεταβλητών που μπορείτε να χρησιμοποιήσετε:

- [[Μεταβλητές#Προκαθορισμένες μεταβλητές|Προκαθορισμένες μεταβλητές]]
- [[Μεταβλητές#Μεταβλητές εντολών|Μεταβλητές εντολών]]
- [[Μεταβλητές#Μεταβλητές meta|Μεταβλητές meta]]
- [[Μεταβλητές#Μεταβλητές επιλογέα|Μεταβλητές επιλογέα]]
- [[Μεταβλητές#Μεταβλητές Schema.org|Μεταβλητές Schema.org]]

## Προκαθορισμένες μεταβλητές

Οι προκαθορισμένες μεταβλητές δημιουργούνται αυτόματα με βάση το περιεχόμενο της σελίδας. Αυτές συνήθως λειτουργούν για τους περισσότερους ιστοτόπους.

Η κύρια μεταβλητή περιεχομένου είναι η `{{content}}`, η οποία περιέχει το περιεχόμενο του άρθρου, ή τις [[Υπογράμμιση ιστοσελίδων|επισημάνσεις]], ή την επιλογή εάν υπάρχει επιλεγμένο κείμενο στη σελίδα. Σημειώστε ότι η `{{content}}` επιχειρεί να εξαγάγει το κύριο περιεχόμενο της σελίδας, το οποίο μπορεί να μην είναι πάντα αυτό που θέλετε. Σε αυτήν την περίπτωση, μπορείτε να χρησιμοποιήσετε άλλες προκαθορισμένες μεταβλητές ή μεταβλητές επιλογέα για να εξαγάγετε το περιεχόμενο που χρειάζεστε.

| Μεταβλητή            | Περιγραφή                                                                                |
| ------------------- | -------------------------------------------------------------------------------------- |
| `{{author}}`        | Συγγραφέας της σελίδας                                                                     |
| `{{content}}`       | Περιεχόμενο άρθρου, [[Highlighter\|επισημάνσεις]] ή επιλογή, σε μορφή Markdown |
| `{{contentHtml}}`   | Περιεχόμενο άρθρου, [[Highlighter\|επισημάνσεις]] ή επιλογή, σε μορφή HTML     |
| `{{date}}`          | Τρέχουσα ημερομηνία, μπορεί να μορφοποιηθεί χρησιμοποιώντας το φίλτρο `date`                                 |
| `{{description}}`   | Περιγραφή ή απόσπασμα                                                                 |
| `{{domain}}`        | Domain                                                                                 |
| `{{favicon}}`       | URL favicon                                                                            |
| `{{fullHtml}}`      | Μη επεξεργασμένο HTML για το πλήρες περιεχόμενο της σελίδας                                             |
| `{{highlights}}`    | [[Highlighter\|Επισημάνσεις]] με κείμενο και χρονικές σημάνσεις                           |
| `{{image}}`         | URL εικόνας κοινωνικής κοινοποίησης                                                                 |
| `{{published}}`     | Ημερομηνία δημοσίευσης, μπορεί να μορφοποιηθεί χρησιμοποιώντας το φίλτρο `date`                               |
| `{{selection}}`     | Επιλογή σε μορφή Markdown                                                           |
| `{{selectionHtml}}` | Επιλογή σε μορφή HTML                                                               |
| `{{site}}`          | Όνομα ιστοτόπου ή εκδότης                                                                 |
| `{{title}}`         | Τίτλος της σελίδας                                                                      |
| `{{time}}`          | Τρέχουσα ημερομηνία και ώρα                                                                  |
| `{{url}}`           | Τρέχον URL                                                                            |
| `{{words}}`         | Αρίθμηση λέξεων                                                                             |

## Μεταβλητές εντολών

Οι μεταβλητές εντολών αξιοποιούν γλωσσικά μοντέλα για εξαγωγή και τροποποίηση δεδομένων χρησιμοποιώντας φυσική γλώσσα. Οι μεταβλητές εντολών απαιτούν να είναι ενεργοποιημένος και ρυθμισμένος ο [[Διερμηνέας|Διερμηνέας]].

Οι μεταβλητές εντολών χρησιμοποιούν τη σύνταξη `{{"μια σύνοψη της σελίδας"}}`. Τα διπλά εισαγωγικά γύρω από την εντολή είναι σημαντικά και διακρίνουν τις εντολές από τις προκαθορισμένες μεταβλητές. Οι απαντήσεις εντολών μπορούν να υποστούν μετεπεξεργασία με [[Φίλτρα|φίλτρα]], π.χ. `{{"μια σύνοψη της σελίδας"|blockquote}}`.

### Πότε να χρησιμοποιείτε μεταβλητές εντολών

Οι μεταβλητές εντολών έχουν το πλεονέκτημα ότι είναι εξαιρετικά ευέλικτες και εύκολες στη σύνταξη, ωστόσο συνοδεύονται από αρκετούς συμβιβασμούς: είναι πιο αργές στην εκτέλεση και μπορεί να έχουν ζητήματα κόστους και απορρήτου ανάλογα με τον [[Διερμηνέας#Μοντέλα|πάροχο]] που επιλέγετε.

Σε αντίθεση με άλλους τύπους μεταβλητών, οι μεταβλητές εντολών πρέπει να επεξεργαστούν από ένα εξωτερικό γλωσσικό μοντέλο, οπότε αντικαθίστανται μόνο αφού εκτελεστεί ο [[Διερμηνέας|Διερμηνέας]].

Είναι καλύτερο να *μην* χρησιμοποιείτε μεταβλητές εντολών εάν τα δεδομένα που θέλετε να εξαγάγετε βρίσκονται σε συνεπή μορφή που θα μπορούσε να εξαχθεί με άλλους τύπους μεταβλητών.

Από την άλλη πλευρά, οι μεταβλητές εντολών μπορεί να είναι χρήσιμες εάν τα δεδομένα που θέλετε να εξαγάγετε βρίσκονται σε *ασυνεπή* μορφή σε διάφορους ιστοτόπους. Για παράδειγμα, μπορείτε να δημιουργήσετε ένα [[Obsidian Web Clipper/Πρότυπα|πρότυπο]] για αποθήκευση βιβλίων που είναι ανεξάρτητο από τον ιστότοπο βιβλίων. Μεταβλητές εντολών όπως `{{"συγγραφέας του βιβλίου"}}` θα λειτουργούν σε οποιονδήποτε ιστότοπο βιβλίων, ενώ οι μεταβλητές επιλογέα συνήθως λειτουργούν μόνο για έναν ιστότοπο.

### Παραδείγματα

Οι εντολές μπορούν να χρησιμοποιήσουν σχεδόν οποιοδήποτε ερώτημα φυσικής γλώσσας. Ανάλογα με το μοντέλο που χρησιμοποιείτε, οι εντολές μπορούν να αναζητούν ή να μεταφράζουν δεδομένα σε διάφορες γλώσσες.

- `{{"μια σύνοψη τριών σημείων με κουκκίδες, μεταφρασμένη στα γαλλικά"}}` για εξαγωγή σημείων με κουκκίδες σχετικά με τη σελίδα και μετάφρασή τους στα γαλλικά.
- `{{"un resumé de la page en trois points"}}` για εξαγωγή τριών σημείων με κουκκίδες χρησιμοποιώντας εντολή στα γαλλικά.

Οι εντολές μπορούν να μετατρέψουν το περιεχόμενο της σελίδας σε JSON που μπορεί να χειριστεί με [[Φίλτρα|φίλτρα]]. Για παράδειγμα:

```
{{"return a JSON object for each tweet, that includes the author, tweet_text, date in YYYY-MM-DD format, and images array (if there are any)"|map:tweet => ({text: tweet.tweet_text, author: tweet.author, date: tweet.date})|template:"${text}\n— [[@${author}]], [[${date}]]\n"}}
```


## Μεταβλητές meta

Οι μεταβλητές meta σας επιτρέπουν να εξαγάγετε δεδομένα από [στοιχεία meta](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/meta) στη σελίδα, συμπεριλαμβανομένων δεδομένων [Open Graph](https://ogp.me/) που χρησιμοποιούνται για τη συμπλήρωση προεπισκοπήσεων κοινωνικής κοινοποίησης.

- Η `{{meta:name}}` επιστρέφει το περιεχόμενο της ετικέτας meta name με το δοθέν όνομα, π.χ. `{{meta:name:description}}` για την ετικέτα meta `description`.
- Η `{{meta:property}}` επιστρέφει το περιεχόμενο της ετικέτας meta property με τη δοθείσα ιδιότητα, π.χ. `{{meta:property:og:title}}` για την ετικέτα meta `og:title`.

## Μεταβλητές επιλογέα

Οι μεταβλητές επιλογέα σας επιτρέπουν να εξαγάγετε κείμενο από στοιχεία στη σελίδα χρησιμοποιώντας [επιλογείς CSS](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_selectors/Selectors_and_combinators).

Η σύνταξη είναι `{{selector:cssSelector?attribute}}`, όπου το `?attribute` είναι προαιρετικό. Εάν δεν καθοριστεί χαρακτηριστικό, επιστρέφεται το περιεχόμενο κειμένου του στοιχείου. Μπορείτε επίσης να χρησιμοποιήσετε `{{selectorHtml:cssSelector}}` για να λάβετε το περιεχόμενο HTML του στοιχείου. Οι μεταβλητές επιλογέα τείνουν να λειτουργούν καλύτερα σε έναν συγκεκριμένο ιστότοπο ή σύνολο ιστοτόπων που έχουν συνεπή δομή HTML.

- Η `{{selector:h1}}` επιστρέφει το περιεχόμενο κειμένου οποιωνδήποτε στοιχείων `h1` στη σελίδα.
- Η `{{selector:.author}}` επιστρέφει το περιεχόμενο κειμένου οποιωνδήποτε στοιχείων `.author` στη σελίδα.
- Η `{{selector:img.hero?src}}` επιστρέφει το χαρακτηριστικό `src` της εικόνας με κλάση `hero`.
- Η `{{selector:a.main-link?href}}` επιστρέφει το χαρακτηριστικό `href` της ετικέτας αγκύρωσης με κλάση `main-link`.
- Η `{{selectorHtml:body|markdown}}` επιστρέφει ολόκληρο το HTML του στοιχείου `body`, μετατρεπόμενο σε Markdown χρησιμοποιώντας το [[Φίλτρα#Επεξεργασία HTML|φίλτρο]] `markdown`.
- Υποστηρίζονται ένθετοι επιλογείς CSS και συνδυαστές εάν χρειάζεστε μεγαλύτερη εξειδίκευση.
- Εάν πολλαπλά στοιχεία ταιριάζουν με τον επιλογέα, επιστρέφεται ένας πίνακας, τον οποίο μπορείτε να επεξεργαστείτε με [[Φίλτρα#Πίνακες και αντικείμενα|φίλτρα πινάκων και αντικειμένων]] όπως `join` ή `map`.

Οι μεταβλητές επιλογέα μπορούν επίσης να χρησιμοποιηθούν απευθείας στη [[Λογική|λογική προτύπων]]:

- Σε βρόχους: `{% for comment in selector:.comment %}...{% endfor %}`
- Σε συνθήκες: `{% if selector:.premium-badge %}...{% endif %}`
- Σε ανάθεση μεταβλητών: `{% set items = selector:.list-item %}`

## Μεταβλητές Schema.org

Οι μεταβλητές schema σας επιτρέπουν να εξαγάγετε δεδομένα από [schema.org](https://schema.org/) JSON-LD στη σελίδα. Τα δεδομένα schema.org μπορούν επίσης να χρησιμοποιηθούν για αυτόματη [[Obsidian Web Clipper/Πρότυπα#Αντιστοίχιση Schema.org|ενεργοποίηση προτύπου]].

- Η `{{schema:@Type:key}}` επιστρέφει την τιμή του κλειδιού από το schema.
- Η `{{schema:@Type:parent.child}}` επιστρέφει την τιμή μιας ένθετης ιδιότητας.
- Η `{{schema:@Type:arrayKey}}` επιστρέφει το πρώτο στοιχείο ενός πίνακα.
- Η `{{schema:@Type:arrayKey[index].property}}` επιστρέφει το στοιχείο στο καθορισμένο ευρετήριο ενός πίνακα.
- Η `{{schema:@Type:arrayKey[*].property}}` επιστρέφει μια συγκεκριμένη ιδιότητα από όλα τα στοιχεία ενός πίνακα.

Μπορείτε επίσης να χρησιμοποιήσετε μια συντομογραφία χωρίς να καθορίσετε τον τύπο schema:

- Η `{{schema:author}}` θα αντιστοιχίσει την πρώτη ιδιότητα `author` που βρέθηκε σε οποιονδήποτε τύπο schema.
- Η `{{schema:name}}` θα αντιστοιχίσει την πρώτη ιδιότητα `name` που βρέθηκε σε οποιονδήποτε τύπο schema.

Αυτή η συντομογραφία είναι ιδιαίτερα χρήσιμη όταν δεν γνωρίζετε ή δεν σας ενδιαφέρει ο συγκεκριμένος τύπος schema, αλλά γνωρίζετε το όνομα της ιδιότητας που αναζητάτε.

Οι ένθετες ιδιότητες και η πρόσβαση σε πίνακες λειτουργούν επίσης, τόσο με όσο και χωρίς καθορισμένο τον `@Type` του schema:

- Η `{{schema:author.name}}` θα βρει την πρώτη ιδιότητα `author` και στη συνέχεια θα αποκτήσει πρόσβαση στην υπο-ιδιότητα `name`.
- Η `{{schema:author[0].name}}` θα αποκτήσει πρόσβαση στο `name` του πρώτου συγγραφέα σε έναν πίνακα συγγραφέων.
- Η `{{schema:author[*].name}}` θα επιστρέψει έναν πίνακα με όλα τα ονόματα συγγραφέων.
