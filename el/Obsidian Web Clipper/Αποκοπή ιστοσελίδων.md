Μόλις εγκαταστήσετε την επέκταση προγράμματος περιήγησης [[Εισαγωγή στο Obsidian Web Clipper|Web Clipper]], μπορείτε να έχετε πρόσβαση σε αυτήν με διάφορους τρόπους, ανάλογα με το πρόγραμμα περιήγησής σας:

1. Το εικονίδιο Obsidian στη γραμμή εργαλείων του προγράμματος περιήγησής σας.
2. Συντομεύσεις πληκτρολογίου, για ενεργοποίηση της επέκτασης από το πληκτρολόγιο.
3. Μενού περιβάλλοντος, κάνοντας δεξί κλικ στην ιστοσελίδα που επισκέπτεστε.

Για να αποθηκεύσετε μια σελίδα στο Obsidian κάντε κλικ στο κουμπί **Προσθήκη στο Obsidian**.

## Καταγραφή σελίδας

Όταν ανοίγετε την επέκταση, το Web Clipper εξάγει δεδομένα από την τρέχουσα ιστοσελίδα ακολουθώντας τις ρυθμίσεις στο [[Obsidian Web Clipper/Πρότυπα|πρότυπό]] σας. Μπορείτε να δημιουργήσετε τα δικά σας πρότυπα και να προσαρμόσετε την έξοδο χρησιμοποιώντας [[Μεταβλητές|μεταβλητές]] και [[Φίλτρα|φίλτρα]].

Από προεπιλογή, το Web Clipper προσπαθεί να εξάγει ευφυώς μόνο το κύριο περιεχόμενο του άρθρου, εξαιρώντας άλλα στοιχεία της σελίδας. Ωστόσο, μπορείτε να παρακάμψετε αυτή τη συμπεριφορά με τους ακόλουθους τρόπους:

- Εάν υπάρχει προσαρμοσμένο πρότυπο, χρησιμοποιεί το πρότυπό σας.
- Εάν υπάρχει επιλογή, χρησιμοποιεί την επιλογή. Μπορείτε να χρησιμοποιήσετε `Ctrl/Cmd+A` για να επιλέξετε ολόκληρη τη σελίδα.
- Εάν υπάρχουν [[Υπογράμμιση ιστοσελίδων|επισημάνσεις]], χρησιμοποιεί τις επισημάνσεις.

## Λήψη εικόνων

Οι εικόνες δεν λαμβάνονται αυτόματα όταν χρησιμοποιείτε το Web Clipper. Αντί αυτού, οι εικόνες συνδέονται με το URL τους στο διαδίκτυο. Αυτό εξοικονομεί χώρο στο θησαυροφυλάκιό σας, αλλά σημαίνει ότι οι εικόνες δεν θα είναι προσβάσιμες χωρίς σύνδεση ή εάν το URL σταματήσει να λειτουργεί.

Μπορείτε να κατεβάσετε εικόνες για οποιοδήποτε αρχείο στο Obsidian χρησιμοποιώντας την [[Παλέτα εντολών|εντολή]] με το όνομα **Λήψη συνημμένων για το τρέχον αρχείο**. Αυτή η εντολή μπορεί επίσης να αντιστοιχιστεί σε μια συντόμευση πληκτρολογίου στο Obsidian.

## Συντομεύσεις πληκτρολογίου

Το Web Clipper περιλαμβάνει συντομεύσεις πληκτρολογίου που μπορείτε να χρησιμοποιήσετε για να επιταχύνετε τη ροή εργασίας σας. Για να αλλάξετε τις αντιστοιχίσεις πλήκτρων, μεταβείτε στις **Ρυθμίσεις Web Clipper** → **Γενικά** και ακολουθήστε τις οδηγίες για το πρόγραμμα περιήγησής σας. Οι αντιστοιχίσεις μπορούν να αλλάξουν σε όλα τα προγράμματα περιήγησης εκτός από το Safari, το οποίο δεν υποστηρίζει επεξεργασία συντομεύσεων.

| Ενέργεια                   | macOS          | Windows/Linux   |
| -------------------------- | -------------- | --------------- |
| Άνοιγμα clipper            | `Cmd+Shift+O`  | `Ctrl+Shift+O`  |
| Γρήγορη αποκοπή           | `Opt+Shift+O`  | `Alt+Shift+O`   |
| Εναλλαγή επισημαντή       | `Opt+Shift+H`  | `Alt+Shift+H`   |

## Λειτουργικότητα διεπαφής

Η διεπαφή του Web Clipper χωρίζεται σε τέσσερις ενότητες:

1. **Κεφαλίδα** όπου μπορείτε να αλλάξετε πρότυπα, να ενεργοποιήσετε την [[Υπογράμμιση ιστοσελίδων|επισήμανση]], τη [[Αναγνώστης|λειτουργία ανάγνωσης]] και να αποκτήσετε πρόσβαση στις ρυθμίσεις.
2. **Ιδιότητες** εμφανίζει τα [[Ιδιότητες|μεταδεδομένα]] που εξήχθησαν από τη σελίδα και θα αποθηκευτούν ως [[Ιδιότητες]] στο Obsidian.
3. **Περιεχόμενο σημείωσης** που θα αποθηκευτεί στο Obsidian.
4. **Υποσέλιδο** σας επιτρέπει να επιλέξετε το θησαυροφυλάκιο και τον φάκελο, και να προσθέσετε στο Obsidian.

Η λειτουργικότητα της κεφαλίδας περιλαμβάνει:

- Αναπτυσσόμενο μενού **Πρότυπο** για εναλλαγή μεταξύ των αποθηκευμένων [[Obsidian Web Clipper/Πρότυπα|προτύπων]] σας που έχουν προστεθεί στις ρυθμίσεις του Web Clipper.
- Κουμπί **Περισσότερα (...)** για εμφάνιση μεταβλητών σελίδας που μπορείτε να χρησιμοποιήσετε σε πρότυπα.
- Κουμπί **Επισημαντής** για ενεργοποίηση της [[Υπογράμμιση ιστοσελίδων|επισήμανσης]].
- Κουμπί **Γρανάζι** για άνοιγμα των ρυθμίσεων του Web Clipper.

Η λειτουργικότητα του υποσέλιδου περιλαμβάνει:

- Κουμπί **Προσθήκη στο Obsidian** για αποθήκευση δεδομένων στο Obsidian.
- Αναπτυσσόμενο μενού **Θησαυροφυλάκιο** για εναλλαγή μεταξύ αποθηκευμένων θησαυροφυλακίων που έχουν προστεθεί στις ρυθμίσεις του Web Clipper.
- Πεδίο **Φάκελος** για να ορίσετε σε ποιον φάκελο θα αποθηκευτεί.
- **Διερμηνέας** για εκτέλεση [[Ερμηνεία ιστοσελίδων|εντολών φυσικής γλώσσας]] στη σελίδα.
