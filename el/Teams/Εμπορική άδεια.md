Το Obsidian είναι δωρεάν στη χρήση και υποστηρίζεται 100% από τους χρήστες. Οι οργανισμοί μπορούν να επιλέξουν να υποστηρίξουν το Obsidian αγοράζοντας εμπορικές άδειες.

Οι οργανισμοί που υποστηρίζουν το Obsidian μπορούν να προβληθούν στη [σελίδα Enterprise](https://obsidian.md/enterprise/) αγοράζοντας 25 ή περισσότερες άδειες.

Οι εμπορικές άδειες **δεν** είναι επιλέξιμες για επιστροφή χρημάτων σύμφωνα με την [[Πολιτική επιστροφής χρημάτων]] μας.

## Αγορά εμπορικής άδειας

Το Obsidian προσφέρει δύο μεθόδους για την [αγορά](https://obsidian.md/pricing) εμπορικής άδειας.

**Αγορά μέσω λογαριασμού Obsidian**

1. Συνδεθείτε ή δημιουργήστε [τον λογαριασμό σας στο Obsidian](https://obsidian.md/account/commercial).
2. Μεταβείτε στην ενότητα **Άδεια**.
3. Κάντε κλικ στο **Αγορά άδειας** και επιλέξτε τον αριθμό θέσεων προς αγορά.
4. Ολοκληρώστε τη διαδικασία πληρωμής.
5. Το κλειδί προϊόντος σας θα εμφανιστεί και μπορεί να ενεργοποιηθεί ακολουθώντας τα παρακάτω βήματα.

Αν χρειάζεστε τιμολόγιο για αυτή τη συναλλαγή, μπορείτε να [[Εισαγωγή στις άδειες και πληρωμές#Δημιουργία τιμολογίου|δημιουργήσετε ένα τιμολόγιο]].

**Αγορά μέσω εντολής αγοράς**

Για αγορές μέσω εντολής αγοράς (PO), επικοινωνήστε με την [[Βοήθεια και υποστήριξη#Επικοινωνία με την υποστήριξη Obsidian|υποστήριξη Obsidian]].

Η εντολή αγοράς σας πρέπει να περιλαμβάνει:

- Επωνυμία της εταιρείας σας (όνομα «χρέωσης σε»)
- Διεύθυνση χρέωσης
- Αριθμός ΦΠΑ (προαιρετικό)
- Όροι πληρωμής (προεπιλογή: NET 30 ημέρες)
- Αριθμός PO/Αναφοράς
- Ηλεκτρονικό ταχυδρομείο παράδοσης αδειών
- Ηλεκτρονικό ταχυδρομείο τιμολογίων για PO
- **Προτιμώμενη μέθοδος πληρωμής**: Τραπεζικό έμβασμα/ACH, Πιστωτική κάρτα ή PayPal

> [!info] Η αγορά εμπορικής άδειας δεν περιλαμβάνει [Obsidian Sync](https://obsidian.md/sync), [Obsidian Publish](https://obsidian.md/publish) ή [άδεια Catalyst](https://obsidian.md/account/catalyst).

## Εφαρμογή εμπορικής άδειας

Για να εφαρμόσετε προαιρετικά μια εμπορική άδεια στην εγκατάσταση του Obsidian σας, ακολουθήστε τα παρακάτω βήματα. Σημειώστε ότι η εφαρμογή εμπορικής άδειας δεν παρέχει κανένα λειτουργικό πλεονέκτημα εντός της εφαρμογής.

1. Ανοίξτε την εφαρμογή Obsidian.
2. Μεταβείτε στις **[[Ρυθμίσεις]] → Γενικά → Εμπορική άδεια**.
3. Κάντε κλικ στο **Ενεργοποίηση** και εισάγετε το κλειδί προϊόντος σας.
4. Ολοκληρώθηκε! Η εφαρμογή είναι πλέον αδειοδοτημένη για εμπορική χρήση.

## Υποστηρικτικοί οργανισμοί

Αν ο οργανισμός σας έχει αγοράσει τουλάχιστον 25 εμπορικές άδειες, μπορείτε να ζητήσετε να προβληθείτε ως [υποστηρικτικός οργανισμός](https://obsidian.md/enterprise/). Αν θέλετε ο οργανισμός σας να προστεθεί ή να αφαιρεθεί από τη σελίδα, παρακαλούμε [[Βοήθεια και υποστήριξη#Επικοινωνία με την υποστήριξη Obsidian|επικοινωνήστε με την υποστήριξη Obsidian]].

Το λογότυπό σας πρέπει να παρέχεται ως αρχείο SVG, να καλύπτει ένα τετράγωνο 32 επί 32 pixel και να χρησιμοποιεί μόνο λευκό χρώμα, χωρίς χρώμα φόντου.

## Συχνές ερωτήσεις

**Μπορώ να αγοράσω μια άδεια για τον εαυτό μου ή πρέπει να ζητήσω από την εταιρεία μου να την αγοράσει;**
Ναι. Μπορείτε να αγοράσετε μια άδεια για τον εαυτό σας μεταβαίνοντας στον [Πίνακα ελέγχου λογαριασμού](https://obsidian.md/account/commercial). Οι περισσότερες εταιρείες σας επιτρέπουν να υποβάλετε αναφορά εξόδων για την αγορά.

Ορισμένες εταιρείες μπορεί να έχουν διαφορετική διαδικασία προμηθειών. Επικοινωνήστε με το τμήμα πληροφορικής σας για να καθορίσετε τις συγκεκριμένες διαδικασίες τους.

**Ποια επωνυμία πρέπει να χρησιμοποιήσω αν αγοράζω την άδεια μόνος/η μου;**
Χρησιμοποιήστε την επωνυμία της εταιρείας σας για την άδεια αν αναμένετε να αποζημιωθείτε από την εταιρεία σας.

Διαφορετικά, χρησιμοποιήστε το νόμιμο όνομά σας ως επωνυμία.

**Μπορώ να αγοράσω εμπορική άδεια για πολλούς υπαλλήλους;**
Ναι. Μπορείτε να αγοράσετε πολλαπλές θέσεις εμπορικής άδειας μέσω του [Πίνακα ελέγχου λογαριασμού](https://obsidian.md/account/commercial). Δεν χρειάζεται κάθε υπάλληλος να έχει δικό του λογαριασμό Obsidian, μπορείτε να χρησιμοποιήσετε έναν μοναδικό λογαριασμό διαχειριστή για να αγοράσετε όλες τις θέσεις.

Μετά την αγορά της άδειας, θα λάβετε ένα κλειδί προϊόντος το οποίο μπορείτε να στείλετε στους υπαλλήλους για να ενεργοποιήσουν την άδειά τους — δείτε τις παραπάνω οδηγίες για την ενεργοποίηση της άδειας.

**Είναι υποχρεωτικό να αγοράσω εμπορική άδεια για κάθε υπάλληλο;**
Όχι. Το Obsidian είναι 100% δωρεάν στη χρήση και επομένως δεν υπάρχει ελάχιστος αριθμός θέσεων που απαιτείται για αγορά.

**Μπορώ να χρησιμοποιήσω την εμπορική άδεια που αγόρασα τόσο στο εταιρικό τηλέφωνό μου όσο και στον εταιρικό υπολογιστή μου;**
Ναι. Η εμπορική άδεια χρεώνεται ανά αριθμό θέσεων, όχι ανά συσκευή. Αυτό σημαίνει ότι μπορείτε να χρησιμοποιήσετε το ίδιο κλειδί προϊόντος σε όλες τις ατομικές εργασιακές σας συσκευές.

**Εργάζομαι σε περιβάλλον υψηλής ασφάλειας, σε απομονωμένο δίκτυο. Πώς μπορώ να εφαρμόσω την εμπορική μου άδεια;**
Σε τέτοια εξειδικευμένα περιβάλλοντα, δεν χρειάζεται να εφαρμόσετε την εμπορική άδεια απευθείας στην εγκατάστασή σας.
