Μάθετε πώς να επεκτείνετε το Obsidian με πρόσθετα που δημιουργήθηκαν από την κοινότητα. Χρησιμοποιήστε πρόσθετα για να προσαρμόσετε το Obsidian στις συγκεκριμένες ανάγκες σας, όπως υποστήριξη πρόσθετων μορφών αρχείων ή ενσωμάτωση με υπηρεσία τρίτων.

> [!warning] Προειδοποίηση
> Τα πρόσθετα κοινότητας εκτελούν κώδικα τρίτων για λογαριασμό σας που θα μπορούσε ενδεχομένως να προκαλέσει ζημιά. Για να μάθετε περισσότερα σχετικά με το τι κάνει η ομάδα του Obsidian για να αποτρέψει επιβλαβή πρόσθετα, ανατρέξτε στην ενότητα [[Ασφάλεια πρόσθετων]].

## Περιήγηση στα πρόσθετα κοινότητας

1. Ανοίξτε τις **[[Ρυθμίσεις]]**.
2. Επιλέξτε **Ενεργοποιήστε τα πρόσθετα κοινότητας**. Για περισσότερες πληροφορίες, ανατρέξτε στην ενότητα [[Ασφάλεια πρόσθετων#Περιορισμένη λειτουργία|Περιορισμένη λειτουργία]].
3. Επιλέξτε **Περιήγηση** για να εμφανιστούν όλα τα διαθέσιμα πρόσθετα κοινότητας.

Χρησιμοποιήστε το πλαίσιο κειμένου για να φιλτράρετε τα πρόσθετα βάσει ονόματος, δημιουργού και περιγραφής.

Μπορείτε επίσης να περιηγηθείτε στα διαθέσιμα πρόσθετα μέσω του προγράμματος περιήγησής σας, μεταβαίνοντας στο [obsidian.md/plugins](https://obsidian.md/plugins).

## Εγκατάσταση πρόσθετου κοινότητας

Για να εγκαταστήσετε ένα πρόσθετο κοινότητας, πρέπει πρώτα να απενεργοποιήσετε την [[Ασφάλεια πρόσθετων#Περιορισμένη λειτουργία|Περιορισμένη λειτουργία]].

1. Ανοίξτε τις **[[Ρυθμίσεις]]**.
2. Στο πλαϊνό μενού, επιλέξτε **Πρόσθετα κοινότητας**.
3. Επιλέξτε **Περιήγηση** για να εξερευνήσετε τα διαθέσιμα πρόσθετα κοινότητας.
4. Επιλέξτε το πρόσθετο που θέλετε να εγκαταστήσετε.
5. Επιλέξτε **Εγκατάσταση**.

Για να χρησιμοποιήσετε ένα εγκατεστημένο πρόσθετο, πρέπει να το ενεργοποιήσετε.

## Ενεργοποίηση πρόσθετου κοινότητας

Είτε επιλέξτε **Ενεργοποίηση** αμέσως μετά την εγκατάσταση, είτε ενεργοποιήστε το από τη λίστα των πρόσθετων κοινότητας στις **[[Ρυθμίσεις]] → Πρόσθετα κοινότητας → Εγκατεστημένο πρόσθετο**.

## Ενημέρωση πρόσθετων

Για λόγους ασφαλείας, τα πρόσθετα κοινότητας δεν ενημερώνονται αυτόματα. Μπορείτε είτε να ενημερώσετε όλα τα πρόσθετα ταυτόχρονα, είτε να ενημερώσετε μεμονωμένα πρόσθετα.

Για να ενημερώσετε όλα τα πρόσθετά σας:

1. Ανοίξτε τις **[[Ρυθμίσεις]]**.
2. Στην ενότητα **Πρόσθετα κοινότητας → Τωρινά πρόσθετα**, επιλέξτε **Έλεγχος για ενημερώσεις**.
3. Αν υπάρχουν διαθέσιμες ενημερώσεις, επιλέξτε **Ενημέρωση όλων**.

Για να ενημερώσετε ένα μεμονωμένο πρόσθετο:

1. Ανοίξτε τις **[[Ρυθμίσεις]]**.
2. Στην ενότητα **Πρόσθετα κοινότητας → Τωρινά πρόσθετα**, επιλέξτε **Έλεγχος για ενημερώσεις**.
3. Στην ενότητα **Εγκατεστημένο πρόσθετο**, επιλέξτε **Ενημέρωση** δίπλα στο πρόσθετο που θέλετε να ενημερώσετε.

## Απεγκατάσταση πρόσθετου κοινότητας

1. Ανοίξτε τις **[[Ρυθμίσεις]]**.
2. Στο πλαϊνό μενού, επιλέξτε **Πρόσθετα κοινότητας**.
3. Στην ενότητα **Εγκατεστημένο πρόσθετο**, επιλέξτε το εικονίδιο κάδου απορριμμάτων ![[lucide-trash-2.svg#icon]] δίπλα στο πρόσθετο που θέλετε να απεγκαταστήσετε.
4. Επιλέξτε **Απεγκατάσταση** για επιβεβαίωση.

## Διαχείριση εγκατεστημένων πρόσθετων

Στις **[[Ρυθμίσεις]] → Πρόσθετα κοινότητας → Εγκατεστημένο πρόσθετο**, κάθε πρόσθετο διαθέτει αρκετά εικονίδια για τη διαχείρισή του:

- **[[Ρυθμίσεις]]** ![[lucide-settings.svg#icon]] — Ανοίξτε τη σελίδα ρυθμίσεων του πρόσθετου αν διαθέτει παραμετροποιήσιμες επιλογές.
- **[[Συντομεύσεις]]** ![[lucide-plus-circle.svg#icon]] — Ορίστε συντομεύσεις πληκτρολογίου για τις εντολές του πρόσθετου.
- **Χρηματοδότηση** ![[lucide-heart.svg#icon]] — Δείτε τη διεύθυνση URL χρηματοδότησης του δημιουργού του πρόσθετου για να υποστηρίξετε τη δουλειά του.
- **Απεγκατάσταση** ![[lucide-trash-2.svg#icon]] — Αφαιρέστε το πρόσθετο από το θησαυροφυλάκιό σας.
- **Εναλλαγή** — Ενεργοποιήστε ή απενεργοποιήστε το πρόσθετο χωρίς να το απεγκαταστήσετε.

Μπορείτε επίσης:

- Να επιλέξετε το εικονίδιο **ανανέωσης** για επαναφόρτωση όλων των πρόσθετων.
- Να επιλέξετε το εικονίδιο **φακέλου** ![[lucide-folder-open.svg#icon]] για να ανοίξετε τον φάκελο πρόσθετων στον [[Φάκελος ρυθμίσεων|φάκελο ρυθμίσεών]] σας.
- Να χρησιμοποιήσετε τη **γραμμή αναζήτησης** για φιλτράρισμα πρόσθετων βάσει ονόματος.
