Το [[Εισαγωγή στο Obsidian Sync|Obsidian Sync]] προσφέρει έναν headless client για συγχρονισμό θησαυροφυλακίων χωρίς τη χρήση της εφαρμογής επιφάνειας εργασίας. Χρήσιμο για CI pipelines, agents και αυτοματοποιημένες ροές εργασίας. Συγχρονίστε τις τελευταίες αλλαγές ή διατηρήστε τα αρχεία συνεχώς ενημερωμένα.

Εγκαταστήστε το [[Obsidian Headless]] **(ανοιχτή beta)** για να αλληλεπιδράσετε με το [[Εισαγωγή στο Obsidian Sync|Obsidian Sync]] από τη γραμμή εντολών χωρίς την εφαρμογή επιφάνειας εργασίας του Obsidian. Το Headless Sync χρησιμοποιεί τις ίδιες [[Ασφάλεια και απόρρητο|προστασίες κρυπτογράφησης και απορρήτου]] με την εφαρμογή επιφάνειας εργασίας, συμπεριλαμβανομένης της κρυπτογράφησης από άκρο σε άκρο.

## Γρήγορη εκκίνηση

> [!error] Δημιουργήστε αντίγραφα ασφαλείας των δεδομένων σας πριν ξεκινήσετε
> 1. Δημιουργείτε πάντα αντίγραφα ασφαλείας των δεδομένων σας πριν ξεκινήσετε σε περίπτωση που συμβεί κάτι απρόσμενο.
> 2. Μην χρησιμοποιείτε *ταυτόχρονα* τον Συγχρονισμό της εφαρμογής επιφάνειας εργασίας και τον Headless Sync στην ίδια συσκευή, καθώς μπορεί να προκαλέσει συγκρούσεις δεδομένων. Χρησιμοποιήστε μόνο μία μέθοδο συγχρονισμού ανά συσκευή.

Εγκαταστήστε το [[Obsidian Headless|Obsidian Headless]] **(ανοιχτή beta)**:

```shell
npm install -g obsidian-headless
```

Πρέπει να έχετε ενεργή [[Πλάνα και όρια αποθήκευσης|συνδρομή Obsidian Sync]].

```shell
# Σύνδεση
ob login

# Εμφάνιση των απομακρυσμένων θησαυροφυλακίων σας
ob sync-list-remote

# Ρύθμιση θησαυροφυλακίου για συγχρονισμό
cd ~/vaults/my-vault
ob sync-setup --vault "My Vault"

# Εκτέλεση εφάπαξ συγχρονισμού
ob sync

# Εκτέλεση συνεχούς συγχρονισμού (παρακολουθεί αλλαγές)
ob sync --continuous
```

## Εντολές

### `ob sync-list-remote`

Εμφανίζει όλα τα απομακρυσμένα θησαυροφυλάκια που είναι διαθέσιμα στον λογαριασμό σας, συμπεριλαμβανομένων των κοινόχρηστων θησαυροφυλακίων.

### `ob sync-list-local`

Εμφανίζει τα τοπικά ρυθμισμένα θησαυροφυλάκια και τις διαδρομές τους.

### `ob sync-create-remote`

Δημιουργεί ένα νέο απομακρυσμένο θησαυροφυλάκιο.

```
ob sync-create-remote --name "Vault Name" [--encryption <standard|e2ee>] [--password <password>] [--region <region>]
```

| Επιλογή | Περιγραφή |
| --- | --- |
| `--name` | Όνομα θησαυροφυλακίου (υποχρεωτικό) |
| `--encryption` | `standard` για διαχειριζόμενη κρυπτογράφηση, `e2ee` για κρυπτογράφηση από άκρο σε άκρο |
| `--password` | Κωδικός κρυπτογράφησης από άκρο σε άκρο (ζητείται αν παραλειφθεί) |
| `--region` | [[Περιοχές Sync\|Περιοχή]] διακομιστή (αυτόματη αν παραλειφθεί) |

### `ob sync-setup`

Ρυθμίζει τον συγχρονισμό μεταξύ ενός τοπικού και ενός απομακρυσμένου θησαυροφυλακίου.

```
ob sync-setup --vault <id-or-name> [--path <local-path>] [--password <password>] [--device-name <name>] [--config-dir <name>]
```

| Επιλογή | Περιγραφή |
| --- | --- |
| `--vault` | Αναγνωριστικό ή όνομα απομακρυσμένου θησαυροφυλακίου (υποχρεωτικό) |
| `--path` | Τοπικός κατάλογος (προεπιλογή: τρέχων κατάλογος) |
| `--password` | Κωδικός κρυπτογράφησης E2E (ζητείται αν παραλειφθεί) |
| `--device-name` | Όνομα συσκευής που εμφανίζεται στο [[Ιστορικό έκδοσης\|ιστορικό έκδοσης συγχρονισμού]] |
| `--config-dir` | Όνομα [[Φάκελος ρυθμίσεων\|φακέλου ρυθμίσεων]] (προεπιλογή: `.obsidian`) |

### `ob sync`

Εκτελεί συγχρονισμό για ένα ρυθμισμένο θησαυροφυλάκιο.

```
ob sync [--path <local-path>] [--continuous]
```

| Επιλογή | Περιγραφή |
| --- | --- |
| `--path` | Διαδρομή τοπικού θησαυροφυλακίου (προεπιλογή: τρέχων κατάλογος) |
| `--continuous` | Εκτέλεση συνεχώς, παρακολουθώντας αλλαγές |

### `ob sync-config`

Προβολή ή αλλαγή [[Ρυθμίσεις Sync και επιλεκτικός συγχρονισμός|ρυθμίσεων συγχρονισμού]] για ένα θησαυροφυλάκιο. Εκτελέστε χωρίς επιλογές για εμφάνιση της τρέχουσας διαμόρφωσης.

```
ob sync-config [--path <local-path>] [options]
```

| Επιλογή                | Περιγραφή                                                                                                                                                                                                                         |
| --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `--path`              | Διαδρομή τοπικού θησαυροφυλακίου (προεπιλογή: τρέχων κατάλογος)                                                                                                                                                                    |
| `--mode`              | Λειτουργία συγχρονισμού: `bidirectional` (προεπιλογή), `pull-only` (μόνο λήψη, αγνόηση τοπικών αλλαγών), ή `mirror-remote` (μόνο λήψη, επαναφορά τοπικών αλλαγών)                                                                   |
| `--conflict-strategy` | `merge` ή `conflict`                                                                                                                                                                                                               |
| `--file-types`        | Τύποι συνημμένων για συγχρονισμό: `image`, `audio`, `video`, `pdf`, `unsupported` (διαχωρισμένοι με κόμμα, κενό για εκκαθάριση)                                                                                                     |
| `--configs`           | Κατηγορίες ρυθμίσεων για συγχρονισμό: `app`, `appearance`, `appearance-data`, `hotkey`, `core-plugin`, `core-plugin-data`, `community-plugin`, `community-plugin-data` (διαχωρισμένες με κόμμα, κενό για απενεργοποίηση συγχρονισμού ρυθμίσεων) |
| `--excluded-folders`  | Φάκελοι για εξαίρεση (διαχωρισμένοι με κόμμα, κενό για εκκαθάριση)                                                                                                                                                                |
| `--device-name`       | Όνομα συσκευής για αναγνώριση αυτού του client στο ιστορικό έκδοσης συγχρονισμού                                                                                                                                                    |
| `--config-dir`        | Όνομα φακέλου ρυθμίσεων (προεπιλογή: `.obsidian`)                                                                                                                                                                                  |

### `ob sync-status`

Εμφανίζει την κατάσταση και τη διαμόρφωση συγχρονισμού για ένα θησαυροφυλάκιο.

```
ob sync-status [--path <local-path>]
```

### `ob sync-unlink`

Αποσυνδέει ένα θησαυροφυλάκιο από τον συγχρονισμό και αφαιρεί τα αποθηκευμένα διαπιστευτήρια.

```
ob sync-unlink [--path <local-path>]
```

## Εγγενείς μονάδες

Το Obsidian Headless περιλαμβάνει ένα προκατασκευασμένο εγγενές πρόσθετο (addon) για τον ορισμό του χρόνου δημιουργίας αρχείων (birthtime) σε Windows και macOS. Αυτό διατηρεί τις αρχικές χρονοσφραγίδες δημιουργίας κατά τη λήψη αρχείων από τον διακομιστή.

Το addon στοχεύει στην έκδοση N-API 3, επομένως τα μεταγλωττισμένα δυαδικά αρχεία είναι σταθερά σε επίπεδο ABI και λειτουργούν σε διάφορες εκδόσεις Node.js χωρίς επαναμεταγλώττιση.

Σε Linux, το birthtime δεν υποστηρίζεται — το addon δεν περιλαμβάνεται και ο συγχρονισμός λειτουργεί κανονικά χωρίς αυτό.

Προκατασκευασμένα δυαδικά αρχεία περιλαμβάνονται για:

- `win32-x64`
- `win32-arm64`
- `win32-ia32`
- `darwin-x64`
- `darwin-arm64`
