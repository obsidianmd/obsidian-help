Ένα θησαυροφυλάκιο είναι ένας φάκελος στο τοπικό σας σύστημα αρχείων όπου το Obsidian αποθηκεύει τις σημειώσεις σας. Μπορείτε να κρατάτε όλες τις σημειώσεις σας σε ένα θησαυροφυλάκιο ή να δημιουργήσετε πολλά θησαυροφυλάκια για κάθε διαφορετικό σας έργο.

Την πρώτη φορά που θα ανοίξετε το Obsidian, θα σας ζητηθεί να προσθέσετε ένα νέο θησαυροφυλάκιο. Έχετε δύο επιλογές, είτε να δημιουργήσετε ένα νέο κενό θησαυροφυλάκιο, είτε να χρησιμοποιήσετε έναν υπάρχοντα φάκελο.

## Δημιουργία κενού θησαυροφυλακίου

Για να δημιουργήσετε ένα νέο κενό θησαυροφυλάκιο:

1. Στα δεξιά του **Δημιουργία νέου φυλάκιου**, κάντε κλικ στο **Δημιουργία**.
2. Στο **Όνομα φυλάκιου**, εισάγετε το όνομα του θησαυροφυλακίου σας.
3. Κάντε κλικ στο **Περιήγηση** για να επιλέξετε πού θα δημιουργηθεί το νέο θησαυροφυλάκιο.
4. Κάντε κλικ στο **Δημιουργία**.

## Άνοιγμα υπάρχοντος φακέλου

Αν έχετε ήδη έναν φάκελο που θέλετε να χρησιμοποιήσετε ως θησαυροφυλάκιο:

1. Στα δεξιά του **Άνοιγμα φακέλου σαν φυλάκιο**, κάντε κλικ στο **Άνοιγμα**.
2. Στο πρόγραμμα περιήγησης αρχείων, επιλέξτε τον φάκελο που θέλετε να χρησιμοποιήσετε ως θησαυροφυλάκιο.
3. Κάντε κλικ στο **Άνοιγμα**.

Αν θέλετε να μάθετε περισσότερα σχετικά με τον τρόπο λειτουργίας των θησαυροφυλακίων, μάθετε [[Πώς αποθηκεύει δεδομένα το Obsidian|πώς αποθηκεύει δεδομένα το Obsidian]].

Τώρα που ρυθμίσατε το θησαυροφυλάκιό σας, είστε έτοιμοι να [[Δημιουργήστε την πρώτη σας σημείωση|δημιουργήσετε την πρώτη σας σημείωση]].
