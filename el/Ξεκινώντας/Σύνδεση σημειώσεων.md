Ενώ το Obsidian είναι εξαιρετικό για τη λήψη σημειώσεων, η πραγματική δύναμη του Obsidian βρίσκεται στη δυνατότητα σύνδεσης των σημειώσεών σας μεταξύ τους. Κατανοώντας πώς μια πληροφορία σχετίζεται με μια άλλη, μπορείτε να βελτιώσετε την ικανότητά σας να τις θυμάστε και να αναπτύξετε βαθύτερες γνώσεις. Σε αυτόν τον οδηγό, θα μάθετε πώς να δημιουργείτε και να πλοηγείστε σε συνδέσμους στο Obsidian.

## Δημιουργία συνδέσμου

Σε αυτό το βήμα, θα δημιουργήσετε δύο σημειώσεις και θα τις συνδέσετε μεταξύ τους χρησιμοποιώντας τη \[\[σύνταξη διπλών αγκυλών\]\].

1. [[Δημιουργήστε την πρώτη σας σημείωση|Δημιουργήστε μια σημείωση]] με το όνομα "Τρεις νόμοι της κίνησης":

   > Οι νόμοι της κίνησης είναι τρεις νόμοι που διατυπώθηκαν από τον Ισαάκ Νεύτωνα, οι οποίοι περιγράφουν τη σχέση μεταξύ της κίνησης ενός αντικειμένου και των δυνάμεων που ασκούνται σε αυτό.

2. Δημιουργήστε μια άλλη σημείωση με το όνομα "Νόμος της Αδράνειας" με το ακόλουθο κείμενο:

   > Ο Νόμος της Αδράνειας είναι ένας από τους

3. Στο τέλος της πρότασης, πατήστε την αριστερή αγκύλη (`[`) δύο φορές στο πληκτρολόγιό σας.
4. Πληκτρολογήστε "τρεις" για να βρείτε την πρώτη σημείωση που δημιουργήσατε.
5. Πατήστε `Enter` για να δημιουργήσετε έναν σύνδεσμο προς την επισημασμένη σημείωση.

   > Ο Νόμος της Αδράνειας είναι ένας από τους \[\[Τρεις νόμοι της κίνησης\]\]

Ανοίξτε τη σημείωση "Τρεις νόμοι της κίνησης" κάνοντας κλικ στον σύνδεσμο ενώ πατάτε `Ctrl` (ή `Cmd` σε macOS).

## Δημιουργία συνδέσμου προς μη υπάρχουσα σημείωση

Μπορείτε να δημιουργήσετε συνδέσμους προς σημειώσεις που δεν υπάρχουν ακόμα, για όταν θέλετε να εμβαθύνετε σε ένα θέμα αργότερα.

1. Στη σημείωσή σας "Τρεις νόμοι της κίνησης", επιλέξτε το κείμενο "Ισαάκ Νεύτωνας".
2. Πατήστε την αριστερή αγκύλη (`[`) δύο φορές στο πληκτρολόγιό σας για να δημιουργήσετε έναν σύνδεσμο. Ο δεύτερος σύνδεσμος έχει πιο αχνό χρώμα για να υποδείξει ότι η σημείωση δεν υπάρχει ακόμα.
3. Δημιουργήστε τη σημείωση κάνοντας κλικ στον σύνδεσμο ενώ πατάτε `Ctrl` (ή `Cmd` σε macOS).

## Πλοήγηση μεταξύ σημειώσεων

Όπως είδατε στα προηγούμενα βήματα, μπορείτε να κάνετε κλικ σε έναν σύνδεσμο ενώ πατάτε `Ctrl` (ή `Cmd` σε macOS) για να μεταβείτε στη συνδεδεμένη σημείωση.

Ένας άλλος τρόπος πλοήγησης μεταξύ σημειώσεων είναι μέσω των _οπισθοσυνδέσμων_. Ένας οπισθοσύνδεσμος σας επιτρέπει να πλοηγηθείτε στην αντίθετη κατεύθυνση ενός υπάρχοντος συνδέσμου.

1. Ανοίξτε τη σημείωση "Ισαάκ Νεύτωνας".
2. Στη δεξιά πλαϊνή μπάρα, κάντε κλικ στην καρτέλα **Οπισθοσύνδεσμοι**.
3. Κάτω από τις **Συνδεδεμένες αναφορές**, κάντε κλικ στην αναφορά στους "Τρεις νόμοι της κίνησης" για να μεταβείτε σε αυτή τη σημείωση.

Ένας άλλος τρόπος πλοήγησης μεταξύ των σημειώσεών σας είναι χρησιμοποιώντας μια οπτική αναπαράσταση του τρόπου σύνδεσης των σημειώσεών σας.

**Επιτραπέζιος υπολογιστής**
1. Στην πάνω δεξιά γωνία της σημείωσης, κάντε κλικ στο **Περισσότερες επιλογές** ![[lucide-more-horizontal.svg#icon]].
2. Επιλέξτε **Άνοιγμα σε συνδεδεμένη προβολή**.
3. Επιλέξτε **Άνοιγμα τοπικού γράφου**.
4. Κάντε κλικ σε οποιονδήποτε κόμβο στον γράφο για να πλοηγηθείτε σε αυτή τη σημείωση.

**Κινητό**
1. Στο πάνω μέρος της σημείωσης, επιλέξτε **Περισσότερες επιλογές** ![[lucide-more-horizontal.svg#icon]].
2. Επιλέξτε **Άνοιγμα τοπικού γράφου**.
3. Κάντε κλικ σε οποιονδήποτε κόμβο στον γράφο για να πλοηγηθείτε σε αυτή τη σημείωση.

## Μάθετε περισσότερα

Η κατανόηση του τρόπου σύνδεσης των σημειώσεών σας γίνεται ολοένα και πιο δύσκολη καθώς το θησαυροφυλάκιό σας μεγαλώνει. Μάθετε πώς να χρησιμοποιείτε την [[Προβολή Γράφου|προβολή γράφου]] για να αποκτήσετε βαθύτερες γνώσεις από τη βάση γνώσεών σας.
