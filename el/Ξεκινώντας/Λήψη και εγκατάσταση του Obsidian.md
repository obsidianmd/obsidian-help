Το Obsidian είναι διαθέσιμο για όλες τις κύριες πλατφόρμες υπολογιστών και κινητών. Αυτοί είναι όλοι οι υποστηριζόμενοι τρόποι λήψης και εγκατάστασης του Obsidian.

## Εγκατάσταση του Obsidian σε Windows

1. Ανοίξτε το πρόγραμμα περιήγησής σας και μεταβείτε στη σελίδα [Λήψη Obsidian](https://obsidian.md/download).
2. Στην ενότητα **Windows**, κάντε κλικ στο **Universal** για να κατεβάσετε το αρχείο εγκατάστασης.
3. Ανοίξτε το αρχείο εγκατάστασης και ακολουθήστε τις οδηγίες.
4. Ανοίξτε το Obsidian όπως θα ανοίγατε οποιαδήποτε άλλη εφαρμογή.

## Εγκατάσταση του Obsidian σε macOS

1. Ανοίξτε το πρόγραμμα περιήγησής σας και μεταβείτε στη σελίδα [Λήψη Obsidian](https://obsidian.md/download).
2. Στην ενότητα **macOS**, κάντε κλικ στο **Universal** για να κατεβάσετε το αρχείο εγκατάστασης.
3. Ανοίξτε το αρχείο εγκατάστασης.
4. Στο παράθυρο που ανοίγει, σύρετε το Obsidian στον φάκελο Applications.
5. Ανοίξτε το Obsidian όπως θα ανοίγατε οποιαδήποτε άλλη εφαρμογή.

## Εγκατάσταση του Obsidian σε Linux

Εάν χρησιμοποιείτε Linux, μπορείτε να εγκαταστήσετε το Obsidian με διάφορους τρόπους. Ακολουθήστε τις οδηγίες για το σύστημα πακέτων που χρησιμοποιείτε.

### Εγκατάσταση του Obsidian μέσω Snap

1. Ανοίξτε το πρόγραμμα περιήγησής σας και μεταβείτε στη σελίδα [Λήψη Obsidian](https://obsidian.md/download).
2. Στην ενότητα **Linux**, κάντε κλικ στο **Snap** για να κατεβάσετε το αρχείο εγκατάστασης.
3. Ανοίξτε ένα τερματικό και πλοηγηθείτε στον φάκελο όπου κατεβάσατε το αρχείο εγκατάστασης.
4. Στο τερματικό, εκτελέστε την ακόλουθη εντολή για να εγκαταστήσετε το πακέτο Snap: (η σημαία `--dangerous` είναι απαραίτητη επειδή η Canonical, η εταιρεία που δημιούργησε το Snap, δεν έχει ελέγξει το πακέτο μας, η σημαία `--classic` επιτρέπει στο Obsidian να έχει πρόσβαση εκτός του sandbox, όπου αποθηκεύονται οι σημειώσεις σας)

   ```bash
   snap install obsidian_<version>_<arch>.snap --dangerous --classic
   ```

5. Ανοίξτε το Obsidian όπως θα ανοίγατε οποιαδήποτε άλλη εφαρμογή.

### Εγκατάσταση του Obsidian μέσω AppImage

1. Ανοίξτε το πρόγραμμα περιήγησής σας και μεταβείτε στη σελίδα [Λήψη Obsidian](https://obsidian.md/download).
2. Στην ενότητα **Linux**, κάντε κλικ στο **AppImage** για να κατεβάσετε το αρχείο εγκατάστασης.
3. Ανοίξτε ένα τερματικό και πλοηγηθείτε στον φάκελο όπου κατεβάσατε το αρχείο εγκατάστασης.
4. Στο τερματικό, εκτελέστε την ακόλουθη εντολή για να ανοίξετε το Obsidian:

   ```bash
   chmod u+x Obsidian-<version>.AppImage
   ./Obsidian-<version>.AppImage --no-sandbox
   ```
Σημείωση: Σε Chromebook, πρέπει να εγκατασταθεί το πακέτο `libnss3-dev` αλλιώς ενδέχεται να λάβετε το σφάλμα `error while loading shared libraries: libnss3.so: cannot open shared object file: No such file or directory`.

### Εγκατάσταση του Obsidian μέσω Flatpak

1. Στο τερματικό σας, εκτελέστε την ακόλουθη εντολή για να εγκαταστήσετε το Obsidian:

   ```bash
   flatpak install flathub md.obsidian.Obsidian
   ```

2. Ανοίξτε το Obsidian εκτελώντας την ακόλουθη εντολή:

   ```bash
   flatpak run md.obsidian.Obsidian
   ```

## Εγκατάσταση του Obsidian σε Android

1. Βρείτε το Obsidian στο [Play Store](https://play.google.com/store/apps/details?id=md.obsidian).
2. Πατήστε **Εγκατάσταση** για να κατεβάσετε την εφαρμογή.
3. Ανοίξτε το Obsidian όπως θα ανοίγατε οποιαδήποτε άλλη εφαρμογή.

Μπορείτε προαιρετικά να κατεβάσετε το APK για Android από τη σελίδα [Λήψη Obsidian](https://obsidian.md/download).

## Εγκατάσταση του Obsidian σε iPhone και iPad

1. Βρείτε το Obsidian στο [App Store](https://apps.apple.com/us/app/obsidian-connected-notes/id1557175442).
2. Πατήστε **Λήψη** για να κατεβάσετε την εφαρμογή.
3. Ανοίξτε το Obsidian όπως θα ανοίγατε οποιαδήποτε άλλη εφαρμογή.
