Μάθετε πώς να χρησιμοποιείτε το στοιχείο HTML [iframe](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/iframe) για να ενσωματώνετε ιστοσελίδες στις σημειώσεις σας.

Για να ενσωματώσετε μια ιστοσελίδα, προσθέστε τα παρακάτω στη σημείωσή σας και αντικαταστήστε το κείμενο υπόδειξης με τη διεύθυνση URL της ιστοσελίδας που θέλετε να ενσωματώσετε:

```html
<iframe src="ΕΙΣΑΓΕΤΕ ΤΗ ΔΙΕΥΘΥΝΣΗ URL ΕΔΩ"></iframe>
```

> [!note] Σημείωση
> Ορισμένες ιστοσελίδες δεν επιτρέπουν την ενσωμάτωσή τους. Αντ' αυτού, ενδέχεται να παρέχουν διευθύνσεις URL που προορίζονται για ενσωμάτωση. Εάν η ιστοσελίδα δεν υποστηρίζει ενσωμάτωση, δοκιμάστε να αναζητήσετε το όνομα της ιστοσελίδας ακολουθούμενο από "embed iframe". Για παράδειγμα, "youtube embed iframe".

> [!tip] Συμβουλή
> Εάν χρησιμοποιείτε τον [[Καμβάς]], μπορείτε να ενσωματώσετε μια ιστοσελίδα σε μια κάρτα. Για περισσότερες πληροφορίες, ανατρέξτε στην ενότητα [[Καμβάς#Προσθήκη κάρτας από ιστοσελίδες]].

## Ενσωμάτωση βίντεο YouTube

Για να ενσωματώσετε ένα βίντεο YouTube, χρησιμοποιήστε την ίδια σύνταξη Markdown με τις [[Βασική σύνταξη μορφοποίησης#Εξωτερικές εικόνες|εξωτερικές εικόνες]]:

```md
![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)
```

![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)

## Ενσωμάτωση tweet

Για να ενσωματώσετε ένα tweet, χρησιμοποιήστε την ίδια σύνταξη Markdown με τις [[Βασική σύνταξη μορφοποίησης#Εξωτερικές εικόνες|εξωτερικές εικόνες]]:

![](https://twitter.com/obsdmd/status/1580548874246443010)
