Αυτή η σελίδα παραθέτει τις προεπιλεγμένες συντομεύσεις πληκτρολογίου για πλοήγηση και επεξεργασία κειμένου στο Obsidian. Αυτές οι συντομεύσεις παρέχονται από το λειτουργικό σας σύστημα ή το πλαίσιο στο οποίο είναι χτισμένο το Obsidian και δεν μπορούν να προσαρμοστούν μέσα στο Obsidian.

Για προσαρμόσιμες συντομεύσεις πληκτρολογίου για εντολές του Obsidian, δείτε [[Συντομεύσεις]].

## Συντομεύσεις Windows και Linux

### Κοινές ενέργειες

| Ενέργεια | Συντόμευση |
|-|-|
| Αντιγραφή | `Ctrl+C` |
| Αποκοπή | `Ctrl+X` |
| Επικόλληση | `Ctrl+V` |
| Επικόλληση χωρίς μορφοποίηση | `Ctrl+Shift+V` |
| Αναίρεση | `Ctrl+Z` |
| Επανάληψη | `Ctrl+Shift+Z` ή `Ctrl+Y` |
| Αντιγραφή παραγράφου | `Ctrl+C` (χωρίς επιλεγμένο κείμενο) |
| Αποκοπή παραγράφου | `Ctrl+X` (χωρίς επιλεγμένο κείμενο) |

### Επεξεργασία κειμένου

| Ενέργεια | Συντόμευση |
|-|-|
| Εισαγωγή νέας γραμμής| `Enter` |
| Διαγραφή του προηγούμενου χαρακτήρα | `Backspace` |
| Διαγραφή του επόμενου χαρακτήρα | `Delete` |
| Διαγραφή της προηγούμενης λέξης | `Ctrl+Backspace` |
| Διαγραφή της επόμενης λέξης | `Ctrl+Delete` |
| Διαγραφή της τρέχουσας γραμμής | `Ctrl+Shift+K` (χωρίς επιλεγμένο κείμενο) |

### Πλοήγηση κειμένου

| Ενέργεια | Συντόμευση |
|-|-|
| Μετακίνηση του δρομέα κατά έναν χαρακτήρα | `Left/→` |
| Μετακίνηση του δρομέα στην αρχή της προηγούμενης λέξης | `Ctrl+←` |
| Μετακίνηση του δρομέα στο τέλος της επόμενης λέξης | `Ctrl+→` |
| Μετακίνηση του δρομέα στην αρχή της τρέχουσας γραμμής | `Home` |
| Μετακίνηση του δρομέα στο τέλος της τρέχουσας γραμμής | `End` |
| Μετακίνηση του δρομέα στην προηγούμενη γραμμή | `↑` |
| Μετακίνηση του δρομέα στην επόμενη γραμμή | `↓` |
| Μετακίνηση του δρομέα στην αρχή της σημείωσης | `Ctrl+Home` |
| Μετακίνηση του δρομέα στο τέλος της σημείωσης | `Ctrl+End` |
| Μετακίνηση του δρομέα μία σελίδα επάνω | `Page up` |
| Μετακίνηση του δρομέα μία σελίδα κάτω | `Page down` |

### Επιλογή κειμένου

| Ενέργεια | Συντόμευση |
|-|-|
| Απλοποίηση επιλογής | `Escape` |
| Επιλογή όλων | `Ctrl+A` |
| Επέκταση επιλογής κατά έναν χαρακτήρα | `Shift+Left/→` |
| Επέκταση επιλογής στην αρχή της προηγούμενης λέξης | `Ctrl+Shift+←` |
| Επέκταση επιλογής στο τέλος της επόμενης λέξης | `Ctrl+Shift+→` |
| Επέκταση επιλογής στην αρχή της τρέχουσας γραμμής | `Shift+Home` |
| Επέκταση επιλογής στο τέλος της τρέχουσας γραμμής | `Shift+End` |
| Επέκταση επιλογής στην αρχή της σημείωσης | `Ctrl+Shift+Home` |
| Επέκταση επιλογής στο τέλος της σημείωσης | `Ctrl+Shift+End` |
| Επέκταση επιλογής μία σελίδα επάνω | `Shift+Page up` |
| Επέκταση επιλογής μία σελίδα κάτω | `Shift+Page down` |

## Συντομεύσεις macOS

### Κοινές ενέργειες

| Ενέργεια | Συντόμευση |
|-|-|
| Αντιγραφή | `Cmd+C` |
| Αποκοπή | `Cmd+X` |
| Επικόλληση | `Cmd+V` |
| Επικόλληση χωρίς μορφοποίηση | `Cmd+Shift+V` |
| Αναίρεση | `Cmd+Z` |
| Επανάληψη | `Cmd+Shift+Z` |
| Αντιγραφή παραγράφου | `Cmd+C` (χωρίς επιλεγμένο κείμενο) |
| Αποκοπή παραγράφου | `Cmd+X` (χωρίς επιλεγμένο κείμενο) |

### Μορφοποίηση κειμένου

| Ενέργεια | Συντόμευση |
| ---- | ---- |
| Έντονο κείμενο | `Cmd+B` |
| Πλάγιο κείμενο | `Cmd+I` |

### Επεξεργασία κειμένου

| Ενέργεια | Συντόμευση |
|-|-|
| Εισαγωγή νέας γραμμής| `Enter` |
| Διαγραφή του προηγούμενου χαρακτήρα | `Backspace` |
| Διαγραφή του επόμενου χαρακτήρα | `Delete` |
| Διαγραφή της προηγούμενης λέξης | `Option+Backspace` |
| Διαγραφή της επόμενης λέξης | `Option+Delete` |
| Διαγραφή μέχρι την αρχή της τρέχουσας γραμμής | `Cmd+Backspace` |
| Διαγραφή μέχρι το τέλος της τρέχουσας γραμμής | `Cmd+Delete` |
| Διαγραφή της τρέχουσας γραμμής | `Cmd+Shift+K` (χωρίς επιλεγμένο κείμενο) |

### Πλοήγηση κειμένου

| Ενέργεια | Συντόμευση |
|-|-|
| Μετακίνηση του δρομέα κατά έναν χαρακτήρα | `Left/→` |
| Μετακίνηση του δρομέα στην αρχή της προηγούμενης λέξης | `Option+←` |
| Μετακίνηση του δρομέα στο τέλος της επόμενης λέξης | `Option+→` |
| Μετακίνηση του δρομέα στην αρχή της τρέχουσας γραμμής | `Cmd+←` |
| Μετακίνηση του δρομέα στο τέλος της τρέχουσας γραμμής | `Cmd+→` |
| Μετακίνηση του δρομέα στην προηγούμενη γραμμή | `↑` |
| Μετακίνηση του δρομέα στην επόμενη γραμμή | `↓` |
| Μετακίνηση του δρομέα στην αρχή της σημείωσης | `Cmd+↑` |
| Μετακίνηση του δρομέα στο τέλος της σημείωσης | `Cmd+↓` |
| Μετακίνηση του δρομέα μία σελίδα επάνω | `Fn+↑` |
| Μετακίνηση του δρομέα μία σελίδα κάτω | `Fn+↓` |
