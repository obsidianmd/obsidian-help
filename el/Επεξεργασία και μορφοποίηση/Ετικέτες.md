Οι ετικέτες είναι λέξεις-κλειδιά ή θέματα που σας βοηθούν να βρίσκετε γρήγορα τις σημειώσεις που θέλετε.

## Προσθήκη ετικέτας σε μια σημείωση

Για να δημιουργήσετε μια ετικέτα, εισάγετε ένα σύμβολο δίεσης (`#`) στον επεξεργαστή, ακολουθούμενο από μια λέξη-κλειδί. Για παράδειγμα, `#meeting`.

Μπορείτε επίσης να προσθέσετε ετικέτες χρησιμοποιώντας την [[Ιδιότητες|ιδιότητα]] `tags`. Οι ετικέτες σε YAML πρέπει πάντα να μορφοποιούνται ως λίστα:

```yaml
---
tags:
  - recipe
  - cooking
---
```

## Εύρεση σημειώσεων χρησιμοποιώντας ετικέτες

Για να βρείτε σημειώσεις χρησιμοποιώντας το πρόσθετο [[Αναζήτηση]], χρησιμοποιήστε τον [[Αναζήτηση#Τελεστές αναζήτησης|τελεστή αναζήτησης]] `tag` στον όρο αναζήτησής σας, για παράδειγμα `tag:#meeting`.

Μπορείτε επίσης να αναζητήσετε ετικέτες κάνοντας κλικ πάνω τους στις σημειώσεις σας.

Για να βρείτε σημειώσεις χρησιμοποιώντας το πρόσθετο [[Παράθυρο ετικετών|Παράθυρο ετικετών]], επιλέξτε **Ετικέτες: Προβολή παραθύρου ετικετών** στην [[Παλέτα εντολών]] και στη συνέχεια επιλέξτε την ετικέτα που θέλετε να αναζητήσετε.

## Ένθετες ετικέτες

Οι ένθετες ετικέτες ορίζουν ιεραρχίες ετικετών που διευκολύνουν την εύρεση και το φιλτράρισμα σχετικών ετικετών.

Δημιουργήστε ένθετες ετικέτες χρησιμοποιώντας καθέτους (`/`) στο όνομα της ετικέτας, για παράδειγμα `#inbox/to-read` και `#inbox/processing`.

- Στην [[Αναζήτηση]], η αναζήτηση `tag:inbox` θα αντιστοιχεί τόσο στο `#inbox` όσο και σε όλες τις ένθετες ετικέτες όπως `#inbox/to-read`.
- Στο [[Παράθυρο ετικετών]], οι ένθετες ετικέτες εμφανίζονται ως ανήκουσες στη γονική τους ετικέτα.
- Στις [[Εισαγωγή στις Βάσεις|Βάσεις]], οι ένθετες ετικέτες αναγνωρίζονται από τη συνάρτηση [[Συναρτήσεις#hasTag|`hasTag`]], επομένως η `file.hasTag("a")` θα αντιστοιχεί τόσο στο `#a` όσο και στο `#a/b`.

## Μορφή ετικέτας

Μπορείτε να χρησιμοποιήσετε οποιονδήποτε από τους παρακάτω χαρακτήρες στις ετικέτες σας:

- Αλφαβητικά γράμματα
- Αριθμούς
- Κάτω παύλα (`_`)
- Ενωτικό (`-`)
- Κάθετο (`/`) για [[#Ένθετες ετικέτες]]
- Ευρέως αποδεκτούς χαρακτήρες Unicode, συμπεριλαμβανομένων emoji και άλλων συμβόλων

Οι ετικέτες πρέπει να περιέχουν τουλάχιστον έναν μη αριθμητικό χαρακτήρα. Για παράδειγμα, η #1984 δεν είναι έγκυρη ετικέτα, αλλά η #y1984 είναι.

Οι ετικέτες δεν κάνουν διάκριση πεζών-κεφαλαίων. Για παράδειγμα, οι #tag και #TAG θα αντιμετωπίζονται ως ταυτόσημες.

> [!note] 
> Οι ετικέτες θα εμφανίζονται με τη χρήση πεζών-κεφαλαίων με την οποία δημιουργήθηκαν αρχικά στο [[Παράθυρο ετικετών]]. 
> Για παράδειγμα, αν δημιουργήσετε #Tag και στη συνέχεια #TAG, θα εμφανίζεται #Tag και για τις δύο. 

Οι ετικέτες δεν μπορούν να περιέχουν κενά. Για να διαχωρίσετε δύο ή περισσότερες λέξεις, μπορείτε αντ' αυτού να χρησιμοποιήσετε τις ακόλουθες μορφές:

- #camelCase
- #PascalCase
- #snake_case
- #kebab-case
