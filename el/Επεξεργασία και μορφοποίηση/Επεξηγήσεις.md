Χρησιμοποιήστε πλαίσια επισήμανσης για να συμπεριλάβετε πρόσθετο περιεχόμενο χωρίς να διακόψετε τη ροή των σημειώσεών σας.

Για να δημιουργήσετε ένα πλαίσιο επισήμανσης, προσθέστε `[!info]` στην πρώτη γραμμή μιας παράθεσης σε μπλοκ, όπου `info` είναι το _αναγνωριστικό τύπου_. Το αναγνωριστικό τύπου καθορίζει την εμφάνιση και την αίσθηση του πλαισίου επισήμανσης. Για να δείτε όλους τους διαθέσιμους τύπους, ανατρέξτε στην ενότητα [[#Υποστηριζόμενοι τύποι]]. Τα πλαίσια επισήμανσης υποστηρίζονται επίσης εγγενώς στο [[Εισαγωγή στο Obsidian Publish|Obsidian Publish]].

```markdown
> [!info] Εδώ είναι ένας τίτλος πλαισίου επισήμανσης
> Εδώ είναι ένα μπλοκ πλαισίου επισήμανσης.
> Υποστηρίζει **Markdown**, [[Εσωτερικοί σύνδεσμοι|Συνδέσμους wiki]] και [[Ενσωμάτωση αρχείων|ενσωματώσεις]]!
> ![[Engelbart.jpg]]
```

> [!info] Εδώ είναι ένας τίτλος πλαισίου επισήμανσης
> Εδώ είναι ένα μπλοκ πλαισίου επισήμανσης.
> Υποστηρίζει **Markdown**, [[Εσωτερικοί σύνδεσμοι|Συνδέσμους wiki]] και [[Ενσωμάτωση αρχείων|ενσωματώσεις]]!
> ![[Engelbart.jpg]]

Μπορείτε να εισαγάγετε ένα προεπιλεγμένο πλαίσιο επισήμανσης `[!note]` χρησιμοποιώντας την [[Παλέτα εντολών|εντολή]] `Εισαγωγή επεξήγησης`. Ο δρομέας τοποθετείται αυτόματα στο πεδίο ονόματος του πλαισίου επισήμανσης, επιτρέποντάς σας να διαγράψετε το προεπιλεγμένο όνομα και να πληκτρολογήσετε ένα νέο πριν επεξεργαστείτε το περιεχόμενο.

Για να τυλίξετε υπάρχον περιεχόμενο σε ένα πλαίσιο επισήμανσης, επιλέξτε το κείμενο (συμπεριλαμβανομένων λιστών, μπλοκ κώδικα κ.λπ.) και εκτελέστε την εντολή `Εισαγωγή επεξήγησης`. Το επιλεγμένο περιεχόμενο θα περικλειστεί αυτόματα στο πλαίσιο επισήμανσης.

Στη [[Προβολές και λειτουργία επεξεργασίας#Ζωντανή Προεπισκόπηση|Ζωντανή Προεπισκόπηση]], μπορείτε επίσης να κάνετε δεξί κλικ στο όνομα του πλαισίου επισήμανσης για να αλλάξετε τον τύπο του.


### Αλλαγή τίτλου

Από προεπιλογή, ο τίτλος του πλαισίου επισήμανσης είναι το αναγνωριστικό τύπου του με κεφαλαία στην αρχή. Μπορείτε να τον αλλάξετε προσθέτοντας κείμενο μετά το αναγνωριστικό τύπου:

```markdown
> [!tip] Τα πλαίσια επισήμανσης μπορούν να έχουν προσαρμοσμένους τίτλους
> Όπως αυτό.
```

> [!tip] Τα πλαίσια επισήμανσης μπορούν να έχουν προσαρμοσμένους τίτλους
> Όπως αυτό.

Μπορείτε ακόμη να παραλείψετε το σώμα για να δημιουργήσετε πλαίσια επισήμανσης μόνο με τίτλο:

```markdown
> [!tip] Πλαίσιο επισήμανσης μόνο με τίτλο
```

### Αναδιπλούμενα πλαίσια επισήμανσης

Μπορείτε να κάνετε ένα πλαίσιο επισήμανσης αναδιπλούμενο προσθέτοντας ένα συν (`+`) ή ένα πλην (`-`) αμέσως μετά το αναγνωριστικό τύπου.

Ένα σύμβολο συν αναπτύσσει το πλαίσιο επισήμανσης από προεπιλογή, ενώ ένα σύμβολο πλην το συμπτύσσει.

```markdown
> [!faq]- Είναι τα πλαίσια επισήμανσης αναδιπλούμενα;
> Ναι! Σε ένα αναδιπλούμενο πλαίσιο επισήμανσης, τα περιεχόμενα αποκρύπτονται όταν το πλαίσιο επισήμανσης είναι συμπτυγμένο.
```

> [!faq]- Είναι τα πλαίσια επισήμανσης αναδιπλούμενα;
> Ναι! Σε ένα αναδιπλούμενο πλαίσιο επισήμανσης, τα περιεχόμενα αποκρύπτονται όταν είναι συμπτυγμένο.

### Ένθετα πλαίσια επισήμανσης

Μπορείτε να ενθέσετε πλαίσια επισήμανσης σε πολλαπλά επίπεδα.

```markdown
> [!question] Μπορούν τα πλαίσια επισήμανσης να είναι ένθετα;
> > [!todo] Ναι!, μπορούν.
> > > [!example] Μπορείτε ακόμη να χρησιμοποιήσετε πολλαπλά επίπεδα ένθεσης.
```

### Προσαρμογή πλαισίων επισήμανσης

Τα [[CSS αποσπάσματα]] και τα [[Πρόσθετα κοινότητας]] μπορούν να ορίσουν προσαρμοσμένα πλαίσια επισήμανσης ή ακόμη και να αντικαταστήσουν την προεπιλεγμένη ρύθμιση.

Για να ορίσετε ένα προσαρμοσμένο πλαίσιο επισήμανσης, δημιουργήστε το ακόλουθο μπλοκ CSS:

```css
.callout[data-callout="custom-question-type"] {
    --callout-color: 0, 0, 0;
    --callout-icon: lucide-alert-circle;
}
```

Η τιμή του χαρακτηριστικού `data-callout` είναι το αναγνωριστικό τύπου που θέλετε να χρησιμοποιήσετε, για παράδειγμα `[!custom-question-type]`.

- Το `--callout-color` ορίζει το χρώμα φόντου χρησιμοποιώντας αριθμούς (0–255) για κόκκινο, πράσινο και μπλε.
- Το `--callout-icon` μπορεί να είναι ένα αναγνωριστικό εικονιδίου από το [lucide.dev](https://lucide.dev) ή ένα στοιχείο SVG.

> [!warning] Σημείωση σχετικά με τις εκδόσεις εικονιδίων lucide
> Το Obsidian ενημερώνει τα εικονίδια Lucide περιοδικά. Η τρέχουσα έκδοση που περιλαμβάνεται εμφανίζεται παρακάτω· χρησιμοποιήστε αυτά ή παλαιότερα εικονίδια σε προσαρμοσμένα πλαίσια επισήμανσης.
> ![[Συντελεστές#^lucide]]

> [!tip] Εικονίδια SVG
> Αντί να χρησιμοποιήσετε ένα εικονίδιο Lucide, μπορείτε επίσης να χρησιμοποιήσετε ένα στοιχείο SVG ως εικονίδιο του πλαισίου επισήμανσης.
>
> ```css
> --callout-icon: '<svg>...custom svg...</svg>';
> ```

### Υποστηριζόμενοι τύποι

Μπορείτε να χρησιμοποιήσετε αρκετούς τύπους πλαισίων επισήμανσης και ψευδώνυμα. Κάθε τύπος έχει διαφορετικό χρώμα φόντου και εικονίδιο.

Για να χρησιμοποιήσετε αυτά τα προεπιλεγμένα στυλ, αντικαταστήστε το `info` στα παραδείγματα με οποιονδήποτε από αυτούς τους τύπους, όπως `[!tip]` ή `[!warning]`. Οι τύποι πλαισίων επισήμανσης μπορούν επίσης να αλλάξουν κάνοντας δεξί κλικ σε ένα πλαίσιο επισήμανσης στη λειτουργία Ζωντανής Προεπισκόπησης.

Εκτός εάν [[#Προσαρμογή πλαισίων επισήμανσης|προσαρμόσετε τα πλαίσια επισήμανσης]], οποιοσδήποτε μη υποστηριζόμενος τύπος θα χρησιμοποιεί προεπιλεγμένα τον τύπο `note`. Το αναγνωριστικό τύπου δεν κάνει διάκριση πεζών-κεφαλαίων.

> [!note]
> ```md
> > [!note]
> > Lorem ipsum dolor sit amet
> ```

> [!abstract]-
> ```md
> > [!abstract]
> > Lorem ipsum dolor sit amet
> ```

Ψευδώνυμα: `summary`, `tldr`

> [!tip]-
