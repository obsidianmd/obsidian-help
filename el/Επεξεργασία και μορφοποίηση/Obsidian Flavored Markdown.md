Το Obsidian προσπαθεί να παρέχει τη μέγιστη δυνατή λειτουργικότητα χωρίς να παραβιάζει υπάρχουσες μορφές. Ως αποτέλεσμα, χρησιμοποιούμε έναν συνδυασμό παραλλαγών του [[Βασική σύνταξη μορφοποίησης|Markdown]].

Το Obsidian υποστηρίζει [CommonMark](https://commonmark.org/), [GitHub Flavored Markdown](https://github.github.com/gfm/) και [LaTeX](https://www.latex-project.org/).

> [!tip]- Markdown μέσα σε HTML
> Το Obsidian δεν αποδίδει σύνταξη Markdown μέσα σε στοιχεία HTML. Αυτή είναι μια σκόπιμη σχεδιαστική επιλογή για βελτιστοποίηση απόδοσης και για τη διατήρηση χαμηλής πολυπλοκότητας του αναλυτή κατά τη διαχείριση μεγάλων εγγράφων.
>
> Για παράδειγμα, η μορφοποίηση Markdown όπως `**bold**` ή `` `code` `` δεν θα υποστεί επεξεργασία μέσα σε `<div>`, `<span>`, `<table>` ή οποιαδήποτε άλλη ετικέτα HTML.
>
> ```md
> <div>
> This **will not** be bold.
> </div>
> ```

### Υποστηριζόμενες επεκτάσεις Markdown

| Σύνταξη         | Περιγραφή                                                                       |
| --------------- | ------------------------------------------------------------------------------- |
| `[[Link]]`      | [[Εσωτερικοί σύνδεσμοι]]                                                       |
| `![[Link]]`     | [[Ενσωμάτωση αρχείων]]                                                          |
| `![[Link#^id]]` | [[Εσωτερικοί σύνδεσμοι#Σύνδεση σε μπλοκ μιας σημείωσης\|Αναφορές μπλοκ]]      |
| `^id`           | [[Εσωτερικοί σύνδεσμοι#Σύνδεση σε μπλοκ μιας σημείωσης\|Ορισμός μπλοκ]]       |
| `[^id]`         | [[Βασική σύνταξη μορφοποίησης#Υποσημειώσεις\|Υποσημειώσεις]]                    |
| `%%Text%%`      | [[Βασική σύνταξη μορφοποίησης#Σχόλια\|Σχόλια]]                                  |
| `~~Text~~`      | [[Βασική σύνταξη μορφοποίησης#Έντονα, πλάγια, επισημάνσεις\|Διακριτή διαγραφή]] |
| `==Text==`      | [[Βασική σύνταξη μορφοποίησης#Έντονα, πλάγια, επισημάνσεις\|Επισημάνσεις]]      |
| `` ``` ``       | [[Βασική σύνταξη μορφοποίησης#Μπλοκ κώδικα\|Μπλοκ κώδικα]]                     |
| `- [ ]`         | [[Βασική σύνταξη μορφοποίησης#Λίστες ελέγχου\|Μη ολοκληρωμένη εργασία]]        |
| `- [x]`         | [[Βασική σύνταξη μορφοποίησης#Λίστες ελέγχου\|Ολοκληρωμένη εργασία]]           |
| `> [!note]`     | [[Επεξηγήσεις]]                                                                 |
| (δείτε σύνδεσμο) | [[Σύνθετη σύνταξη μορφοποίησης#Πίνακες\|Πίνακες]]                             |
