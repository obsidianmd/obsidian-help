Οι [[Εισαγωγή στις Βάσεις|Βάσεις]] σας επιτρέπουν να δημιουργείτε προβολές τύπου βάσης δεδομένων για τις σημειώσεις σας. Δείτε πώς μπορείτε να δημιουργήσετε μια βάση και να την ενσωματώσετε σε μια σημείωση. Κάθε βάση μπορεί να έχει μία ή περισσότερες [[Προβολές|προβολές]] για να εμφανίζει πληροφορίες με διαφορετικούς τρόπους.

## Δημιουργία νέας βάσης

**Παλέτα εντολών:**

1. Ανοίξτε την **Παλέτα εντολών**.
2. Επιλέξτε
	- **Βάσεις: Δημιουργία νέας βάσης** για να δημιουργήσετε μια βάση στον ίδιο φάκελο με το ενεργό αρχείο.
	- **Βάσεις: Εισαγωγή νέας βάσης** για να δημιουργήσετε μια βάση και να την ενσωματώσετε στο τρέχον αρχείο.

**Εξερεύνηση αρχείων:**

1. Στην Εξερεύνηση αρχείων, κάντε δεξί κλικ στον φάκελο στον οποίο θέλετε να δημιουργήσετε τη βάση.
2. Επιλέξτε **Νέα βάση**.

**Κορδέλα:**

- Στο κατακόρυφο μενού κορδέλας, επιλέξτε **Δημιουργία νέας βάσης** για να δημιουργήσετε μια βάση στον ίδιο φάκελο με το ενεργό αρχείο.

## Ενσωμάτωση βάσης

### Ενσωμάτωση αρχείου βάσης

Μπορείτε να ενσωματώσετε αρχεία βάσης σε [[Ενσωμάτωση αρχείων|οποιοδήποτε άλλο αρχείο]] χρησιμοποιώντας τη σύνταξη `![[Αρχείο.base]]`. Για να καθορίσετε την προεπιλεγμένη προβολή χρησιμοποιήστε `![[Αρχείο.base#Προβολή]]`.

### Ενσωμάτωση βάσης ως μπλοκ κώδικα

Οι βάσεις μπορούν επίσης να ενσωματωθούν απευθείας σε μια σημείωση χρησιμοποιώντας ένα μπλοκ κώδικα `base` και τη [[Σύνταξη Βάσεων|σύνταξη βάσεων]].

~~~yaml
```base
filters:
  and:
    - file.hasTag("example")
views:
  - type: table
    name: Table
```
~~~
