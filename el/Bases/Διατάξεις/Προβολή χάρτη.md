Ο Χάρτης είναι ένας τύπος [[Προβολές|προβολής]] που μπορείτε να χρησιμοποιήσετε στις [[Εισαγωγή στις Βάσεις|Βάσεις]]. Απαιτεί την εγκατάσταση του [πρόσθετου Maps](obsidian://show-plugin?id=maps).

Επιλέξτε ![[lucide-map.svg#icon]]  **Χάρτης** από το μενού προβολής για να εμφανίσετε αρχεία ως διαδραστικό χάρτη με δείκτες για κάθε αρχείο και μια προεπισκόπηση που εμφανίζει τις ιδιότητες αυτού του αρχείου.

![[bases-map-places.png#interface]]

## Εγκατάσταση του πρόσθετου Maps

Οι προβολές χάρτη απαιτούν Obsidian 1.10. Το [πρόσθετο Maps](obsidian://show-plugin?id=maps) είναι ένα επίσημο [[Πρόσθετα κοινότητας|πρόσθετο κοινότητας]] που μπορείτε να κατεβάσετε ξεχωριστά.

1. Ακολουθήστε τις οδηγίες στο [[Πρόσθετα κοινότητας#Εγκατάσταση ενός πρόσθετου κοινότητας]]
2. Κατεβάστε και ενεργοποιήστε το [Maps](obsidian://show-plugin?id=maps) από τη λίστα

## Παράδειγμα

Για να ξεκινήσετε, δοκιμάστε να δημιουργήσετε μια σημείωση με τίτλο **Eiffel Tower** και αντιγράψτε τις ακόλουθες ιδιότητες σε αυτήν:

```yaml
---
coordinates:
  - "48.85837"
  - "2.294481"
icon: "landmark"
color: "red"
tags:
  - places
---
```

Δείτε τι σημαίνει ο παραπάνω κώδικας:

| Ιδιότητα      | Τιμή                     |                                                                                                                                                                          |
| ------------- | ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `coordinates` | `48.85837`<br>`2.294481` | Οι συντεταγμένες αποθηκεύονται ως `γεωγραφικό πλάτος, γεωγραφικό μήκος`. Μπορείτε να λάβετε συντεταγμένες κάνοντας δεξί κλικ σε μια τοποθεσία στον χάρτη και επιλέγοντας **Αντιγραφή συντεταγμένων**. |
| `icon`        | `landmark`               | Το όνομα ενός εικονιδίου από τη [βιβλιοθήκη Lucide](https://lucide.dev/).                                                                                                |
| `color`       | `red`                    | Μια έγκυρη τιμή CSS: hex, RGB, ονομασμένο χρώμα κ.λπ.                                                                                                                    |
| `tags`        | `places`                 | Η ετικέτα που θα χρησιμοποιήσουμε για να βρούμε δείκτες χάρτη στη βάση μας.                                                                                              |
Τώρα δημιουργήστε μια προβολή χάρτη με ένα φίλτρο για την ετικέτα `places` και ορίστε συντεταγμένες δεικτών, εικονίδιο και χρώμα χρησιμοποιώντας τις ιδιότητες που αναφέρονται παραπάνω.

Μπορείτε επίσης να ανοίξετε αυτά τα [αρχεία παραδείγματος](https://github.com/obsidianmd/obsidian-maps/tree/master/examples) στο Obsidian για να δείτε λειτουργικές προβολές χάρτη με δείκτες, εικονίδια και χρώματα ήδη ρυθμισμένα.

## Ρυθμίσεις

Οι ρυθμίσεις προβολής χάρτη μπορούν να διαμορφωθούν στις [[Προβολές#Ρυθμίσεις προβολής|Ρυθμίσεις προβολής]].

- Ύψος ενσωμάτωσης
- Κεντρικές συντεταγμένες
- Περιορισμοί μεγέθυνσης
- Συντεταγμένες, χρώμα και εικονίδιο δεικτών
- Φόντο

### Δείκτες

#### Συντεταγμένες

Για να εμφανίσετε πινέζες στον χάρτη, μεταβείτε στις [[Προβολές#Ρυθμίσεις προβολής|ρυθμίσεις προβολής]] και επιλέξτε μια ιδιότητα **συντεταγμένων δεικτών**. Η ιδιότητα πρέπει να περιέχει συντεταγμένες γεωγραφικού πλάτους και μήκους. Γίνονται δεκτές οι ακόλουθες μορφές:

```yaml
# Ιδιότητα κειμένου
coordinates: "lat, lng"

# Ιδιότητα λίστας
coordinates:
  - "lat"
  - "lng"
```

Αν αποθηκεύετε τις συντεταγμένες ως ξεχωριστές ιδιότητες `latitude` και `longitude`, μπορείτε να τις συνδυάσετε με μια ιδιότητα [[Σύνταξη Βάσεων#Τύποι|τύπου]] ορίζοντάς την ως πίνακα συντεταγμένων χρησιμοποιώντας τον ακόλουθο τύπο: `[latitude, longitude]`.

#### Εικονίδια

Προσθέστε εικονίδια στους δείκτες ορίζοντας μια ιδιότητα **εικονιδίων δεικτών**. Για παράδειγμα, μπορείτε να προσθέσετε μια ιδιότητα με όνομα `icon` στις σημειώσεις σας και να της δώσετε τιμές όπως `landmark` ή `utensils` από την ενσωματωμένη [βιβλιοθήκη Lucide](https://lucide.dev/icons/) του Obsidian.

##### Χρήση τύπου για ορισμό εικονιδίων

Ας πούμε ότι θέλετε όλα τα εστιατόρια να έχουν το ίδιο εικονίδιο στον χάρτη:

1. Δημιουργήστε μια σημείωση με τίτλο **Restaurants** και προσθέστε μια ιδιότητα με όνομα `icon` με την τιμή `utensils`.
2. Δώστε στις σημειώσεις εστιατορίων μια ιδιότητα με όνομα `type` που συνδέεται με τη σημείωση `[[Restaurants]]`.
3. Προσθέστε μια ιδιότητα τύπου με όνομα `Type icon` στη βάση σας με τον ακόλουθο κώδικα:
	```js
	list(type)[0].asFile().properties.icon
	```
4. Επιλέξτε το `Type icon` ως εικονίδιο δείκτη στις ρυθμίσεις προβολής.

Voilà! Τώρα ο χάρτης σας θα εμφανίζει εικονίδια από τον *τύπο* του τόπου, όχι από τον ίδιο τον τόπο.

#### Χρώματα

Ορίστε το χρώμα των δεικτών. Δέχεται τιμές ως RGB `rgb(0,0,0)`, HEX `#000` ή μεταβλητές CSS όπως `var(--color-blue)`. Όπως στο παράδειγμα εικονιδίων παραπάνω, μπορείτε να χρησιμοποιήσετε μια ιδιότητα τύπου για να ορίσετε χρώματα δυναμικά.

### Φόντο

#### Πλακίδια χάρτη

Τα πλακίδια χάρτη είναι ένας τυπικός τρόπος εμφάνισης ψηφιακών χαρτών. Υπάρχουν αρκετές υπηρεσίες που μπορείτε να χρησιμοποιήσετε για να προσαρμόσετε χάρτες με μοναδικά στυλ, χρώματα και γραμματοσειρές. Το Maps υποστηρίζει τόσο raster όσο και vector πλακίδια και δέχεται τα περισσότερα URL πλακιδίων, συμπεριλαμβανομένων των URL TileJSON.

Το [OpenFreeMap](https://openfreemap.org/) προσφέρει μερικά στυλ που μπορείτε να χρησιμοποιήσετε δωρεάν. Δοκιμάστε να χρησιμοποιήσετε ένα από τα ακόλουθα URL στη ρύθμιση **Πλακίδια χάρτη**:

| Όνομα    | URL                                              |
| -------- | ------------------------------------------------ |
| Dark     | `https://tiles.openfreemap.org/styles/dark`      |
| Positron | `https://tiles.openfreemap.org/styles/positron`  |
| Liberty  | ``https://tiles.openfreemap.org/styles/liberty`` |
#### Χρήσιμοι σύνδεσμοι

- [Maputnik](https://maputnik.github.io/) για προσαρμογή πλακιδίων χάρτη.
- [Protomaps](https://protomaps.com/) για αυτο-φιλοξενία πλακιδίων χάρτη.
- Άλλες φιλοξενούμενες υπηρεσίες με δωρεάν βαθμίδα περιλαμβάνουν τα [MapTiler](https://www.maptiler.com/) και [Mapbox](https://www.mapbox.com/).


## Συμβουλές

Μπορείτε να δημιουργήσετε συνδέσμους προς δημοφιλείς υπηρεσίες χαρτών χρησιμοποιώντας [[Τύποι]]. Για παράδειγμα, η πινέζα σας μπορεί να εμφανίζει έναν σύνδεσμο προς το Google Maps χρησιμοποιώντας τον ακόλουθο τύπο:

```js
link("https://www.google.com/maps/search/" + file.name.replace(" ","+"),"Google Maps")
```

Το [πρόσθετο Maps](https://github.com/obsidianmd/obsidian-maps) είναι ανοιχτού κώδικα. Μπορείτε να βοηθήσετε συνεισφέροντας αναφορές σφαλμάτων, αιτήματα χαρακτηριστικών και pull requests.
