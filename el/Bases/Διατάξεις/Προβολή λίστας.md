Η Λίστα είναι ένας τύπος [[Προβολές|προβολής]] που μπορείτε να χρησιμοποιήσετε στις [[Εισαγωγή στις Βάσεις|Βάσεις]].

Επιλέξτε ![[lucide-list.svg#icon]]  **Λίστα** από το μενού προβολής για να εμφανίσετε αρχεία ως [[Βασική σύνταξη μορφοποίησης#Λίστες|λίστα]] με κουκκίδες ή αριθμημένους δείκτες.

## Ρυθμίσεις

Οι ρυθμίσεις προβολής λίστας μπορούν να τροποποιηθούν στις [[Προβολές#Ρυθμίσεις προβολής|Ρυθμίσεις προβολής]].

- Δείκτες: κουκκίδες, αριθμοί ή κανένας
- Εσοχή ιδιοτήτων
- Διαχωριστικά για ιδιότητες

### Δείκτες

Επιλέξτε τον τύπο δείκτη λίστας που θέλετε να χρησιμοποιήσετε: κουκκίδες, αριθμοί ή κανένας.

### Εσοχή ιδιοτήτων

Όταν είναι ενεργοποιημένο, οποιεσδήποτε ιδιότητες επιλέξετε στο μενού **Ιδιότητες** θα εμφανίζονται ως στοιχεία με εσοχή κάτω από το κύριο στοιχείο λίστας.

Για να αλλάξετε το κύριο στοιχείο λίστας, μετακινήστε μια ιδιότητα στο μενού **Ιδιότητες** στην κορυφή, σύροντάς την ή χρησιμοποιώντας τη συντόμευση `Alt + Up/Down`.

### Διαχωριστικά

Όταν η επιλογή **ένθετες ιδιότητες** είναι απενεργοποιημένη, αυτή η ρύθμιση ορίζει έναν χαρακτήρα για τον διαχωρισμό των ιδιοτήτων κάθε στοιχείου λίστας. Η προεπιλογή είναι το κόμμα.
