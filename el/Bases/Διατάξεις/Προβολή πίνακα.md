Ο πίνακας είναι ένας τύπος [[Προβολές|προβολής]] που μπορείτε να χρησιμοποιήσετε στις [[Εισαγωγή στις Βάσεις|Βάσεις]].

Επιλέξτε ![[lucide-table.svg#icon]]  **Πίνακας** από το μενού προβολής για να εμφανίσετε αρχεία ως πίνακα με μία γραμμή για κάθε αρχείο και στήλες για τις [[Ιδιότητες]] αυτού του αρχείου.

![Παράδειγμα βάσης που εμφανίζει προβολή πίνακα με μια λίστα βιβλίων](bases-noshadow.png#interface)

## Ρυθμίσεις

Οι ρυθμίσεις της προβολής πίνακα μπορούν να διαμορφωθούν στις [[Προβολές#Ρυθμίσεις προβολής|Ρυθμίσεις προβολής]].

### Ύψος γραμμής

Το ύψος γραμμής σάς επιτρέπει να εμφανίζετε περισσότερες πληροφορίες. Επιλέξτε μεταξύ **χαμηλό**, **μεσαίο**, **υψηλό** και **πολύ υψηλό**.

## Συνόψεις

Μπορείτε να προσθέσετε συνόψεις σε μια στήλη πίνακα για να υπολογίσετε γρήγορα τιμές όπως σύνολα, μέσους όρους ή μετρήσεις για τις γραμμές που είναι αυτή τη στιγμή ορατές στην προβολή.

Οι συνόψεις συνδέονται με την προβολή, όχι με τη βάση. Κάθε προβολή μπορεί να εμφανίζει διαφορετικές συνόψεις για την ίδια στήλη.

### Προσθήκη σύνοψης

1. Κάντε δεξί κλικ στην κεφαλίδα της στήλης σε μια προβολή πίνακα.
2. Επιλέξτε ![[lucide-calculator.svg#icon]] **Σύνοψη…**.
3. Επιλέξτε μία από τις ενσωματωμένες συναρτήσεις σύνοψης ή επιλέξτε ![[lucide-square-function.svg#icon]] **Προσθήκη σύνοψης** για να ορίσετε τη δική σας.

Η σύνοψη εμφανίζεται στο κάτω μέρος της στήλης. Όταν τα αποτελέσματα είναι [[Προβολές#Ταξινόμηση και ομαδοποίηση αποτελεσμάτων|ομαδοποιημένα]], η σύνοψη κάθε ομάδας εμφανίζεται στο πάνω μέρος της ομάδας.

Μόλις προστεθεί η γραμμή σύνοψης, μπορείτε να προσθέσετε περισσότερες συνόψεις για άλλες στήλες κάνοντας κλικ στο κελί σύνοψης. Η γραμμή σύνοψης αποκρύπτεται αν αφαιρεθούν όλες οι συνόψεις.

### Ενσωματωμένες συνόψεις

Οι ακόλουθες συνόψεις είναι διαθέσιμες από προεπιλογή. Οι επιλογές μπορεί να διαφέρουν ανάλογα με τον τύπο ιδιότητας.

#### Όλοι οι τύποι ιδιοτήτων

- **Κενό**: πλήθος γραμμών χωρίς τιμή.
- **Συμπληρωμένο**: πλήθος γραμμών με τιμή.
- **Μοναδική**: αριθμός διακριτών τιμών.

#### Αριθμοί

- **Μέσος όρος**: μέσος όρος όλων των αριθμητικών τιμών.
- **Μέγιστο**: μεγαλύτερη τιμή.
- **Διάμεσος**: διάμεση τιμή.
- **Ελάχιστο**: μικρότερη τιμή.
- **Εύρος**: διαφορά μεταξύ μέγιστου και ελάχιστου.
- **Τυπική απόκλιση**: τυπική απόκλιση.
- **Άθροισμα**: σύνολο όλων των τιμών.

#### Ημερομηνίες

- **Παλαιότερη**: η μικρότερη/παλαιότερη ημερομηνία.
- **Πιο πρόσφατη**: η μεγαλύτερη/πιο πρόσφατη ημερομηνία.
- **Εύρος**: διαφορά μεταξύ παλαιότερης και πιο πρόσφατης.

#### Πλαίσιο ελέγχου

- **Επιλεγμένο**: αριθμός γραμμών όπου το πλαίσιο ελέγχου είναι ενεργό.
- **Μη επιλεγμένο**: αριθμός γραμμών όπου το πλαίσιο ελέγχου είναι ανενεργό.

### Προσαρμοσμένες συνόψεις

Μπορείτε να ορίσετε τη δική σας σύνοψη χρησιμοποιώντας έναν τύπο:

1. Στο μενού ![[lucide-calculator.svg#icon]] **Σύνοψη…**, επιλέξτε ![[lucide-square-function.svg#icon]] **Προσθήκη σύνοψης**.
2. Δώστε ένα όνομα στη σύνοψη.
3. Εισαγάγετε έναν τύπο. Ο τύπος εκτελείται πάνω στη λίστα τιμών αυτής της στήλης (για παράδειγμα, χρησιμοποιώντας μια [[Συναρτήσεις|συνάρτηση]] όπως `values.reduce(...)`).
4. Αποθηκεύστε τη σύνοψη.

Οι προσαρμοσμένες συνόψεις είναι χρήσιμες όταν χρειάζεστε έναν υπολογισμό που δεν καλύπτεται από τις ενσωματωμένες επιλογές.

## Συντομεύσεις

Μπορείτε να μετακινηθείτε γρήγορα σε μια προβολή πίνακα χρησιμοποιώντας τις ακόλουθες συντομεύσεις ποντικιού και [[Συντομεύσεις επεξεργασίας|πληκτρολογίου]].

- Κλικ με Shift δημιουργεί μια επιλογή κελιών.
