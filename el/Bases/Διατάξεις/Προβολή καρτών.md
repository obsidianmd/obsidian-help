Οι Κάρτες είναι ένας τύπος [[Προβολές|προβολής]] που μπορείτε να χρησιμοποιήσετε στις [[Εισαγωγή στις Βάσεις|Βάσεις]].

Επιλέξτε ![[lucide-layout-grid.svg#icon]]  **Κάρτες** από το μενού προβολής για να εμφανίσετε αρχεία σε διάταξη πλέγματος τύπου συλλογής με προαιρετικές εικόνες εξωφύλλου.

## Ρυθμίσεις

Οι ρυθμίσεις της προβολής καρτών μπορούν να τροποποιηθούν στις [[Προβολές#Ρυθμίσεις προβολής|Ρυθμίσεις προβολής]].

- Μέγεθος κάρτας
- Ιδιότητα εικόνας
- Προσαρμογή εικόνας
- Αναλογία διαστάσεων εικόνας

### Μέγεθος κάρτας

Καθορίζει το πλάτος μιας κάρτας.

### Ιδιότητα εικόνας

Οι κάρτες υποστηρίζουν μια προαιρετική εικόνα εξωφύλλου, η οποία είναι μια [[Ιδιότητες|ιδιότητα]] που εμφανίζεται ως εικόνα στο πάνω μέρος της κάρτας. Η ιδιότητα μπορεί να είναι οποιοδήποτε από τα ακόλουθα:

- Ένας σύνδεσμος σε ένα τοπικό [[Επισυνάψεις|συνημμένο]] `"[[link/to/attachment.jpg]]"`
- Ένας εξωτερικός σύνδεσμος (URL)
- Ένας δεκαεξαδικός κωδικός χρώματος (`#000000`)

### Προσαρμογή εικόνας

Εάν έχετε ρυθμίσει μια ιδιότητα εικόνας, αυτή η επιλογή θα καθορίσει πώς θα εμφανίζεται η εικόνα στην κάρτα.

- **Κάλυψη:** Η εικόνα γεμίζει το πλαίσιο περιεχομένου της κάρτας. Εάν δεν χωράει, η εικόνα θα περικοπεί.
- **Περιεχόμενο:** Η εικόνα κλιμακώνεται μέχρι να χωρέσει μέσα στο πλαίσιο περιεχομένου της κάρτας. Η εικόνα δεν θα περικοπεί.

### Αναλογία διαστάσεων εικόνας

Το ύψος της εικόνας εξωφύλλου καθορίζεται από την αναλογία διαστάσεών της. Η προεπιλεγμένη αναλογία διαστάσεων είναι 1:1, που σημαίνει ότι όλες οι εικόνες σας θα είναι τετράγωνες. Προσαρμόστε αυτή την επιλογή για να κάνετε την εικόνα πιο κοντή ή πιο ψηλή.
