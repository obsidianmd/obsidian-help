Οι [[Προβολές|Προβολές]] σας επιτρέπουν να οργανώνετε τις πληροφορίες σε μια [[Εισαγωγή στις Βάσεις|Βάση]] με πολλούς τρόπους. Μια βάση μπορεί να περιέχει αρκετές προβολές, και κάθε προβολή μπορεί να έχει μια μοναδική ρύθμιση για εμφάνιση, ταξινόμηση και φιλτράρισμα αρχείων.

Για παράδειγμα, μπορεί να θέλετε να δημιουργήσετε μια βάση με το όνομα «Βιβλία» που να έχει ξεχωριστές προβολές για «Λίστα ανάγνωσης» και «Πρόσφατα ολοκληρωμένα».

## Γραμμή εργαλείων

Στην κορυφή μιας βάσης υπάρχει μια γραμμή εργαλείων που σας επιτρέπει να αλληλεπιδράτε με τις προβολές και τα αποτελέσματά τους.

- ![[lucide-table.svg#icon]] **Μενού προβολής** — δημιουργία, επεξεργασία και εναλλαγή προβολών.
- **Αποτελέσματα** — περιορισμός, αντιγραφή και εξαγωγή αρχείων.
- ![[lucide-arrow-up-down.svg#icon]] **Ταξινόμηση** — ταξινόμηση και ομαδοποίηση αρχείων.
- ![[lucide-list-filter.svg#icon]] **Φίλτρο** — φιλτράρισμα αρχείων.
- ![[lucide-list.svg#icon]] **Ιδιότητες** — επιλογή ιδιοτήτων για εμφάνιση και δημιουργία [[Τύποι|τύπων]].
- ![[lucide-search.svg#icon]] **Αναζήτηση** — αναζήτηση στοιχείων χρησιμοποιώντας τις εμφανιζόμενες ιδιότητές τους.
- ![[lucide-plus.svg#icon]] **Νέο** — δημιουργία νέου αρχείου στην τρέχουσα προβολή.

## Προσθήκη και εναλλαγή προβολών

Υπάρχουν δύο τρόποι για να προσθέσετε μια προβολή σε μια βάση:

- Κάντε κλικ στο όνομα της προβολής επάνω αριστερά και επιλέξτε ![[lucide-plus.svg#icon]] **Προσθήκη προβολής**.
- Χρησιμοποιήστε την [[Παλέτα εντολών|παλέτα εντολών]] και επιλέξτε **Bases: Add view**.

Η πρώτη προβολή στη λίστα προβολών σας θα φορτωθεί ως προεπιλογή. Σύρετε τις προβολές από το εικονίδιό τους για να αλλάξετε τη σειρά τους.

## Ρυθμίσεις προβολής

Κάθε προβολή έχει τις δικές της επιλογές ρύθμισης. Για να επεξεργαστείτε τις ρυθμίσεις προβολής:

1. Κάντε κλικ στο όνομα της προβολής επάνω αριστερά.
2. Κάντε κλικ στο δεξί βέλος δίπλα στην προβολή που θέλετε να ρυθμίσετε.

Εναλλακτικά, κάντε *δεξί κλικ* στο όνομα της προβολής στη γραμμή εργαλείων της βάσης για γρήγορη πρόσβαση στις ρυθμίσεις προβολής.

## Φίλτρα

Ανοίξτε το μενού ![[lucide-list-filter.svg#icon]] **Φίλτρο** στην κορυφή μιας βάσης για να προσθέσετε φίλτρα.

Μια βάση χωρίς φίλτρα εμφανίζει όλα τα αρχεία στο θησαυροφυλάκιό σας. Τα φίλτρα περιορίζουν τα αποτελέσματα ώστε να εμφανίζονται μόνο αρχεία που πληρούν συγκεκριμένα κριτήρια. Για παράδειγμα, μπορείτε να χρησιμοποιήσετε φίλτρα για να εμφανίζετε μόνο αρχεία με μια συγκεκριμένη [[Ετικέτες|ετικέτα]] ή μέσα σε έναν συγκεκριμένο φάκελο. Υπάρχουν πολλοί διαθέσιμοι τύποι φίλτρων.

Τα φίλτρα μπορούν να εφαρμοστούν σε όλες τις προβολές μιας βάσης ή μόνο σε μία μεμονωμένη προβολή, επιλέγοντας από τις δύο ενότητες στο μενού ![[lucide-list-filter.svg#icon]] **Φίλτρο**.

- **Όλες οι προβολές** εφαρμόζει φίλτρα σε όλες τις προβολές της βάσης.
- **Αυτή η προβολή** εφαρμόζει φίλτρα στην ενεργή προβολή.

#### Συστατικά ενός φίλτρου

Τα φίλτρα έχουν τρία συστατικά:

1. **Ιδιότητα** — σας επιτρέπει να επιλέξετε μια [[Ιδιότητες|ιδιότητα]] στο θησαυροφυλάκιό σας, συμπεριλαμβανομένων των [[Σύνταξη Βάσεων#Ιδιότητες αρχείου|ιδιοτήτων αρχείου]].
2. **Τελεστής** — σας επιτρέπει να επιλέξετε πώς θα συγκρίνετε τις συνθήκες. Η λίστα των διαθέσιμων τελεστών εξαρτάται από τον τύπο ιδιότητας (κείμενο, ημερομηνία, αριθμός, κ.λπ.)
3. **Τιμή** — σας επιτρέπει να επιλέξετε την τιμή με την οποία γίνεται η σύγκριση. Οι τιμές μπορούν να περιλαμβάνουν μαθηματικά και [[Συναρτήσεις|συναρτήσεις]].

#### Σύνδεσμοι

- **Όλα τα παρακάτω ισχύουν** είναι μια δήλωση `και` — τα αποτελέσματα θα εμφανίζονται μόνο αν πληρούνται *όλες* οι συνθήκες στην ομάδα φίλτρων.
- **Οποιοδήποτε από τα παρακάτω ισχύει** είναι μια δήλωση `ή` — τα αποτελέσματα θα εμφανίζονται αν πληρούται *οποιαδήποτε* από τις συνθήκες στην ομάδα φίλτρων.
- **Κανένα από τα παρακάτω δεν ισχύει** είναι μια δήλωση `not` — τα αποτελέσματα δεν θα εμφανίζονται αν πληρούται *οποιαδήποτε* από τις συνθήκες στην ομάδα φίλτρων.

#### Ομάδες φίλτρων

Οι ομάδες φίλτρων σας επιτρέπουν να δημιουργείτε πιο σύνθετη λογική δημιουργώντας συνδυασμούς συνδέσμων.

#### Σύνθετος επεξεργαστής φίλτρων

Κάντε κλικ στο κουμπί κώδικα ![[lucide-code-xml.svg#icon]] για να χρησιμοποιήσετε τον **σύνθετο επεξεργαστή φίλτρων**. Αυτό εμφανίζει την ακατέργαστη [[Σύνταξη Βάσεων|σύνταξη]] του φίλτρου και μπορεί να χρησιμοποιηθεί με πιο σύνθετες [[Συναρτήσεις|συναρτήσεις]] που δεν μπορούν να εμφανιστούν μέσω της διεπαφής σημείωσε-και-κάνε-κλικ.

## Ταξινόμηση και ομαδοποίηση αποτελεσμάτων

Ανοίξτε το μενού ![[lucide-arrow-up-down.svg#icon]] **Ταξινόμηση** για να ταξινομήσετε και να ομαδοποιήσετε τα αποτελέσματα σε μια προβολή.

Μπορείτε να τακτοποιήσετε τα αποτελέσματα κατά μία ή περισσότερες ιδιότητες σε αύξουσα ή φθίνουσα σειρά. Αυτό διευκολύνει την καταχώριση σημειώσεων κατά όνομα, χρόνο τελευταίας επεξεργασίας ή οποιαδήποτε άλλη ιδιότητα — συμπεριλαμβανομένων τύπων.

Μπορείτε επίσης να ομαδοποιήσετε τα αποτελέσματα κατά μια ιδιότητα για να οργανώσετε παρόμοια στοιχεία σε οπτικά ξεχωριστές ενότητες. Επί του παρόντος, το Obsidian υποστηρίζει ομαδοποίηση μόνο κατά μία ιδιότητα.

### Προσθήκη ταξινόμησης

1. Ανοίξτε το μενού ![[lucide-arrow-up-down.svg#icon]] **Ταξινόμηση** στην κορυφή της προβολής.
2. Επιλέξτε την ιδιότητα κατά την οποία θέλετε να ταξινομήσετε (ή ομαδοποιήσετε).
3. Αν έχετε πολλαπλές ταξινομήσεις, σύρετέ τες πάνω ή κάτω χρησιμοποιώντας τη λαβή ![[lucide-grip-vertical.svg#icon]] για να αλλάξετε την προτεραιότητά τους.

Οι επιλογές ταξινόμησης αποτελεσμάτων εξαρτώνται από τον τύπο ιδιότητας:

- **Κείμενο**: ταξινόμηση *αλφαβητικά* (Α→Ω) ή σε *αντίστροφη αλφαβητική σειρά* (Ω→Α).
- **Αριθμός**: ταξινόμηση από *μικρότερο σε μεγαλύτερο* (0→1) ή *μεγαλύτερο σε μικρότερο* (1→0).
- **Ημερομηνία και ώρα**: ταξινόμηση *από παλιά σε νέα* ή *από νέα σε παλιά*.

### Αφαίρεση ταξινόμησης

1. Ανοίξτε το μενού ![[lucide-arrow-up-down.svg#icon]] **Ταξινόμηση** στην κορυφή της προβολής.
2. Κάντε κλικ στο κουμπί ![[lucide-trash-2.svg#icon]] κάδου απορριμμάτων δίπλα στην ταξινόμηση ή ομαδοποίηση που θέλετε να αφαιρέσετε.

## Περιορισμός, αντιγραφή και εξαγωγή αποτελεσμάτων

### Περιορισμός αποτελεσμάτων

Το μενού *αποτελέσματα* εμφανίζει τον αριθμό των αποτελεσμάτων στην προβολή. Κάντε κλικ στο κουμπί αποτελεσμάτων για να περιορίσετε τον αριθμό των αποτελεσμάτων και να αποκτήσετε πρόσβαση σε πρόσθετες ενέργειες.

### Αντιγραφή στο πρόχειρο

Αυτή η ενέργεια αντιγράφει την προβολή στο πρόχειρό σας. Μόλις βρίσκεται στο πρόχειρό σας, μπορείτε να την επικολλήσετε σε ένα αρχείο Markdown ή σε άλλες εφαρμογές εγγράφων, συμπεριλαμβανομένων λογιστικών φύλλων όπως Google Sheets, Excel και Numbers.

### Εξαγωγή CSV

Αυτή η ενέργεια αποθηκεύει ένα CSV της τρέχουσας προβολής σας.

## Ενσωμάτωση μιας προβολής

Μπορείτε να ενσωματώσετε αρχεία βάσης σε [[Ενσωμάτωση αρχείων|οποιοδήποτε άλλο αρχείο]] χρησιμοποιώντας τη σύνταξη `![[Αρχείο.base]]`. Θα χρησιμοποιηθεί η πρώτη προβολή στη λίστα. Μπορείτε να αλλάξετε τη σειρά σύροντας τις προβολές στο μενού προβολής.

Για να καθορίσετε την προεπιλεγμένη προβολή για μια ενσωμάτωση, χρησιμοποιήστε `![[Αρχείο.base#Προβολή]]`.
