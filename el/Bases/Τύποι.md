Οι τύποι σας επιτρέπουν να δημιουργείτε υπολογιστικές ιδιότητες στις [[Εισαγωγή στις Βάσεις|Βάσεις]] χρησιμοποιώντας δεδομένα από άλλες [[Ιδιότητες|ιδιότητες]]. Μπορείτε να εκτελείτε υπολογισμούς, να χειρίζεστε κείμενο, να εργάζεστε με ημερομηνίες και πολλά άλλα.

## Τι μπορούν να κάνουν οι τύποι

Οι τύποι μπορούν να σας βοηθήσουν να:

- **Υπολογίζετε τιμές**, προσθέτετε τιμές, υπολογίζετε σύνολα ή εκτελείτε μαθηματικές πράξεις.
- **Χειρίζεστε κείμενο**, συνδυάζετε αλφαριθμητικά, αλλάζετε πεζά/κεφαλαία ή εξάγετε υποαλφαριθμητικά.
- **Εργάζεστε με ημερομηνίες**, υπολογίζετε χρονικές διαφορές, μορφοποιείτε ημερομηνίες ή προσδιορίζετε προθεσμίες.
- **Εφαρμόζετε λογική**, χρησιμοποιείτε συνθήκες για να εμφανίζετε διαφορετικές τιμές.
- **Επεξεργάζεστε λίστες**, φιλτράρετε, ταξινομείτε, αντιστοιχίζετε ή συγκεντρώνετε δεδομένα λίστας.

## Δημιουργία ιδιότητας τύπου

Για να δημιουργήσετε μια ιδιότητα τύπου:

1. Στη βάση σας, κάντε κλικ στο **Ιδιότητες** στη γραμμή εργαλείων.
2. Κάντε κλικ στο **Προσθήκη τύπου** στο κάτω μέρος του μενού.
3. Εισάγετε ένα όνομα για την ιδιότητα τύπου.
4. Πληκτρολογήστε τον τύπο σας στο πεδίο **Τύπος**.
5. Κλείστε το παράθυρο διαλόγου.

Ο επεξεργαστής τύπων θα συμπληρώνει αυτόματα ονόματα [[Συναρτήσεις|συναρτήσεων]] και ιδιοτήτων καθώς πληκτρολογείτε για να επικυρώσει τη σύνταξη του τύπου σας. Ένα πράσινο σημάδι ελέγχου εμφανίζεται όταν ο τύπος σας είναι έγκυρος.

Μόλις δημιουργηθεί, μπορείτε να χρησιμοποιήσετε μια ιδιότητα τύπου όπως οποιαδήποτε άλλη ιδιότητα στη βάση σας. Προσθέστε την σε [[Προβολές|προβολές]], χρησιμοποιήστε την σε φίλτρα, ταξινομήστε με βάση αυτήν και πολλά άλλα.

## Σύνταξη τύπου

Στον επεξεργαστή τύπων, πληκτρολογήστε μια έκφραση χρησιμοποιώντας ιδιότητες, τελεστές και συναρτήσεις.

### Αναφορά σε ιδιότητες

Μπορείτε να αναφέρεστε σε διαφορετικούς τύπους ιδιοτήτων στους τύπους σας:

- **Ιδιότητες σημείωσης** — Ιδιότητες από τα [[Ιδιότητες|μεταδεδομένα αρχής]] μιας σημείωσης.
- **Ιδιότητες αρχείου** — Ενσωματωμένες ιδιότητες όπως `file.name`, `file.size` ή `file.mtime`.
- **Ιδιότητες τύπων** — Άλλοι τύποι χρησιμοποιώντας `formula.formula_name`.

**Παραδείγματα:**

- `price * quantity` — πολλαπλασιασμός δύο ιδιοτήτων σημείωσης
- `file.name + " - " + description` — συνδυασμός ονόματος αρχείου με ιδιότητα σημείωσης
- `formula.price_per_unit * 1.1` — χρήση άλλης ιδιότητας τύπου

### Χρήση τελεστών

**Αριθμητικοί τελεστές** εκτελούν μαθηματικές πράξεις σε αριθμούς:

- `price + tax` — πρόσθεση
- `price - discount` — αφαίρεση
- `price * quantity` — πολλαπλασιασμός
- `price / quantity` — διαίρεση
- `(part / whole) * 100` — χρήση παρενθέσεων για σειρά πράξεων

**Τελεστές σύγκρισης** συγκρίνουν τιμές:

- `price > 100` — μεγαλύτερο από
- `age < 18` — μικρότερο από
- `status == "Done"` — ίσο με
- `status != "Done"` — διάφορο από
- `file.mtime > now() - '7d'` — σύγκριση ημερομηνιών

**Λογικοί τελεστές** συνδυάζουν λογικές συνθήκες:

- `!completed` — άρνηση
- `price > 0 && quantity > 0` — και
- `urgent || important` — ή

Μάθετε περισσότερα στη [[Σύνταξη Βάσεων#Τελεστές|Σύνταξη Βάσεων]].

### Χρήση συναρτήσεων

Οι συναρτήσεις εκτελούν λειτουργίες σε τιμές. Οι διαθέσιμες συναρτήσεις εξαρτώνται από τον τύπο της τιμής με την οποία εργάζεστε. Δείτε την πλήρη λίστα των [[Συναρτήσεις|Συναρτήσεων]].

**Κοινές κατηγορίες συναρτήσεων:**

- **Καθολικές συναρτήσεις** — `if()`, `now()`, `date()`, `link()`, `max()`, `min()`
- **Συναρτήσεις αλφαριθμητικών** — `contains()`, `replace()`, `split()`, `lower()`, `title()`
- **Συναρτήσεις αριθμών** — `round()`, `ceil()`, `floor()`, `abs()`, `toFixed()`
- **Συναρτήσεις ημερομηνιών** — `format()`, `relative()`, `date()`, `time()`
- **Συναρτήσεις λιστών** — `filter()`, `map()`, `sort()`, `join()`, `unique()`

**Παραδείγματα:**

- `if(price, "$" + price.toFixed(2), "")` για ορισμό συνθήκης με μορφοποίηση αριθμού.
- `file.name.lower()` για μετατροπή σε πεζά.
- `tags.contains("urgent")` για έλεγχο αν η λίστα ετικετών περιέχει μια τιμή.
- `due_date.format("YYYY-MM-DD")` για μορφοποίηση ημερομηνίας.

## Παραδείγματα τύπων

### Υπολογισμός προθεσμίας

Ορισμός της ημερομηνίας λήξης ενός έργου σε 2 εβδομάδες μετά την ημερομηνία έναρξης:

```js
start_date + "2w"
```

### Εμφάνιση κατάστασης εκπρόθεσμου

Εμφάνιση "Εκπρόθεσμο" αν η ημερομηνία λήξης έχει περάσει και η κατάσταση δεν είναι "Done":

```js
if(due_date < now() && status != "Done", "Overdue", "")
```

### Μορφοποίηση νομίσματος

Εμφάνιση τιμής με 2 δεκαδικά ψηφία και σύμβολο νομίσματος:

```js
if(price, "$" + price.toFixed(2), "")
```

### Μέτρηση στοιχείων λίστας

Μέτρηση του αριθμού στοιχείων σε μια ιδιότητα λίστας:

```js
tasks.length
```

### Υπολογισμός βαθμολογίας προτεραιότητας

Συνδυασμός πολλαπλών παραγόντων σε μια βαθμολογία προτεραιότητας:

```js
(impact * urgency) / effort
```

### Συνδυασμός πεδίων κειμένου

Δημιουργία πλήρους ονόματος από όνομα και επώνυμο:

```js
first_name + " " + last_name
```

### Υπολογισμός συνολικού κόστους

Πολλαπλασιασμός μηνιαίου κόστους επί αριθμό μηνών κατοχής:

```js
monthlyUses * formula.Owned.round()
```

## Τύποι δεδομένων

Οι τύποι λειτουργούν με διαφορετικούς τύπους δεδομένων:

- **Αλφαριθμητικά** — Κείμενο μέσα σε εισαγωγικά: `"hello"` ή `'world'`
- **Αριθμοί** — Αριθμητικές τιμές: `42`, `3.14`, `(2 + 2)`
- **Λογικές τιμές** — Αληθές ή ψευδές: `true`, `false`
- **Ημερομηνίες** — Δημιουργούνται με `date()`, `today()` ή `now()`
- **Λίστες** — Συλλογές τιμών: `[1, 2, 3]`
- **Αντικείμενα** — Ζεύγη κλειδιού-τιμής: `{"name": "value"}`

Ο τύπος εξόδου ενός τύπου καθορίζεται από τα δεδομένα και τις συναρτήσεις που χρησιμοποιούνται.

## Αναφορά σε άλλους τύπους

Οι τύποι μπορούν να αναφέρονται σε άλλους τύπους, δημιουργώντας παράγωγους υπολογισμούς. Για παράδειγμα, αν έχετε έναν τύπο με όνομα `price_per_unit`:

```js
price / quantity
```

Μπορείτε να αναφερθείτε σε αυτόν σε έναν άλλο τύπο:

```js
formula.price_per_unit * 1.1
```

> [!warning] Αποφύγετε τις κυκλικές αναφορές
> Ένας τύπος δεν μπορεί να αναφέρεται στον εαυτό του άμεσα ή έμμεσα μέσω άλλων τύπων.
