Οι Βάσεις είναι ένα [[Κύρια πρόσθετα|βασικό πρόσθετο]] που σας επιτρέπει να δημιουργείτε προβολές τύπου βάσης δεδομένων για τις σημειώσεις σας. Χρησιμοποιώντας μια βάση μπορείτε να προβάλλετε, να επεξεργάζεστε, να ταξινομείτε και να φιλτράρετε αρχεία και τις [[Ιδιότητες|ιδιότητές]] τους. Οι Βάσεις μπορούν να σας βοηθήσουν να οργανώσετε τα πάντα, από έργα έως ταξιδιωτικά σχέδια, λίστες ανάγνωσης και πολλά άλλα.

Κάθε βάση μπορεί να έχει πολλές [[Προβολές|προβολές]] με διαφορετικές διατάξεις, όπως πίνακες και κάρτες. Παρακάτω βρίσκεται ένα παράδειγμα προβολής πίνακα όπου κάθε γραμμή είναι ένα αρχείο και κάθε στήλη είναι μια ιδιότητα αυτού του αρχείου.

![Παράδειγμα βάσης που εμφανίζει προβολή πίνακα με μια λίστα βιβλίων](bases-noshadow.png#interface)

Όλα τα δεδομένα στις Βάσεις του Obsidian αποθηκεύονται στα τοπικά σας αρχεία [[Βασική σύνταξη μορφοποίησης|Markdown]] και τις [[Ιδιότητες|ιδιότητές]] τους. Οι προβολές περιγράφονται από τη [[Σύνταξη Βάσεων]], η οποία μπορεί να αποθηκευτεί ως αρχείο `.base` ή να [[Δημιουργία βάσης#Ενσωμάτωση βάσης|ενσωματωθεί]] σε μπλοκ κώδικα μέσα στα αρχεία Markdown σας.

## Πώς να χρησιμοποιήσετε τις βάσεις

- [[Δημιουργία βάσης]]
	- Μάθετε πώς να δημιουργείτε και να ενσωματώνετε μια βάση.
- [[Προβολές]]
	- Εξερευνήστε διατάξεις, φιλτράρετε, ταξινομήστε και ομαδοποιήστε ιδιότητες.
- [[Τύποι]]
	- Δημιουργήστε υπολογισμούς χρησιμοποιώντας δεδομένα από άλλες ιδιότητες.
- [[Συναρτήσεις]]
	- Συναρτήσεις που μπορείτε να χρησιμοποιήσετε σε τύπους και φίλτρα.
- [[Σύνταξη Βάσεων]]
	- Πώς μορφοποιούνται τα αρχεία βάσεων.

## Τύποι προβολών

Οι προβολές μπορούν να εμφανίζονται με διαφορετικές διατάξεις. Πρόσθετες διατάξεις μπορούν να προστεθούν μέσω [[Πρόσθετα κοινότητας]].

- [[Προβολή πίνακα|Πίνακας]]
	- Εμφανίστε αρχεία ως γραμμές με στήλες που δείχνουν ιδιότητες αυτών των αρχείων.
- [[Προβολή λίστας|Λίστα]]
	- Εμφανίστε αρχεία ως λίστα με δείκτες κουκκίδων ή αριθμών.
- [[Προβολή καρτών|Κάρτες]]
	- Εμφανίστε αρχεία ως πλέγμα. Σας επιτρέπει να δημιουργείτε προβολές τύπου γκαλερί με εικόνες.
- [[Προβολή χάρτη|Χάρτης]]
	- Εμφανίστε αρχεία ως καρφίτσες σε έναν διαδραστικό χάρτη.
