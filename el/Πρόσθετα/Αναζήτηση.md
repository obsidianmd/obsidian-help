Η Αναζήτηση είναι ένα [[Κύρια πρόσθετα|βασικό πρόσθετο]] που σας βοηθά να βρείτε δεδομένα στο θησαυροφυλάκιό σας στο Obsidian χρησιμοποιώντας όρους αναζήτησης και τελεστές για τον περιορισμό των αποτελεσμάτων.

Από προεπιλογή, μπορείτε να βρείτε την Αναζήτηση στην αριστερή πλαϊνή μπάρα ![[lucide-search.svg#icon]]. Μπορείτε επίσης να ανοίξετε την Αναζήτηση πατώντας `Ctrl+Shift+F` (Windows/Linux) ή `Command+Shift+F` (macOS).

- **Αναζήτηση επιλεγμένου κειμένου**: Εάν επιλέξετε κείμενο στον επεξεργαστή και ανοίξετε την Αναζήτηση με τη συντόμευση πληκτρολογίου, η Αναζήτηση σας εμφανίζει τα αποτελέσματα για το επιλεγμένο κείμενο.
- **Αναζήτηση πρόσφατων όρων αναζήτησης**: Ανοίξτε την Αναζήτηση με κενό όρο αναζήτησης για να εμφανίσετε τους πρόσφατους όρους αναζήτησης. Κάντε κλικ σε οποιονδήποτε από αυτούς για να χρησιμοποιήσετε ξανά τον όρο αναζήτησης.

> [!info] Φάκελοι που εξαιρούνται
> Τα αρχεία που ταιριάζουν με τα μοτίβα [[Ρυθμίσεις#Excluded files|Φάκελοι που εξαιρούνται]] δεν θα εμφανίζονται στα αποτελέσματα αναζήτησης.

## Όροι αναζήτησης

Ένας όρος αναζήτησης είναι η λέξη ή η φράση που εισάγετε στο πεδίο αναζήτησης. Η εκμάθηση αποτελεσματικής σύνταξης όρων αναζήτησης μπορεί να σας βοηθήσει να βρείτε γρήγορα αυτό που ψάχνετε, ακόμη και σε μεγάλα θησαυροφυλάκια. Το Obsidian αναζητά μόνο στα περιεχόμενα σημειώσεων και καμβάδων.

> [!tip]- Αναζήτηση διαδρομών και ονομάτων αρχείων
> Από προεπιλογή, μπορείτε να αναζητήσετε μόνο τις διαδρομές και τα ονόματα αρχείων σημειώσεων και καμβάδων. Για να αναζητήσετε μια διαδρομή ή ένα όνομα αρχείου οποιουδήποτε αρχείου στο θησαυροφυλάκιο, χρησιμοποιήστε τον τελεστή `path` ή `file`.

Κάθε λέξη στον όρο αναζήτησης αντιστοιχίζεται ανεξάρτητα μέσα σε κάθε αρχείο. Για να αναζητήσετε μια ακριβή φράση, περικλείστε τη σε εισαγωγικά, για παράδειγμα `"star wars"`. Για να αναζητήσετε κείμενο σε εισαγωγικά μέσα σε μια ακριβή φράση, μπορείτε να κάνετε _διαφυγή_ των εισαγωγικών προσθέτοντας μια ανάστροφη κάθετο (`\`) μπροστά από το εισαγωγικό, για παράδειγμα `"they said \"hello\" to each other"`.

Μπορείτε να ελέγξετε εάν θα επιστρέφονται αρχεία που περιέχουν _όλες_ τις λέξεις του όρου αναζήτησης ή _οποιαδήποτε_ από τις λέξεις:

- `meeting work` επιστρέφει αρχεία που περιέχουν τόσο `meeting` όσο και `work`.
- `meeting OR work` επιστρέφει αρχεία που περιέχουν είτε `meeting` είτε `work`.

Μπορείτε ακόμα να συνδυάσετε τα δύο στον ίδιο όρο αναζήτησης.

- `meeting work OR meetup personal` επιστρέφει αρχεία για εργασιακές συναντήσεις και προσωπικές συναντήσεις.

Μπορείτε να χρησιμοποιήσετε παρενθέσεις για να ελέγξετε την προτεραιότητα κάθε έκφρασης.

- `meeting (work OR meetup) personal` επιστρέφει αρχεία που περιέχουν `meeting`, `personal` και είτε `work` είτε `meetup`.

Για να εξαιρέσετε ή να αρνηθείτε μια λέξη από τα αποτελέσματα αναζήτησης, προσθέστε μια παύλα (`-`) μπροστά της:

- `meeting -work` επιστρέφει αρχεία που περιέχουν `meeting` αλλά όχι `work`.

Μπορείτε να εξαιρέσετε πολλαπλές εκφράσεις:

- `meeting -work -meetup` επιστρέφει αρχεία που περιέχουν `meeting` αλλά όχι `work` ή `meetup`.

Μπορείτε να εξαιρέσετε έναν συνδυασμό εκφράσεων χρησιμοποιώντας παρενθέσεις:

- `meeting -(work meetup)` επιστρέφει αρχεία που περιέχουν `meeting` αλλά όχι _και τα δύο_ `work` και `meetup`.

Για να φιλτράρετε αποτελέσματα χρησιμοποιώντας τους τελεστές μικρότερο (`<`) και μεγαλύτερο (`>`), περικλείστε τους σε αγκύλες (`[]`) ή εισαγωγικά (`""`):

- `meeting [duration:<5]` επιστρέφει αρχεία όπου υπάρχει η λέξη meeting και η διάρκεια είναι μικρότερη από 5.
- `meeting [duration:>5]` επιστρέφει αρχεία όπου υπάρχει η λέξη meeting και η διάρκεια είναι μεγαλύτερη από 5.

> [!tip]- Επεξήγηση όρου αναζήτησης
> Εάν χρειάζεται να αποσφαλματώσετε έναν σύνθετο όρο αναζήτησης, μπορείτε να κάνετε κλικ στο **Επεξήγηση όρου αναζήτησης** στην Αναζήτηση για μια εξήγηση του όρου αναζήτησής σας.

## Τελεστές αναζήτησης

Οι τελεστές αναζήτησης επιτρέπουν πιο λεπτομερείς όρους αναζήτησης για να φιλτράρετε τα αποτελέσματά σας ακόμη περισσότερο.

Ορισμένοι τελεστές σας επιτρέπουν ακόμη να προσθέσετε ένα ένθετο ερώτημα αναζήτησης μέσα σε παρενθέσεις, για παράδειγμα: `task:(call OR email)`.

| Τελεστής αναζήτησης | Περιγραφή                                                                                                                                                                                                                                                                                                        |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `file:`              | Εύρεση κειμένου στο όνομα αρχείου. Αντιστοιχεί σε οποιοδήποτε αρχείο στο θησαυροφυλάκιο.<p/>Παράδειγμα: `file:.jpg` ή `file:202209`.                                                                                                                                                                            |
| `path:`              | Εύρεση κειμένου στη διαδρομή αρχείου. Αντιστοιχεί σε οποιοδήποτε αρχείο στο θησαυροφυλάκιο.<p/>Παράδειγμα: `path:"Daily notes/2022-07"`.                                                                                                                                                                        |
| `content:`           | Εύρεση κειμένου στο περιεχόμενο αρχείου.<p/>Παράδειγμα: `content:"happy cat"`.                                                                                                                                                                                                                                    |
| `match-case:`        | Αντιστοίχιση με διάκριση πεζών-κεφαλαίων.<p/>Παράδειγμα: `match-case:HappyCat`.                                                                                                                                                                                                                                  |
| `ignore-case:`       | Αντιστοίχιση χωρίς διάκριση πεζών-κεφαλαίων.<p/>Παράδειγμα: `ignore-case:ikea`.                                                                                                                                                                                                                                  |
| `tag:`               | Εύρεση ετικέτας σε αρχείο.<p/>Παράδειγμα: `tag:#work`.<p/>Λάβετε υπόψη ότι η αναζήτηση `tag:#work` δεν θα επιστρέψει αποτελέσματα για `#myjob/work`.<br /><br />**Σημείωση**: Δεδομένου ότι ο `tag:` αγνοεί αντιστοιχίες σε μπλοκ κώδικα και σε μη-Markdown περιεχόμενο, είναι συχνά ταχύτερος και ακριβέστερος από μια κανονική αναζήτηση πλήρους κειμένου για `#work`. |
| `line:`              | Εύρεση αρχείων που περιέχουν τουλάχιστον μία γραμμή που αντιστοιχεί στο `x`.<p/>Παράδειγμα: `line:(mix flour)`.<p/><br>**Σημείωση:** Η χρήση `-line` αντιστρέφει την αναζήτηση, δηλαδή βρίσκει αρχεία όπου καμία γραμμή δεν αντιστοιχεί στο `x`.                                                                |
| `block:`             | Εύρεση αντιστοιχιών στο ίδιο μπλοκ.<p/>Παράδειγμα: `block:(dog cat)`.<p/>**Σημείωση**: Δεδομένου ότι ο `block:` απαιτεί από την Αναζήτηση να αναλύσει το περιεχόμενο Markdown σε κάθε αρχείο, μπορεί να κάνει τον όρο αναζήτησής σας να χρειαστεί περισσότερο χρόνο για να ολοκληρωθεί.                           |
| `section:`           | Εύρεση αντιστοιχιών στην ίδια ενότητα (κείμενο μεταξύ δύο επικεφαλίδων).<p/>Παράδειγμα: `section:(dog cat)`.                                                                                                                                                                                                     |
| `task:`              | Εύρεση αντιστοιχιών σε μια [[Βασική σύνταξη μορφοποίησης#Task lists\|εργασία]] σε επίπεδο μπλοκ.<p/>Παράδειγμα: `task:call`.                                                                                                                                                                                     |
| `task-todo:`         | Εύρεση αντιστοιχιών σε μια *μη ολοκληρωμένη* [[Βασική σύνταξη μορφοποίησης#Task lists\|εργασία]] σε επίπεδο μπλοκ.<p/>Παράδειγμα: `task-todo:call`.                                                                                                                                                              |
| `task-done:`         | Εύρεση αντιστοιχιών σε μια *ολοκληρωμένη* [[Βασική σύνταξη μορφοποίησης#Task lists\|εργασία]] σε επίπεδο μπλοκ.<p/>Παράδειγμα: `task-done:call`.                                                                                                                                                                 |

## Αναζήτηση ιδιοτήτων

Μπορείτε να χρησιμοποιήσετε δεδομένα αποθηκευμένα στις [[Ιδιότητες]] στους όρους αναζήτησής σας.

Χρησιμοποιήστε αγκύλες γύρω από ένα όνομα ιδιότητας `[property]` για να επιστρέψετε αρχεία με αυτή την ιδιότητα:

- `[aliases]` επιστρέφει αρχεία που περιέχουν την ιδιότητα `aliases`

Χρησιμοποιήστε αγκύλες και άνω-κάτω τελεία `[property:value]` για να επιστρέψετε αρχεία με αυτή την ιδιότητα και τιμή:

- `[aliases:Name]` επιστρέφει αρχεία όπου η τιμή της ιδιότητας `aliases` είναι `Name`

Χρησιμοποιήστε `null` ως τιμή για να βρείτε ιδιότητες που δεν έχουν τιμή:

- `[aliases:null]` επιστρέφει αρχεία όπου η ιδιότητα `aliases` υπάρχει αλλά δεν έχει τιμή

> [!info]+ Κενές τιμές
> Ο τελεστής `null` λειτουργεί όταν μια ιδιότητα είναι κενή (π.χ. `aliases: `), αλλά όχι όταν η ιδιότητα περιέχει κενά εισαγωγικά (`""`) ή κενές αγκύλες (`[]`).

Τόσο η ιδιότητα όσο και η τιμή επιτρέπουν υπο-ερωτήματα, όπως παρενθέσεις για ομαδοποίηση, τον τελεστή `OR`, διπλά εισαγωγικά για ακριβή αντιστοίχιση και κανονικές εκφράσεις.

- `[status:Draft OR Published]` επιστρέφει αρχεία όπου η τιμή της ιδιότητας `status` είναι `Draft` ή `Published`

## Αλλαγή διάκρισης πεζών-κεφαλαίων

Από προεπιλογή, οι όροι αναζήτησης δεν κάνουν διάκριση πεζών-κεφαλαίων. Εάν θέλετε να αναζητήσετε με ακριβή αντιστοίχιση πεζών-κεφαλαίων, επιλέξτε **Ταίριασμα πεζών-κεφαλαίων** ![[obsidian-icon-upper-lowercase.svg#icon]] μέσα στη γραμμή αναζήτησης.

Αυτή η ρύθμιση μπορεί να εναλλαχθεί. Εάν το εικονίδιο **Ταίριασμα πεζών-κεφαλαίων** είναι επισημασμένο, αυτό σημαίνει ότι εκτελείτε αυτή τη στιγμή αναζήτηση με διάκριση πεζών-κεφαλαίων.

## Αλλαγή σειράς ταξινόμησης αποτελεσμάτων

1. Εισάγετε έναν [[#Όροι αναζήτησης|όρο αναζήτησης]].
2. Κάτω από το πεδίο αναζήτησης, επιλέξτε το αναπτυσσόμενο μενού στα δεξιά.
3. Επιλέξτε τη σειρά ταξινόμησης που θέλετε. Η προεπιλογή είναι "Ταξιν. κατά όνομα αρχείου (A στο Ω)".

Οι ακόλουθες επιλογές είναι διαθέσιμες:

- Ταξιν. κατά όνομα αρχείου (A στο Ω)
- Ταξιν. κατά όνομα αρχείου (Ω στο Α)
- Ταξιν. κατά ημερομηνία αλλαγής (πιο πρόσφατο)
- Ταξιν. κατά ημερομηνία αλλαγής (παλαιότερο)
- Ημερομηνία δημιουργίας (πιο πρόσφατο)
- Ημερομηνία δημιουργίας (παλαιότερο)

## Αντιγραφή αποτελεσμάτων αναζήτησης

1. Εισάγετε έναν [[#Όροι αναζήτησης|όρο αναζήτησης]].
2. Κάτω από το πεδίο αναζήτησης, επιλέξτε το εικονίδιο με τις τρεις τελείες δίπλα στον αριθμό αποτελεσμάτων.
3. Επιλέξτε **Αντιγραφή αποτελέσματος αναζήτησης**.

## Χρήση κανονικών εκφράσεων

Μια κανονική έκφραση είναι ένα σύνολο χαρακτήρων που περιγράφει ένα μοτίβο κειμένου. Για να χρησιμοποιήσετε κανονικές εκφράσεις στον όρο αναζήτησής σας, περικλείστε την έκφραση με κάθετες (`/`).

- `/\d{4}-\d{2}-\d{2}/` αντιστοιχεί σε μια ημερομηνία ISO 8601, όπως 2022-01-01.

Μπορείτε ακόμα να συνδυάσετε κανονικές εκφράσεις με τελεστές αναζήτησης:

- `path:/\d{4}-\d{2}-\d{2}/` επιστρέφει αρχεία με ημερομηνία στη διαδρομή αρχείου.

Για περισσότερες πληροφορίες σχετικά με τη σύνταξη κανονικών εκφράσεων, ανατρέξτε στον [Πρακτικό οδηγό Regex](https://www.freecodecamp.org/news/practical-regex-guide-with-real-life-examples/) του FreeCodeCamp ή στις [Κανονικές εκφράσεις](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Regular_Expressions) της Mozilla.

> [!info]+ Κανονικές εκφράσεις τύπου JavaScript
> Οι κανονικές εκφράσεις υπάρχουν σε διαφορετικές παραλλαγές που μπορεί να φαίνονται διαφορετικές μεταξύ τους. Το Obsidian χρησιμοποιεί κανονικές εκφράσεις τύπου JavaScript.

## Ρύθμιση παραμέτρων αναζήτησης

Για να ρυθμίσετε την Αναζήτηση, επιλέξτε **Ρυθμίσεις αναζήτησης** ![[lucide-sliders-horizontal.svg#icon]] στη δεξιά πλευρά της γραμμής αναζήτησης για να δείτε τις εναλλαγές.

| Ρύθμιση                         | Περιγραφή                                                                   |
|----------------------------------|-----------------------------------------------------------------------------|
| **Επεξήγηση όρου αναζήτησης**   | Αναλύει τους όρους αναζήτησης και τους εξηγεί σε απλό κείμενο.             |
| **Κλείσιμο αποτελεσμάτων**      | Εναλλάσσει εάν θα εμφανίζεται το πλαίσιο αναζήτησης.                       |
| **Προβολή παραπάνω πλαισίου**   | Επεκτείνει το αποτέλεσμα αναζήτησης για να εμφανίσει περισσότερο κείμενο γύρω από την αντιστοίχιση. |

## Ενσωμάτωση αποτελεσμάτων αναζήτησης σε μια σημείωση

Για να ενσωματώσετε αποτελέσματα αναζήτησης σε μια σημείωση, προσθέστε ένα μπλοκ κώδικα `query`:

````
```query
embed OR search
```
````

Το [[Εισαγωγή στο Obsidian Publish|Obsidian Publish]] δεν υποστηρίζει ενσωματωμένα [[Περιορισμοί Publish#Search|αποτελέσματα αναζήτησης]]. Για να δείτε ένα ζωντανό παράδειγμα απόδοσης, χρησιμοποιήστε το παραπάνω μπλοκ κώδικα μέσα στο θησαυροφυλάκιό σας.
