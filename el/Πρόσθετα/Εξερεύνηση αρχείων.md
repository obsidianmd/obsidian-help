Η εξερεύνηση αρχείων είναι ένα [[Κύρια πρόσθετα|βασικό πρόσθετο]] που σας επιτρέπει να διαχειρίζεστε αρχεία και φακέλους μέσα στο θησαυροφυλάκιό σας. Μπορείτε να περιηγηθείτε σε σημειώσεις και άλλες [[Αποδεκτές μορφές αρχείων]] στο θησαυροφυλάκιό σας και να εκτελέσετε πολλές κοινές λειτουργίες αρχείων:

- Δημιουργία, διαγραφή και μετονομασία αρχείων και φακέλων.
- Μετακίνηση αρχείων και φακέλων με μεταφορά και απόθεση.
- Χρήση του μενού περιβάλλοντος για πρόσβαση σε όλες τις διαθέσιμες λειτουργίες.

> [!tip]- Μεταφορά και απόθεση αρχείων
> Μπορείτε να σύρετε ένα αρχείο από την εξερεύνηση αρχείων στη σημείωσή σας για να δημιουργήσετε έναν σύνδεσμο προς αυτό, ή να σύρετε ένα αρχείο σε έναν φάκελο στην εξερεύνηση αρχείων για να το αντιγράψετε.

## Δημιουργία νέας σημείωσης

Για να δημιουργήσετε μια νέα σημείωση στην προεπιλεγμένη τοποθεσία για νέες σημειώσεις:

1. Κάντε κλικ στο **Νέα σημείωση** ![[lucide-pen-line.svg#icon]] στο πάνω μέρος της εξερεύνησης αρχείων.
2. Πληκτρολογήστε το όνομα της σημείωσης και μετά πατήστε Enter.

> [!tip]- Αλλαγή προεπιλεγμένης τοποθεσίας
> Μπορείτε να αλλάξετε την προεπιλεγμένη τοποθεσία για νέες σημειώσεις στις **[[Ρυθμίσεις]] → [[Ρυθμίσεις#Αρχείο|Αρχείο]] → [[Ρυθμίσεις#Tοποθεσία για καινούργιες σημειώσεις|Tοποθεσία για καινούργιες σημειώσεις]]**.

Για να δημιουργήσετε μια νέα σημείωση σε συγκεκριμένο φάκελο:

1. Κάντε δεξί κλικ στον φάκελο και μετά κάντε κλικ στο **Νέα σημείωση**.
2. Πληκτρολογήστε το όνομα της σημείωσης και μετά πατήστε Enter.

## Δημιουργία νέου φακέλου

Για να δημιουργήσετε έναν νέο φάκελο στη ρίζα του θησαυροφυλακίου σας:

1. Κάντε κλικ στο **Νέος φάκελος** ![[lucide-folder-plus.svg#icon]] στο πάνω μέρος της εξερεύνησης αρχείων.
2. Πληκτρολογήστε το όνομα του φακέλου και μετά πατήστε Enter.

Για να δημιουργήσετε υποφάκελο:

1. Κάντε δεξί κλικ στον φάκελο μέσα στον οποίο θέλετε να δημιουργήσετε τον υποφάκελο και μετά κάντε κλικ στο **Νέος φάκελος**.
2. Πληκτρολογήστε το όνομα του φακέλου και μετά πατήστε Enter.

## Αλλαγή σειράς ταξινόμησης

Για να αλλάξετε τη σειρά ταξινόμησης των αρχείων σας:

1.  Επιλέξτε **Αλλαγή σειράς ταξινόμησης** ![[lucide-arrow-up-narrow-wide.svg#icon]] στο πάνω μέρος της εξερεύνησης αρχείων.
2. Επιλέξτε πώς θέλετε να ταξινομηθούν τα αρχεία σας. Μπορείτε να ταξινομήσετε σε αύξουσα ή φθίνουσα σειρά κατά όνομα αρχείου, χρόνος τροποποίησης ή χρόνος δημιουργίας.

## Αυτόματη αποκάλυψη ενεργού αρχείου

Όταν ανοίγετε μια σημείωση, η εξερεύνηση αρχείων μπορεί αυτόματα να κάνει κύλιση και να επισημάνει αυτή τη σημείωση στο δέντρο φακέλων. Αυτό σας βοηθά να παρακολουθείτε πού βρίσκεται η ενεργή σημείωσή σας μέσα στο θησαυροφυλάκιό σας.

Για εναλλαγή της αυτόματης αποκάλυψης:

- Επιλέξτε **Αυτόματη αποκάλυψη ενεργού αρχείου** ![[lucide-gallery-vertical.svg#icon]] στο πάνω μέρος της εξερεύνησης αρχείων.

Όταν είναι ενεργοποιημένο, η εξερεύνηση αρχείων θα ακολουθεί αυτόματα και θα αποκαλύπτει την τρέχουσα ανοιχτή σημείωση.

## Ανάπτυξη ή σύμπτυξη όλων των φακέλων

Μπορείτε να αναπτύξετε ή να συμπτύξετε όλους τους φακέλους στην εξερεύνηση αρχείων ταυτόχρονα.

Για ανάπτυξη όλων των φακέλων:

- Επιλέξτε **Ανάπτυξη όλων** ![[lucide-chevrons-up-down.svg#icon]] στο πάνω μέρος της εξερεύνησης αρχείων.

Για σύμπτυξη όλων των φακέλων:

- Επιλέξτε **Σύμπτυξη όλων** ![[lucide-chevrons-down-up.svg#icon]] στο πάνω μέρος της εξερεύνησης αρχείων.

## Διαγραφή αρχείου ή φακέλου

1. Κάντε δεξί κλικ στο αρχείο που θέλετε να διαγράψετε και μετά κάντε κλικ στο **Διαγραφή**.
2. Εάν σας ζητηθεί να επιβεβαιώσετε ότι θέλετε να διαγράψετε το αρχείο, κάντε κλικ στο **Διαγραφή**.

Για περισσότερες πληροφορίες, ανατρέξτε στο [[Διαχείριση σημειώσεων#Διαγραφή σημείωσης|Διαγραφή σημείωσης]].

## Μετονομασία αρχείου ή φακέλου

1. Κάντε δεξί κλικ στο αρχείο που θέλετε να μετονομάσετε και μετά κάντε κλικ στο **Μετονομασία**.
2. Πληκτρολογήστε το νέο όνομα και μετά πατήστε Enter.

Για περισσότερες πληροφορίες, ανατρέξτε στο [[Διαχείριση σημειώσεων#Μετονομασία σημείωσης|Μετονομασία σημείωσης]].

## Μετακίνηση αρχείου ή φακέλου

Για να μετακινήσετε ένα αρχείο ή φάκελο, μπορείτε να χρησιμοποιήσετε μεταφορά και απόθεση ή το μενού περιβάλλοντος.

**Μεταφορά και απόθεση:**

- Σύρετε ένα αρχείο ή φάκελο στον φάκελο στον οποίο θέλετε να το μετακινήσετε.
- Με `Alt-Click` (Windows/Linux) ή `Opt-Click` (macOS) μπορείτε να επιλέξετε πολλαπλά μεμονωμένα αρχεία και να τα σύρετε σε άλλον φάκελο. Εάν βρίσκονται όλα σε σειρά, μπορείτε να χρησιμοποιήσετε `Shift-Click` για αυτό.

**Μενού περιβάλλοντος:**

1. Κάντε δεξί κλικ σε ένα αρχείο και μετά επιλέξτε **Μετακίνηση αρχείου σε...**.
2. Αναζητήστε το όνομα του φακέλου στον οποίο θέλετε να μετακινήσετε το αρχείο και μετά επιλέξτε τον από τη λίστα.
