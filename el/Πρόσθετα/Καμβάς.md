Ο Καμβάς είναι ένα [[Κύρια πρόσθετα|βασικό πρόσθετο]] για οπτική σημειογραφία. Σας δίνει άπειρο χώρο για να τοποθετήσετε σημειώσεις και να τις συνδέσετε με άλλες σημειώσεις, συνημμένα και ιστοσελίδες.

Η οπτική σημειογραφία σας βοηθά να κατανοήσετε τις σημειώσεις σας οργανώνοντάς τες σε δισδιάστατο χώρο. Συνδέστε σημειώσεις με γραμμές και ομαδοποιήστε σχετικές σημειώσεις μαζί για να κατανοήσετε καλύτερα τη σχέση μεταξύ τους.

Τα δεδομένα Canvas που δημιουργείτε στο Obsidian αποθηκεύονται ως αρχεία `.canvas` χρησιμοποιώντας την ανοιχτή μορφή αρχείου [JSON Canvas](https://jsoncanvas.org/).

## Δημιουργία νέου καμβά

Για να ξεκινήσετε να χρησιμοποιείτε τον Καμβά, πρέπει πρώτα να δημιουργήσετε ένα αρχείο για να φιλοξενήσει τον καμβά σας. Μπορείτε να δημιουργήσετε έναν νέο καμβά χρησιμοποιώντας τις ακόλουθες μεθόδους.

**Παλέτα εντολών:**

1. Ανοίξτε την [[Παλέτα εντολών]].
2. Επιλέξτε **Canvas: Δημιουργία νέου καμβά** για να δημιουργήσετε έναν καμβά στον ίδιο φάκελο με το ενεργό αρχείο.

**Εξερεύνηση αρχείων:**

- Στην [[Εξερεύνηση αρχείων]], κάντε δεξί κλικ στον φάκελο στον οποίο θέλετε να δημιουργήσετε τον καμβά.
- Επιλέξτε **Νέος καμβάς**.

**Κορδέλα:**

- Στο κατακόρυφο μενού κορδέλας, επιλέξτε **Δημιουργία νέου καμβά** ![[lucide-layout-dashboard.svg#icon]] για να δημιουργήσετε έναν καμβά στον ίδιο φάκελο με το ενεργό αρχείο.

> [!note] Η επέκταση αρχείου .canvas
> Το Obsidian αποθηκεύει τα δεδομένα του καμβά σας ως αρχεία `.canvas` χρησιμοποιώντας μια ανοιχτή μορφή αρχείου που ονομάζεται [JSON Canvas](https://jsoncanvas.org/).

## Προσθήκη καρτών

Μπορείτε να σύρετε αρχεία στον καμβά σας από το Obsidian ή από άλλες εφαρμογές. Για παράδειγμα, αρχεία Markdown, εικόνες, ήχο, PDF ή ακόμα και μη αναγνωρισμένους τύπους αρχείων.

### Προσθήκη καρτών κειμένου

Μπορείτε να προσθέσετε κάρτες μόνο κειμένου που δεν αναφέρονται σε αρχείο. Μπορείτε να χρησιμοποιήσετε Markdown, συνδέσμους και μπλοκ κώδικα όπως ακριβώς σε μια σημείωση.

Για να προσθέσετε μια νέα κάρτα κειμένου στον καμβά σας:

- Επιλέξτε ή σύρετε το εικονίδιο κενού αρχείου στο κάτω μέρος του καμβά.

Μπορείτε επίσης να προσθέσετε κάρτες κειμένου κάνοντας διπλό κλικ στον καμβά.

Για να μετατρέψετε μια κάρτα κειμένου σε αρχείο:

1. Κάντε δεξί κλικ στην κάρτα κειμένου και στη συνέχεια επιλέξτε **Μετατροπή σε αρχείο...**.
2. Εισαγάγετε το όνομα σημείωσης και στη συνέχεια επιλέξτε **Αποθήκευση**.

> [!note] Σημείωση
> Οι κάρτες μόνο κειμένου δεν εμφανίζονται στους [[Οπισθοσύνδεσμοι|Οπισθοσυνδέσμους]]. Για να εμφανιστούν, πρέπει να τις μετατρέψετε σε αρχείο.

### Προσθήκη καρτών από σημειώσεις

Για να προσθέσετε μια σημείωση από το θησαυροφυλάκιό σας στον καμβά σας:

1. Επιλέξτε ή σύρετε το εικονίδιο εγγράφου στο κάτω μέρος του καμβά.
2. Επιλέξτε τη σημείωση που θέλετε να προσθέσετε.

Μπορείτε επίσης να προσθέσετε σημειώσεις από το μενού περιβάλλοντος του καμβά:

1. Κάντε δεξί κλικ στον καμβά και στη συνέχεια επιλέξτε **Προσθήκη σημείωσης από φυλάκιο**.
2. Επιλέξτε τη σημείωση που θέλετε να προσθέσετε.

Ή, μπορείτε να τις προσθέσετε στον καμβά σύροντας το αρχείο από την [[Εξερεύνηση αρχείων]].

### Προσθήκη καρτών από πολυμέσα

Για να προσθέσετε πολυμέσα από το θησαυροφυλάκιό σας στον καμβά σας:

1. Επιλέξτε ή σύρετε το εικονίδιο αρχείου εικόνας στο κάτω μέρος του καμβά.
2. Επιλέξτε το αρχείο πολυμέσου που θέλετε να προσθέσετε.

Μπορείτε επίσης να προσθέσετε πολυμέσα από το μενού περιβάλλοντος του καμβά:

1. Κάντε δεξί κλικ στον καμβά και στη συνέχεια επιλέξτε **Προσθήκη πολυμέσου από φυλάκιο**.
2. Επιλέξτε το αρχείο πολυμέσου που θέλετε να προσθέσετε.

Ή, μπορείτε να τα προσθέσετε στον καμβά σύροντας το αρχείο από την [[Εξερεύνηση αρχείων]].

### Προσθήκη καρτών από ιστοσελίδες

Για να ενσωματώσετε μια ιστοσελίδα στον καμβά σας:

1. Κάντε δεξί κλικ στον καμβά και στη συνέχεια επιλέξτε **Προσθήκη ιστοσελίδας**.
2. Εισαγάγετε το URL της ιστοσελίδας και στη συνέχεια επιλέξτε **Αποθήκευση**.

Μπορείτε επίσης να επιλέξετε ένα URL στο πρόγραμμα περιήγησής σας και στη συνέχεια να το σύρετε στον καμβά για να το ενσωματώσετε σε μια κάρτα.

Για να ανοίξετε την ιστοσελίδα στο πρόγραμμα περιήγησής σας, πατήστε `Ctrl` (ή `Cmd` σε macOS) και επιλέξτε την ετικέτα της κάρτας. Ή, κάντε δεξί κλικ στην κάρτα και επιλέξτε **Άνοιγμα στο πρόγραμμα περιήγησης**.

### Προσθήκη καρτών από φακέλους

Σύρετε έναν φάκελο από την εξερεύνηση αρχείων για να προσθέσετε όλα τα αρχεία σε αυτόν τον φάκελο στον καμβά.

### Επεξεργασία κάρτας

Κάντε διπλό κλικ σε μια κάρτα κειμένου ή σημείωσης για να ξεκινήσετε την επεξεργασία της. Κάντε κλικ εκτός της κάρτας για να σταματήσετε την επεξεργασία. Μπορείτε επίσης να πατήσετε `Escape` για να σταματήσετε την επεξεργασία μιας κάρτας.

Μπορείτε επίσης να επεξεργαστείτε μια κάρτα κάνοντας δεξί κλικ πάνω της και επιλέγοντας **Επεξεργασία**.

### Διαγραφή κάρτας

Αφαιρέστε τις επιλεγμένες κάρτες κάνοντας δεξί κλικ σε οποιαδήποτε από αυτές και στη συνέχεια επιλέγοντας **Διαγραφή**. Ή, πατήστε `Backspace` (ή `Delete` σε macOS).

Μπορείτε επίσης να επιλέξετε **Αφαίρεση** ![[lucide-trash-2.svg#icon]] στα στοιχεία ελέγχου επιλογής πάνω από την επιλογή σας.

### Εναλλαγή καρτών

Μπορείτε να αντικαταστήσετε μια κάρτα σημείωσης ή πολυμέσων με μια άλλη κάρτα του ίδιου τύπου.

Για να αντικαταστήσετε μια κάρτα σημείωσης:

1. Κάντε δεξί κλικ στην κάρτα που θέλετε να αντικαταστήσετε.
2. Επιλέξτε **Εναλλαγή αρχείου**.
3. Επιλέξτε τη σημείωση με την οποία θέλετε να αντικαταστήσετε.

## Επιλογή καρτών

Επιλέξτε κάρτες στον καμβά κάνοντας κλικ πάνω τους. Μπορείτε να επιλέξετε πολλαπλές κάρτες σύροντας μια επιλογή γύρω τους.

Μπορείτε επίσης να προσθέσετε και να αφαιρέσετε κάρτες από μια υπάρχουσα επιλογή πατώντας `Shift` και επιλέγοντάς τες.

Πατήστε `Ctrl+a` (ή `Cmd+a` σε macOS) για να επιλέξετε όλες τις κάρτες στον καμβά.

Για να κάνετε κύλιση στο περιεχόμενο μιας κάρτας, πρέπει πρώτα να την επιλέξετε.

### Τακτοποίηση καρτών

Σύρετε μια επιλεγμένη κάρτα για να τη μετακινήσετε.

Πατήστε `Alt` (ή `Option` σε macOS) και σύρετε για να διπλασιάσετε την επιλογή.

Μπορείτε να πατήσετε `Shift` κατά τη μεταφορά για να μετακινηθείτε μόνο σε μία κατεύθυνση.

Πατήστε `Space` κατά τη μετακίνηση μιας επιλογής για να απενεργοποιήσετε την αυτόματη στοίχιση.

Η επιλογή μιας κάρτας τη μετακινεί στο προσκήνιο.

### Αλλαγή μεγέθους κάρτας

Σύρετε οποιαδήποτε πλευρά μιας κάρτας για να αλλάξετε το μέγεθός της.

Μπορείτε να πατήσετε `Space` κατά την αλλαγή μεγέθους για να απενεργοποιήσετε την αυτόματη στοίχιση.

Για να διατηρήσετε την αναλογία διαστάσεων κατά την αλλαγή μεγέθους, πατήστε `Shift` κατά την αλλαγή μεγέθους.

## Σύνδεση καρτών

Σχεδιάστε γραμμές μεταξύ καρτών για να δημιουργήσετε σχέσεις μεταξύ τους. Χρησιμοποιήστε χρώματα και ετικέτες για να περιγράψετε πώς σχετίζονται μεταξύ τους.

### Σύνδεση δύο καρτών

Για να συνδέσετε δύο κάρτες με μια κατευθυνόμενη γραμμή:

1. Τοποθετήστε τον δρομέα πάνω σε μία από τις πλευρές μιας κάρτας μέχρι να δείτε έναν γεμάτο κύκλο.
2. Σύρετε τον κύκλο στην πλευρά μιας διαφορετικής κάρτας για να τις συνδέσετε.

> [!tip] Συμβουλή
> Εάν σύρετε τη γραμμή χωρίς να τη συνδέσετε σε άλλη κάρτα, μπορείτε στη συνέχεια να προσθέσετε την κάρτα στην οποία θέλετε να τη συνδέσετε.

### Αποσύνδεση δύο καρτών

Για να αφαιρέσετε τη σύνδεση μεταξύ δύο καρτών:

1. Τοποθετήστε τον δρομέα πάνω σε μια γραμμή σύνδεσης μέχρι να εμφανιστούν δύο μικροί κύκλοι στη γραμμή.
2. Σύρετε έναν από τους κύκλους μακριά από την κάρτα χωρίς να τον συνδέσετε σε άλλη.

Μπορείτε επίσης να αποσυνδέσετε δύο κάρτες κάνοντας δεξί κλικ στη γραμμή μεταξύ τους και στη συνέχεια επιλέγοντας **Αφαίρεση**. Ή, επιλέγοντας τη γραμμή και στη συνέχεια πατώντας `Backspace` (ή `Delete` σε macOS).

### Σύνδεση μιας κάρτας σε διαφορετική κάρτα

Για να μετακινήσετε ένα από τα άκρα μιας γραμμής σύνδεσης:

1. Τοποθετήστε τον δρομέα πάνω σε μια γραμμή σύνδεσης μέχρι να εμφανιστούν δύο μικροί κύκλοι στη γραμμή.
2. Σύρετε τον κύκλο πάνω από το άκρο που θέλετε να επανασυνδέσετε, σε μια άλλη κάρτα.

### Πλοήγηση σε σύνδεση

Εάν δύο συνδεδεμένες κάρτες απέχουν πολύ μεταξύ τους, μπορείτε να πλοηγηθείτε στην πηγή ή στον στόχο της σύνδεσης κάνοντας δεξί κλικ στη γραμμή και στη συνέχεια επιλέγοντας **Μετάβαση στον στόχο** ή **Μετάβαση στην πηγή**.

### Προσθήκη ετικέτας σε σύνδεση

Μπορείτε να προσθέσετε μια ετικέτα σε μια γραμμή για να περιγράψετε τη σχέση μεταξύ δύο καρτών.

Για να προσθέσετε ετικέτα σε μια σύνδεση:

1. Κάντε διπλό κλικ στη γραμμή.
2. Εισαγάγετε την ετικέτα και στη συνέχεια πατήστε `Escape` ή κάντε κλικ οπουδήποτε στον καμβά.

Μπορείτε επίσης να προσθέσετε ετικέτα σε μια σύνδεση επιλέγοντάς τη και στη συνέχεια επιλέγοντας **Επεξεργασία ετικέτας** από τα στοιχεία ελέγχου επιλογής.

Για να επεξεργαστείτε μια ετικέτα σύνδεσης, κάντε διπλό κλικ στη γραμμή ή κάντε δεξί κλικ στη γραμμή και στη συνέχεια επιλέξτε **Επεξεργασία ετικέτας**.

### Αλλαγή χρώματος κάρτας ή σύνδεσης

1. Επιλέξτε τις κάρτες ή τις συνδέσεις στις οποίες θέλετε να αλλάξετε χρώμα.
2. Στα στοιχεία ελέγχου επιλογής, επιλέξτε **Ορισμός χρώματος** ![[lucide-palette.svg#icon]].
3. Επιλέξτε ένα χρώμα.

## Ομαδοποίηση καρτών

### Ομαδοποίηση επιλεγμένων καρτών

Για να δημιουργήσετε μια κενή ομάδα:

- Κάντε δεξί κλικ στον καμβά και στη συνέχεια επιλέξτε **Δημιουργία ομάδας**.

Για να ομαδοποιήσετε σχετικές κάρτες:

1. Επιλέξτε τις κάρτες.
2. Κάντε δεξί κλικ σε οποιαδήποτε από τις επιλεγμένες κάρτες και στη συνέχεια επιλέξτε **Δημιουργία ομάδας**.

**Μετονομασία ομάδας:** Κάντε διπλό κλικ στο όνομα της ομάδας για να το επεξεργαστείτε και στη συνέχεια πατήστε `Enter` για αποθήκευση.

## Πλοήγηση στον καμβά

Καθώς αρχίζετε να προσθέτετε περισσότερες κάρτες στον καμβά σας, θα θέλετε να κατανοήσετε πώς μπορείτε να πλοηγηθείτε στον καμβά για να δείτε ένα μέρος του. Μάθετε πώς να κάνετε παν και μεγέθυνση για να μετακινηθείτε στον καμβά με ευκολία.

### Παν στον καμβά

Για να μετακινήσετε τον καμβά κατακόρυφα και οριζόντια, γνωστό και ως _παν_, μπορείτε να χρησιμοποιήσετε οποιαδήποτε από τις ακόλουθες προσεγγίσεις:

- Πατήστε `Space` και σύρετε τον καμβά.
- Σύρετε τον καμβά χρησιμοποιώντας το μεσαίο κουμπί του ποντικιού.
- Κάντε κύλιση με το ποντίκι για παν κατακόρυφα και πατήστε `Shift` κατά την κύλιση για παν οριζόντια.

### Μεγέθυνση του καμβά

Για μεγέθυνση του καμβά, πατήστε `Space` ή `Ctrl` (ή `Cmd` σε macOS) και κάντε κύλιση χρησιμοποιώντας τον τροχό του ποντικιού. Ή, επιλέξτε **Μεγέθυνση** ![[lucide-plus.svg#icon]] και **Σμίκρυνση** ![[lucide-minus.svg#icon]] από τα στοιχεία ελέγχου ζουμ στην πάνω δεξιά γωνία.

#### Ζουμ στο ταίριασμα

Για να κάνετε ζουμ στον καμβά ώστε κάθε στοιχείο να είναι ορατό, επιλέξτε **Ζουμ στο ταίριασμα** ![[lucide-maximize.svg#icon]]. Ή, χρησιμοποιήστε τη συντόμευση πληκτρολογίου `Shift+1`.

#### Ζουμ στην επιλογή

Για να κάνετε ζουμ στον καμβά ώστε όλα τα επιλεγμένα στοιχεία να είναι ορατά, κάντε δεξί κλικ σε μια επιλεγμένη κάρτα και στη συνέχεια επιλέξτε **Ζουμ στην επιλογή**. Ή, χρησιμοποιήστε μια συντόμευση πληκτρολογίου πατώντας `Shift+2`.

#### Επαναφορά εστίασης

Για να αλλάξετε το επίπεδο ζουμ πίσω στην προεπιλογή, επιλέξτε **Επαναφορά εστίασης** στα στοιχεία ελέγχου ζουμ στην πάνω δεξιά γωνία.
