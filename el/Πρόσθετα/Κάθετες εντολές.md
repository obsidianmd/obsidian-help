Οι εντολές κάθετου είναι ένα [[Κύρια πρόσθετα|βασικό πρόσθετο]] που σας επιτρέπει να εκτελείτε εντολές στον επεξεργαστή πληκτρολογώντας μια κάθετο (`/`) στην αρχή μιας γραμμής ή μετά από οποιοδήποτε κενό.

Για να εκτελέσετε μια εντολή χρησιμοποιώντας τις εντολές κάθετου:

1. Στον επεξεργαστή, πληκτρολογήστε μια κάθετο (`/`).
2. Εισαγάγετε το όνομα της εντολής που θέλετε να εκτελέσετε για να αναζητήσετε τη λίστα διαθέσιμων εντολών.
3. Πλοηγηθείτε στην εντολή χρησιμοποιώντας τα βελάκια.
4. Πατήστε Enter.

Για να βγείτε από την αναζήτηση εντολών κάθετου χωρίς να εκτελέσετε κάποια εντολή, πατήστε `Esc` ή το πλήκτρο `Space`.

> [!TIP] Ασαφής αντιστοίχιση
> Όπως και η [[Παλέτα εντολών]], οι εντολές κάθετου υποστηρίζουν ασαφή αντιστοίχιση που σας επιτρέπει να αναζητάτε εντολές ακόμα κι αν δεν γνωρίζετε τα ακριβή ονόματά τους. Για παράδειγμα, πληκτρολογήστε "scf" για να βρείτε την εντολή **Αποθήκευση τρέχοντος αρχείου**.
