Οι Διαφάνειες είναι ένα [[Κύρια πρόσθετα|βασικό πρόσθετο]] που σας επιτρέπει να δημιουργείτε παρουσιάσεις από τις σημειώσεις σας. Για να ξεκινήσετε μια παρουσίαση:

**Μενού αρχείου:**

- Κάντε δεξί κλικ στην καρτέλα μιας σημείωσης και κάντε κλικ στο **Έναρξη παρουσίασης**.

**Παλέτα εντολών:**

1. Πατήστε `Ctrl+P` (ή `Cmd+P` σε macOS) για να ανοίξετε την [[Παλέτα εντολών]].
2. Αναζητήστε την εντολή **Έναρξη παρουσίασης**.
3. Πατήστε `Enter` με την εντολή επιλεγμένη για να ξεκινήσετε μια παρουσίαση χρησιμοποιώντας την ενεργή σημείωση.

Για να πλοηγηθείτε μεταξύ των διαφανειών, κάντε κλικ στα αριστερά και δεξιά βέλη στην κάτω δεξιά γωνία, ή πατήστε τα πλήκτρα αριστερού και δεξιού βέλους στο πληκτρολόγιό σας.

Μπορείτε επίσης να προχωρήσετε στην επόμενη διαφάνεια πατώντας το πλήκτρο διαστήματος.

Για να σταματήσετε την παρουσίαση, πατήστε `Escape` ή κάντε κλικ στον σταυρό στην πάνω δεξιά γωνία της παρουσίασης.

Μπορείτε να χρησιμοποιήσετε οποιοδήποτε έγκυρο αρχείο Markdown ως παρουσίαση. Για να διαχωρίσετε τις διαφάνειες, εισάγετε `---` στην αρχή μιας γραμμής περιτριγυρισμένο από κενές γραμμές.

```md
# Παρουσιάσεις με Διαφάνειες

Μια επίδειξη για το πώς να δημιουργείτε παρουσιάσεις χρησιμοποιώντας τις Διαφάνειες.

---

## Μορφοποίηση

Μπορείτε να χρησιμοποιήσετε κανονική μορφοποίηση Markdown, όπως *πλάγιο* και **έντονο** κείμενο.

---

## Διαφάνειες

Χρησιμοποιήστε `---` για να διαχωρίσετε τις διαφάνειες.
```
