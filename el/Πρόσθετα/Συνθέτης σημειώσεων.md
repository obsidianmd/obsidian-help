Ο Συνθέτης σημειώσεων είναι ένα [[Κύρια πρόσθετα|βασικό πρόσθετο]] που σας επιτρέπει να συγχωνεύσετε δύο σημειώσεις ή να εξαγάγετε μέρος μιας σημείωσης σε μια νέα σημείωση.

## Συγχώνευση σημειώσεων

Η συγχώνευση σημειώσεων προσθέτει μια σημείωση σε μια άλλη και αφαιρεί την πρώτη. Ο Συνθέτης σημειώσεων ενημερώνει όλους τους συνδέσμους ώστε να αναφέρονται στη συγχωνευμένη σημείωση.

Όταν επιλέγετε τη σημείωση στην οποία θα γίνει η συγχώνευση, μπορείτε να επιλέξετε μεταξύ των ακόλουθων μεθόδων:

- `Enter`: Προσθέτει τη σημείωση προέλευσης στο _τέλος_ της σημείωσης προορισμού.
- `Shift+Enter`: Προσθέτει τη σημείωση προέλευσης στην _αρχή_ της σημείωσης προορισμού.
- `Ctrl+Enter` (ή `Cmd+Enter` σε macOS): Δημιουργεί μια νέα σημείωση με το περιεχόμενο της σημείωσης προέλευσης.

Για να συγχωνεύσετε την ενεργή σημείωση με μια άλλη σημείωση στο θησαυροφυλάκιό σας:

**Εξερεύνηση αρχείων**

1. Στην Εξερεύνηση αρχείων, κάντε δεξί κλικ στη σημείωση που θέλετε να συγχωνεύσετε.
2. Κάντε κλικ στο **Συγχώνευση ολόκληρου αρχείου με...**.
3. Επιλέξτε τη σημείωση στην οποία θέλετε να γίνει η συγχώνευση.
4. Κάντε κλικ στο **Συγχώνευση** για επιβεβαίωση.

**Παλέτα εντολών**

1. Ανοίξτε την [[Παλέτα εντολών]].
2. Επιλέξτε **Συνθέτης σημειώσεων: Συγχώνευση τρέχοντος αρχείου με άλλο αρχείο...**.
3. Επιλέξτε τη σημείωση στην οποία θέλετε να γίνει η συγχώνευση.
4. Κάντε κλικ στο **Συγχώνευση** για επιβεβαίωση.

> [!tip] Συμβουλή
> Από προεπιλογή, ο Συνθέτης σημειώσεων σας ζητά επιβεβαίωση κατά τη συγχώνευση σημειώσεων. Εάν απενεργοποιήσετε την επιβεβαίωση και συγχωνεύσετε μια σημείωση κατά λάθος, μπορείτε ακόμα να την ανακτήσετε με το πρόσθετο [[Ανάκτηση αρχείων]].

## Εξαγωγή σημείωσης

Όταν επιλέγετε τη σημείωση στην οποία θα εξαχθεί η επιλογή, μπορείτε να επιλέξετε μεταξύ των ακόλουθων μεθόδων:

- `Enter`: Προσθέτει το επιλεγμένο κείμενο στο _τέλος_ της σημείωσης προορισμού.
- `Shift+Enter`: Προσθέτει το επιλεγμένο κείμενο στην _αρχή_ της σημείωσης προορισμού.
- `Ctrl+Enter` (ή `Cmd+Enter` σε macOS): Δημιουργεί μια νέα σημείωση με το επιλεγμένο κείμενο.

Για να εξαγάγετε κείμενο σε μια νέα σημείωση:

**Επεξεργαστής**

1. Ενώ βρίσκεστε στη **Λειτουργία επεξεργασίας**, επιλέξτε το κείμενο που θέλετε να εξαγάγετε.
2. Κάντε δεξί κλικ στο επιλεγμένο κείμενο.
3. Κάντε κλικ στο **Εξαγωγή τρέχουσας επιλογής...**.
4. Επιλέξτε τη σημείωση στην οποία θέλετε να γίνει η εξαγωγή.

**Παλέτα εντολών**

1. Ενώ βρίσκεστε στη **Λειτουργία επεξεργασίας**, επιλέξτε το κείμενο που θέλετε να εξαγάγετε.
2. Ανοίξτε την [[Παλέτα εντολών]].
3. Επιλέξτε **Συνθέτης σημειώσεων: Εξαγωγή τρέχουσας επιλογής...**.
4. Επιλέξτε τη σημείωση στην οποία θέλετε να γίνει η εξαγωγή.

> [!tip] Συμβουλή
> Από προεπιλογή, ο Συνθέτης σημειώσεων αντικαθιστά το εξαγόμενο κείμενο με έναν σύνδεσμο προς τη σημείωση προορισμού. Στις ρυθμίσεις, μπορείτε επίσης να αλλάξετε ώστε να γίνεται [[Ενσωμάτωση αρχείων|ενσωμάτωση]] της σημείωσης προορισμού, ή να μην αφήνεται τίποτα.

## Αρχείο προτύπου

Διαμορφώνοντας ένα πρότυπο, μπορείτε να προσαρμόσετε το περιεχόμενο πριν το προσθέσετε στη νέα σημείωση. Για να χρησιμοποιήσετε ένα πρότυπο, εισαγάγετε μια **Τοποθεσία προτύπου αρχείου** στις ρυθμίσεις του πρόσθετου.

Το πρότυπο μπορεί να περιέχει τις ακόλουθες μεταβλητές:

| Μεταβλητή          | Περιγραφή                                                                                                                                              |
|-------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| `{{content}}`     | Το περιεχόμενο για συγχώνευση ή η εξαγόμενη επιλογή κειμένου. Εάν δεν συμπεριλάβετε αυτή τη μεταβλητή, ο Συνθέτης σημειώσεων προσθέτει το περιεχόμενο στο τέλος του προτύπου. |
| `{{fromTitle}}`   | Όνομα της σημείωσης προέλευσης.                                                                                                                                 |
| `{{newTitle}}`    | Όνομα της σημείωσης προορισμού. Για παράδειγμα, για να προσθέσετε το όνομα αρχείου ως επικεφαλίδα στην αρχή του αρχείου.                                                     |
| `{{date:FORMAT}}` | Ημερομηνία δημιουργίας της νέας σημείωσης. Για παράδειγμα, `{{date:YYYY-MM-DD}}`.                                                                                       |
