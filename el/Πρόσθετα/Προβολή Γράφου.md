Η Προβολή Γράφου είναι ένα [[Κύρια πρόσθετα|βασικό πρόσθετο]] που σας επιτρέπει να οπτικοποιείτε τις σχέσεις μεταξύ των σημειώσεων στο θησαυροφυλάκιό σας.

Για να ανοίξετε την Προβολή Γράφου, κάντε κλικ στο **Άνοιγμα προβολής γράφου** στην [[Κορδέλα]].

- Οι κύκλοι αντιπροσωπεύουν σημειώσεις, ή _κόμβους_.
- Οι γραμμές αντιπροσωπεύουν [[Εσωτερικοί σύνδεσμοι|εσωτερικούς συνδέσμους]] μεταξύ δύο κόμβων.

Όσο περισσότεροι κόμβοι αναφέρονται σε έναν συγκεκριμένο κόμβο, τόσο μεγαλύτερος γίνεται.

Για να αλληλεπιδράσετε με σημειώσεις στον γράφο:

- Αιωρηθείτε πάνω από κάθε κύκλο για να επισημάνετε τις συνδέσεις αυτής της σημείωσης.
- Κάντε κλικ σε μια σημείωση στον γράφο για να ανοίξετε αυτή τη σημείωση.
- Κάντε δεξί κλικ σε μια σημείωση για να ανοίξετε ένα μενού περιβάλλοντος με τις διαθέσιμες ενέργειες για αυτή τη σημείωση.

Για να πλοηγηθείτε στον γράφο:

- Κάντε μεγέθυνση και σμίκρυνση χρησιμοποιώντας τον τροχό κύλισης του ποντικιού σας, ή χρησιμοποιώντας τα πλήκτρα `+` και `-`.
- Μετακινήστε τον γράφο σέρνοντάς τον με τον δρομέα του ποντικιού σας, ή χρησιμοποιώντας τα πλήκτρα βελών.

Μπορείτε να κρατήσετε πατημένο το Shift ενώ χρησιμοποιείτε το πληκτρολόγιο για να επιταχύνετε τις κινήσεις.

## Ρυθμίσεις

Για να ανοίξετε τις ρυθμίσεις του γράφου, κάντε κλικ στο εικονίδιο γραναζιού στην επάνω δεξιά γωνία της προβολής γράφου.

Κάντε κλικ στο **Επαναφορά στις προεπιλεγμένες ρυθμίσεις** στην επάνω δεξιά γωνία του πλαισίου ρυθμίσεων για να επαναφέρετε τυχόν αλλαγές που κάνατε.

### Φίλτρα

Αυτή η ενότητα ελέγχει ποιοι κόμβοι εμφανίζονται στον γράφο.

- **Αναζήτηση αρχείων** σας επιτρέπει να φιλτράρετε σημειώσεις βάσει ενός όρου αναζήτησης. Για να μάθετε πώς μπορείτε να γράψετε πιο προχωρημένους όρους αναζήτησης, ανατρέξτε στην [[Αναζήτηση]].
- **Ετικέτες** εναλλάσσει αν θα εμφανίζονται οι ετικέτες στον γράφο.
- **Επισυνάψεις** εναλλάσσει αν θα εμφανίζονται τα συνημμένα στον γράφο.
- **Μόνο υπαρκτά αρχεία** εναλλάσσει αν θα εμφανίζονται σημειώσεις που υπάρχουν στο θησαυροφυλάκιό σας. Δεδομένου ότι μια σημείωση δεν χρειάζεται να υπάρχει για να συνδεθεί με αυτήν, αυτό μπορεί να βοηθήσει στον περιορισμό του γράφου σε σημειώσεις που πραγματικά έχετε στο θησαυροφυλάκιό σας.
- **Ορφανά** εναλλάσσει αν θα εμφανίζονται σημειώσεις χωρίς κανέναν σύνδεσμο.

> [!info] Φάκελοι που εξαιρούνται
> Τα αρχεία που ταιριάζουν με τα μοτίβα [[Ρυθμίσεις#Φάκελοι που εξαιρούνται|Φάκελοι που εξαιρούνται]] δεν θα εμφανίζονται στην Προβολή Γράφου.

### Ομάδες

Δημιουργήστε ομάδες σημειώσεων για να τις διακρίνετε μεταξύ τους χρησιμοποιώντας χρώμα.

Για να δημιουργήσετε μια νέα ομάδα:

1. Κάντε κλικ στο **Νέα ομάδα**.
2. Στο πλαίσιο αναζήτησης, πληκτρολογήστε έναν όρο αναζήτησης για τις σημειώσεις που θέλετε να προσθέσετε στην ομάδα.
3. Κάντε κλικ στον χρωματιστό κύκλο για να δώσετε στην ομάδα ένα χρώμα.

Για να μάθετε πώς μπορείτε να γράψετε πιο προχωρημένους όρους αναζήτησης, ανατρέξτε στην [[Αναζήτηση]].

### Προβολή

Αυτή η ενότητα ελέγχει τον τρόπο οπτικοποίησης κόμβων και συνδέσμων στον γράφο.

- **Βέλη** εναλλάσσει αν θα εμφανίζεται η κατεύθυνση κάθε συνδέσμου.
- **Όριο εξασθένισης κειμένου** ελέγχει τη διαφάνεια του κειμένου για το όνομα κάθε σημείωσης.
- **Μέγεθος κόμβου** ελέγχει το μέγεθος του κύκλου που αντιπροσωπεύει κάθε σημείωση.
- **Πάχος ακμών** ελέγχει το πλάτος γραμμής για κάθε σύνδεσμο.
- **Κινούμενο σχέδιο** ξεκινά μια [[#Εκκίνηση κινούμενης απεικόνισης χρονοδιαγράμματος|κινούμενη απεικόνιση χρονοδιαγράμματος]].

### Δυνάμεις

Αυτή η ενότητα ελέγχει τις δυνάμεις που ενεργούν σε κάθε κόμβο στον γράφο.

- **Κεντρική δύναμη** ελέγχει πόσο συμπαγής είναι ο γράφος. Μια υψηλότερη τιμή δημιουργεί έναν πιο κυκλικό γράφο.
- **Δύναμη απώθησης** ελέγχει πόσο ένας κόμβος απωθεί τους άλλους κόμβους μακριά από αυτόν.
- **Δύναμη ακμής** ελέγχει την έλξη σε κάθε σύνδεσμο. Αν ο σύνδεσμος ήταν ένα λαστιχάκι, η δύναμη ακμής ελέγχει πόσο σφιχτό ή χαλαρό είναι το λαστιχάκι.
- **Απόσταση ακμής** ελέγχει το μήκος των γραμμών μεταξύ κάθε σημείωσης.

## Τοπικός Γράφος

Για να ανοίξετε μια τοπική Προβολή Γράφου, χρησιμοποιήστε την εντολή **Άνοιγμα τοπικού γράφου**. Ενώ η Προβολή Γράφου εμφανίζει όλες τις σημειώσεις στο θησαυροφυλάκιό σας, ένας τοπικός γράφος σας δείχνει τις σημειώσεις που συνδέονται με την ενεργή σημείωση.

Η τοπική Προβολή Γράφου μπορεί να χρησιμοποιήσει όλες τις [[#Ρυθμίσεις]] που είναι διαθέσιμες στην καθολική Προβολή Γράφου. Επιπλέον, μπορείτε να αλλάξετε το βάθος του τοπικού γράφου. Κάθε επίπεδο βάθους θα εμφανίζει σημειώσεις συνδεδεμένες με τις σημειώσεις που αποκαλύφθηκαν στο προηγούμενο βάθος. Για να ελέγξετε το βάθος του τοπικού γράφου, χρησιμοποιήστε τον ολισθητή στο επάνω μέρος του πίνακα ρυθμίσεων φίλτρων του τοπικού γράφου.
