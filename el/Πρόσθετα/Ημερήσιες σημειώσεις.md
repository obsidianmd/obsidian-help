Οι Ημερήσιες σημειώσεις είναι ένα [[Κύρια πρόσθετα|βασικό πρόσθετο]] που ανοίγει μια σημείωση με βάση τη σημερινή ημερομηνία ή τη δημιουργεί αν δεν υπάρχει. Χρησιμοποιήστε τις ημερήσιες σημειώσεις για να δημιουργήσετε ημερολόγια, λίστες εκκρεμοτήτων ή καθημερινά αρχεία καταγραφής για πράγματα που ανακαλύψατε κατά τη διάρκεια της ημέρας.

Για να ανοίξετε τη σημερινή ημερήσια σημείωση, μπορείτε:

- Να κάνετε κλικ στο **Άνοιγμα σημερινής σημείωσης** ![[lucide-calendar.svg#icon]] στην [[Κορδέλα|κορδέλα]].
- Να εκτελέσετε την εντολή **Άνοιγμα σημερινής σημείωσης** από την [[Παλέτα εντολών]].
- Να [[Συντομεύσεις#Ορισμός συντόμευσης|χρησιμοποιήσετε μια συντόμευση πληκτρολογίου]] για την εντολή **Άνοιγμα σημερινής σημείωσης**.

Από προεπιλογή, το Obsidian δημιουργεί μια νέα κενή σημείωση με όνομα τη σημερινή ημερομηνία σε μορφή ΕΕΕΕ-ΜΜ-ΗΗ.

> [!tip] Αν προτιμάτε να έχετε τις ημερήσιες σημειώσεις σας σε ξεχωριστό φάκελο, μπορείτε να ρυθμίσετε τη <u>Νέα τοποθεσία αρχείου</u> στις ρυθμίσεις προσθέτων για να αλλάξετε πού δημιουργεί το Obsidian νέες ημερήσιες σημειώσεις.

> [!example]- Αυτόματοι υποφάκελοι
> Μπορείτε να οργανώσετε αυτόματα τις ημερήσιες σημειώσεις σας σε φακέλους χρησιμοποιώντας τη λειτουργία **Μορφή ημερομηνίας**.
> 
> Για παράδειγμα, αν ορίσετε τη μορφή ημερομηνίας ως `YYYY/MMMM/YYYY-MMM-DD`, οι σημειώσεις σας θα δημιουργούνται ως `2023/January/2023-Jan-01`. 
> 
> Μπορείτε να εξερευνήσετε περισσότερες επιλογές μορφοποίησης στον ιστότοπο τεκμηρίωσης του [momentJS](https://momentjs.com/docs/#/displaying/format/).

## Δημιουργία ημερήσιας σημείωσης από πρότυπο

Αν οι ημερήσιες σημειώσεις σας έχουν την ίδια δομή, μπορείτε να χρησιμοποιήσετε ένα [[Πρόσθετα/Πρότυπα|πρότυπο]] για να προσθέσετε προκαθορισμένο περιεχόμενο στις ημερήσιες σημειώσεις σας κατά τη δημιουργία τους.

1. Δημιουργήστε μια νέα σημείωση με όνομα "Ημερήσιο πρότυπο" με το ακόλουθο κείμενο (ή ό,τι σας φαίνεται λογικό!):

   ```md
   # {{date:YYYY-MM-DD}}

   ## Εργασίες

   - [ ]
   ```

2. Ανοίξτε τις **[[Ρυθμίσεις]]**.
3. Στην πλαϊνή μπάρα, κάντε κλικ στο **Ημερήσιες σημειώσεις** κάτω από τις **Ρυθμίσεις προσθέτων**.
4. Στο πεδίο κειμένου δίπλα στην **Τοποθεσία προτύπου αρχείου**, επιλέξτε τη σημείωση "Ημερήσιο πρότυπο".

Το Obsidian θα χρησιμοποιήσει το πρότυπο την επόμενη φορά που θα δημιουργήσετε μια νέα ημερήσια σημείωση.

## Ημερήσιες σημειώσεις και ιδιότητες

Όταν το πρόσθετο Ημερήσιες σημειώσεις είναι ενεργοποιημένο και υπάρχει μια ιδιότητα ημερομηνίας σε οποιαδήποτε σημείωση, το Obsidian θα επιχειρήσει αυτόματα να δημιουργήσει έναν σύνδεσμο προς την ημερήσια σημείωση για τη συγκεκριμένη ημέρα. Για παράδειγμα, αν μια σημείωση με τίτλο `example.md` περιλαμβάνει μια ιδιότητα ημερομηνίας όπως `2023-01-01`, αυτή η ημερομηνία θα μετατραπεί σε σύνδεσμο με δυνατότητα κλικ στην ενότητα [[Προβολές και λειτουργία επεξεργασίας#Ζωντανή προεπισκόπηση|ζωντανής προεπισκόπησης]].
