Το πρόσθετο [[Κύρια πρόσθετα|Παλέτα εντολών]] σας επιτρέπει να εκτελέσετε οποιαδήποτε εντολή απευθείας από το πληκτρολόγιό σας. Σας επιτρέπει επίσης να εξερευνήσετε όλες τις διαθέσιμες εντολές και τις [[Συντομεύσεις|συντομεύσεις πληκτρολογίου]] τους.

Για να εκτελέσετε μια εντολή από την παλέτα εντολών:

1. Πατήστε `Ctrl+P` (ή `Cmd+P` σε macOS) για να ανοίξετε την παλέτα εντολών. Μπορείτε επίσης να την ανοίξετε από την [[Κορδέλα]] με το εικονίδιο παλέτας εντολών ![[lucide-terminal.svg#icon]].
2. Πληκτρολογήστε το όνομα της εντολής που θέλετε να εκτελέσετε.
3. Πλοηγηθείτε στην εντολή χρησιμοποιώντας τα πλήκτρα βελών.
4. Πατήστε `Enter`.

> [!tip] Συμβουλή
> Η παλέτα εντολών υποστηρίζει _ασαφή αντιστοίχιση_, επιτρέποντάς σας να αναζητήσετε εντολές ακόμα κι αν δεν γνωρίζετε τα ακριβή ονόματά τους. Για παράδειγμα, πληκτρολογώντας "scf" θα βρείτε την εντολή **S**ave **c**urrent **f**ile.

Από την **έκδοση 1.8.3**, οι πρόσφατα χρησιμοποιημένες εντολές εμφανίζονται στην κορυφή της παλέτας εντολών. Ωστόσο, αυτές οι πρόσφατα χρησιμοποιημένες εντολές εξακολουθούν να υπόκεινται στον αλγόριθμο _ασαφούς αντιστοίχισης_. Όταν αρχίσετε να φιλτράρετε, οι πιο σύντομες εντολές θα έχουν προτεραιότητα έναντι των πρόσφατα χρησιμοποιημένων.

## Καρφιτσωμένες εντολές

Μπορείτε να καρφιτσώσετε τις εντολές που χρησιμοποιείτε συχνά στην κορυφή της παλέτας εντολών για γρήγορη πρόσβαση χωρίς να χρειάζεται να πληκτρολογήσετε το όνομά τους.

> [!tip] Συμβουλή
> Αν θέλετε να εκτελείτε γρήγορα εντολές που χρησιμοποιείτε συχνά, μπορείτε επίσης να [[Συντομεύσεις#Ορισμός συντόμευσης|ορίσετε συντομεύσεις πληκτρολογίου]] γι' αυτές.

### Καρφίτσωμα εντολής

1. Ανοίξτε τις **[[Ρυθμίσεις]]**.
2. Στην πλαϊνή μπάρα, κάντε κλικ στην **Παλέτα εντολών** κάτω από τις **Ρυθμίσεις προσθέτων**.
3. Δίπλα στην επιλογή **Νέο καρφιτσωμένη εντολή**, κάντε κλικ στο **Επιλογή εντολής**.
4. Επιλέξτε την εντολή που θέλετε να καρφιτσώσετε από τη λίστα.
5. Πατήστε `Enter`.

### Ξεκαρφίτσωμα εντολής

1. Ανοίξτε τις **[[Ρυθμίσεις]]**.
2. Στην πλαϊνή μπάρα, κάντε κλικ στην **Παλέτα εντολών** κάτω από τις **Ρυθμίσεις προσθέτων**.
3. Κάτω από τις **Καρφιτσωμένες εντολές**, κάντε κλικ στο εικονίδιο σταυρού δίπλα στην εντολή που θέλετε να ξεκαρφιτσώσετε.
