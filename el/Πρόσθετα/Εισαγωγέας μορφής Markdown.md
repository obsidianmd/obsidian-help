Ο Εισαγωγέας μορφής Markdown είναι ένα [[Κύρια πρόσθετα|βασικό πρόσθετο]] που σας επιτρέπει να μετατρέψετε Markdown από άλλες εφαρμογές στη μορφή Obsidian. Σας επιτρέπει επίσης να μετατρέψετε ορισμένες [[Ιδιότητες]] σε νέες απαιτούμενες μορφές.

> [!warning] Προειδοποίηση
> Ο Εισαγωγέας μορφής Markdown μετατρέπει ολόκληρο το θησαυροφυλάκιό σας βάσει των ρυθμίσεών σας. [[Δημιουργία αντιγράφων ασφαλείας των αρχείων Obsidian|Δημιουργήστε αντίγραφα ασφαλείας των αρχείων Obsidian]] πριν εκτελέσετε τη μετατροπή.

Για να μετατρέψετε όλες τις σημειώσεις στο θησαυροφυλάκιό σας:

1. Στην [[Παλέτα εντολών]], επιλέξτε **Άνοιγμα εισαγωγέα**. Αυτό μπορεί επίσης να βρεθεί στην [[Κορδέλα]] με το εικονίδιο **Άνοιγμα εισαγωγέα** ![[lucide-binary.svg#icon]].
2. Ενεργοποιήστε τις μορφές που θέλετε να μετατρέψετε.
3. Κάντε κλικ στο **Εκκίνηση Μετατροπής**.

Για περισσότερες πληροφορίες, ανατρέξτε στη [[Βασική σύνταξη μορφοποίησης]].

## Υποστηριζόμενες μορφές

### Roam Research

Ο Εισαγωγέας μορφής Markdown μπορεί να μετατρέψει την ακόλουθη σύνταξη Roam Research:

- **Ετικέτες**: Μετατρέπει `#tag` και `#[[tag]]` σε `[[tag]]`
- **Επισήμανση**: Μετατρέπει `^^highlight^^` σε `==highlight==`
- **Στοιχεία TODO**: Μετατρέπει `{{[[TODO]]}}` σε `[ ]`

### Bear

Ο Εισαγωγέας μορφής Markdown μπορεί να μετατρέψει την ακόλουθη σύνταξη Bear:

- **Επισήμανση**: Μετατρέπει `::highlight::` σε `==highlight==`

### Zettelkasten

Ο Εισαγωγέας μορφής Markdown μπορεί να μετατρέψει την ακόλουθη σύνταξη Zettelkasten:

- **Πλήρεις σύνδεσμοι**: Μετατρέπει `[[UID]]` σε `[[UID File Name]]`
- **Εμφανίσιμοι σύνδεσμοι**: Μετατρέπει `[[UID]]` σε `[[UID File Name|File Name]]`

### [[Ιδιότητες]]

Από την έκδοση Obsidian `1.9.3`, ο Εισαγωγέας μορφής Markdown μπορεί να μετατρέψει [[Ιδιότητες#Καταργημένες ιδιότητες|καταργημένες μορφές ιδιοτήτων]] στην τρέχουσα μορφή:

**Ψευδώνυμα**

```yaml
# Πριν

alias: My Note Title

# Μετά

aliases:
  - My Note Title
```

**Ετικέτες**

```yaml
# Πριν

tag: project, important

# Μετά

tags:
  - project
  - important
```

**Κλάσεις CSS**

```yaml
# Πριν

cssclass: custom-style

# Μετά

cssclasses:
  - custom-style
```
