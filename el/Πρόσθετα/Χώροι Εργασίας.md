Οι Χώροι Εργασίας είναι ένα [[Κύρια πρόσθετα|βασικό πρόσθετο]] που σας επιτρέπει να διαχειρίζεστε και να εναλλάσσετε μεταξύ διαφορετικών διατάξεων εφαρμογής ανάλογα με την εργασία σας, για παράδειγμα ημερολόγιο, ανάγνωση ή γραφή.

Ένας χώρος εργασίας περιέχει πληροφορίες σχετικά με τα ανοιχτά αρχεία και τις καρτέλες, καθώς και το πλάτος και την ορατότητα κάθε πλαϊνής μπάρας.

## Αποθήκευση χώρου εργασίας

**Κορδέλα:**

1. Στην κορδέλα, κάντε κλικ στο **Διαχείριση διάταξης χώρου εργασίας** ![[lucide-panels-top-left.svg#icon]].
2. Εισάγετε ένα όνομα για τον χώρο εργασίας.
3. Επιλέξτε **Αποθήκευση**.

**Παλέτα εντολών:**

1. Πατήστε `Ctrl+P` (ή `Cmd+P` σε macOS) για να ανοίξετε την [[Παλέτα εντολών|Παλέτα εντολών]].
2. Αναζητήστε την εντολή **Διαχείριση διάταξης χώρου εργασίας**.
3. Πατήστε `Enter` με την εντολή επιλεγμένη.
4. Εισάγετε ένα όνομα για τον χώρο εργασίας.
5. Επιλέξτε **Αποθήκευση**.

Για να ενημερώσετε έναν υπάρχοντα χώρο εργασίας, αποθηκεύστε τον χώρο εργασίας με το ίδιο όνομα με τον χώρο εργασίας που θέλετε να ενημερώσετε.

## Φόρτωση χώρου εργασίας

**Κορδέλα:**

1. Στην κορδέλα, κάντε κλικ στο **Διαχείριση διάταξης χώρου εργασίας** ![[lucide-panels-top-left.svg#icon]].
2. Δίπλα στον χώρο εργασίας που θέλετε να φορτώσετε, επιλέξτε **Φόρτωση**.

**Παλέτα εντολών:**

1. Πατήστε `Ctrl+P` (ή `Cmd+P` σε macOS) για να ανοίξετε την [[Παλέτα εντολών|Παλέτα εντολών]].
2. Αναζητήστε την εντολή **Διαχείριση διάταξης χώρου εργασίας**.
3. Πατήστε `Enter` με την εντολή επιλεγμένη.
4. Επιλέξτε τον χώρο εργασίας για φόρτωση.
