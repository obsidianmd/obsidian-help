Η Προβολή ιδιοτήτων είναι ένα [[Κύρια πρόσθετα|βασικό πρόσθετο]] που προσθέτει δύο προβολές πλαϊνής μπάρας τις οποίες μπορείτε να χρησιμοποιήσετε για να διαχειριστείτε τις [[Ιδιότητες]]:

- **Ιδιότητες αρχείου** εμφανίζει μια προβολή των ιδιοτήτων για την ενεργή σημείωση.
- **Όλες οι ιδιότητες** εμφανίζει μια λίστα με όλες τις ιδιότητες στο θησαυροφυλάκιό σας και τον τύπο τους.

## Προβολή όλων των ιδιοτήτων

Με την προβολή «Όλες οι ιδιότητες» μπορείτε να:

- Ταξινομήσετε ιδιότητες κατά όνομα ή κατά συχνότητα (συνολικός αριθμός χρήσεων στο θησαυροφυλάκιό σας).
- Κάνετε κλικ σε μια ιδιότητα για να ανοίξετε την προβολή [[Αναζήτηση]] με την σύνταξη [[Αναζήτηση#Αναζήτηση ιδιοτήτων|αναζήτησης ιδιοτήτων]] προσυμπληρωμένη.
- Κάνετε δεξί κλικ σε μια ιδιότητα για να τη μετονομάσετε καθολικά.
