Οι συντομεύσεις πληκτρολογίου είναι προσαρμόσιμοι συνδυασμοί πλήκτρων για εντολές του Obsidian. Μπορείτε να αντιστοιχίσετε οποιονδήποτε συνδυασμό πλήκτρων για να ενεργοποιήσετε εντολές όπως άνοιγμα αρχείων, εισαγωγή προτύπων ή εναλλαγή πρόσθετων.

Οι συντομεύσεις πληκτρολογίου διαφέρουν από τις [[Συντομεύσεις επεξεργασίας|συντομεύσεις πληκτρολογίου του συστήματος]] (όπως `Ctrl+C` για αντιγραφή), οι οποίες παρέχονται από το λειτουργικό σας σύστημα και δεν μπορούν να προσαρμοστούν μέσα στο Obsidian.

## Προβολή συντομεύσεων

Ο γρηγορότερος τρόπος να δείτε τη συντόμευση που έχει οριστεί για μια εντολή είναι να βρείτε την εντολή στην [[Παλέτα εντολών]]. Αν εμφανίζεται μια συντόμευση πληκτρολογίου δίπλα στην εντολή, σημαίνει ότι έχει οριστεί.

Μπορείτε επίσης να προβάλετε και να διαχειριστείτε όλες τις συντομεύσεις στις **[[Ρυθμίσεις]] → Συντομεύσεις**.

## Ορισμός συντόμευσης

Για να προσθέσετε ή να τροποποιήσετε μια συντόμευση πληκτρολογίου:

1. Ανοίξτε **[[Ρυθμίσεις]] → Συντομεύσεις**.
2. Βρείτε την εντολή στη λίστα ή χρησιμοποιήστε το φίλτρο αναζήτησης.
3. Επιλέξτε το εικονίδιο συν (+) δίπλα στην εντολή.
4. Πατήστε τον συνδυασμό πλήκτρων που θέλετε να χρησιμοποιήσετε.
5. Επιλέξτε **Αποθήκευση** για επιβεβαίωση.

Μπορείτε να αντιστοιχίσετε πολλαπλούς συνδυασμούς συντομεύσεων σε μία εντολή επιλέγοντας ξανά το εικονίδιο συν (+) μετά τον ορισμό της πρώτης συντόμευσης.

## Αφαίρεση συντόμευσης

Για να αφαιρέσετε μια συντόμευση πληκτρολογίου:

1. Ανοίξτε **[[Ρυθμίσεις]] → Συντομεύσεις**.
2. Βρείτε την εντολή με τη συντόμευση που θέλετε να αφαιρέσετε.
3. Επιλέξτε το εικονίδιο X δίπλα στον συνδυασμό πλήκτρων.

## Φιλτράρισμα συντομεύσεων

Για να εμφανίσετε μόνο εντολές που έχουν αντιστοιχισμένες συντομεύσεις πληκτρολογίου, επιλέξτε το εικονίδιο φίλτρου στις **[[Ρυθμίσεις]] → Συντομεύσεις**.

## Διατάξεις πληκτρολογίου εκτός ΗΠΑ

Οι συντομεύσεις πληκτρολογίου εμφανίζονται όπως θα φαίνονταν σε διάταξη πληκτρολογίου ΗΠΑ. Ακόμα κι αν ο εμφανιζόμενος συνδυασμός δεν ταιριάζει με τη διάταξη του πληκτρολογίου σας, η συντόμευση θα λειτουργεί βάσει των πραγματικών πλήκτρων που πατήσατε (εφόσον δεν αλλάξετε τη διάταξη του πληκτρολογίου σας).

## Προεπιλεγμένες συντομεύσεις πληκτρολογίου

Για μια ολοκληρωμένη λίστα με τις προεπιλεγμένες συντομεύσεις πληκτρολογίου για επεξεργασία και πλοήγηση, δείτε [[Συντομεύσεις επεξεργασίας]].
