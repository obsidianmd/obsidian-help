Οι καρτέλες στο Obsidian λειτουργούν όπως και οι καρτέλες σε άλλες εφαρμογές, όπως τα προγράμματα περιήγησης ιστού.

Μπορείτε να ανοίξετε όσες καρτέλες θέλετε στο Obsidian. Μπορείτε επίσης να τακτοποιήσετε τις καρτέλες για να δημιουργήσετε προσαρμοσμένες διατάξεις που διατηρούνται μέχρι την επόμενη φορά που θα ανοίξετε την εφαρμογή.

> [!note] Καρτέλες σε κινητό
> Στο κινητό, μπορείτε να διαχειριστείτε τις ανοιχτές καρτέλες από τον μετρητή καρτελών στη [[Εφαρμογή κινητού#Γραμμή πλοήγησης|Γραμμή πλοήγησης]].

## Άνοιγμα νέας καρτέλας

Στο πάνω μέρος του παραθύρου της εφαρμογής, δίπλα στην τελευταία καρτέλα στα δεξιά, επιλέξτε **Νέα καρτέλα** ![[lucide-plus.svg#icon]]. Ή χρησιμοποιήστε μια συντόμευση πληκτρολογίου:

- **Windows και Linux:** `Ctrl+t`
- **macOS:** `Cmd+t`

## Άνοιγμα συνδέσμου

Επιλέξτε έναν σύνδεσμο στο Obsidian για να τον ανοίξετε στην ενεργή καρτέλα.

Για να ανοίξετε έναν σύνδεσμο σε νέα καρτέλα, πατήστε `Ctrl` (ή `Cmd` στο macOS) και επιλέξτε τον σύνδεσμο.

Ακολουθούν όλα τα πλήκτρα τροποποίησης που μπορείτε να χρησιμοποιήσετε για να ανοίξετε συνδέσμους με διάφορους τρόπους:

|Ενέργεια|MacOS|Windows/Linux|
|---|---|---|
|**Πλοήγηση**|_Κανένα_|_Κανένα_|
|**Νέα καρτέλα**|`⌘` (+ `Shift` στη λειτουργία πηγαίου κώδικα)|`Ctrl` (+ `Shift` στη λειτουργία πηγαίου κώδικα)|
|**Νέα ομάδα καρτελών**|`⌘` `⌥`| `Ctrl` `Alt`|
|**Νέο παράθυρο**|`⌘` `⌥` `Shift`|`Ctrl` `Alt` `Shift`|

## Οργάνωση καρτελών και παραθύρων

Κάθε καρτέλα ανήκει σε μια _ομάδα καρτελών_. Μπορείτε να σύρετε και να αποθέσετε καρτέλες για να τις αναδιατάξετε μέσα σε μια ομάδα καρτελών, να τις μετακινήσετε σε διαφορετική ομάδα καρτελών ή να δημιουργήσετε μια νέα ομάδα καρτελών. Στην επιφάνεια εργασίας, μπορείτε να σύρετε καρτέλες έξω από το παράθυρό τους για να τις ανοίξετε σε ξεχωριστό [[Αναδυόμενα παράθυρα|αναδυόμενο παράθυρο]].

Οι καρτέλες στις πλαϊνές μπάρες εμφανίζουν μόνο το εικονίδιο. Περάστε το ποντίκι πάνω από το εικονίδιο για να εμφανιστεί μια επεξήγηση με τον τίτλο της καρτέλας.

### Τακτοποίηση καρτελών

Για να αλλάξετε τη σειρά των καρτελών σας, σύρετε την καρτέλα κατά μήκος των καρτελών στην ομάδα καρτελών.

Καθώς σύρετε μια καρτέλα, οι _ζώνες απόθεσης_—περιοχές στις οποίες μπορείτε να μετακινήσετε την καρτέλα—επισημαίνονται. Η ζώνη απόθεσης καθορίζει πού θα εισαχθεί η καρτέλα. Ορισμένες καρτέλες μπορούν να βρίσκονται μόνο σε μία από τις πλαϊνές μπάρες.

### Διαχωρισμός ομάδας καρτελών

Κάντε δεξί κλικ σε μια καρτέλα και επιλέξτε **Χώρισμα Δεξιά** ή **Χώρισμα Κάτω** για να δημιουργήσετε μια νέα ομάδα καρτελών με αυτή την καρτέλα.

Μπορείτε επίσης να διαχωρίσετε μια ομάδα καρτελών σύροντας μια καρτέλα στο κάτω μέρος μιας άλλης καρτέλας.

### Αλλαγή μεγέθους ομάδας καρτελών

Για να αλλάξετε το μέγεθος μιας ομάδας καρτελών, τοποθετήστε τον δρομέα πάνω σε μια άκρη της ομάδας καρτελών. Η άκρη επισημαίνεται όταν μπορεί να συρθεί για αλλαγή μεγέθους.

Μπορείτε να αλλάξετε το μέγεθος των πλαϊνών μπαρών με παρόμοιο τρόπο για να δημιουργήσετε περισσότερο χώρο για τις ομάδες καρτελών στη μέση.

### Μετακίνηση καρτέλας σε νέο παράθυρο

**Μεταφορά και απόθεση:**

- Επιλέξτε και σύρετε την καρτέλα έξω από το παράθυρο της εφαρμογής για να την ανοίξετε σε νέο παράθυρο.

**Παλέτα εντολών:**

- Ανοίξτε την παλέτα εντολών και επιλέξτε **Μετακίνηση παραθύρου σε νέο παράθυρο**.

### Μετακίνηση καρτέλας σε διαφορετικό παράθυρο

Για να μετακινήσετε μια καρτέλα σε άλλο υπάρχον παράθυρο, σύρετε την καρτέλα στο παράθυρο στο οποίο θέλετε να τη μετακινήσετε.

### Καρφίτσωμα καρτέλας

Για να καρφιτσώσετε μια καρτέλα στον κύριο επεξεργαστή, κάντε δεξί κλικ στην καρτέλα και επιλέξτε **Καρφίτσωμα**. Οι σύνδεσμοι σε μια καρφιτσωμένη καρτέλα ανοίγουν πάντα σε ξεχωριστή καρτέλα.

Για να ξεκαρφιτσώσετε μια καρφιτσωμένη καρτέλα στον κύριο επεξεργαστή, κάντε δεξί κλικ στην καρτέλα και επιλέξτε **Ξεκαρφίτσωμα**.

Για να καρφιτσώσετε ή να ξεκαρφιτσώσετε μια καρτέλα στην πλαϊνή μπάρα, δείτε [[Πλαϊνή μπάρα#Καρφίτσωμα καρτελών|καρφίτσωμα καρτελών στην πλαϊνή μπάρα]].

## Εναλλαγή σε διαφορετική καρτέλα

Επιλέξτε μια καρτέλα για να μεταβείτε σε αυτή. Ή χρησιμοποιήστε μια συντόμευση πληκτρολογίου:

| Μετάβαση σε                     | MacOS            | Windows/Linux        |
|----------------------------------|------------------|----------------------|
| **Επόμενη καρτέλα**             | `⌃`+`⇥`         | `Ctrl`+`Tab`         |
| **Προηγούμενη καρτέλα**         | `⌃`+`⇧`+`⇥`    | `Ctrl`+`Shift`+`Tab` |
| **Πρώτη καρτέλα στα αριστερά**  | `⌘`+`1`          | `Ctrl`+`1`           |
| **2η έως 8η καρτέλα**           | `⌘`+`2`..`8`     | `Ctrl`+`2`..`8`      |
| **Τελευταία καρτέλα στα δεξιά** | `⌘`+`9`          | `Ctrl`+`9`           |
| **Πρόσφατα κλεισμένη καρτέλα**  | `⌘`+`⇧`+`t`     | `Ctrl`+`Shift`+`t`   |

## Στοίβαξη ομάδων καρτελών

Μπορείτε να στοιβάξετε καρτέλες ώστε να ολισθαίνουν πάνω σε άλλες καρτέλες στην ίδια ομάδα καρτελών.

Για να στοιβάξετε σημειώσεις, επιλέξτε το κάτω βέλος στην επάνω δεξιά γωνία της ομάδας καρτελών και μετά επιλέξτε **Στοίβαξη καρτελών**.

![tab-stacks](https://user-images.githubusercontent.com/693981/188205363-0f24b2a5-3706-4a8c-b38b-7a66baa68ce6.gif)

Η στοίβαξη καρτελών είναι εμπνευσμένη από τις [συρόμενες σημειώσεις του Andy Matuschak](https://notes.andymatuschak.org/).

## Συνδεδεμένες προβολές

Οι _συνδεδεμένες προβολές_ είναι καρτέλες που αναφέρονται σε μια διαφορετική καρτέλα. Όταν αλλάζει το περιεχόμενο της αναφερόμενης καρτέλας, αλλάζει και η συνδεδεμένη προβολή.

Για καρτέλες σημειώσεων, μπορείτε να χρησιμοποιήσετε τα ακόλουθα πρόσθετα ως συνδεδεμένες προβολές:

- [[Προβολή Γράφου]] (τοπικό)
- [[Οπισθοσύνδεσμοι]]
- [[Περίγραμμα]]

Για να ανοίξετε μια συνδεδεμένη προβολή για μια καρτέλα σημείωσης:

1. Επιλέξτε **Περισσότερες επιλογές** ![[lucide-more-horizontal.svg#icon]] στην επάνω δεξιά γωνία της σημείωσης.
2. Στην ενότητα **Άνοιγμα σε συνδεδεμένη προβολή**, επιλέξτε τη συνδεδεμένη προβολή που θέλετε να ανοίξετε.
