Το Obsidian έχει δύο πλαϊνές μπάρες: μία αριστερή πλαϊνή μπάρα και μία δεξιά πλαϊνή μπάρα. Οι πλαϊνές μπάρες περιέχουν καρτέλες που δημιουργούνται από πρόσθετα, όπως τους [[Οπισθοσύνδεσμοι|Οπισθοσυνδέσμους]], τους [[Εξερχόμενοι σύνδεσμοι|Εξερχόμενους συνδέσμους]] και την [[Εξερεύνηση αρχείων|Εξερεύνηση αρχείων]]. Στην επιφάνεια εργασίας, μπορείτε επίσης να σύρετε σημειώσεις στην πλαϊνή μπάρα για να τις κρατήσετε ορατές δίπλα στον κύριο επεξεργαστή σας.

Στην επιφάνεια εργασίας και σε μεγαλύτερα tablet, η αριστερή πλαϊνή μπάρα περιλαμβάνει την [[Κορδέλα|Κορδέλα]]. Σε κινητά και μικρότερα tablet, οι πλαϊνές μπάρες είναι [[#Άνοιγμα κρυφών πλαϊνών μπαρών|συμπτυγμένες από προεπιλογή]].

## Άνοιγμα κρυφών πλαϊνών μπαρών

### Επιφάνεια εργασίας και μεγαλύτερα tablet

Στην επιφάνεια εργασίας και σε μεγαλύτερα tablet, ανοίξτε μια συμπτυγμένη πλαϊνή μπάρα επιλέγοντας το εικονίδιο αριστερής ανάπτυξης ![[obsidian-icon-side-bar-left.svg#icon]] ή δεξιάς ανάπτυξης ![[obsidian-icon-side-bar-right.svg#icon]].

### Κινητά και μικρότερα tablet

Σε κινητά και μικρότερα tablet, μπορείτε να ανοίξετε τις κρυφές πλαϊνές μπάρες με τους εξής τρόπους:

- Σύρετε αριστερά ή δεξιά
- Επιλέξτε το εικονίδιο αριστερής ανάπτυξης ![[obsidian-icon-side-bar-left.svg#icon]] (μόνο αριστερή πλαϊνή μπάρα)
- Χρησιμοποιήστε τις επιλογές *Εναλλαγή αριστερής* ή *Εναλλαγή δεξιάς* από την [[Παλέτα εντολών|Παλέτα εντολών]].

## Καρτέλες

Από προεπιλογή, τα πρόσθετα δημιουργούν νέες καρτέλες αντί για νέες [[#Ομάδες καρτελών]].

Μπορείτε να εναλλάσσεστε μεταξύ καρτελών επιλέγοντας το εικονίδιο. Μια υπόδειξη εμφανίζεται για να σας βοηθήσει να αναγνωρίσετε τις καρτέλες.

Σε κάθε πλαίσιο, μπορείτε να βλέπετε μόνο μία καρτέλα κάθε φορά. Ορισμένες ενέργειες εμφανίζουν αυτόματα μια καρτέλα. Για παράδειγμα, όταν επιλέγετε μια ετικέτα, η καρτέλα της ανοίγει.

### Άνοιγμα ή επανάνοιγμα καρτελών

Υπάρχουν αρκετοί τρόποι για να προσθέσετε καρτέλες στις πλαϊνές μπάρες σας:

- **Ενεργοποίηση πρόσθετου**: Πολλά πρόσθετα δημιουργούν αυτόματα καρτέλες στην πλαϊνή μπάρα όταν ενεργοποιούνται. Μεταβείτε στις **Ρυθμίσεις → Πρόσθετα κοινότητας** (ή **Κύρια πρόσθετα**) και ενεργοποιήστε το πρόσθετο για την καρτέλα που θέλετε.
- **Εκτέλεση εντολής**: Χρησιμοποιήστε την [[Παλέτα εντολών|Παλέτα εντολών]] για να αναζητήσετε εντολές που ανοίγουν συγκεκριμένες καρτέλες. Για παράδειγμα, αναζητήστε "Οπισθοσύνδεσμοι" ή "Εξερχόμενοι σύνδεσμοι" για να ανοίξετε αυτές τις καρτέλες.
- **Σύρετε μια σημείωση** (μόνο στην επιφάνεια εργασίας): Στην επιφάνεια εργασίας, μπορείτε να σύρετε μια σημείωση από την [[Εξερεύνηση αρχείων|Εξερεύνηση αρχείων]] ή από μια καρτέλα του επεξεργαστή στην πλαϊνή μπάρα για να την ανοίξετε εκεί.

Αν έχετε κλείσει μια καρτέλα και θέλετε να την επαναφέρετε, αυτές οι ίδιες μέθοδοι θα την ανοίξουν ξανά.

### Κλείσιμο καρτελών

Ορισμένες καρτέλες μπορούν να κλείσουν μέσω του μενού δεξιού κλικ, ενώ άλλες όχι. Αν δεν θέλετε να βλέπετε μια καρτέλα που δεν μπορεί να κλείσει, σκεφτείτε να απενεργοποιήσετε το σχετικό πρόσθετο ή να συμπτύξετε την πλαϊνή μπάρα επιλέγοντας το εικονίδιο βέλους.

### Αναδιάταξη καρτελών

Μπορείτε να σύρετε και να αποθέσετε τα εικονίδια καρτελών για να τα αναδιατάξετε μέσα σε μια ομάδα καρτελών.

Στην επιφάνεια εργασίας, μπορείτε επίσης να σύρετε καρτέλες από την πλαϊνή μπάρα στην κύρια περιοχή του επεξεργαστή για να τις ανοίξετε εκεί.

### Καρφίτσωμα καρτελών

Μπορείτε να καρφιτσώσετε μια καρτέλα στην πλαϊνή μπάρα για να ελέγχετε πώς ενημερώνεται όταν αλληλεπιδράτε με άλλα μέρη της εφαρμογής.

Για να καρφιτσώσετε ή να ξεκαρφιτσώσετε μια καρτέλα στην πλαϊνή μπάρα, σύρετε την καρτέλα προς ή από την πλαϊνή μπάρα.

Η συμπεριφορά μιας καρφιτσωμένης καρτέλας μέσα στην πλαϊνή μπάρα εξαρτάται από τον τύπο περιεχομένου της:

- **Σημειώσεις και Βάσεις**: Μια καρφιτσωμένη καρτέλα σημείωσης ή βάσης παραμένει στη θέση της. Νέες σημειώσεις ανοίγουν ως ξεχωριστές καρτέλες στον κύριο επεξεργαστή αντί να αντικαθιστούν την καρφιτσωμένη καρτέλα.
- **Πλαίσια** (όπως [[Οπισθοσύνδεσμοι|Οπισθοσύνδεσμοι]] ή [[Εξερχόμενοι σύνδεσμοι|Εξερχόμενοι σύνδεσμοι]]): Ένα καρφιτσωμένο πλαίσιο παραμένει εστιασμένο στην τελευταία επιλεγμένη σημείωση. Η επιλογή διαφορετικής σημείωσης δεν ενημερώνει το καρφιτσωμένο πλαίσιο.

Για πληροφορίες σχετικά με το καρφίτσωμα καρτελών στον κύριο επεξεργαστή, δείτε [[Καρτέλες#Καρφίτσωμα καρτέλας|Καρφίτσωμα καρτέλας]].

## Ομάδες καρτελών

Μπορείτε να έχετε πολλαπλές ομάδες καρτελών στην πλαϊνή μπάρα.

### Δημιουργία ομάδων καρτελών

Για να δημιουργήσετε μια νέα ομάδα καρτελών, σύρετε το εικονίδιο μιας καρτέλας πάνω ή κάτω από μια υπάρχουσα ομάδα καρτελών. Η επισημασμένη περιοχή υποδεικνύει πού θα δημιουργηθεί η νέα ομάδα καρτελών.

### Κλείσιμο ομάδων καρτελών

Για να κλείσετε μια ομάδα καρτελών, μπορείτε να [[#Κλείσιμο καρτελών|κλείσετε όλες τις καρτέλες σε αυτήν]], ή να σύρετε όλες τις καρτέλες της ομάδας από τα εικονίδιά τους και να τις αποθέσετε σε υπάρχουσες ομάδες καρτελών.

Όταν επισημαίνεται μόνο η περιοχή καρτελών μιας ομάδας καρτελών, η καρτέλα που σύρεται θα συγχωνευτεί με τις άλλες καρτέλες σε αυτήν την ομάδα.
