Υπάρχουν πολλοί τρόποι μεταφοράς και απόθεσης στοιχείων στο Obsidian. Αυτό περιλαμβάνει καρτέλες, αρχεία, φακέλους και περιεχόμενο.

### Μεταφορά καρτελών

- Μπορείτε να [[Καρτέλες#Τακτοποίηση καρτελών|τακτοποιήσετε καρτέλες]] και να χωρίσετε ομάδες καρτελών στην κύρια περιοχή περιεχομένου και στις [[Πλαϊνή μπάρα|πλαϊνές μπάρες]].

### Πηγές μεταφοράς

- Μπορείτε να σύρετε ένα αρχείο ή πολλά αρχεία από την [[Εξερεύνηση αρχείων|εξερεύνηση αρχείων]].
- Μπορείτε να σύρετε ένα αρχείο από ένα αποτέλεσμα [[Αναζήτηση|αναζήτησης]].
- Μπορείτε να σύρετε ένα αρχείο από τους [[Οπισθοσύνδεσμοι|οπισθοσυνδέσμους]] ή τις μη συνδεδεμένες αναφορές.
- Μπορείτε να σύρετε ένα αρχείο από έναν σύνδεσμο μέσα στη σημείωση, σε λειτουργία προεπισκόπησης.

### Προορισμοί απόθεσης

- Μπορείτε να αφήσετε ένα αρχείο σε μια κεφαλίδα καρτέλας για να ανοίξετε το αρχείο εκεί. Κρατήστε `Alt` (ή `Shift` σε macOS) για απόθεση οπουδήποτε στην καρτέλα.
- Μπορείτε να αφήσετε ένα αρχείο σε έναν φάκελο στην [[Εξερεύνηση αρχείων|εξερεύνηση αρχείων]] για μετακίνηση του αρχείου εκεί.
- Μπορείτε να αφήσετε ένα αρχείο σε έναν επεξεργαστή για να το εισάγετε ως σύνδεσμο. Ο σύνδεσμος που δημιουργείται ακολουθεί επίσης τις προτιμήσεις σας, όπως σχετικές διαδρομές ή χρήση συνδέσμων Markdown.
- Μπορείτε να αφήσετε ένα αρχείο στην καρτέλα [[Σελιδοδείκτες]] για να το προσθέσετε ως σελιδοδείκτη.

### Μεταφορά από εξωτερικές εφαρμογές στο Obsidian

- Μπορείτε να σύρετε οποιοδήποτε περιεχόμενο HTML από τον περιηγητή στο Obsidian για αυτόματη μετατροπή σε Markdown
- Μπορείτε να μεταφέρετε και να αφήσετε οποιαδήποτε αρχεία από τον εξερευνητή αρχείων του λειτουργικού σας συστήματος ώστε το Obsidian να εισάγει (δημιουργία αντιγράφου) αυτά τα αρχεία στον φάκελο συνημμένων του θησαυροφυλακίου σας και να τα εισάγει ως εσωτερικούς συνδέσμους. Κρατήστε `Ctrl` σε Windows/Linux ή `Option` σε macOS για να δημιουργήσετε απόλυτους συνδέσμους `file:///` προς αυτά τα αρχεία αντί να εισάγετε ένα αντίγραφο.

### Απόθεση αρχείων εκτός Obsidian

- Αυτό δημιουργεί μια διεύθυνση URL `obsidian://` στην εξωτερική εφαρμογή για τη σημείωση Obsidian που σύρατε.
