Les onglets dans Obsidian fonctionnent de manière très similaire aux onglets d'autres applications, comme les navigateurs web.

Vous pouvez ouvrir autant d'onglets que vous le souhaitez dans Obsidian. Vous pouvez également organiser les onglets pour créer des dispositions personnalisées qui persistent jusqu'à la prochaine ouverture de l'application.

> [!note] Onglets sur mobile
> Sur mobile, vous pouvez gérer les onglets ouverts depuis le compteur d'onglets dans la [[Application mobile#Barre de navigation|barre de navigation]].

## Ouvrir un nouvel onglet

En haut de la fenêtre de l'application, à côté du dernier onglet sur la droite, sélectionnez **Nouvel onglet** ( ![[lucide-plus.svg#icon]] ). Ou utilisez un raccourci clavier :

- **Windows et Linux :** `Ctrl+t`
- **macOS :** `Cmd+t`

## Ouvrir un lien

Sélectionnez un lien dans Obsidian pour l'ouvrir dans l'onglet actif.

Pour ouvrir un lien dans un nouvel onglet, appuyez sur `Ctrl` (ou `Cmd` sur macOS) et sélectionnez le lien.

Voici toutes les touches de modification que vous pouvez utiliser pour ouvrir des liens de différentes manières :

|Action|macOS|Windows/Linux|
|---|---|---|
|**Naviguer**|_Aucune_|_Aucune_|
|**Nouvel onglet**|`⌘` (+ `Shift` en mode source)|`Ctrl` (+ `Shift` en mode source)|
|**Nouveau groupe d'onglets**|`⌘` `⌥`| `Ctrl` `Alt`|
|**Nouvelle fenêtre**|`⌘` `⌥` `Shift`|`Ctrl` `Alt` `Shift`|

## Organiser vos onglets et fenêtres

Chaque onglet appartient à un _groupe d'onglets_. Vous pouvez glisser-déposer les onglets pour les réorganiser au sein d'un groupe d'onglets, les déplacer vers un autre groupe d'onglets ou créer un nouveau groupe d'onglets. Sur ordinateur, vous pouvez faire glisser les onglets hors de leur fenêtre pour les ouvrir dans une [[Fenêtres détachées|fenêtre détachée]] séparée.

Les onglets dans les barres latérales n'affichent que l'icône. Survolez l'icône pour afficher une infobulle avec le titre de l'onglet.

### Organiser les onglets

Pour changer l'ordre de vos onglets, faites glisser l'onglet le long des onglets dans le groupe d'onglets.

Lorsque vous faites glisser un onglet, les _zones de dépôt_ — zones sur lesquelles vous pouvez déplacer l'onglet — sont mises en surbrillance. La zone de dépôt détermine où insérer l'onglet. Certains onglets ne peuvent se trouver que dans l'une des barres latérales.

### Diviser un groupe d'onglets

Faites un clic droit sur un onglet et sélectionnez **Diviser à droite** ou **Diviser vers le bas** pour créer un nouveau groupe d'onglets avec cet onglet.

Vous pouvez également diviser un groupe d'onglets en faisant glisser un onglet vers le bas d'un autre onglet.

### Redimensionner un groupe d'onglets

Pour redimensionner un groupe d'onglets, placez le curseur sur un bord du groupe d'onglets. Le bord est mis en surbrillance lorsqu'il peut être glissé pour redimensionner.

Vous pouvez redimensionner les barres latérales de manière similaire pour donner plus d'espace aux groupes d'onglets au centre.

### Déplacer un onglet vers une nouvelle fenêtre

**Glisser-déposer :**

- Sélectionnez et faites glisser l'onglet en dehors de la fenêtre de l'application pour l'ouvrir dans une nouvelle fenêtre.

**Palette de commandes :**

- Ouvrez la palette de commandes, et sélectionnez **Déplacer l'onglet actuel vers une nouvelle fenêtre**.

### Déplacer un onglet vers une autre fenêtre

Pour déplacer un onglet vers une autre fenêtre existante, faites glisser l'onglet vers la fenêtre dans laquelle vous souhaitez le déplacer.

### Épingler un onglet

Pour épingler un onglet dans l'éditeur principal, faites un clic droit sur l'onglet et sélectionnez **Épingler**. Les liens dans un onglet épinglé s'ouvrent toujours dans un onglet séparé.

Pour désépingler un onglet épinglé dans l'éditeur principal, faites un clic droit sur l'onglet et sélectionnez **Désépingler**.

Pour épingler ou désépingler un onglet dans la barre latérale, consultez [[Barre latérale#Épingler des onglets|épingler des onglets dans la barre latérale]].

## Basculer vers un autre onglet

Sélectionnez un onglet pour y basculer. Ou utilisez un raccourci clavier :

| Basculer vers                     | macOS            | Windows/Linux        |
|-----------------------------------|------------------|----------------------|
| **Onglet suivant**                | `⌃`+`⇥`         | `Ctrl`+`Tab`         |
| **Onglet précédent**              | `⌃`+`⇧`+`⇥`    | `Ctrl`+`Shift`+`Tab` |
| **Premier onglet à gauche**       | `⌘`+`1`          | `Ctrl`+`1`           |
| **2e au 8e onglet**               | `⌘`+`2`..`8`     | `Ctrl`+`2`..`8`      |
| **Dernier onglet à droite**       | `⌘`+`9`          | `Ctrl`+`9`           |
| **Onglet récemment fermé**        | `⌘`+`⇧`+`t`     | `Ctrl`+`Shift`+`t`   |

## Empiler les groupes d'onglets

Vous pouvez empiler les onglets pour les faire glisser par-dessus d'autres onglets dans le même groupe d'onglets.

Pour empiler les notes, sélectionnez la flèche vers le bas dans le coin supérieur droit du groupe d'onglets, puis sélectionnez **Empiler les notes**.

![tab-stacks](https://user-images.githubusercontent.com/693981/188205363-0f24b2a5-3706-4a8c-b38b-7a66baa68ce6.gif)

L'empilement d'onglets est inspiré des [notes glissantes d'Andy Matuschak](https://notes.andymatuschak.org/).

## Vues liées

Les _vues liées_ sont des onglets qui font référence à un autre onglet. Lorsque le contenu de l'onglet référencé change, la vue liée change également.

Pour les onglets de notes, vous pouvez utiliser les modules suivants comme vues liées :

- [[Vue graphique]] (locale)
- [[Rétroliens]]
- [[Plan]]

Pour ouvrir une vue liée pour un onglet de note :

1. Sélectionnez **Plus d'options** ( ![[lucide-more-horizontal.svg#icon]] ) dans le coin supérieur droit de la note.
2. Sous **Ouvrir une vue liée**, sélectionnez la vue liée que vous souhaitez ouvrir.

## Enregistrer les dispositions

Vous pouvez enregistrer et restaurer les dispositions de fenêtres en utilisant le module [[Espaces de travail]].
