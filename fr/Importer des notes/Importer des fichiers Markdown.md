Obsidian utilise des fichiers Markdown `.md` comme format principal pour les notes. Cela facilite l'importation de fichiers Markdown dans Obsidian. De nombreuses applications peuvent éditer des fichiers Markdown ou exporter vers ce format.

Il existe plusieurs façons d'ajouter des fichiers Markdown à votre coffre Obsidian :

1. Glisser-déposer des fichiers ou des dossiers dans la fenêtre de l'[[Explorateur de fichiers]]
2. Utiliser le gestionnaire de fichiers de votre système (par exemple l'Explorateur Windows ou le Finder sur macOS), et déplacer vos fichiers directement dans le dossier du coffre Obsidian.

### Conversion depuis d'autres variantes de Markdown

Obsidian utilise le [[Obsidian Flavored Markdown]]. Certaines applications utilisent une syntaxe Markdown légèrement différente. Vous pouvez utiliser l'[[Importateur depuis des fichiers Markdown]] pour convertir vos fichiers vers la syntaxe Markdown d'Obsidian.
