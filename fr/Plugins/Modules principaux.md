Cette page référence les modules principaux, lesquels sont pré-installés lorsque vous téléchargez Obsidian. Les modules principaux sont officiellement créés et mis à jours par l'équipe d'Obsidian, et inclus dans l'application.

Certains modules principaux sont désactivés par défaut. Vous pouvez les activer dans **[[Paramètres]]** **→** **Plugins principaux.** 

L'équipe d'Obsidian gère aussi la maintenance et le développement de [[Modules principaux#Autres modules|certains modules communautaires disponibles en sources ouvertes]]. 


> [!NOTE] Plugin communautaires
> Si vous ne trouvez pas ici le module dont vous avez besoin, vous pouvez jeter un oeil aux[[Modules complémentaires| modules construits par la communauté d'utilisateurs]], qui sont beaucoup plus nombreux et variés. Vous pouvez aussi [en créer un vous même](https://docs.obsidian.md/Plugins/Getting+started/Build+a+plugin).

## Liste des modules principaux

- **[[Afficheur Web]]** : 
	- Ouvrez des pages Web à l'intérieur d'Obsidian. 
- **[[Aperçu de page]]** : 
	- Prévisualisez le contenu d'une note en plaçant votre souris par-dessus un lien. 
- **[[Bases]]** :
	- Créez un affichage personnalisé que vous pouvez éditer et auxquels vous pouvez appliquer des règles de tri et de filtres.
- **[[Canvas]]** :
	- Organisez visuellement vos notes et créez des schémas dans un espace infini pour déployer vos idées.
- **[[Commandes slash]]** :
	- Déclenchez des commandes Slash dans l'éditeur avec la touche barre oblique `/`. 
- **[[Compositeur de notes]]** : 
	- Fusionnez deux notes ou divisez en une en deux. 
- **[[Créateur de notes uniques]]** : 
	- Créez des notes avec des préfixes d'horodateurs uniques, pour des workflows tels que Zettelkasten ou ou Slip box. 
- **[[Diapositives]]** :
	- Réalisez des présentations à partir de vos notes.
- **[[Enregistreur audio]]** :
	- Enregistrez des pistes audios directement dans une note.
- **[[Espaces de travail]]** :
	- Enregistrez différentes dispositions d'espace de travail et passez de l'une à l'autre rapidement. 
- **[[Explorateur de fichiers]]** :
	- Naviguez dans les fichiers et les dossier de votre coffre.
- **[[Liens sortants]]** : 
	- Affichez tous les liens apparaissant dans la note que vous consultez.
- **[[Importateur depuis des fichiers Markdown|Convertisseur]]** :
	- Convertissez des fichiers Markdown venant d'autres applications en un [[Le format Markdown dans Obsidian|Markdown adapté à Obsidian]]. 
- **[[Modèles]]** :
	- Insérez automatiquement du contenu prédéfini lors de la création de notes.
- **[[Compteur de mots]]** :
	- Affichez le nombre de mots et de caractères présents dans la note que vous consultez.
- **[[Note aléatoire]]** : 
	- Ouvrez une note au hasard.
- **[[Notes quotidiennes]]** :
	- Créez et ouvrez des notes basées sur la date du jour.
- **[[Palette de commandes]]** :
	- Accédez rapidement à toutes les commandes possible à l'aide d'un [[Raccourcis clavier|raccourci clavier]]. 
- **[[Plan]]** :
	- Affichez la table des matières de la note que vous consultez.
- **[[Publier]]** :
	- Publier vos notes sur internet en tant que site web, wiki ou documentation.
- **[[Rechercher]]** : 
	- Trouver des fichiers dans votre coffre.
- **[[Restaurateur de fichiers]]** : 
	- Restaurez votre travail perdu grâce à des sauvegardes régulières.
- **[[Rétroliens]]** :
	- Visualisez tous les liens mentionnés dans une note.
- **[[Sélecteur rapide]]** : 
	- Cherchez, créez et ouvrez des notes à partir d'un [[Raccourcis clavier|raccourci clavier]]. 
- **[[Signets]]** :
	- Sauvegardez des notes, des titres, des recherches et bien plus encore. 
- **[[Sync]]** :
	- Synchronisez vos notes entre différents appareils.
- **[[Volet de mots-clés]]** :
	- Listez tous les mots-clés (tags) de votre coffre.  
- **[[Propriétés]]** : 
	- Listez tous les propriétés de votre coffre, et voyez celles de la note active. 
- [[Vue graphique]] :
	- Visualisez les liens entre les notes de votre coffre.
- **[[Affichage des pieds-de-page]]**
	- Affichez une liste des pieds-de-page présents dans la note que vous consultez.

## Autres modules

L'équipe d'Obsidian participe aussi à la maintenance et au développement de certains [[modules complémentaires]] :
- **[[Module d'importation]] (Importer) :**
	- Convertissez des fichiers provenant d'autres applications et de formats non pris en charge par Obsidian. 
- **[[Maps]]** :
	- Ajoutez une vue cartographique au module [[Bases]].
