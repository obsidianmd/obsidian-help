Le web clipper d'Obsidian est une extension de navigateur internet gratuite qui vous permet de surligner, annoter et sauvegarder dans votre coffre des le contenu des pages internet.

> [!warning] Nécessite Obsidian 1.7.2 ou version antérieure
> Le web clipper fonctionne avec des fonctionnalités ajoutés au [[Obsidian URI]] de la version 1.7.2 d'Obsidian. Assurez-vous donc d'utiliser la version 1.7.2 d'Obsidian ou une version antérieur si vous souhaitez utiliser le web clipper.

# Installer le web clipper

Tout d'abord, installez le web clipper d'Obsidian à partir du catalogue d'extensions officiel de votre navigateur : 
- [Chrome](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf) ;
- [Firefox](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/) ;
- [Safari](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363) ;
- [Edge](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme).

# Politique de confidentialité

Le web clipper d'Obsidian sauvegarde le contenu localement dans votre coffre d'Obsidian selon les principe de notre [politique de confidentialité](https://obsidian.md/privacy). Vous données ne sont pas collectées et nous ne collectons aucune autre information. [Le code source est ouvert](https://github.com/obsidianmd/obsidian-clipper) et vous pouvez donnez votre avis pour ses modifications.
