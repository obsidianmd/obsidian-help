Le module des **rétroliens** (ou des **occurrences liées**) montre comment la note que vous êtes en train de consulter est référencée dans d'autres notes. Après l'avoir activé, vous verrez le nombre de liens dirigés vers cette note dans la barre latérale droite.

Pour voir les rétroliens, développez la barre latérale droite, puis cliquez sur "**Rétroliens pour [nom de la note actuelle]**" ![[obsidian-icon-links-coming-in.svg]]. Vous devriez ensuite voir un panneau qui montre les rétroliens de la note actuelle :
![[rétroliens.png]] 
Sous la mention "**Occurrences liées**" vous pouvez voir le nom de toutes les notes contenant un lien vers la note actuelle, avec le nombre de fois que ce lien apparaît dans chaque note et au total.

En cliquant sur la petite flèche à gauche du nom de chaque note, vous pourrez voir un extrait du contexte textuel dans lequel se trouve chaque lien. Vous pouvez l'agrandir à votre guise en cliquant sur les flèches haut et bas. Notez que si elle n'apparaissent pas, c'est que le lien se trouve à une extrémité de la note.

![[rétroliens-contexte.png]]

En développant la section "**Occurrences non liées**", vous pouvez également voir où le nom de la note est mentionné, mais pas explicitement lié (en d'autres termes, mentionné sans les crochets).
