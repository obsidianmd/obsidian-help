Diapositives est un [[Modules principaux|module principal]] qui vous permet de créer des présentations à partir de vos notes. Pour démarrer une présentation :

**Menu du fichier :**

- Faites un clic droit sur l'onglet d'une note et cliquez sur **Démarrer la présentation**.

**Palette de commandes :**

1. Appuyez sur `Ctrl+P` (ou `Cmd+P` sur macOS) pour ouvrir la [[Palette de commandes]].
2. Recherchez la commande **Démarrer la présentation**.
3. Appuyez sur `Entrée` avec la commande sélectionnée pour démarrer une présentation à partir de la note active.

Pour naviguer entre les diapositives, cliquez sur les flèches gauche et droite dans le coin inférieur droit, ou appuyez sur les touches fléchées gauche et droite de votre clavier.

Vous pouvez également passer à la diapositive suivante en appuyant sur la barre d'espace.

Pour arrêter la présentation, appuyez sur `Échap` ou cliquez sur la croix dans le coin supérieur droit de la présentation.

Vous pouvez utiliser n'importe quel fichier Markdown valide comme présentation. Pour séparer les diapositives, saisissez `---` au début d'une ligne entourée de lignes vides.

```md
# Présentations avec Diapositives

Une démonstration sur la façon de créer des présentations avec Diapositives.

---

## Mise en forme

Vous pouvez utiliser la mise en forme Markdown habituelle, comme le texte en *italique* et en **gras**.

---

## Diapositives

Utilisez `---` pour séparer les diapositives.
```
