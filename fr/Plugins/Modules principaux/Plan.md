Plan est un [[Modules principaux|module principal]] qui fait la liste de tous les titres présents dans la note que vous consultez. Le plan se trouve par défaut dans le **volet rétractable de droite**, mais vous pouvez le déplacer par glisser-déposer si vous le souhaitez.

Vous pouvez faire une **recherche textuelle** pour filtrer les titres qui s'affichent et utiliser la fonction **Tout réduire / Tout déplier**, comme avec l'[[explorateur de fichiers]]. 

Pour aller directement dans la section d'un titre, cliquez sur celui-ci dans le plan. Pour déplacer des sections, faites un **clic glisser-déposer** sur le titre que vous souhaitez déplacer à partir du plan.
