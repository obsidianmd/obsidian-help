Le sélecteur rapide est un [[Modules principaux|module principal]] qui vous permet de d'ouvrir de nouvelles notes et même d'en créer de manière fluide et rapide.

## Ouvrir le sélecteur rapide 

Il existe plusieurs façon d'ouvrir le sélecteur rapide :
- [[Raccourcis clavier|Raccourci clavier]] : `Ctrl/Cmd + O`.
- Dans le bandeau latéral situé le long du côté droit de l'écran, cliquez sur **Ouvrir le sélecteur rapide**. ![[icone-sélecteur-rapide.svg]]
- Sur mobile, quand vous n'êtes pas en mode édition, cliquez sur l'icône "**Plus**"  situé vers le haut.

![[sélecteur-rapide.png]]

## Trouver une note avec le sélecteur rapide et l'ouvrir

Une fois que vous avez ouvert le sélecteur rapide, commencez simplement à taper le nom du fichier que vous recherchez et il apparaîtra sous le champs de recherche. Utilisez les flèches haut et bas pour parcourir les résultats, `Enter` pour ouvrir la note et `Echap` pour quitter le sélecteur rapide. Vous pouvez aussi ouvrir la note avec la souris. Si vous avez défini un ou plusieurs alias dans les [[propriétés]] de la note, alors vous pouvez vous en servir pour la rechercher. 

> [!NOTE] Utiliser le sélecteur rapide pour basculer d'une note à l'autre
> S'il n'y a rien d'écrit dans le champs de recherche, les fichiers consultés le plus récemment s'afficheront dans le sélecteur rapide. Vous pouvez donc vous en servir pour basculer d'une note à l'autre à l'aide du clavier, même si elles sont déjà ouvertes.

> [!INFO] Fichiers exclus
> Les fichiers correspondant à vos paramètres d'[[Paramètres#Fichiers exclus|exclusion de fichiers]] s'afficheront après les autres. 

Si le nom que vous avez inscrit dans le sélecteur rapide ne correspond à aucun résultat existant, vous pouvez appuyer sur la touche `Entrer` pour **créer une nouvelle note avec ce nom**. S'il y a quand même quelques résultats similaires mais ne correspondant pas tout à fait à ce que vous voulez, il vous faudra appuyer sur les touches `maj + Entrer` pour créer une nouvelle note plutôt que d'ouvrir celle qui apparaît en première dans la liste.

Appuyez sur les touches `Opt/Ctrl + Entrer` pour **ouvrir la note dans un nouvel onglet**.

Appuyez sur les touches `Ctrl + Alt + Entrer` (Windows/Linux) ou `Opt + Cmd + Entrer` (macOS) pour **ouvrir la note dans une nouvelle fenêtre vers la droite**.

> [!INFO] Information
> La fonctionnalité de saisie semi-automatique passe à un algorithme de résultats plus simples quand le nombre d'éléments dans le coffre atteint un nombre de 10 000. Cela permet au logiciel de maintenir des performances optimales.
