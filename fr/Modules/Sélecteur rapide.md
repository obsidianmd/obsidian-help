Le sélecteur rapide est un [[Modules principaux|module principal]] qui vous permet de rechercher et d'ouvrir des notes en utilisant uniquement votre clavier.

## Ouvrir le sélecteur rapide

Il existe plusieurs façons d'ouvrir le sélecteur rapide, lorsqu'il est activé :
- Appuyez sur `Ctrl+O` (ou `Cmd+O` sur macOS) pour ouvrir le sélecteur rapide ;
- Dans le ruban, cliquez sur **Ouvrir le sélecteur rapide** ( ![[lucide-file-search.svg#icon]] ) ;
- Sur mobile, lorsque vous n'êtes pas en train de modifier une note, appuyez sur l'icône plus en bas au centre de l'application.

## Ouvrir une note avec le sélecteur rapide

1. [[#Ouvrir le sélecteur rapide]].
2. Tapez pour rechercher une note par nom ou alias.
3. Naviguez vers la note à l'aide des touches fléchées.
4. Appuyez sur `Entrée` pour ouvrir la note sélectionnée.

> [!info] 
> La fonctionnalité de complétion automatique bascule vers un algorithme de résultats plus simple lorsque le coffre atteint 10 000 éléments afin de maintenir des performances optimales de l'application. 
^search-autocomplete-large

Si le texte ne correspond à aucune note, vous pouvez appuyer sur `Entrée` pour créer une note portant ce nom. Même si le texte correspond à une ou plusieurs notes similaires, vous pouvez toujours créer une note avec le nom exact en appuyant sur `Maj+Entrée`.

Vous pouvez ouvrir la note sélectionnée dans un nouvel onglet en appuyant sur `Ctrl+Entrée` (ou `Cmd+Entrée` sur macOS).

> [!tip] Basculer entre deux notes
> Si le terme de recherche est vide, le sélecteur rapide affiche les notes les plus récentes. Basculez entre deux notes en ouvrant le sélecteur rapide, en appuyant sur la touche `Bas`, puis en appuyant sur `Entrée`.


> [!info] Fichiers exclus
> Les fichiers correspondant à vos motifs de [[Paramètres#Fichiers exclus|Fichiers exclus]] sont dépriorisés dans les résultats du sélecteur rapide.
