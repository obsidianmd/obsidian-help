La [[Modules principaux|Vue des notes de bas de page]] est un [[Modules principaux|module principal]] qui liste toutes les notes de bas de page dans la note active.

Cliquez sur une note de bas de page pour éditer son texte. Vous pouvez également naviguer vers la position de la note de bas de page dans la note.
