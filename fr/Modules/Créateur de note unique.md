Le Créateur de note unique est un [[Modules principaux|module principal]] qui vous permet de créer des notes avec des noms basés sur l'horodatage, également connues sous le nom de notes Zettelkasten.

Par exemple, si vous créez une nouvelle note à 09:45 le 1er janvier 2024, le Créateur de note unique crée une note avec `202401010945` comme nom. Si une note portant le même nom existe déjà, la nouvelle note utilise le prochain horodatage disponible.

## Créer une note unique

**Ruban :**

- Dans le ruban, cliquez sur **Créer une nouvelle note unique** ( ![[obsidian-icon-sheets-in-box.svg#icon]] ).

**Palette de commandes :**

1. Appuyez sur `Ctrl+P` (ou `Cmd+P` sur macOS) pour ouvrir la [[Palette de commandes]].
2. Recherchez la commande **Créer une nouvelle note unique**.
3. Appuyez sur `Entrée` lorsque la commande est sélectionnée.

## Créer des notes uniques à partir d'un modèle

Par défaut, les nouvelles notes uniques sont vides. Pour utiliser un autre fichier comme modèle :

- Ouvrez les **[[Paramètres]]**.
- Dans la barre latérale, cliquez sur **Modules principaux → Créateur de note unique**.
- Dans **Emplacement du fichier modèle**, entrez le fichier que vous souhaitez utiliser comme modèle.

Pour plus d'informations sur la rédaction de modèles, consultez [[Modèles]].
