Le Plan est un [[Modules principaux|module principal]] qui liste les entêtes de la note active.

Pour naviguer vers cette section dans la note, cliquez sur l'entête dans le plan.

Pour réorganiser les sections dans la note, cliquez et glissez l'entête dans le plan.
