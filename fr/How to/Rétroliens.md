Les rétroliens sont l'une des fonctionnalités les plus puissantes d'Obsidian. Ils vous permettent de voir quels sont les liens venant d'autres notes qui pointent vers la note que vous être en train de consulter. Notez qu'il s'agit de l'un des [[modules principaux]] d'Obsidian. Il est activé par défaut, mais vous pouvez le désactiver si vous le souhaitez en allant dans les **Paramètres des Modules principaux**. 

# Comment voir les rétroliens d'une note ?

Pour voir les rétroliens d'une note, cliquez sur l'onglet **Rétroliens ![[obsidian-icon-links-coming-in.svg]]**  dans la barre latérale droite. Vous pouvez aussi l'ouvrir en passant par la [[palette de commandes]] et en activant la commande **Rétroliens : Afficher le volet des rétroliens** ou en cliquant sur **Plus d'option** (les trois points verticaux en haut à droite de l'onglet de la note active) **→** **Ouvrir la vue liée** **→** **Ouvrir les rétroliens**.

Vous pouvez également créer un volet séparé de rétroliens dédié à une note dans le **Menu Plus d'options**. Ce sera un [[volet lié]] connecté à la note qui pourra être déplacé n'importe où, comme décrit dans [[Travailler avec plusieurs volets]]. Vous trouverez une option équivalente dans la [[Palette de commandes]].

## Afficher les rétroliens à l'intérieur d'une note

Plutôt que d'afficher les rétroliens dans un onglets séparé de la note, vous pouvez les faire apparaître tout en bas de celle-ci.

Pour ce faire, ouvrez la [[palette de commandes]], puis sélectionnez **Rétroliens : Basculer les rétroliens dans le document**. Cette commande vous permet de faire cela de façon ponctuelle dans une seule note et vous pouvez enlever les rétroliens à tout moment en réactivant cette même commande. 

Mais si vous voulez que cette fonctionnalité s'active automatiquement à chaque fois que vous ouvrez une note, c'est possible aussi. Pour cela, allez dans les **Paramètres → Modules principaux** puis cliquez sur l'engrenage à côté du plugin **Rétroliens** et cochez **Rétroliens dans le document**. 

Une fois cela fait et quelle que soit la méthode utilisée, vous devriez voir tout en bas de votre note quelque chose comme ceci :

# Utiliser les rétroliens 

Le volet rétroliens comprend deux sections : **Occurences liées** et **Occurences non liées** :
- Les **Occurrences liées** sont les rétroliens pointant vers les notes contenant un ou plusieurs liens actifs pointant vers la note que vous êtes en train de consulter.
- Les **Occurrences non liées** sont les rétroliens pointant vers des notes qui contiennent le nom de la note que vous êtes en train de consulter sans que ce ne soit des liens actifs.

Cela donne accès aux fonctionnalités suivantes : 
![[outils-rétroliens.png]] 
1. **"Réduire les résultats"** permet d'afficher/masquer un extrait du contenu des notes présentes dans la liste servant à montrer le lien dans son contexte. 
2. **"Afficher plus de contexte"** permet d'agrandir/réduire la taille de l'extrait visible. Vous pouvez aussi l'agrandir à votre guise avec les deux petites flèches situées à droite de l'extrait. 
3. **"Modifier l'ordre de tri"** permet de changer l'ordre dans lequel apparaissent les résultats au sein d'une même section (alphabétique, date d'édition, etc.). 
4. **"Afficher le filtre de recherche"** permet d'afficher/cacher un champs de recherche textuelle avec lequel vous pouvez faire en sorte que n'apparaissent que les résultats correspondant à un ou plusieurs mots-clés. Pour en savoir plus, consultez la note sur les fonctionnalités de [[Recherche]].
