Les onglets peuvent être liés entre eux. Cela a deux effets principaux :
1. Par défaut, lors de l'ouverture d'un nouveau [[Modes édition et aperçu|volet d'aperçu]] depuis une note qui est en mode édition, l'onglet se trouvant dans le nouveau volet est lié à l'ancien. Cela veut dire que chaque modifications que vous apportez dans l'éditeur apparaîtront dans l'aperçu en temps réel. Cela signifie également que faire défiler l'un fera défiler l'autre.
2. Lorsqu'un nouveau [[Rétroliens|volet des rétroliens]] est créé à partir d'une note, il sera lié à cette note. Cela signifie que les rétroliens affichés seront toujours ceux associés à la note qui se trouve dans ce volet. Ceci est utile si vous souhaitez référencer un ensemble de rétroliens tout en laissant le volet de rétroliens normal se mettre à jour avec n'importe quelle autre note sur laquelle vous travaillez.

Les onglets liés sont identifiés par une icône de lien. Vous pouvez survoler le lien avec la souris pour voir à quel onglet il est lié, ou cliquer sur l'icône pour le libérer, afin qu'ils ne soient plus liés. Cela ne fermera pas l'onglet précédemment lié.
