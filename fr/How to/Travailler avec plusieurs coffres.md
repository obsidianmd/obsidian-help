Chaque ensemble de notes dans Obsidian est appelé  un *"coffre"*. Ce coffre correspond à un dossier principal appelé *"dossier racine"* (un terme qu'on rencontre parfois quand on modifie certains paramètres). Le dossier racine, ou coffre, contient tous les fichiers et dossiers qui apparaîtrons quand vous entrerez dans votre coffre. En ce moment, vous vous trouvez dans le coffre `Aide d'Obsidian`, qui est simplement un coffre inclus avec le logiciel.

Vous pouvez ouvrir un autre coffre n'importe où sur votre ordinateur, et il mémorisera ses propres paramètres et modules. Ceci est utile, par exemple, si vous avez un coffre dans lequel vous conservez des notes, et un autre dans lequel vous rédigez de longs documents.

Le Coffre d'Aide est stocké dans un répertoire spécial Obsidian (voir [[Comment Obsidian stocke les données]]), mais à part cela, c'est un répertoire de fichiers comme les autres. Pour y revenir lorsque vous êtes dans un autre coffre, cliquez sur le symbole en forme de point d'interrogation, en-dessous de l'[[explorateur de fichiers]]. 

# Travailler avec plusieurs coffres
 

Le ***"profil de coffre"**** ou ***"gestionnaire de coffres"*** vous permet de passer facilement d'un coffre à l'autre. Pour l'ouvrir, cliquez sur le nom du coffre dans lequel vous vous trouvez (en-dessous de l'[[explorateur de fichiers]] et à côté du symbole "**![[symbole-chevrons.svg]]**" , puis cliquez sur **"Gérer les coffres"**. Vous pouvez aussi faire cela en sélectionnant "**Ouvrir un autre coffre**" à partir de la [[palette de commandes]] ou avec un raccourci clavier que vous aurez paramétré vous-même.  ^b613b4

Vous êtes maintenant dans votre **gestionnaire de coffre**. 
À gauche, s'affiche tous les coffres que vous avez déjà créés. Il vous suffit de cliquer sur leur nom pour les ouvrir. Vous pouvez aussi faire toute une série d'autres actions en cliquant sur "**Plus d'options** ![[symbole-plus-d'options.svg]]" . 
À droite, trois options principales :
- **Créer un nouveau coffre** vous permet de créer un coffre en créant un nouveau dossier racine, celui-ci sera vide puisque vous n'aurez pas encore créés ou ajouté de fichiers à l'intérieur ; ^5f73c2
- **Ouvrir un dossier comme coffre** vous permet de créer un coffre à partir d'un dossier déjà existant, celui-ci contiendra tous les fichiers qu'il y avait déjà dans ce dossier avant la création du coffre, même ceux dont le format n'est pas pris en charge (vous pouvez choisir de les rendre visibles ou non dans les paramètres) ; ^9dd295
- **Se connecter à [[Obsidian Sync]]** vous permet d'ouvrir un coffre qui existe déjà et qui est synchronisés entre plusieurs appareil via le service [[Obsidian Sync]]. 
En dessous, deux autres options intéressantes s'offrent à vous. L'une vous permet de choisir en quelle langue sera le logiciel. L'autre, le petit bouton en forme de point d'interrogation, vous permet d'accéder à différents services d'aide. Parmi ceux-ci, il y a :
- **La documentation d'aide du site officiel [Obsidian Help](https://help.obsidian.md/).** La version française n'existe pas encore, mais elle est en cours de traduction. Si vous voulez participer à sa traduction, consultez la page [[Contribuer sur Obsidian]]. 
- **Le forum Discord**, dont une partie en réservée aux francophones (vers le bas dans l'interface). Vous pouvez y aller pour poser vos questions aux-quelles la documentation d'aide n'aura pas su répondre. La communauté y est très active !
- **Le forum officiel**, un site où vous pourrez trouver des questions-réponses entre membres de la communauté.
- **Le coffre de test** est une partie de la documentation d'aide qui est incluse automatiquement dans Obsidian quand vous téléchargez le logiciel. Si vous lisez ces lignes dans un coffre Obsidian, vous vous y trouvez déjà ! ;) Notez par ailleurs qu'une fois que vous l'avez ouvert pour la première fois, il apparaît à gauche dans le gestionnaire de coffres. Vous pouvez même l'ouvrir sans avoir à passer par le gestionnaire de coffres, directement en cliquant sur le petit icône "**![[symbole-chevrons.svg]]**". 

## Créer un nouveau coffre 

Pour créer un nouveau coffre, vous pouvez ouvrir un dossier déjà existant en tant que coffre, ou en créer un nouveau qui correspondra à un coffre. 
1. Pour ce faire, suivez les mêmes instructions que [[Gérer les coffres#^b613b4|celles données précédemment pour accéder au sélectionneur de coffre]]. 
2. Une fois cela fait, sélectionner "**[[Gérer les coffres#^5f73c2|Créer]]**" ou "**[[Gérer les coffres#^9dd295|Ouvrir]]**" selon votre souhait (voir les deux liens précédent pour savoir quel choix faire). 
	- Si vous cliquez sur "**Créer**" :
		1. Dans "**Nom du coffre**", entrez le nom que vous souhaitez donner à votre coffre ;
		2.  Cliquez sur **Parcourir** pour choisir l'emplacement du coffre dans votre ordinateur ;
		3. Cliquez à nouveau sur **Créer**
	- Si vous cliquez sur "**Ouvrir**", choisissez simplement dans la fenêtre d'exploration qui s'ouvre quel dossier vous voulez transformer en coffre. 


> [!NOTE] Ouvrir un coffre à partir d'Obsidian Sync
> Si vous voulez ouvrir un coffre à partir d'Obsidian Sync, référez-vous plutôt à la page [[Paramétrer Obsidian]]. 


>Les [[Liens internes]] ne peuvent pas être partagés entre différents coffres. Si vous avez un coffre qui se trouve dans un autre coffre, cela peut prêter à confusion, nous vous recommandons donc de ne pas le faire.

# Renommer un coffre

Etant donné que le nom d'un coffre et celui de son dossier racine sont les mêmes, renommer un coffre revient également à renommer le dossier racine.
1. Sur votre ordinateur, ouvrez Obsidian ;
2. En bas à gauche, cliquez sur le petit symbole "**![[symbole-chevrons.svg]]**" (ou sur le nom du coffre actuellement ouvrir qui se trouve juste à côté) ;
3. Dans la fenêtre contextuel qui s'ouvre, sélectionnez "**Gérer les coffres**" ;
4. Dans la liste des coffres existants, sélectionnez **Plus d'options** "![[symbole-plus-d'options.svg]]" à côté du coffre que vous souhaitez renommer ; 
5. Sélectionnez **Renommer le coffre** ; 
6. Entrez le nouveau nom du coffre, puis cliquez sur ``Entrer``. 

# Déplacer un coffre

1. Sur votre ordinateur, ouvrez Obsidian ;
2. En bas à gauche, sélectionnez le sélectionneur de coffres "**![[symbole-chevrons.svg]]**" ;
3. Dans la fenêtre contextuel qui s'ouvre, sélectionnez "**Gérer les coffres**" ;
4. Fermez la fenêtre du coffre actif, mais laissez celle du sélectionneur de coffres ouverte ;
5. Dans la liste des coffres existants, sélectionnez **Plus d'options** "![[symbole-plus-d'options.svg]]" à côté du coffre que vous souhaitez déplacer ; 
6. Sélectionnez **Déplacer le coffre**, puis sélectionnez le nouvel emplacement. 

Certains systèmes d'exploitation ne permettent pas de déplacer un coffre avec le sélectionneur de coffres. Dans ce cas, vous devez le déplacer manuellement : 
1. Fermez Obsidian ;
2. Déplacez votre coffre à un nouvel emplacement ; 
3. Réouvrez votre coffre ;
4. Cliquez sur le sélectionneur de coffres "**![[symbole-chevrons.svg]]**" ;
5. Dans la fenêtre contextuelle qui s'ouvre, sélectionnez **Gérer les coffres** ;
6. Naviguez et sélectionnez le nouveau dossier correspondant à votre coffre ;
7. Cliquez sur **Ouvrir** ;
8. Vérifiez que le contenu du coffre soit resté inchangé. Si nécessaire, réactivez les [[modules complémentaires]] en allant dans les **[[Paramètres]]** **→** **Modules complémentaires** **→** **Désactiver le mode restreint.** 

# Supprimer un coffre

Si vous supprimez un coffre depuis la liste des coffres du gestionnaire de coffres, cela ne supprime pas son contenu. Mais si vous le faites depuis l'[[explorateur de fichiers]] de votre système d'exploitation, son contenu sera supprimé.

Voici les étapes à suivre pour faire la suppression uniquement dans le **gestionnaire de coffres** :
1. Sur votre ordinateur, ouvrez Obsidian ;
2. En bas à gauche, sélectionnez le sélectionneur de coffres "**![[symbole-chevrons.svg]]**" ;
3. Dans la fenêtre contextuel qui s'ouvre, sélectionnez "**Gérer les coffres**" ;
4. Dans la liste des coffres existants, sélectionnez **Plus d'options** "![[symbole-plus-d'options.svg]]" à côté du coffre que vous souhaitez enlever de la liste ; 
5. Sélectionnez **Supprimer de la liste**.

# Transférer des paramètres d'un coffre à un autre

Pour avoir les mêmes paramètres dans plusieurs coffres différents, pas besoin de tous les reparamétrer à la main, ce serait bien trop fastidieux ! À partir de l'[[explorateur de fichiers]] (ou d'un terminal), copier le dossier qui s'appelle `.obsidian` à partir du dossier racine du coffre source et collez le dans le dossier racine du coffre destinataire. Pour que les changements soient effectifs, il ne vous reste plus qu'à redémarrer Obsidian, et le tour est joué !

> [!NOTE] Et si je ne trouve pas le dossier `.obsidian` ?  
> Bon nombre de systèmes d'exploitation cachent par défaut les dossiers dont le nom commence par un point. C'est parce que ce sont des dossiers de configuration, et les développeurs de certains systèmes d'exploitation partent du principe que les utilisateur n'ont pas besoin d'accéder à ces dossiers. 
> Une solution simple pour résoudre ce problème est, si votre système d'exploitation le permet, de faire une modification dans les paramètres de celui-ci pour que ces dossiers deviennent visibles. 
> Si vous êtes à la recherche d'une autre solution, nous vous suggérons de consulter les notes [[paramétrer le coffre]] et [[fichiers de configuration]] ainsi que le site internet de votre système d'exploitation.
