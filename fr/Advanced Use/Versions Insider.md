Si vous avez souscris à une [[licence Catalyst]], vous avez accès aux nouvelles mise à jours avant leur mise à disposition sur le site Internet pour tout le monde. 


> [!WARNING] Attention 
> Les versions Beta incluent de nouvelles choses mais peuvent être moins stables. N'activez pas cela si vous préférez avoir une expérience plus fiable avec Obsidian. 
> 
> Ayez à l'esprit que les modules communautaires et leurs développeurs reçoivent les versions Beta en même temps que les personnes titulaires d'une licence Catalyst. Soyez patients avec les développeurs qui doivent faire de nouvelles mises à jours pour rendre leurs modules compatibles avec les nouvelles mises à jours. 

# Activer les versions Beta sur ordinateur

Pour recevoir les versions Beta dès qu'elles sont disponibles, suivez les étapes suivantes : 
1. Ouvez les [[Paramètres]] ;
2. Dans la barre latérale, sélectionnez **Général** ;
3. Sous **Compte → Votre Compte**, sélectionnez **Se connecter** ;
4. Dans **Email**, renseignez votre adresse email ; 
5. Dans **Mot de passe**, renseignez votre mot de passe ; 
6. Une fois que vous vous êtes inscrit, retournez dans les Paramètres ; 
7. Dans **Général → Application**, activez **Recevoir les versions Beta** ; 
8. Cliquez sur **Vérifier les mises à jours**, puis **Relancer**. 

# Activer les versions Beta sur appareil mobile

Pour trouver les instructions vous expliquant comment installer les versions Beta sur votre appareil mobile, suivez ces étapes : 
1. Télécharger et installez [Discord](https://discord.com/) ; 
2. Rejoignez le [serveur Discord Obidian](https://discord.com/invite/obsidianmd) ; 
3. Obtenez votre [[Licence Catalyst#|badge Discord]] vous permettant d'accéder aux salons des membres Beta ; 
4. Dans le salon `#insider-welcome`, vous trouverez les instructions à suivre pour accéder à votre téléchargement en fonction du type d'appareil que vous utilisez.

# Signaler des problèmes et autres retours d'expérience

Si vous remarquez un dysfonctionnement dans une versions Beta, veuillez s'il vous plaît en faire part à l'équipe d'Obsidian pour que nous puissions la corriger. Avant de faire cela, référez-vous au [forum](https://forum.obsidian.md/) et à notre Discord pour vérifier que cela n'a pas déjà été signalé par quelqu'un d'autre. 

Pour signaler un problème, utilisez l'un de ces médiums : 
- Sur Discord, faites le signalement dans le salon ``#insider-release``. 
- Dans le forum, créez un nouveau sujet (topic) sous [Signalement de bugs (Bugs report)](https://forum.obsidian.md/c/bug-reports/7).

Quand vous signalez un problème, renseignez la version de mise à jours du logiciel et le système d'exploitation que vous utilisez. Vous pouvez trouver la version de mise à jours dans **[[Paramètres]] → À propos → Application → Version**. 

# Retourner à la version publique sur ordinateur 

Si vous souhaitez revenir à la version publique d'Obsidian sur votre ordinateur (et donc quitter la version Beta), suivez les étapes suivantes : 
1. Désactiver **Accès à la version Beta** :
	1. Ouvrir les **Paramètres** ;
	2. Dan la barre latérale, sélectionner **Général** ;
	3. Sous **Application**, désactiver **Recevoir les versions Beta**.
2. Fermer Obsidian. 
3. Supprimer le fichier ``obsidian-VERSION.asar ``, sachant que ``VERSION`` correspond à une version d'Obsidian :
	1. Sur Windows, le fichier à supprimer est ``%APPDATA%\obsidian\obsidian-VERSION.asar`` ;
	2. Sur Mac OS : ``~/Library/Application Support/obsidian/obsidian-VERSION.asar`` ;
	3. Sur Linux : `~/.config/obsidian/obsidian-VERSION.asar`.
4. Redémarrez Obsidian. 

# Revenir à la version publique sur mobile 

Pour revenir à la version publique sur mobile (et donc quittez la version Beta), suivez les étapes suivantes : 
1. Faites une sauvegarde des données qui se trouvent dans votre coffre ; 
2. Désinstallez Obsidian ; 
3. Réinstallez Obsidian ; 
4. Récupérez vos données sauvegardées en les réinsérant dans votre coffre, ou, le cas échéant, en activant la synchronisation d'[[Obsidian Sync]] ;
5. Ouvrez Obsdian.
