[[Obsidian Web Clipperの紹介|Web Clipper]]には、ウェブページから不要な要素を取り除き、メインの記事コンテンツのみをクリーンで読みやすいフォーマットで表示する、**リーダー**と呼ばれるビルトインのリーディングモードが含まれています。

## リーディングモードを有効にする

リーダーは以下の方法で有効にできます：

- ![[lucide-book-icon.svg#icon]] 拡張機能パネルの**本のアイコン**。
- **キーボードショートカット**（拡張機能の設定でカスタマイズ可能）。
- 閲覧中のウェブページを右クリックして**コンテキストメニュー**から。

また、![[obsidian.svg#icon]] 拡張機能ボタンをワンクリックでリーダーを開くように設定することもできます。Web Clipperの設定で**一般** → **動作** → **開く動作**に移動し、**リーダー**を選択してください。

上記のすべてのオプションでリーダーを終了し、元のページに戻すこともできます。

## 仕組み

リーダーはページからメインコンテンツを抽出します。元のページは以下を含むクリーンなリーディングビューに置き換えられます：

- 記事のタイトル、著者、公開日、ソースドメイン。
- 画像とフォーマットが保持されたメインの記事コンテンツ。
- 複数の見出しがある記事のサイドバーのアウトライン。
- コードブロックのシンタックスハイライト。
- クリックするとインラインで表示される脚注ポップオーバー。

## リーディングビューをカスタマイズする

拡張機能の設定でリーダーのエクスペリエンスをカスタマイズできます。一部のオプションはリーダーモード内で ![[lucide-case-sensitive.svg#icon]] アイコンをクリックしても利用できます。

| 設定 | 説明 |
| ---------------- | ----------------------------------- |
| フォント | システムにインストールされているフォントを追加します。 |
| フォントサイズ | テキストサイズを調整します。 |
| 行の&nbsp;高さ | 行間のスペース。 |
| 行&nbsp;幅 | コンテンツエリアの最大幅。 |
| 外観 | ライトまたはダークの外観を選択します。 |
| テーマ | カラーを選択します。 |
| カスタムCSS | ページに適用される追加のCSS。 |
