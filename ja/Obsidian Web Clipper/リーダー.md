[[Obsidian Web Clipperの紹介|Web Clipper]]には、ウェブページから不要な要素を取り除き、メインの記事コンテンツのみをクリーンで読みやすいフォーマットで表示する、リーダーと呼ばれるビルトインのリーディングモードが含まれています。

## リーディングモードを有効にする

リーダーは以下の方法で有効にできます：

- 拡張機能パネルヘッダーの**本のアイコン**。
- キーボードショートカット`Alt+Shift+R`（拡張機能の設定でカスタマイズ可能）。
- リーダーモードを再度切り替えると、元のページに戻ります。

## 仕組み

リーダーはページからメインコンテンツを抽出します。元のページは以下を含むクリーンなリーディングビューに置き換えられます：

- 記事のタイトル、著者、公開日、ソースドメイン。
- 画像とフォーマットが保持されたメインの記事コンテンツ。
- 複数の見出しがある記事のサイドバーのアウトライン。
- コードブロックのシンタックスハイライト。
- クリックするとインラインで表示される脚注ポップオーバー。

## リーディングビューをカスタマイズする

リーダーモードには非表示の設定バーがあり、ページの右上隅にホバーすると表示され、リーディングエクスペリエンスをカスタマイズするコントロールが含まれています。

| 設定 | 説明 |
| ----------------- | ------------------------------------------------------------------------------------------ |
| フォントサイズ | テキストサイズを調整します。 |
| 行幅 | コンテンツエリアの最大幅。 |
| 行の高さ | 行間のスペース。 |
| カラー&nbsp;スキーム | |
| テーマ | 自動はシステムの設定に従います。または常にライトモードかダークモードを選択できます。 |
