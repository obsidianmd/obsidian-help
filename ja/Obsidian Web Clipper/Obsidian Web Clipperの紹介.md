Obsidian Web Clipperは、ページをハイライトしてウェブコンテンツを保管庫に保存できる無料のブラウザ拡張機能です。

> [!warning]- Obsidian 1.7.2以上が必要です
> Web Clipperは、Obsidian 1.7.2で[[Obsidian URI]]に追加された機能に依存しています。Obsidian 1.7.2以上をお使いください。

## Web Clipperのインストール

お使いのブラウザの公式ディレクトリからObsidian Web Clipper拡張機能を入手してください：

- [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
	- Chrome、Brave、Arc、Orion、その他のChromiumベースのブラウザ向け。
- [Firefox Add-Ons](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
	- FirefoxおよびFirefox Mobile向け。
- [Safari Extensions](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
	- macOS、iOS、iPadOS向け。
- [Edge Add-Ons](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
	- Microsoft Edge向け。

## Web Clipperの使い方

- [[ウェブページのクリップ]]
	- ページからコンテンツを保存する方法を学びましょう。
- [[ウェブページのハイライト|ハイライター]]
	- 重要な箇所をハイライトし、保存したい要素を選択します。
- [[リーダー|リーダー]]
	- ウェブページをすっきりとした読みやすいビューで閲覧します。
- [[ウェブページの解釈|インタープリター]]
	- 自然言語プロンプトを使用してページ上のデータをキャプチャ・修正します。
- [[Web Clipperのトラブルシューティング|トラブルシューティング]]
	- Web Clipperの一般的な問題を解決します。
- [[Obsidian Web Clipper/テンプレート|テンプレート]]
	- 特定のウェブサイト用のカスタムテンプレートを作成します。
- [[変数]]
	- テンプレートにページデータを事前に入力します。
- [[フィルター]]
	- テンプレート変数の出力を変更します。
- [[ロジック]]
	- テンプレートで条件分岐、ループ、変数の設定を使用します。

## プライバシー

Obsidian Web ClipperはコンテンツをローカルのあなたのObsidian保管庫に保存し、当社の[プライバシーポリシー](https://obsidian.md/privacy)に従います。あなたのデータは収集されず、利用状況の指標も一切収集しません。コードは[オープンソース](https://github.com/obsidianmd/obsidian-clipper)であり、監査可能です。
