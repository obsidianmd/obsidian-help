[[Obsidian Web Clipperの紹介|Web Clipper]]を使うと、ウェブページ上のテキストをハイライトし、Obsidianに保存したい要素を選択できます。ハイライトは保存されるため、ページに戻ったときに再確認できます。

ハイライトは拡張機能を開いたときに[[ウェブページのクリップ|キャプチャ]]してObsidianに保存できます。

## ハイライターをオンにする

ブラウザに応じて、いくつかの方法でハイライトをオンにできます：

- 拡張機能パネルのハイライターアイコン。
- ホットキーで、キーボードから拡張機能を認証する。
- コンテキストメニューで、訪問中のウェブページを右クリックする。

ハイライトをオンにすると、テキスト、画像、ハイライトしたい要素を選択できます。

## ハイライターの設定

Web Clipperの設定にアクセスして、ハイライターの挙動を変更できます。ここではハイライトを`.json`ファイルにエクスポートすることもできます。

`{{content}}`[[変数|変数]]を使って、クリップしたノートにハイライトを挿入するオプションは3つあります：

- **ページ内容をハイライトする** — `==highlight==`という[[Obsidian Flavored Markdown|構文]]でテキストに直接ハイライトを追加します。
- **ページコンテンツを置換する** — ページコンテンツなしで、ハイライトのリストを返します。
- **何もしない** — ハイライトなしで元のコンテンツを返します。

`{{highlights}}`変数を使って、テンプレートにハイライトを直接追加できます。例：

```
{{highlights|map: item => item.text|join:"\n\n"}}
```
