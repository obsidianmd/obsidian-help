[[Obsidian Web Clipperの紹介|Web Clipper]]を使うと、ウェブページ上のテキストをハイライトし、Obsidianに保存したい要素を選択できます。ハイライトは保存されるため、ページに戻ったときに確認できます。

ハイライトは拡張機能を使ってObsidianに保存できます。ハイライターのオプションは拡張機能の設定で定義できます。

## ハイライターを使用する

ハイライターはライブウェブページまたは[[リーダー]]で使用できます。リーダービューでは、テキストを選択するとハイライトするオプションが表示されます。

ハイライターを有効にすることで、選択範囲を自動的にハイライトに追加することもできます：

- 拡張機能パネルまたはリーダーツールバーのハイライターアイコン。
- ホットキーで、キーボードから拡張機能の設定でカスタマイズ可能。
- コンテキストメニューで、訪問中のウェブページを右クリックする。

ハイライターをオンにすると、選択したテキスト、画像、要素がハイライトに追加されます。上記のすべての方法でハイライターを終了することもできます。

## ハイライトをノートに追加する

クリップしたノートにハイライトを挿入する方法は3つあります：

- **ページ内容をハイライトする** — `==highlight==`という[[Obsidian Flavored Markdown|構文]]でテキストに直接ハイライトを追加します。
- **ページコンテンツを置換する** — ページコンテンツなしで、ハイライトのリストを返します。
- **何もしない** — ハイライトなしで元のコンテンツを返します。
 
これらのオプションはテンプレート内の`{{content}}`[[変数|変数]]を変更します。`{{highlights}}`変数を使って、テンプレートにハイライトを直接追加することもできます。例：

```
{{highlights|map: item => item.text|join:"\n\n"}}
```

## ハイライトの表示と検索

作成したすべてのハイライトは、ハイライトページで確認できます。このページは**設定** → **ハイライター**から開けます。

## ハイライトのエクスポート

ハイライトは設定ページまたはハイライトページから`.json`ファイルにエクスポートできます。
