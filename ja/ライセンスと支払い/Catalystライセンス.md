Catalystライセンスは、ベータバージョンへの早期アクセスを提供し、Obsidianの継続的な開発を支援するための1回限りの購入です。[Catalystライセンスを購入](https://obsidian.md/buy/catalyst)することでObsidianをサポートできます。

特典には以下が含まれます：

- Obsidianの[[#早期アクセスバージョンを有効にする|新バージョン]]への早期アクセス
- 限定の[[#インサイダーチャンネルに参加する|インサイダーチャンネル]]へのアクセス
- 限定の[[#フォーラムバッジを取得する|フォーラムバッジ]]と[[#Discordバッジを取得する|Discordバッジ]]

## Catalystティア

Catalystには、貢献額に応じた3つのティアがあります：*Insider*、*Supporter*、*VIP*。

| 特典                                                                               | Insider | Supporter |   VIP    |
| ---------------------------------------------------------------------------------- | :-----: | :-------: | :------: |
| 新バージョンへの早期アクセス                                                       |    ✓    |     ✓     |    ✓     |
| 限定開発チャンネル `#insider-desktop` と `#insider-mobile` へのアクセス             |    ✓    |     ✓     |    ✓     |
| 限定ラウンジチャンネル `#vip-lounge` へのアクセス                                   |         |     ✓     |    ✓     |
| Insiderバッジ                                                                      |    ✓    |           |          |
| Supporterバッジ                                                                    |         |     ✓     |          |
| VIPバッジ                                                                          |         |           |    ✓     |
| **貢献額（USD）**                                                                  | **$25** |  **$50**  | **$100** |


> [!info] Catalystティアのアップグレード
> すでにCatalystライセンスをお持ちの場合、現在のティアと新しいティアの差額を支払うことで上位のCatalystティアにアップグレードできます。

## Catalyst特典へのアクセス

Catalystライセンスを購入すると、さまざまな特典にアクセスできます。ここでは特典の有効化方法を説明します。

### フォーラムバッジを取得する

**注意：** Obsidianアカウントとフォーラムアカウントをリンクするには、両方のアカウントで同じメールアドレスを使用する必要があります。

1. まだの場合は、[フォーラム](https://forum.obsidian.md/)に参加してください。
2. [Obsidianアカウントダッシュボード](https://obsidian.md/account/catalyst)にアクセスしてください。
3. **Catalyst**の下にある**Get forum badge**を選択してください。

### Discordバッジを取得する

1. まだの場合は、[Discordサーバー](https://discord.gg/veuWUTm)に参加してください。
2. [Obsidianアカウントダッシュボード](https://obsidian.md/account/catalyst)にアクセスしてください。
3. **Catalyst**の下にある**Get Discord badge**を選択してください。

### インサイダーチャンネルに参加する

InsiderはObsidianチームや他のInsiderと限定のDiscordチャンネルでチャットできます。[[#Discordバッジを取得する]]の後、自動的にインサイダーチャンネルに参加します。

### 早期アクセスバージョンを有効にする

[[早期アクセスバージョン]]の受け取り方法をご覧ください。
