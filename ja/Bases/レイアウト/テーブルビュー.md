テーブルは、[[Basesの紹介|Bases]]で使用できる[[ビュー]]の一種です。

ビューメニューから ![[lucide-table.svg#icon]] **テーブル**を選択すると、各ファイルが行として表示され、そのファイルの[[プロパティ]]が列として表示されるテーブル形式でファイルを閲覧できます。

![書籍のリストをテーブルビューで表示したベースの例](bases-noshadow.png#interface)

## 設定

テーブルビューの設定は[[ビュー#ビュー設定|ビュー設定]]で設定できます。

### 行の高さ

行の高さを変更すると、より多くの情報を表示できます。**低**、**中**、**高**、**特に高**から選択してください。

## 要約

テーブルの列に要約を追加して、ビューに現在表示されている行の合計、平均、カウントなどの値をすばやく計算できます。

要約はベースではなくビューに紐づけられています。各ビューで同じ列に対して異なる要約を表示できます。

### 要約の追加

1. テーブルビューの列ヘッダーを右クリックします。
2. ![[lucide-calculator.svg#icon]] **要約…**を選択します。
3. 組み込みの要約関数のいずれかを選択するか、![[lucide-square-function.svg#icon]] **サマリーを追加**を選択して独自の要約を定義します。

要約は列の下部に表示されます。結果が[[ビュー#結果の並べ替えとグループ化|グループ化]]されている場合、各グループの要約はグループの上部に表示されます。

要約バーが追加されると、要約セルをクリックして他の列にも要約を追加できます。すべての要約が削除されると、要約バーは非表示になります。

### 組み込みの要約

以下の要約がデフォルトで利用できます。プロパティタイプによって利用可能なオプションが異なる場合があります。

#### すべてのプロパティタイプ

- **値がありません**: 値のない行の数。
- **Filled**: 値のある行の数。
- **ユニーク**: 異なる値の数。

#### 数値

- **Average**: すべての数値の平均。
- **Max**: 最大値。
- **Median**: 中央値。
- **Min**: 最小値。
- **Range**: 最大値と最小値の差。
- **Stddev**: 標準偏差。
- **Sum**: すべての値の合計。

#### 日付

- **Earliest**: 最も小さい/古い日付。
- **Latest**: 最も大きい/新しい日付。
- **Range**: 最も古い日付と最も新しい日付の差。

#### チェックボックス

- **Checked**: チェックボックスがオンの行の数。
- **Unchecked**: チェックボックスがオフの行の数。

### カスタム要約

数式を使って独自の要約を定義できます:

1. ![[lucide-calculator.svg#icon]] **要約…**メニューで、![[lucide-square-function.svg#icon]] **サマリーを追加**を選択します。
2. 要約に名前を付けます。
3. 数式を入力します。数式はその列の値のリストに対して実行されます（例えば、`values.reduce(...)` のような[[関数]]を使用します）。
4. 要約を保存します。

カスタム要約は、組み込みのオプションではカバーされない計算が必要な場合に便利です。

## ショートカット

以下のマウスおよび[[編集ショートカット|キーボードショートカット]]を使って、テーブルビュー内をすばやく移動できます。

- Shift+クリックでセル選択を作成します。
- セル選択を右クリックすると、それらのファイルに対する追加のアクションにアクセスできます。

| --------------------------------------------------------------------------------------------------------------- | ------------------- | ------------------ |
| 選択したセルをコピー                                                                                         | `Ctrl+C`            | `Cmd+C`            |
| 選択したセルを貼り付け                                                                                        | `Ctrl+V`            | `Cmd+V`            |
| プロパティの変更を元に戻す                                                                                      | `Ctrl+Z`            | `Cmd+Z`            |
| プロパティの変更をやり直す                                                                                      | `Ctrl+Shift+Z`      | `Cmd+Shift+Z`      |
| 現在のグループのすべてのセルを選択                                                                           | `Ctrl+A`            | `Cmd+A`            |
| 指定した方向のすべてのセルを選択                                                                           | `Ctrl+Shift+Arrow`  | `Ctrl+Shift+Arrow` |
| 列を選択                                                                                               | `Ctrl+Space`        |                    |
| 行を選択                                                                                                  | `Shift+Space`       |                    |
| 現在のセルにフォーカス — チェックボックスの場合はチェックの切り替え、数式の場合は数式エディタを開く | `Enter`             |                    |
| 最初の列に移動                                                                                          | `Home`              |                    |
| 最後の列に移動                                                                                           | `End`               |                    |
| ページの高さ分だけ上下に移動                                                                             | `PageUp`,`PageDown` |                    |
| 現在のセル選択を解除                                                                                | `Esc`               |                    |
| 現在のセルをクリア                                                                                         | `Backspace`         |                    |
| 次のセルに移動                                                                                             | `Tab`               |                    |
| 前のセルに移動                                                                                         | `Shift-Tab`         |                    |
