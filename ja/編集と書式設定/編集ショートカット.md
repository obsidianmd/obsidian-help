このページでは、Obsidianでのテキストのナビゲーションと編集に使用するデフォルトのキーボードショートカットを一覧にしています。これらのショートカットはオペレーティングシステムまたはObsidianが構築されているフレームワークによって提供されており、Obsidian内でカスタマイズすることはできません。

Obsidianのコマンドに対するカスタマイズ可能なキーボードショートカットについては、[[ホットキー]]を参照してください。

## WindowsおよびLinuxのショートカット

### 一般的な操作

| 操作 | ショートカット |
|-|-|
| コピー | `Ctrl+C` |
| 切り取り | `Ctrl+X` |
| 貼り付け | `Ctrl+V` |
| 書式なしで貼り付け | `Ctrl+Shift+V` |
| 元に戻す | `Ctrl+Z` |
| やり直す | `Ctrl+Shift+Z` または `Ctrl+Y` |
| 段落をコピー | `Ctrl+C`（テキスト未選択時） |
| 段落を切り取り | `Ctrl+X`（テキスト未選択時） |

### テキスト編集

| 操作 | ショートカット |
|-|-|
| 新しい行を挿入 | `Enter` |
| 前の文字を削除 | `Backspace` |
| 次の文字を削除 | `Delete` |
| 前の単語を削除 | `Ctrl+Backspace` |
| 次の単語を削除 | `Ctrl+Delete` |
| 現在の行を削除 | `Ctrl+Shift+K`（テキスト未選択時） |

### テキストナビゲーション

| 操作 | ショートカット |
|-|-|
| カーソルを1文字移動 | `Left/→` |
| カーソルを前の単語の先頭に移動 | `Ctrl+←` |
| カーソルを次の単語の末尾に移動 | `Ctrl+→` |
| カーソルを現在の行の先頭に移動 | `Home` |
| カーソルを現在の行の末尾に移動 | `End` |
| カーソルを前の行に移動 | `↑` |
| カーソルを次の行に移動 | `↓` |
| カーソルをノートの先頭に移動 | `Ctrl+Home` |
| カーソルをノートの末尾に移動 | `Ctrl+End` |
| カーソルを1ページ上に移動 | `Page up` |
| カーソルを1ページ下に移動 | `Page down` |

### テキスト選択

| 操作 | ショートカット |
|-|-|
| 選択を解除 | `Escape` |
| すべてを選択 | `Ctrl+A` |
| 選択を1文字拡張 | `Shift+Left/→` |
| 選択を前の単語の先頭まで拡張 | `Ctrl+Shift+←` |
| 選択を次の単語の末尾まで拡張 | `Ctrl+Shift+→` |
| 選択を現在の行の先頭まで拡張 | `Shift+Home` |
| 選択を現在の行の末尾まで拡張 | `Shift+End` |
| 選択をノートの先頭まで拡張 | `Ctrl+Shift+Home` |
| 選択をノートの末尾まで拡張 | `Ctrl+Shift+End` |
| 選択を1ページ上に拡張 | `Shift+Page up` |
| 選択を1ページ下に拡張 | `Shift+Page down` |

## macOSのショートカット

### 一般的な操作

| 操作 | ショートカット |
|-|-|
| コピー | `Cmd+C` |
| 切り取り | `Cmd+X` |
| 貼り付け | `Cmd+V` |
| 書式なしで貼り付け | `Cmd+Shift+V` |
| 元に戻す | `Cmd+Z` |
| やり直す | `Cmd+Shift+Z` |
| 段落をコピー | `Cmd+C`（テキスト未選択時） |
| 段落を切り取り | `Cmd+X`（テキスト未選択時） |

### テキスト書式設定

| 操作 | ショートカット |
| ---- | ---- |
| ボールドテキスト | `Cmd+B` |
| イタリックテキスト | `Cmd+I` |

### テキスト編集

| 操作 | ショートカット |
|-|-|
| 新しい行を挿入 | `Enter` |
| 前の文字を削除 | `Backspace` |
| 次の文字を削除 | `Delete` |
| 前の単語を削除 | `Option+Backspace` |
| 次の単語を削除 | `Option+Delete` |
| 現在の行の先頭まで削除 | `Cmd+Backspace` |
| 現在の行の末尾まで削除 | `Cmd+Delete` |
| 現在の行を削除 | `Cmd+Shift+K`（テキスト未選択時） |

### テキストナビゲーション

| 操作 | ショートカット |
|-|-|
| カーソルを1文字移動 | `Left/→` |
| カーソルを前の単語の先頭に移動 | `Option+←` |
| カーソルを次の単語の末尾に移動 | `Option+→` |
| カーソルを現在の行の先頭に移動 | `Cmd+←` |
| カーソルを現在の行の末尾に移動 | `Cmd+→` |
| カーソルを前の行に移動 | `↑` |
| カーソルを次の行に移動 | `↓` |
| カーソルをノートの先頭に移動 | `Cmd+↑` |
| カーソルをノートの末尾に移動 | `Cmd+↓` |
| カーソルを1ページ上に移動 | `Fn+↑` |
| カーソルを1ページ下に移動 | `Fn+↓` |

### テキスト選択

| 操作                                                 | ショートカット                   |
| ------------------------------------------------------ | -------------------------- |
| 選択を解除                                     | `Escape`                   |
| すべてを選択                                             | `Cmd+A`                    |
| 選択を1文字拡張                         | `Shift+Left/→`   |
| 選択を前の単語の先頭まで拡張 | `Option+Shift+←`  |
| 選択を次の単語の末尾まで拡張           | `Option+Shift+→` |
| 選択を現在の行の先頭まで拡張  | `Cmd+Shift+←`              |
| 選択を現在の行の末尾まで拡張        | `Cmd+Shift+→`    |
| 選択をノートの先頭まで拡張          | `Cmd+Shift+↑`       |
| 選択をノートの末尾まで拡張                | `Cmd+Shift+↓`     |
| 選択を1ページ上に拡張                           | `Ctrl+Shift+↑`      |
| 選択を1ページ下に拡張                         | `Ctrl+Shift+↓`    |
