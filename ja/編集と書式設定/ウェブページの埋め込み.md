[iframe](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/iframe) HTML要素を使用して、ノートにウェブページを埋め込む方法を学びましょう。

ウェブページを埋め込むには、ノートに以下を追加し、プレースホルダーテキストを埋め込みたいウェブページのURLに置き換えてください：

```html
<iframe src="INSERT YOUR URL HERE"></iframe>
```

> [!note] 注意
> 一部のウェブサイトでは埋め込みが許可されていません。その代わりに、埋め込み用のURLが提供されている場合があります。ウェブサイトが埋め込みに対応していない場合は、そのウェブサイトの名前に「embed iframe」を付けて検索してみてください。例えば、「youtube embed iframe」のように検索します。

> [!tip] ヒント
> [[Canvas]]を使用している場合は、カードにウェブページを埋め込むことができます。詳細については、[[Canvas#ウェブページからカードを追加する]]を参照してください。

## YouTube動画の埋め込み

YouTube動画を埋め込むには、[[基本的な書式構文#外部画像|外部画像]]と同じ Markdown 構文を使用します：

```md
![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)
```

![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)

## ツイートの埋め込み

ツイートを埋め込むには、[[基本的な書式構文#外部画像|外部画像]]と同じ Markdown 構文を使用します：

```md
![](https://twitter.com/obsdmd/status/1580548874246443010)
```

![](https://twitter.com/obsdmd/status/1580548874246443010)
