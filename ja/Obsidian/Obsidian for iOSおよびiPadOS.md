iOSおよびiPadOS向けのObsidianモバイルアプリは、iPhoneやiPadに強力なノートテイキング機能をもたらします。[Apple App Store](https://apps.apple.com/us/app/obsidian-connected-notes/id1557175442)からダウンロードできます。

このページでは、ウィジェット、Siri連携、ショートカットなど、iOS固有の機能について説明します。

## 同期

デバイス間でのノート同期については、[[デバイス間でノートを同期する]]をご覧ください。

## ウィジェット

Obsidian for iOSでは、保管庫に対してすばやくアクションを実行できるいくつかのウィジェットを提供しています。

> [!note] 注意
> ウィジェットはiOSおよびiPadOS 18以降で利用可能です。
> アプリのロック解除に「Face IDを要求」を使用している場合、ウィジェットは利用できません。


### ロック画面およびコントロールセンターのウィジェット

ロック画面およびコントロールセンターのウィジェットでは、以下のことができます:
- 新規ノートを作成する
- 特定のノートを開く
- デイリーノートを開く
- 検索を開く
- Obsidianを開く

### ホーム画面のウィジェット

ホーム画面のウィジェットでは、以下のことができます:
- ノートを作成する
- ノートを表示する
- デイリーノートを開く

### ウィジェットのカスタマイズ

使用する保管庫の選択や、開く特定のノートの指定など、ワークフローに合わせてウィジェットをカスタマイズできます。

- **ホーム画面のウィジェット:** ウィジェットをタップアンドホールドし、**ウィジェットを編集**を選択します。
- **ロック画面のウィジェット:** ロック画面をタッチアンドホールドし、**カスタマイズ**をタップしてロック画面を選択し、カスタマイズしたいウィジェットをタップします。
- **コントロールセンターのウィジェット:** コントロールセンターを開き、左上の**+**ボタンをタップして編集を開始し、カスタマイズしたいウィジェットをタップします。

## ショートカット

ObsidianはAppleのショートカットアプリと連携しており、強力な自動化を作成できます。利用可能なショートカットは以下の通りです:

- **ノートを開く** — 保管庫内の特定のノートを開きます
- **新規ノートを作成する** — 保管庫に新しいノートを作成します
- **デイリーノートを開く** — 今日のデイリーノートに直接ジャンプします
- **デイリーノートにキャプチャ** — Obsidianアプリを開かずにデイリーノートにテキストを追加または先頭に挿入します
- **ブックマークにキャプチャ** — Obsidianアプリを開かずにブックマーク済みのノートにテキストを追加または先頭に挿入します

キャプチャショートカットは、バックグラウンドでノートにコンテンツを追加できるため、すばやいノートテイキングに特に便利です。

## 共有シート

Obsidianの共有シートを使うと、ウェブページからコンテンツをキャプチャできます。YouTubeやその他のソーシャルネットワークなどのアプリでも動作します。

> [!note]
> - ネイティブの共有シートはiOSおよびiPadOS 18以降で利用可能です。
> - このセクションで説明する共有シートの機能にはObsidian 1.13.0以降が必要です。

共有シートを使って、他のアプリからObsidianにすばやくコンテンツを送信できます:
1. 他のアプリで**共有**ボタンをタップします。
2. **Obsidian**を選択します。
3. ロケーションを選択します。
4. キャプチャしたコンテンツを確認または編集します。
5. **保存**をタップします。

![[ios-share-sheet-extension.png|400]]

### ロケーション

ロケーションを使うと、保存する前に共有コンテンツの送信先を決めることができます。

ロケーションでキャプチャできる送信先:
- **新規ノート** — 保管庫またはフォルダに新しいノートを作成します。
- **デイリーノート** — 今日のデイリーノートにコンテンツを追加または先頭に挿入します。
- **ブックマーク済みノート** — ブックマーク済みのノートにコンテンツを追加または先頭に挿入します。
- **ノート** — 保管庫内の既存のノートを選択します。
- **新規ブックマーク** — 共有URLをObsidianのブックマークに保存します。

![[ios-share-sheet-locations.png|400]]

### ロケーションのカスタマイズ

記事を受信箱に保存したり、引用をデイリーノートに追加したり、リンクをブックマークに追加するなど、よく使うワークフロー用にロケーションを作成できます。

ロケーションをカスタマイズするには:

1. iOS共有シートからObsidianを開きます。
2. ツールバーの現在のロケーションをタップします。
3. **+**ボタンをタップして新しいロケーションを作成するか、既存のロケーションを選択して編集します。
4. 保管庫、動作、およびオプションの設定を選択します。

`動作`の種類に応じて、以下のようなオプションを設定できます:
- フォルダ
- テンプレート
- ブックマークグループ
- 追加または先頭挿入の位置
- 共有リンクで**フルテキスト**をキャプチャするか**URL**のみをキャプチャするか

![[ios-share-sheet-add-location.png|400]]

### 共有時のテンプレートの使用

共有シートからコンテンツを共有する際にテンプレートを使用できます。テンプレートを使うと、ページタイトル、著者、ソースウェブサイト、公開日などの詳細でキャプチャしたウェブコンテンツをフォーマットできます。

テンプレート付きのロケーションを設定するには:

1. iOS共有シートからObsidianを開きます。
2. ツールバーの現在のロケーションをタップします。
3. **+**ボタンをタップして新しいロケーションを作成します。
4. ロケーションの名前を入力します。
5. 保管庫を選択します。
6. **動作**を**新規ノート**に設定します。
7. **オプション**セクションで**テンプレート**をタップします。
8. テンプレートとして使用する保管庫内のノートを選択します。
9. **保存**をタップしてロケーションを保存します。

![[ios-share-sheet-set-template.png|400]]

このロケーションを使ってリンクを共有すると、Obsidianはまずテンプレートを適用し、その後に共有コンテンツを追加します。

サポートされているテンプレートプレースホルダー:

| プレースホルダー | 説明 |
| --- | --- |
| `{{author}}` | 記事の著者 |
| `{{description}}` | 記事の説明または概要 |
| `{{domain}}` | ウェブサイトのドメイン名 |
| `{{favicon}}` | ウェブサイトのファビコンのURL |
| `{{image}}` | 記事のメイン画像のURL |
| `{{published}}` | デフォルトの日付形式を使用した記事の公開日 |
| `{{published: YYYY-MM-DD}}` | カスタム日付形式を使用した公開日 |
| `{{site}}` | ウェブサイトの名前 |
| `{{title}}` | 記事のタイトル |
| `{{url}}` | 記事のURL |
| `{{wordCount}}` | 抽出されたコンテンツの総語数 |

標準のテンプレート日付・時刻プレースホルダーも使用できます:

| プレースホルダー | 説明 |
| --- | --- |
| `{{date}}` | 現在の日付 |
| `{{date: YYYY-MM-DD}}` | カスタム形式を使用した現在の日付 |
| `{{time}}` | 現在の時刻 |
| `{{time: HH:mm}}` | カスタム形式を使用した現在の時刻 |

## Siri連携

Siriの音声コマンドを使ってObsidianと対話できます:

- "Capture using Obsidian"
- "Capture to Obsidian"
- "Open my daily note in Obsidian"
- "Search in Obsidian"

## Spotlight連携

iOSのSpotlightで「Obsidian」を検索すると、クイックアクションが表示されます:
- 新規ノート
- 検索
- デイリーノート
