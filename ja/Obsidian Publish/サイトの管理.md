このページでは、[[Obsidian Publishの概要|Obsidian Publish]]サイトの管理方法について説明します。

サイトとは、Obsidian Publishによってホストされ、Obsidian Publishアドレスまたは[[カスタムドメイン|カスタムドメイン]]を通じてオンラインで利用可能なノートのコレクションです。

## 新しいサイトの作成

> [!note] 同時に保持できるサイトの数は、Obsidian Publishのサブスクリプションによって決まります。新しいサイトを作成する前に、サブスクリプションで許可されていることを確認してください。

1. アプリケーションウィンドウの左側にあるリボンメニューで、**変更の公開**（ ![[lucide-send.svg#icon]] ）をクリックします。
2. 既存のサイトがある場合は、**サイトの切り替え**（ ![[lucide-repeat.svg#icon]] ）をクリックします。
3. **サイトID**に、サイトのパスを入力します。例えば、IDが`my-amazing-site`のサイトは、publish.obsidian.md/my-amazing-siteからアクセスできます。
4. **作成**をクリックします。

## 既存のサイトの削除

> [!note] サイトを削除しても、ノートは保管庫に残ります。

1. アプリケーションウィンドウの左側にあるリボンメニューで、**変更の公開**（ ![[lucide-send.svg#icon]] ）をクリックします。
2. **サイトの切り替え**（ ![[lucide-repeat.svg#icon]] ）をクリックします。
3. 削除したいサイトの右側にある**サイトを削除**（ ![[lucide-x.svg#icon]] ）をクリックします。
4. **削除**をクリックして確認します。

## サイト間の切り替え

1. アプリケーションウィンドウの左側にあるリボンメニューで、**変更の公開**（ ![[lucide-send.svg#icon]] ）をクリックします。
2. **サイトの切り替え**（ ![[lucide-repeat.svg#icon]] ）をクリックします。
3. 切り替えたいサイトの右側にある**選択**をクリックします。

## サイトIDの変更

1. アプリケーションウィンドウの左側にあるリボンメニューで、**変更の公開**（ ![[lucide-send.svg#icon]] ）をクリックします。
2. **サイトの切り替え**（ ![[lucide-repeat.svg#icon]] ）をクリックします。
3. 編集したいサイトの右側にある**サイトIDを編集**（ ![[lucide-edit-3.svg#icon]] ）をクリックします。
4. **サイトID**に、サイトの新しいIDを入力します。
5. **変更**をクリックします。

## サイトオプションの表示

1. アプリケーションウィンドウの左側にあるリボンメニューで、**変更の公開**（ ![[lucide-send.svg#icon]] ）をクリックします。
2. **変更の公開**ダイアログで、**サイトオプションの変更**（ ![[lucide-cog.svg#icon]] ）をクリックします。

## サイトオプション

### 一般

| オプション                          | タイプ   | 説明                                                                                                                |
| ------------------------------- | ------ | -------------------------------------------------------------------------------------------------------------------------- |
| サイト名称                       | 入力  | Obsidian Publishサイトの公開名およびタイトルです。                                                                   |
| ホームページファイル                   | 入力  | ランディングページとして使用するMarkdownファイルの場所です。                                                    |
| ロゴ                            | 入力  | サイトバナーとして使用する画像です。画像は[[コンテンツの公開#ノートの公開\|公開されている]]必要があります。 |
| サイトコラボレーション              | ボタン | 公開ノートの編集アクセスを許可する他のユーザーを宣言します。ユーザーはObsidianアカウントを持っている必要があります。             |
| カスタムドメイン                   | ボタン | [[カスタムドメイン]]                                                                                                         |
| 検索エンジンによるインデックス化を無効 | トグル | `robots.txt`ファイルを追加して、検索エンジンによるサイトのクロールを防止します。                                   |

### 外観

| オプション            | タイプ     | 説明                                                                               |
|-------------------|----------|-------------------------------------------------------------------------------------------|
| テーマ             | ドロップダウン | サイトでのテーマの表示方法を選択します。**ライト**、**ダーク**、または**システムテーマを適用**。 |
| ライトテーマとダークテーマの切り替え | トグル   | サイト上のトグルボタンでユーザーが**ライト**/**ダーク**モードを切り替えられるようにします。          |

## リーディングエクスペリエンス

| オプション               | タイプ   | 説明                                                                                |
|----------------------|--------|--------------------------------------------------------------------------------------------|
| プレビューをホバー表示   | トグル | アクティブな内部リンクにホバーした際のページプレビューの有効・無効を切り替えます。 |
| ページタイトルを非表示にする      | トグル | 公開ノートのインラインタイトル表示の有効・無効を切り替えます。                    |
| 読みやすい行の長さ | トグル | サイト内の読みやすい行の長さの有効・無効を切り替えます。                                   |
| 厳密な改行   | トグル | サイト上での単一改行の表示の有効・無効を切り替えます。                    |
| スライディングウィンドウを使用          | トグル | サイトでの[[タブ#タブグループのスタック\|スタックタブ]]の有効・無効を切り替えます。    |

### コンポーネント

| オプション                 | タイプ   | 説明                                                                                                                          |
|------------------------|--------|--------------------------------------------------------------------------------------------------------------------------------------|
| ナビゲーションを表示        | トグル | Publishサイトでの[[ファイルエクスプローラー]]ビューの有効・無効を切り替えます。                                                                   |
| ナビゲーションをカスタマイズ   | ボタン | **ナビゲーションを表示**が有効の場合、ファイルの表示順序を[[サイトのカスタマイズ#ナビゲーションのカスタマイズ\|カスタマイズ]]します。 |
| 検索バーを表示        | トグル | サイト上の検索バーの有効・無効を切り替えます。                                                                                         |
| グラフビューを表示        | トグル | サイトの右サイドバー内のグラフビューの有効・無効を切り替えます。                                                              |
| 目次を表示 | トグル | サイトでの[[アウトライン|目次]]ビューの有効・無効を切り替えます。                                                              |
| バックリンクを表示         | トグル | サイトでの[[バックリンク]]の有効・無効を切り替えます。                                                                                        |

### 他のサイト設定

| オプション                         | タイプ   | 説明                                            |
| ------------------------------ | ------ | ------------------------------------------------------ |
| パスワード                      | ボタン | サイト全体へのアクセスを制限するための[[Obsidian Publish/セキュリティとプライバシー#サイトパスワードの追加\|パスワードを設定]]します。 |
| Googleアナリティクスのトラッキングコード | 入力  | **カスタムドメインURLのみ**。Googleアナリティクスのサイトトラッキングコードをここに入力します。                                                       |
