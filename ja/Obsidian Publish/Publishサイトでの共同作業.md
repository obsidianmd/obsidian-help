他のObsidianユーザーと[[Obsidian Publishの概要|Obsidian Publish]]サイトで共同作業する方法を学びましょう。友人や同僚をコラボレーターとして追加することで、サイトへの変更を公開できるようになります。

Obsidian Publishの有効なサブスクリプションが必要なのはサイトオーナーのみです。コラボレーターには[Obsidianアカウント](https://obsidian.md/account)のみが必要です。

> [!warning] 共有サイトに変更を公開する前に、必ず[[#コラボレーター間で変更を同期する|コラボレーター間で変更を同期]]してください。そうしないと、他のコラボレーターの変更を上書きしてしまうリスクがあります。

## サイトにコラボレーターを追加する

1. [[リボン]]で**パブリッシュによる変更の公開**（![[lucide-send.svg#icon]]）を選択するか、[[コマンドパレット]]を開いて **Publish: Publish changes...** と入力します。
2. **パブリッシュによる変更の公開**ダイアログで、**サイトオプションの変更** ![[lucide-cog.svg#icon]] をクリックします。
3. **サイトコラボレーション**の横にある**管理**を選択します。
4. **ユーザーを招待する**にコラボレーターのメールアドレスを入力します。
5. **追加**を選択します。

## サイトからコラボレーターを削除する

1. [[リボン]]で**パブリッシュによる変更の公開**（![[lucide-send.svg#icon]]）を選択するか、[[コマンドパレット]]を開いて **Publish: Publish changes...** と入力します。
2. **パブリッシュによる変更の公開**ダイアログで、**サイトオプションの変更**（ ![[lucide-cog.svg#icon]] ）をクリックします。
3. **サイトコラボレーション**の横にある**管理**を選択します。
4. 削除したいコラボレーターの横にある**ユーザーを取り除く**（ ![[lucide-x.svg#icon]] ）を選択します。

## コラボレーター間で変更を同期する

Obsidian Publishは、公開された変更をローカル保管庫間で自動的に同期しません。代わりに、コラボレーターは他のコラボレーターからの変更を手動で同期する必要があります。

ライブサイトの変更でローカルノートを更新するには：

1. [[リボン]]で**パブリッシュによる変更の公開**（![[lucide-send.svg#icon]]）を選択するか、[[コマンドパレット]]を開いて **Publish: Publish changes...** と入力します。
2. 同期したい変更を右クリックまたは長押しし、**ライブバージョンを使用**を選択します。**これにより、ローカル保管庫のノートが上書きされます。**

> [!tip] [[Obsidian Syncの紹介|Obsidian Sync]]や[git](https://git-scm.com/)など、他のツールを使用して保管庫間の変更を同期することを推奨します。

## 権限

以下の表は、オーナーとコラボレーターが利用可能なサイト権限の一覧です：

| アクション                           | コラボレーター | オーナー |
|------------------------------------|:------------:|:-----:|
| 新しいページの公開                    | ✓            | ✓     |
| 公開済みページへの変更の公開            | ✓            | ✓     |
| ページの非公開化                      | ✓            | ✓     |
| サイトオプションの設定                 |              | ✓     |
| 権限の管理                           |              | ✓     |
