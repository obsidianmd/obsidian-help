[[Obsidian Publishの概要|Obsidian Publish]]のサイトは、検索エンジンや[[ソーシャルメディアリンクプレビュー|ソーシャルシェアリングカード]]向けに自動的に最適化されます。説明文、パーマリンク、画像を使用してページメタデータをカスタマイズすることもできます。

[[サイトの管理#サイトオプション|サイトオプション]]で検索エンジンによるインデックス化を無効にすることもできます。

### Google Search Console

GoogleでSEOを追跡するには、[Google Search Console](https://search.google.com/search-console)にサイトを登録してください。また、おすすめの手順に従って[[アナリティクス|Google Analyticsを設定]]することもできます。

### サイトマップ

Publishサイトのサイトマップは`/sitemap.xml`で利用できます。例：
https://help.obsidian.md/sitemap.xml

サイトマップはRSSフィードとして`/rss.xml`でも利用できます：
https://help.obsidian.md/rss.xml

### パーマリンク

サイトに恒久的なURLを定義するために[[パーマリンク]]の使用を検討してください。

### メタデータ

ページメタデータは[[プロパティ#Obsidian Publish用のプロパティ|プロパティ]]を使用してカスタマイズでき、[[ソーシャルメディアリンクプレビュー|ソーシャルシェアリングカード]]に役立ちます。

| プロパティ      | 説明                                                                                                       |
| ------------- | ----------------------------------------------------------------------------------------------------------------- |
| `publish`     | [[コンテンツの公開#公開するデータを自動的に選択する\|公開するデータを自動的に選択する]]を参照してください。 |
| `permalink`   | [[パーマリンク\|パーマリンク]]を参照してください。                                                                                   |
| `description` | [[ソーシャルメディアリンクプレビュー#説明文\|説明文]]を参照してください。                                                      |
| `image`       | [[ソーシャルメディアリンクプレビュー#画像\|画像]]を参照してください。                                                                  |
| `cover`       | [[ソーシャルメディアリンクプレビュー#画像\|画像]]を参照してください。                                                                  |
