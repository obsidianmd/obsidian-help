[[Obsidian Publishの概要|Obsidian Publish]]サイトは、検索エンジンや[[ソーシャルメディアリンクプレビュー|ソーシャル共有カード]]に対して自動的に最適化されます。また、説明文、パーマリンク、画像を使ってページメタデータをカスタマイズすることもできます。

[[サイトの管理#サイトオプション|サイトオプション]]で検索エンジンによるインデックス化を無効にすることもできます。

### Google Search Console

GoogleでSEOを追跡するには、[Google Search Console](https://search.google.com/search-console)にサイトを登録してください。また、おすすめの手順に従って[[アナリティクス|Google Analyticsを設定]]することもできます。

### サイトマップ

Publishサイトのサイトマップは`/sitemap.xml`で利用できます。例：
https://help.obsidian.md/sitemap.xml

サイトマップはRSSフィードとして`/rss.xml`でも利用できます：
https://help.obsidian.md/rss.xml

### パーマリンク

[[パーマリンク]]を使用して、サイトの永続的なURLを定義することを検討してください。

### メタデータ

ページメタデータは[[プロパティ#Obsidian Publishのプロパティ|プロパティ]]を使ってカスタマイズでき、[[ソーシャルメディアリンクプレビュー|ソーシャル共有カード]]に役立ちます。

| プロパティ      | 説明                                                                                                       |
| ------------- | ----------------------------------------------------------------------------------------------------------------- |
| `publish`     | [[コンテンツの公開#公開するノートの自動選択\|公開するノートの自動選択]]を参照。 |
| `permalink`   | [[パーマリンク\|パーマリンク]]を参照。                                                                                   |
| `description` | [[ソーシャルメディアリンクプレビュー#説明文\|説明文]]を参照。                                                      |
| `image`       | [[ソーシャルメディアリンクプレビュー#画像\|画像]]を参照。                                                                  |
| `cover`       | [[ソーシャルメディアリンクプレビュー#画像\|画像]]を参照。                                                                  |
