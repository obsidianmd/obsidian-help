Trang này liệt kê các phím tắt bàn phím mặc định để điều hướng và chỉnh sửa văn bản trong Obsidian. Các phím tắt này được cung cấp bởi hệ điều hành hoặc framework mà Obsidian được xây dựng trên đó, và không thể tùy chỉnh trong Obsidian.

Để xem các phím tắt có thể tùy chỉnh cho các lệnh Obsidian, hãy xem [[Phím tắt]].

## Phím tắt Windows và Linux

### Thao tác phổ biến

| Thao tác | Phím tắt |
|-|-|
| Sao chép | `Ctrl+C` |
| Cắt | `Ctrl+X` |
| Dán | `Ctrl+V` |
| Dán không giữ định dạng | `Ctrl+Shift+V` |
| Hoàn tác | `Ctrl+Z` |
| Làm lại | `Ctrl+Shift+Z` hoặc `Ctrl+Y` |
| Sao chép đoạn văn | `Ctrl+C` (khi không có văn bản được chọn) |
| Cắt đoạn văn | `Ctrl+X` (khi không có văn bản được chọn) |

### Chỉnh sửa văn bản

| Thao tác | Phím tắt |
|-|-|
| Chèn dòng mới | `Enter` |
| Xóa ký tự trước đó | `Backspace` |
| Xóa ký tự tiếp theo | `Delete` |
| Xóa từ trước đó | `Ctrl+Backspace` |
| Xóa từ tiếp theo | `Ctrl+Delete` |
| Xóa dòng hiện tại | `Ctrl+Shift+K` (khi không có văn bản được chọn) |

### Điều hướng văn bản

| Thao tác | Phím tắt |
|-|-|
| Di chuyển con trỏ một ký tự | `Left/→` |
| Di chuyển con trỏ đến đầu từ trước đó | `Ctrl+←` |
| Di chuyển con trỏ đến cuối từ tiếp theo | `Ctrl+→` |
| Di chuyển con trỏ đến đầu dòng hiện tại | `Home` |
| Di chuyển con trỏ đến cuối dòng hiện tại | `End` |
| Di chuyển con trỏ lên dòng trước | `↑` |
| Di chuyển con trỏ xuống dòng tiếp theo | `↓` |
| Di chuyển con trỏ đến đầu ghi chú | `Ctrl+Home` |
| Di chuyển con trỏ đến cuối ghi chú | `Ctrl+End` |
| Di chuyển con trỏ lên một trang | `Page up` |
| Di chuyển con trỏ xuống một trang | `Page down` |

### Chọn văn bản

| Thao tác | Phím tắt |
|-|-|
| Thu gọn vùng chọn | `Escape` |
| Chọn tất cả | `Ctrl+A` |
| Mở rộng vùng chọn một ký tự | `Shift+Left/→` |
| Mở rộng vùng chọn đến đầu từ trước đó | `Ctrl+Shift+←` |
| Mở rộng vùng chọn đến cuối từ tiếp theo | `Ctrl+Shift+→` |
| Mở rộng vùng chọn đến đầu dòng hiện tại | `Shift+Home` |
| Mở rộng vùng chọn đến cuối dòng hiện tại | `Shift+End` |
| Mở rộng vùng chọn đến đầu ghi chú | `Ctrl+Shift+Home` |
| Mở rộng vùng chọn đến cuối ghi chú | `Ctrl+Shift+End` |
| Mở rộng vùng chọn lên một trang | `Shift+Page up` |
| Mở rộng vùng chọn xuống một trang | `Shift+Page down` |

## Phím tắt macOS

### Thao tác phổ biến

| Thao tác | Phím tắt |
|-|-|
| Sao chép | `Cmd+C` |
| Cắt | `Cmd+X` |
| Dán | `Cmd+V` |
| Dán không giữ định dạng | `Cmd+Shift+V` |
| Hoàn tác | `Cmd+Z` |
| Làm lại | `Cmd+Shift+Z` |
| Sao chép đoạn văn | `Cmd+C` (khi không có văn bản được chọn) |
| Cắt đoạn văn | `Cmd+X` (khi không có văn bản được chọn) |

### Định dạng văn bản

| Thao tác | Phím tắt |
| ---- | ---- |
| In đậm | `Cmd+B` |
| In nghiêng | `Cmd+I` |

### Chỉnh sửa văn bản

| Thao tác | Phím tắt |
|-|-|
| Chèn dòng mới | `Enter` |
| Xóa ký tự trước đó | `Backspace` |
| Xóa ký tự tiếp theo | `Delete` |
| Xóa từ trước đó | `Option+Backspace` |
| Xóa từ tiếp theo | `Option+Delete` |
| Xóa đến đầu dòng hiện tại | `Cmd+Backspace` |
| Xóa đến cuối dòng hiện tại | `Cmd+Delete` |
| Xóa dòng hiện tại | `Cmd+Shift+K` (khi không có văn bản được chọn) |

### Điều hướng văn bản

| Thao tác | Phím tắt |
|-|-|
| Di chuyển con trỏ một ký tự | `Left/→` |
| Di chuyển con trỏ đến đầu từ trước đó | `Option+←` |
| Di chuyển con trỏ đến cuối từ tiếp theo | `Option+→` |
| Di chuyển con trỏ đến đầu dòng hiện tại | `Cmd+←` |
| Di chuyển con trỏ đến cuối dòng hiện tại | `Cmd+→` |
| Di chuyển con trỏ lên dòng trước | `↑` |
| Di chuyển con trỏ xuống dòng tiếp theo | `↓` |
| Di chuyển con trỏ đến đầu ghi chú | `Cmd+↑` |
| Di chuyển con trỏ đến cuối ghi chú | `Cmd+↓` |
| Di chuyển con trỏ lên một trang | `Fn+↑` |
| Di chuyển con trỏ xuống một trang | `Fn+↓` |

### Chọn văn bản

| Thao tác                                               | Phím tắt                   |

| Mở rộng vùng chọn đến cuối dòng hiện tại              | `Cmd+Shift+→`    |

| Mở rộng vùng chọn đến cuối ghi chú                    | `Cmd+Shift+↓`     |
