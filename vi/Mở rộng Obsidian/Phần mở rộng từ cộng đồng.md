Tìm hiểu cách mở rộng Obsidian với các plugin được xây dựng bởi cộng đồng. Sử dụng plugin để điều chỉnh Obsidian phù hợp với nhu cầu cụ thể của bạn, chẳng hạn như hỗ trợ thêm các định dạng tệp bổ sung hoặc tích hợp với dịch vụ bên thứ ba.

> [!warning] Cảnh báo
> Plugin cộng đồng chạy mã của bên thứ ba thay mặt bạn và có thể gây hại. Để tìm hiểu thêm về những gì đội ngũ Obsidian làm để ngăn chặn các plugin có hại, hãy tham khảo [[Bảo mật plugin]].

## Duyệt plugin cộng đồng

1. Mở **[[Cài đặt]]**.
2. Chọn **Bật phần mở rộng từ cộng đồng**. Để biết thêm thông tin, hãy tham khảo [[Bảo mật plugin#Chế độ hạn chế|Chế độ hạn chế]].
3. Chọn **Duyệt** để liệt kê tất cả các plugin cộng đồng có sẵn.

Sử dụng hộp văn bản để lọc plugin dựa trên tên, tác giả và mô tả.

Bạn cũng có thể duyệt các plugin có sẵn trong trình duyệt của mình tại [obsidian.md/plugins](https://obsidian.md/plugins).

## Cài đặt plugin cộng đồng

Để cài đặt plugin cộng đồng, trước tiên bạn phải tắt [[Bảo mật plugin#Chế độ hạn chế|Chế độ hạn chế]].

1. Mở **[[Cài đặt]]**.
2. Trong menu bên, chọn **Phần mở rộng từ cộng đồng**.
3. Chọn **Duyệt** để khám phá các plugin cộng đồng có sẵn.
4. Chọn plugin bạn muốn cài đặt.
5. Chọn **Cài đặt**.

Để sử dụng plugin đã cài đặt, bạn cần kích hoạt nó.

## Kích hoạt plugin cộng đồng

Bạn có thể chọn **Kích hoạt** ngay sau khi cài đặt, hoặc kích hoạt từ danh sách Plugin cộng đồng trong **[[Cài đặt]] → Phần mở rộng từ cộng đồng → Phần mở rộng đã cài đặt**.

## Cập nhật plugin

Vì lý do bảo mật, plugin cộng đồng không tự động cập nhật. Bạn có thể cập nhật tất cả plugin cùng lúc hoặc cập nhật từng plugin riêng lẻ.

Để cập nhật tất cả plugin:

1. Mở **[[Cài đặt]]**.
2. Trong **Phần mở rộng từ cộng đồng → Phần mở rộng hiện tại**, chọn **Kiểm tra cập nhật**.
3. Nếu có bản cập nhật, chọn **Cập nhật tất cả**.

Để cập nhật từng plugin riêng lẻ:

1. Mở **[[Cài đặt]]**.
2. Trong **Phần mở rộng từ cộng đồng → Phần mở rộng hiện tại**, chọn **Kiểm tra cập nhật**.
3. Trong **Phần mở rộng đã cài đặt**, chọn **Cập nhật** bên cạnh plugin bạn muốn cập nhật.

## Gỡ bỏ cài đặt plugin cộng đồng

1. Mở **[[Cài đặt]]**.
2. Trong menu bên, chọn **Phần mở rộng từ cộng đồng**.
3. Trong **Phần mở rộng đã cài đặt**, chọn biểu tượng thùng rác ![[lucide-trash-2.svg#icon]] bên cạnh plugin bạn muốn gỡ bỏ.
4. Chọn **Gỡ bỏ cài đặt** để xác nhận.

## Quản lý plugin đã cài đặt

Trong **[[Cài đặt]] → Phần mở rộng từ cộng đồng → Phần mở rộng đã cài đặt**, mỗi plugin có một số biểu tượng để quản lý:

- **[[Cài đặt]]** ![[lucide-settings.svg#icon]] — Mở trang cài đặt của plugin nếu nó có các tùy chọn có thể cấu hình.
- **[[Phím tắt]]** ![[lucide-plus-circle.svg#icon]] — Đặt phím tắt cho các lệnh của plugin.
- **Tài trợ** ![[lucide-heart.svg#icon]] — Xem URL tài trợ của tác giả plugin để hỗ trợ công việc của họ.
- **Gỡ bỏ cài đặt** ![[lucide-trash-2.svg#icon]] — Xóa plugin khỏi kho của bạn.
- **Chuyển đổi** — Kích hoạt hoặc vô hiệu hóa plugin mà không cần gỡ bỏ cài đặt.

Bạn cũng có thể:

- Chọn biểu tượng **làm mới** để tải lại tất cả plugin.
- Chọn biểu tượng **thư mục** ![[lucide-folder-open.svg#icon]] để mở thư mục plugin trong [[Thư mục cấu hình|thư mục cấu hình]] của bạn.
- Sử dụng **thanh tìm kiếm** để lọc plugin theo tên.
