Nếu bạn là nhà phát triển, bạn có thể đóng góp cho Obsidian theo nhiều cách khác nhau.

### Plugin và chủ đề

Bạn có thể phát triển [[Phần mở rộng từ cộng đồng]] và [[Chủ đề]] bằng Typescript và CSS. Tìm hiểu thêm bằng cách truy cập [Tài liệu dành cho Nhà phát triển Obsidian](https://docs.obsidian.md).

### Các dự án mã nguồn mở

Obsidian phát triển và duy trì một số dự án mã nguồn mở. Bạn có thể gửi pull request với các bản sửa lỗi và cải tiến đến các kho GitHub sau:

- [JSON Canvas](https://github.com/obsidianmd/jsoncanvas) (MIT), một định dạng tệp mở cho dữ liệu canvas vô hạn.
- [Obsidian Importer](https://github.com/obsidianmd/obsidian-importer) (MIT), một plugin chuyển đổi dữ liệu từ nhiều định dạng khác nhau sang tệp Markdown văn bản thuần.
- [Obsidian Maps](https://github.com/obsidianmd/obsidian-maps) (MIT), một plugin hiển thị bản đồ trong Obsidian Cơ sở.
- [Obsidian Web Clipper](https://github.com/obsidianmd/obsidian-clipper) (MIT), một tiện ích mở rộng trình duyệt để tô sáng và thu thập trang web thành tệp Markdown văn bản thuần.
- [Defuddle](https://github.com/kepano/defuddle) (MIT), trình phân tích cú pháp HTML-sang-Markdown được sử dụng bởi Obsidian Web Clipper.

### Tài liệu

Để giúp cải thiện [tài liệu dành cho nhà phát triển](https://docs.obsidian.md/Home), hãy gửi pull request đến kho [obsidian-developer-docs](https://github.com/obsidianmd/obsidian-developer-docs).

Để giúp cải thiện tài liệu này, hãy gửi pull request đến kho [obsidian-help](https://github.com/obsidianmd/obsidian-help).
