Trang này giải thích cách quản lý các trang web [[Giới thiệu về Obsidian Publish|Obsidian Publish]].

Một trang web là một tập hợp các ghi chú được lưu trữ bởi Obsidian Publish và có sẵn trực tuyến, thông qua địa chỉ Obsidian Publish hoặc [[Tên miền tùy chỉnh|tên miền tùy chỉnh]].

## Tạo trang web mới

> [!note] Số lượng trang web mà bạn có thể có cùng lúc được xác định bởi gói đăng ký Obsidian Publish của bạn. Trước khi tạo trang web mới, hãy đảm bảo rằng gói đăng ký của bạn cho phép điều đó.

1. Trong thanh công cụ, ở bên trái cửa sổ ứng dụng, nhấp vào **Xuất bản thay đổi** ![[lucide-send.svg#icon]].
2. Nếu bạn đã có trang web hiện tại, nhấp vào **Chuyển đổi trang web** ![[lucide-repeat.svg#icon]].
3. Trong **ID trang web**, nhập đường dẫn bạn muốn cho trang web của mình. Ví dụ, một trang web có ID `my-amazing-site` sẽ có sẵn tại publish.obsidian.md/my-amazing-site.
4. Nhấp vào **Tạo mới**.

## Xóa trang web hiện có

> [!note] Các ghi chú vẫn còn trong kho của bạn ngay cả sau khi bạn xóa trang web.

1. Trong thanh công cụ, ở bên trái cửa sổ ứng dụng, nhấp vào **Xuất bản thay đổi** ![[lucide-send.svg#icon]].
2. Nhấp vào **Chuyển đổi trang web** ![[lucide-repeat.svg#icon]].
3. Nhấp vào **Xóa trang web** ![[lucide-x.svg#icon]] ở bên phải trang web bạn muốn xóa.
4. Nhấp vào **Xóa** để xác nhận.

## Chuyển đổi giữa các trang web

1. Trong thanh công cụ, ở bên trái cửa sổ ứng dụng, nhấp vào **Xuất bản thay đổi** ![[lucide-send.svg#icon]].
2. Nhấp vào **Chuyển đổi trang web** ![[lucide-repeat.svg#icon]].
3. Nhấp vào **Chọn** ở bên phải trang web bạn muốn chuyển sang.

## Thay đổi ID trang web

1. Trong thanh công cụ, ở bên trái cửa sổ ứng dụng, nhấp vào **Xuất bản thay đổi** ![[lucide-send.svg#icon]].
2. Nhấp vào **Chuyển đổi trang web** ![[lucide-repeat.svg#icon]].
3. Nhấp vào **Sửa ID trang web** ![[lucide-edit-3.svg#icon]] ở bên phải trang web bạn muốn chỉnh sửa.
4. Trong **ID trang web**, nhập ID mới cho trang web của bạn.
5. Nhấp vào **Thay đổi**.

## Xem cài đặt trang web

1. Trong thanh công cụ, ở bên trái cửa sổ ứng dụng, nhấp vào **Xuất bản thay đổi** ![[lucide-send.svg#icon]].
2. Trong hộp thoại **Xuất bản thay đổi**, nhấp vào **Thay đổi cài đặt trang web** ![[lucide-cog.svg#icon]].

## Cài đặt trang web

### Chung

| Tùy chọn                                    | Loại      | Mô tả                                                                                                                                              |
| -------------------------------------------- | --------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| Tên trang web                                | Nhập liệu | Tên công khai và tiêu đề của trang web Obsidian Publish của bạn.                                                                                   |
| Tập tin trang chủ                            | Nhập liệu | Vị trí của tệp Markdown mà bạn muốn sử dụng làm trang đích.                                                                                      |
| Logo                                         | Nhập liệu | Hình ảnh bạn muốn sử dụng làm biểu ngữ trang web. Hình ảnh phải [[Xuất bản nội dung của bạn#Publish notes\|được xuất bản]].                       |
| Hợp tác trang web                            | Nút        | Khai báo những người dùng khác mà bạn muốn cho phép chỉnh sửa các ghi chú đã xuất bản của bạn. Người dùng phải có tài khoản Obsidian.             |
| Tên miền tùy chỉnh                          | Nút        | [[Tên miền tùy chỉnh]]                                                                                                                             |
| Cấm chỉ số hệ thống tìm kiếm               | Bật/Tắt   | Ngăn các công cụ tìm kiếm thu thập dữ liệu trang web của bạn bằng cách thêm tệp `robots.txt`.                                                    |

### Giao diện

| Tùy chọn                        | Loại        | Mô tả                                                                                              |
|----------------------------------|-------------|-----------------------------------------------------------------------------------------------------|
| Chủ đề                          | Danh sách   | Chọn cách chủ đề hiển thị trên trang web của bạn; **Sáng**, **Tối**, hoặc **Tùy chỉnh theo hệ thống**. |
| Chuyển đổi ánh sáng/ tối        | Bật/Tắt     | Cho phép người dùng chuyển đổi chế độ **Sáng**/**Tối** bằng nút bật/tắt trên trang web của bạn.   |

## Trải nghiệm đọc

| Ẩn tiêu đề trang                  | Bật/Tắt | Bật hoặc tắt hiển thị tiêu đề nội tuyến của ghi chú đã xuất bản.                                       |
| Độ dài dòng dễ đọc                | Bật/Tắt | Bật hoặc tắt độ dài dòng dễ đọc trên trang web của bạn.                                                |
| Ngắt dòng nghiêm ngặt             | Bật/Tắt | Bật hoặc tắt hiển thị ngắt dòng đơn trên trang web của bạn.                                            |
| Sử dụng chế độ cửa sổ trượt       | Bật/Tắt | Bật hoặc tắt [[Thẻ#Stack tab groups\|Thẻ xếp chồng]] trên trang web của bạn.                           |
