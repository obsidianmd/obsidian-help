Az Obsidian lehetővé teszi a jegyzeteid áttelepítését a Craftből. Ez a Craft adataidat tartós Markdown fájlokká alakítja, amelyeket az Obsidiannal és sok más alkalmazással is használhatsz.

## Adatok exportálása a Craftből

Nyisd meg a Craft alkalmazást az asztali számítógépeden:

1. Lépj az „Organize" vagy „All Documents" nézetbe
2. Kattints a hárompontos menüre
3. Válaszd az Export lehetőséget.
4. Válaszd a Markdown formátumot.

A Craftben lévő jegyzeteid számától függően ez a folyamat eltarthat egy ideig.

## Craft adatok importálása az Obsidianba

Innen kövesd a [[Markdown fájlok importálása]] útmutatót.

## Hibaelhárítás

A Craft Docs nem szabványos szintaxist használ bizonyos hivatkozásokhoz. Az ezek áttelepítéséhez szükséges forrásokat az [Importáló GitHub tárolóban](https://github.com/obsidianmd/obsidian-importer/issues/27) találod.
