Az Obsidian Markdown `.md` fájlokat használ a jegyzetek elsődleges formátumaként. Ez megkönnyíti a Markdown fájlok importálását az Obsidianba. Számos alkalmazás képes Markdown fájlokat szerkeszteni vagy ebbe a formátumba exportálni.

Többféleképpen adhat hozzá Markdown fájlokat az Obsidian széfjéhez:

1. Húzza és ejtse a fájlokat vagy mappákat a [[Fájlkezelő]] ablakba
2. Használja a rendszer fájlkezelőjét (pl. Windows Intéző vagy Finder macOS-en), és helyezze át a fájlokat közvetlenül az Obsidian széf mappájába.

### Konvertálás más Markdown változatokból

Az Obsidian az [[Obsidian Flavored Markdown]] formátumot használja. Egyes alkalmazások kissé eltérő Markdown szintaxist használnak. A [[Formátum-konvertáló]] segítségével átalakíthatja fájljait az Obsidian Markdown szintaxisára.
