Ez az oldal az Obsidianban szöveg navigálásához és szerkesztéséhez használt alapértelmezett billentyűkombinációkat sorolja fel. Ezeket a billentyűkombinációkat az operációs rendszer vagy az Obsidian alapjául szolgáló keretrendszer biztosítja, és nem testreszabhatók az Obsidianon belül.

Az Obsidian parancsokhoz tartozó testreszabható billentyűkombinációkért lásd: [[Gyorsbillentyűk]].

## Windows és Linux billentyűkombinációk

### Általános műveletek

| Művelet | Billentyűkombináció |
|-|-|
| Másolás | `Ctrl+C` |
| Kivágás | `Ctrl+X` |
| Beillesztés | `Ctrl+V` |
| Beillesztés formázás nélkül | `Ctrl+Shift+V` |
| Visszavonás | `Ctrl+Z` |
| Újra | `Ctrl+Shift+Z` vagy `Ctrl+Y` |
| Bekezdés másolása | `Ctrl+C` (kijelölt szöveg nélkül) |
| Bekezdés kivágása | `Ctrl+X` (kijelölt szöveg nélkül) |

### Szövegszerkesztés

| Művelet | Billentyűkombináció |
|-|-|
| Új sor beszúrása | `Enter` |
| Előző karakter törlése | `Backspace` |
| Következő karakter törlése | `Delete` |
| Előző szó törlése | `Ctrl+Backspace` |
| Következő szó törlése | `Ctrl+Delete` |
| Aktuális sor törlése | `Ctrl+Shift+K` (kijelölt szöveg nélkül) |

### Szövegnavigáció

| Művelet | Billentyűkombináció |
|-|-|
| Kurzor mozgatása egy karakterrel | `Left/→` |
| Kurzor mozgatása az előző szó elejére | `Ctrl+←` |
| Kurzor mozgatása a következő szó végére | `Ctrl+→` |
| Kurzor mozgatása az aktuális sor elejére | `Home` |
| Kurzor mozgatása az aktuális sor végére | `End` |
| Kurzor mozgatása az előző sorra | `↑` |
| Kurzor mozgatása a következő sorra | `↓` |
| Kurzor mozgatása a jegyzet elejére | `Ctrl+Home` |
| Kurzor mozgatása a jegyzet végére | `Ctrl+End` |
| Kurzor mozgatása egy oldallal feljebb | `Page up` |
| Kurzor mozgatása egy oldallal lejjebb | `Page down` |

### Szövegkijelölés

| Művelet | Billentyűkombináció |
|-|-|
| Kijelölés egyszerűsítése | `Escape` |
| Mindent kijelöl | `Ctrl+A` |
| Kijelölés kiterjesztése egy karakterrel | `Shift+Left/→` |
| Kijelölés kiterjesztése az előző szó elejéig | `Ctrl+Shift+←` |
| Kijelölés kiterjesztése a következő szó végéig | `Ctrl+Shift+→` |
| Kijelölés kiterjesztése az aktuális sor elejéig | `Shift+Home` |
| Kijelölés kiterjesztése az aktuális sor végéig | `Shift+End` |
| Kijelölés kiterjesztése a jegyzet elejéig | `Ctrl+Shift+Home` |
| Kijelölés kiterjesztése a jegyzet végéig | `Ctrl+Shift+End` |
| Kijelölés kiterjesztése egy oldallal feljebb | `Shift+Page up` |
| Kijelölés kiterjesztése egy oldallal lejjebb | `Shift+Page down` |

## macOS billentyűkombinációk

### Általános műveletek

| Művelet | Billentyűkombináció |
|-|-|
| Másolás | `Cmd+C` |
| Kivágás | `Cmd+X` |
| Beillesztés | `Cmd+V` |
| Beillesztés formázás nélkül | `Cmd+Shift+V` |
| Visszavonás | `Cmd+Z` |
| Újra | `Cmd+Shift+Z` |
| Bekezdés másolása | `Cmd+C` (kijelölt szöveg nélkül) |
| Bekezdés kivágása | `Cmd+X` (kijelölt szöveg nélkül) |

### Szövegformázás

| Művelet | Billentyűkombináció |
| ---- | ---- |
| Félkövér szöveg | `Cmd+B` |
| Dőlt szöveg | `Cmd+I` |

### Szövegszerkesztés

| Művelet | Billentyűkombináció |
|-|-|
| Új sor beszúrása | `Enter` |
| Előző karakter törlése | `Backspace` |
| Következő karakter törlése | `Delete` |
| Előző szó törlése | `Option+Backspace` |
| Következő szó törlése | `Option+Delete` |
| Törlés az aktuális sor elejéig | `Cmd+Backspace` |
| Törlés az aktuális sor végéig | `Cmd+Delete` |
| Aktuális sor törlése | `Cmd+Shift+K` (kijelölt szöveg nélkül) |

### Szövegnavigáció

| Művelet | Billentyűkombináció |
|-|-|
| Kurzor mozgatása egy karakterrel | `Left/→` |
| Kurzor mozgatása az előző szó elejére | `Option+←` |
| Kurzor mozgatása a következő szó végére | `Option+→` |
| Kurzor mozgatása az aktuális sor elejére | `Cmd+←` |
| Kurzor mozgatása az aktuális sor végére | `Cmd+→` |
| Kurzor mozgatása az előző sorra | `↑` |
| Kurzor mozgatása a következő sorra | `↓` |
| Kurzor mozgatása a jegyzet elejére | `Cmd+↑` |
| Kurzor mozgatása a jegyzet végére | `Cmd+↓` |
| Kurzor mozgatása egy oldallal feljebb | `Fn+↑` |
| Kurzor mozgatása egy oldallal lejjebb | `Fn+↓` |
