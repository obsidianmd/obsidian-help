A Licença Catalyst é uma compra única que dá acesso antecipado a versões beta e ajuda a apoiar o desenvolvimento contínuo do Obsidian. Você pode apoiar o Obsidian [comprando uma Licença Catalyst](https://obsidian.md/buy/catalyst).

Os benefícios incluem:

- Acesso antecipado a [[#Ativar versões de acesso antecipado|novas versões]] do Obsidian
- Acesso a [[#Fazer parte dos nossos canais Insider|canais Insider]] exclusivos.
- [[#Obter seu distintivo do fórum|Distintivos de fórum]] e [[#Obter seu distintivo do Discord|distintivos do Discord]] exclusivos.

## Níveis Catalyst

Existem três níveis de Catalyst baseados no valor da sua contribuição: *Insider*, *Supporter* e *VIP*.

| Benefícios                                                                                    | Insider | Supporter |   VIP    |
| --------------------------------------------------------------------------------------------- | :-----: | :-------: | :------: |
| Acesso antecipado a novas versões                                                             |    ✓    |     ✓     |    ✓     |
| Acesso a canais exclusivos de desenvolvimento, `#insider-desktop` e `#insider-mobile`         |    ✓    |     ✓     |    ✓     |
| Acesso ao canal exclusivo lounge, `#vip-lounge`                                               |         |     ✓     |    ✓     |
| Distintivo Insider                                                                            |    ✓    |           |          |
| Distintivo Supporter                                                                          |         |     ✓     |          |
| Distintivo VIP                                                                                |         |           |    ✓     |
| **Contribuição (USD)**                                                                        | **$25** |  **$50**  | **$100** |


> [!info] Atualize seu nível Catalyst
> Se você já possui uma Licença Catalyst, pode atualizar para um nível Catalyst superior pagando a diferença de preço entre o seu nível atual e o novo.

## Acessar benefícios Catalyst

Após comprar uma Licença Catalyst, você pode acessar diversos benefícios. Saiba como ativá-los aqui.

### Obter seu distintivo do fórum

**Nota:** Para conectar sua conta Obsidian e sua conta do fórum, você deve usar o mesmo endereço de e-mail em ambas as contas.

1. Se ainda não o fez, faça parte do nosso [fórum](https://forum.obsidian.md/).
2. Acesse o [painel da sua conta Obsidian](https://obsidian.md/account/catalyst).
3. Em **Catalyst**, selecione **Get forum badge**.

### Obter seu distintivo do Discord

1. Se ainda não o fez, faça parte do nosso [servidor Discord](https://discord.gg/veuWUTm).
2. Acesse o [painel da sua conta Obsidian](https://obsidian.md/account/catalyst).
3. Em **Catalyst**, selecione **Get Discord badge**.

### Fazer parte dos nossos canais Insider

Insiders podem conversar com a equipe do Obsidian e outros Insiders em nossos canais exclusivos do Discord. Você automaticamente entra nos canais Insider após [[#Obter seu distintivo do Discord]].
