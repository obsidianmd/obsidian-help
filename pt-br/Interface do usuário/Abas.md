As abas no Obsidian funcionam de forma semelhante às abas em outros aplicativos, como navegadores web.

Você pode abrir quantas abas quiser no Obsidian. Também é possível organizar as abas para criar leiautes personalizados que persistem até a próxima vez que você abrir o aplicativo.

> [!note] Abas no celular
> No celular, você pode gerenciar as abas abertas a partir do contador de abas na [[Aplicativo móvel#Barra de navegação|Barra de navegação]].

## Abrir uma nova aba

No topo da janela do aplicativo, ao lado da última aba à direita, selecione **Nova aba** ( ![[lucide-plus.svg#icon]] ). Ou use um atalho de teclado:

- **Windows e Linux:** `Ctrl+t`
- **macOS:** `Cmd+t`

## Abrir um link

Selecione um link no Obsidian para abri-lo na aba ativa.

Para abrir um link em uma nova aba, pressione `Ctrl` (ou `Cmd` no macOS) e selecione o link.

A seguir estão todas as teclas modificadoras que você pode usar para abrir links de várias maneiras:

|Ação|MacOS|Windows/Linux|
|---|---|---|
|**Navegar**|_Nenhum_|_Nenhum_|
|**Nova aba**|`⌘` (+ `Shift` no Modo de origem)|`Ctrl` (+ `Shift` no Modo de origem)|
|**Novo grupo de abas**|`⌘` `⌥`| `Ctrl` `Alt`|
|**Nova janela**|`⌘` `⌥` `Shift`|`Ctrl` `Alt` `Shift`|

## Organizar suas abas e janelas

Cada aba pertence a um _grupo de abas_. Você pode arrastar e soltar abas para reorganizá-las dentro de um grupo de abas, movê-las para um grupo diferente ou criar um novo grupo de abas. No desktop, você pode arrastar abas para fora da janela para abri-las em uma [[Janelas destacáveis|janela destacável]] separada.

As abas nas barras laterais mostram apenas o ícone. Passe o cursor sobre o ícone para exibir uma dica com o título da aba.

### Organizar abas

Para alterar a ordem das suas abas, arraste a aba ao longo das abas no grupo de abas.

Ao arrastar uma aba, _zonas de soltar_—áreas para onde você pode mover a aba—ficam destacadas. A zona de soltar determina onde inserir a aba. Algumas abas só podem estar em uma das barras laterais.

### Dividir um grupo de abas

Clique com o botão direito em uma aba e selecione **Dividir para a direita** ou **Dividir para baixo** para criar um novo grupo de abas com essa aba.

Você também pode dividir um grupo de abas arrastando uma aba para a parte inferior de outra aba.

### Redimensionar um grupo de abas

Para redimensionar um grupo de abas, passe o cursor sobre a borda do grupo de abas. A borda fica destacada quando pode ser arrastada para redimensionar.

Você pode redimensionar as barras laterais de forma semelhante para dar mais espaço aos grupos de abas no centro.

### Mover aba para uma nova janela

**Arrastar e soltar:**

- Selecione e arraste a aba para fora da janela do aplicativo para abri-la em uma nova janela.

**Paleta de comandos:**

- Abra a Paleta de comandos e selecione **Mover painel atual para uma nova janela**.

### Mover uma aba para uma janela diferente

Para mover uma aba para outra janela existente, arraste a aba para a janela para a qual deseja movê-la.

### Fixar uma aba

Para fixar uma aba no editor principal, clique com o botão direito na aba e selecione **Fixar**. Links em uma aba fixada sempre abrem em uma aba separada.

Para desafixar uma aba fixada no editor principal, clique com o botão direito na aba e selecione **Desafixar**.

Para fixar ou desafixar uma aba na barra lateral, consulte [[Barra lateral#Fixar abas|fixar abas na barra lateral]].

## Alternar para uma aba diferente

Selecione uma aba para alternar para ela. Ou use um atalho de teclado:

| Alternar para               | MacOS            | Windows/Linux        |
|-----------------------------|------------------|----------------------|
| **Próxima aba**             | `⌃`+`⇥`         | `Ctrl`+`Tab`         |
| **Aba anterior**            | `⌃`+`⇧`+`⇥`    | `Ctrl`+`Shift`+`Tab` |
| **Primeira aba à esquerda** | `⌘`+`1`          | `Ctrl`+`1`           |
| **2ª até 8ª aba**           | `⌘`+`2`..`8`     | `Ctrl`+`2`..`8`      |
| **Última aba à direita**    | `⌘`+`9`          | `Ctrl`+`9`           |
| **Aba fechada recentemente**| `⌘`+`⇧`+`t`     | `Ctrl`+`Shift`+`t`   |

## Empilhar grupos de abas

Você pode empilhar abas para deslizá-las sobre outras abas no mesmo grupo de abas.

Para empilhar notas, selecione a seta para baixo no canto superior direito do grupo de abas e então selecione **Empilhar notas**.

![tab-stacks](https://user-images.githubusercontent.com/693981/188205363-0f24b2a5-3706-4a8c-b38b-7a66baa68ce6.gif)

As pilhas de abas são inspiradas nas [notas deslizantes de Andy Matuschak](https://notes.andymatuschak.org/).

## Visualizações vinculadas

_Visualizações vinculadas_ são abas que fazem referência a uma aba diferente. Quando o conteúdo da aba referenciada muda, a visualização vinculada também muda.

Para abas de notas, você pode usar os seguintes plugins como visualizações vinculadas:

- [[Visão de grafo]] (local)
- [[Links inversos]]
- [[Sumário]]

Para abrir uma visualização vinculada para uma aba de nota:

1. Selecione **Mais opções** ( ![[lucide-more-horizontal.svg#icon]] ) no canto superior direito da nota.
2. Em **Abrir visualização vinculada**, selecione a visualização vinculada que deseja abrir.

## Salvar leiautes

Você pode salvar e restaurar leiautes de janelas usando o plugin [[Áreas de trabalho]].
