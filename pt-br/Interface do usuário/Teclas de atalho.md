DESCRIPTION: Teclas de atalho são atalhos de teclado personalizáveis para comandos do Obsidian. Você pode atribuir qualquer combinação de teclas para acionar comandos como abrir arquivos, inserir modelos ou alternar plugins.

Teclas de atalho são diferentes dos [[Atalhos de edição|atalhos de teclado do sistema]] (como `Ctrl+C` para copiar), que são fornecidos pelo seu sistema operacional e não podem ser personalizados no Obsidian.

## Visualizar teclas de atalho

A maneira mais rápida de ver a tecla de atalho definida para um comando é encontrar o comando na [[Paleta de comandos]]. Se houver uma tecla de atalho listada para esse comando, ela está definida.

Você também pode visualizar e administrar todas as teclas de atalho em **[[Configurações]] → Teclas de atalho**.

## Definir uma tecla de atalho

Para adicionar ou modificar uma tecla de atalho:

1. Abra **[[Configurações]] → Teclas de atalho**.
2. Encontre o comando na lista ou use o filtro de pesquisa.
3. Selecione o ícone de mais (+) ao lado do comando.
4. Pressione a combinação de teclas que deseja usar.
5. Selecione **Salvar** para confirmar.

Você pode atribuir múltiplas combinações de teclas de atalho a um único comando selecionando o ícone de mais (+) novamente após definir a primeira tecla de atalho.

## Remover uma tecla de atalho

Para remover uma tecla de atalho:

1. Abra **[[Configurações]] → Teclas de atalho**.
2. Encontre o comando com a tecla de atalho que deseja remover.
3. Selecione o ícone X ao lado da combinação de teclas.

## Filtrar teclas de atalho

Para mostrar apenas os comandos que possuem teclas de atalho atribuídas, selecione o ícone de filtro em **[[Configurações]] → Teclas de atalho**.

## Layouts de teclado não americanos

As teclas de atalho são exibidas como apareceriam em um layout de teclado americano (US). Mesmo que a combinação exibida não corresponda ao layout do seu teclado, a tecla de atalho funcionará com base nas teclas reais que você pressionou (desde que você não mude o layout do seu teclado).

## Atalhos de teclado padrão

Para uma lista completa dos atalhos de teclado padrão para edição e navegação, consulte [[Atalhos de edição]].
