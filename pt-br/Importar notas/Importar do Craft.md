O Obsidian permite que você migre suas notas do Craft. Isso converterá seus dados do Craft em arquivos Markdown duráveis, que você pode usar com o Obsidian e muitos outros aplicativos.

## Exportar seus dados do Craft

Abra o aplicativo Craft no seu computador desktop:

1. Vá para a visualização "Organize" ou "All Documents"
2. Clique no menu de três pontos
3. Selecione a opção Exportar.
4. Escolha Markdown.

Dependendo do número de notas que você tem no Craft, esse processo pode levar algum tempo.

## Importar seus dados do Craft para o Obsidian

A partir daqui, você pode seguir as instruções de [[Importar arquivos Markdown]].

## Solução de problemas

O Craft Docs usa sintaxe não padrão para certos links. Você pode encontrar recursos para migrar esses links no [repositório GitHub do Importador](https://github.com/obsidianmd/obsidian-importer/issues/27).
