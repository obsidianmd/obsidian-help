Aprenda como estender o Obsidian com plugins criados pela comunidade. Use plugins para adaptar o Obsidian às suas necessidades específicas, como suporte a formatos de arquivo adicionais ou integração com serviços de terceiros.

> [!warning] Aviso
> Plugins da comunidade executam código de terceiros em seu nome, o que pode potencialmente causar danos. Para saber mais sobre o que a equipe do Obsidian faz para prevenir plugins prejudiciais, consulte [[Segurança de plugins]].

## Selecionar plugins da comunidade

1. Abra as **[[Configurações]]**.
2. Selecione **Ligar os plugins da comunidade**. Para mais informações, consulte [[Segurança de plugins#Modo restrito|Modo restrito]].
3. Selecione **Selecionar** para listar todos os plugins da comunidade disponíveis.

Use a caixa de texto para filtrar plugins com base em nome, autor e descrição.

Você também pode explorar plugins disponíveis no seu navegador, acessando [obsidian.md/plugins](https://obsidian.md/pt-BR/plugins).

## Instalar um plugin da comunidade

Para instalar um plugin da comunidade, você deve primeiro desativar o [[Segurança de plugins#Modo restrito|Modo restrito]].

1. Abra as **[[Configurações]]**.
2. No menu lateral, selecione **Plugins da comunidade**.
3. Selecione **Selecionar** para explorar os plugins da comunidade disponíveis.
4. Selecione o plugin que deseja instalar.
5. Selecione **Instalar**.

Para usar um plugin instalado, você precisa ativá-lo.

## Ativar um plugin da comunidade

Selecione **Ativar** diretamente após instalá-lo, ou ative-o a partir da lista de Plugins da comunidade em **[[Configurações]] → Plugins da comunidade → Plugins instalados**.

## Atualizar plugins

Por questões de segurança, plugins da comunidade não são atualizados automaticamente. Você pode atualizar todos os plugins de uma vez ou atualizar plugins individualmente.

Para atualizar todos os seus plugins:

1. Abra as **[[Configurações]]**.
2. Em **Plugins da comunidade → Plugins atuais**, selecione **Verificar por atualizações**.
3. Se houver atualizações disponíveis, selecione **Atualizar tudo**.

Para atualizar um plugin individual:

1. Abra as **[[Configurações]]**.
2. Em **Plugins da comunidade → Plugins atuais**, selecione **Verificar por atualizações**.
3. Em **Plugins instalados**, selecione **Atualizar** ao lado do plugin que deseja atualizar.

## Desinstalar um plugin da comunidade

1. Abra as **[[Configurações]]**.
2. No menu lateral, selecione **Plugins da comunidade**.
3. Em **Plugins instalados**, selecione o ícone da lixeira ![[lucide-trash-2.svg#icon]] ao lado do plugin que deseja desinstalar.
4. Selecione **Desinstalar** para confirmar.

## Administrar plugins instalados

Em **[[Configurações]] → Plugins da comunidade → Plugins instalados**, cada plugin possui vários ícones para gerenciá-lo:

- **[[Configurações]]** ![[lucide-settings.svg#icon]] — Abrir a página de configurações do plugin, caso possua opções configuráveis.
- **[[Teclas de atalho]]** ![[lucide-plus-circle.svg#icon]] — Definir atalhos de teclado para os comandos do plugin.
- **Financiamento** ![[lucide-heart.svg#icon]] — Visualizar a URL de financiamento do autor do plugin para apoiar seu trabalho.
- **Desinstalar** ![[lucide-trash-2.svg#icon]] — Remover o plugin do seu cofre.
- **Alternar** — Ativar ou desativar o plugin sem desinstalá-lo.

Você também pode:

- Selecionar o ícone de **recarregar** para recarregar todos os plugins.
- Selecionar o ícone de **pasta** ![[lucide-folder-open.svg#icon]] para abrir a pasta de plugins na sua [[Pasta de configuração|pasta de configuração]].
- Usar a **barra de pesquisa** para filtrar plugins por nome.
