O Criador de nota única é um [[Plugins nativos|plugin nativo]] que permite criar notas com nomes baseados em tempo, também conhecidas como notas Zettelkasten.

Por exemplo, se você criar uma nova nota às 09:45 do dia 1º de janeiro de 2024, o Criador de nota única cria uma nota com `202401010945` como nome. Se uma nota com o mesmo nome já existir, a nova nota usa o próximo timestamp disponível.

## Criar uma nota única

**Menu lateral:**

- No menu lateral, clique em **Criar nova nota única** ![[obsidian-icon-sheets-in-box.svg#icon]].

**Paleta de comandos:**

1. Pressione `Ctrl+P` (ou `Cmd+P` no macOS) para abrir a [[Paleta de comandos]].
2. Pesquise pelo comando **Criar nova nota única**.
3. Pressione `Enter` com o comando selecionado.

## Criar notas únicas a partir de um modelo

Por padrão, novas notas únicas estão vazias. Para usar outro arquivo como modelo:

- Abra as **[[Configurações]]**.
- Na barra lateral, clique em **Plugins nativos → Criador de nota única**.
- Em **Localização do arquivo de modelo**, insira o arquivo que você deseja usar como modelo.

Para mais informações sobre como escrever modelos, consulte [[Plugins/Modelos]].
