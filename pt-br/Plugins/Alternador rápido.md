Navegação rápida é um [[Plugins nativos|plugin nativo]] que permite pesquisar e abrir notas usando apenas o teclado.

## Abrir a Navegação rápida

Existem várias maneiras de abrir a Navegação rápida, quando está ativada:
- Pressione `Ctrl+O` (ou `Cmd+O` no macOS) para abrir a Navegação rápida;
- No menu lateral, clique em **Abrir a troca rápida** ( ![[lucide-file-search.svg#icon]] );
- No celular, quando você não está editando uma nota, toque no ícone de mais na parte inferior central do aplicativo.

## Abrir uma nota com a Navegação rápida

1. [[#Abrir a Navegação rápida]].
2. Digite para pesquisar uma nota por nome ou apelido.
3. Navegue até a nota usando as teclas de seta.
4. Pressione `Enter` para abrir a nota selecionada.

> [!info] 
> A funcionalidade de autocompletar muda para um algoritmo de resultados mais simples quando o cofre atinge 10.000 itens para manter o desempenho ideal do aplicativo. 
^search-autocomplete-large

Se o texto não corresponder a nenhuma nota, você pode pressionar `Enter` para criar uma nota com esse nome. Mesmo que o texto corresponda a uma ou mais notas semelhantes, você ainda pode criar uma nota com o nome exato pressionando `Shift+Enter`.

Você pode abrir a nota selecionada em uma nova aba pressionando `Ctrl+Enter` (ou `Cmd+Enter` no macOS).

> [!tip] Alternar entre duas notas
> Se o termo de pesquisa estiver vazio, a Navegação rápida mostra as notas mais recentes. Alterne entre duas notas abrindo a Navegação rápida, pressionando a seta `Para baixo` e depois pressionando `Enter`.


> [!info] Arquivos excluídos
> Arquivos que correspondem aos seus padrões de [[Configurações#Arquivos excluídos|Arquivos excluídos]] são despriorizados nos resultados da Navegação rápida.
