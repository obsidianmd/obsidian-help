Se você conhece outro idioma, preferencialmente como nativo, pode ajudar a traduzir a interface de usuário do Obsidian ou a documentação.

Para ajudar a traduzir a interface de usuário, envie um pull request para o [repositório de traduções](https://github.com/obsidianmd/obsidian-translations).

Para ajudar a melhorar nossa documentação, envie um pull request para o repositório [obsidian-help](https://github.com/obsidianmd/obsidian-help). Certifique-se de seguir o [[Guia de estilo#Traduções|guia de estilo para traduções]] ao fazer alterações.
