Obsidian Web Clipper é uma extensão de navegador gratuita que permite destacar páginas e salvar conteúdo da web no seu cofre.

## Instalar o Web Clipper

Obtenha a extensão Obsidian Web Clipper no diretório oficial do seu navegador:

- [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
	- Para Chrome, Brave, Arc, Orion e outros navegadores baseados em Chromium.
- [Firefox Add-Ons](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
	- Para Firefox e Firefox Mobile.
- [Safari Extensions](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
	- Para macOS, iOS e iPadOS.
- [Edge Add-Ons](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
	- Para Microsoft Edge.

## Privacidade

O Obsidian Web Clipper salva conteúdo localmente no seu cofre do Obsidian e segue nossa [política de privacidade](https://obsidian.md/pt-BR/privacy). Seus dados não são coletados e não reunimos nenhuma métrica de uso. O código é [código aberto](https://github.com/obsidianmd/obsidian-clipper) e auditável.
