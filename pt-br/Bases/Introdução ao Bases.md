Bases é um [[Plugins nativos|plugin nativo]] que permite criar visualizações semelhantes a bancos de dados das suas notas. Usando uma base, você pode visualizar, editar, ordenar e filtrar arquivos e suas [[Propriedades|propriedades]]. Bases pode ajudar a organizar tudo, desde projetos até planos de viagem, listas de leitura e muito mais.

Cada base pode ter várias [[Visualizações|visualizações]] com diferentes leiautes, como tabelas e cartões. Abaixo está um exemplo de visualização de tabela onde cada linha é um arquivo, e cada coluna é uma propriedade desse arquivo.

![Exemplo de uma base mostrando uma visualização de tabela com uma lista de livros](bases-noshadow.png#interface)

Todos os dados no Obsidian Bases são armazenados nos seus arquivos Markdown locais e suas [[Propriedades|propriedades]]. As visualizações são descritas pela [[Sintaxe de Bases]], que pode ser salva como um arquivo `.base` ou [[Criar uma base#Incorporar uma base|incorporada]] em blocos de código dentro dos seus arquivos Markdown.

## Tipos de visualização

As visualizações podem ser exibidas com diferentes leiautes. Leiautes adicionais podem ser adicionados por [[Plugins da comunidade]].

- [[Visualização de tabela|Tabela]]
	- Exiba arquivos como linhas com colunas que mostram propriedades desses arquivos.
- [[Visualização de lista|Lista]]
	- Exiba arquivos como uma lista com marcadores ou numeração.
- [[Visualização de cartões|Cartões]]
	- Exiba arquivos como uma grade. Permite criar visualizações tipo galeria com imagens.
- [[Visualização de mapa|Mapa]]
	- Exiba arquivos como marcadores em um mapa interativo.
