Cartões é um tipo de [[Visualizações|visualização]] que você pode usar no [[Introdução ao Bases|Bases]].

Selecione ![[lucide-layout-grid.svg#icon]]  **Cartões** no menu de visualização para exibir arquivos em um leiaute de grade estilo galeria com imagens de capa opcionais.

## Configurações

As configurações da visualização de cartões podem ser configuradas em [[Visualizações#Configurações de visualização|Configurações de visualização]].

- Tamanho do cartão
- Propriedade de imagem
- Ajuste de imagem
- Proporção da imagem

### Tamanho do cartão

Define a largura de um cartão.

### Propriedade de imagem

Cartões suportam uma imagem de capa opcional, que é uma [[Propriedades|propriedade]] exibida como imagem no topo do cartão. A propriedade pode ser qualquer uma das seguintes:

- Um link para um [[Anexos|anexo]] local `"[[link/para/anexo.jpg]]"`
- Um link externo (URL)
- Um código de cor hexadecimal (`#000000`)

### Ajuste de imagem

Se você tiver uma propriedade de imagem configurada, esta opção determinará como a imagem será exibida no cartão.

- **Cobrir:** A imagem preenche a caixa de conteúdo do cartão. Se não couber, a imagem será cortada.
- **Conter:** A imagem é redimensionada até caber na caixa de conteúdo do cartão. A imagem não será cortada.

### Proporção da imagem

A altura da imagem de capa é determinada pela sua proporção. A proporção padrão é 1:1, o que significa que todas as suas imagens serão quadradas. Ajuste esta opção para tornar a imagem mais baixa ou mais alta.
