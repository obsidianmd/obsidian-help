Espiar página é um [[Plugins nativos|plugin nativo]] que permite visualizar uma página quando você passa o cursor sobre um link interno, sem precisar navegar até essa página.

Por padrão, passar o cursor sobre um link exibirá a prévia desse arquivo no [[Explorador de arquivos]], [[Pesquisa]], [[Links inversos]] e mais. Para visualizar uma página enquanto estiver no Editor, pressione `Ctrl` (ou `Cmd` no macOS) enquanto passa o cursor sobre o link.

Também existe uma opção para exigir `Ctrl` (ou `Cmd` no macOS) enquanto passa o cursor para ver o arquivo. As configurações dessa opção estão em Plugins nativos > Espiar página.

> [!hint]
> A pré-visualização está **ativada por padrão**. Alternar a opção do plugin irá **desativar o funcionamento**.
