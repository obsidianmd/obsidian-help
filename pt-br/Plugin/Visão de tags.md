Painel de etiquetas é um [[Plugins nativos|plugin nativo]] que lista todas as etiquetas no seu cofre e o número de notas para cada etiqueta.

Clique em uma etiqueta para pesquisá-la usando a [[Pesquisa]].

Para alternar uma etiqueta no termo de pesquisa, clique na etiqueta enquanto pressiona `Ctrl` (ou `Cmd` no macOS).

O plugin oferece as seguintes opções:

- **Mudar ordenação** determina se as etiquetas devem ser ordenadas por **Nome da etiqueta** ou **Frequência**.
- **Mostrar tags aninhadas** alterna entre exibir etiquetas aninhadas como uma árvore ou como uma lista plana.
- **Expandir tudo** e **Recolher todos** expande e recolhe a hierarquia de etiquetas aninhadas.

Você pode escolher expandir ou recolher cada nível aninhado separadamente clicando na seta ao lado dele.
