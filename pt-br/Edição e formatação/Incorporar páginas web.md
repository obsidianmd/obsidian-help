Aprenda como usar o elemento HTML [iframe](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/iframe) para incorporar páginas web em suas notas.

Para incorporar uma página web, adicione o seguinte em sua nota e substitua o texto de exemplo pela URL da página web que você deseja incorporar:

```html
<iframe src="INSIRA SUA URL AQUI"></iframe>
```

> [!note] Nota
> Alguns sites não permitem que você os incorpore. Em vez disso, eles podem fornecer URLs destinadas a incorporá-los. Se o site não suportar incorporação, tente pesquisar o nome do site seguido de "embed iframe". Por exemplo, "youtube embed iframe".

> [!tip] Dica
> Se você estiver usando [[Canvas]], pode incorporar uma página web em um cartão. Para mais informações, consulte [[Canvas#Adicionar cartões a partir de páginas web]].

## Incorporar um vídeo do YouTube

Para incorporar um vídeo do YouTube, use a mesma sintaxe Markdown de [[Sintaxe de formatação básica#Imagens externas|imagens externas]]:

```md
![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)
```

![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)

## Incorporar um tweet

Para incorporar um tweet, use a mesma sintaxe Markdown de [[Sintaxe de formatação básica#Imagens externas|imagens externas]]:

```md
![](https://twitter.com/obsdmd/status/1580548874246443010)
```

![](https://twitter.com/obsdmd/status/1580548874246443010)
