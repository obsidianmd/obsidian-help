Esta página lista os atalhos de teclado padrão para navegar e editar texto no Obsidian. Esses atalhos são fornecidos pelo seu sistema operacional ou pelo framework no qual o Obsidian é construído, e não podem ser personalizados dentro do Obsidian.

Para atalhos de teclado personalizáveis para comandos do Obsidian, veja [[Teclas de atalho]].

## Atalhos do Windows e Linux

### Ações comuns

| Ação | Atalho |
|-|-|
| Copiar | `Ctrl+C` |
| Recortar | `Ctrl+X` |
| Colar | `Ctrl+V` |
| Colar sem formatação | `Ctrl+Shift+V` |
| Desfazer | `Ctrl+Z` |
| Refazer | `Ctrl+Shift+Z` ou `Ctrl+Y` |
| Copiar parágrafo | `Ctrl+C` (sem texto selecionado) |
| Recortar parágrafo | `Ctrl+X` (sem texto selecionado) |

### Edição de texto

| Ação | Atalho |
|-|-|
| Inserir nova linha | `Enter` |
| Excluir o caractere anterior | `Backspace` |
| Excluir o próximo caractere | `Delete` |
| Excluir a palavra anterior | `Ctrl+Backspace` |
| Excluir a próxima palavra | `Ctrl+Delete` |
| Excluir a linha atual | `Ctrl+Shift+K` (sem texto selecionado) |

### Navegação de texto

| Ação | Atalho |
|-|-|
| Mover o cursor um caractere | `Left/→` |
| Mover o cursor para o início da palavra anterior | `Ctrl+←` |
| Mover o cursor para o final da próxima palavra | `Ctrl+→` |
| Mover o cursor para o início da linha atual | `Home` |
| Mover o cursor para o final da linha atual | `End` |
| Mover o cursor para a linha anterior | `↑` |
| Mover o cursor para a próxima linha | `↓` |
| Mover o cursor para o início da nota | `Ctrl+Home` |
| Mover o cursor para o final da nota | `Ctrl+End` |
| Mover o cursor uma página acima | `Page up` |
| Mover o cursor uma página abaixo | `Page down` |

### Seleção de texto

| Ação | Atalho |
|-|-|
| Simplificar seleção | `Escape` |
| Selecionar tudo | `Ctrl+A` |
| Estender seleção um caractere | `Shift+Left/→` |
| Estender seleção até o início da palavra anterior | `Ctrl+Shift+←` |
| Estender seleção até o final da próxima palavra | `Ctrl+Shift+→` |
| Estender seleção até o início da linha atual | `Shift+Home` |
| Estender seleção até o final da linha atual | `Shift+End` |
| Estender seleção até o início da nota | `Ctrl+Shift+Home` |
| Estender seleção até o final da nota | `Ctrl+Shift+End` |
| Estender seleção uma página acima | `Shift+Page up` |
| Estender seleção uma página abaixo | `Shift+Page down` |

## Atalhos do macOS

### Ações comuns

| Ação | Atalho |
|-|-|

| Recortar | `Cmd+X` |
| Colar | `Cmd+V` |
| Colar sem formatação | `Cmd+Shift+V` |
| Desfazer | `Cmd+Z` |
| Refazer | `Cmd+Shift+Z` |
| Copiar parágrafo | `Cmd+C` (sem texto selecionado) |
| Recortar parágrafo | `Cmd+X` (sem texto selecionado) |

### Formatação de texto

| Ação | Atalho |
| ---- | ---- |
| Texto em negrito | `Cmd+B` |
| Texto em itálico | `Cmd+I` |

### Edição de texto

| Ação | Atalho |
|-|-|

| Excluir até o início da linha atual | `Cmd+Backspace` |
| Excluir até o final da linha atual | `Cmd+Delete` |
| Excluir a linha atual | `Cmd+Shift+K` (sem texto selecionado) |

### Navegação de texto

| Ação | Atalho |
|-|-|
| Mover o cursor um caractere | `Left/→` |
| Mover o cursor para o início da palavra anterior | `Option+←` |
| Mover o cursor para o final da próxima palavra | `Option+→` |
| Mover o cursor para o início da linha atual | `Cmd+←` |
| Mover o cursor para o final da linha atual | `Cmd+→` |
| Mover o cursor para a linha anterior | `↑` |
| Mover o cursor para a próxima linha | `↓` |
| Mover o cursor para o início da nota | `Cmd+↑` |
| Mover o cursor para o final da nota | `Cmd+↓` |
| Mover o cursor uma página acima | `Fn+↑` |
| Mover o cursor uma página abaixo | `Fn+↓` |
