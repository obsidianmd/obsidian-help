Apreneu a utilitzar l'element HTML [iframe](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/iframe) per incrustar pàgines web a les vostres notes.

Per incrustar una pàgina web, afegiu el següent a la vostra nota i substituïu el text de mostra per l'URL de la pàgina web que voleu incrustar:

```html
<iframe src="INSERIU EL VOSTRE URL AQUÍ"></iframe>
```

> [!note] Nota
> Alguns llocs web no permeten que els incrusteu. En canvi, poden proporcionar URL pensats per a la seva incrustació. Si el lloc web no admet la incrustació, proveu de cercar el nom del lloc web seguit de "embed iframe". Per exemple, "youtube embed iframe".

> [!tip] Consell
> Si utilitzeu [[Canvas]], podeu incrustar una pàgina web en una targeta. Per a més informació, consulteu [[Canvas#Afegeix targetes des de pàgines web]].

## Incrustar un vídeo de YouTube

Per incrustar un vídeo de YouTube, utilitzeu la mateixa sintaxi Markdown que les [[Sintaxi de format bàsic#Imatges externes|imatges externes]]:

```md
![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)
```

![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)

## Incrustar un tuit

Per incrustar un tuit, utilitzeu la mateixa sintaxi Markdown que les [[Sintaxi de format bàsic#Imatges externes|imatges externes]]:

```md
![](https://twitter.com/obsdmd/status/1580548874246443010)
```
