Aquesta pàgina llista les dreceres de teclat per defecte per navegar i editar text a Obsidian. Aquestes dreceres les proporciona el vostre sistema operatiu o el framework sobre el qual està construït Obsidian, i no es poden personalitzar dins d'Obsidian.

Per a dreceres de teclat personalitzables per a ordres d'Obsidian, consulteu [[Tecles d'accés ràpid]].

## Dreceres de Windows i Linux

### Accions comunes

| Acció | Drecera |
|-|-|
| Copia | `Ctrl+C` |
| Retalla | `Ctrl+X` |
| Enganxa | `Ctrl+V` |
| Enganxa sense format | `Ctrl+Shift+V` |
| Desfés | `Ctrl+Z` |
| Refés | `Ctrl+Shift+Z` o `Ctrl+Y` |
| Copia el paràgraf | `Ctrl+C` (sense text seleccionat) |
| Retalla el paràgraf | `Ctrl+X` (sense text seleccionat) |

### Edició de text

| Acció | Drecera |
|-|-|
| Insereix una nova línia | `Enter` |
| Suprimeix el caràcter anterior | `Backspace` |
| Suprimeix el caràcter següent | `Delete` |
| Suprimeix la paraula anterior | `Ctrl+Backspace` |
| Suprimeix la paraula següent | `Ctrl+Delete` |
| Suprimeix la línia actual | `Ctrl+Shift+K` (sense text seleccionat) |

### Navegació de text

| Acció | Drecera |
|-|-|
| Mou el cursor un caràcter | `Left/→` |
| Mou el cursor a l'inici de la paraula anterior | `Ctrl+←` |
| Mou el cursor al final de la paraula següent | `Ctrl+→` |
| Mou el cursor a l'inici de la línia actual | `Home` |
| Mou el cursor al final de la línia actual | `End` |
| Mou el cursor a la línia anterior | `↑` |
| Mou el cursor a la línia següent | `↓` |
| Mou el cursor a l'inici de la nota | `Ctrl+Home` |
| Mou el cursor al final de la nota | `Ctrl+End` |
| Mou el cursor una pàgina amunt | `Page up` |
| Mou el cursor una pàgina avall | `Page down` |

### Selecció de text

| Acció | Drecera |
|-|-|
| Simplifica la selecció | `Escape` |
| Selecciona-ho tot | `Ctrl+A` |
| Amplia la selecció un caràcter | `Shift+Left/→` |
| Amplia la selecció fins a l'inici de la paraula anterior | `Ctrl+Shift+←` |
| Amplia la selecció fins al final de la paraula següent | `Ctrl+Shift+→` |
| Amplia la selecció fins a l'inici de la línia actual | `Shift+Home` |
| Amplia la selecció fins al final de la línia actual | `Shift+End` |
| Amplia la selecció fins a l'inici de la nota | `Ctrl+Shift+Home` |
| Amplia la selecció fins al final de la nota | `Ctrl+Shift+End` |
| Amplia la selecció una pàgina amunt | `Shift+Page up` |
| Amplia la selecció una pàgina avall | `Shift+Page down` |
