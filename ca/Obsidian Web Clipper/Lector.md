[[Introducció a Obsidian Web Clipper|Web Clipper]] inclou un mode de lectura integrat anomenat Lector que elimina el desordre de les pàgines web, presentant únicament el contingut principal de l'article en un format net i llegible.

## Activar el mode de lectura

Podeu activar el Lector de diverses maneres:

- La **icona de llibre** a la capçalera del panell de l'extensió.
- La drecera de teclat `Alt+Shift+R`, personalitzable a la configuració de l'extensió.
- Activar el mode de lectura de nou restaurarà la pàgina original.

## Com funciona

El Lector extreu el contingut principal de la pàgina. La pàgina original es substitueix per una vista de lectura neta que inclou:

- El títol de l'article, l'autor, la data de publicació i el domini d'origen.
- El contingut principal de l'article, amb imatges i format preservats.
- Un esquema a la barra lateral per a articles amb múltiples encapçalaments.
- Ressaltat de sintaxi per a blocs de codi.
- Finestres emergents de notes a peu de pàgina que es mostren en línia en fer-hi clic.

## Personalitzar la vista de lectura

El mode Lector inclou una barra de configuració oculta que podeu mostrar passant el cursor per la cantonada superior dreta de la pàgina, amb controls per personalitzar l'experiència de lectura.

| Configuració              | Descripció                                                                                           |
| ------------------------- | ---------------------------------------------------------------------------------------------------- |
| Mida de la font           | Ajusta la mida del text.                                                                             |
| Amplada de línia          | Amplada màxima de l'àrea de contingut.                                                               |
| Alçada de línia           | Espai entre línies.                                                                                  |
| Esquema&nbsp;de&nbsp;color |                                                                                                      |
| Tema                      | Automàtic segueix la preferència del sistema. O escull usar sempre el mode clar o el mode fosc.       |
