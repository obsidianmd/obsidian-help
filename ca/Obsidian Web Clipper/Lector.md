[[Introducció a Obsidian Web Clipper|Web Clipper]] inclou un mode de lectura integrat anomenat **Lector** que elimina el desordre de les pàgines web, presentant únicament el contingut principal de l'article en un format net i llegible.

## Activar el mode de lectura

Podeu activar el Lector de diverses maneres:

- ![[lucide-book-icon.svg#icon]] **Icona de llibre** al panell de l'extensió.
- **Drecera de teclat** personalitzable a la configuració de l'extensió.
- **Menú contextual** fent clic dret a la pàgina web que esteu visitant.

També podeu configurar el botó de l'extensió ![[obsidian.svg#icon|20]] perquè obri el Lector amb un sol clic. A la configuració del Web Clipper, aneu a **General** → **Behavior** → **Open behavior** i trieu **Reader**.

Totes les opcions anteriors també us permeten sortir del Lector i restaurar la pàgina original.

## Com funciona

El Lector extreu el contingut principal de la pàgina. La pàgina original es substitueix per una vista de lectura neta que inclou:

- El títol de l'article, l'autor, la data de publicació i el domini d'origen.
- El contingut principal de l'article, amb imatges i format preservats.
- Un esquema a la barra lateral per a articles amb múltiples encapçalaments.
- Ressaltat de sintaxi per a blocs de codi.
- Finestres emergents de notes a peu de pàgina que es mostren en línia en fer-hi clic.

## Personalitzar la vista de lectura

Personalitzeu la vostra experiència amb el Lector a la configuració de l'extensió. Algunes opcions també estan disponibles dins del mode Lector fent clic a la icona ![[lucide-case-sensitive.svg#icon]].

| Configuració              | Descripció                                        |
| ------------------------- | ------------------------------------------------- |
| Font                      | Afegiu fonts instal·lades al vostre sistema.      |
| Mida de la font           | Ajusta la mida del text.                          |
| Alçada&nbsp;de&nbsp;línia | Espai entre línies.                               |
| Amplada&nbsp;de&nbsp;línia | Amplada màxima de l'àrea de contingut.            |
| Aparença                  | Trieu aparença clara o fosca.                     |
| Tema                      | Trieu els colors.                                 |
| CSS personalitzat         | CSS addicional aplicat a la pàgina.               |
