Obsidian Web Clipper és una extensió de navegador gratuïta que et permet ressaltar pàgines i desar contingut web a la teva cambra forta.

## Instal·lar Web Clipper

Obtén l'extensió Obsidian Web Clipper des del directori oficial per al teu navegador:

- [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
	- Per a Chrome, Brave, Arc, Orion i altres navegadors basats en Chromium.
- [Firefox Add-Ons](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
	- Per a Firefox i Firefox Mobile.
- [Safari Extensions](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
	- Per a macOS, iOS i iPadOS.
- [Edge Add-Ons](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
	- Per a Microsoft Edge.

- [[Retallar pàgines web]]
	- Comença i aprèn com desar contingut de les pàgines.

- [[Lector|Lector]]

- [[Interpretar pàgines web|Intèrpret]]
	- Utilitza indicacions en llenguatge natural per capturar i modificar dades d'una pàgina.
- [[Resolució de problemes del Web Clipper|Resolució de problemes]]
	- Resol problemes habituals amb Web Clipper.
- [[Obsidian Web Clipper/Plantilles|Plantilles]]
	- Crea plantilles personalitzades per a llocs web específics.
- [[Variables]]
	- Omple prèviament les dades de la pàgina a la teva plantilla.
- [[Filtres]]

- [[Lògica]]

## Privacitat

Obsidian Web Clipper desa el contingut localment a la teva cambra forta d'Obsidian i segueix la nostra [política de privacitat](https://obsidian.md/privacy). Les teves dades no es recopilen, i no recollim cap mètrica d'ús. El codi és de [codi obert](https://github.com/obsidianmd/obsidian-clipper) i auditable.
