Obsidian utilitza fitxers Markdown `.md` com a format principal per a les notes. Això facilita la importació de fitxers Markdown a Obsidian. Moltes aplicacions poden editar fitxers Markdown o exportar a aquest format.

Hi ha diverses maneres d'afegir fitxers Markdown a la vostra cambra forta d'Obsidian:

1. Arrossegueu i deixeu anar fitxers o carpetes a la finestra de l'[[Explorador de fitxers]]
2. Utilitzeu el navegador de fitxers del vostre sistema (p. ex. l'Explorador de Windows o el Finder a macOS), i moveu els vostres fitxers directament a la carpeta de la cambra forta d'Obsidian.

### Convertir des d'altres variants de Markdown

Obsidian utilitza [[Obsidian Flavored Markdown]]. Algunes aplicacions utilitzen una sintaxi de Markdown lleugerament diferent. Podeu utilitzar l'[[Importador de format Markdown]] per convertir els vostres fitxers a la sintaxi Markdown d'Obsidian.
