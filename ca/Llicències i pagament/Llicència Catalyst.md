La llicència Catalyst és una compra única que us dóna accés anticipat a versions beta i ajuda a donar suport al desenvolupament continuat d'Obsidian. Podeu donar suport a Obsidian [comprant una llicència Catalyst](https://obsidian.md/buy/catalyst).

Els beneficis inclouen:

- Accés anticipat a [[#Habilitar versions d'accés anticipat|noves versions]] d'Obsidian
- Accés a [[#Uneix-te als nostres canals Insider|canals Insider]] exclusius.
- [[#Obtén la teva insígnia del fòrum|Insígnies del fòrum]] i [[#Obtén la teva insígnia de Discord|insígnies de Discord]] exclusives.

## Nivells de Catalyst

Hi ha tres nivells de Catalyst basats en la quantitat que contribuïu: *Insider*, *Supporter* i *VIP*.

| Beneficis                                | Insider | Supporter |   VIP    |
| ---------------------------------------- | :-----: | :-------: | :------: |
| Accés anticipat a noves versions         |    ✓    |     ✓     |    ✓     |
| Accés a canals de desenvolupament exclusius |    ✓    |     ✓     |    ✓     |
| Accés al canal exclusiu de sala VIP      |         |     ✓     |    ✓     |
| Insígnia Insider                         |    ✓    |           |          |
| Insígnia Supporter                       |         |     ✓     |          |
| Insígnia VIP                             |         |           |    ✓     |
| **Contribució (USD)**                    | **$25** |  **$50**  | **$100** |


> [!info] Actualitza el teu nivell de Catalyst
> Si ja teniu una llicència Catalyst, podeu actualitzar a un nivell de Catalyst superior pagant la diferència de preu entre el vostre nivell actual i el nou.

## Accedir als beneficis de Catalyst

Un cop hàgiu comprat una llicència Catalyst, podeu accedir a una sèrie de beneficis. Apreneu com habilitar-los aquí.

### Obtén la teva insígnia del fòrum

**Nota:** Per connectar el vostre compte d'Obsidian i el vostre compte del fòrum, heu d'utilitzar la mateixa adreça de correu electrònic per a ambdós comptes.

1. Si encara no ho heu fet, uniu-vos al nostre [fòrum](https://forum.obsidian.md/).
2. Aneu al [tauler del vostre compte d'Obsidian](https://obsidian.md/account/catalyst).
3. Sota **Catalyst**, seleccioneu **Get forum badge**.

### Obtén la teva insígnia de Discord

1. Si encara no ho heu fet, uniu-vos al nostre [servidor de Discord](https://discord.gg/veuWUTm).
2. Aneu al [tauler del vostre compte d'Obsidian](https://obsidian.md/account/catalyst).
3. Sota **Catalyst**, seleccioneu **Get Discord badge**.

### Uneix-te als nostres canals Insider

Els Insiders poden xatejar amb l'equip d'Obsidian i altres Insiders als nostres canals exclusius de Discord. Us uniu automàticament als canals Insider després d'[[#Obtén la teva insígnia de Discord]].

- [vip-lounge](https://discord.com/channels/686053708261228577/714234422379151400) per a discussions amb membres de Catalyst.
- [insider-desktop-release](https://discord.com/channels/686053708261228577/716028884885307432) per a les últimes versions d'accés anticipat a escriptori.
- [insider-mobile-release](https://discord.com/channels/686053708261228577/817515900349448202) per a les últimes versions d'accés anticipat a mòbil.
