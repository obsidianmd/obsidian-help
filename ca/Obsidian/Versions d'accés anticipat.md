Obteniu accés anticipat a les properes versions activant les _versions d'accés anticipat_. Les versions d'accés anticipat només estan disponibles per als usuaris amb una [[Llicència Catalyst]].

> [!warning] Advertència
> Les versions d'accés anticipat són versions beta. Inclouen noves funcionalitats però poden ser menys estables. No activeu les versions d'accés anticipat si preferiu una experiència més fiable.
> 
> Tingueu en compte que els desenvolupadors de connectors de la comunitat i temes reben les versions d'accés anticipat al mateix temps que tothom. Sigueu pacients amb els desenvolupadors que necessiten fer actualitzacions per donar suport a les noves funcionalitats.

## Activar les versions d'accés anticipat per a escriptori

Per rebre les versions d'accés anticipat tan aviat com estiguin disponibles, seguiu aquests passos:

1. Obriu **[[Configuració]]**.
2. A la barra lateral, seleccioneu **General**.
3. Sota **Compte → El teu compte**, seleccioneu **Inicia sessió**.
4. A **Correu electrònic**, introduïu el vostre correu electrònic.
5. A **Contrasenya**, introduïu la vostra contrasenya.
6. Un cop hàgiu iniciat sessió, torneu a **[[Configuració]]**.
7. A la barra lateral, seleccioneu **General**.
8. Sota **Aplicació**, activeu **Rebre versions de prova**.
9. Feu clic a **Comprovar actualitzacions** i després a **Torna a iniciar**.

## Instal·lar les versions d'accés anticipat en dispositius mòbils

Per trobar instruccions sobre com instal·lar les versions d'accés anticipat al vostre dispositiu mòbil, seguiu aquests passos:

1. Baixeu i instal·leu [Discord](<https://discord.com>).
2. Uniu-vos al [servidor de Discord d'Obsidian](https://discord.gg/obsidianmd).
3. [[Llicència Catalyst#Obteniu la vostra insígnia de Discord|Obteniu la vostra insígnia de Discord]] per accedir als canals Insider.
4. Al canal `#insider-welcome`, trobareu instruccions per accedir a la vostra descàrrega segons el vostre tipus de dispositiu.

## Informar de problemes i altres comentaris

Si descobriu un problema en una versió d'accés anticipat, considereu informar-ne a l'equip d'Obsidian. Abans d'informar d'un problema, cerqueu al [fòrum](https://forum.obsidian.md/) o a Discord per veure si algú ja l'ha reportat.

Per informar d'un problema, utilitzeu un dels canals següents:

- A Discord, informeu del problema al canal `#insider-release` corresponent.
- Al fòrum, creeu un nou tema sota [Bug reports](https://forum.obsidian.md/c/bug-reports/7).

Quan informeu d'un problema, incloeu la versió de compilació i el sistema operatiu on l'esteu executant. Podeu trobar la versió de compilació a **[[Configuració]] → Quant a → Aplicació → Versió actual**.

## Tornar a les versions públiques a l'escriptori

Per tornar a utilitzar les versions públiques (no d'accés anticipat) a l'escriptori:

1. Desactiveu les versions d'accés anticipat.
   1. Obriu **[[Configuració]]**.
   2. A la barra lateral, seleccioneu **General**.
   3. Sota **Aplicació**, desactiveu **Rebre versions de prova**.
2. Tanqueu Obsidian.
3. Suprimiu el fitxer `obsidian-VERSION.asar`, on `VERSION` és la versió d'Obsidian.
   - Windows: `%APPDATA%\obsidian\obsidian-VERSION.asar`
   - Mac: `~/Library/Application Support/obsidian/obsidian-VERSION.asar`
   - Linux: `~/.config/obsidian/obsidian-VERSION.asar`
4. Reinicieu Obsidian.

## Tornar a les versions públiques al mòbil

Per tornar a utilitzar les versions públiques (no d'accés anticipat) al mòbil:

1. Feu una còpia de seguretat de les dades de la vostra cambra forta
2. Desinstal·leu Obsidian
3. Reinstal·leu Obsidian des de la Play Store o l'App Store d'Apple
4. Restaureu les dades de la vostra cambra forta des de les còpies de seguretat
5. Obriu Obsidian
