מחבר הערות הוא [[תוספי ליבה|תוסף ליבה]] המאפשר לכם למזג שתי הערות או לחלץ חלק מהערה להערה חדשה.

## מיזוג הערות

מיזוג הערות מוסיף הערה לאחרת ומסיר את הראשונה. מחבר הערות מעדכן את כל הקישורים כך שיצביעו להערה הממוזגת.

כאשר אתם בוחרים את ההערה למיזוג, תוכלו לבחור בין השיטות הבאות:

- `Enter`: מוסיף את הערת המקור ב*סוף* הערת היעד.
- `Shift+Enter`: מוסיף את הערת המקור ב*תחילת* הערת היעד.
- `Ctrl+Enter` (או `Cmd+Enter` ב-macOS): יוצר הערה חדשה עם תוכן הערת המקור.

למיזוג ההערה הפעילה עם הערה אחרת בכספת שלכם:

**סייר הקבצים**

1. בסייר הקבצים, לחצו לחיצה ימנית על ההערה שברצונכם למזג.
2. לחצו על **מיזוג הקובץ כולו עם...**.
3. בחרו את ההערה שאליה ברצונכם למזג.
4. לחצו על **למזג** לאישור.

**לוח פקודות**

1. פתחו את [[לוח פקודות|לוח הפקודות]].
2. בחרו **מחבר הערות: מיזוג קובץ נוכחי עם קובץ אחר...**.
3. בחרו את ההערה שאליה ברצונכם למזג.
4. לחצו על **למזג** לאישור.

> [!tip] עצה
> כברירת מחדל, מחבר הערות מבקש מכם אישור בעת מיזוג הערות. אם תשביתו את האישור, ותמזגו הערה בטעות, תוכלו עדיין לשחזר אותה באמצעות תוסף [[שחזור קובץ]].

## חילוץ הערה

כאשר אתם בוחרים את ההערה לחילוץ הבחירה אליה, תוכלו לבחור בין השיטות הבאות:

- `Enter`: מוסיף את הטקסט הנבחר ב*סוף* הערת היעד.
- `Shift+Enter`: מוסיף את הטקסט הנבחר ב*תחילת* הערת היעד.
- `Ctrl+Enter` (או `Cmd+Enter` ב-macOS): יוצר הערה חדשה עם הטקסט הנבחר.

לחילוץ טקסט להערה חדשה:

**עורך**

1. בזמן שאתם ב**עורך**, סמנו את הטקסט שברצונכם לחלץ.
2. לחצו לחיצה ימנית על הטקסט הנבחר.
3. לחצו על **חלץ את הבחירה הנוכחית...**.
4. בחרו את ההערה שאליה ברצונכם לחלץ.

**לוח פקודות**

1. בזמן שאתם ב**עורך**, סמנו את הטקסט שברצונכם לחלץ.
2. פתחו את [[לוח פקודות|לוח הפקודות]].
3. בחרו **מחבר הערות: חלץ את הבחירה הנוכחית...**.
4. בחרו את ההערה שאליה ברצונכם לחלץ.

> [!tip] עצה
> כברירת מחדל, מחבר הערות מחליף את הטקסט שחולץ בקישור להערת היעד. בהגדרות, תוכלו גם לשנות כך שיתבצע [[הטמעת קבצים|הטמעה]] של הערת היעד, או שלא יישאר דבר במקומו.

## קובץ תבנית

על ידי הגדרת תבנית, תוכלו להתאים אישית את התוכן לפני הוספתו להערה החדשה. לשימוש בתבנית, הזינו **מיקום קובץ התבנית** בהגדרות התוסף.

התבנית יכולה להכיל את המשתנים הבאים:

| משתנה              | תיאור                                                                                                                                                  |
|-------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| `{{content}}`     | התוכן למיזוג, או הטקסט הנבחר שחולץ. אם לא תכללו משתנה זה, מחבר הערות מוסיף את התוכן בתחתית התבנית. |
| `{{fromTitle}}`   | שם הערת המקור.                                                                                                                                 |
| `{{newTitle}}`    | שם הערת היעד. לדוגמא, כדי להוסיף את שם הקובץ ככותרת בראש הקובץ.                                                     |
| `{{date:FORMAT}}` | תאריך יצירת ההערה החדשה. לדוגמא, `{{date:YYYY-MM-DD}}`.                                                                                       |
