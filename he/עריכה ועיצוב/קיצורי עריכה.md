דף זה מפרט קיצורי מקלדת ברירת מחדל לניווט ולעריכת טקסט ב-Obsidian. קיצורים אלה מסופקים על ידי מערכת ההפעלה שלך או על ידי המסגרת שעליה בנוי Obsidian, ולא ניתן להתאים אותם בתוך Obsidian.

לקיצורי מקלדת הניתנים להתאמה אישית עבור פקודות Obsidian, ראו [[מקשי קיצור]].

## קיצורי Windows ו-Linux

### פעולות נפוצות

| פעולה | קיצור |
|-|-|
| העתק | `Ctrl+C` |
| גזור | `Ctrl+X` |
| הדבק | `Ctrl+V` |
| הדבק ללא עיצוב | `Ctrl+Shift+V` |
| בטל | `Ctrl+Z` |
| בצע שוב | `Ctrl+Shift+Z` או `Ctrl+Y` |
| העתק פסקה | `Ctrl+C` (ללא טקסט נבחר) |
| גזור פסקה | `Ctrl+X` (ללא טקסט נבחר) |

### עריכת טקסט

| פעולה | קיצור |
|-|-|
| הכנס שורה חדשה | `Enter` |
| מחק את התו הקודם | `Backspace` |
| מחק את התו הבא | `Delete` |
| מחק את המילה הקודמת | `Ctrl+Backspace` |
| מחק את המילה הבאה | `Ctrl+Delete` |
| מחק את השורה הנוכחית | `Ctrl+Shift+K` (ללא טקסט נבחר) |

### ניווט בטקסט

| פעולה | קיצור |
|-|-|
| הזז את הסמן תו אחד | `Left/→` |
| הזז את הסמן לתחילת המילה הקודמת | `Ctrl+←` |
| הזז את הסמן לסוף המילה הבאה | `Ctrl+→` |
| הזז את הסמן לתחילת השורה הנוכחית | `Home` |
| הזז את הסמן לסוף השורה הנוכחית | `End` |
| הזז את הסמן לשורה הקודמת | `↑` |
| הזז את הסמן לשורה הבאה | `↓` |
| הזז את הסמן לתחילת ההערה | `Ctrl+Home` |
| הזז את הסמן לסוף ההערה | `Ctrl+End` |
| הזז את הסמן עמוד אחד למעלה | `Page up` |
| הזז את הסמן עמוד אחד למטה | `Page down` |

### בחירת טקסט

| פעולה | קיצור |
|-|-|
| פשט בחירה | `Escape` |
| בחר הכל | `Ctrl+A` |
| הרחב בחירה תו אחד | `Shift+Left/→` |
| הרחב בחירה לתחילת המילה הקודמת | `Ctrl+Shift+←` |
| הרחב בחירה לסוף המילה הבאה | `Ctrl+Shift+→` |
| הרחב בחירה לתחילת השורה הנוכחית | `Shift+Home` |
| הרחב בחירה לסוף השורה הנוכחית | `Shift+End` |
| הרחב בחירה לתחילת ההערה | `Ctrl+Shift+Home` |
| הרחב בחירה לסוף ההערה | `Ctrl+Shift+End` |
| הרחב בחירה עמוד אחד למעלה | `Shift+Page up` |
| הרחב בחירה עמוד אחד למטה | `Shift+Page down` |

## קיצורי macOS

### פעולות נפוצות

| פעולה | קיצור |
|-|-|
| העתק | `Cmd+C` |
| גזור | `Cmd+X` |
| הדבק | `Cmd+V` |
| הדבק ללא עיצוב | `Cmd+Shift+V` |
| בטל | `Cmd+Z` |
| בצע שוב | `Cmd+Shift+Z` |
| העתק פסקה | `Cmd+C` (ללא טקסט נבחר) |
| גזור פסקה | `Cmd+X` (ללא טקסט נבחר) |

### עיצוב טקסט

| פעולה | קיצור |
| ---- | ---- |
| טקסט מודגש | `Cmd+B` |
| טקסט נטוי | `Cmd+I` |

### עריכת טקסט

| פעולה | קיצור |
|-|-|
| הכנס שורה חדשה | `Enter` |
| מחק את התו הקודם | `Backspace` |
| מחק את התו הבא | `Delete` |
| מחק את המילה הקודמת | `Option+Backspace` |
| מחק את המילה הבאה | `Option+Delete` |
| מחק עד תחילת השורה הנוכחית | `Cmd+Backspace` |
| מחק עד סוף השורה הנוכחית | `Cmd+Delete` |
| מחק את השורה הנוכחית | `Cmd+Shift+K` (ללא טקסט נבחר) |

### ניווט בטקסט

| פעולה | קיצור |
|-|-|
| הזז את הסמן תו אחד | `Left/→` |
| הזז את הסמן לתחילת המילה הקודמת | `Option+←` |
| הזז את הסמן לסוף המילה הבאה | `Option+→` |
| הזז את הסמן לתחילת השורה הנוכחית | `Cmd+←` |
| הזז את הסמן לסוף השורה הנוכחית | `Cmd+→` |
| הזז את הסמן לשורה הקודמת | `↑` |
| הזז את הסמן לשורה הבאה | `↓` |
| הזז את הסמן לתחילת ההערה | `Cmd+↑` |
| הזז את הסמן לסוף ההערה | `Cmd+↓` |
| הזז את הסמן עמוד אחד למעלה | `Fn+↑` |
| הזז את הסמן עמוד אחד למטה | `Fn+↓` |
