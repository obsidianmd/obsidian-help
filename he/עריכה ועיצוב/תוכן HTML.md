Obsidian תומך ב-HTML כדי לאפשר לך להציג את ההערות שלך בדרך שאתה רוצה, או אפילו [[הטמעת דפי אינטרנט|להטמיע דפי אינטרנט]]. הרשאת HTML בתוך ההערות שלך כרוכה בסיכונים. כדי למנוע מקוד זדוני לגרום נזק, Obsidian _מחטא_ כל HTML בהערות שלך.

> [!example]
> האלמנט `<script>` בדרך כלל מאפשר לך להריץ JavaScript בכל פעם שהוא נטען. אם Obsidian לא היה מחטא HTML, תוקף יכול היה לשכנע אותך להדביק טקסט המכיל JavaScript שמחלץ מידע רגיש מהמחשב שלך ושולח אותו בחזרה אליו.

עם זאת, מכיוון שתחביר Markdown אינו תומך בכל צורות העיצוב, שימוש ב-HTML מחוטא יכול להיות דרך נוספת לשיפור איכות ההערות שלך. כללנו כמה מהשימושים הנפוצים יותר של HTML.

## מגבלות HTML

ל-Obsidian יש מגבלות ספציפיות בעת שימוש ב-HTML בהערות שלך:

### אין Markdown בתוך HTML

Obsidian אינו מעבד תחביר Markdown בתוך אלמנטי HTML. זוהי בחירת עיצוב מכוונת לאופטימיזציית ביצועים ולשמירה על מורכבות מנתח נמוכה בעת ניהול מסמכים גדולים.

לדוגמה, זה לא יעבוד כצפוי:

```md
<div>
This **will not** be bold and this `will not` be code.
</div>
```

### בלוקי HTML חייבים להיות עצמאיים

בלוקי HTML חייבים להיות שלמים ולא יכולים להכיל שורות ריקות בתוכם. שורות ריקות ישברו את בלוק ה-HTML.

זה יעבוד:

```md
<table>
<tr>
<td>Content here</td>
</tr>
</table>
```

זה לא יעבוד כראוי:

```md
<table>

<tr>

<td>Content here</td>

</tr>

</table>
```

### כאשר Markdown נראה כאילו הוא עובד בתוך HTML

לחלק מתגיות HTML מוטבעות כמו `<span>` או `<a>` יש פונקציונליות מוגבלת והן עשויות להיראות כאילו הן מעבדות Markdown, אך זה לא באמת מה שקורה. ה-Markdown מעובד מחוץ להקשר של ה-HTML.

לפרטים נוספים על האופן שבו Obsidian מטפל ב-Markdown, ראה [[Obsidian Flavored Markdown]].

## שימושים נפוצים ב-HTML

> [!info] פרטים נוספים על שימוש ב-`<iframe>` ניתן למצוא ב-[[הטמעת דפי אינטרנט]].

### הערות

[[תחביר עיצוב בסיסי#הערות|הערות Markdown]] הן הדרך המועדפת להוספת הערות מוסתרות בתוך ההערות שלך. עם זאת, חלק מהשיטות להמרת הערות Markdown, כגון [Pandoc](https://pandoc.org), מספקות תמיכה מוגבלת בהערות Markdown. במקרים אלה, אתה יכול להשתמש ב-`<!-- HTML Comment -->` במקום!

### קו תחתון

אם אתה צריך להוסיף קו תחתון לפריט בהערות שלך, אתה יכול להשתמש ב-`<u>Example</u>` כדי ליצור <u>טקסט עם קו תחתון</u>.

### Span/Div

ניתן להשתמש בתגיות span ו-div כדי להחיל מחלקות מותאמות אישית מ[[קטעי CSS|קטע CSS]], או עיצוב מוגדר מותאם אישית, על אזור נבחר של טקסט. לדוגמה, שימוש ב-`<span style="font-family: cursive">your text</span>` יכול לאפשר לך <span style="font-family: cursive">לשנות את הגופן שלך</span> במהירות.

## קו חוצה

צריך למחוק <s>טקסט כלשהו</s>? השתמש ב-`<s>this</s>` כדי להוסיף לו קו חוצה.
