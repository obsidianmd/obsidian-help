למד כיצד ניתן להטמיע הערות אחרות ומדיה בתוך ההערות שלך. על ידי הטמעת קבצים בהערות שלך, תוכל לעשות שימוש חוזר בתוכן ברחבי הכספת.

כדי להטמיע קובץ בכספת שלך, הוסף סימן קריאה (`!`) לפני [[קישורים פנימיים|קישור פנימי]]. ניתן להטמיע קבצים בכל אחד מ[[פורמטי קבצים נתמכים]].

> [!tip] הטמעה בגרירה ושחרור
> בשולחן העבודה, ניתן גם לגרור ולשחרר קבצים נתמכים ישירות לתוך ההערה כדי להטמיע אותם אוטומטית.

## הטמעת הערה בתוך הערה אחרת

כדי להטמיע הערה:

```md
![[קישורים פנימיים]]
```

ניתן גם להטמיע קישורים ל[[קישורים פנימיים#קישור לכותרת בהערה|כותרות]] ו[[קישורים פנימיים#קישור לבלוק בהערה|בלוקים]].

```md
![[קישורים פנימיים#^b15695]]
```

הטקסט שלהלן הוא דוגמה לבלוק מוטמע:

![[קישורים פנימיים#^b15695]]

## הטמעת תמונה בהערה

כדי להטמיע תמונה:

```md
![[Engelbart.jpg]]
```

![[Engelbart.jpg#outline]]

ניתן לשנות את ממדי התמונה על ידי הוספת `|640x480` ליעד הקישור, כאשר 640 הוא הרוחב ו-480 הוא הגובה.

```md
![[Engelbart.jpg|100x145]]
```

אם מציינים רק את הרוחב, התמונה תשתנה בהתאם ליחס הגובה-רוחב המקורי שלה. לדוגמה, `![[Engelbart.jpg|100]]`.

![[Engelbart.jpg#outline|100]]

ניתן גם להטמיע תמונה המתארחת באופן חיצוני באמצעות קישור Markdown. ניתן לשלוט ברוחב ובגובה באותו אופן כמו קישור ויקי.

```md
![250](https://publish-01.obsidian.md/access/f786db9fac45774fa4f0d8112e232d67/Attachments/Engelbart.jpg)
```

![250](https://publish-01.obsidian.md/access/f786db9fac45774fa4f0d8112e232d67/Attachments/Engelbart.jpg)

## הטמעת PDF בהערה

כדי להטמיע PDF:

```md
![[Document.pdf]]
```

ניתן גם לפתוח עמוד ספציפי ב-PDF על ידי הוספת `#page=N` ליעד הקישור, כאשר `N` הוא מספר העמוד:

```md
![[Document.pdf#page=3]]
```

ניתן גם לציין את הגובה בפיקסלים עבור מציג ה-PDF המוטמע, על ידי הוספת `#height=[number]` לקישור. לדוגמה:

```md
![[Document.pdf#height=400]]
```

## הטמעת רשימה בהערה

כדי להטמיע רשימה מהערה אחרת, תחילה הוסף [[קישורים פנימיים#קישור לבלוק בהערה|מזהה בלוק]] לרשימה שלך:

```md

- פריט רשימה 1
- פריט רשימה 2

^my-list-id
```

לאחר מכן קשר לרשימה באמצעות מזהה הבלוק:

```md
![[My note#^my-list-id]]
```
