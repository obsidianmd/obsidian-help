## דרישות מוקדמות

כדי להשתמש באנליטיקה באתר ה-[[מבוא ל-Obsidian Publish|Obsidian Publish]] שלך, אתה צריך [[תחומים מותאמים אישית|תחום מותאם אישית]].

> [!important] חשוב
> לפני שאתה מפעיל אנליטיקה, וודא שהחוקים והתקנות המקומיים שלך מאפשרים לך לעקוב אחר המבקרים שלך. בהתאם לכלים שאתה משתמש בהם, ייתכן שתצטרך להוסיף באנר הסכמה לאתר שלך.

## Google Analytics

כדי להפעיל Google Analytics עבור אתר ה-Obsidian Publish שלך:

1. בסרגל הכלים, משמאל לחלון האפליקציה, לחץ על **פרסם שינויים** ![[lucide-send.svg#icon]].
2. בתיבת הדו-שיח **פרסם שינויים**, לחץ על **שנה אפשרויות אתר** ![[lucide-cog.svg#icon]].
3. ב**קוד מעקב של Google Analytics**, הזן את קוד המעקב שלך.

כדי להשתמש ב-Google Tag Manager במקום ב-Google Analytics, השתמש ב-JavaScript מותאם אישית כדי להוסיף סקריפטים משלך.

## Plausible, Fathom ושירותי אנליטיקה אחרים

אתה יכול להוסיף את רוב שירותי האנליטיקה לאתר שלך דרך קובץ ה-[[התאמה אישית של האתר|publish.js]] שלך. וודא שאתה מחליף את `yourdomain.com` בתחום שלך, ואת ה-`src` של הסקריפט בסקריפט מספק האנליטיקה שלך.

הנה דוגמא באמצעות [Plausible Analytics](https://plausible.io/)

```js
var analyticsScript = document.createElement('script');
analyticsScript.defer = true;
analyticsScript.setAttribute('data-domain', 'yourdomain.com');
analyticsScript.src = 'https://plausible.io/js/plausible.js';
document.head.appendChild(analyticsScript);
```

 ניתן להשתמש באותה גישה עבור [Fathom Analytics](https://usefathom.com/). שים לב לשינוי מ-`data-domain` ל-`data-site` — ספקי אנליטיקה שונים עשויים להשתמש בפורמט שונה לאופן שבו יש להכניס את הסקריפט.

```javascript
var fathom = analyticsScript.createElement('script');
analyticsScript.defer = true;
analyticsScript.setAttribute('data-site', 'yourdomain.com');
analyticsScript.src = 'https://cdn.usefathom.com/script.js';
document.head.appendChild(analyticsScript);
```

## פתרון בעיות

כדי לוודא שהאתר שלך משתמש בשירות האנליטיקה שלך, השבת תוספי דפדפן לחסימת פרסומות, כגון uBlock Origin, שעשויים לחסום את הרצת סקריפט המעקב.
