Obsidian Headless **(בטא פתוחה)** הוא לקוח ללא ממשק גרפי עבור שירותי Obsidian. הוא מאפשר לכם [[סנכרון ללא ממשק|לסנכרן כספות]] משורת הפקודה ללא האפליקציה השולחנית, עם כל היתרונות של מהירות, פרטיות ו[[מבוא ל-Obsidian Sync|הצפנה מקצה לקצה של Obsidian Sync]].

סיבות לשימוש ב-Obsidian Headless:

- אוטומציה של גיבויים מרוחקים.
- אוטומציה של פרסום אתר אינטרנט.
- מתן גישה לכלים אוטונומיים (agentic) לכספת ללא גישה למחשב המלא שלכם.
- סנכרון כספת צוותית משותפת לשרת שמזין כלים אחרים.
- הפעלת אוטומציות מתוזמנות, למשל צבירת הערות יומיות לסיכומים שבועיים, תיוג אוטומטי וכו'.

> [!info] Obsidian Headless לעומת Obsidian CLI
> [[Obsidian CLI]] שולט באפליקציית Obsidian השולחנית מהטרמינל שלכם. Obsidian Headless הוא לקוח עצמאי שרץ באופן בלתי תלוי, ללא צורך באפליקציה השולחנית.

## התקנה

Obsidian Headless **(בטא פתוחה)** דורש Node.js 22 ומעלה. התקינו אותו מ-[npm](https://www.npmjs.com/package/obsidian-headless):

```shell
npm install -g obsidian-headless
```

## אימות

### התחברות

```shell
ob login
```

אם כבר מחוברים, `ob login` מציג את פרטי החשבון שלכם. כדי להחליף חשבונות, העבירו `--email` ו/או `--password` כדי להתחבר שוב.

```
ob login [--email <email>] [--password <password>] [--mfa <code>]
```

כל האפשרויות הן אינטראקטיביות כשלא מצוינות — דוא״ל וסיסמה יתבקשו, ואימות דו-שלבי יתבקש אוטומטית אם מופעל בחשבון.

כדי להתנתק ולנקות אישורים מאוחסנים:

```shell
ob logout
```
