## גלובליות

פונקציות גלובליות משמשות ללא סוג.

### `escapeHTML()`

`escapeHTML(html: string): string`

- מסמנת תווים מיוחדים במחרוזת כדי להפוך אותה לבטוחה להכללה ב-HTML.

### `date()`

`date(date: string): date`

- `date(string): date` מפענחת את המחרוזת שסופקה ומחזירה אובייקט תאריך.
- מחרוזת ה-`date` צריכה להיות בפורמט `YYYY-MM-DD HH:mm:ss`.

### `duration()`

`duration(value: string): duration`

- מפענחת מחרוזת כמשך זמן. ראה את [[תחביר Bases#חשבון תאריכים|סעיף חשבון התאריכים]] לפורמט מחרוזת ה-`value`.
- אין צורך לפענח משכי זמן באופן מפורש בעת ביצוע חשבון תאריכים (לדוגמה, `now() + '1d'`), אך יש צורך בכך בעת ביצוע חשבון על משכי זמן (לדוגמה, `now() + (duration('1d') * 2)`).
- בעת ביצוע חשבון על משכי זמן עם סקלרים, משך הזמן חייב להיות בצד שמאל. לדוגמה `duration('5h') * 2`, במקום `2 * duration('5h')`.

### `file()`

`file(path: string | file | url): file`

- מחזירה אובייקט קובץ עבור הקובץ או הנתיב הנתון.
- דוגמה: `file(link("[[filename]]"))` או `file("path to file")`.

### `html()`

`html(html: string): html`

- ממירה מחרוזת לקטע קוד שמוצג כ-HTML.

### `if()`

`if(condition: any, trueResult: any, falseResult?: any): any`

- `condition` הוא התנאי שייבדק.
- `trueResult` הוא הפלט אם התנאי נכון.
- `falseResult` הוא הפלט האופציונלי אם התנאי שקרי. אם לא ניתן, מניחים שהוא `null`.
- מחזירה את `trueResult` אם `condition` הוא נכון, או ערך אמתי (truthy), או `falseResult` אחרת.
- דוגמה: `if(isModified, "Modified", "Unmodified")`

### `image()`

`image(path: string | file | url): image`

- מחזירה אובייקט תמונה שמציג את התמונה בתצוגה.
- דוגמה: `image(image-property)` או `image("https://obsidian.md/images/obsidian-logo-gradient.svg")`

### `icon()`

`icon(name: string): icon`

- מחזירה ערך שיוצג כסמל בתצוגה. שם הסמל חייב להתאים לסמל Lucide נתמך.
- דוגמה: `icon("arrow-right")`.

### `link()`

`link(path: string | file, display?: value): Link`

- מפענחת מחרוזת `path` ומחזירה אובייקט קישור שמוצג כקישור לנתיב הנתון.
- ניתן לספק באופן אופציונלי את הפרמטר `display` כדי לשנות את הטקסט שהקישור מציג.

### `list()`

`list(element: any): List`

- אם האלמנט שסופק הוא רשימה, מחזירה אותו ללא שינוי.
- אחרת, עוטפת את ה-`element` שסופק ברשימה, ויוצרת רשימה עם אלמנט יחיד.
- פונקציה זו יכולה להיות מועילה כאשר מאפיין מכיל תערובת של מחרוזות או רשימות ברחבי הכספת.
- דוגמה: `list("value")` מחזירה `["value"]`.

### `max()`

`max(value1: number, value2: number...): number`

- מחזירה את הגדול מבין כל המספרים שסופקו.

### `min()`

`min(value1: number, value2: number...): number`

- מחזירה את הקטן מבין כל המספרים שסופקו.

### `now()`

`now(): date`

- `now()` מחזירה אובייקט תאריך המייצג את הרגע הנוכחי.

### `number()`

`number(input: any): number`

- מנסה להחזיר את הערך שסופק כמספר.
- אובייקטי תאריך יוחזרו כמילישניות מאז תחילת עידן Unix.
- ערכים בוליאניים יחזירו 1 או 0.
- מחרוזות יפוענחו למספר ויחזירו שגיאה אם התוצאה אינה חוקית.
- דוגמה, `number("3.4")` מחזירה `3.4`.

### `duration()`

`duration(value: string): duration`

- מפענחת מחרוזת כמשך זמן. ראה את [[תחביר Bases#חשבון תאריכים|סעיף חשבון התאריכים]] לפורמט מחרוזת ה-`value`.
- אין צורך לפענח משכי זמן באופן מפורש בעת ביצוע חשבון תאריכים (לדוגמה, `now() + '1d'`), אך יש צורך בכך בעת ביצוע חשבון על משכי זמן (לדוגמה, `now() + (duration('1d') * 2)`).
- בעת ביצוע חשבון על משכי זמן עם סקלרים, משך הזמן חייב להיות בצד שמאל. לדוגמה `duration('5h') * 2`, במקום `2 * duration('5h')`.

### `today()`

`today(): date`

- `today()` מחזירה אובייקט תאריך המייצג את התאריך הנוכחי. חלק השעה מוגדר לאפס.

### `random()`

`random(): number`

- `random()` מחזירה מספר אקראי בין 0 ל-1.
- יצירת המספר מתרעננת בכל פעם שתצוגה נטענת. מעבר בין תצוגות משנה את המספר האקראי.

## כל סוג

פונקציות שניתן להשתמש בהן עם כל ערך. זה כולל מחרוזות (למשל `"hello"`), מספרים (למשל `42`), רשימות (למשל `[1,2,3]`), אובייקטים, ועוד.

### `isTruthy()`

`any.isTruthy(): boolean`

- מחזירה את הערך מומר לבוליאני.
- דוגמה: `1.isTruthy()` מחזירה `true`.

### `isType()`

`any.isType(type: string): boolean`

- מחזירה true אם הערך הוא מהסוג שסופק.
- דוגמה: `"example".isType("string")` ו-`true.isType("boolean")` שניהם מחזירים true.

### `toString()`

`any.toString(): string`

- מחזירה את ייצוג המחרוזת של כל ערך.
- דוגמה: `123.toString()` מחזירה `"123"`.

## תאריך

פונקציות שניתן להשתמש בהן עם תאריך ושעה כגון `date("2025-05-27")`. ניתן לבצע השוואות תאריכים באמצעות [[תחביר Bases#חשבון תאריכים|חשבון תאריכים]].

### שדות

השדות הבאים זמינים עבור תאריכים:

| שדה                | סוג      | תיאור                        |
| ------------------ | -------- | ---------------------------- |
| `date.year`        | `number` | השנה של התאריך               |
| `date.month`       | `number` | החודש של התאריך (1–12)       |
| `date.day`         | `number` | היום בחודש                   |
| `date.hour`        | `number` | השעה (0–23)                  |
| `date.minute`      | `number` | הדקה (0–59)                  |
| `date.second`      | `number` | השנייה (0–59)                |
| `date.millisecond` | `number` | המילישנייה (0–999)           |

### `date()`

`date.date(): date`

- מחזירה אובייקט תאריך ללא חלק השעה.
- דוגמה: `now().date().format("YYYY-MM-DD HH:mm:ss")` מחזירה מחרוזת כגון "2025-12-31 00:00:00"

### `format()`

`date.format(format: string): string`

- `format` הוא מחרוזת הפורמט (למשל, `"YYYY-MM-DD"`).
- מחזירה את התאריך מעוצב לפי מחרוזת פורמט של Moment.js.
- דוגמה: `date.format("YYYY-MM-DD")` מחזירה `"2025-05-27"`.

### `time()`

`date.time(): string`

- מחזירה את השעה.
- דוגמה: `now().time()` מחזירה מחרוזת כגון "23:59:59"

### `relative()`

`date.relative(): string`

- מחזירה השוואה קריאה של התאריך לתאריך ושעה הנוכחיים.
- דוגמה: `file.mtime.relative()` מחזירה ערך כגון `3 days ago`.

### `isEmpty()`

`date.isEmpty(): boolean`

- מחזירה false.

## מחרוזת

פונקציות שניתן להשתמש בהן עם רצף תווים כגון `"hello"`.

### שדות

| שדה             | סוג      | תיאור                        |
| --------------- | -------- | ---------------------------- |
| `string.length` | `number` | מספר התווים במחרוזת           |

### `contains()`

`string.contains(value: string): boolean`

- `value` הוא תת-המחרוזת לחיפוש.
- מחזירה true אם המחרוזת מכילה את `value`.
- דוגמה: `"hello".contains("ell")` מחזירה `true`.

### `containsAll()`

`string.containsAll(...values: string): boolean`

- `values` הם תת-מחרוזת אחת או יותר לחיפוש.
- מחזירה true אם המחרוזת מכילה את כל ה-`values`.
- דוגמה: `"hello".containsAll("h", "e")` מחזירה `true`.

### `containsAny()`

`string.containsAny(...values: string): boolean`

- `values` הם תת-מחרוזת אחת או יותר לחיפוש.
- מחזירה true אם המחרוזת מכילה לפחות אחד מה-`values`.
- דוגמה: `"hello".containsAny("x", "y", "e")` מחזירה `true`.

### `endsWith()`

`string.endsWith(query: string): boolean`

- `query` הוא המחרוזת לבדיקה בסוף.
- מחזירה true אם מחרוזת זו מסתיימת ב-`query`.
- דוגמה: `"hello".endsWith("lo")` מחזירה `true`.

### `isEmpty()`

`string.isEmpty(): boolean`

- מחזירה true אם למחרוזת אין תווים, או שהיא לא קיימת.
- דוגמה: `"Hello world".isEmpty()` מחזירה `false`.
- דוגמה: `"".isEmpty()` מחזירה `true`.

### `lower()`

`string.lower(): string`

- מחזירה את המחרוזת מומרת לאותיות קטנות.

### `replace()`

`string.replace(pattern: string | Regexp, replacement: string): string`

- `pattern` הוא הערך לחיפוש במחרוזת היעד.
- `replacement` הוא הערך שיחליף דפוסים שנמצאו.
- אם `pattern` הוא מחרוזת, כל המופעים של הדפוס יוחלפו.
- אם `pattern` הוא Regexp, הדגל `g` קובע אם רק המופע הראשון או כל המופעים יוחלפו.
- דוגמה: `""a:b:c:d".replace(/:/, "-")` מחזירה `"a-b,c,d"`, בעוד ש-`"a:b:c:d".replace(/:/g, "-")` מחזירה `"a-b-c-d"`.

### `repeat()`

`string.repeat(count: number): string`

- `count` הוא מספר הפעמים לחזרה על המחרוזת.
- דוגמה: `"123".repeat(2)` מחזירה `"123123"`

### `reverse()`

`string.reverse(): string`

- הופכת את המחרוזת.
- דוגמה: `"hello".reverse()` מחזירה `"olleh"`.

### `slice()`

`string.slice(start: number, end?: number): string`

- `start` הוא אינדקס ההתחלה הכולל.
- `end` הוא אינדקס הסוף האופציונלי הלא-כולל.
- מחזירה תת-מחרוזת מ-`start` (כולל) עד `end` (לא כולל).
- דוגמה: `"hello".slice(1, 4)` מחזירה `"ell"`.
- אם `end` מושמט, חותכת עד סוף המחרוזת.

### `split()`

`string.split(separator: string | Regexp, n?: number): list`

- `separator` הוא המפריד לפיצול המחרוזת.
- `n` הוא מספר אופציונלי. אם סופק, התוצאה תכלול את `n` האלמנטים הראשונים.
- מחזירה רשימה של תתי-מחרוזות.
- דוגמה: `"a,b,c,d".split(",", 3)` או `"a,b,c,d".split(/,/, 3)` מחזירה `["a", "b", "c"]`.

### `startsWith()`

`string.startsWith(query: string): boolean`

- `query` הוא המחרוזת לבדיקה בהתחלה.
- מחזירה true אם מחרוזת זו מתחילה ב-`query`.
- דוגמה: `"hello".startsWith("he")` מחזירה `true`.

### `title()`

`string.title(): string`

- ממירה את המחרוזת לאותיות כותרת (האות הראשונה של כל מילה גדולה).
- דוגמה: `"hello world".title()` מחזירה `"Hello World"`.

### `trim()`

`string.trim(): string`

- מסירה רווחים משני קצוות המחרוזת.
- דוגמה: `"  hi  ".trim()` מחזירה `"hi"`.

## מספר

פונקציות שניתן להשתמש בהן עם ערכים מספריים כגון `42`, `3.14`.

### `abs()`

`number.abs(): number`

- מחזירה את הערך המוחלט של המספר.
- דוגמה: `(-5).abs()` מחזירה `5`.

### `ceil()`

`number.ceil(): number`

- מעגלת את המספר כלפי מעלה למספר שלם הקרוב ביותר.
- דוגמה: `(2.1).ceil()` מחזירה `3`.

### `floor()`

`number.floor(): number`

- מעגלת את המספר כלפי מטה למספר שלם הקרוב ביותר.
- דוגמה: `(2.9).floor()` מחזירה `2`.

### `isEmpty()`

`number.isEmpty(): boolean`

- מחזירה true אם המספר אינו קיים.
- דוגמה: `5.isEmpty()` מחזירה `false`.

### `round()`

`number.round(digits: number): number`

- מעגלת את המספר למספר שלם הקרוב ביותר.
- באופן אופציונלי, ניתן לספק פרמטר `digits` כדי לעגל למספר ספרות עשרוניות זה.
- דוגמה: `(2.5).round()` מחזירה `3`, ו-`(2.3333).round(2)` מחזירה `2.33`.

### `toFixed()`

`number.toFixed(precision: number): string`

- `precision` הוא מספר המקומות העשרוניים.
- מחזירה מחרוזת עם המספר בסימון נקודה קבועה.
- דוגמה: `(3.14159).toFixed(2)` מחזירה `"3.14"`.

## רשימה

פונקציות שניתן להשתמש בהן עם רשימה מסודרת של אלמנטים כגון `[1, 2, 3]`.

### שדות

| שדה           | סוג      | תיאור                        |
| ------------- | -------- | ---------------------------- |
| `list.length` | `number` | מספר האלמנטים ברשימה          |

### `contains()`

`list.contains(value: any): boolean`

- `value` הוא האלמנט לחיפוש.
- מחזירה true אם הרשימה מכילה את `value`.
- דוגמה: `[1,2,3].contains(2)` מחזירה `true`.

### `containsAll()`

`list.containsAll(...values: any): boolean`

- `values` הם אלמנט אחד או יותר לחיפוש.
- מחזירה true אם הרשימה מכילה את כל ה-`values`.
- דוגמה: `[1,2,3].containsAll(2,3)` מחזירה `true`.

### `containsAny()`

`list.containsAny(...values: any): boolean`

- `values` הם אלמנט אחד או יותר לחיפוש.
- מחזירה true אם הרשימה מכילה לפחות אחד מה-`values`.
- דוגמה: `[1,2,3].containsAny(3,4)` מחזירה `true`.

### `filter()`

`list.filter(value: Boolean): list`

- מסננת את אלמנטי הרשימה על ידי קריאה לפונקציית מסנן, המשתמשת במשתנים `index` ו-`value`, ומחזירה ערך בוליאני האם האלמנט צריך להישמר.
- `value` הוא הערך של פריט ברשימה.
- `index` הוא האינדקס של הערך הנוכחי.
- דוגמה: `[1,2,3,4].filter(value > 2)` מחזירה `[3,4]`.

### `flat()`

`list.flat(): list`

- משטחת רשימות מקוננות לרשימה יחידה.
- דוגמה: `[1,[2,3]].flat()` מחזירה `[1,2,3]`.

### `isEmpty()`

`list.isEmpty(): boolean`

- מחזירה true אם לרשימה אין אלמנטים.
- דוגמה: `[1,2,3].isEmpty()` מחזירה `false`.

### `join()`

`list.join(separator: string): string`

- `separator` הוא המחרוזת להוספה בין אלמנטים.
- מחברת את כל אלמנטי הרשימה למחרוזת יחידה.
- דוגמה: `[1,2,3].join(",")` מחזירה `"1,2,3"`.

### `map()`

`list.map(value: Any): list`

- ממירה כל אלמנט ברשימה על ידי קריאה לפונקציית המרה, המשתמשת במשתנים `index` ו-`value`, ומחזירה את הערך החדש שיוצב ברשימה.
- `value` הוא הערך של פריט ברשימה.
- `index` הוא האינדקס של הערך הנוכחי.
- דוגמה: `[1,2,3,4].map(value + 1)` מחזירה `[2,3,4,5]`.

### `reduce()`

`list.reduce(expression: Any, acc: Any): Any`

- מצמצמת את אלמנטי הרשימה לערך יחיד על ידי הרצת ביטוי עבור כל אלמנט. הביטוי יכול להשתמש במשתנים `index`, `value` ו-`acc` (הצובר), וצריך להחזיר את ערך הצובר הבא.
- `expression` מוערך עבור כל אלמנט ברשימה.
- `value` הוא הערך של הפריט הנוכחי ברשימה.
- `index` הוא האינדקס של הפריט הנוכחי.
- `acc` הוא הערך המצטבר עד כה.
- דוגמה (סכום): `[1,2,3].reduce(acc + value, 0)` מחזירה `6`.
- דוגמה (מקסימום): `values.filter(value.isType("number")).reduce(if(acc == null || value > acc, value, acc), null)` מחזירה את המספר הגדול ביותר, או `null` אם אין.

### `reverse()`

`list.reverse(): list`

- הופכת את הרשימה במקום.
- דוגמה: `[1,2,3].reverse()` מחזירה `[3,2,1]`.

### `slice()`

`list.slice(start: number, end?: number): list`

- `start` הוא אינדקס ההתחלה הכולל.
- `end` הוא אינדקס הסוף האופציונלי הלא-כולל.
- מחזירה עותק רדוד של חלק מהרשימה מ-`start` (כולל) עד `end` (לא כולל).
- דוגמה: `[1,2,3,4].slice(1,3)` מחזירה `[2,3]`.
- אם `end` מושמט, חותכת עד סוף הרשימה.

### `sort()`

`list.sort(): list`

- ממיינת אלמנטי רשימה מהקטן לגדול.
- דוגמה: `[3, 1, 2].sort()` מחזירה `[1, 2, 3]`.
- דוגמה: `["c", "a", "b"].sort()` מחזירה `["a", "b", "c"]`.

### `unique()`

`list.unique(): list`

- מסירה אלמנטים כפולים.
- דוגמה: `[1,2,2,3].unique()` מחזירה `[1,2,3]`.

## קישור

פונקציות שניתן להשתמש בהן על קישור. ניתן ליצור קישורים מקובץ (`file.asLink()`) או מנתיב (`link("path")`).

### `asFile()`

`link.asFile(): file`

- מחזירה אובייקט קובץ אם הקישור מפנה לקובץ מקומי תקף.
- דוגמה: `link("[[filename]]").asFile()`

### `linksTo()`

`link.linksTo(file): boolean`

- מחזירה האם לקובץ המיוצג על ידי ה-`link` יש קישור ל-`file`.

## קובץ

פונקציות שניתן להשתמש בהן עם קובץ בכספת.

### שדות

השדות הבאים זמינים עבור קבצים:

| שדה               | סוג      | תיאור                                                   |
| ----------------- | -------- | ------------------------------------------------------- |
| `file.name`       | `string` | שם הקובץ.                                               |
| `file.basename`   | `string` | שם הקובץ ללא סיומת.                                      |
| `file.path`       | `string` | הנתיב המלא לקובץ, יחסית לשורש הכספת.                     |
| `file.folder`     | `string` | הנתיב המלא לתיקיית האב.                                 |
| `file.ext`        | `string` | סיומת הקובץ.                                            |
| `file.size`       | `number` | גודל הקובץ, בבתים.                                      |
| `file.properties` | `object` | מאפייני ההערה של קובץ זה.                                |
| `file.tags`       | `list`   | התגיות של קובץ זה. כולל תגיות מוטבעות.                    |
| `file.links`      | `list`   | הקישורים הפנימיים בתוך קובץ זה.                          |
| `file.ctime`      | `date`   | חותמת זמן של מועד יצירת הקובץ.                           |
| `file.mtime`      | `date`   | חותמת זמן של מועד שינוי הקובץ האחרון.                     |

### `asLink()`

`file.asLink(display?: string): Link`

- `display` טקסט תצוגה אופציונלי לקישור.
- מחזירה אובייקט קישור שמוצג כקישור פעיל.
- דוגמה: `file.asLink()`

### `hasLink()`

`file.hasLink(otherFile: file | string): boolean`

- `otherFile` הוא אובייקט קובץ אחר או נתיב מחרוזת לבדיקה.
- מחזירה true אם ל-`file` יש קישור ל-`otherFile`.
- דוגמה: `file.hasLink(otherFile)` מחזירה `true` אם יש קישור מ-`file` ל-`otherFile`.

### `hasProperty()`

`file.hasProperty(name: string): boolean`

- מחזירה true אם להערה יש את מאפיין הקובץ הנתון.

### `hasTag()`

`file.hasTag(...values: string): boolean`

- `values` הם שם תגית אחד או יותר.
- מחזירה true אם לקובץ יש אחת מהתגיות ב-`values`.
- דוגמה: `file.hasTag("tag1", "tag2")` מחזירה `true` אם לקובץ יש תגית `#tag1` או `#tag2`. זה כולל גם [[תגים#תגיות מקוננות|תגיות מקוננות]], כגון `#tag1/a` או `#tag2/b`.

### `inFolder()`

`file.inFolder(folder: string): boolean`

- `folder` הוא שם התיקייה לבדיקה.
- מחזירה true אם הקובץ נמצא בתיקייה המצוינת או באחת מתיקיות המשנה שלה.
- דוגמה: `file.inFolder("notes")` מחזירה `true`.

## אובייקט

פונקציות שניתן להשתמש בהן עם אוסף של זוגות מפתח-ערך כגון `{"a": 1, "b": 2}`.

### `isEmpty()`

`object.isEmpty(): boolean`

- מחזירה true אם לאובייקט אין מאפיינים משלו.
- דוגמה: `{}.isEmpty()` מחזירה `true`.

### `keys()`

`object.keys(): list`

- מחזירה רשימה המכילה את המפתחות של האובייקט.

### `values()`

`object.values(): list`

- מחזירה רשימה המכילה את הערכים של האובייקט.

## ביטוי רגולרי

פונקציות שניתן להשתמש בהן עם דפוס ביטוי רגולרי. דוגמה: `/abc/`.

### `matches()`

`regexp.matches(value: string): boolean`

- `value` הוא המחרוזת לבדיקה.
- מחזירה true אם הביטוי הרגולרי תואם את `value`.
- דוגמה: `/abc/.matches("abcde")` מחזירה `true`.
