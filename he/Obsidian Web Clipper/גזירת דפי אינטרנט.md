לאחר שתתקין את תוסף הדפדפן [[מבוא ל-Obsidian Web Clipper|Web Clipper]], תוכל לגשת אליו במספר דרכים, בהתאם לדפדפן שלך:

1. סמל Obsidian בסרגל הכלים של הדפדפן.
2. מקשי קיצור, להפעלת התוסף מהמקלדת.
3. תפריט הקשר, על ידי לחיצה ימנית על דף האינטרנט שבו אתה מבקר.

כדי לשמור דף ל-Obsidian לחץ על כפתור **הוסף ל-Obsidian**.

## לכידת דף

כאשר אתה פותח את התוסף, Web Clipper מחלץ נתונים מדף האינטרנט הנוכחי בהתאם להגדרות ב[[תבניות|תבנית]] שלך. תוכל ליצור תבניות משלך, ולהתאים את הפלט באמצעות [[משתנים]] ו[[מסננים]].

כברירת מחדל Web Clipper מנסה לחלץ בצורה חכמה רק את תוכן המאמר העיקרי, תוך החרגת אלמנטים אחרים בדף. עם זאת, תוכל לדרוס התנהגות זו בדרכים הבאות:

- אם קיימת תבנית מותאמת אישית, הוא משתמש בתבנית שלך.
- אם קיימת בחירה, הוא משתמש בבחירה. תוכל להשתמש ב-`Ctrl/Cmd+A` כדי לבחור את כל הדף.
- אם קיימות [[הדגשת דפי אינטרנט|הדגשות]], הוא משתמש בהדגשות.

## הורדת תמונות

תמונות אינן מורדות אוטומטית כאשר אתה משתמש ב-Web Clipper. במקום זאת, התמונות מקשרות לכתובת ה-URL שלהן באינטרנט. זה חוסך מקום בכספת שלך אך משמעות הדבר שהתמונות לא יהיו נגישות במצב לא מקוון, או אם כתובת ה-URL תפסיק לעבוד.

תוכל להוריד תמונות עבור כל קובץ ב-Obsidian באמצעות [[לוח פקודות|פקודה]] בשם **הורד קבצים מצורפים עבור הקובץ הנוכחי**. ניתן גם למפות פקודה זו לקיצור מקלדת ב-Obsidian.

## מקשי קיצור

Web Clipper כולל קיצורי מקלדת שבהם תוכל להשתמש כדי לייעל את זרימת העבודה שלך. כדי לשנות מיפויי מקשים עבור אל **הגדרות Web Clipper** → **כללי** ועקוב אחר ההוראות עבור הדפדפן שלך. ניתן לשנות מיפויים עבור כל הדפדפנים למעט Safari שאינו תומך בעריכת מקשי קיצור.

| פעולה                    | macOS          | Windows/Linux   |
| ------------------------ | -------------- | --------------- |
| פתח גוזר                | `Cmd+Shift+O`  | `Ctrl+Shift+O`  |
| גזירה מהירה             | `Opt+Shift+O`  | `Alt+Shift+O`   |
| הפעל/כבה מסמן           | `Opt+Shift+H`  | `Alt+Shift+H`   |

## פונקציונליות הממשק

ממשק Web Clipper מחולק לארבעה חלקים:

1. **כותרת עליונה** שבה תוכל להחליף תבניות, להפעיל [[הדגשת דפי אינטרנט|הדגשה]], [[קורא|מצב קריאה]], ולגשת להגדרות.
2. **מאפיינים** מציג את ה[[מאפיינים|מטא-נתונים]] שחולצו מהדף שיישמרו כ[[מאפיינים]] ב-Obsidian.
3. **תוכן הפתק** שיישמר ב-Obsidian.
4. **כותרת תחתונה** מאפשרת לך לבחור את הכספת והתיקייה, ולהוסיף ל-Obsidian.

פונקציונליות הכותרת העליונה כוללת:

- תפריט נפתח **תבנית** למעבר בין [[תבניות|התבניות]] השמורות שהוספת בהגדרות Web Clipper.
- כפתור **עוד (...)** להצגת משתני עמוד שתוכל להשתמש בהם בתבניות.
- כפתור **מסמן** להפעלת [[הדגשת דפי אינטרנט|הדגשה]].
- כפתור **גלגל שיניים** לפתיחת הגדרות Web Clipper.

פונקציונליות הכותרת התחתונה כוללת:

- כפתור **הוסף ל-Obsidian** לשמירת נתונים ב-Obsidian.
- תפריט נפתח **כספת** למעבר בין כספות שמורות שהוספת בהגדרות Web Clipper.
- שדה **תיקייה** להגדרת התיקייה שבה יישמר הקובץ.
- **מתרגם** להרצת [[פרש דפי אינטרנט|הנחיות בשפה טבעית]] על הדף.
