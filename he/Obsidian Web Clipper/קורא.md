[[מבוא ל-Obsidian Web Clipper|Web Clipper]] כולל מצב קריאה מובנה בשם קורא שמסיר עומסים מדפי אינטרנט, ומציג רק את תוכן המאמר העיקרי בפורמט נקי וקריא.

## הפעלת מצב קריאה

ניתן להפעיל את הקורא במספר דרכים:

- **סמל הספר** בכותרת חלונית התוסף.
- קיצור מקלדת `Alt+Shift+R`, שניתן להתאמה אישית בהגדרות התוסף.
- לחיצה חוזרת על מצב קורא תשחזר את הדף המקורי.

## כיצד זה עובד

הקורא מחלץ את התוכן העיקרי מהדף. הדף המקורי מוחלף בתצוגת קריאה נקייה הכוללת:

- כותרת המאמר, המחבר, תאריך הפרסום ודומיין המקור.
- תוכן המאמר העיקרי, עם שימור של תמונות ועיצוב.
- מתאר בסרגל הצד עבור מאמרים עם כותרות מרובות.
- הדגשת תחביר עבור בלוקי קוד.
- חלונות קופצים של הערות שוליים המוצגים בתוך השורה בלחיצה.

## התאמה אישית של תצוגת הקריאה

מצב קורא כולל סרגל הגדרות מוסתר שניתן להציג על ידי ריחוף בפינה הימנית העליונה של הדף, עם פקדים להתאמה אישית של חווית הקריאה.

| הגדרה            | תיאור                                                                                         |
| ----------------- | --------------------------------------------------------------------------------------------- |
| גודל גופן        | התאמת גודל הטקסט.                                                                            |
| רוחב שורה        | רוחב מרבי של אזור התוכן.                                                                     |
| גובה שורה        | ריווח בין שורות.                                                                              |
| ערכת&nbsp;צבעים  |                                                                                               |
| ערכת נושא        | אוטומטי עוקב אחר העדפת המערכת שלך. או בחר להשתמש תמיד במצב בהיר או כהה.                     |
