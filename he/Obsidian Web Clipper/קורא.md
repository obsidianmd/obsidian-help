[[מבוא ל-Obsidian Web Clipper|Web Clipper]] כולל מצב קריאה מובנה בשם **קורא** שמסיר עומסים מדפי אינטרנט, ומציג רק את תוכן המאמר העיקרי בפורמט נקי וקריא.

## הפעלת מצב קריאה

ניתן להפעיל את הקורא במספר דרכים:

- ![[lucide-book-icon.svg#icon]] **סמל הספר** בחלונית התוסף.
- **קיצור מקלדת** שניתן להתאמה אישית בהגדרות התוסף.
- **תפריט הקשר** על ידי לחיצה ימנית על דף האינטרנט שבו אתם מבקרים.

ניתן גם להגדיר את כפתור התוסף ![[obsidian.svg#icon]] לפתוח את הקורא בלחיצה אחת. בהגדרות Web Clipper עברו אל **General** → **Behavior** → **Open behavior** ובחרו **Reader**.

כל האפשרויות לעיל גם מאפשרות לצאת מהקורא ולשחזר את הדף המקורי.

## כיצד זה עובד

הקורא מחלץ את התוכן העיקרי מהדף. הדף המקורי מוחלף בתצוגת קריאה נקייה הכוללת:

- כותרת המאמר, המחבר, תאריך הפרסום ודומיין המקור.
- תוכן המאמר העיקרי, עם שימור של תמונות ועיצוב.
- מתאר בסרגל הצד עבור מאמרים עם כותרות מרובות.
- הדגשת תחביר עבור בלוקי קוד.
- חלונות קופצים של הערות שוליים המוצגים בתוך השורה בלחיצה.

## התאמה אישית של תצוגת הקריאה

התאימו את חוויית הקורא שלכם בהגדרות התוסף. חלק מהאפשרויות זמינות גם בתוך מצב הקורא על ידי לחיצה על הסמל ![[lucide-case-sensitive.svg#icon]].

| הגדרה             | תיאור                                    |
| ------------------ | ---------------------------------------- |
| גופן              | הוספת גופנים המותקנים במערכת שלכם.       |
| גודל גופן         | התאמת גודל הטקסט.                        |
| גובה&nbsp;שורה    | ריווח בין שורות.                          |
| רוחב&nbsp;שורה    | רוחב מרבי של אזור התוכן.                 |
| מראה              | בחירת מראה בהיר או כהה.                  |
| ערכת נושא         | בחירת הצבעים.                            |
| CSS מותאם אישית   | CSS נוסף המוחל על הדף.                   |
