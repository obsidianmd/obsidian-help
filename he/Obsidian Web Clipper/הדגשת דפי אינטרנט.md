[[מבוא ל-Obsidian Web Clipper|Web Clipper]] מאפשר לך להדגיש טקסט בדפי אינטרנט, ולבחור את האלמנטים שברצונך לשמור ל-Obsidian. ההדגשות שלך נשמרות, כך שתוכל לחזור אליהן כשתבקר שוב בדף.

ניתן [[גזירת דפי אינטרנט|ללכוד]] הדגשות ולשמור אותן ל-Obsidian כשאתה פותח את התוסף.

## הפעלת המסמן

ניתן להפעיל את ההדגשה במספר דרכים, בהתאם לדפדפן שלך:

- סמל המסמן בחלונית התוסף.
- מקשי קיצור, להפעלת התוסף מהמקלדת.
- תפריט הקשר, על ידי לחיצה ימנית על דף האינטרנט שבו אתה מבקר.

לאחר שההדגשה מופעלת, תוכל לבחור טקסט, תמונות ואלמנטים שברצונך להדגיש.

## הגדרות המסמן

ניתן לשנות את התנהגות המסמן על ידי מעבר להגדרות Web Clipper. כאן תוכל גם לייצא את ההדגשות שלך לקובץ `.json`.

ישנן שלוש אפשרויות להכנסת הדגשות להערה הגזורה שלך באמצעות ה[[משתנים|משתנה]] `{{content}}`:

- **סמן תוכן בדף** — מוסיף הדגשות ישירות לטקסט עם ה[[Obsidian Flavored Markdown|תחביר]] `==highlight==`.
- **החלף תוכן פתק** — מחזיר רשימת הדגשות, ללא תוכן הדף.
- **אל תעשה כלום** — מחזיר את התוכן המקורי ללא הדגשות.

ניתן להוסיף הדגשות ישירות לתבנית שלך באמצעות המשתנה `{{highlights}}`, לדוגמא:

```
{{highlights|map: item => item.text|join:"\n\n"}}
```
