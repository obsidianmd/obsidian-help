Obsidian משתמש בקבצי Markdown בפורמט `.md` כפורמט העיקרי להערות. זה מקל על ייבוא קבצי Markdown אל Obsidian. אפליקציות רבות יכולות לערוך קבצי Markdown או לייצא לפורמט זה.

ישנן מספר דרכים להוסיף קבצי Markdown לכספת ה-Obsidian שלך:

1. גרור ושחרר קבצים או תיקיות לחלון [[סייר קבצים|סייר הקבצים]]
2. השתמש בסייר הקבצים של המערכת שלך (למשל סייר Windows או Finder ב-macOS), והעבר את הקבצים ישירות לתיקיית כספת ה-Obsidian.

### המרה מגרסאות אחרות של Markdown

Obsidian משתמש ב-[[Obsidian Flavored Markdown]]. חלק מהאפליקציות משתמשות בתחביר Markdown מעט שונה. תוכל להשתמש ב-[[יבואן פורמט Markdown]] כדי להמיר את הקבצים שלך לתחביר ה-Markdown של Obsidian.
