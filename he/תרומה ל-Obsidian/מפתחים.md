אם אתה מפתח, תוכל לתרום ל-Obsidian במגוון דרכים.

### תוספים וערכות נושא

תוכל לפתח [[תוספים קהילתיים]] ו[[ערכות נושא]] באמצעות Typescript ו-CSS. למד עוד בכך שתעבור ל[תיעוד המפתחים של Obsidian](https://docs.obsidian.md).

### פרויקטי קוד פתוח

Obsidian מפתחת ומתחזקת מספר פרויקטי קוד פתוח. תוכל לשלוח בקשות משיכה (pull requests) עם תיקוני באגים ושיפורים למאגרי GitHub הבאים:

- [JSON Canvas](https://github.com/obsidianmd/jsoncanvas) ‏(MIT), פורמט קובץ פתוח לנתוני Canvas אינסופי.
- [Obsidian Importer](https://github.com/obsidianmd/obsidian-importer) ‏(MIT), תוסף שממיר נתונים ממגוון רחב של פורמטים לקבצי Markdown בטקסט רגיל.
- [Obsidian Maps](https://github.com/obsidianmd/obsidian-maps) ‏(MIT), תוסף שמציג מפות ב-Obsidian Bases.
- [Obsidian Web Clipper](https://github.com/obsidianmd/obsidian-clipper) ‏(MIT), תוסף דפדפן להדגשה ולכידה של דפי אינטרנט לקבצי Markdown בטקסט רגיל.
- [Defuddle](https://github.com/kepano/defuddle) ‏(MIT), מנתח HTML-ל-Markdown המשמש את Obsidian Web Clipper.

### תיעוד

כדי לסייע בשיפור [תיעוד המפתחים](https://docs.obsidian.md/Home) שלנו, שלח בקשת משיכה למאגר [obsidian-developer-docs](https://github.com/obsidianmd/obsidian-developer-docs).

כדי לסייע בשיפור תיעוד זה, שלח בקשת משיכה למאגר [obsidian-help](https://github.com/obsidianmd/obsidian-help).
