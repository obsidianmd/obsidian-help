[[מבוא ל-Obsidian Sync|Obsidian Sync]] מציע לקוח ללא ממשק לסנכרון כספות ללא שימוש באפליקציית שולחן העבודה. שימושי לצינורות CI, סוכנים, ותהליכי עבודה אוטומטיים. סנכרנו את השינויים האחרונים או שמרו על קבצים מעודכנים באופן רציף.

התקינו את [[Obsidian Headless]] **(בטא פתוחה)** כדי לתקשר עם [[מבוא ל-Obsidian Sync|Obsidian Sync]] משורת הפקודה ללא אפליקציית שולחן העבודה של Obsidian. סנכרון ללא ממשק משתמש באותן [[אבטחה ופרטיות|הגנות הצפנה ופרטיות]] כמו אפליקציית שולחן העבודה, כולל הצפנה מקצה לקצה.

## התחלה מהירה

> [!error] גבו את הנתונים שלכם לפני שאתם מתחילים
> 1. תמיד גבו את הנתונים שלכם לפני שאתם מתחילים למקרה שמשהו בלתי צפוי יקרה.
> 2. אל תשתמשו *גם* בסנכרון אפליקציית שולחן העבודה וגם בסנכרון ללא ממשק על אותו מכשיר, כיוון שזה עלול לגרום לקונפליקטים בנתונים. השתמשו רק בשיטת סנכרון אחת לכל מכשיר.

התקינו את [[Obsidian Headless|Obsidian Headless]] **(בטא פתוחה)**:

```shell
npm install -g obsidian-headless
```

עליכם להחזיק [[מסלולים ומגבלות אחסון|מנוי Obsidian Sync]] פעיל.

```shell
# התחברות
ob login

# הצגת הכספות המרוחקות שלך
ob sync-list-remote

# הגדרת כספת לסנכרון
cd ~/vaults/my-vault
ob sync-setup --vault "My Vault"

# הרצת סנכרון חד-פעמי
ob sync

# הרצת סנכרון רציף (מאזין לשינויים)
ob sync --continuous
```

## פקודות

### `ob sync-list-remote`

הצגת כל הכספות המרוחקות הזמינות לחשבון שלכם, כולל כספות משותפות.

### `ob sync-list-local`

הצגת כספות מוגדרות מקומית והנתיבים שלהן.

### `ob sync-create-remote`

יצירת כספת מרוחקת חדשה.

```
ob sync-create-remote --name "Vault Name" [--encryption <standard|e2ee>] [--password <password>] [--region <region>]
```

| אפשרות | תיאור |
| --- | --- |
| `--name` | שם הכספת (נדרש) |
| `--encryption` | `standard` להצפנה מנוהלת, `e2ee` להצפנה מקצה לקצה |
| `--password` | סיסמת הצפנה מקצה לקצה (תתבקשו אם לא צוינה) |
| `--region` | [[אזורי סנכרון|אזור]] שרת (אוטומטי אם לא צוין) |

### `ob sync-setup`

הגדרת סנכרון בין כספת מקומית לכספת מרוחקת.

```
ob sync-setup --vault <id-or-name> [--path <local-path>] [--password <password>] [--device-name <name>] [--config-dir <name>]
```

| אפשרות | תיאור |
| --- | --- |
| `--vault` | מזהה או שם כספת מרוחקת (נדרש) |
| `--path` | תיקייה מקומית (ברירת מחדל: התיקייה הנוכחית) |
| `--password` | סיסמת הצפנה מקצה לקצה (תתבקשו אם לא צוינה) |
| `--device-name` | שם התקן המוצג ב[[היסטוריית גרסאות|היסטוריית גרסאות הסנכרון]] |
| `--config-dir` | שם [[תיקיית תצורה|תיקיית תצורה]] (ברירת מחדל: `.obsidian`) |

### `ob sync`

הרצת סנכרון עבור כספת מוגדרת.

```
ob sync [--path <local-path>] [--continuous]
```

| אפשרות | תיאור |
| --- | --- |
| `--path` | נתיב כספת מקומית (ברירת מחדל: התיקייה הנוכחית) |
| `--continuous` | הרצה רציפה, מאזינה לשינויים |

### `ob sync-config`

הצגה או שינוי [[הגדרות סנכרון וסנכרון סלקטיבי|הגדרות סנכרון]] עבור כספת. הריצו ללא אפשרויות כדי להציג את התצורה הנוכחית.

```
ob sync-config [--path <local-path>] [options]
```

| אפשרות                | תיאור                                                                                                                                                                                                          |
| --------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `--path`              | נתיב כספת מקומית (ברירת מחדל: התיקייה הנוכחית)                                                                                                                                                                  |
| `--mode`              | מצב סנכרון: `bidirectional` (ברירת מחדל), `pull-only` (הורדה בלבד, התעלמות משינויים מקומיים), או `mirror-remote` (הורדה בלבד, החזרת שינויים מקומיים)                                                              |
| `--conflict-strategy` | `merge` או `conflict`                                                                                                                                                                                          |
| `--file-types`        | סוגי קבצים מצורפים לסנכרון: `image`, `audio`, `video`, `pdf`, `unsupported` (מופרדים בפסיקים, ריק לניקוי)                                                                                                    |
| `--configs`           | קטגוריות תצורה לסנכרון: `app`, `appearance`, `appearance-data`, `hotkey`, `core-plugin`, `core-plugin-data`, `community-plugin`, `community-plugin-data` (מופרדים בפסיקים, ריק להשבתת סנכרון תצורה) |
| `--excluded-folders`  | תיקיות להחרגה (מופרדות בפסיקים, ריק לניקוי)                                                                                                                                                           |
| `--device-name`       | שם התקן לזיהוי לקוח זה בהיסטוריית גרסאות הסנכרון                                                                                                                                                                |
| `--config-dir`        | שם תיקיית תצורה (ברירת מחדל: `.obsidian`)                                                                                                                                                                   |

### `ob sync-status`

הצגת סטטוס סנכרון ותצורה עבור כספת.

```
ob sync-status [--path <local-path>]
```

### `ob sync-unlink`

ניתוק כספת מהסנכרון והסרת פרטי התחברות מאוחסנים.

```
ob sync-unlink [--path <local-path>]
```

## מודולים מקוריים

Obsidian Headless כולל תוסף מקורי מוכן מראש להגדרת זמן יצירת קובץ (birthtime) ב-Windows וב-macOS. זה שומר על חותמות זמן יצירה מקוריות בעת הורדת קבצים מהשרת.

התוסף מכוון ל-N-API גרסה 3, כך שהבינאריים המקומפלים יציבים ב-ABI ועובדים בגרסאות Node.js שונות ללא הידור מחדש.

ב-Linux, birthtime אינו נתמך — התוסף אינו כלול והסנכרון פועל כרגיל בלעדיו.

בינאריים מוכנים מראש כלולים עבור:

- `win32-x64`
- `win32-arm64`
- `win32-ia32`
- `darwin-x64`
- `darwin-arm64`
