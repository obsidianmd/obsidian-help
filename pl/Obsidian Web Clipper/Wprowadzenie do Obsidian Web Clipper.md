Obsidian Web Clipper to darmowe rozszerzenie przeglądarki, które pozwala wyróżniać strony i zapisywać treści internetowe do Twojego skarbca.

## Instalacja Web Clipper

Pobierz rozszerzenie Obsidian Web Clipper z oficjalnego katalogu dla Twojej przeglądarki:

- [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
	- Dla Chrome, Brave, Arc, Orion i innych przeglądarek opartych na Chromium.
- [Firefox Add-Ons](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
	- Dla Firefox i Firefox Mobile.
- [Safari Extensions](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
	- Dla macOS, iOS i iPadOS.
- [Edge Add-Ons](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
	- Dla Microsoft Edge.

## Jak korzystać z Web Clipper

- [[Wycinanie stron internetowych]]
	- Rozpocznij i dowiedz się, jak zapisywać treści ze stron.
- [[Wyróżnianie stron internetowych|Zakreślacz]]

- [[Czytnik|Czytnik]]

- [[Interpretowanie stron internetowych|Tłumacz]]
	- Używaj zapytań w języku naturalnym, aby przechwytywać i modyfikować dane na stronie.
- [[Rozwiązywanie problemów z Web Clipper|Rozwiązywanie problemów]]
	- Rozwiąż typowe problemy z Web Clipper.
- [[Obsidian Web Clipper/Szablony|Szablony]]
	- Twórz niestandardowe szablony dla konkretnych stron internetowych.
- [[Zmienne]]
	- Automatycznie wypełniaj dane strony w swoim szablonie.
- [[Filtry]]
	- Modyfikuj wynik zmiennych szablonu.
- [[Logika]]

## Prywatność

Obsidian Web Clipper zapisuje treści lokalnie w Twoim skarbcu Obsidian i stosuje się do naszej [polityki prywatności](https://obsidian.md/privacy). Twoje dane nie są zbierane i nie gromadzimy żadnych metryk użytkowania. Kod jest [otwartym oprogramowaniem (open-source)](https://github.com/obsidianmd/obsidian-clipper) i dostępny do audytu.
