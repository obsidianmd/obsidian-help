Obsidian używa plików Markdown `.md` jako głównego formatu notatek. Dzięki temu łatwo jest importować pliki Markdown do Obsidian. Wiele aplikacji może edytować pliki Markdown lub eksportować do tego formatu.

Istnieje kilka sposobów na dodanie plików Markdown do skarbca Obsidian:

1. Przeciągnij i upuść pliki lub foldery do okna [[Eksplorator plików|Eksploratora plików]]
2. Użyj systemowej przeglądarki plików (np. Eksploratora Windows lub Findera na macOS) i przenieś pliki bezpośrednio do folderu skarbca Obsidian.

### Konwersja z innych odmian Markdown

Obsidian używa [[Obsidian Flavored Markdown]]. Niektóre aplikacje stosują nieco inną składnię Markdown. Możesz użyć [[Konwerter formatowania|Konwertera formatowania]], aby przekonwertować swoje pliki na składnię Markdown używaną przez Obsidian.
