Ta strona zawiera listę domyślnych skrótów klawiszowych do nawigacji i edycji tekstu w Obsidian. Te skróty są dostarczane przez system operacyjny lub framework, na którym zbudowany jest Obsidian, i nie można ich dostosowywać w Obsidian.

Aby poznać konfigurowalne skróty klawiszowe dla poleceń Obsidian, zobacz [[Skróty klawiszowe]].

## Skróty dla Windows i Linux

### Typowe czynności

| Czynność | Skrót |
|-|-|
| Kopiuj | `Ctrl+C` |
| Wytnij | `Ctrl+X` |
| Wklej | `Ctrl+V` |
| Wklej bez formatowania | `Ctrl+Shift+V` |
| Cofnij | `Ctrl+Z` |
| Ponów | `Ctrl+Shift+Z` lub `Ctrl+Y` |
| Kopiuj akapit | `Ctrl+C` (bez zaznaczonego tekstu) |
| Wytnij akapit | `Ctrl+X` (bez zaznaczonego tekstu) |

### Edycja tekstu

| Czynność | Skrót |
|-|-|
| Wstaw nową linię | `Enter` |
| Usuń poprzedni znak | `Backspace` |
| Usuń następny znak | `Delete` |
| Usuń poprzednie słowo | `Ctrl+Backspace` |
| Usuń następne słowo | `Ctrl+Delete` |
| Usuń bieżącą linię | `Ctrl+Shift+K` (bez zaznaczonego tekstu) |

### Nawigacja po tekście

| Czynność | Skrót |
|-|-|
| Przesuń kursor o jeden znak | `Left/→` |
| Przesuń kursor na początek poprzedniego słowa | `Ctrl+←` |
| Przesuń kursor na koniec następnego słowa | `Ctrl+→` |
| Przesuń kursor na początek bieżącej linii | `Home` |
| Przesuń kursor na koniec bieżącej linii | `End` |
| Przesuń kursor do poprzedniej linii | `↑` |
| Przesuń kursor do następnej linii | `↓` |
| Przesuń kursor na początek notatki | `Ctrl+Home` |
| Przesuń kursor na koniec notatki | `Ctrl+End` |
| Przesuń kursor o jedną stronę w górę | `Page up` |
| Przesuń kursor o jedną stronę w dół | `Page down` |

### Zaznaczanie tekstu

| Czynność | Skrót |
|-|-|
| Uprość zaznaczenie | `Escape` |
| Zaznacz wszystko | `Ctrl+A` |
| Rozszerz zaznaczenie o jeden znak | `Shift+Left/→` |
| Rozszerz zaznaczenie do początku poprzedniego słowa | `Ctrl+Shift+←` |
| Rozszerz zaznaczenie do końca następnego słowa | `Ctrl+Shift+→` |
| Rozszerz zaznaczenie do początku bieżącej linii | `Shift+Home` |
| Rozszerz zaznaczenie do końca bieżącej linii | `Shift+End` |
| Rozszerz zaznaczenie do początku notatki | `Ctrl+Shift+Home` |
| Rozszerz zaznaczenie do końca notatki | `Ctrl+Shift+End` |
| Rozszerz zaznaczenie o jedną stronę w górę | `Shift+Page up` |
| Rozszerz zaznaczenie o jedną stronę w dół | `Shift+Page down` |

|-|-|

| Wytnij akapit | `Cmd+X` (bez zaznaczonego tekstu) |
