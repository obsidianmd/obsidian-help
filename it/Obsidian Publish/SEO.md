I siti [[Introduction to Obsidian Publish|Obsidian Publish]] sono automaticamente ottimizzati per i motori di ricerca e le [[Social media link previews|schede di condivisione social]]. Puoi anche personalizzare i metadati delle pagine con descrizioni, permalink e immagini.

Nota che puoi non consentire l'indicizzazione nei motori di ricerca nelle [[Manage sites#Opzioni sito|opzioni sito]].

### Google Search Console

Per monitorare la SEO con Google, registra il tuo sito su [Google Search Console](https://search.google.com/search-console). Puoi anche seguire i passaggi consigliati per [[Analytics|configurare Google Analytics]].

### Sitemap

La sitemap del tuo sito Publish è disponibile all'indirizzo `/sitemap.xml`, ad esempio:
https://help.obsidian.md/sitemap.xml

La sitemap è disponibile anche come feed RSS all'indirizzo `/rss.xml`:
https://help.obsidian.md/rss.xml

### Permalink

Considera l'uso dei [[Permalinks|Permalink]] per definire URL permanenti sul tuo sito.

### Metadati

I metadati delle pagine possono essere personalizzati utilizzando le [[Properties#Proprietà per Obsidian Publish|Proprietà]], che possono essere utili per le [[Social media link previews|schede di condivisione social]].

| Proprietà     | Descrizione                                                                                                       |
| ------------- | ----------------------------------------------------------------------------------------------------------------- |
| `publish`     | Vedi [[Publish your content#Selezionare automaticamente le note da pubblicare\|Selezionare automaticamente le note da pubblicare]]. |
| `permalink`   | Vedi [[Permalinks\|Permalink]].                                                                                   |
| `description` | Vedi [[Social media link previews#Descrizione\|Descrizione]].                                                     |
| `image`       | Vedi [[Social media link previews#Immagine\|Immagine]].                                                           |
| `cover`       | Vedi [[Social media link previews#Immagine\|Immagine]].                                                           |
