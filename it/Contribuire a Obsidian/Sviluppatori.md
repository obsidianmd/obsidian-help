Se sei uno sviluppatore puoi contribuire a Obsidian in vari modi.

### Plugin e temi

Puoi sviluppare [[Plugin della comunità|Plugin della community]] e [[Temi]] utilizzando Typescript e CSS. Scopri di più visitando la [Documentazione per sviluppatori di Obsidian](https://docs.obsidian.md).

### Progetti open source

Obsidian sviluppa e mantiene diversi progetti open source. Puoi inviare pull request con correzioni di bug e miglioramenti ai seguenti repository GitHub:

- [JSON Canvas](https://github.com/obsidianmd/jsoncanvas) (MIT), un formato di file aperto per dati di canvas infinito.
- [Obsidian Importer](https://github.com/obsidianmd/obsidian-importer) (MIT), un plugin che converte dati da un'ampia varietà di formati in file Markdown in testo normale.
- [Obsidian Maps](https://github.com/obsidianmd/obsidian-maps) (MIT), un plugin che visualizza mappe in Obsidian Base.
- [Obsidian Web Clipper](https://github.com/obsidianmd/obsidian-clipper) (MIT), un'estensione del browser per evidenziare e catturare pagine web in file Markdown in testo normale.
- [Defuddle](https://github.com/kepano/defuddle) (MIT), il parser da HTML a Markdown utilizzato da Obsidian Web Clipper.

### Documentazione

Per contribuire a migliorare la nostra [documentazione per sviluppatori](https://docs.obsidian.md/Home), invia una pull request al repository [obsidian-developer-docs](https://github.com/obsidianmd/obsidian-developer-docs).

Per contribuire a migliorare questa documentazione, invia una pull request al repository [obsidian-help](https://github.com/obsidianmd/obsidian-help).

### Community

Il canale [plugin-dev](https://discord.com/channels/686053708261228577/840286264964022302) sul nostro server Discord ufficiale è il luogo dove puoi discutere lo sviluppo di plugin e aiutare altri sviluppatori con i loro plugin.
