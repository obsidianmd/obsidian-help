Puoi gestire file e cartelle in diversi modi, usando le [[Hotkeys|scorciatoie da tastiera]], i [[Command palette|comandi]], o l'[[File explorer|esplora file]].

## Creare una nuova nota

Per creare un nuovo file:

1. Premi `Ctrl+N` (o `Cmd+N` su macOS).
2. Inserisci il nome della nota e poi premi `Invio` per iniziare a modificarla.

Puoi anche creare note usando l'[[File explorer#Creare una nuova nota|esplora file]], o selezionando **Crea nuova nota** dalla [[Command palette|tavolozza dei comandi]].

> [!hint] Limitazioni dei caratteri di sistema
> Obsidian rispetta le limitazioni sui nomi dei file del sistema operativo su cui crei la nota. Se prevedi di [[Sync your notes across devices|sincronizzare le tue note tra dispositivi]], assicurati che i nomi dei file siano [compatibili con altri sistemi operativi](https://stackoverflow.com/q/1976007).
^blockquote-system-limitation

## Rinominare una nota

Per rinominare una nota attiva:

1. Seleziona il nome della nota nella parte superiore dell'editor (o premi `F2`).
2. Inserisci il nuovo nome e poi premi `Invio`.

Quando rinomini un file, Obsidian aggiorna automaticamente tutti i collegamenti a quel file.

Puoi rinominare una nota o una cartella senza aprirla, utilizzando l'[[File explorer#Rinominare un file o una cartella|esplora file]]

## Eliminare una nota

Per eliminare una nota, seleziona **Altre opzioni → Eliminazione file** in alto a destra di una nota attiva.

Oppure, seleziona **Elimina file attuale** dalla [[Command palette|tavolozza dei comandi]].

Puoi anche eliminare una nota o una cartella, utilizzando l'[[File explorer#Eliminare un file o una cartella|esplora file]].

> [!note] Cosa succede ai file dopo averli eliminati?
> Per modificare cosa succede ai file eliminati, seleziona una delle seguenti opzioni in **[[Settings|Impostazioni]] → File e collegamenti**:
>
> - **Cestino di sistema**: Per impostazione predefinita, i file eliminati finiscono nel cestino di sistema del tuo sistema operativo. Per ripristinare un file, usa il tuo gestore di file preferito.
> - **Cestino di Obsidian**: Puoi inviare i file eliminati in una cartella `.trash` nella tua cassaforte.
> - **Elimina definitivamente**: I file vengono eliminati immediatamente senza alcuna possibilità di ripristino.
