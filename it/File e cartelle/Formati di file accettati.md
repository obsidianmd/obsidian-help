Obsidian supporta i seguenti tipi di file:

- **Markdown**: `.md`  
- **Base:** `.base`
- **JSON Canvas**: `.canvas` ([Scopri di più](https://jsoncanvas.org/))  
- **Immagini**: `.avif`, `.bmp`, `.gif`, `.jpeg`, `.jpg`, `.png`, `.svg`, `.webp`  
- **Audio**: `.flac`, `.m4a`, `.mp3`, `.ogg`, `.wav`, `.webm`, `.3gp`  
- **Video**: `.mkv`, `.mov`, `.mp4`, `.ogv`, `.webm`  
- **PDF**: `.pdf`  

Puoi estendere Obsidian per supportare altri formati di file utilizzando i [[Plugin della comunità]].

### Incorporare file

Molti tipi di file — tra cui immagini, audio, video e PDF — possono essere incorporati direttamente nelle note.

- Su **Desktop**, trascina e rilascia un file supportato in una nota per incorporarlo automaticamente.  
- Su **Mobile**, usa le opzioni per gli allegati per inserire o collegare un file.

Consulta [[Incorporare file]] per ulteriori opzioni e sintassi di incorporamento.

> [!NOTE]
> Il supporto per audio, video e altri file dipende dai codec disponibili sul tuo dispositivo.
