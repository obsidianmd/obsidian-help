La cartella di configurazione di Obsidian contiene tutti i file di impostazioni relativi alla tua [[Gestire le casseforti|cassaforte]].

Per impostazione predefinita, la cartella di configurazione si chiama `.obsidian` e si trova nella cartella della cassaforte. Quando si utilizza un servizio di sincronizzazione, o se si desidera testare profili diversi nella stessa cassaforte, potrebbe essere utile cambiare la cartella di configurazione.

## Accedere alla cartella di configurazione

### Desktop

Su desktop, la cartella `.obsidian` si trova nella cartella della cassaforte. Puoi accedervi tramite il file manager del tuo sistema.

### Mobile

L'accesso alla cartella di configurazione varia a seconda della piattaforma:

**iOS e iPadOS**

Non esiste un modo predefinito per accedere alla cartella `.obsidian` su iOS o iPadOS. Per visualizzare e modificare file e cartelle nascosti, è necessaria un'app di terze parti come:

- **Taio** — App gratuita con funzionalità di base per visualizzare e modificare nomi di cartelle e file.
- **Textastic** — App a pagamento con funzionalità avanzate di modifica dei file.

**Android**

Per accedere alla cartella `.obsidian` su Android:

1. Apri il file manager di sistema.
2. Attiva l'opzione "Mostra file nascosti" nelle impostazioni.
3. Naviga fino alla cartella della cassaforte. La cartella `.obsidian` apparirà al livello principale.

> [!tip]- File manager alternativi
> Se il tuo file manager predefinito non dispone dell'opzione "Mostra file nascosti", installa un'app file manager che includa questa funzionalità.

## Cambiare la cartella di configurazione

Per impostare la cartella di configurazione:

1. Apri **[[Impostazioni]] → File e collegamenti**.
2. In **Cambia cartella di configurazione**, digita il nome del tuo profilo, iniziando con un punto (`.`). Ad esempio, `.obsidian-awesome`.
3. Riavvia Obsidian affinché le modifiche abbiano effetto.

> [!info] Eventuali impostazioni presenti nella cartella di configurazione non verranno trasferite nella nuova cartella di configurazione. Tuttavia, la cartella di configurazione precedente rimarrà all'interno della cartella della cassaforte.
