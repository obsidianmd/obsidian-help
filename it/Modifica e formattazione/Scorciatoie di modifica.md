Questa pagina elenca le scorciatoie da tastiera predefinite per navigare e modificare il testo in Obsidian. Queste scorciatoie sono fornite dal sistema operativo o dal framework su cui è costruito Obsidian e non possono essere personalizzate all'interno di Obsidian.

Per le scorciatoie da tastiera personalizzabili per i comandi di Obsidian, consulta [[Tasti di scelta rapida]].

## Scorciatoie per Windows e Linux

### Azioni comuni

| Azione | Scorciatoia |
|-|-|
| Copia | `Ctrl+C` |
| Taglia | `Ctrl+X` |
| Incolla | `Ctrl+V` |
| Incolla senza formattazione | `Ctrl+Shift+V` |
| Annulla | `Ctrl+Z` |
| Ripeti | `Ctrl+Shift+Z` o `Ctrl+Y` |
| Copia paragrafo | `Ctrl+C` (senza testo selezionato) |
| Taglia paragrafo | `Ctrl+X` (senza testo selezionato) |

### Modifica del testo

| Azione | Scorciatoia |
|-|-|
| Inserisci nuova riga | `Enter` |
| Elimina il carattere precedente | `Backspace` |
| Elimina il carattere successivo | `Delete` |
| Elimina la parola precedente | `Ctrl+Backspace` |
| Elimina la parola successiva | `Ctrl+Delete` |
| Elimina la riga corrente | `Ctrl+Shift+K` (senza testo selezionato) |

### Navigazione del testo

| Azione | Scorciatoia |
|-|-|
| Sposta il cursore di un carattere | `Left/→` |
| Sposta il cursore all'inizio della parola precedente | `Ctrl+←` |
| Sposta il cursore alla fine della parola successiva | `Ctrl+→` |
| Sposta il cursore all'inizio della riga corrente | `Home` |
| Sposta il cursore alla fine della riga corrente | `End` |
| Sposta il cursore alla riga precedente | `↑` |
| Sposta il cursore alla riga successiva | `↓` |
| Sposta il cursore all'inizio della nota | `Ctrl+Home` |
| Sposta il cursore alla fine della nota | `Ctrl+End` |
| Sposta il cursore di una pagina verso l'alto | `Page up` |
| Sposta il cursore di una pagina verso il basso | `Page down` |

### Selezione del testo

| Azione | Scorciatoia |
|-|-|
| Semplifica selezione | `Escape` |
| Seleziona tutto | `Ctrl+A` |
| Estendi selezione di un carattere | `Shift+Left/→` |
| Estendi selezione all'inizio della parola precedente | `Ctrl+Shift+←` |
| Estendi selezione alla fine della parola successiva | `Ctrl+Shift+→` |
| Estendi selezione all'inizio della riga corrente | `Shift+Home` |
| Estendi selezione alla fine della riga corrente | `Shift+End` |
| Estendi selezione all'inizio della nota | `Ctrl+Shift+Home` |
| Estendi selezione alla fine della nota | `Ctrl+Shift+End` |
| Estendi selezione di una pagina verso l'alto | `Shift+Page up` |
| Estendi selezione di una pagina verso il basso | `Shift+Page down` |

## Scorciatoie per macOS

### Azioni comuni

| Azione | Scorciatoia |
|-|-|
| Copia | `Cmd+C` |
| Taglia | `Cmd+X` |
| Incolla | `Cmd+V` |
| Incolla senza formattazione | `Cmd+Shift+V` |
| Annulla | `Cmd+Z` |
| Ripeti | `Cmd+Shift+Z` |
| Copia paragrafo | `Cmd+C` (senza testo selezionato) |
| Taglia paragrafo | `Cmd+X` (senza testo selezionato) |

### Formattazione del testo

| Azione | Scorciatoia |
| ---- | ---- |
| Testo in grassetto | `Cmd+B` |
| Testo in corsivo | `Cmd+I` |

### Modifica del testo

| Azione | Scorciatoia |
|-|-|
| Inserisci nuova riga | `Enter` |
| Elimina il carattere precedente | `Backspace` |
| Elimina il carattere successivo | `Delete` |
| Elimina la parola precedente | `Option+Backspace` |
| Elimina la parola successiva | `Option+Delete` |
| Elimina fino all'inizio della riga corrente | `Cmd+Backspace` |
| Elimina fino alla fine della riga corrente | `Cmd+Delete` |
| Elimina la riga corrente | `Cmd+Shift+K` (senza testo selezionato) |

### Navigazione del testo

| Azione | Scorciatoia |
|-|-|
| Sposta il cursore di un carattere | `Left/→` |
| Sposta il cursore all'inizio della parola precedente | `Option+←` |
| Sposta il cursore alla fine della parola successiva | `Option+→` |
| Sposta il cursore all'inizio della riga corrente | `Cmd+←` |
| Sposta il cursore alla fine della riga corrente | `Cmd+→` |
| Sposta il cursore alla riga precedente | `↑` |
| Sposta il cursore alla riga successiva | `↓` |
| Sposta il cursore all'inizio della nota | `Cmd+↑` |
| Sposta il cursore alla fine della nota | `Cmd+↓` |
| Sposta il cursore di una pagina verso l'alto | `Fn+↑` |
| Sposta il cursore di una pagina verso il basso | `Fn+↓` |
