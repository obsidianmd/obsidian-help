Puoi importare [[Formati di file accettati|formati di file accettati]], o _allegati_, nella tua cassaforte, come immagini, file audio o PDF. Gli allegati sono file normali a cui puoi accedere tramite il tuo file system. Gli allegati possono essere [[Incorporare file|incorporati]].

## Aggiungere un allegato

Puoi aggiungere allegati alla tua cassaforte in diversi modi. Solo i [[Formati di file accettati|formati di file accettati]] possono essere aggiunti.

> [!todo]- Copiare e incollare allegati
> Puoi incollare gli allegati direttamente nelle tue note. Obsidian crea un file con il contenuto incollato nella posizione predefinita degli allegati e lo [[Incorporare file|incorpora]] nella nota.

> [!todo]- Trascinare e rilasciare allegati
> Se trascini un file dal tuo file system in un editor aperto, Obsidian copia il file nella posizione predefinita degli allegati e lo [[Incorporare file|incorpora]] nella nota.

 > [!todo]- Scaricare allegati nella cartella della cassaforte
 > Puoi scaricare un allegato direttamente nella tua cassaforte, ad esempio se [[Importa note#Altri formati|importi dal browser]], o da altre app che salvano file direttamente nel tuo file system.

## Cambiare la posizione predefinita degli allegati

Per impostazione predefinita, gli allegati vengono aggiunti alla radice della tua cassaforte.

Puoi cambiare la posizione predefinita degli allegati in **[[Impostazioni|Impostazioni]] → File e collegamenti → Posizione predefinita dei nuovi allegati**.

- **Cartella principale del vault** aggiunge l'allegato alla radice della tua cassaforte.
- **Cartella personalizzata** aggiunge l'allegato in una cartella specificata.
- **Stessa cartella del file** aggiunge l'allegato nella stessa cartella della nota in cui lo hai aggiunto.
- **Sottocartella della cartella attuale** aggiunge gli allegati in una cartella specificata accanto alla nota in cui hai aggiunto l'allegato. Se non esiste, Obsidian la crea quando aggiungi un allegato.
