Scopri come usare l'elemento HTML [iframe](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/iframe) per incorporare pagine web nelle tue note.

Per incorporare una pagina web, aggiungi quanto segue nella tua nota e sostituisci il testo segnaposto con l'URL della pagina web che desideri incorporare:

```html
<iframe src="INSERISCI IL TUO URL QUI"></iframe>
```

> [!note] Nota
> Alcuni siti web non consentono di incorporarli. Potrebbero invece fornire URL specifici pensati per l'incorporamento. Se il sito web non supporta l'incorporamento, prova a cercare il nome del sito web seguito da "embed iframe". Ad esempio, "youtube embed iframe".

> [!tip] Suggerimento
> Se stai usando [[Lavagna]], puoi incorporare una pagina web in una scheda. Per maggiori informazioni, consulta [[Lavagna#Aggiungere schede da pagine web]].

## Incorporare un video di YouTube

Per incorporare un video di YouTube, usa la stessa sintassi Markdown delle [[Sintassi di formattazione di base#Immagini esterne|immagini esterne]]:

```md
![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)
```

![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)

## Incorporare un tweet

Per incorporare un tweet, usa la stessa sintassi Markdown delle [[Sintassi di formattazione di base#Immagini esterne|immagini esterne]]:

```md
![](https://twitter.com/obsdmd/status/1580548874246443010)
```
