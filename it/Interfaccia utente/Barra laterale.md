Obsidian ha due barre laterali: una barra laterale sinistra e una barra laterale destra. Le barre laterali contengono schede create dai plugin, come [[Backlinks]], [[Outgoing links]] e l'[[Esplora file]]. Su desktop, è anche possibile trascinare le note nella barra laterale per mantenerle visibili accanto all'editor principale.

Su desktop e tablet più grandi, la barra laterale sinistra include la [[Barra multifunzione]]. Su mobile e tablet più piccoli, le barre laterali sono [[#Aprire le barre laterali nascoste|compresse per impostazione predefinita]].

## Aprire le barre laterali nascoste

### Desktop e tablet più grandi

Su desktop e tablet più grandi, aprire una barra laterale compressa selezionando l'icona espandi sinistra ( ![[obsidian-icon-side-bar-left.svg#icon]] ) o espandi destra ( ![[obsidian-icon-side-bar-right.svg#icon]] ).

### Mobile e tablet più piccoli

Su mobile e tablet più piccoli, è possibile aprire le barre laterali nascoste nei seguenti modi:

- Scorrere a sinistra o a destra
- Selezionare l'icona espandi sinistra ( ![[obsidian-icon-side-bar-left.svg#icon]] ) (solo barra laterale sinistra)
- Utilizzare le opzioni *Attiva/disattiva sinistra* o *Attiva/disattiva destra* dalla [[Riquadro comandi]].

## Schede

Per impostazione predefinita, i plugin creano nuove schede invece di nuovi [[#Gruppi di schede]].

È possibile passare da una scheda all'altra selezionando l'icona. Un suggerimento appare per aiutare a identificare le schede.

In ogni pannello, è possibile visualizzare solo una scheda alla volta. Alcune azioni portano automaticamente una scheda in primo piano. Ad esempio, quando si seleziona un'etichetta, la relativa scheda si apre.

### Aprire o riaprire le schede

Esistono diversi modi per aggiungere schede alle barre laterali:

- **Abilitare un plugin**: Molti plugin creano automaticamente schede nella barra laterale quando vengono abilitati. Andare su **Impostazioni → Plugin della comunità** (o **Plugin principali**) e abilitare il plugin per la scheda desiderata.
- **Eseguire un comando**: Utilizzare la [[Riquadro comandi]] per cercare comandi che aprono schede specifiche. Ad esempio, cercare "Backlinks" o "Outgoing links" per aprire quelle schede.
- **Trascinare una nota** (solo desktop): Su desktop, è possibile trascinare una nota dall'[[Esplora file]] o da una scheda dell'editor nella barra laterale per aprirla lì.

Se una scheda è stata chiusa e si desidera recuperarla, gli stessi metodi la riapriranno.

### Chiudere le schede

Alcune schede possono essere chiuse tramite il menu contestuale, altre no. Se non si desidera visualizzare una scheda che non può essere chiusa, considerare la disabilitazione del plugin corrispondente o comprimere la barra laterale selezionando l'icona freccia.

### Riorganizzare le schede

È possibile trascinare e rilasciare le icone delle schede per riorganizzarle all'interno di un gruppo di schede.

Su desktop, è anche possibile trascinare le schede dalla barra laterale nell'area dell'editor principale per aprirle lì.

### Appuntare le schede

È possibile appuntare una scheda nella barra laterale per controllare come si aggiorna quando si interagisce con altre parti dell'app.

Per appuntare o sbloccare una scheda nella barra laterale, trascinare la scheda verso o dalla barra laterale.

Il comportamento di una scheda appuntata nella barra laterale dipende dal tipo di contenuto:

- **Note e Base**: Una nota o scheda base appuntata rimane al suo posto. Le nuove note si aprono come schede separate nell'editor principale invece di sostituire la scheda appuntata.
- **Pannelli** (come [[Backlinks]] o [[Outgoing links]]): Un pannello appuntato rimane focalizzato sull'ultima nota selezionata. Selezionare una nota diversa non aggiorna il pannello appuntato.

Per informazioni sull'appuntare schede nell'editor principale, consultare [[Schede#Appuntare una scheda|Appuntare una scheda]].

## Gruppi di schede

È possibile avere più gruppi di schede nella barra laterale.

### Creare gruppi di schede

Per creare un nuovo gruppo di schede, trascinare l'icona di una scheda sopra o sotto un gruppo di schede esistente. L'area evidenziata indica dove verrà creato il nuovo gruppo di schede.

### Chiudere gruppi di schede

Per chiudere un gruppo di schede, è possibile [[#Chiudere le schede|chiudere tutte le schede al suo interno]], oppure trascinare tutte le schede del gruppo tramite le loro icone e rilasciarle in gruppi di schede esistenti.

Quando è evidenziata solo l'area delle schede di un gruppo, la scheda trascinata verrà unita alle altre schede di quel gruppo.
