Base è un [[Plugin principali|plugin principale]] che ti permette di creare viste simili a database delle tue note. Utilizzando un base puoi visualizzare, modificare, ordinare e filtrare file e le loro [[Proprietà|proprietà]]. Base può aiutarti a organizzare qualsiasi cosa, dai progetti ai piani di viaggio, liste di lettura e altro ancora.

Ogni base può avere diverse [[Viste|viste]] con diversi layout come tabelle e schede. Di seguito è riportato un esempio di vista tabella dove ogni riga è un file e ogni colonna è una proprietà di quel file.

![Esempio di un base che mostra una vista tabella con un elenco di libri](bases-noshadow.png#interface)

Tutti i dati in Obsidian Base sono archiviati nei tuoi file [[Sintassi di formattazione di base|Markdown]] locali e nelle loro [[Proprietà|proprietà]]. Le viste sono descritte dalla [[Sintassi di Base|Sintassi di Base]], che può essere salvata come file `.base` o [[Crea una base#Incorporare un base|incorporata]] in blocchi di codice all'interno dei tuoi file Markdown.

## Tipi di vista

Le viste possono essere visualizzate con diversi layout. Layout aggiuntivi possono essere aggiunti tramite i [[Plugin della comunità]].

- [[Vista tabella|Tabella]]
	- Visualizza i file come righe con colonne che mostrano le proprietà di quei file.
- [[Vista lista|Elenco]]
	- Visualizza i file come un elenco con marcatori puntati o numerati.
- [[Vista schede|Schede]]
	- Visualizza i file come una griglia. Permette di creare viste simili a gallerie con immagini.
- [[Vista mappa|Mappa]]
	- Visualizza i file come segnaposto su una mappa interattiva.
