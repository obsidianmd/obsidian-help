Elenco è un tipo di [[Views|vista]] che puoi utilizzare in [[Introduction to Bases|Base]].

Seleziona ![[lucide-list.svg#icon]]  **Elenco** dal menu delle viste per visualizzare i file come un [[Basic formatting syntax#Elenchi|elenco]] con marcatori a punti elenco o numerati.

## Impostazioni

Le impostazioni della vista elenco possono essere configurate nelle [[Views#Impostazioni della vista|Impostazioni della vista]].

- Marcatori: punti elenco, numero o nessuno
- Rientra proprietà
- Separatori per le proprietà

### Marcatori

Scegli il tipo di marcatore dell'elenco che vuoi utilizzare: punti elenco, numero o nessuno.

### Rientra proprietà

Quando attivato, qualsiasi proprietà selezionata nel menu **Proprietà** verrà visualizzata come elemento rientrato rispetto all'elemento principale dell'elenco.

Per cambiare l'elemento principale dell'elenco, sposta una proprietà nel menu **Proprietà** in cima, trascinandola o utilizzando la scorciatoia `Alt + Su/Giù`.

### Separatori

Quando **proprietà annidate** è disattivato, questa impostazione definisce un carattere per separare le proprietà di ciascun elemento dell'elenco. Il valore predefinito è una virgola.
