Obsidian ti permette di migrare facilmente le tue note da Bear utilizzando il [[Importatore|plugin Importer]]. Questo convertirà i tuoi dati di Bear in file Markdown durevoli, che potrai utilizzare con Obsidian e molte altre app.

## Esporta i tuoi dati da Bear

Obsidian utilizza il formato di backup di Bear, i file `.bear2bk`. Puoi trovare le istruzioni di Bear per creare il backup [sul sito web di Bear](https://bear.app/faq/backup-restore/).

1. Apri Bear.
2. Crea un backup
	1. Su macOS: **File** → **Backup notes**.
	2. Su iOS: clicca sul menu altro (⋮) in cima alla barra laterale e scegli **Backup all notes**.
3. Scegli una posizione per il tuo file di backup.
4. Clicca **Export notes**.
5. Dovresti ora avere un file `.bear2bk` che contiene tutte le tue note.

## Importa i tuoi dati di Bear in Obsidian

Avrai bisogno del plugin ufficiale di Obsidian [[Importatore]], che puoi [installare qui](obsidian://show-plugin?id=obsidian-importer).

1. Apri **[[Impostazioni|Impostazioni]]**.
2. Vai su **Plugin della comunità** e [installa Importer](obsidian://show-plugin?id=obsidian-importer).
3. Abilita il plugin Importer.
4. Apri il plugin **Importer** utilizzando la tavolozza dei comandi o l'icona nella barra degli strumenti.
5. Sotto **Formattazione file** scegli **Bear (.bear2bk)**.
6. Seleziona la posizione del tuo file di backup di Bear.
7. Clicca **Importa** e attendi il completamento dell'importazione.
8. Hai finito!
