Obsidian utilizza file Markdown `.md` come formato principale per le note. Questo rende facile importare file Markdown in Obsidian. Molte app possono modificare file Markdown o esportare in questo formato.

Esistono diversi modi per aggiungere file Markdown alla tua cassaforte di Obsidian:

1. Trascina e rilascia file o cartelle nella finestra dell'[[Esplora file]]
2. Usa il file browser del tuo sistema (ad esempio Esplora risorse su Windows o Finder su macOS) e sposta i tuoi file direttamente nella cartella della cassaforte di Obsidian.

### Conversione da altre varianti di Markdown

Obsidian utilizza [[Obsidian Flavored Markdown]]. Alcune app utilizzano una sintassi Markdown leggermente diversa. Puoi usare lo [[Strumento importazione Markdown]] per convertire i tuoi file nella sintassi Markdown di Obsidian.
