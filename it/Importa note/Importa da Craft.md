Obsidian ti permette di migrare le tue note da Craft. Questo convertirà i tuoi dati di Craft in file Markdown durevoli, che potrai utilizzare con Obsidian e molte altre app.

## Esporta i tuoi dati da Craft

Apri l'app Craft sul tuo computer desktop:

1. Vai alla vista "Organize" o "All Documents"
2. Fai clic sul menu con i tre puntini
3. Seleziona l'opzione Esporta.
4. Scegli Markdown.

A seconda del numero di note che hai in Craft, questo processo potrebbe richiedere un po' di tempo.

## Importa i tuoi dati di Craft in Obsidian

Da qui puoi seguire le istruzioni per [[Importare file Markdown]].

## Risoluzione dei problemi

Craft Docs utilizza una sintassi non standard per alcuni collegamenti. Puoi trovare risorse per migrarli nel [repository GitHub di Importer](https://github.com/obsidianmd/obsidian-importer/issues/27).
