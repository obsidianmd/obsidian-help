Note univoche è un [[Plugin principali|plugin principale]] che consente di creare note con nomi basati sul tempo, note conosciute anche come note Zettelkasten.

Ad esempio, se crei una nuova nota alle 09:45 del 1° gennaio 2024, Note univoche crea una nota con `202401010945` come nome. Se esiste già una nota con lo stesso nome, la nuova nota utilizza il successivo timestamp disponibile.

## Creare una nota univoca

**Barra degli strumenti:**

- Nella barra degli strumenti, fai clic su **Nuova nota univoca** ![[obsidian-icon-sheets-in-box.svg#icon]].

**Tavolozza dei comandi:**

1. Premi `Ctrl+P` (o `Cmd+P` su macOS) per aprire la [[Riquadro comandi|tavolozza dei comandi]].
2. Cerca il comando **Nuova nota univoca**.
3. Premi `Invio` con il comando selezionato.

## Creare note univoche da un modello

Per impostazione predefinita, le nuove note univoche sono vuote. Per utilizzare un altro file come modello:

- Apri le **[[Impostazioni|Impostazioni]]**.
- Nella barra laterale, fai clic su **Plugin principali → Note univoche**.
- In **Posizione file modello**, inserisci il file che desideri utilizzare come modello.

Per ulteriori informazioni su come scrivere modelli, consulta [[Modelli|Modelli]].
