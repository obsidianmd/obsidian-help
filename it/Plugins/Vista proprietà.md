Vista proprietà è un [[Plugin principali|plugin principale]] che aggiunge due viste nella barra laterale che puoi utilizzare per gestire le [[Proprietà]]:

- **Proprietà file** mostra una vista delle proprietà per la nota attiva.
- **Tutte le proprietà** mostra un elenco di tutte le proprietà nella tua cassaforte e il loro tipo.

## Vista tutte le proprietà

Con la vista "Tutte le proprietà" puoi:

- Ordinare le proprietà per nome o per frequenza (numero totale di utilizzi nella tua cassaforte).
- Fare clic su una proprietà per aprire la vista [[Cerca]] con la sintassi di [[Cerca#Cerca proprietà|ricerca proprietà]] precompilata.
- Fare clic con il tasto destro su una proprietà per rinominarla globalmente.
