Los atajos de teclado son combinaciones de teclas personalizables para los comandos de Obsidian. Puedes asignar cualquier combinación de teclas para ejecutar comandos como abrir archivos, insertar plantillas o activar complementos.

Los atajos de teclado son diferentes de los [[Atajos de edición|atajos de teclado del sistema]] (como `Ctrl+C` para copiar), que son proporcionados por tu sistema operativo y no se pueden personalizar en Obsidian.

## Ver atajos de teclado

La forma más rápida de ver el atajo de teclado asignado a un comando es buscar el comando en la [[Paleta de comandos]]. Si hay un atajo de teclado listado para ese comando, está configurado.

También puedes ver y administrar todos los atajos de teclado en **[[Configuración]] → Atajos de teclado**.

## Configurar un atajo de teclado

Para añadir o modificar un atajo de teclado:

1. Abre **[[Configuración]] → Atajos de teclado**.
2. Busca el comando en la lista o usa el filtro de búsqueda.
3. Selecciona el icono de más (+) junto al comando.
4. Presiona la combinación de teclas que deseas usar.
5. Selecciona **Guardar** para confirmar.

Puedes asignar múltiples combinaciones de atajos de teclado a un solo comando seleccionando el icono de más (+) nuevamente después de configurar el primer atajo.

## Eliminar un atajo de teclado

Para eliminar un atajo de teclado:

1. Abre **[[Configuración]] → Atajos de teclado**.
2. Busca el comando con el atajo de teclado que deseas eliminar.
3. Selecciona el icono X junto a la combinación de teclas.

## Filtrar atajos de teclado

Para mostrar solo los comandos que tienen atajos de teclado asignados, selecciona el icono de filtro en **[[Configuración]] → Atajos de teclado**.

## Distribuciones de teclado no estadounidenses

Los atajos de teclado se muestran como aparecerían en una distribución de teclado estadounidense. Incluso si la combinación mostrada no coincide con tu distribución de teclado, el atajo funcionará basándose en las teclas reales que presionaste (siempre y cuando no cambies tu distribución de teclado).

## Atajos de teclado predeterminados

Para una lista completa de atajos de teclado predeterminados para edición y navegación, consulta [[Atajos de edición]].
