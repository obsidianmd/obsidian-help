Si eres desarrollador puedes contribuir a Obsidian de varias maneras.

### Complementos y temas

Puedes desarrollar [[Complementos de la comunidad]] y [[Temas]] usando Typescript y CSS. Aprende más visitando la [Documentación para desarrolladores de Obsidian](https://docs.obsidian.md).

### Proyectos de código abierto

Obsidian desarrolla y mantiene varios proyectos de código abierto. Puedes enviar pull requests con correcciones de errores y mejoras a los siguientes repositorios de GitHub:

- [JSON Canvas](https://github.com/obsidianmd/jsoncanvas) (MIT), un formato de archivo abierto para datos de lienzo infinito.
- [Obsidian Importer](https://github.com/obsidianmd/obsidian-importer) (MIT), un complemento que convierte datos de una amplia variedad de formatos a archivos Markdown de texto sin formato.
- [Obsidian Maps](https://github.com/obsidianmd/obsidian-maps) (MIT), un complemento que muestra mapas en Obsidian Bases.
- [Obsidian Web Clipper](https://github.com/obsidianmd/obsidian-clipper) (MIT), una extensión de navegador para resaltar y capturar páginas web en archivos Markdown de texto sin formato.
- [Defuddle](https://github.com/kepano/defuddle) (MIT), el analizador de HTML a Markdown utilizado por Obsidian Web Clipper.

### Documentación

Para ayudar a mejorar nuestra [documentación para desarrolladores](https://docs.obsidian.md/Home), envía un pull request al repositorio [obsidian-developer-docs](https://github.com/obsidianmd/obsidian-developer-docs).

Para ayudar a mejorar esta documentación, envía un pull request al repositorio [obsidian-help](https://github.com/obsidianmd/obsidian-help).

### Comunidad

El canal [plugin-dev](https://discord.com/channels/686053708261228577/840286264964022302) en nuestro servidor oficial de Discord es donde puedes discutir sobre el desarrollo de complementos y ayudar a otros desarrolladores con sus complementos.
