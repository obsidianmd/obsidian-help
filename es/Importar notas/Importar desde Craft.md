Obsidian te permite migrar tus notas desde Craft. Esto convertirá tus datos de Craft a archivos Markdown duraderos, que puedes usar con Obsidian y muchas otras aplicaciones.

## Exportar tus datos desde Craft

Abre la aplicación de Craft en tu computadora de escritorio:

1. Ve a la vista "Organize" o "All Documents"
2. Haz clic en el menú de tres puntos
3. Selecciona la opción Exportar.
4. Elige Markdown.

Dependiendo del número de notas que tengas en Craft, este proceso puede tomar un tiempo.

## Importar tus datos de Craft a Obsidian

Desde aquí puedes seguir las instrucciones de [[Importar archivos Markdown]].

## Solución de problemas

Craft Docs utiliza una sintaxis no estándar para ciertos enlaces. Puedes encontrar recursos para migrarlos en el [repositorio de GitHub del Importador](https://github.com/obsidianmd/obsidian-importer/issues/27).
