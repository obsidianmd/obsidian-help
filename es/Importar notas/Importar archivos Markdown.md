Obsidian utiliza archivos Markdown `.md` como formato principal para las notas. Esto facilita la importación de archivos Markdown a Obsidian. Muchas aplicaciones pueden editar archivos Markdown o exportar a este formato.

Hay varias formas de añadir archivos Markdown a tu bóveda de Obsidian:

1. Arrastra y suelta archivos o carpetas en la ventana del [[Explorador de archivos]]
2. Usa el explorador de archivos de tu sistema (por ejemplo, el Explorador de Windows o Finder en macOS), y mueve tus archivos directamente a la carpeta de la bóveda de Obsidian.

### Convertir desde otras variantes de Markdown

Obsidian utiliza [[Obsidian Flavored Markdown]]. Algunas aplicaciones usan una sintaxis Markdown ligeramente diferente. Puedes usar el [[Importador de formato Markdown|Conversor de formato]] para convertir tus archivos a la sintaxis Markdown de Obsidian.
