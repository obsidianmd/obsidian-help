Bases es un [[Complementos principales|complemento principal]] que te permite crear vistas tipo base de datos de tus notas. Usando una base puedes ver, editar, ordenar y filtrar archivos y sus [[Propiedades|propiedades]]. Bases puede ayudarte a organizar todo, desde proyectos hasta planes de viaje, listas de lectura y más.

Cada base puede tener varias [[Vistas|vistas]] con diferentes disposiciones como tablas y tarjetas. A continuación se muestra un ejemplo de vista de tabla donde cada fila es un archivo, y cada columna es una propiedad de ese archivo.

![Ejemplo de una base mostrando una vista de tabla con una lista de libros](bases-noshadow.png#interface)

Todos los datos en Obsidian Bases se almacenan en tus archivos [[Sintaxis de formato básico|Markdown]] locales y sus [[Propiedades|propiedades]]. Las vistas se describen mediante la [[Sintaxis de Bases]], que puede guardarse como un archivo `.base` o [[Crear una base#Incrustar una base|incrustarse]] en bloques de código dentro de tus archivos Markdown.

## Tipos de vista

Las vistas pueden mostrarse con diferentes disposiciones. Se pueden añadir disposiciones adicionales mediante [[Complementos de la comunidad]].

- [[Vista de tabla|Tabla]]
	- Muestra archivos como filas con columnas que muestran propiedades de esos archivos.
- [[Vista de lista|Lista]]
	- Muestra archivos como una lista con viñetas o marcadores numerados.
- [[Vista de tarjetas|Tarjetas]]
	- Muestra archivos como una cuadrícula. Te permite crear vistas tipo galería con imágenes.
- [[Vista de mapa|Mapa]]
	- Muestra archivos como marcadores en un mapa interactivo.
