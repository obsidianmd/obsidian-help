Vista de lista es un tipo de [[Vistas|vista]] que puedes usar en [[Introducción a Bases|Bases]].

Selecciona ![[lucide-list.svg#icon]]  **Lista** desde el menú de vista para mostrar archivos como una [[Sintaxis de formato básico#Listas|lista]] con marcadores de viñetas o numerados.

## Ajustes

Los ajustes de la vista de lista se pueden configurar en [[Vistas#Ajustes de vista|Ajustes de vista]].

- Marcadores: viñetas, números o ninguno
- Aplicar sangría a las propiedades
- Separadores para propiedades

### Marcadores

Elige el tipo de marcador de lista que deseas usar: viñetas, números o ninguno.

### Aplicar sangría a las propiedades

Cuando está activado, cualquier propiedad que selecciones en el menú **Propiedades** se mostrará como elementos con sangría del elemento principal de la lista.

Para cambiar tu elemento principal de la lista, mueve una propiedad en el menú **Propiedades** a la parte superior, arrastrándola o usando el atajo `Alt + Arriba/Abajo`.

### Separadores

Cuando **propiedades anidadas** está desactivado, este ajuste define un carácter para separar las propiedades de cada elemento de la lista. Por defecto es una coma.
