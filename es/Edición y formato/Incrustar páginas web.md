Aprende a usar el elemento HTML [iframe](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/iframe) para incrustar páginas web en tus notas.

Para incrustar una página web, añade lo siguiente en tu nota y reemplaza el texto de marcador con la URL de la página web que deseas incrustar:

```html
<iframe src="INSERTA TU URL AQUÍ"></iframe>
```

> [!note] Nota
> Algunos sitios web no permiten que los incrustes. En su lugar, pueden proporcionar URLs destinadas a incrustarlos. Si el sitio web no admite la incrustación, intenta buscar el nombre del sitio web seguido de "embed iframe". Por ejemplo, "youtube embed iframe".

> [!tip] Consejo
> Si estás usando [[Canvas]], puedes incrustar una página web en una tarjeta. Para más información, consulta [[Canvas#Agregar tarjetas desde páginas web]].

## Incrustar un vídeo de YouTube

Para incrustar un vídeo de YouTube, usa la misma sintaxis Markdown que las [[Sintaxis de formato básico#Imágenes externas|imágenes externas]]:

```md
![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)
```

![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)

## Incrustar un tweet

Para incrustar un tweet, usa la misma sintaxis Markdown que las [[Sintaxis de formato básico#Imágenes externas|imágenes externas]]:

```md
![](https://twitter.com/obsdmd/status/1580548874246443010)
```

![](https://twitter.com/obsdmd/status/1580548874246443010)
