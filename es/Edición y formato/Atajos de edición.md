Esta página lista los atajos de teclado predeterminados para navegar y editar texto en Obsidian. Estos atajos son proporcionados por tu sistema operativo o el framework sobre el que está construido Obsidian, y no pueden personalizarse dentro de Obsidian.

Para atajos de teclado personalizables para comandos de Obsidian, consulta [[Teclas de acceso rápido]].

## Atajos de Windows y Linux

### Acciones comunes

| Acción | Atajo |
|-|-|
| Copiar | `Ctrl+C` |
| Cortar | `Ctrl+X` |
| Pegar | `Ctrl+V` |
| Pegar sin formato | `Ctrl+Shift+V` |
| Deshacer | `Ctrl+Z` |
| Rehacer | `Ctrl+Shift+Z` o `Ctrl+Y` |
| Copiar párrafo | `Ctrl+C` (sin texto seleccionado) |
| Cortar párrafo | `Ctrl+X` (sin texto seleccionado) |

### Edición de texto

| Acción | Atajo |
|-|-|
| Insertar nueva línea| `Enter` |
| Eliminar el carácter anterior | `Backspace` |
| Eliminar el carácter siguiente | `Delete` |
| Eliminar la palabra anterior | `Ctrl+Backspace` |
| Eliminar la palabra siguiente | `Ctrl+Delete` |
| Eliminar la línea actual | `Ctrl+Shift+K` (sin texto seleccionado) |

### Navegación de texto

| Acción | Atajo |
|-|-|
| Mover el cursor un carácter | `Left/→` |
| Mover el cursor al inicio de la palabra anterior | `Ctrl+←` |
| Mover el cursor al final de la palabra siguiente | `Ctrl+→` |
| Mover el cursor al inicio de la línea actual | `Home` |
| Mover el cursor al final de la línea actual | `End` |
| Mover el cursor a la línea anterior | `↑` |
| Mover el cursor a la línea siguiente | `↓` |
| Mover el cursor al inicio de la nota | `Ctrl+Home` |
| Mover el cursor al final de la nota | `Ctrl+End` |
| Mover el cursor una página hacia arriba | `Page up` |
| Mover el cursor una página hacia abajo | `Page down` |

### Selección de texto

| Acción | Atajo |
|-|-|
| Simplificar selección | `Escape` |
| Seleccionar todo | `Ctrl+A` |
| Extender selección un carácter | `Shift+Left/→` |
| Extender selección al inicio de la palabra anterior | `Ctrl+Shift+←` |
| Extender selección al final de la palabra siguiente | `Ctrl+Shift+→` |
| Extender selección al inicio de la línea actual | `Shift+Home` |
| Extender selección al final de la línea actual | `Shift+End` |
| Extender selección al inicio de la nota | `Ctrl+Shift+Home` |
| Extender selección al final de la nota | `Ctrl+Shift+End` |
| Extender selección una página hacia arriba | `Shift+Page up` |
| Extender selección una página hacia abajo | `Shift+Page down` |

## Atajos de macOS

### Acciones comunes

| Acción | Atajo |
|-|-|
| Copiar | `Cmd+C` |
| Cortar | `Cmd+X` |
| Pegar | `Cmd+V` |
| Pegar sin formato | `Cmd+Shift+V` |
| Deshacer | `Cmd+Z` |
| Rehacer | `Cmd+Shift+Z` |
| Copiar párrafo | `Cmd+C` (sin texto seleccionado) |
| Cortar párrafo | `Cmd+X` (sin texto seleccionado) |

### Formato de texto

| Acción | Atajo |
| ---- | ---- |
| Texto en negrita | `Cmd+B` |
| Texto en cursiva | `Cmd+I` |

### Edición de texto

| Acción | Atajo |
|-|-|
| Insertar nueva línea| `Enter` |
| Eliminar el carácter anterior | `Backspace` |
| Eliminar el carácter siguiente | `Delete` |

| Eliminar la palabra siguiente | `Option+Delete` |
| Eliminar hasta el inicio de la línea actual | `Cmd+Backspace` |
| Eliminar hasta el final de la línea actual | `Cmd+Delete` |
| Eliminar la línea actual | `Cmd+Shift+K` (sin texto seleccionado) |

### Navegación de texto

| Acción | Atajo |
|-|-|
| Mover el cursor un carácter | `Left/→` |
| Mover el cursor al inicio de la palabra anterior | `Option+←` |
| Mover el cursor al final de la palabra siguiente | `Option+→` |
| Mover el cursor al inicio de la línea actual | `Cmd+←` |
| Mover el cursor al final de la línea actual | `Cmd+→` |
| Mover el cursor a la línea anterior | `↑` |
| Mover el cursor a la línea siguiente | `↓` |
| Mover el cursor al inicio de la nota | `Cmd+↑` |
| Mover el cursor al final de la nota | `Cmd+↓` |
| Mover el cursor una página hacia arriba | `Fn+↑` |
| Mover el cursor una página hacia abajo | `Fn+↓` |

| Extender selección al final de la palabra siguiente     | `Option+Shift+→` |
