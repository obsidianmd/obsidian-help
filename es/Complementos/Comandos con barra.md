Comandos con barra es un [[Complementos principales|complemento principal]] que te permite ejecutar comandos en el editor escribiendo una barra diagonal (`/`) al inicio de una línea o después de cualquier espacio en blanco.

Para ejecutar un comando usando Comandos con barra:

1. En el editor, escribe una barra diagonal (`/`).
2. Ingresa el nombre del comando que deseas ejecutar para buscar en la lista de comandos disponibles.
3. Navega hasta el comando usando las teclas de flecha.
4. Presiona Enter.

Para salir de la búsqueda de Comandos con barra sin ejecutar un comando, presiona `Esc` o la tecla `Espacio`.

> [!TIP] Coincidencia difusa
> Al igual que la [[Paleta de comandos]], Comandos con barra admite coincidencia difusa, lo que te permite buscar comandos incluso si no conoces sus nombres exactos. Por ejemplo, escribe "gaa" para encontrar el comando **Guardar el archivo actual**.
