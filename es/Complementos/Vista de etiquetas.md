El panel de Etiquetas es un [[Complementos principales|complemento principal]] que lista todas las etiquetas en tu bóveda y el número de notas para cada etiqueta.

Haz clic en una etiqueta para buscarla usando [[Búsqueda]].

Para alternar una etiqueta en el término de búsqueda, haz clic en la etiqueta mientras presionas `Ctrl` (o `Cmd` en macOS).

El complemento proporciona las siguientes opciones:

- **Cambiar el orden** determina si se ordenan las etiquetas por **Nombre de etiqueta** o **Frecuencia**.
- **Mostrar etiquetas anidadas** alterna entre mostrar las etiquetas anidadas como un árbol o como una lista plana.
- **Expandir todo** y **Colapsar todo** expande y contrae la jerarquía de etiquetas anidadas.

Puedes elegir expandir o contraer cada nivel anidado por separado haciendo clic en la flecha junto a él.
