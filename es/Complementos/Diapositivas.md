Diapositivas es un [[Complementos principales|complemento principal]] que te permite crear presentaciones a partir de tus notas. Para iniciar una presentación:

**Menú de archivo:**

- Haz clic derecho en la pestaña de una nota y selecciona **Empezar presentación**.

**Paleta de comandos:**

1. Presiona `Ctrl+P` (o `Cmd+P` en macOS) para abrir la [[Paleta de comandos]].
2. Busca el comando **Empezar presentación**.
3. Presiona `Enter` con el comando seleccionado para iniciar una presentación usando la nota activa.

Para navegar entre diapositivas, haz clic en las flechas izquierda y derecha en la esquina inferior derecha, o presiona las teclas de flecha izquierda y derecha en tu teclado.

También puedes avanzar a la siguiente diapositiva presionando la barra espaciadora.

Para detener la presentación, presiona `Escape` o haz clic en la cruz en la esquina superior derecha de la presentación.

Puedes usar cualquier archivo Markdown válido como presentación. Para separar diapositivas, ingresa `---` al inicio de una línea rodeada de líneas vacías.

```md
# Presentaciones usando Diapositivas

Una demostración de cómo crear presentaciones usando Diapositivas.

---

## Formato

Puedes usar formato Markdown regular, como texto en *cursiva* y **negrita**.

---

## Diapositivas

Usa `---` para separar diapositivas.
```
