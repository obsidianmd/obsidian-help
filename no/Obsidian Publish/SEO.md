[[Introduksjon til Obsidian Publish|Obsidian Publish]]-nettsteder er automatisk optimalisert for søkemotorer og [[Forhåndsvisning av lenker på sosiale medier|delingskort for sosiale medier]]. Du kan også tilpasse sidemetadata med beskrivelser, permalenker og bilder.

Merk at du kan ikke tillate indeksering av søkemotorer i dine [[Administrer nettsteder#Nettstedsalternativer|nettstedsalternativer]].

### Google Search Console

For å spore SEO med Google, registrer nettstedet ditt på [Google Search Console](https://search.google.com/search-console). Du kan også følge våre anbefalte trinn for å [[Analyse|sette opp Google Analytics]].

### Nettstedskart

Nettstedskartet for Publish-nettstedet ditt er tilgjengelig på `/sitemap.xml`, for eksempel:
https://help.obsidian.md/sitemap.xml

Nettstedskartet er også tilgjengelig som en RSS-feed på `/rss.xml`:
https://help.obsidian.md/rss.xml

### Permalenker

Vurder å bruke [[Permalenker]] for å definere permanente URL-er på nettstedet ditt.

### Metadata

Sidemetadata kan tilpasses ved hjelp av [[Egenskaper#Egenskaper for Obsidian Publish|Egenskaper]], som kan være nyttig for [[Forhåndsvisning av lenker på sosiale medier|delingskort for sosiale medier]].

| Egenskap      | Beskrivelse                                                                                                       |
| ------------- | ----------------------------------------------------------------------------------------------------------------- |
| `publish`     | Se [[Publiser innholdet ditt#Velg data som skal publiseres automatisk\|Velg data som skal publiseres automatisk]]. |
| `permalink`   | Se [[Permalenker\|Permalenker]].                                                                                   |
| `description` | Se [[Forhåndsvisning av lenker på sosiale medier#Beskrivelse\|Beskrivelse]].                                       |
| `image`       | Se [[Forhåndsvisning av lenker på sosiale medier#Bilde\|Bilde]].                                                   |
| `cover`       | Se [[Forhåndsvisning av lenker på sosiale medier#Bilde\|Bilde]].                                                   |
