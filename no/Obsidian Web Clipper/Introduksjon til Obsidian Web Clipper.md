Obsidian Web Clipper er en gratis nettleserutvidelse som lar deg utheve sider og lagre nettinnhold til hvelvet ditt.

## Installer Web Clipper

Hent Obsidian Web Clipper-utvidelsen fra den offisielle katalogen for din nettleser:

- [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
	- For Chrome, Brave, Arc, Orion og andre Chromium-baserte nettlesere.
- [Firefox Add-Ons](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
	- For Firefox og Firefox Mobile.
- [Safari Extensions](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
	- For macOS, iOS og iPadOS.
- [Edge Add-Ons](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
	- For Microsoft Edge.

## Personvern

Obsidian Web Clipper lagrer innhold lokalt til Obsidian-hvelvet ditt og følger vår [personvernpolicy](https://obsidian.md/privacy). Dataene dine samles ikke inn, og vi henter ingen bruksstatistikk. Koden er [åpen kildekode](https://github.com/obsidianmd/obsidian-clipper) og kan revideres.
