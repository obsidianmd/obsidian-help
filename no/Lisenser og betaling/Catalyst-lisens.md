Catalyst-lisensen er et engangskjøp som gir deg tidlig tilgang til betaversjoner og bidrar til å støtte den fortsatte utviklingen av Obsidian. Du kan støtte Obsidian ved å [kjøpe en Catalyst-lisens](https://obsidian.md/buy/catalyst).

Fordelene inkluderer:

- Tidlig tilgang til [[#Aktiver tidlig tilgang-versjoner|nye versjoner]] av Obsidian
- Tilgang til eksklusive [[#Bli med i våre Insider-kanaler|Insider-kanaler]].
- Eksklusive [[#Få ditt forummerke|forummerker]] og [[#Få ditt Discord-merke|Discord-merker]].

## Catalyst-nivåer

Det finnes tre nivåer av Catalyst basert på beløpet du bidrar med: *Insider*, *Supporter* og *VIP*.

| Fordeler                                 | Insider | Supporter |   VIP    |
| ---------------------------------------- | :-----: | :-------: | :------: |
| Tidlig tilgang til nye versjoner         |    ✓    |     ✓     |    ✓     |
| Tilgang til eksklusive utviklingskanaler |    ✓    |     ✓     |    ✓     |
| Tilgang til eksklusiv loungekanal        |         |     ✓     |    ✓     |
| Insider-merke                            |    ✓    |           |          |
| Supporter-merke                          |         |     ✓     |          |
| VIP-merke                                |         |           |    ✓     |
| **Bidrag (USD)**                         | **$25** |  **$50**  | **$100** |


> [!info] Oppgrader Catalyst-nivået ditt
> Hvis du allerede eier en Catalyst-lisens, kan du oppgradere til et høyere Catalyst-nivå ved å betale prisforskjellen mellom ditt nåværende og det nye nivået.

## Få tilgang til Catalyst-fordeler

Når du har kjøpt en Catalyst-lisens, kan du få tilgang til en rekke fordeler. Lær hvordan du aktiverer dem her.

### Få ditt forummerke

**Merk:** For å koble Obsidian-kontoen din og forumkontoen din, må du bruke den samme e-postadressen for begge kontoene.

1. Hvis du ikke allerede har gjort det, bli med i vårt [forum](https://forum.obsidian.md/).
2. Gå til [Obsidian-kontooversikten din](https://obsidian.md/account/catalyst).
3. Under **Catalyst**, velg **Get forum badge**.

### Få ditt Discord-merke

1. Hvis du ikke allerede har gjort det, bli med på vår [Discord-server](https://discord.gg/veuWUTm).
2. Gå til [Obsidian-kontooversikten din](https://obsidian.md/account/catalyst).
3. Under **Catalyst**, velg **Get Discord badge**.

### Bli med i våre Insider-kanaler

Insidere kan chatte med Obsidian-teamet og andre Insidere i våre eksklusive Discord-kanaler. Du blir automatisk med i Insider-kanalene etter at du [[#Få ditt Discord-merke|får ditt Discord-merke]].

- [vip-lounge](https://discord.com/channels/686053708261228577/714234422379151400) for diskusjoner med Catalyst-medlemmer.
- [insider-desktop-release](https://discord.com/channels/686053708261228577/716028884885307432) for de nyeste tidlig tilgang-utgivelsene på desktop.
- [insider-mobile-release](https://discord.com/channels/686053708261228577/817515900349448202) for de nyeste tidlig tilgang-utgivelsene på mobil.
