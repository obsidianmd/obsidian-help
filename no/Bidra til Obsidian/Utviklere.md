Hvis du er utvikler kan du bidra til Obsidian på ulike måter.

### Utvidelser og temaer

Du kan utvikle [[Community-utvidelser]] og [[Temaer]] ved hjelp av Typescript og CSS. Les mer ved å gå til [Obsidians utviklerdokumentasjon](https://docs.obsidian.md).

### Åpen kildekode-prosjekter

Obsidian utvikler og vedlikeholder flere åpen kildekode-prosjekter. Du kan sende inn pull requests med feilrettinger og forbedringer til følgende GitHub-repositorier:

- [JSON Canvas](https://github.com/obsidianmd/jsoncanvas) (MIT), et åpent filformat for uendelig canvas-data.
- [Obsidian Importer](https://github.com/obsidianmd/obsidian-importer) (MIT), en utvidelse som konverterer data fra en rekke formater til ren tekst Markdown-filer.
- [Obsidian Maps](https://github.com/obsidianmd/obsidian-maps) (MIT), en utvidelse som viser kart i Obsidian Bases.
- [Obsidian Web Clipper](https://github.com/obsidianmd/obsidian-clipper) (MIT), en nettleserutvidelse for å utheve og fange nettsider til ren tekst Markdown-filer.
- [Defuddle](https://github.com/kepano/defuddle) (MIT), HTML-til-Markdown-parseren som brukes av Obsidian Web Clipper.

### Dokumentasjon

For å bidra til å forbedre [utviklerdokumentasjonen](https://docs.obsidian.md/Home) vår, send inn en pull request til repositoriet [obsidian-developer-docs](https://github.com/obsidianmd/obsidian-developer-docs).

For å bidra til å forbedre denne dokumentasjonen, send inn en pull request til repositoriet [obsidian-help](https://github.com/obsidianmd/obsidian-help).

### Fellesskap

Kanalen [plugin-dev](https://discord.com/channels/686053708261228577/840286264964022302) på vår offisielle Discord-server er stedet hvor du kan diskutere utvidelsesutvikling og hjelpe andre utviklere med utvidelsene deres.
