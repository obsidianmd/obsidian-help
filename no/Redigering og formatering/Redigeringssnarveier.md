Denne siden lister standard hurtigtaster for navigering og redigering av tekst i Obsidian. Disse snarveiene leveres av operativsystemet ditt eller rammeverket Obsidian er bygget på, og kan ikke tilpasses i Obsidian.

For tilpassbare hurtigtaster for Obsidian-kommandoer, se [[Hurtigtaster]].

## Windows- og Linux-snarveier

### Vanlige handlinger

| Handling | Snarvei |
|-|-|
| Kopier | `Ctrl+C` |
| Klipp ut | `Ctrl+X` |
| Lim inn | `Ctrl+V` |
| Lim inn uten formatering | `Ctrl+Shift+V` |
| Angre | `Ctrl+Z` |
| Gjør om | `Ctrl+Shift+Z` eller `Ctrl+Y` |
| Kopier avsnitt | `Ctrl+C` (uten valgt tekst) |
| Klipp ut avsnitt | `Ctrl+X` (uten valgt tekst) |

### Tekstredigering

| Handling | Snarvei |
|-|-|
| Sett inn ny linje | `Enter` |
| Slett forrige tegn | `Backspace` |
| Slett neste tegn | `Delete` |
| Slett forrige ord | `Ctrl+Backspace` |
| Slett neste ord | `Ctrl+Delete` |
| Slett gjeldende linje | `Ctrl+Shift+K` (uten valgt tekst) |

### Tekstnavigasjon

| Handling | Snarvei |
|-|-|
| Flytt markøren ett tegn | `Left/→` |
| Flytt markøren til begynnelsen av forrige ord | `Ctrl+←` |
| Flytt markøren til slutten av neste ord | `Ctrl+→` |
| Flytt markøren til begynnelsen av gjeldende linje | `Home` |
| Flytt markøren til slutten av gjeldende linje | `End` |
| Flytt markøren til forrige linje | `↑` |
| Flytt markøren til neste linje | `↓` |
| Flytt markøren til begynnelsen av notatet | `Ctrl+Home` |
| Flytt markøren til slutten av notatet | `Ctrl+End` |
| Flytt markøren én side opp | `Page up` |
| Flytt markøren én side ned | `Page down` |

### Tekstvalg

| Handling | Snarvei |
|-|-|
| Forenkle utvalg | `Escape` |
| Velg alt | `Ctrl+A` |
| Utvid utvalg med ett tegn | `Shift+Left/→` |
| Utvid utvalg til begynnelsen av forrige ord | `Ctrl+Shift+←` |
| Utvid utvalg til slutten av neste ord | `Ctrl+Shift+→` |
| Utvid utvalg til begynnelsen av gjeldende linje | `Shift+Home` |
| Utvid utvalg til slutten av gjeldende linje | `Shift+End` |
| Utvid utvalg til begynnelsen av notatet | `Ctrl+Shift+Home` |
| Utvid utvalg til slutten av notatet | `Ctrl+Shift+End` |
| Utvid utvalg én side opp | `Shift+Page up` |
| Utvid utvalg én side ned | `Shift+Page down` |

## macOS-snarveier

### Vanlige handlinger

| Handling | Snarvei |
|-|-|
| Kopier | `Cmd+C` |
| Klipp ut | `Cmd+X` |
| Lim inn | `Cmd+V` |
| Lim inn uten formatering | `Cmd+Shift+V` |
| Angre | `Cmd+Z` |
| Gjør om | `Cmd+Shift+Z` |
| Kopier avsnitt | `Cmd+C` (uten valgt tekst) |
| Klipp ut avsnitt | `Cmd+X` (uten valgt tekst) |

### Tekstformatering

| Handling | Snarvei |
| ---- | ---- |
| Fet tekst | `Cmd+B` |
| Kursiv tekst | `Cmd+I` |

### Tekstredigering

| Handling | Snarvei |
|-|-|
| Sett inn ny linje | `Enter` |
| Slett forrige tegn | `Backspace` |
| Slett neste tegn | `Delete` |
| Slett forrige ord | `Option+Backspace` |
| Slett neste ord | `Option+Delete` |
| Slett til begynnelsen av gjeldende linje | `Cmd+Backspace` |
| Slett til slutten av gjeldende linje | `Cmd+Delete` |
| Slett gjeldende linje | `Cmd+Shift+K` (uten valgt tekst) |

### Tekstnavigasjon

| Handling | Snarvei |
|-|-|
| Flytt markøren ett tegn | `Left/→` |
| Flytt markøren til begynnelsen av forrige ord | `Option+←` |
| Flytt markøren til slutten av neste ord | `Option+→` |
| Flytt markøren til begynnelsen av gjeldende linje | `Cmd+←` |
| Flytt markøren til slutten av gjeldende linje | `Cmd+→` |
| Flytt markøren til forrige linje | `↑` |
| Flytt markøren til neste linje | `↓` |
| Flytt markøren til begynnelsen av notatet | `Cmd+↑` |
| Flytt markøren til slutten av notatet | `Cmd+↓` |
| Flytt markøren én side opp | `Fn+↑` |
| Flytt markøren én side ned | `Fn+↓` |

### Tekstvalg

| Handling                                               | Snarvei                    |
| ------------------------------------------------------ | -------------------------- |
| Forenkle utvalg                                        | `Escape`                   |
| Velg alt                                               | `Cmd+A`                    |
| Utvid utvalg med ett tegn                              | `Shift+Left/→`   |
| Utvid utvalg til begynnelsen av forrige ord            | `Option+Shift+←`  |
| Utvid utvalg til slutten av neste ord                  | `Option+Shift+→` |
| Utvid utvalg til begynnelsen av gjeldende linje        | `Cmd+Shift+←`              |
| Utvid utvalg til slutten av gjeldende linje            | `Cmd+Shift+→`    |
| Utvid utvalg til begynnelsen av notatet                | `Cmd+Shift+↑`       |
| Utvid utvalg til slutten av notatet                    | `Cmd+Shift+↓`     |
| Utvid utvalg én side opp                               | `Ctrl+Shift+↑`      |
| Utvid utvalg én side ned                               | `Ctrl+Shift+↓`    |
