A licença Catalyst é uma compra única que lhe dá acesso antecipado a versões beta e ajuda a apoiar o desenvolvimento contínuo do Obsidian. Pode apoiar o Obsidian ao [comprar uma licença Catalyst](https://obsidian.md/buy/catalyst).

Os benefícios incluem:

- Acesso antecipado a [[#Ativar versões de acesso antecipado|novas versões]] do Obsidian
- Acesso a [[#Junte-se aos nossos canais Insider|canais Insider]] exclusivos.
- [[#Obter o seu emblema no fórum|Emblemas de fórum]] e [[#Obter o seu emblema no Discord|emblemas de Discord]] exclusivos.

## Níveis Catalyst

Existem três níveis de Catalyst com base no montante que contribui: *Insider*, *Supporter* e *VIP*.

| Benefícios                                       | Insider | Supporter |   VIP    |
| ------------------------------------------------ | :-----: | :-------: | :------: |
| Acesso antecipado a novas versões                |    ✓    |     ✓     |    ✓     |
| Acesso a canais exclusivos de desenvolvimento    |    ✓    |     ✓     |    ✓     |
| Acesso ao canal exclusivo lounge                 |         |     ✓     |    ✓     |
| Emblema Insider                                  |    ✓    |           |          |
| Emblema Supporter                                |         |     ✓     |          |
| Emblema VIP                                      |         |           |    ✓     |
| **Contribuição (USD)**                           | **$25** |  **$50**  | **$100** |


> [!info] Atualizar o seu nível Catalyst
> Se já possui uma licença Catalyst, pode atualizar para um nível Catalyst superior pagando a diferença de preço entre o seu nível atual e o novo nível.

## Aceder aos benefícios Catalyst

Depois de ter comprado uma licença Catalyst, pode aceder a uma variedade de benefícios. Saiba como ativá-los aqui.

### Obter o seu emblema no fórum

**Nota:** Para conectar a sua conta Obsidian e a sua conta do fórum, deve usar o mesmo endereço de e-mail para ambas as contas.

1. Se ainda não o fez, junte-se ao nosso [fórum](https://forum.obsidian.md/).
2. Vá ao [painel da sua conta Obsidian](https://obsidian.md/account/catalyst).
3. Em **Catalyst**, selecione **Get forum badge**.

### Obter o seu emblema no Discord

1. Se ainda não o fez, junte-se ao nosso [servidor Discord](https://discord.gg/veuWUTm).
2. Vá ao [painel da sua conta Obsidian](https://obsidian.md/account/catalyst).
3. Em **Catalyst**, selecione **Get Discord badge**.

### Junte-se aos nossos canais Insider

Os Insiders podem conversar com a equipa do Obsidian e outros Insiders nos nossos canais exclusivos do Discord. Junta-se automaticamente aos canais Insider depois de [[#Obter o seu emblema no Discord]].

- [vip-lounge](https://discord.com/channels/686053708261228577/714234422379151400) para discussões com membros Catalyst.
- [insider-desktop-release](https://discord.com/channels/686053708261228577/716028884885307432) para as últimas versões de acesso antecipado no desktop.
- [insider-mobile-release](https://discord.com/channels/686053708261228577/817515900349448202) para as últimas versões de acesso antecipado no mobile.
