Os atalhos de teclado são atalhos de teclado personalizáveis para comandos do Obsidian. Pode atribuir qualquer combinação de teclas para acionar comandos como abrir ficheiros, inserir modelos ou ativar/desativar plugins.

Os atalhos de teclado são diferentes dos [[Atalhos de edição|atalhos de teclado do sistema]] (como `Ctrl+C` para copiar), que são fornecidos pelo seu sistema operativo e não podem ser personalizados no Obsidian.

## Ver atalhos de teclado

A forma mais rápida de ver o atalho de teclado definido para um comando é encontrar o comando na [[Paleta de comando]]. Se houver um atalho de teclado listado para esse comando, está definido.

Também pode ver e gerir todos os atalhos de teclado em **[[Definições]] → Atalhos de teclado**.

## Definir um atalho de teclado

Para adicionar ou modificar um atalho de teclado:

1. Abra **[[Definições]] → Atalhos de teclado**.
2. Encontre o comando na lista ou utilize o filtro de pesquisa.
3. Selecione o ícone de mais (+) ao lado do comando.
4. Prima a combinação de teclas que pretende utilizar.
5. Selecione **Guardar** para confirmar.

Pode atribuir múltiplas combinações de atalhos de teclado a um único comando selecionando o ícone de mais (+) novamente após definir o primeiro atalho.

## Remover um atalho de teclado

Para remover um atalho de teclado:

1. Abra **[[Definições]] → Atalhos de teclado**.
2. Encontre o comando com o atalho de teclado que pretende remover.
3. Selecione o ícone X ao lado da combinação de teclas.

## Filtrar atalhos de teclado

Para mostrar apenas comandos que têm atalhos de teclado atribuídos, selecione o ícone de filtro em **[[Definições]] → Atalhos de teclado**.

## Disposições de teclado não americanas

Os atalhos de teclado são apresentados como apareceriam numa disposição de teclado americana (US). Mesmo que a combinação apresentada não corresponda à disposição do seu teclado, o atalho de teclado funcionará com base nas teclas reais que premiu (desde que não altere a disposição do teclado).

## Atalhos de teclado predefinidos

Para uma lista completa dos atalhos de teclado predefinidos para edição e navegação, consulte [[Atalhos de edição]].
