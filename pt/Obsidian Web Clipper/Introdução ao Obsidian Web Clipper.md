Obsidian Web Clipper é uma extensão de navegador gratuita que permite destacar páginas e guardar conteúdo web no seu cofre.

## Instalar o Web Clipper

Obtenha a extensão Obsidian Web Clipper a partir do diretório oficial para o seu navegador:

- [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
	- Para Chrome, Brave, Arc, Orion e outros navegadores baseados em Chromium.
- [Firefox Add-Ons](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
	- Para Firefox e Firefox Mobile.
- [Safari Extensions](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
	- Para macOS, iOS e iPadOS.
- [Edge Add-Ons](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
	- Para Microsoft Edge.

## Privacidade

Obsidian Web Clipper guarda conteúdo localmente no seu cofre Obsidian e segue a nossa [política de privacidade](https://obsidian.md/privacy). Os seus dados não são recolhidos e não reunimos quaisquer métricas de utilização. O código é de [código aberto](https://github.com/obsidianmd/obsidian-clipper) e auditável.
