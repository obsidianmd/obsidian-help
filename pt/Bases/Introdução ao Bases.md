Bases é um [[Plugins Base|plugin principal]] que permite criar vistas semelhantes a bases de dados das suas notas. Usando uma base, pode visualizar, editar, ordenar e filtrar ficheiros e as suas [[Propriedades|propriedades]]. As Bases podem ajudá-lo a organizar tudo, desde projetos a planos de viagem, listas de leitura e muito mais.

Cada base pode ter várias [[Vistas|vistas]] com diferentes esquemas, como tabelas e cartões. Abaixo está um exemplo de vista de tabela onde cada linha é um ficheiro e cada coluna é uma propriedade desse ficheiro.

![Exemplo de uma base mostrando uma vista de tabela com uma lista de livros](bases-noshadow.png#interface)

Todos os dados no Obsidian Bases são armazenados nos seus ficheiros [[Sintaxe de formatação básica|Markdown]] locais e nas suas [[Propriedades|propriedades]]. As vistas são descritas pela [[Sintaxe de Bases]], que pode ser guardada como um ficheiro `.base` ou [[Criar uma base#Incorporar uma base|incorporada]] em blocos de código dentro dos seus ficheiros Markdown.

## Tipos de vista

As vistas podem ser apresentadas com diferentes esquemas. Esquemas adicionais podem ser adicionados por [[Plugins da comunidade]].

- [[Vista de Tabela|Tabela]]
	- Apresenta ficheiros como linhas com colunas que mostram propriedades desses ficheiros.
- [[Vista de Lista|Lista]]
	- Apresenta ficheiros como uma lista com marcadores ou numeração.
- [[Vista de Cartões|Cartões]]
	- Apresenta ficheiros como uma grelha. Permite criar vistas semelhantes a galerias com imagens.
- [[Vista de mapa|Mapa]]
	- Apresenta ficheiros como marcadores num mapa interativo.
