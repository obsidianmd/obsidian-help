Lista é um tipo de [[Vistas|vista]] que pode usar no [[Introdução ao Bases|Bases]].

Selecione ![[lucide-list.svg#icon]]  **Lista** no menu de vista para apresentar ficheiros como uma [[Sintaxe de formatação básica#Listas|lista]] com marcadores ou numeração.

## Definições

As definições da vista de lista podem ser configuradas nas [[Vistas#Definições de vista|Definições de vista]].

- Marcadores: marcadores, números ou nenhum
- Avançar propriedades
- Separadores para propriedades

### Marcadores

Escolha o tipo de marcador de lista que pretende usar: marcadores, números ou nenhum.

### Avançar propriedades

Quando ativado, quaisquer propriedades que selecionar no menu **Propriedades** serão apresentadas como itens com recuo relativamente ao item principal da lista.

Para mudar o item principal da lista, mova uma propriedade no menu **Propriedades** para o topo, arrastando-a ou usando o atalho `Alt + Cima/Baixo`.

### Separadores

Quando **propriedades aninhadas** está desativado, esta definição define um caráter para separar as propriedades de cada item da lista. O valor predefinido é uma vírgula.
