Pré-visualização de página é um [[Plugins Base|plugin principal]] que permite pré-visualizar uma página quando passa o cursor sobre uma ligação interna, sem precisar de navegar para essa página.

Por predefinição, ao passar o cursor sobre uma ligação, será feita a pré-visualização desse ficheiro no [[Explorador de ficheiros]], na [[Pesquisar|Pesquisa]], nos [[Links inversos]], e mais. Para pré-visualizar uma página enquanto está no Editor, pressione `Ctrl` (ou `Cmd` no macOS) enquanto passa o cursor sobre a ligação.

Existe também uma opção para exigir `Ctrl` (ou `Cmd` no macOS) enquanto passa o cursor para ver o ficheiro. As definições para esta opção encontram-se em Plugins Base > Pré-visualização de página.

> [!hint]
> A pré-visualização está **ativada por predefinição**. Alternar a opção do plugin irá **desativar o comportamento**.
