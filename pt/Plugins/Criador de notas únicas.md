O Criador de notas únicas é um [[Plugins Base|plugin principal]] que permite criar notas com nomes baseados no tempo, também conhecidas como notas Zettelkasten.

Por exemplo, se criar uma nova nota às 09:45 de 1 de janeiro de 2024, o Criador de notas únicas cria uma nota com `202401010945` como nome. Se já existir uma nota com o mesmo nome, a nova nota utiliza o próximo carimbo de data/hora disponível.

## Criar uma nota única

**Barra de ferramentas:**

- Na barra de ferramentas, clique em **Crie uma nova nota única** ![[obsidian-icon-sheets-in-box.svg#icon]].

**Paleta de comandos:**

1. Prima `Ctrl+P` (ou `Cmd+P` no macOS) para abrir a [[Paleta de comando]].
2. Pesquise pelo comando **Crie uma nova nota única**.
3. Prima `Enter` com o comando selecionado.

## Criar notas únicas a partir de um modelo

Por predefinição, as novas notas únicas estão vazias. Para utilizar outro ficheiro como modelo:

- Abra as **[[Definições]]**.
- Na barra lateral, clique em **Plugins Base → Criador de notas únicas**.
- Em **Localização do ficheiro modelo**, introduza o ficheiro que pretende utilizar como modelo.

Para mais informações sobre como escrever modelos, consulte [[Plugins/Modelos]].
