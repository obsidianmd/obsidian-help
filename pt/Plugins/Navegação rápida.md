O Alternador rápido é um [[Plugins Base|plugin principal]] que permite pesquisar e abrir notas usando apenas o teclado.

## Abrir o Alternador rápido

Existem várias formas de abrir o Alternador rápido, quando está ativado:
- Pressione `Ctrl+O` (ou `Cmd+O` no macOS) para abrir o Alternador rápido;
- Na barra de ferramentas, clique em **Abrir Quick Switcher** ![[lucide-file-search.svg#icon]];
- No móvel, quando não está a editar uma nota, toque no ícone de mais no centro inferior da aplicação.

## Abrir uma nota com o Alternador rápido

1. [[#Abrir o Alternador rápido]].
2. Escreva para pesquisar uma nota por nome ou alcunha.
3. Navegue até à nota usando as teclas de seta.
4. Pressione `Enter` para abrir a nota selecionada.

> [!info] 
> A funcionalidade de preenchimento automático muda para um algoritmo de resultados mais simples quando o cofre atinge 10.000 itens, para manter o desempenho ideal da aplicação. 
^search-autocomplete-large

Se o texto não corresponder a nenhuma nota, pode pressionar `Enter` para criar uma nota com esse nome. Mesmo que o texto corresponda a uma ou mais notas semelhantes, ainda pode criar uma nota com o nome exato pressionando `Shift+Enter`.

Pode abrir a nota selecionada num novo separador pressionando `Ctrl+Enter` (ou `Cmd+Enter` no macOS).

> [!tip] Alternar entre duas notas
> Se o termo de pesquisa estiver vazio, o Alternador rápido mostra as notas mais recentes. Alterne entre duas notas abrindo o Alternador rápido, pressionando a seta `Baixo` e depois pressionando `Enter`.


> [!info] Ficheiros excluídos
> Os ficheiros que correspondam aos padrões de [[Definições#Ficheiros excluídos|Ficheiros excluídos]] são despriorizados nos resultados do Alternador rápido.
