O Painel de tags é um [[Plugins Base|plugin principal]] que lista todas as etiquetas no seu cofre e o número de notas para cada etiqueta.

Clique numa etiqueta para pesquisá-la utilizando a [[Pesquisar|Pesquisa]].

Para alternar uma etiqueta no termo de pesquisa, clique na etiqueta enquanto pressiona `Ctrl` (ou `Cmd` no macOS).

O plugin disponibiliza as seguintes opções:

- **Mudar ordem** determina se as etiquetas são ordenadas por **Nome da etiqueta** ou **Frequência**.
- **Mostrar nested tags** alterna entre apresentar etiquetas aninhadas como uma árvore ou como uma lista plana.
- **Expandir tudo** e **Recolher tudo** expande e recolhe a hierarquia de etiquetas aninhadas.

Pode optar por expandir ou recolher cada nível aninhado separadamente clicando na seta ao lado do mesmo.
