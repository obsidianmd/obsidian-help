Áreas de trabalho é um [[Plugins Base|plugin principal]] que permite gerir e alternar entre diferentes esquemas da aplicação dependendo da sua tarefa, por exemplo, diário, leitura ou escrita.

Uma área de trabalho contém informação sobre ficheiros e separadores abertos, e a largura e visibilidade de cada barra lateral.

## Guardar uma área de trabalho

**Barra de ferramentas:**

1. Na barra de ferramentas, clique em **Gerir esquema da área de trabalho** ![[lucide-panels-top-left.svg#icon]].
2. Introduza um nome para a área de trabalho.
3. Selecione **Guardar**.

**Paleta de comandos:**

1. Prima `Ctrl+P` (ou `Cmd+P` no macOS) para abrir a [[Paleta de comando]].
2. Pesquise pelo comando **Gerir esquema da área de trabalho**.
3. Prima `Enter` com o comando selecionado.
4. Introduza um nome para a área de trabalho.
5. Selecione **Guardar**.

Para atualizar uma área de trabalho existente, guarde a área de trabalho com o mesmo nome da área de trabalho que pretende atualizar.

## Carregar uma área de trabalho

**Barra de ferramentas:**

1. Na barra de ferramentas, clique em **Gerir esquema da área de trabalho** ![[lucide-panels-top-left.svg#icon]].
2. Junto à área de trabalho que pretende carregar, selecione **Carregar**.

**Paleta de comandos:**

1. Prima `Ctrl+P` (ou `Cmd+P` no macOS) para abrir a [[Paleta de comando]].
2. Pesquise pelo comando **Gerir esquema da área de trabalho**.
3. Prima `Enter` com o comando selecionado.
4. Selecione a área de trabalho a carregar.

## Eliminar uma área de trabalho

**Barra de ferramentas:**

1. Na barra de ferramentas, clique em **Gerir esquema da área de trabalho** ![[lucide-panels-top-left.svg#icon]].
2. Junto à área de trabalho que pretende eliminar, selecione **Apagar esquema** ![[lucide-x.svg#icon]].

**Paleta de comandos:**

1. Prima `Ctrl+P` (ou `Cmd+P` no macOS) para abrir a [[Paleta de comando]].
2. Pesquise pelo comando **Gerir esquema da área de trabalho**.
3. Prima `Enter` com o comando selecionado.
4. Junto à área de trabalho que pretende eliminar, selecione **Apagar esquema** ![[lucide-x.svg#icon]].
