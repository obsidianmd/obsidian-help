Comandos de barra é um [[Plugins Base|plugin principal]] que permite executar comandos no editor ao escrever uma barra (`/`) no início de uma linha ou após qualquer espaço em branco.

Para executar um comando usando Comandos de barra:

1. No editor, escreva uma barra (`/`).
2. Introduza o nome do comando que pretende executar para pesquisar a lista de comandos disponíveis.
3. Navegue até ao comando usando as teclas de seta.
4. Prima Enter.

Para sair da pesquisa de Comandos de barra sem invocar um comando, prima `Esc` ou a tecla `Espaço`.

> [!TIP] Correspondência aproximada
> Tal como a [[Paleta de comando]], os Comandos de barra suportam correspondência aproximada, o que permite pesquisar comandos mesmo que não saiba os seus nomes exatos. Por exemplo, escreva "gfa" para encontrar o comando **Guardar ficheiro atual**.
