Diapositivos é um [[Plugins Base|plugin principal]] que lhe permite criar apresentações a partir das suas notas. Para começar uma apresentação:

**Menu de ficheiro:**

- Clique com o botão direito no separador de uma nota e clique em **Começar a apresentação**.

**Paleta de comandos:**

1. Prima `Ctrl+P` (ou `Cmd+P` no macOS) para abrir a [[Paleta de comando]].
2. Pesquise pelo comando **Começar a apresentação**.
3. Prima `Enter` com o comando selecionado para iniciar uma apresentação usando a nota ativa.

Para navegar entre diapositivos, clique nas setas esquerda e direita no canto inferior direito, ou prima as teclas de seta esquerda e direita no seu teclado.

Também pode avançar para o próximo diapositivo premindo a Barra de espaços.

Para parar a apresentação, prima `Escape` ou clique na cruz no canto superior direito da apresentação.

Pode usar qualquer ficheiro Markdown válido como apresentação. Para separar diapositivos, introduza `---` no início de uma linha rodeada por linhas em branco.

```md
# Apresentações usando Diapositivos

Uma demonstração de como construir apresentações usando Diapositivos.

---

## Formatação

Pode usar formatação Markdown regular, como texto *enfatizado* e **negrito**.

---

## Diapositivos

Use `---` para separar diapositivos.
```
