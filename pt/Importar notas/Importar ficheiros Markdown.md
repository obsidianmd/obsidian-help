O Obsidian utiliza ficheiros Markdown `.md` como formato principal para notas. Isto torna fácil importar ficheiros Markdown para o Obsidian. Muitas aplicações podem editar ficheiros Markdown ou exportar para este formato.

Existem várias formas de adicionar ficheiros Markdown ao seu cofre do Obsidian:

1. Arrastar e largar ficheiros ou pastas na janela do [[Explorador de ficheiros]]
2. Utilizar o explorador de ficheiros do sistema (por exemplo, o Explorador do Windows ou o Finder no macOS) e mover os ficheiros diretamente para a pasta do cofre do Obsidian.

### Converter a partir de outras variantes de Markdown

O Obsidian utiliza [[Obsidian Flavored Markdown]]. Algumas aplicações usam uma sintaxe Markdown ligeiramente diferente. Pode utilizar o [[Importador de formato Markdown]] para converter os seus ficheiros para a sintaxe Markdown do Obsidian.
