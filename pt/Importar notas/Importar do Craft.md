O Obsidian permite-lhe migrar as suas notas do Craft. Isto irá converter os seus dados do Craft em ficheiros Markdown duráveis, que pode utilizar com o Obsidian e muitas outras aplicações.

## Exportar os seus dados do Craft

Abra a aplicação Craft no seu computador de secretária:

1. Vá à vista "Organize" ou "All Documents"
2. Clique no menu de três pontos
3. Selecione a opção Exportar.
4. Escolha Markdown.

Dependendo do número de notas que tem no Craft, este processo pode demorar algum tempo.

## Importar os seus dados do Craft para o Obsidian

A partir daqui pode seguir as instruções de [[Importar ficheiros Markdown]].

## Resolução de problemas

O Craft Docs utiliza sintaxe não padronizada para certas ligações. Pode encontrar recursos para migrar essas ligações no [repositório GitHub do Importador](https://github.com/obsidianmd/obsidian-importer/issues/27).
