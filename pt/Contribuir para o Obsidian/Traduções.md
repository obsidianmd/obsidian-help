Se souber outro idioma, de preferência nativamente, pode ajudar a traduzir a interface de utilizador do Obsidian ou a documentação.

Para ajudar a traduzir a interface de utilizador, submeta um pull request no [repositório de traduções](https://github.com/obsidianmd/obsidian-translations).

Para ajudar a melhorar a nossa documentação, submeta um pull request no repositório [obsidian-help](https://github.com/obsidianmd/obsidian-help). Certifique-se de seguir o [[Guia de estilo#Traduções|guia de estilo para traduções]] ao fazer alterações.
