Se é programador, pode contribuir para o Obsidian de várias formas.

### Plugins e temas

Pode desenvolver [[Plugins da comunidade]] e [[Temas]] utilizando Typescript e CSS. Saiba mais consultando a [Documentação para Programadores do Obsidian](https://docs.obsidian.md).

### Projetos de código aberto

O Obsidian desenvolve e mantém vários projetos de código aberto. Pode submeter pull requests com correções de erros e melhorias nos seguintes repositórios GitHub:

- [JSON Canvas](https://github.com/obsidianmd/jsoncanvas) (MIT), um formato de ficheiro aberto para dados de tela infinita.
- [Obsidian Importer](https://github.com/obsidianmd/obsidian-importer) (MIT), um plugin que converte dados de uma grande variedade de formatos para ficheiros Markdown em texto simples.
- [Obsidian Maps](https://github.com/obsidianmd/obsidian-maps) (MIT), um plugin que apresenta mapas no Obsidian Bases.
- [Obsidian Web Clipper](https://github.com/obsidianmd/obsidian-clipper) (MIT), uma extensão de navegador para destacar e capturar páginas web para ficheiros Markdown em texto simples.
- [Defuddle](https://github.com/kepano/defuddle) (MIT), o conversor de HTML para Markdown utilizado pelo Obsidian Web Clipper.

### Documentação

Para ajudar a melhorar a nossa [documentação para programadores](https://docs.obsidian.md/Home), submeta um pull request no repositório [obsidian-developer-docs](https://github.com/obsidianmd/obsidian-developer-docs).

Para ajudar a melhorar esta documentação, submeta um pull request no repositório [obsidian-help](https://github.com/obsidianmd/obsidian-help).

### Comunidade

O canal [plugin-dev](https://discord.com/channels/686053708261228577/840286264964022302) no nosso servidor oficial do Discord é onde pode discutir o desenvolvimento de plugins e ajudar outros programadores com os seus plugins.
