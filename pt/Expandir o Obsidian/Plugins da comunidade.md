Aprenda como estender o Obsidian com plugins criados pela comunidade. Use plugins para adaptar o Obsidian às suas necessidades específicas, como suporte para formatos de ficheiro adicionais ou integração com um serviço de terceiros.

> [!warning] Aviso
> Os plugins da comunidade executam código de terceiros em seu nome, que pode potencialmente causar danos. Para saber mais sobre o que a equipa do Obsidian faz para prevenir plugins prejudiciais, consulte [[Segurança de plugins]].

## Explorar plugins da comunidade

1. Abra as **[[Definições]]**.
2. Selecione **Ligar os plugins da comunidade**. Para mais informações, consulte [[Segurança de plugins#Modo restrito|Modo restrito]].
3. Selecione **Selecionar** para listar todos os plugins da comunidade disponíveis.

Use a caixa de texto para filtrar plugins com base no nome, autor e descrição.

Também pode explorar os plugins disponíveis no seu navegador, acedendo a [obsidian.md/plugins](https://obsidian.md/plugins).

## Instalar um plugin da comunidade

Para instalar um plugin da comunidade, primeiro tem de desativar o [[Segurança de plugins#Modo restrito|Modo restrito]].

1. Abra as **[[Definições]]**.
2. No menu lateral, selecione **Plugins da comunidade**.
3. Selecione **Selecionar** para explorar os plugins da comunidade disponíveis.
4. Selecione o plugin que pretende instalar.
5. Selecione **Instalar**.

Para usar um plugin instalado, precisa de o ativar.

## Ativar um plugin da comunidade

Selecione **Ativar** diretamente após a instalação, ou ative-o a partir da lista de Plugins da comunidade em **[[Definições]] → Plugins da comunidade → Plugins instalados**.

## Atualizar plugins

Por motivos de segurança, os plugins da comunidade não são atualizados automaticamente. Pode atualizar todos os plugins de uma vez ou atualizar plugins individualmente.

Para atualizar todos os seus plugins:

1. Abra as **[[Definições]]**.
2. Em **Plugins da comunidade → Plugins atuais**, selecione **Procurar por atualizações**.
3. Se existirem atualizações disponíveis, selecione **Atualizar todos**.

Para atualizar um plugin individual:

1. Abra as **[[Definições]]**.
2. Em **Plugins da comunidade → Plugins atuais**, selecione **Procurar por atualizações**.
3. Em **Plugins instalados**, selecione **Atualizar** junto ao plugin que pretende atualizar.

## Desinstalar um plugin da comunidade

1. Abra as **[[Definições]]**.
2. No menu lateral, selecione **Plugins da comunidade**.
3. Em **Plugins instalados**, selecione o ícone do lixo ![[lucide-trash-2.svg#icon]] junto ao plugin que pretende desinstalar.
4. Selecione **Desinstalar** para confirmar.

## Gerir plugins instalados

Em **[[Definições]] → Plugins da comunidade → Plugins instalados**, cada plugin tem vários ícones para o gerir:

- **[[Definições]]** ![[lucide-settings.svg#icon]] — Abrir a página de definições do plugin, se tiver opções configuráveis.
- **[[Atalhos de teclado]]** ![[lucide-plus-circle.svg#icon]] — Definir atalhos de teclado para os comandos do plugin.
- **Financiamento** ![[lucide-heart.svg#icon]] — Ver o URL de financiamento do autor do plugin para apoiar o seu trabalho.
- **Desinstalar** ![[lucide-trash-2.svg#icon]] — Remover o plugin do seu cofre.
- **Alternar** — Ativar ou desativar o plugin sem o desinstalar.

Também pode:

- Selecionar o ícone de **recarregar** para recarregar todos os plugins.
- Selecionar o ícone de **pasta** ![[lucide-folder-open.svg#icon]] para abrir a pasta de plugins na sua [[Pasta de configuração|pasta de configuração]].
- Usar a **barra de pesquisa** para filtrar plugins por nome.
