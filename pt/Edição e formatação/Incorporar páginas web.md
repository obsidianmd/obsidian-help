Saiba como usar o elemento HTML [iframe](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/iframe) para incorporar páginas web nas suas notas.

Para incorporar uma página web, adicione o seguinte na sua nota e substitua o texto de exemplo pelo URL da página web que pretende incorporar:

```html
<iframe src="INSIRA O SEU URL AQUI"></iframe>
```

> [!note] Nota
> Alguns sites não permitem que os incorpore. Em vez disso, podem fornecer URLs destinados a serem usados para incorporação. Se o site não suportar incorporação, tente pesquisar pelo nome do site seguido de "embed iframe". Por exemplo, "youtube embed iframe".

> [!tip] Dica
> Se estiver a usar o [[Canvas]], pode incorporar uma página web num cartão. Para mais informações, consulte [[Canvas#Adicionar cartões a partir de páginas web]].

## Incorporar um vídeo do YouTube

Para incorporar um vídeo do YouTube, use a mesma sintaxe Markdown das [[Sintaxe de formatação básica#Imagens externas|imagens externas]]:

```md
![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)
```

![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)

## Incorporar um tweet

Para incorporar um tweet, use a mesma sintaxe Markdown das [[Sintaxe de formatação básica#Imagens externas|imagens externas]]:

```md
![](https://twitter.com/obsdmd/status/1580548874246443010)
```

![](https://twitter.com/obsdmd/status/1580548874246443010)
