Esta página lista os atalhos de teclado predefinidos para navegar e editar texto no Obsidian. Estes atalhos são fornecidos pelo seu sistema operativo ou pela framework na qual o Obsidian é construído, e não podem ser personalizados dentro do Obsidian.

Para atalhos de teclado personalizáveis para comandos do Obsidian, consulte [[Atalhos de teclado]].

## Atalhos para Windows e Linux

### Ações comuns

| Ação | Atalho |
|-|-|
| Copiar | `Ctrl+C` |
| Cortar | `Ctrl+X` |
| Colar | `Ctrl+V` |
| Colar sem formatação | `Ctrl+Shift+V` |
| Anular | `Ctrl+Z` |
| Refazer | `Ctrl+Shift+Z` ou `Ctrl+Y` |
| Copiar parágrafo | `Ctrl+C` (sem texto selecionado) |
| Cortar parágrafo | `Ctrl+X` (sem texto selecionado) |

### Edição de texto

| Ação | Atalho |
|-|-|
| Inserir nova linha | `Enter` |
| Eliminar o carácter anterior | `Backspace` |
| Eliminar o carácter seguinte | `Delete` |
| Eliminar a palavra anterior | `Ctrl+Backspace` |
| Eliminar a palavra seguinte | `Ctrl+Delete` |
| Eliminar a linha atual | `Ctrl+Shift+K` (sem texto selecionado) |

### Navegação de texto

| Ação | Atalho |
|-|-|
| Mover o cursor um carácter | `Left/→` |
| Mover o cursor para o início da palavra anterior | `Ctrl+←` |
| Mover o cursor para o fim da palavra seguinte | `Ctrl+→` |
| Mover o cursor para o início da linha atual | `Home` |
| Mover o cursor para o fim da linha atual | `End` |
| Mover o cursor para a linha anterior | `↑` |
| Mover o cursor para a linha seguinte | `↓` |
| Mover o cursor para o início da nota | `Ctrl+Home` |
| Mover o cursor para o fim da nota | `Ctrl+End` |
| Mover o cursor uma página acima | `Page up` |
| Mover o cursor uma página abaixo | `Page down` |

### Seleção de texto

| Ação | Atalho |
|-|-|
| Simplificar seleção | `Escape` |
| Selecionar tudo | `Ctrl+A` |
| Estender seleção um carácter | `Shift+Left/→` |
| Estender seleção até ao início da palavra anterior | `Ctrl+Shift+←` |
| Estender seleção até ao fim da palavra seguinte | `Ctrl+Shift+→` |
| Estender seleção até ao início da linha atual | `Shift+Home` |
| Estender seleção até ao fim da linha atual | `Shift+End` |
| Estender seleção até ao início da nota | `Ctrl+Shift+Home` |
| Estender seleção até ao fim da nota | `Ctrl+Shift+End` |
| Estender seleção uma página acima | `Shift+Page up` |
| Estender seleção uma página abaixo | `Shift+Page down` |

| Mover o cursor para a linha anterior | `↑` |
| Mover o cursor para a linha seguinte | `↓` |
