Obsidian Web Clipper er en gratis browserudvidelse, som gør dig i stand til at markere websider og gemme deres indhold i din boks.

> [!warning]- Kræver Obsidian 1.7.2 eller nyere
> Web Clipper afhænger af funktioner, der er tilføjet til [[Obsidian URI]]-formatet i Obsidian 1.7.2. Sørg for, at du benytter Obsidian 1.7.2 eller nyere.

## Installér Web Clipper

Hent Obsidian Web Clipper-udvidelsen fra det officielle katalog for din browser:

- [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
	- Til Chrome, Brave, Arc, Orion og andre Chromium-baserede browsere.
- [Firefox Add-Ons](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
	- Til Firefox og Firefox Mobile.
- [Safari Extensions](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
	- Til macOS, iOS og iPadOS.
- [Edge Add-Ons](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
	- Til Microsoft Edge.

## Sådan bruger du Web Clipper

- [[Klip websider]]
	- Kom i gang og lær, hvordan du gemmer indhold fra sider.
- [[Fremhævning|Fremhævning]]
	- Fremhæv vigtige passager og vælg elementer, du vil gemme.
- [[Læser|Læser]]
	- Læs websider i en ren, distraktionsfri visning.
- [[Fortolker|Fortolker]]
	- Brug naturlige sprogkommandoer til at indfange og ændre data på en side.
- [[Fejlfinding af Web Clipper|Fejlfinding]]
	- Løs almindelige problemer med Web Clipper.
- [[Obsidian Web Clipper/Skabeloner|Skabeloner]]
	- Opret tilpassede skabeloner til specifikke websteder.
- [[Variabler]]
	- Forudfyld sidedata i din skabelon.
- [[Filtre]]
	- Ændr outputtet af skabelonvariabler.
- [[Logik]]
	- Brug betingelser, løkker og angiv variabler i skabeloner.

## Privatliv

Obsidian Web Clipper gemmer indhold lokalt i din Obsidian-boks og følger vores [privatlivspolitik](https://obsidian.md/privacy). Dine data indsamles ikke, og vi indsamler ingen brugsstatistikker. Koden er [open source](https://github.com/obsidianmd/obsidian-clipper) og kan auditeres.
