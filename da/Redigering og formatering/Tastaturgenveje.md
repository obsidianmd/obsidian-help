Her er en oversigt over tastaturgenveje, som du kan anvende til at navigere med og redigere tekst i dine noter. Du kan også oprette [[Genvejstaster|brugerdefinerede genvejstaster]].

## Windows og Linux tastaturgenveje

### Almindelige kommandoer

| Funktion | Genvej |
|-|-|
| Kopiér | `Ctrl+C` |
| Klip | `Ctrl+X` |
| Indsæt | `Ctrl+V` |
| Indsæt uden formattering | `Ctrl+Skift+V` |
| Fortryd | `Ctrl+Z` |
| Gendan | `Ctrl+Skift+Z` or `Ctrl+Y` |
| Kopiér afsnit | `Ctrl+C` (uden noget markere tekst) |
| Klip afsnit | `Ctrl+X` (uden noget markere tekst) |

### Teksredigering

| Funktion | Genvej |
|-|-|
| Iddsæt ny linje| `Retur` |
| Slet forrige tegn | `Tilbage` |
| Slet næste tegn | `Slet` |
| Slet forrige ord | `Ctrl+Tilbage` |
| Slet næste ord | `Ctrl+Slet` |
| Slet nuværende linje | `Ctrl+Shift+K` (uden noget markere tekst) |

### Tekstnavigation

| Funktion | Genvej |
|-|-|
| Flyt markøren et tegn | `Venstre/højre pil` |
| Flyt markøren til begyndelsen af forrige ord | `Ctrl+Venstre pil` |
| Flyt markøren til slutningen af næste ord | `Ctrl+Højre pil` |
| Flyt markøren til begyndelsen af nuværende linje | `Hjem` |
| Flyt markøren til slutningen af nuværende linje | `Slut` |
| Flyt markøren til forrige linje | `Pil op` |
| Flyt markøren til næste linje | `Pil ned` |
| Flyt markøren til begyndelsen af noten | `Ctrl+Hjem` |
| Flyt markøren til slutningen af noten | `Ctrl+Slut` |
| Flyt markøren en side op | `Side op` |
| Flyt markøren en side ned | `Side ned` |

### Tekstmarkering

| Funktion | Genvej |
|-|-|
| Ophæv markering | `Escape` |
| Vælg alt | `Ctrl+A` |
| Udvid markering med et tegn | `Shift+Venstre/højre pil` |
| Udvid markering til begyndelsen af forrige ord | `Ctrl+kift+Venstre pil` |
| Udvid markering til slutningen af næste ord | `Ctrl+Skift+Højre pil` |
| Udvid markering til begyndelsen af nuværende linje | `Skift+Hjem` |
| Udvid markering til slutningen af nuværende linje | `Skift+Slut` |
| Udvid markering til begyndelsen af noten | `Ctrl+Skift+Hjem` |
| Udvid markering til slutningen af noten | `Ctrl+Skift+Hjem` |
| Udvid markering en side op | `Skift+Side op` |
| Udvid markering en side ned | `Skift+Side ned` |

## macOS tastaturgenveje

### Almindelige kommandoer

| Funktion | Genvej |
|-|-|
| Kopier | `Cmd+C` |
| Klip | `Cmd+X` |
| Indsæt | `Cmd+V` |
| Indsæt uden formaterinbg | `Cmd+Skift+V` |
| Fortryd | `Cmd+Z` |
| Gendan | `Cmd+Skift+Z` |
| Kopier afsnit | `Cmd+C` (uden noget markere tekst) |
| Klip afsnit | `Cmd+X` (uden noget markere tekst) |

### Tekstformattering

| Funktion | Genvej |
| ---- | ---- |
| Fed tekst | `Cmd+B` |
| Kursiv tekst | `Cmd+I` |

### Tekstredigering

| Funktion | Genvej |
|-|-|
| Indsæt ny linje| `Retur` |
| Slet forrige tegn | `Tilbage` |
| Slet næste tegn | `Slet` |
| Slet forrige ord | `Option+Tilbage` |
| Slet næste ord | `Option+Slet` |
| Slet til begyndelsen af nuværende linje | `Cmd+Tilbage` |
| Slet til slutningen af nuværende linje | `Cmd+Slet` |
| Slet nuværende linje | `Cmd+Skift+K` (uden noget markere tekst) |

### Tekstnavigation

| Funktion | Genvej |
|-|-|
| Flyt markøren et tegn | `Venstre/højre pil` |
| Flyt markøren til begyndelsen af forrige ord | `Option+Venstre pil` |
| Flyt markøren til slutningen af næste ord | `Option+Højre pil` |
| Flyt markøren til begyndelsen af nuværende linje | `Cmd+Venstre pil` |
| Flyt markøren til slutningen af nuværende linje | `Cmd+Højre pil` |
| Flyt markøren til forrige linje | `Pil op` |
| Flyt markøren til næste linje | `Pil ned` |
| Flyt markøren til begyndelsen af noten | `Cmd+Pil op` |
| Flyt markøren til slutningen af noten | `Cmd+Pil ned` |
| Flyt markøren en side op | `Fn+Pil op` |
| Flyt markøren en side ned | `Fn+Pil ned` |

### Tekstmarkering

| Funktion | Genvej |
|-|-|
| Ophæv markering | `Escape` |
| Vælg alt | `Cmd+A` |
| Udvid markering en karakter | `Skift+Venstre/højre pil` |
| Udvid markering til begyndelsen af forrige ord | `Option+Skift+Venstre pil` |
| Udvid markering til slutningen af næste ord | `Option+Skift+Højre pil` |
| Udvid markering til begyndelsen af nuværende linje | `Cmd+Skift+Venstre pil` |
| Udvid markering til slutningen af nuværende linje | `Cmd+Skift+Højre pil` |
| Udvid markering til begyndelsen af noten | `Cmd+Skift+Pil op` |
| Udvid markering til slutningen af noten | `Cmd+Skift+Pil ned` |
| Udvid markering en side op | `Ctrl+Skift+Pil op` |
| Udvid markering en side ned | `Ctrl+Skift+Pil ned` |
