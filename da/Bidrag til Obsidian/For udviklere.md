Hvis du er udvikler, kan du bidrage til Obsidian på flere måder.

### Plugins og temaer

Du kan udvikle [[Fællesskabsplugins|fællesskabsplugins]] og [[Temaer|temaer]] på forskellige måder.

### Open source projekter

Obsidian udvikler og vedligeholder flere open source projekter. Du kan bidrage med fejlrettelser og forbedringer til følgende GitHub projekter:

- [Obsidian Web Clipper](https://github.com/obsidianmd/obsidian-clipper) (MIT), er en browser udvidelse, der kan markere og udklippe websider over til enkle tekst Markdown filer.
- [Obsidian Importer](https://github.com/obsidianmd/obsidian-importer) (MIT), er et plugin, der kan konvertere data fra en lang række formater til enkle tekst Markdown filer.
- [JSON Canvas](https://github.com/obsidianmd/jsoncanvas) (MIT), er et åbent fil format, der definerer uendelig canvas data.

### Dokumentation

Du kan forbedre vores [udviklerdokumentation](https://docs.obsidian.md/Home) på engelsk ved at indsende et "pull request" i GitHub projektet [obsidian-developer-docs](https://github.com/obsidianmd/obsidian-developer-docs).

### Fælleskabet

I [plugin-dev](https://discord.com/channels/686053708261228577/840286264964022302) kanalen på vores officielle Discord server kan du diskutere plugin udvikling og hjælpe andre udviklere med deres plugins.
