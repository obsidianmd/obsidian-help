[[Introduktion til Obsidian Publish|Obsidian Publish]]-websteder er automatisk optimeret til søgemaskiner og [[Forhåndsvisning af links på sociale medier|kort til deling på sociale medier]]. Du kan også tilpasse sidemetadata med beskrivelser, permalinks og billeder.

Bemærk, at du kan forhindre søgemaskineindeksering i dine [[Administrer sider#Side-indstillinger|side-indstillinger]].

### Google Search Console

For at spore SEO med Google kan du registrere dit websted på [Google Search Console](https://search.google.com/search-console). Du kan også følge vores anbefalede trin til at [[Analyse|opsætte Google Analytics]].

### Sitemap

Sitemappet for dit Publish-websted er tilgængeligt på `/sitemap.xml`, for eksempel:
https://help.obsidian.md/sitemap.xml

Sitemappet er også tilgængeligt som et RSS-feed på `/rss.xml`:
https://help.obsidian.md/rss.xml

### Permalinks

Overvej at bruge [[Permalinks]] til at definere permanente URL'er på dit websted.

### Metadata

Sidemetadata kan tilpasses ved hjælp af [[Egenskaber#Egenskaber for Obsidian Publish|Egenskaber]], hvilket kan være nyttigt til [[Forhåndsvisning af links på sociale medier|kort til deling på sociale medier]].

| Egenskab      | Beskrivelse                                                                                                       |
| ------------- | ----------------------------------------------------------------------------------------------------------------- |
| `publish`     | Se [[Udgiv dit indhold#Automatically select data to publish\|Vælg automatisk data til udgivelse]]. |
| `permalink`   | Se [[Permalinks\|Permalinks]].                                                                                   |
| `description` | Se [[Forhåndsvisning af links på sociale medier#Description\|Beskrivelse]].                                                      |
| `image`       | Se [[Forhåndsvisning af links på sociale medier#Image\|Billede]].                                                                  |
| `cover`       | Se [[Forhåndsvisning af links på sociale medier#Image\|Billede]].                                                                  |
