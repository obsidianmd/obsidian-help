iOS 및 iPadOS용 Obsidian 모바일 앱은 iPhone과 iPad에서 강력한 노트 작성 기능을 제공해요. [Apple App Store](https://apps.apple.com/us/app/obsidian-connected-notes/id1557175442)에서 다운로드할 수 있어요.

이 페이지에서는 위젯, Siri 통합, 단축어 등 iOS 관련 기능을 다뤄요.

## 동기화

기기 간 노트 동기화에 대한 정보는 [[기기 간 노트 동기화]]를 참조하세요.

## 위젯

iOS용 Obsidian은 보관함에서 빠른 작업을 수행할 수 있는 여러 위젯을 제공해요.

> [!note] 참고
> 위젯은 iOS 및 iPadOS 18 이상에서 사용할 수 있어요.
> "Face ID 요구"를 사용하여 앱을 잠금 해제하는 경우 위젯을 사용할 수 없어요.


### 잠금 화면 및 제어 센터 위젯

잠금 화면 및 제어 센터 위젯으로 다음 작업을 수행할 수 있어요:
- 새 노트 생성
- 특정 노트 열기
- 일일 노트 열기
- 검색 열기
- Obsidian 열기

### 홈 화면 위젯

홈 화면 위젯으로 다음 작업을 수행할 수 있어요:
- 노트 생성
- 노트 보기
- 일일 노트 열기

### 위젯 사용자 지정

사용할 보관함을 선택하거나 열 특정 노트를 지정하는 등 워크플로에 맞게 위젯을 사용자 지정할 수 있어요.

- **홈 화면 위젯:** 위젯을 길게 탭한 다음 **위젯 편집**을 선택해요.
- **잠금 화면 위젯:** 잠금 화면을 길게 터치하고 **사용자화**를 탭한 다음 잠금 화면을 선택하고 사용자 지정할 위젯을 탭해요.
- **제어 센터 위젯:** 제어 센터를 열고 왼쪽 상단의 **+** 버튼을 탭하여 편집을 시작한 다음 사용자 지정할 위젯을 탭해요.

## 단축어

Obsidian은 Apple의 단축어 앱과 통합되어 강력한 자동화를 만들 수 있어요. 사용 가능한 단축어는 다음과 같아요:

- **노트 열기** — 보관함에서 특정 노트를 열어요
- **새 노트 생성** — 보관함에 새 노트를 생성해요
- **일일 노트 열기** — 오늘의 일일 노트로 바로 이동해요
- **일일 노트에 캡처** — Obsidian 앱을 열지 않고 일일 노트에 텍스트를 추가하거나 앞에 삽입해요
- **북마크에 캡처** — Obsidian 앱을 열지 않고 북마크된 노트에 텍스트를 추가하거나 앞에 삽입해요

캡처 단축어는 백그라운드에서 노트에 콘텐츠를 추가할 수 있어 빠른 노트 작성에 특히 유용해요.

## 공유 시트

Obsidian의 공유 시트를 사용하면 웹 페이지에서 콘텐츠를 캡처할 수 있어요. YouTube 및 기타 소셜 네트워크와 같은 앱에서도 작동해요.

> [!note]
> - 네이티브 공유 시트는 iOS 및 iPadOS 18 이상에서 사용할 수 있어요.
> - 이 섹션에서 설명하는 공유 시트 기능은 Obsidian 1.13.0 이상이 필요해요.

공유 시트를 사용하여 다른 앱에서 Obsidian으로 콘텐츠를 빠르게 보내려면:
1. 다른 앱에서 **공유** 버튼을 탭해요.
2. **Obsidian**을 선택해요.
3. 위치를 선택해요.
4. 캡처된 콘텐츠를 검토하거나 편집해요.
5. **저장**을 탭해요.

![[ios-share-sheet-extension.png|400]]

### 위치

위치를 사용하면 저장하기 전에 공유된 콘텐츠가 어디로 이동할지 결정할 수 있어요.

위치로 캡처할 수 있는 대상:
- **새 노트** — 보관함 또는 폴더에 새 노트를 생성해요.
- **일일 노트** — 오늘의 일일 노트에 콘텐츠를 추가하거나 앞에 삽입해요.
- **북마크된 노트** — 북마크된 노트에 콘텐츠를 추가하거나 앞에 삽입해요.
- **노트** — 보관함에서 기존 노트를 선택해요.
- **새 북마크** — 공유된 URL을 Obsidian 북마크에 저장해요.

![[ios-share-sheet-locations.png|400]]

### 위치 사용자 지정

기사를 수신함에 저장하거나, 인용문을 일일 노트에 추가하거나, 링크를 북마크에 추가하는 등 일반적인 워크플로를 위한 위치를 만들 수 있어요.

위치를 사용자 지정하려면:

1. iOS 공유 시트에서 Obsidian을 열어요.
2. 툴바에서 현재 위치를 탭해요.
3. **+** 버튼을 탭하여 새 위치를 만들거나, 기존 위치를 선택하여 편집해요.
4. 보관함, 동작 및 선택적 설정을 선택해요.

`동작` 유형에 따라 다음과 같은 옵션을 설정할 수 있어요:
- 폴더
- 템플릿
- 북마크 그룹
- 추가 또는 앞에 삽입 위치
- 공유된 링크가 **전체 텍스트**를 캡처할지 **URL**만 캡처할지 여부

![[ios-share-sheet-add-location.png|400]]

### 공유 시 템플릿 사용

공유 시트에서 콘텐츠를 공유할 때 템플릿을 사용할 수 있어요. 템플릿을 사용하면 페이지 제목, 작성자, 출처 웹사이트, 게시 날짜 등의 세부 정보로 캡처된 웹 콘텐츠의 형식을 지정할 수 있어요.

템플릿이 포함된 위치를 설정하려면:

1. iOS 공유 시트에서 Obsidian을 열어요.
2. 툴바에서 현재 위치를 탭해요.
3. **+** 버튼을 탭하여 새 위치를 만들어요.
4. 위치의 이름을 입력해요.
5. 보관함을 선택해요.
6. **동작**을 **새 노트**로 설정해요.
7. **선택 사항** 섹션에서 **템플릿**을 탭해요.
8. 보관함에서 템플릿으로 사용할 노트를 선택해요.
9. **저장**을 탭하여 위치를 저장해요.

![[ios-share-sheet-set-template.png|400]]

이 위치를 사용하여 링크를 공유하면 Obsidian이 먼저 템플릿을 적용한 다음 공유된 콘텐츠를 추가해요.

지원되는 템플릿 플레이스홀더:

| 플레이스홀더 | 설명 |
| --- | --- |
| `{{author}}` | 기사의 작성자 |
| `{{description}}` | 기사의 설명 또는 요약 |
| `{{domain}}` | 웹사이트의 도메인 이름 |
| `{{favicon}}` | 웹사이트 파비콘의 URL |
| `{{image}}` | 기사 메인 이미지의 URL |
| `{{published}}` | 기본 날짜 형식을 사용한 기사의 게시 날짜 |
| `{{published: YYYY-MM-DD}}` | 사용자 지정 날짜 형식을 사용한 게시 날짜 |
| `{{site}}` | 웹사이트의 이름 |
| `{{title}}` | 기사의 제목 |
| `{{url}}` | 기사 URL |
| `{{wordCount}}` | 추출된 콘텐츠의 총 단어 수 |

표준 템플릿 날짜 및 시간 플레이스홀더도 사용할 수 있어요:

| 플레이스홀더 | 설명 |
| --- | --- |
| `{{date}}` | 현재 날짜 |
| `{{date: YYYY-MM-DD}}` | 사용자 지정 형식을 사용한 현재 날짜 |
| `{{time}}` | 현재 시간 |
| `{{time: HH:mm}}` | 사용자 지정 형식을 사용한 현재 시간 |

## Siri 통합

Siri 음성 명령을 사용하여 Obsidian과 상호작용할 수 있어요:

- "Capture using Obsidian"
- "Capture to Obsidian"
- "Open my daily note in Obsidian"
- "Search in Obsidian"

## Spotlight 통합

iOS Spotlight에서 "Obsidian"을 검색하면 빠른 작업을 볼 수 있어요:
- 새 노트
- 검색
- 일일 노트
