[[Obsidian Web Clipper 소개|Web Clipper]] 브라우저 확장 프로그램을 설치하면, 브라우저에 따라 여러 가지 방법으로 접근할 수 있습니다:

1. 브라우저 툴바의 Obsidian 아이콘.
2. 키보드 단축키로 확장 프로그램 활성화.
3. 방문 중인 웹 페이지를 마우스 오른쪽 버튼으로 클릭하여 컨텍스트 메뉴 사용.

Obsidian에 페이지를 저장하려면 **Obsidian에 추가하기** 버튼을 클릭하세요.

## 페이지 캡처

확장 프로그램을 열면, Web Clipper가 [[Obsidian Web Clipper/템플릿|템플릿]] 설정에 따라 현재 웹 페이지에서 데이터를 추출합니다. 자신만의 템플릿을 만들고 [[변수]]와 [[필터]]를 사용하여 출력을 사용자 지정할 수 있습니다.

기본적으로 Web Clipper는 페이지의 다른 요소를 제외하고 주요 기사 콘텐츠만 지능적으로 추출하려고 시도합니다. 그러나 다음과 같은 방법으로 이 동작을 재정의할 수 있습니다:

- 사용자 지정 템플릿이 있으면 해당 템플릿을 사용합니다.
- 선택 영역이 있으면 선택 영역을 사용합니다. `Ctrl/Cmd+A`를 사용하여 전체 페이지를 선택할 수 있습니다.
- [[웹 페이지 하이라이트|하이라이트]]가 있으면 하이라이트를 사용합니다.

## 이미지 다운로드

Web Clipper를 사용할 때 이미지는 자동으로 다운로드되지 않습니다. 대신 이미지는 웹 기반 URL로 링크됩니다. 이는 보관함의 공간을 절약하지만, 오프라인에서 접근하거나 URL이 작동하지 않을 경우 이미지를 사용할 수 없다는 것을 의미합니다.

Obsidian에서 **현재 파일의 첨부파일 다운로드하기**라는 [[명령어 팔레트|명령]]을 사용하여 모든 파일의 이미지를 다운로드할 수 있습니다. 이 명령은 Obsidian에서 키보드 단축키에 매핑할 수도 있습니다.

## 단축키

Web Clipper에는 워크플로를 빠르게 수행하는 데 사용할 수 있는 키보드 단축키가 포함되어 있습니다. 키 매핑을 변경하려면 **Web Clipper 설정** → **일반**으로 이동하여 브라우저에 맞는 지침을 따르세요. Safari를 제외한 모든 브라우저에서 매핑을 변경할 수 있으며, Safari는 단축키 편집을 지원하지 않습니다.

| 동작                  | macOS         | Windows/Linux  |
| ----------------------- | ------------- | -------------- |
| 클리퍼 열기            | `Cmd+Shift+O` | `Ctrl+Shift+O` |
| 퀵 클립              | `Opt+Shift+O` | `Alt+Shift+O`  |
| 하이라이터 모드 전환 | `Opt+Shift+H` | `Alt+Shift+H`  |

## 인터페이스 기능

Web Clipper 인터페이스는 네 개의 섹션으로 나뉩니다:

1. **헤더**: 템플릿 전환, [[웹 페이지 하이라이트|하이라이트]] 켜기, [[리더|읽기 모드]], 설정 접근이 가능합니다.
2. **속성**: 페이지에서 추출된 [[속성|메타데이터]]를 보여주며, Obsidian에서 [[속성]]으로 저장됩니다.
3. **노트 내용**: Obsidian에 저장될 내용입니다.
4. **푸터**: 보관함과 폴더를 선택하고 Obsidian에 추가할 수 있습니다.

헤더 기능에는 다음이 포함됩니다:

- **템플릿** 드롭다운으로 Web Clipper 설정에서 추가한 저장된 [[Obsidian Web Clipper/템플릿|템플릿]] 간에 전환할 수 있습니다.
- **더 보기 (...)** 버튼으로 템플릿에서 사용할 수 있는 페이지 변수를 표시합니다.
- **하이라이터** 버튼으로 [[웹 페이지 하이라이트|하이라이트]]를 켤 수 있습니다.
- **톱니바퀴** 버튼으로 Web Clipper 설정을 열 수 있습니다.

푸터 기능에는 다음이 포함됩니다:

- **Obsidian에 추가하기** 버튼으로 Obsidian에 데이터를 저장합니다.
- **보관함** 드롭다운으로 Web Clipper 설정에서 추가한 저장된 보관함 간에 전환할 수 있습니다.
- **폴더** 필드로 저장할 폴더를 정의합니다.
- **인터프리터**로 페이지에서 [[웹 페이지 해석|자연어 프롬프트]]를 실행합니다.
