[[Obsidian Web Clipper 소개|Web Clipper]]에는 웹 페이지에서 불필요한 요소를 제거하고 주요 기사 콘텐츠만 깔끔하고 읽기 쉬운 형식으로 보여주는 리더라는 내장 읽기 모드가 포함되어 있습니다.

## 읽기 모드 활성화

다음과 같은 방법으로 리더를 활성화할 수 있습니다:

- 확장 프로그램 패널 헤더의 **책 아이콘**.
- 키보드 단축키 `Alt+Shift+R`, 확장 프로그램 설정에서 사용자 지정 가능.
- 리더 모드를 다시 토글하면 원래 페이지로 복원됩니다.

## 작동 방식

리더는 페이지에서 주요 콘텐츠를 추출합니다. 원래 페이지는 다음을 포함하는 깔끔한 읽기 뷰로 대체됩니다:

- 기사 제목, 작성자, 게시 날짜 및 원본 도메인.
- 이미지와 서식이 유지된 주요 기사 콘텐츠.
- 여러 제목이 있는 기사의 경우 사이드바에 개요 표시.
- 코드 블록의 구문 강조.
- 클릭 시 인라인으로 표시되는 각주 팝오버.

## 읽기 뷰 사용자 지정

리더 모드에는 페이지 오른쪽 상단 모서리에 마우스를 호버하면 표시되는 숨겨진 설정 바가 있으며, 읽는 방식을 사용자 지정할 수 있는 컨트롤이 포함되어 있습니다.

| 설정              | 설명                                                                                       |
| ----------------- | ------------------------------------------------------------------------------------------ |
| 폰트 크기         | 텍스트 크기를 조정합니다.                                                                  |
| 줄 너비           | 콘텐츠 영역의 최대 너비입니다.                                                             |
| 줄 높이           | 줄 사이 간격입니다.                                                                        |
| 색상&nbsp;스킴    |                                                                                            |
| 테마              | 자동은 시스템 기본값 설정을 따릅니다. 또는 항상 라이트 모드나 다크 모드를 사용하도록 선택할 수 있습니다. |
