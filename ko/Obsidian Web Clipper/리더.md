[[Obsidian Web Clipper 소개|Web Clipper]]에는 웹 페이지에서 불필요한 요소를 제거하고 주요 기사 콘텐츠만 깔끔하고 읽기 쉬운 형식으로 보여주는 **리더**라는 내장 읽기 모드가 포함되어 있어요.

## 읽기 모드 활성화

다음과 같은 방법으로 리더를 활성화할 수 있어요:

- ![[lucide-book-icon.svg#icon]] 확장 프로그램 패널의 **책 아이콘**.
- 확장 프로그램 설정에서 사용자 지정 가능한 **키보드 단축키**.
- 방문 중인 웹 페이지를 마우스 오른쪽 버튼으로 클릭하여 **컨텍스트 메뉴**.

![[obsidian.svg#icon]] 확장 프로그램 버튼을 클릭 한 번으로 리더를 열도록 설정할 수도 있어요. Web Clipper 설정에서 **일반** → **동작** → **열기 동작**으로 이동하여 **리더**를 선택하세요.

위의 모든 옵션을 통해 리더를 종료하고 원래 페이지를 복원할 수도 있어요.

## 작동 방식

리더는 페이지에서 주요 콘텐츠를 추출해요. 원래 페이지는 다음을 포함하는 깔끔한 읽기 뷰로 대체돼요:

- 기사 제목, 작성자, 게시 날짜 및 원본 도메인.
- 이미지와 서식이 유지된 주요 기사 콘텐츠.
- 여러 제목이 있는 기사의 경우 사이드바에 개요 표시.
- 코드 블록의 구문 강조.
- 클릭 시 인라인으로 표시되는 각주 팝오버.

## 읽기 뷰 사용자 지정

확장 프로그램 설정에서 리더 경험을 사용자 지정하세요. 일부 옵션은 리더 모드 내에서 ![[lucide-case-sensitive.svg#icon]] 아이콘을 클릭하여 사용할 수도 있어요.

| 설정              | 설명                                    |
| ----------------- | --------------------------------------- |
| 폰트              | 시스템에 설치된 폰트를 추가해요.        |
| 폰트 크기         | 텍스트 크기를 조정해요.                 |
| 줄&nbsp;높이      | 줄 사이 간격이에요.                     |
| 줄&nbsp;너비      | 콘텐츠 영역의 최대 너비예요.            |
| 외관              | 라이트 또는 다크 외관을 선택해요.       |
| 테마              | 색상을 선택해요.                        |
| 사용자 지정 CSS   | 페이지에 적용되는 추가 CSS예요.         |
