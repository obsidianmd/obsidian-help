[[Obsidian Web Clipper 소개|Web Clipper]]를 사용하면 웹 페이지에서 텍스트를 하이라이트하고 Obsidian에 저장할 요소를 선택할 수 있어요. 하이라이트는 저장되므로 페이지를 다시 방문할 때 확인할 수 있어요.

하이라이트는 확장 프로그램을 사용하여 Obsidian에 저장할 수 있어요. 확장 프로그램 설정에서 하이라이터 옵션을 정의할 수 있어요.

## 하이라이터 사용하기

하이라이터는 실시간 웹 페이지 또는 [[리더]]에서 사용할 수 있어요. Reader 보기에서는 텍스트를 선택하면 하이라이트할 수 있는 옵션이 제공돼요.

하이라이터를 활성화하여 선택 항목을 자동으로 하이라이트에 추가할 수도 있어요:

- 확장 프로그램 패널 또는 Reader 도구 모음의 ![[lucide-highlighter.svg#icon]] **하이라이터 아이콘**.
- 확장 프로그램 설정에서 사용자 지정할 수 있는 **키보드 단축키**.
- 방문 중인 웹 페이지를 마우스 오른쪽 버튼으로 클릭하여 **컨텍스트 메뉴** 사용.

하이라이터가 켜지면 선택한 텍스트, 이미지 및 요소가 하이라이트에 추가돼요. 위의 모든 방법으로 하이라이터를 종료할 수도 있어요.

## 노트에 하이라이트 추가하기

클리핑된 노트에 하이라이트를 삽입하는 세 가지 옵션이 있어요:

- **페이지 내용 인라인 하이라이트** — [[Obsidian 스타일 Markdown|구문]] `==highlight==`를 사용하여 텍스트에 직접 하이라이트를 추가해요.
- **페이지 내용 바꾸기** — 페이지 내용 없이 하이라이트 목록을 반환해요.
- **하이라이트 없음** — 하이라이트 없이 원본 내용을 반환해요.
 
이 옵션들은 템플릿의 `{{content}}` [[변수|변수]]를 변경해요. `{{highlights}}` 변수를 사용하여 템플릿에 직접 하이라이트를 추가할 수도 있어요. 예시:

```
{{highlights|map: item => item.text|join:"\n\n"}}
```

## 하이라이트 보기 및 검색

생성한 모든 하이라이트는 하이라이트 페이지에서 확인할 수 있어요. **설정** → **하이라이터**로 이동하여 이 페이지를 열 수 있어요.

## 하이라이트 내보내기

하이라이트는 설정 페이지 또는 하이라이트 페이지에서 `.json` 파일로 내보낼 수 있어요.
