[[Obsidian Web Clipper 소개|Web Clipper]]를 사용하면 웹 페이지에서 텍스트를 하이라이트하고 Obsidian에 저장할 요소를 선택할 수 있습니다. 하이라이트는 저장되므로 페이지를 다시 방문할 때 확인할 수 있습니다.

하이라이트는 확장 프로그램을 열 때 [[웹 페이지 클리핑|캡처]]하여 Obsidian에 저장할 수 있습니다.

## 하이라이터 켜기

브라우저에 따라 여러 방법으로 하이라이트를 켤 수 있습니다:

- 확장 프로그램 패널의 하이라이터 아이콘.
- 키보드 단축키로 확장 프로그램을 활성화.
- 방문 중인 웹 페이지를 마우스 오른쪽 버튼으로 클릭하여 컨텍스트 메뉴 사용.

하이라이트를 켜면 텍스트, 이미지 및 하이라이트할 요소를 선택할 수 있습니다.

## 하이라이터 설정

Web Clipper 설정으로 이동하여 하이라이터 동작을 변경할 수 있습니다. 여기에서 하이라이트를 `.json` 파일로 내보낼 수도 있습니다.

`{{content}}` [[변수|변수]]를 통해 클리핑된 노트에 하이라이트를 삽입하는 세 가지 옵션이 있습니다:

- **페이지 내용 인라인 하이라이트** — [[Obsidian 스타일 Markdown|구문]] `==highlight==`를 사용하여 텍스트에 직접 하이라이트를 추가합니다.
- **페이지 내용 바꾸기** — 페이지 내용 없이 하이라이트 목록을 반환합니다.
- **하이라이트 없음** — 하이라이트 없이 원본 내용을 반환합니다.

`{{highlights}}` 변수를 사용하여 템플릿에 직접 하이라이트를 추가할 수 있습니다. 예시:

```
{{highlights|map: item => item.text|join:"\n\n"}}
```
