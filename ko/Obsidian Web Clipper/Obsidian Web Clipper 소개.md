Obsidian Web Clipper는 페이지를 하이라이트하고 웹 콘텐츠를 보관함에 저장할 수 있는 무료 브라우저 확장 프로그램이에요.

## Web Clipper 설치

사용 중인 브라우저의 공식 디렉터리에서 Obsidian Web Clipper 확장 프로그램을 받으세요:

- [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
	- Chrome, Brave, Arc, Orion 및 기타 Chromium 기반 브라우저용.
- [Firefox Add-Ons](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
	- Firefox 및 Firefox Mobile용.
- [Safari Extensions](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
	- macOS, iOS, iPadOS용.
- [Edge Add-Ons](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
	- Microsoft Edge용.

## Web Clipper 사용 방법

- [[웹 페이지 클리핑]]
	- 시작하기 및 페이지에서 콘텐츠를 저장하는 방법을 알아보세요.
- [[웹 페이지 하이라이트|하이라이터]]
	- 중요한 구절을 하이라이트하고 저장할 요소를 선택하세요.
- [[리더|리더]]
	- 깔끔하고 집중할 수 있는 환경에서 웹 페이지를 읽으세요.
- [[웹 페이지 해석|인터프리터]]
	- 자연어 프롬프트를 사용하여 페이지의 데이터를 캡처하고 수정하세요.
- [[Web Clipper 문제 해결|문제 해결]]
	- Web Clipper의 일반적인 문제를 해결하세요.
- [[Obsidian Web Clipper/템플릿|템플릿]]
	- 특정 웹사이트를 위한 커스텀 템플릿을 만드세요.
- [[변수]]
	- 템플릿에 페이지 데이터를 미리 채우세요.
- [[필터]]
	- 템플릿 변수의 출력을 수정하세요.
- [[로직]]
	- 템플릿에서 조건문, 반복문을 사용하고 변수를 설정하세요.

## 개인정보 보호

Obsidian Web Clipper는 콘텐츠를 Obsidian 보관함에 로컬로 저장하며 [개인정보 보호정책](https://obsidian.md/ko/privacy)을 따라요. 귀하의 데이터는 수집되지 않으며, 사용 지표도 수집하지 않아요. 코드는 [오픈 소스](https://github.com/obsidianmd/obsidian-clipper)이며 감사할 수 있어요.
