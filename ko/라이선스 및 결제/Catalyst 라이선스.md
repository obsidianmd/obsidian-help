Catalyst 라이선스는 베타 버전에 대한 얼리 액세스를 제공하고 Obsidian의 지속적인 개발을 지원하는 일회성 구매입니다. [Catalyst 라이선스 구매](https://obsidian.md/buy/catalyst)를 통해 Obsidian을 후원할 수 있습니다.

혜택은 다음과 같습니다:

- Obsidian [[#얼리 액세스 버전 활성화|새 버전]]에 대한 얼리 액세스
- 독점 [[#Insider 채널 참여|Insider 채널]] 접근 권한
- 독점 [[#포럼 배지 받기|포럼 배지]] 및 [[#Discord 배지 받기|Discord 배지]]

## Catalyst 등급

기여 금액에 따라 세 가지 Catalyst 등급이 있습니다: *Insider*, *Supporter*, *VIP*.

| 혜택                                                                               | Insider | Supporter |   VIP    |
| ---------------------------------------------------------------------------------- | :-----: | :-------: | :------: |
| 새 버전에 대한 얼리 액세스                                                         |    ✓    |     ✓     |    ✓     |
| 독점 개발 채널 접근, `#insider-desktop` 및 `#insider-mobile`                       |    ✓    |     ✓     |    ✓     |
| 독점 라운지 채널 접근, `#vip-lounge`                                               |         |     ✓     |    ✓     |
| Insider 배지                                                                       |    ✓    |           |          |
| Supporter 배지                                                                     |         |     ✓     |          |
| VIP 배지                                                                           |         |           |    ✓     |
| **기여 금액 (USD)**                                                                | **$25** |  **$50**  | **$100** |


> [!info] Catalyst 등급 업그레이드
> 이미 Catalyst 라이선스를 소유하고 있다면 현재 등급과 새 등급 간의 가격 차액을 지불하여 더 높은 Catalyst 등급으로 업그레이드할 수 있습니다.

## Catalyst 혜택 이용

Catalyst 라이선스를 구매하면 다양한 혜택을 이용할 수 있습니다. 여기에서 활성화 방법을 알아보세요.

### 포럼 배지 받기

**참고:** Obsidian 계정과 포럼 계정을 연결하려면 두 계정 모두 동일한 이메일 주소를 사용해야 합니다.

1. 아직 가입하지 않았다면 [포럼](https://forum.obsidian.md/)에 참여하세요.
2. [Obsidian 계정 대시보드](https://obsidian.md/account/catalyst)로 이동하세요.
3. **Catalyst** 아래에서 **Get forum badge**를 선택하세요.

### Discord 배지 받기

1. 아직 가입하지 않았다면 [Discord 서버](https://discord.gg/veuWUTm)에 참여하세요.
2. [Obsidian 계정 대시보드](https://obsidian.md/account/catalyst)로 이동하세요.
3. **Catalyst** 아래에서 **Get Discord badge**를 선택하세요.

### Insider 채널 참여

Insider는 독점 Discord 채널에서 Obsidian 팀 및 다른 Insider와 대화할 수 있습니다. [[#Discord 배지 받기]] 후 자동으로 Insider 채널에 참여하게 됩니다.

[[얼리 액세스 버전]]을 받는 방법을 알아보세요.
