리본은 Obsidian 내에서 자주 사용하는 명령을 위한 공간이에요.

데스크톱에서는 [[사이드바#숨겨진 사이드바 열기|왼쪽 사이드바]]에 위치하며, 왼쪽 사이드바가 닫혀 있어도 계속 표시돼요.

모바일에서는 노트를 편집하지 않을 때 앱 오른쪽 하단의 **메뉴 옵션** ( ![[lucide-menu.svg#icon]] )을 탭하여 [[모바일 앱#리본 액션|리본 액션]]에 접근할 수 있어요.

## 동작

각 동작은 리본의 아이콘으로 표시돼요. 이 아이콘 위에 호버하면 툴팁이 표시되며, 선택하면 관련 동작이 실행돼요. 모바일에서는 이 아이콘을 탭하여 실행할 수 있어요.

1. **[[보관함 관리|보관함 전환기]]** 열기 ( ![[obsidian-icon-vault-switcher.svg#icon]] ).
2. **도움말** 접근 ( ![[lucide-help-circle.svg#icon]] ).
3. **[[설정]]** 열기 ( ![[lucide-settings.svg#icon]] ).

모바일 버전에서는 리본에 빠른 접근 동작을 설정할 수 있어요. 자세한 내용은 [[#모바일|모바일 섹션]] 및 [[모바일 앱]]을 참조하세요.

## 리본 커스터마이즈

커스터마이즈된 리본 레이아웃은 세션 간에 기억돼요. 앱 설정(`workspace.json` 및 `workspace-mobile.json` 파일)이 동기화될 때 다른 기기 및 모바일 앱과 동기화돼요.

### 데스크톱

데스크톱 버전에서는 다음과 같이 리본을 커스터마이즈할 수 있어요:

- 아이콘을 드래그 앤 드롭하여 리본 동작의 순서를 재배열해요.
- 특정 동작을 숨기려면 리본의 빈 공간을 마우스 오른쪽 버튼으로 클릭하고 숨기려는 동작의 체크를 해제해요.

#### 리본 숨기기

리본을 숨기는 방법은 두 가지예요:

- 리본을 마우스 오른쪽 버튼으로 클릭하고 **리본 숨기기**를 선택해요.
- **[[설정]]** → **외형**을 열고, **고급 설정**까지 스크롤한 다음 **리본 표시하기**를 비활성화해요.

#### 리본 표시하기

리본을 숨겼다가 다시 표시하려면:

1. **[[설정]]** → **외형**을 열어요.
2. **고급 설정**까지 스크롤해요.
3. **리본 표시하기**를 활성화해요.
4. 리본이 창 왼쪽에 다시 나타나요.

> [!tip] 사용자 지정 CSS
> 커뮤니티 테마나 사용자 지정 CSS를 사용하는 경우, 커스텀 스타일 규칙에 의해 리본이 숨겨지지 않는지 확인하세요.

### 모바일

모바일 버전의 Obsidian에서는 빠른 접근 동작 수행, 항목 재배열 및 리본 메뉴 커스터마이즈가 가능해요. 이러한 커스터마이즈에 접근하려면 다음 단계를 따르세요:

1. Obsidian의 **[[설정]]** ( ![[lucide-cog.svg#icon]] )을 열어요.
2. **외형** 섹션으로 이동해요.
3. **고급 설정** 옵션까지 스크롤해요.
4. **리본 설정** 항목 아래의 **관리** 버튼을 선택해요.

#### 빠른 접근

기본적으로 빠른 접근 옵션은 *짧게 누르기*로 "리본 메뉴 열기"가 설정되어 있어요. 다른 옵션을 빠른 접근으로 선택하면:

- 선택한 옵션은 *짧게 누르기*로 실행돼요.
- 리본 메뉴 접근은 *길게 누르기*로 변경돼요.

> [!info] 빠른 접근 옵션은 Obsidian 설정과 활성화된 코어 플러그인에 따라 달라져요.

#### 재배열 및 표시 여부

리본 메뉴의 항목 순서를 변경하려면 **드래그 앤 드롭** ( ![[lucide-menu.svg#icon]] ) 버튼을 길게 누른 채 선택한 항목을 위아래로 이동해요.

빨간색 **제거** ( ![[lucide-minus-circle.svg#icon]] ) 또는 초록색 **추가** ( ![[lucide-plus-circle.svg#icon]] ) 버튼을 사용하여 리본 메뉴의 항목 표시 여부를 전환할 수 있어요.
