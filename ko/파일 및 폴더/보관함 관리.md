**보관함**은 파일 시스템에 있는 폴더로, 노트, [[첨부 파일]], 그리고 Obsidian 전용 설정이 담긴 [[설정 폴더]]를 포함해요. 보관함에 대한 자세한 정보는 [[Obsidian의 데이터 저장 방식]]을 참조하세요.

**보관함 전환기**를 사용하여 보관함을 관리할 수 있어요. *보관함 프로필*은 Obsidian을 처음 열 때 표시돼요.

기존 보관함에서 보관함 전환기를 열려면, [[사이드바|왼쪽 사이드바]] 하단의 **보관함 프로필** ( ![[lucide-chevrons-up-down.svg#icon]])을 선택하세요. 또는 [[명령어 팔레트]]에서 **다른 보관함 열기**를 선택하세요.

## 새 보관함 생성

1. 컴퓨터에서 Obsidian을 열어요.
2. 왼쪽 하단에서 **보관함 프로필** ( ![[lucide-chevrons-up-down.svg#icon]])을 선택해요.
3. 컨텍스트 메뉴가 나타나요. **보관함 관리하기...**를 선택해요.
3. **새 원격 보관함 생성하기** 오른쪽의 **생성**을 선택해요.
4. **보관함 이름**에 보관함 이름을 입력해요.
5. **탐색**을 클릭하여 새 보관함이 생성될 위치를 선택해요.
6. **생성**을 클릭해요.

## 기존 폴더에서 보관함 생성

1. 컴퓨터에서 Obsidian을 열어요.
2. 왼쪽 하단에서 **보관함 프로필** ( ![[lucide-chevrons-up-down.svg#icon]])을 선택해요.
3. 컨텍스트 메뉴가 나타나요. **보관함 관리하기...**를 선택해요.
3. **보관함 폴더 열기** 오른쪽의 **열기**를 클릭해요.
4. 파일 브라우저에서 보관함으로 사용할 폴더를 선택해요.
5. **열기**를 클릭해요.

> [!tip] Obsidian Sync에서 보관함 열기
> Obsidian Sync를 사용하여 원격 보관함을 열려면 [[Obsidian Sync 설정하기]]를 참조하세요.

## 보관함 이름 바꾸기

보관함의 이름과 기본 폴더의 이름이 동일하므로, 보관함 이름을 바꾸면 폴더 이름도 변경돼요.

1. 컴퓨터에서 Obsidian을 열어요.
2. 왼쪽 하단에서 **보관함 프로필** ( ![[lucide-chevrons-up-down.svg#icon]])을 선택해요.
3. 컨텍스트 메뉴가 나타나요. **보관함 관리하기...**를 선택해요.
4. 보관함 목록에서 이름을 바꾸려는 보관함 옆의 **다른 옵션** ( ![[lucide-more-horizontal.svg#icon]] )을 선택해요.
5. **보관함 이름 바꾸기**를 선택해요.
6. 보관함의 새 이름을 입력하고 `Enter`를 눌러요.

## 보관함을 다른 폴더로 이동

1. 컴퓨터에서 Obsidian을 열어요.
2. 왼쪽 하단에서 **보관함 프로필** ( ![[lucide-chevrons-up-down.svg#icon]])을 선택해요.
3. 컨텍스트 메뉴가 나타나요. **보관함 관리하기...**를 선택해요.
4. 현재 보관함 창을 닫되, **보관함 관리** 창은 열어둬요.
5. 보관함 목록에서 이동하려는 보관함 옆의 **다른 옵션** ( ![[lucide-more-horizontal.svg#icon]] )을 선택해요.
6. **보관함 이동**을 선택한 다음 새 위치를 선택해요.

일부 운영 체제에서는 보관함 전환기로 보관함을 이동할 수 없어요. 이 경우 수동으로 보관함을 이동해야 해요:

1. Obsidian을 닫아요.
2. 보관함 폴더를 새 위치로 이동해요. 다른 서비스가 동기화하는 폴더는 피하세요.
3. Obsidian을 다시 열어요.
4. 왼쪽 하단의 **보관함 프로필** 아이콘 ( ![[lucide-chevrons-up-down.svg#icon]])을 클릭해요.
5. 팝업 메뉴에서 **보관함 관리하기...**를 선택해요.
6. **보관함 폴더 열기** 옆의 **열기**를 클릭해요.
7. 새 보관함 폴더를 찾아 선택해요.
8. **열기**를 클릭해요.
9. 보관함 내용이 변경되지 않았는지 확인해요. 필요한 경우 **[[설정]] → 커뮤니티 플러그인 → 제한 모드 끄기**로 이동하여 커뮤니티 플러그인을 다시 활성화해요.

## 보관함 제거

보관함을 제거하면 보관함 목록에서만 제거돼요.

1. 컴퓨터에서 Obsidian을 열어요.
2. 왼쪽 하단에서 **보관함 프로필** ( ![[lucide-chevrons-up-down.svg#icon]])을 선택해요.
3. 컨텍스트 메뉴가 나타나요. **보관함 관리하기...**를 선택해요.
4. 보관함 목록에서 제거하려는 보관함 옆의 **다른 옵션** ( ![[lucide-more-horizontal.svg#icon]])을 선택해요.
5. **목록에서 삭제**를 선택해요.

## 다른 보관함으로 설정 전달

다른 보관함에서 동일한 설정을 사용하려면, 선호하는 파일 관리자(또는 터미널)를 사용하여 원본 보관함의 루트에 있는 `.obsidian` 폴더를 대상 보관함의 루트로 복사해요.

변경 사항을 적용하려면 Obsidian을 다시 시작해야 할 수 있어요.

> [!note] `.obsidian` 폴더는 어디에 있나요?
> 기본적으로 대부분의 운영 체제는 마침표(`.`)로 시작하는 폴더를 숨겨요. `.obsidian` 폴더와 접근 방법에 대한 자세한 내용은 [[Obsidian의 데이터 저장 방식#보관함 설정|보관함 설정]] 및 [[설정 폴더|설정 폴더]]를 참조하세요.
