보관함은 Obsidian이 노트를 저장하는 로컬 파일 시스템의 폴더예요. 모든 노트를 하나의 보관함에 보관하거나, 각기 다른 프로젝트별로 여러 보관함을 만들 수 있어요.

Obsidian을 처음 열면 새 보관함을 추가하라는 메시지가 표시돼요. 새로운 빈 보관함을 생성하거나 기존 폴더를 사용하는 두 가지 옵션이 있어요.

## 빈 보관함 생성

새로운 빈 보관함을 생성하려면:

1. **새 원격 보관함 생성하기** 오른쪽에서 **생성**을 클릭해요.
2. **보관함 이름**에 보관함의 이름을 입력해요.
3. **탐색**을 클릭하여 새 보관함이 생성될 위치를 선택해요.
4. **생성**을 클릭해요.

## 기존 폴더 열기

보관함으로 사용하려는 폴더가 이미 있는 경우:

1. **보관함 폴더 열기** 오른쪽에서 **열기**를 클릭해요.
2. 파일 브라우저에서 보관함으로 사용할 폴더를 선택해요.
3. **열기**를 클릭해요.

보관함이 어떻게 작동하는지 더 알고 싶다면, [[Obsidian의 데이터 저장 방식]]을 참고하세요.

보관함 설정이 완료되었으므로, 이제 [[첫 번째 노트 만들기|첫 번째 노트를 만들]] 준비가 되었어요.
