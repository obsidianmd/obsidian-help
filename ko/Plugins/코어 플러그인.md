이 페이지는 Obsidian에 기본으로 설치되어 제공되는 코어 플러그인 목록입니다. 코어 플러그인은 Obsidian 팀에서 공식적으로 개발하고 지원하며, 앱에 포함되어 있습니다.

일부 코어 플러그인은 기본적으로 비활성화되어 있습니다. **[[설정]] → 코어 플러그인**에서 활성화할 수 있습니다.

Obsidian 팀은 특정 [[#기타 플러그인|오픈 소스 커뮤니티 플러그인]]의 개발도 관리하고 있습니다.

> [!tip] 커뮤니티 플러그인
> 원하는 플러그인을 여기에서 찾을 수 없다면, [[커뮤니티 플러그인|커뮤니티에서 만든 플러그인]]을 확인해 보세요. 직접 [플러그인을 개발](https://docs.obsidian.md/Plugins/Getting+started/Build+a+plugin)할 수도 있습니다.

### 모든 코어 플러그인

- [[음성 녹음]]
	- 노트에서 직접 오디오를 녹음하고 저장합니다.
- [[백링크]]
	- 노트의 모든 링크와 링크되지 않은 호출을 확인합니다.
- [[베이스 소개|베이스]]
	- 속성을 사용하여 파일을 편집, 정렬, 필터링할 수 있는 사용자 지정 뷰를 생성합니다.
- [[북마크]]
	- 노트, 제목, 검색 등에 대한 링크를 저장합니다.
- [[Canvas]]
	- 무한한 공간에서 아이디어를 시각적으로 배치하여 노트를 정리합니다.
- [[명령어 팔레트]]
	- 키보드에서 빠르게 명령에 접근합니다.
- [[일일 노트]]
	- 현재 날짜를 기반으로 노트를 생성하고 엽니다.
- [[파일 탐색기]]
	- 보관함 내의 파일과 폴더를 탐색합니다.
- [[파일 복원]]
	- 정기적인 스냅샷에서 작업을 복원합니다.
- [[마크다운 형식 불러오기]]
	- 다른 앱의 Markdown을 Obsidian 형식으로 변환합니다.
- [[그래프 뷰]]
	- 보관함 내 노트 간의 관계를 시각화합니다.
- [[노트 작성기]]
	- 두 노트를 병합하거나 하나를 둘로 분할합니다.
- [[나가는 링크]]
	- 활성 노트의 모든 링크를 표시합니다.
- [[개요]]
	- 활성 노트의 목차를 표시합니다.
- [[페이지 미리보기]]
	- 링크 위에 마우스를 올려 노트 내용을 미리 봅니다.
- [[속성 화면]]
	- 보관함의 모든 속성을 나열하고, 활성 노트의 속성을 확인합니다.
- [[Obsidian Publish 소개|Publish]]
	- 노트를 웹사이트, 위키 또는 문서로 호스팅합니다.
- [[빠른 전환기]]
	- 키보드에서 노트를 검색, 생성, 열기합니다.
- [[무작위 노트]]
	- 보관함에서 무작위 노트를 엽니다.
- [[검색]]
	- 보관함에서 파일을 찾습니다.
- [[슬래시 명령어]]
	- `/` 키를 사용하여 편집기 내에서 명령을 실행합니다.
- [[슬라이드]]
	- 노트로 프레젠테이션을 만듭니다.
- [[Obsidian Sync 소개|Sync]]
	- 기기 간에 노트를 동기화합니다.
- [[태그 패널|태그 패널]]
	- 보관함의 모든 태그를 나열합니다.
- [[Plugins/템플릿|템플릿]]
	- 노트에 미리 정의된 콘텐츠를 삽입합니다.
- [[고유 노트 생성]]
	- 시간 코드 제목을 사용하여 고유한 노트를 생성합니다.
- [[웹 뷰어]]
	- Obsidian에서 외부 링크를 엽니다.
- [[단어 개수]]
	- 단어 수와 문자 수를 표시합니다.
- [[작업 공간]]
	- 레이아웃을 저장하고 전환합니다.

### 기타 플러그인

Obsidian 팀은 커뮤니티 플러그인 스토어를 통해 제공되는 플러그인도 관리하고 있습니다:

- [[Importer]]
	- 다양한 앱과 형식의 파일을 Obsidian으로 변환합니다.
- [[지도 뷰|Maps]]
	- Obsidian 베이스에 지도 뷰를 추가합니다.
