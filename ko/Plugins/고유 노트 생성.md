고유 노트 생성은 시간 기반 이름으로 노트를 만들 수 있는 [[코어 플러그인]]이에요. Zettelkasten 노트라고도 해요.

예를 들어, 2024년 1월 1일 09:45에 새 노트를 만들면, 고유 노트 생성은 `202401010945`를 이름으로 하는 노트를 만들어요. 동일한 이름의 노트가 이미 존재하면, 새 노트는 다음 사용 가능한 타임스탬프를 사용해요.

## 고유 노트 만들기

**리본:**

- 리본에서 **새 고유 노트 생성** ( ![[obsidian-icon-sheets-in-box.svg#icon]] )을 클릭해요.

**명령어 팔레트:**

1. `Ctrl+P` (macOS에서는 `Cmd+P`)를 눌러 [[명령어 팔레트]]를 열어요.
2. **새 고유 노트 생성** 명령을 검색해요.
3. 명령이 선택된 상태에서 `Enter`를 눌러요.

## 템플릿으로 고유 노트 만들기

기본적으로 새 고유 노트는 비어 있어요. 다른 파일을 템플릿으로 사용하려면:

- **[[설정]]**을 열어요.
- 사이드바에서 **코어 플러그인 → 고유 노트 생성**을 클릭해요.
- **템플릿 파일 위치**에서 템플릿으로 사용할 파일을 입력해요.

템플릿 작성 방법에 대한 자세한 내용은 [[템플릿]]을 참조하세요.
