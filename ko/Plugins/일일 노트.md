일일 노트는 오늘 날짜를 기반으로 노트를 열거나, 존재하지 않으면 새로 만드는 [[코어 플러그인|코어 플러그인]]입니다. 일일 노트를 사용하여 일지, 할 일 목록, 또는 하루 동안 발견한 것들의 일일 기록을 만들 수 있습니다.

오늘의 일일 노트를 열려면 다음 중 하나를 수행하세요:

- [[리본|리본]]에서 **오늘의 일일 노트 열기**( ![[lucide-calendar.svg#icon]] )를 클릭합니다.
- [[명령어 팔레트]]에서 **오늘의 일일 노트 열기**를 실행합니다.
- **오늘의 일일 노트 열기** 명령에 [[단축키#단축키 설정|키보드 단축키를 사용]]합니다.

기본적으로 Obsidian은 YYYY-MM-DD 형식의 오늘 날짜로 이름이 지정된 새 빈 노트를 생성합니다.

> [!tip] 일일 노트를 별도의 폴더에 보관하려면, 플러그인 옵션에서 <u>새 파일 위치</u>를 설정하여 Obsidian이 새 일일 노트를 생성하는 위치를 변경할 수 있습니다.

> [!example]- 자동 하위 폴더
> **날짜 형식** 기능을 사용하여 일일 노트를 폴더로 자동 정리할 수 있습니다.
> 
> 예를 들어, 날짜 형식을 `YYYY/MMMM/YYYY-MMM-DD`로 설정하면, 노트가 `2023/January/2023-Jan-01`로 생성됩니다.
> 
> [momentJS](https://momentjs.com/docs/#/displaying/format/) 문서 사이트에서 더 많은 서식 옵션을 확인할 수 있습니다.

## 템플릿으로 일일 노트 생성

일일 노트가 동일한 구조를 가지고 있다면, [[Plugins/템플릿|템플릿]]을 사용하여 일일 노트를 생성할 때 미리 정의된 콘텐츠를 추가할 수 있습니다.

1. 다음 텍스트로 "Daily template"이라는 새 노트를 만드세요 (또는 본인에게 맞는 내용으로!):

   ```md
   # {{date:YYYY-MM-DD}}

   ## 작업

   - [ ]
   ```

2. **[[설정]]**을 엽니다.
3. 사이드바에서 **플러그인 옵션** 아래의 **일일 노트**를 클릭합니다.
4. **템플릿 파일 위치** 옆의 텍스트 상자에서 "Daily template" 노트를 선택합니다.

Obsidian은 다음에 새 일일 노트를 생성할 때 해당 템플릿을 사용합니다.

## 일일 노트와 속성

일일 노트 플러그인이 활성화되어 있고 노트 안에 날짜 속성이 있으면, Obsidian은 자동으로 해당 날짜의 일일 노트에 대한 링크를 생성하려고 시도합니다. 예를 들어, `example.md`라는 노트에 `2023-01-01`과 같은 날짜 속성이 포함되어 있으면, 이 날짜는 [[뷰와 편집 모드#실시간 미리보기|실시간 미리보기]] 섹션에서 클릭 가능한 링크로 변환됩니다.
