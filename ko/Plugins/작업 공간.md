작업 공간은 작업에 따라 다양한 앱 레이아웃을 관리하고 전환할 수 있게 해주는 [[코어 플러그인|코어 플러그인]]이에요. 예를 들어 일기 작성, 읽기, 글쓰기 등에 맞는 레이아웃을 설정할 수 있어요.

작업 공간에는 열려 있는 파일과 탭, 각 사이드바의 너비 및 표시 여부에 대한 정보가 포함돼요.

## 작업 공간 저장

**리본:**

1. 리본에서 **작업 공간 레이아웃 관리**( ![[lucide-panels-top-left.svg#icon]] )를 클릭해요.
2. 작업 공간의 이름을 입력해요.
3. **저장**을 선택해요.

**명령어 팔레트:**

1. `Ctrl+P`(macOS에서는 `Cmd+P`)를 눌러 [[명령어 팔레트]]를 열어요.
2. **작업 공간 레이아웃 관리** 명령을 검색해요.
3. 명령이 선택된 상태에서 `Enter`를 눌러요.
4. 작업 공간의 이름을 입력해요.
5. **저장**을 선택해요.

기존 작업 공간을 업데이트하려면 업데이트하려는 작업 공간과 동일한 이름으로 저장하세요.

## 작업 공간 불러오기

**리본:**

1. 리본에서 **작업 공간 레이아웃 관리**( ![[lucide-panels-top-left.svg#icon]] )를 클릭해요.
2. 불러오려는 작업 공간 옆에서 **불러오기**를 선택해요.

**명령어 팔레트:**

1. `Ctrl+P`(macOS에서는 `Cmd+P`)를 눌러 [[명령어 팔레트]]를 열어요.
2. **작업 공간 레이아웃 관리** 명령을 검색해요.
3. 명령이 선택된 상태에서 `Enter`를 눌러요.
4. 불러올 작업 공간을 선택해요.

## 작업 공간 삭제

**리본:**

1. 리본에서 **작업 공간 레이아웃 관리**( ![[lucide-panels-top-left.svg#icon]] )를 클릭해요.
2. 삭제하려는 작업 공간 옆에서 **레이아웃 삭제**( ![[lucide-x.svg#icon]] )를 선택해요.

**명령어 팔레트:**

1. `Ctrl+P`(macOS에서는 `Cmd+P`)를 눌러 [[명령어 팔레트]]를 열어요.
2. **작업 공간 레이아웃 관리** 명령을 검색해요.
3. 명령이 선택된 상태에서 `Enter`를 눌러요.
4. 삭제하려는 작업 공간 옆에서 **레이아웃 삭제**( ![[lucide-x.svg#icon]] )를 선택해요.
