파일 탐색기는 보관함 내의 파일과 폴더를 관리할 수 있는 [[코어 플러그인|코어 플러그인]]이에요. 보관함에 있는 노트와 기타 [[지원 파일 형식]]을 탐색하고 다양한 일반적인 파일 작업을 수행할 수 있어요:

- 파일과 폴더를 생성, 삭제, 이름 바꾸기할 수 있어요.
- 드래그 앤 드롭으로 파일과 폴더를 이동할 수 있어요.
- 컨텍스트 메뉴를 사용하여 사용 가능한 모든 작업에 접근할 수 있어요.

> [!tip]- 파일 드래그 앤 드롭
> 파일 탐색기에서 파일을 노트로 드래그하여 링크를 생성하거나, 파일을 파일 탐색기의 폴더로 드래그하여 복사할 수 있어요.

## 새 노트 만들기

새 노트를 기본 위치에 생성하려면:

1. 파일 탐색기 상단의 **새 노트** ![[lucide-pen-line.svg#icon]]를 클릭해요.
2. 노트 이름을 입력한 다음 Enter를 눌러요.

> [!tip]- 기본 위치 변경
> **[[설정]] → [[설정#파일 및 링크|파일 및 링크]] → [[설정#새 노트를 생성하는 위치|새 노트를 생성하는 위치]]**에서 새 노트의 기본 위치를 변경할 수 있어요.

특정 폴더에 새 노트를 만들려면:

1. 폴더를 우클릭한 다음 **새 노트**를 클릭해요.
2. 노트 이름을 입력한 다음 Enter를 눌러요.

## 새 폴더 만들기

보관함의 루트에 새 폴더를 만들려면:

1. 파일 탐색기 상단의 **새 폴더** ![[lucide-folder-plus.svg#icon]]를 클릭해요.
2. 폴더 이름을 입력한 다음 Enter를 눌러요.

하위 폴더를 만들려면:

1. 하위 폴더를 만들 폴더를 우클릭한 다음 **새 폴더**를 클릭해요.
2. 폴더 이름을 입력한 다음 Enter를 눌러요.

## 정렬 순서 변경

파일의 정렬 순서를 변경하려면:

1.  파일 탐색기 상단의 **정렬 순서 변경** ![[lucide-arrow-up-narrow-wide.svg#icon]]을 선택해요.
2. 파일을 정렬할 방법을 선택해요. 파일 이름, 수정된 시간 또는 생성된 시간별로 오름차순이나 내림차순으로 정렬할 수 있어요.

## 활성 파일 자동 표시

노트를 열면 파일 탐색기가 자동으로 해당 노트로 스크롤하고 폴더 트리에서 강조 표시할 수 있어요. 이를 통해 보관함 내에서 활성 노트의 위치를 추적할 수 있어요.

자동 표시를 전환하려면:

- 파일 탐색기 상단의 **활성 파일 자동 표시** ![[lucide-gallery-vertical.svg#icon]]를 선택해요.

활성화되면 파일 탐색기가 현재 열린 노트를 자동으로 따라가며 표시해요.

## 모든 폴더 펼치기 또는 접기

파일 탐색기의 모든 폴더를 한 번에 펼치거나 접을 수 있어요.

모든 폴더를 펼치려면:

- 파일 탐색기 상단의 **모두 펼치기** ![[lucide-chevrons-up-down.svg#icon]]를 선택해요.

모든 폴더를 접으려면:

- 파일 탐색기 상단의 **모두 접기** ![[lucide-chevrons-down-up.svg#icon]]를 선택해요.

## 파일 또는 폴더 삭제

1. 삭제할 파일을 우클릭한 다음 **삭제**를 클릭해요.
2. 파일 삭제를 확인하라는 메시지가 표시되면 **삭제**를 클릭해요.

자세한 내용은 [[노트 관리#노트 삭제|노트 삭제]]를 참조하세요.

## 파일 또는 폴더 이름 바꾸기

1. 이름을 바꿀 파일을 우클릭한 다음 **이름 바꾸기**를 클릭해요.
2. 새 이름을 입력한 다음 Enter를 눌러요.

자세한 내용은 [[노트 관리#노트 이름 바꾸기|노트 이름 바꾸기]]를 참조하세요.

## 파일 또는 폴더 이동

파일이나 폴더를 이동하려면 드래그 앤 드롭 또는 컨텍스트 메뉴를 사용할 수 있어요.

**드래그 앤 드롭:**

- 파일이나 폴더를 이동하려는 폴더로 드래그해요.
- `Alt-Click` (Windows/Linux) 또는 `Opt-Click` (macOS)으로 여러 개별 파일을 선택하여 다른 폴더로 드래그할 수 있어요. 파일이 모두 연속으로 나열되어 있으면 `Shift-Click`을 사용할 수 있어요.

**컨텍스트 메뉴:**

1. 파일을 우클릭한 다음 **파일을 다음 위치로 이동...**을 선택해요.
2. 파일을 이동할 폴더의 이름을 검색한 다음 목록에서 선택해요.
