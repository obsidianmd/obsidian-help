웹 뷰어는 데스크탑에서 Obsidian 내부에서 외부 링크를 열 수 있게 해주는 [[코어 플러그인|코어 플러그인]]이에요. 앱을 떠나지 않고 링크 콘텐츠를 읽을 수 있어 웹 리서치 프로젝트에서 멀티태스킹이 더 쉬워져요.

외부 링크는 [[탭|탭]]으로 열리며, 재배치하거나 분할하고 [[팝아웃 창|팝아웃 창]]에서 열 수 있어요. [[Canvas]] 파일에 임베드된 웹 페이지 카드도 웹 뷰어 탭으로 열 수 있어요.

웹 뷰어는 기본 브라우저를 대체하는 것이 아니에요. 웹 뷰어는 Obsidian 내에서 리서치를 위해 웹 페이지에 빠르게 접근하는 방법을 제공해요. 하지만 전용 브라우저의 완전한 기능, 보안 제어 또는 확장성을 제공하지는 않아요.

## 리더 뷰

안경 아이콘을 클릭하면 웹 페이지의 일반 텍스트 버전을 볼 수 있어요. 이 기능은 Firefox용으로 개발된 Mozilla의 Readability 라이브러리를 사용하여 콘텐츠를 정리해요.

## 보관함에 저장

더 보기 작업 아이콘을 클릭하면 웹 페이지를 보관함에 저장할 수 있어요. **[[설정]]** → **웹 뷰어**에서 저장 페이지 위치를 사용자 지정할 수 있어요.

## 광고 차단

웹 뷰어는 기본적으로 광고를 차단해요. [Easylist](https://easylist.to/)와 같은 목록을 추가하여 광고 차단 규칙을 사용자 지정할 수 있어요.

## 보안

타사 Obsidian 플러그인을 사용하는 경우, 민감한 작업이나 비밀번호로 보호된 웹사이트에는 웹 뷰어 대신 기본 브라우저를 사용하는 것을 권장해요.

웹 뷰어는 [[Canvas]]에 웹 페이지를 임베드할 수 있게 해주는 동일한 [Chromium](https://developer.chrome.com/docs/apps/reference/webviewTag) 기능을 기반으로 해요. 웹 뷰어는 안전하게 구현되었는지 확인하기 위해 [독립적인 감사](https://obsidian.md/blog/cure53-second-client-audit/)를 받았어요.

Obsidian 플러그인은 [[플러그인 보안#플러그인 기능|샌드박스화되지 않으며]] 앱에 대한 깊은 제어 권한을 가져요. 이 설계는 강력한 기능을 가능하게 하지만 보안상의 절충도 동반해요. Obsidian이 실행 중인 동안 타사 플러그인은 웹 뷰어의 쿠키에 완전히 접근할 수 있어요.
