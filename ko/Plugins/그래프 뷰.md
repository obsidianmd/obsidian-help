그래프 뷰는 보관함에 있는 노트 간의 관계를 시각화할 수 있게 해주는 [[코어 플러그인|코어 플러그인]]입니다.

그래프 뷰를 열려면 [[리본]]에서 **그래프 뷰 열기**를 클릭하세요.

- 원은 노트, 즉 _노드_를 나타냅니다.
- 선은 두 노드 사이의 [[내부 링크]]를 나타냅니다.

주어진 노드를 참조하는 노드가 많을수록 해당 노드는 더 커집니다.

그래프에서 노트와 상호작용하려면:

- 각 원 위에 호버하여 해당 노트의 연결을 강조 표시합니다.
- 그래프에서 노트를 클릭하여 해당 노트를 엽니다.
- 노트를 마우스 오른쪽 버튼으로 클릭하여 해당 노트에 사용할 수 있는 작업이 포함된 컨텍스트 메뉴를 엽니다.

그래프를 탐색하려면:

- 마우스 스크롤 휠을 사용하거나 `+` 및 `-` 키를 사용하여 확대 및 축소합니다.
- 마우스 커서로 드래그하거나 방향키를 사용하여 그래프를 이동합니다.

키보드를 사용할 때 Shift를 누르고 있으면 이동 속도를 높일 수 있습니다.

## 설정

그래프 설정을 열려면 그래프 뷰의 오른쪽 상단 모서리에 있는 톱니바퀴 아이콘을 클릭하세요.

변경한 내용을 초기화하려면 설정 상자 오른쪽 상단의 **기본 설정으로 되돌리기**를 클릭하세요.

### 필터

이 섹션은 그래프에 표시할 노드를 제어합니다.

- **파일 검색**을 사용하면 검색어를 기반으로 노트를 필터링할 수 있습니다. 더 고급 검색어를 작성하는 방법에 대해 알아보려면 [[검색]]을 참조하세요.
- **태그**는 그래프에 태그를 표시할지 여부를 전환합니다.
- **첨부 파일**은 그래프에 첨부 파일을 표시할지 여부를 전환합니다.
- **존재하는 파일만 표시**는 보관함에 존재하는 노트만 표시할지 여부를 전환합니다. 노트에 링크하기 위해 해당 노트가 존재할 필요는 없으므로, 이 옵션은 보관함에 실제로 있는 노트로 그래프를 제한하는 데 도움이 됩니다.
- **미연결 노트**는 링크가 없는 노트를 표시할지 여부를 전환합니다.

> [!info] 제외하는 파일
> [[설정#제외하는 파일|제외하는 파일]] 패턴과 일치하는 파일은 그래프 뷰에 표시되지 않습니다.

### 그룹

색상을 사용하여 노트를 서로 구별할 수 있도록 노트 그룹을 생성합니다.

새 그룹을 생성하려면:

1. **새 북마크 그룹**을 클릭합니다.
2. 검색 상자에 그룹에 추가하려는 노트의 검색어를 입력합니다.
3. 색상이 있는 원을 클릭하여 그룹에 색상을 지정합니다.

더 고급 검색어를 작성하는 방법에 대해 알아보려면 [[검색]]을 참조하세요.

### 표시

이 섹션은 그래프에서 노드와 링크를 시각화하는 방법을 제어합니다.

- **화살표**는 각 링크의 방향을 표시할지 여부를 전환합니다.
- **텍스트 표시 시작점**은 각 노트 이름의 텍스트 투명도를 제어합니다.
- **노드 크기**는 각 노트를 나타내는 원의 크기를 제어합니다.
- **링크 두께**는 각 링크의 선 너비를 제어합니다.
- **애니메이션 재생**은 [[#타임랩스 애니메이션 시작|타임랩스 애니메이션]]을 시작합니다.

### 장력

이 섹션은 그래프에서 각 노드에 작용하는 힘을 제어합니다.

- **중심 장력**은 그래프가 얼마나 밀집되는지를 제어합니다. 값이 높을수록 더 원형에 가까운 그래프가 생성됩니다.
- **반발력**은 노드가 다른 노드를 얼마나 밀어내는지를 제어합니다.
- **링크 장력**은 각 링크에 대한 당기는 힘을 제어합니다. 링크가 고무줄이라면, 링크 장력은 고무줄이 얼마나 팽팽한지 또는 느슨한지를 제어합니다.
- **링크 거리**는 각 노트 사이의 선 길이를 제어합니다.

## 로컬 그래프

로컬 그래프 뷰를 열려면 **로컬 그래프 열기** 명령을 사용하세요. 그래프 뷰가 보관함의 모든 노트를 표시하는 반면, 로컬 그래프 뷰는 활성 노트에 연결된 노트를 표시합니다.

로컬 그래프 뷰는 전역 그래프 뷰에서 사용할 수 있는 모든 [[#설정]]을 사용할 수 있습니다. 또한 로컬 그래프의 깊이를 변경할 수 있습니다. 각 깊이 레벨은 이전 깊이에서 표시된 노트에 연결된 노트를 보여줍니다. 로컬 그래프 깊이를 제어하려면 로컬 그래프 필터 설정 패널 상단의 슬라이더를 사용하세요.
