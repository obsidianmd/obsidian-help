Canvas는 시각적 노트 작성을 위한 [[코어 플러그인|코어 플러그인]]입니다. 노트를 배치하고 다른 노트, 첨부 파일, 웹 페이지와 연결할 수 있는 무한한 공간을 제공합니다.

시각적 노트 작성은 노트를 2D 공간에 정리하여 노트를 더 잘 이해하는 데 도움을 줍니다. 노트를 선으로 연결하고 관련 노트를 그룹으로 묶어 노트 간의 관계를 더 잘 파악할 수 있습니다.

Obsidian에서 생성하는 Canvas 데이터는 개방형 파일 형식인 [JSON Canvas](https://jsoncanvas.org/)를 사용하여 `.canvas` 파일로 저장됩니다.

## 새 캔버스 만들기

Canvas를 사용하려면 먼저 캔버스를 담을 파일을 만들어야 합니다. 다음 방법으로 새 캔버스를 만들 수 있습니다.

**명령어 팔레트:**

1. [[명령어 팔레트]]를 엽니다.
2. **Canvas: 새 캔버스 만들기**를 선택하여 활성 파일과 같은 폴더에 캔버스를 생성합니다.

**파일 탐색기:**

- [[파일 탐색기]]에서 캔버스를 만들 폴더를 마우스 오른쪽 버튼으로 클릭합니다.
- **새 캔버스**를 선택합니다.

**리본:**

- 세로 리본 메뉴에서 **새 캔버스 만들기**( ![[lucide-layout-dashboard.svg#icon]] )를 선택하여 활성 파일과 같은 폴더에 캔버스를 생성합니다.

> [!note] .canvas 확장자
> Obsidian은 [JSON Canvas](https://jsoncanvas.org/)라는 개방형 파일 형식을 사용하여 캔버스 데이터를 `.canvas` 파일로 저장합니다.

## 카드 추가

Obsidian이나 다른 애플리케이션에서 파일을 캔버스로 드래그할 수 있습니다. 예를 들어 Markdown 파일, 이미지, 오디오, PDF, 또는 인식되지 않는 파일 유형도 가능합니다.

### 텍스트 카드 추가

파일을 참조하지 않는 텍스트 전용 카드를 추가할 수 있습니다. 노트에서처럼 Markdown, 링크, 코드 블록을 사용할 수 있습니다.

캔버스에 새 텍스트 카드를 추가하려면:

- 캔버스 하단의 빈 파일 아이콘을 선택하거나 드래그합니다.

캔버스를 더블 클릭하여 텍스트 카드를 추가할 수도 있습니다.

텍스트 카드를 파일로 변환하려면:

1. 텍스트 카드를 마우스 오른쪽 버튼으로 클릭하고 **파일로 변환...**을 선택합니다.
2. 노트 이름을 입력하고 **저장**을 선택합니다.

> [!note] 참고
> 텍스트 전용 카드는 [[백링크]]에 표시되지 않습니다. 표시하려면 파일로 변환해야 합니다.

### 노트에서 카드 추가

보관함의 노트를 캔버스에 추가하려면:

1. 캔버스 하단의 문서 아이콘을 선택하거나 드래그합니다.
2. 추가할 노트를 선택합니다.

캔버스 컨텍스트 메뉴에서도 노트를 추가할 수 있습니다:

1. 캔버스를 마우스 오른쪽 버튼으로 클릭하고 **보관함의 노트 추가**를 선택합니다.
2. 추가할 노트를 선택합니다.

또는 [[파일 탐색기]]에서 파일을 드래그하여 캔버스에 추가할 수 있습니다.

### 미디어에서 카드 추가

보관함의 미디어를 캔버스에 추가하려면:

1. 캔버스 하단의 이미지 파일 아이콘을 선택하거나 드래그합니다.
2. 추가할 미디어 파일을 선택합니다.

캔버스 컨텍스트 메뉴에서도 미디어를 추가할 수 있습니다:

1. 캔버스를 마우스 오른쪽 버튼으로 클릭하고 **보관함의 미디어 추가**를 선택합니다.
2. 추가할 미디어 파일을 선택합니다.

또는 [[파일 탐색기]]에서 파일을 드래그하여 캔버스에 추가할 수 있습니다.

### 웹 페이지에서 카드 추가

캔버스에 웹 페이지를 삽입하려면:

1. 캔버스를 마우스 오른쪽 버튼으로 클릭하고 **웹페이지 추가**를 선택합니다.
2. 웹 페이지의 URL을 입력하고 **저장**을 선택합니다.

브라우저에서 URL을 선택한 후 캔버스로 드래그하여 카드에 삽입할 수도 있습니다.

브라우저에서 웹 페이지를 열려면 `Ctrl`(macOS에서는 `Cmd`)을 누르고 카드 라벨을 선택합니다. 또는 카드를 마우스 오른쪽 버튼으로 클릭하고 **브라우저에서 열기**를 선택합니다.

### 폴더에서 카드 추가

파일 탐색기에서 폴더를 드래그하면 해당 폴더의 모든 파일을 캔버스에 추가할 수 있습니다.

### 카드 편집

텍스트 또는 노트 카드를 더블 클릭하여 편집을 시작합니다. 카드 바깥을 클릭하면 편집이 중지됩니다. `Escape`를 눌러서도 카드 편집을 중지할 수 있습니다.

카드를 마우스 오른쪽 버튼으로 클릭하고 **편집**을 선택하여 편집할 수도 있습니다.

### 카드 삭제

선택한 카드를 마우스 오른쪽 버튼으로 클릭하고 **삭제**를 선택하여 제거합니다. 또는 `Backspace`(macOS에서는 `Delete`)를 누릅니다.

선택 컨트롤에서 **제거**( ![[lucide-trash-2.svg#icon]] )를 선택할 수도 있습니다.

### 카드 교체

노트 또는 미디어 카드를 같은 유형의 다른 카드로 교체할 수 있습니다.

노트 카드를 교체하려면:

1. 교체할 카드를 마우스 오른쪽 버튼으로 클릭합니다.
2. **파일 교체**를 선택합니다.
3. 교체할 노트를 선택합니다.

## 카드 선택

캔버스에서 카드를 클릭하여 선택합니다. 여러 카드를 선택하려면 드래그하여 선택 영역을 그립니다.

`Shift`를 누르고 카드를 클릭하여 기존 선택에서 카드를 추가하거나 제거할 수 있습니다.

`Ctrl+a`(macOS에서는 `Cmd+a`)를 눌러 캔버스의 모든 카드를 선택합니다.

카드의 내용을 스크롤하려면 먼저 카드를 선택해야 합니다.

### 카드 배열

선택한 카드를 드래그하여 이동합니다.

`Alt`(macOS에서는 `Option`)를 누르고 드래그하여 선택 항목을 복제합니다.

드래그 중 `Shift`를 누르면 한 방향으로만 이동합니다.

선택 항목을 이동하는 중에 `Space`를 누르면 스냅을 비활성화합니다.

카드를 선택하면 앞으로 이동합니다.

### 카드 크기 조절

카드의 가장자리를 드래그하여 크기를 조절합니다.

크기 조절 중 `Space`를 누르면 스냅을 비활성화합니다.

크기 조절 중 `Shift`를 누르면 가로세로 비율을 유지합니다.

## 카드 연결

카드 사이에 선을 그려 관계를 만듭니다. 색상과 라벨을 사용하여 서로의 관계를 설명합니다.

### 두 카드 연결

두 카드를 방향이 있는 선으로 연결하려면:

1. 카드의 가장자리 위에 커서를 올려 채워진 원이 나타날 때까지 기다립니다.
2. 원을 다른 카드의 가장자리로 드래그하여 연결합니다.

> [!tip] 팁
> 다른 카드에 연결하지 않고 선을 드래그하면, 연결할 카드를 나중에 추가할 수 있습니다.

### 두 카드 연결 끊기

두 카드 간의 연결을 제거하려면:

1. 연결선 위에 커서를 올려 선 위에 두 개의 작은 원이 나타날 때까지 기다립니다.
2. 원 중 하나를 카드에서 다른 곳으로 드래그합니다.

연결선을 마우스 오른쪽 버튼으로 클릭하고 **제거**를 선택하여 두 카드의 연결을 끊을 수도 있습니다. 또는 선을 선택하고 `Backspace`(macOS에서는 `Delete`)를 누릅니다.

### 다른 카드에 연결

연결선의 한쪽 끝을 이동하려면:

1. 연결선 위에 커서를 올려 선 위에 두 개의 작은 원이 나타날 때까지 기다립니다.
2. 다시 연결할 끝의 원을 다른 카드로 드래그합니다.

### 연결 탐색

연결된 두 카드가 멀리 떨어져 있는 경우, 선을 마우스 오른쪽 버튼으로 클릭하고 **대상으로 이동** 또는 **소스로 이동**을 선택하여 연결의 소스나 대상으로 이동할 수 있습니다.

### 연결에 라벨 추가

두 카드 간의 관계를 설명하기 위해 선에 라벨을 추가할 수 있습니다.

연결에 라벨을 추가하려면:

1. 선을 더블 클릭합니다.
2. 라벨을 입력하고 `Escape`를 누르거나 캔버스의 아무 곳이나 클릭합니다.

선택 컨트롤에서 **라벨 수정**을 선택하여 연결에 라벨을 추가할 수도 있습니다.

연결 라벨을 편집하려면 선을 더블 클릭하거나, 선을 마우스 오른쪽 버튼으로 클릭하고 **라벨 수정**을 선택합니다.

### 카드 또는 연결의 색상 변경

1. 색상을 변경할 카드 또는 연결을 선택합니다.
2. 선택 컨트롤에서 **색상 설정**( ![[lucide-palette.svg#icon]] )을 선택합니다.
3. 색상을 선택합니다.

## 카드 그룹화

### 선택한 카드 그룹화

빈 그룹을 만들려면:

- 캔버스를 마우스 오른쪽 버튼으로 클릭하고 **그룹 생성**을 선택합니다.

관련 카드를 그룹화하려면:

1. 카드를 선택합니다.
2. 선택한 카드 중 하나를 마우스 오른쪽 버튼으로 클릭하고 **그룹 생성**을 선택합니다.

**그룹 이름 바꾸기:** 그룹 이름을 더블 클릭하여 편집하고, `Enter`를 눌러 저장합니다.

## 캔버스 탐색

캔버스에 카드를 더 많이 추가하면 캔버스의 특정 부분을 보기 위해 탐색하는 방법을 알아야 합니다. 이동과 확대를 사용하여 캔버스를 쉽게 이동하는 방법을 알아보세요.

### 캔버스 이동

캔버스를 수직 및 수평으로 이동(_패닝_)하려면 다음 방법 중 하나를 사용합니다:

- `Space`를 누르고 캔버스를 드래그합니다.
- 마우스 가운데 버튼으로 캔버스를 드래그합니다.
- 마우스를 스크롤하여 수직으로 이동하고, `Shift`를 누르며 스크롤하여 수평으로 이동합니다.

### 캔버스 확대

캔버스를 확대하려면 `Space` 또는 `Ctrl`(macOS에서는 `Cmd`)을 누르고 마우스 휠로 스크롤합니다. 또는 오른쪽 상단의 확대 컨트롤에서 **확대**( ![[lucide-plus.svg#icon]] ) 및 **축소**( ![[lucide-minus.svg#icon]] )를 선택합니다.

#### 화면 맞춤 확대

모든 항목이 보이도록 캔버스를 확대하려면 **화면 맞춤 확대**( ![[lucide-maximize.svg#icon]] )를 선택합니다. 또는 키보드 단축키 `Shift+1`을 사용합니다.

#### 선택 맞춤 확대

선택한 모든 항목이 보이도록 캔버스를 확대하려면 선택한 카드를 마우스 오른쪽 버튼으로 클릭하고 **선택 맞춤 확대**를 선택합니다. 또는 키보드 단축키 `Shift+2`를 사용합니다.

#### 화면 비율 초기화

배율을 기본값으로 되돌리려면 오른쪽 상단의 확대 컨트롤에서 **화면 비율 초기화**를 선택합니다.
