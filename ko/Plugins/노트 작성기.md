노트 작성기는 두 노트를 병합하거나 노트의 일부를 새 노트로 추출할 수 있는 [[코어 플러그인|코어 플러그인]]이에요.

## 노트 병합

노트 병합은 하나의 노트를 다른 노트에 추가하고 첫 번째 노트를 제거해요. 노트 작성기는 병합된 노트를 참조하도록 모든 링크를 업데이트해요.

병합할 노트를 선택할 때 다음 방법 중 선택할 수 있어요:

- `Enter`: 원본 노트를 대상 노트의 _끝_에 추가해요.
- `Shift+Enter`: 원본 노트를 대상 노트의 _시작_에 추가해요.
- `Ctrl+Enter` (macOS에서는 `Cmd+Enter`): 원본 노트의 내용으로 새 노트를 생성해요.

활성 노트를 보관함의 다른 노트와 병합하려면:

**파일 탐색기**

1. 파일 탐색기에서 병합하려는 노트를 마우스 오른쪽 버튼으로 클릭해요.
2. **전체 파일을 다음으로 합치기...**를 클릭해요.
3. 병합할 대상 노트를 선택해요.
4. **병합하기**를 클릭하여 확인해요.

**명령어 팔레트**

1. [[명령어 팔레트]]를 열어요.
2. **노트 작성기: 현재 파일을 다음 파일로 합치기...**를 선택해요.
3. 병합할 대상 노트를 선택해요.
4. **병합하기**를 클릭하여 확인해요.

> [!tip] 팁
> 기본적으로 노트 작성기는 노트를 병합할 때 확인을 요청해요. 확인을 비활성화한 후 실수로 노트를 병합한 경우에도 [[파일 복원]] 플러그인으로 복원할 수 있어요.

## 노트 추출

선택 영역을 추출할 노트를 선택할 때 다음 방법 중 선택할 수 있어요:

- `Enter`: 선택한 텍스트를 대상 노트의 _끝_에 추가해요.
- `Shift+Enter`: 선택한 텍스트를 대상 노트의 _시작_에 추가해요.
- `Ctrl+Enter` (macOS에서는 `Cmd+Enter`): 선택한 텍스트로 새 노트를 생성해요.

텍스트를 새 노트로 추출하려면:

**편집기**

1. **편집기**에서 추출하려는 텍스트를 선택해요.
2. 선택한 텍스트를 마우스 오른쪽 버튼으로 클릭해요.
3. **현재 선택 영역 추출...**을 클릭해요.
4. 추출할 대상 노트를 선택해요.

**명령어 팔레트**

1. **편집기**에서 추출하려는 텍스트를 선택해요.
2. [[명령어 팔레트]]를 열어요.
3. **노트 작성기: 현재 선택 영역 추출...**을 선택해요.
4. 추출할 대상 노트를 선택해요.

> [!tip] 팁
> 기본적으로 노트 작성기는 추출된 텍스트를 대상 노트로의 링크로 대체해요. 설정에서 대상 노트를 [[파일 임베드|임베드]]하도록 변경하거나, 아무것도 남기지 않도록 변경할 수도 있어요.

## 템플릿 파일

템플릿을 설정하면 새 노트에 추가하기 전에 내용을 사용자 지정할 수 있어요. 템플릿을 사용하려면 플러그인 설정에서 **템플릿 파일 위치**를 입력하세요.

템플릿에는 다음 변수를 포함할 수 있어요:

| 변수                | 설명                                                                                                                                              |
|-------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| `{{content}}`     | 병합할 내용 또는 추출한 텍스트 선택 영역이에요. 이 변수를 포함하지 않으면 노트 작성기는 템플릿 하단에 내용을 추가해요. |
| `{{fromTitle}}`   | 원본 노트의 이름이에요.                                                                                                                                 |
| `{{newTitle}}`    | 대상 노트의 이름이에요. 예를 들어, 파일 상단에 파일 이름을 제목으로 추가할 수 있어요.                                                     |
| `{{date:FORMAT}}` | 새 노트의 생성 날짜예요. 예: `{{date:YYYY-MM-DD}}`.                                                                                       |
