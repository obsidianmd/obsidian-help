파일을 다른 이름으로 참조하고 싶다면, 노트에 _별칭_을 추가하는 것을 고려하세요. 별칭은 노트의 대체 이름이에요.

별칭은 약어, 닉네임, 또는 다른 언어로 노트를 참조할 때 사용해요.

한 곳에서만 링크의 표시 방식을 변경하고 싶다면, [[내부 링크#링크 표시 텍스트 변경하기|링크 표시 텍스트 변경하기]] 방법을 참고하세요.

![[내부 링크#^callout-internal-links-link-text]]

## 노트에 별칭 추가하기

노트에 별칭을 추가하려면, 노트의 [[속성]]에 `aliases` 속성을 추가하세요. 별칭은 항상 YAML에서 목록 형식으로 작성해야 해요.

```md
---
aliases:
  - Doggo
  - Woofer
  - Yapper
---

# Dog
```

## 별칭을 사용하여 노트에 링크하기

별칭을 사용하여 노트에 링크하려면:

1. [[내부 링크]]에서 별칭을 입력하기 시작하세요. 모든 별칭이 제안 목록에 나타나며, 옆에 곡선 화살표 아이콘이 표시돼요.
2. `Enter`를 눌러 별칭을 선택하세요.

Obsidian은 별칭을 커스텀 표시 텍스트로 사용하여 링크를 생성해요. 예를 들어 `[[Artificial Intelligence|AI]]`와 같아요.

> [!note] 참고
> Obsidian은 별칭을 링크 대상으로만 사용하는 것(`[[AI]]`)이 아니라, 위키 링크 형식을 사용하는 다른 애플리케이션과의 호환성을 보장하기 위해 `[[Artificial Intelligence|AI]]` 링크 형식을 사용해요.

## 별칭에 대한 링크되지 않은 호출 찾기

[[백링크]]를 사용하면 별칭의 링크되지 않은 호출을 찾을 수 있어요.

예를 들어, "Artificial intelligence"의 별칭으로 "AI"를 설정한 후, 다른 노트에서 "AI"가 언급된 곳을 확인할 수 있어요.

링크되지 않은 호출을 별칭에 링크하면, Obsidian은 해당 호출을 별칭을 표시 텍스트로 사용하는 [[내부 링크]]로 변환해요.
