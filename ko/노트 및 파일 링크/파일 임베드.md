노트에 다른 노트와 미디어를 임베드하는 방법을 알아보세요. 노트에 파일을 임베드하면 보관함 전체에서 콘텐츠를 재사용할 수 있어요.

보관함의 파일을 임베드하려면 [[내부 링크|내부 링크]] 앞에 느낌표(`!`)를 추가하세요. [[지원 파일 형식]]에 해당하는 모든 파일을 임베드할 수 있어요.

> [!tip] 드래그 앤 드롭으로 임베드하기
> 데스크톱에서는 지원되는 파일을 노트로 직접 드래그 앤 드롭하여 자동으로 임베드할 수도 있어요.

## 다른 노트에 노트 임베드하기

노트를 임베드하려면:

```md
![[내부 링크]]
```

[[내부 링크#노트의 제목으로 링크|제목]]과 [[내부 링크#노트의 블록으로 링크|블록]]에 대한 링크도 임베드할 수 있어요.

```md
![[내부 링크#^b15695]]
```

아래 텍스트는 임베드된 블록의 예시예요:

![[내부 링크#^b15695]]

## 노트에 이미지 임베드하기

이미지를 임베드하려면:

```md
![[Engelbart.jpg]]
```

![[Engelbart.jpg#outline]]

링크 대상에 `|640x480`을 추가하여 이미지 크기를 변경할 수 있어요. 여기서 640은 너비이고 480은 높이예요.

```md
![[Engelbart.jpg|100x145]]
```

너비만 지정하면 이미지는 원래 가로세로 비율에 따라 조정돼요. 예를 들어, `![[Engelbart.jpg|100]]`.

![[Engelbart.jpg#outline|100]]

Markdown 링크를 사용하여 외부에 호스팅된 이미지를 임베드할 수도 있어요. 위키 링크와 동일한 방식으로 너비와 높이를 제어할 수 있어요.

```md
![250](https://publish-01.obsidian.md/access/f786db9fac45774fa4f0d8112e232d67/Attachments/Engelbart.jpg)
```

![250](https://publish-01.obsidian.md/access/f786db9fac45774fa4f0d8112e232d67/Attachments/Engelbart.jpg)

## 노트에 오디오 파일 임베드하기

오디오 파일을 임베드하려면:

```md
![[Excerpt from Mother of All Demos (1968).ogg]]
```

![[Excerpt from Mother of All Demos (1968).ogg]]

## 노트에 PDF 임베드하기

PDF를 임베드하려면:

```md
![[Document.pdf]]
```

링크 대상에 `#page=N`을 추가하여 PDF의 특정 페이지를 열 수도 있어요. 여기서 `N`은 페이지 번호예요:

```md
![[Document.pdf#page=3]]
```

`#height=[number]`를 링크 대상에 추가하여 임베드된 PDF 뷰어의 높이를 픽셀 단위로 지정할 수도 있어요. 예를 들어:

```md
![[Document.pdf#height=400]]
```

## 노트에 목록 임베드하기

다른 노트의 목록을 임베드하려면 먼저 목록에 [[내부 링크#노트의 블록으로 링크|블록 식별자]]를 추가하세요:

```md

- 목록 항목 1
- 목록 항목 2

^my-list-id
```

그런 다음 블록 식별자를 사용하여 목록에 링크하세요:

```md
![[My note#^my-list-id]]
```
