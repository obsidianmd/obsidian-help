[iframe](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/iframe) HTML 요소를 사용하여 노트에 웹 페이지를 삽입하는 방법을 알아보세요.

웹 페이지를 삽입하려면 노트에 다음을 추가하고 자리 표시자 텍스트를 삽입하려는 웹 페이지의 URL로 바꾸세요:

```html
<iframe src="여기에 URL을 입력하세요"></iframe>
```

> [!note] 참고
> 일부 웹사이트는 임베드를 허용하지 않아요. 대신 임베드용으로 만들어진 URL을 제공하는 경우가 있어요. 웹사이트가 임베드를 지원하지 않는 경우, 웹사이트 이름 뒤에 "embed iframe"을 붙여 검색해 보세요. 예를 들어, "youtube embed iframe"과 같이 검색해요.

> [!tip] 팁
> [[Canvas]]를 사용하는 경우, 카드에 웹 페이지를 삽입할 수 있어요. 자세한 내용은 [[Canvas#웹 페이지에서 카드 추가]]를 참고하세요.

## YouTube 동영상 삽입

YouTube 동영상을 삽입하려면 [[기본 서식 구문#외부 이미지|외부 이미지]]와 동일한 Markdown 구문을 사용하세요:

```md
![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)
```

![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)
