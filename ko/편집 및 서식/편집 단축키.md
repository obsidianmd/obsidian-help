이 페이지는 Obsidian에서 텍스트를 탐색하고 편집하기 위한 기본 키보드 단축키를 나열합니다. 이 단축키는 운영 체제 또는 Obsidian이 기반으로 하는 프레임워크에서 제공되며, Obsidian 내에서 사용자 정의할 수 없습니다.

Obsidian 명령에 대한 사용자 정의 가능한 키보드 단축키는 [[단축키]]를 참조하세요.

## Windows 및 Linux 단축키

### 일반 동작

| 동작 | 단축키 |
|-|-|
| 복사 | `Ctrl+C` |
| 잘라내기 | `Ctrl+X` |
| 붙여넣기 | `Ctrl+V` |
| 서식 없이 붙여넣기 | `Ctrl+Shift+V` |
| 실행 취소 | `Ctrl+Z` |
| 다시 실행 | `Ctrl+Shift+Z` 또는 `Ctrl+Y` |
| 문단 복사 | `Ctrl+C` (텍스트를 선택하지 않은 상태) |
| 문단 잘라내기 | `Ctrl+X` (텍스트를 선택하지 않은 상태) |

### 텍스트 편집

| 동작 | 단축키 |
|-|-|
| 새 줄 삽입 | `Enter` |
| 이전 문자 삭제 | `Backspace` |
| 다음 문자 삭제 | `Delete` |
| 이전 단어 삭제 | `Ctrl+Backspace` |
| 다음 단어 삭제 | `Ctrl+Delete` |
| 현재 줄 삭제 | `Ctrl+Shift+K` (텍스트를 선택하지 않은 상태) |

### 텍스트 탐색

| 동작 | 단축키 |
|-|-|
| 커서를 한 문자 이동 | `Left/→` |
| 커서를 이전 단어의 시작으로 이동 | `Ctrl+←` |
| 커서를 다음 단어의 끝으로 이동 | `Ctrl+→` |
| 커서를 현재 줄의 시작으로 이동 | `Home` |
| 커서를 현재 줄의 끝으로 이동 | `End` |
| 커서를 이전 줄로 이동 | `↑` |
| 커서를 다음 줄로 이동 | `↓` |
| 커서를 노트의 시작으로 이동 | `Ctrl+Home` |
| 커서를 노트의 끝으로 이동 | `Ctrl+End` |
| 커서를 한 페이지 위로 이동 | `Page up` |
| 커서를 한 페이지 아래로 이동 | `Page down` |

### 텍스트 선택

| 동작 | 단축키 |
|-|-|
| 선택 해제 | `Escape` |
| 전체 선택 | `Ctrl+A` |
| 선택 영역을 한 문자 확장 | `Shift+Left/→` |
| 선택 영역을 이전 단어의 시작까지 확장 | `Ctrl+Shift+←` |
| 선택 영역을 다음 단어의 끝까지 확장 | `Ctrl+Shift+→` |
| 선택 영역을 현재 줄의 시작까지 확장 | `Shift+Home` |
| 선택 영역을 현재 줄의 끝까지 확장 | `Shift+End` |
| 선택 영역을 노트의 시작까지 확장 | `Ctrl+Shift+Home` |
| 선택 영역을 노트의 끝까지 확장 | `Ctrl+Shift+End` |
| 선택 영역을 한 페이지 위로 확장 | `Shift+Page up` |
| 선택 영역을 한 페이지 아래로 확장 | `Shift+Page down` |

## macOS 단축키

### 일반 동작

| 동작 | 단축키 |
|-|-|
| 복사 | `Cmd+C` |
| 잘라내기 | `Cmd+X` |
| 붙여넣기 | `Cmd+V` |
| 서식 없이 붙여넣기 | `Cmd+Shift+V` |
| 실행 취소 | `Cmd+Z` |
| 다시 실행 | `Cmd+Shift+Z` |
| 문단 복사 | `Cmd+C` (텍스트를 선택하지 않은 상태) |
| 문단 잘라내기 | `Cmd+X` (텍스트를 선택하지 않은 상태) |

### 텍스트 서식

| 동작 | 단축키 |
| ---- | ---- |
| 굵게 | `Cmd+B` |
| 기울임꼴 | `Cmd+I` |

### 텍스트 편집

| 동작 | 단축키 |
|-|-|
| 새 줄 삽입 | `Enter` |
| 이전 문자 삭제 | `Backspace` |
| 다음 문자 삭제 | `Delete` |
| 이전 단어 삭제 | `Option+Backspace` |
| 다음 단어 삭제 | `Option+Delete` |
| 현재 줄의 시작까지 삭제 | `Cmd+Backspace` |
| 현재 줄의 끝까지 삭제 | `Cmd+Delete` |
| 현재 줄 삭제 | `Cmd+Shift+K` (텍스트를 선택하지 않은 상태) |

### 텍스트 탐색

| 동작 | 단축키 |
|-|-|
| 커서를 한 문자 이동 | `Left/→` |
| 커서를 이전 단어의 시작으로 이동 | `Option+←` |
| 커서를 다음 단어의 끝으로 이동 | `Option+→` |
| 커서를 현재 줄의 시작으로 이동 | `Cmd+←` |
| 커서를 현재 줄의 끝으로 이동 | `Cmd+→` |
| 커서를 이전 줄로 이동 | `↑` |
| 커서를 다음 줄로 이동 | `↓` |
| 커서를 노트의 시작으로 이동 | `Cmd+↑` |
| 커서를 노트의 끝으로 이동 | `Cmd+↓` |
| 커서를 한 페이지 위로 이동 | `Fn+↑` |
| 커서를 한 페이지 아래로 이동 | `Fn+↓` |

### 텍스트 선택

| 선택 해제                                               | `Escape`                   |

| 선택 영역을 다음 단어의 끝까지 확장                         | `Option+Shift+→` |
| 선택 영역을 현재 줄의 시작까지 확장                         | `Cmd+Shift+←`              |
| 선택 영역을 현재 줄의 끝까지 확장                           | `Cmd+Shift+→`    |
| 선택 영역을 노트의 시작까지 확장                            | `Cmd+Shift+↑`       |
| 선택 영역을 노트의 끝까지 확장                              | `Cmd+Shift+↓`     |
