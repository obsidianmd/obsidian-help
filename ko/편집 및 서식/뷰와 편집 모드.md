Obsidian에서는 _뷰_와 _모드_를 사용하여 Markdown으로 작성된 노트를 읽고 편집하는 방식을 제어할 수 있어요.

- **뷰**는 노트를 읽기와 편집 사이에서 전환해요.
- **모드**는 편집 중 Markdown이 표시되는 방식을 제어해요.

> [!note] 참고
> 기본적으로 Obsidian은 새 탭을 편집 모드로 열어요. **[[설정]] → 편집기 → 새 탭에 사용하는 기본 화면**에서 변경할 수 있어요.

## 읽기 보기

*읽기 보기*는 Markdown 구문 없이 노트를 표시하여, 집중적으로 검토할 수 있는 깔끔하고 읽기 쉬운 형식을 제공해요.

*읽기 보기*로 전환하려면:

- 편집기 오른쪽 상단의 화면 전환 아이콘![[lucide-book-icon.svg#icon]]을 클릭해요.
- 또는 상태 표시줄의 대화형 상태 아이콘 ![[lucide-edit-3.svg#icon]] 또는 ![[lucide-code-xml.svg#icon]] 을 클릭하고 **읽기 보기**를 선택해요.
- 또는 `Ctrl+E`(macOS에서는 `Cmd+E`)를 눌러요.

> [!note] 참고
> **[[설정]] → 외형 → 탭 타이틀바 표시**를 활성화하면 화면 전환 아이콘이 표시돼요.
> 
> 이 옵션이 비활성화된 경우, **[[설정]] → 편집기 → 편집기 상태 보기**를 켜면 상태 표시줄에서 뷰를 전환할 수 있어요.

> [!tip] 나란히 보기
> 노트를 *편집기*와 *읽기 보기*에서 동시에 열려면, `Ctrl`(macOS에서는 `Cmd`)을 누른 채 화면 전환 아이콘을 클릭해요.

## 편집기

*편집기*에서는 노트를 변경할 수 있어요.

*편집기*에서는 *편집 모드*가 Markdown 표시 방식을 결정해요. *실시간 미리보기*와 *원본 모드*, 두 가지 *편집 모드* 중 하나를 선택할 수 있어요.

### 실시간 미리보기

*실시간 미리보기*는 대부분의 Markdown 구문을 숨기면서 서식이 적용된 텍스트를 인라인으로 표시해요. 커서가 서식이 적용된 콘텐츠에 들어가면 편집을 위해 기본 구문이 표시돼요.

*실시간 미리보기*로 전환하려면:

- 편집기 오른쪽 상단의 화면 전환 아이콘![[lucide-edit-3.svg#icon]]을 클릭해요.
- 또는 상태 표시줄의 대화형 상태 아이콘 ![[lucide-book-icon.svg#icon]] 또는 ![[lucide-code-xml.svg#icon]] 을 클릭하고 **실시간 미리보기**를 선택해요.
- 또는 `Ctrl+E`(macOS에서는 `Cmd+E`)를 눌러요.
- 또는 [[명령어 팔레트|명령]] **편집/미리보기 화면 전환**을 사용해요.

> [!note] 참고
> 기본적으로 편집기는 *실시간 미리보기*로 설정되어 있어요. **[[설정]] → 편집기 → 기본 편집 모드**에서 변경할 수 있어요.

> [!tip] 대부분의 경우, 실시간 미리보기를 사용하면 [[#읽기 보기]]로 전환할 필요가 없어요.

### 원본 모드

*원본 모드*는 모든 Markdown 구문을 작성된 그대로 표시해요. 일반 텍스트를 선호하거나 정밀한 서식 제어가 필요할 때 사용해요.

*원본 모드*로 전환하려면:
- 상태 표시줄의 대화형 상태 아이콘 ![[lucide-book-icon.svg#icon]] 또는 ![[lucide-edit-3.svg#icon]] 을 클릭하고 **원본 모드**를 선택해요.

> [!note] 참고
> 기본적으로 편집기는 *실시간 미리보기*로 설정되어 있어요. **[[설정]] → 편집기 → 기본 편집 모드**에서 *원본 모드*로 변경할 수 있어요.
> 
> *원본 모드*로 전환하려면 추가로 다음 방법을 사용할 수 있어요:
> 
> - 노트 오른쪽 상단의 화면 전환 아이콘![[lucide-edit-3.svg#icon]]을 클릭해요.
> - 또는 `Ctrl+E`(macOS에서는 `Cmd+E`)를 눌러요.
> - 또는 [[명령어 팔레트|명령]] **편집/미리보기 화면 전환**을 사용해요.

> [!tip] *편집 모드* 전환
> *실시간 미리보기*와 *원본 모드* 사이를 빠르게 전환하려면, [[단축키|키보드 단축키]]를 [[명령어 팔레트|명령]] **실시간 미리보기/원본 모드 전환**에 설정할 수 있어요.
