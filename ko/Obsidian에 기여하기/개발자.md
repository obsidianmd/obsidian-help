개발자라면 다양한 방법으로 Obsidian에 기여할 수 있습니다.

### 플러그인 및 테마

Typescript와 CSS를 사용하여 [[커뮤니티 플러그인]]과 [[테마]]를 개발할 수 있습니다. 자세한 내용은 [Obsidian 개발자 문서](https://docs.obsidian.md)를 참조하세요.

### 오픈 소스 프로젝트

Obsidian은 여러 오픈 소스 프로젝트를 개발하고 유지 관리합니다. 다음 GitHub 저장소에 버그 수정 및 개선 사항에 대한 풀 리퀘스트를 제출할 수 있습니다:

- [JSON Canvas](https://github.com/obsidianmd/jsoncanvas) (MIT), 무한 캔버스 데이터를 위한 개방형 파일 형식.
- [Obsidian Importer](https://github.com/obsidianmd/obsidian-importer) (MIT), 다양한 형식의 데이터를 일반 텍스트 Markdown 파일로 변환하는 플러그인.
- [Obsidian Maps](https://github.com/obsidianmd/obsidian-maps) (MIT), Obsidian 베이스에서 지도를 표시하는 플러그인.
- [Obsidian Web Clipper](https://github.com/obsidianmd/obsidian-clipper) (MIT), 웹 페이지를 강조 표시하고 일반 텍스트 Markdown 파일로 캡처하는 브라우저 확장 프로그램.
- [Defuddle](https://github.com/kepano/defuddle) (MIT), Obsidian Web Clipper에서 사용하는 HTML-to-Markdown 파서.

### 문서

[개발자 문서](https://docs.obsidian.md/Home)를 개선하는 데 도움을 주시려면 [obsidian-developer-docs](https://github.com/obsidianmd/obsidian-developer-docs) 저장소에 풀 리퀘스트를 제출하세요.

이 문서를 개선하는 데 도움을 주시려면 [obsidian-help](https://github.com/obsidianmd/obsidian-help) 저장소에 풀 리퀘스트를 제출하세요.
