이 페이지는 [[Obsidian Publish 소개|Obsidian Publish]] 사이트를 관리하는 방법을 설명해요.

사이트는 Obsidian Publish에서 호스팅되며, Obsidian Publish 주소 또는 [[사용자 지정 도메인|사용자 지정 도메인]]을 통해 온라인으로 접근할 수 있는 노트 모음이에요.

## 새 사이트 만들기

> [!note] 동시에 보유할 수 있는 사이트 수는 Obsidian Publish 구독에 따라 결정돼요. 새 사이트를 만들기 전에 구독이 이를 허용하는지 확인하세요.

1. 앱 창 왼쪽의 리본에서 **변경 사항 게재** ( ![[lucide-send.svg#icon]] )를 클릭해요.
2. 이미 기존 사이트가 있는 경우 **사이트 전환** ( ![[lucide-repeat.svg#icon]] )을 클릭해요.
3. **사이트 ID**에 사이트에 사용할 경로를 입력해요. 예를 들어, ID가 `my-amazing-site`인 사이트는 publish.obsidian.md/my-amazing-site에서 접근할 수 있어요.
4. **생성**을 클릭해요.

## 기존 사이트 삭제

> [!note] 사이트를 삭제해도 보관함의 노트는 그대로 남아 있어요.

1. 앱 창 왼쪽의 리본에서 **변경 사항 게재** ( ![[lucide-send.svg#icon]] )를 클릭해요.
2. **사이트 전환** ( ![[lucide-repeat.svg#icon]] )을 클릭해요.
3. 삭제할 사이트 오른쪽의 **사이트 삭제** ( ![[lucide-x.svg#icon]] )를 클릭해요.
4. **삭제**를 클릭하여 확인해요.

## 사이트 간 전환

1. 앱 창 왼쪽의 리본에서 **변경 사항 게재** ( ![[lucide-send.svg#icon]] )를 클릭해요.
2. **사이트 전환** ( ![[lucide-repeat.svg#icon]] )을 클릭해요.
3. 전환할 사이트 오른쪽의 **선택**을 클릭해요.

## 사이트 ID 변경

1. 앱 창 왼쪽의 리본에서 **변경 사항 게재** ( ![[lucide-send.svg#icon]] )를 클릭해요.
2. **사이트 전환** ( ![[lucide-repeat.svg#icon]] )을 클릭해요.
3. 편집할 사이트 오른쪽의 **사이트 ID 수정** ( ![[lucide-edit-3.svg#icon]] )을 클릭해요.
4. **사이트 ID**에 사이트의 새 ID를 입력해요.
5. **변경**을 클릭해요.

## 사이트 옵션 보기

1. 앱 창 왼쪽의 리본에서 **변경 사항 게재** ( ![[lucide-send.svg#icon]] )를 클릭해요.
2. **변경 사항 게재** 대화 상자에서 **사이트 옵션 변경** ( ![[lucide-cog.svg#icon]] )을 클릭해요.

## 사이트 옵션

### 일반

| 옵션                          | 유형   | 설명                                                                                                                |
| ------------------------------- | ------ | -------------------------------------------------------------------------------------------------------------------------- |
| 사이트 이름                       | 입력  | Obsidian Publish 사이트의 공개 이름 및 제목이에요.                                                                   |
| 홈페이지 파일                   | 입력  | 랜딩 페이지로 사용할 Markdown 파일의 위치예요.                                                    |
| 로고                            | 입력  | 사이트 배너로 사용할 이미지예요. 이미지는 [[콘텐츠 게재#노트 게재\|게재된 상태]]여야 해요. |
| 사이트 공동 작업              | 버튼 | 게재된 노트를 편집할 수 있도록 다른 사용자를 지정해요. 사용자는 Obsidian 계정이 있어야 해요.             |
| 사용자 지정 도메인                   | 버튼 | [[사용자 지정 도메인]]                                                                                                         |
| 검색 엔진 인덱싱을 사용하지 않음 | 토글 | `robots.txt` 파일을 추가하여 검색 엔진이 사이트를 크롤링하지 못하도록 해요.                                   |

### 외형

| 옵션            | 유형     | 설명                                                                               |
|-------------------|----------|-------------------------------------------------------------------------------------------|
| 테마             | 드롭다운 | 사이트에서 테마가 표시되는 방식을 선택해요: **라이트**, **다크**, 또는 **시스템 테마 적용**. |
| 라이트/다크 테마 전환 | 토글   | 사용자가 사이트에서 토글 버튼으로 **라이트**/**다크** 모드를 전환할 수 있도록 해요.          |

## 읽는 방식

| 옵션               | 유형   | 설명                                                                                |
|----------------------|--------|--------------------------------------------------------------------------------------------|
| 마우스 오버 시 미리보기 표시   | 토글 | 활성화된 내부 링크에 마우스를 올렸을 때 페이지 미리보기 기능을 활성화 또는 비활성화해요. |
| 페이지 제목 숨기기      | 토글 | 게재된 노트의 인라인 제목 표시를 활성화 또는 비활성화해요.                    |
| 읽기 쉬운 행 길이 | 토글 | 사이트 내 읽기 쉬운 행 길이를 활성화 또는 비활성화해요.                                   |
| 엄격한 줄 바꿈   | 토글 | 사이트에서 단일 줄 바꿈 표시를 활성화 또는 비활성화해요.                    |
| 슬라이딩 창          | 토글 | 사이트에서 [[탭#탭 그룹 쌓기\|스택 탭]]을 활성화 또는 비활성화해요.    |

### 구성 요소

| 옵션                 | 유형   | 설명                                                                                                                          |
|------------------------|--------|--------------------------------------------------------------------------------------------------------------------------------------|
| 내비게이션 표시        | 토글 | Publish 사이트에서 [[파일 탐색기]] 뷰를 활성화 또는 비활성화해요.                                                                   |
| 사용자 지정 내비게이션   | 버튼 | **내비게이션 표시**가 활성화된 경우 파일 나열 순서를 [[사이트 커스터마이즈#내비게이션 사용자 지정\|사용자 지정]]해요. |
| 검색바 표시        | 토글 | 사이트에서 검색바를 활성화 또는 비활성화해요.                                                                                         |
| 그래프 화면 표시        | 토글 | 사이트 오른쪽 사이드바에서 그래프 뷰를 활성화 또는 비활성화해요.                                                              |
| 목차 표시 | 토글 | 사이트에서 [[개요\|목차]] 뷰를 활성화 또는 비활성화해요.                                                              |
| 백링크 보기         | 토글 | 사이트에서 [[백링크]]를 활성화 또는 비활성화해요.                                                                                        |

### 기타 사이트 설정

| 옵션                         | 유형   | 설명                                            |
| ------------------------------ | ------ | ------------------------------------------------------ |
| 비밀번호                      | 버튼 | 전체 사이트에 대한 접근을 제한하기 위해 [[Obsidian Publish/보안 및 개인정보 보호#사이트 비밀번호 추가\|비밀번호를 설정]]해요. |
| Google Analytics 추적 코드 | 입력  | **사용자 지정 도메인 URL 전용**. Google Analytics 사이트 추적 코드를 여기에 입력해요.                                                       |
