이 페이지에서는 게재된 콘텐츠를 관리하는 방법을 설명합니다. 사이트 스타일링을 사용자 지정하는 방법은 [[사이트 사용자 지정]]을 참조하세요.

## 전제 조건

- Obsidian 계정. 계정이 없다면 [회원가입](https://obsidian.md/auth?returnto=%2Faccount%2Fpublish#signup)하세요.
- 활성화된 Obsidian Publish 구독. 구독이 없다면 [계정 대시보드](https://obsidian.md/account/publish)에서 구독하세요.
- **Publish** 코어 플러그인이 [[Obsidian Publish 설정#Obsidian Publish 활성화|활성화]]되어 있어야 합니다.
- [[사이트 관리#새 사이트 만들기|Publish 사이트]]가 생성되어 있어야 합니다.

## 노트 게재

1. **리본**에서 **변경사항 게재**(![[lucide-send.svg#icon]])를 선택합니다.
2. **변경사항 게재** 대화 상자에서 **추가**를 선택하여 게재되지 않은 모든 노트를 확인합니다.
3. 게재할 노트를 선택합니다.
4. **게재**를 선택합니다.

## 노트 게재 취소

노트를 게재 취소해도 로컬 보관함에는 그대로 남아 있습니다.

1. **리본**에서 **변경사항 게재**(![[lucide-send.svg#icon]])를 선택합니다.
2. **변경사항 게재** 대화 상자에서 **게재 완료**를 선택하여 게재된 모든 노트를 확인합니다.
3. 게재를 취소할 노트를 선택합니다.
4. **게재**를 선택합니다.

## 게재된 노트 업데이트

1. **리본**에서 **변경사항 게재**(![[lucide-send.svg#icon]])를 선택합니다.
2. **변경사항 게재** 대화 상자에서 **변경된 사항**을 선택하여 마지막 게재 이후 수정된 모든 노트를 확인합니다.
3. 업데이트할 노트를 선택합니다.
4. **게재**를 선택합니다.

> [!hint] 이름이 변경되었거나 제거된 노트 및 이미지를 Publish에서 삭제하는 작업은 이 단계에서 이루어집니다. 안전을 위해 자동으로 선택되지 않으므로 해당 데이터를 삭제하려면 체크박스를 수동으로 선택해야 합니다.

## 링크된 데이터 게재

다른 노트로의 링크나 임베드된 이미지가 포함된 노트를 게재할 때, 링크된 노트도 함께 게재하지 않으면 깨진 링크가 발생할 수 있습니다. **Obsidian Publish**는 이미 선택한 노트에서 링크된 미디어를 자동으로 선택하여 이를 방지합니다.

링크된 모든 노트를 포함하려면 **변경사항 게재** 대화 상자에서 **링크된 파일 추가**를 선택하세요.

게재하기 전에 업데이트된 선택 항목을 검토하여 아직 게재할 준비가 되지 않은 데이터가 포함되어 있지 않은지 확인하세요.

> [!tip] **링크된 파일 추가** 기능은 [[#데이터 무시]]에서 정의된 제외 항목을 따릅니다.

## 게재할 데이터 자동 선택

노트의 [[속성]]에 `publish: true`를 설정하면 새 노트 또는 변경된 노트로 게재 시 자동으로 포함됩니다.

특정 폴더의 노트와 링크된 이미지를 **포함** 폴더로 추가하여 자동으로 선택할 수도 있습니다:

1. **리본**에서 **변경사항 게재**(![[lucide-send.svg#icon]])를 선택하거나 [[명령어 팔레트]]를 열고 **Publish: Publish changes...**를 입력합니다.
2. **개제 필터 관리**(![[lucide-filter.svg#icon]]) 아이콘을 선택합니다.
3. **포함되는 폴더** 섹션에서 **관리**를 선택합니다.
4. 추천 목록에서 포함할 폴더를 선택합니다.
5. 폴더가 포함 목록에 추가됩니다.
6. 완료되면 **완료**를 선택합니다.

### 데이터 무시

Obsidian Publish에서 노트를 무시하려면 노트의 [[속성]]에 `publish: false`를 설정하세요. 해당 노트는 더 이상 게재할 노트 목록에 나타나지 않습니다.

특정 폴더의 노트와 이미지를 **제외** 폴더로 추가하여 자동으로 무시할 수도 있습니다:

1. **리본**에서 **변경사항 게재**(![[lucide-send.svg#icon]])를 선택하거나 [[명령어 팔레트]]를 열고 **Publish: Publish changes...**를 입력합니다.
2. **개제 필터 관리**(![[lucide-filter.svg#icon]]) 아이콘을 선택합니다.
3. **제외되는 폴더** 섹션에서 **관리**를 선택합니다.
4. 추천 목록에서 제외할 폴더를 선택합니다.
5. 폴더가 제외 목록에 추가됩니다.
6. 완료되면 **완료**를 선택합니다.

> [!note] `publish: true`는 제외된 폴더보다 우선합니다
> 파일에 `publish: true`가 설정되어 있으면 제외된 폴더나 필터에 있더라도 여전히 게재됩니다. 이는 `publish: true`가 더 구체적인 제어를 제공하기 때문입니다.
