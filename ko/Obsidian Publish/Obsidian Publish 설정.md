이 가이드에서는 보관함에 [[Obsidian Publish 소개|Obsidian Publish]]를 활성화하고, 사이트를 생성하고, 노트를 게재하는 방법을 안내합니다.

## 전제 조건

- Obsidian 계정. 계정이 없다면 [회원가입](https://obsidian.md/auth?returnto=%2Faccount%2Fpublish#signup)하세요.
- 활성화된 Obsidian Publish 구독. 구독이 없다면 [계정 대시보드](https://obsidian.md/account/publish)에서 구독하세요.

## Obsidian 계정으로 로그인

1. **[[설정]]**을 엽니다.
2. 사이드바에서 **일반**을 선택합니다.
3. **계정 → 내 계정**에서 **로그인**을 선택합니다.
4. **이메일**에 이메일을 입력합니다.
5. **비밀번호**에 비밀번호를 입력합니다.
6. **로그인**을 클릭합니다.

## Obsidian Publish 활성화

1. **[[설정]]**을 엽니다.
2. 사이드바에서 **코어 플러그인**을 선택합니다.
3. **Publish**를 활성화합니다.

### Publish 열기

명령어 팔레트, 키보드 단축키 또는 리본을 통해 Publish에 접근할 수 있습니다.

#### 명령어 팔레트로 열기

> [!info] 이 옵션을 사용하려면 [[명령어 팔레트]] 코어 플러그인이 활성화되어 있어야 합니다.

**데스크톱/태블릿** ![[lucide-monitor-check.svg#icon]]

1. **명령어 팔레트**를 엽니다 ( ![[lucide-terminal.svg#icon]] ).
2. "Publish"를 입력합니다.
3. **Publish: 변경사항 게재...** 옵션을 선택합니다.

**변경사항 게재** 창이 새 팝업으로 나타납니다.

**모바일** ![[obsidian-icon-smartphone.svg#icon]]

1. [[리본]]을 엽니다 ( ![[lucide-menu.svg#icon]] ).
2. **명령어 팔레트**를 엽니다 ( ![[lucide-terminal.svg#icon]] ).
3. "Publish"를 입력합니다.
4. **Publish: 변경사항 게재...** 옵션을 선택합니다.

**변경사항 게재** 메뉴가 슬라이드업으로 나타납니다.

#### 키보드 단축키로 열기

1. **[[설정]]**을 엽니다.
2. **옵션** 카테고리에서 **단축키**를 선택합니다.
3. 단축키 검색창에 "Publish"를 입력합니다.
4. **Publish: 변경사항 게재...** 옆에 원하는 키보드 단축키를 지정합니다.
5. **[[설정]]**을 닫습니다.
6. 키보드 단축키를 사용하여 **변경사항 게재**를 엽니다.

#### 리본으로 열기

[[리본]]을 통해서도 Obsidian Publish에 접근할 수 있습니다.

**데스크톱/태블릿** ![[lucide-monitor-check.svg#icon]]

1. 왼쪽 [[사이드바]]를 엽니다 ( ![[obsidian-icon-side-bar-left.svg#icon]] ).
2. **변경사항 게재** ( ![[lucide-send.svg#icon]] ) 아이콘을 선택합니다.

**변경사항 게재** 창이 새 팝업으로 나타납니다.

**모바일** ![[obsidian-icon-smartphone.svg#icon]]

1. [[리본]]을 엽니다 ( ![[lucide-menu.svg#icon]] ).
2. **변경사항 게재** ( ![[lucide-send.svg#icon]] ) 아이콘을 선택합니다.

**변경사항 게재** 메뉴가 슬라이드업으로 나타납니다.

## 새 사이트 생성

1. [[#Publish 열기]]
2. **사이트 ID**에 사이트에 사용할 경로를 입력합니다. 예를 들어, ID가 `my-amazing-site`인 사이트는 `publish.obsidian.md/my-amazing-site`에서 접근할 수 있습니다.
3. **생성**을 선택합니다.

### 사이트 이름 설정

1. [[#Publish 열기]]
2. **사이트 옵션 변경** ( ![[lucide-cog.svg#icon]] )을 선택합니다.
3. **일반** 아래의 **사이트 이름**에 사이트 이름을 입력합니다.
4. **사이트 설정 저장**을 선택합니다.

> [!abstract] **추가 읽기**: [[사이트 관리]], [[사이트 커스터마이즈|사이트 사용자 지정]]

## 게재할 노트 선택

1. [[#Publish 열기]]
2. **추가**를 선택하여 아직 게재하지 않은 모든 노트를 확인합니다.
3. 게재할 노트를 선택합니다.
4. **게재**를 선택합니다.

> [!abstract] **추가 읽기**: [[콘텐츠 게재]], [[보안 및 개인정보 보호|보안 및 개인정보 보호]], [[Publish 제한 사항|Publish 제한 사항]]

Obsidian Publish 사이트를 설정하고 첫 번째 노트를 게재했습니다. 이 가이드에서 이전에 설정한 사이트 주소로 이동하여 독자들과 공유해 보세요.

## 다음 단계

첫 번째 노트를 게재했으므로, 다음 작업을 진행할 수 있습니다:
- 사이트에 사용자 지정 도메인 설정하기: [[사용자 지정 도메인]]
- 고급 사이트 사용자 지정에 대해 알아보기: [[사이트 커스터마이즈|사이트 사용자 지정]]
- 소셜 미디어 카드 추가 방법 알아보기: [[소셜 미디어 링크 미리보기]]
