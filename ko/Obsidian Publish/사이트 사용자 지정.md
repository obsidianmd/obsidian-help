이 페이지에서는 [[Obsidian Publish 소개|Obsidian Publish]] 사이트의 모양과 느낌을 사용자 지정하는 방법을 설명합니다.

## 정적 에셋

다음 파일을 사이트에 [[콘텐츠 게재#노트 게재|게재]]하여 사이트를 사용자 지정할 수 있습니다:

- `publish.css` — 사용자 지정 CSS 추가
- `publish.js` — 사용자 지정 JavaScript 추가
- `favicon-32x32.png` — 파비콘 설정

**참고:**

- [Publish용 CSS 변수](https://docs.obsidian.md/Reference/CSS+variables/Publish/Publish)는 문서 사이트에서 확인할 수 있습니다.
- Obsidian은 CSS 또는 JavaScript 파일을 지원하지 않으므로, 다른 애플리케이션을 사용하여 생성하고 편집해야 합니다.
- `publish.css`와 `publish.js`는 모두 보관함의 루트 디렉터리(`/`)에 있어야 합니다.
- 기본적으로 `publish.css`와 `publish.js`는 파일 탐색기에 표시되지 않지만, **변경사항 게재** 대화 상자에서 게재할 수 있습니다.
- `publish.js`로 사용자 지정 JavaScript를 사용하려면 [[사용자 지정 도메인]]이 필요합니다.

파비콘의 경우, Obsidian Publish는 다음 명명 규칙을 지원합니다. 여기서 `32`는 아이콘 크기(픽셀 단위)를 나타냅니다:

- `favicon-32.png`
- `favicon-32x32.png`
- `favicon.ico`

다음 크기 중 하나 이상을 제공하는 것을 권장합니다:

- `favicon-32x32.png`
- `favicon-128x128.png`
- `favicon-152x152.png`
- `favicon-167x167.png`
- `favicon-180x180.png`
- `favicon-192x192.png`
- `favicon-196x196.png`

파비콘은 사이트에 게재되기만 하면 보관함 내 어디에든 자유롭게 배치할 수 있습니다.

## 커뮤니티 테마 사용

사이트에 커뮤니티 테마를 사용하려면:

1. OS의 기본 파일 탐색기에서 보관함을 엽니다.
2. 보관함 설정 폴더(기본값: `.obsidian`)로 이동합니다.
3. `themes` 폴더를 엽니다.
4. 사이트에 사용할 테마의 CSS 파일을 복사합니다.
5. 보관함의 루트 폴더에 파일을 붙여넣습니다.
6. CSS 파일 이름을 `publish.css`로 바꿉니다.
7. `publish.css`를 [[콘텐츠 게재#노트 게재|게재]]합니다.

**참고:**

- 몇 분 내에 스타일이 업데이트되지 않으면 브라우저 캐시를 새로 고침해 보세요.
- [[사이트 관리#사이트 옵션 보기|사이트 옵션]]에서 라이트 모드와 다크 모드를 전환할 수 있습니다.
- 많은 커뮤니티 테마가 사용자 지정 스타일링을 위해 **Style Settings**를 사용하지만, 이 설정은 Obsidian Publish에서 작동하지 않습니다.

> [!tip] 테마 개발
> 원하는 테마를 찾을 수 없나요? [Publish 테마 만들기](https://docs.obsidian.md/Themes/Obsidian+Publish+themes/Build+a+Publish+theme) 방법을 직접 배워보세요.

## UI 기능 활성화

그래프 뷰나 목차 등 사이트의 여러 UI 기능을 토글할 수 있습니다.

[[사이트 관리#사이트 옵션 보기|사이트 옵션]]의 **읽는 방식** 및 **구성 요소** 섹션에서 사용 가능한 UI 기능을 탐색하세요.

### 내비게이션 사용자 지정

Obsidian Publish에서는 Publish [[파일 탐색기]] 내의 파일과 폴더의 내비게이션 순서와 표시를 사용자 지정할 수 있습니다. 내비게이션 항목은 기본적으로 게재된 순서대로 나열됩니다. 게재되지 않은 노트는 이 창에 표시되지 않습니다.

#### 사용자 지정 내비게이션 옵션에 접근하기

1. [[리본]]에서 **변경사항 게재**(![[lucide-send.svg#icon]])를 선택하거나 [[명령어 팔레트]]를 열고 **Publish: Publish changes...**를 입력합니다.
2. **변경사항 게재** 대화 상자에서 **사이트 옵션 변경**( ![[lucide-cog.svg#icon]] )을 선택합니다.
3. **구성 요소 설정**에서 **사용자 지정 내비게이션** 옆의 **관리** 버튼을 선택합니다.

**사이트 옵션 변경** 창 위에 **내비게이션**이라는 제목의 새 팝업 창이 나타납니다.

#### 내비게이션 항목 조정

**내비게이션 미리보기** 섹션에서 게재된 콘텐츠의 표시 순서를 조정할 수 있습니다.

1. 조정할 폴더 또는 노트를 선택합니다.
2. 원하는 위치까지 노트 또는 폴더를 위아래로 드래그합니다.
3. **내비게이션** 창의 오른쪽 하단에서 **완료**를 선택합니다.

Publish가 내비게이션 변경사항을 사이트에 전송합니다.

#### 내비게이션 항목 숨기기 및 숨기기 해제

게재했지만 내비게이션에 표시하고 싶지 않은 노트나 폴더가 있는 경우, 해당 항목을 숨길 수 있습니다.

1. 조정할 폴더 또는 노트를 선택합니다.
2. 마우스 오른쪽 버튼을 클릭하고 **내비게이션에서 숨기기**를 선택합니다. 항목이 **내비게이션 미리보기**에서 사라집니다.
3. **내비게이션** 창의 오른쪽 하단에서 **완료**를 선택합니다.

Publish가 내비게이션 변경사항을 사이트에 전송합니다.

> [!tip] **내비게이션 미리보기** 제목 오른쪽의 체크박스를 선택하여 **숨겨진 항목 표시**를 할 수 있습니다.

## FAQ

**내비게이션에서 파일을 한 폴더에서 다른 폴더로 이동할 수 있나요?**

아니요. 폴더 내 노트의 파일 내비게이션 구조는 유지되어야 합니다. 폴더 내(보관함 루트 포함) 노트의 순서와 다른 폴더 내의 폴더 순서는 조정할 수 있습니다.

**완료를 선택하기 전에 여러 노트와 폴더의 순서를 편집할 수 있나요?**

예.

**이 변경사항을 되돌리려면 어떻게 하나요?**

- **표시 순서**: **내비게이션 항목 표시 순서** 옆의 **기본값 복원** 아이콘(반시계 방향 회전 화살표)을 선택합니다. 내비게이션 항목이 알파벳 순으로 복원됩니다.
- **숨김 상태**: **페이지나 폴더를 내비게이션 영역으로부터 숨깁니다.** 옆의 **기본값 복원** 아이콘(반시계 방향 회전 화살표)을 선택합니다. 숨겨진 내비게이션 항목이 표시 상태로 복원됩니다.
