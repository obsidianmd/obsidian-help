목록은 [[베이스 소개|베이스]]에서 사용할 수 있는 [[보기|뷰]]의 한 유형이에요.

뷰 메뉴에서 ![[lucide-list.svg#icon]]  **목록**을 선택하면 파일을 글머리 기호 또는 번호 매기기 마커가 포함된 [[기본 서식 구문#목록|목록]] 형태로 표시할 수 있어요.

## 설정

목록 뷰 설정은 [[보기#뷰 설정|뷰 설정]]에서 설정할 수 있어요.

- 마커: 글머리 기호, 숫자 또는 없음
- 속성 들여쓰기
- 속성 구분자

### 마커

사용할 목록 마커 유형을 선택해요: 글머리 기호, 숫자, 또는 없음.

### 속성 들여쓰기

활성화하면 **속성** 메뉴에서 선택한 속성이 기본 목록 항목의 들여쓰기된 항목으로 표시돼요.

기본 목록 항목을 변경하려면 **속성** 메뉴에서 속성을 드래그하거나 `Alt + Up/Down` 단축키를 사용하여 맨 위로 이동하세요.

### 구분자

**속성 들여쓰기**가 비활성화되어 있을 때, 이 설정은 각 목록 항목의 속성을 구분하는 문자를 정의해요. 기본값은 쉼표예요.
