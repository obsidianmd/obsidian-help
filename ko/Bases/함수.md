## 전역

전역 함수는 유형 없이 사용됩니다.

### `escapeHTML()`

`escapeHTML(html: string): string`

- 문자열의 특수 문자를 이스케이프하여 HTML에 안전하게 포함할 수 있도록 합니다.

### `date()`

`date(date: string): date`

- `date(string): date`는 제공된 문자열을 파싱하여 날짜 객체를 반환합니다.
- `date` 문자열은 `YYYY-MM-DD HH:mm:ss` 형식이어야 합니다.

### `duration()`

`duration(value: string): duration`

- 문자열을 기간(duration)으로 파싱합니다. `value` 문자열 형식은 [[베이스 구문#날짜 연산|날짜 연산 섹션]]을 참조하세요.
- 날짜 연산을 수행할 때 기간을 명시적으로 파싱할 필요는 없지만(예: `now() + '1d'`), 기간에 대한 연산을 수행할 때는 필요합니다(예: `now() + (duration('1d') * 2)`).
- 기간과 스칼라의 연산을 수행할 때 기간이 왼쪽에 있어야 합니다. 예를 들어 `2 * duration('5h')` 대신 `duration('5h') * 2`를 사용합니다.

### `file()`

`file(path: string | file | url): file`

- 주어진 파일 또는 경로에 대한 파일 객체를 반환합니다.
- 예시: `file(link("[[filename]]"))` 또는 `file("path to file")`.

### `html()`

`html(html: string): html`

- 문자열을 HTML로 렌더링되는 코드 스니펫으로 변환합니다.

### `if()`

`if(condition: any, trueResult: any, falseResult?: any): any`

- `condition`은 평가할 조건입니다.
- `trueResult`는 조건이 참일 때의 출력입니다.
- `falseResult`는 조건이 거짓일 때의 선택적 출력입니다. 주어지지 않으면 `null`로 간주됩니다.
- `condition`이 참이거나 참으로 평가되는 값이면 `trueResult`를 반환하고, 그렇지 않으면 `falseResult`를 반환합니다.
- 예시: `if(isModified, "Modified", "Unmodified")`

### `image()`

`image(path: string | file | url): image`

- 뷰에서 이미지를 렌더링하는 이미지 객체를 반환합니다.
- 예시: `image(image-property)` 또는 `image("https://obsidian.md/images/obsidian-logo-gradient.svg")`

### `icon()`

`icon(name: string): icon`

- 뷰에서 아이콘으로 렌더링되는 값을 반환합니다. 아이콘 이름은 지원되는 Lucide 아이콘과 일치해야 합니다.
- 예시: `icon("arrow-right")`.

### `link()`

`link(path: string | file, display?: value): Link`

- 문자열 `path`를 파싱하고 주어진 경로로 링크를 렌더링하는 Link 객체를 반환합니다.
- 선택적으로 `display` 매개변수를 제공하여 링크에 표시되는 텍스트를 변경할 수 있습니다.

### `list()`

`list(element: any): List`

- 제공된 요소가 목록이면 수정 없이 반환합니다.
- 그렇지 않으면 제공된 `element`를 목록으로 감싸 단일 요소를 가진 목록을 생성합니다.
- 이 함수는 보관함 전체에서 속성이 문자열과 목록이 혼합되어 포함된 경우에 유용할 수 있습니다.
- 예시: `list("value")`는 `["value"]`를 반환합니다.

### `max()`

`max(value1: number, value2: number...): number`

- 제공된 모든 숫자 중 가장 큰 값을 반환합니다.

### `min()`

`min(value1: number, value2: number...): number`

- 제공된 모든 숫자 중 가장 작은 값을 반환합니다.

### `now()`

`now(): date`

- `now()`는 현재 순간을 나타내는 날짜 객체를 반환합니다.

### `number()`

`number(input: any): number`

- 제공된 값을 숫자로 반환하려고 시도합니다.
- 날짜 객체는 유닉스 에포크 이후의 밀리초로 반환됩니다.
- 불리언은 1 또는 0을 반환합니다.
- 문자열은 숫자로 파싱되며 결과가 유효하지 않으면 오류를 반환합니다.
- 예시: `number("3.4")`는 `3.4`를 반환합니다.

### `duration()`

`duration(value: string): duration`

- 문자열을 기간(duration)으로 파싱합니다. `value` 문자열 형식은 [[베이스 구문#날짜 연산|날짜 연산 섹션]]을 참조하세요.
- 날짜 연산을 수행할 때 기간을 명시적으로 파싱할 필요는 없지만(예: `now() + '1d'`), 기간에 대한 연산을 수행할 때는 필요합니다(예: `now() + (duration('1d') * 2)`).
- 기간과 스칼라의 연산을 수행할 때 기간이 왼쪽에 있어야 합니다. 예를 들어 `2 * duration('5h')` 대신 `duration('5h') * 2`를 사용합니다.

### `today()`

`today(): date`

- `today()`는 현재 날짜를 나타내는 날짜 객체를 반환합니다. 시간 부분은 0으로 설정됩니다.

## Any

모든 값에 사용할 수 있는 함수입니다. 여기에는 문자열(예: `"hello"`), 숫자(예: `42`), 목록(예: `[1,2,3]`), 객체 등이 포함됩니다.

### `isTruthy()`

`any.isTruthy(): boolean`

- 값을 불리언으로 변환하여 반환합니다.
- 예시: `1.isTruthy()`는 `true`를 반환합니다.

### `isType()`

`any.isType(type: string): boolean`

- 값이 제공된 유형인 경우 true를 반환합니다.
- 예시: `"example".isType("string")`과 `true.isType("boolean")`은 모두 true를 반환합니다.

### `toString()`

`any.toString(): string`

- 모든 값의 문자열 표현을 반환합니다.
- 예시: `123.toString()`는 `"123"`을 반환합니다.

## 날짜

`date("2025-05-27")`과 같은 날짜 및 시간에 사용할 수 있는 함수입니다. 날짜 비교는 [[베이스 구문#날짜 연산|날짜 연산]]을 사용하여 수행할 수 있습니다.

### 필드

다음 필드는 날짜에 사용할 수 있습니다:

| 필드                | 유형       | 설명                  |
| ------------------- | -------- | ------------------- |
| `date.year`         | `number` | 날짜의 연도              |
| `date.month`        | `number` | 날짜의 월 (1–12)        |
| `date.day`          | `number` | 날짜의 일               |
| `date.hour`         | `number` | 시 (0–23)            |
| `date.minute`       | `number` | 분 (0–59)            |
| `date.second`       | `number` | 초 (0–59)            |
| `date.millisecond`  | `number` | 밀리초 (0–999)         |

### `date()`

`date.date(): date`

- 시간이 제거된 날짜 객체를 반환합니다.
- 예시: `now().date().format("YYYY-MM-DD HH:mm:ss")`는 "2025-12-31 00:00:00"과 같은 문자열을 반환합니다.

### `format()`

`date.format(format: string): string`

- `format`은 서식 문자열입니다(예: `"YYYY-MM-DD"`).
- Moment.js 서식 문자열로 지정된 대로 서식이 지정된 날짜를 반환합니다.
- 예시: `date.format("YYYY-MM-DD")`는 `"2025-05-27"`을 반환합니다.

### `time()`

`date.time(): string`

- 시간을 반환합니다.
- 예시: `now().time()`는 "23:59:59"와 같은 문자열을 반환합니다.

### `relative()`

`date.relative(): string`

- 현재 날짜/시간과 비교한 읽기 쉬운 표현을 반환합니다.
- 예시: `file.mtime.relative()`는 `3 days ago`와 같은 값을 반환합니다.

### `isEmpty()`

`date.isEmpty(): boolean`

- false를 반환합니다.

## 문자열

`"hello"`와 같은 문자 시퀀스에 사용할 수 있는 함수입니다.

### 필드

| 필드              | 유형       | 설명              |
| --------------- | -------- | --------------- |
| `string.length` | `number` | 문자열의 문자 수       |

### `contains()`

`string.contains(value: string): boolean`

- `value`는 검색할 부분 문자열입니다.
- 문자열에 `value`가 포함되어 있으면 true를 반환합니다.
- 예시: `"hello".contains("ell")`는 `true`를 반환합니다.

### `containsAll()`

`string.containsAll(...values: string): boolean`

- `values`는 검색할 하나 이상의 부분 문자열입니다.
- 문자열에 모든 `values`가 포함되어 있으면 true를 반환합니다.
- 예시: `"hello".containsAll("h", "e")`는 `true`를 반환합니다.

### `containsAny()`

`string.containsAny(...values: string): boolean`

- `values`는 검색할 하나 이상의 부분 문자열입니다.
- 문자열에 `values` 중 하나 이상이 포함되어 있으면 true를 반환합니다.
- 예시: `"hello".containsAny("x", "y", "e")`는 `true`를 반환합니다.

### `endsWith()`

`string.endsWith(query: string): boolean`

- `query`는 끝에서 확인할 문자열입니다.
- 이 문자열이 `query`로 끝나면 true를 반환합니다.
- 예시: `"hello".endsWith("lo")`는 `true`를 반환합니다.

### `isEmpty()`

`string.isEmpty(): boolean`

- 문자열에 문자가 없거나 존재하지 않으면 true를 반환합니다.
- 예시: `"Hello world".isEmpty()`는 `false`를 반환합니다.
- 예시: `"".isEmpty()`는 `true`를 반환합니다.

### `lower()`

`string.lower(): string`

- 소문자로 변환된 문자열을 반환합니다.

### `replace()`

`string.replace(pattern: string | Regexp, replacement: string): string`

- `pattern`은 대상 문자열에서 검색할 값입니다.
- `replacement`는 발견된 패턴을 대체할 값입니다.
- `pattern`이 문자열이면 패턴의 모든 항목이 대체됩니다.
- `pattern`이 정규식이면 `g` 플래그에 따라 첫 번째 항목만 또는 모든 항목이 대체됩니다.
- 예시: `""a:b:c:d".replace(/:/, "-")`는 `"a-b,c,d"`를 반환하고, `"a:b:c:d".replace(/:/g, "-")`는 `"a-b-c-d"`를 반환합니다.

### `repeat()`

`string.repeat(count: number): string`

- `count`는 문자열을 반복할 횟수입니다.
- 예시: `"123".repeat(2)`는 `"123123"`을 반환합니다.

### `reverse()`

`string.reverse(): string`

- 문자열을 뒤집습니다.
- 예시: `"hello".reverse()`는 `"olleh"`를 반환합니다.

### `slice()`

`string.slice(start: number, end?: number): string`

- `start`는 포함 시작 인덱스입니다.
- `end`는 선택적 배제 끝 인덱스입니다.
- `start`(포함)부터 `end`(배제)까지의 부분 문자열을 반환합니다.
- 예시: `"hello".slice(1, 4)`는 `"ell"`을 반환합니다.
- `end`가 생략되면 문자열의 끝까지 슬라이스합니다.

### `split()`

`string.split(separator: string | Regexp, n?: number): list`

- `separator`는 문자열을 분할하는 구분자입니다.
- `n`은 선택적 숫자입니다. 제공되면 결과에 처음 `n`개의 요소가 포함됩니다.
- 부분 문자열의 목록을 반환합니다.
- 예시: `"a,b,c,d".split(",", 3)` 또는 `"a,b,c,d".split(/,/, 3)`는 `["a", "b", "c"]`를 반환합니다.

### `startsWith()`

`string.startsWith(query: string): boolean`

- `query`는 시작 부분에서 확인할 문자열입니다.
- 이 문자열이 `query`로 시작하면 true를 반환합니다.
- 예시: `"hello".startsWith("he")`는 `true`를 반환합니다.

### `title()`

`string.title(): string`

- 문자열을 제목 대소문자(각 단어의 첫 글자를 대문자)로 변환합니다.
- 예시: `"hello world".title()`는 `"Hello World"`를 반환합니다.

### `trim()`

`string.trim(): string`

- 문자열 양쪽 끝의 공백을 제거합니다.
- 예시: `"  hi  ".trim()`는 `"hi"`를 반환합니다.

## 숫자

`42`, `3.14`와 같은 숫자 값에 사용할 수 있는 함수입니다.

### `abs()`

`number.abs(): number`

- 숫자의 절대값을 반환합니다.
- 예시: `(-5).abs()`는 `5`를 반환합니다.

### `ceil()`

`number.ceil(): number`

- 숫자를 가장 가까운 정수로 올림합니다.
- 예시: `(2.1).ceil()`는 `3`을 반환합니다.

### `floor()`

`number.floor(): number`

- 숫자를 가장 가까운 정수로 내림합니다.
- 예시: `(2.9).floor()`는 `2`를 반환합니다.

### `isEmpty()`

`number.isEmpty(): boolean`

- 숫자가 존재하지 않으면 true를 반환합니다.
- 예시: `5.isEmpty()`는 `false`를 반환합니다.

### `round()`

`number.round(digits: number): number`

- 숫자를 가장 가까운 정수로 반올림합니다.
- 선택적으로 `digits` 매개변수를 제공하여 해당 소수 자릿수로 반올림할 수 있습니다.
- 예시: `(2.5).round()`는 `3`을 반환하고, `(2.3333).round(2)`는 `2.33`을 반환합니다.

### `toFixed()`

`number.toFixed(precision: number): string`

- `precision`은 소수점 이하 자릿수입니다.
- 고정 소수점 표기법으로 된 숫자의 문자열을 반환합니다.
- 예시: `(3.14159).toFixed(2)`는 `"3.14"`를 반환합니다.

## 목록

`[1, 2, 3]`과 같은 요소의 순서가 있는 목록에 사용할 수 있는 함수입니다.

### 필드

| 필드            | 유형       | 설명              |
| ------------- | -------- | --------------- |
| `list.length` | `number` | 목록의 요소 수        |

### `contains()`

`list.contains(value: any): boolean`

- `value`는 검색할 요소입니다.
- 목록에 `value`가 포함되어 있으면 true를 반환합니다.
- 예시: `[1,2,3].contains(2)`는 `true`를 반환합니다.

### `containsAll()`

`list.containsAll(...values: any): boolean`

- `values`는 검색할 하나 이상의 요소입니다.
- 목록에 모든 `values`가 포함되어 있으면 true를 반환합니다.
- 예시: `[1,2,3].containsAll(2,3)`는 `true`를 반환합니다.

### `containsAny()`

`list.containsAny(...values: any): boolean`

- `values`는 검색할 하나 이상의 요소입니다.
- 목록에 `values` 중 하나 이상이 포함되어 있으면 true를 반환합니다.
- 예시: `[1,2,3].containsAny(3,4)`는 `true`를 반환합니다.

### `filter()`

`list.filter(value: Boolean): list`

- `index`와 `value` 변수를 사용하는 필터 함수를 호출하여 이 목록의 요소를 필터링하며, 요소를 유지할지 여부에 대한 불리언 값을 반환합니다.
- `value`는 목록의 항목 값입니다.
- `index`는 현재 값의 인덱스입니다.
- 예시: `[1,2,3,4].filter(value > 2)`는 `[3,4]`를 반환합니다.

### `flat()`

`list.flat(): list`

- 중첩된 목록을 단일 목록으로 평탄화합니다.
- 예시: `[1,[2,3]].flat()`는 `[1,2,3]`을 반환합니다.

### `isEmpty()`

`list.isEmpty(): boolean`

- 목록에 요소가 없으면 true를 반환합니다.
- 예시: `[1,2,3].isEmpty()`는 `false`를 반환합니다.

### `join()`

`list.join(separator: string): string`

- `separator`는 요소 사이에 삽입할 문자열입니다.
- 모든 목록 요소를 단일 문자열로 결합합니다.
- 예시: `[1,2,3].join(",")`는 `"1,2,3"`을 반환합니다.

### `map()`

`list.map(value: Any): list`

- `index`와 `value` 변수를 사용하는 변환 함수를 호출하여 이 목록의 각 요소를 변환하며, 목록에 배치될 새 값을 반환합니다.
- `value`는 목록의 항목 값입니다.
- `index`는 현재 값의 인덱스입니다.
- 예시: `[1,2,3,4].map(value + 1)`는 `[2,3,4,5]`를 반환합니다.

### `reduce()`

`list.reduce(expression: Any, acc: Any): Any`

- 각 요소에 대해 표현식을 실행하여 이 목록의 요소를 단일 값으로 줄입니다. 표현식은 `index`, `value`, `acc`(누적기) 변수를 사용할 수 있으며 다음 누적기 값을 반환해야 합니다.
- `expression`은 목록의 모든 요소에 대해 평가됩니다.
- `value`는 목록의 현재 항목 값입니다.
- `index`는 현재 항목의 인덱스입니다.
- `acc`는 지금까지 누적된 값입니다.
- 예시 (합계): `[1,2,3].reduce(acc + value, 0)`는 `6`을 반환합니다.
- 예시 (최대값): `values.filter(value.isType("number")).reduce(if(acc == null || value > acc, value, acc), null)`는 가장 큰 숫자를 반환하거나, 없으면 `null`을 반환합니다.

### `reverse()`

`list.reverse(): list`

- 목록을 제자리에서 뒤집습니다.
- 예시: `[1,2,3].reverse()`는 `[3,2,1]`을 반환합니다.

### `slice()`

`list.slice(start: number, end?: number): list`

- `start`는 포함 시작 인덱스입니다.
- `end`는 선택적 배제 끝 인덱스입니다.
- `start`(포함)부터 `end`(배제)까지 목록 일부의 얕은 복사본을 반환합니다.
- 예시: `[1,2,3,4].slice(1,3)`는 `[2,3]`을 반환합니다.
- `end`가 생략되면 목록의 끝까지 슬라이스합니다.

### `sort()`

`list.sort(): list`

- 목록 요소를 가장 작은 것부터 가장 큰 것으로 정렬합니다.
- 예시: `[3, 1, 2].sort()`는 `[1, 2, 3]`을 반환합니다.
- 예시: `["c", "a", "b"].sort()`는 `["a", "b", "c"]`를 반환합니다.

### `unique()`

`list.unique(): list`

- 중복 요소를 제거합니다.
- 예시: `[1,2,2,3].unique()`는 `[1,2,3]`을 반환합니다.

## 링크

링크에 사용할 수 있는 함수입니다. 링크는 파일(`file.asLink()`) 또는 경로(`link("path")`)에서 생성할 수 있습니다.

### `asFile()`

`link.asFile(): file`

- 링크가 유효한 로컬 파일을 참조하는 경우 파일 객체를 반환합니다.
- 예시: `link("[[filename]]").asFile()`

### `linksTo()`

`link.linksTo(file): boolean`

- `link`가 나타내는 파일이 `file`로의 링크를 가지고 있는지 여부를 반환합니다.

## 파일

보관함의 파일에 사용할 수 있는 함수입니다.

### 필드

다음 필드는 파일에 사용할 수 있습니다:

| 필드                | 유형       | 설명                           |
| ----------------- | -------- | ---------------------------- |
| `file.name`       | `string` | 이 파일의 이름입니다.                 |
| `file.basename`   | `string` | 확장자를 제외한 이 파일의 이름입니다.        |
| `file.path`       | `string` | 보관함 루트를 기준으로 한 이 파일의 전체 경로입니다. |
| `file.folder`     | `string` | 상위 폴더의 전체 경로입니다.             |
| `file.ext`        | `string` | 이 파일의 확장자입니다.                |
| `file.size`       | `number` | 이 파일의 크기(바이트)입니다.            |
| `file.properties` | `object` | 이 파일의 노트 속성입니다.              |
| `file.tags`       | `list`   | 이 파일의 태그입니다. 인라인 태그를 포함합니다.  |
| `file.links`      | `list`   | 이 파일 내의 내부 링크입니다.            |
| `file.ctime`      | `date`   | 이 파일이 생성된 타임스탬프입니다.          |
| `file.mtime`      | `date`   | 이 파일이 마지막으로 수정된 타임스탬프입니다.    |

### `asLink()`

`file.asLink(display?: string): Link`

- `display`는 링크의 선택적 표시 텍스트입니다.
- 작동하는 링크로 렌더링되는 Link 객체를 반환합니다.
- 예시: `file.asLink()`

### `hasLink()`

`file.hasLink(otherFile: file | string): boolean`

- `otherFile`은 확인할 다른 파일 객체 또는 문자열 경로입니다.
- `file`이 `otherFile`로 링크되어 있으면 true를 반환합니다.
- 예시: `file.hasLink(otherFile)`는 `file`에서 `otherFile`로의 링크가 있으면 `true`를 반환합니다.

### `hasProperty()`

`file.hasProperty(name: string): boolean`

- 노트에 주어진 파일 속성이 있으면 true를 반환합니다.

### `hasTag()`

`file.hasTag(...values: string): boolean`

- `values`는 하나 이상의 태그 이름입니다.
- 파일에 `values`의 태그 중 하나라도 있으면 true를 반환합니다.
- 예시: `file.hasTag("tag1", "tag2")`는 파일에 태그 `#tag1` 또는 `#tag2`가 있으면 `true`를 반환합니다. `#tag1/a` 또는 `#tag2/b`와 같은 [[태그#중첩 태그|중첩 태그]]도 포함됩니다.

### `inFolder()`

`file.inFolder(folder: string): boolean`

- `folder`는 확인할 폴더 이름입니다.
- 파일이 지정된 폴더 또는 그 하위 폴더에 있으면 true를 반환합니다.
- 예시: `file.inFolder("notes")`는 `true`를 반환합니다.

## 객체

`{"a": 1, "b": 2}`와 같은 키-값 쌍의 컬렉션에 사용할 수 있는 함수입니다.

### `isEmpty()`

`object.isEmpty(): boolean`

- 객체에 자체 속성이 없으면 true를 반환합니다.
- 예시: `{}.isEmpty()`는 `true`를 반환합니다.

### `keys()`

`object.keys(): list`

- 객체의 키를 포함하는 목록을 반환합니다.

### `values()`

`object.values(): list`

- 객체의 값을 포함하는 목록을 반환합니다.

## 정규 표현식

정규 표현식 패턴에 사용할 수 있는 함수입니다. 예시: `/abc/`.

### `matches()`

`regexp.matches(value: string): boolean`

- `value`는 테스트할 문자열입니다.
- 정규 표현식이 `value`와 일치하면 true를 반환합니다.
- 예시: `/abc/.matches("abcde")`는 `true`를 반환합니다.
