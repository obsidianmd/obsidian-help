베이스는 노트의 데이터베이스와 유사한 뷰를 생성할 수 있는 [[코어 플러그인|코어 플러그인]]이에요. 베이스를 사용하면 파일과 해당 [[속성|속성]]을 보고, 편집하고, 정렬하고, 필터링할 수 있어요. 베이스는 프로젝트부터 여행 계획, 읽기 목록 등 모든 것을 정리하는 데 도움이 돼요.

각 베이스에는 표와 카드 같은 다양한 레이아웃의 여러 [[보기|보기]]를 가질 수 있어요. 아래는 각 행이 파일이고 각 열이 해당 파일의 속성인 표 뷰의 예시예요.

![도서 목록이 포함된 표 뷰를 보여주는 베이스 예시](bases-noshadow.png#interface)

Obsidian 베이스의 모든 데이터는 로컬 [[기본 서식 구문|Markdown]] 파일과 해당 [[속성]]에 저장돼요. 뷰는 [[베이스 구문]]으로 기술되며, `.base` 파일로 저장하거나 Markdown 파일 내의 코드 블록에 [[베이스 생성#베이스 임베드|임베드]]할 수 있어요.

## 베이스 사용하기

- [[베이스 생성]]
	- 베이스를 생성하고 임베드하는 방법을 알아봐요.
- [[보기]]
	- 레이아웃, 필터, 정렬 및 그룹 속성을 살펴봐요.
- [[수식]]
	- 다른 속성의 데이터를 사용하여 계산을 생성해요.
- [[함수]]
	- 수식과 필터에서 사용할 수 있는 함수예요.
- [[베이스 구문]]
	- 베이스 파일의 서식 지정 방식이에요.

## 뷰 유형

뷰는 다양한 레이아웃으로 표시할 수 있어요. [[커뮤니티 플러그인]]으로 추가 레이아웃을 더할 수 있어요.

- [[표 뷰|표]]
	- 파일을 행으로 표시하고, 해당 파일의 속성을 열로 보여줘요.
- [[목록 뷰|목록]]
	- 파일을 글머리 기호 또는 번호 포함 마커가 있는 목록으로 표시해요.
- [[카드 뷰|카드]]
	- 파일을 그리드로 표시해요. 이미지를 포함한 갤러리 형태의 뷰를 만들 수 있어요.
- [[지도 뷰|지도]]
	- 파일을 인터랙티브 지도 위의 핀으로 표시해요.
