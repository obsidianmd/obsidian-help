[[베이스 소개|베이스]]를 사용하면 노트의 데이터베이스와 같은 뷰를 생성할 수 있어요. 여기에서는 베이스를 생성하고 노트에 임베드하는 방법을 설명해요. 모든 베이스는 하나 이상의 [[보기|보기]]를 가질 수 있어 정보를 다양한 방식으로 표시할 수 있어요.

## 새 베이스 생성하기

**명령어 팔레트:**

1. **명령어 팔레트**를 열어요.
2. 선택해요.
	- **베이스: 새 베이스 생성하기**를 선택하면 활성 파일과 같은 폴더에 베이스가 생성돼요.
	- **베이스: 새 베이스 삽입하기**를 선택하면 베이스를 생성하고 현재 파일에 임베드해요.

**파일 탐색기:**

1. 파일 탐색기에서 베이스를 생성할 폴더를 마우스 오른쪽 버튼으로 클릭해요.
2. **새 베이스**를 선택해요.

**리본:**

- 세로 리본 설정에서 **새 베이스 생성하기**를 선택하면 활성 파일과 같은 폴더에 베이스가 생성돼요.

## 베이스 임베드하기

### 베이스 파일 임베드하기

`![[File.base]]` 구문을 사용하여 [[파일 임베드|다른 파일]]에 베이스 파일을 임베드할 수 있어요. 기본 보기를 지정하려면 `![[File.base#View]]`를 사용하세요.

### 코드 블록으로 베이스 임베드하기

베이스는 `base` 코드 블록과 [[베이스 구문|베이스 구문]]을 사용하여 노트에 직접 임베드할 수도 있어요.

~~~yaml
```base
filters:
  and:
    - file.hasTag("example")
views:
  - type: table
    name: Table
```
~~~
