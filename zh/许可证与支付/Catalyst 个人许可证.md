Catalyst 许可证是一次性购买的许可证，它让你能够抢先体验测试版本，并帮助支持 Obsidian 的持续开发。你可以通过[购买 Catalyst 许可证](https://obsidian.md/zh/buy/catalyst)来支持 Obsidian。

购买后你将享有以下权益：

- 抢先体验 Obsidian 的[[#启用内部版本|新版本]]
- 进入专属的[[#加入 Insider 频道|Insider 频道]]。
- 获得专属的[[#获取论坛徽章|论坛徽章]]和 [[#获取 Discord 徽章|Discord 徽章]]。

## Catalyst 等级

Catalyst 许可证根据你的贡献金额分为三个等级：*Insider*、*Supporter* 和 *VIP*。

| 权益                                                                               | Insider | Supporter |   VIP    |
| ---------------------------------------------------------------------------------- | :-----: | :-------: | :------: |
| 抢先体验新版本                                                                       |    ✓    |     ✓     |    ✓     |
| 进入专属开发频道 `#insider-desktop` 和 `#insider-mobile` |    ✓    |     ✓     |    ✓     |
| 进入专属休息室频道 `#vip-lounge`                                  |         |     ✓     |    ✓     |
| Insider 徽章                                                                      |    ✓    |           |          |
| Supporter 徽章                                                                    |         |     ✓     |          |
| VIP 徽章                                                                          |         |           |    ✓     |
| **贡献金额（美元）**                                                             | **$25** |  **$50**  | **$100** |


> [!info] 升级你的 Catalyst 等级
> 如果你已经拥有 Catalyst 许可证，可以通过支付当前等级与新等级之间的差价来升级到更高的 Catalyst 等级。

## 获取 Catalyst 权益

购买 Catalyst 许可证后，你可以享受一系列权益。以下介绍如何启用这些权益。

### 获取论坛徽章

**注意：** 要关联你的 Obsidian 账户和论坛账户，两个账户必须使用相同的邮箱地址。

1. 如果你还没有注册，请先加入我们的[论坛](https://forum.obsidian.md/)。
2. 前往你的 [Obsidian 账户管理页面](https://obsidian.md/zh/account/catalyst)。
3. 在 **Catalyst** 下，选择 **Get forum badge**。

### 获取 Discord 徽章

1. 如果你还没有加入，请先加入我们的 [Discord 服务器](https://discord.gg/veuWUTm)。
2. 前往你的 [Obsidian 账户管理页面](https://obsidian.md/zh/account/catalyst)。
3. 在 **Catalyst** 下，选择 **Get Discord badge**。

### 加入 Insider 频道

Insider 可以在我们专属的 Discord 频道中与 Obsidian 团队及其他 Insider 交流。在你[[#获取 Discord 徽章]]后，将自动加入 Insider 频道。
