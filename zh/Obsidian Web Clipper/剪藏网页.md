安装 [[Obsidian Web Clipper 简介|Obsidian Web Clipper]] 浏览器扩展后，你可以通过以下方式访问它（具体取决于你的浏览器）：

1. 浏览器工具栏中的 Obsidian 图标。
2. 快捷键，通过键盘激活扩展。
3. 右键菜单，右键点击你正在浏览的网页。

要将页面保存到 Obsidian，请点击**添加到 Obsidian**按钮。

## 捕获页面

打开扩展时，Obsidian Web Clipper 会根据你的[[Obsidian Web Clipper/模板|模板]]设置从当前网页中提取数据。你可以创建自己的模板，并使用[[变量|变量]]和[[过滤器|筛选]]来自定义输出。

默认情况下，Obsidian Web Clipper 会尝试智能提取页面的主要文章内容，排除页面上的其他元素。不过，你可以通过以下方式覆盖此行为：

- 如果存在自定义模板，则使用你的模板。
- 如果存在选中内容，则使用选中内容。你可以使用 `Ctrl/Cmd+A` 选择整个页面。
- 如果存在任何[[高亮器|高亮内容]]，则使用高亮内容。

## 下载图片

使用 Obsidian Web Clipper 时，图片不会自动下载。图片会链接到其网络 URL。这可以节省仓库空间，但意味着在离线状态下或 URL 失效时图片将无法访问。

你可以在 Obsidian 中使用[[命令面板|命令]]**下载当前文件内的所有附件**来下载任何文件中的图片。此命令也可以在 Obsidian 中映射为快捷键。

## 快捷键

Obsidian Web Clipper 包含一些键盘快捷键，可以帮助你加速工作流程。要更改键位映射，请前往 **Web Clipper 设置** → **常规**，并按照浏览器的说明进行操作。所有浏览器都可以更改键位映射，但 Safari 除外，因为它不支持编辑快捷键。

| 操作             | macOS         | Windows/Linux  |
| ------------------ | ------------- | -------------- |
| 打开 Clipper       | `Cmd+Shift+O` | `Ctrl+Shift+O` |
| 快速剪藏         | `Opt+Shift+O` | `Alt+Shift+O`  |
| 切换高亮器 | `Opt+Shift+H` | `Alt+Shift+H`  |
| 切换阅读器      | `Opt+Shift+R` | `Alt+Shift+R`  |

## 界面功能

Obsidian Web Clipper 界面分为四个部分：

1. **顶部栏**，你可以在此切换模板、开启[[高亮器|高亮]]、[[阅读器|阅读视图]]，以及访问设置。
2. **属性**，显示从页面中提取的[[属性|元数据]]，这些数据将作为[[属性]]保存在 Obsidian 中。
3. **笔记内容**，将保存到 Obsidian 中的内容。
4. **底部栏**，允许你选择仓库和文件夹，并添加到 Obsidian。

顶部栏功能包括：

- ![[lucide-chevrons-up-down.svg#icon]] **模板**切换器，用于使用在扩展设置中保存的[[Obsidian Web Clipper/模板|模板]]。
- ![[lucide-more-horizontal.svg#icon]] **更多**按钮，显示可在模板中使用的页面变量。
- ![[lucide-highlighter.svg#icon]] **高亮**按钮，用于开启[[高亮器|高亮]]。
- ![[lucide-book-icon.svg#icon]] **阅读器**按钮，用于开启[[阅读器|阅读视图]]。
- ![[lucide-picture-in-picture-2.svg#icon]] **嵌入**按钮，用于将 Web Clipper 从弹出窗口移入页面中。
- ![[lucide-settings.svg#icon]] **设置**按钮，用于打开扩展设置。

底部栏功能包括：

- **添加到 Obsidian** 按钮，将数据保存到 Obsidian。
- **仓库**下拉菜单，用于在 Web Clipper 设置中保存的仓库之间切换。
- **文件夹**字段，用于定义保存到哪个文件夹。
- **解释器**，用于在页面上运行[[解释器|自然语言提示]]。
