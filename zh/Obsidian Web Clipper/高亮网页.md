[[Obsidian Web Clipper 简介|Web Clipper]] 允许你高亮网页上的文本，并选择你想要保存到 Obsidian 的元素。你的高亮内容会被保存，这样当你再次访问页面时可以重新查看。

当你打开扩展程序时，高亮内容可以被[[剪藏网页|捕获]]并保存到 Obsidian。

## 开启高亮

你可以通过多种方式开启高亮功能，具体取决于你的浏览器：

- 扩展面板中的高亮图标。
- 快捷键，通过键盘激活扩展程序。
- 右键菜单，在你正在浏览的网页上右键点击。

开启高亮功能后，你可以选择想要高亮的文本、图片和元素。

## 高亮设置

你可以在 Web Clipper 设置中更改高亮的行为。在这里你还可以将高亮内容导出为 `.json` 文件。

通过 `{{content}}` [[变量|变量]]将高亮内容插入到剪藏笔记中有三个选项：

- **高亮页面内容** — 使用 [[Obsidian 风格的 Markdown 语法|语法]] `==highlight==` 直接在文本中添加高亮。
- **替换页面内容** — 返回高亮列表，不包含任何页面内容。
- **不做任何操作** — 返回不含高亮的原始内容。

你可以使用 `{{highlights}}` 变量将高亮内容直接添加到模板中，例如：

```
{{highlights|map: item => item.text|join:"\n\n"}}
```
