[[Obsidian Web Clipper 简介|Web Clipper]] 内置了一个名为"阅读器"的阅读模式，可以去除网页中的杂乱元素，仅以简洁、易读的格式呈现文章主要内容。

## 激活阅读模式

你可以通过以下方式激活阅读器：

- 扩展面板标题栏中的**书本图标**。
- 键盘快捷键 `Alt+Shift+R`，可在扩展设置中自定义。
- 再次切换阅读模式将恢复原始页面。

## 工作原理

阅读器会从页面中提取主要内容。原始页面将被替换为简洁的阅读视图，包括：

- 文章标题、作者、发布日期和来源域名。
- 文章主要内容，保留图片和格式。
- 对于包含多个标题的文章，侧边栏中会显示大纲。
- 代码块的语法高亮。
- 点击脚注时会以行内弹出框的形式显示。

## 自定义阅读视图

阅读模式包含一个隐藏的设置栏，将鼠标悬停在页面右上角即可显示，其中提供了自定义阅读体验的控件。

| 设置项            | 描述                                                                                       |
| ----------------- | ------------------------------------------------------------------------------------------ |
| 字体大小          | 调整文本大小。                                                                             |
| 行宽              | 内容区域的最大宽度。                                                                       |
| 行高              | 行与行之间的间距。                                                                         |
| 配色方案          |                                                                                            |
| 主题              | "自动"会跟随系统偏好设置。或选择总是使用浅色模式或深色模式。                               |
