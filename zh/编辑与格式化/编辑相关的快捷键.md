本页列出了在 Obsidian 中导航和编辑文本的默认键盘快捷键。这些快捷键由您的操作系统或 Obsidian 所基于的框架提供，无法在 Obsidian 内自定义。

如需了解可自定义的 Obsidian 命令快捷键，请参阅[[快捷键]]。

## Windows 和 Linux 快捷键

### 常用操作

| 操作 | 快捷键 |
|-|-|
| 复制 | `Ctrl+C` |
| 剪切 | `Ctrl+X` |
| 粘贴 | `Ctrl+V` |
| 无格式粘贴 | `Ctrl+Shift+V` |
| 撤销 | `Ctrl+Z` |
| 重做 | `Ctrl+Shift+Z` 或 `Ctrl+Y` |
| 复制段落 | `Ctrl+C`（未选中文本时） |
| 剪切段落 | `Ctrl+X`（未选中文本时） |

### 文本编辑

| 操作 | 快捷键 |
|-|-|
| 插入新行 | `Enter` |
| 删除前一个字符 | `Backspace` |
| 删除后一个字符 | `Delete` |
| 删除前一个单词 | `Ctrl+Backspace` |
| 删除后一个单词 | `Ctrl+Delete` |
| 删除当前行 | `Ctrl+Shift+K`（未选中文本时） |

### 文本导航

| 操作 | 快捷键 |
|-|-|
| 移动光标一个字符 | `Left/→` |
| 移动光标到前一个单词开头 | `Ctrl+←` |
| 移动光标到后一个单词末尾 | `Ctrl+→` |
| 移动光标到当前行开头 | `Home` |
| 移动光标到当前行末尾 | `End` |
| 移动光标到上一行 | `↑` |
| 移动光标到下一行 | `↓` |
| 移动光标到笔记开头 | `Ctrl+Home` |
| 移动光标到笔记末尾 | `Ctrl+End` |
| 向上翻一页 | `Page up` |
| 向下翻一页 | `Page down` |

### 文本选择

| 操作 | 快捷键 |
|-|-|
| 简化选择 | `Escape` |
| 选中所有元素 | `Ctrl+A` |
| 扩展选择一个字符 | `Shift+Left/→` |
| 扩展选择到前一个单词开头 | `Ctrl+Shift+←` |
| 扩展选择到后一个单词末尾 | `Ctrl+Shift+→` |
| 扩展选择到当前行开头 | `Shift+Home` |
| 扩展选择到当前行末尾 | `Shift+End` |
| 扩展选择到笔记开头 | `Ctrl+Shift+Home` |
| 扩展选择到笔记末尾 | `Ctrl+Shift+End` |
| 向上扩展选择一页 | `Shift+Page up` |
| 向下扩展选择一页 | `Shift+Page down` |

## macOS 快捷键

### 常用操作

| 操作 | 快捷键 |
|-|-|
| 复制 | `Cmd+C` |
| 剪切 | `Cmd+X` |
| 粘贴 | `Cmd+V` |
| 无格式粘贴 | `Cmd+Shift+V` |
| 撤销 | `Cmd+Z` |
| 重做 | `Cmd+Shift+Z` |
| 复制段落 | `Cmd+C`（未选中文本时） |
| 剪切段落 | `Cmd+X`（未选中文本时） |

### 文本格式

| 操作 | 快捷键 |
| ---- | ---- |
| 加粗文本 | `Cmd+B` |
| 斜体文本 | `Cmd+I` |

### 文本编辑

| 操作 | 快捷键 |
|-|-|
| 插入新行 | `Enter` |
| 删除前一个字符 | `Backspace` |
| 删除后一个字符 | `Delete` |
| 删除前一个单词 | `Option+Backspace` |
| 删除后一个单词 | `Option+Delete` |
| 删除到当前行开头 | `Cmd+Backspace` |
| 删除到当前行末尾 | `Cmd+Delete` |
| 删除当前行 | `Cmd+Shift+K`（未选中文本时） |

### 文本导航

| 操作 | 快捷键 |
|-|-|
| 移动光标一个字符 | `Left/→` |
| 移动光标到前一个单词开头 | `Option+←` |
| 移动光标到后一个单词末尾 | `Option+→` |
| 移动光标到当前行开头 | `Cmd+←` |
| 移动光标到当前行末尾 | `Cmd+→` |
| 移动光标到上一行 | `↑` |
| 移动光标到下一行 | `↓` |
| 移动光标到笔记开头 | `Cmd+↑` |
| 移动光标到笔记末尾 | `Cmd+↓` |
| 向上翻一页 | `Fn+↑` |
| 向下翻一页 | `Fn+↓` |

### 文本选择

| 操作                                                 | 快捷键                   |
| ------------------------------------------------------ | -------------------------- |
| 简化选择                                     | `Escape`                   |
| 选中所有元素                                             | `Cmd+A`                    |
| 扩展选择一个字符                         | `Shift+Left/→`   |
| 扩展选择到前一个单词开头 | `Option+Shift+←`  |
| 扩展选择到后一个单词末尾           | `Option+Shift+→` |
| 扩展选择到当前行开头  | `Cmd+Shift+←`              |
| 扩展选择到当前行末尾        | `Cmd+Shift+→`    |
| 扩展选择到笔记开头          | `Cmd+Shift+↑`       |
| 扩展选择到笔记末尾                | `Cmd+Shift+↓`     |
| 向上扩展选择一页                           | `Ctrl+Shift+↑`      |
| 向下扩展选择一页                         | `Ctrl+Shift+↓`    |
