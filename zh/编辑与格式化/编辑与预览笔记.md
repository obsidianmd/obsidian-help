Obsidian 允许你使用_视图_和_模式_来控制如何阅读和编辑以 Markdown 编写的笔记。

- **视图**用于在阅读和编辑笔记之间切换。
- **模式**控制编辑时 Markdown 的显示方式。

> [!note] 注意
> 默认情况下，Obsidian 以编辑模式打开新标签页。你可以在**[[设置]] → 编辑器 → 新标签页的默认视图**中更改此设置。

## 阅读视图

*阅读视图*不显示 Markdown 语法，提供干净、易读的格式，便于专注审阅。

切换到*阅读视图*：

- 点击编辑器右上角的视图切换器 ![[lucide-book-icon.svg#icon]] 。
- 或点击状态栏中的交互式状态图标 ![[lucide-edit-3.svg#icon]] 或 ![[lucide-code-xml.svg#icon]] ，然后选择**阅读视图**。
- 或按 `Ctrl+E`（macOS 上为 `Cmd+E`）。

> [!note] 注意
> 启用**[[设置]] → 外观 → 显示标签页标题栏**即可看到视图切换器图标。
> 
> 如果该选项已禁用，请开启**[[设置]] → 编辑器 → 视图状态**以从状态栏切换视图。

> [!tip] 并排查看
> 要同时以*编辑模式*和*阅读视图*打开一篇笔记，按住 `Ctrl`（macOS 上为 `Cmd`）并点击视图切换器。

## 编辑模式

*编辑模式*允许你对笔记进行更改。

在*编辑模式*下，*编辑模式*定义了 Markdown 的显示方式。你可以选择两种*编辑模式*之一：*实时预览*或*源码模式*。

### 实时预览

*实时预览*在行内显示格式化文本，同时隐藏大部分 Markdown 语法。当光标进入格式化内容时，底层语法会变为可见以便编辑。

切换到*实时预览*：

- 点击编辑器右上角的视图切换器 ![[lucide-edit-3.svg#icon]] 。
- 或点击状态栏中的交互式状态图标 ![[lucide-book-icon.svg#icon]] 或 ![[lucide-code-xml.svg#icon]] ，然后选择**实时预览**。
- 或按 `Ctrl+E`（macOS 上为 `Cmd+E`）。
- 或使用[[命令面板|命令]]**切换编辑/预览视图**。

> [!note] 注意
> 默认情况下，编辑模式设置为*实时预览*。你可以在**[[设置]] → 编辑器 → 默认编辑模式**中更改此设置。

> [!tip] 在许多情况下，实时预览可以消除切换到[[#阅读视图]]的需要。

### 源码模式

*源码模式*完全按照原样显示所有 Markdown 语法。如果你偏好纯文本或需要精确的格式控制，可以使用它。

切换到*源码模式*：
- 点击状态栏中的交互式状态图标 ![[lucide-book-icon.svg#icon]] 或 ![[lucide-edit-3.svg#icon]]  ，然后选择**源码模式**。

> [!note] 注意
> 默认情况下，编辑模式设置为*实时预览*。你可以在**[[设置]] → 编辑器 → 默认编辑模式**中将其更改为*源码模式*。
> 
> 切换到*源码模式*后，你还可以通过以下方式操作：
> 
> - 点击笔记右上角的视图切换器 ![[lucide-edit-3.svg#icon]] 。
> - 或按 `Ctrl+E`（macOS 上为 `Cmd+E`）。
> - 或使用[[命令面板|命令]]**切换编辑/预览视图**。

> [!tip] 切换*编辑模式*
> 要在*实时预览*和*源码模式*之间快速切换，你可以为[[命令面板|命令]]**切换实时阅览/源码模式**设置一个[[快捷键]]。
