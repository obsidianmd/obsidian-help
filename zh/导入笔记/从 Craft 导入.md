Obsidian 支持从 Craft 中迁移笔记。该功能会将你的 Craft 数据转换为持久的 Markdown 文件，你可以在 Obsidian 和许多其他应用中使用这些文件。

## 从 Craft 导出数据

在桌面电脑上打开 Craft 应用：

1. 前往"整理"或"所有文档"视图
2. 点击三点菜单
3. 选择导出选项。
4. 选择 Markdown。

根据你在 Craft 中笔记的数量，此过程可能需要一些时间。

## 将 Craft 数据导入 Obsidian

接下来，你可以按照[[导入 Markdown 文件]]的说明进行操作。

## 故障排查

Craft Docs 对某些链接使用了非标准语法。你可以在 [Importer GitHub 仓库](https://github.com/obsidianmd/obsidian-importer/issues/27) 中找到迁移这些链接的相关资源。
