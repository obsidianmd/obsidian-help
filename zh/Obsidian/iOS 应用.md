Obsidian 移动端应用适用于 iOS 和 iPadOS，为你的 iPhone 和 iPad 带来强大的笔记功能。你可以从 [Apple App Store](https://apps.apple.com/us/app/obsidian-connected-notes/id1557175442) 下载。

本页面介绍 iOS 特有的功能，包括小组件、Siri 集成和快捷指令。

## 小组件

Obsidian iOS 版提供了多种小组件，方便你快速操作仓库。

> [!note] 注意
> 小组件适用于 iOS 和 iPadOS 18 及更高版本。
> 使用"要求面容 ID"解锁应用时，小组件不可用。

### 锁定屏幕和控制中心小组件

锁定屏幕和控制中心小组件允许你：
- 新建笔记
- 打开特定笔记
- 打开日记
- 打开搜索
- 打开 Obsidian

### 主屏幕小组件

主屏幕小组件允许你：
- 新建笔记
- 查看笔记
- 打开日记

### 自定义小组件

你可以自定义小组件以适应你的工作流程，例如选择要使用的仓库或指定要打开的特定笔记。

- **主屏幕小组件：** 长按小组件，然后选择**编辑小组件**。
- **锁定屏幕小组件：** 长按锁定屏幕，点击**自定义**，选择锁定屏幕，然后点击要自定义的小组件。
- **控制中心小组件：** 打开控制中心，点击左上角的 **+** 按钮开始编辑，然后点击要自定义的小组件。

## 快捷指令

Obsidian 与 Apple 快捷指令应用集成，允许你创建强大的自动化操作。可用的快捷指令包括：

- **打开笔记** — 打开仓库中的特定笔记
- **新建笔记** — 在仓库中创建新笔记
- **打开日记** — 直接跳转到今天的日记
- **速记到日记** — 在不打开 Obsidian 应用的情况下，将文本追加或前置到日记中
- **速记到书签** — 在不打开 Obsidian 应用的情况下，将文本追加或前置到已添加书签的笔记中

速记类快捷指令对于快速记录特别有用，因为它们允许你在后台向笔记中添加内容。

## 共享面板

Obsidian 的共享面板可让你从网页中捕获内容。它也适用于 YouTube 和其他社交网络等应用。

> [!note]
> - 原生共享面板适用于 iOS 和 iPadOS 18 及更高版本。
> - 本节中描述的共享面板功能需要 Obsidian 1.13.0 或更高版本。

使用共享面板从其他应用快速将内容发送到 Obsidian：
1. 在其他应用中，点击**共享**按钮。
2. 选择 **Obsidian**。
3. 选择一个位置。
4. 查看或编辑捕获的内容。
5. 点击**保存**。

![[ios-share-sheet-extension.png|400]]

### 位置

位置可让你在保存之前决定共享内容的去向。

位置可以捕获到：
- **新笔记** — 在仓库或文件夹中创建新笔记。
- **日记** — 将内容追加或前置到今天的日记中。
- **已加书签的笔记** — 将内容追加或前置到已添加书签的笔记中。
- **笔记** — 选择仓库中已有的笔记。
- **新书签** — 将共享的 URL 保存到 Obsidian 书签。

![[ios-share-sheet-locations.png|400]]

### 自定义位置

你可以为常见工作流程创建位置，例如将文章保存到收件箱、将引用追加到日记中，或将链接添加到书签。

自定义位置的步骤：

1. 从 iOS 共享面板打开 Obsidian。
2. 点击工具栏中的当前位置。
3. 点击 **+** 按钮创建新位置，或选择已有位置进行编辑。
4. 选择仓库、行为和可选设置。

根据`行为`类型，你可以配置以下选项：
- 文件夹
- 模板
- 书签组
- 追加或前置位置
- 共享链接是捕获**全文**还是仅捕获 **URL**

![[ios-share-sheet-add-location.png|400]]

### 共享时使用模板

你可以在通过共享面板共享内容时使用模板。模板可让你使用页面标题、作者、来源网站和发布日期等详细信息来格式化捕获的网页内容。

设置带模板的位置的步骤：

1. 从 iOS 共享面板打开 Obsidian。
2. 点击工具栏中的当前位置。
3. 点击 **+** 按钮创建新位置。
4. 输入位置名称。
5. 选择仓库。
6. 将**行为**设置为**新笔记**。
7. 在**可选**部分，点击**模板**。
8. 从仓库中选择一个笔记作为模板。
9. 点击**保存**以保存位置。

![[ios-share-sheet-set-template.png|400]]

使用此位置共享链接时，Obsidian 会先应用模板，然后添加共享内容。

支持的模板占位符：

| 占位符 | 描述 |
| --- | --- |
| `{{author}}` | 文章作者 |
| `{{description}}` | 文章的描述或摘要 |
| `{{domain}}` | 网站域名 |
| `{{favicon}}` | 网站 favicon 的 URL |
| `{{image}}` | 文章主图的 URL |
| `{{published}}` | 文章发布日期，使用默认日期格式 |
| `{{published: YYYY-MM-DD}}` | 使用自定义日期格式的发布日期 |
| `{{site}}` | 网站名称 |
| `{{title}}` | 文章标题 |
| `{{wordCount}}` | 提取内容的总字数 |

你还可以使用标准模板日期和时间占位符：

| 占位符 | 描述 |
| --- | --- |
| `{{date}}` | 当前日期 |
| `{{date: YYYY-MM-DD}}` | 使用自定义格式的当前日期 |
| `{{time}}` | 当前时间 |
| `{{time: HH:mm}}` | 使用自定义格式的当前时间 |

## Siri 集成

你可以使用 Siri 语音命令与 Obsidian 交互：

- "Capture using Obsidian"
- "Capture to Obsidian"
- "Open my daily note in Obsidian"
- "Search in Obsidian"

## 聚焦搜索集成

当你在 iOS 聚焦搜索中搜索"Obsidian"时，你会看到快捷操作：
- 新建笔记
- 搜索
- 日记
