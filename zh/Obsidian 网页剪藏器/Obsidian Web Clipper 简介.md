Obsidian Web Clipper 是一款免费的浏览器扩展，可让你高亮页面并将网页内容保存到你的仓库中。

## 安装 Obsidian Web Clipper

从你所使用浏览器的官方扩展商店获取 Obsidian Web Clipper 扩展：

- [Chrome 应用商店](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
	- 适用于 Chrome、Brave、Arc、Orion 及其他基于 Chromium 的浏览器。
- [Firefox 附加组件](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
	- 适用于 Firefox 和 Firefox Mobile。
- [Safari 扩展](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
	- 适用于 macOS、iOS 和 iPadOS。
- [Edge 附加组件](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
	- 适用于 Microsoft Edge。

## 隐私

Obsidian Web Clipper 将内容本地保存到你的 Obsidian 仓库中，并遵循我们的[隐私政策](https://obsidian.md/zh/privacy)。你的数据不会被收集，我们也不会采集任何使用指标。代码是[开源](https://github.com/obsidianmd/obsidian-clipper)的，可供审查。
