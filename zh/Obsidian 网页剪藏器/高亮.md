[[Obsidian Web Clipper 简介|Web Clipper]] 允许你高亮网页上的文本，并选择你想要保存到 Obsidian 的元素。高亮内容会被保存，这样当你再次访问页面时可以查看。

高亮内容可以通过扩展程序保存到 Obsidian。你可以在扩展程序设置中定义高亮选项。

## 使用高亮工具

高亮工具可以在实时网页或[[阅读器]]中使用。在阅读器视图中，创建文本选区后可以选择将其高亮。

你也可以通过激活高亮工具来自动将选中内容添加到高亮中：

- 扩展面板或阅读器工具栏中的 ![[lucide-highlighter.svg#icon]] **高亮图标**。
- **快捷键**，可在扩展程序设置中自定义。
- **右键菜单**，在你正在浏览的网页上右键点击。

开启高亮工具后，任何选中的文本、图片和元素都会被添加到你的高亮中。以上所有方式也可以用来退出高亮工具。

## 将高亮添加到笔记

将高亮内容插入到剪藏笔记中有三个选项：

- **高亮页面内容** — 使用 [[Obsidian 风格的 Markdown 语法|语法]] `==highlight==` 直接在文本中添加高亮。
- **替换页面内容** — 返回高亮列表，不包含任何页面内容。
- **不做任何操作** — 返回不含高亮的原始内容。
 
这些选项会改变模板中的 `{{content}}` [[变量|变量]]。你也可以使用 `{{highlights}}` 变量将高亮内容直接添加到模板中，例如：

```
{{highlights|map: item => item.text|join:"\n\n"}}
```

## 查看和搜索高亮

你创建的所有高亮内容都可以在高亮页面中查看。你可以通过 **设置** → **高亮工具** 打开此页面。

## 导出高亮

高亮内容可以导出为 `.json` 文件，可以在设置页面或高亮页面中进行导出。
