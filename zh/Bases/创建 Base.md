[[Introduction to Bases|数据库]]允许你为笔记创建类似数据库的视图。以下是创建数据库并将其嵌入笔记的方法。每个数据库可以拥有一个或多个[[Views|视图]]，以不同方式展示信息。

## 新建数据库

**命令面板：**

1. 打开**命令面板**。
2. 选择
	- **数据库：新建数据库**，在当前文件所在文件夹中创建数据库。
	- **数据库：新建数据库并插入**，创建数据库并将其嵌入到当前文件中。

**文件浏览器：**

1. 在文件浏览器中，右键点击你想要创建数据库的文件夹。
2. 选择**新建数据库**。

**功能区：**

- 在左侧功能区设置中，选择**新建数据库**，在当前文件所在文件夹中创建数据库。

## 嵌入数据库

### 嵌入数据库文件

你可以使用 `![[File.base]]` 语法将数据库文件嵌入到[[插入文件|任何其他文件]]中。要指定默认视图，请使用 `![[File.base#View]]`。

### 以代码块形式嵌入数据库

数据库也可以通过 `base` 代码块和[[Bases syntax|数据库语法]]直接嵌入到笔记中。

~~~yaml
```base
filters:
  and:
    - file.hasTag("example")
views:
  - type: table
    name: Table
```
~~~
