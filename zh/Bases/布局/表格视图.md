表格是[[视图|视图]]的一种类型，可用于[[Bases 简介|数据库]]。

从视图菜单中选择 ![[lucide-table.svg#icon]] **表格**，即可将文件显示为表格形式——每个文件占一行，每列对应该文件的[[属性]]。

![数据库表格视图示例，展示一个书目列表](bases-noshadow.png#interface)

## 设置

表格视图的设置可以在[[视图#视图设置|视图设置]]中进行配置。

### 行高

行高可以让你显示更多信息。可选择**低**、**中**、**高**和**超高**。

## 汇总

你可以为表格列添加汇总，以快速计算当前视图中可见行的总计、平均值或计数等值。

汇总与视图绑定，而非与数据库绑定。每个视图可以为同一列显示不同的汇总。

### 添加汇总

1. 在表格视图中右键点击列标题。
2. 选择 ![[lucide-calculator.svg#icon]] **汇总…**。
3. 选择一个内置汇总函数，或选择 ![[lucide-square-function.svg#icon]] **添加汇总**来自定义。

汇总显示在列底部。当结果被[[视图#排序与分组结果|分组]]时，每个分组的汇总会显示在分组顶部。

添加汇总栏后，你可以点击汇总单元格来为其他列添加更多汇总。如果所有汇总都被移除，汇总栏将自动隐藏。

### 内置汇总

以下汇总默认可用。可用选项可能因属性类型而异。

#### 所有属性类型

- **没有值**：无值的行数。
- **已填写**：有值的行数。
- **时间戳**：不同值的数量。

#### 数字

- **平均值**：所有数值的平均值。
- **最大值**：最大值。
- **中位数**：中位数。
- **最小值**：最小值。
- **范围**：最大值与最小值之差。
- **标准差**：标准差。
- **求和**：所有值的总计。

#### 日期

- **最早**：最小/最早的日期。
- **最晚**：最大/最近的日期。
- **范围**：最早与最晚之间的差值。

#### 复选框

- **已选中**：复选框为开启状态的行数。
- **未选中**：复选框为关闭状态的行数。

### 自定义汇总

你可以使用公式定义自己的汇总：

1. 在 ![[lucide-calculator.svg#icon]] **汇总…** 菜单中，选择 ![[lucide-square-function.svg#icon]] **添加汇总**。
2. 为汇总命名。
3. 输入公式。公式会对该列中的值列表执行运算（例如，使用 `values.reduce(...)` 等[[函数|函数]]）。
4. 保存汇总。

当内置选项无法满足你的计算需求时，自定义汇总非常实用。

## 快捷键

你可以使用以下鼠标和[[编辑相关的快捷键|键盘快捷键]]在表格视图中快速移动。

- 按住 Shift 并点击可创建单元格选区。
- 右键点击单元格选区可访问这些文件的附加操作。

| 操作                                                                                           | 快捷键               | macOS               |
| ---------------------------------------------------------------------------------------------- | -------------------- | ------------------- |
| 复制选中的单元格                                                                               | `Ctrl+C`             | `Cmd+C`             |
| 粘贴选中的单元格                                                                               | `Ctrl+V`             | `Cmd+V`             |
| 撤销属性更改                                                                                   | `Ctrl+Z`             | `Cmd+Z`             |
| 重做属性更改                                                                                   | `Ctrl+Shift+Z`       | `Cmd+Shift+Z`       |
| 选中当前分组中的所有单元格                                                                     | `Ctrl+A`             | `Cmd+A`             |
| 向指定方向选中所有单元格                                                                       | `Ctrl+Shift+Arrow`   | `Ctrl+Shift+Arrow`  |
| 选中该列                                                                                       | `Ctrl+Space`         |                     |
| 选中该行                                                                                       | `Shift+Space`        |                     |
| 聚焦当前单元格——对于复选框，切换选中状态；对于公式，打开公式编辑器                             | `Enter`              |                     |
| 跳转到第一列                                                                                   | `Home`               |                     |
| 跳转到最后一列                                                                                 | `End`                |                     |
| 按页面高度上下导航                                                                             | `PageUp`,`PageDown`  |                     |
| 清除当前单元格选区                                                                             | `Esc`                |                     |
| 清除当前单元格内容                                                                             | `Backspace`          |                     |
| 跳转到后一个单元格                                                                             | `Tab`                |                     |
| 跳转到前一个单元格                                                                             | `Shift-Tab`          |                     |
