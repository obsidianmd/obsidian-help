如果你是一名开发者，你可以通过多种方式为 Obsidian 做出贡献。

### 插件与主题

你可以使用 TypeScript 和 CSS 开发[[第三方插件]]和[[主题]]。请前往 [Obsidian 开发者文档](https://docs.obsidian.md)了解更多。

### 开源项目

Obsidian 开发并维护着多个开源项目。你可以向以下 GitHub 仓库提交包含错误修复和功能增强的拉取请求（Pull Request）：

- [JSON Canvas](https://github.com/obsidianmd/jsoncanvas)（MIT），一种用于无限白板数据的开放文件格式。
- [Obsidian Importer](https://github.com/obsidianmd/obsidian-importer)（MIT），一个可以将多种格式的数据转换为纯文本 Markdown 文件的插件。
- [Obsidian Maps](https://github.com/obsidianmd/obsidian-maps)（MIT），一个在 Obsidian 数据库中显示地图的插件。
- [Obsidian Web Clipper](https://github.com/obsidianmd/obsidian-clipper)（MIT），一个用于高亮和剪藏网页为纯文本 Markdown 文件的浏览器扩展。
- [Defuddle](https://github.com/kepano/defuddle)（MIT），Obsidian Web Clipper 使用的 HTML 转 Markdown 解析器。

### 文档

如果你想帮助改进我们的[开发者文档](https://docs.obsidian.md/Home)，请向 [obsidian-developer-docs](https://github.com/obsidianmd/obsidian-developer-docs) 仓库提交拉取请求。

如果你想帮助改进本文档，请向 [obsidian-help](https://github.com/obsidianmd/obsidian-help) 仓库提交拉取请求。

### 社区

我们官方 Discord 服务器上的 [plugin-dev](https://discord.com/channels/686053708261228577/840286264964022302) 频道是讨论插件开发以及帮助其他开发者解决插件问题的地方。
