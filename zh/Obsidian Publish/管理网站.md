本页介绍如何管理 [[发布服务简介|Obsidian Publish]] 网站。

网站是由 Obsidian Publish 托管并在线提供的笔记集合，可以通过 Obsidian Publish 地址或[[自定义域名]]访问。

## 创建新网站

> [!note] 你可以同时拥有的网站数量取决于你的 Obsidian Publish 订阅。在创建新网站之前，请确保你的订阅允许创建。

1. 在应用窗口左侧的功能区中，点击**发布更改**（ ![[lucide-send.svg#icon]] ）。
2. 如果你已有网站，点击**切换网站**（ ![[lucide-repeat.svg#icon]] ）。
3. 在**网站 ID** 中，输入你想要的网站路径。例如，ID 为 `my-amazing-site` 的网站可通过 publish.obsidian.md/my-amazing-site 访问。
4. 点击**创建**。

## 删除现有网站

> [!note] 即使删除网站，笔记仍会保留在你的仓库中。

1. 在应用窗口左侧的功能区中，点击**发布更改**（ ![[lucide-send.svg#icon]] ）。
2. 点击**切换网站**（ ![[lucide-repeat.svg#icon]] ）。
3. 点击要删除的网站右侧的**删除网站**（ ![[lucide-x.svg#icon]] ）。
4. 点击**删除**确认。

## 在网站之间切换

1. 在应用窗口左侧的功能区中，点击**发布更改**（ ![[lucide-send.svg#icon]] ）。
2. 点击**切换网站**（ ![[lucide-repeat.svg#icon]] ）。
3. 点击要切换到的网站右侧的**选择**。

## 更改网站 ID

1. 在应用窗口左侧的功能区中，点击**发布更改**（ ![[lucide-send.svg#icon]] ）。
2. 点击**切换网站**（ ![[lucide-repeat.svg#icon]] ）。
3. 点击要编辑的网站右侧的**编辑网站 ID**（ ![[lucide-edit-3.svg#icon]] ）。
4. 在**网站 ID** 中，输入网站的新 ID。
5. 点击**更改**。

## 查看网站设置

1. 在应用窗口左侧的功能区中，点击**发布更改**（ ![[lucide-send.svg#icon]] ）。
2. 在**发布更改**对话框中，点击**更改网站设置**（ ![[lucide-cog.svg#icon]] ）。

## 网站设置

### 常规

| 选项                          | 类型   | 描述                                                                                                                |
| ------------------------------- | ------ | -------------------------------------------------------------------------------------------------------------------------- |
| 网站名称                       | 输入  | 你的 Obsidian Publish 网站的公开名称和标题。                                                                                   |
| 主页                   | 输入  | 你希望作为着陆页的 Markdown 文件的位置。                                                    |
| 网站标志                            | 输入  | 你希望作为网站横幅的图片。该图片必须[[发布笔记与取消发布#发布笔记\|已发布]]。 |
| 协作              | 按钮 | 声明你希望授予编辑已发布笔记权限的其他用户。用户必须拥有 Obsidian 账户。             |
| 自定义域名                   | 按钮 | [[自定义域名]]                                                                                                         |
| 禁止搜索引擎索引 | 开关 | 通过添加 `robots.txt` 文件，阻止遵守规范的搜索引擎抓取你的网站。                                   |

### 外观

| 选项            | 类型     | 描述                                                                               |
|-------------------|----------|-------------------------------------------------------------------------------------------|
| 主题             | 下拉菜单 | 选择网站主题的呈现方式：**浅色**、**深色**或**跟随系统**。 |
| 主题配色切换按钮 | 开关   | 允许用户通过网站上的切换按钮在**浅色**/**深色**模式之间切换。          |

## 阅读体验

| 选项               | 类型   | 描述                                                                                |
|----------------------|--------|--------------------------------------------------------------------------------------------|
| 页面预览   | 开关 | 启用或禁用悬停在活动内部链接上时的页面预览功能。 |
| 隐藏页内标题      | 开关 | 启用或禁用已发布笔记的行内标题显示。                    |
| 限制行宽 | 开关 | 启用或禁用网站内的限制行宽功能。                                   |
| 严格换行   | 开关 | 启用或禁用网站上单个换行符的显示。                    |
| 堆叠标签页          | 开关 | 启用或禁用网站上的[[标签页#堆叠标签页组\|堆叠标签页]]功能。    |

### 组件

| 选项                 | 类型   | 描述                                                                                                                          |
|------------------------|--------|--------------------------------------------------------------------------------------------------------------------------------------|
| 显示文件列表        | 开关 | 启用或禁用发布网站上的[[文件列表]]视图。                                                                   |
| 自定义文件列表   | 按钮 | 当**显示文件列表**开启时，[[自定义网站界面#自定义文件列表\|自定义]]文件的排列顺序。 |
| 显示搜索栏        | 开关 | 启用或禁用网站上的搜索栏。                                                                                         |
| 显示局部关系图        | 开关 | 启用或禁用网站右侧边栏中的关系图谱。                                                              |
| 显示大纲 | 开关 | 启用或禁用网站上的[[大纲\|目录]]视图。                                                              |
| 显示反向链接         | 开关 | 启用或禁用网站上的[[反向链接]]。                                                                                        |

### 其他设置

| 选项                         | 类型   | 描述                                            |
| ------------------------------ | ------ | ------------------------------------------------------ |
| 密码                      | 按钮 | [[安全与隐私#添加网站密码\|设置密码]]以限制对整个网站的访问。 |
| 谷歌分析代码 | 输入  | **仅限自定义域名**。在此处放置你的 Google Analytics 网站跟踪代码。                                                       |
