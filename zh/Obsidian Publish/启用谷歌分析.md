## 前提条件

要在你的 [[发布服务简介|Obsidian Publish]] 网站上使用分析功能，你需要一个[[自定义域名]]。

> [!important] 重要
> 在启用分析功能之前，请确保当地法律法规允许你追踪访客。根据你使用的工具，你可能需要在网站上添加一个用户同意横幅。

## 谷歌分析

要为你的 Obsidian Publish 网站启用谷歌分析：

1. 在应用窗口左侧的功能区中，点击**发布更改**（ ![[lucide-send.svg#icon]] ）。
2. 在**发布更改**对话框中，点击**更改网站设置**（ ![[lucide-cog.svg#icon]] ）。
3. 在**谷歌分析代码**中，输入你的跟踪代码。

如果想使用 Google Tag Manager 而非谷歌分析，可以使用自定义 JavaScript 来添加你自己的脚本。

## Plausible、Fathom 及其他分析服务

你可以通过 [[自定义网站界面|publish.js]] 文件将大多数分析服务添加到你的网站。请确保将 `yourdomain.com` 替换为你的域名，并将脚本的 `src` 替换为你的分析服务提供商提供的脚本地址。

以下是使用 [Plausible Analytics](https://plausible.io/) 的示例：

```js
var analyticsScript = document.createElement('script');
analyticsScript.defer = true;
analyticsScript.setAttribute('data-domain', 'yourdomain.com');
analyticsScript.src = 'https://plausible.io/js/plausible.js';
document.head.appendChild(analyticsScript);
```

同样的方法也可以用于 [Fathom Analytics](https://usefathom.com/)。请注意 `data-domain` 变成了 `data-site`——不同的分析服务提供商对脚本插入方式可能有不同的格式要求。

```javascript
var fathom = analyticsScript.createElement('script');
analyticsScript.defer = true;
analyticsScript.setAttribute('data-site', 'yourdomain.com');
analyticsScript.src = 'https://cdn.usefathom.com/script.js';
document.head.appendChild(analyticsScript);
```

## 故障排除

要验证你的网站是否正在使用分析服务，请禁用任何可能阻止跟踪脚本运行的广告拦截浏览器扩展，例如 uBlock Origin。
