了解如何与其他 Obsidian 用户在你的 [[发布服务简介|Obsidian Publish]] 网站上进行协作。通过将你的朋友和同事添加为协作者，他们可以向你的网站发布更改。

只有网站所有者需要拥有有效的 Obsidian Publish 订阅。协作者只需要一个 [Obsidian 账户](https://obsidian.md/zh/account)。

> [!warning] 在向共享网站发布更改之前，请确保[[#在协作者之间同步修改|在协作者之间同步修改]]。否则，你可能会覆盖其他协作者的更改。

## 将协作者添加到网站

1. 在[[功能区]]中，选择**发布更改**（![[lucide-send.svg#icon]]），或打开[[命令面板]]并输入 **Publish: Publish changes…**
2. 在**发布更改**对话框中，点击**更改网站设置** ![[lucide-cog.svg#icon]]。
3. 在**协作**旁边，选择**管理**。
4. 在**邀请用户**中，输入协作者的邮箱。
5. 选择**添加**。

## 从网站移除协作者

1. 在[[功能区]]中，选择**发布更改**（![[lucide-send.svg#icon]]），或打开[[命令面板]]并输入 **Publish: Publish changes…**
2. 在**发布更改**对话框中，点击**更改网站设置**（![[lucide-cog.svg#icon]]）。
3. 在**协作**旁边，选择**管理**。
4. 在你想要移除的协作者旁边，选择**移除用户**（![[lucide-x.svg#icon]]）。

## 在协作者之间同步修改

Obsidian Publish 不会自动在本地仓库之间同步已发布的更改。协作者需要手动同步其他协作者的更改。

要使用线上网站的更改来更新本地笔记：

1. 在[[功能区]]中，选择**发布更改**（![[lucide-send.svg#icon]]），或打开[[命令面板]]并输入 **Publish: Publish changes…**
2. 右键单击或长按你想要同步的更改，然后选择**同步网站版本**。**这将覆盖你本地仓库中的笔记。**

> [!tip] 我们建议你使用其他工具在仓库之间同步更改，例如 [[Obsidian 官方同步简介|Obsidian Sync]] 或 [git](https://git-scm.com/)。

## 权限

下表列出了所有者和协作者可用的网站权限：

| 操作                     | 协作者 | 所有者 |
|--------------------------|:------:|:------:|
| 发布新页面               | ✓      | ✓      |
| 发布已发布页面的更改     | ✓      | ✓      |
| 取消发布页面             | ✓      | ✓      |
| 设置网站选项             |        | ✓      |
| 管理权限                 |        | ✓      |
