Obsidian Web Clipper adalah ekstensi peramban gratis yang memungkinkan Anda menyorot halaman dan menyimpan konten web ke brankas Anda.

## Pasang Web Clipper

Dapatkan ekstensi Obsidian Web Clipper dari direktori resmi untuk peramban Anda:

- [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
	- Untuk Chrome, Brave, Arc, Orion, dan peramban berbasis Chromium lainnya.
- [Firefox Add-Ons](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
	- Untuk Firefox dan Firefox Mobile.
- [Safari Extensions](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
	- Untuk macOS, iOS, dan iPadOS.
- [Edge Add-Ons](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
	- Untuk Microsoft Edge.

## Privasi

Obsidian Web Clipper menyimpan konten secara lokal ke brankas Obsidian Anda dan mengikuti [kebijakan privasi](https://obsidian.md/privacy) kami. Data Anda tidak dikumpulkan, dan kami tidak mengumpulkan metrik penggunaan apa pun. Kodenya bersifat [sumber terbuka](https://github.com/obsidianmd/obsidian-clipper) dan dapat diaudit.
