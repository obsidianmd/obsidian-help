Obsidian dapat digunakan sebagai aplikasi pencatatan untuk individu, dan sebagai alat kolaborasi untuk tim Anda.

## Menggunakan Obsidian dengan tim Anda

- [[Lisensi Komersial]]
	- Dukung Obsidian dengan membeli lisensi komersial.
- [[Menerapkan Obsidian untuk tim Anda|Menerapkan Obsidian]]
	- Cara memasang, memperbarui, dan menyesuaikan Obsidian di seluruh tim Anda.
- [[Sync untuk tim]]
	- Cara menyinkronkan data di seluruh tim Anda.
- [[Publish untuk tim]]
	- Cara mempublikasikan data dari brankas tim Anda.
