Halaman ini mencantumkan pintasan keyboard bawaan untuk menavigasi dan menyunting teks di Obsidian. Pintasan ini disediakan oleh sistem operasi Anda atau kerangka kerja yang digunakan Obsidian, dan tidak dapat dikustomisasi di dalam Obsidian.

Untuk pintasan keyboard yang dapat dikustomisasi untuk perintah Obsidian, lihat [[Pintasan]].

## Pintasan Windows dan Linux

### Tindakan umum

| Tindakan | Pintasan |
|-|-|
| Salin | `Ctrl+C` |
| Potong | `Ctrl+X` |
| Tempel | `Ctrl+V` |
| Tempel tanpa pemformatan | `Ctrl+Shift+V` |
| Urungkan | `Ctrl+Z` |
| Ulangi | `Ctrl+Shift+Z` atau `Ctrl+Y` |
| Salin paragraf | `Ctrl+C` (tanpa teks yang dipilih) |
| Potong paragraf | `Ctrl+X` (tanpa teks yang dipilih) |

### Penyuntingan teks

| Tindakan | Pintasan |
|-|-|
| Sisipkan baris baru | `Enter` |
| Hapus karakter sebelumnya | `Backspace` |
| Hapus karakter berikutnya | `Delete` |
| Hapus kata sebelumnya | `Ctrl+Backspace` |
| Hapus kata berikutnya | `Ctrl+Delete` |
| Hapus baris saat ini | `Ctrl+Shift+K` (tanpa teks yang dipilih) |

### Navigasi teks

| Tindakan | Pintasan |
|-|-|
| Pindahkan kursor satu karakter | `Left/→` |
| Pindahkan kursor ke awal kata sebelumnya | `Ctrl+←` |
| Pindahkan kursor ke akhir kata berikutnya | `Ctrl+→` |
| Pindahkan kursor ke awal baris saat ini | `Home` |
| Pindahkan kursor ke akhir baris saat ini | `End` |
| Pindahkan kursor ke baris sebelumnya | `↑` |
| Pindahkan kursor ke baris berikutnya | `↓` |
| Pindahkan kursor ke awal catatan | `Ctrl+Home` |
| Pindahkan kursor ke akhir catatan | `Ctrl+End` |
| Pindahkan kursor satu halaman ke atas | `Page up` |
| Pindahkan kursor satu halaman ke bawah | `Page down` |

### Pemilihan teks

| Tindakan | Pintasan |
|-|-|
| Sederhanakan pilihan | `Escape` |
| Pilih semua | `Ctrl+A` |
| Perluas pilihan satu karakter | `Shift+Left/→` |
| Perluas pilihan ke awal kata sebelumnya | `Ctrl+Shift+←` |
| Perluas pilihan ke akhir kata berikutnya | `Ctrl+Shift+→` |
| Perluas pilihan ke awal baris saat ini | `Shift+Home` |
| Perluas pilihan ke akhir baris saat ini | `Shift+End` |
| Perluas pilihan ke awal catatan | `Ctrl+Shift+Home` |
| Perluas pilihan ke akhir catatan | `Ctrl+Shift+End` |
| Perluas pilihan satu halaman ke atas | `Shift+Page up` |
| Perluas pilihan satu halaman ke bawah | `Shift+Page down` |

## Pintasan macOS

### Tindakan umum

| Tindakan | Pintasan |
|-|-|
| Salin | `Cmd+C` |
| Potong | `Cmd+X` |
| Tempel | `Cmd+V` |
| Tempel tanpa pemformatan | `Cmd+Shift+V` |
| Urungkan | `Cmd+Z` |
| Ulangi | `Cmd+Shift+Z` |
| Salin paragraf | `Cmd+C` (tanpa teks yang dipilih) |
| Potong paragraf | `Cmd+X` (tanpa teks yang dipilih) |

### Pemformatan teks

| Tindakan | Pintasan |
| ---- | ---- |
| Teks tebal | `Cmd+B` |
| Teks miring | `Cmd+I` |

### Penyuntingan teks

| Tindakan | Pintasan |
|-|-|
| Sisipkan baris baru | `Enter` |
| Hapus karakter sebelumnya | `Backspace` |
| Hapus karakter berikutnya | `Delete` |
| Hapus kata sebelumnya | `Option+Backspace` |
| Hapus kata berikutnya | `Option+Delete` |
| Hapus ke awal baris saat ini | `Cmd+Backspace` |
| Hapus ke akhir baris saat ini | `Cmd+Delete` |
| Hapus baris saat ini | `Cmd+Shift+K` (tanpa teks yang dipilih) |

### Navigasi teks

| Tindakan | Pintasan |
|-|-|
| Pindahkan kursor satu karakter | `Left/→` |
| Pindahkan kursor ke awal kata sebelumnya | `Option+←` |
| Pindahkan kursor ke akhir kata berikutnya | `Option+→` |
| Pindahkan kursor ke awal baris saat ini | `Cmd+←` |
| Pindahkan kursor ke akhir baris saat ini | `Cmd+→` |
| Pindahkan kursor ke baris sebelumnya | `↑` |
| Pindahkan kursor ke baris berikutnya | `↓` |
| Pindahkan kursor ke awal catatan | `Cmd+↑` |
| Pindahkan kursor ke akhir catatan | `Cmd+↓` |
| Pindahkan kursor satu halaman ke atas | `Fn+↑` |
| Pindahkan kursor satu halaman ke bawah | `Fn+↓` |

### Pemilihan teks

| Tindakan                                                | Pintasan                   |
| ------------------------------------------------------- | -------------------------- |
| Sederhanakan pilihan                                    | `Escape`                   |
| Pilih semua                                             | `Cmd+A`                    |
| Perluas pilihan satu karakter                           | `Shift+Left/→`   |
| Perluas pilihan ke awal kata sebelumnya                 | `Option+Shift+←`  |
| Perluas pilihan ke akhir kata berikutnya                | `Option+Shift+→` |
| Perluas pilihan ke awal baris saat ini                  | `Cmd+Shift+←`              |
| Perluas pilihan ke akhir baris saat ini                 | `Cmd+Shift+→`    |
| Perluas pilihan ke awal catatan                         | `Cmd+Shift+↑`       |
| Perluas pilihan ke akhir catatan                        | `Cmd+Shift+↓`     |
| Perluas pilihan satu halaman ke atas                    | `Ctrl+Shift+↑`      |
| Perluas pilihan satu halaman ke bawah                   | `Ctrl+Shift+↓`    |
