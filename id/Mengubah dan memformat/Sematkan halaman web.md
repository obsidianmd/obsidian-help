Pelajari cara menggunakan elemen HTML [iframe](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/iframe) untuk menyematkan halaman web di catatan Anda.

Untuk menyematkan halaman web, tambahkan kode berikut di catatan Anda dan ganti teks placeholder dengan URL halaman web yang ingin Anda sematkan:

```html
<iframe src="MASUKKAN URL ANDA DI SINI"></iframe>
```

> [!note] Catatan
> Beberapa situs web tidak mengizinkan Anda menyematkannya. Sebagai gantinya, mereka mungkin menyediakan URL yang ditujukan untuk menyematkannya. Jika situs web tidak mendukung penyematan, coba cari nama situs web tersebut diikuti dengan "embed iframe". Misalnya, "youtube embed iframe".

> [!tip] Tip
> Jika Anda menggunakan [[Canvas]], Anda dapat menyematkan halaman web di sebuah kartu. Untuk informasi lebih lanjut, lihat [[Canvas#Tambah kartu dari halaman web]].

## Menyematkan video YouTube

Untuk menyematkan video YouTube, gunakan sintaksis Markdown yang sama seperti [[Sintaksis format dasar#Gambar eksternal|gambar eksternal]]:

```md
![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)
```

![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)

## Menyematkan tweet

Untuk menyematkan tweet, gunakan sintaksis Markdown yang sama seperti [[Sintaksis format dasar#Gambar eksternal|gambar eksternal]]:

```md
![](https://twitter.com/obsdmd/status/1580548874246443010)
```
