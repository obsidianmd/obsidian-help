Anda dapat mengimpor [[Format file yang diterima]], atau _lampiran_, ke brankas Anda, seperti gambar, file audio, atau PDF. Lampiran adalah file biasa yang dapat Anda akses menggunakan sistem file Anda. Lampiran dapat [[Sematkan file|disematkan]].

## Menambahkan lampiran

Anda dapat menambahkan lampiran ke brankas Anda dengan berbagai cara. Hanya [[Format file yang diterima]] yang dapat ditambahkan.

> [!todo]- Salin dan tempel lampiran
> Anda dapat menempelkan lampiran langsung ke catatan Anda. Obsidian membuat file dengan konten yang ditempelkan di lokasi lampiran bawaan dan [[Sematkan file|menyematkannya]] di catatan.

> [!todo]- Seret dan lepas lampiran
> Jika Anda menyeret file dari sistem file Anda ke editor yang terbuka, Obsidian menyalin file tersebut ke lokasi lampiran bawaan dan [[Sematkan file|menyematkannya]] di catatan.

## Mengubah lokasi lampiran bawaan

Secara bawaan, lampiran ditambahkan ke root brankas Anda.

Anda dapat mengubah lokasi lampiran bawaan di **[[Pengaturan]] → Files & Links → Default location for new attachments**.

- **Vault folder** menambahkan lampiran ke root brankas Anda.
- **In the folder specified below** menambahkan lampiran ke folder yang ditentukan.
- **Same folder as current file** menambahkan lampiran ke folder yang sama dengan catatan tempat Anda menambahkannya.
- **In subfolder under current folder** menambahkan lampiran ke folder yang ditentukan di samping catatan tempat Anda menambahkan lampiran. Jika folder tersebut belum ada, Obsidian akan membuatnya saat Anda menambahkan lampiran.
