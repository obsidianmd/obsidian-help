Obsidian memungkinkan Anda mengontrol cara membaca dan mengedit catatan yang ditulis dalam Markdown menggunakan _tampilan_ dan _mode_.

- **Tampilan** beralih antara membaca dan mengedit catatan Anda.
- **Mode** mengontrol bagaimana Markdown ditampilkan saat Anda mengedit.

> [!note] Catatan
> Secara bawaan, Obsidian membuka tab baru dalam mode pengeditan. Anda dapat mengubah ini di **[[Pengaturan]] → Editor → Tampilan bawaan untuk tab baru**.

## Tampilan baca

*Tampilan baca* menampilkan catatan Anda tanpa sintaksis Markdown, menawarkan format yang bersih dan mudah dibaca untuk tinjauan yang fokus.

Untuk beralih ke *Tampilan baca*:

- Klik peralih tampilan ![[lucide-book-icon.svg#icon]] di pojok kanan atas editor.
- Atau klik ikon status interaktif ( ![[lucide-edit-3.svg#icon]] atau ![[lucide-code-xml.svg#icon]] ) di bilah status dan pilih **Tampilan baca**.
- Atau tekan `Ctrl+E` (`Cmd+E` di macOS).

> [!note] Catatan
> Aktifkan **[[Pengaturan]] → Tampilan → Tampilkan tab bilah judul** untuk melihat ikon peralih tampilan.
> 
> Jika itu dinonaktifkan, aktifkan **[[Pengaturan]] → Editor → Status Pengubah** untuk beralih tampilan dari bilah status.

> [!tip] Tampilkan berdampingan
> Untuk membuka catatan dalam *Tampilan pengeditan* dan *Tampilan baca* secara bersamaan, tahan `Ctrl` (atau `Cmd` di macOS) dan klik peralih tampilan.

## Tampilan pengeditan

*Tampilan pengeditan* memungkinkan Anda melakukan perubahan pada catatan Anda.

Saat dalam *Tampilan pengeditan*, *Mode pengeditan* menentukan bagaimana Markdown ditampilkan. Anda dapat memilih salah satu dari dua *Mode pengeditan*: *Pratinjau Langsung* atau *Mode Sumber*.

### Pratinjau Langsung

*Pratinjau Langsung* menampilkan teks terformat secara inline sambil menyembunyikan sebagian besar sintaksis Markdown. Ketika kursor Anda memasuki konten terformat, sintaksis yang mendasarinya menjadi terlihat untuk pengeditan.

Untuk beralih ke *Pratinjau Langsung*:

- Klik peralih tampilan ![[lucide-edit-3.svg#icon]] di pojok kanan atas editor.
- Atau klik ikon status interaktif ( ![[lucide-book-icon.svg#icon]] atau ![[lucide-code-xml.svg#icon]] ) di bilah status dan pilih **Pratinjau Langsung**.
- Atau tekan `Ctrl+E` (`Cmd+E` di macOS).
- Atau gunakan [[Palet perintah|perintah]] **Ubah mode pengubah/pratinjau**.

> [!note] Catatan
> Secara bawaan, tampilan pengeditan diatur ke *Pratinjau Langsung*. Anda dapat mengubah ini di **[[Pengaturan]] → Editor → Mode ubah bawaan**.

> [!tip] Dalam banyak kasus, Pratinjau Langsung dapat menghilangkan kebutuhan untuk beralih ke [[#Tampilan baca]].

### Mode Sumber

*Mode Sumber* menampilkan semua sintaksis Markdown persis seperti yang ditulis. Gunakan ini jika Anda lebih suka teks biasa atau membutuhkan kontrol format yang presisi.

Untuk beralih ke *Mode Sumber*:
- Klik ikon status interaktif ( ![[lucide-book-icon.svg#icon]] atau ![[lucide-edit-3.svg#icon]] ) di bilah status dan pilih **Mode Sumber**.

> [!note] Catatan
> Secara bawaan, tampilan pengeditan diatur ke *Pratinjau Langsung*. Ubah ini ke *Mode Sumber* di **[[Pengaturan]] → Editor → Mode ubah bawaan**.
> 
> Untuk beralih ke *Mode Sumber*, sekarang Anda juga dapat:
> 
> - Klik peralih tampilan ![[lucide-edit-3.svg#icon]] di pojok kanan atas catatan Anda.
> - Atau tekan `Ctrl+E` (`Cmd+E` di macOS).
> - Atau gunakan [[Palet perintah|perintah]] **Ubah mode pengubah/pratinjau**.

> [!tip] Beralih *Mode pengeditan*
> Untuk beralih antara *Pratinjau Langsung* dan *Mode Sumber* dengan cepat, Anda dapat mengatur [[Pintasan|Pintasan keyboard]] untuk [[Palet perintah|perintah]] **Pengalih Pratinjau Nyata/Mode Sumber**.
