Lisensi Catalyst adalah pembelian satu kali yang memberi Anda akses awal ke versi beta dan membantu mendukung pengembangan Obsidian yang berkelanjutan. Anda dapat mendukung Obsidian dengan [membeli lisensi Catalyst](https://obsidian.md/buy/catalyst).

Manfaat yang didapatkan:

- Akses awal ke [[#Aktifkan versi akses awal|versi baru]] Obsidian
- Akses ke [[#Gabung ke kanal Insider kami|kanal Insider]] eksklusif.
- [[#Dapatkan lencana forum Anda|Lencana forum]] dan [[#Dapatkan lencana Discord Anda|lencana Discord]] eksklusif.

## Tingkatan Catalyst

Ada tiga tingkatan Catalyst berdasarkan jumlah kontribusi Anda: *Insider*, *Supporter*, dan *VIP*.

| Manfaat                                                                                    | Insider | Supporter |   VIP    |
| ------------------------------------------------------------------------------------------ | :-----: | :-------: | :------: |
| Akses awal ke versi baru                                                                   |    ✓    |     ✓     |    ✓     |
| Akses ke kanal pengembangan eksklusif, `#insider-desktop` dan `#insider-mobile`             |    ✓    |     ✓     |    ✓     |
| Akses ke kanal lounge eksklusif, `#vip-lounge`                                             |         |     ✓     |    ✓     |
| Lencana Insider                                                                            |    ✓    |           |          |
| Lencana Supporter                                                                          |         |     ✓     |          |
| Lencana VIP                                                                                |         |           |    ✓     |
| **Kontribusi (USD)**                                                                       | **$25** |  **$50**  | **$100** |


> [!info] Tingkatkan tingkatan Catalyst Anda
> Jika Anda sudah memiliki lisensi Catalyst, Anda dapat meningkatkan ke tingkatan Catalyst yang lebih tinggi dengan membayar selisih harga antara tingkatan Anda saat ini dan tingkatan baru.

## Akses manfaat Catalyst

Setelah Anda membeli lisensi Catalyst, Anda dapat mengakses berbagai manfaat. Pelajari cara mengaktifkannya di sini.

### Dapatkan lencana forum Anda

**Catatan:** Untuk menghubungkan akun Obsidian dan akun forum Anda, Anda harus menggunakan alamat email yang sama untuk kedua akun.

1. Jika belum, bergabunglah ke [forum](https://forum.obsidian.md/) kami.
2. Buka [dasbor akun Obsidian](https://obsidian.md/account/catalyst) Anda.
3. Di bagian **Catalyst**, pilih **Get forum badge**.

### Dapatkan lencana Discord Anda

1. Jika belum, bergabunglah ke [server Discord](https://discord.gg/veuWUTm) kami.
2. Buka [dasbor akun Obsidian](https://obsidian.md/account/catalyst) Anda.
3. Di bagian **Catalyst**, pilih **Get Discord badge**.

### Gabung ke kanal Insider kami

Insider dapat mengobrol dengan tim Obsidian dan Insider lainnya di kanal Discord eksklusif kami. Anda secara otomatis bergabung ke kanal Insider setelah Anda [[#Dapatkan lencana Discord Anda]].

### Aktifkan versi akses awal

Pelajari cara menerima [[Versi akses awal]].
