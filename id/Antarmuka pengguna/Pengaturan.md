Pengaturan memungkinkan Anda menyesuaikan pengalaman Obsidian Anda. Konfigurasikan opsi umum, preferensi editor, tampilan, pintasan keyboard, dan kelola plugin inti maupun komunitas.

## Buka Pengaturan

### Desktop

Di [[Bilah samping#Buka bilah samping tersembunyi|bilah samping kiri]], pilih **[[Pengaturan]]** ![[lucide-cog.svg#icon]]. Anda juga dapat membuka Pengaturan dengan [[Palet perintah]].

### Seluler

Di bilah samping kiri, pilih **[[Pengaturan]]** ![[lucide-cog.svg#icon]].

## Organisasi pengaturan

Pengaturan diorganisasikan ke dalam kategori berikut:

- **Pengaturan** — Pengaturan inti aplikasi termasuk [[#Umum]], [[#Editor]], [[#Berkas dan tautan]], [[#Tampilan]], dan [[#Pintasan]].
- **[[Plugin inti]]** — Plugin bawaan yang sudah tersedia di Obsidian.
- **[[Plugin komunitas]]** — Plugin pihak ketiga yang dipasang dari komunitas.

## Umum

Halaman Umum berisi informasi versi, pengaturan pembaruan, preferensi bahasa, dan manajemen akun.

### Versi dan pembaruan

Lihat versi Obsidian dan versi penginstal Anda saat ini di bagian atas halaman. Pilih **Cek pembaruan** untuk memeriksa versi baru secara manual, atau pilih **[Baca log perubahan](<https://obsidian.md/changelog/>)** untuk melihat catatan rilis.

**Perbarui otomatis**

Saat diaktifkan, Obsidian secara otomatis memeriksa versi baru dan memberi tahu Anda ketika pembaruan tersedia.

**Dapatkan versi insider**

Perbarui otomatis ke [[Versi akses awal]] terbaru. Versi ini mencakup fitur baru tetapi mungkin kurang stabil.

> [!info]+ Lisensi Catalyst diperlukan
> Opsi ini hanya tersedia untuk pengguna dengan [[Lisensi Catalyst]].

### Bahasa

Pilih bahasa pilihan Anda untuk antarmuka Obsidian. Pilih **[[Terjemahan|Pelajari cara menambahkan bahasa baru ke Obsidian]]** untuk berkontribusi terjemahan.

### Bantuan

Pilih **[[Beranda|Buka]]** untuk mengakses sumber bantuan, termasuk dokumentasi, forum komunitas, dan panduan pemecahan masalah.

### Akun

Kelola akun dan lisensi Obsidian Anda.

**Akun Anda**

Lihat informasi akun Anda, termasuk nama pengguna dan email. Pilih **[Kelola](https://obsidian.md/account)** untuk mengakses pengaturan akun, atau pilih **Log Out** untuk keluar dari aplikasi.

**Lisensi Catalyst**

Lihat status lisensi Catalyst Anda. Jika Anda memiliki [[Lisensi Catalyst]] aktif, tingkat lisensi Anda akan muncul di sini.

**Lisensi komersial**

Jika Anda mendukung [Obsidian untuk kerja](<https://obsidian.md/blog/free-for-work/>), pilih **Aktifkan** untuk memasukkan kunci lisensi komersial, atau pilih **[Beli](https://obsidian.md/account/commercial)** untuk membeli lisensi. Pilih **[[Lisensi Komersial|Pelajari lebih lanjut]]** untuk informasi tentang lisensi komersial.

### Lanjutan

#### Beri tahu jika startup lebih lama dari perkiraan

Saat diaktifkan, Obsidian menampilkan notifikasi jika startup sangat lambat. Notifikasi tersebut menyertakan informasi diagnostik tentang penyebab keterlambatan. Pilih ikon timer ![[lucide-timer.svg#icon]] di sebelah pengaturan ini untuk [[Bantuan dan dukungan#Periksa waktu startup|memeriksa waktu startup Anda]].

## Editor

Halaman Editor berisi pengaturan untuk cara Anda melihat dan mengedit catatan.

### Selalu fokus ke tab baru

Saat membuka tautan di tab baru, langsung beralih ke tab tersebut.

### Tampilan bawaan untuk tab baru

Tampilan bawaan saat tab file Markdown baru dibuka. Pilih antara [[Tampilan dan mode penyuntingan#Tampilan baca|Tampilan baca]] atau [[Tampilan dan mode penyuntingan#Tampilan pengeditan|Pengubah]].

### Mode ubah bawaan

Mode bawaan pengubah pada saat tab baru dimulai. Pilih antara [[Tampilan dan mode penyuntingan#Pratinjau langsung|Pratinjau langsung]] atau [[Tampilan dan mode penyuntingan#Mode sumber|Mode sumber]].

### Status Pengubah

Tampilkan tombol mode pengeditan di [[Bilah status|bilah status]].

### Tampilan

#### Panjang baris yang dapat dibaca

Membatasi panjang maksimum baris. Lebih sedikit konten terlihat, tetapi teks panjang lebih mudah dibaca.

#### Pemisah baris ketat

Spesifikasi Markdown mengabaikan [[Sintaksis format dasar#Pemisah baris|pemisah baris]] tunggal dalam tampilan baca. Matikan ini untuk membuat pemisah baris tunggal terlihat.

#### Properti di dokumen

Pilih bagaimana [[Properti|properti]] ditampilkan di bagian atas catatan. Pilih **sumber** untuk menampilkan properti sebagai YAML mentah.

#### Lipat judul

[[Melipat|Lipat]] semua konten di bawah judul dengan memilih ikon perkecil di sebelahnya.

### Lipat indentasi

[[Melipat|Lipat]] bagian dari indentasi, seperti daftar, dengan memilih ikon perkecil.

#### Nomor baris

Tampilkan nomor baris di sisi.

#### Bantuan indentasi

Tampilkan garis hubungan vertikal di antara item daftar.

#### Kanan-ke-Kiri (RTL)

Atur arah teks dari catatan untuk ditampilkan secara kanan-ke-kiri.

#### Pasangkan tanda kurung otomatis

Memasangkan tanda kurung dan tanda kutip secara otomatis.

#### Pasangkan sintaks Markdown otomatis

Memasangkan simbol otomatis untuk tebal, miring, kode, dan lainnya.

#### Daftar cerdas

Secara otomatis mengatur indentasi dan penempatan item daftar.

#### Gunakan tab

Gunakan tab untuk indentasi dengan menekan tombol Tab. Matikan ini untuk menggunakan 4 spasi sebagai indentasi.

#### Ubah HTML otomatis

Secara otomatis mengubah HTML ke Markdown saat menempel dan seret-dan-lepas dari halaman web.

Gunakan `Ctrl/Cmd+Shift+V` untuk menempel HTML tanpa mengonversi.

### Perilaku

#### Pemeriksa ejaan

Aktifkan pemeriksa ejaan. Pilih ikon roda gigi untuk menghapus item tersimpan dari kamus kustom.

#### Bahasa pemeriksa ejaan

**Windows dan Linux:**

Pilih bahasa yang digunakan oleh pemeriksa ejaan. Pilih ikon plus (+) untuk menambahkan bahasa. Untuk menghapus bahasa, pilih X di sebelahnya.

**macOS:**

Pemeriksa ejaan bawaan secara otomatis mendeteksi bahasa yang digunakan oleh OS.

#### Ukuran tab

Jumlah spasi yang setara dengan tab.

### Lanjutan

#### Key binding Vim

Memungkinkan Anda untuk menggunakan key binding Vim dalam pengubah sumber.

## Berkas dan tautan

Halaman Berkas dan tautan berisi pengaturan untuk manajemen file, tautan, dan perilaku brankas.

### Lokasi bawaan untuk catatan baru

Di mana menaruh setiap catatan yang baru dibuat. Opsi meliputi:

- **Folder vault** — Catatan dibuat di root brankas Anda.
- **Satu folder dengan berkas sekarang** — Catatan dibuat di folder yang sama dengan catatan yang sedang aktif.
- **Dalam folder yang dikhususkan ini** — Catatan dibuat di folder spesifik yang Anda pilih.

### Lokasi bawaan untuk lampiran baru

Di mana [[Lampiran|lampiran]] yang baru ditambahkan ditempatkan. Opsi meliputi:

- **Folder vault** — Lampiran ditempatkan di root brankas Anda.
- **Dalam folder yang dikhususkan ini** — Lampiran ditempatkan di folder spesifik yang Anda pilih.
- **Satu folder dengan berkas sekarang** — Lampiran ditempatkan di folder yang sama dengan catatan yang sedang aktif.
- **Di subfolder di bawah folder sekarang** — Lampiran ditempatkan di subfolder dalam folder catatan saat ini.

### Tautan

#### Format tautan baru

Tautan apa yang disisipkan saat menghasilkan [[Tautan internal|tautan internal]] secara otomatis. Opsi meliputi:

- **Alur terdekat jika memungkinkan** — Menggunakan alur unik terpendek ke file yang ditautkan.
- **Alur relatif ke file** — Menggunakan alur relatif terhadap file saat ini.
- **Alur absolut dalam brankas** — Menggunakan alur lengkap dari root brankas.

#### Secara otomatis perbarui tautan internal

Saat diaktifkan, Obsidian secara otomatis memperbarui tautan internal ketika Anda mengganti nama file. Matikan ini untuk diminta memperbarui tautan setelah mengganti nama.

#### Gunakan Wikilink

Hasilkan Wikilink secara otomatis untuk `[[tautan]]` dan `![[gambar]]` alih-alih tautan dan gambar Markdown. Nonaktifkan opsi ini untuk menghasilkan tautan Markdown sebagai gantinya.

#### Tampilkan semua jenis berkas

Tampilkan berkas dengan ekstensi apa pun meski jika Obsidian tidak dapat membuka mereka, jadi Anda dapat menghubungkan ke mereka dan melihat mereka di Penjelajah berkas dan Peralih cepat.

### Tempat sampah

#### Konfirmasi penghapusan file

Tanyakan sebelum menghapus file.

#### Berkas terhapus

Apa yang terjadi pada berkas setelah Anda hapus. Opsi meliputi:

- **Pindahkan ke tempat sampah sistem** — File dipindahkan ke tempat sampah sistem operasi Anda.
- **Pindahkan ke tempat sampah Obsidian** — File dipindahkan ke folder `.trash` di brankas Anda.
- **Hapus secara permanen** — File dihapus secara permanen dan tidak dapat dipulihkan.

### Lanjutan

#### Berkas yang dikecualikan

Berkas yang dikecualikan akan disembunyikan di [[Cari]], [[Tampilan grafik]], dan Sebutan yang belum tertaut ([[Backlink]] dan [[Tautan Keluar]]), dan kurang terlihat di [[Peralih cepat]] dan saran tautan. Pilih **Kelola** untuk mengonfigurasi berkas yang dikecualikan.

#### Timpa folder konfigurasi

Gunakan [[Folder konfigurasi|folder konfigurasi]] yang berbeda dari bawaan. Harus dimulai dengan titik.

#### Izinkan callback URI

Aktifkan penggunaan x-callback-url melalui x-success atau x-error saat menangani [[Obsidian URI|URI Obsidian]].

#### Bangun ulang cache brankas

Membangun ulang [[Cara Obsidian menyimpan data#Cache metadata|cache metadata]] brankas. Membangun ulang cache dapat memakan waktu beberapa detik hingga beberapa menit tergantung pada ukuran brankas Anda. Pilih **Bangun ulang** untuk memulai proses.

## Tampilan

Halaman Tampilan berisi pengaturan untuk menyesuaikan tampilan Obsidian.

### Tema dasar

Pilih skema warna bawaan Obsidian. Opsi meliputi:

- **Mengikuti sistem** — Menyesuaikan mode terang atau gelap sistem operasi Anda.
- **Terang** — Menggunakan mode terang.
- **Gelap** — Menggunakan mode gelap.

### Warna aksen

Pilih warna aksen yang digunakan pada aplikasi. Pilih pemilih warna atau ikon atur ulang untuk menyesuaikan.

### Tema

Kelola [[Tema|tema]] yang terpasang dan telusuri tema komunitas. Pilih **Kelola** untuk melihat dan memasang tema. Dropdown menampilkan tema yang sedang aktif. Ikon folder ![[lucide-folder-open.svg#icon]] membuka folder tema di [[Folder konfigurasi|folder konfigurasi]] Anda.

### Tema komunitas terkini

Menampilkan jumlah tema komunitas yang telah Anda pasang.

### Font

#### Font antarmuka

Atur font dasar untuk semua bagian Obsidian. Pilih **Kelola** untuk memilih font kustom atau masukkan nama font.

#### Font teks

Atur font untuk tampilan pengubah dan baca. Pilih **Kelola** untuk memilih font kustom atau masukkan nama font.

#### Font monospace

Atur font untuk tempat-tempat seperti blok kode dan frontmatter. Pilih **Kelola** untuk memilih font kustom atau masukkan nama font.

#### Ukuran font

Ukuran font dalam piksel yang mempengaruhi editor dan pratinjau. Sesuaikan menggunakan slider.

#### Pengaturan cepat ukuran font

Sesuaikan ukuran font menggunakan `Ctrl+Scroll` (Windows/Linux) atau `Cmd+Scroll` (macOS), atau menggunakan gerakan cubit-perbesar trackpad.

### Antarmuka

#### Judul sebaris

Menampilkan nama file sebagai judul yang dapat diedit secara sebaris dengan konten file. Judul sebaris berfungsi sebagai judul di dalam Obsidian tetapi tidak menambahkan judul ke catatan Anda.

#### Tampilkan tab bilah judul

Menampilkan judul pada bagian atas setiap tab.

#### Tampilkan bilah alat

Tampilkan toolbar vertikal di sisi jendela. Pelajari lebih lanjut tentang [[Bilah alat|bilah alat]].

#### Menu ribbon

Atur perintah apa yang tampil pada menu ribbon. Pilih **Kelola** untuk menyesuaikan.

> [!info]+ Konfigurasi ribbon seluler
> Di seluler, kustomisasi ribbon bekerja secara berbeda. Lihat [[Bilah alat#Seluler|Kustomisasi ribbon di seluler]] untuk detail.

### Lanjutan

#### Tingkat pembesaran

Atur tingkat pembesaran dari keseluruhan aplikasi. Sesuaikan menggunakan slider.

#### Menu bawaan

Menu pada aplikasi akan menyesuaikan sistem operasi. Mereka tidak akan dipengaruhi oleh tema Anda.

#### Gaya bingkai jendela

Tentukan gaya bilah judul dari jendela Obsidian. Memerlukan pemuatan ulang secara penuh untuk mendapatkan hasil. Opsi meliputi:

- **Bingkai Obsidian** — Menggunakan bingkai jendela kustom Obsidian.
- **Bingkai bawaan** — Menggunakan bingkai jendela bawaan sistem operasi Anda.
- **Bingkai tersembunyi** — Menyembunyikan bilah judul sepenuhnya.

#### Ikon aplikasi kustom

Atur ikon kustom untuk aplikasi. Pilih **Pilih** untuk memilih file ikon (`.icns`, `.ico`, `.png`, atau `.svg`). Pelajari lebih lanjut tentang [[Tampilan#Ikon aplikasi kustom|ikon aplikasi kustom]].

#### Jendela transparan

Aktifkan efek transparan untuk memberi kesan lebih dalam. Paling baik digunakan dengan mode gelap. Tidak didukung di Linux.

> [!info]+ Khusus macOS
> Fitur ini terutama dirancang untuk macOS. Pelajari lebih lanjut tentang [[Tampilan#Transparansi|transparansi]].

#### Akselerasi Perangkat Keras

Aktifkan Akselerasi Perangkat Keras, menggunakan GPU Anda untuk membuat Obsidian lebih lancar. Jika Anda menonaktifkan ini, performa aplikasi dapat sangat menurun tetapi mungkin menyelesaikan beberapa masalah langka.

### Cuplikan CSS

Kelola dan lihat [[Cuplikan CSS|cuplikan CSS]] yang tersimpan di brankas Anda. Cuplikan CSS disimpan di `/vault/.obsidian/snippets/`. Pilih ikon muat ulang untuk memuat ulang cuplikan, atau ikon folder ![[lucide-folder-open.svg#icon]] untuk membuka folder cuplikan.

## Pintasan

Halaman [[Pintasan]] memungkinkan Anda melihat, mengatur, dan menyesuaikan [[Pintasan penyuntingan|pintasan keyboard]] untuk perintah di Obsidian.

Gunakan bilah pencarian untuk memfilter perintah, dan pilih ikon filter untuk hanya menampilkan perintah yang sudah memiliki pintasan. Untuk menambahkan pintasan, pilih ikon plus (+) di sebelah perintah. Untuk menghapus pintasan, pilih ikon X di sebelah kombinasi keyboard.

## Plugin inti

[[Plugin inti]] adalah plugin bawaan yang sudah tersedia di Obsidian. Setiap plugin menambahkan fungsionalitas spesifik ke aplikasi.

Untuk mengaktifkan atau menonaktifkan plugin inti, pilih tombol toggle di sebelah nama plugin. Pilih ikon plus ![[lucide-plus-circle.svg#icon]] untuk melihat [[Pintasan]] untuk plugin tersebut. Pilih ikon pengaturan ![[lucide-settings.svg#icon]] untuk membuka halaman pengaturan plugin.

Gunakan bilah pencarian untuk memfilter plugin berdasarkan nama.

## Plugin komunitas

[[Plugin komunitas]] adalah plugin pihak ketiga yang dibuat oleh komunitas Obsidian. Telusuri dan pasang plugin dengan memilih **Telusuri**.

### Mode terbatas

[[Keamanan plugin#Mode terbatas|Mode terbatas]] menonaktifkan semua plugin komunitas untuk keamanan. Pilih **Nyalakan** untuk mengaktifkan mode terbatas.

### Plugin saat ini

Menampilkan jumlah plugin komunitas yang telah Anda pasang. Pilih **Cek pembaruan** untuk memeriksa pembaruan plugin.

### Plugin terpasang

Lihat dan kelola plugin komunitas yang terpasang. Gunakan bilah pencarian untuk memfilter plugin berdasarkan nama. Setiap plugin memiliki ikon untuk pengaturan, pintasan, pendanaan, penghapusan pemasangan, dan toggle untuk mengaktifkan atau menonaktifkannya.
