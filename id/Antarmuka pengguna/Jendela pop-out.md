> [!compatibility] Fitur ini hanya tersedia di Desktop.

Anda dapat membuka _jendela pop-out_ terpisah di Obsidian. Setiap jendela pop-out terkait dengan jendela brankas masing-masing. Jika Anda menutup jendela brankas, semua jendela pop-out yang terkait juga akan tertutup.

File hanya dapat dipindahkan antar jendela yang terkait dengan brankas yang sama.

## Membuka file di jendela baru

**Penjelajah berkas:**

Di [[Penjelajah berkas]], klik kanan pada catatan dan pilih **Buka di jendela baru**.

**Palet perintah:**

Di [[Palet perintah]], pilih **Buka panel terkini ke jendela baru**.

**Tab:**

Klik kanan pada tab dan pilih **Buka di jendela baru**. Atau, pilih **Opsi lain** ![[lucide-ellipsis-vertical.svg#icon]] di sudut kanan atas editor, lalu pilih **Buka di jendela baru**.

> [!tip] Anda juga dapat klik kanan tautan di editor dan pilih **Buka di jendela baru**.

## Memindahkan file ke jendela lain

**Penjelajah berkas:**

Di [[Penjelajah berkas]], seret file ke grup tab di jendela tujuan.

**Palet perintah:**

Di [[Palet perintah]], pilih **Pindahkan panel terkini ke jendela baru**.

**Tab:**

Seret tab ke grup tab di jendela tujuan. Atau, klik kanan tab dan pilih **Pindahkan ke Jendela Baru**.
