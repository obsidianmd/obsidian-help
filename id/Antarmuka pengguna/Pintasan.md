Pintasan keyboard adalah pintasan papan ketik yang dapat disesuaikan untuk perintah Obsidian. Anda dapat menetapkan kombinasi keyboard apa pun untuk memicu perintah seperti membuka file, menyisipkan templat, atau mengaktifkan plugin.

Pintasan keyboard berbeda dari [[Pintasan penyuntingan|pintasan papan ketik sistem]] (seperti `Ctrl+C` untuk menyalin), yang disediakan oleh sistem operasi Anda dan tidak dapat disesuaikan di Obsidian.

## Melihat pintasan keyboard

Cara tercepat untuk melihat pintasan keyboard yang ditetapkan untuk suatu perintah adalah dengan mencari perintah tersebut di [[Palet perintah]]. Jika ada pintasan keyboard yang tercantum untuk perintah tersebut, berarti sudah ditetapkan.

Anda juga dapat melihat dan mengelola semua pintasan keyboard di **[[Pengaturan]] → Pintasan**.

## Menetapkan pintasan keyboard

Untuk menambah atau mengubah pintasan keyboard:

1. Buka **[[Pengaturan]] → Pintasan**.
2. Cari perintah dalam daftar atau gunakan filter pencarian.
3. Pilih ikon tambah (+) di sebelah perintah.
4. Tekan kombinasi keyboard yang ingin Anda gunakan.
5. Pilih **Simpan** untuk mengonfirmasi.

Anda dapat menetapkan beberapa kombinasi pintasan keyboard ke satu perintah dengan memilih ikon tambah (+) lagi setelah menetapkan pintasan keyboard pertama.

## Menghapus pintasan keyboard

Untuk menghapus pintasan keyboard:

1. Buka **[[Pengaturan]] → Pintasan**.
2. Cari perintah yang memiliki pintasan keyboard yang ingin Anda hapus.
3. Pilih ikon X di sebelah kombinasi keyboard.

## Memfilter pintasan keyboard

Untuk menampilkan hanya perintah yang memiliki pintasan keyboard yang ditetapkan, pilih ikon filter di **[[Pengaturan]] → Pintasan**.

## Tata letak keyboard non-AS

Pintasan keyboard ditampilkan seperti yang akan muncul pada tata letak keyboard AS. Meskipun kombinasi yang ditampilkan tidak sesuai dengan tata letak keyboard Anda, pintasan keyboard akan berfungsi berdasarkan tombol yang sebenarnya Anda tekan (selama Anda tidak mengubah tata letak keyboard Anda).

## Pintasan papan ketik bawaan

Untuk daftar lengkap pintasan papan ketik bawaan untuk penyuntingan dan navigasi, lihat [[Pintasan penyuntingan]].
