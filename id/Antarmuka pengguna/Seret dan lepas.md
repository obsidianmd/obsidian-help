Ada banyak cara untuk menyeret dan melepas elemen di Obsidian. Ini termasuk tab, file, folder, dan konten.

### Menyeret tab

- Anda dapat [[Tab#Atur tab|mengatur tab]] dan membagi grup tab di area konten utama dan di [[Bilah samping|bilah samping]].

### Sumber penyeretan

- Anda dapat menyeret file, atau beberapa file dari [[Penjelajah berkas|penjelajah berkas]].
- Anda dapat menyeret file dari hasil [[Cari|pencarian]].
- Anda dapat menyeret file dari [[Backlink|backlink]] atau referensi yang tidak tertaut.
- Anda dapat menyeret file dari tautan di dalam catatan, dalam mode pratinjau.

### Tujuan pelepasan

- Anda dapat melepas file pada header tab untuk membuka file di sana. Tahan `Alt` (atau `Shift` di macOS) untuk melepas di mana saja dalam tab.
- Anda dapat melepas file pada folder di [[Penjelajah berkas|penjelajah berkas]] untuk memindahkan file ke sana.
- Anda dapat melepas file ke dalam editor untuk menyisipkannya sebagai tautan. Tautan yang dihasilkan juga mengikuti preferensi Anda, seperti jalur relatif, atau menggunakan Tautan Markdown.
- Anda dapat melepas file pada tab [[Penanda]] untuk mem-bookmark-nya.

### Menyeret dari luar Obsidian

- Anda dapat menyeret konten HTML apa pun dari browser ke Obsidian untuk secara otomatis mengonversinya menjadi Markdown
- Anda dapat menyeret dan melepas file apa pun dari penjelajah file bawaan sistem Anda agar Obsidian mengimpor (membuat salinan) file tersebut ke folder lampiran brankas Anda dan menyisipkannya sebagai tautan internal. Tahan `Ctrl` di Windows/Linux atau `Option` di macOS untuk membuat tautan absolut `file:///` ke file tersebut alih-alih mengimpor salinannya.

### Melepas file di luar Obsidian

- Ini membuat URL `obsidian://` di aplikasi eksternal untuk catatan Obsidian yang Anda seret.
