### Skema warna

Obsidian hadir dengan dua skema warna, Terang dan Gelap. Secara bawaan, skema warna diatur untuk menyesuaikan dengan sistem operasi Anda. Anda dapat mengubahnya dengan membuka **[[Pengaturan]] → Tampilan**.

### Tema kustom

Anda dapat mengubah tampilan Obsidian dengan menggunakan [[Tema|tema kustom]]. Telusuri ratusan tema buatan komunitas dengan membuka **[[Pengaturan]] → Tampilan**, lalu klik **Kelola** untuk melihat dan memasang tema.

Anda juga dapat menyesuaikan tema menggunakan [[Cuplikan CSS]] dan [membuat tema Anda sendiri](https://docs.obsidian.md/Themes/App+themes/Build+a+theme).

### Ikon aplikasi kustom

Anda dapat mengubah ikon aplikasi desktop Obsidian. Coba editor ikon yang ditampilkan di [blog kami](https://obsidian.md/blog/new-obsidian-icon/) untuk menyesuaikan warna ikon sesuai keinginan Anda.

Untuk mengubah ikon aplikasi Obsidian, ikuti langkah-langkah berikut:

1. Buka **[[Pengaturan]]**.
2. Pilih **Tampilan** di sebelah kiri.
3. Gulir ke bawah ke bagian **Lanjutan**.
4. Di bawah **Ikon aplikasi kustom** klik **Pilih**.
5. Pilih ikon yang diinginkan dari sistem file Anda. Format file yang diterima adalah `.icns`, `.ico`, `.png`, dan `.svg`.
6. Klik **Mulai ulang**.
7. Setelah Obsidian dimulai ulang, ikon baru Anda akan muncul di bilah tugas pada desktop dan di layar pada perangkat seluler.

> [!tip] Ikon aplikasi kustom di bilah tugas Windows
> Untuk sistem operasi Windows tertentu, langkah tambahan mungkin diperlukan untuk menyesuaikan ikon aplikasi kustom di bilah tugas.
> 
> Ikuti langkah-langkah berikut:
> - Klik kanan ikon Obsidian di bilah tugas atau pintasan desktop Anda.
> - Klik kanan nama Obsidian (lewati langkah ini untuk ikon desktop).
> - Pilih **Properties**.
> - Klik **Change Icon**.
> - Temukan ikon di sistem file Anda.
> - Klik **OK**.
> 
> Setelah itu, Anda memiliki opsi untuk memulai ulang `explorer.exe` melalui task manager atau memulai ulang komputer Anda untuk menerapkan perubahan yang telah dibuat.

### Transparansi

Di MacOS Anda dapat membuat jendela Obsidian menjadi transparan dengan mengaktifkan opsi ini di tab **Tampilan** pada pengaturan.

> [!info]- Windows dan transparansi
> Tim Obsidian menghapus transparansi di Windows untuk versi 1.15.11 karena penghapusan fitur tersebut oleh Electron.
