[[Pengantar Obsidian Publish|Obsidian Publish]] memungkinkan Anda menyimpan catatan dan berbagai jenis media, termasuk gambar dan klip video, dengan batas 4 GB per situs. Untuk detail lebih lanjut, silakan lihat [[Format file yang diterima|format file yang diterima]] kami.

![[Batasan Publish#^publish-media-limit]]

Halaman ini menyediakan tips untuk membantu Anda mengoptimalkan file media yang disimpan di Publish.

## Gambar

Saat mengunggah gambar langsung ke Publish, sebaiknya gunakan format `.svg`, `.webp`, dan `.png`. Kami juga merekomendasikan untuk mengoptimalkan semua gambar guna mengurangi ukuran file sambil mempertahankan kualitas visual.

![[Panduan gaya#Optimisasi]]

Jika Anda menggunakan layanan eksternal untuk menyimpan gambar, kami tetap menyarankan untuk menyesuaikan ukuran gambar sesuai kebutuhan berdasarkan [[Sematkan file#Sematkan gambar di dalam catatan|instruksi penyematan]] kami.

## PDF

PDF memiliki berbagai ukuran dan kompleksitas. Untuk mengoptimalkan performa PDF yang disematkan di Publish, kami menyarankan untuk [mengompres file](https://www.adobe.com/acrobat/hub/how-to-make-pdfs-smaller.html) dan menghapus gambar yang tidak diperlukan. Sebagian besar editor PDF menyediakan kemampuan ini.

## Video dan audio

Publish tidak dioptimalkan untuk menyimpan video, GIF, atau file audio berukuran besar. Menyematkan video yang disimpan dan ditautkan secara internal tidak disarankan jika Anda berencana menyimpan banyak file jenis ini.

Untuk menyimpan file dalam jumlah terbatas, kami menyarankan menggunakan format berikut:

**Video**
- `.mp4`
- `.webm`
- `.hevc` atau `.av1` _(Catatan: Belum didukung secara luas)_

Audio
- `.aac`
- `.mp3`
- `.opus`
