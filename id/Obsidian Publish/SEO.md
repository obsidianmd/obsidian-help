Situs [[Pengantar Obsidian Publish|Obsidian Publish]] secara otomatis dioptimalkan untuk mesin pencari dan [[Pratinjau tautan media sosial|kartu berbagi sosial]]. Anda juga dapat menyesuaikan metadata halaman dengan deskripsi, permalink, dan gambar.

Perlu diketahui bahwa Anda dapat melarang pengindeksan mesin pencari, di [[Kelola situs#Opsi situs|opsi situs]] Anda.

### Google Search Console

Untuk melacak SEO dengan Google, daftarkan situs Anda di [Google Search Console](https://search.google.com/search-console). Anda juga dapat mengikuti langkah yang kami rekomendasikan untuk [[Analitik|menyiapkan Google Analytics]].

### Peta Situs

Peta situs untuk situs Publish Anda tersedia di `/sitemap.xml`, contohnya:
https://help.obsidian.md/sitemap.xml

Peta situs juga tersedia sebagai umpan RSS di `/rss.xml`:
https://help.obsidian.md/rss.xml

### Permalink

Pertimbangkan untuk menggunakan [[Permalink]] untuk menentukan URL permanen di situs Anda.

### Metadata

Metadata halaman dapat dikustomisasi menggunakan [[Properti#Properti untuk Obsidian Publish|Properti]], yang berguna untuk [[Pratinjau tautan media sosial|kartu berbagi sosial]].

| Properti      | Deskripsi                                                                                                         |
| ------------- | ----------------------------------------------------------------------------------------------------------------- |
| `publish`     | Lihat [[Publikasikan konten Anda#Secara otomatis memilih data untuk dipublikasikan\|Secara otomatis memilih data untuk dipublikasikan]]. |
| `permalink`   | Lihat [[Permalink\|Permalink]].                                                                                   |
| `description` | Lihat [[Pratinjau tautan media sosial#Deskripsi\|Deskripsi]].                                                     |
| `image`       | Lihat [[Pratinjau tautan media sosial#Gambar\|Gambar]].                                                           |
| `cover`       | Lihat [[Pratinjau tautan media sosial#Gambar\|Gambar]].                                                           |
