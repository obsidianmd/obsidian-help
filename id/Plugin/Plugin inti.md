Halaman ini mencantumkan plugin inti yang sudah terpasang bersama Obsidian. Plugin inti dibangun dan didukung secara resmi oleh tim Obsidian, dan sudah termasuk di dalam aplikasi.

Beberapa plugin inti dinonaktifkan secara bawaan. Anda dapat mengaktifkannya melalui **[[Pengaturan]] → Plugin inti**.

Tim Obsidian juga memelihara pengembangan beberapa [[#Plugin lainnya|plugin komunitas sumber terbuka]] tertentu.

> [!tip] Plugin komunitas
> Jika Anda tidak menemukan plugin yang dicari di sini, lihat beberapa [[Plugin komunitas|plugin yang dibangun oleh komunitas]]. Anda juga dapat [Membuat plugin](https://docs.obsidian.md/Plugins/Getting+started/Build+a+plugin) sendiri.

### Semua plugin inti

- [[Backlink]]
	- Lihat semua tautan dan sebutan yang belum tertaut dari sebuah catatan.
- [[Pengenalan Basis|Basis]]
	- Buat tampilan kustom yang memungkinkan Anda mengubah, mengurutkan, dan memfilter file menggunakan propertinya.
- [[Canvas]]
	- Atur catatan secara visual dengan ruang tak terbatas untuk menata ide.
- [[Cari]]
	- Temukan file di brankas Anda.
- [[Catatan acak]]
	- Membuka catatan acak di brankas Anda.
- [[Catatan harian]]
	- Buat dan buka catatan berdasarkan tanggal saat ini.
- [[Importir format Markdown]]
	- Konversi format Markdown dari aplikasi lain ke format Obsidian.
- [[Kerangka]]
	- Tampilkan daftar isi untuk catatan yang aktif.
- [[Komposer catatan]]
	- Gabungkan dua catatan atau pisahkan satu menjadi dua.
- [[Palet perintah]]
	- Akses perintah dengan cepat dari keyboard Anda.
- [[Panel tag|Panel tag]]
	- Daftar semua tag di brankas Anda.
- [[Pembuat catatan unik]]
	- Buat catatan unik menggunakan judul berkode waktu.
- [[Pemulih berkas]]
	- Pulihkan pekerjaan Anda dari snapshot reguler.
- [[Penampil web]]
	- Buka tautan eksternal di Obsidian.
- [[Penanda]]
	- Simpan tautan ke catatan, judul, pencarian, dan lainnya.
- [[Penghitung kata]]
	- Tampilkan jumlah kata dan karakter.
- [[Penjelajah berkas]]
	- Telusuri file dan folder di dalam brankas Anda.
- [[Peralih cepat]]
	- Cari, buat, dan buka catatan dari keyboard Anda.
- [[Perekam suara]]
	- Rekam dan simpan rekaman audio langsung di dalam catatan.
- [[Perintah garis miring]]
	- Jalankan perintah di dalam editor menggunakan tombol `/`.
- [[Pratinjau halaman]]
	- Pratinjau isi catatan dengan mengarahkan kursor ke tautan.
- [[Pengantar Obsidian Publish|Publish]]
	- Host catatan Anda sebagai situs web, wiki, atau dokumentasi.
- [[Ruang kerja]]
	- Simpan tata letak dan beralih di antaranya.
- [[Slide]]
	- Buat presentasi dari catatan Anda.
- [[Pengantar Obsidian Sync|Sync]]
	- Sinkronisasi catatan Anda antar perangkat.
- [[Tampilan grafik]]
	- Visualisasikan hubungan antar catatan di dalam brankas Anda.
- [[Tampilan properti]]
	- Daftar semua properti di brankas Anda, dan lihat properti untuk catatan yang aktif.
- [[Tautan Keluar]]
	- Tampilkan semua tautan untuk catatan yang aktif.
- [[Plugin/Template|Template]]
	- Sisipkan konten yang sudah ditentukan sebelumnya ke dalam catatan Anda.

### Plugin lainnya

Tim Obsidian juga memelihara plugin yang tersedia melalui toko Plugin komunitas:

- [[Importer]]
	- Konversi file dari berbagai aplikasi dan format ke Obsidian.
- [[Tampilan peta|Maps]]
	- Tambahkan tampilan peta ke Obsidian Basis.
