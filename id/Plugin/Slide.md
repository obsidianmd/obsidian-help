Slide adalah [[Plugin inti|plugin inti]] yang memungkinkan Anda membuat presentasi dari catatan Anda. Untuk memulai presentasi:

**Menu file:**

- Klik kanan tab catatan dan klik **Mulai presentasi**.

**Palet perintah:**

1. Tekan `Ctrl+P` (atau `Cmd+P` di macOS) untuk membuka [[Palet perintah]].
2. Cari perintah **Mulai presentasi**.
3. Tekan `Enter` dengan perintah yang dipilih untuk memulai presentasi menggunakan catatan yang aktif.

Untuk berpindah antar slide, klik panah kiri dan kanan di pojok kanan bawah, atau tekan tombol panah kiri dan kanan pada keyboard Anda.

Anda juga dapat melanjutkan ke slide berikutnya dengan menekan Spasi.

Untuk menghentikan presentasi, tekan `Escape` atau klik tanda silang di pojok kanan atas presentasi.

Anda dapat menggunakan file Markdown yang valid sebagai presentasi. Untuk memisahkan slide, masukkan `---` di awal baris yang diapit oleh baris kosong.

```md
# Presentasi menggunakan Slide

Demo cara membuat presentasi menggunakan Slide.

---

## Pemformatan

Anda dapat menggunakan pemformatan Markdown biasa, seperti teks *miring* dan **tebal**.

---

## Slide

Gunakan `---` untuk memisahkan slide.
```
