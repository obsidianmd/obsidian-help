Penanda adalah [[Plugin inti|plugin inti]] yang memungkinkan Anda mengakses item yang sering digunakan dengan cepat. Penanda adalah "pintasan" yang langsung membawa Anda ke item yang dibookmark.

Anda dapat menambahkan penanda ke hal-hal berikut:

- File
- Folder
- Grafik
- Pencarian
- Judul
- Blok
- Tautan

Buka penanda dengan memilihnya di tab Penanda.

Anda dapat mengurutkan ulang penanda dengan memilih dan menyeretnya.

Anda dapat membookmark item dan mengorganisirnya ke dalam _grup penanda_ untuk menemukannya dengan lebih mudah. Anda dapat membuka dan menutup grup penanda untuk menampilkan atau menyembunyikan penanda di dalamnya.

> [!todo]- Tampilkan penanda
> Anda dapat menemukan item yang dibookmark dengan mengklik **Penanda** ![[lucide-bookmark.svg#icon]] di bilah sisi kiri.
>
> Atau, buka tab Penanda dengan menjalankan **Bookmarks: Show bookmarks** dari [[Palet perintah]].

## Menambahkan penanda

Menambahkan penanda akan membuka dialog **Tambah penanda** di mana Anda dapat mengatur judul opsional dan menetapkannya ke grup penanda.

Ketika Anda membookmark sebuah item, **Tambah penanda** berubah menjadi **Ubah penanda** dan sebagai gantinya memungkinkan Anda mengedit penanda tersebut.

> [!todo]- Bookmark sebuah file
>
> **Penanda**:
>
> - Pilih **Bookmark the active tab** ![[lucide-bookmark-plus.svg#icon]] di bagian atas tab Penanda.
>
> **Penjelajah berkas**:
>
> - Klik kanan sebuah file di [[Penjelajah berkas]] dan klik **Bookmark**.
>
> **Palet perintah**:
>
> 1. Tekan `Ctrl+P` (atau `Cmd+P` di macOS) untuk membuka [[Palet perintah]].
> 2. Cari perintah **Bookmark**.
> 3. Tekan `Enter` dengan perintah yang dipilih untuk menambahkan penanda.

> [!todo]- Bookmark sebuah folder
>
> **Penjelajah berkas**:
>
> - Klik kanan sebuah folder di [[Penjelajah berkas]] dan klik **Bookmark**.

> [!todo]- Bookmark istilah pencarian
>
> **Pencarian**:
>
> 1. Masukkan [[Cari#Istilah pencarian|istilah pencarian]].
> 2. Di bawah kolom pencarian, pilih ikon tiga titik di samping jumlah hasil.
> 3. Pilih **Bookmark**.

> [!todo]- Bookmark sebuah grafik
> Untuk membookmark sebuah grafik, klik kanan tab dengan tampilan grafik dan pilih **Bookmark**.
>
> Anda tidak dapat membookmark grafik lokal.

> [!todo]- Bookmark sebuah judul
> Untuk membookmark sebuah judul, klik kanan judul tersebut dan pilih **Bookmark this heading**.
>
> **Palet perintah**:
>
> 1. Tekan `Ctrl+P` (atau `Cmd+P` di macOS) untuk membuka [[Palet perintah]].
> 2. Pilih perintah **Bookmark heading under cursor**.

> [!todo]- Bookmark sebuah blok
>
> **Palet perintah**:
>
> 1. Tekan `Ctrl+P` (atau `Cmd+P` di macOS) untuk membuka [[Palet perintah]].
> 2. Pilih perintah **Bookmark block under cursor**.

> [!todo]- Bookmark sebuah tautan
>
> 1. Aktifkan [[Penampil web]] di [[Plugin inti]].
> 2. Buka [[Penampil web]] dan navigasikan ke tautan yang ingin Anda bookmark.
> 3. Di bilah alamat, pilih ikon tiga titik di samping ikon pembaca.
> 4. Pilih **Bookmark**.
>
> **Palet perintah**:
>
> 1. Tekan `Ctrl+P` (atau `Cmd+P` di macOS) dalam panel [[Penampil web]] untuk membuka [[Palet perintah]].
> 2. Pilih perintah **Bookmark**.

## Menghapus penanda

Di tab Penanda, klik kanan penanda dan pilih **Hapus**.

Anda juga dapat menghapus penanda dari dialog **Ubah penanda**.

## Menambahkan grup penanda

**Penanda**:

- Pilih **New bookmark group** ![[lucide-folder-plus.svg#icon]] di bagian atas tab Penanda.

Untuk memindahkan penanda ke grup lain, pilih dan seret ke grup yang Anda inginkan.

## Menghapus grup penanda

Untuk menghapus grup penanda dan semua penanda di dalamnya, klik kanan grup penanda dan pilih **Hapus**.

## Bookmark beberapa file

Anda dapat menambahkan beberapa penanda sekaligus.

**Penjelajah berkas**:

1. Pilih beberapa file dengan menekan `Alt` dan memilih setiap file, atau tekan `Shift` untuk memilih rentang file.
2. Klik kanan salah satu file yang dipilih dan kemudian pilih **Bookmark**.

**Opsi grup tab**:

1. Di sudut kanan atas grup tab, pilih panah bawah.
2. Pilih **Bookmark 3 tabs**, di mana 3 adalah jumlah tab dalam grup tab tersebut.
