Pratinjau halaman adalah [[Plugin inti|plugin inti]] yang memungkinkan Anda melihat pratinjau halaman saat Anda mengarahkan kursor ke tautan internal, tanpa perlu menavigasi ke halaman tersebut.

Secara bawaan, mengarahkan kursor ke tautan akan menampilkan pratinjau file tersebut di [[Penjelajah berkas]], [[Cari]], [[Backlink]], dan lainnya. Untuk melihat pratinjau halaman saat dalam tampilan Pengubah, tekan `Ctrl` (atau `Cmd` di macOS) sambil mengarahkan kursor ke tautan.

Terdapat juga opsi untuk mewajibkan `Ctrl` (atau `Cmd` di macOS) saat mengarahkan kursor untuk melihat file. Pengaturan untuk opsi ini berada di bawah Plugin inti > Pratinjau halaman.

> [!hint]
> Pratinjau **aktif secara bawaan**. Mengubah opsi plugin akan **menonaktifkan perilaku tersebut**.
