**Vault sandbox Obsidian** adalah fitur yang memungkinkan Anda menjelajahi berbagai fungsionalitas tanpa memengaruhi data yang sudah ada. Vault ini berguna baik sebagai alat pembelajaran maupun untuk debug masalah. Vault ini membantu menentukan apakah masalah disebabkan oleh plugin, tema, atau aplikasi itu sendiri.

> [!note] Vault sandbox tidak tersedia di perangkat seluler. Namun, Anda dapat mengunduh salinannya dari repositori [Obsidian Help](https://github.com/obsidianmd/obsidian-help) untuk digunakan di perangkat seluler.


## Membuka vault sandbox

Anda dapat membuka vault sandbox dengan beberapa cara. Dalam semua kasus, vault ini dibuka secara terpisah dari brankas Anda saat ini.

**Bilah Sisi Kiri**

1. Buka **bilah sisi kiri** ![[obsidian-icon-side-bar-left.svg#icon]].
2. Di samping **peralih brankas**, tekan ikon **Bantuan** ![[obsidian-icon-help.svg#icon]].
3. Pilih **Buka** di samping opsi **Vault Sandbox**.

**Palet Perintah**

1. Aktifkan plugin inti **palet perintah**.
2. Buka **palet perintah**.
3. Cari **Open sandbox vault**.
4. Tekan **Enter** atau pilih item tersebut untuk membuka vault sandbox.
## Menutup vault sandbox

Untuk menutup vault sandbox, tutup jendela brankas **Obsidian Sandbox**.
