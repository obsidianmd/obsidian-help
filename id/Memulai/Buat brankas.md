Brankas adalah folder di sistem file lokal tempat Obsidian menyimpan catatan Anda. Anda dapat menyimpan semua catatan dalam satu brankas, atau membuat beberapa brankas untuk setiap proyek yang berbeda.

Saat pertama kali membuka Obsidian, Anda akan diminta untuk menambahkan brankas baru. Anda memiliki dua pilihan, yaitu membuat brankas kosong baru, atau menggunakan folder yang sudah ada.

## Buat brankas kosong

Untuk membuat brankas kosong baru:

1. Di sebelah kanan **Buat vault baru**, klik **Buat**.
2. Di **Nama vault**, masukkan nama brankas Anda.
3. Klik **Telusuri** untuk memilih lokasi brankas baru Anda akan dibuat.
4. Klik **Buat**.

## Buka folder yang sudah ada

Jika Anda sudah memiliki folder yang ingin digunakan sebagai brankas:

1. Di sebelah kanan **Buka folder sebagai vault**, klik **Buka**.
2. Di penjelajah file, pilih folder yang ingin Anda gunakan sebagai brankas.
3. Klik **Buka**.

Jika Anda ingin mengetahui lebih lanjut tentang cara kerja brankas, pelajari bagaimana [[Cara Obsidian menyimpan data]].

Sekarang setelah brankas Anda sudah disiapkan, Anda siap untuk [[Buat catatan pertama Anda]].
