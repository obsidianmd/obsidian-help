Aplikasi datang dan pergi, tetapi data Anda harus bertahan. Obsidian menggunakan file Markdown teks biasa non-proprietary yang disimpan secara lokal di perangkat Anda. Anda memiliki kendali penuh atas data Anda, yang berarti Anda dapat menggunakan Obsidian secara offline dan beralih ke aplikasi lain dengan mudah jika diperlukan.
