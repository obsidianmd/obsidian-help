Obsidian menggunakan file Markdown `.md` sebagai format utama untuk catatan. Ini memudahkan Anda untuk mengimpor file Markdown ke dalam Obsidian. Banyak aplikasi yang dapat menyunting file Markdown atau mengekspor ke format ini.

Ada beberapa cara untuk menambahkan file Markdown ke brankas Obsidian Anda:

1. Seret dan lepas file atau folder ke jendela [[Penjelajah berkas]]
2. Gunakan penjelajah file sistem Anda (misalnya Windows Explorer atau Finder di macOS), dan pindahkan file Anda langsung ke folder brankas Obsidian.

### Mengonversi dari varian Markdown lainnya

Obsidian menggunakan [[Obsidian Flavored Markdown]]. Beberapa aplikasi menggunakan sintaksis Markdown yang sedikit berbeda. Anda dapat menggunakan [[Importir format Markdown]] untuk mengonversi file Anda ke sintaksis Markdown Obsidian.
