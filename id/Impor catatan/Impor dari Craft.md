Obsidian memungkinkan Anda untuk memigrasikan catatan dari Craft. Proses ini akan mengonversi data Craft Anda menjadi file Markdown yang tahan lama, yang dapat Anda gunakan dengan Obsidian dan banyak aplikasi lainnya.

## Ekspor data Anda dari Craft

Buka aplikasi Craft di komputer desktop Anda:

1. Buka tampilan "Organize", atau "All Documents"
2. Klik menu tiga titik
3. Pilih opsi Ekspor.
4. Pilih Markdown.

Tergantung pada jumlah catatan yang Anda miliki di Craft, proses ini mungkin memerlukan waktu cukup lama.

## Impor data Craft Anda ke Obsidian

Dari sini Anda dapat mengikuti instruksi [[Impor file Markdown]].

## Pemecahan masalah

Craft Docs menggunakan sintaksis non-standar untuk tautan tertentu. Anda dapat menemukan sumber daya untuk memigrasikan tautan tersebut di [repositori GitHub Importer](https://github.com/obsidianmd/obsidian-importer/issues/27).
