Pelajari cara memperluas Obsidian dengan plugin yang dibuat oleh komunitas. Gunakan plugin untuk menyesuaikan Obsidian agar sesuai dengan kebutuhan spesifik Anda, seperti dukungan untuk format file tambahan atau integrasi dengan layanan pihak ketiga.

> [!warning] Peringatan
> Plugin komunitas menjalankan kode pihak ketiga atas nama Anda yang berpotensi menimbulkan kerugian. Untuk mempelajari lebih lanjut tentang apa yang dilakukan tim Obsidian untuk mencegah plugin berbahaya, lihat [[Keamanan plugin]].

## Menelusuri plugin komunitas

1. Buka **[[Pengaturan]]**.
2. Pilih **Aktifkan plugin komunitas**. Untuk informasi lebih lanjut, lihat [[Keamanan plugin#Mode terbatas|Mode Terbatas]].
3. Pilih **Telusuri** untuk menampilkan daftar semua plugin komunitas yang tersedia.

Gunakan kotak teks untuk memfilter plugin berdasarkan nama, pembuat, dan deskripsi.

Anda juga dapat menelusuri plugin yang tersedia di peramban Anda, dengan mengunjungi [obsidian.md/plugins](https://obsidian.md/plugins).

## Memasang plugin komunitas

Untuk memasang plugin komunitas, Anda harus terlebih dahulu menonaktifkan [[Keamanan plugin#Mode terbatas|Mode Terbatas]].

1. Buka **[[Pengaturan]]**.
2. Di menu samping, pilih **Plugin komunitas**.
3. Pilih **Telusuri** untuk menjelajahi plugin komunitas yang tersedia.
4. Pilih plugin yang ingin Anda pasang.
5. Pilih **Pasang**.

Untuk menggunakan plugin yang sudah terpasang, Anda perlu mengaktifkannya.

## Mengaktifkan plugin komunitas

Pilih **Aktifkan** langsung setelah memasangnya, atau aktifkan dari daftar Plugin komunitas di bawah **[[Pengaturan]] → Plugin komunitas → Plugin terpasang**.

## Memperbarui plugin

Untuk tujuan keamanan, plugin komunitas tidak diperbarui secara otomatis. Anda dapat memperbarui semua plugin sekaligus, atau memperbarui plugin satu per satu.

Untuk memperbarui semua plugin Anda:

1. Buka **[[Pengaturan]]**.
2. Di bawah **Plugin komunitas → Plugin saat ini**, pilih **Cek pembaruan**.
3. Jika pembaruan tersedia, pilih **Perbarui semua**.

Untuk memperbarui plugin tertentu:

1. Buka **[[Pengaturan]]**.
2. Di bawah **Plugin komunitas → Plugin saat ini**, pilih **Cek pembaruan**.
3. Di bawah **Plugin terpasang**, pilih **Perbarui** di sebelah plugin yang ingin Anda perbarui.

## Menghapus pemasangan plugin komunitas

1. Buka **[[Pengaturan]]**.
2. Di menu samping, pilih **Plugin komunitas**.
3. Di bawah **Plugin terpasang**, pilih ikon tempat sampah ![[lucide-trash-2.svg#icon]] di sebelah plugin yang ingin Anda hapus pemasangannya.
4. Pilih **Hapus pemasangan** untuk mengonfirmasi.

## Mengelola plugin terpasang

Di bawah **[[Pengaturan]] → Plugin komunitas → Plugin terpasang**, setiap plugin memiliki beberapa ikon untuk mengelolanya:

- **[[Pengaturan]]** ![[lucide-settings.svg#icon]] — Buka halaman pengaturan plugin jika memiliki opsi yang dapat dikonfigurasi.
- **[[Pintasan]]** ![[lucide-plus-circle.svg#icon]] — Atur pintasan keyboard untuk perintah plugin.
- **Pendanaan** ![[lucide-heart.svg#icon]] — Lihat URL pendanaan pembuat plugin untuk mendukung pekerjaan mereka.
- **Hapus pemasangan** ![[lucide-trash-2.svg#icon]] — Hapus plugin dari brankas Anda.
- **Toggle** — Aktifkan atau matikan plugin tanpa menghapus pemasangannya.

Anda juga dapat:

- Pilih ikon **muat ulang** untuk memuat ulang semua plugin.
- Pilih ikon **folder** ![[lucide-folder-open.svg#icon]] untuk membuka folder plugin di [[Folder konfigurasi|folder konfigurasi]] Anda.
- Gunakan **bilah pencarian** untuk memfilter plugin berdasarkan nama.
