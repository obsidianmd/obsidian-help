Basis adalah [[Plugin inti|plugin inti]] yang memungkinkan Anda membuat tampilan mirip basis data dari catatan Anda. Menggunakan basis, Anda dapat melihat, mengubah, mengurutkan, dan menyaring file beserta [[Properti|properti]]-nya. Basis dapat membantu Anda mengorganisasi segala hal mulai dari proyek hingga rencana perjalanan, daftar bacaan, dan lainnya.

Setiap basis dapat memiliki beberapa [[Tampilan|tampilan]] dengan tata letak berbeda seperti tabel dan kartu. Di bawah ini adalah contoh tampilan tabel di mana setiap baris adalah file, dan setiap kolom adalah properti dari file tersebut.

![Contoh basis menampilkan tampilan tabel dengan daftar buku](bases-noshadow.png#interface)

Semua data dalam Obsidian Basis disimpan dalam file [[Sintaksis format dasar|Markdown]] lokal Anda beserta [[Properti|properti]]-nya. Tampilan dijelaskan oleh [[Sintaksis Basis]], yang dapat disimpan sebagai file `.base` atau [[Buat basis#Sematkan basis|disematkan]] dalam blok kode di dalam file Markdown Anda.

## Cara menggunakan basis

- [[Buat basis]]
	- Pelajari cara membuat dan menyematkan basis.
- [[Tampilan]]
	- Jelajahi tata letak, filter, urutkan, dan kelompokkan properti.
- [[Rumus]]
	- Buat perhitungan menggunakan data dari properti lain.
- [[Fungsi]]
	- Fungsi yang dapat Anda gunakan dalam rumus dan filter.
- [[Sintaksis Basis]]
	- Bagaimana file basis diformat.

## Jenis tampilan

Tampilan dapat ditampilkan dengan tata letak yang berbeda. Tata letak tambahan dapat ditambahkan oleh [[Plugin komunitas]].

- [[Tampilan tabel|Tabel]]
	- Menampilkan file sebagai baris dengan kolom yang menunjukkan properti dari file tersebut.
- [[Tampilan daftar|Daftar]]
	- Menampilkan file sebagai daftar dengan penanda berpoin atau bernomor.
- [[Tampilan kartu|Kartu]]
	- Menampilkan file sebagai kotak grid. Memungkinkan Anda membuat tampilan mirip galeri dengan gambar.
- [[Tampilan peta|Peta]]
	- Menampilkan file sebagai pin pada peta interaktif.
