Obsidian mendukung jenis file berikut:

- **Markdown**: `.md`  
- **Basis:** `.base`
- **JSON Canvas**: `.canvas` ([Pelajari lebih lanjut](https://jsoncanvas.org/))  
- **Gambar**: `.avif`, `.bmp`, `.gif`, `.jpeg`, `.jpg`, `.png`, `.svg`, `.webp`  
- **Audio**: `.flac`, `.m4a`, `.mp3`, `.ogg`, `.wav`, `.webm`, `.3gp`  
- **Video**: `.mkv`, `.mov`, `.mp4`, `.ogv`, `.webm`  
- **PDF**: `.pdf`  

Anda dapat memperluas Obsidian untuk mendukung format file lain menggunakan [[Plugin komunitas]].

### Menyematkan file

Banyak jenis file — termasuk gambar, audio, video, dan PDF — dapat disematkan langsung ke dalam catatan Anda.

- Di **Desktop**, seret dan lepas file yang didukung ke dalam catatan untuk menyematkannya secara otomatis.  
- Di **Seluler**, gunakan opsi lampiran untuk menyisipkan atau menautkan file.

Lihat [[Sematkan file]] untuk opsi dan sintaksis penyematan lainnya.

> [!NOTE]
> Dukungan untuk audio, video, dan file lainnya bergantung pada kodek yang tersedia di perangkat Anda.
