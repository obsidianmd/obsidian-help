Folder konfigurasi Obsidian berisi semua file pengaturan yang berkaitan dengan [[Kelola brankas|brankas]] Anda.

Secara bawaan, folder konfigurasi Anda diberi nama `.obsidian` dan ditempatkan di folder vault. Saat menggunakan layanan sinkronisasi, atau jika Anda ingin menguji profil yang berbeda di brankas yang sama, Anda mungkin ingin mengubah folder konfigurasi Anda.

## Mengakses folder konfigurasi Anda

### Desktop

Di desktop, folder `.obsidian` terletak di folder vault Anda. Anda dapat mengaksesnya melalui pengelola file sistem Anda.

### Seluler

Cara mengakses folder konfigurasi berbeda berdasarkan platform:

**iOS dan iPadOS**

Tidak ada cara bawaan untuk mengakses folder `.obsidian` di iOS atau iPadOS. Untuk melihat dan mengedit file dan folder tersembunyi, Anda memerlukan aplikasi pihak ketiga seperti:

- **Taio** — Aplikasi gratis dengan fitur dasar untuk melihat dan mengedit nama folder dan file.
- **Textastic** — Aplikasi berbayar dengan kemampuan pengeditan file tingkat lanjut.

**Android**

Untuk mengakses folder `.obsidian` di Android:

1. Buka pengelola file sistem Anda.
2. Aktifkan toggle "Tampilkan file tersembunyi" di pengaturan.
3. Navigasi ke folder vault Anda. Folder `.obsidian` akan muncul di tingkat teratas.

> [!tip]- Pengelola file alternatif
> Jika pengelola file bawaan Anda tidak memiliki opsi "Tampilkan file tersembunyi", pasang aplikasi pengelola file yang menyertakan fitur ini.

## Mengubah folder konfigurasi Anda

Untuk mengatur folder konfigurasi Anda:

1. Buka **[[Pengaturan]] → Berkas & Tautan**.
2. Di **Timpa folder konfigurasi**, ketik nama profil Anda, dimulai dengan tanda titik (`.`). Misalnya, `.obsidian-awesome`.
3. Mulai ulang Obsidian agar perubahan diterapkan.

> [!info] Pengaturan apa pun dalam folder konfigurasi Anda tidak akan ditransfer ke folder konfigurasi baru Anda. Namun, folder konfigurasi sebelumnya akan tetap berada di dalam folder vault Anda.
