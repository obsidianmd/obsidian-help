**Brankas** adalah folder di sistem file Anda yang berisi catatan, [[Lampiran|lampiran]], dan [[Folder konfigurasi|folder konfigurasi]] dengan pengaturan khusus Obsidian. Untuk informasi lebih lanjut tentang brankas, lihat [[Cara Obsidian menyimpan data]].

Anda dapat mengelola brankas menggunakan **Peralih Brankas**. *Profil Brankas* terbuka saat pertama kali Anda membuka Obsidian.

Untuk membuka peralih brankas dari brankas yang sudah ada, pilih **Profil Brankas** ![[lucide-chevrons-up-down.svg#icon]] di bagian bawah [[Bilah samping|bilah sisi kiri]]. Atau, pilih **Buka vault lain** dari [[Palet perintah|palet perintah]].

## Buat brankas baru

1. Di komputer Anda, buka Obsidian.
2. Di kiri bawah, pilih **Profil Brankas** ![[lucide-chevrons-up-down.svg#icon]].
3. Menu konteks akan muncul. Pilih **Kelola brankas...**.
3. Di sebelah kanan **Buat vault baru**, pilih **Buat**.
4. Di **Nama vault**, masukkan nama brankas Anda.
5. Klik **Telusuri** untuk memilih lokasi brankas baru Anda akan dibuat.
6. Klik **Buat**.

## Buat brankas dari folder yang sudah ada

1. Di komputer Anda, buka Obsidian.
2. Di kiri bawah, pilih **Profil Brankas** ![[lucide-chevrons-up-down.svg#icon]].
3. Menu konteks akan muncul. Pilih **Kelola brankas...**.
3. Di sebelah kanan **Buka folder sebagai vault**, klik **Buka**.
4. Di penjelajah file, pilih folder yang ingin Anda gunakan sebagai brankas.
5. Klik **Buka**.

> [!tip] Buka vault dari Obsidian Sync
> Jika Anda ingin membuka brankas jarak jauh menggunakan Obsidian Sync, lihat [[Menyiapkan Obsidian Sync]].

## Ganti nama brankas

Karena nama brankas dan folder dasarnya sama, mengganti nama brankas juga mengganti nama foldernya.

1. Di komputer Anda, buka Obsidian.
2. Di kiri bawah, pilih **Profil Brankas** ![[lucide-chevrons-up-down.svg#icon]].
3. Menu konteks akan muncul. Pilih **Kelola brankas...**.
4. Di daftar brankas, pilih **Opsi lain** ![[lucide-more-horizontal.svg#icon]] di samping brankas yang ingin Anda ganti namanya.
5. Pilih **Ganti nama vault**.
6. Masukkan nama baru untuk brankas, dan tekan `Enter`.

## Pindahkan brankas ke folder lain

1. Di komputer Anda, buka Obsidian.
2. Di kiri bawah, pilih **Profil Brankas** ![[lucide-chevrons-up-down.svg#icon]].
3. Menu konteks akan muncul. Pilih **Kelola brankas...**.
4. Tutup jendela brankas saat ini, biarkan jendela **Kelola brankas** tetap terbuka.
5. Di daftar brankas, pilih **Opsi lain** ![[lucide-more-horizontal.svg#icon]] di samping brankas yang ingin Anda pindahkan.
6. Pilih **Pindahkan vault**, lalu pilih lokasi baru.

Beberapa sistem operasi tidak mengizinkan brankas dipindahkan dengan Peralih Brankas. Dalam kasus ini, Anda perlu memindahkan brankas secara manual:

1. Tutup Obsidian.
2. Pindahkan folder brankas Anda ke lokasi baru, hindari folder yang disinkronkan oleh layanan lain.
3. Buka kembali Obsidian.
4. Klik ikon **Profil Brankas** di kiri bawah ![[lucide-chevrons-up-down.svg#icon]].
5. Dari menu pop-up, pilih **Kelola brankas...**.
6. Di samping **Buka folder sebagai vault**, klik **Buka**.
7. Navigasikan ke dan pilih folder brankas baru Anda.
8. Klik **Buka**.
9. Verifikasi bahwa isi brankas tidak berubah. Jika perlu, aktifkan kembali plugin komunitas dengan menavigasi ke **[[Pengaturan]] → Plugin Komunitas → Matikan mode terbatas**.

## Hapus brankas

Menghapus brankas hanya menghapusnya dari daftar brankas.

1. Di komputer Anda, buka Obsidian.
2. Di kiri bawah, pilih **Profil Brankas** ![[lucide-chevrons-up-down.svg#icon]].
3. Menu konteks akan muncul. Pilih **Kelola brankas...**.
4. Di daftar brankas, pilih **Opsi lain** ![[lucide-more-horizontal.svg#icon]] di samping brankas yang ingin Anda hapus.
5. Pilih **Hapus dari daftar**.

## Transfer pengaturan ke brankas lain

Untuk menggunakan pengaturan yang sama untuk brankas lain, gunakan pengelola file (atau terminal) pilihan Anda untuk menyalin folder `.obsidian` dari root brankas sumber ke root brankas tujuan.

Anda mungkin perlu memulai ulang Obsidian untuk menerapkan perubahan.

> [!note] Di mana saya menemukan folder `.obsidian`?
> Secara bawaan, sebagian besar sistem operasi menyembunyikan folder yang dimulai dengan titik (`.`). Untuk informasi lebih lanjut tentang folder `.obsidian` dan cara mengaksesnya, lihat [[Cara Obsidian menyimpan data#Pengaturan brankas|pengaturan brankas]] dan [[Folder konfigurasi|folder konfigurasi]].
