Perintah slash adalah [[Plugin inti|plugin inti]] yang memungkinkan Anda menjalankan perintah di editor dengan mengetikkan garis miring (`/`) di awal baris atau setelah spasi kosong.

Untuk menjalankan perintah menggunakan perintah slash:

1. Di editor, ketik garis miring (`/`).
2. Masukkan nama perintah yang ingin Anda jalankan untuk mencari daftar perintah yang tersedia.
3. Navigasikan ke perintah menggunakan tombol panah.
4. Tekan Enter.

Untuk keluar dari pencarian perintah slash tanpa menjalankan perintah, tekan `Esc` atau tombol `Spasi`.

> [!TIP] Pencocokan fuzzy
> Seperti [[Palet perintah]], perintah slash mendukung pencocokan fuzzy yang memungkinkan Anda mencari perintah meskipun Anda tidak mengetahui nama persisnya. Misalnya, ketik "scf" untuk menemukan perintah **Simpan berkas**.
