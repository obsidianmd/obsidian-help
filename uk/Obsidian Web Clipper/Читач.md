[[Вступ до Obsidian Web Clipper|Web Clipper]] включає вбудований режим читання під назвою Читач, який прибирає зайве з вебсторінок, представляючи лише основний вміст статті в чистому, зручному для читання форматі.

## Активація режиму читання

Ви можете активувати Читач кількома способами:

- **Значок книги** у заголовку панелі розширення.
- Сполучення клавіш `Alt+Shift+R`, яке можна налаштувати в параметрах розширення.
- Повторне перемикання режиму читання відновить оригінальну сторінку.

## Як це працює

Читач витягує основний вміст зі сторінки. Оригінальна сторінка замінюється чистим поданням для читання, яке включає:

- Назву статті, автора, дату публікації та домен-джерело.
- Основний вміст статті зі збереженими зображеннями та форматуванням.
- Структуру на бічній панелі для статей з кількома заголовками.
- Підсвічування синтаксису для блоків коду.
- Спливні підказки для виносок, які відображаються в тексті при натисканні.

## Налаштування подання для читання

Режим Читача включає приховану панель налаштувань, яку можна відобразити, навівши курсор на правий верхній кут сторінки, з елементами керування для налаштування зручності читання.

| Налаштування              | Опис                                                                                                |
| ------------------------- | --------------------------------------------------------------------------------------------------- |
| Розмір шрифту             | Налаштування розміру тексту.                                                                        |
| Ширина рядка              | Максимальна ширина області вмісту.                                                                  |
| Висота рядка              | Відстань між рядками.                                                                               |
| Колірна&nbsp;схема        |                                                                                                     |
| Тема                      | Автоматично слідує за системними налаштуваннями. Або оберіть постійне використання світлої чи темної теми. |
