[[Вступ до Obsidian Web Clipper|Web Clipper]] включає вбудований режим читання під назвою **Читач**, який прибирає зайве з вебсторінок, представляючи лише основний вміст статті в чистому, зручному для читання форматі.

## Активація режиму читання

Ви можете активувати Читач кількома способами:

- ![[lucide-book-icon.svg#icon]] **Значок книги** на панелі розширення.
- **Сполучення клавіш**, яке можна налаштувати в параметрах розширення.
- **Контекстне меню** при натисканні правою кнопкою миші на вебсторінці, яку ви переглядаєте.

Ви також можете налаштувати кнопку розширення ![[obsidian.svg#icon]] для відкриття Читача одним натисканням. У налаштуваннях Web Clipper перейдіть до **General** → **Behavior** → **Open behavior** і оберіть **Reader**.

Усі вищезазначені варіанти також дозволяють вийти з Читача та відновити оригінальну сторінку.

## Як це працює

Читач витягує основний вміст зі сторінки. Оригінальна сторінка замінюється чистим поданням для читання, яке включає:

- Назву статті, автора, дату публікації та домен-джерело.
- Основний вміст статті зі збереженими зображеннями та форматуванням.
- Структуру на бічній панелі для статей з кількома заголовками.
- Підсвічування синтаксису для блоків коду.
- Спливні підказки для виносок, які відображаються в тексті при натисканні.

## Налаштування подання для читання

Налаштуйте зручність читання в параметрах розширення. Деякі параметри також доступні безпосередньо в режимі Читача при натисканні на значок ![[lucide-case-sensitive.svg#icon]].

| Налаштування              | Опис                                            |
| ------------------------- | ----------------------------------------------- |
| Шрифт                     | Додайте шрифти, встановлені у вашій системі.    |
| Розмір шрифту             | Налаштування розміру тексту.                    |
| Висота&nbsp;рядка         | Відстань між рядками.                           |
| Ширина&nbsp;рядка         | Максимальна ширина області вмісту.              |
| Зовнішній вигляд          | Оберіть світлий або темний зовнішній вигляд.    |
| Тема                      | Оберіть кольори.                                |
| Власний CSS               | Додатковий CSS, застосований до сторінки.        |
