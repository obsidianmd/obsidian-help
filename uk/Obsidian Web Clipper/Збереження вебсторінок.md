Після встановлення розширення [[Вступ до Obsidian Web Clipper|Web Clipper]] для браузера ви можете отримати до нього доступ кількома способами, залежно від вашого браузера:

1. Значок Obsidian на панелі інструментів браузера.
2. Сполучення клавіш для активації розширення з клавіатури.
3. Контекстне меню, яке відкривається натисканням правої кнопки миші на вебсторінці, яку ви відвідуєте.

Щоб зберегти сторінку в Obsidian, натисніть кнопку **Додати в Obsidian**.

## Збереження сторінки

Коли ви відкриваєте розширення, Web Clipper витягує дані з поточної вебсторінки відповідно до налаштувань у вашому [[Шаблони|шаблоні]]. Ви можете створювати власні шаблони та налаштовувати вивід за допомогою [[Змінні|змінних]] та [[Фільтри|фільтрів]].

За замовчуванням Web Clipper намагається інтелектуально витягти лише основний вміст статті, виключаючи інші елементи на сторінці. Однак ви можете змінити цю поведінку наступними способами:

- Якщо присутній особистий шаблон, використовується ваш шаблон.
- Якщо є виділений текст, використовується виділення. Ви можете натиснути `Ctrl/Cmd+A`, щоб виділити всю сторінку.
- Якщо є будь-які [[Виділення вебсторінок|виділення]], використовуються вони.

## Завантаження зображень

Зображення не завантажуються автоматично при використанні Web Clipper. Натомість зображення посилаються на їх вебадресу. Це економить місце у вашому сховищі, але означає, що зображення не будуть доступні в офлайн-режимі або якщо URL перестане працювати.

Ви можете завантажити зображення для будь-якого файлу в Obsidian за допомогою [[Меню команд|команди]] з назвою **Завантажити вкладення для поточного файлу**. Цю команду також можна призначити на сполучення клавіш в Obsidian.

## Сполучення клавіш

Web Clipper включає комбінації клавіш, які можна використовувати для прискорення робочого процесу. Щоб змінити призначення клавіш, перейдіть до **Налаштування Web Clipper** → **Загальне** та дотримуйтесь інструкцій для вашого браузера. Призначення можна змінити для всіх браузерів, окрім Safari, який не підтримує редагування сполучень клавіш.

| Дія                     | macOS          | Windows/Linux   |
| ----------------------- | -------------- | --------------- |
| Відкрити кліпер         | `Cmd+Shift+O`  | `Ctrl+Shift+O`  |
| Швидке вирізання        | `Opt+Shift+O`  | `Alt+Shift+O`   |
| Перемкнути виділення     | `Opt+Shift+H`  | `Alt+Shift+H`   |

## Функціональність інтерфейсу

Інтерфейс Web Clipper поділений на чотири розділи:

1. **Заголовок** — де ви можете перемикати шаблони, вмикати [[Виділення вебсторінок|виділення]], [[Читач|режим читання]] та отримувати доступ до налаштувань.
2. **Властивості** — показує [[Властивості|метадані]], витягнуті зі сторінки, які будуть збережені як [[Властивості]] в Obsidian.
3. **Вміст нотатки** — який буде збережений в Obsidian.
4. **Нижня панель** — дозволяє вибрати сховище та теку, а також додати в Obsidian.

Функціональність заголовка включає:

- Випадаючий список **Шаблон** для перемикання між збереженими [[Шаблони|шаблонами]], доданими в налаштуваннях Web Clipper.
- Кнопка **Більше (...)** для відображення змінних сторінки, які можна використовувати в шаблонах.
- Кнопка **Виділення** для ввімкнення [[Виділення вебсторінок|виділення]].
- Кнопка **Шестерня** для відкриття налаштувань Web Clipper.

Функціональність нижньої панелі включає:

- Кнопка **Додати в Obsidian** для збереження даних в Obsidian.
- Випадаючий список **Сховище** для перемикання між збереженими сховищами, доданими в налаштуваннях Web Clipper.
- Поле **Тека** для визначення теки, в яку зберігати.
- **Інтерпретатор** для запуску [[Інтерпретація вебсторінок|запитів природною мовою]] на сторінці.
