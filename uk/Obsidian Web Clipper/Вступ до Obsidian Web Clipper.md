Obsidian Web Clipper — це безоплатне розширення для браузера, яке дозволяє виділяти сторінки та зберігати вебвміст до вашого сховища.

## Встановлення Web Clipper

Отримайте розширення Obsidian Web Clipper з офіційного каталогу для вашого браузера:

- [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
	- Для Chrome, Brave, Arc, Orion та інших браузерів на основі Chromium.
- [Firefox Add-Ons](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
	- Для Firefox та Firefox Mobile.
- [Safari Extensions](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
	- Для macOS, iOS та iPadOS.
- [Edge Add-Ons](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
	- Для Microsoft Edge.

## Як використовувати Web Clipper

- [[Збереження вебсторінок]]
	- Почніть роботу та дізнайтеся, як зберігати вміст зі сторінок.
- [[Виділення вебсторінок|Виділення]]
	- Виділяйте важливі фрагменти та обирайте елементи, які хочете зберегти.
- [[Читач|Читач]]
	- Читайте вебсторінки в чистому вигляді без відволікань.
- [[Інтерпретація вебсторінок|Інтерпретатор]]
	- Використовуйте запити природною мовою для захоплення та модифікації даних на сторінці.
- [[Вирішення проблем Web Clipper|Вирішення проблем]]
	- Вирішуйте поширені проблеми з Web Clipper.
- [[Obsidian Web Clipper/Шаблони|Шаблони]]
	- Створюйте власні шаблони для конкретних вебсайтів.
- [[Змінні]]
	- Попередньо заповнюйте дані сторінки у вашому шаблоні.
- [[Фільтри]]
	- Змінюйте вихідні дані змінних шаблону.
- [[Логіка]]
	- Використовуйте умовні конструкції, цикли та встановлюйте змінні в шаблонах.

## Конфіденційність

Obsidian Web Clipper зберігає вміст локально у вашому сховищі Obsidian і відповідає нашій [політиці конфіденційності](https://obsidian.md/privacy). Ваші дані не збираються, і ми не збираємо жодних метрик використання. Код є [відкритим](https://github.com/obsidianmd/obsidian-clipper) і доступний для перевірки.
