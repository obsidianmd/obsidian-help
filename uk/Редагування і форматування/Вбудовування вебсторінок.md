Дізнайтеся, як використовувати HTML-елемент [iframe](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/iframe) для вбудовування вебсторінок у ваші нотатки.

Щоб вбудувати вебсторінку, додайте наступний код у вашу нотатку, замінивши текст-заповнювач на URL вебсторінки, яку ви хочете вбудувати:

```html
<iframe src="ВСТАВТЕ ВАШ URL СЮДИ"></iframe>
```

> [!note] Примітка
> Деякі вебсайти не дозволяють вбудовувати їх. Натомість вони можуть надавати спеціальні URL, призначені для вбудовування. Якщо вебсайт не підтримує вбудовування, спробуйте пошукати назву вебсайту з додаванням "embed iframe". Наприклад, "youtube embed iframe".

> [!tip] Підказка
> Якщо ви використовуєте [[Полотно|Canvas]], ви можете вбудувати вебсторінку в картку. Для отримання додаткової інформації зверніться до [[Полотно#Додавання карток з вебсторінок]].

## Вбудовування відео з YouTube

Щоб вбудувати відео з YouTube, використовуйте той самий синтаксис Markdown, що й для [[Базовий синтаксис форматування#Зовнішні зображення|зовнішніх зображень]]:

```md
![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)
```

![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)

## Вбудовування твіту

Щоб вбудувати твіт, використовуйте той самий синтаксис Markdown, що й для [[Базовий синтаксис форматування#Зовнішні зображення|зовнішніх зображень]]:

```md
![](https://twitter.com/obsdmd/status/1580548874246443010)
```
