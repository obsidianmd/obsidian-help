На цій сторінці наведено типові сполучення клавіш для навігації та редагування тексту в Obsidian. Ці комбінації надаються вашою операційною системою або фреймворком, на якому побудовано Obsidian, і не можуть бути налаштовані в межах Obsidian.

Для налаштовуваних сполучень клавіш для команд Obsidian див. [[Сполучення клавіш]].

## Комбінації для Windows та Linux

### Загальні дії

| Дія | Комбінація |
|-|-|
| Копіювати | `Ctrl+C` |
| Вирізати | `Ctrl+X` |
| Вставити | `Ctrl+V` |
| Вставити без форматування | `Ctrl+Shift+V` |
| Скасувати | `Ctrl+Z` |
| Повторити | `Ctrl+Shift+Z` або `Ctrl+Y` |
| Копіювати абзац | `Ctrl+C` (без виділеного тексту) |
| Вирізати абзац | `Ctrl+X` (без виділеного тексту) |

### Редагування тексту

| Дія | Комбінація |
|-|-|
| Вставити новий рядок | `Enter` |
| Видалити попередній символ | `Backspace` |
| Видалити наступний символ | `Delete` |
| Видалити попереднє слово | `Ctrl+Backspace` |
| Видалити наступне слово | `Ctrl+Delete` |
| Видалити поточний рядок | `Ctrl+Shift+K` (без виділеного тексту) |

### Навігація по тексту

| Дія | Комбінація |
|-|-|
| Перемістити курсор на один символ | `Left/→` |
| Перемістити курсор на початок попереднього слова | `Ctrl+←` |
| Перемістити курсор на кінець наступного слова | `Ctrl+→` |
| Перемістити курсор на початок поточного рядка | `Home` |
| Перемістити курсор на кінець поточного рядка | `End` |
| Перемістити курсор на попередній рядок | `↑` |
| Перемістити курсор на наступний рядок | `↓` |
| Перемістити курсор на початок нотатки | `Ctrl+Home` |
| Перемістити курсор на кінець нотатки | `Ctrl+End` |
| Перемістити курсор на одну сторінку вгору | `Page up` |
| Перемістити курсор на одну сторінку вниз | `Page down` |

### Виділення тексту

| Дія | Комбінація |
|-|-|
| Спростити виділення | `Escape` |
| Вибрати все | `Ctrl+A` |
| Розширити виділення на один символ | `Shift+Left/→` |
| Розширити виділення до початку попереднього слова | `Ctrl+Shift+←` |
| Розширити виділення до кінця наступного слова | `Ctrl+Shift+→` |
| Розширити виділення до початку поточного рядка | `Shift+Home` |
| Розширити виділення до кінця поточного рядка | `Shift+End` |
| Розширити виділення до початку нотатки | `Ctrl+Shift+Home` |
| Розширити виділення до кінця нотатки | `Ctrl+Shift+End` |
| Розширити виділення на одну сторінку вгору | `Shift+Page up` |
| Розширити виділення на одну сторінку вниз | `Shift+Page down` |
