Obsidian прагне до максимальної функціональності без порушення існуючих форматів. Тому ми використовуємо комбінацію різновидів [[Базовий синтаксис форматування|Markdown]].

Obsidian підтримує [CommonMark](https://commonmark.org/), [GitHub Flavored Markdown](https://github.github.com/gfm/) та [LaTeX](https://www.latex-project.org/).

> [!tip]- Markdown всередині HTML
> Obsidian не обробляє синтаксис Markdown всередині HTML-елементів. Це свідоме дизайнерське рішення для оптимізації продуктивності та зменшення складності парсера при роботі з великими документами.
>
> Наприклад, форматування Markdown, як-от `**bold**` або `` `code` ``, не буде оброблено всередині `<div>`, `<span>`, `<table>` або будь-яких інших HTML-тегів.
>
> ```md
> <div>
> This **will not** be bold.
> </div>
> ```

### Підтримувані розширення Markdown

| Синтаксис       | Опис                                                                                          |
| --------------- | --------------------------------------------------------------------------------------------- |
| `[[Link]]`      | [[Внутрішні посилання]]                                                                       |
| `![[Link]]`     | [[Вбудовування файлів]]                                                                       |
| `![[Link#^id]]` | [[Внутрішні посилання#Посилання на блок у нотатці\|Посилання на блоки]]                       |
| `^id`           | [[Внутрішні посилання#Посилання на блок у нотатці\|Визначення блоку]]                         |
| `[^id]`         | [[Базовий синтаксис форматування#Виноски\|Виноски]]                                           |
| `%%Text%%`      | [[Базовий синтаксис форматування#Коментарі\|Коментарі]]                                      |
| `~~Text~~`      | [[Базовий синтаксис форматування#Жирний, курсив, виділення\|Закреслення]]                     |
| `==Text==`      | [[Базовий синтаксис форматування#Жирний, курсив, виділення\|Виділення]]                       |
| `` ``` ``       | [[Базовий синтаксис форматування#Блоки коду\|Блоки коду]]                                     |
| `- [ ]`         | [[Базовий синтаксис форматування#Списки задач\|Незавершене завдання]]                          |
| `- [x]`         | [[Базовий синтаксис форматування#Списки задач\|Завершене завдання]]                            |
| `> [!note]`     | [[Виноски]]                                                                                   |
| (див. посилання)| [[Розширений синтаксис форматування#Таблиці\|Таблиці]]                                        |
