Ліцензія Catalyst — це одноразова покупка, яка надає ранній доступ до бета-версій та допомагає підтримувати подальшу розробку Obsidian. Ви можете підтримати Obsidian, [придбавши ліцензію Catalyst](https://obsidian.md/buy/catalyst).

Переваги включають:

- Ранній доступ до [[#Увімкнення версій раннього доступу|нових версій]] Obsidian
- Доступ до ексклюзивних [[#Приєднуйтесь до наших каналів Insider|каналів Insider]].
- Ексклюзивні [[#Отримайте значок на форумі|значки на форумі]] та [[#Отримайте значок у Discord|значки в Discord]].

## Рівні Catalyst

Існує три рівні Catalyst залежно від суми вашого внеску: *Insider*, *Supporter* та *VIP*.

| Переваги                                    | Insider | Supporter |   VIP    |
| -------------------------------------------- | :-----: | :-------: | :------: |
| Ранній доступ до нових версій                |    ✓    |     ✓     |    ✓     |
| Доступ до ексклюзивних каналів розробки      |    ✓    |     ✓     |    ✓     |
| Доступ до ексклюзивного каналу відпочинку    |         |     ✓     |    ✓     |
| Значок Insider                               |    ✓    |           |          |
| Значок Supporter                             |         |     ✓     |          |
| Значок VIP                                   |         |           |    ✓     |
| **Внесок (USD)**                             | **$25** |  **$50**  | **$100** |


> [!info] Підвищення рівня Catalyst
> Якщо ви вже маєте ліцензію Catalyst, ви можете підвищити свій рівень Catalyst, сплативши різницю між вартістю поточного та нового рівня.

## Доступ до переваг Catalyst

Після придбання ліцензії Catalyst ви можете скористатися низкою переваг. Дізнайтеся, як їх увімкнути, тут.

### Отримайте значок на форумі

**Примітка:** Щоб зв'язати свій обліковий запис Obsidian з обліковим записом на форумі, необхідно використовувати однакову електронну пошту для обох облікових записів.

1. Якщо ви ще цього не зробили, приєднайтесь до нашого [форуму](https://forum.obsidian.md/).
2. Перейдіть до [панелі облікового запису Obsidian](https://obsidian.md/account/catalyst).
3. У розділі **Catalyst** виберіть **Get forum badge**.

### Отримайте значок у Discord

1. Якщо ви ще цього не зробили, приєднайтесь до нашого [сервера Discord](https://discord.gg/veuWUTm).
2. Перейдіть до [панелі облікового запису Obsidian](https://obsidian.md/account/catalyst).
3. У розділі **Catalyst** виберіть **Get Discord badge**.

### Приєднуйтесь до наших каналів Insider

Учасники Insider можуть спілкуватися з командою Obsidian та іншими учасниками Insider в наших ексклюзивних каналах Discord. Ви автоматично приєднуєтесь до каналів Insider після того, як [[#Отримайте значок у Discord]].

- [vip-lounge](https://discord.com/channels/686053708261228577/714234422379151400) для обговорень з учасниками Catalyst.
- [insider-desktop-release](https://discord.com/channels/686053708261228577/716028884885307432) для останніх релізів раннього доступу на десктопі.
- [insider-mobile-release](https://discord.com/channels/686053708261228577/817515900349448202) для останніх релізів раннього доступу на мобільних пристроях.

### Увімкнення версій раннього доступу

Дізнайтеся, як отримати [[Версії раннього доступу|версії раннього доступу]].
