Obsidian Headless **(відкрита бета)** — це безголовий клієнт для сервісів Obsidian. Він дозволяє [[Автономна синхронізація|синхронізувати сховища]] з командного рядка без настільного застосунку, зберігаючи всі переваги швидкості, конфіденційності та наскрізного шифрування [[Вступ до Obsidian Sync|Obsidian Sync]].

Причини, з яких ви можете використовувати Obsidian Headless:

- Автоматизація віддалених резервних копій.
- Автоматизація публікації вебсайту.
- Надання агентним інструментам доступу до сховища без доступу до всього комп'ютера.
- Синхронізація спільного командного сховища з сервером, що передає дані іншим інструментам.
- Запуск запланованих автоматизацій, наприклад, агрегація щоденних нотаток у тижневі підсумки, автоматичне додавання міток тощо.

> [!info] Obsidian Headless проти Obsidian CLI
> [[Obsidian CLI]] керує настільним застосунком Obsidian з вашого терміналу. Obsidian Headless — це автономний клієнт, який працює незалежно, без потреби у настільному застосунку.

## Встановлення

Obsidian Headless **(відкрита бета)** потребує Node.js 22 або новішої версії. Встановіть його з [npm](https://www.npmjs.com/package/obsidian-headless):

```shell
npm install -g obsidian-headless
```

## Автентифікація

### Увійти

```shell
ob login
```

Якщо ви вже увійшли, `ob login` відобразить інформацію про ваш обліковий запис. Щоб змінити обліковий запис, передайте `--email` та/або `--password` для повторного входу.

```
ob login [--email <email>] [--password <password>] [--mfa <code>]
```

Усі параметри є інтерактивними, якщо їх не вказано — електронна пошта та пароль запитуються автоматично, а двофакторна автентифікація запитується автоматично, якщо вона увімкнена для облікового запису.

Щоб вийти та очистити збережені облікові дані:

```shell
ob logout
```
