Ця сторінка містить перелік поширених проблем, з якими ви можете зіткнутися під час використання [[Вступ до Obsidian Publish|Obsidian Publish]], та способи їх вирішення.

Обов'язково ознайомтеся спершу з [[Медіафайли]] та [[Обмеження Publish|Обмеження Publish]].

## Загальне

### Публікація нотаток

**Я отримую помилку хешу під час спроби опублікувати нотатку.**

Чи використовуєте ви [[Додатки спільноти|додаток спільноти]], який змінює час модифікації файлу при оновленні? Якщо так, цей додаток може конфліктувати з Publish. Будь ласка, надішліть звіт про помилку розробнику додатка для вирішення проблеми.

**Я отримую дивну мережеву помилку, і у мене дуже великий сайт Publish.**

Ймовірно, нам потрібно перевірити вашу базу даних. Будь ласка, [[Довідка та підтримка#Зв'язок з підтримкою Obsidian|зверніться до підтримки Obsidian]] за допомогою.

## CSS та теми

**Мій CSS у [[Тека конфігурації|теці конфігурації]] не працює на Publish. Чому?**

Publish не зчитує дані з теки конфігурації. Натомість вам потрібно створити файл `publish.css` у кореневому каталозі вашого сховища для публікації. Дізнатися більше про це можна в розділі [[Налаштування вашого сайту]].

**Мій CSS виглядає на Publish інакше, ніж у застосунку. Чому?**

CSS Obsidian Publish не повністю збігається з CSS у застосунку. Ми рекомендуємо [розробляти будь-який CSS та теми](https://docs.obsidian.md/Themes/Obsidian+Publish+themes/About+Obsidian+Publish+themes) спеціально для Publish з нуля.

Загалом те, що працює для [[Подання та режими редагування#Режим читання|режиму читання]], з великою ймовірністю працюватиме й на Publish.
