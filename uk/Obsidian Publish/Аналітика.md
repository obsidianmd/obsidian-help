## Передумови

Для використання аналітики на вашому сайті [[Вступ до Obsidian Publish|Obsidian Publish]] вам потрібен [[Особисті домени|особистий домен]].

> [!important] Важливо
> Перш ніж увімкнути аналітику, переконайтеся, що місцеві закони та нормативні акти дозволяють відстежувати відвідувачів. Залежно від інструментів, які ви використовуєте, може знадобитися додати банер згоди на ваш сайт.

## Google Analytics

Щоб увімкнути Google Analytics для вашого сайту Obsidian Publish:

1. На стрічковому меню, зліва від вікна застосунку, натисніть **Опублікувати зміни** ![[lucide-send.svg#icon]].
2. У діалоговому вікні **Опублікувати зміни** натисніть **Налаштування сайту** ![[lucide-cog.svg#icon]].
3. У полі **Трекінг код Google Analytics** введіть свій код відстеження.

Щоб використовувати Google Tag Manager замість Google Analytics, скористайтеся власним JavaScript для додавання своїх скриптів.

## Plausible, Fathom та інші сервіси аналітики

Ви можете додати більшість сервісів аналітики на свій сайт через файл [[Налаштування вашого сайту|publish.js]]. Обов'язково замініть `yourdomain.com` на ваш домен, а атрибут `src` скрипта — на скрипт від вашого провайдера аналітики.

Ось приклад використання [Plausible Analytics](https://plausible.io/)

```js
var analyticsScript = document.createElement('script');
analyticsScript.defer = true;
analyticsScript.setAttribute('data-domain', 'yourdomain.com');
analyticsScript.src = 'https://plausible.io/js/plausible.js';
document.head.appendChild(analyticsScript);
```

Такий самий підхід можна використовувати для [Fathom Analytics](https://usefathom.com/). Зверніть увагу на заміну `data-domain` на `data-site` — різні провайдери аналітики можуть мати різний формат вставки скрипта.

```javascript
var fathom = analyticsScript.createElement('script');
analyticsScript.defer = true;
analyticsScript.setAttribute('data-site', 'yourdomain.com');
analyticsScript.src = 'https://cdn.usefathom.com/script.js';
document.head.appendChild(analyticsScript);
```

## Вирішення проблем

Щоб перевірити, чи ваш сайт використовує сервіс аналітики, вимкніть будь-які розширення браузера для блокування реклами, такі як uBlock Origin, які можуть блокувати виконання скрипта відстеження.
