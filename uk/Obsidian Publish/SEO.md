Сайти [[Вступ до Obsidian Publish|Obsidian Publish]] автоматично оптимізуються для пошукових систем та [[Попередній перегляд посилань у соціальних мережах|карток для поширення у соціальних мережах]]. Ви також можете налаштувати метадані сторінки за допомогою описів, постійних посилань та зображень.

Зверніть увагу, що ви можете заборонити індексацію пошуковикам у [[Управління сайтами#Налаштування сайту|налаштуваннях сайту]].

### Google Search Console

Щоб відстежувати SEO за допомогою Google, зареєструйте свій сайт у [Google Search Console](https://search.google.com/search-console). Ви також можете дотримуватися наших рекомендованих кроків для [[Аналітика|налаштування Google Analytics]].

### Карта сайту

Карта сайту для вашого сайту Publish доступна за адресою `/sitemap.xml`, наприклад:
https://help.obsidian.md/sitemap.xml

Карта сайту також доступна як RSS-канал за адресою `/rss.xml`:
https://help.obsidian.md/rss.xml

### Постійні посилання

Розгляньте можливість використання [[Постійні посилання|постійних посилань]] для визначення незмінних URL-адрес на вашому сайті.

### Метадані

Метадані сторінки можна налаштувати за допомогою [[Властивості#Властивості для Obsidian Publish|властивостей]], що може бути корисним для [[Попередній перегляд посилань у соціальних мережах|карток для поширення у соціальних мережах]].

| Властивість   | Опис                                                                                                              |
| ------------- | ----------------------------------------------------------------------------------------------------------------- |
| `publish`     | Див. [[Публікація вашого контенту#Автоматичний вибір даних для публікації\|Автоматичний вибір даних для публікації]]. |
| `permalink`   | Див. [[Постійні посилання\|Постійні посилання]].                                                                  |
| `description` | Див. [[Попередній перегляд посилань у соціальних мережах#Опис\|Опис]].                                            |
| `image`       | Див. [[Попередній перегляд посилань у соціальних мережах#Зображення\|Зображення]].                                |
| `cover`       | Див. [[Попередній перегляд посилань у соціальних мережах#Зображення\|Зображення]].                                |
