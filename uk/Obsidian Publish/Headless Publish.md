[[Вступ до Obsidian Publish|Obsidian Publish]] пропонує безголовий клієнт для публікації сховищ без використання настільного застосунку. Це зручно для CI-пайплайнів та автоматизованих робочих процесів. Публікуйте свої останні зміни за розкладом або як частину пайплайну збірки.

Встановіть [[Obsidian Headless]] **(відкрита бета)**, щоб взаємодіяти з [[Вступ до Obsidian Publish|Obsidian Publish]] з командного рядка без настільного застосунку Obsidian.

## Швидкий старт

Встановіть [[Obsidian Headless|Obsidian Headless]] **(відкрита бета)**:

```shell
npm install -g obsidian-headless
```

У вас має бути активна [[Вступ до Obsidian Publish|підписка на Obsidian Publish]].

```shell
# Увійти
ob login

# Переглянути список ваших сайтів Publish
ob publish-list-sites

# Під'єднати локальне сховище до сайту Publish
cd ~/vaults/my-vault
ob publish-setup --site "my-site"

# Переглянути зміни без публікації
ob publish --dry-run

# Опублікувати зміни
ob publish
```

## Команди

### `ob publish-list-sites`

Показати всі сайти Publish, доступні для вашого облікового запису.

### `ob publish-create-site`

Створити новий сайт Publish.

```
ob publish-create-site --slug <slug>
```

| Параметр | Опис |
| --- | --- |
| `--slug` | Ідентифікатор URL для вашого сайту (наприклад, `my-notes` створює `publish.obsidian.md/my-notes`) |

### `ob publish-setup`

Під'єднати локальне сховище до сайту Publish.

```
ob publish-setup [--site <id-or-slug>] [--path <local-path>]
```

| Параметр | Опис |
| --- | --- |
| `--site` | ID або slug сайту |
| `--path` | Шлях до локального сховища (за замовчуванням: поточна тека) |

### `ob publish`

Опублікувати зміни сховища на ваш сайт. За замовчуванням включаються лише файли з `publish: true` у їхніх початкових метаданих.

```
ob publish [--path <local-path>] [--all] [--dry-run] [--yes]
```

| Параметр | Опис |
| --- | --- |
| `--path` | Шлях до локального сховища (за замовчуванням: поточна тека) |
| `--all` | Включити всі файли, а не лише позначені для публікації |
| `--dry-run` | Показати зміни без публікації |
| `--yes` | Опублікувати без запиту підтвердження |

### `ob publish-config`

Переглянути або змінити налаштування включення/виключення тек для сховища. Запустіть без параметрів, щоб відобразити поточну конфігурацію.

```
ob publish-config [--path <local-path>] [options]
```

| Параметр | Опис |
| --- | --- |
| `--path` | Шлях до локального сховища (за замовчуванням: поточна тека) |
| `--includes` | Теки для включення, через кому (порожній рядок для очищення) |
| `--excludes` | Теки для виключення, через кому (порожній рядок для очищення) |

### `ob publish-site-options`

Переглянути або оновити загальні налаштування відображення та навігації сайту. Запустіть без параметрів, щоб відобразити поточні налаштування.

```
ob publish-site-options [--path <local-path>] [options]
```

| Параметр | Опис |
| --- | --- |
| `--path` | Шлях до локального сховища (за замовчуванням: поточна тека) |
| `--site-name` | Назва сайту |
| `--index-file` | Шлях до файлу домашньої сторінки |
| `--logo` | Шлях до файлу логотипу (порожній рядок для очищення) |
| `--show-navigation` | Показ бічної панелі навігації |
| `--show-graph` | Показ графа |
| `--show-outline` | Показ змісту |
| `--show-search` | Показ пошуку |
| `--show-backlinks` | Показ зворотних посилань |
| `--show-hover-preview` | Показ попереднього перегляду |
| `--show-theme-toggle` | Показ перемикача теми |
| `--default-theme` | Тема за замовчуванням: `light` або `dark` |
| `--readable-line-length` | Обмежити максимальну довжину рядка |
| `--strict-line-breaks` | Подвійний дільник рядків |
| `--hide-title` | Приховати вбудований заголовок |
| `--sliding-window` | Режим ковзного вікна |
| `--nav-order` | Порядок навігації, шляхи через кому (порожній рядок для очищення) |
| `--nav-hidden` | Приховані елементи навігації, шляхи через кому (порожній рядок для очищення) |

### `ob publish-unlink`

Від'єднати сховище від сайту Publish.

```
ob publish-unlink [--path <local-path>]
```
