## Глобальні

Глобальні функції використовуються без типу.

### `escapeHTML()`

`escapeHTML(html: string): string`

- Екранує спеціальні символи в рядку, щоб зробити його безпечним для включення в HTML.

### `date()`

`date(date: string): date`

- `date(string): date` аналізує наданий рядок і повертає об'єкт дати.
- Рядок `date` має бути у форматі `YYYY-MM-DD HH:mm:ss`.

### `duration()`

`duration(value: string): duration`

- Аналізує рядок як тривалість. Зверніться до [[Синтаксис Бази#Арифметика дат|розділу арифметики дат]] щодо формату рядка `value`.
- Тривалості не потрібно явно аналізувати під час арифметики дат (наприклад, `now() + '1d'`), але це потрібно під час арифметики з тривалостями (наприклад, `now() + (duration('1d') * 2)`).
- При арифметиці тривалостей зі скалярами тривалість має бути зліва. Наприклад, `duration('5h') * 2`, а не `2 * duration('5h')`.

### `file()`

`file(path: string | file | url): file`

- Повертає об'єкт файлу для заданого файлу або шляху.
- Приклад: `file(link("[[filename]]"))` або `file("path to file")`.

### `html()`

`html(html: string): html`

- Перетворює рядок на фрагмент коду, який відображається як HTML.

### `if()`

`if(condition: any, trueResult: any, falseResult?: any): any`

- `condition` — умова, що оцінюється.
- `trueResult` — результат, якщо умова істинна.
- `falseResult` — необов'язковий результат, якщо умова хибна. Якщо не вказано, приймається значення `null`.
- Повертає `trueResult`, якщо `condition` є істинним або truthy-значенням, інакше повертає `falseResult`.
- Приклад: `if(isModified, "Modified", "Unmodified")`

### `image()`

`image(path: string | file | url): image`

- Повертає об'єкт зображення, який відображає зображення в поданні.
- Приклад: `image(image-property)` або `image("https://obsidian.md/images/obsidian-logo-gradient.svg")`

### `icon()`

`icon(name: string): icon`

- Повертає значення, яке відображається як значок у поданні. Назва значка має відповідати підтримуваному значку Lucide.
- Приклад: `icon("arrow-right")`.

### `link()`

`link(path: string | file, display?: value): Link`

- Аналізує рядок `path` і повертає об'єкт Link, який відображається як посилання на заданий шлях.
- За бажанням вкажіть параметр `display`, щоб змінити текст посилання.

### `list()`

`list(element: any): List`

- Якщо наданий елемент є списком, повертає його без змін.
- В іншому випадку обгортає наданий `element` у список, створюючи список з одним елементом.
- Ця функція може бути корисною, коли властивість містить суміш рядків або списків у сховищі.
- Приклад: `list("value")` повертає `["value"]`.

### `max()`

`max(value1: number, value2: number...): number`

- Повертає найбільше з усіх наданих чисел.

### `min()`

`min(value1: number, value2: number...): number`

- Повертає найменше з усіх наданих чисел.

### `now()`

`now(): date`

- `now()` повертає об'єкт дати, що представляє поточний момент.

### `number()`

`number(input: any): number`

- Намагається повернути надане значення як число.
- Об'єкти дат повертаються як мілісекунди з початку епохи Unix.
- Булеві значення повертають 1 або 0.
- Рядки аналізуються в число і повертають помилку, якщо результат недійсний.
- Приклад: `number("3.4")` повертає `3.4`.

### `duration()`

`duration(value: string): duration`

- Аналізує рядок як тривалість. Зверніться до [[Синтаксис Бази#Арифметика дат|розділу арифметики дат]] щодо формату рядка `value`.
- Тривалості не потрібно явно аналізувати під час арифметики дат (наприклад, `now() + '1d'`), але це потрібно під час арифметики з тривалостями (наприклад, `now() + (duration('1d') * 2)`).
- При арифметиці тривалостей зі скалярами тривалість має бути зліва. Наприклад, `duration('5h') * 2`, а не `2 * duration('5h')`.

### `today()`

`today(): date`

- `today()` повертає об'єкт дати, що представляє поточну дату. Часова частина встановлена на нуль.

### `random()`

`random(): number`

- `random()` повертає випадкове число від 0 до 1.
- Генерація числа оновлюється щоразу при завантаженні подання. Перехід між поданнями змінює випадкове число.

## Будь-який

Функції, які можна використовувати з будь-яким значенням. Це включає рядки (наприклад, `"hello"`), числа (наприклад, `42`), списки (наприклад, `[1,2,3]`), об'єкти тощо.

### `isTruthy()`

`any.isTruthy(): boolean`

- Повертає значення, приведене до булевого типу.
- Приклад: `1.isTruthy()` повертає `true`.

### `isType()`

`any.isType(type: string): boolean`

- Повертає true, якщо значення має наданий тип.
- Приклад: `"example".isType("string")` та `true.isType("boolean")` обидва повертають true.

### `toString()`

`any.toString(): string`

- Повертає рядкове представлення будь-якого значення.
- Приклад: `123.toString()` повертає `"123"`.

## Дата

Функції, які можна використовувати з датою і часом, наприклад `date("2025-05-27")`. Порівняння дат можна виконувати за допомогою [[Синтаксис Бази#Арифметика дат|арифметики дат]].

### Поля

Наступні поля доступні для дат:

| Поле               | Тип      | Опис                         |
| ------------------ | -------- | ---------------------------- |
| `date.year`        | `number` | Рік дати                     |
| `date.month`       | `number` | Місяць дати (1–12)           |
| `date.day`         | `number` | День місяця                  |
| `date.hour`        | `number` | Година (0–23)                |
| `date.minute`      | `number` | Хвилина (0–59)               |
| `date.second`      | `number` | Секунда (0–59)               |
| `date.millisecond` | `number` | Мілісекунда (0–999)          |

### `date()`

`date.date(): date`

- Повертає об'єкт дати з видаленим часом.
- Приклад: `now().date().format("YYYY-MM-DD HH:mm:ss")` повертає рядок, наприклад "2025-12-31 00:00:00"

### `format()`

`date.format(format: string): string`

- `format` — рядок формату (наприклад, `"YYYY-MM-DD"`).
- Повертає дату, відформатовану відповідно до рядка формату Moment.js.
- Приклад: `date.format("YYYY-MM-DD")` повертає `"2025-05-27"`.

### `time()`

`date.time(): string`

- Повертає час.
- Приклад: `now().time()` повертає рядок, наприклад "23:59:59"

### `relative()`

`date.relative(): string`

- Повертає зрозуміле порівняння дати з поточним моментом часу.
- Приклад: `file.mtime.relative()` повертає значення на кшталт `3 days ago`.

### `isEmpty()`

`date.isEmpty(): boolean`

- Повертає false.

## Рядок

Функції, які можна використовувати з послідовністю символів, наприклад `"hello"`.

### Поля

| Поле            | Тип      | Опис                            |
| --------------- | -------- | ------------------------------- |
| `string.length` | `number` | Кількість символів у рядку      |

### `contains()`

`string.contains(value: string): boolean`

- `value` — підрядок для пошуку.
- Повертає true, якщо рядок містить `value`.
- Приклад: `"hello".contains("ell")` повертає `true`.

### `containsAll()`

`string.containsAll(...values: string): boolean`

- `values` — один або кілька підрядків для пошуку.
- Повертає true, якщо рядок містить усі `values`.
- Приклад: `"hello".containsAll("h", "e")` повертає `true`.

### `containsAny()`

`string.containsAny(...values: string): boolean`

- `values` — один або кілька підрядків для пошуку.
- Повертає true, якщо рядок містить принаймні одне з `values`.
- Приклад: `"hello".containsAny("x", "y", "e")` повертає `true`.

### `endsWith()`

`string.endsWith(query: string): boolean`

- `query` — рядок для перевірки в кінці.
- Повертає true, якщо цей рядок закінчується на `query`.
- Приклад: `"hello".endsWith("lo")` повертає `true`.

### `isEmpty()`

`string.isEmpty(): boolean`

- Повертає true, якщо рядок не має символів або відсутній.
- Приклад: `"Hello world".isEmpty()` повертає `false`.
- Приклад: `"".isEmpty()` повертає `true`.

### `lower()`

`string.lower(): string`

- Повертає рядок, перетворений у нижній регістр.

### `replace()`

`string.replace(pattern: string | Regexp, replacement: string): string`

- `pattern` — значення для пошуку в цільовому рядку.
- `replacement` — значення для заміни знайдених шаблонів.
- Якщо `pattern` є рядком, усі входження шаблону будуть замінені.
- Якщо `pattern` є Regexp, прапор `g` визначає, чи замінюється лише перше або всі входження.
- Приклад: `""a:b:c:d".replace(/:/, "-")` повертає `"a-b,c,d"`, тоді як `"a:b:c:d".replace(/:/g, "-")` повертає `"a-b-c-d"`.

### `repeat()`

`string.repeat(count: number): string`

- `count` — кількість повторень рядка.
- Приклад: `"123".repeat(2)` повертає `"123123"`

### `reverse()`

`string.reverse(): string`

- Перевертає рядок.
- Приклад: `"hello".reverse()` повертає `"olleh"`.

### `slice()`

`string.slice(start: number, end?: number): string`

- `start` — включний початковий індекс.
- `end` — необов'язковий виключний кінцевий індекс.
- Повертає підрядок від `start` (включно) до `end` (виключно).
- Приклад: `"hello".slice(1, 4)` повертає `"ell"`.
- Якщо `end` опущено, зріз виконується до кінця рядка.

### `split()`

`string.split(separator: string | Regexp, n?: number): list`

- `separator` — розділювач для розбиття рядка.
- `n` — необов'язкове число. Якщо вказано, результат міститиме перші `n` елементів.
- Повертає список підрядків.
- Приклад: `"a,b,c,d".split(",", 3)` або `"a,b,c,d".split(/,/, 3)` повертає `["a", "b", "c"]`.

### `startsWith()`

`string.startsWith(query: string): boolean`

- `query` — рядок для перевірки на початку.
- Повертає true, якщо цей рядок починається з `query`.
- Приклад: `"hello".startsWith("he")` повертає `true`.

### `title()`

`string.title(): string`

- Перетворює рядок у регістр заголовка (перша літера кожного слова велика).
- Приклад: `"hello world".title()` повертає `"Hello World"`.

### `trim()`

`string.trim(): string`

- Видаляє пробіли з обох кінців рядка.
- Приклад: `"  hi  ".trim()` повертає `"hi"`.

## Число

Функції, які можна використовувати з числовими значеннями, наприклад `42`, `3.14`.

### `abs()`

`number.abs(): number`

- Повертає абсолютне значення числа.
- Приклад: `(-5).abs()` повертає `5`.

### `ceil()`

`number.ceil(): number`

- Округлює число вгору до найближчого цілого.
- Приклад: `(2.1).ceil()` повертає `3`.

### `floor()`

`number.floor(): number`

- Округлює число вниз до найближчого цілого.
- Приклад: `(2.9).floor()` повертає `2`.

### `isEmpty()`

`number.isEmpty(): boolean`

- Повертає true, якщо число відсутнє.
- Приклад: `5.isEmpty()` повертає `false`.

### `round()`

`number.round(digits: number): number`

- Округлює число до найближчого цілого.
- За бажанням вкажіть параметр `digits` для округлення до відповідної кількості десяткових знаків.
- Приклад: `(2.5).round()` повертає `3`, а `(2.3333).round(2)` повертає `2.33`.

### `toFixed()`

`number.toFixed(precision: number): string`

- `precision` — кількість десяткових знаків.
- Повертає рядок із числом у нотації з фіксованою точкою.
- Приклад: `(3.14159).toFixed(2)` повертає `"3.14"`.

## Список

Функції, які можна використовувати з упорядкованим списком елементів, наприклад `[1, 2, 3]`.

### Поля

| Поле          | Тип      | Опис                            |
| ------------- | -------- | ------------------------------- |
| `list.length` | `number` | Кількість елементів у списку    |

### `contains()`

`list.contains(value: any): boolean`

- `value` — елемент для пошуку.
- Повертає true, якщо список містить `value`.
- Приклад: `[1,2,3].contains(2)` повертає `true`.

### `containsAll()`

`list.containsAll(...values: any): boolean`

- `values` — один або кілька елементів для пошуку.
- Повертає true, якщо список містить усі `values`.
- Приклад: `[1,2,3].containsAll(2,3)` повертає `true`.

### `containsAny()`

`list.containsAny(...values: any): boolean`

- `values` — один або кілька елементів для пошуку.
- Повертає true, якщо список містить принаймні одне зі значень `values`.
- Приклад: `[1,2,3].containsAny(3,4)` повертає `true`.

### `filter()`

`list.filter(value: Boolean): list`

- Фільтрує елементи цього списку за допомогою функції фільтрації, яка використовує змінні `index` і `value` та повертає булеве значення, що визначає, чи слід зберігати елемент.
- `value` — значення елемента в списку.
- `index` — індекс поточного значення.
- Приклад: `[1,2,3,4].filter(value > 2)` повертає `[3,4]`.

### `flat()`

`list.flat(): list`

- Вирівнює вкладені списки в один список.
- Приклад: `[1,[2,3]].flat()` повертає `[1,2,3]`.

### `isEmpty()`

`list.isEmpty(): boolean`

- Повертає true, якщо список не має елементів.
- Приклад: `[1,2,3].isEmpty()` повертає `false`.

### `join()`

`list.join(separator: string): string`

- `separator` — рядок, що вставляється між елементами.
- Об'єднує всі елементи списку в один рядок.
- Приклад: `[1,2,3].join(",")` повертає `"1,2,3"`.

### `map()`

`list.map(value: Any): list`

- Перетворює кожен елемент цього списку за допомогою функції перетворення, яка використовує змінні `index` і `value` та повертає нове значення для розміщення в списку.
- `value` — значення елемента в списку.
- `index` — індекс поточного значення.
- Приклад: `[1,2,3,4].map(value + 1)` повертає `[2,3,4,5]`.

### `reduce()`

`list.reduce(expression: Any, acc: Any): Any`

- Зводить елементи цього списку до одного значення, виконуючи вираз для кожного елемента. Вираз може використовувати змінні `index`, `value` та `acc` (акумулятор) і має повертати наступне значення акумулятора.
- `expression` обчислюється для кожного елемента в списку.
- `value` — значення поточного елемента в списку.
- `index` — індекс поточного елемента.
- `acc` — накопичене значення на даний момент.
- Приклад (сума): `[1,2,3].reduce(acc + value, 0)` повертає `6`.
- Приклад (максимум): `values.filter(value.isType("number")).reduce(if(acc == null || value > acc, value, acc), null)` повертає найбільше число або `null`, якщо таких немає.

### `reverse()`

`list.reverse(): list`

- Перевертає список на місці.
- Приклад: `[1,2,3].reverse()` повертає `[3,2,1]`.

### `slice()`

`list.slice(start: number, end?: number): list`

- `start` — включний початковий індекс.
- `end` — необов'язковий виключний кінцевий індекс.
- Повертає поверхневу копію частини списку від `start` (включно) до `end` (виключно).
- Приклад: `[1,2,3,4].slice(1,3)` повертає `[2,3]`.
- Якщо `end` опущено, зріз виконується до кінця списку.

### `sort()`

`list.sort(): list`

- Сортує елементи списку від найменшого до найбільшого.
- Приклад: `[3, 1, 2].sort()` повертає `[1, 2, 3]`.
- Приклад: `["c", "a", "b"].sort()` повертає `["a", "b", "c"]`.

### `unique()`

`list.unique(): list`

- Видаляє повторювані елементи.
- Приклад: `[1,2,2,3].unique()` повертає `[1,2,3]`.

## Посилання

Функції, які можна використовувати з посиланням. Посилання можна створити з файлу (`file.asLink()`) або шляху (`link("path")`).

### `asFile()`

`link.asFile(): file`

- Повертає об'єкт файлу, якщо посилання вказує на дійсний локальний файл.
- Приклад: `link("[[filename]]").asFile()`

### `linksTo()`

`link.linksTo(file): boolean`

- Повертає, чи файл, представлений `link`, має посилання на `file`.

## Файл

Функції, які можна використовувати з файлом у сховищі.

### Поля

Наступні поля доступні для файлів:

| Поле              | Тип      | Опис                                                    |
| ----------------- | -------- | ------------------------------------------------------- |
| `file.name`       | `string` | Назва цього файлу.                                      |
| `file.basename`   | `string` | Назва цього файлу без розширення.                       |
| `file.path`       | `string` | Повний шлях до цього файлу відносно кореня сховища.     |
| `file.folder`     | `string` | Повний шлях до батьківської теки.                       |
| `file.ext`        | `string` | Розширення цього файлу.                                 |
| `file.size`       | `number` | Розмір цього файлу в байтах.                            |
| `file.properties` | `object` | Властивості нотатки для цього файлу.                    |
| `file.tags`       | `list`   | Мітки цього файлу. Включає вбудовані мітки.             |
| `file.links`      | `list`   | Внутрішні посилання в цьому файлі.                      |
| `file.ctime`      | `date`   | Мітка часу створення цього файлу.                       |
| `file.mtime`      | `date`   | Мітка часу останньої зміни цього файлу.                 |

### `asLink()`

`file.asLink(display?: string): Link`

- `display` — необов'язковий текст відображення для посилання.
- Повертає об'єкт Link, що відображається як функціональне посилання.
- Приклад: `file.asLink()`

### `hasLink()`

`file.hasLink(otherFile: file | string): boolean`

- `otherFile` — інший об'єкт файлу або рядковий шлях для перевірки.
- Повертає true, якщо `file` посилається на `otherFile`.
- Приклад: `file.hasLink(otherFile)` повертає `true`, якщо є посилання з `file` на `otherFile`.

### `hasProperty()`

`file.hasProperty(name: string): boolean`

- Повертає true, якщо нотатка має задану властивість файлу.

### `hasTag()`

`file.hasTag(...values: string): boolean`

- `values` — одна або кілька назв міток.
- Повертає true, якщо файл має будь-яку з міток у `values`.
- Приклад: `file.hasTag("tag1", "tag2")` повертає `true`, якщо файл має мітку `#tag1` або `#tag2`. Це також включає будь-які [[Мітки#Вкладені мітки|вкладені мітки]], такі як `#tag1/a` або `#tag2/b`.

### `inFolder()`

`file.inFolder(folder: string): boolean`

- `folder` — назва теки для перевірки.
- Повертає true, якщо файл знаходиться у вказаній теці або одній з її підтек.
- Приклад: `file.inFolder("notes")` повертає `true`.

## Об'єкт

Функції, які можна використовувати з колекцією пар ключ-значення, наприклад `{"a": 1, "b": 2}`.

### `isEmpty()`

`object.isEmpty(): boolean`

- Повертає true, якщо об'єкт не має власних властивостей.
- Приклад: `{}.isEmpty()` повертає `true`.

### `keys()`

`object.keys(): list`

- Повертає список, що містить ключі об'єкта.

### `values()`

`object.values(): list`

- Повертає список, що містить значення об'єкта.

## Регулярний вираз

Функції, які можна використовувати з шаблоном регулярного виразу. Приклад: `/abc/`.

### `matches()`

`regexp.matches(value: string): boolean`

- `value` — рядок для перевірки.
- Повертає true, якщо регулярний вираз збігається з `value`.
- Приклад: `/abc/.matches("abcde")` повертає `true`.
