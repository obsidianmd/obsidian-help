Obsidian використовує файли Markdown `.md` як основний формат для нотаток. Це спрощує імпорт файлів Markdown до Obsidian. Багато застосунків можуть редагувати файли Markdown або експортувати в цей формат.

Є кілька способів додати файли Markdown до вашого сховища Obsidian:

1. Перетягніть файли або теки у вікно [[Файловий провідник|Файлового провідника]]
2. Використайте файловий менеджер вашої системи (наприклад, Провідник Windows або Finder на macOS) і перемістіть файли безпосередньо до теки сховища Obsidian.

### Конвертація з інших різновидів Markdown

Obsidian використовує [[Obsidian Flavored Markdown]]. Деякі застосунки використовують дещо інший синтаксис Markdown. Ви можете скористатися [[Імпорт Markdown-файлів|Імпортом Markdown-файлів]], щоб конвертувати ваші файли до синтаксису Markdown Obsidian.
