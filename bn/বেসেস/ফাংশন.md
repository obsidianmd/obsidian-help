## Global

Global ফাংশন কোনো টাইপ ছাড়াই ব্যবহার করা হয়।

### `escapeHTML()`

`escapeHTML(html: string): string`

- HTML-এ অন্তর্ভুক্তির জন্য নিরাপদ করতে একটি স্ট্রিংয়ে বিশেষ অক্ষর এস্কেপ করে।

### `date()`

`date(date: string): date`

- `date(string): date` প্রদত্ত স্ট্রিং পার্স করে এবং একটি date অবজেক্ট ফেরত দেয়।
- `date` স্ট্রিংটি `YYYY-MM-DD HH:mm:ss` ফরম্যাটে হওয়া উচিত।

### `duration()`

`duration(value: string): duration`

- একটি স্ট্রিংকে duration হিসেবে পার্স করে। `value` স্ট্রিং ফরম্যাটের জন্য [[বেসেস সিনট্যাক্স#Date arithmetic|তারিখ গণিত বিভাগ]] দেখুন।
- তারিখ গণিতের জন্য সুস্পষ্ট পার্সিং প্রয়োজন নেই (উদাহরণস্বরূপ, `now() + '1d'`), তবে duration-এর উপর গণিত করার সময় এটি প্রয়োজন (উদাহরণস্বরূপ, `now() + (duration('1d') * 2)`)।
- স্কেলারসহ duration-এর উপর গণিত করার সময়, duration অবশ্যই বামে থাকতে হবে। উদাহরণস্বরূপ `duration('5h') * 2`, `2 * duration('5h')`-এর পরিবর্তে।

### `file()`

`file(path: string | file | url): file`

- প্রদত্ত ফাইল বা পাথের জন্য একটি file অবজেক্ট ফেরত দেয়।
- উদাহরণ: `file(link("[[filename]]"))` বা `file("path to file")`।

### `html()`

`html(html: string): html`

- একটি স্ট্রিংকে এমন একটি কোড স্নিপেটে রূপান্তরিত করে যা HTML হিসেবে রেন্ডার হয়।

### `if()`

`if(condition: any, trueResult: any, falseResult?: any): any`

- `condition` হলো মূল্যায়ন করার এক্সপ্রেশন।
- `trueResult` হলো `condition` সত্য হলে আউটপুট।
- `falseResult` হলো `condition` মিথ্যা হলে ঐচ্ছিক আউটপুট। বাদ দিলে, ডিফল্টরূপে `null` হয়।
- `condition` সত্য বা truthy হলে `trueResult` ফেরত দেয়, নয়তো `falseResult` ফেরত দেয়।
- উদাহরণ: `if(isModified, "Modified", "Unmodified")`

### `image()`

`image(path: string | file | url): image`

- একটি image অবজেক্ট ফেরত দেয় যা ভিউতে ছবিটি রেন্ডার করে।
- উদাহরণ: `image(image-property)` বা `image("https://obsidian.md/images/obsidian-logo-gradient.svg")`।

### `icon()`

`icon(name: string): icon`

- এমন একটি মান ফেরত দেয় যা একটি ভিউতে আইকন হিসেবে রেন্ডার হবে। আইকনের নাম অবশ্যই একটি সমর্থিত Lucide আইকনের সাথে মিলতে হবে।
- উদাহরণ: `icon("arrow-right")`।

### `link()`

`link(path: string | file, display?: value): Link`

- একটি স্ট্রিং `path` পার্স করে এবং প্রদত্ত পাথের একটি লিঙ্ক হিসেবে রেন্ডার হওয়া একটি Link অবজেক্ট ফেরত দেয়।
- ঐচ্ছিকভাবে লিঙ্কের প্রদর্শনের টেক্সট নির্ধারণ করতে `display` প্যারামিটার প্রদান করুন।

### `list()`

`list(element: any): List`

- প্রদত্ত উপাদানটি একটি লিস্ট হলে, এটি অপরিবর্তিতভাবে ফেরত দেয়।
- নয়তো, প্রদত্ত `element` একটি লিস্টে মোড়ানো হয়, একটি একক উপাদানসহ একটি লিস্ট তৈরি করে।
- ভল্ট জুড়ে একটি প্রপার্টিতে স্ট্রিং বা লিস্টের মিশ্রণ থাকলে এই ফাংশন ব্যবহার করুন।
- উদাহরণ: `list("value")` `["value"]` ফেরত দেয়।

### `max()`

`max(value1: number, value2: number...): number`

- প্রদত্ত সব সংখ্যার মধ্যে সবচেয়ে বড়টি ফেরত দেয়।

### `min()`

`min(value1: number, value2: number...): number`

- প্রদত্ত সব সংখ্যার মধ্যে সবচেয়ে ছোটটি ফেরত দেয়।

### `now()`

`now(): date`

- বর্তমান মুহূর্তের জন্য একটি date অবজেক্ট ফেরত দেয়।

### `number()`

`number(input: any): number`

- প্রদত্ত মানটিকে একটি সংখ্যা হিসেবে ফেরত দেওয়ার চেষ্টা করে।
- Date অবজেক্টকে Unix epoch থেকে মিলিসেকেন্ড হিসেবে ফেরত দেয়।
- বুলিয়ানকে `1` বা `0` হিসেবে ফেরত দেয়।
- স্ট্রিংকে সংখ্যা হিসেবে পার্স করে, এবং স্ট্রিংটি বৈধ সংখ্যা না হলে একটি এরর ফেরত দেয়।
- উদাহরণ: `number("3.4")` `3.4` ফেরত দেয়।

### `today()`

`today(): date`

- বর্তমান তারিখের জন্য একটি date অবজেক্ট ফেরত দেয়। সময়ের অংশ মধ্যরাত হিসেবে নির্ধারিত থাকে।

### `random()`

`random(): number`

- 0 এবং 1-এর মধ্যে একটি র‍্যান্ডম সংখ্যা ফেরত দেয়।
- একটি ভিউ লোড হলে সংখ্যা তৈরি রিফ্রেশ হয়। ভিউয়ের মধ্যে নেভিগেট করলে র‍্যান্ডম সংখ্যা পরিবর্তিত হয়।

## Any type

যেকোনো মানের সাথে ব্যবহার করা যায় এমন ফাংশন। এর মধ্যে স্ট্রিং (যেমন `"hello"`), সংখ্যা (যেমন `42`), লিস্ট (যেমন `[1,2,3]`), অবজেক্ট, এবং আরও অনেক কিছু অন্তর্ভুক্ত।

### `isTruthy()`

`any.isTruthy(): boolean`

- মানটিকে বুলিয়ানে রূপান্তরিত করে ফেরত দেয়।
- উদাহরণ: `1.isTruthy()` `true` ফেরত দেয়।

### `isType()`

`any.isType(type: string): boolean`

- মানটি প্রদত্ত টাইপের হলে true ফেরত দেয়।
- উদাহরণ: `"example".isType("string")` এবং `true.isType("boolean")` উভয়ই true ফেরত দেয়।

### `toString()`

`any.toString(): string`

- যেকোনো মানের স্ট্রিং প্রতিনিধিত্ব ফেরত দেয়।
- উদাহরণ: `123.toString()` `"123"` ফেরত দেয়।

## Date type

`date("2025-05-27")`-এর মতো একটি তারিখ ও সময়ের সাথে ব্যবহার করা যায় এমন ফাংশন। তারিখ তুলনা [[বেসেস সিনট্যাক্স#Date arithmetic|তারিখ গণিত]] ব্যবহার করে করা যায়।

### ফিল্ড

তারিখের জন্য নিম্নলিখিত ফিল্ডগুলো উপলব্ধ:

| ফিল্ড              | টাইপ     | বর্ণনা                  |
| ------------------ | -------- | ---------------------------- |
| `date.year`        | `number` | তারিখের বছর         |
| `date.month`       | `number` | তারিখের মাস (1–12) |
| `date.day`         | `number` | মাসের দিন         |
| `date.hour`        | `number` | ঘণ্টা (0–23)              |
| `date.minute`      | `number` | মিনিট (0–59)            |
| `date.second`      | `number` | সেকেন্ড (0–59)            |
| `date.millisecond` | `number` | মিলিসেকেন্ড (0–999)      |

### `date()`

`date.date(): date`

- সময় বাদ দিয়ে একটি date অবজেক্ট ফেরত দেয়।
- উদাহরণ: `now().date().format("YYYY-MM-DD HH:mm:ss")` "2025-12-31 00:00:00"-এর মতো একটি স্ট্রিং ফেরত দেয়।

### `format()`

`date.format(format: string): string`

- `format` হলো ফরম্যাট স্ট্রিং (যেমন, `"YYYY-MM-DD"`)।
- একটি Moment.js ফরম্যাট স্ট্রিং অনুযায়ী ফরম্যাট করা তারিখ ফেরত দেয়।
- উদাহরণ: `date.format("YYYY-MM-DD")` `"2025-05-27"` ফেরত দেয়।

### `time()`

`date.time(): string`

- সময়ের অংশ একটি স্ট্রিং হিসেবে ফেরত দেয়।
- উদাহরণ: `now().time()` "23:59:59"-এর মতো একটি স্ট্রিং ফেরত দেয়।

### `relative()`

`date.relative(): string`

- বর্তমান তারিখ-সময়ের সাথে তারিখের একটি পাঠযোগ্য তুলনা ফেরত দেয়।
- উদাহরণ: `file.mtime.relative()` `3 days ago`-এর মতো একটি মান ফেরত দেয়।

### `isEmpty()`

`date.isEmpty(): boolean`

- false ফেরত দেয়।

## String type

`"hello"`-এর মতো অক্ষরের একটি ধারার সাথে ব্যবহার করা যায় এমন ফাংশন।

### ফিল্ড

| ফিল্ড           | টাইপ     | বর্ণনা                            |
| --------------- | -------- | -------------------------------------- |
| `string.length` | `number` | স্ট্রিংয়ের অক্ষর সংখ্যা |

### `contains()`

`string.contains(value: string): boolean`

- `value` হলো খোঁজার জন্য সাব-স্ট্রিং।
- স্ট্রিংয়ে `value` থাকলে true ফেরত দেয়।
- উদাহরণ: `"hello".contains("ell")` `true` ফেরত দেয়।

### `containsAll()`

`string.containsAll(...values: string): boolean`

- `values` হলো খোঁজার জন্য এক বা একাধিক সাব-স্ট্রিং।
- স্ট্রিংয়ে সব `values` থাকলে true ফেরত দেয়।
- উদাহরণ: `"hello".containsAll("h", "e")` `true` ফেরত দেয়।

### `containsAny()`

`string.containsAny(...values: string): boolean`

- `values` হলো খোঁজার জন্য এক বা একাধিক সাব-স্ট্রিং।
- স্ট্রিংয়ে `values`-এর মধ্যে অন্তত একটি থাকলে true ফেরত দেয়।
- উদাহরণ: `"hello".containsAny("x", "y", "e")` `true` ফেরত দেয়।

### `endsWith()`

`string.endsWith(query: string): boolean`

- `query` হলো শেষে যাচাই করার স্ট্রিং।
- এই স্ট্রিংটি `query` দিয়ে শেষ হলে true ফেরত দেয়।
- উদাহরণ: `"hello".endsWith("lo")` `true` ফেরত দেয়।

### `isEmpty()`

`string.isEmpty(): boolean`

- স্ট্রিংয়ে কোনো অক্ষর না থাকলে, বা এটি উপস্থিত না থাকলে true ফেরত দেয়।
- উদাহরণ: `"Hello world".isEmpty()` `false` ফেরত দেয়।
- উদাহরণ: `"".isEmpty()` `true` ফেরত দেয়।

### `lower()`

`string.lower(): string`

- স্ট্রিংটিকে লোয়ারকেসে রূপান্তরিত করে ফেরত দেয়।

### `replace()`

`string.replace(pattern: string | Regexp, replacement: string): string`

- `pattern` হলো টার্গেট স্ট্রিংয়ে খোঁজার মান।
- `replacement` হলো পাওয়া প্যাটার্নগুলো প্রতিস্থাপন করার মান। `pattern` একটি Regexp হলে, আপনি `replacement`-এ `$1`, `$2`, ইত্যাদি ব্যবহার করে ক্যাপচার গ্রুপ রেফারেন্স করতে পারেন।
- `pattern` একটি স্ট্রিং হলে, প্যাটার্নের সব উপস্থিতি প্রতিস্থাপিত হবে।
- `pattern` একটি Regexp হলে, `g` ফ্ল্যাগ নির্ধারণ করে শুধু প্রথমটি নাকি সব উপস্থিতি প্রতিস্থাপিত হবে।
- উদাহরণ: `"a:b:c:d".replace(/:/, "-")` `"a-b:c:d"` ফেরত দেয়, যেখানে `"a:b:c:d".replace(/:/g, "-")` `"a-b-c-d"` ফেরত দেয়।
- ক্যাপচার গ্রুপসহ উদাহরণ: `"John Smith".replace(/(\w+) (\w+)/, "$2, $1")` `"Smith, John"` ফেরত দেয়।

### `repeat()`

`string.repeat(count: number): string`

- `count` হলো স্ট্রিং কতবার পুনরাবৃত্তি করা হবে তার সংখ্যা।
- উদাহরণ: `"123".repeat(2)` `"123123"` ফেরত দেয়।

### `reverse()`

`string.reverse(): string`

- স্ট্রিংটি বিপরীত করে।
- উদাহরণ: `"hello".reverse()` `"olleh"` ফেরত দেয়।

### `slice()`

`string.slice(start: number, end?: number): string`

- `start` হলো ইনক্লুসিভ শুরুর ইনডেক্স।
- `end` হলো ঐচ্ছিক এক্সক্লুসিভ শেষের ইনডেক্স।
- `start` (ইনক্লুসিভ) থেকে `end` (এক্সক্লুসিভ) পর্যন্ত একটি সাব-স্ট্রিং ফেরত দেয়।
- উদাহরণ: `"hello".slice(1, 4)` `"ell"` ফেরত দেয়।
- `end` বাদ দিলে, স্ট্রিংয়ের শেষ পর্যন্ত স্লাইস করে।

### `split()`

`string.split(separator: string | Regexp, n?: number): list`

- `separator` হলো স্ট্রিং বিভক্ত করার জন্য ডিলিমিটার।
- `n` হলো একটি ঐচ্ছিক সংখ্যা। প্রদান করা হলে, ফলাফলে প্রথম `n` উপাদান থাকবে।
- সাব-স্ট্রিংয়ের একটি লিস্ট ফেরত দেয়।
- উদাহরণ: `"a,b,c,d".split(",", 3)` বা `"a,b,c,d".split(/,/, 3)` `["a", "b", "c"]` ফেরত দেয়।

### `startsWith()`

`string.startsWith(query: string): boolean`

- `query` হলো শুরুতে যাচাই করার স্ট্রিং।
- এই স্ট্রিংটি `query` দিয়ে শুরু হলে true ফেরত দেয়।
- উদাহরণ: `"hello".startsWith("he")` `true` ফেরত দেয়।

### `title()`

`string.title(): string`

- স্ট্রিংটিকে টাইটেল কেসে রূপান্তরিত করে (প্রতিটি শব্দের প্রথম অক্ষর বড় হাতের করে)।
- উদাহরণ: `"hello world".title()` `"Hello World"` ফেরত দেয়।

### `trim()`

`string.trim(): string`

- স্ট্রিংয়ের উভয় প্রান্ত থেকে হোয়াইটস্পেস সরিয়ে দেয়।
- উদাহরণ: `"  hi  ".trim()` `"hi"` ফেরত দেয়।

## Number type

`42`, `3.14`-এর মতো সাংখ্যিক মানের সাথে ব্যবহার করা যায় এমন ফাংশন।

### `abs()`

`number.abs(): number`

- সংখ্যাটির পরম মান ফেরত দেয়।
- উদাহরণ: `(-5).abs()` `5` ফেরত দেয়।

### `ceil()`

`number.ceil(): number`

- সংখ্যাটিকে নিকটতম পূর্ণসংখ্যায় উপরের দিকে রাউন্ড করে।
- উদাহরণ: `(2.1).ceil()` `3` ফেরত দেয়।

### `floor()`

`number.floor(): number`

- সংখ্যাটিকে নিকটতম পূর্ণসংখ্যায় নিচের দিকে রাউন্ড করে।
- উদাহরণ: `(2.9).floor()` `2` ফেরত দেয়।

### `isEmpty()`

`number.isEmpty(): boolean`

- সংখ্যাটি উপস্থিত না থাকলে true ফেরত দেয়।
- উদাহরণ: `5.isEmpty()` `false` ফেরত দেয়।

### `round()`

`number.round(digits: number): number`

- সংখ্যাটিকে নিকটতম পূর্ণসংখ্যায় রাউন্ড করে।
- ঐচ্ছিকভাবে, কতটি দশমিক স্থানে রাউন্ড করতে হবে তা নির্ধারণ করতে একটি `digits` প্যারামিটার প্রদান করুন।
- উদাহরণ: `(2.5).round()` `3` ফেরত দেয়, এবং `(2.3333).round(2)` `2.33` ফেরত দেয়।

### `toFixed()`

`number.toFixed(precision: number): string`

- `precision` হলো দশমিক স্থানের সংখ্যা।
- ফিক্সড-পয়েন্ট নোটেশনে সংখ্যাসহ একটি স্ট্রিং ফেরত দেয়।
- উদাহরণ: `(3.14159).toFixed(2)` `"3.14"` ফেরত দেয়।

## List type

`[1, 2, 3]`-এর মতো উপাদানের একটি সাজানো লিস্টের সাথে ব্যবহার করা যায় এমন ফাংশন।

### ফিল্ড

| ফিল্ড         | টাইপ     | বর্ণনা                        |
| ------------- | -------- | ---------------------------------- |
| `list.length` | `number` | লিস্টের উপাদান সংখ্যা |

### `contains()`

`list.contains(value: any): boolean`

- `value` হলো খোঁজার জন্য উপাদান।
- লিস্টে `value` থাকলে true ফেরত দেয়।
- উদাহরণ: `[1,2,3].contains(2)` `true` ফেরত দেয়।

### `containsAll()`

`list.containsAll(...values: any): boolean`

- `values` হলো খোঁজার জন্য এক বা একাধিক উপাদান।
- লিস্টে সব `values` থাকলে true ফেরত দেয়।
- উদাহরণ: `[1,2,3].containsAll(2,3)` `true` ফেরত দেয়।

### `containsAny()`

`list.containsAny(...values: any): boolean`

- `values` হলো খোঁজার জন্য এক বা একাধিক উপাদান।
- লিস্টে `values`-এর মধ্যে অন্তত একটি থাকলে true ফেরত দেয়।
- উদাহরণ: `[1,2,3].containsAny(3,4)` `true` ফেরত দেয়।

### `filter()`

`list.filter(value: Boolean): list`

- লিস্ট ফিল্টার করে এবং শুধুমাত্র সেই উপাদান রাখে যেখানে এক্সপ্রেশনটি সত্য।
- `value` হলো লিস্টের একটি আইটেমের মান।
- `index` হলো বর্তমান মানের ইনডেক্স।
- উদাহরণ: `[1,2,3,4].filter(value > 2)` `[3,4]` ফেরত দেয়।

### `flat()`

`list.flat(): list`

- একটি নেস্টেড লিস্টকে একটি একক লিস্টে সমতল করে।
- উদাহরণ: `[1,[2,3]].flat()` `[1,2,3]` ফেরত দেয়।

### `isEmpty()`

`list.isEmpty(): boolean`

- লিস্টে কোনো উপাদান না থাকলে true ফেরত দেয়।
- উদাহরণ: `[1,2,3].isEmpty()` `false` ফেরত দেয়।

### `join()`

`list.join(separator: string): string`

- `separator` হলো উপাদানগুলোর মাঝে যুক্ত করার স্ট্রিং।
- সব লিস্ট উপাদানকে একটি একক স্ট্রিংয়ে যুক্ত করে।
- উদাহরণ: `[1,2,3].join(",")` `"1,2,3"` ফেরত দেয়।

### `map()`

`list.map(value: Any): list`

- একটি এক্সপ্রেশন ব্যবহার করে লিস্টের প্রতিটি উপাদান রূপান্তর করে।
- `value` হলো লিস্টের একটি আইটেমের মান।
- `index` হলো বর্তমান মানের ইনডেক্স।
- উদাহরণ: `[1,2,3,4].map(value + 1)` `[2,3,4,5]` ফেরত দেয়।

### `reduce()`

`list.reduce(expression: Any, acc: Any): Any`

- প্রতিটি উপাদানের জন্য একটি এক্সপ্রেশন চালিয়ে লিস্টটিকে একটি একক মানে হ্রাস করে। এক্সপ্রেশনটি অবশ্যই `acc`-এর পরবর্তী মান ফেরত দিতে হবে। বর্তমান উপাদানের জন্য `value`, এর অবস্থানের জন্য `index`, এবং এখন পর্যন্ত সঞ্চিত ফলাফলের জন্য `acc` ব্যবহার করুন।
- `expression` লিস্টের প্রতিটি উপাদানের জন্য মূল্যায়িত হয়।
- `value` হলো লিস্টের বর্তমান আইটেমের মান।
- `index` হলো বর্তমান আইটেমের ইনডেক্স।
- `acc` হলো এখন পর্যন্ত সঞ্চিত মান।
- উদাহরণ (যোগফল): `[1,2,3].reduce(acc + value, 0)` `6` ফেরত দেয়।
- উদাহরণ (সর্বোচ্চ): `values.filter(value.isType("number")).reduce(if(acc == null || value > acc, value, acc), null)` সবচেয়ে বড় সংখ্যা, অথবা কোনোটি না থাকলে `null` ফেরত দেয়।

### `reverse()`

`list.reverse(): list`

- লিস্টটি জায়গাতেই বিপরীত করে।
- উদাহরণ: `[1,2,3].reverse()` `[3,2,1]` ফেরত দেয়।

### `slice()`

`list.slice(start: number, end?: number): list`

- `start` হলো ইনক্লুসিভ শুরুর ইনডেক্স।
- `end` হলো ঐচ্ছিক এক্সক্লুসিভ শেষের ইনডেক্স।
- `start` (ইনক্লুসিভ) থেকে `end` (এক্সক্লুসিভ) পর্যন্ত লিস্টের একটি অংশের একটি শ্যালো কপি ফেরত দেয়।
- উদাহরণ: `[1,2,3,4].slice(1,3)` `[2,3]` ফেরত দেয়।
- `end` বাদ দিলে, লিস্টের শেষ পর্যন্ত স্লাইস করে।

### `sort()`

`list.sort(): list`

- লিস্টের উপাদান ছোট থেকে বড় ক্রমে সাজায়।
- উদাহরণ: `[3, 1, 2].sort()` `[1, 2, 3]` ফেরত দেয়।
- উদাহরণ: `["c", "a", "b"].sort()` `["a", "b", "c"]` ফেরত দেয়।

### `unique()`

`list.unique(): list`

- ডুপ্লিকেট উপাদান সরিয়ে দেয়।
- উদাহরণ: `[1,2,2,3].unique()` `[1,2,3]` ফেরত দেয়।

## Link type

একটি লিঙ্কের সাথে ব্যবহার করা যায় এমন ফাংশন। লিঙ্ক একটি ফাইল থেকে (`file.asLink()`) অথবা একটি পাথ থেকে (`link("path")`) তৈরি করা যায়।

### `asFile()`

`link.asFile(): file`

- লিঙ্কটি একটি বৈধ লোকাল ফাইল নির্দেশ করলে একটি file অবজেক্ট ফেরত দেয়।
- উদাহরণ: `link("[[filename]]").asFile()`।

### `linksTo()`

`link.linksTo(file): boolean`

- `link`-এর প্রতিনিধিত্বকারী ফাইলটির `file`-এ একটি লিঙ্ক থাকলে `true` ফেরত দেয়।

## File type

ভল্টের একটি ফাইলের সাথে ব্যবহার করা যায় এমন ফাংশন।

### ফিল্ড

ফাইলের জন্য নিম্নলিখিত ফিল্ডগুলো উপলব্ধ:

| ফিল্ড             | টাইপ     | বর্ণনা                                             |
| ----------------- | -------- | ------------------------------------------------------- |
| `file.name`       | `string` | এই ফাইলের নাম।                                  |
| `file.basename`   | `string` | ফাইল এক্সটেনশন ছাড়া এই ফাইলের নাম।       |
| `file.path`       | `string` | এই ফাইলের সম্পূর্ণ পাথ, ভল্ট রুটের তুলনায়। |
| `file.folder`     | `string` | প্যারেন্ট ফোল্ডারের সম্পূর্ণ পাথ।                     |
| `file.ext`        | `string` | এই ফাইলের ফাইল এক্সটেনশন।                       |
| `file.size`       | `number` | এই ফাইলের আকার, বাইটে।                        |
| `file.properties` | `object` | এই ফাইলের নোট প্রপার্টি।                      |
| `file.tags`       | `list`   | এই ফাইলের ট্যাগ। ইনলাইন ট্যাগসহ।           |
| `file.links`      | `list`   | এই ফাইলের মধ্যে থাকা ইন্টারনাল লিঙ্ক।                    |
| `file.ctime`      | `date`   | এই ফাইল কখন তৈরি হয়েছিল তার টাইমস্ট্যাম্প।                |
| `file.mtime`      | `date`   | এই ফাইল সর্বশেষ কখন পরিবর্তিত হয়েছিল তার টাইমস্ট্যাম্প।          |

### `asLink()`

`file.asLink(display?: string): Link`

- `display` হলো লিঙ্কের জন্য ঐচ্ছিক প্রদর্শনের টেক্সট।
- একটি Link অবজেক্ট ফেরত দেয় যা একটি কার্যকরী লিঙ্ক হিসেবে রেন্ডার হয়।
- উদাহরণ: `file.asLink()`

### `hasLink()`

`file.hasLink(otherFile: file | string): boolean`

- `otherFile` হলো যাচাই করার জন্য অন্য একটি file অবজেক্ট বা স্ট্রিং পাথ।
- `file` থেকে `otherFile`-এ একটি লিঙ্ক থাকলে true ফেরত দেয়।
- উদাহরণ: `file.hasLink(otherFile)` `file` থেকে `otherFile`-এ একটি লিঙ্ক থাকলে `true` ফেরত দেয়।

### `hasProperty()`

`file.hasProperty(name: string): boolean`

- ফাইলে প্রদত্ত প্রপার্টি থাকলে `true` ফেরত দেয়।

### `hasTag()`

`file.hasTag(...values: string): boolean`

- `values` হলো এক বা একাধিক ট্যাগের নাম।
- `values`-এর যেকোনো ট্যাগ ফাইলে থাকলে true ফেরত দেয়।
- উদাহরণ: `file.hasTag("tag1", "tag2")` ফাইলে `#tag1` বা `#tag2` ট্যাগ থাকলে `true` ফেরত দেয়। এতে যেকোনো [[ট্যাগ#Nested tags|নেস্টেড ট্যাগও]] অন্তর্ভুক্ত, যেমন `#tag1/a` বা `#tag2/b`।

### `inFolder()`

`file.inFolder(folder: string): boolean`

- `folder` হলো যাচাই করার ফোল্ডারের নাম।
- ফাইলটি নির্দিষ্ট ফোল্ডারে বা এর একটি সাব-ফোল্ডারে থাকলে true ফেরত দেয়।
- উদাহরণ: `file.inFolder("notes")` `true` ফেরত দেয়।

## Object type

`{"a": 1, "b": 2}`-এর মতো কী-ভ্যালু জোড়ার একটি সংকলনের সাথে ব্যবহার করা যায় এমন ফাংশন।

### `isEmpty()`

`object.isEmpty(): boolean`

- অবজেক্টে কোনো নিজস্ব প্রপার্টি না থাকলে true ফেরত দেয়।
- উদাহরণ: `{}.isEmpty()` `true` ফেরত দেয়।

### `keys()`

`object.keys(): list`

- অবজেক্টের কী-গুলোর একটি লিস্ট ফেরত দেয়।

### `values()`

`object.values(): list`

- অবজেক্টের মানগুলোর একটি লিস্ট ফেরত দেয়।

## Regular expression type

একটি রেগুলার এক্সপ্রেশন প্যাটার্নের সাথে ব্যবহার করা যায় এমন ফাংশন। উদাহরণ: `/abc/`।

### `matches()`

`regexp.matches(value: string): boolean`

- `value` হলো যাচাই করার স্ট্রিং।
- রেগুলার এক্সপ্রেশনটি `value`-এর সাথে মিলে গেলে true ফেরত দেয়।
- উদাহরণ: `/abc/.matches("abcde")` `true` ফেরত দেয়।
