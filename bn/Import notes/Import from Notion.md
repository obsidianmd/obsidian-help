Obsidian আপনাকে [[Importer|Importer প্লাগইন]] ব্যবহার করে সহজেই Notion থেকে আপনার নোট মাইগ্রেট করতে দেয়। এটি আপনার Notion ডেটাকে টেকসই Markdown ফাইলে রূপান্তরিত করবে যা আপনি অফলাইনে Obsidian এবং আরও অনেক অ্যাপের সাথে ব্যবহার করতে পারবেন।

Obsidian আপনার Notion ডেটা ইম্পোর্ট করার দুটি উপায় প্রদান করে:

1. **API import** আপনার সম্পূর্ণ ওয়ার্কস্পেস সংরক্ষণ করে, যার মধ্যে রয়েছে ডেটাবেস এবং ফর্মুলা যা [[Introduction to Bases|Bases]]-এ রূপান্তরিত হয়, তবে এর জন্য একটি Notion ইন্টিগ্রেশন টোকেন এবং ইন্টারনেট সংযোগ প্রয়োজন।
2. **File import** ডেটাবেস সংরক্ষণ করে না তবে API টোকেন বা ইন্টারনেট সংযোগের প্রয়োজন হয় না।

## API import

### একটি Notion API ইন্টিগ্রেশন টোকেন তৈরি করুন

API-এর মাধ্যমে আপনার Notion ডেটা অ্যাক্সেস করতে আপনার একটি ইন্টিগ্রেশন টোকেন প্রয়োজন। এই ধাপটি সম্পূর্ণ করতে প্রায় ২ মিনিট সময় লাগে।

টোকেনটি সংখ্যা এবং অক্ষরের একটি দীর্ঘ স্ট্রিং যা সাধারণত `ntn_...` দিয়ে শুরু হয়, যা আপনাকে Notion থেকে আপনার ডেটা ডাউনলোড করার অনুমতি দেবে।

1. আপনার [Notion Integrations](https://www.notion.so/profile/integrations/internal) ড্যাশবোর্ডে সাইন ইন করুন।
2. **New integration** নির্বাচন করুন।

![[notion-integration.png#interface]]

2. আপনার ইন্টিগ্রেশনকে একটি নাম দিন, যেমন "Personal"। যেকোনো নাম ব্যবহার করা যেতে পারে।
3. আপনি যে ওয়ার্কস্পেস এক্সপোর্ট করতে চান তা নির্বাচন করুন।
4. **Save**-এ ক্লিক করুন এবং **Configure integration settings**-এ চালিয়ে যান।
5. **Configuration** ট্যাবে আপনার API টোকেন **Internal Integration Secret** ফিল্ডে অ্যাক্সেসযোগ্য। 
6. **Show** তারপর **Copy** নির্বাচন করুন।
7. টোকেনটি আপনার পাসওয়ার্ড ম্যানেজারের মতো নিরাপদ কোনো জায়গায় সংরক্ষণ করুন।

![[notion-token.png#interface]]

এরপর, আপনি যে Notion পেজ এবং ডেটাবেস ইম্পোর্ট করতে চান তাতে আপনার ইন্টিগ্রেশনকে অ্যাক্সেস দিন।

1. আপনার সবেমাত্র তৈরি করা ইন্টিগ্রেশনের **Access** ট্যাবে যান।
2. **Edit access**-এ ক্লিক করুন।
3. আপনি যে পেজ এবং ডেটাবেস ইম্পোর্ট করতে চান তা যোগ করুন।

এখন আপনি Obsidian Importer ব্যবহার করে আপনার ডেটা রূপান্তর করতে পারেন।

### API-এর মাধ্যমে আপনার Notion ডেটা ইম্পোর্ট করুন

আপনার অফিসিয়াল Obsidian [[Importer]] প্লাগইন প্রয়োজন হবে, যা আপনি [এখানে ইনস্টল করতে পারেন](obsidian://show-plugin?id=obsidian-importer)।

1. **[[Settings]]** খুলুন।
2. **Community Plugins**-এ যান এবং [Importer ইনস্টল করুন](obsidian://show-plugin?id=obsidian-importer)।
3. Importer প্লাগইন চালু করুন।
4. কমান্ড প্যালেট বা রিবন আইকন ব্যবহার করে **Importer** প্লাগইন খুলুন।
5. **File format**-এর অধীনে **Notion (API)** নির্বাচন করুন
6. **API token**-এর অধীনে, Notion থেকে আপনার **Internal Integration Secret** পেস্ট করুন।
7. আপনি যে ডেটাবেস এবং পেজ ইম্পোর্ট করতে চান তা নির্বাচন করতে **Load**-এ ক্লিক করুন।
8. ইম্পোর্ট বিকল্পগুলি পর্যালোচনা ও সম্পাদনা করুন।
9. **Import** নির্বাচন করুন এবং ইম্পোর্ট সম্পূর্ণ না হওয়া পর্যন্ত অপেক্ষা করুন
10. আপনার কাজ শেষ!

### সীমাবদ্ধতা

> [!info] API import নতুন
> Notion API ইম্পোর্টার নতুন। Notion ওয়ার্কস্পেসের জটিলতার কারণে, কিছু প্রান্তিক ক্ষেত্র বিবেচনা করা নাও হতে পারে। রূপান্তরে সমস্যা হলে, [একটি বাগ রিপোর্ট জমা দিন](https://github.com/obsidianmd/obsidian-importer/issues) যাতে আমরা এটি উন্নত করতে পারি। 

Notion API রেট সীমার কারণে, বড় ওয়ার্কস্পেস ইম্পোর্ট করতে যথেষ্ট সময় লাগতে পারে। অনুগ্রহ করে ধৈর্য ধরুন।

Notion API-এর সীমাবদ্ধতার কারণে কিছু ডেটা পাওয়া যায় না বা রূপান্তরিত করা যায় না:

- প্রতিটি ডেটাবেসের শুধুমাত্র প্রাথমিক ভিউ ইম্পোর্ট করা হয়।
- [লিঙ্কড ডেটা সোর্স](https://developers.notion.com/docs/working-with-databases#additional-types-of-databases) ইম্পোর্ট করা হয় না: *"Notion's API does not currently support linked data sources. When sharing a database with your integration, make sure it contains the original data source!"*
- `People` ফাংশন: `name()` এবং `email()`
- `Text` ফাংশন: `style()` এবং `unstyle()`

এছাড়াও, Importer নিচের পরিবর্তনগুলি করবে:

- সাব-পেজ বা ডেটাবেস ছাড়া পেজগুলি `[filename]/[filename].md`-এর পরিবর্তে `[filename].md` হিসেবে ইম্পোর্ট করা হবে।
- ডেটাবেসগুলি সবসময় `[database name]` নামের ফোল্ডার হিসেবে উপস্থাপিত হয় যার ভেতরে একটি `[database name].base` ফাইল থাকে।

## File import

File import হলো আপনার Notion ডেটা ইম্পোর্ট করার একটি বিকল্প উপায়। এই পদ্ধতি ডেটাবেস সংরক্ষণ করে না তবে API টোকেন বা ইন্টারনেট সংযোগের প্রয়োজন হয় না।

### Notion থেকে আপনার ডেটা এক্সপোর্ট করুন

ইম্পোর্টের জন্য আপনার ডেটা প্রস্তুত করতে, আপনাকে Notion-এর HTML এক্সপোর্ট ফরম্যাট ব্যবহার করে আপনার সম্পূর্ণ ওয়ার্কস্পেস এক্সপোর্ট করতে হবে। আমরা সুপারিশ করি যে আপনি Notion-এর Markdown এক্সপোর্ট ব্যবহার করবেন না কারণ এটি গুরুত্বপূর্ণ ডেটা বাদ দেয়। সম্পূর্ণ ওয়ার্কস্পেস কনটেন্ট এক্সপোর্ট করতে আপনার অবশ্যই Notion ওয়ার্কস্পেসে অ্যাডমিন অ্যাক্সেস থাকতে হবে।

1. Notion সাইডবারের শীর্ষে থাকা **[[Settings]]**-এ যান।
2. **Workspace**-এর অধীনে **General** নির্বাচন করুন।
3. **Export all workspace content** খুঁজে বের করে নির্বাচন করুন।
4. **Export format**-এর অধীনে **HTML** নির্বাচন করুন।
5. **Include everything** নির্বাচন করুন।
6. **Create folders for subpages** চালু করুন।
7. আপনি ইমেইলের মাধ্যমে বা সরাসরি ব্রাউজারে একটি `.zip` ফাইল পাবেন।

![[notion-export.png#interface]]

![[notion-export-2.png#interface]]

### আপনার Notion .zip ফাইল ইম্পোর্ট করুন

আপনার অফিসিয়াল Obsidian [[Importer]] প্লাগইন প্রয়োজন হবে, যা আপনি [এখানে ইনস্টল করতে পারেন](obsidian://show-plugin?id=obsidian-importer)।

1. **[[Settings]]** খুলুন।
2. **Community Plugins**-এ যান এবং [Importer ইনস্টল করুন](obsidian://show-plugin?id=obsidian-importer)।
3. Importer প্লাগইন চালু করুন।
4. কমান্ড প্যালেট বা রিবন আইকন ব্যবহার করে **Importer** প্লাগইন খুলুন।
5. **File format**-এর অধীনে **Notion (.zip)** নির্বাচন করুন
6. আপনি যে Notion ফাইলগুলি ইম্পোর্ট করতে চান তার `.zip` ফাইলটি বেছে নিন। *আপনার সমস্ত Notion একসাথে ইম্পোর্ট করার পরামর্শ দেওয়া হয় যাতে অভ্যন্তরীণ লিঙ্কগুলি সঠিকভাবে সংশোধন করা যায়।*
7. _ঐচ্ছিকভাবে_, ইম্পোর্টের জন্য একটি ফোল্ডার নির্বাচন করুন। আপনার Notion পেজ এবং ডেটাবেস এই ফোল্ডারের ভেতরে নেস্ট করা হবে।
8. Notion কাঠামো বজায় রাখতে **Save parent pages in subfolders** চালু করুন। *মনে রাখবেন Notion-এ আপনি ফোল্ডারে কনটেন্ট লিখতে পারেন, Obsidian-এ এটি সম্ভব নয় এবং এই পেজগুলি ফোল্ডারের অধীনে একটি সাব-পেজ হিসেবে যোগ করা হবে।*
9. **Import** নির্বাচন করুন এবং ইম্পোর্ট সম্পূর্ণ না হওয়া পর্যন্ত অপেক্ষা করুন
10. আপনার কাজ শেষ!

### সমস্যা সমাধান

Notion থেকে ইম্পোর্ট করার সময় আপনার যদি কোনো সমস্যা হয়:

- নিশ্চিত করুন যে আপনি Notion-এ এক্সপোর্ট ফরম্যাট হিসেবে **HTML** ব্যবহার করছেন, **Markdown নয়**।
- ইম্পোর্টের সময় Obsidian যদি স্থবির হয়ে যায় বলে মনে হয়, কমিউনিটি প্লাগইন বন্ধ করে আবার চেষ্টা করুন।

অন্য কোনো সমস্যায় পড়েছেন? অন্যরা এটির সম্মুখীন হয়েছেন কিনা তা দেখতে [Importer রিপোজিটরি](https://github.com/obsidianmd/obsidian-importer/issues) সার্চ করুন।

#### বড় ওয়ার্কস্পেস ইম্পোর্ট করা

আপনি যদি একাধিক গিগাবাইট ডেটা সহ একটি ওয়ার্কস্পেস ইম্পোর্ট করছেন, তাহলে Notion থেকে এক্সপোর্টে নেস্টেড `.zip` ফাইল থাকতে পারে। এই ক্ষেত্রে আপনি এই রকম একটি ইম্পোর্ট এরর মেসেজ দেখতে পারেন:

```
Import failed {id}.zip/{id}-Part-1.zip undefined.
```

আপনি যদি এই এরর দেখতে পান, তাহলে আপনি Notion থেকে ফাইলটি আনজিপ করতে পারেন, এবং তারপর নেস্টেড `Export-{id}-Part-1.zip` ফাইলগুলি ইম্পোর্ট করতে পারেন।
