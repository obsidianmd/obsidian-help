Obsidian নোটের প্রধান ফরম্যাট হিসেবে Markdown `.md` ফাইল ব্যবহার করে। এটি Obsidian-এ Markdown ফাইল ইম্পোর্ট করা সহজ করে তোলে। অনেক অ্যাপ Markdown ফাইল সম্পাদনা করতে পারে বা এই ফরম্যাটে এক্সপোর্ট করতে পারে।

আপনার Obsidian ভল্টে Markdown ফাইল যোগ করার বেশ কয়েকটি উপায় রয়েছে:

1. [[File explorer]] উইন্ডোতে ফাইল বা ফোল্ডার টেনে এনে ছেড়ে দিন
2. আপনার সিস্টেম ফাইল ব্রাউজার ব্যবহার করুন (যেমন Windows-এ Windows Explorer বা macOS-এ Finder), এবং আপনার ফাইলগুলি সরাসরি Obsidian ভল্ট ফোল্ডারে সরিয়ে নিন।

### Markdown-এর অন্যান্য ফ্লেভার থেকে রূপান্তর করা

Obsidian [[Obsidian Flavored Markdown]] ব্যবহার করে। কিছু অ্যাপ সামান্য ভিন্ন Markdown সিনট্যাক্স ব্যবহার করে। আপনি আপনার ফাইলগুলি Obsidian-এর Markdown সিনট্যাক্সে রূপান্তর করতে [[Format converter]] ব্যবহার করতে পারেন।
