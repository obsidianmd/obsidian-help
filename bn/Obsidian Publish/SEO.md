[[Introduction to Obsidian Publish|Obsidian Publish]] সাইটগুলো স্বয়ংক্রিয়ভাবে সার্চ ইঞ্জিন এবং [[Social media link previews|সোশ্যাল শেয়ারিং কার্ড]]-এর জন্য অপ্টিমাইজ করা থাকে। আপনি বর্ণনা, পার্মালিঙ্ক, এবং ছবি দিয়ে পৃষ্ঠার মেটাডেটাও কাস্টমাইজ করতে পারেন।

লক্ষ্য করুন যে আপনি আপনার [[Manage sites#Site Options|সাইট অপশনে]] সার্চ ইঞ্জিন ইনডেক্সিং নিষিদ্ধ করতে পারেন।

### Google Search Console

Google-এর সাথে SEO ট্র্যাক করতে, [Google Search Console](https://search.google.com/search-console)-এ আপনার সাইট নিবন্ধন করুন। আপনি আমাদের সুপারিশকৃত পদক্ষেপগুলো অনুসরণ করেও [[Analytics|Google Analytics সেট আপ]] করতে পারেন।

### সাইটম্যাপ

আপনার Publish সাইটের জন্য সাইটম্যাপ `/sitemap.xml`-এ উপলব্ধ, উদাহরণস্বরূপ:
https://help.obsidian.md/sitemap.xml

সাইটম্যাপ একটি RSS ফিড হিসেবেও `/rss.xml`-এ উপলব্ধ:
https://help.obsidian.md/rss.xml

### পার্মালিঙ্ক

আপনার সাইটে স্থায়ী URL নির্ধারণ করতে [[Permalinks]] ব্যবহার করার কথা বিবেচনা করুন।

### মেটাডেটা

পৃষ্ঠার মেটাডেটা [[Properties#Properties for Obsidian Publish|প্রপার্টি]] ব্যবহার করে কাস্টমাইজ করা যায়, যা [[Social media link previews|সোশ্যাল শেয়ারিং কার্ড]]-এর জন্য উপযোগী হতে পারে।

| প্রপার্টি      | বিবরণ                                                                                                       |
| ------------- | ----------------------------------------------------------------------------------------------------------------- |
| `publish`     | দেখুন [[Publish your content#Automatically select data to publish\|প্রকাশের জন্য স্বয়ংক্রিয়ভাবে ডেটা নির্বাচন করুন]]। |
| `permalink`   | দেখুন [[Permalinks\|পার্মালিঙ্ক]]।                                                                                   |
| `description` | দেখুন [[Social media link previews#Description\|বর্ণনা]]।                                                      |
| `image`       | দেখুন [[Social media link previews#Image\|ছবি]]।                                                                  |
| `cover`       | দেখুন [[Social media link previews#Image\|ছবি]]।                                                                  |
