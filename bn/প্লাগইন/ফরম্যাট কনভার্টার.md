Markdown ফরম্যাট ইম্পোর্টার একটি [[কোর প্লাগইন|কোর প্লাগইন]] যা আপনাকে অন্য অ্যাপ্লিকেশন থেকে Markdown-কে Obsidian ফরম্যাটে রূপান্তর করতে দেয়। এটি আপনাকে নির্দিষ্ট [[প্রপার্টি]]-কে নতুন প্রয়োজনীয় ফরম্যাটে রূপান্তর করতেও দেয়। 

> [!warning] সতর্কতা
> Markdown ফরম্যাট ইম্পোর্টার আপনার সেটিংসের উপর ভিত্তি করে আপনার সম্পূর্ণ ভল্ট রূপান্তর করে। রূপান্তর সম্পাদনের আগে [[আপনার Obsidian ফাইল ব্যাকআপ করুন]] করুন।

আপনার ভল্টের সব নোট রূপান্তর করতে:

1. [[কমান্ড প্যালেট]]-এ, **Open format converter** নির্বাচন করুন। এটি [[রিবন]]-এও **Open format converter** ![[lucide-binary.svg#icon]] আইকন দিয়ে পাওয়া যায়।
2. আপনি যে ফরম্যাটগুলি রূপান্তর করতে চান তা সক্রিয় করুন।
3. **Start conversion**-এ ক্লিক করুন।

আরও তথ্যের জন্য, [[মৌলিক ফরম্যাটিং সিনট্যাক্স]] দেখুন।

## সমর্থিত ফরম্যাট

### Roam Research

Markdown ফরম্যাট ইম্পোর্টার নিম্নলিখিত Roam Research সিনট্যাক্স রূপান্তর করতে পারে:

- **ট্যাগ**: `#tag` এবং `#[[tag]]`-কে `[[tag]]`-এ রূপান্তর করে
- **হাইলাইট**: `^^highlight^^`-কে `==highlight==`-এ রূপান্তর করে
- **TODO আইটেম**: `{{[[TODO]]}}`-কে `[ ]`-এ রূপান্তর করে

### Bear

Markdown ফরম্যাট ইম্পোর্টার নিম্নলিখিত Bear সিনট্যাক্স রূপান্তর করতে পারে:

- **হাইলাইট**: `::highlight::`-কে `==highlight==`-এ রূপান্তর করে

### Zettelkasten

Markdown ফরম্যাট ইম্পোর্টার নিম্নলিখিত Zettelkasten সিনট্যাক্স রূপান্তর করতে পারে:

- **সম্পূর্ণ লিঙ্ক**: `[[UID]]`-কে `[[UID File Name]]`-এ রূপান্তর করে
- **প্রেটি লিঙ্ক**: `[[UID]]`-কে `[[UID File Name|File Name]]`-এ রূপান্তর করে

### [[প্রপার্টি]]

Obsidian `1.9.3` অনুযায়ী, Markdown ফরম্যাট ইম্পোর্টার [[প্রপার্টি#ডেপ্রিকেটেড প্রপার্টি|ডেপ্রিকেটেড প্রপার্টি]] ফরম্যাটগুলিকে বর্তমান ফরম্যাটে রূপান্তর করতে পারে:

**Aliases**

```yaml
# Before

alias: My Note Title

# After

aliases:
  - My Note Title
```

**Tags**

```yaml
# Before

tag: project, important

# After

tags:
  - project
  - important
```

**CSS Classes**

```yaml
# Before

cssclass: custom-style

# After

cssclasses:
  - custom-style
```
