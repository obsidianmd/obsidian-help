این صفحه میانبرهای پیش‌فرض صفحه‌کلید برای ناوبری و ویرایش متن در Obsidian را فهرست می‌کند. این میانبرها توسط سیستم‌عامل یا فریم‌ورکی که Obsidian بر اساس آن ساخته شده ارائه می‌شوند و قابل سفارشی‌سازی در Obsidian نیستند.

برای میانبرهای صفحه‌کلید قابل سفارشی‌سازی مربوط به دستورات Obsidian، به [[میانبرها]] مراجعه کنید.

## میانبرهای ویندوز و لینوکس

### عملیات رایج

| عملیات | میانبر |
|-|-|
| کپی | `Ctrl+C` |
| برش | `Ctrl+X` |
| چسباندن | `Ctrl+V` |
| چسباندن بدون قالب‌بندی | `Ctrl+Shift+V` |
| واگرد | `Ctrl+Z` |
| انجام دوباره | `Ctrl+Shift+Z` یا `Ctrl+Y` |
| کپی پاراگراف | `Ctrl+C` (بدون متن انتخاب‌شده) |
| برش پاراگراف | `Ctrl+X` (بدون متن انتخاب‌شده) |

### ویرایش متن

| عملیات | میانبر |
|-|-|
| درج خط جدید | `Enter` |
| حذف کاراکتر قبلی | `Backspace` |
| حذف کاراکتر بعدی | `Delete` |
| حذف کلمه قبلی | `Ctrl+Backspace` |
| حذف کلمه بعدی | `Ctrl+Delete` |
| حذف خط فعلی | `Ctrl+Shift+K` (بدون متن انتخاب‌شده) |

### ناوبری متن

| عملیات | میانبر |
|-|-|
| جابجایی نشانگر یک کاراکتر | `Left/→` |
| جابجایی نشانگر به ابتدای کلمه قبلی | `Ctrl+←` |
| جابجایی نشانگر به انتهای کلمه بعدی | `Ctrl+→` |
| جابجایی نشانگر به ابتدای خط فعلی | `Home` |
| جابجایی نشانگر به انتهای خط فعلی | `End` |
| جابجایی نشانگر به خط قبلی | `↑` |
| جابجایی نشانگر به خط بعدی | `↓` |
| جابجایی نشانگر به ابتدای یادداشت | `Ctrl+Home` |
| جابجایی نشانگر به انتهای یادداشت | `Ctrl+End` |
| جابجایی نشانگر یک صفحه بالا | `Page up` |
| جابجایی نشانگر یک صفحه پایین | `Page down` |

### انتخاب متن

| عملیات | میانبر |
|-|-|
| ساده‌سازی انتخاب | `Escape` |
| انتخاب همه | `Ctrl+A` |
| گسترش انتخاب یک کاراکتر | `Shift+Left/→` |
| گسترش انتخاب تا ابتدای کلمه قبلی | `Ctrl+Shift+←` |
| گسترش انتخاب تا انتهای کلمه بعدی | `Ctrl+Shift+→` |
| گسترش انتخاب تا ابتدای خط فعلی | `Shift+Home` |
| گسترش انتخاب تا انتهای خط فعلی | `Shift+End` |
| گسترش انتخاب تا ابتدای یادداشت | `Ctrl+Shift+Home` |
| گسترش انتخاب تا انتهای یادداشت | `Ctrl+Shift+End` |
| گسترش انتخاب یک صفحه بالا | `Shift+Page up` |
| گسترش انتخاب یک صفحه پایین | `Shift+Page down` |
