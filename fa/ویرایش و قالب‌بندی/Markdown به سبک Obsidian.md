Obsidian تلاش می‌کند حداکثر قابلیت را بدون شکستن هیچ‌یک از قالب‌بندی‌های موجود ارائه دهد. به همین دلیل، ما از ترکیبی از گونه‌های مختلف [[سینتکس قالب‌بندی پایه|Markdown]] استفاده می‌کنیم.

Obsidian از [CommonMark](https://commonmark.org/)، [GitHub Flavored Markdown](https://github.github.com/gfm/) و [LaTeX](https://www.latex-project.org/) پشتیبانی می‌کند.

> [!tip]- Markdown درون HTML
> Obsidian سینتکس Markdown را درون عناصر HTML رندر نمی‌کند. این یک تصمیم طراحی آگاهانه برای بهینه‌سازی عملکرد و کاهش پیچیدگی تجزیه‌گر هنگام مدیریت اسناد بزرگ است.
>
> به عنوان نمونه، قالب‌بندی‌های Markdown مانند `**bold**` یا `` `code` `` درون `<div>`، `<span>`، `<table>` یا هر تگ HTML دیگری پردازش نخواهند شد.
>
> ```md
> <div>
> This **will not** be bold.
> </div>
> ```

### افزونه‌های Markdown پشتیبانی‌شده

| سینتکس          | توضیحات                                                               |
| --------------- | --------------------------------------------------------------------- |
| `[[Link]]`      | [[پیوندهای داخلی]]                                                    |
| `![[Link]]`     | [[جاسازی فایل‌ها]]                                                    |
| `![[Link#^id]]` | [[پیوندهای داخلی#پیوند به یک بلوک در یادداشت\|ارجاع‌های بلوکی]]       |
| `^id`           | [[پیوندهای داخلی#پیوند به یک بلوک در یادداشت\|تعریف یک بلوک]]        |
| `[^id]`         | [[سینتکس قالب‌بندی پایه#پانویس‌ها\|پانویس‌ها]]                        |
| `%%Text%%`      | [[سینتکس قالب‌بندی پایه#یادداشت‌ها\|یادداشت‌ها]]                      |
| `~~Text~~`      | [[سینتکس قالب‌بندی پایه#پررنگ، کج‌نویس، برجسته‌سازی\|خط‌خورده]]       |
| `==Text==`      | [[سینتکس قالب‌بندی پایه#پررنگ، کج‌نویس، برجسته‌سازی\|برجسته‌سازی]]    |
| `` ``` ``       | [[سینتکس قالب‌بندی پایه#بلوک‌های کد\|بلوک‌های کد]]                    |
| `- [ ]`         | [[سینتکس قالب‌بندی پایه#فهرست کارها\|کار ناتمام]]                     |
| `- [x]`         | [[سینتکس قالب‌بندی پایه#فهرست کارها\|کار تکمیل‌شده]]                  |
| `> [!note]`     | [[فراخوان‌ها]]                                                         |
