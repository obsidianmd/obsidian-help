Obsidian از فایل‌های Markdown با پسوند `.md` به‌عنوان قالب اصلی یادداشت‌ها استفاده می‌کند. این کار درج فایل‌های Markdown به Obsidian را آسان می‌سازد. بسیاری از اپلیکیشن‌ها قادر به ویرایش فایل‌های Markdown یا استخراج به این قالب هستند.

چندین روش برای افزودن فایل‌های Markdown به گاوصندوق Obsidian وجود دارد:

1. فایل‌ها یا پوشه‌ها را به پنجره [[مرورگر فایل]] بکشید و رها کنید
2. از مرورگر فایل سیستم خود (مانند Windows Explorer یا Finder در macOS) استفاده کنید و فایل‌هایتان را مستقیماً به پوشه‌ی اصلی خزانه Obsidian منتقل کنید.

### تبدیل از سایر انواع Markdown

Obsidian از [[Obsidian Flavored Markdown]] استفاده می‌کند. برخی اپلیکیشن‌ها از سینتکس Markdown کمی متفاوت استفاده می‌کنند. می‌توانید از [[واردکننده‌ی قالب مارک‌داون]] برای تبدیل فایل‌هایتان به سینتکس Markdown مورد استفاده Obsidian بهره بگیرید.
