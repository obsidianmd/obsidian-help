پس از نصب افزونه مرورگر [[معرفی Obsidian Web Clipper|Web Clipper]]، بسته به مرورگر خود می‌توانید به چند روش به آن دسترسی داشته باشید:

1. آیکون Obsidian در نوار ابزار مرورگر شما.
2. میانبرهای صفحه‌کلید، برای فعال‌سازی افزونه از طریق صفحه‌کلید.
3. منوی زمینه، با کلیک راست روی صفحه وبی که در حال بازدید آن هستید.

برای ذخیره یک صفحه در Obsidian، روی دکمه **به ابسیدین اضافه کنید** کلیک کنید.

## ضبط یک صفحه

وقتی افزونه را باز می‌کنید، Web Clipper داده‌ها را از صفحه وب فعلی بر اساس تنظیمات [[الگوها|قالب]] شما استخراج می‌کند. شما می‌توانید قالب‌های خود را بسازید و خروجی را با استفاده از [[متغیرها]] و [[پالایه‌ها]] سفارشی کنید.

به طور پیش‌فرض Web Clipper تلاش می‌کند تنها محتوای اصلی مقاله را به صورت هوشمند استخراج کند و سایر عناصر صفحه را حذف نماید. با این حال، می‌توانید این رفتار را به روش‌های زیر تغییر دهید:

- اگر یک قالب سفارشی وجود داشته باشد، از قالب شما استفاده می‌کند.
- اگر متنی انتخاب شده باشد، از آن انتخاب استفاده می‌کند. می‌توانید از `Ctrl/Cmd+A` برای انتخاب کل صفحه استفاده کنید.
- اگر [[مارکر|هایلایت‌هایی]] وجود داشته باشد، از هایلایت‌ها استفاده می‌کند.

## بارگیری تصاویر

تصاویر هنگام استفاده از Web Clipper به صورت خودکار بارگیری نمی‌شوند. در عوض، تصاویر به URL وب‌محور خود پیوند می‌دهند. این کار فضای گاوصندوق شما را صرفه‌جویی می‌کند، اما به این معنی است که تصاویر به صورت آفلاین یا در صورت از کار افتادن URL قابل دسترسی نخواهند بود.

شما می‌توانید تصاویر هر فایلی را در Obsidian با استفاده از [[فرمان‌دان|دستور]] به نام **بارگیری پیوست‌ها برای فایل فعلی** بارگیری کنید. این دستور همچنین می‌تواند در Obsidian به یک میانبر صفحه‌کلید اختصاص داده شود.

## میانبرها

Web Clipper شامل میانبرهای صفحه‌کلیدی است که می‌توانید برای سرعت بخشیدن به جریان کار خود از آن‌ها استفاده کنید. برای تغییر نگاشت کلیدها به **تنظیمات Web Clipper** ← **عمومی** بروید و دستورالعمل‌های مرورگر خود را دنبال کنید. نگاشت‌ها برای همه مرورگرها قابل تغییر هستند به جز Safari که از ویرایش میانبرها پشتیبانی نمی‌کند.

| عملکرد                  | macOS         | Windows/Linux  |
| ----------------------- | ------------- | -------------- |
| گشودن کلیپر            | `Cmd+Shift+O` | `Ctrl+Shift+O` |
| کلیپ سریع              | `Opt+Shift+O` | `Alt+Shift+O`  |
| فعال/غیرفعال‌سازی مارکر | `Opt+Shift+H` | `Alt+Shift+H`  |
| فعال/غیرفعال‌سازی خواننده | `Opt+Shift+R` | `Alt+Shift+R`  |

## قابلیت‌های رابط کاربری

رابط کاربری Web Clipper به چهار بخش تقسیم شده است:

1. **سربرگ** جایی که می‌توانید بین قالب‌ها جابجا شوید، [[مارکر|هایلایت]]، [[خواننده|حالت خواندن]] را فعال کنید و به تنظیمات دسترسی داشته باشید.
2. **ویژگی‌ها** [[ویژگی‌ها|فراداده]] استخراج‌شده از صفحه را نشان می‌دهد که به عنوان [[ویژگی‌ها]] در Obsidian ذخیره خواهد شد.
3. **محتوای یادداشت** که در Obsidian ذخیره خواهد شد.
4. **پاورقی** به شما اجازه می‌دهد گاوصندوق و پوشه را انتخاب کنید و به Obsidian اضافه نمایید.

قابلیت‌های سربرگ شامل:

- ![[lucide-chevrons-up-down.svg#icon]] سوئیچ **قالب** برای استفاده از [[الگوها|قالب‌های]] ذخیره‌شده که در تنظیمات افزونه اضافه شده‌اند.
- ![[lucide-more-horizontal.svg#icon]] دکمه **بیشتر** برای نمایش متغیرهای صفحه که می‌توانید در قالب‌ها استفاده کنید.
- ![[lucide-highlighter.svg#icon]] دکمه **مارکر** برای فعال‌سازی [[مارکر|هایلایت]].
- ![[lucide-book-icon.svg#icon]] دکمه **خواننده** برای فعال‌سازی [[خواننده|حالت خواندن]].
- ![[lucide-picture-in-picture-2.svg#icon]] دکمه **جاسازی** برای انتقال Web Clipper از پنجره بازشو به داخل صفحه.
- ![[lucide-settings.svg#icon]] دکمه **تنظیمات** برای گشودن تنظیمات افزونه.

قابلیت‌های پاورقی شامل:

- دکمه **به ابسیدین اضافه کنید** برای ذخیره داده‌ها در Obsidian.
- منوی کشویی **گاوصندوق** برای جابجایی بین گاوصندوق‌های ذخیره‌شده که در تنظیمات Web Clipper اضافه شده‌اند.
- فیلد **پوشه** برای تعیین پوشه‌ای که فایل در آن ذخیره شود.
- **تفسیرگر** برای اجرای [[تفسیرگر|دستورات زبان طبیعی]] روی صفحه.
