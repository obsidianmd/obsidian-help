Obsidian Web Clipper یک افزونه رایگان مرورگر است که به شما امکان می‌دهد صفحات را برجسته‌سازی کنید و محتوای وب را در گاوصندوق خود ذخیره کنید.

## نصب Web Clipper

افزونه Obsidian Web Clipper را از فهرست رسمی مرورگر خود دریافت کنید:

- [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
	- برای Chrome، Brave، Arc، Orion و سایر مرورگرهای مبتنی بر Chromium.
- [Firefox Add-Ons](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
	- برای Firefox و Firefox Mobile.
- [Safari Extensions](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
	- برای macOS، iOS و iPadOS.
- [Edge Add-Ons](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
	- برای Microsoft Edge.

## نحوه استفاده از Web Clipper

- [[برش صفحات وب]]
	- شروع کنید و یاد بگیرید چگونه محتوا را از صفحات ذخیره کنید.
- [[هایلایت صفحات وب|مارکر]]
	- بخش‌های مهم را برجسته‌سازی کنید و عناصری که می‌خواهید ذخیره کنید را انتخاب نمایید.
- [[خواننده|خواننده]]
	- صفحات وب را در نمایی تمیز و بدون حواس‌پرتی بخوانید.
- [[تفسیر صفحات وب|تفسیرگر]]
	- از دستورات زبان طبیعی برای گرفتن و تغییر داده‌ها در یک صفحه استفاده کنید.
- [[عیب‌یابی Web Clipper|عیب‌یابی]]
	- مشکلات رایج Web Clipper را برطرف کنید.
- [[Obsidian Web Clipper/الگوها|الگوها]]
	- الگوهای سفارشی برای وبگاه‌های خاص بسازید.
- [[متغیرها]]
	- داده‌های صفحه را در الگوی خود پیش‌بارگذاری کنید.
- [[پالایه‌ها]]
	- خروجی متغیرهای الگو را تغییر دهید.
- [[منطق]]
	- از شرط‌ها، حلقه‌ها و تعریف متغیرها در الگوها استفاده کنید.

## حریم خصوصی

Obsidian Web Clipper محتوا را به صورت محلی در گاوصندوق Obsidian شما ذخیره می‌کند و از [سیاست حریم خصوصی](https://obsidian.md/privacy) ما پیروی می‌کند. داده‌های شما جمع‌آوری نمی‌شود و ما هیچ معیار استفاده‌ای را گردآوری نمی‌کنیم. کد برنامه [متن‌باز](https://github.com/obsidianmd/obsidian-clipper) و قابل بازبینی است.
