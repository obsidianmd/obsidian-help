[[معرفی Obsidian Web Clipper|Web Clipper]] شامل یک حالت خواندن داخلی به نام **خواننده** است که شلوغی‌های صفحات وب را حذف کرده و تنها محتوای اصلی مقاله را در قالبی تمیز و خوانا ارائه می‌دهد.

## فعال کردن حالت خواندن

می‌توانید خواننده را به چند روش فعال کنید:

- ![[lucide-book-icon.svg#icon]] **آیکون کتاب** در پنل افزونه.
- **میانبر صفحه‌کلید** قابل سفارشی‌سازی در تنظیمات افزونه.
- **منوی زمینه** با کلیک راست روی صفحه وبی که بازدید می‌کنید.

همچنین می‌توانید دکمه افزونه ![[obsidian.svg#icon]] را طوری تنظیم کنید که خواننده را با یک کلیک باز کند. در تنظیمات Web Clipper به **General** ← **Behavior** ← **Open behavior** بروید و **Reader** را انتخاب کنید.

تمام گزینه‌های بالا همچنین به شما امکان می‌دهند از خواننده خارج شده و صفحه اصلی را بازیابی کنید.

## نحوه عملکرد

خواننده محتوای اصلی را از صفحه استخراج می‌کند. صفحه اصلی با یک نمای خواندن تمیز جایگزین می‌شود که شامل موارد زیر است:

- عنوان مقاله، نویسنده، تاریخ انتشار و دامنه منبع.
- محتوای اصلی مقاله، با حفظ تصاویر و قالب‌بندی.
- طرح کلی در نوار کناری برای مقالاتی با چندین سرفصل.
- برجسته‌سازی سینتکس برای بلوک‌های کد.
- پاپ‌آورهای پانوشت که با کلیک کردن به صورت درون‌خطی نمایش داده می‌شوند.

## سفارشی‌سازی نمای خواندن

تجربه خواننده خود را در تنظیمات افزونه سفارشی کنید. برخی گزینه‌ها همچنین در حالت خواننده با کلیک بر آیکون ![[lucide-case-sensitive.svg#icon]] در دسترس هستند.

| تنظیم              | توضیح                                    |
| ------------------ | ---------------------------------------- |
| قلم                | افزودن قلم‌های نصب‌شده روی سیستم شما.     |
| اندازه‌ی قلم       | تنظیم اندازه متن.                        |
| ارتفاع&nbsp;خط     | فاصله بین خطوط.                          |
| عرض&nbsp;خط        | حداکثر عرض ناحیه محتوا.                  |
| ظاهر               | انتخاب ظاهر روشن یا تاریک.              |
| پوسته              | انتخاب رنگ‌ها.                           |
| CSS سفارشی         | CSS اضافی اعمال‌شده بر صفحه.             |
