Obsidian používa Markdown súbory `.md` ako primárny formát pre poznámky. Vďaka tomu je jednoduché importovať Markdown súbory do Obsidian. Mnohé aplikácie dokážu upravovať Markdown súbory alebo exportovať do tohto formátu.

Existuje niekoľko spôsobov, ako pridať Markdown súbory do vášho trezoru Obsidian:

1. Presuňte súbory alebo priečinky do okna [[Prieskumník súborov|Prieskumníka súborov]]
2. Použite systémový prehliadač súborov (napr. Windows Explorer alebo Finder v macOS) a presuňte súbory priamo do priečinka trezoru Obsidian.

### Konverzia z iných variantov Markdown

Obsidian používa [[Obsidian Flavored Markdown]]. Niektoré aplikácie používajú mierne odlišnú syntax Markdown. Môžete použiť [[Prevodník formátov]] na konverziu vašich súborov do syntaxe Markdown v Obsidian.
