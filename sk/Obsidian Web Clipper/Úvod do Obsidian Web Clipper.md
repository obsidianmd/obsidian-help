Obsidian Web Clipper je bezplatné rozšírenie prehliadača, ktoré vám umožňuje zvýrazňovať stránky a ukladať webový obsah do vášho trezora.

## Inštalácia Web Clipper

Získajte rozšírenie Obsidian Web Clipper z oficiálneho adresára pre váš prehliadač:

- [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
	- Pre Chrome, Brave, Arc, Orion a ďalšie prehliadače založené na Chromium.
- [Firefox Add-Ons](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
	- Pre Firefox a Firefox Mobile.
- [Safari Extensions](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
	- Pre macOS, iOS a iPadOS.
- [Edge Add-Ons](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
	- Pre Microsoft Edge.

## Ako používať Web Clipper

- [[Zachytávanie webových stránok]]
	- Začnite a naučte sa, ako ukladať obsah zo stránok.
- [[Zvýrazňovač|Zvýrazňovač]]
	- Zvýraznite dôležité pasáže a vyberte prvky, ktoré chcete uložiť.
- [[Čítačka|Čítačka]]
	- Čítajte webové stránky v čistom zobrazení bez rušivých prvkov.
- [[Interpret|Interpret]]
	- Použite výzvy v prirodzenom jazyku na zachytenie a úpravu dát na stránke.
- [[Riešenie problémov s Web Clipper|Riešenie problémov]]
	- Vyriešte bežné problémy s Web Clipper.
- [[Obsidian Web Clipper/Šablóny|Šablóny]]
	- Vytvorte vlastné šablóny pre konkrétne webové stránky.
- [[Premenné]]
	- Predvyplňte dáta stránky vo vašej šablóne.
- [[Filtre]]
	- Upravte výstup premenných šablóny.
- [[Logika]]
	- Použite podmienky, cykly a nastavte premenné v šablónach.

## Súkromie

Obsidian Web Clipper ukladá obsah lokálne do vášho trezora Obsidian a riadi sa našimi [zásadami ochrany osobných údajov](https://obsidian.md/privacy). Vaše dáta nie sú zbierané a nezhromažďujeme žiadne metriky používania. Kód je [otvorený zdrojový kód](https://github.com/obsidianmd/obsidian-clipper) a je možné ho auditovať.
