Ak ste vývojár, môžete prispieť k Obsidian rôznymi spôsobmi.

### Pluginy a témy

Môžete vyvíjať [[Komunitné pluginy]] a [[Témy]] pomocou Typescript a CSS. Viac informácií nájdete v [dokumentácii pre vývojárov Obsidian](https://docs.obsidian.md).

### Projekty s otvoreným zdrojovým kódom

Obsidian vyvíja a spravuje niekoľko projektov s otvoreným zdrojovým kódom. Môžete odosielať pull requesty s opravami chýb a vylepšeniami do nasledujúcich GitHub repozitárov:

- [JSON Canvas](https://github.com/obsidianmd/jsoncanvas) (MIT), otvorený formát súborov pre dáta nekonečného plátna.
- [Obsidian Importer](https://github.com/obsidianmd/obsidian-importer) (MIT), plugin, ktorý konvertuje dáta z rôznych formátov do obyčajných textových Markdown súborov.
- [Obsidian Maps](https://github.com/obsidianmd/obsidian-maps) (MIT), plugin, ktorý zobrazuje mapy v Obsidian Databázach.
- [Obsidian Web Clipper](https://github.com/obsidianmd/obsidian-clipper) (MIT), rozšírenie prehliadača na zvýrazňovanie a zachytávanie webových stránok do obyčajných textových Markdown súborov.
- [Defuddle](https://github.com/kepano/defuddle) (MIT), parser z HTML do Markdown používaný Obsidian Web Clipper.

### Dokumentácia

Ak chcete pomôcť vylepšiť našu [dokumentáciu pre vývojárov](https://docs.obsidian.md/Home), odošlite pull request do repozitára [obsidian-developer-docs](https://github.com/obsidianmd/obsidian-developer-docs).

Ak chcete pomôcť vylepšiť túto dokumentáciu, odošlite pull request do repozitára [obsidian-help](https://github.com/obsidianmd/obsidian-help).

### Komunita

Kanál [plugin-dev](https://discord.com/channels/686053708261228577/840286264964022302) na našom oficiálnom Discord serveri je miesto, kde môžete diskutovať o vývoji pluginov a pomáhať ostatným vývojárom s ich pluginmi.
