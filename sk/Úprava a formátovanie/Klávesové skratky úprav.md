Táto stránka obsahuje predvolené klávesové skratky na navigáciu a úpravu textu v Obsidian. Tieto skratky sú poskytované vaším operačným systémom alebo frameworkom, na ktorom je Obsidian postavený, a nie je možné ich v rámci Obsidian prispôsobiť.

Pre prispôsobiteľné klávesové skratky príkazov Obsidian si pozrite [[Klávesové skratky]].

## Skratky pre Windows a Linux

### Bežné akcie

| Akcia | Skratka |
|-|-|
| Kopírovať | `Ctrl+C` |
| Vystrihnúť | `Ctrl+X` |
| Prilepiť | `Ctrl+V` |
| Prilepiť bez formátovania | `Ctrl+Shift+V` |
| Späť | `Ctrl+Z` |
| Znova | `Ctrl+Shift+Z` alebo `Ctrl+Y` |
| Kopírovať odsek | `Ctrl+C` (bez vybraného textu) |
| Vystrihnúť odsek | `Ctrl+X` (bez vybraného textu) |

### Úprava textu

| Akcia | Skratka |
|-|-|
| Vložiť nový riadok | `Enter` |
| Odstrániť predchádzajúci znak | `Backspace` |
| Odstrániť nasledujúci znak | `Delete` |
| Odstrániť predchádzajúce slovo | `Ctrl+Backspace` |
| Odstrániť nasledujúce slovo | `Ctrl+Delete` |
| Odstrániť aktuálny riadok | `Ctrl+Shift+K` (bez vybraného textu) |

### Navigácia v texte

| Akcia | Skratka |
|-|-|
| Posunúť kurzor o jeden znak | `Left/→` |
| Posunúť kurzor na začiatok predchádzajúceho slova | `Ctrl+←` |
| Posunúť kurzor na koniec nasledujúceho slova | `Ctrl+→` |
| Posunúť kurzor na začiatok aktuálneho riadku | `Home` |
| Posunúť kurzor na koniec aktuálneho riadku | `End` |
| Posunúť kurzor na predchádzajúci riadok | `↑` |
| Posunúť kurzor na nasledujúci riadok | `↓` |
| Posunúť kurzor na začiatok poznámky | `Ctrl+Home` |
| Posunúť kurzor na koniec poznámky | `Ctrl+End` |
| Posunúť kurzor o jednu stránku hore | `Page up` |
| Posunúť kurzor o jednu stránku dole | `Page down` |

### Výber textu

| Akcia | Skratka |
|-|-|
| Zjednodušiť výber | `Escape` |
| Vybrať všetko | `Ctrl+A` |
| Rozšíriť výber o jeden znak | `Shift+Left/→` |
| Rozšíriť výber na začiatok predchádzajúceho slova | `Ctrl+Shift+←` |
| Rozšíriť výber na koniec nasledujúceho slova | `Ctrl+Shift+→` |
| Rozšíriť výber na začiatok aktuálneho riadku | `Shift+Home` |
| Rozšíriť výber na koniec aktuálneho riadku | `Shift+End` |
| Rozšíriť výber na začiatok poznámky | `Ctrl+Shift+Home` |
| Rozšíriť výber na koniec poznámky | `Ctrl+Shift+End` |
| Rozšíriť výber o jednu stránku hore | `Shift+Page up` |
| Rozšíriť výber o jednu stránku dole | `Shift+Page down` |

## Skratky pre macOS

### Bežné akcie

| Akcia | Skratka |
|-|-|
| Kopírovať | `Cmd+C` |
| Vystrihnúť | `Cmd+X` |
| Prilepiť | `Cmd+V` |
| Prilepiť bez formátovania | `Cmd+Shift+V` |
| Späť | `Cmd+Z` |
| Znova | `Cmd+Shift+Z` |
| Kopírovať odsek | `Cmd+C` (bez vybraného textu) |
| Vystrihnúť odsek | `Cmd+X` (bez vybraného textu) |

### Formátovanie textu

| Akcia | Skratka |
| ---- | ---- |
| Tučný text | `Cmd+B` |
| Text kurzívou | `Cmd+I` |

### Úprava textu

| Akcia | Skratka |
|-|-|
| Vložiť nový riadok | `Enter` |
| Odstrániť predchádzajúci znak | `Backspace` |
| Odstrániť nasledujúci znak | `Delete` |
| Odstrániť predchádzajúce slovo | `Option+Backspace` |
| Odstrániť nasledujúce slovo | `Option+Delete` |
| Odstrániť po začiatok aktuálneho riadku | `Cmd+Backspace` |
| Odstrániť po koniec aktuálneho riadku | `Cmd+Delete` |
| Odstrániť aktuálny riadok | `Cmd+Shift+K` (bez vybraného textu) |

### Navigácia v texte

| Akcia | Skratka |
|-|-|
| Posunúť kurzor o jeden znak | `Left/→` |
| Posunúť kurzor na začiatok predchádzajúceho slova | `Option+←` |
| Posunúť kurzor na koniec nasledujúceho slova | `Option+→` |
| Posunúť kurzor na začiatok aktuálneho riadku | `Cmd+←` |
| Posunúť kurzor na koniec aktuálneho riadku | `Cmd+→` |
| Posunúť kurzor na predchádzajúci riadok | `↑` |
| Posunúť kurzor na nasledujúci riadok | `↓` |
| Posunúť kurzor na začiatok poznámky | `Cmd+↑` |
| Posunúť kurzor na koniec poznámky | `Cmd+↓` |
| Posunúť kurzor o jednu stránku hore | `Fn+↑` |
| Posunúť kurzor o jednu stránku dole | `Fn+↓` |

### Výber textu

| ------------------------------------------------------ | -------------------------- |
| Zjednodušiť výber                                      | `Escape`                   |
| Vybrať všetko                                          | `Cmd+A`                    |
| Rozšíriť výber o jeden znak                            | `Shift+Left/→`   |
| Rozšíriť výber na začiatok predchádzajúceho slova     | `Option+Shift+←`  |
| Rozšíriť výber na koniec nasledujúceho slova           | `Option+Shift+→` |
| Rozšíriť výber na začiatok aktuálneho riadku           | `Cmd+Shift+←`              |
| Rozšíriť výber na koniec aktuálneho riadku             | `Cmd+Shift+→`    |
| Rozšíriť výber na začiatok poznámky                    | `Cmd+Shift+↑`       |
| Rozšíriť výber na koniec poznámky                      | `Cmd+Shift+↓`     |
| Rozšíriť výber o jednu stránku hore                    | `Ctrl+Shift+↑`      |
| Rozšíriť výber o jednu stránku dole                    | `Ctrl+Shift+↓`    |
