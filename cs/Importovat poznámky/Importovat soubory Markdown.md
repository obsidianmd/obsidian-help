Obsidian používá soubory Markdown `.md` jako primární formát pro poznámky. Díky tomu je snadné importovat soubory Markdown do Obsidian. Mnoho aplikací umí upravovat soubory Markdown nebo exportovat do tohoto formátu.

Existuje několik způsobů, jak přidat soubory Markdown do vašeho trezoru Obsidian:

1. Přetáhněte soubory nebo složky do okna [[Průzkumník souborů|Průzkumníka souborů]]
2. Použijte systémový správce souborů (např. Průzkumník Windows nebo Finder na macOS) a přesuňte soubory přímo do složky trezoru Obsidian.

### Převod z jiných variant Markdown

Obsidian používá [[Obsidian Flavored Markdown]]. Některé aplikace používají mírně odlišnou syntaxi Markdown. Můžete použít [[Importér Markdown formátu]] k převodu souborů do syntaxe Markdown používané Obsidian.
