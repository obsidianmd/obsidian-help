Tato stránka obsahuje výchozí klávesové zkratky pro navigaci a úpravu textu v Obsidian. Tyto zkratky jsou poskytovány vaším operačním systémem nebo frameworkem, na kterém je Obsidian postaven, a nelze je v rámci Obsidian přizpůsobit.

Přizpůsobitelné klávesové zkratky pro příkazy Obsidian naleznete v části [[Klávesové zkratky]].

## Zkratky pro Windows a Linux

### Běžné akce

| Akce | Zkratka |
|-|-|
| Kopírovat | `Ctrl+C` |
| Vyjmout | `Ctrl+X` |
| Vložit | `Ctrl+V` |
| Vložit bez formátování | `Ctrl+Shift+V` |
| Zpět | `Ctrl+Z` |
| Znovu | `Ctrl+Shift+Z` nebo `Ctrl+Y` |
| Kopírovat odstavec | `Ctrl+C` (bez vybraného textu) |
| Vyjmout odstavec | `Ctrl+X` (bez vybraného textu) |

### Úprava textu

| Akce | Zkratka |
|-|-|
| Vložit nový řádek | `Enter` |
| Smazat předchozí znak | `Backspace` |
| Smazat následující znak | `Delete` |
| Smazat předchozí slovo | `Ctrl+Backspace` |
| Smazat následující slovo | `Ctrl+Delete` |
| Smazat aktuální řádek | `Ctrl+Shift+K` (bez vybraného textu) |

### Navigace v textu

| Akce | Zkratka |
|-|-|
| Přesunout kurzor o jeden znak | `Left/→` |
| Přesunout kurzor na začátek předchozího slova | `Ctrl+←` |
| Přesunout kurzor na konec následujícího slova | `Ctrl+→` |
| Přesunout kurzor na začátek aktuálního řádku | `Home` |
| Přesunout kurzor na konec aktuálního řádku | `End` |
| Přesunout kurzor na předchozí řádek | `↑` |
| Přesunout kurzor na následující řádek | `↓` |
| Přesunout kurzor na začátek poznámky | `Ctrl+Home` |
| Přesunout kurzor na konec poznámky | `Ctrl+End` |
| Přesunout kurzor o stránku nahoru | `Page up` |
| Přesunout kurzor o stránku dolů | `Page down` |

### Výběr textu

| Akce | Zkratka |
|-|-|
| Zjednodušit výběr | `Escape` |
| Vybrat vše | `Ctrl+A` |
| Rozšířit výběr o jeden znak | `Shift+Left/→` |
| Rozšířit výběr na začátek předchozího slova | `Ctrl+Shift+←` |
| Rozšířit výběr na konec následujícího slova | `Ctrl+Shift+→` |
| Rozšířit výběr na začátek aktuálního řádku | `Shift+Home` |
| Rozšířit výběr na konec aktuálního řádku | `Shift+End` |
| Rozšířit výběr na začátek poznámky | `Ctrl+Shift+Home` |
| Rozšířit výběr na konec poznámky | `Ctrl+Shift+End` |
| Rozšířit výběr o stránku nahoru | `Shift+Page up` |
| Rozšířit výběr o stránku dolů | `Shift+Page down` |

## Zkratky pro macOS

### Běžné akce

| Akce | Zkratka |
|-|-|
| Kopírovat | `Cmd+C` |
| Vyjmout | `Cmd+X` |
| Vložit | `Cmd+V` |
| Vložit bez formátování | `Cmd+Shift+V` |
| Zpět | `Cmd+Z` |
| Znovu | `Cmd+Shift+Z` |
| Kopírovat odstavec | `Cmd+C` (bez vybraného textu) |
| Vyjmout odstavec | `Cmd+X` (bez vybraného textu) |

### Formátování textu

| Akce | Zkratka |
| ---- | ---- |
| Tučný text | `Cmd+B` |
| Text kurzívou | `Cmd+I` |

### Úprava textu

| Akce | Zkratka |
|-|-|
| Vložit nový řádek | `Enter` |
| Smazat předchozí znak | `Backspace` |
| Smazat následující znak | `Delete` |
| Smazat předchozí slovo | `Option+Backspace` |
| Smazat následující slovo | `Option+Delete` |
| Smazat po začátek aktuálního řádku | `Cmd+Backspace` |
| Smazat po konec aktuálního řádku | `Cmd+Delete` |
| Smazat aktuální řádek | `Cmd+Shift+K` (bez vybraného textu) |

### Navigace v textu

| Akce | Zkratka |
|-|-|

| Přesunout kurzor na začátek předchozího slova | `Option+←` |
| Přesunout kurzor na konec následujícího slova | `Option+→` |
| Přesunout kurzor na začátek aktuálního řádku | `Cmd+←` |
| Přesunout kurzor na konec aktuálního řádku | `Cmd+→` |
| Přesunout kurzor na předchozí řádek | `↑` |
| Přesunout kurzor na následující řádek | `↓` |
| Přesunout kurzor na začátek poznámky | `Cmd+↑` |
| Přesunout kurzor na konec poznámky | `Cmd+↓` |
| Přesunout kurzor o stránku nahoru | `Fn+↑` |
| Přesunout kurzor o stránku dolů | `Fn+↓` |
