Naučte se používat HTML element [iframe](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/iframe) k vkládání webových stránek do vašich poznámek.

Pro vložení webové stránky přidejte do poznámky následující kód a nahraďte zástupný text URL adresou webové stránky, kterou chcete vložit:

```html
<iframe src="ZDE VLOŽTE SVOU URL"></iframe>
```

> [!note] Poznámka
> Některé webové stránky neumožňují jejich vložení. Místo toho mohou poskytovat URL adresy určené speciálně pro vkládání. Pokud webová stránka nepodporuje vkládání, zkuste vyhledat název webové stránky následovaný výrazem „embed iframe". Například „youtube embed iframe".

> [!tip] Tip
> Pokud používáte [[Canvas]], můžete vložit webovou stránku do karty. Další informace najdete v části [[Canvas#Přidání karet z webových stránek]].

## Vložení videa z YouTube

Pro vložení videa z YouTube použijte stejnou Markdown syntaxi jako pro [[Základní syntaxe formátování#Externí obrázky|externí obrázky]]:

```md
![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)
```
