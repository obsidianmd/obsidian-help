Pokud jste vývojář, můžete k Obsidian přispět různými způsoby.

### Pluginy a motivy

Můžete vyvíjet [[Komunitní pluginy]] a [[Motivy]] pomocí TypeScriptu a CSS. Zjistěte více na [dokumentaci pro vývojáře Obsidian](https://docs.obsidian.md).

### Open source projekty

Obsidian vyvíjí a spravuje několik open source projektů. Můžete odesílat pull requesty s opravami chyb a vylepšeními do následujících GitHub repozitářů:

- [JSON Canvas](https://github.com/obsidianmd/jsoncanvas) (MIT), otevřený souborový formát pro data nekonečného plátna.
- [Obsidian Importer](https://github.com/obsidianmd/obsidian-importer) (MIT), plugin, který převádí data z široké škály formátů do souborů v prostém textu Markdown.
- [Obsidian Maps](https://github.com/obsidianmd/obsidian-maps) (MIT), plugin, který zobrazuje mapy v Obsidian Základnách.
- [Obsidian Web Clipper](https://github.com/obsidianmd/obsidian-clipper) (MIT), rozšíření prohlížeče pro zvýrazňování a ukládání webových stránek do souborů v prostém textu Markdown.
- [Defuddle](https://github.com/kepano/defuddle) (MIT), parser HTML-do-Markdown používaný Obsidian Web Clipper.

### Dokumentace

Chcete-li pomoci vylepšit naši [dokumentaci pro vývojáře](https://docs.obsidian.md/Home), odešlete pull request do repozitáře [obsidian-developer-docs](https://github.com/obsidianmd/obsidian-developer-docs).

Chcete-li pomoci vylepšit tuto dokumentaci, odešlete pull request do repozitáře [obsidian-help](https://github.com/obsidianmd/obsidian-help).

### Komunita

Kanál [plugin-dev](https://discord.com/channels/686053708261228577/840286264964022302) na našem oficiálním Discord serveru je místo, kde můžete diskutovat o vývoji pluginů a pomáhat ostatním vývojářům s jejich pluginy.
