Obsidian Web Clipper je bezplatné rozšíření prohlížeče, které vám umožňuje zvýrazňovat stránky a ukládat webový obsah do vašeho trezoru.

## Instalace Web Clipper

Získejte rozšíření Obsidian Web Clipper z oficiálního adresáře pro váš prohlížeč:

- [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
	- Pro Chrome, Brave, Arc, Orion a další prohlížeče založené na Chromiu.
- [Firefox Add-Ons](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
	- Pro Firefox a Firefox Mobile.
- [Safari Extensions](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
	- Pro macOS, iOS a iPadOS.
- [Edge Add-Ons](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
	- Pro Microsoft Edge.

## Soukromí

Obsidian Web Clipper ukládá obsah lokálně do vašeho trezoru Obsidian a řídí se našimi [zásadami ochrany osobních údajů](https://obsidian.md/privacy). Vaše data nejsou shromažďována a nesledujeme žádné metriky používání. Kód je [open source](https://github.com/obsidianmd/obsidian-clipper) a je možné ho auditovat.
