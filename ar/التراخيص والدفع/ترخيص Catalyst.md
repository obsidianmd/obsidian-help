ترخيص Catalyst هو عملية شراء لمرة واحدة تمنحك وصولاً مبكراً إلى الإصدارات التجريبية وتساعد في دعم التطوير المستمر لـ Obsidian. يمكنك دعم Obsidian عن طريق [شراء ترخيص Catalyst](https://obsidian.md/buy/catalyst).

تشمل المزايا:

- الوصول المبكر إلى [[#تمكين إصدارات الوصول المبكر|الإصدارات الجديدة]] من Obsidian
- الوصول إلى [[#انضم إلى قنوات Insider الخاصة بنا|قنوات Insider]] الحصرية.
- [[#احصل على شارة المنتدى|شارات المنتدى]] و[[#احصل على شارة Discord|شارات Discord]] الحصرية.

## مستويات Catalyst

هناك ثلاثة مستويات من Catalyst بناءً على مبلغ مساهمتك: *Insider* و*Supporter* و*VIP*.

| المزايا                                                                            | Insider | Supporter |   VIP    |
| ---------------------------------------------------------------------------------- | :-----: | :-------: | :------: |
| الوصول المبكر إلى الإصدارات الجديدة                                               |    ✓    |     ✓     |    ✓     |
| الوصول إلى قنوات التطوير الحصرية، `#insider-desktop` و`#insider-mobile`           |    ✓    |     ✓     |    ✓     |
| الوصول إلى قناة الصالون الحصرية، `#vip-lounge`                                    |         |     ✓     |    ✓     |
| شارة Insider                                                                      |    ✓    |           |          |
| شارة Supporter                                                                    |         |     ✓     |          |
| شارة VIP                                                                          |         |           |    ✓     |
| **المساهمة (دولار أمريكي)**                                                       | **$25** |  **$50**  | **$100** |


> [!info] ترقية مستوى Catalyst الخاص بك
> إذا كنت تمتلك بالفعل ترخيص Catalyst، يمكنك الترقية إلى مستوى Catalyst أعلى عن طريق دفع فرق السعر بين مستواك الحالي والمستوى الجديد.

## الوصول إلى مزايا Catalyst

بمجرد شرائك ترخيص Catalyst، يمكنك الوصول إلى مجموعة من المزايا. تعرّف على كيفية تمكينها هنا.

### احصل على شارة المنتدى

**ملاحظة:** لربط حساب Obsidian الخاص بك بحساب المنتدى، يجب استخدام نفس عنوان البريد الإلكتروني لكلا الحسابين.

1. إذا لم تكن قد فعلت ذلك بالفعل، انضم إلى [المنتدى](https://forum.obsidian.md/).
2. انتقل إلى [لوحة تحكم حساب Obsidian](https://obsidian.md/account/catalyst).
3. ضمن **Catalyst**، اختر **Get forum badge**.

### احصل على شارة Discord

1. إذا لم تكن قد فعلت ذلك بالفعل، انضم إلى [خادم Discord](https://discord.gg/veuWUTm).
2. انتقل إلى [لوحة تحكم حساب Obsidian](https://obsidian.md/account/catalyst).
3. ضمن **Catalyst**، اختر **Get Discord badge**.

### انضم إلى قنوات Insider الخاصة بنا

يمكن لأعضاء Insider التواصل مع فريق Obsidian وأعضاء Insider الآخرين في قنوات Discord الحصرية. ستنضم تلقائياً إلى قنوات Insider بعد [[#احصل على شارة Discord|الحصول على شارة Discord]].

### تمكين إصدارات الوصول المبكر

تعرّف على كيفية الحصول على [[إصدارات الوصول المبكر|إصدارات الوصول المبكر]].
