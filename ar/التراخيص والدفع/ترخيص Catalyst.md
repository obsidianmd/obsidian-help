ترخيص Catalyst هو عملية شراء لمرة واحدة تمنحك وصولًا مبكرًا إلى الإصدارات التجريبية وتساعد في دعم التطوير المستمر لـ Obsidian. يمكنك دعم Obsidian عن طريق [شراء ترخيص Catalyst](https://obsidian.md/ar/buy/catalyst).

تشمل المزايا:

- الوصول المبكر إلى [[#تمكين إصدارات الوصول المبكر|الإصدارات الجديدة]] من Obsidian
- الوصول إلى [[#الانضمام إلى قنوات Insider الحصرية|قنوات Insider]] الحصرية.
- [[#الحصول على شارة المنتدى|شارات المنتدى]] و[[#الحصول على شارة Discord|شارات Discord]] الحصرية.

## مستويات Catalyst

هناك ثلاثة مستويات من Catalyst بناءً على مبلغ مساهمتك: *Insider*، و*Supporter*، و*VIP*.

| المزايا                                  | Insider | Supporter |   VIP    |
| ---------------------------------------- | :-----: | :-------: | :------: |
| الوصول المبكر إلى الإصدارات الجديدة     |    ✓    |     ✓     |    ✓     |
| الوصول إلى قنوات التطوير الحصرية        |    ✓    |     ✓     |    ✓     |
| الوصول إلى قناة الصالون الحصرية         |         |     ✓     |    ✓     |
| شارة Insider                            |    ✓    |           |          |
| شارة Supporter                          |         |     ✓     |          |
| شارة VIP                                |         |           |    ✓     |
| **المساهمة (بالدولار الأمريكي)**         | **$25** |  **$50**  | **$100** |


> [!info] ترقية مستوى Catalyst الخاص بك
> إذا كنت تمتلك بالفعل ترخيص Catalyst، يمكنك الترقية إلى مستوى Catalyst أعلى بدفع فرق السعر بين مستواك الحالي والمستوى الجديد.

## الوصول إلى مزايا Catalyst

بمجرد شراء ترخيص Catalyst، يمكنك الوصول إلى مجموعة من المزايا. تعرّف على كيفية تمكينها هنا.

### الحصول على شارة المنتدى

**ملاحظة:** لربط حساب Obsidian الخاص بك بحساب المنتدى، يجب استخدام نفس عنوان البريد الإلكتروني لكلا الحسابين.

1. إذا لم تكن قد فعلت ذلك بعد، انضم إلى [المنتدى](https://forum.obsidian.md/).
2. انتقل إلى [لوحة تحكم حساب Obsidian](https://obsidian.md/ar/account/catalyst).
3. ضمن **Catalyst**، حدد **Get forum badge**.

### الحصول على شارة Discord

1. إذا لم تكن قد فعلت ذلك بعد، انضم إلى [خادم Discord](https://discord.gg/veuWUTm).
2. انتقل إلى [لوحة تحكم حساب Obsidian](https://obsidian.md/ar/account/catalyst).
3. ضمن **Catalyst**، حدد **Get Discord badge**.

### الانضمام إلى قنوات Insider الحصرية

يمكن لأعضاء Insider الدردشة مع فريق Obsidian وأعضاء Insider الآخرين في قنوات Discord الحصرية الخاصة بنا. ستنضم تلقائيًا إلى قنوات Insider بعد [[#الحصول على شارة Discord]].

- [vip-lounge](https://discord.com/channels/686053708261228577/714234422379151400) للنقاشات مع أعضاء Catalyst.
- [insider-desktop-release](https://discord.com/channels/686053708261228577/716028884885307432) لأحدث إصدارات الوصول المبكر على سطح المكتب.
- [insider-mobile-release](https://discord.com/channels/686053708261228577/817515900349448202) لأحدث إصدارات الوصول المبكر على الهاتف المحمول.
