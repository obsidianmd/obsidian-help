يتضمن [[مقدمة عن قاصّ أوبسيديان للويب|قاصّ أوبسيديان للويب]] وضع قراءة مدمجًا يُسمى **القارئ** يزيل الفوضى من صفحات الويب، ويعرض فقط محتوى المقال الرئيسي بتنسيق نظيف وسهل القراءة.

## تفعيل وضع القراءة

يمكنك تفعيل القارئ بعدة طرق:

- ![[lucide-book-icon.svg#icon]] **أيقونة الكتاب** في لوحة الإضافة.
- **اختصار لوحة المفاتيح** قابل للتخصيص في إعدادات الإضافة.
- **قائمة السياق** عن طريق النقر بزر الفأرة الأيمن على صفحة الويب التي تزورها.

يمكنك أيضًا ضبط زر إضافة ![[obsidian.svg#icon|20]] لفتح القارئ بنقرة واحدة. في إعدادات قاصّ الويب انتقل إلى **عام** ← **السلوك** ← **سلوك الفتح** واختر **القارئ**.

جميع الخيارات أعلاه تتيح لك أيضًا الخروج من القارئ واستعادة الصفحة الأصلية.

## كيف يعمل

يستخرج القارئ المحتوى الرئيسي من الصفحة. يتم استبدال الصفحة الأصلية بعرض قراءة نظيف يتضمن:

- عنوان المقال، والمؤلف، وتاريخ النشر، ونطاق المصدر.
- محتوى المقال الرئيسي، مع الحفاظ على الصور والتنسيق.
- مخطط تفصيلي في الشريط الجانبي للمقالات التي تحتوي على عناوين متعددة.
- تمييز بناء الجملة لكتل التعليمات البرمجية.
- نوافذ منبثقة للحواشي السفلية تُعرض مضمنة عند النقر عليها.

## تخصيص عرض القراءة

خصّص تجربة القارئ في إعدادات الإضافة. بعض الخيارات متاحة أيضًا داخل وضع القارئ عن طريق النقر على أيقونة ![[lucide-case-sensitive.svg#icon]].

| الإعداد              | الوصف                                  |
| -------------------- | -------------------------------------- |
| الخط                 | إضافة خطوط مثبتة على نظامك.           |
| حجم الخط             | ضبط حجم النص.                          |
| ارتفاع&nbsp;السطر    | المسافة بين الأسطر.                    |
| عرض&nbsp;السطر       | العرض الأقصى لمنطقة المحتوى.           |
| المظهر               | اختيار المظهر الفاتح أو الداكن.        |
| السمة                | اختيار الألوان.                        |
| CSS مخصص             | CSS إضافي يُطبَّق على الصفحة.          |
