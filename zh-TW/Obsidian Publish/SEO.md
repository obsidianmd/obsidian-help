[[Obsidian Publish 介紹|Obsidian Publish]] 網站會自動針對搜尋引擎和[[社群媒體連結預覽|社群分享卡片]]進行最佳化。你也可以使用描述、永久連結和圖片來自訂頁面中繼資料。

請注意，你可以在[[管理網站#網站選項|網站選項]]中禁止搜尋引擎索引。

### Google Search Console

若要使用 Google 追蹤 SEO，請在 [Google Search Console](https://search.google.com/search-console) 註冊你的網站。你也可以參考我們推薦的步驟來[[網站分析|設定 Google Analytics]]。

### Sitemap

你的 Publish 網站的 Sitemap 可在 `/sitemap.xml` 取得，例如：
https://help.obsidian.md/sitemap.xml

Sitemap 也可以作為 RSS feed 在 `/rss.xml` 取得：
https://help.obsidian.md/rss.xml

### 永久連結

建議使用[[永久連結]]來為你的網站定義永久網址。

### 中繼資料

頁面中繼資料可以使用[[屬性#Obsidian Publish 的屬性|屬性]]來自訂，這對於[[社群媒體連結預覽|社群分享卡片]]十分有用。

| 屬性            | 說明                                                                                                              |
| ------------- | ----------------------------------------------------------------------------------------------------------------- |
| `publish`     | 請參閱[[發佈內容#自動選取要發佈的資料\|自動選取要發佈的資料]]。 |
| `permalink`   | 請參閱[[永久連結\|永久連結]]。                                                                                   |
| `description` | 請參閱[[社群媒體連結預覽#說明\|說明]]。                                                      |
| `image`       | 請參閱[[社群媒體連結預覽#圖片\|圖片]]。                                                                  |
| `cover`       | 請參閱[[社群媒體連結預覽#圖片\|圖片]]。                                                                  |
