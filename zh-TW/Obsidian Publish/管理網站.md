本頁說明如何管理 [[Obsidian Publish 介紹|Obsidian Publish]] 網站。

網站是由 Obsidian Publish 託管並於線上可用的筆記集合，可透過 Obsidian Publish 位址或[[自訂網域|自訂網域]]存取。

## 建立新網站

> [!note] 你同時可擁有的網站數量取決於你的 Obsidian Publish 訂閱方案。建立新網站前，請確認你的訂閱方案允許此操作。

1. 在應用程式視窗左側的功能區中，點擊 **發佈變更** ![[lucide-send.svg#icon]]。
2. 如果你已有現有網站，請點擊 **切換網站** ![[lucide-repeat.svg#icon]]。
3. 在 **網站 ID** 中，輸入你想要的網站路徑。例如，ID 為 `my-amazing-site` 的網站可透過 publish.obsidian.md/my-amazing-site 存取。
4. 點擊 **建立**。

## 刪除現有網站

> [!note] 即使刪除網站後，筆記仍會保留在你的保管庫中。

1. 在應用程式視窗左側的功能區中，點擊 **發佈變更** ![[lucide-send.svg#icon]]。
2. 點擊 **切換網站** ![[lucide-repeat.svg#icon]]。
3. 在你要刪除的網站右側，點擊 **刪除網站** ![[lucide-x.svg#icon]]。
4. 點擊 **刪除** 以確認。

## 在網站之間切換

1. 在應用程式視窗左側的功能區中，點擊 **發佈變更** ![[lucide-send.svg#icon]]。
2. 點擊 **切換網站** ![[lucide-repeat.svg#icon]]。
3. 在你要切換到的網站右側，點擊 **挑選**。

## 變更網站 ID

1. 在應用程式視窗左側的功能區中，點擊 **發佈變更** ![[lucide-send.svg#icon]]。
2. 點擊 **切換網站** ![[lucide-repeat.svg#icon]]。
3. 在你要編輯的網站右側，點擊 **編輯網站 ID** ![[lucide-edit-3.svg#icon]]。
4. 在 **網站 ID** 中，輸入新的網站 ID。
5. 點擊 **變更**。

## 檢視網站選項

1. 在應用程式視窗左側的功能區中，點擊 **發佈變更** ![[lucide-send.svg#icon]]。
2. 在 **發佈變更** 對話框中，點擊 **變更網站選項** ![[lucide-cog.svg#icon]]。

## 網站選項

### 一般

| 選項                          | 類型   | 描述                                                                                                                |
| ------------------------------- | ------ | -------------------------------------------------------------------------------------------------------------------------- |
| 網站名稱                       | 輸入  | 你的 Obsidian Publish 網站的公開名稱和標題。                                                                   |
| 首頁檔案                   | 輸入  | 你想作為首頁的 Markdown 檔案位置。                                                    |
| 標誌                            | 輸入  | 你想作為網站橫幅的圖片。該圖片必須[[發佈內容#發佈筆記\|已發佈]]。 |
| 網站協作              | 按鈕 | 宣告你想授予編輯已發佈筆記權限的其他使用者。使用者必須擁有 Obsidian 帳戶。             |
| 自訂網域                   | 按鈕 | [[自訂網域]]                                                                                                         |
| 禁止搜尋引擎索引 | 切換 | 透過新增 `robots.txt` 檔案來防止遵守規則的搜尋引擎爬取你的網站。                                   |

### 外觀

| 選項            | 類型     | 描述                                                                               |
|-------------------|----------|-------------------------------------------------------------------------------------------|
| 主題             | 下拉選單 | 選擇你的主題在網站上的呈現方式：**明亮**、**暗黑** 或 **跟隨作業系統**。 |
| 切換亮色/暗色主題 | 切換   | 允許使用者透過網站上的切換按鈕切換 **明亮**/**暗黑** 模式。          |

## 閱讀體驗

| 選項               | 類型   | 描述                                                                                |
|----------------------|--------|--------------------------------------------------------------------------------------------|
| 顯示懸停預覽   | 切換 | 啟用或停用懸停在啟用的內部連結上時的頁面預覽功能。 |
| 隱藏頁面標題      | 切換 | 啟用或停用已發佈筆記的行內標題顯示。                    |
| 縮減行寬 | 切換 | 啟用或停用網站中的縮減行寬。                                   |
| 精確的換行符號   | 切換 | 啟用或停用在網站上顯示單一換行符號。                    |
| 使用捲動 (sliding) 視窗          | 切換 | 啟用或停用網站上的[[標籤頁#堆疊分頁群組\|堆疊分頁]]。    |

### 元件

| 選項                 | 類型   | 描述                                                                                                                          |
|------------------------|--------|--------------------------------------------------------------------------------------------------------------------------------------|
| 顯示導覽        | 切換 | 啟用或停用發佈網站上的[[檔案管理器]]檢視。                                                                   |
| 自訂導覽   | 按鈕 | 在 **顯示導覽** 開啟時，[[自訂網站#自訂導覽\|自訂]]檔案的排列順序。 |
| 顯示搜尋列        | 切換 | 啟用或停用網站上的搜尋列。                                                                                         |
| 顯示關聯圖        | 切換 | 啟用或停用網站右側邊欄中的圖譜檢視。                                                              |
| 顯示筆記目錄 | 切換 | 啟用或停用網站上的[[大綱\|目錄]]檢視。                                                              |
| 顯示反向連結         | 切換 | 啟用或停用網站上的[[反向連結]]。                                                                                        |

### 其他網站設定

| 選項                         | 類型   | 描述                                            |
| ------------------------------ | ------ | ------------------------------------------------------ |
| 密碼                      | 按鈕 | [[Obsidian Publish/安全性與隱私#新增網站密碼\|設定密碼]]以限制整個網站的存取。 |
| Google Analytics 追蹤程式碼 | 輸入  | **僅限自訂網域網址**。在此處放置你的 Google Analytics 網站追蹤程式碼。                                                       |
