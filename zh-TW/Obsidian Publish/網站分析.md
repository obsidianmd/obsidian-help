## 先決條件

若要在 [[Obsidian Publish 介紹|Obsidian Publish]] 網站上使用網站分析功能，你需要一個[[自訂網域]]。

> [!important] 重要
> 在啟用網站分析之前，請確認你當地的法律法規允許追蹤訪客。根據你使用的工具，你可能需要在網站上新增同意橫幅。

## Google Analytics

若要為你的 Obsidian Publish 網站啟用 Google Analytics：

1. 在功能區中，點擊應用程式視窗左側的**發佈變更** ![[lucide-send.svg#icon]]。
2. 在**發佈變更**對話框中，點擊**變更網站選項** ![[lucide-cog.svg#icon]]。
3. 在 **Google Analytics 追蹤程式碼**中，輸入你的追蹤程式碼。

若要使用 Google Tag Manager 而非 Google Analytics，請使用自訂 JavaScript 來新增你自己的指令碼。

## Plausible、Fathom 及其他分析服務

你可以透過 [[自訂網站|publish.js]] 檔案將大多數分析服務新增到你的網站。請確保將 `yourdomain.com` 替換為你的網域，並將指令碼 `src` 替換為分析服務提供商的指令碼。

以下是使用 [Plausible Analytics](https://plausible.io/) 的範例：

```js
var analyticsScript = document.createElement('script');
analyticsScript.defer = true;
analyticsScript.setAttribute('data-domain', 'yourdomain.com');
analyticsScript.src = 'https://plausible.io/js/plausible.js';
document.head.appendChild(analyticsScript);
```

同樣的方法也可以用於 [Fathom Analytics](https://usefathom.com/)。請注意 `data-domain` 已變更為 `data-site`——不同的分析服務提供商可能有不同的指令碼插入格式。

```javascript
var fathom = analyticsScript.createElement('script');
analyticsScript.defer = true;
analyticsScript.setAttribute('data-site', 'yourdomain.com');
analyticsScript.src = 'https://cdn.usefathom.com/script.js';
document.head.appendChild(analyticsScript);
```

## 疑難排解

若要驗證你的網站是否正在使用你的分析服務，請停用任何可能阻擋追蹤指令碼執行的廣告封鎖瀏覽器擴充功能，例如 uBlock Origin。
