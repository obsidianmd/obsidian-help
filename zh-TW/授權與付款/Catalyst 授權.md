Catalyst 授權是一次性購買，讓你可以搶先體驗測試版本，同時支持 Obsidian 的持續開發。你可以透過[購買 Catalyst 授權](https://obsidian.md/buy/catalyst)來支持 Obsidian。

權益包括：

- 搶先體驗 [[#啟用搶先體驗版本|新版本]] 的 Obsidian
- 進入專屬的 [[#加入我們的 Insider 頻道|Insider 頻道]]。
- 專屬的[[#取得論壇徽章|論壇徽章]]和 [[#取得 Discord 徽章|Discord 徽章]]。

## Catalyst 等級

Catalyst 依據贊助金額分為三個等級：*Insider*、*Supporter* 和 *VIP*。

| 權益                     | Insider | Supporter |   VIP    |
| ------------------------ | :-----: | :-------: | :------: |
| 搶先體驗新版本           |    ✓    |     ✓     |    ✓     |
| 進入專屬開發頻道         |    ✓    |     ✓     |    ✓     |
| 進入專屬休息室頻道       |         |     ✓     |    ✓     |
| Insider 徽章             |    ✓    |           |          |
| Supporter 徽章           |         |     ✓     |          |
| VIP 徽章                 |         |           |    ✓     |
| **贊助金額（美元）**     | **$25** |  **$50**  | **$100** |


> [!info] 升級你的 Catalyst 等級
> 如果你已經擁有 Catalyst 授權，可以透過支付目前等級與新等級之間的差額來升級到更高的 Catalyst 等級。

## 使用 Catalyst 權益

購買 Catalyst 授權後，你可以享有一系列權益。以下說明如何啟用這些權益。

### 取得論壇徽章

**注意：** 若要連結你的 Obsidian 帳戶和論壇帳戶，兩個帳戶必須使用相同的電子郵件地址。

1. 如果你還沒有加入，請先加入我們的[論壇](https://forum.obsidian.md/)。
2. 前往你的 [Obsidian 帳戶儀表板](https://obsidian.md/account/catalyst)。
3. 在 **Catalyst** 下方，選擇 **Get forum badge**。

### 取得 Discord 徽章

1. 如果你還沒有加入，請先加入我們的 [Discord 伺服器](https://discord.gg/veuWUTm)。
2. 前往你的 [Obsidian 帳戶儀表板](https://obsidian.md/account/catalyst)。
3. 在 **Catalyst** 下方，選擇 **Get Discord badge**。

### 加入我們的 Insider 頻道

Insider 可以在我們專屬的 Discord 頻道中與 Obsidian 團隊及其他 Insider 交流。當你[[#取得 Discord 徽章]]後，就會自動加入 Insider 頻道。

- [vip-lounge](https://discord.com/channels/686053708261228577/714234422379151400) 供 Catalyst 成員討論交流。
- [insider-desktop-release](https://discord.com/channels/686053708261228577/716028884885307432) 提供最新的桌面版搶先體驗版本。
- [insider-mobile-release](https://discord.com/channels/686053708261228577/817515900349448202) 提供最新的行動版搶先體驗版本。
