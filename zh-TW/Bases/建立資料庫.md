[[資料庫介紹|資料庫]]讓你建立類似資料庫的筆記檢視。以下說明如何建立資料庫並將其嵌入筆記中。每個資料庫可以有一個或多個[[檢視]]，以不同方式顯示資訊。

## 建立新的資料庫

**命令面板：**

1. 開啟**命令面板**。
2. 選擇
	- **Bases: Create new base** 在與目前檔案相同的資料夾中建立資料庫。
	- **Bases: Insert new base** 建立資料庫並嵌入至目前檔案中。

**檔案總管：**

1. 在檔案總管中，右鍵點擊你要建立資料庫的資料夾。
2. 選擇**新 base**。

**功能區：**

- 在垂直功能區選單中，選擇**建立新的 base** 在與目前檔案相同的資料夾中建立資料庫。

## 嵌入資料庫

### 嵌入資料庫檔案

你可以使用 `![[File.base]]` 語法將資料庫檔案嵌入到[[嵌入檔案|任何其他檔案]]中。若要指定預設檢視，請使用 `![[File.base#View]]`。

### 以程式碼區塊嵌入資料庫

資料庫也可以使用 `base` 程式碼區塊和[[資料庫語法]]直接嵌入筆記中。

~~~yaml
```base
filters:
  and:
    - file.hasTag("example")
views:
  - type: table
    name: Table
```
~~~
