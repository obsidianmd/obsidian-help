Obsidian 行動應用程式適用於 iOS 和 iPadOS，將強大的筆記功能帶到您的 iPhone 和 iPad 上。您可以從 [Apple App Store](https://apps.apple.com/us/app/obsidian-connected-notes/id1557175442) 下載。

本頁面涵蓋 iOS 專屬功能，包括小工具、Siri 整合和捷徑。

## 小工具

Obsidian iOS 版提供多種小工具，讓您對保管庫進行快速操作。

> [!note] 注意
> 小工具適用於 iOS 和 iPadOS 18 及更高版本。
> 使用「要求 Face ID」解鎖應用程式時，小工具不可用。


### 鎖定畫面與控制中心小工具

鎖定畫面和控制中心小工具允許您：
- 建立新筆記
- 開啟特定筆記
- 開啟每日筆記
- 開啟搜尋
- 開啟 Obsidian

### 主畫面小工具

主畫面小工具允許您：
- 建立筆記
- 檢視筆記
- 開啟每日筆記

### 自訂小工具

您可以自訂小工具以配合您的工作流程，例如選擇要使用的保管庫或指定要開啟的特定筆記。

- **主畫面小工具：** 點按並長按小工具，然後選擇**編輯小工具**。
- **鎖定畫面小工具：** 觸碰並長按鎖定畫面，點按**自訂**，選擇鎖定畫面，然後點按您要自訂的小工具。
- **控制中心小工具：** 開啟控制中心，點按左上角的 **+** 按鈕開始編輯，然後點按您要自訂的小工具。

## 捷徑

Obsidian 整合了 Apple 的捷徑應用程式，讓您建立強大的自動化操作。可用的捷徑包括：

- **開啟筆記** — 開啟保管庫中的特定筆記
- **建立新筆記** — 在保管庫中建立新筆記
- **開啟每日筆記** — 直接跳至今天的日記
- **擷取至日記** — 將文字附加或前置到日記中，無需開啟 Obsidian 應用程式
- **擷取至書籤** — 將文字附加或前置到已加入書籤的筆記中，無需開啟 Obsidian 應用程式

擷取捷徑特別適合快速筆記，因為它們允許您在背景中將內容新增到筆記。

## 分享表單

Obsidian 的分享表單讓您可以擷取網頁內容。它也適用於 YouTube 和其他社群網路等應用程式。

> [!note]
> - 原生分享表單適用於 iOS 和 iPadOS 18 及更高版本。
> - 本節描述的分享表單功能需要 Obsidian 1.13.0 或更新版本。

使用分享表單快速將其他應用程式的內容傳送到 Obsidian：
1. 在其他應用程式中，點按**分享**按鈕。
2. 選擇 **Obsidian**。
3. 選擇一個位置。
4. 檢閱或編輯擷取的內容。
5. 點按**儲存**。

![[ios-share-sheet-extension.png|400]]

### 位置

位置讓您在儲存之前決定分享的內容要傳送到哪裡。

位置可以擷取至：
- **新筆記** — 在保管庫或資料夾中建立新筆記。
- **每日筆記** — 將內容附加或前置到今天的日記。
- **已加入書籤的筆記** — 將內容附加或前置到已加入書籤的筆記。
- **筆記** — 選擇保管庫中現有的筆記。
- **新書籤** — 將分享的 URL 儲存到 Obsidian 書籤。

![[ios-share-sheet-locations.png|400]]

### 自訂位置

您可以為常見的工作流程建立位置，例如將文章儲存到收件匣、將引用附加到每日筆記，或將連結加入書籤。

若要自訂位置：

1. 從 iOS 分享表單開啟 Obsidian。
2. 點按工具列中目前的位置。
3. 點按 **+** 按鈕建立新位置，或選擇現有位置進行編輯。
4. 選擇保管庫、行為和可選設定。

根據「行為」類型，您可以設定以下選項：
- 資料夾
- 範本
- 書籤群組
- 附加或前置位置
- 分享的連結是擷取**完整文字**還是僅擷取 **URL**

![[ios-share-sheet-add-location.png|400]]

### 分享時使用範本

您可以在從分享表單分享內容時使用範本。範本讓您以頁面標題、作者、來源網站和發布日期等詳細資訊來格式化擷取的網頁內容。

若要設定帶有範本的位置：

1. 從 iOS 分享表單開啟 Obsidian。
2. 點按工具列中目前的位置。
3. 點按 **+** 按鈕建立新位置。
4. 輸入位置的名稱。
5. 選擇保管庫。
6. 將**行為**設定為**新筆記**。
7. 在**可選**區段中，點按**範本**。
8. 從保管庫中選擇一個筆記作為範本。
9. 點按**儲存**以儲存位置。

![[ios-share-sheet-set-template.png|400]]

當您使用此位置分享連結時，Obsidian 會先套用範本，然後加入分享的內容。

支援的範本佔位符：

| 佔位符 | 說明 |
| --- | --- |
| `{{author}}` | 文章的作者 |
| `{{description}}` | 文章的描述或摘要 |
| `{{domain}}` | 網站的網域名稱 |
| `{{favicon}}` | 網站 favicon 的 URL |
| `{{image}}` | 文章主圖片的 URL |
| `{{published}}` | 文章的發布日期，使用預設日期格式 |
| `{{published: YYYY-MM-DD}}` | 使用自訂日期格式的發布日期 |
| `{{site}}` | 網站名稱 |
| `{{title}}` | 文章標題 |
| `{{wordCount}}` | 擷取內容的總字數 |

您也可以使用標準範本日期和時間佔位符：

| 佔位符 | 說明 |
| --- | --- |
| `{{date}}` | 目前日期 |
| `{{date: YYYY-MM-DD}}` | 使用自訂格式的目前日期 |
| `{{time}}` | 目前時間 |
| `{{time: HH:mm}}` | 使用自訂格式的目前時間 |

您可以使用 Siri 語音指令與 Obsidian 互動：

- "Capture using Obsidian"
- "Capture to Obsidian"
- "Open my daily note in Obsidian"
- "Search in Obsidian"

- 新增筆記
- 搜尋
- 每日筆記
