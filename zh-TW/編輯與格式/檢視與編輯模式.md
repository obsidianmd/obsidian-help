Obsidian 讓你使用_檢視_和_模式_來控制如何閱讀和編輯以 Markdown 撰寫的筆記。

- **檢視**可在閱讀和編輯筆記之間切換。
- **模式**控制編輯時 Markdown 的顯示方式。

> [!note] 備註
> 預設情況下，Obsidian 會以編輯模式開啟新分頁。你可以在**[[設定]] → 編輯器 → 新分頁的預設檢視模式**中變更此設定。

## 閱讀檢視

*閱讀檢視*顯示不含 Markdown 語法的筆記，提供乾淨、易讀的格式以便專注檢閱。

切換至*閱讀檢視*：

- 點擊編輯器右上角的檢視切換器 ![[lucide-book-icon.svg#icon]]。
- 或點擊狀態列中的互動狀態圖示 ![[lucide-edit-3.svg#icon]] 或 ![[lucide-code-xml.svg#icon]]，然後選擇**閱讀檢視**。
- 或按 `Ctrl+E`（macOS 上為 `Cmd+E`）。

> [!note] 備註
> 啟用**[[設定]] → 外觀 → 顯示分頁標題列**即可看到檢視切換器圖示。
> 
> 如果已停用該選項，請開啟**[[設定]] → 編輯器 → 編輯器狀態**，以便從狀態列切換檢視。

> [!tip] 並排檢視
> 若要同時以*編輯*和*閱讀檢視*開啟筆記，請按住 `Ctrl`（macOS 上為 `Cmd`）並點擊檢視切換器。

## 編輯檢視

*編輯檢視*讓你對筆記進行變更。

在*編輯檢視*中，*編輯模式*決定 Markdown 的顯示方式。你可以選擇兩種*編輯模式*之一：*即時預覽*或*原始碼模式*。

### 即時預覽

*即時預覽*會內嵌顯示格式化文字，同時隱藏大部分 Markdown 語法。當游標進入格式化內容時，底層語法會變為可見以供編輯。

切換至*即時預覽*：

- 點擊編輯器右上角的檢視切換器 ![[lucide-edit-3.svg#icon]]。
- 或點擊狀態列中的互動狀態圖示 ![[lucide-book-icon.svg#icon]] 或 ![[lucide-code-xml.svg#icon]]，然後選擇**即時預覽**。
- 或按 `Ctrl+E`（macOS 上為 `Cmd+E`）。
- 或使用[[命令面板|命令]]**切換編輯/預覽模式**。

> [!note] 備註
> 預設情況下，編輯檢視設定為*即時預覽*。你可以在**[[設定]] → 編輯器 → 預設編輯模式**中變更此設定。

> [!tip] 在許多情況下，即時預覽可以免去切換至[[#閱讀檢視]]的需要。

### 原始碼模式

*原始碼模式*會完全按照撰寫的方式顯示所有 Markdown 語法。如果你偏好純文字或需要精確的格式控制，請使用此模式。

切換至*原始碼模式*：
- 點擊狀態列中的互動狀態圖示 ![[lucide-book-icon.svg#icon]] 或 ![[lucide-edit-3.svg#icon]]，然後選擇**原始碼模式**。

> [!note] 備註
> 預設情況下，編輯檢視設定為*即時預覽*。若要變更為*原始碼模式*，請至**[[設定]] → 編輯器 → 預設編輯模式**。
> 
> 切換至*原始碼模式*後，你還可以透過以下方式切換：
> 
> - 點擊筆記右上角的檢視切換器 ![[lucide-edit-3.svg#icon]]。
> - 或按 `Ctrl+E`（macOS 上為 `Cmd+E`）。
> - 或使用[[命令面板|命令]]**切換編輯/預覽模式**。

> [!tip] 切換*編輯模式*
> 若要在*即時預覽*和*原始碼模式*之間快速切換，你可以為[[命令面板|命令]]**切換實際預覽/原始碼模式**設定一個[[快速鍵]]。
