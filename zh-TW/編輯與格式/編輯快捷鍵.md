本頁列出 Obsidian 中用於導覽和編輯文字的預設鍵盤快捷鍵。這些快捷鍵由你的作業系統或 Obsidian 所使用的框架提供，無法在 Obsidian 內自訂。

若要自訂 Obsidian 命令的鍵盤快捷鍵，請參閱[[快速鍵]]。

## Windows 和 Linux 快捷鍵

### 常用操作

| 操作 | 快捷鍵 |
|-|-|
| 複製 | `Ctrl+C` |
| 剪下 | `Ctrl+X` |
| 貼上 | `Ctrl+V` |
| 貼上為純文字 | `Ctrl+Shift+V` |
| 復原 | `Ctrl+Z` |
| 重做 | `Ctrl+Shift+Z` 或 `Ctrl+Y` |
| 複製段落 | `Ctrl+C`（未選取文字時） |
| 剪下段落 | `Ctrl+X`（未選取文字時） |

### 文字編輯

| 操作 | 快捷鍵 |
|-|-|
| 插入新行 | `Enter` |
| 刪除前一個字元 | `Backspace` |
| 刪除下一個字元 | `Delete` |
| 刪除前一個單字 | `Ctrl+Backspace` |
| 刪除下一個單字 | `Ctrl+Delete` |
| 刪除目前行 | `Ctrl+Shift+K`（未選取文字時） |

### 文字導覽

| 操作 | 快捷鍵 |
|-|-|
| 移動游標一個字元 | `Left/→` |
| 移動游標至前一個單字的開頭 | `Ctrl+←` |
| 移動游標至下一個單字的結尾 | `Ctrl+→` |
| 移動游標至目前行的開頭 | `Home` |
| 移動游標至目前行的結尾 | `End` |
| 移動游標至上一行 | `↑` |
| 移動游標至下一行 | `↓` |
| 移動游標至筆記開頭 | `Ctrl+Home` |
| 移動游標至筆記結尾 | `Ctrl+End` |
| 移動游標向上一頁 | `Page up` |
| 移動游標向下一頁 | `Page down` |

### 文字選取

| 操作 | 快捷鍵 |
|-|-|
| 簡化選取 | `Escape` |
| 全選 | `Ctrl+A` |
| 擴展選取一個字元 | `Shift+Left/→` |
| 擴展選取至前一個單字的開頭 | `Ctrl+Shift+←` |
| 擴展選取至下一個單字的結尾 | `Ctrl+Shift+→` |
| 擴展選取至目前行的開頭 | `Shift+Home` |
| 擴展選取至目前行的結尾 | `Shift+End` |
| 擴展選取至筆記開頭 | `Ctrl+Shift+Home` |
| 擴展選取至筆記結尾 | `Ctrl+Shift+End` |
| 擴展選取向上一頁 | `Shift+Page up` |
| 擴展選取向下一頁 | `Shift+Page down` |

## macOS 快捷鍵

### 常用操作

| 操作 | 快捷鍵 |
|-|-|
| 複製 | `Cmd+C` |
| 剪下 | `Cmd+X` |
| 貼上 | `Cmd+V` |
| 貼上為純文字 | `Cmd+Shift+V` |
| 復原 | `Cmd+Z` |
| 重做 | `Cmd+Shift+Z` |
| 複製段落 | `Cmd+C`（未選取文字時） |
| 剪下段落 | `Cmd+X`（未選取文字時） |

### 文字格式

| 操作 | 快捷鍵 |
| ---- | ---- |
| 粗體文字 | `Cmd+B` |
| 斜體文字 | `Cmd+I` |

### 文字編輯

| 操作 | 快捷鍵 |
|-|-|
| 插入新行 | `Enter` |
| 刪除前一個字元 | `Backspace` |
| 刪除下一個字元 | `Delete` |
| 刪除前一個單字 | `Option+Backspace` |
| 刪除下一個單字 | `Option+Delete` |
| 刪除至目前行的開頭 | `Cmd+Backspace` |
| 刪除至目前行的結尾 | `Cmd+Delete` |
| 刪除目前行 | `Cmd+Shift+K`（未選取文字時） |

### 文字導覽

| 操作 | 快捷鍵 |
|-|-|
| 移動游標一個字元 | `Left/→` |
| 移動游標至前一個單字的開頭 | `Option+←` |
| 移動游標至下一個單字的結尾 | `Option+→` |
| 移動游標至目前行的開頭 | `Cmd+←` |
| 移動游標至目前行的結尾 | `Cmd+→` |
| 移動游標至上一行 | `↑` |
| 移動游標至下一行 | `↓` |
| 移動游標至筆記開頭 | `Cmd+↑` |
| 移動游標至筆記結尾 | `Cmd+↓` |
| 移動游標向上一頁 | `Fn+↑` |
| 移動游標向下一頁 | `Fn+↓` |

### 文字選取

| 操作                                                 | 快捷鍵                   |
| ------------------------------------------------------ | -------------------------- |
| 簡化選取                                     | `Escape`                   |
| 全選                                             | `Cmd+A`                    |
| 擴展選取一個字元                         | `Shift+Left/→`   |
| 擴展選取至前一個單字的開頭 | `Option+Shift+←`  |
| 擴展選取至下一個單字的結尾           | `Option+Shift+→` |
| 擴展選取至目前行的開頭  | `Cmd+Shift+←`              |
| 擴展選取至目前行的結尾        | `Cmd+Shift+→`    |
| 擴展選取至筆記開頭          | `Cmd+Shift+↑`       |
| 擴展選取至筆記結尾                | `Cmd+Shift+↓`     |
| 擴展選取向上一頁                           | `Ctrl+Shift+↑`      |
| 擴展選取向下一頁                         | `Ctrl+Shift+↓`    |
