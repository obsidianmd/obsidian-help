了解如何使用 [iframe](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/iframe) HTML 元素在筆記中嵌入網頁。

若要嵌入網頁，請在筆記中加入以下內容，並將佔位文字替換為你想嵌入的網頁 URL：

```html
<iframe src="INSERT YOUR URL HERE"></iframe>
```

> [!note] 備註
> 某些網站不允許你嵌入它們。它們可能會提供專門用於嵌入的 URL。如果網站不支援嵌入，請嘗試搜尋該網站名稱加上「embed iframe」。例如「youtube embed iframe」。

> [!tip] 小技巧
> 如果你正在使用 [[Canvas]]，可以在卡片中嵌入網頁。更多資訊請參閱 [[Canvas#從網頁新增卡片]]。

## 嵌入 YouTube 影片

若要嵌入 YouTube 影片，請使用與[[基本格式語法#外部圖片|外部圖片]]相同的 Markdown 語法：

```md
![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)
```

![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)

## 嵌入推文

若要嵌入推文，請使用與[[基本格式語法#外部圖片|外部圖片]]相同的 Markdown 語法：

```md
![](https://twitter.com/obsdmd/status/1580548874246443010)
```

![](https://twitter.com/obsdmd/status/1580548874246443010)
