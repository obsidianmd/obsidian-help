Obsidian 使用 Markdown `.md` 檔案作為筆記的主要格式。這使得將 Markdown 檔案匯入 Obsidian 變得十分容易。許多應用程式可以編輯 Markdown 檔案或匯出為此格式。

有幾種方式可以將 Markdown 檔案加入你的 Obsidian 保管庫：

1. 將檔案或資料夾拖放到[[檔案管理器]]視窗中
2. 使用系統的檔案瀏覽器（例如 Windows 檔案總管或 macOS 的 Finder），將檔案直接移動到 Obsidian 儲存庫的根目錄中。

### 從其他 Markdown 風格轉換

Obsidian 使用 [[Obsidian 風格 Markdown]]。部分應用程式使用略有不同的 Markdown 語法。你可以使用[[Markdown 格式轉換器]]將檔案轉換為 Obsidian 的 Markdown 語法。
