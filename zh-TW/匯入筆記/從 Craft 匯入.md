Obsidian 可讓你從 Craft 遷移筆記。這會將你的 Craft 資料轉換為耐久的 Markdown 檔案，你可以在 Obsidian 及許多其他應用程式中使用。

## 從 Craft 匯出資料

在桌面電腦上開啟 Craft 應用程式：

1. 前往「Organize」或「All Documents」檢視
2. 點選三點選單
3. 選擇匯出選項。
4. 選擇 Markdown。

根據你在 Craft 中的筆記數量，這個過程可能需要一些時間。

## 將 Craft 資料匯入 Obsidian

接下來你可以依照[[匯入 Markdown 檔案]]的說明進行操作。

## 疑難排解

Craft Docs 對某些連結使用非標準語法。你可以在 [Importer GitHub 儲存庫](https://github.com/obsidianmd/obsidian-importer/issues/27) 找到遷移這些連結的相關資源。
