Obsidian 讓你能輕鬆使用官方的[[匯入工具|匯入工具外掛程式]]匯入個別 HTML 檔案與整個 HTML 檔案資料夾。如果你想將整個網站儲存為 [[Obsidian 風格 Markdown|Markdown]] 格式，或是從匯出為 HTML 的工具匯入資料，這項功能非常實用。

如果你正在從特定應用程式匯入筆記，建議先檢視[[匯入工具|匯入工具外掛程式]]支援的應用程式清單，確認是否有更適合的轉換器能更完整地保留你要遷移的資料。

## 將 HTML 資料匯入 Obsidian

你需要安裝官方的 Obsidian [[匯入工具]]外掛程式，可以[在此安裝](obsidian://show-plugin?id=obsidian-importer)。

1. 開啟**[[設定]]**。
2. 前往**社群外掛程式**並[安裝匯入工具](obsidian://show-plugin?id=obsidian-importer)。
3. 啟用匯入工具外掛程式。
4. 使用命令面板或功能區圖示開啟**匯入工具**外掛程式。
5. 在**檔案格式**下選擇 **HTML (.html)**。
6. 選取你的 HTML 檔案或資料夾的位置。
7. 點擊**匯入**並等待匯入完成。
8. 完成！

### 匯入設定

- **附件大小限制**：跳過匯入超過指定大小的附件。
- **最小圖片大小**：跳過匯入任一維度小於指定大小的圖片。可用於跳過圖示和標誌。
