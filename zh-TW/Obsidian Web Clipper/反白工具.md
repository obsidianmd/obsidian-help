[[Obsidian Web Clipper 簡介|Web Clipper]] 讓你可以在網頁上反白文字，並選擇要儲存到 Obsidian 的元素。反白內容會被保存，方便你回到頁面時檢視。

反白內容可以透過擴充功能儲存到 Obsidian。你可以在擴充功能設定中定義反白標記選項。

## 使用反白標記工具

反白標記工具可用於即時網頁或[[閱讀器]]。在閱讀器檢視中，建立文字選取範圍會提供反白選項。

你也可以透過啟用反白標記工具，自動將選取範圍加入反白內容：

- 擴充功能面板或閱讀器工具列中的 ![[lucide-highlighter.svg#icon]] **反白標記圖示**。
- **快捷鍵**，可在擴充功能設定中自訂。
- **右鍵選單**，在你正瀏覽的網頁上按下右鍵。

開啟反白標記功能後，任何選取的文字、圖片和元素都會被加入反白內容。以上所有方法也可以用來關閉反白標記工具。

## 將反白內容加入筆記

將反白內容插入擷取的筆記中，有三種選項：

- **反白頁面內容** — 使用 [[Obsidian 風格 Markdown|語法]] `==highlight==` 直接在文字中加入反白標記。
- **取代頁面內容** — 傳回反白內容清單，不包含任何頁面內容。
- **不執行任何操作** — 傳回不含反白標記的原始內容。
 
這些選項會變更範本中的 `{{content}}` [[變數|變數]]。你也可以使用 `{{highlights}}` 變數直接將反白內容加入範本中，例如：

```
{{highlights|map: item => item.text|join:"\n\n"}}
```

## 檢視與搜尋反白內容

你建立的所有反白內容都可以在反白內容頁面中檢視。你可以前往**設定** → **反白標記**來開啟此頁面。

## 匯出反白內容

反白內容可以匯出為 `.json` 檔案，可在設定頁面或反白內容頁面中進行。
