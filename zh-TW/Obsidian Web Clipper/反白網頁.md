[[Obsidian Web Clipper 簡介|Web Clipper]] 讓你可以在網頁上反白文字，並選擇要儲存到 Obsidian 的元素。你的反白內容會被保存，方便你回到頁面時重新檢視。

反白內容可以在你開啟擴充功能時被[[擷取網頁|擷取]]並儲存到 Obsidian。

## 開啟反白標記工具

你可以透過多種方式開啟反白標記功能，視瀏覽器而定：

- 擴充功能面板中的反白標記圖示。
- 快捷鍵，從鍵盤啟用擴充功能。
- 右鍵選單，在你正瀏覽的網頁上按下右鍵。

開啟反白標記功能後，你可以選取想要反白的文字、圖片和元素。

## 反白標記設定

你可以前往 Web Clipper 設定來變更反白標記的行為。你也可以在此將反白內容匯出為 `.json` 檔案。

透過 `{{content}}` [[變數|變數]]將反白內容插入擷取的筆記中，有三種選項：

- **反白頁面內容** — 使用 [[Obsidian 風格 Markdown|語法]] `==highlight==` 直接在文字中加入反白標記。
- **取代頁面內容** — 傳回反白內容清單，不包含任何頁面內容。
- **不執行任何操作** — 傳回不含反白標記的原始內容。

你可以使用 `{{highlights}}` 變數直接將反白內容加入範本中，例如：

```
{{highlights|map: item => item.text|join:"\n\n"}}
```
