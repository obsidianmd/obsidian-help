Obsidian Web Clipper 是一款免費的瀏覽器擴充功能，可讓你反白頁面內容並將網頁資料儲存到你的保管庫中。

## 安裝 Web Clipper

從你的瀏覽器官方目錄取得 Obsidian Web Clipper 擴充功能：

- [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
	- 適用於 Chrome、Brave、Arc、Orion 及其他基於 Chromium 的瀏覽器。
- [Firefox Add-Ons](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
	- 適用於 Firefox 與 Firefox Mobile。
- [Safari Extensions](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
	- 適用於 macOS、iOS 與 iPadOS。
- [Edge Add-Ons](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
	- 適用於 Microsoft Edge。

## 隱私權

Obsidian Web Clipper 會將內容以本機方式儲存到你的 Obsidian 保管庫，並遵循我們的[隱私權政策](https://obsidian.md/privacy)。你的資料不會被蒐集，我們也不會收集任何使用指標。程式碼為[開放原始碼](https://github.com/obsidianmd/obsidian-clipper)，可供審閱。
