安裝 [[Obsidian Web Clipper 簡介|Web Clipper]] 瀏覽器擴充功能後，你可以透過以下方式存取它（視瀏覽器而定）：

1. 瀏覽器工具列中的 Obsidian 圖示。
2. 快捷鍵，從鍵盤啟用擴充功能。
3. 右鍵選單，在你正在瀏覽的網頁上點擊右鍵。

若要將頁面儲存到 Obsidian，請點擊**加入 Obsidian** 按鈕。

## 擷取頁面

當你開啟擴充功能時，Web Clipper 會根據你的[[模板|範本]]設定，從目前網頁中擷取資料。你可以建立自己的範本，並使用[[變數]]和[[過濾設定]]自訂輸出內容。

預設情況下，Web Clipper 會嘗試智慧擷取頁面的主要文章內容，排除頁面上的其他元素。不過，你可以透過以下方式覆寫此行為：

- 如果存在自訂範本，則使用你的範本。
- 如果有選取文字，則使用選取的內容。你可以使用 `Ctrl/Cmd+A` 選取整個頁面。
- 如果有任何[[螢光筆|反白]]內容，則使用反白的部分。

## 下載圖片

使用 Web Clipper 時，圖片不會自動下載。圖片會連結到其網路 URL。這可以節省保管庫的空間，但這表示圖片在離線時或 URL 失效後將無法存取。

你可以在 Obsidian 中使用[[命令面板|命令]]中名為**下載目前檔案的附件**的命令來下載任何檔案的圖片。此命令也可以在 Obsidian 中對應到快捷鍵。

## 快捷鍵

Web Clipper 包含鍵盤快捷鍵，可用來加速你的工作流程。若要變更按鍵對應，請前往 **Web Clipper 設定** → **一般**，並按照你的瀏覽器說明操作。除了 Safari 不支援編輯快捷鍵外，所有瀏覽器都可以變更對應設定。

| 動作             | macOS         | Windows/Linux  |
| ------------------ | ------------- | -------------- |
| 開啟擷取器       | `Cmd+Shift+O` | `Ctrl+Shift+O` |
| 快速擷取         | `Opt+Shift+O` | `Alt+Shift+O`  |
| 切換螢光筆 | `Opt+Shift+H` | `Alt+Shift+H`  |
| 切換閱讀器      | `Opt+Shift+R` | `Alt+Shift+R`  |

## 介面功能

Web Clipper 介面分為四個區段：

1. **頁首**：你可以在此切換範本、開啟[[螢光筆|反白]]、[[閱讀器|閱讀模式]]，以及存取設定。
2. **屬性**：顯示從頁面擷取的[[屬性|中繼資料]]，這些資料將作為[[屬性]]儲存到 Obsidian。
3. **筆記內容**：將儲存到 Obsidian 的內容。
4. **頁尾**：可讓你選擇保管庫和資料夾，並加入 Obsidian。

頁首功能包括：

- ![[lucide-chevrons-up-down.svg#icon]] **範本**切換器，可使用擴充功能設定中新增的已儲存[[模板|範本]]。
- ![[lucide-more-horizontal.svg#icon]] **更多**按鈕，顯示你可以在範本中使用的頁面變數。
- ![[lucide-highlighter.svg#icon]] **螢光筆**按鈕，開啟[[螢光筆|反白]]功能。
- ![[lucide-book-icon.svg#icon]] **閱讀器**按鈕，開啟[[閱讀器|閱讀檢視]]。
- ![[lucide-picture-in-picture-2.svg#icon]] **嵌入**按鈕，將 Web Clipper 從彈出視窗移入頁面中。
- ![[lucide-settings.svg#icon]] **設定**按鈕，開啟擴充功能設定。

頁尾功能包括：

- **加入 Obsidian** 按鈕，將資料儲存到 Obsidian。
- **保管庫**下拉選單，在 Web Clipper 設定中新增的已儲存保管庫之間切換。
- **資料夾**欄位，定義要儲存到哪個資料夾。
- **解讀器**，在頁面上執行[[解讀器|自然語言提示]]。
