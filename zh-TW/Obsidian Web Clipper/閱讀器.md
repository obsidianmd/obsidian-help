[[Obsidian Web Clipper 簡介|Web Clipper]] 內建了一個名為「閱讀器」的閱讀模式，能夠去除網頁上的雜亂內容，僅以乾淨、易讀的格式呈現主要文章內容。

## 啟用閱讀模式

你可以透過以下方式啟用閱讀器：

- 擴充功能面板標題列中的**書本圖示**。
- 鍵盤快捷鍵 `Alt+Shift+R`，可在擴充功能設定中自訂。
- 再次切換閱讀模式即可還原原始頁面。

## 運作方式

閱讀器會擷取頁面的主要內容。原始頁面將替換為乾淨的閱讀檢視，包含：

- 文章標題、作者、發佈日期和來源網域。
- 主要文章內容，保留圖片和格式。
- 側邊欄中的大綱（適用於含有多個標題的文章）。
- 程式碼區塊的語法高亮。
- 點擊後以行內方式顯示的腳註彈出視窗。

## 自訂閱讀檢視

閱讀模式包含一個隱藏的設定列，將滑鼠懸停在頁面右上角即可顯示，其中提供了自訂閱讀體驗的控制項。

| 設定              | 說明                                                                                       |
| ----------------- | ------------------------------------------------------------------------------------------ |
| 字型大小          | 調整文字大小。                                                                             |
| 行寬              | 內容區域的最大寬度。                                                                       |
| 行高              | 行與行之間的間距。                                                                         |
| 色彩&nbsp;配置    |                                                                                            |
| 主題              | 自動會跟隨你的系統偏好設定。或挑選總是使用明亮模式或暗黑模式。                             |
