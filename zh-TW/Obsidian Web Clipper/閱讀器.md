[[Obsidian Web Clipper 簡介|Web Clipper]] 內建了一個名為**閱讀器**的閱讀模式，能夠去除網頁上的雜亂內容，僅以乾淨、易讀的格式呈現主要文章內容。

## 啟用閱讀模式

你可以透過以下方式啟用閱讀器：

- ![[lucide-book-icon.svg#icon]] 擴充功能面板中的**書本圖示**。
- **鍵盤快捷鍵**，可在擴充功能設定中自訂。
- **右鍵選單**，在你正在瀏覽的網頁上按右鍵即可使用。

你也可以將 ![[obsidian.svg#icon|20]] 擴充功能按鈕設定為一鍵開啟閱讀器。在 Web Clipper 設定中前往**一般** → **行為** → **開啟行為**，然後選擇**閱讀器**。

以上所有選項也可以用來退出閱讀器並還原原始頁面。

## 運作方式

閱讀器會擷取頁面的主要內容。原始頁面將替換為乾淨的閱讀檢視，包含：

- 文章標題、作者、發佈日期和來源網域。
- 主要文章內容，保留圖片和格式。
- 側邊欄中的大綱（適用於含有多個標題的文章）。
- 程式碼區塊的語法高亮。
- 點擊後以行內方式顯示的腳註彈出視窗。

## 自訂閱讀檢視

在擴充功能設定中自訂你的閱讀器體驗。部分選項也可在閱讀模式中點擊 ![[lucide-case-sensitive.svg#icon]] 圖示來使用。

| 設定             | 說明                               |
| ---------------- | ---------------------------------- |
| 字型             | 新增安裝在你系統上的字型。         |
| 字型大小         | 調整文字大小。                     |
| 行&nbsp;高       | 行與行之間的間距。                 |
| 行&nbsp;寬       | 內容區域的最大寬度。               |
| 外觀             | 選擇明亮或暗黑外觀。              |
| 主題             | 選擇色彩配置。                     |
| 自訂 CSS         | 套用至頁面的額外 CSS。             |
