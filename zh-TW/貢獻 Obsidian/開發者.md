如果您是開發者，可以透過多種方式為 Obsidian 做出貢獻。

### 外掛程式與佈景主題

您可以使用 Typescript 和 CSS 開發[[社群外掛程式]]和[[佈景主題]]。前往 [Obsidian 開發者文件](https://docs.obsidian.md)深入了解。

### 開放原始碼專案

Obsidian 開發並維護多個開放原始碼專案。您可以向以下 GitHub 儲存庫提交包含錯誤修正和功能增強的 Pull Request：

- [JSON Canvas](https://github.com/obsidianmd/jsoncanvas)（MIT），一種用於無限白板資料的開放檔案格式。
- [Obsidian Importer](https://github.com/obsidianmd/obsidian-importer)（MIT），一個可將多種格式的資料轉換為純文字 Markdown 檔案的外掛程式。
- [Obsidian Maps](https://github.com/obsidianmd/obsidian-maps)（MIT），一個在 Obsidian 資料庫中顯示地圖的外掛程式。
- [Obsidian Web Clipper](https://github.com/obsidianmd/obsidian-clipper)（MIT），一個用於反白和擷取網頁為純文字 Markdown 檔案的瀏覽器擴充功能。
- [Defuddle](https://github.com/kepano/defuddle)（MIT），Obsidian Web Clipper 使用的 HTML 轉 Markdown 解析器。

### 文件

若要協助改善我們的[開發者文件](https://docs.obsidian.md/Home)，請向 [obsidian-developer-docs](https://github.com/obsidianmd/obsidian-developer-docs) 儲存庫提交 Pull Request。

若要協助改善本文件，請向 [obsidian-help](https://github.com/obsidianmd/obsidian-help) 儲存庫提交 Pull Request。

### 社群

我們官方 Discord 伺服器上的 [plugin-dev](https://discord.com/channels/686053708261228577/840286264964022302) 頻道是您可以討論外掛程式開發並協助其他開發者解決外掛程式問題的地方。
