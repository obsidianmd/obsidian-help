## កំណត់ហេតុសកម្មភាព Sync

Obsidian Sync រួមបញ្ចូលកំណត់ហេតុសមកាលកម្មលម្អិតដែលតាមដានអន្តរកម្មទាំងអស់រវាងឯកសារមូលដ្ឋានរបស់អ្នក និង vault ពីចម្ងាយ។ កំណត់ហេតុនេះបង្ហាញការផ្ទុកឡើង ការទាញយក ការលុប និងបញ្ហាណាមួយដូចជាជម្លោះការរួមបញ្ចូល ឬបញ្ហាភ្ជាប់។

**ចូលប្រើកំណត់ហេតុសកម្មភាព៖**
- ចុចរូបតំណាងស្ថានភាព sync នៅក្នុងរបារស្ថានភាព
- ទៅកាន់ **[[ការកំណត់]] → Sync → Activity log**
- ប្រើ **ក្ដារលាយពាក្យបញ្ជា → Sync: Open activity log**

កំណត់ហេតុនេះផ្តល់ពេលវេលា និងព័ត៌មានលម្អិតសម្រាប់ប្រតិបត្តិការសមកាលកម្មនីមួយៗ ធ្វើឱ្យវាមានប្រយោជន៍សម្រាប់ការដោះស្រាយបញ្ហាសមកាលកម្ម។

> [!warning] កំណត់ហេតុ Sync មិនត្រូវបានរក្សាទុកបន្ទាប់ពី Obsidian ត្រូវបានបិទទេ។ ប្រសិនបើអ្នកជួបបញ្ហា សូមប្រាកដថាអ្នកចម្លងកំណត់ហេតុ _មុនពេល_ បិទកម្មវិធី។

កំណត់ហេតុនេះចែកសារជាប្រភេទដូចខាងក្រោម៖

- [[#សារទូទៅ]]
- [[#សារកំហុស]]
- [[#សារដែលបានរំលង]]
- [[#សារគណនី]]

អ្នកអាចត្រងកំណត់ហេតុ Sync តាម **ទាំងអស់** **កំហុស** **បានរំលង** និង **ជម្លោះការរួមបញ្ចូល**។ លើសពីនេះ អ្នកអាចស្វែងរកកំណត់ហេតុ Sync ដោយប្រើប្រអប់ស្វែងរកនៅក្នុងបង្អួច Sync។

> [!summary] យើងបានរួមបញ្ចូលសារមួយចំនួនដែលអ្នកទំនងជាបានជួបខាងក្រោម។ បញ្ជីនេះមិនទាំងអស់ទេ។ ប្រសិនបើអ្នកកំពុងជួបបញ្ហា ហើយមានសារកំណត់ហេតុ sync ដែលអ្នកមិនប្រាកដ សូម [[ជំនួយ និងការគាំទ្រ#Contact Obsidian support|ទាក់ទងផ្នែកគាំទ្រ Obsidian]]។

### សារទូទៅ

ទាំងនេះគឺជាសារប្រចាំថ្ងៃធម្មតាដែលអ្នកអាចជួប។

**Connecting to server**  
Obsidian កំពុងព្យាយាមភ្ជាប់ទៅ [[Obsidian Sync/សុវត្ថិភាព និងឯកជនភាព#Where can I find my current Sync server and where is it hosted?|ម៉ាស៊ីនមេ Sync]] នៃ vault ពីចម្ងាយរបស់អ្នក។

**Connected to server. Detecting changes...**  
Obsidian បានបង្កើតការភ្ជាប់ ហើយកំពុងប្រៀបធៀប vault មូលដ្ឋានជាមួយ vault ពីចម្ងាយដើម្បីកំណត់ថាតើការផ្លាស់ប្តូរចាំបាច់ឬអត់។

> [!info] សារនេះក៏អាចបង្ហាញពីបញ្ហា Sync ផ្សេងទៀតផងដែរ។ ប្រសិនបើអ្នកឃើញវាម្តងហើយម្តងទៀត ហើយជឿថានៅមានធាតុដែលត្រូវសមកាលកម្ម សូម [[ជំនួយ និងការគាំទ្រ#Contact Obsidian support|ទាក់ទងផ្នែកគាំទ្រ Obsidian]]។

**Fully synced**  
- Vault មូលដ្ឋាន និង vault ពីចម្ងាយត្រូវបានសមកាលកម្មពេញលេញ។

**Merging conflicted file**  
ជម្លោះមួយត្រូវបានរកឃើញក្នុងពេលសមកាលកម្ម ហើយឯកសារត្រូវបានរួមបញ្ចូលជំនួសឱ្យការសរសេរជាន់លើ។ សូមមើល [[ដោះស្រាយបញ្ហា Obsidian Sync#ការដោះស្រាយជម្លោះ|ការដោះស្រាយជម្លោះ]] សម្រាប់ព័ត៌មានបន្ថែម។ ប្រសិនបើការរួមបញ្ចូលមិនចង់បាន អ្នកអាចស្ដារកំណែមុនតាមរយៈ [[ប្រវត្តិកំណែ]] ឬ [[ការស្តារឯកសារឡើងវិញ]]។

**Rejected server change**  
ការផ្លាស់ប្តូរនៅលើ vault ពីចម្ងាយចាស់ជាងកំណែនៅលើឧបករណ៍មូលដ្ឋានរបស់អ្នក ដូច្នេះកំណែមូលដ្ឋានត្រូវបានរក្សាទុក ហើយការផ្លាស់ប្តូរពីចម្ងាយត្រូវបានមិនអើពើ។

### សារកំហុស

ទាំងនេះគឺជាសារដែលរៀបរាប់ពីកំហុសក្នុងការសមកាលកម្មឯកសារ។

**Out of memory**  
បញ្ហានេះជាធម្មតាកើតឡើងនៅលើឧបករណ៍ចល័តនៅពេលដែលមិនមានទំហំផ្ទុក ឬអង្គចងចាំគ្រប់គ្រាន់ដើម្បីទាញយកឯកសារ។ វាកើតមានជារឿយៗជាមួយឯកសារធំ ដូចជាវីដេអូ។

### សារដែលបានរំលង

ទាំងនេះគឺជាសារដែលរៀបរាប់ពីអ្វីដែលត្រូវបានរំលង និងមូលហេតុ។

**Unable to download file with illegal name**

ឯកសារមាន [តួអក្សរពិសេស ឬអនុសញ្ញាដាក់ឈ្មោះ](https://stackoverflow.com/questions/1976007/what-characters-are-forbidden-in-windows-and-linux-directory-names) ដែលមិនត្រូវបានអនុញ្ញាតនៅលើប្រព័ន្ធប្រតិបត្តិការទទួល។ សម្រាប់ភាពងាយស្រួល អ្នកអាចប្តូរឈ្មោះឯកសារនៅលើឧបករណ៍ប្រភពដើមដើម្បីដកតួអក្សរពិសេសទាំងអស់ លើកលែងតែ `-` និង `_`។

សូមចំណាំថានេះក៏រួមបញ្ចូលឯកសារដែលមានចំណុច `.` ច្រើននៅក្នុងឈ្មោះរបស់ពួកគេនៅលើឧបករណ៍ Android ផងដែរ។

### សារគណនី

ទាំងនេះគឺជាសារដែលទាក់ទងនឹងការផ្លាស់ប្តូរនៅក្នុងការជាវ ឬគណនីរបស់អ្នក។

**Vault limit exceeded**  
គណនីរបស់អ្នកបានលើសពី [[សំណួរដែលគេសួរញឹកញាប់#How large can each remote vault be|ទំហំផ្ទុកអតិបរមា]]។ ឯកសារភ្ជាប់ និងប្រវត្តិកំណែរួមចំណែកដល់ទំហំនេះ។ ទោះបីជា vault របស់អ្នកហាក់ដូចជាតូចជាងដែនកំណត់ក៏ដោយ កំណែចាស់ និងឯកសារអាចធ្វើឱ្យវាលើស។

ដើម្បីកាត់បន្ថយទំហំ vault៖
1. បើក **[[ការកំណត់]] → Sync**។
2. ប្រើជម្រើសនៅក្រោម **Vault size over limit** ដើម្បីលុបឯកសារធំ។

**Vault not found**  
`{"res":"err","msg":"Vault not found."}`

កំហុសនេះអាចកើតឡើងក្នុងករណីទាំងនេះ៖

1. Vault ត្រូវបានលុបពីឧបករណ៍ផ្សេង។
2. ការជាវ sync មិនសកម្មជាង 30 ថ្ងៃ ធ្វើឱ្យ vault ពីចម្ងាយត្រូវបានដកចេញ។
