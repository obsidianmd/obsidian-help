Obsidian бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҖбҹ’бһ“бһ»бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһ‘бһ¶бҹҶбһ„бһ–бҹҒбһӣбһӮбҹ’бһҳбһ¶бһ“бһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһңбһ·бһ’бһёбһҹбһҳбһҖбһ¶бһӣбҹ”

бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһ“бҹҒбҹҮбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҠбһ”бҹӢбһңбһ·бһ’бһёбһҹбһҳбһҖбһ¶бһӣбһ‘бһјбһ‘бҹ… бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„бһӮбһ“бҹ’бһӣбһ№бҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡбһ”бһ¶бһҸбҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһ’бһ¶бһ“бһ¶бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҡбһӣбһјбһ“бҹ”

бһҷбһҫбһ„бһҖбҹҸбһҺбҹӮбһ“бһ¶бҹҶбһұбҹ’бһҷбһўбһ¶бһ“бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ [[бһ”бһҳбҹ’бһҡбһ»бһ„бһ‘бһ»бһҖбһҜбһҖбһҹбһ¶бһҡ Obsidian бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ|бһ”бһҳбҹ’бһҡбһ»бһ„бһ‘бһ»бһҖ]] бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҡбһ–бһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

## бһңбһ·бһ’бһёбһҹбһҳбһҖбһ¶бһӣ

бһҮбһ¶бһҳбһҪбһҷ Obsidian бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҮбһ¶бһҜбһҖбһҹбһ¶бһҡбһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһҠбҹӮбһӣбһ бҹ…бһҗбһ¶ [[Vault бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһ“бһ·бһ„ Vault бһ–бһёбһ…бһҳбҹ’бһ„бһ¶бһҷ|vault]]бҹ” бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҳбһ¶бһ“бһңбһ·бһ’бһёбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһҹбһҳбһҖбһ¶бһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ”

бһ“бҹҒбҹҮбһҮбһ¶бһңбһ·бһ’бһёбһҹбһҳбһҖбһ¶бһӣбһ‘бһјбһ‘бҹ…бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһҠбҹ„бһҷбһҹбһҳбһ¶бһҮбһ·бһҖбһҹбһ бһӮбһҳбһ“бҹҚ Obsidian:

1. **бһҹбһҳбһҖбһ¶бһӣбһҠбҹҶбһ”бһјбһ„бһ—бһ¶бһӮбһё**: [[#Obsidian Sync]]
2. **бһҹбһҳбһҖбһ¶бһӣ cloud бһ—бһ¶бһӮбһёбһ‘бһёбһ”бһё**: [[#iCloud]], [[#OneDrive]], бһ“бһ·бһ„ [[#Google Drive]]
3. **бһҹбһҳбһҖбһ¶бһӣбһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢ**: [[#Syncthing]]
4. **бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹҶбһҺбҹӮ**: [[#Git]] бһ“бһ·бһ„ [[#Working Copy]]

бһҮбһҳбҹ’бһҡбһҫбһҹбһҹбһҳбһҖбһ¶бһӣбһҮбҹҶбһ“бһҪбһҹбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ‘бҹҖбһҸбһўбһ¶бһ…бһҡбһҖбһ”бһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ [бһҗбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбҹҶбһ“бһҪбһҷбһҹбһ бһӮбһҳбһ“бҹҚ](https://community.obsidian.md/search?type=plugin&categories=syncing)бҹ”

## Obsidian Sync

**бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣбһҺбҹӮбһ“бһ¶бҹҶ**: `Windows`, `macOS`, `Linux`, `iOS`, `Android`

бһңбһ·бһ’бһёбһҹбһҳбһҖбһ¶бһӣбһҠбҹҸбһҸбҹ’бһҡбһ„бҹӢ бһ“бһ·бһ„бһҳбһ¶бһ“бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһ”бҹҶбһ•бһ»бһҸбһӮбһәбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҠбҹҶбһ”бһјбһ„бһ—бһ¶бһӮбһёбһҡбһ”бһҹбҹӢбһҷбһҫбһ„: [[бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһўбҹҶбһ–бһё Obsidian Sync|Obsidian Sync]]бҹ”

Obsidian Sync бһ’бҹ’бһңбһҫбһұбҹ’бһҷ vault бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ’бҹ’бһңбһҫбһҹбһҳбһҖбһ¶бһӣбһҶбҹ’бһӣбһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ vault бһ–бһёбһ…бһҳбҹ’бһ„бһ¶бһҷбһҠбһҫбһҳбҹ’бһ”бһёбһ…бһҳбҹ’бһӣбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ” бһ…бҹ’бһ”бһ¶бһ”бҹӢбһ…бһҳбҹ’бһӣбһ„бһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢбһ“бҹ…бһҸбҹӮбһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбһ¶бһ“бһ·бһ…бҹ’бһ…бҹ”

бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳ [[бһҠбҹҶбһЎбһҫбһ„ Obsidian Sync|бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҠбҹҶбһЎбһҫбһ„]] бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢ Obsidian Syncбҹ”

## iCloud

**бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣбһҺбҹӮбһ“бһ¶бҹҶ**: `macOS`, `iOS`, `iPadOS`

iCloud бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бҹ’бһңбһҫбһҹбһҳбһҖбһ¶бһӣ vault бһҡбһңбһ¶бһ„ iOS бһ“бһ·бһ„ macOSбҹ” бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮ **iCloud Drive бһӣбһҫ Windows** бһўбһ¶бһ…бһ“бһ¶бҹҶбһұбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһҹбҹ’бһ‘бһҪбһ“ бһ¬бһҒбһјбһ…бһҜбһҖбһҹбһ¶бһҡбҹ”

**бһҡбһ”бҹҖбһ”бһ”бһ„бҹ’бһҖбһҫбһҸбһ“бһ·бһ„бһ‘бһ»бһҖ vault бһҖбҹ’бһ“бһ»бһ„ iCloud Drive**:

- **бһ”бһҫбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ iCloud Drive**:
    - бһӣбһҫ macOS: бһ…бһјбһӣ **System Preferences вҶ’ Apple ID вҶ’ iCloud вҶ’ iCloud Drive**
    - бһӣбһҫ iOS: бһ…бһјбһӣ **Settings вҶ’ [бһҲбҹ’бһҳбҹ„бҹҮбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ] вҶ’ iCloud вҶ’ iCloud Drive**
- **бһ”бһ„бҹ’бһҖбһҫбһҸ vault бһҗбҹ’бһҳбһёбһҖбҹ’бһ“бһ»бһ„ iCloud**:
    - бһӣбһҫ macOS:
        1. бһ”бһҫбһҖ **Obsidian** бһ бһҫбһҷбһҮбҹ’бһҡбһҫбһҹ **Create new vault**
        2. бһҖбҹ’бһ“бһ»бһ„ file picker бһҡбһ»бһҖбһҡбһҖбһ‘бҹ… **iCloud Drive вҶ’ Obsidian**
        3. бһ”бһ„бҹ’бһҖбһҫбһҸбһҗбһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ vault бһ бһҫбһҷбһҠбҹҶбһЎбһҫбһ„бһҲбҹ’бһҳбҹ„бҹҮ
        4. бһҮбҹ’бһҡбһҫбһҹ **Create** бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһ”бҹӢ
    - бһӣбһҫ iOS:
        1. бһ”бһҫбһҖ **Obsidian** бһ бһҫбһҷбһ…бһ»бһ… **Create new vault**
        2. бһ”бһүбҹ’бһ…бһјбһӣбһҲбҹ’бһҳбҹ„бҹҮ vault
        3. бһ…бһ»бһ…бһ”бһҫбһҖ **Store in iCloud**
        4. бһ…бһ»бһ… **Create**
- **бһ”бһҫбһҖ vault бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ Apple бһ•бҹ’бһҹбҹҒбһ„**:
    - бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ macOS бһ¬ iOS бһ•бҹ’бһҹбҹҒбһ„ бһ”бһҫбһҖ **Obsidian** бһ…бһјбһӣ [[бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„вҖӢбһңбҹүбһ¶бһӣбҹӢ|бһ”бҹ’бһҸбһјбһҡ vault]] бһ бһҫбһҷбһҮбҹ’бһҡбһҫбһҹ **Open folder as vault**бҹ” бһҡбһ»бһҖбһҡбһҖбһ‘бҹ… **iCloud Drive вҶ’ Obsidian**

> [!warning]+ бһ‘бһёбһҸбһ¶бҹҶбһ„бһҗбһҸбһӣбһҫ iOS бһ“бһ·бһ„ iPadOS
> бһ“бҹ…бһ–бҹҒбһӣбһ”бҹ’бһҡбһҫ iCloud бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ mobile бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶ vault бһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһ‘бһёбһҸбһ¶бҹҶбһ„бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң: `iCloud Drive/Obsidian/[бһҲбҹ’бһҳбҹ„бҹҮ Vault бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ]`бҹ”
>

> [!info] бһ‘бһ»бһҖбһҜбһҖбһҹбһ¶бһҡбһўбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҖбҹ’бһҡбҹ…бһ”бһҺбҹ’бһҠбһ¶бһү
> бһҳбһ»бһ“бһ“бһ№бһ„бһ”бҹ’бһҡбһҫ OneDrive бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһҗбһҸ vault бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢ **Always keep on this device**бҹ” бһҖбһ¶бһҡбһ“бҹҒбҹҮбһҖбһ¶бһҡбһ–бһ¶бһҡ OneDrive бһ–бһёбһҖбһ¶бһҡ offload бһҜбһҖбһҹбһ¶бһҡ бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһұбҹ’бһҷ Obsidian бһӮбһ·бһҸбһҗбһ¶бһҜбһҖбһҹбһ¶бһҡбһ”бһ¶бһҸбҹӢбҹ”

**бһҡбһ”бҹҖбһ”бһ”бһ„бҹ’бһҖбһҫбһҸбһ“бһ·бһ„бһ‘бһ»бһҖ vault бһҖбҹ’бһ“бһ»бһ„ OneDrive**:

   - бһӣбһҫ macOS: бһ‘бһ¶бһүбһҷбһҖ OneDrive app бһ бһҫбһҷбһ…бһјбһӣ
2. **бһ”бһ„бҹ’бһҖбһҫбһҸ vault бһҗбҹ’бһҳбһёбһҖбҹ’бһ“бһ»бһ„ OneDrive**:

3. **бһ”бһҫбһҖ vault бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһҹбҹҒбһ„**:

> [!info] бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҹбһҳбһҖбһ¶бһӣ Android
> OneDrive бһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһҳбһ·бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Android бҹ” бһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһ”бҹ’бһҡбһҫ app бһҠбһјбһ…бһҮбһ¶ [Dropsync](https://play.google.com/store/apps/details?id=com.ttxapps.dropsync) бһ¬ [FolderSync](https://play.google.com/store/apps/details?id=dk.tacit.android.foldersync.lite)бҹ”

> [!tip] бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһў
> - бһҡбһҖбҹ’бһҹбһ¶бһҜбһҖбһҹбһ¶бһҡ vault бһұбҹ’бһҷ **Available Offline** бһҮбһ¶бһ“бһ·бһ…бҹ’бһ… бһҠбҹ„бһҷбһ…бһ»бһ…бһ”бҹ’бһҠбһјбһҡбһҡбһјбһ”бһҗбһҸбһӣбһҫбһҗбһҸ бһ бһҫбһҷбһҮбҹ’бһҡбһҫбһҹ **Always keep on this device**

## Syncthing

**бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣбһҺбҹӮбһ“бһ¶бҹҶ**: `Windows`, `macOS`, `Linux`

Syncthing бһӮбһәбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһҳбһҖбһ¶бһӣбһҜбһҖбһҹбһ¶бһҡбһҠбҹӮбһӣбһҳбһ·бһ“бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷ cloud storageбҹ” бһңбһ¶бһ’бҹ’бһңбһҫбһҹбһҳбһҖбһ¶бһӣ vault бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҡбһңбһ¶бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҸбһ¶бһҳбһ”бһҺбҹ’бһҠбһ¶бһүбҹ”

> [!info]+ бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡ Android
> Syncthing app бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡ Android бһҳбһ·бһ“бһҳбҹӮбһ“бһҳбһ¶бһ“бһҖбһ¶бһҡбһҗбҹӮбһ‘бһ¶бҹҶбһ‘бҹҖбһҸбһ бһҫбһҷбҹ” бһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶бһҷбҹүбһ¶бһ„бһҺбһ¶ бһҖбһ¶бһҡ fork бһҹбһ бһӮбһҳбһ“бҹҚ [Syncthing-Fork](https://github.com/Catfriend1/syncthing-android) бһ“бҹ…бһҸбҹӮбһ”бһ“бҹ’бһҸбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ бһ бһҫбһҷбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһӣбһҫ Android бһ”бһ¶бһ“бҹ”

**бһҡбһ”бҹҖбһ”бһ”бһ„бҹ’бһҖбһҫбһҸбһ“бһ·бһ„бһ‘бһ»бһҖ vault бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Syncthing**:

1. **бһҠбҹҶбһЎбһҫбһ„ Syncthing**:
   - бһҠбҹҶбһЎбһҫбһ„ Syncthing бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ“бһёбһҳбһҪбһҷбҹ—бҹ” бһҹбһјбһҳбһҳбһҫбһӣ [бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡ Syncthing](https://syncthing.net/) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ
   - бһӣбһҫ Android бһҠбҹҶбһЎбһҫбһ„ [Syncthing-Fork](https://github.com/Catfriend1/syncthing-android)
2. **бһ”бһ„бҹ’бһҖбһҫбһҸбһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢ shared folder**:
   - бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ:
     1. бһ”бһҫбһҖ Syncthing бһ бһҫбһҷбһ”бһ„бҹ’бһҖбһҫбһҸ shared folderбҹ” бһҖбҹҶбһҺбһҸбҹӢ folder path бһ‘бҹ… vault Obsidian
     2. бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶ folder бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҮбҹ’бһҡбһҫбһҹбһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ
     3. бһҖбҹҶбһҺбһҸбҹӢ folder syncing preferences (бһ§. **Send & Receive** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ bidirectional)
3. **бһ”бһҫбһҖ vault бһҖбҹ’бһ“бһ»бһ„ Obsidian**:
   - бһ“бҹ…бһ–бҹҒбһӣ folder бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһҹбһҳбһҖбһ¶бһӣбһҶбҹ’бһӣбһ„бһ§бһ”бһҖбһҡбһҺбҹҚ бһ”бһҫбһҖ **Obsidian** бһ…бһјбһӣ [[бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„вҖӢбһңбҹүбһ¶бһӣбҹӢ|бһ”бҹ’бһҸбһјбһҡ vault]] бһ бһҫбһҷбһҮбҹ’бһҡбһҫбһҹ **Open folder as vault**

> [!info] бһ—бһ¶бһ–бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҡбһ”бһҹбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ
> Syncthing бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһ“бҹ…бһ–бҹҒбһӣбһ§бһ”бһҖбһҡбһҺбҹҚбһҷбҹүбһ¶бһ„бһ бҹ„бһ…бһҺбһ¶бһҹбҹӢбһҳбһҪбһҷ бһҸбҹӮбһ„бһҸбҹӮбһ”бһҫбһҖ бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҹбһҳбһҖбһ¶бһӣбһҠбҹӮбһӣбһҳбһ·бһ“бһҡбҹҶбһҒбһ¶бһ“бҹ”

> [!tip] бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһў
> - бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ local sync бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһ”бһҺбҹ’бһҠбһ¶бһүбһҸбҹӮбһҳбһҪбһҷ
> - бһҠбһҖ `.obsidian` бһ–бһёбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҹбһҳбһҖбһ¶бһӣ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бһ¶бһ“бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҠбһ¶бһ…бҹӢбһҠбҹ„бһҷбһЎбҹӮбһҖ
> - бһ”бҹ’бһҡбһҫ ignore patterns бһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡбһҹбһҳбһҖбһ¶бһӣбһҜбһҖбһҹбһ¶бһҡбһ”бһҺбҹ’бһҠбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“ бһ¬ backup

## Git

**бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣбһҺбҹӮбһ“бһ¶бҹҶ**: `Windows`, `macOS`, `Linux`

**Git** бһӮбһәбһҮбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹҶбһҺбҹӮ бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡ бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһ бһҖбһ¶бһҡ бһ“бһ·бһ„бһ’бҹ’бһңбһҫбһҹбһҳбһҖбһ¶бһӣ vault бһҸбһ¶бһҳбһҡбһҷбҹҲ repository бһҠбһјбһ…бһҮбһ¶ GitHub, GitLab бһ¬ server бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бҹ”

**бһҡбһ”бҹҖбһ”бһ’бҹ’бһңбһҫбһҹбһҳбһҖбһ¶бһӣ vault бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Git**:

1. **бһҠбҹҶбһЎбһҫбһ„ remote repository**:
    - бһ”бһ„бҹ’бһҖбһҫбһҸ repository бһӣбһҫ Git hosting platform (бһ§. GitHub, GitLab бһ¬ server бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“)
2. **бһ’бҹ’бһңбһҫбһҹбһҳбһҖбһ¶бһӣ vault**:
    1. бһ”бһҫбһҖ terminal бһ¬ Git GUI (бһ§. GitKraken, Sourcetree)
    2. Initialize Git repository бһҖбҹ’бһ“бһ»бһ„бһҗбһҸ vault бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ `git init`
    3. бһ”бһ“бҹ’бһҗбҹӮбһҳ remote repository: `git remote add origin [URL]`
    4. Commit бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡ: `git add .` бһ“бһ·бһ„ `git commit -m "бһҹбһ¶бһҡ"`
    5. Push бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡ: `git push origin main`
3. **Pull бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһҹбҹҒбһ„**:
    - Clone repository бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһҹбҹҒбһ„ бһ бһҫбһҷ pull бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ `git pull origin main`

> [!info] бһҸбҹ’бһҡбһјбһңбһ’бҹ’бһңбһҫбһҹбһҳбһҖбһ¶бһӣбһҠбҹ„бһҷбһҒбҹ’бһӣбһҪбһ“бһҜбһ„
> Git бһ•бҹ’бһҠбһӣбҹӢбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹҶбһҺбҹӮбһҠбҹҸбһҒбҹ’бһӣбһ¶бҹҶбһ„ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҖбһ¶бһҡбһҹбһҳбһҖбһ¶бһӣбһҳбһ·бһ“бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ” бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһң push бһ“бһ·бһ„ pull бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһҠбҹ„бһҷбһҠбҹғбҹ”

### Working Copy

**бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣбһҺбҹӮбһ“бһ¶бҹҶ**: `iOS`
**бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ**: [[#Git]]

**Working Copy** бһӮбһәбһҮбһ¶ Git client бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ iOS бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷ clone, commit бһ“бһ·бһ„ push бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһ‘бҹ… Git repositoryбҹ” бһңбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Git vault sync бһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶бһҳбһ»бһҒбһ„бһ¶бһҡбһҒбҹ’бһӣбҹҮбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбһ¶бһҡбһ‘бһјбһ‘бһ¶бһҸбҹӢбһҖбҹ’бһ“бһ»бһ„ appбҹ”

**бһҡбһ”бҹҖбһ”бһ’бҹ’бһңбһҫбһҹбһҳбһҖбһ¶бһӣ vault бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Working Copy**:

1. **бһҠбҹҶбһЎбһҫбһ„ Working Copy**:
    - бһ‘бһ¶бһүбһҷбһҖ app **[Working Copy](https://apps.apple.com/us/app/working-copy-git-client/id896694807)** бһӣбһҫ iPhone бһ¬ iPad
2. **Clone Git repository**:
    - бһ”бһҫбһҖ Working Copy бһ…бһ»бһ… **Add Repository** бһ бһҫбһҷбһ”бһүбҹ’бһ…бһјбһӣ URL repository (бһ§. GitHub, GitLab)
3. **бһ—бҹ’бһҮбһ¶бһ”бҹӢ repository бһ‘бҹ… Obsidian**:
    - бһ—бҹ’бһҮбһ¶бһ”бҹӢ repository folder бһҠбҹӮбһӣ clone бһ‘бҹ… vault бһ‘бһ‘бҹҒбһҖбҹ’бһ“бһ»бһ„ **Obsidian**
4. **Commit бһ“бһ·бһ„ push бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡ**:
    - бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҖбҹӮбһ”бҹ’бһҡбҹӮбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҖбҹ’бһ“бһ»бһ„ Obsidian бһ”бҹ’бһҡбһҫ Working Copy бһҠбһҫбһҳбҹ’бһ”бһё **Commit** бһ“бһ·бһ„ **Push**
    - бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһҹбҹҒбһ„ pull бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Git бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһҹбһҳбһҖбһ¶бһӣ vault

> [!info] бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҹбһ бһӮбһҳбһ“бҹҚ
> бһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶ Working Copy бһҳбһ·бһ“бһҳбһ¶бһ“бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡ бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһңбһ¶бһҠбҹ„бһҷбһҮбҹ„бһӮбһҮбҹҗбһҷбҹ”

**бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһҒбҹ’бһүбһ»бҹҶбһҸбҹ’бһҡбһјбһңбһ‘бһ»бһҖ "Available Offline"?**

**бһҸбһҫбһҒбҹ’бһүбһ»бҹҶбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҒбһ»бһҹбҹ—бһӮбҹ’бһ“бһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ vault бһҠбҹ„бһҷбһҡбһ”бҹҖбһ”бһҺбһ¶?**
