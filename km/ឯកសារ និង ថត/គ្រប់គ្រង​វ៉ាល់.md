**Vault** бһӮбһәбһҮбһ¶бһҗбһҸбһҳбһҪбһҷбһ“бҹ…бһӣбһҫбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҜбһҖбһҹбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹӮбһӣбһ•бҹ’бһ‘бһ»бһҖбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶ [[бһҜбһҖбһҹбһ¶бһҡбһ—бҹ’бһҮбһ¶бһ”бҹӢ]] бһ“бһ·бһ„ [[бһҗбһҸбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’]] бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҡбһ”бһҹбҹӢ Obsidianбҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһё vault бһҹбһјбһҳбһҷбҹ„бһ„бһ‘бҹ… [[бһҡбһ”бҹҖбһ”бһҠбҹӮбһӣ Obsidian бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ]]бҹ”

бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ vault бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ **Vault switcher**бҹ” *Vault profile* бһ“бһ№бһ„бһ”бһҫбһҖбһ“бҹ…бһ–бҹҒбһӣбһҠбҹҶбһ”бһјбһ„бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһҫбһҖ Obsidianбҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҫбһҖ vault switcher бһ–бһё vault бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢ бһҹбһјбһҳбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ **Vault profile** ![[lucide-chevrons-up-down.svg#icon]] бһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹғ [[бһҡбһ”бһ¶бһҡбһ…бҹҶбһ бҹҖбһ„|бһҡбһ”бһ¶бһҡбһ…бҹҶбһ бҹҖбһ„бһҒбһ¶бһ„бһҶбҹ’бһңбҹҒбһ„]]бҹ” бһ¬ бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ **Open another vault** бһ–бһё [[бһҖбҹ’бһҠбһ¶бһҡбһӣбһ¶бһҷбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶]]бҹ”

## бһ”бһ„бҹ’бһҖбһҫбһҸ vault бһҗбҹ’бһҳбһё

1. бһ“бҹ…бһӣбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ”бһҫбһҖ Obsidianбҹ”
2. бһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҶбҹ’бһңбҹҒбһ„ бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ **Vault profile** ![[lucide-chevrons-up-down.svg#icon]]бҹ”
3. бһҳбҹүбһәбһ“бһ»бһҷбһ”бһҡбһ·бһ”бһ‘бһ“бһ№бһ„бһӣбҹҒбһ…бһЎбһҫбһ„бҹ” бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ **Manage Vaults...**бҹ”
3. бһ“бҹ…бһҒбһ¶бһ„бһҹбҹ’бһҸбһ¶бҹҶбһ“бҹғ **Create new vault** бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ **Create**бҹ”
4. бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ **Vault name** бһ”бһүбҹ’бһ…бһјбһӣбһҲбҹ’бһҳбҹ„бҹҮ vault бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
5. бһ…бһ»бһ… **Browse** бһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбҹӮбһӣ vault бһҗбҹ’бһҳбһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҖбһҫбһҸбҹ”
6. бһ…бһ»бһ… **Create**бҹ”

## бһ”бһ„бҹ’бһҖбһҫбһҸ vault бһ–бһёбһҗбһҸбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢ

1. бһ“бҹ…бһӣбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ”бһҫбһҖ Obsidianбҹ”
2. бһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҶбҹ’бһңбҹҒбһ„ бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ **Vault profile** ![[lucide-chevrons-up-down.svg#icon]]бҹ”
3. бһҳбҹүбһәбһ“бһ»бһҷбһ”бһҡбһ·бһ”бһ‘бһ“бһ№бһ„бһӣбҹҒбһ…бһЎбһҫбһ„бҹ” бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ **Manage Vaults...**бҹ”
3. бһ“бҹ…бһҒбһ¶бһ„бһҹбҹ’бһҸбһ¶бҹҶбһ“бҹғ **Open folder as vault** бһ…бһ»бһ… **Open**бҹ”
4. бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҜбһҖбһҹбһ¶бһҡ бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҗбһҸбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бҹ’бһҡбһҫбһҮбһ¶ vault бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
5. бһ…бһ»бһ… **Open**бҹ”

> [!tip] бһ”бһҫбһҖ vault бһ–бһё Obsidian Sync
> бһ”бһҫбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бһҫбһҖ vault бһ–бһёбһ…бһҳбҹ’бһ„бһ¶бһҷбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Obsidian Sync бһҹбһјбһҳбһҷбҹ„бһ„бһ‘бҹ… [[бһҠбҹҶбһЎбһҫбһ„ Obsidian Sync]]бҹ”

## бһ”бҹ’бһҸбһјбһҡбһҲбҹ’бһҳбҹ„бҹҮ vault

бһҠбҹ„бһҷбһҹбһ¶бһҡбһҲбҹ’бһҳбҹ„бҹҮ vault бһ“бһ·бһ„бһҗбһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӮбһәбһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһҖбһ¶бһҡбһ”бҹ’бһҸбһјбһҡбһҲбҹ’бһҳбҹ„бҹҮ vault бһҖбҹҸбһ”бҹ’бһҸбһјбһҡбһҲбҹ’бһҳбҹ„бҹҮбһҗбһҸбһҠбҹӮбһҡбҹ”

1. бһ“бҹ…бһӣбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ”бһҫбһҖ Obsidianбҹ”
2. бһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҶбҹ’бһңбҹҒбһ„ бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ **Vault profile** ![[lucide-chevrons-up-down.svg#icon]]бҹ”
3. бһҳбҹүбһәбһ“бһ»бһҷбһ”бһҡбһ·бһ”бһ‘бһ“бһ№бһ„бһӣбҹҒбһ…бһЎбһҫбһ„бҹ” бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ **Manage Vaults...**бҹ”
4. бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһүбҹ’бһҮбһё vault бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ **More options** ![[lucide-more-horizontal.svg#icon]] бһ“бҹ…бһҮбһ¶бһ”бҹӢ vault бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бҹ’бһҸбһјбһҡбһҲбҹ’бһҳбҹ„бҹҮбҹ”
5. бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ **Rename vault**бҹ”
6. бһ”бһүбҹ’бһ…бһјбһӣбһҲбҹ’бһҳбҹ„бҹҮбһҗбҹ’бһҳбһёбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ vault бһҡбһҪбһ…бһ…бһ»бһ… `Enter`бҹ”

## бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бһё vault бһ‘бҹ…бһҗбһҸбһ•бҹ’бһҹбҹҒбһ„

1. бһ“бҹ…бһӣбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ”бһҫбһҖ Obsidianбҹ”
2. бһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҶбҹ’бһңбҹҒбһ„ бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ **Vault profile** ![[lucide-chevrons-up-down.svg#icon]]бҹ”
3. бһҳбҹүбһәбһ“бһ»бһҷбһ”бһҡбһ·бһ”бһ‘бһ“бһ№бһ„бһӣбҹҒбһ…бһЎбһҫбһ„бҹ” бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ **Manage Vaults...**бҹ”
4. бһ”бһ·бһ‘бһ”бһ„бҹ’бһўбһҪбһ… vault бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ бһҠбҹ„бһҷбһ‘бһ»бһҖбһ”бһ„бҹ’бһўбһҪбһ… **Manage Vaults** бһ”бһҫбһҖбһ…бҹ„бһӣбҹ”
5. бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһүбҹ’бһҮбһё vault бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ **More options** ![[lucide-more-horizontal.svg#icon]] бһ“бҹ…бһҮбһ¶бһ”бҹӢ vault бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бһёбҹ”
6. бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ **Move vault** бһҡбһҪбһ…бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ‘бһёбһҸбһ¶бҹҶбһ„бһҗбҹ’бһҳбһёбҹ”

бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҳбһ·бһ“бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бһё vault бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Vault Switcher бһ‘бҹҒбҹ” бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бһё vault бһҠбҹ„бһҷбһҠбҹғбҹ–

1. бһ”бһ·бһ‘ Obsidianбҹ”
2. бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бһёбһҗбһҸ vault бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ…бһ‘бһёбһҸбһ¶бҹҶбһ„бһҗбҹ’бһҳбһё бһҠбҹ„бһҷбһҮбҹҖбһҹбһңбһ¶бһ„бһҗбһҸбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҳбһҖбһ¶бһӣбһҖбһҳбҹ’бһҳбһҠбҹ„бһҷбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ•бҹ’бһҹбҹҒбһ„бҹ”
3. бһ”бһҫбһҖ Obsidian бһЎбһҫбһ„бһңбһ·бһүбҹ”
4. бһ…бһ»бһ…бһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„ **Vault profile** бһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҶбҹ’бһңбҹҒбһ„ ![[lucide-chevrons-up-down.svg#icon]]бҹ”
5. бһ–бһёбһҳбҹүбһәбһ“бһ»бһҷбһӣбҹҒбһ…бһЎбһҫбһ„ бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ **Manage Vaults...**бҹ”
6. бһ“бҹ…бһҮбһ¶бһ”бҹӢ **Open folder as vault** бһ…бһ»бһ… **Open**бҹ”
7. бһҡбһ»бһҖбһҡбһҖ бһ“бһ·бһ„бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҗбһҸ vault бһҗбҹ’бһҳбһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
8. бһ…бһ»бһ… **Open**бҹ”
9. бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҗбһ¶бһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡ vault бһҳбһ·бһ“бһҳбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбҹ” бһ”бһҫбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢ бһ”бһҫбһҖбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбҹҶбһ“бһҪбһҷбһҹбһ бһӮбһҳбһ“бҹҚбһЎбһҫбһ„бһңбһ·бһүбһҠбҹ„бһҷбһ…бһјбһӣбһ‘бҹ… **[[бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ]] вҶ’ Community Plugins вҶ’ Turn restricted mode off**бҹ”

## бһӣбһ»бһ” vault

4. бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһүбҹ’бһҮбһё vault бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ **More options** ![[lucide-more-horizontal.svg#icon]] бһ“бҹ…бһҮбһ¶бһ”бҹӢ vault бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһӣбһ»бһ”бҹ”
5. бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ **Remove from list**бҹ”

## бһ•бҹ’бһ‘бҹҒбһҡбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ‘бҹ… vault бһҳбһҪбһҷбһ‘бҹҖбһҸ

бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ vault бһҳбһҪбһҷбһ‘бҹҖбһҸ бһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҜбһҖбһҹбһ¶бһҡ (бһ¬ terminal) бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһҳбҹ’бһӣбһ„бһҗбһҸ `.obsidian` бһ–бһёбһ«бһҹбһ“бҹғ vault бһ”бҹ’бһҡбһ—бһ–бһ‘бҹ…бһ«бһҹбһ“бҹғ vault бһӮбҹ„бһӣбһҠбҹ…бҹ”

бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһҸбҹ’бһҡбһјбһңбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ Obsidian бһЎбһҫбһ„бһңбһ·бһүбһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбҹ”
