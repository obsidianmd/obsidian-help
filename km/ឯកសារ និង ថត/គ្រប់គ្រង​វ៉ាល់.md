**Vault** គឺជាថតមួយនៅលើប្រព័ន្ធឯកសាររបស់អ្នកដែលផ្ទុកកំណត់ត្រា [[ឯកសារភ្ជាប់]] និង [[ថតកំណត់រចនាសម្ព័ន្ធ]] ជាមួយការកំណត់ជាក់លាក់របស់ Obsidian។ សម្រាប់ព័ត៌មានបន្ថែមអំពី vault សូមយោងទៅ [[របៀបដែល Obsidian រក្សាទុកទិន្នន័យ]]។

អ្នកអាចគ្រប់គ្រង vault របស់អ្នកដោយប្រើ **Vault switcher**។ *Vault profile* នឹងបើកនៅពេលដំបូងដែលអ្នកបើក Obsidian។

ដើម្បីបើក vault switcher ពី vault ដែលមានស្រាប់ សូមជ្រើសរើស **Vault profile** ![[lucide-chevrons-up-down.svg#icon]] នៅខាងក្រោមនៃ [[របារចំហៀង|របារចំហៀងខាងឆ្វេង]]។ ឬ ជ្រើសរើស **Open another vault** ពី [[ក្ដារលាយពាក្យបញ្ជា]]។

## បង្កើត vault ថ្មី

1. នៅលើកុំព្យូទ័ររបស់អ្នក បើក Obsidian។
2. នៅខាងក្រោមឆ្វេង ជ្រើសរើស **Vault profile** ![[lucide-chevrons-up-down.svg#icon]]។
3. ម៉ឺនុយបរិបទនឹងលេចឡើង។ ជ្រើសរើស **Manage Vaults...**។
3. នៅខាងស្តាំនៃ **Create new vault** ជ្រើសរើស **Create**។
4. នៅក្នុង **Vault name** បញ្ចូលឈ្មោះ vault របស់អ្នក។
5. ចុច **Browse** ដើម្បីជ្រើសរើសកន្លែងដែល vault ថ្មីរបស់អ្នកនឹងត្រូវបង្កើត។
6. ចុច **Create**។

## បង្កើត vault ពីថតដែលមានស្រាប់

1. នៅលើកុំព្យូទ័ររបស់អ្នក បើក Obsidian។
2. នៅខាងក្រោមឆ្វេង ជ្រើសរើស **Vault profile** ![[lucide-chevrons-up-down.svg#icon]]។
3. ម៉ឺនុយបរិបទនឹងលេចឡើង។ ជ្រើសរើស **Manage Vaults...**។
3. នៅខាងស្តាំនៃ **Open folder as vault** ចុច **Open**។
4. នៅក្នុងកម្មវិធីរុករកឯកសារ ជ្រើសរើសថតដែលអ្នកចង់ប្រើជា vault របស់អ្នក។
5. ចុច **Open**។

> [!tip] បើក vault ពី Obsidian Sync
> បើអ្នកចង់បើក vault ពីចម្ងាយដោយប្រើ Obsidian Sync សូមយោងទៅ [[ដំឡើង Obsidian Sync]]។

## ប្តូរឈ្មោះ vault

ដោយសារឈ្មោះ vault និងថតមូលដ្ឋានគឺដូចគ្នា ការប្តូរឈ្មោះ vault ក៏ប្តូរឈ្មោះថតដែរ។

1. នៅលើកុំព្យូទ័ររបស់អ្នក បើក Obsidian។
2. នៅខាងក្រោមឆ្វេង ជ្រើសរើស **Vault profile** ![[lucide-chevrons-up-down.svg#icon]]។
3. ម៉ឺនុយបរិបទនឹងលេចឡើង។ ជ្រើសរើស **Manage Vaults...**។
4. នៅក្នុងបញ្ជី vault ជ្រើសរើស **More options** ![[lucide-more-horizontal.svg#icon]] នៅជាប់ vault ដែលអ្នកចង់ប្តូរឈ្មោះ។
5. ជ្រើសរើស **Rename vault**។
6. បញ្ចូលឈ្មោះថ្មីសម្រាប់ vault រួចចុច `Enter`។

## ផ្លាស់ទី vault ទៅថតផ្សេង

1. នៅលើកុំព្យូទ័ររបស់អ្នក បើក Obsidian។
2. នៅខាងក្រោមឆ្វេង ជ្រើសរើស **Vault profile** ![[lucide-chevrons-up-down.svg#icon]]។
3. ម៉ឺនុយបរិបទនឹងលេចឡើង។ ជ្រើសរើស **Manage Vaults...**។
4. បិទបង្អួច vault បច្ចុប្បន្ន ដោយទុកបង្អួច **Manage Vaults** បើកចោល។
5. នៅក្នុងបញ្ជី vault ជ្រើសរើស **More options** ![[lucide-more-horizontal.svg#icon]] នៅជាប់ vault ដែលអ្នកចង់ផ្លាស់ទី។
6. ជ្រើសរើស **Move vault** រួចជ្រើសរើសទីតាំងថ្មី។

ប្រព័ន្ធប្រតិបត្តិការមួយចំនួនមិនអនុញ្ញាតឱ្យផ្លាស់ទី vault ដោយប្រើ Vault Switcher ទេ។ ក្នុងករណីទាំងនេះ អ្នកនឹងត្រូវផ្លាស់ទី vault ដោយដៃ៖

1. បិទ Obsidian។
2. ផ្លាស់ទីថត vault របស់អ្នកទៅទីតាំងថ្មី ដោយជៀសវាងថតដែលត្រូវបានសមកាលកម្មដោយសេវាកម្មផ្សេង។
3. បើក Obsidian ឡើងវិញ។
4. ចុចរូបតំណាង **Vault profile** នៅខាងក្រោមឆ្វេង ![[lucide-chevrons-up-down.svg#icon]]។
5. ពីម៉ឺនុយលេចឡើង ជ្រើសរើស **Manage Vaults...**។
6. នៅជាប់ **Open folder as vault** ចុច **Open**។
7. រុករក និងជ្រើសរើសថត vault ថ្មីរបស់អ្នក។
8. ចុច **Open**។
9. ផ្ទៀងផ្ទាត់ថាខ្លឹមសារ vault មិនមានការផ្លាស់ប្តូរ។ បើចាំបាច់ បើកកម្មវិធីជំនួយសហគមន៍ឡើងវិញដោយចូលទៅ **[[ការកំណត់]] → Community Plugins → Turn restricted mode off**។

## លុប vault

ការលុប vault គ្រាន់តែលុបវាចេញពីបញ្ជី vault ប៉ុណ្ណោះ។

1. នៅលើកុំព្យូទ័ររបស់អ្នក បើក Obsidian។
2. នៅខាងក្រោមឆ្វេង ជ្រើសរើស **Vault profile** ![[lucide-chevrons-up-down.svg#icon]]។
3. ម៉ឺនុយបរិបទនឹងលេចឡើង។ ជ្រើសរើស **Manage Vaults...**។
4. នៅក្នុងបញ្ជី vault ជ្រើសរើស **More options** ![[lucide-more-horizontal.svg#icon]] នៅជាប់ vault ដែលអ្នកចង់លុប។
5. ជ្រើសរើស **Remove from list**។

## ផ្ទេរការកំណត់ទៅ vault មួយទៀត

ដើម្បីប្រើការកំណត់ដូចគ្នាសម្រាប់ vault មួយទៀត ប្រើកម្មវិធីគ្រប់គ្រងឯកសារ (ឬ terminal) ដើម្បីចម្លងថត `.obsidian` ពីឫសនៃ vault ប្រភពទៅឫសនៃ vault គោលដៅ។

អ្នកប្រហែលជាត្រូវចាប់ផ្តើម Obsidian ឡើងវិញដើម្បីអនុវត្តការផ្លាស់ប្តូរ។
