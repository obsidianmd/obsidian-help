You can manage files and folders in several ways, using [[ផ្លូវកាត់ក្តារចុច]], [[ក្ដារលាយពាក្យបញ្ជា|commands]], or [[កម្មវិធីរុករកឯកសារ]].

## Create a new note

To create a new file:

1. Press `Ctrl+N` (or `Cmd+N` on macOS).
2. Enter the name of the note and then press `Enter` to start editing the note.

You can also create notes using [[កម្មវិធីរុករកឯកសារ#Create a new note|File explorer]], or by selecting **Create new note** from the [[ក្ដារលាយពាក្យបញ្ជា]].

> [!hint] System character limitation
> Obsidian will respect the filename limitations of the operating system you create the note on. If you plan to [[សមកាលកម្មកំណត់ត្រារបស់អ្នកឆ្លងឧបករណ៍|sync your notes across devices]], make sure your filenames are [safe for other operating systems](https://stackoverflow.com/q/1976007).
^blockquote-system-limitation

## Rename a note

To rename an active note:

1. Select the name of the note at the top of the editor (or press `F2`).
2. Enter the new name and then press `Enter`.

When you rename a file, Obsidian automatically updates all the links to that file.

You can rename a note or folder without opening it, by using [[កម្មវិធីរុករកឯកសារ#Rename a file or folder|File explorer]]

## Delete a note

To delete a note, select **More options → Delete file** at the upper right of an active note.

Or, select **Delete current file** from the [[ក្ដារលាយពាក្យបញ្ជា]].

You can also delete a note or folder, using the [[កម្មវិធីរុករកឯកសារ#Delete a file or folder|File explorer]].

> [!note] What happens to files after I delete them?
> To change what happens to deleted files, select one of the following options under **[[ការកំណត់]] → Files & Links**:
>
> - **System trash**: By default, deleted files end up in the system trash for your operating system. To restore a file, use your preferred file manager.
> - **Obsidian trash**: You can send deleted files to a `.trash` folder in your vault.
> - **Permanently delete**: Files are immediately deleted without any means to restore them.
