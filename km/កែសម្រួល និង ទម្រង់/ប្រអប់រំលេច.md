ប្រើប្រអប់រំលេចដើម្បីបញ្ចូលមាតិកាបន្ថែមដោយមិនបំបែកលំហូរនៃកំណត់ត្រារបស់អ្នក។

ដើម្បីបង្កើតប្រអប់រំលេច សូមបន្ថែម `[!info]` ទៅបន្ទាត់ដំបូងនៃ blockquote ដែល `info` គឺជា _ឧត្តមសញ្ញាប្រភេទ_។ ឧត្តមសញ្ញាប្រភេទកំណត់រូបរាង និងអារម្មណ៍របស់ប្រអប់រំលេច។ ដើម្បីមើលប្រភេទទាំងអស់ដែលអាចប្រើបាន សូមមើល [[#ប្រភេទដែលគាំទ្រ]]។ ប្រអប់រំលេចក៏ត្រូវបានគាំទ្រដោយផ្ទាល់នៅលើ [[ការណែនាំអំពី Obsidian Publish|Obsidian Publish]]។

ដើម្បីរុំមាតិកាដែលមានស្រាប់ក្នុងប្រអប់រំលេច សូមជ្រើសរើសអត្ថបទ (រួមទាំងបញ្ជី ប្លុកកូដ។ល។) រួចដំណើរការពាក្យបញ្ជា `Insert callout`។ មាតិកាដែលបានជ្រើសរើសនឹងត្រូវបានរុំក្នុងប្រអប់រំលេចដោយស្វ័យប្រវត្តិ។

### ប្រអប់រំលេចដែលអាចបត់បាន

អ្នកអាចធ្វើឱ្យប្រអប់រំលេចអាចបត់បានដោយបន្ថែមសញ្ញាបូក (`+`) ឬសញ្ញាដក (`-`) ដោយផ្ទាល់បន្ទាប់ពីឧត្តមសញ្ញាប្រភេទ។

សញ្ញាបូកពន្លាតប្រអប់រំលេចតាមលំនាំដើម ហើយសញ្ញាដកបង្រួមវាវិញ។

```markdown
> [!faq]- តើប្រអប់រំលេចអាចបត់បានទេ?
> បាន! នៅក្នុងប្រអប់រំលេចដែលអាចបត់បាន មាតិកាត្រូវបានលាក់នៅពេលប្រអប់រំលេចត្រូវបានបង្រួម។
```

### ប្ដូរប្រអប់រំលេចតាមតម្រូវការ

[[ស្នីប៉ែត CSS]] និង [[កម្មវិធីជំនួយសហគមន៍]] អាចកំណត់ប្រអប់រំលេចផ្ទាល់ខ្លួន ឬថែមទាំងសរសេរជាន់លើការកំណត់រចនាសម្ព័ន្ធលំនាំដើម។

ដើម្បីកំណត់ប្រអប់រំលេចផ្ទាល់ខ្លួន សូមបង្កើតប្លុក CSS ខាងក្រោម៖

```css
.callout[data-callout="custom-question-type"] {
    --callout-color: 0, 0, 0;
    --callout-icon: lucide-alert-circle;
}
```

តម្លៃនៃ attribute `data-callout` គឺជាឧត្តមសញ្ញាប្រភេទដែលអ្នកចង់ប្រើ ឧទាហរណ៍ `[!custom-question-type]`។

- `--callout-color` កំណត់ពណ៌ផ្ទៃខាងក្រោយដោយប្រើលេខ (0–255) សម្រាប់ក្រហម បៃតង និងខៀវ។
- `--callout-icon` អាចជា ID រូបតំណាងពី [lucide.dev](https://lucide.dev) ឬធាតុ SVG។

> [!warning] ចំណាំអំពីកំណែរូបតំណាង lucide
> Obsidian ធ្វើបច្ចុប្បន្នភាពរូបតំណាង Lucide ជាទៀងទាត់។ កំណែបច្ចុប្បន្នដែលបានរួមបញ្ចូលត្រូវបានបង្ហាញខាងក្រោម; សូមប្រើរូបតំណាងទាំងនេះ ឬមុនទាំងនេះក្នុងប្រអប់រំលេចផ្ទាល់ខ្លួន។
> ![[ឥណទាន#^lucide]]

> [!tip] រូបតំណាង SVG
> ជំនួសឱ្យការប្រើរូបតំណាង Lucide អ្នកក៏អាចប្រើធាតុ SVG ជារូបតំណាងប្រអប់រំលេចផងដែរ។
>
> ```css
> --callout-icon: '<svg>...custom svg...</svg>';
> ```
