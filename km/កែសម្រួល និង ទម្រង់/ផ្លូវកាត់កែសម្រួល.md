ទំព័រនេះរាយបញ្ជីផ្លូវកាត់ក្តារចុចលំនាំដើមសម្រាប់ការរុករក និងកែសម្រួលអត្ថបទក្នុង Obsidian។ ផ្លូវកាត់ទាំងនេះផ្តល់ដោយប្រព័ន្ធប្រតិបត្តិការរបស់អ្នក ឬក្របខ័ណ្ឌដែល Obsidian បានបង្កើតឡើង ហើយមិនអាចប្ដូរតាមបំណងនៅក្នុង Obsidian បានទេ។

សម្រាប់ផ្លូវកាត់ក្តារចុចដែលអាចប្ដូរតាមបំណងសម្រាប់ពាក្យបញ្ជា Obsidian សូមមើល [[ផ្លូវកាត់ក្តារចុច]]។

## ផ្លូវកាត់សម្រាប់ Windows និង Linux

### សកម្មភាពទូទៅ

| សកម្មភាព | ផ្លូវកាត់ |
|-|-|
| ចម្លង | `Ctrl+C` |
| កាត់ | `Ctrl+X` |
| បិទភ្ជាប់ | `Ctrl+V` |
| បិទភ្ជាប់ដោយគ្មានទម្រង់ | `Ctrl+Shift+V` |
| មិនធ្វើវិញ | `Ctrl+Z` |
| ធ្វើវិញ | `Ctrl+Shift+Z` ឬ `Ctrl+Y` |
| ចម្លងកថាខណ្ឌ | `Ctrl+C` (ដោយគ្មានអត្ថបទដែលបានជ្រើសរើស) |
| កាត់កថាខណ្ឌ | `Ctrl+X` (ដោយគ្មានអត្ថបទដែលបានជ្រើសរើស) |
