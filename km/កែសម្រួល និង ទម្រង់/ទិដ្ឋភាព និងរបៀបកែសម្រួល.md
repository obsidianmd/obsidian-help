Obsidian អនុញ្ញាតឱ្យអ្នកគ្រប់គ្រងរបៀបដែលអ្នកអាន និងកែសម្រួលកំណត់ត្រាដែលសរសេរជា Markdown ដោយប្រើ _ទិដ្ឋភាព_ និង _របៀប_ ។

- **ទិដ្ឋភាព** ប្ដូររវាងការអាន និងការកែសម្រួលកំណត់ត្រារបស់អ្នក។
- **របៀប** គ្រប់គ្រងរបៀបដែល Markdown បង្ហាញនៅពេលអ្នកកំពុងកែសម្រួល។

> [!note] ចំណាំ
> តាមលំនាំដើម Obsidian បើកផ្ទាំងថ្មីនៅក្នុងរបៀបកែសម្រួល។ អ្នកអាចផ្លាស់ប្ដូរវានៅក្រោម **[[ការកំណត់]] → កម្មវិធីកែ → ទិដ្ឋភាពលំនាំដើមសម្រាប់ផ្ទាំងថ្មី**។

## ទិដ្ឋភាពអាន

*ទិដ្ឋភាពអាន* បង្ហាញកំណត់ត្រារបស់អ្នកដោយគ្មានវាក្យសម្ពន្ធ Markdown ដែលផ្ដល់ទម្រង់ស្អាត អានបានសម្រាប់ការពិនិត្យដោយផ្ចង់អារម្មណ៍។

ដើម្បីប្ដូរទៅ *ទិដ្ឋភាពអាន*៖

- ចុចឧបករណ៍ប្ដូរទិដ្ឋភាព ![[lucide-book-icon.svg#icon]] នៅជ្រុងខាងលើស្ដាំនៃកម្មវិធីកែ។
- ឬចុចរូបតំណាងស្ថានភាពអន្តរកម្ម ![[lucide-edit-3.svg#icon]] ឬ ![[lucide-code-xml.svg#icon]] នៅរបារស្ថានភាព ហើយជ្រើសរើស **ទិដ្ឋភាពអាន**។
- ឬចុច `Ctrl+E` (`Cmd+E` នៅលើ macOS)។

> [!note] ចំណាំ
> បើក **[[ការកំណត់]] → រូបរាង → បង្ហាញរបារចំណងជើងផ្ទាំង** ដើម្បីឃើញរូបតំណាងឧបករណ៍ប្ដូរទិដ្ឋភាព។
> 
> ប្រសិនបើវាត្រូវបានបិទ សូមបើក **[[ការកំណត់]] → កម្មវិធីកែ → បង្ហាញរបៀបកែសម្រួលនៅរបារស្ថានភាព** ដើម្បីប្ដូរទិដ្ឋភាពពីរបារស្ថានភាព។

> [!tip] មើលផ្គូផ្គង
> ដើម្បីបើកកំណត់ត្រាមួយក្នុង *ទិដ្ឋភាពកែសម្រួល* និង *ទិដ្ឋភាពអាន* ក្នុងពេលតែមួយ សូមសង្កត់ `Ctrl` (ឬ `Cmd` នៅលើ macOS) ហើយចុចឧបករណ៍ប្ដូរទិដ្ឋភាព។

## ទិដ្ឋភាពកែសម្រួល

*ទិដ្ឋភាពកែសម្រួល* អនុញ្ញាតឱ្យអ្នកធ្វើការផ្លាស់ប្ដូរកំណត់ត្រារបស់អ្នក។

នៅពេលស្ថិតក្នុង *ទិដ្ឋភាពកែសម្រួល* *របៀបកែសម្រួល* កំណត់របៀបដែល Markdown ត្រូវបានបង្ហាញ។ អ្នកអាចជ្រើសរើស *របៀបកែសម្រួល* មួយក្នុងចំណោមពីរ៖ *ការមើលផ្ទាល់* ឬ *របៀបប្រភព*។

### ការមើលផ្ទាល់

*ការមើលផ្ទាល់* បង្ហាញអត្ថបទដែលបានធ្វើទម្រង់នៅក្នុងបន្ទាត់ ខណៈពេលដែលលាក់វាក្យសម្ពន្ធ Markdown ភាគច្រើន។ នៅពេលទស្សន៍បញ្ជីរបស់អ្នកចូលទៅក្នុងមាតិកាដែលបានធ្វើទម្រង់ វាក្យសម្ពន្ធមូលដ្ឋានត្រូវបានបង្ហាញសម្រាប់ការកែសម្រួល។

ដើម្បីប្ដូរទៅ *ការមើលផ្ទាល់*៖

- ចុចឧបករណ៍ប្ដូរទិដ្ឋភាព ![[lucide-edit-3.svg#icon]] នៅជ្រុងខាងលើស្ដាំនៃកម្មវិធីកែ។

- ឬប្រើ [[ក្ដារលាយពាក្យបញ្ជា|ពាក្យបញ្ជា]] **ប្ដូរទិដ្ឋភាពអាន**។

> [!note] ចំណាំ
> តាមលំនាំដើម ទិដ្ឋភាពកែសម្រួលត្រូវបានកំណត់ជា *ការមើលផ្ទាល់*។ អ្នកអាចផ្លាស់ប្ដូរវានៅក្រោម **[[ការកំណត់]] → កម្មវិធីកែ → របៀបកែសម្រួលលំនាំដើម**។
