Obsidian бһӮбһ¶бҹҶбһ‘бҹ’бһҡ HTML бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһ бһ¶бһүбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбһ¶бһҳбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бһ¶бһ“ бһ¬вҖӢбһҹбһјбһҳбҹ’бһ”бһёбһҸбҹӮ[[бһ”бһ„бҹ’бһҖбһ”бҹӢбһҜбһҖбһҹбһ¶бһҡ|бһ”бһ„бҹ’бһҖбһ”бҹӢбһ‘бҹҶбһ–бҹҗбһҡбһңбҹӮбһ”]]бҹ” бһҖбһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸ HTML бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһ бһ¶бһ“бһ·бһ—бҹҗбһҷбҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбһјбһҠбһ–бҹ’бһҷбһ¶бһ”бһ¶бһ‘бһҳбһ·бһ“бһұбҹ’бһҷбһ”бһ„бҹ’бһҖбһҡбһ•бһӣбһўбһ¶бһҖбҹ’бһҡбһҖбҹӢ Obsidian _sanitize_ (бһҠбһҖбһ…бҹҒбһү) HTML бһҺбһ¶бһҳбһҪбһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

> бһ’бһ¶бһҸбһ» `<script>` бһҮбһ¶бһ’бһҳбҹ’бһҳбһҸбһ¶бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҡбһҸбҹӢ JavaScript бһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһҠбҹӮбһӣбһңбһ¶бһ•бҹ’бһ‘бһ»бһҖбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫ Obsidian бһҳбһ·бһ“ sanitize HTML бһўбҹ’бһ“бһҖбһңбһ¶бһҷбһ”бҹ’бһҡбһ бһ¶бһҡбһўбһ¶бһ…бһ”бһүбҹ’бһ…бһ»бҹҮбһ”бһүбҹ’бһ…бһјбһӣбһўбҹ’бһ“бһҖбһұбҹ’бһҷбһ”бһ·бһ‘бһ—бҹ’бһҮбһ¶бһ”бҹӢбһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹӮбһӣбһҳбһ¶бһ“ JavaScript бһҠбҹӮбһӣбһҹбҹ’бһҡбһ„бҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһ–бһёбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһ•бҹ’бһүбһҫбһңбһ¶бһ‘бҹ…бһ–бһҪбһҖбһӮбҹҒбҹ”

бһҠбһјбһ…бҹ’бһ“бҹ„бҹҮ бһҠбҹ„бһҷбһҹбһ¶бһҡбһңбһ¶бһҖбҹ’бһҷбһҹбһҳбҹ’бһ–бһ“бҹ’бһ’ Markdown бһҳбһ·бһ“бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ HTML бһҠбҹӮбһӣ sanitize бһҡбһҪбһ…бһ бһҫбһҷбһўбһ¶бһ…бһҮбһ¶бһңбһ·бһ’бһёбһҳбһҪбһҷбһ‘бҹҖбһҸбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһ“бһӮбһ»бһҺбһ—бһ¶бһ–бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҷбһҫбһ„бһ”бһ¶бһ“бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ HTML бһ‘бһјбһ‘бҹ…бһҒбҹ’бһӣбҹҮбҹ”

## бһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢ HTML

Obsidian бһҳбһ¶бһ“бһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһ“бҹ…бһ–бҹҒбһӣбһ”бҹ’бһҡбһҫ HTML бһҖбҹ’бһ“бһ»бһ„бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–

### бһӮбҹ’бһҳбһ¶бһ“ Markdown бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ HTML

Obsidian бһҳбһ·бһ“ render бһңбһ¶бһҖбҹ’бһҷбһҹбһҳбҹ’бһ–бһ“бҹ’бһ’ Markdown бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ’бһ¶бһҸбһ» HTML бһ‘бҹҒбҹ” бһ“бҹҒбҹҮбһҮбһ¶бһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһҠбҹ„бһҷбһ…бҹҒбһҸбһ“бһ¶ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– бһ“бһ·бһ„бһҡбһҖбҹ’бһҹбһ¶бһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбҹ’бһҡбһҹбҹ’бһҳбһ¶бһү parser бһұбҹ’бһҷбһ‘бһ¶бһ” бһ“бҹ…бһ–бҹҒбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҜбһҖбһҹбһ¶бһҡбһҠбҹҸбһ’бҹҶбҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһ“бҹҒбҹҮбһ“бһ№бһ„бһҳбһ·бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбһ¶бһҳбһҖбһ¶бһҡбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбҹ–

### бһ”бҹ’бһӣбһ»бһҖ HTML бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲ self-contained

бһ”бҹ’бһӣбһ»бһҖ HTML бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһ–бҹҒбһүбһӣбҹҒбһү бһ бһҫбһҷбһҳбһ·бһ“бһўбһ¶бһ…бһҳбһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ‘бһ‘бҹҒбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ–бһҪбһҖбһңбһ¶бһ”бһ¶бһ“бһ‘бҹҒбҹ” бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ‘бһ‘бҹҒбһ“бһ№бһ„бһ”бҹҶбһ”бҹӮбһҖбһ”бҹ’бһӣбһ»бһҖ HTMLбҹ”

### бһ“бҹ…бһ–бҹҒбһӣ Markdown бһ бһ¶бһҖбҹӢбһҠбһјбһ…бһҮбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ HTML

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸбһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣ Obsidian бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ Markdown бһҹбһјбһҳбһҳбһҫбһӣ [[Obsidian Flavored Markdown]]бҹ”

### бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢ

[[бһңбһ¶бһҖбҹ’бһҷбһҹбһҳбҹ’бһ–бһ“бҹ’бһ’бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“#бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢ|бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢ Markdown]] бһӮбһәбһҮбһ¶бһңбһ·бһ’бһёбһҠбҹӮбһӣбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһӣбһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһ‘бҹ„бҹҮбһҮбһ¶бһҷбҹүбһ¶бһ„бһҺбһ¶ бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ”бҹҶбһ”бҹ’бһӣбҹӮбһ„бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶ Markdown бһҒбҹ’бһӣбҹҮ бһҠбһјбһ…бһҮбһ¶ [Pandoc](https://pandoc.org) бһҳбһ¶бһ“бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢ Markdownбҹ” бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫ `<!-- HTML Comment -->` бһҮбҹҶбһ“бһҪбһҹбһңбһ·бһү!

### бһӮбһјбһҹбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҖбҹ’бһҡбҹ„бһҳ

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһӮбһјбһҹбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҖбҹ’бһҡбҹ„бһҳбһҳбһ’бҹ’бһҷбһ”бһ‘бһ—бҹ’бһӣбһ¶бһҳбҹ—бһҖбҹ’бһ“бһ»бһ„бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫ `<u>Example</u>` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸ <u>бһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҖбҹ’бһҡбҹ„бһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ</u>бҹ”

бһҹбҹ’бһӣбһ¶бһҖ Span бһ“бһ·бһ„ div бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҗбҹ’бһ“бһ¶бһҖбҹӢ custom бһ–бһё[[бһҹбҹ’бһ“бһёбһ”бҹүбҹӮбһҸ CSS|бһҹбҹ’бһ“бһёбһ”бҹүбҹӮбһҸ CSS]] бһ¬бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһ‘бҹ…бһӣбһҫбһҸбҹҶбһ”бһ“бҹӢбһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹӮбһӣбһ”бһ¶бһ“бһҮбҹ’бһҡбһҫбһҹбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ `<span style="font-family: cursive">your text</span>` бһўбһ¶бһ…бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖ <span style="font-family: cursive">бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ–бһ»бһҳбҹ’бһ–бһўбһҖбҹ’бһҹбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ</span> бһҷбҹүбһ¶бһ„бһҡбһ бҹҗбһҹбҹ”

## бһӮбһјбһҹбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҸбҹӢ

бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһӮбһјбһҹбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҸбҹӢ<s>бһўбһҸбҹ’бһҗбһ”бһ‘бһҒбҹ’бһӣбҹҮ</s>бһ‘бҹҒ? бһ”бҹ’бһҡбһҫ `<s>this</s>` бһҠбһҫбһҳбҹ’бһ”бһёбһӮбһјбһҹбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҸбҹӢбһңбһ¶бҹ”
