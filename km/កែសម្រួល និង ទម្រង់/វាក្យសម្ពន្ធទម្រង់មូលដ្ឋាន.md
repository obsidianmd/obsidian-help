រៀនពីរបៀបអនុវត្តទម្រង់មូលដ្ឋានទៅកំណត់ត្រារបស់អ្នក ដោយប្រើ [Markdown](https://daringfireball.net/projects/markdown/)។ សម្រាប់វាក្យសម្ពន្ធទម្រង់កម្រិតខ្ពស់បន្ថែមទៀត សូមយោងទៅ [[វាក្យសម្ពន្ធទម្រង់កម្រិតខ្ពស់]]។

## កថាខណ្ឌ

ដើម្បីបង្កើតកថាខណ្ឌក្នុង Markdown សូមប្រើ **បន្ទាត់ទទេ** ដើម្បីបែងចែកប្លុកអត្ថបទ។ ប្លុកអត្ថបទនីមួយៗដែលបែងចែកដោយបន្ទាត់ទទេត្រូវបានចាត់ទុកជាកថាខណ្ឌដាច់ដោយឡែក។

```md
This is a paragraph.

This is another paragraph.
```

This is a paragraph.

### ការបំបែកបន្ទាត់

តាមលំនាំដើមក្នុង Obsidian ការចុច `Enter` ម្តងនឹងបង្កើតបន្ទាត់ថ្មីក្នុងកំណត់ត្រារបស់អ្នក ប៉ុន្តែវាត្រូវបានចាត់ទុកជា *ការបន្ត* នៃកថាខណ្ឌដដែលនៅក្នុងលទ្ធផលដែលបង្ហាញ ដោយអនុវត្តតាមអាកប្បកិរិយា Markdown ធម្មតា។ ដើម្បីបញ្ចូលការបំបែកបន្ទាត់ *នៅក្នុង* កថាខណ្ឌមួយដោយមិនចាប់ផ្តើមកថាខណ្ឌថ្មី អ្នកអាច៖

- បន្ថែម **គម្លាតពីរ** នៅចុងបន្ទាត់មុនពេលចុច `Enter` ឬ
- ប្រើផ្លូវកាត់ `Shift+Enter` ដើម្បីបញ្ចូលការបំបែកបន្ទាត់ដោយផ្ទាល់។

> [!question]- ហេតុអ្វីការចុច `Enter` ច្រើនដងមិនបង្កើតការបំបែកបន្ទាត់បន្ថែមក្នុងទិដ្ឋភាពអាន?
> ក្នុង Markdown ការចុច `Enter` តែមួយត្រូវបានមិនអើពើ ហើយការចុច `Enter` ជាប់គ្នាច្រើនដងផ្តល់លទ្ធផលត្រឹមតែកថាខណ្ឌថ្មីមួយ។ អាកប្បកិរិយានេះស្របតាមច្បាប់ soft wrap របស់ Markdown ដែលបន្ទាត់ទទេបន្ថែមមិនបង្កើតការបំបែកបន្ទាត់ ឬកថាខណ្ឌបន្ថែមទេ — ពួកវាត្រូវបានបង្រួមទៅជាការបំបែកកថាខណ្ឌតែមួយ។ នេះជារបៀបដែល Markdown ដំណើរការអត្ថបទតាមលំនាំដើម ដើម្បីធានាថាកថាខណ្ឌហូរដោយធម្មជាតិដោយគ្មានការបំបែកដែលមិនបានរំពឹងទុក​។

Obsidian រួមបញ្ចូលការកំណត់ **[[ការកំណត់#Strict line breaks|ការបំបែកបន្ទាត់តឹងរ៉ឹង]]** ដែលធ្វើឱ្យ Obsidian អនុវត្តតាមវិធីសម្បទា Markdown ស្តង់ដារសម្រាប់ការបំបែកបន្ទាត់។

ដើម្បីបើកមុខងារនេះ៖

1. បើក **[[ការកំណត់]]**។
2. ទៅផ្ទាំង **កម្មវិធីកែសម្រួល**។
3. បើក **ការបំបែកបន្ទាត់តឹងរ៉ឹង**។

នៅពេល **ការបំបែកបន្ទាត់តឹងរ៉ឹង** ត្រូវបានបើកក្នុង Obsidian ការបំបែកបន្ទាត់មានអាកប្បកិរិយាបីយ៉ាងខុសៗគ្នា អាស្រ័យលើរបៀបដែលបន្ទាត់ត្រូវបានបែងចែក៖

**ការចុច Enter តែមួយដោយគ្មានគម្លាត**: ការចុច `Enter` តែមួយដោយគ្មានគម្លាតនៅចុងបន្ទាត់នឹងរួមបញ្ចូលបន្ទាត់ពីរដាច់ដោយឡែកទៅជាបន្ទាត់តែមួយនៅពេលបង្ហាញ។

```md
line one
line two
```

បង្ហាញជា៖

line one line two

**ការចុច Enter តែមួយដោយមានគម្លាតពីរ ឬច្រើនជាងនៅចុងបន្ទាត់**: បើអ្នកបន្ថែមគម្លាតពីរ ឬច្រើនជាងនៅចុងបន្ទាត់ទីមួយមុនពេលចុច `Enter` បន្ទាត់ទាំងពីរនៅតែជាផ្នែកនៃកថាខណ្ឌដដែល ប៉ុន្តែត្រូវបានបំបែកដោយការបំបែកបន្ទាត់ (ធាតុ HTML `<br>`)។ យើងនឹងប្រើសញ្ញា underscore ពីរដើម្បីតំណាងឱ្យគម្លាតក្នុងឧទាហរណ៍នេះ។

```md
line three__  
line four
```

បង្ហាញជា៖

line three<br>
line four

**ការចុច Enter ពីរដង (ដោយមាន ឬគ្មានគម្លាតនៅចុងបន្ទាត់)**: ការចុច `Enter` ពីរដង (ឬច្រើនជាង) បែងចែកបន្ទាត់ទៅជាកថាខណ្ឌពីរដាច់ដោយឡែក (ធាតុ HTML `<p>`) ដោយមិនគិតពីថាតើអ្នកបន្ថែមគម្លាតនៅចុងបន្ទាត់ទីមួយឬអត់។

```md
line five

line six
```

បង្ហាញជា៖

<p>line five</p>
<p>line six</p>

## ចំណងជើង

ដើម្បីបង្កើតចំណងជើង សូមបន្ថែមសញ្ញា `#` រហូតដល់ប្រាំមួយ មុនអត្ថបទចំណងជើងរបស់អ្នក។ ចំនួនសញ្ញា `#` កំណត់កម្រិតចំណងជើង (ដូចដែលបង្ហាញក្នុង [[គ្រោង]])។

```md
# This is a heading 1
## This is a heading 2
### This is a heading 3
#### This is a heading 4
##### This is a heading 5
###### This is a heading 6
```

%% These headings use HTML to avoid cluttering the Outline/Table of contents %%
<h1>នេះជាចំណងជើង 1</h1>
<h2>នេះជាចំណងជើង 2</h2>
<h3>នេះជាចំណងជើង 3</h3>
<h4>នេះជាចំណងជើង 4</h4>
<h5>នេះជាចំណងជើង 5</h5>
<h6>នេះជាចំណងជើង 6</h6>

## ដិត ទ្រេត ពណ៌ចំណាំ

ទម្រង់អត្ថបទក៏អាចត្រូវបានអនុវត្តដោយប្រើ [[ផ្លូវកាត់កែសម្រួល]] ផងដែរ។

| រចនាប័ទ្ម | វាក្យសម្ពន្ធ | ឧទាហរណ៍ | លទ្ធផល |
|-|-|-|-|
| ដិត | `** **` ឬ `__ __` | `**Bold text**` | **Bold text** |
| ទ្រេត | `* *` ឬ `_ _`  | `*Italic text*` | *Italic text* |
| ឆូតកាត់ | `~~ ~~` |  `~~Striked out text~~` | ~~Striked out text~~ |
| ពណ៌ចំណាំ | `== ==` |  `==Highlighted text==` | ==Highlighted text== |
| ដិត និងទ្រេតដាក់ក្នុង | `** **` និង `_ _`  | `**Bold text and _nested italic_ text**` | **Bold text and _nested italic_ text** |
| ដិត និងទ្រេត | `*** ***` ឬ `___ ___` |  `***Bold and italic text***` | ***Bold and italic text*** |

ទម្រង់អាចត្រូវបានបង្ខំឱ្យបង្ហាញជាអត្ថបទធម្មតាដោយបន្ថែម backslash `\` នៅពីមុខវា។

\*\*This line will not be bold\*\*

```markdown
\*\*This line will not be bold\*\*
```

\**This line will be italic and show the asterisks*\*

```markdown
\**This line will be italic and show the asterisks*\*
```

- Markdown: `[Three laws of motion](Three%20laws%20of%20motion.md)`

## តំណភ្ជាប់ខាងក្រៅ

បើអ្នកចង់តភ្ជាប់ទៅ URL ខាងក្រៅ អ្នកអាចបង្កើតតំណភ្ជាប់ inline ដោយដាក់អត្ថបទតំណភ្ជាប់ក្នុងវង់ក្រចក (`[ ]`) រួចដាក់ URL ក្នុងវង់ក្រចក (`( )`)។

```md
[Obsidian Help](https://help.obsidian.md)
```

[Obsidian Help](https://help.obsidian.md)

អ្នកក៏អាចបង្កើតតំណភ្ជាប់ខាងក្រៅទៅឯកសារក្នុង vault ផ្សេងទៀត ដោយតភ្ជាប់ទៅ [[Obsidian URI|Obsidian URI]]។

```md

```

### គេចគម្លាតទទេក្នុងតំណភ្ជាប់

