រូបមន្តអនុញ្ញាតឱ្យអ្នកបង្កើតលក្ខណៈសម្បត្តិដែលបានគណនានៅក្នុង [[ការណែនាំអំពី Bases|Bases]] ដោយប្រើទិន្នន័យពី[[លក្ខណៈសម្បត្តិ|លក្ខណៈសម្បត្តិ]]ផ្សេងទៀត។ អ្នកអាចអនុវត្តការគណនា រៀបចំអត្ថបទ ធ្វើការជាមួយកាលបរិច្ឆេទ និងច្រើនទៀត។

## អ្វីដែលរូបមន្តអាចធ្វើបាន

រូបមន្តអាចជួយអ្នក៖

- **គណនាតម្លៃ** បន្ថែមតម្លៃ គណនាសរុប ឬអនុវត្តប្រតិបត្តិការគណិតវិទ្យា។
- **រៀបចំអត្ថបទ** ផ្សំខ្សែអក្សរ ប្ដូរអក្សរធំតូច ឬដកខ្សែអក្សររង។
- **ធ្វើការជាមួយកាលបរិច្ឆេទ** គណនាភាពខុសគ្នានៃពេលវេលា ទម្រង់កាលបរិច្ឆេទ ឬកំណត់ថ្ងៃកំណត់។
- **អនុវត្តតក្កវិទ្យា** ប្រើសេចក្ដីថ្លែងការណ៍មានលក្ខខណ្ឌដើម្បីបង្ហាញតម្លៃផ្សេងៗ។
- **ដំណើរការបញ្ជី** ត្រង តម្រៀប ផែនទី ឬប្រមូលផ្តុំទិន្នន័យបញ្ជី។

## បង្កើតលក្ខណៈសម្បត្តិរូបមន្ត

ដើម្បីបង្កើតលក្ខណៈសម្បត្តិរូបមន្ត៖

1. នៅក្នុង base របស់អ្នក ចុច **Properties** នៅក្នុងរបារឧបករណ៍។
2. ចុច **Add formula** នៅខាងក្រោមម៉ឺនុយ។
3. វាយបញ្ចូលឈ្មោះសម្រាប់លក្ខណៈសម្បត្តិរូបមន្តរបស់អ្នក។
4. វាយរូបមន្តរបស់អ្នកនៅក្នុងវាល **Formula**។
5. បិទប្រអប់។

កម្មវិធីកែរូបមន្តនឹងបំពេញស្វ័យប្រវត្តិឈ្មោះ[[អនុគមន៍|អនុគមន៍]] និងលក្ខណៈសម្បត្តិនៅពេលអ្នកវាយបញ្ចូល ដើម្បីផ្ទៀងផ្ទាត់វាក្យសម org រូបមន្តរបស់អ្នក។ សញ្ញាធីកពណ៌បៃតងនឹងបង្ហាញនៅពេលរូបមន្តរបស់អ្នកត្រឹមត្រូវ។

នៅពេលបង្កើតរួច អ្នកអាចប្រើលក្ខណៈសម្បត្តិរូបមន្តដូចជាលក្ខណៈសម្បត្តិផ្សេងទៀតនៅក្នុង base របស់អ្នក។ បន្ថែមវាទៅ[[ទិដ្ឋភាព|ទិដ្ឋភាព]] ប្រើវានៅក្នុងតម្រង តម្រៀបតាមវា និងច្រើនទៀត។

## សរសេររូបមន្ត

នៅក្នុងកម្មវិធីកែរូបមន្ត វាយកន្សោមដោយប្រើលក្ខណៈសម្បត្តិ សញ្ញាប្រមាណវិធី និងអនុគមន៍។

### យោងលក្ខណៈសម្បត្តិ

អ្នកអាចយោងប្រភេទលក្ខណៈសម្បត្តិផ្សេងៗក្នុងរូបមន្តរបស់អ្នក៖

- **លក្ខណៈសម្បត្តិកំណត់ត្រា** — លក្ខណៈសម្បត្តិពី [[លក្ខណៈសម្បត្តិ|frontmatter]] របស់កំណត់ត្រា។
- **លក្ខណៈសម្បត្តិឯកសារ** — លក្ខណៈសម្បត្តិដែលភ្ជាប់មកដូចជា `file.name`, `file.size`, ឬ `file.mtime`។
- **លក្ខណៈសម្បត្តិរូបមន្ត** — រូបមន្តផ្សេងទៀតដោយប្រើ `formula.formula_name`។

**ឧទាហរណ៍៖**

- `price * quantity` — គុណលក្ខណៈសម្បត្តិកំណត់ត្រាពីរ
- `file.name + " - " + description` — ផ្សំឈ្មោះឯកសារជាមួយលក្ខណៈសម្បត្តិកំណត់ត្រា
- `formula.price_per_unit * 1.1` — ប្រើលក្ខណៈសម្បត្តិរូបមន្តផ្សេងទៀត

### ប្រើសញ្ញាប្រមាណវិធី

**សញ្ញាប្រមាណវិធីនព្វន្ធ** អនុវត្តគណិតវិទ្យាលើលេខ៖

- `price + tax` — បូក
- `price - discount` — ដក
- `price * quantity` — គុណ
- `price / quantity` — ចែក
- `(part / whole) * 100` — ប្រើវង់ក្រចកសម្រាប់លំដាប់នៃប្រតិបត្តិការ

**សញ្ញាប្រមាណវិធីប្រៀបធៀប** ប្រៀបធៀបតម្លៃ៖

- `price > 100` — ធំជាង
- `age < 18` — តូចជាង
- `status == "Done"` — ស្មើ
- `status != "Done"` — មិនស្មើ
- `file.mtime > now() - '7d'` — ប្រៀបធៀបកាលបរិច្ឆេទ

**សញ្ញាប្រមាណវិធីប៊ូលីន** ផ្សំលក្ខខណ្ឌតក្កវិទ្យា៖

- `!completed` — មិន
- `price > 0 && quantity > 0` — និង
- `urgent || important` — ឬ

ស្វែងយល់បន្ថែមក្នុង [[វាក្យសម្ពន្ធ Bases#សញ្ញាប្រមាណវិធី|វាក្យសម្ពន្ធ Bases]]។

### ប្រើអនុគមន៍

អនុគមន៍អនុវត្តប្រតិបត្តិការលើតម្លៃ។ អនុគមន៍ដែលមានអាស្រ័យលើប្រភេទតម្លៃដែលអ្នកកំពុងធ្វើការជាមួយ។ សូមមើលបញ្ជីពេញលេញនៃ[[អនុគមន៍]]។

**ប្រភេទអនុគមន៍ទូទៅ៖**

- **អនុគមន៍សកល** — `if()`, `now()`, `date()`, `link()`, `max()`, `min()`
- **អនុគមន៍ខ្សែអក្សរ** — `contains()`, `replace()`, `split()`, `lower()`, `title()`
- **អនុគមន៍លេខ** — `round()`, `ceil()`, `floor()`, `abs()`, `toFixed()`
- **អនុគមន៍កាលបរិច្ឆេទ** — `format()`, `relative()`, `date()`, `time()`
- **អនុគមន៍បញ្ជី** — `filter()`, `map()`, `sort()`, `join()`, `unique()`

**ឧទាហរណ៍៖**

- `if(price, "$" + price.toFixed(2), "")` ដើម្បីកំណត់លក្ខខណ្ឌជាមួយការធ្វើទម្រង់លេខ។
- `file.name.lower()` ដើម្បីបម្លែងទៅជាអក្សរតូច។
- `tags.contains("urgent")` ដើម្បីពិនិត្យមើលថាតើបញ្ជីស្លាកមានតម្លៃមួយឬអត់។
- `due_date.format("YYYY-MM-DD")` ដើម្បីធ្វើទម្រង់កាលបរិច្ឆេទ។

## ឧទាហរណ៍រូបមន្ត

### គណនាថ្ងៃកំណត់

កំណត់ថ្ងៃផុតកំណត់របស់គម្រោងជា ២ សប្តាហ៍បន្ទាប់ពីថ្ងៃចាប់ផ្តើម៖

```js
start_date + "2w"
```

### បង្ហាញស្ថានភាពហួសកំណត់

បង្ហាញ "Overdue" ប្រសិនបើថ្ងៃកំណត់បានកន្លងផុត ហើយស្ថានភាពមិនមែន "Done"៖

```js
if(due_date < now() && status != "Done", "Overdue", "")
```

### ធ្វើទម្រង់រូបិយប័ណ្ណ

បង្ហាញតម្លៃជាមួយខ្ទង់ទសភាគ ២ និងនិមិត្តសញ្ញារូបិយប័ណ្ណ៖

```js
if(price, "$" + price.toFixed(2), "")
```

### រាប់ធាតុក្នុងបញ្ជី

រាប់ចំនួនធាតុក្នុងលក្ខណៈសម្បត្តិបញ្ជី៖

```js
tasks.length
```

### គណនាពិន្ទុអាទិភាព

ផ្សំកត្តាច្រើនជាពិន្ទុអាទិភាព៖

```js
(impact * urgency) / effort
```

### ផ្សំវាលអត្ថបទ

បង្កើតឈ្មោះពេញពីនាមខ្លួន និងត្រកូល៖

```js
first_name + " " + last_name
```

### គណនាថ្លៃសរុប

គុណថ្លៃប្រចាំខែនឹងចំនួនខែដែលមាន៖

```js
monthlyUses * formula.Owned.round()
```

## ប្រភេទទិន្នន័យ

រូបមន្តដំណើរការជាមួយប្រភេទទិន្នន័យផ្សេងៗ៖

- **ខ្សែអក្សរ** — អត្ថបទនៅក្នុងសញ្ញាសម្រង់៖ `"hello"` ឬ `'world'`
- **លេខ** — តម្លៃជាលេខ៖ `42`, `3.14`, `(2 + 2)`
- **ប៊ូលីន** — ពិត ឬមិនពិត៖ `true`, `false`
- **កាលបរិច្ឆេទ** — បង្កើតដោយ `date()`, `today()`, ឬ `now()`
- **បញ្ជី** — សម្រង់នៃតម្លៃ៖ `[1, 2, 3]`
- **វត្ថុ** — គូគន្លឹះ-តម្លៃ៖ `{"name": "value"}`

ប្រភេទលទ្ធផលនៃរូបមន្តត្រូវបានកំណត់ដោយទិន្នន័យ និងអនុគមន៍ដែលបានប្រើ។

## យោងរូបមន្តផ្សេងទៀត

រូបមន្តអាចយោងរូបមន្តផ្សេងទៀត ដើម្បីបង្កើតការគណនាដែលបានមកពី។ ឧទាហរណ៍ ប្រសិនបើអ្នកមានរូបមន្តមួយហៅថា `price_per_unit`៖

```js
price / quantity
```

អ្នកអាចយោងវាក្នុងរូបមន្តផ្សេងទៀត៖

```js
formula.price_per_unit * 1.1
```

> [!warning] ជៀសវាងការយោងរង្វង់
> រូបមន្តមួយមិនអាចយោងខ្លួនឯងដោយផ្ទាល់ ឬដោយប្រយោលតាមរយៈរូបមន្តផ្សេងទៀតបានទេ។
