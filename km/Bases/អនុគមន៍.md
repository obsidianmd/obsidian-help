## ជាសកល

អនុគមន៍ជាសកលត្រូវបានប្រើដោយគ្មានប្រភេទ។

### `escapeHTML()`

`escapeHTML(html: string): string`

- គេចចេញពីតួអក្សរពិសេសក្នុងខ្សែអក្សរ ដើម្បីធ្វើឱ្យវាមានសុវត្ថិភាពសម្រាប់ការបញ្ចូលក្នុង HTML។

### `date()`

`date(date: string): date`

- `date(string): date` បកស្រាយខ្សែអក្សរដែលបានផ្ដល់ ហើយត្រឡប់វត្ថុកាលបរិច្ឆេទ។
- ខ្សែអក្សរ `date` គួរតែស្ថិតក្នុងទម្រង់ `YYYY-MM-DD HH:mm:ss`។

### `duration()`

`duration(value: string): duration`

- បកស្រាយខ្សែអក្សរជារយៈពេល។ សូមមើល [[វាក្យសម្ពន្ធ Bases#ការគណនាកាលបរិច្ឆេទ|ផ្នែកការគណនាកាលបរិច្ឆេទ]] សម្រាប់ទម្រង់ខ្សែអក្សរ `value`។
- រយៈពេលមិនចាំបាច់ត្រូវបានបកស្រាយយ៉ាងច្បាស់នៅពេលអនុវត្តការគណនាកាលបរិច្ឆេទ (ឧទាហរណ៍ `now() + '1d'`) ប៉ុន្តែត្រូវការ នៅពេលអនុវត្តការគណនាលើរយៈពេល (ឧទាហរណ៍ `now() + (duration('1d') * 2)`)។
- នៅពេលអនុវត្តការគណនាលើរយៈពេលជាមួយចំនួនដែលកំណត់ទម្រង់ (scalars) រយៈពេលត្រូវតែស្ថិតនៅខាងឆ្វេង។ ឧទាហរណ៍ `duration('5h') * 2` មិនមែន `2 * duration('5h')`។

### `file()`

`file(path: string | file | url): file`

- ត្រឡប់វត្ថុឯកសារសម្រាប់ឯកសារ ឬ path ដែលបានផ្ដល់។
- ឧទាហរណ៍៖ `file(link("[[filename]]"))` ឬ `file("path to file")`។

### `html()`

`html(html: string): html`

- បំប្លែងខ្សែអក្សរទៅជាផ្នែកកូដដែលបង្ហាញជា HTML។

### `if()`

`if(condition: any, trueResult: any, falseResult?: any): any`

- `condition` គឺជាលក្ខខណ្ឌដែលត្រូវវាយតម្លៃ។
- `trueResult` គឺជាលទ្ធផលប្រសិនបើលក្ខខណ្ឌជាពិត។
- `falseResult` គឺជាលទ្ធផលជម្រើសប្រសិនបើលក្ខខណ្ឌជាមិនពិត។ ប្រសិនបើមិនបានបញ្ជាក់ នោះវានឹងត្រូវបានចាត់ទុកជា `null`។
- ត្រឡប់ `trueResult` ប្រសិនបើ `condition` ជាពិត ឬជាតម្លៃពិត (truthy) ហើយត្រឡប់ `falseResult` ក្នុងករណីផ្សេង។
- ឧទាហរណ៍៖ `if(isModified, "Modified", "Unmodified")`

### `image()`

`image(path: string | file | url): image`

- ត្រឡប់វត្ថុរូបភាពដែលនឹងបង្ហាញរូបភាពក្នុង view។
- ឧទាហរណ៍៖ `image(image-property)` ឬ `image("https://obsidian.md/images/obsidian-logo-gradient.svg")`

### `icon()`

`icon(name: string): icon`

- ត្រឡប់តម្លៃដែលនឹងបង្ហាញជាអាយខុនក្នុង view។ ឈ្មោះអាយខុនត្រូវតែត្រូវគ្នានឹងអាយខុន Lucide ដែលគាំទ្រ។
- ឧទាហរណ៍៖ `icon("arrow-right")`។

### `link()`

`link(path: string | file, display?: value): Link`

- បកស្រាយ `path` ជាខ្សែអក្សរ ហើយត្រឡប់វត្ថុ Link ដែលបង្ហាញជាតំណភ្ជាប់ទៅ path ដែលបានផ្ដល់។
- ជម្រើស ផ្ដល់ប៉ារ៉ាម៉ែត្រ `display` ដើម្បីផ្លាស់ប្ដូរអត្ថបទដែលតំណភ្ជាប់បង្ហាញ។

### `list()`

`list(element: any): List`

- ប្រសិនបើធាតុដែលបានផ្ដល់ជាបញ្ជី ត្រឡប់វាដោយមិនផ្លាស់ប្ដូរ។
- ក្នុងករណីផ្សេង រុំ `element` ដែលបានផ្ដល់ក្នុងបញ្ជី ដោយបង្កើតបញ្ជីដែលមានធាតុតែមួយ។
- អនុគមន៍នេះអាចមានប្រយោជន៍នៅពេល property មានការលាយបញ្ចូលគ្នារវាងខ្សែអក្សរ ឬបញ្ជីនៅទូទាំង vault។
- ឧទាហរណ៍៖ `list("value")` ត្រឡប់ `["value"]`។

### `max()`

`max(value1: number, value2: number...): number`

- ត្រឡប់ចំនួនធំបំផុតក្នុងចំណោមចំនួនទាំងអស់ដែលបានផ្ដល់។

### `min()`

`min(value1: number, value2: number...): number`

- ត្រឡប់ចំនួនតូចបំផុតក្នុងចំណោមចំនួនទាំងអស់ដែលបានផ្ដល់។

### `now()`

`now(): date`

- `now()` ត្រឡប់វត្ថុកាលបរិច្ឆេទដែលតំណាងឱ្យពេលបច្ចុប្បន្ន។

### `number()`

`number(input: any): number`

- ព្យាយាមត្រឡប់តម្លៃដែលបានផ្ដល់ជាចំនួន។
- វត្ថុកាលបរិច្ឆេទនឹងត្រូវបានត្រឡប់ជាមិល្លីវិនាទីចាប់តាំងពី unix epoch។
- Boolean នឹងត្រឡប់ 1 ឬ 0។
- ខ្សែអក្សរនឹងត្រូវបានបកស្រាយជាចំនួន ហើយត្រឡប់កំហុសប្រសិនបើលទ្ធផលមិនត្រឹមត្រូវ។
- ឧទាហរណ៍ `number("3.4")` ត្រឡប់ `3.4`។

### `duration()`

`duration(value: string): duration`

- បកស្រាយខ្សែអក្សរជារយៈពេល។ សូមមើល [[វាក្យសម្ពន្ធ Bases#ការគណនាកាលបរិច្ឆេទ|ផ្នែកការគណនាកាលបរិច្ឆេទ]] សម្រាប់ទម្រង់ខ្សែអក្សរ `value`។
- រយៈពេលមិនចាំបាច់ត្រូវបានបកស្រាយយ៉ាងច្បាស់នៅពេលអនុវត្តការគណនាកាលបរិច្ឆេទ (ឧទាហរណ៍ `now() + '1d'`) ប៉ុន្តែត្រូវការ នៅពេលអនុវត្តការគណនាលើរយៈពេល (ឧទាហរណ៍ `now() + (duration('1d') * 2)`)។
- នៅពេលអនុវត្តការគណនាលើរយៈពេលជាមួយចំនួនដែលកំណត់ទម្រង់ (scalars) រយៈពេលត្រូវតែស្ថិតនៅខាងឆ្វេង។ ឧទាហរណ៍ `duration('5h') * 2` មិនមែន `2 * duration('5h')`។

### `today()`

`today(): date`

- `today()` ត្រឡប់វត្ថុកាលបរិច្ឆេទដែលតំណាងឱ្យថ្ងៃបច្ចុប្បន្ន។ ផ្នែកពេលវេលាត្រូវបានកំណត់ជាសូន្យ។

### `random()`

`random(): number`

- `random()` ត្រឡប់លេខចៃដន្យចន្លោះ 0 និង 1។
- ការបង្កើតលេខត្រូវបានធ្វើបច្ចុប្បន្នភាពរាល់ពេលដែលទិដ្ឋភាពត្រូវបានផ្ទុក។ ការរុករករវាងទិដ្ឋភាពផ្លាស់ប្តូរលេខចៃដន្យ។

## ណាមួយ

អនុគមន៍ដែលអ្នកអាចប្រើជាមួយតម្លៃណាមួយ។ នេះរួមមានខ្សែអក្សរ (ឧ. `"hello"`) ចំនួន (ឧ. `42`) បញ្ជី (ឧ. `[1,2,3]`) វត្ថុ និងច្រើនទៀត។

### `isTruthy()`

`any.isTruthy(): boolean`

- ត្រឡប់តម្លៃដែលបំប្លែងទៅជា boolean។
- ឧទាហរណ៍៖ `1.isTruthy()` ត្រឡប់ `true`។

### `isType()`

`any.isType(type: string): boolean`

- ត្រឡប់ true ប្រសិនបើតម្លៃជាប្រភេទដែលបានផ្ដល់។
- ឧទាហរណ៍៖ `"example".isType("string")` និង `true.isType("boolean")` ទាំងពីរត្រឡប់ true។

### `toString()`

`any.toString(): string`

- ត្រឡប់ការតំណាងជាខ្សែអក្សររបស់តម្លៃណាមួយ។
- ឧទាហរណ៍៖ `123.toString()` ត្រឡប់ `"123"`។

## កាលបរិច្ឆេទ

អនុគមន៍ដែលអ្នកអាចប្រើជាមួយកាលបរិច្ឆេទ និងពេលវេលា ដូចជា `date("2025-05-27")`។ ការប្រៀបធៀបកាលបរិច្ឆេទអាចធ្វើបានដោយប្រើ [[វាក្យសម្ពន្ធ Bases#ការគណនាកាលបរិច្ឆេទ|ការគណនាកាលបរិច្ឆេទ]]។

### វាល

វាលខាងក្រោមអាចប្រើបានសម្រាប់កាលបរិច្ឆេទ៖

| វាល                | ប្រភេទ   | ការពណ៌នា                         |
| ------------------ | -------- | --------------------------------- |
| `date.year`        | `number` | ឆ្នាំនៃកាលបរិច្ឆេទ               |
| `date.month`       | `number` | ខែនៃកាលបរិច្ឆេទ (1–12)           |
| `date.day`         | `number` | ថ្ងៃនៃខែ                          |
| `date.hour`        | `number` | ម៉ោង (0–23)                       |
| `date.minute`      | `number` | នាទី (0–59)                       |
| `date.second`      | `number` | វិនាទី (0–59)                     |
| `date.millisecond` | `number` | មិល្លីវិនាទី (0–999)              |

### `date()`

`date.date(): date`

- ត្រឡប់វត្ថុកាលបរិច្ឆេទដោយដកផ្នែកពេលវេលាចេញ។
- ឧទាហរណ៍៖ `now().date().format("YYYY-MM-DD HH:mm:ss")` ត្រឡប់ខ្សែអក្សរដូចជា "2025-12-31 00:00:00"

### `format()`

`date.format(format: string): string`

- `format` គឺជាខ្សែអក្សរទម្រង់ (ឧ. `"YYYY-MM-DD"`)។
- ត្រឡប់កាលបរិច្ឆេទដែលបានធ្វើទ្រង់ទ្រាយតាមខ្សែអក្សរទម្រង់ Moment.js ។
- ឧទាហរណ៍៖ `date.format("YYYY-MM-DD")` ត្រឡប់ `"2025-05-27"`។

### `time()`

`date.time(): string`

- ត្រឡប់ពេលវេលា។
- ឧទាហរណ៍៖ `now().time()` ត្រឡប់ខ្សែអក្សរដូចជា "23:59:59"

### `relative()`

`date.relative(): string`

- ត្រឡប់ការប្រៀបធៀបដែលអាចអានបានរបស់កាលបរិច្ឆេទទៅនឹងកាលបរិច្ឆេទ-ពេលវេលាបច្ចុប្បន្ន។
- ឧទាហរណ៍៖ `file.mtime.relative()` ត្រឡប់តម្លៃដូចជា `3 days ago`។

### `isEmpty()`

`date.isEmpty(): boolean`

- ត្រឡប់ false។

## ខ្សែអក្សរ

អនុគមន៍ដែលអ្នកអាចប្រើជាមួយបន្តរអក្សរ ដូចជា `"hello"`។

### វាល

| វាល             | ប្រភេទ   | ការពណ៌នា                               |
| --------------- | -------- | --------------------------------------- |
| `string.length` | `number` | ចំនួនអក្សរក្នុងខ្សែអក្សរ               |

### `contains()`

`string.contains(value: string): boolean`

- `value` គឺជារង-ខ្សែអក្សរដែលត្រូវស្វែងរក។
- ត្រឡប់ true ប្រសិនបើខ្សែអក្សរមាន `value`។
- ឧទាហរណ៍៖ `"hello".contains("ell")` ត្រឡប់ `true`។

### `containsAll()`

`string.containsAll(...values: string): boolean`

- `values` គឺជារង-ខ្សែអក្សរមួយ ឬច្រើនដែលត្រូវស្វែងរក។
- ត្រឡប់ true ប្រសិនបើខ្សែអក្សរមាន `values` ទាំងអស់។
- ឧទាហរណ៍៖ `"hello".containsAll("h", "e")` ត្រឡប់ `true`។

### `containsAny()`

`string.containsAny(...values: string): boolean`

- `values` គឺជារង-ខ្សែអក្សរមួយ ឬច្រើនដែលត្រូវស្វែងរក។
- ត្រឡប់ true ប្រសិនបើខ្សែអក្សរមានយ៉ាងហោចណាស់មួយក្នុង `values`។
- ឧទាហរណ៍៖ `"hello".containsAny("x", "y", "e")` ត្រឡប់ `true`។

### `endsWith()`

`string.endsWith(query: string): boolean`

- `query` គឺជាខ្សែអក្សរដែលត្រូវពិនិត្យនៅចុងបញ្ចប់។
- ត្រឡប់ true ប្រសិនបើខ្សែអក្សរនេះបញ្ចប់ដោយ `query`។
- ឧទាហរណ៍៖ `"hello".endsWith("lo")` ត្រឡប់ `true`។

### `isEmpty()`

`string.isEmpty(): boolean`

- ត្រឡប់ true ប្រសិនបើខ្សែអក្សរមិនមានអក្សរ ឬមិនមាន។
- ឧទាហរណ៍៖ `"Hello world".isEmpty()` ត្រឡប់ `false`។
- ឧទាហរណ៍៖ `"".isEmpty()` ត្រឡប់ `true`។

### `lower()`

`string.lower(): string`

- ត្រឡប់ខ្សែអក្សរដែលបំប្លែងទៅជាអក្សរតូច។

### `replace()`

`string.replace(pattern: string | Regexp, replacement: string): string`

- `pattern` គឺជាតម្លៃដែលត្រូវស្វែងរកក្នុងខ្សែអក្សរគោលដៅ។
- `replacement` គឺជាតម្លៃដែលត្រូវជំនួស pattern ដែលរកឃើញ។
- ប្រសិនបើ `pattern` ជាខ្សែអក្សរ ការកើតឡើងទាំងអស់នៃ pattern នឹងត្រូវបានជំនួស។
- ប្រសិនបើ `pattern` ជា Regexp ជម្រើស `g` កំណត់ថាតើត្រូវជំនួសតែការកើតឡើងដំបូង ឬការកើតឡើងទាំងអស់។
- ឧទាហរណ៍៖ `"a:b:c:d".replace(/:/, "-")` ត្រឡប់ `"a-b,c,d"` ចំណែក `"a:b:c:d".replace(/:/g, "-")` ត្រឡប់ `"a-b-c-d"`។

### `repeat()`

`string.repeat(count: number): string`

- `count` គឺជាចំនួនដងដែលត្រូវបង្វិលខ្សែអក្សរ។
- ឧទាហរណ៍៖ `"123".repeat(2)` ត្រឡប់ `"123123"`

### `reverse()`

`string.reverse(): string`

- បញ្ច្រាស់ខ្សែអក្សរ។
- ឧទាហរណ៍៖ `"hello".reverse()` ត្រឡប់ `"olleh"`។

### `slice()`

`string.slice(start: number, end?: number): string`

- `start` គឺជាចំណុចចាប់ផ្ដើមបញ្ចូល (inclusive)។
- `end` គឺជាចំណុចបញ្ចប់ជម្រើស (exclusive)។
- ត្រឡប់រង-ខ្សែអក្សរពី `start` (inclusive) ដល់ `end` (exclusive)។
- ឧទាហរណ៍៖ `"hello".slice(1, 4)` ត្រឡប់ `"ell"`។
- ប្រសិនបើ `end` ត្រូវបានលុបចោល នឹងកាត់ដល់ចុងខ្សែអក្សរ។

### `split()`

`string.split(separator: string | Regexp, n?: number): list`

- `separator` គឺជាអ្នករំគំខ្សែអក្សរ។
- `n` គឺជាចំនួនជម្រើស។ ប្រសិនបើបានផ្ដល់ លទ្ធផលនឹងមាន `n` ធាតុដំបូង។
- ត្រឡប់បញ្ជីនៃរង-ខ្សែអក្សរ។
- ឧទាហរណ៍៖ `"a,b,c,d".split(",", 3)` ឬ `"a,b,c,d".split(/,/, 3)` ត្រឡប់ `["a", "b", "c"]`។

### `startsWith()`

`string.startsWith(query: string): boolean`

- `query` គឺជាខ្សែអក្សរដែលត្រូវពិនិត្យនៅដើម។
- ត្រឡប់ true ប្រសិនបើខ្សែអក្សរនេះចាប់ផ្ដើមដោយ `query`។
- ឧទាហរណ៍៖ `"hello".startsWith("he")` ត្រឡប់ `true`។

### `title()`

`string.title(): string`

- បំប្លែងខ្សែអក្សរទៅជា title case (អក្សរធំដំបូងនៃពាក្យនីមួយៗ)។
- ឧទាហរណ៍៖ `"hello world".title()` ត្រឡប់ `"Hello World"`។

### `trim()`

`string.trim(): string`

- លុបចន្លោះពីចុងទាំងពីររបស់ខ្សែអក្សរ។
- ឧទាហរណ៍៖ `"  hi  ".trim()` ត្រឡប់ `"hi"`។

## លេខ

អនុគមន៍ដែលអ្នកអាចប្រើជាមួយតម្លៃលេខ ដូចជា `42`, `3.14`។

### `abs()`

`number.abs(): number`

- ត្រឡប់តម្លៃដាច់ខាតនៃចំនួន។
- ឧទាហរណ៍៖ `(-5).abs()` ត្រឡប់ `5`។

### `ceil()`

`number.ceil(): number`

- បង្វិលចំនួនឡើងទៅចំនួនគត់ដែលនៅជិតបំផុត។
- ឧទាហរណ៍៖ `(2.1).ceil()` ត្រឡប់ `3`។

### `floor()`

`number.floor(): number`

- បង្វិលចំនួនចុះទៅចំនួនគត់ដែលនៅជិតបំផុត។
- ឧទាហរណ៍៖ `(2.9).floor()` ត្រឡប់ `2`។

### `isEmpty()`

`number.isEmpty(): boolean`

- ត្រឡប់ true ប្រសិនបើចំនួនមិនមាន។
- ឧទាហរណ៍៖ `5.isEmpty()` ត្រឡប់ `false`។

### `round()`

`number.round(digits: number): number`

- បង្វិលចំនួនទៅចំនួនគត់ដែលនៅជិតបំផុត។
- ជម្រើស ផ្ដល់ប៉ារ៉ាម៉ែត្រ `digits` ដើម្បីបង្វិលទៅចំនួនខ្ទង់ទសភាគនោះ។
- ឧទាហរណ៍៖ `(2.5).round()` ត្រឡប់ `3` ហើយ `(2.3333).round(2)` ត្រឡប់ `2.33`។

### `toFixed()`

`number.toFixed(precision: number): string`

- `precision` គឺជាចំនួនខ្ទង់ទសភាគ។
- ត្រឡប់ខ្សែអក្សរជាមួយចំនួននៅក្នុងទំរង់ fixed-point។
- ឧទាហរណ៍៖ `(3.14159).toFixed(2)` ត្រឡប់ `"3.14"`។

## បញ្ជី

អនុគមន៍ដែលអ្នកអាចប្រើជាមួយបញ្ជីធាតុតាមលំដាប់ ដូចជា `[1, 2, 3]`។

### វាល

| វាល           | ប្រភេទ   | ការពណ៌នា                        |
| ------------- | -------- | -------------------------------- |
| `list.length` | `number` | ចំនួនធាតុក្នុងបញ្ជី              |

### `contains()`

`list.contains(value: any): boolean`

- `value` គឺជាធាតុដែលត្រូវស្វែងរក។
- ត្រឡប់ true ប្រសិនបើបញ្ជីមាន `value`។
- ឧទាហរណ៍៖ `[1,2,3].contains(2)` ត្រឡប់ `true`។

### `containsAll()`

`list.containsAll(...values: any): boolean`

- `values` គឺជាធាតុមួយ ឬច្រើនដែលត្រូវស្វែងរក។
- ត្រឡប់ true ប្រសិនបើបញ្ជីមាន `values` ទាំងអស់។
- ឧទាហរណ៍៖ `[1,2,3].containsAll(2,3)` ត្រឡប់ `true`។

### `containsAny()`

`list.containsAny(...values: any): boolean`

- `values` គឺជាធាតុមួយ ឬច្រើនដែលត្រូវស្វែងរក។
- ត្រឡប់ true ប្រសិនបើបញ្ជីមានយ៉ាងហោចណាស់មួយក្នុង `values`។
- ឧទាហរណ៍៖ `[1,2,3].containsAny(3,4)` ត្រឡប់ `true`។

### `filter()`

`list.filter(value: Boolean): list`

- ត្រងធាតុរបស់បញ្ជីនេះដោយហៅអនុគមន៍ filter ដែលប្រើអថេរ `index` និង `value` ហើយត្រឡប់តម្លៃ boolean ដើម្បីកំណត់ថាតើធាតុគួរត្រូវរក្សាទុក។
- `value` គឺជាតម្លៃនៃធាតុក្នុងបញ្ជី។
- `index` គឺជាលំដាប់នៃតម្លៃបច្ចុប្បន្ន។
- ឧទាហរណ៍៖ `[1,2,3,4].filter(value > 2)` ត្រឡប់ `[3,4]`។

### `flat()`

`list.flat(): list`

- បំបែកបញ្ជីដែលឋានានុក្រមទៅជាបញ្ជីតែមួយ។
- ឧទាហរណ៍៖ `[1,[2,3]].flat()` ត្រឡប់ `[1,2,3]`។

### `isEmpty()`

`list.isEmpty(): boolean`

- ត្រឡប់ true ប្រសិនបើបញ្ជីមិនមានធាតុ។
- ឧទាហរណ៍៖ `[1,2,3].isEmpty()` ត្រឡប់ `false`។

### `join()`

`list.join(separator: string): string`

- `separator` គឺជាខ្សែអក្សរដែលត្រូវបញ្ចូលរវាងធាតុ។
- ភ្ជាប់ធាតុបញ្ជីទាំងអស់ទៅជាខ្សែអក្សរតែមួយ។
- ឧទាហរណ៍៖ `[1,2,3].join(",")` ត្រឡប់ `"1,2,3"`។

### `map()`

`list.map(value: Any): list`

- បំប្លែងធាតុនីមួយៗរបស់បញ្ជីនេះដោយហៅអនុគមន៍បំប្លែង ដែលប្រើអថេរ `index` និង `value` ហើយត្រឡប់តម្លៃថ្មីដែលត្រូវដាក់ក្នុងបញ្ជី។
- `value` គឺជាតម្លៃនៃធាតុក្នុងបញ្ជី។
- `index` គឺជាលំដាប់នៃតម្លៃបច្ចុប្បន្ន។
- ឧទាហរណ៍៖ `[1,2,3,4].map(value + 1)` ត្រឡប់ `[2,3,4,5]`។

### `reduce()`

`list.reduce(expression: Any, acc: Any): Any`

- បំបួសធាតុរបស់បញ្ជីនេះទៅជាតម្លៃតែមួយដោយដំណើរការ expression មួយសម្រាប់ធាតុនីមួយៗ។ expression អាចប្រើអថេរ `index`, `value` និង `acc` (ការប្រមូល) ហើយគួរត្រឡប់តម្លៃការប្រមូលបន្ទាប់។
- `expression` ត្រូវបានវាយតម្លៃសម្រាប់ធាតុនីមួយៗក្នុងបញ្ជី។
- `value` គឺជាតម្លៃនៃធាតុបច្ចុប្បន្នក្នុងបញ្ជី។
- `index` គឺជាលំដាប់នៃធាតុបច្ចុប្បន្ន។
- `acc` គឺជាតម្លៃប្រមូលបន្ថែម។
- ឧទាហរណ៍ (ផលបូក)៖ `[1,2,3].reduce(acc + value, 0)` ត្រឡប់ `6`។
- ឧទាហរណ៍ (អតិបរមា)៖ `values.filter(value.isType("number")).reduce(if(acc == null || value > acc, value, acc), null)` ត្រឡប់ចំនួនធំបំផុត ឬ `null` ប្រសិនបើមិនមាន។

### `reverse()`

`list.reverse(): list`

- បញ្ច្រាស់បញ្ជីនៅទីតាំង។
- ឧទាហរណ៍៖ `[1,2,3].reverse()` ត្រឡប់ `[3,2,1]`។

### `slice()`

`list.slice(start: number, end?: number): list`

- `start` គឺជាចំណុចចាប់ផ្ដើមបញ្ចូល (inclusive)។
- `end` គឺជាចំណុចបញ្ចប់ជម្រើស (exclusive)។
- ត្រឡប់ច្បាប់ចម្លងរាក់នៃផ្នែកនៃបញ្ជីពី `start` (inclusive) ដល់ `end` (exclusive)។
- ឧទាហរណ៍៖ `[1,2,3,4].slice(1,3)` ត្រឡប់ `[2,3]`។
- ប្រសិនបើ `end` ត្រូវបានលុបចោល នឹងកាត់ដល់ចុងបញ្ជី។

### `sort()`

`list.sort(): list`

- តម្រៀបធាតុបញ្ជីពីតូចទៅធំ។
- ឧទាហរណ៍៖ `[3, 1, 2].sort()` ត្រឡប់ `[1, 2, 3]`។
- ឧទាហរណ៍៖ `["c", "a", "b"].sort()` ត្រឡប់ `["a", "b", "c"]`។

### `unique()`

`list.unique(): list`

- លុបធាតុស្ទួន។
- ឧទាហរណ៍៖ `[1,2,2,3].unique()` ត្រឡប់ `[1,2,3]`។

## តំណភ្ជាប់

អនុគមន៍ដែលអ្នកអាចប្រើជាមួយតំណភ្ជាប់។ តំណភ្ជាប់អាចបង្កើតពីឯកសារ (`file.asLink()`) ឬ path (`link("path")`)។

### `asFile()`

`link.asFile(): file`

- ត្រឡប់វត្ថុឯកសារប្រសិនបើតំណភ្ជាប់ចង្អុលទៅឯកសារក្នុងតំបន់ដែលត្រឹមត្រូវ។
- ឧទាហរណ៍៖ `link("[[filename]]").asFile()`

### `linksTo()`

`link.linksTo(file): boolean`

- ត្រឡប់ថាតើឯកសារដែលតំណាងដោយ `link` មានតំណភ្ជាប់ទៅ `file` ។

## ឯកសារ

អនុគមន៍ដែលអ្នកអាចប្រើជាមួយឯកសារក្នុង vault។

### វាល

វាលខាងក្រោមអាចប្រើបានសម្រាប់ឯកសារ៖

| វាល               | ប្រភេទ   | ការពណ៌នា                                                          |
| ----------------- | -------- | ------------------------------------------------------------------ |
| `file.name`       | `string` | ឈ្មោះឯកសារនេះ។                                                    |
| `file.basename`   | `string` | ឈ្មោះឯកសារនេះដោយគ្មានផ្នែករបស់ extension។                        |
| `file.path`       | `string` | pathពេញទៅឯកសារនេះ ទាក់ទងទៅ root vault។                           |
| `file.folder`     | `string` | path ពេញទៅ folder មេ។                                             |
| `file.ext`        | `string` | extension ឯកសារសម្រាប់ឯកសារនេះ។                                  |
| `file.size`       | `number` | ទំហំឯកសារនេះ ជា bytes។                                            |
| `file.properties` | `object` | លក្ខណៈសម្បត្តិកំណត់ចំណាំសម្រាប់ឯកសារនេះ។                         |
| `file.tags`       | `list`   | ស្លាកសម្រាប់ឯកសារនេះ។ រួមមានស្លាក inline។                        |
| `file.links`      | `list`   | តំណភ្ជាប់ផ្ទៃក្នុងក្នុងឯកសារនេះ។                                  |
| `file.ctime`      | `date`   | ពេលវេលានៃពេលដែលឯកសារនេះត្រូវបានបង្កើត។                           |
| `file.mtime`      | `date`   | ពេលវេលានៃពេលដែលឯកសារនេះត្រូវបានកែប្រែចុងក្រោយ។                   |

### `asLink()`

`file.asLink(display?: string): Link`

- `display` អត្ថបទបង្ហាញជម្រើសសម្រាប់តំណភ្ជាប់។
- ត្រឡប់វត្ថុ Link ដែលបង្ហាញជាតំណភ្ជាប់ដែលដំណើរការបាន។
- ឧទាហរណ៍៖ `file.asLink()`

### `hasLink()`

`file.hasLink(otherFile: file | string): boolean`

- `otherFile` គឺជាវត្ថុឯកសារផ្សេង ឬ string path ដែលត្រូវពិនិត្យ។
- ត្រឡប់ true ប្រសិនបើ `file` ភ្ជាប់ទៅ `otherFile`។
- ឧទាហរណ៍៖ `file.hasLink(otherFile)` ត្រឡប់ `true` ប្រសិនបើមានតំណភ្ជាប់ពី `file` ទៅ `otherFile`។

### `hasProperty()`

`file.hasProperty(name: string): boolean`

- ត្រឡប់ true ប្រសិនបើកំណត់ចំណាំមាន property ឯកសារដែលបានផ្ដល់។

### `hasTag()`

`file.hasTag(...values: string): boolean`

- `values` គឺជាឈ្មោះស្លាកមួយ ឬច្រើន។
- ត្រឡប់ true ប្រសិនបើឯកសារមានស្លាកណាមួយក្នុង `values`។
- ឧទាហរណ៍៖ `file.hasTag("tag1", "tag2")` ត្រឡប់ `true` ប្រសិនបើឯកសារមានស្លាក `#tag1` ឬ `#tag2`។ វារួមមាន [[ស្លាក#ស្លាករំខានក្នុង|ស្លាករំខានក្នុង]] ផងដែរ ដូចជា `#tag1/a` ឬ `#tag2/b`។

### `inFolder()`

`file.inFolder(folder: string): boolean`

- `folder` គឺជាឈ្មោះ folder ដែលត្រូវពិនិត្យ។
- ត្រឡប់ true ប្រសិនបើឯកសារស្ថិតនៅក្នុង folder ដែលបានបញ្ជាក់ ឬ sub-folder ណាមួយរបស់វា។
- ឧទាហរណ៍៖ `file.inFolder("notes")` ត្រឡប់ `true`។

## វត្ថុ

អនុគមន៍ដែលអ្នកអាចប្រើជាមួយបណ្ដុំ key-value ដូចជា `{"a": 1, "b": 2}`។

### `isEmpty()`

`object.isEmpty(): boolean`

- ត្រឡប់ true ប្រសិនបើវត្ថុមិនមាន property ផ្ទាល់ខ្លួន។
- ឧទាហរណ៍៖ `{}.isEmpty()` ត្រឡប់ `true`។

### `keys()`

`object.keys(): list`

- ត្រឡប់បញ្ជីដែលមានសោ (keys) របស់វត្ថុ។

### `values()`

`object.values(): list`

- ត្រឡប់បញ្ជីដែលមានតម្លៃរបស់វត្ថុ។

## កន្សោមទៀងទាត់

អនុគមន៍ដែលអ្នកអាចប្រើជាមួយ pattern កន្សោមទៀងទាត់។ ឧទាហរណ៍៖ `/abc/`។

### `matches()`

`regexp.matches(value: string): boolean`

- `value` គឺជាខ្សែអក្សរដែលត្រូវសាកល្បង។
- ត្រឡប់ true ប្រសិនបើកន្សោមទៀងទាត់ត្រូវគ្នា `value`។
- ឧទាហរណ៍៖ `/abc/.matches("abcde")` ត្រឡប់ `true`។
