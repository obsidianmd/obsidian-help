## សកល

អនុគមន៍សកលត្រូវបានប្រើដោយមិនចាំបាច់មានប្រភេទ។

### `escapeHTML()`

`escapeHTML(html: string): string`

- គេចពីតួអក្សរពិសេសនៅក្នុងខ្សែអក្សរដើម្បីធ្វើឱ្យវាមានសុវត្ថិភាពសម្រាប់បញ្ចូលនៅក្នុង HTML។

### `date()`

`date(date: string): date`

- `date(string): date` ញែកខ្សែអក្សរដែលបានផ្តល់ និងត្រឡប់វត្ថុកាលបរិច្ឆេទ។
- ខ្សែអក្សរ `date` គួរតែមានទម្រង់ `YYYY-MM-DD HH:mm:ss`។

### `duration()`

`duration(value: string): duration`

- ញែកខ្សែអក្សរជារយៈពេល។ សូមមើល [[វាក្យសម្ពន្ធ Bases#គណិតវិទ្យាកាលបរិច្ឆេទ|ផ្នែកគណិតវិទ្យាកាលបរិច្ឆេទ]] សម្រាប់ទម្រង់ខ្សែអក្សរ `value`។
- រយៈពេលមិនចាំបាច់ញែកដោយច្បាស់ពេលធ្វើគណិតវិទ្យាកាលបរិច្ឆេទ (ឧទាហរណ៍ `now() + '1d'`) ប៉ុន្តែត្រូវការពេលធ្វើគណិតវិទ្យាលើរយៈពេល (ឧទាហរណ៍ `now() + (duration('1d') * 2)`)។
- ពេលធ្វើគណិតវិទ្យាលើរយៈពេលជាមួយស្កាលែ រយៈពេលត្រូវតែនៅខាងឆ្វេង។ ឧទាហរណ៍ `duration('5h') * 2` ជំនួសឱ្យ `2 * duration('5h')`។

### `file()`

`file(path: string | file | url): file`

- ត្រឡប់វត្ថុឯកសារសម្រាប់ឯកសារឬផ្លូវដែលបានផ្តល់។
- ឧទាហរណ៍: `file(link("[[filename]]"))` ឬ `file("path to file")`។

### `html()`

`html(html: string): html`

- បម្លែងខ្សែអក្សរទៅជាកូដដែលបង្ហាញជា HTML។

### `if()`

`if(condition: any, trueResult: any, falseResult?: any): any`

- `condition` គឺជាលក្ខខណ្ឌដែលត្រូវវាយតម្លៃ។
- `trueResult` គឺជាលទ្ធផលប្រសិនបើលក្ខខណ្ឌពិត។
- `falseResult` គឺជាលទ្ធផលជម្រើសប្រសិនបើលក្ខខណ្ឌមិនពិត។ បើមិនបានផ្តល់ វានឹងត្រូវបានចាត់ទុកជា `null`។
- ត្រឡប់ `trueResult` ប្រសិនបើ `condition` ពិត ឬជាតម្លៃពិត ឬ `falseResult` បើមិនដូច្នេះ។
- ឧទាហរណ៍: `if(isModified, "Modified", "Unmodified")`

### `image()`

`image(path: string | file | url): image`

- ត្រឡប់វត្ថុរូបភាពដែលនឹងបង្ហាញរូបភាពនៅក្នុងទិដ្ឋភាព។
- ឧទាហរណ៍: `image(image-property)` ឬ `image("https://obsidian.md/images/obsidian-logo-gradient.svg")`

### `icon()`

`icon(name: string): icon`

- ត្រឡប់តម្លៃដែលនឹងបង្ហាញជារូបតំណាងនៅក្នុងទិដ្ឋភាព។ ឈ្មោះរូបតំណាងត្រូវតែត្រូវគ្នាជាមួយរូបតំណាង Lucide ដែលគាំទ្រ។
- ឧទាហរណ៍: `icon("arrow-right")`។

### `link()`

`link(path: string | file, display?: value): Link`

- ញែកខ្សែអក្សរ `path` និងត្រឡប់វត org ថុ Link ដែលបង្ហាញជាតំណភ្ជាប់ទៅផ្លូវដែលបានផ្តល់។
- អ្នកអាចផ្តល់ប៉ារ៉ាម៉ែត្រ `display` ដើម្បីផ្លាស់ប្តូរអត្ថបទដែលតំណភ្ជាប់បង្ហាញ។

### `list()`

`list(element: any): List`

- ប្រសិនបើធាតុដែលបានផ្តល់ជាបញ្ជី ត្រឡប់វាដោយមិនផ្លាស់ប្តូរ។
- បើមិនដូច្នេះ រុំ `element` ដែលបានផ្តល់នៅក្នុងបញ្ជី បង្កើតបញ្ជីដែលមានធាតុតែមួយ។
- អនុគមន៍នេះអាចមានប្រយោជន៍នៅពេលដែល property មានការលាយបញ្ចូលគ្នានៃខ្សែអក្សរឬបញ្ជីនៅទូទាំង vault។
- ឧទាហរណ៍: `list("value")` ត្រឡប់ `["value"]`។

### `max()`

`max(value1: number, value2: number...): number`

- ត្រឡប់លេខធំបំផុតក្នុងចំណោមលេខទាំងអស់ដែលបានផ្តល់។

### `min()`

`min(value1: number, value2: number...): number`

- ត្រឡប់លេខតូចបំផុតក្នុងចំណោមលេខទាំងអស់ដែលបានផ្តល់។

### `now()`

`now(): date`

- `now()` ត្រឡប់វត្ថុកាលបរិច្ឆេទដែលតំណាងឱ្យពេលវេលាបច្ចុប្បន្ន។

### `number()`

`number(input: any): number`

- ព្យាយាមត្រឡប់តម្លៃដែលបានផ្តល់ជាលេខ។
- វត្ថុកាលបរិច្ឆេទនឹងត្រូវបានត្រឡប់ជាមីលីវិនាទីចាប់ពី unix epoch។
- Boolean នឹងត្រឡប់ 1 ឬ 0។
- ខ្សែអក្សរនឹងត្រូវបានញែកទៅជាលេខ ហើយត្រឡប់កំហុសប្រសិនបើលទ្ធផលមិនត្រឹមត្រូវ។
- ឧទាហរណ៍ `number("3.4")` ត org ឡប់ `3.4`។

### `duration()`

`duration(value: string): duration`

- ញែកខ្សែអក្សរជារយៈពេល។ សូមមើល [[វាក្យសម្ពន្ធ Bases#គណិតវិទ្យាកាលបរិច្ឆេទ|ផ្នែកគណិតវិទ្យាកាលបរិច្ឆេទ]] សម្រាប់ទម្រង់ខ្សែអក្សរ `value`។
- រយៈពេលមិនចាំបាច់ញែកដោយច្បាស់ពេលធ្វើគណិតវិទ្យាកាលបរិច្ឆេទ (ឧទាហរណ៍ `now() + '1d'`) ប៉ុន្តែត្រូវការពេលធ្វើគណិតវិទ្យាលើរយៈពេល (ឧទាហរណ៍ `now() + (duration('1d') * 2)`)។
- ពេលធ្វើគណិតវិទ្យាលើរយៈពេលជាមួយស្កាលែ រយៈពេលត្រូវតែនៅខាងឆ្វេង។ ឧទាហរណ៍ `duration('5h') * 2` ជំនួសឱ្យ `2 * duration('5h')`។

### `today()`

`today(): date`

- `today()` ត្រឡប់វត្ថុកាលបរិច្ឆេទដែលតំណាងឱ្យកាលបរិច្ឆេទបច្ចុប្បន្ន។ ផ្នែកពេលវេលាត្រូវបានកំណត់ទៅសូន្យ។

## ប្រភេទទូទៅ

អនុគមន៍ដែលអ្នកអាចប្រើជាមួយតម្លៃណាមួយ។ រួមបញ្ចូលខ្សែអក្សរ (ឧ. `"hello"`) លេខ (ឧ. `42`) បញ្ជី (ឧ. `[1,2,3]`) វត្ថុ និងច្រើនទៀត។

### `isTruthy()`

`any.isTruthy(): boolean`

- ត org ឡប់តម្លៃដែលបានបម្លែងទៅជា boolean។
- ឧទាហរណ៍: `1.isTruthy()` ត្រឡប់ `true`។

### `isType()`

`any.isType(type: string): boolean`

- ត org ឡប់ true ប្រសិនបើតម្លៃជាប្រភេទដែលបានផ្តល់។
- ឧទាហរណ៍: `"example".isType("string")` និង `true.isType("boolean")` ទាំងពីរត្រឡប់ true។

### `toString()`

`any.toString(): string`

- ត org ឡប់ការតំណាងខ្សែអក្សរនៃតម្លៃណាមួយ។
- ឧទាហរណ៍: `123.toString()` ត org ឡប់ `"123"`។

## កាលបរិច្ឆេទ

អនុគមន៍ដែលអ org នកអាចប org រើជាមួយកាលបរិច org ឆេទ និងពេលវេលាដូចជា `date("2025-05-27")`។ ការប org រៀបធៀបកាលបរិច org ឆេទអាចធ org វើបានដោយប org រើ [[វាក org យសម org ពន org ធ Bases#គណិតវិទ org យាកាលបរិច org ឆេទ|គណិតវិទ org យាកាលបរិច org ឆេទ]]។

### វាល

វាលខាងក org រោមអាចប org រើបានសម org រាប org កាលបរិច org ឆេទ:

| វាល              | ប org រភេទ     | ការពិពណ org នា                  |
| ------------------ | -------- | ---------------------------- |
| `date.year`        | `number` | ឆ org នាំនៃកាលបរិច org ឆេទ         |
| `date.month`       | `number` | ខែនៃកាលបរិច org ឆេទ (1–12) |
| `date.day`         | `number` | ថ org ងៃនៃខែ         |
| `date.hour`        | `number` | ម org ៉ org (0–23)              |
| `date.minute`      | `number` | នាទី (0–59)            |
| `date.second`      | `number` | វិនាទី (0–59)            |
| `date.millisecond` | `number` | មីលីវិនាទី (0–999)      |

### `date()`

`date.date(): date`

- ត org ឡប org វត org ថុកាលបរិច org ឆេទដោយដកពេលវេលាចេញ។
- ឧទាហរណ org : `now().date().format("YYYY-MM-DD HH:mm:ss")` ត org ឡប org ខ org សែអក org សរដូចជា "2025-12-31 00:00:00"

### `format()`

`date.format(format: string): string`

- `format` គឺជាខ org សែអក org សរទម org រង org (ឧ org ., `"YYYY-MM-DD"`)។
- ត org ឡប org កាលបរិច org ឆេទដែលបានធ org វើទម org រង org តាមខ org សែអក org សរទម org រង org Moment.js។
- ឧទាហរណ org : `date.format("YYYY-MM-DD")` ត org ឡប org `"2025-05-27"`។

### `time()`

`date.time(): string`

- ត org ឡប org ពេលវេលា។
- ឧទាហរណ org : `now().time()` ត org ឡប org ខ org សែអក org សរដូចជា "23:59:59"

### `relative()`

`date.relative(): string`

- ត org ឡប org ការប org រៀបធៀបដែលអាចអានបាននៃកាលបរិច org ឆេទជាមួយកាលបរិច org ឆេទពេលវេលាបច org ចុប org បន org ន org ។
- ឧទាហរណ org : `file.mtime.relative()` ត org ឡប org តម org លៃដូចជា `3 days ago`។

### `isEmpty()`

`date.isEmpty(): boolean`

- ត org ឡប org false។
