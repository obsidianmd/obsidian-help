[[ការណែនាំអំពី Bases|Bases]] អនុញ្ញាតឱ្យអ្នកបង្កើតទិដ្ឋភាពស្រដៀងនឹងមូលដ្ឋានទិន្នន័យពីកំណត់ត្រារបស់អ្នក។ នេះជារបៀបដែលអ្នកអាចបង្កើត base និងបង្កប់វានៅក្នុងកំណត់ត្រា។ រាល់ base អាចមាន[[ទិដ្ឋភាព|ទិដ្ឋភាព]]មួយ ឬច្រើន ដើម្បីបង្ហាញព័ត៌មានក្នុងវិធីផ្សេងៗគ្នា។

## បង្កើត base ថ្មី

**ក្ដារលាយពាក្យបញ្ជា:**

1. បើក **ក្ដារលាយពាក្យបញ្ជា**។
2. ជ្រើសរើស
	- **Bases: Create new base** ដើម្បីបង្កើត base ក្នុងថតដូចគ្នាជាមួយឯកសារសកម្ម។
	- **Bases: Insert new base** ដើម្បីបង្កើត base និងបង្កប់វាក្នុងឯកសារបច្ចុប្បន្ន។

**កម្មវិធីរុករកឯកសារ:**

1. នៅក្នុងកម្មវិធីរុករកឯកសារ សូមចុចកណ្ដុរស្ដាំលើថតដែលអ្នកចង់បង្កើត base ។
2. ជ្រើសរើស **New base**។

**រីបិន:**

- នៅក្នុងម៉ឺនុយរីបិនបញ្ឈរ ជ្រើសរើស **Create new base** ដើម្បីបង្កើត base ក្នុងថតដូចគ្នាជាមួយឯកសារសកម្ម។
