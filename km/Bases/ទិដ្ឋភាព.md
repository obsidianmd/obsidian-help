ទិដ្ឋភាពអនុញ្ញាតឱ្យអ្នករៀបចំព័ត៌មាននៅក្នុង [[ការណែនាំអំពី Bases|Base]] តាមវិធីជាច្រើន។ Base មួយអាចមានទិដ្ឋភាពជាច្រើន ហើយទិដ្ឋភាពនីមួយៗអាចមានការកំណត់រចនាសម្ព័ន្ធពិសេសដើម្បីបង្ហាញ តម្រៀប និងច្រោះឯកសារ។

ឧទាហរណ៍ អ្នកអាចចង់បង្កើត base មួយឈ្មោះ "សៀវភៅ" ដែលមានទិដ្ឋភាពដាច់ដោយឡែកសម្រាប់ "បញ្ជីអាន" និង "បញ្ចប់ថ្មីៗ"។

## របារឧបករណ៍

នៅផ្នែកខាងលើនៃ base មានរបារឧបករណ៍ដែលអនុញ្ញាតឱ្យអ្នកធ្វើអន្តរកម្មជាមួយទិដ្ឋភាព និងលទ្ធផលរបស់វា។

- ![[lucide-table.svg#icon]] **ម៉ឺនុយទិដ្ឋភាព** — បង្កើត កែសម្រួល និងប្តូរទិដ្ឋភាព។
- **លទ្ធផល** — កំណត់ ចម្លង និងនាំចេញឯកសារ។
- ![[lucide-arrow-up-down.svg#icon]] **តម្រៀប** — តម្រៀប និងដាក់ជាក្រុមឯកសារ។
- ![[lucide-list-filter.svg#icon]] **ច្រោះ** — ច្រោះឯកសារ។
- ![[lucide-list.svg#icon]] **លក្ខណៈសម្បត្តិ** — ជ្រើសរើសលក្ខណៈសម្បត្តិដើម្បីបង្ហាញ និងបង្កើត[[រូបមន្ត]]។
- ![[lucide-search.svg#icon]] **ស្វែងរក** — ស្វែងរកធាតុដោយប្រើលក្ខណៈសម្បត្តិដែលបង្ហាញ។
- ![[lucide-plus.svg#icon]] **ថ្មី** — បង្កើតឯកសារថ្មីនៅក្នុងទិដ្ឋភាពបច្ចុប្បន្ន។

## បន្ថែម និងប្តូរទិដ្ឋភាព

មានវិធីពីរដើម្បីបន្ថែមទិដ្ឋភាពទៅ base មួយ៖

- ចុចលើឈ្មោះទិដ្ឋភាពនៅខាងឆ្វេងផ្នែកខាងលើ ហើយជ្រើសរើស ![[lucide-plus.svg#icon]] **បន្ថែមទិដ្ឋភាព**។
- ប្រើ [[ក្ដារលាយពាក្យបញ្ជា]] ហើយជ្រើសរើស **Bases: Add view**។

ទិដ្ឋភាពដំបូងក្នុងបញ្ជីទិដ្ឋភាពរបស់អ្នកនឹងផ្ទុកតាមលំនាំដើម។ អូសទិដ្ឋភាពដោយរូបតំណាងរបស់វាដើម្បីប្តូរលំដាប់។

## ការកំណត់ទិដ្ឋភាព

ទិដ្ឋភាពនីមួយៗមានជម្រើសកំណត់រចនាសម្ព័ន្ធផ្ទាល់ខ្លួន។ ដើម្បីកែសម្រួលការកំណត់ទិដ្ឋភាព៖

1. ចុចលើឈ្មោះទិដ្ឋភាពនៅខាងឆ្វេងផ្នែកខាងលើ។
2. ចុចព្រួញស្តាំនៅជាប់ទិដ្ឋភាពដែលអ្នកចង់កំណត់រចនាសម្ព័ន្ធ។

ជម្រើសផ្សេងទៀត *ចុចស្តាំ* លើឈ្មោះទិដ្ឋភាពនៅក្នុងរបារឧបករណ៍របស់ base ដើម្បីចូលដំណើរការការកំណត់ទិដ្ឋភាពបានរហ័ស។

## តម្រង

បើកម៉ឺនុយ ![[lucide-list-filter.svg#icon]] **ច្រោះ** នៅផ្នែកខាងលើនៃ base ដើម្បីបន្ថែមតម្រង។

Base មួយដែលគ្មានតម្រងបង្ហាញឯកសារទាំងអស់នៅក្នុង vault របស់អ្នក។ តម្រងកាត់បន្ថយលទ្ធផលដើម្បីបង្ហាញតែឯកសារដែលបំពេញលក្ខខណ្ឌជាក់លាក់ប៉ុណ្ណោះ។ ឧទាហរណ៍ អ្នកអាចប្រើតម្រងដើម្បីបង្ហាញតែឯកសារដែលមាន[[ស្លាក]]ជាក់លាក់ ឬនៅក្នុងថតជាក់លាក់។ ប្រភេទតម្រងជាច្រើនមានសម្រាប់ប្រើ។

តម្រងអាចអនុវត្តចំពោះទិដ្ឋភាពទាំងអស់នៅក្នុង base មួយ ឬតែទិដ្ឋភាពតែមួយដោយជ្រើសរើសពីផ្នែកពីរនៅក្នុងម៉ឺនុយ ![[lucide-list-filter.svg#icon]] **ច្រោះ**។

- **ទិដ្ឋភាពទាំងអស់** អនុវត្តតម្រងចំពោះទិដ្ឋភាពទាំងអស់នៅក្នុង base។
- **ទិដ្ឋភាពនេះ** អនុវត្តតម្រងចំពោះទិដ្ឋភាពសកម្ម។

#### សមាសធាតុនៃតម្រង

តម្រងមានសមាសធាតុបី៖

1. **លក្ខណៈសម្បត្តិ** — អនុញ្ញាតឱ្យអ្នកជ្រើសរើស[[លក្ខណៈសម្បត្តិ]]នៅក្នុង vault របស់អ្នក រួមទាំង[[វាក្យសម្ពន្ធ Bases#លក្ខណៈសម្បត្តិឯកសារ|លក្ខណៈសម្បត្តិឯកសារ]]។
2. **សញ្ញាប្រមាណវិធី** — អនុញ្ញាតឱ្យអ្នកជ្រើសរើសវិធីប្រៀបធៀបលក្ខខណ្ឌ។ បញ្ជីសញ្ញាប្រមាណវិធីដែលមានអាស្រ័យលើប្រភេទលក្ខណៈសម្បត្តិ (អត្ថបទ កាលបរិច្ឆេទ លេខ ល។)
3. **តម្លៃ** — អនុញ្ញាតឱ្យអ្នកជ្រើសរើសតម្លៃដែលអ្នកកំពុងប្រៀបធៀប។ តម្លៃអាចរួមបញ្ចូលគណិតវិទ្យា និង[[អនុគមន៍]]។

#### សភាពផ្សំ

- **ទាំងអស់ខាងក្រោមពិត** គឺជាសេចក្តីថ្លែង `and` — លទ្ធផលនឹងបង្ហាញតែប្រសិនបើលក្ខខណ្ឌ*ទាំងអស់*ក្នុងក្រុមតម្រងត្រូវបានបំពេញ។
- **ណាមួយខាងក្រោមពិត** គឺជាសេចក្តីថ្លែង `or` — លទ្ធផលនឹងបង្ហាញប្រសិនបើលក្ខខណ្ឌ*ណាមួយ*ក្នុងក្រុមតម្រងត្រូវបានបំពេញ។
- **គ្មានខាងក្រោមពិត** គឺជាសេចក្តីថ្លែង `not` — លទ្ធផលនឹងមិនបង្ហាញប្រសិនបើលក្ខខណ្ឌ*ណាមួយ*ក្នុងក្រុមតម្រងត្រូវបានបំពេញ។

#### ក្រុមតម្រង

ក្រុមតម្រងអនុញ្ញាតឱ្យអ្នកបង្កើតតក្កវិទ្យាស្មុគស្មាញជាងមុនដោយបង្កើតការផ្សំនៃសភាពផ្សំ។

#### កម្មវិធីកែតម្រងកម្រិតខ្ពស់

ចុចប៊ូតុងកូដ ![[lucide-code-xml.svg#icon]] ដើម្បីប្រើកម្មវិធីកែ**តម្រងកម្រិតខ្ពស់**។ នេះបង្ហាញ[[វាក្យសម្ពន្ធ Bases|វាក្យសម្ពន្ធ]]ឆៅសម្រាប់តម្រង ហើយអាចប្រើជាមួយ[[អនុគមន៍]]ស្មុគស្មាញជាងមុនដែលមិនអាចបង្ហាញដោយប្រើចំណុចប្រទាក់ចុច-និង-ជ្រើសរើស។

## តម្រៀប និងដាក់ជាក្រុមលទ្ធផល

បើកម៉ឺនុយ ![[lucide-arrow-up-down.svg#icon]] **តម្រៀប** ដើម្បីតម្រៀប និងដាក់ជាក្រុមលទ្ធផលក្នុងទិដ្ឋភាពមួយ។

អ្នកអាចរៀបចំលទ្ធផលតាមលក្ខណៈសម្បត្តិមួយ ឬច្រើនតាមលំដាប់ឡើង ឬចុះ។ នេះធ្វើឱ្យងាយស្រួលក្នុងការរាយកំណត់ត្រាតាមឈ្មោះ ពេលវេលាកែសម្រួលចុងក្រោយ ឬលក្ខណៈសម្បត្តិផ្សេងទៀត — រួមទាំងរូបមន្ត។

អ្នកក៏អាចដាក់ជាក្រុមលទ្ធផលតាមលក្ខណៈសម្បត្តិមួយដើម្បីរៀបចំធាតុស្រដៀងគ្នាទៅជាផ្នែកដែលមើលឃើញខុសគ្នា។ បច្ចុប្បន្ន Obsidian គាំទ្រការដាក់ជាក្រុមតាមលក្ខណៈសម្បត្តិតែមួយប៉ុណ្ណោះ។

2. ជ្រើសរើសលក្ខណៈសម្បត្តិដែលអ្នកចង់តម្រៀប (ឬដាក់ជាក្រុម) តាម។
3. ប្រសិនបើអ្នកមានការតម្រៀបច្រើន អូសវាឡើង ឬចុះដោយប្រើ ![[lucide-grip-vertical.svg#icon]] ចំណុចទាញដើម្បីប្តូរអាទិភាព។

ជម្រើសសម្រាប់លំដាប់លទ្ធផលអាស្រ័យលើប្រភេទលក្ខណៈសម្បត្តិ៖

- **អត្ថបទ**: តម្រៀបតាម*អក្ខរក្រម* (A→Z) ឬតាម*លំដាប់អក្ខរក្រមបញ្ច្រាស* (Z→A)។
- **លេខ**: តម្រៀបពី*តូចបំផុតទៅធំបំផុត* (0→1) ឬ*ធំបំផុតទៅតូចបំផុត* (1→0)។
- **កាលបរិច្ឆេទ និងពេលវេលា**: តម្រៀបតាម*ចាស់ទៅថ្មី* ឬ*ថ្មីទៅចាស់*។

### លុបការតម្រៀប

1. បើកម៉ឺនុយ ![[lucide-arrow-up-down.svg#icon]] **តម្រៀប** នៅផ្នែកខាងលើនៃទិដ្ឋភាព។
2. ចុចប៊ូតុង ![[lucide-trash-2.svg#icon]] ធុងសម្រាម នៅជាប់ការតម្រៀប ឬក្រុមដែលអ្នកចង់លុប។

## កំណត់ ចម្លង និងនាំចេញលទ្ធផល

### កំណត់លទ្ធផល

ម៉ឺនុយ *លទ្ធផល* បង្ហាញចំនួនលទ្ធផលក្នុងទិដ្ឋភាព។ ចុចប៊ូតុងលទ្ធផលដើម្បីកំណត់ចំនួនលទ្ធផល និងចូលដំណើរការសកម្មភាពបន្ថែម។

### ចម្លងទៅក្ដារតម្បៀតភ្ជាប់

សកម្មភាពនេះចម្លងទិដ្ឋភាពទៅក្ដារតម្បៀតភ្ជាប់របស់អ្នក។ នៅពេលនៅក្នុងក្ដារតម្បៀតភ្ជាប់ អ្នកអាចបិទភ្ជាប់វាទៅក្នុងឯកសារ Markdown ឬទៅក្នុងកម្មវិធីឯកសារផ្សេងទៀតរួមទាំងសៀវភៅបញ្ជីដូចជា Google Sheets, Excel និង Numbers។

### នាំចេញ CSV

សកម្មភាពនេះរក្សាទុក CSV នៃទិដ្ឋភាពបច្ចុប្បន្នរបស់អ្នក។
