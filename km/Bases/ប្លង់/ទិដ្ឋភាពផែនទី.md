## ដំឡើងកម្មវិធីជំនួយ Maps

ទិដ្ឋភាពផែនទីត្រូវការ Obsidian 1.10។ [កម្មវិធីជំនួយ Maps](obsidian://show-plugin?id=maps) គឺជា[[កម្មវិធីជំនួយសហគមន៍|កម្មវិធីជំនួយសហគមន៍]]ផ្លូវការដែលអ្នកអាចទាញយកដោយឡែក។

1. អនុវត្តតាមការណែនាំក្នុង [[កម្មវិធីជំនួយសហគមន៍#ដំឡើងកម្មវិធីជំនួយសហគមន៍]]
2. ទាញយក និងបើក [Maps](obsidian://show-plugin?id=maps) ពីបញ្ជី

## ឧទាហរណ៍

ដើម្បីចាប់ផ្តើម សាកល្បងបង្កើតកំណត់ត្រាមួយឈ្មោះ **Eiffel Tower** ហើយចម្លងលក្ខណៈសម្បត្តិខាងក្រោមទៅក្នុងវា៖

```yaml
---
coordinates:
  - "48.85837"
  - "2.294481"
icon: "landmark"
color: "red"
tags:
  - places
---
```

នេះជាអត្ថន័យនៃកូដខាងលើ៖

| លក្ខណៈសម្បត្តិ | តម្លៃ                    |                                                                                                                                                      |
| ------------- | ------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| `coordinates` | `48.85837`<br>`2.294481` | កូអរដោនេត្រូវបានរក្សាទុកជា `latitude, longitude`។ អ្នកអាចយកកូអរដោនេដោយចុចកណ្តុរស្តាំលើទីតាំងមួយនៅលើផែនទី ហើយជ្រើសរើស **Copy coordinates**។ |
| `icon`        | `landmark`               | ឈ្មោះរូបតំណាងពី [បណ្ណាល័យ Lucide](https://lucide.dev/)។                                                                                  |
| `color`       | `red`                    | តម្លៃ CSS ត្រឹមត្រូវ៖ hex, RGB, ឈ្មោះពណ៌ ។ល។                                                                                                       |
| `tags`        | `places`                 | ស្លាកដែលយើងនឹងប្រើដើម្បីស្វែងរកសញ្ញាសម្គាល់ផែនទីក្នុង base របស់យើង។                                                                                                   |
ឥឡូវបង្កើតទិដ្ឋភាពផែនទីមួយជាមួយតម្រងសម្រាប់ស្លាក `places` ហើយកំណត់កូអរដោនេសញ្ញាសម្គាល់ រូបតំណាង និងពណ៌ ដោយប្រើលក្ខណៈសម្បត្តិដែលបានរាយខាងលើ។

អ្នកក៏អាចបើក[ឯកសារឧទាហរណ៍](https://github.com/obsidianmd/obsidian-maps/tree/master/examples)ទាំងនេះក្នុង Obsidian ដើម្បីមើលទិដ្ឋភាពផែនទីដែលដំណើរការជាមួយសញ្ញាសម្គាល់ រូបតំណាង និងពណ៌ដែលបានកំណត់រួចហើយ។

## ការកំណត់

ការកំណត់ទិដ្ឋភាពផែនទីអាចកំណត់រចនាសម្ព័ន្ធក្នុង [[ទិដ្ឋភាព#ការកំណត់ទិដ្ឋភាព|ការកំណត់ទិដ្ឋភាព]]។

- កម្ពស់បង្កប់
- កូអរដោនេកណ្តាល
- កំហិតការពង្រីក
- កូអរដោនេសញ្ញាសម្គាល់ ពណ៌ និងរូបតំណាង
- ផ្ទៃខាងក្រោយ

### សញ្ញាសម្គាល់

#### កូអរដោនេ

ដើម្បីបង្ហាញម្ជុលនៅលើផែនទី សូមចូលទៅ[[ទិដ្ឋភាព#ការកំណត់ទិដ្ឋភាព|ការកំណត់ទិដ្ឋភាព]] ហើយជ្រើសរើសលក្ខណៈសម្បត្តិ **marker coordinates**។ លក្ខណៈសម្បត្តិត្រូវតែមានកូអរដោនេ latitude និង longitude។ ទម្រង់ខាងក្រោមត្រូវបានទទួលយក៖

```yaml
# លក្ខណៈសម្បត្តិអត្ថបទ
coordinates: "lat, lng"

# លក្ខណៈសម្បត្តិបញ្ជី
coordinates:
  - "lat"
  - "lng"
```

ប្រសិនបើអ្នករក្សាទុកកូអរដោនេជាលក្ខណៈសម្បត្តិ `latitude` និង `longitude` ដាច់ដោយឡែកពីគ្នា អ្នកអាចផ្សំពួកវាជាមួយលក្ខណៈសម្បត្តិ[[វាក្យសម្ពន្ធ Bases#រូបមន្ត|រូបមន្ត]]ដោយកំណត់វាជាអារេនៃកូអរដោនេដោយប្រើរូបមន្តខាងក្រោម៖ `[latitude, longitude]`។

#### រូបតំណាង

បន្ថែមរូបតំណាងទៅសញ្ញាសម្គាល់ដោយកំណត់លក្ខណៈសម្បត្តិ **marker icons**។ ឧទាហរណ៍ អ្នកអាចបន្ថែមលក្ខណៈសម្បត្តិមួយឈ្មោះ `icon` ទៅកំណត់ត្រារបស់អ្នក ហើយផ្តល់តម្លៃដូចជា `landmark` ឬ `utensils` ពី[បណ្ណាល័យ Lucide](https://lucide.dev/icons/) ដែលភ្ជាប់មកជាមួយ Obsidian។

##### ប្រើរូបមន្តដើម្បីកំណត់រូបតំណាង

ឧបមាថាអ្នកចង់ឱ្យភោជនីយដ្ឋានទាំងអស់មានរូបតំណាងដូចគ្នានៅលើផែនទី៖

1. បង្កើតកំណត់ត្រាមួយឈ្មោះ **Restaurants** ហើយបន្ថែមលក្ខណៈសម្បត្តិមួយឈ្មោះ `icon` ដែលមានតម្លៃ `utensils`។
2. ផ្តល់ឱ្យកំណត់ត្រាភោជនីយដ្ឋាននូវលក្ខណៈសម្បត្តិមួយឈ្មោះ `type` ដែលភ្ជាប់ទៅកំណត់ត្រា `[[Restaurants]]`។
3. បន្ថែមលក្ខណៈសម្បត្តិរូបមន្តមួយឈ្មោះ `Type icon` ទៅ base របស់អ្នកជាមួយកូដខាងក្រោម៖
	```js
	list(type)[0].asFile().properties.icon
	```
4. ជ្រើសរើស `Type icon` ជារូបតំណាងសញ្ញាសម្គាល់របស់អ្នកក្នុងការកំណត់ទិដ្ឋភាព។

ហើយចប់! ឥឡូវផែនទីរបស់អ្នកនឹងបង្ហាញរូបតំណាងពី *ប្រភេទ* នៃកន្លែង មិនមែនពីកន្លែងខ្លួនឯងទេ។

#### ពណ៌

កំណត់ពណ៌នៃសញ្ញាសម្គាល់។ ទទួលយកតម្លៃជា RGB `rgb(0,0,0)`, HEX `#000`, ឬអថេរ CSS ដូចជា `var(--color-blue)`។ ដូចក្នុងឧទាហរណ៍រូបតំណាងខាងលើ អ្នកអាចប្រើលក្ខណៈសម្បត្តិរូបមន្តដើម្បីកំណត់ពណ៌ដោយស្វ័យប្រវត្តិ។

### ផ្ទៃខាងក្រោយ

#### ក្បឿងផែនទី

ក្បឿងផែនទីគឺជាវិធីស្តង់ដារដើម្បីបង្ហាញផែនទីឌីជីថល។ មានសេវាកម្មជាច្រើនដែលអ្នកអាចប្រើដើម្បីប្ដូរផែនទីជាមួយរចនាប័ទ្ម ពណ៌ និងពុម្ពអក្សរពិសេស។ Maps គាំទ្រទាំងក្បឿង raster និង vector ហើយទទួលយក URL ក្បឿងភាគច្រើន រួមទាំង URL TileJSON។

[OpenFreeMap](https://openfreemap.org/) ផ្តល់រចនាប័ទ្មមួយចំនួនដែលអ្នកអាចប្រើដោយឥតគិតថ្លៃ។ សាកល្បងប្រើ URL មួយក្នុងចំណោមខាងក្រោមក្នុងការកំណត់ **Map tiles**៖

| ឈ្មោះ     | URL                                              |
| -------- | ------------------------------------------------ |
| Dark     | `https://tiles.openfreemap.org/styles/dark`      |
| Positron | `https://tiles.openfreemap.org/styles/positron`  |
| Liberty  | ``https://tiles.openfreemap.org/styles/liberty`` |
#### តំណភ្ជាប់មានប្រយោជន៍

- [Maputnik](https://maputnik.github.io/) សម្រាប់ប្ដូរក្បឿងផែនទីតាមតម្រូវការ។
- [Protomaps](https://protomaps.com/) សម្រាប់បង្ហោះក្បឿងផែនទីដោយខ្លួនឯង។
- សេវាកម្មបង្ហោះផ្សេងទៀតដែលមានកម្រិតឥតគិតថ្លៃរួមទាំង [MapTiler](https://www.maptiler.com/) និង [Mapbox](https://www.mapbox.com/)។


## គន្លឹះ

អ្នកអាចភ្ជាប់ទៅសេវាកម្មផែនទីពេញនិយមដោយប្រើ [[រូបមន្ត]]។ ឧទាហរណ៍ ម្ជុលរបស់អ្នកអាចបង្ហាញតំណភ្ជាប់ទៅ Google Maps ដោយប្រើរូបមន្តខាងក្រោម៖

```js
link("https://www.google.com/maps/search/" + file.name.replace(" ","+"),"Google Maps")
```
