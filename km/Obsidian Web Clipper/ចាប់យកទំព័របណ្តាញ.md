នៅពេលអ្នកដំឡើង [[ការណែនាំអំពី Obsidian Web Clipper|Web Clipper]] ផ្នែកបន្ថែមកម្មវិធីរុករក អ្នកអាចចូលប្រើវាបានច្រើនវិធី អាស្រ័យលើកម្មវិធីរុករករបស់អ្នក៖

1. រូបតំណាង Obsidian នៅក្នុងរបារឧបករណ៍កម្មវិធីរុករករបស់អ្នក។
2. ផ្លូវកាត់ក្តារចុច ដើម្បីធ្វើឱ្យផ្នែកបន្ថែមសកម្មពីក្តារចុចរបស់អ្នក។
3. ម៉ឺនុយបរិបទ ដោយចុចកណ្ដុរស្ដាំលើទំព័រវែបដែលអ្នកកំពុងចូលមើល។

ដើម្បីរក្សាទុកទំព័រទៅក្នុង Obsidian ចុចប៊ូតុង **Add to Obsidian**។

## ចាប់យកទំព័រ

នៅពេលអ្នកបើកផ្នែកបន្ថែម Web Clipper ស្រង់ទិន្នន័យពីទំព័រវែបបច្ចុប្បន្នតាមការកំណត់នៅក្នុង[[Obsidian Web Clipper/ពុម្ព|ពុម្ព]]របស់អ្នក។ អ្នកអាចបង្កើតពុម្ពផ្ទាល់ខ្លួន និងប្ដូរលទ្ធផលតាមតម្រូវការដោយប្រើ[[អថេរ]] និង[[តម្រង]]។

តាមលំនាំដើម Web Clipper ព្យាយាមស្រង់យកតែមាតិកាអត្ថបទសំខាន់ដោយឆ្លាតវៃ ដោយមិនរាប់បញ្ចូលធាតុផ្សេងទៀតនៅលើទំព័រ។ ទោះយ៉ាងណា អ្នកអាចបដិសេធឥរិយាបថនេះតាមវិធីខាងក្រោម៖

- បើពុម្ពផ្ទាល់ខ្លួនមាន វាប្រើពុម្ពរបស់អ្នក។
- បើមានការជ្រើសរើសអត្ថបទ វាប្រើការជ្រើសរើសនោះ។ អ្នកអាចប្រើ `Ctrl/Cmd+A` ដើម្បីជ្រើសរើសទំព័រទាំងមូល។
- បើមាន[[ពណ៌ចំណាំទំព័របណ្តាញ|ការបន្លិច]]ណាមួយ វាប្រើការបន្លិចនោះ។

## ទាញយករូបភាព

រូបភាពមិនត្រូវបានទាញយកដោយស្វ័យប្រវត្តិនៅពេលអ្នកប្រើ Web Clipper ទេ។ ជំនួសមកវិញ រូបភាពភ្ជាប់ទៅ URL នៅលើវែបរបស់ពួកវា។ នេះជួយសន្សំទំហំក្នុង vault របស់អ្នក ប៉ុន្តែវាមានន័យថារូបភាពនឹងមិនអាចចូលប្រើបាន offline ឬប្រសិនបើ URL ឈប់ដំណើរការ។

អ្នកអាចទាញយករូបភាពសម្រាប់ឯកសារណាមួយក្នុង Obsidian ដោយប្រើ[[ក្ដារលាយពាក្យបញ្ជា|ពាក្យបញ្ជា]]ដែលមានឈ្មោះ **Download attachments for current file**។ ពាក្យបញ្ជានេះក៏អាចកំណត់ទៅផ្លូវកាត់ក្តារចុចក្នុង Obsidian ផងដែរ។

## ផ្លូវកាត់ក្តារចុច

Web Clipper រួមបញ្ចូលផ្លូវកាត់ក្តារចុចដែលអ្នកអាចប្រើដើម្បីបង្កើនល្បឿនដំណើរការការងាររបស់អ្នក។ ដើម្បីប្ដូរការកំណត់គ្រាប់ចុច ទៅកាន់ **Web Clipper Settings** → **General** ហើយធ្វើតាមការណែនាំសម្រាប់កម្មវិធីរុករករបស់អ្នក។ ការកំណត់អាចប្ដូរបានសម្រាប់កម្មវិធីរុករកទាំងអស់លើកលែង Safari ដែលមិនគាំទ្រការកែសម្រួលផ្លូវកាត់ក្តារចុច។

| សកម្មភាព                  | macOS         | Windows/Linux  |
| ----------------------- | ------------- | -------------- |
| បើក clipper            | `Cmd+Shift+O` | `Ctrl+Shift+O` |
| ចាប់យករហ័ស              | `Opt+Shift+O` | `Alt+Shift+O`  |
| បិទ/បើករបៀបបន្លិច | `Opt+Shift+H` | `Alt+Shift+H`  |

## មុខងារចំណុចប្រទាក់

ចំណុចប្រទាក់ Web Clipper ត្រូវបានបែងចែកជាបួនផ្នែក៖

1. **បឋមកថា** ដែលអ្នកអាចប្ដូរពុម្ព បើក[[ពណ៌ចំណាំទំព័របណ្តាញ|ការបន្លិច]] [[កម្មវិធីអាន|របៀបអាន]] និងចូលប្រើការកំណត់។
2. **លក្ខណៈសម្បត្តិ** បង្ហាញ[[លក្ខណៈសម្បត្តិ|មេតាដាតា]]ដែលស្រង់ចេញពីទំព័រ ដែលនឹងត្រូវបានរក្សាទុកជា[[លក្ខណៈសម្បត្តិ]]ក្នុង Obsidian។
3. **មាតិកាកំណត់ត្រា** ដែលនឹងត្រូវបានរក្សាទុកទៅ Obsidian។
4. **បាតកថា** អនុញ្ញាតឱ្យអ្នកជ្រើសរើស vault និងថត ហើយបន្ថែមទៅ Obsidian។

- ប៊ូតុង **ច្រើនទៀត (...)** ដើម្បីបង្ហាញអថេរទំព័រដែលអ្នកអាចប្រើក្នុងពុម្ព។
- ប៊ូតុង **កម្មវិធីបន្លិច** ដើម្បីបើក[[ពណ៌ចំណាំទំព័របណ្តាញ|ការបន្លិច]]។
