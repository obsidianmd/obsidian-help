[[ការណែនាំអំពី Obsidian Web Clipper|Web Clipper]] គាំទ្រតក្កវិទ្យាពុម្ពសម្រាប់លក្ខខណ្ឌ វដ្ត និងការកំណត់អថេរ។ វាក្យសម្ពន្ធនេះត្រូវបានបំផុសគំនិតពីភាសាពុម្ព [Twig](https://twig.symfony.com/) និង [Liquid](https://shopify.github.io/liquid/)។

> [!warning] ត្រូវការកំណែចុងក្រោយបំផុត
> មុខងារតក្កវិទ្យាត្រូវការ Obsidian Web Clipper 1.0.0 ដែលមិនទាន់ត្រូវបានអនុម័តនៅលើហាងផ្នែកបន្ថែមទាំងអស់នៅឡើយ។

## លក្ខខណ្ឌ

ប្រើ `{% if %}` ដើម្បីរួមបញ្ចូលមាតិកាតាមលក្ខខណ្ឌដោយផ្អែកលើអថេរ ឬកន្សោម។

```twig
{% if author %}
Author: {{author}}
{% endif %}
```

ប្រើ `{% else %}` ដើម្បីផ្តល់មាតិកាជំនួស និង `{% elseif %}` ដើម្បីភ្ជាប់លក្ខខណ្ឌច្រើន:

```twig
{% if status == "published" %}
Live article
{% elseif status == "draft" %}
Draft article
{% else %}
Unknown status
{% endif %}
```

### សញ្ញាប្រមាណវិធីប្រៀបធៀប

សញ្ញាប្រមាណវិធីប្រៀបធៀបខាងក្រោមត្រូវបានគាំទ្រ:

| សញ្ញាប្រមាណវិធី | ការពិពណ៌នា |
|----------|-------------|
| `==` | ស្មើនឹង |
| `!=` | មិនស្មើនឹង |
| `>` | ធំជាង |
| `<` | តូចជាង |
| `>=` | ធំជាង ឬស្មើនឹង |
| `<=` | តូចជាង ឬស្មើនឹង |
| `contains` | ពិនិត្យថាតើខ្សែអក្សរមានខ្សែអក្សររង ឬអារេមានតម្លៃ |

ឧទាហរណ៍:
- `{% if title == "Home" %}` — ការប្រៀបធៀបខ្សែអក្សរ
- `{% if price >= 100 %}` — ការប្រៀបធៀបលេខ
- `{% if title contains "Review" %}` — ពិនិត្យខ្សែអក្សររង
- `{% if tags contains "important" %}` — សមាជិកភាពអារេ

### សញ្ញាប្រមាណវិធីតក្កវិទ្យា

ផ្សំលក្ខខណ្ឌដោយប្រើសញ្ញាប្រមាណវិធីតក្កវិទ្យា:

| សញ្ញាប្រមាណវិធី | ជម្រើសផ្សេង | ការពិពណ៌នា                         |
| -------- | ----------- | ----------------------------------- |
| `and`    | `&&`        | លក្ខខណ្ឌទាំងពីរត្រូវតែពិត        |
| `or`     | \|\|        | យ៉ាងហោចណាស់លក្ខខណ្ឌមួយត្រូវតែពិត |
| `not`    | `!`         | បដិសេធលក្ខខណ្ឌ                 |

ឧទាហរណ៍:
- `{% if author and published %}` — ទាំងពីរត្រូវតែមាន
- `{% if draft or archived %}` — លក្ខខណ្ឌណាមួយ
- `{% if not hidden %}` — បដិសេធ
- `{% if (premium or featured) and published %}` — លក្ខខណ្ឌដែលបានដាក់ក្រុម

### ភាពពិត

នៅពេលអថេរត្រូវបានប្រើដោយគ្មានសញ្ញាប្រមាណវិធីប្រៀបធៀប វាត្រូវបានវាយតម្លៃសម្រាប់ "ភាពពិត":

- `false`, `null`, `undefined`, ខ្សែអក្សរទទេ `""`, និង `0` ត្រូវបានចាត់ទុកជា **មិនពិត**។
- អារេទទេ `[]` ត្រូវបានចាត់ទុកជា **មិនពិត**។
- អ្វីៗផ្សេងទៀតគឺ **ពិត**។

```twig
{% if content %}
Has content
{% endif %}
```

## កំណត់អថេរ

ប្រើ `{% set %}` ដើម្បីបង្កើត ឬកែប្រែអថេរនៅក្នុងពុម្ពរបស់អ្នក:

```twig
{% set slug = title|lower|replace:" ":"-" %}
File: {{slug}}.md
```

អថេរអាចត្រូវបានកំណត់ទៅ:
- អថេរផ្សេង: `{% set name = author %}`
- តម្លៃត្រង់: `{% set count = 5 %}` ឬ `{% set label = "Draft" %}`
- កន្សោមជាមួយតម្រង: `{% set excerpt = content|truncate:100 %}`
- លទ្ធផល selector: `{% set comments = selector:.comment %}`

អថេរដែលកំណត់ដោយ `{% set %}` អាចត្រូវបានប្រើនៅក្នុងតក្កវិទ្យាពុម្ពបន្តបន្ទាប់ និងនៅក្នុងលទ្ធផល `{{variable}}`។

## តម្លៃជំនួស

ប្រើសញ្ញាប្រមាណវិធី `??` ដើម្បីផ្តល់តម្លៃជំនួសនៅពេលអថេរទទេ ឬមិនត្រូវបានកំណត់:

```twig
{{title ?? "Untitled"}}
```

ប្រសិនបើ `title` ទទេ មិនត្រូវបានកំណត់ ឬមិនពិត តម្លៃជំនួស `"Untitled"` នឹងត្រូវបានប្រើជំនួសវិញ។

នេះគឺជាទម្រង់កាត់សម្រាប់សេចក្តីថ្លែង `if` ដែលស្មើគ្នា:

```twig
{% if title %}{{title}}{% else %}Untitled{% endif %}
```

### ការភ្ជាប់តម្លៃជំនួស

អ្នកអាចភ្ជាប់តម្លៃជំនួសច្រើន:

```twig
{{title ?? headline ?? "No title"}}
```

នេះនឹងប្រើ `title` ប្រសិនបើមាន បើមិនដូច្នោះ `headline` បើមិនដូច្នោះខ្សែអក្សរ `"No title"`។

### ជាមួយតម្រង

តម្រងភ្ជាប់យ៉ាងតឹងជាង `??` ដូច្នេះតម្រងត្រូវបានអនុវត្តមុនការពិនិត្យជំនួស:

```twig
{{title|upper ?? "UNTITLED"}}
```

នេះអនុវត្ត `upper` ទៅ `title` ជាមុនសិន បន្ទាប់មកត្រឡប់ទៅ `"UNTITLED"` ប្រសិនបើលទ្ធផលទទេ។ ដើម្បីអនុវត្តតម្រងទៅតម្លៃជំនួស ប្រើវង់ក្រចក ឬកន្សោមដាច់ដោយឡែក:

```twig
{{title ?? "Untitled"|lower}}
```

នេះនឹងប្រើ `title` ប្រសិនបើមាន បើមិនដូច្នោះអនុវត្ត `lower` ទៅតម្លៃជំនួស ដែលផ្តល់លទ្ធផល `"untitled"`។

## វដ្ត

ប្រើ `{% for %}` ដើម្បីធ្វើឡើងវិញលើអារេ:

```twig
{% for item in schema:author %}
- {{item.name}}
{% endfor %}
```

### ប្រភពវដ្ត

អ្នកអាចធ្វើវដ្តលើ:
- អារេ Schema: `{% for item in schema:author %}`
- លទ្ធផល Selector: `{% for comment in selector:.comment %}`
- អថេរដែលបានកំណត់ពីមុន: `{% set items = selector:.item %}{% for item in items %}`

### អថេរវដ្ត

នៅក្នុងវដ្ត អ្នកមានសិទ្ធិចូលប្រើវត្ថុ `loop` ដែលមានលក្ខណៈសម្បត្តិខាងក្រោម:

| អថេរ | ការពិពណ៌នា |
|----------|-------------|
| `loop.index` | ការធ្វើឡើងវិញបច្ចុប្បន្ន (ចាប់ផ្តើមពី 1) |
| `loop.index0` | ការធ្វើឡើងវិញបច្ចុប្បន្ន (ចាប់ផ្តើមពី 0) |
| `loop.first` | `true` ប្រសិនបើជាការធ្វើឡើងវិញដំបូង |
| `loop.last` | `true` ប្រសិនបើជាការធ្វើឡើងវិញចុងក្រោយ |
| `loop.length` | ចំនួនសរុបនៃធាតុ |

```twig
{% for tag in tags %}
{{loop.index}}. {{tag}}
{% if loop.last %} (end of list){% endif %}
{% endfor %}
```

សម្រាប់ភាពឆបគ្នាថយក្រោយ អ្នកក៏អាចប្រើ `item_index` (ដែល `item` គឺជាឈ្មោះអថេរវដ្តរបស់អ្នក) ដើម្បីទទួលបានទីតាំងចាប់ផ្តើមពី 0:

```twig
{% for tag in tags %}
{{tag_index}}. {{tag}}
{% endfor %}
```

### ការចូលប្រើធាតុអារេតាមសន្ទស្សន៍

ប្រើសញ្ញាវង់ក្រចកការ៉េដើម្បីចូលប្រើធាតុអារេតាមសន្ទស្សន៍:

```twig
{{items[0]}}
{{items[loop.index0]}}
```

នេះមានប្រយោជន៍នៅពេលអ្នកត្រូវការចូលប្រើធាតុពីអារេច្រើនស្របគ្នា:

```twig
{% set transcripts = selector:.transcript-text %}
{% set timestamps = selector:.timestamp %}

{% for line in transcripts %}
{{timestamps[loop.index0]}} - {{line}}
{% endfor %}
```

សញ្ញាវង់ក្រចកការ៉េក៏ដំណើរការជាមួយលក្ខណៈសម្បត្តិវត្ថុផងដែរ:

```twig
{{user["name"]}}
{{data["my-key"]}}
```

### វដ្តស្រទាប់

វដ្តអាចត្រូវបានស្រទាប់សម្រាប់រចនាសម្ព័ន្ធទិន្នន័យស្មុគ្រស្មាញ:

```twig
{% for section in sections %}
## {{section.title}}
{% for item in section.items %}
- {{item}}
{% endfor %}
{% endfor %}
```

## ផ្សំតក្កវិទ្យា

លក្ខខណ្ឌ និងវដ្តអាចត្រូវបានផ្សំជាមួយគ្នា:

```twig
{% for item in items %}
{% if item.active %}
- {{item.name}}
{% endif %}
{% endfor %}
```

## លំដាប់នៃការវាយតម្លៃ

តក្កវិទ្យាពុម្ពត្រូវបានដំណើរការតាមលំដាប់ខាងក្រោម:

1. **តក្កវិទ្យាពុម្ព** — `{% if %}`, `{% for %}`, `{% set %}`, និង `{{variables}}` ត្រូវបានវាយតម្លៃជាមុនសិន
2. **អថេរ prompt** — [[អថេរ#អថេរ prompt|អថេរ prompt]] ដូចជា `{{"summarize this"|prompt}}` ត្រូវបានផ្ញើទៅ Interpreter បន្ទាប់ពីតក្កវិទ្យាពុម្ពបញ្ចប់

នេះមានន័យថាអ្នកអាចប្រើតក្កវិទ្យាពុម្ពដើម្បីបង្កើត prompt ដោយថាមវន្ត ប៉ុន្តែលទ្ធផល prompt មិនមានសម្រាប់ប្រើក្នុងលក្ខខណ្ឌ ឬវដ្តទេ។
