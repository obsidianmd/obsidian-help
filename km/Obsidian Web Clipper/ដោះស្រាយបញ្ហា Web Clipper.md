## бһ‘бһјбһ‘бҹ…

### бһҳбһ¶бһҸбһ·бһҖбһ¶бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһ”бһ¶бһҸбҹӢ

бһҸбһ¶бһҳбһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳ Web Clipper бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһ…бһ¶бһ”бҹӢбһҷбһҖбһҳбһ¶бһҸбһ·бһҖбһ¶бһ–бһёбһ‘бҹҶбһ–бҹҗбһҡбһҠбҹ„бһҷбһҶбҹ’бһӣбһ¶бһҸбһңбҹғбҹ” бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһңбһ¶бһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһҳбһ·бһ“бһҮбҹ„бһӮбһҮбҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһ“бҹ…бһӮбҹ’бһҡбһ”бҹӢбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһЎбһҫбһҷбҹ”

Web Clipper бһ”бҹ’бһҡбһҫ [Defuddle](https://github.com/kepano/defuddle) бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһ”бҹӢбһҷбһҖбһҸбҹӮбһҳбһ¶бһҸбһ·бһҖбһ¶бһ…бһҳбҹ’бһ”бһ„бһҡбһ”бһҹбҹӢбһ‘бҹҶбһ–бҹҗбһҡбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ” бһңбһ¶бһҳбһ·бһ“бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣ header, footer бһ“бһ·бһ„бһ’бһ¶бһҸбһ»бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бҹҖбһҸбһ‘бҹҒ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ–бҹҒбһӣбһҒбҹ’бһӣбҹҮбһңбһ¶бһўбһ¶бһ…бһҳбһ¶бһ“бһ—бһ¶бһ–бһҡбһ№бһҸбһ”бһ“бҹ’бһҸбһ№бһ„бһ–бҹҒбһҖ бһ бһҫбһҷбһӣбһ»бһ”бһҳбһ¶бһҸбһ·бһҖбһ¶бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…[бһҡбһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚ bug](https://github.com/kepano/defuddle) бһ‘бҹ… Defuddleбҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһҡбҹҶбһӣбһ„ Defuddle бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Web Clipper бһҹбһјбһҳбһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

- бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһўбһҸбҹ’бһҗбһ”бһ‘ бһ¬бһ”бҹ’бһҡбһҫ `Cmd/Ctrl+A` бһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһўбһҸбҹ’бһҗбһ”бһ‘бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ”
- [[бһ–бһҺбҹҢбһ…бҹҶбһҺбһ¶бҹҶбһ‘бҹҶбһ–бҹҗбһҡбһ”бһҺбҹ’бһҸбһ¶бһү|бһҡбҹҶбһӣбҹҒбһ…бһҳбһ¶бһҸбһ·бһҖбһ¶]] бһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ…бһ¶бһ”бҹӢбһҷбһҖбһҷбҹүбһ¶бһ„бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбҹ”
- бһ”бҹ’бһҡбһҫ [[Obsidian Web Clipper/бһ–бһ»бһҳбҹ’бһ–|бһӮбҹҶбһҡбһјбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“]] бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһ“бҹ„бҹҮбҹ”

### бһҳбһ·бһ“бһҳбһ¶бһ“бһҳбһ¶бһҸбһ·бһҖбһ¶бһ”бһ„бҹ’бһ бһ¶бһүбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Obsidian

бһ”бһҫбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһғбһҫбһүбһҳбһ¶бһҸбһ·бһҖбһ¶бһҺбһ¶бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„ Obsidian бһ“бҹ…бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһ…бһ»бһ… **Add to Obsidian**бҹ–

- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҖбҹҶбһ бһ»бһҹбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ [[бһҮбҹҶбһ“бһҪбһҷ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡ#бһ…бһ¶бһ”бҹӢбһҷбһҖбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ» Console|бһҖбһ»бһ„бһҹбһјбһӣбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ]] бһҡбһ”бһҹбҹӢ Obsidianбҹ”
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҗбһ¶бһҲбҹ’бһҳбҹ„бҹҮ vault бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ Web Clipper бһҸбҹ’бһҡбһјбһңбһӮбҹ’бһ“бһ¶бһҷбҹүбһ¶бһ„бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҮбһ¶бһҳбһҪбһҷ *бһҲбҹ’бһҳбҹ„бҹҮ vault* бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҖбҹ’бһ“бһ»бһ„ Obsidian *бһҳбһ·бһ“бһҳбҹӮбһ“бһ•бҹ’бһӣбһјбһң vault бһ‘бҹҒ*бҹ”
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҗбһ¶бһҲбҹ’бһҳбҹ„бҹҮбһҗбһҸбһҜбһҖбһҹбһ¶бһҡбһҳбһ¶бһ“бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ”

## Linux

#### Obsidian бһҳбһ·бһ“бһ”бһҫбһҖ

- бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһ–бһ·бһ’бһёбһҖбһ¶бһҡ [[Obsidian URI]] [[Obsidian URI#бһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮ Obsidian URI|бһ”бһ¶бһ“бһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮбһ бһҫбһҷ]]бҹ”
- бһ”бһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ Firefox бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҸбҹ’бһҡбһјбһң[бһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮбһңбһ¶бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ](https://kb.mozillazine.org/Register_protocol)бҹ”

#### Obsidian бһ”бһҫбһҖбһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ¶бһ“бһҸбҹӮбһҲбҹ’бһҳбҹ„бҹҮбһҜбһҖбһҹбһ¶бһҡбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ

бһңбһ¶бһ‘бҹҶбһ“бһ„бһҮбһ¶ Obsidian бһҳбһ·бһ“бһўбһ¶бһ…бһ…бһјбһӣбһ”бҹ’бһҡбһҫ clipboard бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫ clipboard бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһҮбһјбһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ… Obsidianбҹ” бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһҠбһӣбҹӢбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“ sandbox бһ“бһ·бһ„бһҹбһ·бһ‘бҹ’бһ’бһ· clipboardбҹ”

бһ”бһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ Wayland бһҹбһјбһҳбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶ Obsidian бһҳбһ¶бһ“бһҹбһ·бһ‘бҹ’бһ’бһ·бһўбһ¶бһ“ clipboard бһ“бҹ…бһ–бҹҒбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһҸбҹ„бһҸбҹ” бһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸбһ“бҹҒбҹҮбһўбһ¶бһ…бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһ„бҹ’бһўбһҪбһ… tiling бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ§. Hyprland бһ¬ Swayбҹ”

бһ”бһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ Hyprlandбҹ–

```ini
# ~/.config/hypr/hyprland.conf
misc {
    focus_on_activate = true
}
```

бһ”бһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ Swayбҹ–

```ini
# ~/.config/sway/config
for_window [class="obsidian"] focus_on_window_activation focus
```

- бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ бһҹбһјбһҳбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ”бҹ’бһҸбһјбһҡбһ‘бҹ… **Legacy mode** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ **Web Clipper Settings** вҶ’ **General**бҹ” бһңбһ¶бһ“бһ№бһ„бһҡбҹҶбһӣбһ„ clipboard бһ бһҫбһҷбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҳбһ¶бһҸбһ·бһҖбһ¶бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҸбһ¶бһҳбһҡбһҷбҹҲ URIбҹ” бһҹбһјбһҳбһҖбһҸбҹӢбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҗбһ¶бһ“бҹҒбҹҮбһ“бһ№бһ„бһҖбҹҶбһҺбһҸбҹӢбһ…бҹҶбһ“бһҪбһ“бһҸбһҪбһўбһҖбҹ’бһҹбһҡбһҠбҹӮбһӣбһўбһ¶бһ…бһ…бһ¶бһ”бҹӢбһҷбһҖбһ”бһ¶бһ“бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ бһ“бһ·бһ„бһҖбһ¶бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷ Linux бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҫбһҖбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳ Web Clipper бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Safariбҹ–

1. бһ…бһјбһӣбһ‘бҹ… Safari бһ…бһ»бһ…бһ”бҹҠбһјбһҸбһ»бһ„бһҒбһ¶бһ„бһҶбҹ’бһңбҹҒбһ„бһ”бҹҶбһ•бһ»бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһ”бһ¶бһҡ URL бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ бһңбһ¶бһҳбһҫбһӣбһ‘бҹ…бһҠбһјбһ…бһҮбһ¶бһ…бһҸбһ»бһҖбҹ„бһҺбһҖбҹӮбһ„бһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ”
2. бһ…бһ»бһ… **Manage Extensions**бҹ”
3. бһ”бһҫбһҖ **Obsidian Web Clipper** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһүбҹ’бһҮбһё Extensionsбҹ”
4. бһ…бҹҒбһүбһ–бһёбһҳбҹүбһәбһ“бһ»бһҷбҹ”
5. бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳ **бһ…бһ»бһ…бһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бһҠбһ»бҹҶбһ•бһ¶бһҹбҹҠбһӣбҹӢ** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһ”бһ¶бһҡ URLбҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷ Web Clipper бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһӮбҹ’бһҡбһ”бҹӢбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбҹ–

1. бһ…бһјбһӣбһ‘бҹ… iOS **[[бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ]]** вҶ’ **Apps** вҶ’ **Safari** вҶ’ **Extensions**бҹ”
2. бһ“бҹ…бһҖбҹ’бһҡбҹ„бһҳ **Permissions** бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһңбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһӮбҹ’бһҡбһ”бҹӢбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷ Obsidian бһ‘бһ‘бһҪбһӣбһҳбһ¶бһҸбһ·бһҖбһ¶ Web Clipper бһҮбһ¶бһ“бһ·бһ…бҹ’бһ…бҹ–

1. бһ…бһјбһӣбһ‘бҹ… iOS **[[бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ]]** вҶ’ **Apps** вҶ’ **Obsidian**бҹ”
2. бһҖбҹҶбһҺбһҸбҹӢ **Paste from other apps** бһ‘бҹ… **Allow**бҹ”
