## បង្កើត ឬកែសម្រួលពុម្ព

ដើម្បី**បង្កើត**ពុម្ព សូមចូលទៅការកំណត់ Web Clipper ហើយចុចប៊ូតុង **New template** នៅរបារចំហៀង។ អ្នកក៏អាច**ស្ទួន**ពុម្បមួយក្នុងម៉ឺនុយសកម្មភាព **More** នៅជ្រុងខាងស្តាំផ្នែកខាងលើ។

ដើម្បី**កែសម្រួល**ពុម្ព សូមជ្រើសរើសពុម្បមួយពីរបារចំហៀង។ ការផ្លាស់ប្តូររបស់អ្នកនឹងត្រូវបានរក្សាទុកដោយស្វ័យប្រវត្តិ។

ពុម្បប្រើ [[អថេរ]] និង [[តម្រង]] ដែលអនុញ្ញាតឱ្យអ្នកកែសម្រួលរបៀបរក្សាទុកមាតិកា។

## នាំចូល និងនាំចេញពុម្ប Web Clipper

ដើម្បីនាំចូលពុម្ប៖

1. បើកផ្នែកបន្ថែម ហើយចុចរូបតំណាង **[[ការកំណត់]]** រាងធ្មេញ។
2. ចូលទៅពុម្បណាមួយក្នុងបញ្ជី។
3. ចុច **Import** នៅខាងស្តាំផ្នែកខាងលើ ឬអូស និងទម្លាក់ឯកសារពុម្ប `.json` របស់អ្នកកន្លែងណាមួយក្នុងតំបន់ពុម្ប។

ដើម្បីនាំចេញពុម្ប សូមចុច **Export** នៅខាងស្តាំផ្នែកខាងលើ។ វានឹងទាញយកឯកសារពុម្ប `.json`។ អ្នកក៏អាចចម្លងទិន្នន័យពុម្បទៅក្ដារតម្បៀតខ្ទាស់តាមរយៈម៉ឺនុយ **More**។

## ការកំណត់ពុម្ប

### អាកប្បកិរិយា

កំណត់របៀបដែលមាតិកាពី Web Clipper នឹងត្រូវបានបន្ថែមទៅ Obsidian៖

- **បង្កើតកំណត់ត្រាថ្មី**
- **បន្ថែមទៅកំណត់ត្រាដែលមានស្រាប់** នៅខាងលើ ឬខាងក្រោម
- **បន្ថែមទៅកំណត់ចំណាំប្រចាំថ្ងៃ** នៅខាងលើ ឬខាងក្រោម (តម្រូវឱ្យកម្មវិធីជំនួយ [[កំណត់ចំណាំប្រចាំថ្ងៃ]] សកម្ម)

### ដំណើរការពុម្បដោយស្វ័យប្រវត្តិ

កម្មវិធីដំណើរការពុម្បអនុញ្ញាតឱ្យអ្នកជ្រើសរើសពុម្បដោយស្វ័យប្រវត្តិដោយផ្អែកលើ URL ទំព័របច្ចុប្បន្ន ឬទិន្នន័យ [schema.org](https://schema.org/)។ អ្នកអាចកំណត់ច្បាប់ច្រើនសម្រាប់ពុម្បនីមួយៗ ដែលបំបែកដោយបន្ទាត់ថ្មី។

ការផ្គូផ្គងដំបូងក្នុងបញ្ជីពុម្បរបស់អ្នកកំណត់ថាពុម្បណាដែលត្រូវប្រើ។ អ្នកអាចអូសពុម្បឡើងលើ និងចុះក្រោមក្នុងការកំណត់ Web Clipper ដើម្បីផ្លាស់ប្តូរលំដាប់ដែលពុម្បត្រូវផ្គូផ្គង។

#### ការផ្គូផ្គង URL សាមញ្ញ

ការផ្គូផ្គងសាមញ្ញដំណើរការពុម្បប្រសិនបើ URL ទំព័របច្ចុប្បន្ន *ចាប់ផ្តើមដោយ* លំនាំដែលបានផ្តល់។ ឧទាហរណ៍៖

- `https://obsidian.md` នឹងផ្គូផ្គង URL ណាមួយដែលចាប់ផ្តើមដោយអត្ថបទនេះ។

#### ការផ្គូផ្គងដោយកន្សោមធម្មតា

អ្នកអាចដំណើរការពុម្បដោយផ្អែកលើលំនាំ URL ស្មុគស្មាញជាងមុនដោយប្រើកន្សោមធម្មតា។ រុំលំនាំ regex របស់អ្នកក្នុងសញ្ញាស្លាស (`/`)។ ចងចាំថាត្រូវគេចពីតួអក្សរពិសេសក្នុងលំនាំ regex (ដូចជា `.` និង `/`) ដោយសញ្ញាច្រកថយ (`\`)។ ឧទាហរណ៍៖

- `/^https:\/\/www\.imdb\.com\/title\/tt\d+\/reference\/?$/` នឹងផ្គូផ្គងទំព័រឯកសារយោង IMDB ណាមួយ។

#### ការផ្គូផ្គង Schema.org

អ្នកអាចដំណើរការពុម្បដោយផ្អែកលើទិន្នន័យ [schema.org](https://schema.org/) ដែលមាននៅលើទំព័រ។ ប្រើបុព្វបទ `schema:` តាមដោយកូនសោ schema ដែលអ្នកចង់ផ្គូផ្គង។ អ្នកអាចបញ្ជាក់តម្លៃរំពឹងទុកបន្ថែម។ ឧទាហរណ៍៖

- `schema:@Recipe` នឹងផ្គូផ្គងទំព័រដែលប្រភេទ schema គឺ "Recipe"។
- `schema:@Recipe.name` នឹងផ្គូផ្គងទំព័រដែល `@Recipe.name` មានវត្តមាន។
- `schema:@Recipe.name=Cookie` នឹងផ្គូផ្គងទំព័រដែល `@Recipe.name` គឺ "Cookie"។

តម្លៃ Schema.org ក៏អាចប្រើដើម្បី [[អថេរ#Schema.org variables|បំពេញទិន្នន័យជាមុនក្នុងពុម្ប]]។
