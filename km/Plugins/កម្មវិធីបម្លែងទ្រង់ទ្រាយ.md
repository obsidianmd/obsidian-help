កម្មវិធីបម្លែងទ្រង់ទ្រាយ គឺជា[[កម្មវិធីជំនួយចម្បង|កម្មវិធីជំនួយចម្បង]]ដែលអនុញ្ញាតឱ្យអ្នកបម្លែង Markdown ពីកម្មវិធីផ្សេងៗទៅទម្រង់ Obsidian។ វាក៏អនុញ្ញាតឱ្យអ្នកបម្លែង[[លក្ខណៈសម្បត្តិ]]មួយចំនួនទៅទម្រង់ថ្មីដែលត្រូវការផងដែរ។

> [!warning] ការព្រមាន
> កម្មវិធីបម្លែងទ្រង់ទ្រាយបម្លែង vault ទាំងមូលរបស់អ្នកដោយផ្អែកលើការកំណត់របស់អ្នក។ [[បម្រុងទុកឯកសារ Obsidian របស់អ្នក]]មុនពេលអ្នកធ្វើការបម្លែង។

ដើម្បីបម្លែងកំណត់ត្រាទាំងអស់ក្នុង vault របស់អ្នក៖

1. នៅក្នុង[[ក្ដារលាយពាក្យបញ្ជា]] ជ្រើសរើស **Open format converter**។ វាក៏អាចរកឃើញនៅក្នុង[[រីបិន]]ជាមួយរូបតំណាង **Open format converter** ![[lucide-binary.svg#icon]] ផងដែរ។
2. បើកទម្រង់ដែលអ្នកចង់បម្លែង។
3. ចុច **Start conversion**។

សម្រាប់ព័ត៌មានបន្ថែម សូមយោងទៅ[[វាក្យសម្ពន្ធទម្រង់មូលដ្ឋាន]]។

## ទម្រង់ដែលគាំទ្រ

### Roam Research

កម្មវិធីបម្លែងទ្រង់ទ្រាយអាចបម្លែងវាក្យសម្ពន្ធ Roam Research ខាងក្រោម៖

- **ស្លាក**: បម្លែង `#tag` និង `#[[tag]]` ទៅ `[[tag]]`
- **ការបន្លិច**: បម្លែង `^^highlight^^` ទៅ `==highlight==`
- **ធាតុ TODO**: បម្លែង `{{[[TODO]]}}` ទៅ `[ ]`

### Bear

កម្មវិធីបម្លែងទ្រង់ទ្រាយអាចបម្លែងវាក្យសម្ពន្ធ Bear ខាងក្រោម៖

- **ការបន្លិច**: បម្លែង `::highlight::` ទៅ `==highlight==`

### Zettelkasten

កម្មវិធីបម្លែងទ្រង់ទ្រាយអាចបម្លែងវាក្យសម្ពន្ធ Zettelkasten ខាងក្រោម៖

- **តំណភ្ជាប់ពេញលេញ**: បម្លែង `[[UID]]` ទៅ `[[UID File Name]]`
- **តំណភ្ជាប់ស្អាត**: បម្លែង `[[UID]]` ទៅ `[[UID File Name|File Name]]`

### [[លក្ខណៈសម្បត្តិ]]

ចាប់ពី Obsidian `1.9.3` កម្មវិធីបម្លែងទ្រង់ទ្រាយអាចបម្លែងទម្រង់[[លក្ខណៈសម្បត្តិ#លក្ខណៈសម្បត្តិដែលលែងប្រើ|លក្ខណៈសម្បត្តិដែលលែងប្រើ]]ទៅទម្រង់បច្ចុប្បន្ន៖

**ឈ្មោះក្លែងក្លាយ**

```yaml
# មុន

alias: My Note Title

# ក្រោយ

aliases:
  - My Note Title
```

**ស្លាក**

```yaml
# មុន

tag: project, important

# ក្រោយ

tags:
  - project
  - important
```

**ថ្នាក់ CSS**

```yaml
# មុន

cssclass: custom-style

# ក្រោយ

cssclasses:
  - custom-style
```
