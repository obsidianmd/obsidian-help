[[អ្នកនិពន្ធកំណត់ត្រា]] គឺជា[[កម្មវិធីជំនួយចម្បង|កម្មវិធីជំនួយចម្បង]]ដែលអនុញ្ញាតឱ្យអ្នកបញ្ចូលកំណត់ត្រាពីរចូលគ្នា ឬដកផ្នែកមួយនៃកំណត់ត្រាទៅជាកំណត់ត្រាថ្មី។

## បញ្ចូលកំណត់ត្រាចូលគ្នា

ការបញ្ចូលកំណត់ត្រាចូលគ្នាគឺបន្ថែមកំណត់ត្រាមួយទៅកំណត់ត្រាមួយទៀត ហើយលុបកំណត់ត្រាដំបូង។ អ្នកនិពន្ធកំណត់ត្រាអាប់ដេតតំណភ្ជាប់ទាំងអស់ដើម្បីយោងទៅកំណត់ត្រាដែលបានបញ្ចូលគ្នា។

នៅពេលអ្នកជ្រើសរើសកំណត់ត្រាដើម្បីបញ្ចូលចូល អ្នកអាចជ្រើសរើសវិធីខាងក្រោម:

- `Enter`: បន្ថែមកំណត់ត្រាប្រភពនៅ _ចុង_ នៃកំណត់ត្រាគោលដៅ។
- `Shift+Enter`: បន្ថែមកំណត់ត្រាប្រភពនៅ _ដើម_ នៃកំណត់ត្រាគោលដៅ។
- `Ctrl+Enter` (ឬ `Cmd+Enter` លើ macOS): បង្កើតកំណត់ត្រាថ្មីមួយដែលមានមាតិកានៃកំណត់ត្រាប្រភព។

ដើម្បីបញ្ចូលកំណត់ត្រាសកម្មជាមួយកំណត់ត្រាផ្សេងទៀតនៅក្នុង vault របស់អ្នក:

**កម្មវិធីរុករកឯកសារ**

1. នៅក្នុងកម្មវិធីរុករកឯកសារ ចុចខាងស្តាំលើកំណត់ត្រាដែលអ្នកចង់បញ្ចូលចូលគ្នា។
2. ចុច **Merge entire file with...**។
3. ជ្រើសរើសកំណត់ត្រាដែលអ្នកចង់បញ្ចូលចូល។
4. ចុច **Merge** ដើម្បីបញ្ជាក់។

**ក្ដារលាយពាក្យបញ្ជា**

1. បើក[[ក្ដារលាយពាក្យបញ្ជា]]។
2. ជ្រើសរើស **Note composer: Merge current file with another file...**។
3. ជ្រើសរើសកំណត់ត្រាដែលអ្នកចង់បញ្ចូលចូល។
4. ចុច **Merge** ដើម្បីបញ្ជាក់។

> [!tip] គន្លឹះ
> តាមលំនាំដើម អ្នកនិពន្ធកំណត់ត្រាសួរអ្នកឱ្យបញ្ជាក់នៅពេលបញ្ចូលកំណត់ត្រាចូលគ្នា។ ប្រសិនបើអ្នកបិទការបញ្ជាក់ ហើយអ្នកបញ្ចូលកំណត់ត្រាចូលគ្នាដោយកំហុស អ្នកនៅតែអាចសង្គ្រោះវាបានដោយប្រើកម្មវិធីជំនួយ [[ការស្តារឯកសារឡើងវិញ]]។

## ដកកំណត់ត្រាចេញ

នៅពេលអ្នកជ្រើសរើសកំណត់ត្រាដើម្បីដកអត្ថបទចេញទៅ អ្នកអាចជ្រើសរើសវិធីខាងក្រោម:

- `Enter`: បន្ថែមអត្ថបទដែលបានជ្រើសរើសនៅ _ចុង_ នៃកំណត់ត្រាគោលដៅ។
- `Shift+Enter`: បន្ថែមអត្ថបទដែលបានជ្រើសរើសនៅ _ដើម_ នៃកំណត់ត្រាគោលដៅ។
- `Ctrl+Enter` (ឬ `Cmd+Enter` លើ macOS): បង្កើតកំណត់ត្រាថ្មីមួយដែលមានអត្ថបទដែលបានជ្រើសរើស។

ដើម្បីដកអត្ថបទទៅកំណត់ត្រាថ្មី:

**កម្មវិធីកែសម្រួល**

1. ខណៈពេលនៅក្នុង **ទិដ្ឋភាពកែសម្រួល** ជ្រើសរើសអត្ថបទដែលអ្នកចង់ដកចេញ។
2. ចុចខាងស្តាំលើអត្ថបទដែលបានជ្រើសរើស។
3. ចុច **Extract current selection...**។
4. ជ្រើសរើសកំណត់ត្រាដែលអ្នកចង់ដកចេញទៅ។

**ក្ដារលាយពាក្យបញ្ជា**

1. ខណៈពេលនៅក្នុង **ទិដ្ឋភាពកែសម្រួល** ជ្រើសរើសអត្ថបទដែលអ្នកចង់ដកចេញ។
2. បើក[[ក្ដារលាយពាក្យបញ្ជា]]។
3. ជ្រើសរើស **Note composer: Extract current selection...**។
4. ជ្រើសរើសកំណត់ត្រាដែលអ្នកចង់ដកចេញទៅ។

> [!tip] គន្លឹះ
> តាមលំនាំដើម អ្នកនិពន្ធកំណត់ត្រាជំនួសអត្ថបទដែលបានដកចេញដោយតំណភ្ជាប់ទៅកំណត់ត្រាគោលដៅ។ នៅក្នុងការកំណត់ អ្នកក៏អាចផ្លាស់ប្តូរដើម្បី[[បង្កប់ឯកសារ|បង្កប់]]កំណត់ត្រាគោលដៅជំនួស ឬទុកទទេ។

## ឯកសារពុម្ព

ដោយកំណត់រចនាសម្ព័ន្ធពុម្ព អ្នកអាចប្ដូរមាតិកាតាមតម្រូវការមុនពេលបន្ថែមវាទៅកំណត់ត្រាថ្មី។ ដើម្បីប្រើពុម្ព សូមបញ្ចូល **Template file location** នៅក្នុងការកំណត់កម្មវិធីជំនួយ។

ពុម្បអាចមានអថេរខាងក្រោម:

| អថេរ              | ការពិពណ៌នា                                                                                                                                              |
|-------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| `{{content}}`     | មាតិកាដែលត្រូវបញ្ចូលចូលគ្នា ឬអត្ថបទដែលបានដកចេញ។ ប្រសិនបើអ្នកមិនរួមបញ្ចូលអថេរនេះ អ្នកនិពន្ធកំណត់ត្រាបន្ថែមមាតិកានៅខាងក្រោមនៃពុម្ព។ |
| `{{fromTitle}}`   | ឈ្មោះកំណត់ត្រាប្រភព។                                                                                                                                 |
| `{{newTitle}}`    | ឈ្មោះកំណត់ត្រាគោលដៅ។ ឧទាហរណ៍ ដើម្បីបន្ថែមឈ្មោះឯកសារជាចំណងជើងនៅផ្នែកខាងលើនៃឯកសារ។                                                     |
| `{{date:FORMAT}}` | កាលបរិច្ឆេទបង្កើតកំណត់ត្រាថ្មី។ ឧទាហរណ៍ `{{date:YYYY-MM-DD}}`។                                                                                       |
