бһ‘бҹҶбһ–бҹҗбһҡбһ“бҹҒбҹҮбһҡбһ¶бһҷбһ”бһүбҹ’бһҮбһёбһ”бһүбҹ’бһ бһ¶бһ‘бһјбһ‘бҹ…бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҮбһҪбһ”бһ”бҹ’бһҡбһ‘бҹҮбһҮбһ¶бһҳбһҪбһҷ [[бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһўбҹҶбһ–бһё Obsidian Publish|Obsidian Publish]] бһ“бһ·бһ„бһңбһ·бһ’бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ–бһҪбһҖбһңбһ¶бҹ”

бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ [[бһҜбһҖбһҹбһ¶бһҡбһҳбҹҒбһҢбҹҖ]] бһ“бһ·бһ„ [[бһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһ”бһҹбҹӢ Publish|бһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһ”бһҹбҹӢ Publish]] бһҮбһ¶бһҳбһ»бһ“бһҹбһ·бһ“бҹ”

## бһ‘бһјбһ‘бҹ…

### бһҖбһ¶бһҡбһ•бҹ’бһҹбһ¶бһҷбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶

**бһҒбҹ’бһүбһ»бҹҶбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбҹҶбһ бһ»бһҹ hash бһ“бҹ…бһ–бҹҒбһӣбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһ•бҹ’бһҹбһ¶бһҷбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бҹ”**

бһҸбһҫбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҡбһҫ [[бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбҹҶбһ“бһҪбһҷбһҹбһ бһӮбһҳбһ“бҹҚ|бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбҹҶбһ“бһҪбһҷбһҹбһ бһӮбһҳбһ“бҹҚ]] бһҠбҹӮбһӣбһҖбҹӮбһ”бҹ’бһҡбҹӮбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҖбҹӮбһ”бҹ’бһҡбҹӮбһҡбһ”бһҹбҹӢбһҜбһҖбһҹбһ¶бһҡбһ“бҹ…бһ–бҹҒбһӣбһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһ¬? бһ”бһҫбһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҳбҹӮбһ“ бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбҹҶбһ“бһҪбһҷбһ“бҹҒбҹҮбһўбһ¶бһ…бһҳбһ¶бһ“бһҮбһҳбҹ’бһӣбҹ„бҹҮбһҮбһ¶бһҳбһҪбһҷ Publishбҹ” бһҹбһјбһҳбһ•бҹ’бһүбһҫбһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһҖбҹҶбһ бһ»бһҹбһ‘бҹ…бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбҹҶбһ“бһҪбһҷбһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бҹ”

**бһҒбҹ’бһүбһ»бҹҶбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбҹҶбһ бһ»бһҹбһ”бһҺбҹ’бһҠбһ¶бһүбһ…бһҳбҹ’бһӣбҹӮбһҖ бһ бһҫбһҷбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡ Publish бһҡбһ”бһҹбҹӢбһҒбҹ’бһүбһ»бҹҶбһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶбһ’бҹҶбһҒбҹ’бһӣбһ¶бҹҶбһ„бҹ”**

бһ‘бҹҶбһ“бһ„бһҮбһ¶бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҹбһјбһҳ [[бһҮбҹҶбһ“бһҪбһҷ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡ#бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһ•бҹ’бһ“бҹӮбһҖбһҮбҹҶбһ“бһҪбһҷ Obsidian|бһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ•бҹ’бһ“бҹӮбһҖбһӮбһ¶бҹҶбһ‘бҹ’бһҡ Obsidian]] бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбҹҶбһ“бһҪбһҷбҹ”

## CSS бһ“бһ·бһ„бһҡбһјбһ”бһҡбһ¶бһ„

**CSS бһҡбһ”бһҹбҹӢбһҒбҹ’бһүбһ»бҹҶбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ [[бһҗбһҸбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’]] бһҳбһ·бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҫ Publishбҹ” бһ бҹҒбһҸбһ»бһўбҹ’бһңбһё?**

Publish бһҳбһ·бһ“бһўбһ¶бһ“бһ–бһёбһҗбһҸбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ‘бҹҒбҹ” бһҮбҹҶбһ“бһҪбһҹбһңбһ·бһү бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҖбһҫбһҸбһҜбһҖбһҹбһ¶бһҡ `publish.css` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸбһ“бҹғ vault бһ•бҹ’бһҹбһ¶бһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһёбһҡбһҝбһ„бһ“бҹҒбҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ [[бһ”бҹ’бһҠбһјбһҡбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбһ¶бһҳбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡ]]бҹ”

**CSS бһҡбһ”бһҹбҹӢбһҒбҹ’бһүбһ»бҹҶбһҳбһҫбһӣбһ‘бҹ…бһҳбһ·бһ“бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ“бҹ…бһӣбһҫ Publish бһҠбһјбһ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бҹҒбҹ” бһ бҹҒбһҸбһ»бһўбҹ’бһңбһё?**

CSS бһҡбһ”бһҹбҹӢ Obsidian Publish бһҳбһ·бһ“бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ”бҹҒбҹҮбһ”бһ·бһ‘бһ“бһ№бһ„бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бҹҒбҹ” бһҷбһҫбһ„бһҺбҹӮбһ“бһ¶бҹҶбһұбҹ’бһҷ [бһўбһ—бһ·бһңбһҢбҹ’бһҚ CSS бһ“бһ·бһ„бһҡбһјбһ”бһҡбһ¶бһ„бһҺбһ¶бһҳбһҪбһҷ](https://docs.obsidian.md/Themes/Obsidian+Publish+themes/About+Obsidian+Publish+themes) бһҮбһ¶бһ–бһ·бһҹбҹҒorg бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Publish бһ–бһёбһҠбҹҶбһ”бһјбһ„бҹ”

бһҮбһ¶ _бһ‘бһјбһ‘бҹ…_ бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ [[бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ– бһ“бһ·бһ„бһҡбһ”бҹҖбһ”бһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣ#бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһўбһ¶бһ“|бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһўбһ¶бһ“]] бһҳбһ¶бһ“бһӣбһ‘бҹ’бһ’бһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҫ Publishбҹ”
