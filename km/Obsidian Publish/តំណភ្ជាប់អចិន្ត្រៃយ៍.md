អ្នកអាចប្តូរឈ្មោះ URL ទៅកំណត់ត្រារបស់អ្នកដោយប្រើ _តំណភ្ជាប់អចិន្ត្រៃយ៍_។

ឧទាហរណ៍ អ្នកអាចប្តូរពី៖

```
https://publish.obsidian.md/username/Company/About+us
```

ទៅជា៖

```
https://publish.obsidian.md/username/about
```

ដើម្បីបង្កើតតំណភ្ជាប់អចិន្ត្រៃយ៍សម្រាប់កំណត់ត្រាមួយ សូមបន្ថែមលក្ខណៈសម្បត្តិ `permalink` ទៅក្នុង [[លក្ខណៈសម្បត្តិ]] របស់អ្នក។

```yaml
---
permalink: about
---
```

ប្រសិនបើនរណាម្នាក់ចូលមើលកំណត់ត្រាដោយប្រើ URL ដើម ពួកគេនឹងត្រូវបញ្ជូនបន្តដោយស្វ័យប្រវត្តិទៅកាន់តំណភ្ជាប់អចិន្ត្រៃយ៍។

## បញ្ជូនបន្តកំណត់ត្រាចាស់

ការប្តូរឈ្មោះ និងលុបកំណត់ត្រាគឺជាផ្នែកធម្មជាតិនៃការថែរក្សា vault ដែលមានជីវិត។ ខណៈពេលដែល Obsidian ធ្វើបច្ចុប្បន្នភាពតំណភ្ជាប់ដោយស្វ័យប្រវត្តិនៅពេលអ្នកផ្លាស់ទីកំណត់ត្រាក្នុង vault មូលដ្ឋានរបស់អ្នក គេហទំព័រផ្សេងទៀតអាចនៅតែភ្ជាប់ទៅកំណត់ត្រាចាស់របស់អ្នកនៅលើគេហទំព័រ [[ការណែនាំអំពី Obsidian Publish|Obsidian Publish]] ដែលបានផ្សាយរបស់អ្នក។ អ្នកអាចបញ្ជូនបន្តអ្នកអានពីកំណត់ត្រាមួយទៅកំណត់ត្រាមួយទៀត។

ស្រមៃថាអ្នកចង់ផ្លាស់ទីកំណត់ត្រាពីថតមួយទៅថតមួយទៀត៖
