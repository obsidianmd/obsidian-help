ទំព័រនេះពន្យល់អំពីរបៀបដែលអ្នកអាចប្ដូរតាមតម្រូវការនូវរូបរាង និងអារម្មណ៍គេហទំព័រ [[ការណែនាំអំពី Obsidian Publish|Obsidian Publish]] របស់អ្នក។

## ទ្រព្យសម្បត្តិថេរ

អ្នកអាចប្ដូរគេហទំព័ររបស់អ្នកតាមតម្រូវការដោយ[[ផ្សាយមាតិការបស់អ្នក#ផ្សាយកំណត់ត្រា|ផ្សាយ]]ឯកសារខាងក្រោមទៅគេហទំព័ររបស់អ្នក៖

- `publish.css` ដើម្បីបន្ថែម CSS ផ្ទាល់ខ្លួន
- `publish.js` ដើម្បីបន្ថែម JavaScript ផ្ទាល់ខ្លួន
- `favicon-32x32.png` ដើម្បីកំណត់ favicon

**កំណត់ចំណាំ៖**

- [អថេរ CSS សម្រាប់ Publish](https://docs.obsidian.md/Reference/CSS+variables/Publish/Publish) អាចរកបាននៅលើគេហទំព័រឯកសាររបស់យើង។
- ដោយសារ Obsidian មិនគាំទ្រឯកសារ CSS ឬ JavaScript អ្នកត្រូវប្រើកម្មវិធីផ្សេងទៀតដើម្បីបង្កើត និងកែសម្រួលពួកវា។
- ទាំង `publish.css` និង `publish.js` ត្រូវតែស្ថិតក្នុងថតឫស (`/`) នៃ vault របស់អ្នក។
- តាមលំនាំដើម `publish.css` និង `publish.js` មិនបង្ហាញក្នុងកម្មវិធីរុករកឯកសារទេ ប៉ុន្តែអ្នកនៅតែអាចផ្សាយពួកវាពីប្រអប់ **Publish changes** បាន។
- ដើម្បីប្រើ JavaScript ផ្ទាល់ខ្លួនជាមួយ `publish.js` អ្នកត្រូវ[[ដូមែនផ្ទាល់ខ្លួន]]។

សម្រាប់ favicons Obsidian Publish គាំទ្រការដាក់ឈ្មោះដូចខាងក្រោម ដែល `32` តំណាងឱ្យទំហំរូបតំណាងគិតជាភីកសែល៖

- `favicon-32.png`
- `favicon-32x32.png`
- `favicon.ico`

យើងណែនាំឱ្យអ្នកផ្ដល់ទំហំមួយ ឬច្រើនខាងក្រោម៖

- `favicon-32x32.png`
- `favicon-128x128.png`
- `favicon-152x152.png`
- `favicon-167x167.png`
- `favicon-180x180.png`
- `favicon-192x192.png`
- `favicon-196x196.png`

អ្នកមានភាពបត់បែនក្នុងការដាក់ favicons កន្លែងណាក៏បានក្នុង vault ដរាបណាពួកវាត្រូវបានផ្សាយទៅគេហទំព័ររបស់អ្នក។

## ប្រើរូបរាងសហគមន៍

ដើម្បីប្រើរូបរាងសហគមន៍មួយសម្រាប់គេហទំព័ររបស់អ្នក៖

1. បើក vault របស់អ្នកក្នុងកម្មវិធីរុករកឯកសារលំនាំដើមសម្រាប់ OS របស់អ្នក។
2. ចូលទៅថតការកំណត់ vault (លំនាំដើម៖ `.obsidian`)។
3. បើកថត `themes`។
4. ចម្លងឯកសារ CSS សម្រាប់រូបរាងដែលអ្នកចង់ប្រើសម្រាប់គេហទំព័ររបស់អ្នក។
5. ដាក់ឯកសារក្នុងថតឫសនៃ vault របស់អ្នក។
6. ប្ដូរឈ្មោះឯកសារ CSS ទៅ `publish.css`។
7. [[ផ្សាយមាតិការបស់អ្នក#ផ្សាយកំណត់ត្រា|ផ្សាយ]] `publish.css`។

**កំណត់ចំណាំ៖**

- បើរចនាប័ទ្មមិនអាប់ដេតក្នុងរយៈពេលប៉ុន្មាននាទី សូមសាកល្បងសម្អាត cache កម្មវិធីរុករកតាមអ៊ីនធឺណិតរបស់អ្នក។
- អ្នកអាចប្ដូររវាងរបៀបភ្លឺ និងងងឹតក្នុង[[គ្រប់គ្រងគេហទំព័រ#ជម្រើសមើលគេហទំព័រ|ជម្រើសគេហទំព័រ]]។
- រូបរាងសហគមន៍ជាច្រើនប្រើ **Style Settings** សម្រាប់រចនាប័ទ្មផ្ទាល់ខ្លួន ប៉ុន្តែការកំណត់ទាំងនេះមិនដំណើរការលើ Obsidian Publish ទេ។

> [!tip] ការអភិវឌ្ឍរូបរាង
> រកមិនឃើញរូបរាងសម្រាប់អ្នកទេ? រៀនពីរបៀប [បង្កើតរូបរាង Publish](https://docs.obsidian.md/Themes/Obsidian+Publish+themes/Build+a+Publish+theme) ដោយខ្លួនឯង។

## បើកដំណើរការមុខងារ UI

អ្នកអាចបើក/បិទមុខងារ UI ជាច្រើនសម្រាប់គេហទំព័ររបស់អ្នក ដូចជាទិដ្ឋភាពក្រាហ្វ ឬតារាងមាតិកា។

រុករកមុខងារ UI ដែលមានក្រោមផ្នែក **បទពិសោធន៍អាន** និង **សមាសធាតុ** ក្នុង[[គ្រប់គ្រងគេហទំព័រ#ជម្រើសមើលគេហទំព័រ|ជម្រើសគេហទំព័រ]]

### ប្ដូរការរុករកតាមតម្រូវការ

នៅក្នុង Obsidian Publish អ្នកមានសមត្ថភាពប្ដូរតាមតម្រូវការនូវលំដាប់ការរុករក និងការបង្ហាញឯកសារ និងថតក្នុង [[កម្មវិធីរុករកឯកសារ]] របស់ Publish។ ធាតុរុករកត្រូវបានរៀបតាមលំដាប់ដែលបានផ្សាយតាមលំនាំដើម។ កំណត់ត្រាដែលមិនបានផ្សាយនឹងមិនបង្ហាញក្នុងផ្ទាំងនេះទេ។

#### ការចូលប្រើជម្រើសប្ដូរការរុករកតាមតម្រូវការ

1. ក្នុង[[រីបិន]] ជ្រើសរើស **Publish changes** ![[lucide-send.svg#icon]] ឬបើក[[ក្ដារលាយពាក្យបញ្ជា]] ហើយវាយ **Publish: Publish changes...**
2. ក្នុងប្រអប់ **Publish changes** ជ្រើសរើស **Change site options** ![[lucide-cog.svg#icon]]។
3. ក្រោម **Components settings** នៅជិត **Customize navigation** ជ្រើសរើសប៊ូតុង **manage**។

បង្អួចលេចចេញថ្មីមួយដែលមានចំណងជើង **Navigation** នឹងបង្ហាញនៅលើបង្អួច **Change site options** របស់អ្នក។

#### កែតម្រូវធាតុរុករក

នៅក្នុងផ្នែកដែលមានស្លាក **Navigation preview** អ្នកអាចកែតម្រូវលំដាប់បង្ហាញមាតិកាដែលបានផ្សាយរបស់អ្នក។

1. ជ្រើសរើសថត ឬកំណត់ត្រាដែលអ្នកចង់កែតម្រូវ។
2. អូសកំណត់ត្រា ឬថតឡើង ឬចុះរហូតដល់វាស្ថិតក្នុងទីតាំងដែលអ្នកចង់បាន។
3. នៅខាងស្ដាំខាងក្រោមនៃបង្អួច **Navigation** ជ្រើសរើស **Done**។

Publish នឹងផ្ញើការផ្លាស់ប្ដូរការរុករករបស់អ្នកទៅគេហទំព័ររបស់អ្នក។

#### លាក់ និងបង្ហាញធាតុរុករក

ប្រសិនបើមានកំណត់ត្រា ឬថតដែលអ្នកបានផ្សាយ ប៉ុន្តែអ្នកមិនចង់ឱ្យវាបង្ហាញក្នុងការរុករករបស់អ្នកទេ អ្នកអាចជ្រើសរើសលាក់ធាតុទាំងនោះជំនួសវិញ។

1. ជ្រើសរើសថត ឬកំណត់ត្រាដែលអ្នកចង់កែតម្រូវ។
2. ចុចស្ដាំ ហើយជ្រើសរើស **Hide in navigation**។ ធាតុនោះគួរតែបាត់ពី **Navigation preview** ឥឡូវនេះ។
3. នៅខាងស្ដាំខាងក្រោមនៃបង្អួច **Navigation** ជ្រើសរើស **Done**។

Publish នឹងផ្ញើការផ្លាស់ប្ដូរការរុករករបស់អ្នកទៅគេហទំព័ររបស់អ្នក។

> [!tip] អ្នកអាច **បង្ហាញឯកសារលាក់** ដោយជ្រើសរើសប្រអប់ធីកនៅខាងស្ដាំនៃចំណងជើង **Navigation Preview**

## សំណួរដែលគេសួរញឹកញាប់

**តើខ្ញុំអាចផ្លាស់ទីឯកសារពីថតមួយទៅថតមួយទៀតក្នុងការរុករកបានទេ?**

ទេ។ រចនាសម្ព័ន្ធរុករកឯកសារសម្រាប់កំណត់ត្រាក្នុងថតត្រូវតែរក្សាទុក។ អ្នកអាចកែតម្រូវលំដាប់កំណត់ត្រាក្នុងថត (រួមទាំងថតឫស vault) និងលំដាប់ថតក្នុងថតផ្សេងទៀត។

**តើខ្ញុំអាចកែសម្រួលលំដាប់កំណត់ត្រា និងថតច្រើនមុនពេលជ្រើសរើស Done បានទេ?**

បាន។

**តើខ្ញុំត្រឡប់ការផ្លាស់ប្ដូរទាំងនេះយ៉ាងដូចម្ដេច?**

- **លំដាប់បង្ហាញ**៖ ជ្រើសរើសរូបតំណាង **Restore Default** (ព្រួញបង្វិលត្រឡប់) នៅជិត **Navigation item display order**។ នេះនឹងស្ដារធាតុរុករករបស់អ្នកទៅលំដាប់អក្សរក្រម។
- **ស្ថានភាពលាក់**៖ ជ្រើសរើសរូបតំណាង **Restore Default** (ព្រួញបង្វិលត្រឡប់) នៅជិត **Hide pages or folders from navigation**។ នេះនឹងស្ដារធាតុរុករកដែលលាក់របស់អ្នកទៅស្ថានភាពដែលមើលឃើញ។
