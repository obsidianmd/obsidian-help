បើអ្នកជាអ្នកអភិវឌ្ឍន៍ អ្នកអាចរួមចំណែកដល់ Obsidian តាមវិធីផ្សេងៗ។

### កម្មវិធីជំនួយ និងរូបរាង

អ្នកអាចអភិវឌ្ឍ [[កម្មវិធីជំនួយសហគមន៍]] និង [[រូបរាង]] ដោយប្រើ Typescript និង CSS។ ស្វែងយល់បន្ថែមដោយចូលទៅកាន់ [ឯកសារអ្នកអភិវឌ្ឍន៍ Obsidian](https://docs.obsidian.md)។

### គម្រោងប្រភពបើកចំហ

Obsidian អភិវឌ្ឍ និងថែរក្សាគម្រោងប្រភពបើកចំហជាច្រើន។ អ្នកអាចដាក់ pull requests ជាមួយនឹងការជួសជុលកំហុស និងការកែលម្អទៅកាន់ repositories នៅ GitHub ខាងក្រោម៖

- [JSON Canvas](https://github.com/obsidianmd/jsoncanvas) (MIT) ទម្រង់ឯកសារបើកចំហសម្រាប់ទិន្នន័យ canvas គ្មានដែនកំណត់។
- [Obsidian Importer](https://github.com/obsidianmd/obsidian-importer) (MIT) កម្មវិធីជំនួយដែលបម្លែងទិន្នន័យពីទម្រង់ផ្សេងៗជាច្រើនទៅជាឯកសារ Markdown ជាអក្សរធម្មតា។
- [Obsidian Maps](https://github.com/obsidianmd/obsidian-maps) (MIT) កម្មវិធីជំនួយដែលបង្ហាញផែនទីនៅក្នុង Obsidian Bases។
- [Obsidian Web Clipper](https://github.com/obsidianmd/obsidian-clipper) (MIT) ផ្នែកបន្ថែមកម្មវិធីអ៊ីនធឺណិតសម្រាប់ពណ៌ចំណាំ និងចាប់យកទំព័របណ្តាញទៅជាឯកសារ Markdown ជាអក្សរធម្មតា។
- [Defuddle](https://github.com/kepano/defuddle) (MIT) កម្មវិធីវិភាគ HTML ទៅ Markdown ដែលប្រើដោយ Obsidian Web Clipper។

### ឯកសារ

ដើម្បីជួយកែលម្អ [ឯកសារអ្នកអភិវឌ្ឍន៍](https://docs.obsidian.md/Home) របស់យើង សូមដាក់ pull request ទៅ repository [obsidian-developer-docs](https://github.com/obsidianmd/obsidian-developer-docs)។

ដើម្បីជួយកែលម្អឯកសារនេះ សូមដាក់ pull request ទៅ repository [obsidian-help](https://github.com/obsidianmd/obsidian-help)។
