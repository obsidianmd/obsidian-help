## ឯកសារ

កំណែដំបូងនៃ vault ជំនួយត្រូវបានសរសេរដោយ Silver។ ផ្នែកក្រោយត្រូវបានសរសេរដោយ [@mediapathic](http://mediapathic.net) ជាមួយនឹងជំនួយដ៏ធំពី Reggie។ យើងក៏បានទទួលការកែតម្រូវពីភ្នែកជាច្រើនគូនៅក្នុងសហគមន៍ Discord និងវេទិកាផងដែរ។

ឯកសារត្រូវបានថែទាំដោយ [Marcus Olsson](https://marcus.se.net/) មួយរយៈ។ បច្ចុប្បន្នវាត្រូវបានថែទាំដោយក្រុម Obsidian។

ឥឡូវនេះយើងទទួលយក pull requests ទៅកាន់ vault ជំនួយនៅក្នុង [repo ឯកសាររបស់យើង](https://github.com/obsidianmd/obsidian-docs/)។ សំណើរបញ្ហាត្រូវបានស្វាគមន៍។

### បន្ថែមឈ្មោះរបស់អ្នក

ប្រសិនបើអ្នកជាអ្នកបកប្រែ សូមធ្វើ pull request ដើម្បីបន្ថែម "ឈ្មោះរបស់អ្នក (ភាសាដែលរួមចំណែក)" ទៅផ្នែកខាងលើ។ សូមរក្សាបញ្ជីឈ្មោះតាមលំដាប់អក្សរក្រម អរគុណ!

ប្រសិនបើឈ្មោះរបស់អ្នកមិនមែនជាភាសាអង់គ្លេស និងមិនអាចរាយតាមលំដាប់អក្សរក្រមបានទេ សូមប្រើទម្រង់នេះ: `your-github-username, [ឈ្មោះរបស់អ្នកជាភាសារបស់អ្នក] (ឈ្មោះភាសា)`។
