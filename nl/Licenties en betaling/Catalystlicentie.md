De Catalyst-licentie is een eenmalige aanschaf die je vroegtijdige toegang geeft tot bètaversies en helpt de voortdurende ontwikkeling van Obsidian te ondersteunen. Je kunt Obsidian ondersteunen door [een Catalyst-licentie aan te schaffen](https://obsidian.md/buy/catalyst).

Voordelen zijn onder andere:

- Vroegtijdige toegang tot [[#Vroege-toegangsversies activeren|nieuwe versies]] van Obsidian
- Toegang tot exclusieve [[#Word lid van onze Insider-kanalen|Insider-kanalen]].
- Exclusieve [[#Ontvang je forumbadge|forumbadges]] en [[#Ontvang je Discord-badge|Discord-badges]].

## Catalyst-niveaus

Er zijn drie niveaus van Catalyst op basis van het bedrag dat je bijdraagt: *Insider*, *Supporter* en *VIP*.

| Voordelen                                                                                      | Insider | Supporter |   VIP    |
| ---------------------------------------------------------------------------------------------- | :-----: | :-------: | :------: |
| Vroegtijdige toegang tot nieuwe versies                                                        |    ✓    |     ✓     |    ✓     |
| Toegang tot exclusieve ontwikkelkanalen, `#insider-desktop` en `#insider-mobile`               |    ✓    |     ✓     |    ✓     |
| Toegang tot exclusief loungekanaal, `#vip-lounge`                                              |         |     ✓     |    ✓     |
| Insider-badge                                                                                  |    ✓    |           |          |
| Supporter-badge                                                                                |         |     ✓     |          |
| VIP-badge                                                                                      |         |           |    ✓     |
| **Bijdrage (USD)**                                                                             | **$25** |  **$50**  | **$100** |


> [!info] Upgrade je Catalyst-niveau
> Als je al een Catalyst-licentie hebt, kun je upgraden naar een hoger Catalyst-niveau door het prijsverschil te betalen tussen je huidige en het nieuwe niveau.

## Toegang tot Catalyst-voordelen

Zodra je een Catalyst-licentie hebt aangeschaft, krijg je toegang tot een reeks voordelen. Lees hier hoe je ze kunt activeren.

### Ontvang je forumbadge

**Opmerking:** Om je Obsidian-account en je forumaccount te verbinden, moet je voor beide accounts hetzelfde e-mailadres gebruiken.

1. Word lid van ons [forum](https://forum.obsidian.md/) als je dat nog niet hebt gedaan.
2. Ga naar je [Obsidian-accountdashboard](https://obsidian.md/account/catalyst).
3. Selecteer onder **Catalyst** de optie **Get forum badge**.

### Ontvang je Discord-badge

1. Word lid van onze [Discord-server](https://discord.gg/veuWUTm) als je dat nog niet hebt gedaan.
2. Ga naar je [Obsidian-accountdashboard](https://obsidian.md/account/catalyst).
3. Selecteer onder **Catalyst** de optie **Get Discord badge**.

### Word lid van onze Insider-kanalen

Insiders kunnen chatten met het Obsidian-team en andere Insiders in onze exclusieve Discord-kanalen. Je wordt automatisch lid van de Insider-kanalen nadat je [[#Ontvang je Discord-badge]].
