Obsidian Web Clipper is een gratis browserextensie waarmee je pagina's kunt markeren en webinhoud kunt opslaan in je kluis.

## Web Clipper installeren

Download de Obsidian Web Clipper-extensie uit de officiële directory voor jouw browser:

- [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
	- Voor Chrome, Brave, Arc, Orion en andere Chromium-gebaseerde browsers.
- [Firefox Add-Ons](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
	- Voor Firefox en Firefox Mobile.
- [Safari Extensions](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
	- Voor macOS, iOS en iPadOS.
- [Edge Add-Ons](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
	- Voor Microsoft Edge.

## Privacy

Obsidian Web Clipper slaat inhoud lokaal op in je Obsidian-kluis en volgt ons [privacybeleid](https://obsidian.md/privacy). Je gegevens worden niet verzameld en we verzamelen geen gebruiksstatistieken. De code is [open source](https://github.com/obsidianmd/obsidian-clipper) en controleerbaar.
