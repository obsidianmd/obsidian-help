Op deze pagina staan de standaard sneltoetsen voor het navigeren en bewerken van tekst in Obsidian. Deze sneltoetsen worden aangeboden door je besturingssysteem of het framework waarop Obsidian is gebouwd, en kunnen niet worden aangepast binnen Obsidian.

Zie [[Sneltoetsen]] voor aanpasbare sneltoetsen voor Obsidian-opdrachten.

## Windows- en Linux-sneltoetsen

### Veelgebruikte acties

| Actie | Sneltoets |
|-|-|
| Kopiëren | `Ctrl+C` |
| Knippen | `Ctrl+X` |
| Plakken | `Ctrl+V` |
| Plakken zonder opmaak | `Ctrl+Shift+V` |
| Ongedaan maken | `Ctrl+Z` |
| Opnieuw | `Ctrl+Shift+Z` of `Ctrl+Y` |
| Alinea kopiëren | `Ctrl+C` (zonder geselecteerde tekst) |
| Alinea knippen | `Ctrl+X` (zonder geselecteerde tekst) |

### Tekst bewerken

| Actie | Sneltoets |
|-|-|
| Nieuwe regel invoegen | `Enter` |
| Vorige teken verwijderen | `Backspace` |
| Volgende teken verwijderen | `Delete` |
| Vorige woord verwijderen | `Ctrl+Backspace` |
| Volgende woord verwijderen | `Ctrl+Delete` |
| Huidige regel verwijderen | `Ctrl+Shift+K` (zonder geselecteerde tekst) |

### Tekstnavigatie

| Actie | Sneltoets |
|-|-|
| Cursor één teken verplaatsen | `Left/→` |
| Cursor naar het begin van het vorige woord verplaatsen | `Ctrl+←` |
| Cursor naar het einde van het volgende woord verplaatsen | `Ctrl+→` |
| Cursor naar het begin van de huidige regel verplaatsen | `Home` |
| Cursor naar het einde van de huidige regel verplaatsen | `End` |
| Cursor naar de vorige regel verplaatsen | `↑` |
| Cursor naar de volgende regel verplaatsen | `↓` |
| Cursor naar het begin van de notitie verplaatsen | `Ctrl+Home` |
| Cursor naar het einde van de notitie verplaatsen | `Ctrl+End` |
| Cursor één pagina omhoog verplaatsen | `Page up` |
| Cursor één pagina omlaag verplaatsen | `Page down` |

### Tekstselectie

| Actie | Sneltoets |
|-|-|
| Selectie vereenvoudigen | `Escape` |
| Alles selecteren | `Ctrl+A` |
| Selectie één teken uitbreiden | `Shift+Left/→` |
| Selectie uitbreiden tot het begin van het vorige woord | `Ctrl+Shift+←` |
| Selectie uitbreiden tot het einde van het volgende woord | `Ctrl+Shift+→` |
| Selectie uitbreiden tot het begin van de huidige regel | `Shift+Home` |
| Selectie uitbreiden tot het einde van de huidige regel | `Shift+End` |
| Selectie uitbreiden tot het begin van de notitie | `Ctrl+Shift+Home` |
| Selectie uitbreiden tot het einde van de notitie | `Ctrl+Shift+End` |
| Selectie één pagina omhoog uitbreiden | `Shift+Page up` |
| Selectie één pagina omlaag uitbreiden | `Shift+Page down` |

## macOS-sneltoetsen

### Veelgebruikte acties

| Actie | Sneltoets |
|-|-|
| Kopiëren | `Cmd+C` |
| Knippen | `Cmd+X` |
| Plakken | `Cmd+V` |
| Plakken zonder opmaak | `Cmd+Shift+V` |
| Ongedaan maken | `Cmd+Z` |
| Opnieuw | `Cmd+Shift+Z` |
| Alinea kopiëren | `Cmd+C` (zonder geselecteerde tekst) |
| Alinea knippen | `Cmd+X` (zonder geselecteerde tekst) |

### Tekstopmaak

| Actie | Sneltoets |
| ---- | ---- |
| Vetgedrukte tekst | `Cmd+B` |
| Cursieve tekst | `Cmd+I` |

### Tekst bewerken

| Actie | Sneltoets |
|-|-|
| Nieuwe regel invoegen | `Enter` |
| Vorige teken verwijderen | `Backspace` |
| Volgende teken verwijderen | `Delete` |

| Volgende woord verwijderen | `Option+Delete` |
| Verwijderen tot het begin van de huidige regel | `Cmd+Backspace` |
| Verwijderen tot het einde van de huidige regel | `Cmd+Delete` |
| Huidige regel verwijderen | `Cmd+Shift+K` (zonder geselecteerde tekst) |

### Tekstnavigatie

| Actie | Sneltoets |
|-|-|

| Cursor naar het begin van het vorige woord verplaatsen | `Option+←` |
| Cursor naar het einde van het volgende woord verplaatsen | `Option+→` |
| Cursor naar het begin van de huidige regel verplaatsen | `Cmd+←` |
| Cursor naar het einde van de huidige regel verplaatsen | `Cmd+→` |
| Cursor naar de vorige regel verplaatsen | `↑` |
| Cursor naar de volgende regel verplaatsen | `↓` |
| Cursor naar het begin van de notitie verplaatsen | `Cmd+↑` |
| Cursor naar het einde van de notitie verplaatsen | `Cmd+↓` |
| Cursor één pagina omhoog verplaatsen | `Fn+↑` |
| Cursor één pagina omlaag verplaatsen | `Fn+↓` |
