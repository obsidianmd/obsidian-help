[[Inleiding tot Obsidian Publish|Obsidian Publish]]-sites worden automatisch geoptimaliseerd voor zoekmachines en [[Linkvoorbeelden op sociale media|kaarten voor sociaal delen]]. Je kunt ook paginametadata aanpassen met beschrijvingen, permalinks en afbeeldingen.

Merk op dat je zoekmachineindexering kunt uitschakelen in je [[Sites beheren#Site-instellingen|site-instellingen]].

### Google Search Console

Om SEO bij te houden met Google, registreer je je site op [Google Search Console](https://search.google.com/search-console). Je kunt ook onze aanbevolen stappen volgen om [[Analyse|Google Analytics in te stellen]].

### Sitemap

De sitemap voor je Publish-site is beschikbaar op `/sitemap.xml`, bijvoorbeeld:
https://help.obsidian.md/sitemap.xml

De sitemap is ook beschikbaar als RSS-feed op `/rss.xml`:
https://help.obsidian.md/rss.xml

### Permalinks

Overweeg om [[Permalinks]] te gebruiken om permanente URL's op je site te definiëren.

### Metadata

Paginametadata kan worden aangepast met behulp van [[Eigenschappen#Eigenschappen voor Obsidian Publish|Eigenschappen]], wat nuttig kan zijn voor [[Linkvoorbeelden op sociale media|kaarten voor sociaal delen]].

| Eigenschap    | Beschrijving                                                                                                      |
| ------------- | ----------------------------------------------------------------------------------------------------------------- |
| `publish`     | Zie [[Je inhoud publiceren#Automatisch gegevens selecteren om te publiceren\|Automatisch gegevens selecteren om te publiceren]]. |
| `permalink`   | Zie [[Permalinks\|Permalinks]].                                                                                   |
| `description` | Zie [[Linkvoorbeelden op sociale media#Beschrijving\|Beschrijving]].                                              |
| `image`       | Zie [[Linkvoorbeelden op sociale media#Afbeelding\|Afbeelding]].                                                  |
| `cover`       | Zie [[Linkvoorbeelden op sociale media#Afbeelding\|Afbeelding]].                                                  |
