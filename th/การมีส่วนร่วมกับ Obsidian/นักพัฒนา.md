หากคุณเป็นนักพัฒนา คุณสามารถมีส่วนร่วมกับ Obsidian ได้หลายวิธี

### ปลั๊กอินและธีม

คุณสามารถพัฒนา[[ปลั๊กอินโดยชุมชน]]และ[[ธีม]]โดยใช้ Typescript และ CSS ข้อมูลเพิ่มเติมได้ที่ [เอกสารประกอบสำหรับนักพัฒนา Obsidian](https://docs.obsidian.md)

### โปรเจกต์โอเพนซอร์ส

Obsidian พัฒนาและดูแลโปรเจกต์โอเพนซอร์สหลายโปรเจกต์ คุณสามารถส่ง pull request สำหรับการแก้ไขบั๊กและการปรับปรุงไปยัง repository บน GitHub ดังนี้:

- [JSON Canvas](https://github.com/obsidianmd/jsoncanvas) (MIT) รูปแบบไฟล์เปิดสำหรับข้อมูลผืนผ้าใบแบบไม่จำกัด
- [Obsidian Importer](https://github.com/obsidianmd/obsidian-importer) (MIT) ปลั๊กอินที่แปลงข้อมูลจากรูปแบบต่างๆ มากมายเป็นไฟล์ Markdown ข้อความธรรมดา
- [Obsidian Maps](https://github.com/obsidianmd/obsidian-maps) (MIT) ปลั๊กอินที่แสดงแผนที่ใน Obsidian ฐาน
- [Obsidian Web Clipper](https://github.com/obsidianmd/obsidian-clipper) (MIT) ส่วนขยายเบราว์เซอร์สำหรับไฮไลต์และจับภาพหน้าเว็บเป็นไฟล์ Markdown ข้อความธรรมดา
- [Defuddle](https://github.com/kepano/defuddle) (MIT) ตัวแปลง HTML เป็น Markdown ที่ใช้โดย Obsidian Web Clipper

### เอกสารประกอบ

หากต้องการช่วยปรับปรุง[เอกสารประกอบสำหรับนักพัฒนา](https://docs.obsidian.md/Home) ให้ส่ง pull request ไปยัง repository [obsidian-developer-docs](https://github.com/obsidianmd/obsidian-developer-docs)

หากต้องการช่วยปรับปรุงเอกสารประกอบนี้ ให้ส่ง pull request ไปยัง repository [obsidian-help](https://github.com/obsidianmd/obsidian-help)
