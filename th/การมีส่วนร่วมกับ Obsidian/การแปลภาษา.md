หากคุณรู้ภาษาอื่น โดยเฉพาะอย่างยิ่งในระดับเจ้าของภาษา คุณสามารถช่วยแปลอินเทอร์เฟซผู้ใช้ของ Obsidian หรือเอกสารประกอบได้

หากต้องการช่วยแปลอินเทอร์เฟซผู้ใช้ ให้ส่ง pull request ไปที่ [คลังการแปลภาษา](https://github.com/obsidianmd/obsidian-translations)

หากต้องการช่วยปรับปรุงเอกสารประกอบของเรา ให้ส่ง pull request ไปที่คลัง [obsidian-help](https://github.com/obsidianmd/obsidian-help) โปรดตรวจสอบให้แน่ใจว่าปฏิบัติตาม[[คู่มือสไตล์#การแปลภาษา|คู่มือสไตล์การแปลภาษา]]เมื่อทำการเปลี่ยนแปลง
