Obsidian Headless **(เปิดเบต้า)** เป็นไคลเอนต์แบบ headless สำหรับบริการ Obsidian ช่วยให้คุณ[[Headless Sync|ซิงค์ห้องนิรภัย]]จากบรรทัดคำสั่งโดยไม่ต้องใช้แอปเดสก์ท็อป พร้อมด้วยความเร็ว ความเป็นส่วนตัว และประโยชน์ของ[[แนะนำ Obsidian Sync|Obsidian Sync]] ที่มาพร้อมกับการเข้ารหัสแบบต้นทางถึงปลายทาง

เหตุผลที่คุณอาจใช้ Obsidian Headless:

- สำรองข้อมูลระยะไกลแบบอัตโนมัติ
- เผยแพร่เว็บไซต์แบบอัตโนมัติ
- ให้เครื่องมือ AI agent เข้าถึงห้องนิรภัยโดยไม่ต้องเข้าถึงคอมพิวเตอร์ทั้งเครื่องของคุณ
- ซิงค์ห้องนิรภัยของทีมที่แชร์ไปยังเซิร์ฟเวอร์ที่เชื่อมต่อกับเครื่องมืออื่น
- รันระบบอัตโนมัติตามกำหนดเวลา เช่น รวบรวมโน้ตประจำวันเป็นสรุปรายสัปดาห์ ติดแท็กอัตโนมัติ ฯลฯ

> [!info] Obsidian Headless vs Obsidian CLI
> [[Obsidian CLI]] ควบคุมแอปเดสก์ท็อป Obsidian จากเทอร์มินัลของคุณ Obsidian Headless เป็นไคลเอนต์แยกต่างหากที่ทำงานได้อย่างอิสระ ไม่ต้องใช้แอปเดสก์ท็อป

## ติดตั้ง

Obsidian Headless **(เปิดเบต้า)** ต้องใช้ Node.js 22 หรือใหม่กว่า ติดตั้งจาก [npm](https://www.npmjs.com/package/obsidian-headless):

```shell
npm install -g obsidian-headless
```

## การยืนยันตัวตน

### ล็อกอิน

```shell
ob login
```

หากล็อกอินอยู่แล้ว `ob login` จะแสดงข้อมูลบัญชีของคุณ หากต้องการเปลี่ยนบัญชี ให้ส่ง `--email` และ/หรือ `--password` เพื่อล็อกอินอีกครั้ง

```
ob login [--email <email>] [--password <password>] [--mfa <code>]
```

ตัวเลือกปรับแต่งทั้งหมดจะเป็นแบบโต้ตอบเมื่อไม่ได้ระบุ — อีเมลและรหัสผ่านจะถูกถาม และ 2FA จะถูกร้องขอโดยอัตโนมัติหากเปิดใช้งานในบัญชี

หากต้องการออกจากระบบและล้างข้อมูลประจำตัวที่จัดเก็บไว้:

```shell
ob logout
```
