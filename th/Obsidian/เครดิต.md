## เอกสารประกอบ

เวอร์ชั่นแรกของคลังวิธีใช้เขียนโดย Silver ส่วนที่เหลือเขียนโดย [@mediapathic](http://mediapathic.net) โดยได้รับความช่วยเหลืออย่างมากจาก Reggie เรายังได้รับการแก้ไขจากสายตาคู่ต่างๆ มากมายในชุมชน Discord และฟอรัม

เอกสารประกอบได้รับการดูแลโดย [Marcus Olsson](https://marcus.se.net/) เป็นระยะหนึ่ง ปัจจุบันดูแลโดยทีม Obsidian

ขณะนี้เรารับ pull request สำหรับคลังวิธีใช้ใน [repo เอกสารของเรา](https://github.com/obsidianmd/obsidian-docs/) ยินดีรับ issues

### เพิ่มชื่อของคุณ

หากคุณเป็นนักแปล กรุณาสร้าง pull request เพื่อเพิ่ม "ชื่อของคุณ (ภาษาที่แปล)" ในส่วนด้านบน กรุณาเรียงรายชื่อตามตัวอักษรด้วย ขอบคุณ!

หากชื่อของคุณไม่ใช่ภาษาอังกฤษและไม่สามารถเรียงตามตัวอักษรได้ ให้ใช้รูปแบบนี้: `your-github-username, [ชื่อของคุณในภาษาของคุณ] (ชื่อภาษา)`
