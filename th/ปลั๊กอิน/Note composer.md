Note composer เป็น[[ปลั๊กอินหลัก]]ที่ช่วยให้คุณผสานสองโน้ตเข้าด้วยกันหรือแยกส่วนหนึ่งของโน้ตออกเป็นโน้ตใหม่

## ผสานโน้ต

การผสานโน้ตจะเพิ่มโน้ตหนึ่งไปยังอีกโน้ตหนึ่งและลบโน้ตแรกออก Note composer จะอัปเดตลิงก์ทั้งหมดให้อ้างอิงไปยังโน้ตที่ผสานแล้ว

เมื่อคุณเลือกโน้ตที่จะผสานเข้า คุณสามารถเลือกระหว่างวิธีต่อไปนี้:

- `Enter`: เพิ่มโน้ตต้นทางที่ _ท้าย_ ของโน้ตปลายทาง
- `Shift+Enter`: เพิ่มโน้ตต้นทางที่ _ต้น_ ของโน้ตปลายทาง
- `Ctrl+Enter` (หรือ `Cmd+Enter` บน macOS): สร้างโน้ตใหม่ที่มีเนื้อหาของโน้ตต้นทาง

วิธีผสานโน้ตที่ใช้งานอยู่กับโน้ตอื่นในห้องนิรภัยของคุณ:

**ตัวสำรวจไฟล์**

1. ในตัวสำรวจไฟล์ คลิกขวาที่โน้ตที่คุณต้องการผสาน
2. คลิก **รวมไฟล์ทั้งไฟล์กับ...**
3. เลือกโน้ตที่คุณต้องการผสานเข้า
4. คลิก **ผสาน** เพื่อยืนยัน

**กระดานคำสั่ง**

1. เปิด[[กระดานคำสั่ง]]
2. เลือก **Note composer: รวมไฟล์ปัจจุบันกับไฟล์อื่น...**
3. เลือกโน้ตที่คุณต้องการผสานเข้า
4. คลิก **ผสาน** เพื่อยืนยัน

> [!tip] เคล็ดลับ
> โดยค่าเริ่มต้น Note composer จะขอให้คุณยืนยันเมื่อผสานโน้ต หากคุณปิดใช้งานการยืนยัน และคุณผสานโน้ตโดยผิดพลาด คุณยังสามารถกู้คืนได้ด้วยปลั๊กอิน[[การกู้คืนข้อมูล]]

## แยกโน้ต

เมื่อคุณเลือกโน้ตที่จะแยกข้อความที่เลือกไป คุณสามารถเลือกระหว่างวิธีต่อไปนี้:

- `Enter`: เพิ่มข้อความที่เลือกที่ _ท้าย_ ของโน้ตปลายทาง
- `Shift+Enter`: เพิ่มข้อความที่เลือกที่ _ต้น_ ของโน้ตปลายทาง
- `Ctrl+Enter` (หรือ `Cmd+Enter` บน macOS): สร้างโน้ตใหม่ที่มีข้อความที่เลือก

วิธีแยกข้อความไปยังโน้ตใหม่:

**เอดิเตอร์**

1. ขณะอยู่ใน **เอดิเตอร์** ให้เลือกข้อความที่คุณต้องการแยก
2. คลิกขวาที่ข้อความที่เลือก
3. คลิก **แยกข้อความที่เลือก...**
4. เลือกโน้ตที่คุณต้องการแยกไป

**กระดานคำสั่ง**

1. ขณะอยู่ใน **เอดิเตอร์** ให้เลือกข้อความที่คุณต้องการแยก
2. เปิด[[กระดานคำสั่ง]]
3. เลือก **Note composer: แยกข้อความที่เลือก...**
4. เลือกโน้ตที่คุณต้องการแยกไป

> [!tip] เคล็ดลับ
> โดยค่าเริ่มต้น Note composer จะแทนที่ข้อความที่แยกด้วยลิงก์ไปยังโน้ตปลายทาง ในการตั้งค่า คุณสามารถเปลี่ยนเป็น[[ฝังไฟล์|ฝัง]]โน้ตปลายทางแทน หรือไม่ทิ้งอะไรไว้เลย

## ไฟล์เทมเพลต

การกำหนดค่าเทมเพลตช่วยให้คุณปรับแต่งเนื้อหาก่อนที่จะเพิ่มลงในโน้ตใหม่ เพื่อใช้เทมเพลต ให้ป้อน **ตำแหน่งไฟล์เทมเพลต** ในการตั้งค่าปลั๊กอิน

เทมเพลตสามารถมีตัวแปรต่อไปนี้:

| ตัวแปร             | คำอธิบาย                                                                                                                                              |
|-------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| `{{content}}`     | เนื้อหาที่จะผสาน หรือข้อความที่แยก หากคุณไม่ใส่ตัวแปรนี้ Note composer จะเพิ่มเนื้อหาที่ด้านล่างของเทมเพลต |
| `{{fromTitle}}`   | ชื่อของโน้ตต้นทาง                                                                                                                                 |
| `{{newTitle}}`    | ชื่อของโน้ตปลายทาง ตัวอย่างเช่น เพื่อเพิ่มชื่อไฟล์เป็นหัวเรื่องที่ด้านบนของไฟล์                                                     |
| `{{date:FORMAT}}` | วันที่สร้างของโน้ตใหม่ ตัวอย่างเช่น `{{date:YYYY-MM-DD}}`                                                                                       |
