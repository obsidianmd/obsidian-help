สูตรช่วยให้คุณสร้างคุณสมบัติที่คำนวณได้ใน[[แนะนำฐาน|ฐาน]]โดยใช้ข้อมูลจาก[[คุณสมบัติ]]อื่นๆ คุณสามารถคำนวณ จัดการข้อความ ทำงานกับวันที่ และอื่นๆ อีกมากมาย

## สิ่งที่สูตรทำได้

สูตรสามารถช่วยคุณ:

- **คำนวณค่า** บวกราคา คำนวณยอดรวม หรือดำเนินการทางคณิตศาสตร์
- **จัดการข้อความ** รวมสตริง เปลี่ยนตัวพิมพ์ หรือแยกสตริงย่อย
- **ทำงานกับวันที่** คำนวณความแตกต่างของเวลา จัดรูปแบบวันที่ หรือกำหนดเส้นตาย
- **ใช้ลอจิก** ใช้คำสั่งเงื่อนไขเพื่อแสดงค่าที่แตกต่างกัน
- **ประมวลผลรายการ** กรอง จัดเรียง แมป หรือรวมข้อมูลรายการ

## สร้างคุณสมบัติสูตร

ในการสร้างคุณสมบัติสูตร:

1. ในฐานของคุณ คลิก **คุณสมบัติ** ในแถบเครื่องมือ
2. คลิก **เพิ่มสูตร** ที่ด้านล่างของเมนู
3. ป้อนชื่อสำหรับคุณสมบัติสูตรของคุณ
4. พิมพ์สูตรของคุณในช่อง **สูตร**
5. ปิดกล่องโต้ตอบ

เอดิเตอร์สูตรจะออโต้คอมพลีทชื่อ[[ฟังก์ชัน]]และคุณสมบัติขณะที่คุณพิมพ์เพื่อตรวจสอบไวยากรณ์สูตรของคุณ เครื่องหมายถูกสีเขียวจะปรากฏเมื่อสูตรของคุณถูกต้อง

เมื่อสร้างแล้ว คุณสามารถใช้คุณสมบัติสูตรเหมือนกับคุณสมบัติอื่นๆ ในฐานของคุณ เพิ่มลงใน[[มุมมอง]] ใช้ในตัวกรอง จัดเรียงตามสูตร และอื่นๆ

## เขียนสูตร

ในเอดิเตอร์สูตร ให้พิมพ์นิพจน์โดยใช้คุณสมบัติ ตัวดำเนินการ และฟังก์ชัน

### อ้างอิงคุณสมบัติ

คุณสามารถอ้างอิงคุณสมบัติประเภทต่างๆ ในสูตรของคุณ:

- **คุณสมบัติของโน้ต** — คุณสมบัติจาก[[คุณสมบัติ|เมทาดาทาตอนต้น]]ของโน้ต
- **คุณสมบัติของไฟล์** — คุณสมบัติในตัว เช่น `file.name`, `file.size` หรือ `file.mtime`
- **คุณสมบัติสูตร** — สูตรอื่นๆ โดยใช้ `formula.formula_name`

**ตัวอย่าง:**

- `price * quantity` — คูณคุณสมบัติของโน้ตสองตัว
- `file.name + " - " + description` — รวมชื่อไฟล์กับคุณสมบัติของโน้ต
- `formula.price_per_unit * 1.1` — ใช้คุณสมบัติสูตรอื่น

### ใช้ตัวดำเนินการ

**ตัวดำเนินการทางคณิตศาสตร์** ดำเนินการคำนวณกับตัวเลข:

- `price + tax` — บวก
- `price - discount` — ลบ
- `price * quantity` — คูณ
- `price / quantity` — หาร
- `(part / whole) * 100` — ใช้วงเล็บสำหรับลำดับการดำเนินการ

**ตัวดำเนินการเปรียบเทียบ** เปรียบเทียบค่า:

- `price > 100` — มากกว่า
- `age < 18` — น้อยกว่า
- `status == "Done"` — เท่ากับ
- `status != "Done"` — ไม่เท่ากับ
- `file.mtime > now() - '7d'` — เปรียบเทียบวันที่

**ตัวดำเนินการบูลีน** รวมเงื่อนไขเชิงลอจิก:

- `!completed` — ไม่
- `price > 0 && quantity > 0` — และ
- `urgent || important` — หรือ

ข้อมูลเพิ่มเติมใน [[ไวยากรณ์ฐาน#ตัวดำเนินการ|ไวยากรณ์ฐาน]]

### ใช้ฟังก์ชัน

ฟังก์ชันดำเนินการกับค่า ฟังก์ชันที่ใช้ได้ขึ้นอยู่กับประเภทของค่าที่คุณทำงานด้วย ดูรายการทั้งหมดของ[[ฟังก์ชัน]]

**หมวดหมู่ฟังก์ชันทั่วไป:**

- **ฟังก์ชันทั่วไป** — `if()`, `now()`, `date()`, `link()`, `max()`, `min()`
- **ฟังก์ชันสตริง** — `contains()`, `replace()`, `split()`, `lower()`, `title()`
- **ฟังก์ชันตัวเลข** — `round()`, `ceil()`, `floor()`, `abs()`, `toFixed()`
- **ฟังก์ชันวันที่** — `format()`, `relative()`, `date()`, `time()`
- **ฟังก์ชันรายการ** — `filter()`, `map()`, `sort()`, `join()`, `unique()`

**ตัวอย่าง:**

- `if(price, "$" + price.toFixed(2), "")` เพื่อกำหนดเงื่อนไขพร้อมการจัดรูปแบบตัวเลข
- `file.name.lower()` เพื่อแปลงเป็นตัวพิมพ์เล็ก
- `tags.contains("urgent")` เพื่อตรวจสอบว่ารายการแท็กมีค่าที่ต้องการหรือไม่
- `due_date.format("YYYY-MM-DD")` เพื่อจัดรูปแบบวันที่

## ตัวอย่างสูตร

### คำนวณเส้นตาย

ตั้งวันครบกำหนดของโปรเจกต์เป็น 2 สัปดาห์หลังจากวันเริ่มต้น:

```js
start_date + "2w"
```

### แสดงสถานะเกินกำหนด

แสดง "Overdue" ถ้าวันครบกำหนดผ่านไปแล้วและสถานะไม่ใช่ "Done":

```js
if(due_date < now() && status != "Done", "Overdue", "")
```

### จัดรูปแบบสกุลเงิน

แสดงราคาพร้อมทศนิยม 2 ตำแหน่งและสัญลักษณ์สกุลเงิน:

```js
if(price, "$" + price.toFixed(2), "")
```

### นับรายการในลิสต์

นับจำนวนรายการในคุณสมบัติแบบรายการ:

```js
tasks.length
```

### คำนวณคะแนนลำดับความสำคัญ

รวมหลายปัจจัยเป็นคะแนนลำดับความสำคัญ:

```js
(impact * urgency) / effort
```

### รวมฟิลด์ข้อความ

สร้างชื่อ-นามสกุลจากชื่อและนามสกุล:

```js
first_name + " " + last_name
```

### คำนวณค่าใช้จ่ายทั้งหมด

คูณค่าใช้จ่ายรายเดือนด้วยจำนวนเดือนที่ถือครอง:

```js
monthlyUses * formula.Owned.round()
```

## ประเภทข้อมูล

สูตรทำงานกับข้อมูลประเภทต่างๆ:

- **สตริง** — ข้อความที่อยู่ในเครื่องหมายคำพูด: `"hello"` หรือ `'world'`
- **ตัวเลข** — ค่าตัวเลข: `42`, `3.14`, `(2 + 2)`
- **บูลีน** — จริงหรือเท็จ: `true`, `false`
- **วันที่** — สร้างด้วย `date()`, `today()` หรือ `now()`
- **รายการ** — คอลเลกชันของค่า: `[1, 2, 3]`
- **อ็อบเจกต์** — คู่คีย์-ค่า: `{"name": "value"}`

ประเภทผลลัพธ์ของสูตรจะถูกกำหนดโดยข้อมูลและฟังก์ชันที่ใช้

## อ้างอิงสูตรอื่น

สูตรสามารถอ้างอิงสูตรอื่นๆ เพื่อสร้างการคำนวณที่ต่อเนื่องกัน ตัวอย่างเช่น ถ้าคุณมีสูตรชื่อ `price_per_unit`:

```js
price / quantity
```

คุณสามารถอ้างอิงมันในสูตรอื่น:

```js
formula.price_per_unit * 1.1
```

> [!warning] หลีกเลี่ยงการอ้างอิงแบบวงกลม
> สูตรไม่สามารถอ้างอิงตัวเองโดยตรงหรือโดยอ้อมผ่านสูตรอื่น
