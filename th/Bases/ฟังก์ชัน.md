ฟังก์ชันถูกใช้ใน[[แนะนำฐาน|ฐาน]]เพื่อจัดการข้อมูลจาก[[คุณสมบัติ]]ใน[[มุมมอง#ตัวกรอง|ตัวกรอง]]และ[[สูตร]] ดู[[ไวยากรณ์ฐาน|ข้อมูลอ้างอิงไวยากรณ์ฐาน]]เพื่อเรียนรู้เพิ่มเติมเกี่ยวกับวิธีใช้ฟังก์ชัน

นอกจากฟังก์ชัน[[ฟังก์ชัน#Global|Global]] แล้ว ฟังก์ชันส่วนใหญ่ขึ้นอยู่กับประเภทของค่าที่คุณต้องการแก้ไข:

## Global

ฟังก์ชัน Global ใช้โดยไม่ต้องระบุประเภท

### `escapeHTML()`

`escapeHTML(html: string): string`

- หลีกเลี่ยงอักขระพิเศษในสตริงเพื่อให้ปลอดภัยสำหรับการรวมใน HTML

### `date()`

`date(date: string): date`

- `date(string): date` แปลงสตริงที่ให้มาและส่งคืนวัตถุวันที่
- สตริง `date` ควรอยู่ในรูปแบบ `YYYY-MM-DD HH:mm:ss`

### `duration()`

`duration(value: string): duration`

- แปลงสตริงเป็นระยะเวลา ดู[[ไวยากรณ์ฐาน#การคำนวณวันที่|ส่วนการคำนวณวันที่]]สำหรับรูปแบบสตริง `value`
- ระยะเวลาไม่จำเป็นต้องแปลงอย่างชัดเจนเมื่อทำการคำนวณวันที่ (เช่น `now() + '1d'`) แต่จำเป็นเมื่อทำการคำนวณกับระยะเวลา (เช่น `now() + (duration('1d') * 2)`)
- เมื่อทำการคำนวณระยะเวลากับสเกลาร์ ระยะเวลาต้องอยู่ทางซ้าย เช่น `duration('5h') * 2` แทนที่จะเป็น `2 * duration('5h')`

### `file()`

`file(path: string | file | url): file`

- ส่งคืนวัตถุไฟล์สำหรับไฟล์หรือเส้นทางที่ให้มา
- ตัวอย่าง: `file(link("[[filename]]"))` หรือ `file("path to file")`

### `html()`

`html(html: string): html`

- แปลงสตริงเป็น snippet โค้ดที่แสดงผลเป็น HTML

### `if()`

`if(condition: any, trueResult: any, falseResult?: any): any`

- `condition` คือเงื่อนไขที่จะถูกประเมิน
- `trueResult` คือผลลัพธ์เมื่อเงื่อนไขเป็นจริง
- `falseResult` คือผลลัพธ์ทางเลือกเมื่อเงื่อนไขเป็นเท็จ หากไม่ได้กำหนด จะถือว่าเป็น `null`
- ส่งคืน `trueResult` ถ้า `condition` เป็นจริง หรือเป็นค่า truthy หรือ `falseResult` ในกรณีอื่น
- ตัวอย่าง: `if(isModified, "Modified", "Unmodified")`

### `image()`

`image(path: string | file | url): image`

- ส่งคืนวัตถุรูปภาพที่จะแสดงผลรูปภาพในมุมมอง
- ตัวอย่าง: `image(image-property)` หรือ `image("https://obsidian.md/images/obsidian-logo-gradient.svg")`

### `icon()`

`icon(name: string): icon`

- ส่งคืนค่าที่จะแสดงผลเป็นไอคอนในมุมมอง ชื่อไอคอนต้องตรงกับไอคอน Lucide ที่รองรับ
- ตัวอย่าง: `icon("arrow-right")`

### `link()`

`link(path: string | file, display?: value): Link`

- แปลงสตริง `path` และส่งคืนวัตถุ Link ที่แสดงผลเป็นลิงก์ไปยังเส้นทางที่กำหนด
- สามารถระบุพารามิเตอร์ `display` เพื่อเปลี่ยนข้อความที่แสดงของลิงก์ได้

### `list()`

`list(element: any): List`

- ถ้าองค์ประกอบที่ให้มาเป็นรายการ จะส่งคืนโดยไม่เปลี่ยนแปลง
- มิฉะนั้น จะห่อ `element` ที่ให้มาในรายการ สร้างรายการที่มีองค์ประกอบเดียว
- ฟังก์ชันนี้มีประโยชน์เมื่อคุณสมบัติมีทั้งสตริงหรือรายการผสมกันทั่วทั้งห้องนิรภัย
- ตัวอย่าง: `list("value")` ส่งคืน `["value"]`

### `max()`

`max(value1: number, value2: number...): number`

- ส่งคืนค่าที่มากที่สุดจากตัวเลขทั้งหมดที่ให้มา

### `min()`

`min(value1: number, value2: number...): number`

- ส่งคืนค่าที่น้อยที่สุดจากตัวเลขทั้งหมดที่ให้มา

### `now()`

`now(): date`

- `now()` ส่งคืนวัตถุวันที่ที่แสดงถึงช่วงเวลาปัจจุบัน

### `number()`

`number(input: any): number`

- พยายามส่งคืนค่าที่ให้มาเป็นตัวเลข
- วัตถุวันที่จะถูกส่งคืนเป็นมิลลิวินาทีนับจาก unix epoch
- ค่าบูลีนจะส่งคืน 1 หรือ 0
- สตริงจะถูกแปลงเป็นตัวเลขและส่งคืนข้อผิดพลาดหากผลลัพธ์ไม่ถูกต้อง
- ตัวอย่าง: `number("3.4")` ส่งคืน `3.4`

### `duration()`

`duration(value: string): duration`

- แปลงสตริงเป็นระยะเวลา ดู[[ไวยากรณ์ฐาน#การคำนวณวันที่|ส่วนการคำนวณวันที่]]สำหรับรูปแบบสตริง `value`
- ระยะเวลาไม่จำเป็นต้องแปลงอย่างชัดเจนเมื่อทำการคำนวณวันที่ (เช่น `now() + '1d'`) แต่จำเป็นเมื่อทำการคำนวณกับระยะเวลา (เช่น `now() + (duration('1d') * 2)`)
- เมื่อทำการคำนวณระยะเวลากับสเกลาร์ ระยะเวลาต้องอยู่ทางซ้าย เช่น `duration('5h') * 2` แทนที่จะเป็น `2 * duration('5h')`

### `today()`

`today(): date`

- `today()` ส่งคืนวัตถุวันที่ที่แสดงถึงวันที่ปัจจุบัน ส่วนเวลาถูกตั้งเป็นศูนย์

### `random()`

`random(): number`

- `random()` ส่งคืนตัวเลขสุ่มระหว่าง 0 ถึง 1
- การสร้างตัวเลขจะรีเฟรชทุกครั้งที่โหลดมุมมอง การสลับระหว่างมุมมองจะเปลี่ยนตัวเลขสุ่ม

## Any

ฟังก์ชันที่คุณสามารถใช้กับค่าใดก็ได้ ซึ่งรวมถึงสตริง (เช่น `"hello"`), ตัวเลข (เช่น `42`), รายการ (เช่น `[1,2,3]`), วัตถุ และอื่นๆ

### `isTruthy()`

`any.isTruthy(): boolean`

- ส่งคืนค่าที่ถูกแปลงเป็นบูลีน
- ตัวอย่าง: `1.isTruthy()` ส่งคืน `true`

### `isType()`

`any.isType(type: string): boolean`

- ส่งคืน true ถ้าค่าเป็นประเภทที่ระบุ
- ตัวอย่าง: `"example".isType("string")` และ `true.isType("boolean")` ทั้งคู่ส่งคืน true

### `toString()`

`any.toString(): string`

- ส่งคืนการแสดงผลสตริงของค่าใดก็ได้
- ตัวอย่าง: `123.toString()` ส่งคืน `"123"`

## Date

ฟังก์ชันที่คุณสามารถใช้กับวันที่และเวลา เช่น `date("2025-05-27")` การเปรียบเทียบวันที่สามารถทำได้โดยใช้[[ไวยากรณ์ฐาน#การคำนวณวันที่|การคำนวณวันที่]]

### ฟิลด์

ฟิลด์ต่อไปนี้ใช้ได้สำหรับวันที่:

| ฟิลด์              | ประเภท   | คำอธิบาย                      |
| ------------------ | -------- | ----------------------------- |
| `date.year`        | `number` | ปีของวันที่                    |
| `date.month`       | `number` | เดือนของวันที่ (1–12)          |
| `date.day`         | `number` | วันของเดือน                    |
| `date.hour`        | `number` | ชั่วโมง (0–23)                |
| `date.minute`      | `number` | นาที (0–59)                   |
| `date.second`      | `number` | วินาที (0–59)                 |
| `date.millisecond` | `number` | มิลลิวินาที (0–999)           |

### `date()`

`date.date(): date`

- ส่งคืนวัตถุวันที่โดยตัดส่วนเวลาออก
- ตัวอย่าง: `now().date().format("YYYY-MM-DD HH:mm:ss")` ส่งคืนสตริงเช่น "2025-12-31 00:00:00"

### `format()`

`date.format(format: string): string`

- `format` คือสตริงรูปแบบ (เช่น `"YYYY-MM-DD"`)
- ส่งคืนวันที่ที่จัดรูปแบบตามสตริงรูปแบบ Moment.js ที่ระบุ
- ตัวอย่าง: `date.format("YYYY-MM-DD")` ส่งคืน `"2025-05-27"`

### `time()`

`date.time(): string`

- ส่งคืนเวลา
- ตัวอย่าง: `now().time()` ส่งคืนสตริงเช่น "23:59:59"

### `relative()`

`date.relative(): string`

- ส่งคืนการเปรียบเทียบที่อ่านง่ายของวันที่กับวันที่เวลาปัจจุบัน
- ตัวอย่าง: `file.mtime.relative()` ส่งคืนค่าเช่น `3 days ago`

### `isEmpty()`

`date.isEmpty(): boolean`

- ส่งคืน false

## String

ฟังก์ชันที่คุณสามารถใช้กับลำดับอักขระ เช่น `"hello"`

### ฟิลด์

| ฟิลด์           | ประเภท   | คำอธิบาย                          |
| --------------- | -------- | --------------------------------- |
| `string.length` | `number` | จำนวนอักขระในสตริง                |

### `contains()`

`string.contains(value: string): boolean`

- `value` คือสตริงย่อยที่ต้องการค้นหา
- ส่งคืน true ถ้าสตริงมี `value`
- ตัวอย่าง: `"hello".contains("ell")` ส่งคืน `true`

### `containsAll()`

`string.containsAll(...values: string): boolean`

- `values` คือสตริงย่อยหนึ่งรายการหรือมากกว่าที่ต้องการค้นหา
- ส่งคืน true ถ้าสตริงมี `values` ทั้งหมด
- ตัวอย่าง: `"hello".containsAll("h", "e")` ส่งคืน `true`

### `containsAny()`

`string.containsAny(...values: string): boolean`

- `values` คือสตริงย่อยหนึ่งรายการหรือมากกว่าที่ต้องการค้นหา
- ส่งคืน true ถ้าสตริงมีอย่างน้อยหนึ่งรายการใน `values`
- ตัวอย่าง: `"hello".containsAny("x", "y", "e")` ส่งคืน `true`

### `endsWith()`

`string.endsWith(query: string): boolean`

- `query` คือสตริงที่ต้องการตรวจสอบที่ส่วนท้าย
- ส่งคืน true ถ้าสตริงนี้ลงท้ายด้วย `query`
- ตัวอย่าง: `"hello".endsWith("lo")` ส่งคืน `true`

### `isEmpty()`

`string.isEmpty(): boolean`

- ส่งคืน true ถ้าสตริงไม่มีอักขระ หรือไม่มีค่า
- ตัวอย่าง: `"Hello world".isEmpty()` ส่งคืน `false`
- ตัวอย่าง: `"".isEmpty()` ส่งคืน `true`

### `lower()`

`string.lower(): string`

- ส่งคืนสตริงที่แปลงเป็นตัวพิมพ์เล็ก

### `replace()`

`string.replace(pattern: string | Regexp, replacement: string): string`

- `pattern` คือค่าที่ต้องการค้นหาในสตริงเป้าหมาย
- `replacement` คือค่าที่จะแทนที่รูปแบบที่พบ
- ถ้า `pattern` เป็นสตริง จะแทนที่ทุกครั้งที่พบรูปแบบ
- ถ้า `pattern` เป็น Regexp แฟล็ก `g` จะกำหนดว่าจะแทนที่เฉพาะครั้งแรกหรือทั้งหมด
- ตัวอย่าง: `""a:b:c:d".replace(/:/, "-")` ส่งคืน `"a-b,c,d"` ในขณะที่ `"a:b:c:d".replace(/:/g, "-")` ส่งคืน `"a-b-c-d"`

### `repeat()`

`string.repeat(count: number): string`

- `count` คือจำนวนครั้งที่ต้องการทำซ้ำสตริง
- ตัวอย่าง: `"123".repeat(2)` ส่งคืน `"123123"`

### `reverse()`

`string.reverse(): string`

- กลับลำดับสตริง
- ตัวอย่าง: `"hello".reverse()` ส่งคืน `"olleh"`

### `slice()`

`string.slice(start: number, end?: number): string`

- `start` คือดัชนีเริ่มต้นแบบรวม
- `end` คือดัชนีสิ้นสุดแบบไม่รวม (ตัวเลือก)
- ส่งคืนสตริงย่อยจาก `start` (รวม) ถึง `end` (ไม่รวม)
- ตัวอย่าง: `"hello".slice(1, 4)` ส่งคืน `"ell"`
- ถ้าละเว้น `end` จะตัดถึงท้ายสตริง

### `split()`

`string.split(separator: string | Regexp, n?: number): list`

- `separator` คือตัวคั่นสำหรับแยกสตริง
- `n` คือตัวเลขทางเลือก ถ้าระบุ ผลลัพธ์จะมี `n` องค์ประกอบแรก
- ส่งคืนรายการของสตริงย่อย
- ตัวอย่าง: `"a,b,c,d".split(",", 3)` หรือ `"a,b,c,d".split(/,/, 3)` ส่งคืน `["a", "b", "c"]`

### `startsWith()`

`string.startsWith(query: string): boolean`

- `query` คือสตริงที่ต้องการตรวจสอบที่ส่วนต้น
- ส่งคืน true ถ้าสตริงนี้เริ่มต้นด้วย `query`
- ตัวอย่าง: `"hello".startsWith("he")` ส่งคืน `true`

### `title()`

`string.title(): string`

- แปลงสตริงเป็น title case (อักษรตัวแรกของแต่ละคำเป็นตัวพิมพ์ใหญ่)
- ตัวอย่าง: `"hello world".title()` ส่งคืน `"Hello World"`

### `trim()`

`string.trim(): string`

- ลบช่องว่างจากทั้งสองปลายของสตริง
- ตัวอย่าง: `"  hi  ".trim()` ส่งคืน `"hi"`

## Number

ฟังก์ชันที่คุณสามารถใช้กับค่าตัวเลข เช่น `42`, `3.14`

### `abs()`

`number.abs(): number`

- ส่งคืนค่าสัมบูรณ์ของตัวเลข
- ตัวอย่าง: `(-5).abs()` ส่งคืน `5`

### `ceil()`

`number.ceil(): number`

- ปัดตัวเลขขึ้นเป็นจำนวนเต็มที่ใกล้ที่สุด
- ตัวอย่าง: `(2.1).ceil()` ส่งคืน `3`

### `floor()`

`number.floor(): number`

- ปัดตัวเลขลงเป็นจำนวนเต็มที่ใกล้ที่สุด
- ตัวอย่าง: `(2.9).floor()` ส่งคืน `2`

### `isEmpty()`

`number.isEmpty(): boolean`

- ส่งคืน true ถ้าตัวเลขไม่มีค่า
- ตัวอย่าง: `5.isEmpty()` ส่งคืน `false`

### `round()`

`number.round(digits: number): number`

- ปัดตัวเลขเป็นจำนวนเต็มที่ใกล้ที่สุด
- สามารถระบุพารามิเตอร์ `digits` เพื่อปัดเป็นจำนวนทศนิยมตามที่ระบุ
- ตัวอย่าง: `(2.5).round()` ส่งคืน `3` และ `(2.3333).round(2)` ส่งคืน `2.33`

### `toFixed()`

`number.toFixed(precision: number): string`

- `precision` คือจำนวนตำแหน่งทศนิยม
- ส่งคืนสตริงที่มีตัวเลขในรูปแบบจุดตายตัว
- ตัวอย่าง: `(3.14159).toFixed(2)` ส่งคืน `"3.14"`

## List

ฟังก์ชันที่คุณสามารถใช้กับรายการที่เรียงลำดับขององค์ประกอบ เช่น `[1, 2, 3]`

### ฟิลด์

| ฟิลด์         | ประเภท   | คำอธิบาย                          |
| ------------- | -------- | --------------------------------- |
| `list.length` | `number` | จำนวนองค์ประกอบในรายการ            |

### `contains()`

`list.contains(value: any): boolean`

- `value` คือองค์ประกอบที่ต้องการค้นหา
- ส่งคืน true ถ้ารายการมี `value`
- ตัวอย่าง: `[1,2,3].contains(2)` ส่งคืน `true`

### `containsAll()`

`list.containsAll(...values: any): boolean`

- `values` คือองค์ประกอบหนึ่งรายการหรือมากกว่าที่ต้องการค้นหา
- ส่งคืน true ถ้ารายการมี `values` ทั้งหมด
- ตัวอย่าง: `[1,2,3].containsAll(2,3)` ส่งคืน `true`

### `containsAny()`

`list.containsAny(...values: any): boolean`

- `values` คือองค์ประกอบหนึ่งรายการหรือมากกว่าที่ต้องการค้นหา
- ส่งคืน true ถ้ารายการมีอย่างน้อยหนึ่งรายการใน `values`
- ตัวอย่าง: `[1,2,3].containsAny(3,4)` ส่งคืน `true`

### `filter()`

`list.filter(value: Boolean): list`

- กรององค์ประกอบของรายการนี้โดยเรียกฟังก์ชันกรอง ซึ่งใช้ตัวแปร `index` และ `value` และส่งคืนค่าบูลีนว่าควรเก็บองค์ประกอบไว้หรือไม่
- `value` คือค่าของรายการในรายการ
- `index` คือดัชนีของค่าปัจจุบัน
- ตัวอย่าง: `[1,2,3,4].filter(value > 2)` ส่งคืน `[3,4]`

### `flat()`

`list.flat(): list`

- ทำให้รายการซ้อนกันแบนเป็นรายการเดียว
- ตัวอย่าง: `[1,[2,3]].flat()` ส่งคืน `[1,2,3]`

### `isEmpty()`

`list.isEmpty(): boolean`

- ส่งคืน true ถ้ารายการไม่มีองค์ประกอบ
- ตัวอย่าง: `[1,2,3].isEmpty()` ส่งคืน `false`

### `join()`

`list.join(separator: string): string`

- `separator` คือสตริงที่จะแทรกระหว่างองค์ประกอบ
- รวมองค์ประกอบทั้งหมดของรายการเป็นสตริงเดียว
- ตัวอย่าง: `[1,2,3].join(",")` ส่งคืน `"1,2,3"`

### `map()`

`list.map(value: Any): list`

- แปลงแต่ละองค์ประกอบของรายการนี้โดยเรียกฟังก์ชันแปลง ซึ่งใช้ตัวแปร `index` และ `value` และส่งคืนค่าใหม่ที่จะวางในรายการ
- `value` คือค่าของรายการในรายการ
- `index` คือดัชนีของค่าปัจจุบัน
- ตัวอย่าง: `[1,2,3,4].map(value + 1)` ส่งคืน `[2,3,4,5]`

### `reduce()`

`list.reduce(expression: Any, acc: Any): Any`

- ลดองค์ประกอบของรายการนี้เป็นค่าเดียวโดยรันนิพจน์สำหรับแต่ละองค์ประกอบ นิพจน์สามารถใช้ตัวแปร `index`, `value` และ `acc` (ตัวสะสม) และควรส่งคืนค่าตัวสะสมถัดไป
- `expression` จะถูกประเมินสำหรับทุกองค์ประกอบในรายการ
- `value` คือค่าของรายการปัจจุบันในรายการ
- `index` คือดัชนีของรายการปัจจุบัน
- `acc` คือค่าสะสมจนถึงปัจจุบัน
- ตัวอย่าง (ผลรวม): `[1,2,3].reduce(acc + value, 0)` ส่งคืน `6`
- ตัวอย่าง (ค่าสูงสุด): `values.filter(value.isType("number")).reduce(if(acc == null || value > acc, value, acc), null)` ส่งคืนตัวเลขที่มากที่สุด หรือ `null` ถ้าไม่มี

### `reverse()`

`list.reverse(): list`

- กลับลำดับรายการ
- ตัวอย่าง: `[1,2,3].reverse()` ส่งคืน `[3,2,1]`

### `slice()`

`list.slice(start: number, end?: number): list`

- `start` คือดัชนีเริ่มต้นแบบรวม
- `end` คือดัชนีสิ้นสุดแบบไม่รวม (ตัวเลือก)
- ส่งคืนสำเนาตื้นของส่วนหนึ่งของรายการจาก `start` (รวม) ถึง `end` (ไม่รวม)
- ตัวอย่าง: `[1,2,3,4].slice(1,3)` ส่งคืน `[2,3]`
- ถ้าละเว้น `end` จะตัดถึงท้ายรายการ

### `sort()`

`list.sort(): list`

- จัดเรียงองค์ประกอบรายการจากน้อยไปมาก
- ตัวอย่าง: `[3, 1, 2].sort()` ส่งคืน `[1, 2, 3]`
- ตัวอย่าง: `["c", "a", "b"].sort()` ส่งคืน `["a", "b", "c"]`

### `unique()`

`list.unique(): list`

- ลบองค์ประกอบที่ซ้ำกัน
- ตัวอย่าง: `[1,2,2,3].unique()` ส่งคืน `[1,2,3]`

## Link

ฟังก์ชันที่คุณสามารถใช้กับลิงก์ ลิงก์สามารถสร้างจากไฟล์ (`file.asLink()`) หรือเส้นทาง (`link("path")`)

### `asFile()`

`link.asFile(): file`

- ส่งคืนวัตถุไฟล์ถ้าลิงก์อ้างถึงไฟล์ท้องถิ่นที่ถูกต้อง
- ตัวอย่าง: `link("[[filename]]").asFile()`

### `linksTo()`

`link.linksTo(file): boolean`

- ส่งคืนว่าไฟล์ที่แสดงโดย `link` มีลิงก์ไปยัง `file` หรือไม่

## File

ฟังก์ชันที่คุณสามารถใช้กับไฟล์ในห้องนิรภัย

### ฟิลด์

ฟิลด์ต่อไปนี้ใช้ได้สำหรับไฟล์:

| ฟิลด์             | ประเภท   | คำอธิบาย                                                |
| ----------------- | -------- | ------------------------------------------------------- |
| `file.name`       | `string` | ชื่อของไฟล์นี้                                           |
| `file.basename`   | `string` | ชื่อของไฟล์นี้โดยไม่มี extension                         |
| `file.path`       | `string` | เส้นทางเต็มไปยังไฟล์นี้ สัมพัทธ์กับรากของห้องนิรภัย       |
| `file.folder`     | `string` | เส้นทางเต็มไปยังโฟลเดอร์หลัก                             |
| `file.ext`        | `string` | extension ของไฟล์นี้                                     |
| `file.size`       | `number` | ขนาดของไฟล์นี้ เป็นไบต์                                  |
| `file.properties` | `object` | คุณสมบัติโน้ตของไฟล์นี้                                   |
| `file.tags`       | `list`   | แท็กของไฟล์นี้ รวมถึงแท็กอินไลน์                         |
| `file.links`      | `list`   | ลิงก์ภายในของไฟล์นี้                                     |
| `file.ctime`      | `date`   | เวลาประทับเมื่อไฟล์นี้ถูกสร้าง                           |
| `file.mtime`      | `date`   | เวลาประทับเมื่อไฟล์นี้ถูกแก้ไขล่าสุด                     |

### `asLink()`

`file.asLink(display?: string): Link`

- `display` ข้อความที่แสดงทางเลือกสำหรับลิงก์
- ส่งคืนวัตถุ Link ที่แสดงผลเป็นลิงก์ที่ใช้งานได้
- ตัวอย่าง: `file.asLink()`

### `hasLink()`

`file.hasLink(otherFile: file | string): boolean`

- `otherFile` คือวัตถุไฟล์อื่นหรือเส้นทางสตริงที่ต้องการตรวจสอบ
- ส่งคืน true ถ้า `file` มีลิงก์ไปยัง `otherFile`
- ตัวอย่าง: `file.hasLink(otherFile)` ส่งคืน `true` ถ้ามีลิงก์จาก `file` ไปยัง `otherFile`

### `hasProperty()`

`file.hasProperty(name: string): boolean`

- ส่งคืน true ถ้าโน้ตมีคุณสมบัติไฟล์ที่กำหนด

### `hasTag()`

`file.hasTag(...values: string): boolean`

- `values` คือชื่อแท็กหนึ่งรายการหรือมากกว่า
- ส่งคืน true ถ้าไฟล์มีแท็กใดก็ได้ใน `values`
- ตัวอย่าง: `file.hasTag("tag1", "tag2")` ส่งคืน `true` ถ้าไฟล์มีแท็ก `#tag1` หรือ `#tag2` ซึ่งยังรวมถึง[[แท็ก#แท็กซ้อน|แท็กซ้อน]]ด้วย เช่น `#tag1/a` หรือ `#tag2/b`

### `inFolder()`

`file.inFolder(folder: string): boolean`

- `folder` คือชื่อโฟลเดอร์ที่ต้องการตรวจสอบ
- ส่งคืน true ถ้าไฟล์อยู่ในโฟลเดอร์ที่ระบุหรือโฟลเดอร์ย่อยของมัน
- ตัวอย่าง: `file.inFolder("notes")` ส่งคืน `true`

## Object

ฟังก์ชันที่คุณสามารถใช้กับคอลเลกชันของคู่คีย์-ค่า เช่น `{"a": 1, "b": 2}`

### `isEmpty()`

`object.isEmpty(): boolean`

- ส่งคืน true ถ้าวัตถุไม่มีคุณสมบัติของตัวเอง
- ตัวอย่าง: `{}.isEmpty()` ส่งคืน `true`

### `keys()`

`object.keys(): list`

- ส่งคืนรายการที่มีคีย์ของวัตถุ

### `values()`

`object.values(): list`

- ส่งคืนรายการที่มีค่าของวัตถุ

## นิพจน์ทั่วไป

ฟังก์ชันที่คุณสามารถใช้กับรูปแบบนิพจน์ทั่วไป ตัวอย่าง: `/abc/`

### `matches()`

`regexp.matches(value: string): boolean`

- `value` คือสตริงที่ต้องการทดสอบ
- ส่งคืน true ถ้านิพจน์ทั่วไปตรงกับ `value`
- ตัวอย่าง: `/abc/.matches("abcde")` ส่งคืน `true`
