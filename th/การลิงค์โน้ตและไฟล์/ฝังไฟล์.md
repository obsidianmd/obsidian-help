ไฟล์ที่ฝังจะแสดงเนื้อหาแบบอินไลน์ในโน้ตและอัปเดตอัตโนมัติเมื่อไฟล์ต้นทางมีการเปลี่ยนแปลง

ในการฝังไฟล์ในห้องนิรภัยของคุณ ให้เพิ่มเครื่องหมายอัศเจรีย์ (`!`) ไว้หน้า[[ลิงค์ภายใน|ลิงก์ภายใน]] คุณสามารถฝังไฟล์ในทุก[[รูปแบบไฟล์ที่รองรับ]]ได้

> [!tip]- ลากและวางเพื่อฝัง
> บนเดสก์ท็อป คุณยังสามารถลากและวางไฟล์ที่รองรับลงในโน้ตของคุณโดยตรงเพื่อฝังโดยอัตโนมัติได้

## ฝังโน้ตในโน้ตอื่น

ในการฝังโน้ต:

```md
![[ลิงค์ภายใน]]
```

คุณยังสามารถฝัง[[ลิงค์ภายใน#ลิงก์ไปยังหัวเรื่องในโน้ต|หัวเรื่อง]]และ[[ลิงค์ภายใน#ลิงก์ไปยังบล็อกในโน้ต|บล็อก]]ที่เฉพาะเจาะจงได้

```md
![[ลิงค์ภายใน#^b15695]]
```

ข้อความด้านล่างเป็นตัวอย่างของบล็อกที่ถูกฝัง:

![[ลิงค์ภายใน#^b15695]]

## ฝังรูปภาพในโน้ต

ในการฝังรูปภาพ:

```md
![[Engelbart.jpg]]
```

![[Engelbart.jpg#outline]]

คุณสามารถเปลี่ยนขนาดรูปภาพได้โดยเพิ่ม `|640x480` ไปยังปลายทางของลิงก์ โดย 640 คือความกว้าง และ 480 คือความสูง

```md
![[Engelbart.jpg|100x145]]
```

หากคุณระบุเฉพาะความกว้าง รูปภาพจะปรับขนาดตามสัดส่วน ตัวอย่างเช่น `![[Engelbart.jpg|100]]`

![[Engelbart.jpg#outline|100]]

คุณยังสามารถฝังรูปภาพที่โฮสต์ภายนอกได้โดยใช้ลิงก์ Markdown คุณสามารถควบคุมความกว้างและความสูงได้โดยใช้ไวยากรณ์เดียวกับลิงก์วิกิ


## ฝัง PDF ในโน้ต

ในการฝัง PDF:

```md
![[Document.pdf]]
```

คุณยังสามารถเปิดหน้าเฉพาะใน PDF ได้โดยเพิ่ม `#page=N` ไปยังปลายทางของลิงก์ โดย `N` คือหมายเลขหน้า:

```md
![[Document.pdf#page=3]]
```

คุณยังสามารถระบุความสูงเป็นพิกเซลสำหรับตัวแสดง PDF ที่ฝังได้โดยเพิ่ม `#height=[number]` ไปยังลิงก์ ตัวอย่างเช่น:

```md
![[Document.pdf#height=400]]
```

## ฝังผืนผ้าใบในโน้ต

ในการฝัง[[Canvas|ผืนผ้าใบ]]:

```md
![[My canvas.canvas]]
```

> [!info]+ ผืนผ้าใบที่ฝังจะแสดงเฉพาะรูปทรง
> ผืนผ้าใบที่ฝังจะแสดงรูปทรงแต่ไม่แสดงข้อความภายในการ์ด หากต้องการดูผืนผ้าใบแบบเต็ม ให้เปิดโดยตรง

## ฝังรายการในโน้ต

ในการฝังรายการจากโน้ตอื่น ก่อนอื่นให้เพิ่ม[[ลิงค์ภายใน#ลิงก์ไปยังบล็อกในโน้ต|ตัวระบุบล็อก]]ให้กับรายการของคุณ:

```md

- รายการข้อ 1
- รายการข้อ 2

^my-list-id
```

จากนั้นลิงก์ไปยังรายการโดยใช้ตัวระบุบล็อก:

```md
![[My note#^my-list-id]]
```
