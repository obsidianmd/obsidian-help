หน้านี้แสดงรายการแป้นพิมพ์ลัดเริ่มต้นสำหรับการนำทางและแก้ไขข้อความใน Obsidian แป้นพิมพ์ลัดเหล่านี้มาจากระบบปฏิบัติการของคุณหรือเฟรมเวิร์กที่ Obsidian ใช้สร้างขึ้น และไม่สามารถปรับแต่งได้ภายใน Obsidian

สำหรับแป้นพิมพ์ลัดที่ปรับแต่งได้สำหรับคำสั่ง Obsidian ดูที่ [[ฮ็อตคีย์]]

## แป้นพิมพ์ลัดสำหรับ Windows และ Linux

### การดำเนินการทั่วไป

| การดำเนินการ | แป้นพิมพ์ลัด |
|-|-|
| คัดลอก | `Ctrl+C` |
| ตัด | `Ctrl+X` |
| วาง | `Ctrl+V` |
| วางโดยไม่มีการจัดรูปแบบ | `Ctrl+Shift+V` |
| Undo | `Ctrl+Z` |
| Redo | `Ctrl+Shift+Z` หรือ `Ctrl+Y` |
| คัดลอกย่อหน้า | `Ctrl+C` (เมื่อไม่ได้เลือกข้อความ) |
| ตัดย่อหน้า | `Ctrl+X` (เมื่อไม่ได้เลือกข้อความ) |

### การแก้ไขข้อความ

| การดำเนินการ | แป้นพิมพ์ลัด |
|-|-|
| แทรกบรรทัดใหม่ | `Enter` |
| ลบอักขระก่อนหน้า | `Backspace` |
| ลบอักขระถัดไป | `Delete` |
| ลบคำก่อนหน้า | `Ctrl+Backspace` |
| ลบคำถัดไป | `Ctrl+Delete` |
| ลบบรรทัดปัจจุบัน | `Ctrl+Shift+K` (เมื่อไม่ได้เลือกข้อความ) |

### การนำทางข้อความ

| การดำเนินการ | แป้นพิมพ์ลัด |
|-|-|
| เลื่อนเคอร์เซอร์หนึ่งอักขระ | `Left/→` |
| เลื่อนเคอร์เซอร์ไปที่จุดเริ่มต้นของคำก่อนหน้า | `Ctrl+←` |
| เลื่อนเคอร์เซอร์ไปที่จุดสิ้นสุดของคำถัดไป | `Ctrl+→` |
| เลื่อนเคอร์เซอร์ไปที่จุดเริ่มต้นของบรรทัดปัจจุบัน | `Home` |
| เลื่อนเคอร์เซอร์ไปที่จุดสิ้นสุดของบรรทัดปัจจุบัน | `End` |
| เลื่อนเคอร์เซอร์ไปบรรทัดก่อนหน้า | `↑` |
| เลื่อนเคอร์เซอร์ไปบรรทัดถัดไป | `↓` |
| เลื่อนเคอร์เซอร์ไปที่จุดเริ่มต้นของโน้ต | `Ctrl+Home` |
| เลื่อนเคอร์เซอร์ไปที่จุดสิ้นสุดของโน้ต | `Ctrl+End` |
| เลื่อนเคอร์เซอร์ขึ้นหนึ่งหน้า | `Page up` |
| เลื่อนเคอร์เซอร์ลงหนึ่งหน้า | `Page down` |

### การเลือกข้อความ

| การดำเนินการ | แป้นพิมพ์ลัด |
|-|-|
| ลดความซับซ้อนของการเลือก | `Escape` |
| เลือกทั้งหมด | `Ctrl+A` |
| ขยายการเลือกหนึ่งอักขระ | `Shift+Left/→` |
| ขยายการเลือกไปที่จุดเริ่มต้นของคำก่อนหน้า | `Ctrl+Shift+←` |
| ขยายการเลือกไปที่จุดสิ้นสุดของคำถัดไป | `Ctrl+Shift+→` |
| ขยายการเลือกไปที่จุดเริ่มต้นของบรรทัดปัจจุบัน | `Shift+Home` |
| ขยายการเลือกไปที่จุดสิ้นสุดของบรรทัดปัจจุบัน | `Shift+End` |
| ขยายการเลือกไปที่จุดเริ่มต้นของโน้ต | `Ctrl+Shift+Home` |
| ขยายการเลือกไปที่จุดสิ้นสุดของโน้ต | `Ctrl+Shift+End` |
| ขยายการเลือกขึ้นหนึ่งหน้า | `Shift+Page up` |
| ขยายการเลือกลงหนึ่งหน้า | `Shift+Page down` |

| การดำเนินการ | แป้นพิมพ์ลัด |
|-|-|
| คัดลอก | `Cmd+C` |
| ตัด | `Cmd+X` |
| วาง | `Cmd+V` |

| ตัดย่อหน้า | `Cmd+X` (เมื่อไม่ได้เลือกข้อความ) |

### การจัดรูปแบบข้อความ

| การดำเนินการ | แป้นพิมพ์ลัด |
| ---- | ---- |
| ข้อความตัวหนา | `Cmd+B` |
| ข้อความตัวเอียง | `Cmd+I` |

### การแก้ไขข้อความ

| การดำเนินการ | แป้นพิมพ์ลัด |
|-|-|
