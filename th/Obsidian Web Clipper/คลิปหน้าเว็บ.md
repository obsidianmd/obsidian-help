เมื่อคุณติดตั้งส่วนขยายเบราว์เซอร์ [[แนะนำ Obsidian Web Clipper|Web Clipper]] แล้ว คุณสามารถเข้าถึงได้หลายวิธี ขึ้นอยู่กับเบราว์เซอร์ของคุณ:

1. ไอคอน Obsidian ในแถบเครื่องมือเบราว์เซอร์ของคุณ
2. ฮ็อตคีย์ เพื่อเปิดใช้งานส่วนขยายจากคีย์บอร์ดของคุณ
3. เมนูบริบท โดยคลิกขวาที่หน้าเว็บที่คุณกำลังเยี่ยมชม

เพื่อเซฟหน้าเว็บไปยัง Obsidian ให้คลิกปุ่ม **เพิ่มไปยัง Obsidian**

## จับภาพหน้าเว็บ

เมื่อคุณเปิดส่วนขยาย Web Clipper จะดึงข้อมูลจากหน้าเว็บปัจจุบันตามการตั้งค่าใน[[Obsidian Web Clipper/เทมเพลต|เทมเพลต]]ของคุณ คุณสามารถสร้างเทมเพลตของคุณเอง และปรับแต่งผลลัพธ์โดยใช้[[ตัวแปร]]และ[[ตัวกรอง]]

โดยค่าเริ่มต้น Web Clipper จะพยายามดึงเฉพาะเนื้อหาบทความหลักอย่างชาญฉลาด โดยไม่รวมองค์ประกอบอื่นๆ บนหน้า อย่างไรก็ตาม คุณสามารถเปลี่ยนการทำงานนี้ได้ด้วยวิธีต่อไปนี้:

- หากมีเทมเพลตกำหนดเองอยู่ จะใช้เทมเพลตของคุณ
- หากมีการเลือกข้อความ จะใช้ส่วนที่เลือก คุณสามารถใช้ `Ctrl/Cmd+A` เพื่อเลือกทั้งหน้า
- หากมี[[ไฮไลต์หน้าเว็บ|ไฮไลต์]]อยู่ จะใช้ไฮไลต์เหล่านั้น

## ดาวน์โหลดรูปภาพ

รูปภาพจะไม่ถูกดาวน์โหลดโดยอัตโนมัติเมื่อคุณใช้ Web Clipper แต่รูปภาพจะลิงก์ไปยัง URL บนเว็บแทน ซึ่งช่วยประหยัดพื้นที่ในห้องนิรภัยของคุณ แต่หมายความว่ารูปภาพจะไม่สามารถเข้าถึงได้แบบออฟไลน์ หรือหาก URL หยุดทำงาน

คุณสามารถดาวน์โหลดรูปภาพสำหรับไฟล์ใดก็ได้ใน Obsidian โดยใช้[[กระดานคำสั่ง|คำสั่ง]]ที่ชื่อว่า **ดาวน์โหลดไฟล์แนบสำหรับไฟล์ปัจจุบัน** คำสั่งนี้ยังสามารถแมปกับฮ็อตคีย์ใน Obsidian ได้อีกด้วย

## ฮ็อตคีย์

Web Clipper มีทางลัดคีย์บอร์ดที่คุณสามารถใช้เพื่อเร่งขั้นตอนการทำงานของคุณ เพื่อเปลี่ยนการแมปคีย์ ให้ไปที่ **การตั้งค่า Web Clipper** → **ทั่วไป** และทำตามคำแนะนำสำหรับเบราว์เซอร์ของคุณ สามารถเปลี่ยนการแมปได้สำหรับเบราว์เซอร์ทั้งหมด ยกเว้น Safari ซึ่งไม่รองรับการแก้ไขฮ็อตคีย์

| การทำงาน                  | macOS         | Windows/Linux  |
| ----------------------- | ------------- | -------------- |
| เปิด clipper            | `Cmd+Shift+O` | `Ctrl+Shift+O` |
| คลิปด่วน              | `Opt+Shift+O` | `Alt+Shift+O`  |
| สลับโหมดไฮไลต์ | `Opt+Shift+H` | `Alt+Shift+H`  |

## ฟังก์ชันการทำงานของอินเทอร์เฟซ

อินเทอร์เฟซ Web Clipper แบ่งออกเป็นสี่ส่วน:

1. **ส่วนหัว** ที่คุณสามารถสลับเทมเพลต เปิด[[ไฮไลต์หน้าเว็บ|ไฮไลต์]] [[อ่าน|โหมดอ่าน]] และเข้าถึงการตั้งค่า
2. **คุณสมบัติ** แสดง[[คุณสมบัติ|เมทาดาทา]]ที่ดึงมาจากหน้าเว็บ ซึ่งจะถูกเซฟเป็น[[คุณสมบัติ]]ใน Obsidian
3. **เนื้อหาบันทึก** ที่จะถูกเซฟไปยัง Obsidian
4. **ส่วนท้าย** ให้คุณเลือกห้องนิรภัยและโฟลเดอร์ และเพิ่มไปยัง Obsidian

ฟังก์ชันการทำงานของส่วนหัวประกอบด้วย:

- ดรอปดาวน์ **เทมเพลต** เพื่อสลับระหว่าง[[Obsidian Web Clipper/เทมเพลต|เทมเพลต]]ที่เซฟไว้ซึ่งเพิ่มในการตั้งค่า Web Clipper
- ปุ่ม **เพิ่มเติม (...)** เพื่อแสดงตัวแปรหน้าที่คุณสามารถใช้ในเทมเพลต
- ปุ่ม **ตัวไฮไลต์** เพื่อเปิด[[ไฮไลต์หน้าเว็บ|ไฮไลต์]]
- ปุ่ม **เฟือง** เพื่อเปิดการตั้งค่า Web Clipper

ฟังก์ชันการทำงานของส่วนท้ายประกอบด้วย:

- ปุ่ม **เพิ่มไปยัง Obsidian** เพื่อเซฟข้อมูลไปยัง Obsidian
- ดรอปดาวน์ **คลัง** เพื่อสลับระหว่างคลังที่เซฟไว้ซึ่งเพิ่มในการตั้งค่า Web Clipper
- ช่อง **โฟลเดอร์** เพื่อกำหนดโฟลเดอร์ที่จะเซฟไปยัง
- **ตัวแปล** เพื่อเรียกใช้[[แปลหน้าเว็บ|พรอมต์ภาษาธรรมชาติ]]บนหน้าเว็บ
