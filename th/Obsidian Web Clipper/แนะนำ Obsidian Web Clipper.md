Obsidian Web Clipper เป็นส่วนขยายเบราว์เซอร์ฟรีที่ให้คุณไฮไลต์หน้าเว็บและบันทึกเนื้อหาเว็บลงในห้องนิรภัยของคุณ

## ติดตั้ง Web Clipper

รับส่วนขยาย Obsidian Web Clipper จากไดเรกทอรีอย่างเป็นทางการสำหรับเบราว์เซอร์ของคุณ:

- [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
	- สำหรับ Chrome, Brave, Arc, Orion และเบราว์เซอร์อื่นๆ ที่ใช้ Chromium
- [Firefox Add-Ons](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
	- สำหรับ Firefox และ Firefox Mobile
- [Safari Extensions](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
	- สำหรับ macOS, iOS และ iPadOS
- [Edge Add-Ons](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
	- สำหรับ Microsoft Edge

## วิธีใช้ Web Clipper

- [[คลิปหน้าเว็บ]]
	- เริ่มต้นใช้งานและเรียนรู้วิธีบันทึกเนื้อหาจากหน้าเว็บ
- [[ตัวไฮไลต์|ตัวไฮไลต์]]
	- ไฮไลต์ข้อความสำคัญและเลือกองค์ประกอบที่คุณต้องการบันทึก
- [[อ่าน|อ่าน]]
	- อ่านหน้าเว็บในมุมมองที่สะอาดตาปราศจากสิ่งรบกวน
- [[ตัวแปล|ตัวแปล]]
	- ใช้คำสั่งภาษาธรรมชาติเพื่อจับภาพและแก้ไขข้อมูลบนหน้าเว็บ
- [[แก้ไขปัญหา Web Clipper|การแก้ไขปัญหา]]
	- แก้ไขปัญหาทั่วไปเกี่ยวกับ Web Clipper
- [[Obsidian Web Clipper/เทมเพลต|เทมเพลต]]
	- สร้างเทมเพลตกำหนดเองสำหรับเว็บไซต์เฉพาะ
- [[ตัวแปร]]
	- กรอกข้อมูลหน้าเว็บล่วงหน้าในเทมเพลตของคุณ
- [[ตัวกรอง]]
	- ปรับแต่งผลลัพธ์ของตัวแปรเทมเพลต
- [[ลอจิก]]
	- ใช้เงื่อนไข ลูป และตั้งค่าตัวแปรในเทมเพลต

## ความเป็นส่วนตัว

Obsidian Web Clipper บันทึกเนื้อหาลงในห้องนิรภัย Obsidian ของคุณในเครื่องและปฏิบัติตาม[นโยบายความเป็นส่วนตัว](https://obsidian.md/privacy)ของเรา ข้อมูลของคุณจะไม่ถูกเก็บรวบรวม และเราไม่ได้รวบรวมตัวชี้วัดการใช้งานใดๆ โค้ดเป็น[โอเพนซอร์ส](https://github.com/obsidianmd/obsidian-clipper)และสามารถตรวจสอบได้
