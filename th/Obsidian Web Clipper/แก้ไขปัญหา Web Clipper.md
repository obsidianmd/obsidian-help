## ทั่วไป

### เนื้อหาบางส่วนหายไป

ตามค่าเริ่มต้น Web Clipper จะพยายามจับเนื้อหาจากหน้าเว็บอย่างชาญฉลาด อย่างไรก็ตาม อาจไม่สำเร็จในทุกเว็บไซต์

Web Clipper ใช้ [Defuddle](https://github.com/kepano/defuddle) เพื่อจับเฉพาะเนื้อหาหลักของหน้า ซึ่งจะไม่รวม header, footer และองค์ประกอบอื่นๆ แต่บางครั้งอาจเข้มงวดเกินไปและลบเนื้อหาที่คุณต้องการเก็บไว้ คุณสามารถ [รายงานบั๊ก](https://github.com/kepano/defuddle) ไปยัง Defuddle

เพื่อข้าม Defuddle ใน Web Clipper ให้ใช้วิธีต่อไปนี้:

- เลือกข้อความ หรือใช้ `Cmd/Ctrl+A` เพื่อเลือกข้อความทั้งหมด
- [[ไฮไลต์หน้าเว็บ|ไฮไลต์เนื้อหา]] เพื่อเลือกสิ่งที่คุณต้องการจับอย่างแม่นยำ
- ใช้ [[Obsidian Web Clipper/เทมเพลต|เทมเพลตกำหนดเอง]] สำหรับเว็บไซต์นั้น

### ไม่มีเนื้อหาปรากฏใน Obsidian

หากคุณไม่เห็นเนื้อหาใดๆ ใน Obsidian เมื่อคุณคลิก **เพิ่มไปยัง Obsidian**:

- ตรวจสอบข้อผิดพลาดใน [[ความช่วยเหลือและการสนับสนุน#บันทึกบันทึกคอนโซล|คอนโซลนักพัฒนา]] ของ Obsidian
- ตรวจสอบว่าชื่อห้องนิรภัยในการตั้งค่า Web Clipper ตรงกับ *ชื่อห้องนิรภัย* ใน Obsidian อย่างแม่นยำ *ไม่ใช่เส้นทางห้องนิรภัย*
- ตรวจสอบว่าชื่อโฟลเดอร์มีรูปแบบที่ถูกต้อง

## Linux

#### Obsidian ไม่เปิดขึ้น

- ตรวจสอบให้แน่ใจว่าโปรโตคอล [[Obsidian URI]] [[Obsidian URI#ลงทะเบียน Obsidian URI|ได้ลงทะเบียนแล้ว]]
- หากคุณใช้ Firefox คุณอาจต้อง [ลงทะเบียนในการตั้งค่าเบราว์เซอร์](https://kb.mozillazine.org/Register_protocol)

#### Obsidian เปิดขึ้นแต่บันทึกเฉพาะชื่อไฟล์เท่านั้น

เป็นไปได้ว่า Obsidian ไม่สามารถเข้าถึงคลิปบอร์ดของคุณได้ การเข้าถึงคลิปบอร์ดจำเป็นสำหรับการส่งข้อมูลจากเบราว์เซอร์ของคุณไปยัง Obsidian การตั้งค่าของคุณอาจส่งผลต่อการ sandbox แอป และสิทธิ์การเข้าถึงคลิปบอร์ด

หากคุณใช้ Wayland ตรวจสอบให้แน่ใจว่า Obsidian มีสิทธิ์อ่านคลิปบอร์ดเมื่อแอปไม่ได้อยู่ในโฟกัส การตั้งค่านี้อาจอยู่ใน tiling window manager ของคุณ เช่น Hyprland หรือ Sway

หากคุณใช้ Hyprland:

```ini
# ~/.config/hypr/hyprland.conf
misc {
    focus_on_activate = true
}
```

หากคุณใช้ Sway:

```ini
# ~/.config/sway/config
for_window [class="obsidian"] focus_on_window_activation focus
```

- หากคุณใช้ Flatpak ลองใช้ [เวอร์ชั่น Obsidian ที่รองรับอย่างเป็นทางการ](https://obsidian.md/download)
- หากคุณใช้ KDE ไปที่ **System Settings** → **Window Management** → **Window Rules** และอนุญาตให้ Obsidian รับโฟกัส [[web-clipper-kde.png|ดูภาพหน้าจอ]]
- เป็นทางเลือกสำรอง ลองเปลี่ยนเป็น **โหมดเก่า** ใน **การตั้งค่า Web Clipper** → **ทั่วไป** ซึ่งจะข้ามคลิปบอร์ดและบันทึกเนื้อหาโดยตรงผ่าน URI โปรดทราบว่าวิธีนี้จะจำกัดจำนวนอักขระที่สามารถคลิปได้ขึ้นอยู่กับเบราว์เซอร์และ Linux distribution ของคุณ

## iOS และ iPadOS

เพื่อเปิดใช้งานส่วนขยาย Web Clipper สำหรับ Safari:

1. ไปที่ Safari แตะปุ่มซ้ายสุดในแถบ URL ของเบราว์เซอร์ ซึ่งดูเหมือนสี่เหลี่ยมผืนผ้าที่มีเส้นอยู่ด้านล่าง
2. แตะ **Manage Extensions**
3. เปิดใช้งาน **Obsidian Web Clipper** ในรายการส่วนขยาย
4. ออกจากเมนู
5. เพื่อใช้ส่วนขยาย **แตะไอคอนชิ้นส่วนจิ๊กซอว์** ในแถบ URL

เพื่ออนุญาตให้ Web Clipper ทำงานบนทุกเว็บไซต์:

1. ไปที่ **[[การตั้งค่า]]** ของ iOS →  **Apps** →  **Safari** →  **Extensions**
2. ภายใต้ **Permissions** อนุญาตให้ทำงานบนทุกเว็บไซต์

เพื่ออนุญาตให้ Obsidian รับเนื้อหาจาก Web Clipper เสมอ:

1. ไปที่ **[[การตั้งค่า]]** ของ iOS →  **Apps** →  **Obsidian**
2. ตั้งค่า **Paste from other apps** เป็น **Allow**
