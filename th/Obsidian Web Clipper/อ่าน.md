[[แนะนำ Obsidian Web Clipper|Web Clipper]] มีโหมดอ่านในตัวที่เรียกว่า Reader ซึ่งจะตัดสิ่งรบกวนออกจากหน้าเว็บ และนำเสนอเฉพาะเนื้อหาบทความหลักในรูปแบบที่สะอาดและอ่านง่าย

## เปิดใช้งานโหมดอ่าน

คุณสามารถเปิดใช้งาน Reader ได้หลายวิธี:

- **ไอคอนหนังสือ** ในส่วนหัวของแผงส่วนขยาย
- ทางลัดแป้นพิมพ์ `Alt+Shift+R` สามารถปรับแต่งได้ในการตั้งค่าส่วนขยาย
- การสลับโหมดอ่านอีกครั้งจะคืนค่าหน้าเว็บเดิม

## วิธีการทำงาน

Reader จะดึงเนื้อหาหลักจากหน้าเว็บ หน้าเว็บเดิมจะถูกแทนที่ด้วยมุมมองการอ่านที่สะอาด ซึ่งประกอบด้วย:

- ชื่อบทความ ผู้เขียน วันที่เผยแพร่ และโดเมนต้นทาง
- เนื้อหาบทความหลัก โดยรูปภาพและการจัดรูปแบบยังคงอยู่
- เค้าโครงในแถบด้านข้างสำหรับบทความที่มีหัวเรื่องหลายรายการ
- การเน้นไวยากรณ์สำหรับบล็อกโค้ด
- ป็อปโอเวอร์เชิงอรรถที่แสดงแบบอินไลน์เมื่อคลิก

## ปรับแต่งมุมมองการอ่าน

โหมด Reader มีแถบการตั้งค่าที่ซ่อนอยู่ ซึ่งคุณสามารถแสดงได้โดยโฮเวอร์ที่มุมขวาบนของหน้าเว็บ พร้อมตัวควบคุมเพื่อปรับแต่งประสบการณ์การอ่าน

| การตั้งค่า | คำอธิบาย |
| --- | --- |
| ขนาดตัวอักษร | ปรับขนาดข้อความ |
| ความกว้างบรรทัด | ความกว้างสูงสุดของพื้นที่เนื้อหา |
| ความสูงบรรทัด | ระยะห่างระหว่างบรรทัด |
| ชุดสี |  |
| ธีม | อัตโนมัติจะตามค่ากำหนดของระบบ หรือเลือกใช้โหมดสีอ่อนหรือโหมดสีเข้มเสมอ |
