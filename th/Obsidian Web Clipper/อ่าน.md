[[แนะนำ Obsidian Web Clipper|Web Clipper]] มีโหมดอ่านในตัวที่เรียกว่า **Reader** ซึ่งจะตัดสิ่งรบกวนออกจากหน้าเว็บ และนำเสนอเฉพาะเนื้อหาบทความหลักในรูปแบบที่สะอาดและอ่านง่าย

## เปิดใช้งานโหมดอ่าน

คุณสามารถเปิดใช้งาน Reader ได้หลายวิธี:

- ![[lucide-book-icon.svg#icon]] **ไอคอนหนังสือ** ในแผงส่วนขยาย
- **ทางลัดแป้นพิมพ์** สามารถปรับแต่งได้ในการตั้งค่าส่วนขยาย
- **เมนูบริบท** โดยคลิกขวาที่หน้าเว็บที่คุณกำลังเยี่ยมชม

คุณยังสามารถตั้งค่าปุ่มส่วนขยาย ![[obsidian.svg#icon]] เพื่อเปิด Reader ในคลิกเดียว ในการตั้งค่า Web Clipper ไปที่ **General** → **Behavior** → **Open behavior** และเลือก **Reader**

ตัวเลือกทั้งหมดข้างต้นยังช่วยให้คุณออกจาก Reader และคืนค่าหน้าเว็บเดิมได้อีกด้วย

## วิธีการทำงาน

Reader จะดึงเนื้อหาหลักจากหน้าเว็บ หน้าเว็บเดิมจะถูกแทนที่ด้วยมุมมองการอ่านที่สะอาด ซึ่งประกอบด้วย:

- ชื่อบทความ ผู้เขียน วันที่เผยแพร่ และโดเมนต้นทาง
- เนื้อหาบทความหลัก โดยรูปภาพและการจัดรูปแบบยังคงอยู่
- เค้าโครงในแถบด้านข้างสำหรับบทความที่มีหัวเรื่องหลายรายการ
- การเน้นไวยากรณ์สำหรับบล็อกโค้ด
- ป็อปโอเวอร์เชิงอรรถที่แสดงแบบอินไลน์เมื่อคลิก

## ปรับแต่งมุมมองการอ่านของคุณ

ปรับแต่งประสบการณ์ Reader ของคุณในการตั้งค่าส่วนขยาย บางตัวเลือกยังสามารถใช้งานได้ภายในโหมด Reader โดยคลิกไอคอน ![[lucide-case-sensitive.svg#icon]]

| การตั้งค่า | คำอธิบาย |
| --- | --- |
| แบบอักษร | เพิ่มแบบอักษรที่ติดตั้งในระบบของคุณ |
| ขนาดตัวอักษร | ปรับขนาดข้อความ |
| ความสูงบรรทัด | ระยะห่างระหว่างบรรทัด |
| ความกว้างบรรทัด | ความกว้างสูงสุดของพื้นที่เนื้อหา |
| รูปลักษณ์ | เลือกรูปลักษณ์สีอ่อนหรือสีเข้ม |
| ธีม | เลือกสี |
| CSS กำหนดเอง | CSS เพิ่มเติมที่ใช้กับหน้าเว็บ |
