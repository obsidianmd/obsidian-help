[[แนะนำ Obsidian Web Clipper|Web Clipper]] ช่วยให้คุณสร้างเทมเพลตที่จับและจัดระเบียบเมทาดาทาจากหน้าเว็บโดยอัตโนมัติ ตัวอย่างเทมเพลตมีอยู่ใน [clipper-templates repo](https://github.com/kepano/clipper-templates)

## สร้างหรือแก้ไขเทมเพลต

หากต้องการ**สร้าง**เทมเพลต ให้ไปที่การตั้งค่า Web Clipper แล้วคลิกปุ่ม **เทมเพลตใหม่** ในแถบด้านข้าง คุณยังสามารถ**ทำซ้ำ**เทมเพลตได้ในเมนู **เพิ่มเติม** ที่มุมขวาบน

หากต้องการ**แก้ไข**เทมเพลต ให้เลือกเทมเพลตจากแถบด้านข้าง การเปลี่ยนแปลงของคุณจะถูกเซฟโดยอัตโนมัติ

เทมเพลตใช้ [[ตัวแปร]] และ [[ตัวกรอง]] ซึ่งช่วยให้คุณปรับแต่งวิธีการเซฟเนื้อหาได้

## นำเข้าและส่งออกเทมเพลต Web Clipper

วิธีนำเข้าเทมเพลต:

1. เปิดส่วนขยายแล้วคลิก ไอคอนรูปเฟือง **[[การตั้งค่า]]**
2. ไปที่เทมเพลตใดก็ได้ในรายการ
3. คลิก **นำเข้า** ที่มุมขวาบน หรือลากและวางไฟล์เทมเพลต `.json` ของคุณไปยังที่ใดก็ได้ในพื้นที่เทมเพลต

หากต้องการส่งออกเทมเพลต ให้คลิก **ส่งออก** ที่มุมขวาบน ซึ่งจะดาวน์โหลดไฟล์เทมเพลต `.json` คุณยังสามารถคัดลอกข้อมูลเทมเพลตไปยังคลิปบอร์ดได้ผ่านเมนู **เพิ่มเติม**

## การตั้งค่าเทมเพลต

### การทำงาน

กำหนดวิธีที่เนื้อหาจาก Web Clipper จะถูกเพิ่มไปยัง Obsidian:

- **สร้างโน้ตใหม่**
- **เพิ่มไปยังโน้ตที่มีอยู่** ที่ด้านบนหรือด้านล่าง
- **เพิ่มไปยังโน้ตประจำวัน** ที่ด้านบนหรือด้านล่าง (ต้องเปิดใช้งานปลั๊กอิน [[โน้ตประจำวัน]])

### กระตุ้นเทมเพลตโดยอัตโนมัติ

ตัวกระตุ้นเทมเพลตช่วยให้คุณเลือกเทมเพลตโดยอัตโนมัติตาม URL ของหน้าปัจจุบันหรือข้อมูล [schema.org](https://schema.org/) คุณสามารถกำหนดกฎหลายข้อสำหรับแต่ละเทมเพลต โดยคั่นด้วยบรรทัดใหม่

การจับคู่แรกในรายการเทมเพลตของคุณจะกำหนดว่าเทมเพลตใดถูกใช้ คุณสามารถลากเทมเพลตขึ้นและลงในการตั้งค่า Web Clipper เพื่อเปลี่ยนลำดับการจับคู่เทมเพลต

#### การจับคู่ URL แบบง่าย

การจับคู่แบบง่ายจะกระตุ้นเทมเพลตหาก URL ของหน้าปัจจุบัน *เริ่มต้นด้วย* รูปแบบที่กำหนด ตัวอย่าง:

- `https://obsidian.md` จะจับคู่กับ URL ใดก็ได้ที่เริ่มต้นด้วยข้อความนี้

#### การจับคู่ด้วยนิพจน์ทั่วไป

คุณสามารถกระตุ้นเทมเพลตตามรูปแบบ URL ที่ซับซ้อนมากขึ้นโดยใช้นิพจน์ทั่วไป ใส่รูปแบบ regex ของคุณในเครื่องหมายสแลช (`/`) อย่าลืม escape อักขระพิเศษในรูปแบบ regex (เช่น `.` และ `/`) ด้วยแบ็คสแลช (`\`) ตัวอย่าง:

- `/^https:\/\/www\.imdb\.com\/title\/tt\d+\/reference\/?$/` จะจับคู่กับหน้าอ้างอิง IMDB ใดก็ได้

#### การจับคู่ Schema.org

คุณสามารถกระตุ้นเทมเพลตตามข้อมูล [schema.org](https://schema.org/) ที่มีอยู่ในหน้า ใช้คำนำหน้า `schema:` ตามด้วยคีย์ schema ที่คุณต้องการจับคู่ คุณสามารถระบุค่าที่คาดหวังได้เป็นทางเลือก ตัวอย่าง:

- `schema:@Recipe` จะจับคู่กับหน้าที่มีประเภท schema เป็น "Recipe"
- `schema:@Recipe.name` จะจับคู่กับหน้าที่มี `@Recipe.name` อยู่
- `schema:@Recipe.name=Cookie` จะจับคู่กับหน้าที่ `@Recipe.name` เป็น "Cookie"

ค่า Schema.org ยังสามารถใช้เพื่อ [[ตัวแปร#ตัวแปร Schema.org|กรอกข้อมูลล่วงหน้าในเทมเพลต]] ได้อีกด้วย

### บริบทตัวแปล

เมื่อเปิดใช้งาน [[แปลหน้าเว็บ|ตัวแปล]] คุณสามารถใช้ [[ตัวแปร#ตัวแปรพรอมต์|ตัวแปรพรอมต์]] เพื่อดึงเนื้อหาหน้าเว็บด้วยภาษาธรรมชาติ สำหรับแต่ละเทมเพลตคุณสามารถกำหนด [[แปลหน้าเว็บ#บริบท|บริบท]] ที่ตัวแปลสามารถเข้าถึงได้
