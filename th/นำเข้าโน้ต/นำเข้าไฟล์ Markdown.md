Obsidian ใช้ไฟล์ Markdown `.md` เป็นรูปแบบหลักสำหรับโน้ต ทำให้การนำเข้าไฟล์ Markdown เข้าสู่ Obsidian ทำได้ง่าย แอปจำนวนมากสามารถแก้ไขไฟล์ Markdown หรือส่งออกเป็นรูปแบบนี้ได้

มีหลายวิธีในการเพิ่มไฟล์ Markdown เข้าไปในห้องนิรภัย Obsidian ของคุณ:

1. ลากและวางไฟล์หรือโฟลเดอร์ลงในหน้าต่าง[[ตัวจัดการไฟล์]]
2. ใช้ตัวจัดการไฟล์ของระบบ (เช่น Windows Explorer หรือ Finder บน macOS) แล้วย้ายไฟล์ของคุณไปยังโฟลเดอร์ห้องนิรภัย Obsidian โดยตรง

### การแปลงจาก Markdown รูปแบบอื่น

Obsidian ใช้ [[Obsidian Flavored Markdown]] แอปบางตัวใช้ไวยากรณ์ Markdown ที่แตกต่างกันเล็กน้อย คุณสามารถใช้[[นำเข้า Markdown]]เพื่อแปลงไฟล์ของคุณให้เป็นไวยากรณ์ Markdown ของ Obsidian
