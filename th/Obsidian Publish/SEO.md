เว็บไซต์ [[แนะนำ Obsidian Publish|Obsidian Publish]] จะถูกปรับให้เหมาะสมสำหรับเครื่องมือค้นหาและ[[พรีวิวลิงค์สื่อสังคม|การ์ดแชร์บนสื่อสังคม]]โดยอัตโนมัติ คุณยังสามารถปรับแต่งเมทาดาทาของหน้าด้วยคำอธิบาย ลิงค์ถาวร และรูปภาพได้อีกด้วย

โปรดทราบว่าคุณสามารถไม่อนุญาตให้ search engine ทำการ index ได้ใน[[จัดการเว็บไซต์#ตัวเลือกเว็บไซต์|ตัวเลือกเว็บไซต์]]ของคุณ

### Google Search Console

หากต้องการติดตาม SEO ด้วย Google ให้ลงทะเบียนเว็บไซต์ของคุณบน [Google Search Console](https://search.google.com/search-console) คุณยังสามารถทำตามขั้นตอนแนะนำของเราเพื่อ[[การวิเคราะห์|ตั้งค่า Google Analytics]] ได้อีกด้วย

### Sitemap

Sitemap สำหรับเว็บไซต์ Publish ของคุณจะอยู่ที่ `/sitemap.xml` ตัวอย่างเช่น:
https://help.obsidian.md/sitemap.xml

Sitemap ยังมีให้ในรูปแบบ RSS feed ที่ `/rss.xml`:
https://help.obsidian.md/rss.xml

### ลิงค์ถาวร

พิจารณาใช้[[ลิงค์ถาวร]]เพื่อกำหนด URL ถาวรบนเว็บไซต์ของคุณ

### เมทาดาทา

เมทาดาทาของหน้าสามารถปรับแต่งได้โดยใช้[[คุณสมบัติ#คุณสมบัติสำหรับ Obsidian Publish|คุณสมบัติ]] ซึ่งอาจเป็นประโยชน์สำหรับ[[พรีวิวลิงค์สื่อสังคม|การ์ดแชร์บนสื่อสังคม]]

| คุณสมบัติ      | คำอธิบาย                                                                                                       |
| ------------- | ----------------------------------------------------------------------------------------------------------------- |
| `publish`     | ดู [[เผยแพร่เนื้อหาของคุณ#เลือกข้อมูลที่จะเผยแพร่โดยอัตโนมัติ\|เลือกข้อมูลที่จะเผยแพร่โดยอัตโนมัติ]] |
| `permalink`   | ดู [[ลิงค์ถาวร\|ลิงค์ถาวร]]                                                                                   |
| `description` | ดู [[พรีวิวลิงค์สื่อสังคม#คำอธิบาย\|คำอธิบาย]]                                                      |
| `image`       | ดู [[พรีวิวลิงค์สื่อสังคม#รูปภาพ\|รูปภาพ]]                                                                  |
| `cover`       | ดู [[พรีวิวลิงค์สื่อสังคม#รูปภาพ\|รูปภาพ]]                                                                  |
