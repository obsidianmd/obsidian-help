หน้านี้แสดงรายการปัญหาทั่วไปที่คุณอาจพบกับ [[แนะนำ Obsidian Publish|Obsidian Publish]] และวิธีแก้ไข

อย่าลืมตรวจสอบ [[ไฟล์มีเดีย]] และ [[ข้อจำกัดของ Publish|ข้อจำกัดของ Publish]] ก่อน

## ทั่วไป

### การเผยแพร่โน้ต

**ฉันได้รับข้อผิดพลาด hash เมื่อพยายามเผยแพร่โน้ต**

คุณกำลังใช้ [[ปลั๊กอินโดยชุมชน|ปลั๊กอินชุมชน]] ที่แก้ไขเวลาที่แก้ไขของไฟล์เมื่ออัปเดตอยู่หรือไม่? หากใช่ ปลั๊กอินนี้อาจขัดแย้งกับ Publish โปรดแจ้งรายงานข้อบกพร่องไปยังผู้พัฒนาปลั๊กอินเพื่อแก้ไขปัญหา

**ฉันได้รับข้อผิดพลาดเครือข่ายที่ผิดปกติ และฉันมีเว็บไซต์ Publish ขนาดใหญ่มาก**

เป็นไปได้ว่าเราจำเป็นต้องตรวจสอบฐานข้อมูลของคุณ โปรด [[ความช่วยเหลือและการสนับสนุน#ติดต่อการสนับสนุน Obsidian|ติดต่อฝ่ายสนับสนุน Obsidian]] เพื่อขอความช่วยเหลือ

## CSS และธีม

**CSS ใน [[โฟลเดอร์การตั้งค่า]] ของฉันไม่ทำงานบน Publish ทำไม?**

Publish ไม่ได้อ่านจากโฟลเดอร์การตั้งค่า แต่คุณจะต้องสร้างไฟล์ `publish.css` ในไดเรกทอรีระดับบนสุดของห้องนิรภัยที่ใช้เผยแพร่ คุณสามารถเรียนรู้เพิ่มเติมได้ที่ [[ปรับแต่งเว็บไซต์ของคุณ]]

**CSS ของฉันบน Publish ไม่เหมือนกับในแอป ทำไม?**

CSS ของ Obsidian Publish ไม่เหมือนกับในแอปพลิเคชันทุกประการ เราแนะนำให้ [พัฒนา CSS และธีม](https://docs.obsidian.md/Themes/Obsidian+Publish+themes/About+Obsidian+Publish+themes) สำหรับ Publish โดยเฉพาะตั้งแต่ต้น

โดย _ทั่วไป_ สิ่งที่ทำงานได้กับ [[มุมมองและโหมดแก้ไข#มุมมองการอ่าน|มุมมองการอ่าน]] มีโอกาสสูงที่จะทำงานได้บน Publish
