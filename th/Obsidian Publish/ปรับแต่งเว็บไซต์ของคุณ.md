หน้านี้อธิบายวิธีที่คุณสามารถปรับแต่งรูปลักษณ์และความรู้สึกของเว็บไซต์ [[แนะนำ Obsidian Publish|Obsidian Publish]] ของคุณ

## ไฟล์สแตติก

คุณสามารถปรับแต่งเว็บไซต์ของคุณได้โดย[[เผยแพร่เนื้อหาของคุณ#เผยแพร่โน้ต|การเผยแพร่]]ไฟล์ต่อไปนี้ไปยังเว็บไซต์ของคุณ:

- `publish.css` เพื่อเพิ่ม CSS กำหนดเอง
- `publish.js` เพื่อเพิ่ม JavaScript กำหนดเอง
- `favicon-32x32.png` เพื่อตั้งค่า favicon

**หมายเหตุ:**

- [ตัวแปร CSS สำหรับ Publish](https://docs.obsidian.md/Reference/CSS+variables/Publish/Publish) สามารถพบได้บนเว็บไซต์เอกสารประกอบของเรา
- เนื่องจาก Obsidian ไม่รองรับไฟล์ CSS หรือ JavaScript คุณจึงต้องใช้แอปพลิเคชันอื่นในการสร้างและแก้ไขไฟล์เหล่านี้
- ทั้ง `publish.css` และ `publish.js` ต้องอยู่ในไดเรกทอรีราก (`/`) ของห้องนิรภัยของคุณ
- โดยค่าเริ่มต้น `publish.css` และ `publish.js` จะไม่ปรากฏในตัวสำรวจไฟล์ แต่คุณยังสามารถเผยแพร่ได้จากกล่องโต้ตอบ **เผยแพร่โน้ตล่าสุด...**
- หากต้องการใช้ JavaScript กำหนดเองกับ `publish.js` คุณต้อง[[โดเมนกำหนดเอง]]

สำหรับ favicon นั้น Obsidian Publish รองรับรูปแบบการตั้งชื่อดังต่อไปนี้ โดยที่ `32` หมายถึงขนาดของไอคอนเป็นพิกเซล:

- `favicon-32.png`
- `favicon-32x32.png`
- `favicon.ico`

เราแนะนำให้คุณจัดเตรียมขนาดต่อไปนี้หนึ่งขนาดหรือมากกว่า:

- `favicon-32x32.png`
- `favicon-128x128.png`
- `favicon-152x152.png`
- `favicon-167x167.png`
- `favicon-180x180.png`
- `favicon-192x192.png`
- `favicon-196x196.png`

คุณสามารถวาง favicon ไว้ที่ใดก็ได้ภายในห้องนิรภัย ตราบใดที่มันถูกเผยแพร่ไปยังเว็บไซต์ของคุณ

## ใช้ธีมโดยชุมชน

หากต้องการใช้ธีมโดยชุมชนสำหรับเว็บไซต์ของคุณ:

1. เปิดห้องนิรภัยของคุณในตัวสำรวจไฟล์เริ่มต้นของระบบปฏิบัติการ
2. ไปที่โฟลเดอร์การตั้งค่าของห้องนิรภัย (ค่าเริ่มต้น: `.obsidian`)
3. เปิดโฟลเดอร์ `themes`
4. คัดลอกไฟล์ CSS ของธีมที่คุณต้องการใช้สำหรับเว็บไซต์ของคุณ
5. วางไฟล์ลงในโฟลเดอร์รากของห้องนิรภัยของคุณ
6. เปลี่ยนชื่อไฟล์ CSS เป็น `publish.css`
7. [[เผยแพร่เนื้อหาของคุณ#เผยแพร่โน้ต|เผยแพร่]] `publish.css`

**หมายเหตุ:**

- หากสไตล์ไม่อัปเดตภายในไม่กี่นาที ให้ลองรีเฟรชแคชของเบราว์เซอร์
- คุณสามารถสลับระหว่างโหมดสีอ่อนและสีเข้มได้ใน[[จัดการเว็บไซต์#ดูตัวเลือกเว็บไซต์|ตัวเลือกเว็บไซต์]]
- ธีมโดยชุมชนหลายตัวใช้ **Style Settings** สำหรับการปรับแต่งสไตล์ แต่การตั้งค่าเหล่านี้ไม่ทำงานบน Obsidian Publish

> [!tip] การพัฒนาธีม
> หาธีมที่ถูกใจไม่เจอ? เรียนรู้วิธี [สร้างธีม Publish](https://docs.obsidian.md/Themes/Obsidian+Publish+themes/Build+a+Publish+theme) ด้วยตัวคุณเอง

## เปิดใช้งานฟีเจอร์ UI

คุณสามารถสลับเปิดปิดฟีเจอร์ UI หลายอย่างสำหรับเว็บไซต์ของคุณ เช่น มุมมองกราฟหรือสารบัญ

เลือกดูฟีเจอร์ UI ที่มีให้ใช้งานภายใต้ส่วน **ประสบการณ์การอ่าน** และ **ส่วนประกอบ** ใน[[จัดการเว็บไซต์#ดูตัวเลือกเว็บไซต์|ตัวเลือกเว็บไซต์]]

### ปรับแต่งการนำทาง

ภายใน Obsidian Publish คุณสามารถปรับแต่งลำดับการนำทางและการแสดงผลของไฟล์และโฟลเดอร์ภายใน [[ตัวจัดการไฟล์]] ของ Publish ได้ รายการนำทางจะถูกแสดงตามลำดับที่เผยแพร่โดยค่าเริ่มต้น โน้ตที่ยังไม่ได้เผยแพร่จะไม่ปรากฏภายในหน้าต่างนี้

#### การเข้าถึงตัวเลือกปรับแต่งการนำทาง

1. ใน [[ริบบอน]] เลือก **เผยแพร่โน้ตล่าสุด...** ![[lucide-send.svg#icon]] หรือเปิด [[กระดานคำสั่ง]] แล้วพิมพ์ **Publish: Publish changes...**
2. ในกล่องโต้ตอบ **เผยแพร่โน้ตล่าสุด...** เลือก **เปลี่ยนการตั้งค่าเว็บไซต์** ![[lucide-cog.svg#icon]]
3. ภายใต้ **การตั้งค่าส่วนประกอบ** ถัดจาก **ปรับแต่งการนำทาง** เลือกปุ่ม **จัดการ**

หน้าต่างป๊อปอัพใหม่ชื่อ **การนำทาง** จะปรากฏเหนือหน้าต่าง **เปลี่ยนการตั้งค่าเว็บไซต์** ของคุณ

#### ปรับรายการนำทาง

ในส่วนที่ระบุว่า **ตัวอย่างการนำทาง** คุณสามารถปรับลำดับการแสดงผลของเนื้อหาที่เผยแพร่ได้

1. เลือกโฟลเดอร์หรือโน้ตที่คุณต้องการปรับ
2. ลากโน้ตหรือโฟลเดอร์ขึ้นหรือลงจนอยู่ในตำแหน่งที่คุณต้องการ
3. ที่มุมล่างขวาของหน้าต่าง **การนำทาง** เลือก **เสร็จสิ้น**

Publish จะส่งการเปลี่ยนแปลงการนำทางของคุณไปยังเว็บไซต์

#### ซ่อนและยกเลิกการซ่อนรายการนำทาง

หากมีโน้ตหรือโฟลเดอร์ที่คุณเผยแพร่แล้ว แต่ไม่ต้องการให้แสดงภายในการนำทาง คุณสามารถเลือกที่จะซ่อนรายการเหล่านั้นแทนได้

1. เลือกโฟลเดอร์หรือโน้ตที่คุณต้องการปรับ
2. คลิกขวาแล้วเลือก **ซ่อนในการนำทาง** รายการนั้นควรหายไปจาก **ตัวอย่างการนำทาง**
3. ที่มุมล่างขวาของหน้าต่าง **การนำทาง** เลือก **เสร็จสิ้น**

Publish จะส่งการเปลี่ยนแปลงการนำทางของคุณไปยังเว็บไซต์

> [!tip] คุณสามารถ **แสดงที่ซ่อนอยู่** ได้โดยเลือกกล่องกาเครื่องหมายทางด้านขวาของชื่อ **ตัวอย่างการนำทาง**

## คำถามที่พบบ่อย

**ฉันสามารถย้ายไฟล์จากโฟลเดอร์หนึ่งไปยังอีกโฟลเดอร์หนึ่งภายในการนำทางได้หรือไม่?**

ไม่ได้ โครงสร้างการนำทางไฟล์สำหรับโน้ตภายในโฟลเดอร์จำเป็นต้องคงไว้ตามเดิม คุณสามารถปรับลำดับโน้ตภายในโฟลเดอร์ (รวมถึงรากของห้องนิรภัย) และลำดับโฟลเดอร์ภายในโฟลเดอร์อื่นได้

**ฉันสามารถแก้ไขลำดับของโน้ตและโฟลเดอร์หลายรายการก่อนเลือกเสร็จสิ้นได้หรือไม่?**

ได้

**ฉันจะย้อนกลับการเปลี่ยนแปลงเหล่านี้ได้อย่างไร?**

- **ลำดับการแสดงผล**: เลือกไอคอน **เรียกคืนค่าเริ่มต้น** (ลูกศรหมุนทวนเข็มนาฬิกา) ถัดจาก **ลำดับการแสดงรายการนำทาง** ซึ่งจะคืนค่ารายการนำทางของคุณเป็นลำดับตามตัวอักษร
- **สถานะการซ่อน**: เลือกไอคอน **เรียกคืนค่าเริ่มต้น** (ลูกศรหมุนทวนเข็มนาฬิกา) ถัดจาก **ซ่อนหน้าหรือโฟลเดอร์จากการนำทาง** ซึ่งจะคืนค่ารายการนำทางที่ซ่อนอยู่ให้กลับมาแสดง
