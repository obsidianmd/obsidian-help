หน้านี้อธิบายวิธีจัดการเว็บไซต์ [[แนะนำ Obsidian Publish|Obsidian Publish]]

เว็บไซต์คือคอลเลกชันของโน้ตที่โฮสต์โดย Obsidian Publish และพร้อมใช้งานออนไลน์ ไม่ว่าจะผ่านที่อยู่ Obsidian Publish หรือ[[โดเมนกำหนดเอง]]

## สร้างเว็บไซต์ใหม่

> [!note] จำนวนเว็บไซต์ที่คุณสามารถมีได้พร้อมกันจะถูกกำหนดโดยการสมัครสมาชิก Obsidian Publish ของคุณ ก่อนที่คุณจะสร้างเว็บไซต์ใหม่ ให้แน่ใจว่าการสมัครสมาชิกของคุณอนุญาต

1. ในริบบอน ทางด้านซ้ายของหน้าต่างแอป คลิก **เผยแพร่การเปลี่ยนแปลง** ![[lucide-send.svg#icon]]
2. หากคุณมีเว็บไซต์อยู่แล้ว คลิก **สลับเว็บไซต์** ![[lucide-repeat.svg#icon]]
3. ใน **Site ID** ให้ป้อนเส้นทางที่คุณต้องการสำหรับเว็บไซต์ของคุณ ตัวอย่างเช่น เว็บไซต์ที่มี ID `my-amazing-site` จะใช้งานได้จาก publish.obsidian.md/my-amazing-site
4. คลิก **สร้าง**

## ลบเว็บไซต์ที่มีอยู่

> [!note] โน้ตจะยังคงอยู่ในห้องนิรภัยของคุณแม้หลังจากที่คุณลบเว็บไซต์แล้ว

1. ในริบบอน ทางด้านซ้ายของหน้าต่างแอป คลิก **เผยแพร่การเปลี่ยนแปลง** ![[lucide-send.svg#icon]]
2. คลิก **สลับเว็บไซต์** ![[lucide-repeat.svg#icon]]
3. คลิก **ลบเว็บไซต์** ![[lucide-x.svg#icon]] ทางด้านขวาของเว็บไซต์ที่คุณต้องการลบ
4. คลิก **ลบ** เพื่อยืนยัน

## สลับระหว่างเว็บไซต์

1. ในริบบอน ทางด้านซ้ายของหน้าต่างแอป คลิก **เผยแพร่การเปลี่ยนแปลง** ![[lucide-send.svg#icon]]
2. คลิก **สลับเว็บไซต์** ![[lucide-repeat.svg#icon]]
3. คลิก **เลือก** ทางด้านขวาของเว็บไซต์ที่คุณต้องการสลับไป

## เปลี่ยน Site ID

1. ในริบบอน ทางด้านซ้ายของหน้าต่างแอป คลิก **เผยแพร่การเปลี่ยนแปลง** ![[lucide-send.svg#icon]]
2. คลิก **สลับเว็บไซต์** ![[lucide-repeat.svg#icon]]
3. คลิก **แก้ไข site ID** ![[lucide-edit-3.svg#icon]] ทางด้านขวาของเว็บไซต์ที่คุณต้องการแก้ไข
4. ใน **Site ID** ให้ป้อน ID ใหม่สำหรับเว็บไซต์ของคุณ
5. คลิก **เปลี่ยนแปลง**

## ดูตัวเลือกเว็บไซต์

1. ในริบบอน ทางด้านซ้ายของหน้าต่างแอป คลิก **เผยแพร่การเปลี่ยนแปลง** ![[lucide-send.svg#icon]]
2. ในกล่องโต้ตอบ **เผยแพร่การเปลี่ยนแปลง** คลิก **เปลี่ยนการตั้งค่าเว็บไซต์** ![[lucide-cog.svg#icon]]
