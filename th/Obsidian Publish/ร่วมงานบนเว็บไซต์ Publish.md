เรียนรู้วิธีทำงานร่วมกันบนเว็บไซต์ [[แนะนำ Obsidian Publish|Obsidian Publish]] ของคุณกับผู้ใช้ Obsidian คนอื่นๆ โดยการเพิ่มเพื่อนและเพื่อนร่วมงานของคุณเป็นผู้ร่วมงาน พวกเขาสามารถเผยแพร่การเปลี่ยนแปลงไปยังเว็บไซต์ของคุณได้

เฉพาะเจ้าของเว็บไซต์เท่านั้นที่ต้องมีการสมัครสมาชิก Obsidian Publish ที่ใช้งานอยู่ ผู้ร่วมงานต้องการเพียง [บัญชี Obsidian](https://obsidian.md/account)

> [!warning] ก่อนที่คุณจะเผยแพร่การเปลี่ยนแปลงไปยังเว็บไซต์ที่แชร์ ให้แน่ใจว่าได้ [[#ซิงค์การเปลี่ยนแปลงระหว่างผู้ร่วมงาน|ซิงค์การเปลี่ยนแปลงระหว่างผู้ร่วมงาน]] ก่อน มิฉะนั้น คุณอาจเสี่ยงที่จะเขียนทับการเปลี่ยนแปลงจากผู้ร่วมงานคนอื่น

## เพิ่มผู้ร่วมงานในเว็บไซต์

1. ใน [[ริบบอน]] เลือก **เผยแพร่โน้ตล่าสุด...** ![[lucide-send.svg#icon]] หรือเปิด [[กระดานคำสั่ง]] แล้วพิมพ์ **Publish: Publish changes...**
2. ในกล่องโต้ตอบ **เผยแพร่โน้ตล่าสุด...** คลิก **เปลี่ยนการตั้งค่าเว็บไซต์** ![[lucide-cog.svg#icon]]
3. ถัดจาก **การทำงานร่วมกันของไซต์** เลือก **จัดการ**
4. ใน **เชิญผู้ใช้** ป้อนอีเมลของผู้ร่วมงาน
5. เลือก **เพิ่ม**

## ลบผู้ร่วมงานออกจากเว็บไซต์

1. ใน [[ริบบอน]] เลือก **เผยแพร่โน้ตล่าสุด...** ![[lucide-send.svg#icon]] หรือเปิด [[กระดานคำสั่ง]] แล้วพิมพ์ **Publish: Publish changes...**
2. ในกล่องโต้ตอบ **เผยแพร่โน้ตล่าสุด...** คลิก **เปลี่ยนการตั้งค่าเว็บไซต์** ![[lucide-cog.svg#icon]]
3. ถัดจาก **การทำงานร่วมกันของไซต์** เลือก **จัดการ**
4. ถัดจากผู้ร่วมงานที่คุณต้องการลบ เลือก **นำผู้ใช้ออก** ![[lucide-x.svg#icon]]

## ซิงค์การเปลี่ยนแปลงระหว่างผู้ร่วมงาน

Obsidian Publish ไม่ได้ซิงค์การเปลี่ยนแปลงที่เผยแพร่แล้วระหว่างคลังในเครื่องโดยอัตโนมัติ แทนที่จะเป็นเช่นนั้น ผู้ร่วมงานจำเป็นต้องซิงค์การเปลี่ยนแปลงจากผู้ร่วมงานคนอื่นด้วยตนเอง

เพื่ออัปเดตโน้ตในเครื่องด้วยการเปลี่ยนแปลงจากเว็บไซต์สด:

1. ใน [[ริบบอน]] เลือก **เผยแพร่โน้ตล่าสุด...** ![[lucide-send.svg#icon]] หรือเปิด [[กระดานคำสั่ง]] แล้วพิมพ์ **Publish: Publish changes...**
2. คลิกขวาหรือกดค้างที่การเปลี่ยนแปลงที่คุณต้องการซิงค์ จากนั้นเลือก **ใช้เวอร์ชันสด** **การดำเนินการนี้จะเขียนทับโน้ตในคลังในเครื่องของคุณ**

> [!tip] เราแนะนำให้คุณใช้เครื่องมืออื่นเพื่อซิงค์การเปลี่ยนแปลงระหว่างคลัง เช่น [[แนะนำ Obsidian Sync|Obsidian Sync]] หรือ [git](https://git-scm.com/)

## สิทธิ์การเข้าถึง

ตารางต่อไปนี้แสดงสิทธิ์การเข้าถึงเว็บไซต์ที่มีสำหรับเจ้าของและผู้ร่วมงาน:

| การดำเนินการ                              | ผู้ร่วมงาน | เจ้าของ |
|--------------------------------------------|:----------:|:-------:|
| เผยแพร่หน้าใหม่                            | ✓          | ✓       |
| เผยแพร่การเปลี่ยนแปลงของหน้าที่เผยแพร่แล้ว | ✓          | ✓       |
| ยกเลิกการเผยแพร่หน้า                       | ✓          | ✓       |
| ตั้งค่าตัวเลือกเว็บไซต์                     |            | ✓       |
| จัดการสิทธิ์การเข้าถึง                      |            | ✓       |
