## ข้อกำหนดเบื้องต้น

ในการใช้การวิเคราะห์บนเว็บไซต์ [[แนะนำ Obsidian Publish|Obsidian Publish]] ของคุณ คุณต้องมี[[โดเมนกำหนดเอง]]

> [!important] สำคัญ
> ก่อนที่คุณจะเปิดใช้งานการวิเคราะห์ ตรวจสอบให้แน่ใจว่ากฎหมายและข้อบังคับในท้องถิ่นของคุณอนุญาตให้ติดตามผู้เข้าชมได้ ขึ้นอยู่กับเครื่องมือที่คุณใช้ คุณอาจต้องเพิ่มแบนเนอร์ขอความยินยอมไปยังเว็บไซต์ของคุณ

## Google Analytics

เพื่อเปิดใช้งาน Google Analytics สำหรับเว็บไซต์ Obsidian Publish ของคุณ:

1. ในริบบอน ทางด้านซ้ายของหน้าต่างแอป คลิก **เผยแพร่การเปลี่ยนแปลง** ![[lucide-send.svg#icon]]
2. ในกล่องโต้ตอบ **เผยแพร่การเปลี่ยนแปลง** คลิก **เปลี่ยนการตั้งค่าเว็บไซต์** ![[lucide-cog.svg#icon]]
3. ใน **Tracking code ของ Google Analytics** ให้ป้อน tracking code ของคุณ

หากต้องการใช้ Google Tag Manager แทน Google Analytics ให้ใช้ JavaScript กำหนดเองเพื่อเพิ่มสคริปต์ของคุณเอง

## Plausible, Fathom และบริการวิเคราะห์อื่นๆ

คุณสามารถเพิ่มบริการวิเคราะห์ส่วนใหญ่ไปยังเว็บไซต์ของคุณผ่านไฟล์ [[ปรับแต่งเว็บไซต์ของคุณ|publish.js]] ตรวจสอบให้แน่ใจว่าเปลี่ยน `yourdomain.com` เป็นโดเมนของคุณ และ `src` ของสคริปต์เป็นสคริปต์จากผู้ให้บริการวิเคราะห์ของคุณ

นี่คือตัวอย่างการใช้ [Plausible Analytics](https://plausible.io/)

```js
var analyticsScript = document.createElement('script');
analyticsScript.defer = true;
analyticsScript.setAttribute('data-domain', 'yourdomain.com');
analyticsScript.src = 'https://plausible.io/js/plausible.js';
document.head.appendChild(analyticsScript);
```

 วิธีเดียวกันนี้สามารถใช้สำหรับ [Fathom Analytics](https://usefathom.com/) ได้เช่นกัน โปรดทราบว่ามีการเปลี่ยนจาก `data-domain` เป็น `data-site` — ผู้ให้บริการวิเคราะห์แต่ละรายอาจมีรูปแบบที่แตกต่างกันสำหรับการแทรกสคริปต์

```javascript
var fathom = analyticsScript.createElement('script');
analyticsScript.defer = true;
analyticsScript.setAttribute('data-site', 'yourdomain.com');
analyticsScript.src = 'https://cdn.usefathom.com/script.js';
document.head.appendChild(analyticsScript);
```

## การแก้ไขปัญหา

เพื่อตรวจสอบว่าเว็บไซต์ของคุณใช้บริการวิเคราะห์อยู่หรือไม่ ให้ปิดใช้งานส่วนขยายเบราว์เซอร์ที่บล็อกโฆษณา เช่น uBlock Origin ที่อาจบล็อกสคริปต์ติดตามไม่ให้ทำงาน
