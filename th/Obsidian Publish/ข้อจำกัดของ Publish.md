## ปลั๊กอินโดยชุมชน

Obsidian Publish รองรับ[[ปลั๊กอินโดยชุมชน]]ในระดับน้อย

ปลั๊กอินที่แสดงผลเป็น Markdown ดิบ เช่น ปลั๊กอิน Waypoint สามารถใช้งานร่วมกับ Publish ได้ เพราะไม่ต้องอาศัยแอปในการแสดงผลข้อมูล

ในขณะที่ปลั๊กอินที่ต้องใช้บล็อกโค้ดของปลั๊กอินในการแสดงผล เช่น Dataview หรือ Fantasy Statblocks จะไม่สามารถทำงานใน Publish ได้ตามค่าเริ่มต้น

## กราฟ

Publish มีการปรับแต่งสีเบื้องต้นสำหรับมุมมองกราฟโดยใช้ CSS คุณสามารถแก้ไขสีของโหนดในไฟล์ `publish.css` โดยใช้ [ตัวแปร CSS ของ Graph View](https://docs.obsidian.md/Reference/CSS+variables/Plugins/Graph)

โปรดทราบว่ากราฟที่เผยแพร่แล้วไม่รองรับตัวเลือกการจัดเรียงและการดูแบบครบถ้วนที่มีในแอป[[มุมมองกราฟ]]

## ไฟล์มีเดีย

Obsidian Publish ไม่ได้ปรับแต่งมาเพื่อการสตรีมวิดีโอหรือไฟล์เสียงขนาดใหญ่ เราได้รวมแนวทางปฏิบัติที่ดีบางประการสำหรับการจัดการ[[ไฟล์มีเดีย|ไฟล์มีเดีย]]ของคุณไว้ในเอกสารเหล่านี้

เพื่อปรับปรุงประสบการณ์สำหรับผู้เยี่ยมชม เราแนะนำให้คุณใช้บริการโฮสต์วิดีโอแทน เช่น YouTube หรือ Vimeo

คุณสามารถอัปโหลดไฟล์ขนาด**สูงสุด 50mb** ไปยังเว็บไซต์ Publish ของคุณ ^publish-media-limit

## PDF

บนอุปกรณ์มือถือ แท็บเล็ต และคอมพิวเตอร์ที่มีหน้าจอขนาดเล็ก คุณอาจพบปัญหาที่ PDF ที่ฝังไว้ไม่โหลดหรือแสดงเฉพาะหน้าแรก สาเหตุเกิดจากข้อจำกัดของตัวแสดงผล PDF บนอุปกรณ์มือถือ

สำหรับเนื้อหาที่มุ่งเป้าไปที่ผู้ใช้มือถือ เราแนะนำให้ใส่ลิงก์ไปยัง PDF ที่โฮสต์ภายนอก หรือรวมลิงก์ภายในที่ให้ผู้ใช้ดาวน์โหลด PDF โดยตรงไปยังอุปกรณ์ของพวกเขา

## ค้นหา

Publish รองรับการค้นหาข้อความธรรมดาในเนื้อหาที่เผยแพร่แล้วในระดับพื้นฐาน ลำดับความสำคัญของผลการค้นหาจะให้กับ:

- ชื่อไฟล์
- นามแฝง
- ชื่อหัวเรื่อง

หลังจากค้นหารายการข้างต้นเพื่อหาที่ตรงกันแล้ว การค้นหาจะรวมข้อความธรรมดาของโน้ตที่เผยแพร่แล้วด้วย

เพื่อปรับปรุงความสามารถในการค้นหาของเว็บไซต์ที่เผยแพร่ แนะนำให้ใช้ชื่อไฟล์ที่สื่อความหมาย ใส่นามแฝงหลายรายการ และเลือกชื่อหัวเรื่องที่สะท้อนเนื้อหาได้อย่างถูกต้อง

นอกจากนี้ Publish ยังไม่รองรับคุณสมบัติ[[ค้นหา#ฝังผลการค้นหาในโน้ต|ผลการค้นหาที่ฝังไว้]]ของแอปในขณะนี้
