Die Catalyst-Lizenz ist ein einmaliger Kauf, der dir frühzeitig Zugang zu Beta-Versionen gibt und die kontinuierliche Entwicklung von Obsidian unterstützt. Du kannst Obsidian unterstützen, indem du [eine Catalyst-Lizenz erwirbst](https://obsidian.md/buy/catalyst).

Zu den Vorteilen gehören:

- Frühzeitiger Zugang zu [[#Vorabversionen aktivieren|neuen Versionen]] von Obsidian
- Zugang zu exklusiven [[#Unseren Insider-Kanälen beitreten|Insider-Kanälen]].
- Exklusive [[#Foren-Badge erhalten|Foren-Badges]] und [[#Discord-Badge erhalten|Discord-Badges]].

## Catalyst-Stufen

Es gibt drei Catalyst-Stufen, basierend auf der Höhe deines Beitrags: *Insider*, *Supporter* und *VIP*.

| Vorteile                                 | Insider | Supporter |   VIP    |
| ---------------------------------------- | :-----: | :-------: | :------: |
| Frühzeitiger Zugang zu neuen Versionen   |    ✓    |     ✓     |    ✓     |
| Zugang zu exklusiven Entwicklungskanälen |    ✓    |     ✓     |    ✓     |
| Zugang zum exklusiven Lounge-Kanal       |         |     ✓     |    ✓     |
| Insider-Badge                            |    ✓    |           |          |
| Supporter-Badge                          |         |     ✓     |          |
| VIP-Badge                                |         |           |    ✓     |
| **Beitrag (USD)**                        | **$25** |  **$50**  | **$100** |


> [!info] Catalyst-Stufe upgraden
> Wenn du bereits eine Catalyst-Lizenz besitzt, kannst du auf eine höhere Catalyst-Stufe upgraden, indem du die Preisdifferenz zwischen deiner aktuellen und der neuen Stufe zahlst.

## Auf Catalyst-Vorteile zugreifen

Sobald du eine Catalyst-Lizenz erworben hast, kannst du auf eine Reihe von Vorteilen zugreifen. Erfahre hier, wie du sie aktivierst.

### Foren-Badge erhalten

**Hinweis:** Um dein Obsidian-Konto und dein Forum-Konto zu verknüpfen, musst du für beide Konten dieselbe E-Mail-Adresse verwenden.

1. Falls noch nicht geschehen, tritt unserem [Forum](https://forum.obsidian.md/) bei.
2. Gehe zu deinem [Obsidian-Konto-Dashboard](https://obsidian.md/account/catalyst).
3. Wähle unter **Catalyst** die Option **Get forum badge**.

### Discord-Badge erhalten

1. Falls noch nicht geschehen, tritt unserem [Discord-Server](https://discord.gg/veuWUTm) bei.
2. Gehe zu deinem [Obsidian-Konto-Dashboard](https://obsidian.md/account/catalyst).
3. Wähle unter **Catalyst** die Option **Get Discord badge**.

### Unseren Insider-Kanälen beitreten

Insider können mit dem Obsidian-Team und anderen Insidern in unseren exklusiven Discord-Kanälen chatten. Du trittst den Insider-Kanälen automatisch bei, nachdem du deinen [[#Discord-Badge erhalten|Discord-Badge erhalten]] hast.

- [vip-lounge](https://discord.com/channels/686053708261228577/714234422379151400) für Diskussionen mit Catalyst-Mitgliedern.
- [insider-desktop-release](https://discord.com/channels/686053708261228577/716028884885307432) für die neuesten Vorabversionen auf dem Desktop.
- [insider-mobile-release](https://discord.com/channels/686053708261228577/817515900349448202) für die neuesten Vorabversionen auf Mobilgeräten.
