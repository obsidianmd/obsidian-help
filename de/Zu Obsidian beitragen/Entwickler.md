Wenn du Entwickler bist, kannst du auf verschiedene Weisen zu Obsidian beitragen.

### Erweiterungen und Themen

Du kannst [[Community-Erweiterungen]] und [[Themen]] mit Typescript und CSS entwickeln. Erfahre mehr in der [Obsidian-Entwicklerdokumentation](https://docs.obsidian.md).

### Open-Source-Projekte

Obsidian entwickelt und pflegt mehrere Open-Source-Projekte. Du kannst Pull Requests mit Fehlerbehebungen und Verbesserungen an die folgenden GitHub-Repositories einreichen:

- [JSON Canvas](https://github.com/obsidianmd/jsoncanvas) (MIT), ein offenes Dateiformat für unendliche Canvas-Daten.
- [Obsidian Importer](https://github.com/obsidianmd/obsidian-importer) (MIT), eine Erweiterung, die Daten aus einer Vielzahl von Formaten in Markdown-Klartextdateien konvertiert.
- [Obsidian Maps](https://github.com/obsidianmd/obsidian-maps) (MIT), eine Erweiterung, die Karten in Obsidian Bases anzeigt.
- [Obsidian Web Clipper](https://github.com/obsidianmd/obsidian-clipper) (MIT), eine Browsererweiterung zum Hervorheben und Erfassen von Webseiten als Markdown-Klartextdateien.
- [Defuddle](https://github.com/kepano/defuddle) (MIT), der HTML-zu-Markdown-Parser, der von Obsidian Web Clipper verwendet wird.

### Dokumentation

Um unsere [Entwicklerdokumentation](https://docs.obsidian.md/Home) zu verbessern, reiche einen Pull Request im Repository [obsidian-developer-docs](https://github.com/obsidianmd/obsidian-developer-docs) ein.

Um diese Dokumentation zu verbessern, reiche einen Pull Request im Repository [obsidian-help](https://github.com/obsidianmd/obsidian-help) ein.

### Community

Der Kanal [plugin-dev](https://discord.com/channels/686053708261228577/840286264964022302) auf unserem offiziellen Discord-Server ist der Ort, an dem du über die Entwicklung von Erweiterungen diskutieren und anderen Entwicklern bei ihren Erweiterungen helfen kannst.
