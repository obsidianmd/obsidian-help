Diese Seite listet die Standard-Tastaturkürzel für die Navigation und Bearbeitung von Text in Obsidian auf. Diese Kürzel werden von deinem Betriebssystem oder dem Framework, auf dem Obsidian basiert, bereitgestellt und können innerhalb von Obsidian nicht angepasst werden.

Für anpassbare Tastaturkürzel für Obsidian-Befehle, siehe [[Tastenkürzel]].

## Windows- und Linux-Kürzel

### Allgemeine Aktionen

| Aktion | Kürzel |
|-|-|
| Kopieren | `Ctrl+C` |
| Ausschneiden | `Ctrl+X` |
| Einfügen | `Ctrl+V` |
| Einfügen ohne Formatierung | `Ctrl+Shift+V` |
| Rückgängig | `Ctrl+Z` |
| Wiederholen | `Ctrl+Shift+Z` oder `Ctrl+Y` |
| Absatz kopieren | `Ctrl+C` (ohne ausgewählten Text) |
| Absatz ausschneiden | `Ctrl+X` (ohne ausgewählten Text) |

### Textbearbeitung

| Aktion | Kürzel |
|-|-|
| Neue Zeile einfügen | `Enter` |
| Vorheriges Zeichen löschen | `Backspace` |
| Nächstes Zeichen löschen | `Delete` |
| Vorheriges Wort löschen | `Ctrl+Backspace` |
| Nächstes Wort löschen | `Ctrl+Delete` |
| Aktuelle Zeile löschen | `Ctrl+Shift+K` (ohne ausgewählten Text) |

### Textnavigation

| Aktion | Kürzel |
|-|-|
| Cursor um ein Zeichen bewegen | `Left/→` |
| Cursor zum Anfang des vorherigen Wortes bewegen | `Ctrl+←` |
| Cursor zum Ende des nächsten Wortes bewegen | `Ctrl+→` |
| Cursor zum Anfang der aktuellen Zeile bewegen | `Home` |
| Cursor zum Ende der aktuellen Zeile bewegen | `End` |
| Cursor zur vorherigen Zeile bewegen | `↑` |
| Cursor zur nächsten Zeile bewegen | `↓` |
| Cursor zum Anfang der Notiz bewegen | `Ctrl+Home` |
| Cursor zum Ende der Notiz bewegen | `Ctrl+End` |
| Cursor eine Seite nach oben bewegen | `Page up` |
| Cursor eine Seite nach unten bewegen | `Page down` |

### Textauswahl

| Aktion | Kürzel |
|-|-|
| Auswahl vereinfachen | `Escape` |
| Alles auswählen | `Ctrl+A` |
| Auswahl um ein Zeichen erweitern | `Shift+Left/→` |
| Auswahl bis zum Anfang des vorherigen Wortes erweitern | `Ctrl+Shift+←` |
| Auswahl bis zum Ende des nächsten Wortes erweitern | `Ctrl+Shift+→` |
| Auswahl bis zum Anfang der aktuellen Zeile erweitern | `Shift+Home` |
| Auswahl bis zum Ende der aktuellen Zeile erweitern | `Shift+End` |
| Auswahl bis zum Anfang der Notiz erweitern | `Ctrl+Shift+Home` |
| Auswahl bis zum Ende der Notiz erweitern | `Ctrl+Shift+End` |
| Auswahl eine Seite nach oben erweitern | `Shift+Page up` |
| Auswahl eine Seite nach unten erweitern | `Shift+Page down` |

## macOS-Kürzel

### Allgemeine Aktionen

| Aktion | Kürzel |
|-|-|
| Kopieren | `Cmd+C` |
| Ausschneiden | `Cmd+X` |
| Einfügen | `Cmd+V` |
| Einfügen ohne Formatierung | `Cmd+Shift+V` |
| Rückgängig | `Cmd+Z` |
| Wiederholen | `Cmd+Shift+Z` |
| Absatz kopieren | `Cmd+C` (ohne ausgewählten Text) |
| Absatz ausschneiden | `Cmd+X` (ohne ausgewählten Text) |

### Textformatierung

| Aktion | Kürzel |
| ---- | ---- |
| Fetter Text | `Cmd+B` |
| Kursiver Text | `Cmd+I` |

### Textbearbeitung

| Aktion | Kürzel |
|-|-|
| Neue Zeile einfügen | `Enter` |
| Vorheriges Zeichen löschen | `Backspace` |
| Nächstes Zeichen löschen | `Delete` |
| Vorheriges Wort löschen | `Option+Backspace` |
| Nächstes Wort löschen | `Option+Delete` |
| Bis zum Anfang der aktuellen Zeile löschen | `Cmd+Backspace` |
| Bis zum Ende der aktuellen Zeile löschen | `Cmd+Delete` |
| Aktuelle Zeile löschen | `Cmd+Shift+K` (ohne ausgewählten Text) |

### Textnavigation

| Aktion | Kürzel |
|-|-|
| Cursor um ein Zeichen bewegen | `Left/→` |
| Cursor zum Anfang des vorherigen Wortes bewegen | `Option+←` |
| Cursor zum Ende des nächsten Wortes bewegen | `Option+→` |
| Cursor zum Anfang der aktuellen Zeile bewegen | `Cmd+←` |
| Cursor zum Ende der aktuellen Zeile bewegen | `Cmd+→` |
| Cursor zur vorherigen Zeile bewegen | `↑` |
| Cursor zur nächsten Zeile bewegen | `↓` |
| Cursor zum Anfang der Notiz bewegen | `Cmd+↑` |
| Cursor zum Ende der Notiz bewegen | `Cmd+↓` |
| Cursor eine Seite nach oben bewegen | `Fn+↑` |
| Cursor eine Seite nach unten bewegen | `Fn+↓` |

| Auswahl bis zum Ende des nächsten Wortes erweitern        | `Option+Shift+→` |
