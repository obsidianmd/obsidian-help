Dieses Glossar enthält gängige Obsidian-Terminologie.

## Alias

Ein **Alias** ist eine Art von [[#Eigenschaft]], die alternative Namen für eine [[#Notiz]] definiert.

## Anhang

Ein **Anhang** ist ein [[Akzeptierte Dateiformate|akzeptiertes Dateiformat]], das außerhalb des Vaults erstellt und später hinzugefügt wurde.

## Befehl

Ein **Befehl** ist eine Aktion, die entweder durch Auswahl aus der [[Befehlspalette]] oder durch Zuweisen eines [[#Tastenkürzel|Tastenkürzels]] ausgeführt werden kann.

## Einbetten

**Einbetten** bedeutet, eine Referenz auf externen Inhalt durch den Inhalt selbst zu ersetzen, zum Beispiel um ein Bild in eine Notiz einzufügen. Siehe auch [[Dateien einbetten]].

## Frontmatter

Frontmatter ist eine Möglichkeit, [[#Eigenschaft|Eigenschaften]] zu definieren, indem [YAML](https://yaml.org/) oder [JSON](https://www.json.org/) am Anfang der Notiz hinzugefügt wird. Siehe auch [[Eigenschaften#Eigenschaftsformat|Eigenschaftsformat]].

## Graph

Ein **Graph** ist eine Visualisierung, die Verbindungen zwischen [[#Notiz|Notizen]] hervorhebt. Siehe auch [[Graph-Ansicht]].

## Tastenkürzel

Ein **Tastenkürzel** ist eine Tastenkombination für einen [[#Befehl]]. Siehe auch [[Tastenkürzel]] und [[Tastenkürzel|Anleitung/Tastenkürzel verwenden]].

## Link

Ein **Link** verweist auf eine andere Notiz oder Datei. Ein [[Interne Links|interner Link]] zeigt auf eine Datei innerhalb des aktuellen Vaults. Ein [[Grundlegende Formatierungssyntax#Externe Links|externer Link]] zeigt auf einen Ort außerhalb des Vaults, typischerweise eine Webseite.

## Hauptbereich

Der **Hauptbereich** ist der zentrale Bereich der Obsidian-App, in dem du hauptsächlich [[#Notiz|Notizen]] bearbeitest.

## Markdown

Markdown ist eine Auszeichnungssprache zur Textformatierung und das primäre Dateiformat für Notizen in Obsidian, `.md`-Dateien. Siehe auch [[Grundlegende Formatierungssyntax]].

## Notiz

Eine **Notiz** ist eine Markdown-Datei innerhalb eines [[#Vault|Vaults]].

## Erweiterung

Eine **Erweiterung** erweitert Obsidian um zusätzliche Funktionen.

- [[Obsidian-Erweiterungen]] werden vom Obsidian-Team geschrieben und sind standardmäßig enthalten.
- [[Community-Erweiterungen]] werden von Drittanbieter-Entwicklern geschrieben und müssen [[Community-Erweiterungen#Eine Community-Erweiterung installieren|installiert]] werden, bevor sie verwendet werden können.

Du kannst [deine eigene Erweiterung erstellen](https://docs.obsidian.md/Plugins/Getting+started/Build+a+plugin).

## Pop-out-Fenster

Standardmäßig werden alle Notizen innerhalb eines Vaults im selben App-Fenster geöffnet. Ein **Pop-out-Fenster** ermöglicht es, Notizen aus demselben Vault in separaten Fenstern zu öffnen, zum Beispiel zur Anzeige auf einem zweiten Bildschirm.

Siehe auch [[Pop-out-Fenster]].

## Eigenschaft

[[Eigenschaften|Eigenschaften]] definieren zusätzliche Informationen über eine Notiz, wie ein Fälligkeitsdatum oder einen Autor.

## Werkzeugleiste

Die **Werkzeugleiste** dient als Container für häufig verwendete Aktionssymbole.

In der Desktop-Version ist sie der vertikale Bereich ganz links.

In der mobilen Version wird sie durch eine Menü-Schaltfläche ![[lucide-menu.svg#icon]] in der [[#Statusleiste]] dargestellt.

## Seitenleiste

Ein Bereich, der unterstützende [[#Ansicht|Ansichten]] enthält, die als [[#Tab|Tabs]] organisiert sind. Eine Seitenleiste kann in mehrere [[#Tab-Gruppe|Tab-Gruppen]] aufgeteilt werden.

Obsidian Desktop hat zwei Seitenleisten, eine auf jeder Seite des [[#Hauptbereich|Hauptbereichs]]. Beide Seitenleisten können über Symbole in der oberen linken und oberen rechten Ecke der Obsidian-App erreicht werden, zusätzlich zum Wischen nach links oder rechts. Das Symbol oben rechts muss gehalten werden, um das Fenster zu öffnen.

## Snippet

Ein **Snippet**, oder [[CSS-Bausteine|CSS-Snippet]], verändert das Erscheinungsbild von Obsidian, genau wie ein [[#Thema]]. Im Gegensatz zu Themen können mehrere Snippets gleichzeitig angewendet werden.

## Statusleiste

Die **Statusleiste** in der Obsidian-Anwendung zeigt wichtige Statistiken und Status an. In der Desktop-Version befindet sie sich in der unteren rechten Ecke, während sie auf mobilen Geräten am unteren Rand der App positioniert ist.


## Tab

Ein **Tab** enthält eine [[#Ansicht]]. Tabs können innerhalb des [[#Hauptbereich|Hauptbereichs]] und der [[#Seitenleiste|Seitenleisten]] neu positioniert werden. Siehe auch [[Tabs]].

## Tab-Gruppe

Eine **Tab-Gruppe** ist eine Sammlung von [[#Tab|Tabs]] im [[#Hauptbereich]]. Tabs in einer Tab-Gruppe können gestapelt werden.

## Tag

Ein **Tag** ist ein Wort, das mit einem Rautezeichen (`#`) beginnt, zum Beispiel `#buch`. Tags werden hauptsächlich verwendet, um verwandte [[#Notiz|Notizen]] zu finden.

## Thema

Ein **Thema** verändert das Erscheinungsbild der Obsidian-App mithilfe von [CSS](https://developer.mozilla.org/en-US/docs/Web/CSS). Du kannst Teile eines Themas mit [[#Snippet|Snippets]] überschreiben.

## Vault

`Aliasse: Lokaler Vault, lokale Daten`

Ein **Vault** ist ein Ordner in deinem Dateisystem, der [[#Notiz|Notizen]] und einen `.obsidian`-Ordner mit Obsidian-spezifischer Konfiguration enthält. Siehe auch [[Wie Obsidian Daten speichert]].

### Remote-Tresor

`Aliasse: Remote-Daten`

Ein [[Lokale und Remote-Tresore|Remote-Tresor]] ist eine Kopie deines lokalen Vaults, die mit [[Einführung in Obsidian Sync|Obsidian Sync]] gepflegt wird. Die Daten des Remote-Tresors werden basierend auf Änderungen an den lokalen Daten aktualisiert.

## Ansicht

Eine **Ansicht** zeigt Informationen an, zum Beispiel die [[Suchen|Suchansicht]].
