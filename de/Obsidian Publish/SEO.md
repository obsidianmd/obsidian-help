[[Einführung in Obsidian Publish|Obsidian Publish]]-Websites werden automatisch für Suchmaschinen und [[Linkvorschau für soziale Medien|Social-Sharing-Karten]] optimiert. Sie können Seitenmetadaten auch mit Beschreibungen, Permalinks und Bildern anpassen.

Beachten Sie, dass Sie die Indizierung durch Suchmaschinen in Ihren [[Websites verwalten#Website-Einstellungen|Website-Einstellungen]] verbieten können.

### Google Search Console

Um SEO mit Google zu verfolgen, registrieren Sie Ihre Website bei der [Google Search Console](https://search.google.com/search-console). Sie können auch unseren empfohlenen Schritten folgen, um [[Analytik|Google Analytics einzurichten]].

### Sitemap

Die Sitemap für Ihre Publish-Website ist unter `/sitemap.xml` verfügbar, zum Beispiel:
https://help.obsidian.md/sitemap.xml

Die Sitemap ist auch als RSS-Feed unter `/rss.xml` verfügbar:
https://help.obsidian.md/rss.xml

### Permalinks

Erwägen Sie die Verwendung von [[Permalinks]], um permanente URLs auf Ihrer Website zu definieren.

### Metadaten

Seitenmetadaten können mithilfe von [[Eigenschaften#Eigenschaften für Obsidian Publish|Eigenschaften]] angepasst werden, was für [[Linkvorschau für soziale Medien|Social-Sharing-Karten]] nützlich sein kann.

| Eigenschaft   | Beschreibung                                                                                                      |
| ------------- | ----------------------------------------------------------------------------------------------------------------- |
| `publish`     | Siehe [[Inhalte veröffentlichen#Notizen automatisch zum Veröffentlichen auswählen\|Notizen automatisch zum Veröffentlichen auswählen]]. |
| `permalink`   | Siehe [[Permalinks\|Permalinks]].                                                                                 |
| `description` | Siehe [[Linkvorschau für soziale Medien#Beschreibung\|Beschreibung]].                                             |
| `image`       | Siehe [[Linkvorschau für soziale Medien#Bild\|Bild]].                                                             |
| `cover`       | Siehe [[Linkvorschau für soziale Medien#Bild\|Bild]].                                                             |
