[[Einführung in Obsidian Web Clipper|Web Clipper]] enthält einen integrierten Lesemodus namens **Leser**, der störende Elemente von Webseiten entfernt und nur den Hauptartikelinhalt in einem übersichtlichen, lesbaren Format darstellt.

## Lesemodus aktivieren

Der Leser kann auf verschiedene Weisen aktiviert werden:

- ![[lucide-book-icon.svg#icon]] **Buchsymbol** im Erweiterungspanel.
- **Tastenkürzel**, anpassbar in den Erweiterungseinstellungen.
- **Kontextmenü** durch Rechtsklick auf die besuchte Webseite.

Du kannst auch die ![[obsidian.svg#icon]] Erweiterungsschaltfläche so einstellen, dass der Leser mit einem Klick geöffnet wird. Gehe in den Web Clipper-Einstellungen zu **Allgemein** → **Verhalten** → **Öffnungsverhalten** und wähle **Leser**.

Alle oben genannten Optionen ermöglichen es auch, den Leser zu beenden und die Originalseite wiederherzustellen.

## Funktionsweise

Der Leser extrahiert den Hauptinhalt der Seite. Die Originalseite wird durch eine übersichtliche Leseansicht ersetzt, die Folgendes enthält:

- Artikeltitel, Autor, Veröffentlichungsdatum und Quell-Domain.
- Den Hauptartikelinhalt mit erhaltenen Bildern und Formatierungen.
- Eine Gliederung in der Seitenleiste für Artikel mit mehreren Überschriften.
- Syntaxhervorhebung für Quelltext-Blöcke.
- Fußnoten-Popover, die beim Anklicken inline angezeigt werden.

## Leseansicht anpassen

Passe dein Leser-Erlebnis in den Erweiterungseinstellungen an. Einige Optionen sind auch im Lesemodus verfügbar, indem du auf das ![[lucide-case-sensitive.svg#icon]] Symbol klickst.

| Einstellung        | Beschreibung                                        |
| ------------------ | --------------------------------------------------- |
| Schriftart         | Auf dem System installierte Schriftarten hinzufügen. |
| Schriftgröße       | Textgröße anpassen.                                 |
| Zeilenhöhe         | Abstand zwischen den Zeilen.                        |
| Zeilenbreite       | Maximale Breite des Inhaltsbereichs.                |
| Erscheinungsbild   | Helles oder dunkles Erscheinungsbild wählen.        |
| Thema              | Farben auswählen.                                   |
| Benutzerdefiniertes&nbsp;CSS | Zusätzliches CSS, das auf die Seite angewendet wird. |
